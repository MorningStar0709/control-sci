## 关键观察

1. **快速收敛**：训练在 100 步附近 loss 骤降（0.9004 → 0.5322），对应 token 准确率从 76% 跳升至 85%
2. **Eval loss 基本稳定**：从 0.894 → 0.890，2 个 epoch 的微调未出现明显过拟合
3. **Token 准确率显著提升**：+15pp，说明模型对控制科学领域特定术语和公式格式的生成能力有实质改善
4. **PPL 改善**：9B 实测 Base PPL=6.48 → QLoRA PPL=4.0, Δ=-38.3%（89 题, 62s）

## 评测结果（⚠️ 反直觉发现）

Phase1 生成（89 题，140min，GPU 100%）+ Phase2 DeepSeek 评分（89 题，4.5min，API 19.9q/m）已完成。

| | **overall** | **A** | **B** | **C** | **D** |
|---|---|---|---|---|---|---|
| QLoRA 9B | **0.4635** | 0.3632 | 0.4840 | 0.6316 | 0.3942 |
| 裸 9B 基线 | 0.6249 | 0.5688 | 0.6104 | 0.6620 | 0.6586 |
| Δ | **-0.1614** | -0.2056 | -0.1264 | -0.0304 | -0.2644 |

### 反直觉发现：领域微调 ≠ 评测提升

- 9B QLoRA 评测 **overall 从 0.6249 跌至 0.4635**，降幅 25.8%
- 与 4B QLoRA 微调一致：4B baseline 0.467 → finetuned 0.407，同样下降
- **A 维退化最严重**（-0.2056，57.9% 零分），C 维几乎无损（-0.0304）
- 26 题零分，3 种典型失败模式：公式错误（~40%）、答案截断（~15%）、计算错误（~35%）
- 参考：4B QLoRA 的 PPL 从 8.4 降至 3.9（-53.6%），9B QLoRA 从 6.48 降至 4.0（-38.3%）。两个尺寸均出现评测退化但 PPL 改善，跨尺寸一致
- C 维幸存原因：跨域泛化题预期答案是开放式，对公式精确度要求低，4-bit 量化对语义理解损失小于对精确公式的记忆损失

**这是有价值的诚实发现**：领域数据注入能改善模型对控制科学语言的拟合度（4B PPL -53.6%, 9B PPL -38.3%，跨尺寸一致），但不能直接转化为评测任务表现——甚至可能因过拟合训练分布而削弱泛化能力。这一发现可作为"技术创新中的诚实发现"写入报告，展示严谨的实验态度。
