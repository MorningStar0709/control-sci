## §4.5.3 API vs 本地模型对比

以 8 个 API 模型（DeepSeek-v4-flash/pro、MiMo-v2-flash/v2.5-pro/v2-pro/v2.5、MiniMax-M2.5-highspeed/M2.7-highspeed）的均值作为 API 基线，与本地部署的 Qwen3.5-9B 进行对比：

| 指标 | API 模型平均 | Qwen3.5-9B（本地） | 差距 |
|------|:-----------:|:------------------:|:----:|
| 总分 | 57.1 | 62.5 | **+5.4** |
| A 维（概念回溯） | 61.9 | 56.9 | **-5.0** |
| B 维（多步推理） | 53.9 | 61.0 | **+7.1** |
| C 维（条件敏感性） | 62.7 | 66.2 | **+3.5** |
| D 维（开放设计） | 49.7 | 65.9 | **+16.2** |

### 对比分析

Qwen3.5-9B 在总分上领先 API 均值 5.4 分，这一结果出人意料——一个 9B 参数的本地模型在科学推理任务上超越了多数商业 API 模型。分维来看：

1. **D 维差距最大（+16.2）**：Qwen3.5-9B 在开放设计维度上的优势最为显著，65.9 分位居全场第二，仅次于 MiMo-v2-flash（73.6 分），远超 API 均值 49.7 分。这表明 Qwen3.5 系列在需要进行综合工程设计的开放性问题上具有突出的能力。

2. **A 维逆差（-5.0）**：Qwen3.5-9B 在 A 维（概念回溯）上落后 API 均值 5.0 分，是唯一一个低于 API 均值的维度。这可能与 Qwen3.5 的训练数据中对控制科学基础概念的覆盖深度有关。

3. **B/C 维领先（+7.1/+3.5）**：在需要多步推理和条件敏感性分析的维度上，Qwen3.5-9B 均显著优于 API 均值，证明其推理链路质量稳定。

**核心启示**：本地模型在科学推理任务上并非必然落后于商业 API。Qwen3.5-9B 以 1/4 到 1/10 的 API 调用成本取得了更强的综合表现，尤其适合对推理质量和 D 维设计能力要求高的科学评测场景。
