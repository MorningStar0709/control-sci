# D.7 Doubling Trick to Remove the Knowledge of T in Algorithm 2

Following the same arguments in Appendix C.7, we divide the total horizon T into consecutive intervals with length $L _ { i } = T _ { i } - T _ { i - 1 }$ , where $T _ { i } = 2 ^ { 2 ^ { i } }$ with $T _ { - 1 } = 0$ , apply Algorithm 2 to each interval, and sum the resulting regret over all the intervals. Moreover, we have

$$
\begin{array}{l} \mathsf {R} _ {\mathbf {x}} (T) = \sum_ {i = 0} ^ {M} \mathsf {R} _ {\mathbf {x}} (L _ {i}) \\ = \sum_ {i = 0} ^ {M} \mathcal {O} \big (\tau (\log^ {4} L _ {i}) \log (L _ {i} / \tau) + \sqrt {L _ {i}} \log^ {2} L _ {i} \big) \\ \leq \sum_ {i = 0} ^ {M} \left(\mathcal {O} (\tau \log (T / \tau) \log^ {4} L _ {i}) + \mathcal {O} (\sqrt {T _ {i}} \log^ {2} T _ {i})\right) \\ \stackrel {(a)} {\leq} \mathcal {O} (\tau \log (T / \tau) \log^ {4} T) + \mathcal {O} (\sqrt {T}) \sum_ {i = 0} ^ {M} \mathcal {O} (2 ^ {2 i}) \\ = \mathcal {O} (\tau \log (T / \tau) \log^ {4} T) + \mathcal {O} (\sqrt {T}) \mathcal {O} (2 ^ {2 \log \log T}) \\ \stackrel {(b)} {=} \mathcal {O} \big (\tau \log (T / \tau) \log^ {4} T + \sqrt {T} \log^ {2} T \big), \\ \end{array}
$$

where (a) follows from the arguments in Appendix C.7 and (b) uses the fact $M = \mathcal { O } ( \log \log T )$ .
