# A.2 Systems with Discontinuous Right-Hand-Side

This section provides some equivalent results for non-smooth systems.

Consider the following differential equation with a discontinuous righthand side:

$$\dot {x} = h (x, t), \tag {A.4}$$

where $h : \Omega \times [ t _ { 0 } , \infty ) \to \mathbb { R } ^ { n } , \Omega \subset \mathbb { R } ^ { n }$ , is Lebesgue measurable and locally essentially bounded, uniformly in t. The Filippov regularization of $f$ is defined as [377]

$$\mathsf {K} [ f ] (x, t) := \bigcap_ {\delta > 0} \bigcap_ {\mu (\bar {N}) = 0} \overline {{\mathrm{co}}} (f (\mathcal {B} (x, \delta) \backslash \bar {N}), t), \tag {A.5}$$

where $\bigcap _ { \mu ( \bar { N } ) = 0 }$ is the intersection over all sets $\bar { N }$ of Lebesgue measure zero, and ${ \overline { { \operatorname { c o } } } } ( E )$ is the convex closure of a set E. We are interested in the Filippov solutions of (A.4):

Definition A.6 ([378]). A function x $\mathbf { \Psi } : \left[ t _ { 0 } , t _ { 1 } \right) \to \mathbb { R } ^ { n }$ , with $t _ { 1 } > t _ { 0 }$ , is called a Filippov solution of (A.4) on $[ t _ { 0 } , t _ { 1 } )$ if x(t) is absolutely continuous and if, for almost all $t \in [ t _ { 0 } , t _ { 1 } )$ , it satisfies ${ \dot { x } } \in \mathsf { K } [ h ] ( x , t )$ , where $\mathsf { K } [ h ] ( x , t )$ is the Filippov regularization of $h ( x , t )$ .

The existence of Filippov solutions is given next.

Proposition A.1 ([379]). Let ${ \dot { x } } \in \mathsf { K } [ h ] ( x , t )$ , where $\mathsf { K } [ h ] ( x , t )$ is the Filippov regularization of $h ( x , t )$ . Let also $h ( x , t )$ be measurable and locally essentially bounded in x over Ω, and measurable in t over $[ t _ { 0 } , \infty )$ . Then, there exists a Fillipov solution $x : [ t _ { 0 } , t _ { 1 } ) \to \mathbb { R } ^ { n }$ of (A.4).

We next provide the definitions for regular functions and generalized gradients.

Definition A.7 ([378]). Given a function $h : \mathbb { R } ^ { m }  \mathbb { R } ^ { n }$ , the right directional derivative of h at $x \in \mathbb { R } ^ { m }$ in the direction of $v \in \mathbb { R } ^ { m }$ is defined as

$$f ^ {\prime} (x, v) := \lim _ {t \to 0 ^ {+}} \frac {f (x + t v) - f (x)}{t}.$$

Additionally, the generalized directional derivative of h at x in the direction of v is defined as

$$f ^ {o} (x, v) := \lim _ {y \to x} \sup _ {t \to 0 ^ {+}} \frac {f (x + t v) - f (y)}{t}.$$
