# C. Proof of Corollary 1

By Assumption 1, Lemma 1 ensures that the pair $( F +$ $L H , G )$ is controllable. As a consequence, there exist $\Omega =$ $\Omega ^ { \top } \succ 0$ and K such that

$$
\Omega \left(F + L \Theta^ {\star} \left[ \begin{array}{c} 0 _ {n \times \mu} \\ I _ {\mu} \end{array} \right] + G K\right) + \left(F + L \Theta^ {\star} \left[ \begin{array}{c} 0 _ {n \times \mu} \\ I _ {\mu} \end{array} \right] + G K\right) ^ {\top} \Omega \preceq - I _ {\mu}. \tag {72}
$$

From (72), Ω and K ensure stabilization for all Θ such that

$$
\Omega L (\Theta - \Theta^ {\star}) \left[ \begin{array}{c} 0 _ {n \times \mu} \\ I _ {\mu} \end{array} \right] + \left[ \begin{array}{l l} 0 _ {\mu \times n} & I _ {\mu} \end{array} \right] (\Theta - \Theta^ {\star}) ^ {\top} (\Omega L) ^ {\top} \preceq \ell I _ {\mu}, \tag {73}
$$

for some $\ell \in ( 0 , 1 )$ . Applying the norms, (73) holds if

$$\left| \Theta - \Theta^ {\star} \right| \leq \frac {\ell}{2 | \Omega L |}. \tag {74}$$

Define

$$\rho^ {\star} := \frac {\ell^ {2}}{1 6 | \Omega L | ^ {2}}. \tag {75}$$

Then, if the data satisfy Assumptions 2 and 3 with $\rho \in$ $[ 0 , \rho ^ { \star } ]$ , the bound (45) implies that (73) holds, thus from the inclusions ${ \pmb { \mathscr { E } } } \subseteq \hat { \pmb { \mathscr { E } } } \subseteq { \pmb { \mathscr { C } } } ( \Omega ^ { - 1 } , K )$ we obtain the statement by invoking Theorem 1. □
