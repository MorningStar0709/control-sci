# 3.2 DeepMartNet for optimal control $u ^ { * }$ and value function $v ( t , { \bf x } )$

Based on the martingale principle theorem for the optimal feedback control, we can extend DeepMartNet to approximate the optimal control by a neural network

$$u _ {t} (\omega) = u _ {t} (\mathbf {X} (\omega)) \sim u _ {\theta_ {1}} (t, \mathbf {X} (\omega)), \tag {3.12}$$

where $u _ { \theta _ { 1 } } ( t , \mathbf { x } ) \in C ( [ 0 , T ] \times R ^ { d } \ )$ will be a neural network approximation for a $d { + 1 }$ dimensional function with network parameters $\theta _ { 1 }$ , and the value function by another network

$$v (t, \mathbf {x}) \sim v _ {\theta_ {2}} (t, \mathbf {x}). \tag {3.13}$$

The loss function will consist of two parts, one for the control network and one for the value network

$$l (\theta_ {1}, \theta_ {2}) = l _ {c t r} (\theta_ {1}) + l _ {v a l} (\theta_ {2})$$

where, similar to (2.16),

$$l _ {c t r} (\theta_ {1}) = l _ {c t r, \mathbf {x} _ {0}} (\theta_ {1})= \frac {1}{N} \sum_ {i = 0} ^ {N - 1} \frac {1}{\left| A _ {i} \right|} \sum_ {m = 1} ^ {\left| A _ {i} \right|} \left(c (X _ {t _ {i}}, t _ {i}, u _ {\theta_ {1}} (t _ {i}, \mathbf {X} _ {i} ^ {(m)})) \Delta t _ {i} + v _ {\theta_ {2}} (t _ {i + 1}, \mathbf {X} _ {i + 1} ^ {(m)}) - v _ {\theta_ {2}} (t _ {i}, \mathbf {X} _ {i} ^ {(m)})\right) ^ {2} \tag {3.14}$$

and, by using Ito formula for $v _ { \theta _ { 2 } } ( t , \mathbf { x } )$ , we can obtain a similar Martingale form for the HJB equation (3.11) and define a similar loss function for the value function $v ( t , \mathbf { x } )$ as in (2.16)

$$
\begin{array}{l} l _ {v a l} (\theta_ {2}) = l _ {v a l, \mathbf {x} _ {0}} (\theta_ {2}) \\ = \frac {1}{N} \sum_ {i = 0} ^ {N - 1} \left(\frac {1}{| A _ {i} |} \sum_ {m = 1} ^ {| A _ {i} |} \binom{v _ {\theta_ {2}} (t _ {i + 1}, \mathbf {X} _ {i + 1} ^ {(m)}) - v _ {\theta_ {2}} (t _ {i}, \mathbf {X} _ {i} ^ {(m)}) +}{H (t _ {i}, \mathbf {X} _ {i} ^ {(m)}, \nabla_ {x} v _ {\theta_ {2}} (t _ {i}, \mathbf {X} _ {i} ^ {(m)}) \sigma (t, \mathbf {X} _ {i} ^ {(m)})) \Delta t _ {i}}\right) ^ {2} \tag {3.15} \\ + \beta \frac {1}{M} \sum_ {m = 1} ^ {M} (v _ {\theta_ {2}} (T, \mathbf {X} _ {N} ^ {(m)}) - \xi (\mathbf {X} _ {N} ^ {(m)})) ^ {2}. \\ \end{array}
$$

Again, for better accuracy globally for the control and value networks, we can define a global loss function with more sampling of the starting points $\begin{array} { r } { \mathbf { x } _ { 0 } ^ { ( k ) } , 1 \le k \le K , } \end{array}$ ,

$$R (\theta_ {1}, \theta_ {2}) = \frac {1}{K} \sum_ {k = 1} ^ {K} \left(l _ {c t r, \mathbf {x} _ {0} ^ {(k)}} (\theta_ {1}) + l _ {v a l, \mathbf {x} _ {0} ^ {(k)}} (\theta_ {2})\right). \tag {3.16}$$

The above approach requires an accurate result for the value function $v ( t _ { i } , \pmb { \chi } _ { i } ^ { ( m ) } )$ in the region explored by the process $\mathbf { \pmb { X } } _ { i } ^ { ( m ) }$ , this could pose a challenge to the DeepMartNet. An alternative approach is to use FBSDE based learning algorithm in [11], which has been shown to be able to meet this requirement.
