$$
\begin{array}{l} \left\| \Sigma_ {t} - \Sigma \right\| = \left\| (A - L C) ^ {t - 1} \right\| \left\| W - \Sigma \right\| \left\| \Psi_ {t, 1} ^ {\top} \right\| \\ \leq \kappa_ {F} \gamma_ {F} ^ {t - 1} \| W - \Sigma \| \sqrt {\frac {\| W + L V L ^ {\top} \| \kappa_ {F} ^ {2}}{\lambda_ {\min} (W) (1 - \gamma_ {F} ^ {2})}}, \\ \end{array}
$$

which proves (E.3).

Finally, we prove (E.4). From (3) and (9), one can show that (see $\mathrm { { e . g . \ [ 3 0 ] ) } }$ for any $t \geq 0$

$$
\begin{array}{l} L _ {t} - L = A \Sigma_ {t} C ^ {\top} (C \Sigma_ {t} C ^ {\top} + V) ^ {- 1} - A \Sigma C ^ {\top} (C \Sigma C ^ {\top} + V) ^ {- 1} \\ = A \Sigma C ^ {\top} \Big ((C \Sigma_ {t} C ^ {\top} + V) ^ {- 1} - (C \Sigma C ^ {\top} + V) ^ {- 1} \Big) + A (\Sigma_ {t} - \Sigma) C ^ {\top} (C \Sigma_ {t} C ^ {\top} + V) ^ {- 1} \\ = A \Sigma C ^ {\top} (C \Sigma C ^ {\top} + V) ^ {- \top} C (\Sigma - \Sigma_ {t}) C ^ {\top} (C \Sigma_ {t} C ^ {\top}) ^ {- 1} - A (\Sigma - \Sigma_ {t}) C ^ {\top} (C \Sigma C ^ {\top} + V) ^ {- 1} \\ = (L C - A) (\Sigma - \Sigma_ {t}) C ^ {\top} (C \Sigma_ {t} C ^ {\top} + V) ^ {- 1}. \\ \end{array}
$$

It follows from (E.3) and Lemma E.4 that for any $t \geq 1$ ,

$$
\begin{array}{l} \left\| L _ {t} - L \right\| \leq \left\| A - L C \right\| \left\| \Sigma_ {t} - \Sigma \right\| \left\| C \right\| \left\| (C \Sigma_ {t} C ^ {\top} + V) ^ {- 1} \right\| \\ \leq \kappa_ {F} ^ {3} \gamma_ {F} ^ {t} \| W - \Sigma \| \sqrt {\frac {\| W + L V L ^ {\top} \|}{\lambda_ {\min} (W) (1 - \gamma_ {F} ^ {2})}} \cdot \frac {\| C \|}{\lambda_ {\min} (V)}, \\ \end{array}
$$

which proves (E.4).
