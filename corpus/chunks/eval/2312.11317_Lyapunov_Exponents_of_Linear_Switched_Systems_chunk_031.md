# 6 Discussion of cases

We end up with some final remarks about which is the optimal strategy depending on the values $\mathrm { t r } ( A ^ { 2 } ) , \mathrm { t r } ( B ^ { 2 } ) , \mathrm { t r } ( A B )$ . As before, we always assume $\operatorname { t r } ( A ^ { 2 } ) \geq \operatorname { t r } ( B ^ { 2 } )$ .

So far, we have obtained that all possible extremals satisfying PMP are in one of the following form:

• constant control;   
• constant singular control;   
• if $\operatorname { t r } ( A ^ { 2 } ) < 0$ , bang-bang periodic control as in Subsection 4.4.1.

As pointed out in Subsection 4.4.2, there is also this other kind of extremals accumulating at singular extremals. However, we saw that their associated principal Lyapunov exponent is at most strictly less then the one associated to singular extremal, so we can omit them from the discussion.

First, assume that $\operatorname { t r } ( A ^ { 2 } ) \geq 0$ . In this case we already saw in Section 5 that if $\operatorname { t r } ( A B ) \geq \operatorname { t r } ( A ^ { 2 } )$ , then the value of the Principal Lyapunov exponent given by the constant singular control is bigger than the one given by the constant control. So, the first two point of the Main Theorem are proved.

We turn now to the $\operatorname { t r } ( A ^ { 2 } ) < 0$ case. We have the following restriction on the possible values of $\mathrm { t r } ( A B )$

Lemma 6.1. $I f \operatorname { t r } ( B ^ { 2 } ) \leq \operatorname { t r } ( A ^ { 2 } ) < 0$ , then $\lvert \mathrm { t r } ( A B ) \rvert \geq \sqrt { \mathrm { t r } ( A ^ { 2 } ) \mathrm { t r } ( B ^ { 2 } ) }$ . In particular $\operatorname { t r } ( [ A , B ] ^ { 2 } ) \geq 0$

Proof. Up to a change of base, we can suppose

$$
A = \left( \begin{array}{c c} 0 & - \lambda \\ \lambda & 0 \end{array} \right), \quad B = P \left( \begin{array}{c c} 0 & - \mu \\ \mu & 0 \end{array} \right) P ^ {- 1}, \quad P = \left( \begin{array}{c c} a & b \\ c & d \end{array} \right) \in \mathrm{SL} _ {2} (\mathbb {R}), \quad \lambda , \mu \in \mathbb {R}.
$$

Since A in this form commutes with rotations, that is matrix of the form

$$
R _ {\theta} = \left( \begin{array}{c c} \cos \theta & - \sin \theta \\ \sin \theta & \cos \theta \end{array} \right), \quad \theta \in [ 0, 2 \pi ],
$$

we can use another change of base given by a rotation matrix to kill one element of P . That is, we can further assume that P is in the form

$$
P = \left( \begin{array}{c c} \alpha & \beta \\ 0 & \frac {1}{\alpha} \end{array} \right), \quad \alpha , \beta \in \mathbb {R}, \quad \alpha \neq 0.
$$
