# B. Proof of Theorem 1

Let $( \hat { U } _ { [ 0 , M - 1 ] } , \hat { X } _ { [ 0 , M - 1 ] } , \hat { Y } _ { [ 0 , M - 1 ] } )$ be an input-stateoutput spectrum of Σ satisfying Assumption 1. Suppose that $\hat { U } _ { [ 0 , M - 1 ] }$ is such that $\hat { V } _ { [ 0 , M - 1 ] } = \bar { \{ ( \hat { U } _ { k } , \hat { \Omega } _ { k } ) \} } _ { k \in \mathbb { Z } _ { [ 0 , M - 1 ] } }$ with $\hat { \Omega } _ { k } = e ^ { \dot { j } \hat { \omega } _ { k } ^ { M } }$ for all $k \in \mathbb { Z } _ { [ 0 , M - 1 ] }$ , is PE of order $L + n _ { x }$ . Note that, due to Assumption 1, Σ˜ is also controllable. By Lemma $2 . ( \mathrm { i } ) , ( \hat { V } _ { [ 0 , M - 1 ] } , \hat { X } _ { [ 0 , M - 1 ] } , \hat { Y } _ { [ 0 , M - 1 ] } )$ is a steadystate input-state-output spectrum of Σ˜ .

(i): Since $\hat { V } _ { [ 0 , M - 1 ] }$ is PE of order $L + n _ { x } .$ it follows from Lemma 1.(i) that

$$
\left[ \begin{array}{c} \Psi_ {1} (\hat {X} _ {[ 0, M - 1 ]}) \\ \Psi_ {L} (\hat {V} _ {[ 0, M - 1 ]}) \end{array} \right]
$$

has full row rank. Rearranging the rows yields Theorem 1.(i).

(ii): It follows from Lemma 1.(ii) that the pair of trajectories $( v _ { [ 0 , L - 1 ] } , y _ { [ 0 , L - 1 ] } )$ is an input-output trajectory of $\tilde { \Sigma }$ , if and only if there exist $G _ { 0 } \in \mathbb { R }$ and $G _ { 1 } \doteq \mathbb { C } ^ { \check { M } - 1 }$ such that

$$
\left[ \begin{array}{c} v _ {[ 0, L - 1 ]} \\ \hdashline y _ {[ 0, L - 1 ]} \end{array} \right] = \left[ \begin{array}{c c} F _ {L} (\hat {V} _ {[ 0, M - 1 ]}) & F _ {L} ^ {*} (\hat {V} _ {[ 1, M - 1 ]}) \\ \hdashline F _ {L} (\hat {Y} _ {[ 0, M - 1 ]}) & F _ {L} ^ {*} (\hat {Y} _ {[ 1, M - 1 ]}) \end{array} \right] \left[ \begin{array}{c} G _ {0} \\ G _ {1} \\ \hdashline G _ {1} ^ {*} \end{array} \right].
$$

Using Lemma 2.(ii) and by rearranging rows, we obtain Theorem 1.(ii). □
