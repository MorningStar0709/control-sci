# Appendix E. PROOF OF PROPOSITION 7

By Lemma 6, the condition (22) holds if and only if

$$T _ {2} (b _ {\mathrm{ini}} - V _ {2} \tilde {b} _ {\mathrm{ini}}) \bmod q = T _ {2} b _ {\mathrm{ini}} - \tilde {b} _ {\mathrm{ini}} \bmod q = 0 \tag {E.1}b _ {v} (t) - \Sigma^ {\dagger} \tilde {b} _ {v} (t) \bmod q = - \Sigma^ {\dagger} \Psi b _ {\xi} (t) \tag {E.2}+ (I - \Sigma^ {\dagger} \Sigma) b _ {\mu} (t) \bmod q, \forall t \geq 0,$$

for some $b _ { \mu } ( t ) \in \mathbb { Z } _ { q } ^ { m + p }$ . It is immediate from (E.1) that $\tilde { b } _ { \mathrm { i n i } } = T _ { 2 } b _ { \mathrm { i n i } }$ mod q. Moreover, multiplying Σ on both sides of (E.2) and using $\Sigma \Sigma ^ { \dagger } = 1$ gives

$$\Sigma b _ {v} (t) - \tilde {b} _ {v} (t) \bmod q = - \Psi b _ {\xi} (t) \bmod q,$$

which is equivalent to (24). In this case, the internal dynamics of $b _ { \xi } ( t )$ is given by

$$
\begin{array}{l} b _ {\xi} (t + 1) = S _ {1} b _ {\xi} (t) + S _ {3} \left(b _ {v} (t) - \Sigma^ {\dagger} \tilde {b} _ {v} (t)\right) \text {   mod   } q \\ = S b _ {\xi} (t) + S _ {3} (I - \Sigma^ {\dagger} \Sigma) b _ {v} (t) \bmod q, \\ \end{array}
$$

with $b _ { \xi } ( 0 ) = T _ { 1 } b _ { \mathsf { i n i } }$ , and this concludes the proof.
