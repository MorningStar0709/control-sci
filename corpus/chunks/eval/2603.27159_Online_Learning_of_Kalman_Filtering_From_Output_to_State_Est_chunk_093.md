Moreover, using similar arguments to those for (C.21), one can show that $| \hat { x } _ { t } ^ { \mathrm { K F } } | = \mathcal { O } ( \sqrt { \log T } )$ for all $t \in \{ 2 s _ { 0 } , \ldots , T - 1 \}$ , under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ , where s0 is given by (C.13) and satisfies $s _ { 0 } = \mathcal { O } ( \log T )$ . Combining the above arguments, we see that under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v } , | \hat { x } _ { t } ^ { \mathrm { K F } } | = \mathcal { O } ( \log ^ { 1 . 5 } T )$ holds for all $t \in \{ 0 , \ldots , T - 1 \}$ . Applying Lemma D.4, we then get from (D.14) that under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ ,

$$\left| \mathbb {E} \big [ (\hat {x} _ {t} - x _ {t}) ^ {2} - (\hat {x} _ {t} ^ {\mathrm{KF}} - x _ {t}) ^ {2} \big ] - \big ((\hat {x} _ {t} - x _ {t}) ^ {2} - (\hat {x} _ {t} ^ {\mathrm{KF}} - x _ {t}) ^ {2} \big) \right|\leq 2 c _ {\mathcal {A}} + 2 5 0 \sigma_ {w} \log \frac {3 T}{\delta} + 2 5 0 \sigma_ {w} (1 + \log T) + \mathcal {O} (\log^ {1. 5} T)= \mathcal {O} (c _ {\mathcal {A}} + \log^ {1. 5} T).$$

One can now apply Lemma E.3 and deduce that under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ , the following holds with probability at least $1 - \delta / 3$ :

$$\mathbb {E} \left[ \mathrm{R} _ {\mathrm{x}} (T) \right] - \mathrm{R} _ {\mathrm{x}} (T)= \sum_ {t = 0} ^ {T - 1} \left(\mathbb {E} \left[ (\hat {x} _ {t} - x _ {t}) ^ {2} - (\hat {x} _ {t} ^ {\mathrm{KF}} - x _ {t}) ^ {2} \right] - \left((\hat {x} _ {t} - x _ {t}) ^ {2} - (\hat {x} _ {t} ^ {\mathrm{KF}} - x _ {t}) ^ {2}\right)\right)\leq \sqrt {2 T \mathcal {O} (c _ {\mathcal {A}} + \log^ {1 . 5} T) \log \frac {3}{\delta}}.$$

We now obtain from a union bound and (36) that the following holds with probability at least $1 - \delta$

$$\mathbb {R} _ {\mathrm{x}} (T) \geq 6 T \sigma_ {w} - \frac {5 6 + 5 0 \sigma_ {w}}{1 8 7 5 \sigma_ {w}} - \mathcal {O} \left(\sqrt {(c _ {\mathcal {A}} + \log^ {1 . 5} T) T \log \frac {3}{\delta}}\right).$$
