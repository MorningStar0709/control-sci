# Concluding remarks

The present contribution considers the stabilization of orbits employing an adaptive fuzzy sliding mode controller. The stability and convergence properties of the closed-loop systems is proven using Lyapunov stability theory and Barbalat’s lemma. As an application of the general formulation, numerical simulations of a nonlinear pendulum with chaotic response is of concern. The control system performance is investigated showing for the tracking of a generic orbit as well as for UPO stabilization. It is shown that the controller needs less effort to stabilize an UPO.
