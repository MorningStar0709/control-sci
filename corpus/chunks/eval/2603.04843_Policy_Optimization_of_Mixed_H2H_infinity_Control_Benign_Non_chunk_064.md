# C.2.1 Proof of the Diffeomorphism from ${ \mathcal { L } } _ { \mathrm { l f t } }$ to $\mathcal { F } _ { \mathrm { c v x } }$

We first provide some auxiliary notations. Denote

$$\mathcal {L} _ {\mathrm{lft}} ^ {0} = \left\{\left(K, \gamma , X\right) \mid K \in \mathrm{R} ^ {m \times n}, \gamma \in \mathrm{R}, X \succ 0 \right\},\mathcal {F} _ {\mathrm{cvx}} ^ {0} = \left\{\left(\gamma , X, Y\right) \mid \gamma \in \mathrm{R}, X \succ 0, Y \in \mathrm{R} ^ {m \times n} \right\}.$$

Evidently, $\mathcal { L } _ { \mathrm { l f t } } ^ { 0 }$ and $\mathcal { F } _ { \mathrm { c v x } } ^ { 0 }$ can be identified with certain open subsets of some Euclidean spaces, and $\mathcal { F } _ { \mathrm { c v x } } ^ { 0 }$ is convex. Also,

$$
\mathcal {L} _ {\mathrm{lft}} = \left\{(K, \gamma , X) \in \mathcal {L} _ {\mathrm{lft}} ^ {0} \left| \begin{array}{c} \gamma \geq \operatorname{tr} (Q _ {2} X) + \operatorname{tr} (R _ {2} K X K ^ {\top}), \\ A _ {K} X + X A _ {K} ^ {\top} + W + \beta^ {- 2} X S _ {K} X \preceq 0 \end{array} \right. \right\}, \tag {C.1}

\mathcal {F} _ {\mathrm{cvx}} = \left\{(\gamma , X, Y) \in \mathcal {F} _ {\mathrm{cvx}} ^ {0} \left| \begin{array}{c} \gamma \geq \operatorname{tr} (Q _ {2} X) + \operatorname{tr} (R _ {2} Y X ^ {- 1} Y ^ {\top}), \\ \mathrm{F} _ {\mathrm{cvx}} \preceq 0 \end{array} \right. \right\}. \tag {C.2}
$$

We can now extend the domnow defined on the larger set $\mathcal { L } _ { \mathrm { l f t } } ^ { 0 }$ of Φ in (25c) so that it is:

$$\Phi (K, \gamma , X) = (\gamma , X, K X), \quad \forall (K, \gamma , X) \in \mathcal {L} _ {\mathrm{lft}} ^ {0}. \tag {C.3}$$

Our subsequent proof consists of showing the following:   
(1) Φ is a $C ^ { 2 }$ diffeomorphism from ${ \mathcal { L } } _ { { \sf l f f } } ^ { 0 }$ to $\mathcal { F } _ { \mathrm { l f t } } ^ { 0 }$   
(2) For any $( K , \gamma , X ) \in \mathcal { \bar { L } } _ { \mathfrak { l f t } } ^ { 0 }$ , we have $( \ddot { K } , \gamma , \dot { X } ) \in \mathcal { L } _ { \mathrm { l f t } }$ if and only $\mathrm { i f } \ \Phi ( K , \gamma , X ) \in \overleftarrow { \mathcal { F } } _ { \mathrm { c v } \mathbf { x } }$ .

Part I. Consider Φ in (C.3) and define the mapping

$$\Psi (\gamma , X, Y) = (Y X ^ {- 1}, \gamma , X), \quad \forall (\gamma , X, Y) \in \mathcal {F} _ {\mathrm{cvx}} ^ {0}. \tag {C.4}$$
