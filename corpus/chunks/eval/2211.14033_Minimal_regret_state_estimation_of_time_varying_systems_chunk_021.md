Table 1. Relative difference in average estimation costs with ${ \mathcal { Q } } = I ,$ , for nine difference patterns. The performance for the stochastic patterns (i.e. the first three) is averaged over 1000 realizations. The best estimator is marked by 1 and sets the baseline for the relative percentage increase.

Table 1 shows that, as expected, $\mathcal { H } _ { 2 }$ and $\mathcal { H } _ { \infty }$ observers give the best performance for Gaussian and worst-case disturbances, respectively. However, observe that, unlike in the work of Sabag and Hassibi (2021), the error of the R observer is not between that of $\mathcal { H } _ { 2 }$ and $\mathcal { H } _ { \infty }$ . For all other disturbance patterns, the R observer achieves a significant improvement compared to the other two.

Note that the difference between the costs of $\mathcal { H } _ { 2 }$ and $\mathcal { H } _ { \infty }$ observers is relatively small compared to the difference between both costs and the one of the R observer. It shows that, for some systems, this new observer can provide a radically different alternative to the two state-of-the-art state estimation methods.
