This command will open the Simulink library browser; Fig. C.1 shows a “screen shot” of the browser. In order to create a new Simulink model, the user must left-mouse click on the “new model” icon as highlighted in the upper left corner of Fig. C.1. Doing so will create a blank screen or working template for constructing the desired block diagram.
