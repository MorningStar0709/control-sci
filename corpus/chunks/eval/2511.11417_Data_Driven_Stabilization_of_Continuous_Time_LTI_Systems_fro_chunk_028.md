# B. Observer Canonical Form

Let $\mathrm { e } _ { i } \in \mathbb { R } ^ { n } , i \in \{ 1 , \dots , n \}$ , be the i-th component of the standard basis of $\mathbb { R } ^ { n }$ and define:

$$
\bar {A} := \left[ \begin{array}{c c} 0 _ {1 \times (n - 1)} & 0 \\ I _ {n - 1} & 0 _ {(n - 1) \times 1} \end{array} \right]. \tag {62}
$$

We observe that A in (3) can be written as

$$A = \bar {A} \otimes I _ {p} + A _ {\mathrm{m}} C = \bar {A} \otimes I _ {p} + A _ {\mathrm{m}} (\mathrm{e} _ {n} ^ {\top} \otimes I _ {p}), \tag {63}$$

where $A _ { \mathrm { m } } : = - [ A _ { 0 } ^ { \top } ~ . . . ~ A _ { n - 1 } ^ { \top } ] ^ { \top }$ . Consider the following permutation matrix:

$$
\Phi := \left[ \begin{array}{l l l l} I _ {p} \otimes \mathrm{e} _ {1} & I _ {p} \otimes \mathrm{e} _ {2} & \dots & I _ {p} \otimes \mathrm{e} _ {n} \end{array} \right] \in \mathbb {R} ^ {n p \times n p}. \tag {64}
$$

Some computations show that $\Phi ^ { \top } ( I _ { p } \otimes \bar { A } ) \Phi = \bar { A } \otimes I _ { p }$ and $( I _ { p } \otimes \mathbf { e } _ { n } ^ { \top } ) \bar { \Phi } = \mathbf { e } _ { n } ^ { \top } \otimes I _ { p } = C .$ , which lead to the observer canonical form:

$$
\begin{array}{l} A _ {0} := \Phi A \Phi^ {\top} = I _ {p} \otimes \bar {A} + \tilde {A} _ {\mathrm{m}} \left(I _ {p} \otimes \mathrm{e} _ {n} ^ {\top}\right) \\ G = G I ^ {\top} - I _ {n} \otimes \bar {I}. \end{array} \tag {65}
C _ {0} := C \Phi^ {\top} = I _ {p} \otimes \mathrm{e} _ {n} ^ {\top},$$

with ${ \tilde { A } } _ { \mathrm { m } } : = \Phi A _ { \mathrm { m } }$
