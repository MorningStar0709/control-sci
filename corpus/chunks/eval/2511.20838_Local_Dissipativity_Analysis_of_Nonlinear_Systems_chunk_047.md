$$
\mathcal {G} _ {\Omega} (x _ {0}): \left\{ \begin{array}{l l} \dot {x} _ {1} & = x _ {2} \\ \dot {x} _ {2} & = - \sin x _ {1} - x _ {2} + u \\ y & = x _ {2}, \end{array} \right. \tag {28}
$$

where $x \in \mathbb { R } ^ { n } , u \in \mathbb { R }$ , and $y \in \mathbb { R }$ . Let $\mathcal { X } \supseteq \Omega$ be the triangulated region about the origin, shown in Figure 3, where the RPI set Ω is found using [17, Theorem 5], where the system is able to have inputs of |u| $< 0 . 1 9 4 \dot { 2 }$ . Problem 2 was applied to the system – solving for the $\mathcal { L } _ { 2 }$ gain specifically. Because the pendulum has a sinusoidal term, SOS techniques [25] cannot be used for analysis. Instead, we compare to the gain found using the global analytical Lyapunov function $\begin{array} { r } { V ( x ) = ( 1 - \cos x _ { 1 } ) \breve { + } \frac { 1 } { 2 } x _ { 2 } ^ { 2 } } \end{array}$ , which results in the analytical gain bound $\gamma \geq 1$ . Figure 2 shows the gain found on Ω for an increasing number of simplices, where the best result is $\gamma = 2 . 2 2 1$ Figure 4 shows an example of a storage function found across Ω with 652 simplices.
