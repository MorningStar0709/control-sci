# B. Proof of Lemma 5

The proof can be done similarly to Theorem 1 and Theorem 2 in [15]. Before proving Lemma 5, we give the preliminary lemma, which is the nonconvex version of Lemma 5.

Lemma 10: Let

$$
\begin{array}{l} \hat {\mathcal {K}} _ {\mathcal {S}} = \left\{K = (E ^ {\top} E) ^ {- 1} E ^ {\top} \tilde {Z} \tilde {Q} ^ {- 1} E \right. \\ = (E ^ {\top} E) ^ {- 1} \sum_ {k = 1} ^ {q} E _ {\mathcal {C} _ {k}} ^ {\top} (\tilde {Z} _ {k} \tilde {Q} _ {k} ^ {- 1}) E _ {\mathcal {C} _ {k}}: \\ \exists \tilde {Z} = \operatorname{blkdiag} (\tilde {Z} _ {1}, \dots \tilde {Z} _ {q}), \\ \tilde {Q} = \operatorname{blkdiag} \left(\tilde {Q} _ {1}, \dots \tilde {Q} _ {q}\right) \succ O, \rho \in \mathbb {R} \\ \end{array}
\text { s.t. } \Phi (\tilde {Q}, \tilde {Z}) + \rho \tilde {R} W \tilde {R} \succ O \} \tag {38}$$

with W in (9),

$$
\Phi (\tilde {Q}, \tilde {Z}) = \left[ \begin{array}{c c} \tilde {Q} & * \\ \tilde {A} \tilde {Q} + \tilde {B} \tilde {Z} & \tilde {Q} \end{array} \right] \tag {39}
$$

and

$$\tilde {R} = \text { blkdiag } (\tilde {Q}, \tilde {Q}). \tag {40}$$

Then, $\hat { \mathcal { K } } _ { S } \subset \mathcal { K } _ { \mathrm { a l l } }$ holds.

Proof: Let $\tilde { K } = \tilde { Z } \tilde { Q } ^ { - 1 }$ . For $( E ^ { \top } E ) ^ { - 1 } E ^ { \top } \tilde { Z } \tilde { Q } ^ { - 1 } E \ \in$ $\hat { \mathcal { K } } _ { S }$ , it can be seen that

$$
\begin{array}{l} O \prec \tilde {R} ^ {- 1} (\Phi (\tilde {Q}, \tilde {Z}) + \rho \tilde {R} M \tilde {R}) \tilde {R} ^ {- 1} \\ = \left[ \begin{array}{c c} \tilde {Q} ^ {- 1} & * \\ \tilde {Q} ^ {- 1} (\tilde {A} + \tilde {B} \tilde {K}) & \tilde {Q} ^ {- 1} \end{array} \right] + \rho M \\ \end{array}
$$

From Finsler’s lemma, by setting $\boldsymbol { K } = ( E ^ { \top } E ) ^ { - 1 } E ^ { \top }$ KE˜ and $P = E ^ { \top } \tilde { Q } ^ { - 1 } E$ , we get the following inequality:

$$
\begin{array}{l} O \prec \left[ \begin{array}{c c} E ^ {\top} & O \\ O & E ^ {\top} \end{array} \right] \left[ \begin{array}{c c} \tilde {Q} ^ {- 1} & * \\ \tilde {Q} ^ {- 1} (\tilde {A} + \tilde {B} \tilde {K}) & \tilde {Q} ^ {- 1} \end{array} \right] \left[ \begin{array}{c c} E & O \\ O & E \end{array} \right] \\ = \left[ \begin{array}{c c} P & * \\ P (A + B K) & P \end{array} \right] = \left[ \begin{array}{c c} P & * \\ A + B K & P ^ {- 1} \end{array} \right] \\ \end{array}
$$

Here, Lemma 3 guarantees $K \in S .$ , thus $\hat { \mathcal { K } } _ { S } \subset \mathcal { K } _ { \mathrm { a l l } }$

Finally, we prove Lemma 5. This is a convexification of Lemma 10.
