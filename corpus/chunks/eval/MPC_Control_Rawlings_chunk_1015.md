Theorem C.33 (Continuity of $U ( \cdot ) )$ . Suppose $\mathcal { Z }$ is a polyhedron in $\mathbb { R } ^ { n } \times \mathbb { U }$ where $\mathbb { U } \subset \mathbb { R } ^ { m }$ is a polytope. Then the set-valued function $U : \mathbb { R } ^ { n }  \mathbb { R } ^ { m }$ defined above is continuous in X.

Proof. By Proposition C.27, the set-valued map $U ( \cdot )$ is outer semicontinuous in X because its graph, Z, is closed. We establish inner semicontinuity using Corollary C.32 above. Let $x , x ^ { \prime }$ be arbitrary points in X and $U ( x )$ and $U ( x ^ { \prime } )$ the associated control constraint sets. Let S be any open set such that $U ( x ) \cap S \neq \emptyset$ and let u be an arbitrary point in $U ( x ) \cap S$ . Because S is open, there exist an $\varepsilon > 0$ such that u ⊕ $\varepsilon { \mathcal { B } } \subset S$ . Let $\varepsilon ^ { \prime } : = \varepsilon / K$ where K is defined in Corollary 1. From Corollary C.32, there exists a $u ^ { \prime } \in U ( x ^ { \prime } )$ such that $| u ^ { \prime } - u | \leq K | x ^ { \prime } - x |$ which implies $| u ^ { \prime } - u | \leq \varepsilon \left( u ^ { \prime } \in u \oplus \varepsilon \mathcal { B } \right)$ for all $x ^ { \prime } \in \mathcal { X }$ such that $| x ^ { \prime } - x | \leq \varepsilon ^ { \prime }$ $( x ^ { \prime } \in ( x \oplus \varepsilon ^ { \prime } { \mathcal { B } } ) \cap { \mathcal { X } } )$ . This implies $u \in \mathcal { U } ( x ^ { \prime } ) \cap S$ for all $x ^ { \prime } \in \mathcal { X }$ such that $| x ^ { \prime } - x | \leq \varepsilon ^ { \prime } \left( x ^ { \prime } \in \left( x \oplus \varepsilon ^ { \prime } \mathcal { B } \right) \cap \mathcal { X } \right)$ . Hence $\mathcal { U } ( x ^ { \prime } ) \cap S \neq \emptyset$ for all $x ^ { \prime } \in ( x \oplus \varepsilon ^ { \prime } { \mathcal { B } } ) \cap \chi$ , so that $\mathcal { U } ( \cdot )$ is inner semicontinuous in X. Since $U ( \cdot )$ is both outer and inner semicontinuous in X, it is continuous in X. 
