# 6.1 Direct Data-Driven Control

In this subsection, we aim to verify the effectiveness of direct data-driven control approach in Theorem 1. The dictionary of observables is selected as

$$z = \Psi (x) = [ x _ {1} x _ {2} \sin (x _ {1}) x _ {2} \cos (x _ {1}) ] ^ {\top}. \tag {71}$$

We collect the data generated under zero and unit vectors, leading to the dataset specified in (48). The data samples are uniformly sampled from the state space $\mathbb { X } =$ $[ - 2 , \bar { 1 0 } ] ^ { 2 }$ under process disturbance given in (70) with $\bar { \delta } \ = \ 0 . 0 1$ , where the total number of data points is $T = 1 0 0 0$ . Additionally, the matrix $W _ { 0 }$ is verified to be of full rank. The approximation error $r ( z , u )$ is bounded by (11) with $c _ { 1 } = c _ { 2 } = 0 . 1$ .

The negative definiteness condition for matrix (31) translates to an LMI, which is solved using the YALMIP toolbox [16] in MATLAB. To ensure the LMI is solvable with the widely used SDP solver MOSEK, we adopt a line search procedure [2], fixing the scalar variable $\nu ,$ solving the LMI, and iterating on the selection of ν if necessary. Accordingly, a feasible controller

$$u _ {\text { direct }} = \left[ - 2. 1 0 6 1 - 3. 4 6 3 5 - 1 3. 4 2 6 4 - 3. 0 4 8 0 \right] \Psi (x) \tag {72}$$

is obtained with the positive scalars $\alpha = 3 , \beta = 3 / 2 .$ , c = 5, $\nu = 1 0 , \ \tau = 0 . 0 2 6 7 , \ \mu = 4 6 . 2 2$ . As shown in $\mathrm { F i g . 1 }$ , closed-loop trajectories are demonstrated to be convergent.
