• $\operatorname { I f } \Gamma _ { 1 } ^ { \infty } = \rho$ and $\Gamma _ { 2 } ^ { \infty } = \sigma$ , then $\mathcal { E } [ \Gamma _ { 1 } ^ { \infty } | \rho ] = \mathcal { E } [ \Gamma _ { 2 } ^ { \infty } | \sigma ] = 0$ and whence

$$0 = \mathcal {E} [ \Gamma_ {1} ^ {\infty} | \rho ] + \mathcal {E} [ \Gamma_ {2} ^ {\infty} | \sigma ] \leq \liminf _ {\tau \to \infty} (\mathcal {E} [ \Gamma_ {1} ^ {\tau} | \rho ] + \mathcal {E} [ \Gamma_ {2} ^ {\tau} | \sigma ]) \leq \liminf _ {\tau \to \infty} \tau (\mathcal {E} [ \Gamma_ {1} ^ {\tau} | \rho ] + \mathcal {E} [ \Gamma_ {2} ^ {\tau} | \sigma ]).$$

• If $\Gamma _ { 1 } ^ { \infty } \neq \rho$ or $\Gamma _ { 2 } ^ { \infty } \neq \sigma$ , then exists $\delta > 0$ such that $\mathcal { E } [ \Gamma _ { 1 } ^ { \infty } | \rho ] + \mathcal { E } [ \Gamma _ { 2 } ^ { \infty } | \sigma ] = \delta$ , which along

with lower semicontinuity gives

$$\liminf _ {\tau \to \infty} \tau (\mathcal {E} [ \Gamma_ {1} ^ {\tau} | \rho ] + \mathcal {E} [ \Gamma_ {2} ^ {\tau} | \sigma ]) \geq \lim _ {\tau \to \infty} \tau \delta = + \infty .$$

Combining both cases above with (3.20), the lim inf inequality follows.

Next, for arbitrary $\Gamma ^ { \infty }$ , take the constant recovery sequence $\{ \Gamma ^ { \infty } \}$ .

• $\operatorname { I f } \ \Gamma _ { 1 } ^ { \infty } = \rho$ and $\Gamma _ { 2 } ^ { \infty } = \sigma$ , then $\mathcal { E } [ \Gamma _ { 1 } ^ { \tau } | \rho ] = \mathcal { E } [ \Gamma _ { 2 } ^ { \tau } | \sigma ] = 0$ .   
• Conversely, if $\Gamma _ { 1 } ^ { \infty } \neq \rho$ or $\Gamma _ { 2 } ^ { \infty } \neq \sigma$ , then again $\mathcal { E } [ \Gamma _ { 1 } ^ { \infty } | \rho ] + \mathcal { E } [ \Gamma _ { 2 } ^ { \infty } | \sigma ] = \delta > 0$

Summarizing both cases, we obtain that the condition (3.16) is satisfied

$$
\begin{array}{l} \lim _ {\tau \to \infty} \mathcal {F} _ {\varepsilon} ^ {\tau} (\Gamma^ {\infty}) = \lim _ {\tau \to \infty} \mathrm{Tr} [ C \Gamma^ {\infty} ] + \varepsilon \mathcal {S} [ \Gamma^ {\infty} ] + \left\{ \begin{array}{l l} 0, & \Gamma_ {1} ^ {\infty} = \rho \text {and} \Gamma_ {2} ^ {\infty} = \sigma \\ \tau \delta , & \text {otherwise} \end{array} \right. = \\ = \mathrm{Tr} [ C \Gamma^ {\infty} ] + \varepsilon \mathcal {S} [ \Gamma^ {\infty} ] + \left\{ \begin{array}{l l} 0, & \Gamma_ {1} ^ {\infty} = \rho \text {and} \Gamma_ {2} ^ {\infty} = \sigma \\ + \infty , & \text {otherwise} \end{array} \right. = \mathcal {F} _ {\varepsilon} ^ {\infty} (\Gamma^ {\infty}), \\ \end{array}
$$

which concludes the proof.
