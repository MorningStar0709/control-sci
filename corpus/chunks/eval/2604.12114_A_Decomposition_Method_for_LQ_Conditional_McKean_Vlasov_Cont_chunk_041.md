# Appendix A.2. Proof of Property (v)

Our goal is to establish that $\mathbb { E } [ ( x _ { t } - H \bar { x } _ { t } ) ^ { \top } Q _ { t } ( x _ { t } - H \bar { x } _ { t } ) ] = \mathbb { E } [ \breve { x } _ { t } ^ { \top } Q _ { t } \breve { x } _ { t } ] + \mathbb { E } \left[ \bar { x } _ { t } ^ { \top } \bar { Q } _ { t } \bar { x } _ { t } \right]$ . Similar to the proof of property (iv), we first decompose $\mathbb { E } [ \breve { x } _ { t } ^ { \top } Q _ { t } \breve { x } _ { t } ]$ into three terms, given by

$$
\begin{array}{l} \mathbb {E} [ \breve {x} _ {t} ^ {\intercal} Q _ {t} \breve {x} _ {t} ] = \mathbb {E} \left[ (x _ {t} - \bar {x} _ {t}) ^ {\intercal} Q _ {t} (x _ {t} - \bar {x} _ {t}) \right] \\ = \mathbb {E} \left[ (x _ {t} - H \bar {x} _ {t} + H \bar {x} _ {t} - \bar {x} _ {t}) ^ {\intercal} Q _ {t} \left(x _ {t} - H \bar {x} _ {t} + H \bar {x} _ {t} - \bar {x} _ {t}\right) \right] \\ \end{array}
= \mathbb {E} \left[ (x _ {t} - H \bar {x} _ {t}) ^ {\intercal} Q _ {t} (x _ {t} - H \bar {x} _ {t}) \right] + \underbrace {\mathbb {E} \left[ (\bar {x} _ {t} - H \bar {x} _ {t}) ^ {\intercal} Q _ {t} (\bar {x} _ {t} - H \bar {x} _ {t}) \right]} _ {D _ {1} ^ {\prime}} - 2 \underbrace {\mathbb {E} \left[ (x _ {t} - H \bar {x} _ {t}) ^ {\intercal} Q _ {t} (\bar {x} _ {t} - H \bar {x} _ {t}) \right]} _ {D _ {2} ^ {\prime}}. \tag {A.5}$$

where

$$
\begin{array}{l} D _ {1} ^ {\prime} = \mathbb {E} \left[ (\bar {x} _ {t} - H \bar {x} _ {t}) ^ {\intercal} Q _ {t} (\bar {x} _ {t} - H \bar {x} _ {t}) \right] = \mathbb {E} \left[ \bar {x} _ {t} ^ {\intercal} (I _ {d} - H) ^ {\intercal} Q _ {t} (I _ {d} - H) \bar {x} _ {t} \right] \\ = \mathbb {E} \left[ \bar {x} _ {t} ^ {\intercal} \bar {Q} _ {t} \bar {x} _ {t} \right], \tag {A.6} \\ \end{array}
$$

and

$$
\begin{array}{l} D _ {2} ^ {\prime} = \mathbb {E} \left[ (x _ {t} - H \bar {x} _ {t}) ^ {\intercal} Q _ {t} (\bar {x} _ {t} - H \bar {x} _ {t}) \right] = \mathbb {E} \left(\mathbb {E} \left[ (x _ {t} - H \bar {x} _ {t}) ^ {\intercal} Q _ {t} (\bar {x} _ {t} - H \bar {x} _ {t}) \mid \mathcal {F} _ {t} ^ {0} \right]\right) \\ = \mathbb {E} \left(\mathbb {E} \left[ (x _ {t} - H \bar {x} _ {t}) ^ {\intercal} \mid \mathcal {F} _ {t} ^ {0} \right] Q _ {t} (\bar {x} _ {t} - H \bar {x} _ {t})\right) \\ = \mathbb {E} \left[ (\bar {x} _ {t} - H \bar {x} _ {t}) ^ {\intercal} Q _ {t} (\bar {x} _ {t} - H \bar {x} _ {t}) \right]. \tag {A.7} \\ \end{array}
$$

Subsequently, (A.5) is expressed as

$$\mathbb {E} \left[ \check {x} _ {t} ^ {\intercal} Q _ {t} \check {x} _ {t} \right] = \mathbb {E} \left[ \left(x _ {t} - H \bar {x} _ {t}\right) ^ {\intercal} Q _ {t} \left(x _ {t} - H \bar {x} _ {t}\right) \right] - \mathbb {E} \left[ \bar {x} _ {t} ^ {\intercal} \bar {Q} _ {t} \bar {x} _ {t} \right]. \tag {A.8}$$

Hence, property (v) holds.
