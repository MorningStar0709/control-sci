Finding an explicit solution of the above is computationally hard as noted in [25]. Therefore, to obtain an approximate solution we discretize (33) into a finite sum and turn γ into a piecewise linear path with boundary conditions. The order of the discretization of γ was set to be 100, meaning γ consisted of 100 piecewise linear steps. The resulting nonlinear program is fed into CasADi using IPOPT as the underlying solver and each run of the problem took less than 0.5 seconds. Figures 1 and 2 show the evolution of 5 different initial value problems of system (30) with the computed control input applied.
