# A.1 Proof of Proposition 1

First, we note that ${ \tilde { V } } ( x ) \leq U ( \xi )$ holds for all ξ. Thus, if $U ( \xi ( t , j ) ) \leq c _ { \mathcal { X } }$ , then it follows that $x ( t , j ) \in \dot { \mathcal { X } }$ . Further since $\hat { x } ( t , k + 1 ) = \hat { x } ( t _ { k } , k )$ for $t _ { k } \leq t \leq t _ { k + 1 }$ and ${ \hat { x } } ( t , k +$ $2 ) \ = \ x ( t _ { k } , k )$ for $t _ { k + 1 } ~ \leq ~ t ~ \leq ~ t _ { k + 2 } ,$ we can conclude that for $r _ { k } \ \preceq \ ( t , j ) \ \preceq \ r _ { k + 2 } , \ x ( t , j ) \in \ \mathcal { X }$ further implies $e ( t , j ) ~ = ~ { \hat { x } } ( t , j ) - x ( t , j ) \in \mathcal { E }$ . Moreover, we note that $s ( r _ { k } ^ { + } ) = e ( r _ { k } ^ { + } )$ and $s ( r _ { k + 1 } ^ { + } ) = - e ( r _ { k + 1 } ^ { + } )$ , i.e. $e ( t , j ) \in \mathcal { E }$ further implies $s ( t , j ) \in \mathcal { E }$ for $r _ { k } ^ { + } \preceq ( t , j ) \preceq r _ { k + 2 }$ .

Next, using (4) and (5) from Condition 1, and (8), we obtain for $x \in { \mathcal { X } } , e \in { \mathcal { E } }$ and $\tau \in [ 0 , t _ { k + 2 } - t _ { k } ]$

$$
\begin{array}{l} \langle \nabla U (\xi), F (\xi) \rangle \\ \leq - \epsilon \tilde {V} (x) - H _ {\ell} ^ {2} (x, e) + \gamma_ {\ell} ^ {2} \tilde {W} (\ell , e, s) \\ + 2 \gamma_ {\ell} \phi_ {\ell} (\tau) \tilde {W} (\ell , e, s) \left[ L _ {\ell} \tilde {W} (\ell , e, s) + H _ {\ell} (x, e) \right] \\ - \gamma_ {\ell} \tilde {W} ^ {2} (\ell , e, s) \left[ 2 \left(L _ {\ell} + \frac {\epsilon}{2}\right) \phi_ {\ell} (\tau) + \gamma_ {\ell} \left(\phi_ {\ell} ^ {2} (\tau) + 1\right) \right] \\ \leq - \epsilon \tilde {V} (x) - H _ {\ell} ^ {2} (x, e) \\ + 2 \gamma_ {\ell} \phi_ {\ell} (\tau) \tilde {W} (\ell , e, s) H _ {\ell} (x, e) - \gamma_ {\ell} ^ {2} \tilde {W} ^ {2} (\ell , e, s) \phi_ {\ell} ^ {2} (\tau) \\ - \epsilon \gamma_ {\ell} \phi_ {\ell} (\tau) \tilde {W} (\ell , e, s) \\ \leq - \epsilon U (\xi). \\ \end{array}
$$

Due to (11), we can conclude that $r ( r _ { k } ^ { + } ) \in \mathcal { X }$ and $e ( r _ { k } ^ { + } ) \in$ E. Thus, we obtain for $t \in [ t _ { k } , t _ { k } + \delta ]$ for sufficiently small $\delta > 0$

$$\frac {d}{d t} U (\xi (t, k + 1)) \leq - \epsilon U (\xi (t, k + 1)) \tag {A.1}$$

and thus due to the comparison Lemma (cf. (Khalil, 2002, p. 102))

$$U (\xi (t, k + 1)) \leq e ^ {- \epsilon (t - t _ {k})} U (\xi (r _ {k} ^ {+})). \tag {A.2}$$

Due to (11), this implies $U ( \xi ( t , k + 1 ) ) < c _ { \mathcal { X } }$ and thus $x ( t , k + \mathrm { i } ) \in \mathcal { X }$ and $e ( t , k + 1 ) \in \mathcal { E }$ . We can now use this argumentation iteratively to conclude that

$$U (\xi (t, k + 1)) \leq e ^ {- \epsilon (t - t _ {k})} U (\xi (r _ {k} ^ {+})) \tag {A.3}$$
