Bounding RTS,1K . $R _ { K } ^ { \mathrm { T S } , 1 }$ Notice that $\{ R _ { K } ^ { \mathrm { T S } , 1 } \} _ { K \geq 0 }$ is a martingale sequence with $\left| R _ { K } ^ { \mathrm { T S , 1 } } - R _ { K - 1 } ^ { \mathrm { T S , 1 } } \right| \ \leq$ $2 \tau _ { 0 } \sigma _ { w } ^ { 2 } \sqrt { n } D$ .
