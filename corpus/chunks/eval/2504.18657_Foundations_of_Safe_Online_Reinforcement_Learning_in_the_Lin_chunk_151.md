proof. Under event $E _ { 2 } ^ { 0 }$ , we have that for sufficiently large $T .$

$$\hat {a} - \hat {b} F _ {\text {opt}} (\hat {\theta} _ {\mathrm{wu}}) \geq a ^ {*} - b ^ {*} F _ {\text {opt}} (\theta^ {*}) - \tilde {O} _ {T} (T ^ {- 1 / 4}) > 0 \tag {146}$$

by Lemma 25 and Lemma 23. Conditional on event $E _ { 2 } ^ { 0 } \cap E _ { \mathrm { E 5 7 } }$ and for sufficiently large $T$ we have the following result:

$$

(147) \\ \end{array}
$$

Because $\theta ^ { * } , D _ { U } , F _ { \mathrm { o p t } } ( \theta ^ { * } ) , K _ { D _ { U } } ^ { \theta ^ { * } }$ are all independent of $T ,$ we can rearrange Equation (147) to get

$$K _ {D _ {U}} ^ {\theta^ {*}} - F _ {\mathrm{opt}} (\theta^ {*}) \leq \tilde {O} _ {T} (T ^ {- 1 / 4}).$$

Combining this with Lemma 23 which states that $| F _ { \mathrm { o p t } } ( \hat { \theta } _ { \mathrm { w u } } ) - F _ { \mathrm { o p t } } ( \theta ^ { * } ) | = \tilde { O } _ { T } ( T ^ { - 1 / 4 } )$ we have that
