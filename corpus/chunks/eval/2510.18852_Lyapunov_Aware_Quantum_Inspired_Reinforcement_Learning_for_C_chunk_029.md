# APPENDIX B IMPLEMENTATION AND REPRODUCIBILITY

All simulation scripts, trained weights, and configuration files have been released on the project’s public repository1 for reproducibility. The repository contains the following core components:

• acc\_env.py – defines the continuous-time adaptive cruise control environment;   
• quantum\_policy.py – implements the variational quantum circuit (VQC) policy network;

• lyapunov.py – provides the Lyapunov function and derivative computation;   
• train.py – trains the LQRL agent using policy gradient and stability penalty;   
• sim\_pygame.py – visualizes the ego and lead vehicles, logs trajectories to CSV, and records MP4 simulation videos;   
• analyze\_results.py – plots trajectories and Lyapunov derivative curves from CSV logs.

The experiment can be reproduced using the following steps:

1) Install dependencies:   
pip install -r requirements.txt

2) Train the LQRL model:   
python train.py

3) Run the simulation and record results:

python sim\_pygame.py

4) Visualize the recorded trajectories:

python analyze\_results.py

All simulations were executed using Python 3.12 with numpy, pygame, and imageio[ffmpeg] libraries. The use of fixed random seeds ensures deterministic and repeatable trajectories for numerical validation.
