# APPENDIX B INTERNAL STABILITY OF SWITCHING ALLOCATION DYNAMICS

Based on the definition of S or σi, the system defined in (16) or (26) constitutes a time-dependent switching system [44], since the sign of $\bar { \tau } ^ { N } ( t )$ determines the dynamics of the allocation. Note that, given (11a) and (12), the sign of $\bar { \tau } ^ { N } ( t )$ equals the sign of $\tau ^ { N } \bar { ( t ) }$ , so the two can be used interchangeably. Given (16), we can write

$$\dot {\zeta} _ {i} = - b _ {i \sigma} \left(\zeta_ {i} - c _ {i} \bar {\tau} ^ {N}\right), \quad i = 1, 2 \tag {40}$$

where

$$
b _ {i \sigma} (t) = \left\{ \begin{array}{l l} k _ {i} (w _ {i} (t) + w _ {3} (t)), & (- 1) ^ {i} \bar {\tau} ^ {N} \prec 0, \\ 0, & \text { otherwise }, \end{array} \right. \tag {41}
c _ {i} (t) = \frac {w _ {3} (t)}{w _ {i} (t) + w _ {3} (t)} - \bar {\alpha}, i = 1, 2.$$

Although in the unforced system $( c _ { i } = 0 )$ we have shown that $\dot { V } ( \zeta ) \leq 0$ (see (19)), it is not possible to establish the existence of a positive definite continuous function $P ( \zeta )$ such that $\dot { V } ( \zeta ) < - P ( \zeta )$ . Consequently, $V ( \zeta )$ does not qualify as a common Lyapunov function across all switching modes [44, Section 2.1] [70]. Nevertheless, in each individual mode of the switching system, one state remains constant while the other decreases, ensuring that $V ( \zeta )$ is non-increasing even under switching (see [44, Section 3.1]), this is sufficient to guarantee internal stability, though asymptotic stability cannot be concluded. In the following, we derive an explicit upper bound on the allocation states and directly prove internal stability.
