The remainder of this section will explain an algorithm that was developed to solve GNEPs using an interior-point method in a previous work [9], and how it has been adapted for the current study. To this end, since we are free to choose g, we suppose that the auxiliary cost function $g$ for player 2 is twice continuously differentiable and for any $\zeta , g ( \zeta , \cdot )$ is convex in $\mu .$ With these conditions, our problem satisfies assumptions A1 and A2 in [9]. For instance, if one chooses $g$ as a linear function of $\mu ,$ then both players face with linear programs. In particular, if $g ( \zeta , \mu ) : = - \langle \zeta , c _ { \mu } \rangle$ , then our problem becomes a zero sum game.

Let us define the functions $\bar { h _ { 1 } } : \mathbb { R } ^ { | \mathsf { X } \times \mathsf { A } | } \times \mathbb { R } ^ { | \mathsf { X } | } \to \mathbb { R } ^ { | \mathsf { X } \times \mathsf { A } | } \times \mathbb { R } ^ { | \mathsf { X } | }$ and $h _ { 2 } : \mathbb { R } ^ { | \mathsf { X } \times \mathsf { A } | } \times \mathbb { R } ^ { | \mathsf { X } | } \to$ $\mathbb { R } ^ { | \mathsf { X } | } \times \mathbb { R } \times \mathbb { R } ^ { | \mathsf { X } | }$ as follows

$$
h _ {1} (\zeta , \mu) := \binom{- \operatorname{Id} \cdot \zeta}{- \hat {\zeta} + (1 - \beta) \mu + \beta   \zeta   p _ {\mu}}, \quad h _ {2} (\zeta , \mu) := \left( \begin{array}{c} - \operatorname{Id} \cdot \mu \\ - \langle \mu , \mathbf {1} \rangle + 1 \\ - \mu + \zeta   p _ {\mu} \end{array} \right)
$$
