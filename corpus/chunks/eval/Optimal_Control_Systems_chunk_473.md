# Singular Values

Let A be an nxm matrix, then the singular values $\sigma$ of the matrix A are defined as the square root values of the eigenvalues ( $\lambda$ ) of the matrix $A^{\prime}A$ , that is

$$\sigma = \sqrt {\lambda (\mathbf {A} ^ {\prime} \mathbf {A})}. \tag {A.2.49}$$

The singular values are usually arranged in the descending order of the magnitude.
