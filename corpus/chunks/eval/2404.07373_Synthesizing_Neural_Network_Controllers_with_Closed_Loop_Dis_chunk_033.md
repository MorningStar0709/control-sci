The Dissipative RINN and LTI controllers are designed with respect to the following uncertain rigid model:

$$
\begin{array}{l} \dot {\boldsymbol {x}} _ {\boldsymbol {r}} (t) = \left[ \begin{array}{c c} 0 & 1 \\ 0 & 0 \end{array} \right] \boldsymbol {x} _ {\boldsymbol {r}} (t) \\ + \left[ \begin{array}{c} 0 \\ \frac {1}{m _ {b} + m _ {r} + \rho L} \end{array} \right] (\boldsymbol {u} (t) + \boldsymbol {d} (t)) + \left[ \begin{array}{c} 1 \\ 0 \end{array} \right] \boldsymbol {w} (t) \\ \end{array}
\boldsymbol {w} (t) = \Delta (\boldsymbol {u} + \boldsymbol {d}) (t)\boldsymbol {y} (t) = \boldsymbol {x} _ {\boldsymbol {b}} (t)\boldsymbol {e} (t) = \boldsymbol {x} _ {\boldsymbol {r}} (t),$$

where $\begin{array} { r } { \boldsymbol { x } _ { r } \ = \ \left\lceil \boldsymbol { x } _ { b } \boldsymbol { \dot { x } } _ { b } \right\rceil ^ { \top } } \end{array}$ and $\| \Delta \| ~ \leq ~ 0 . 1$ . The normbounded $\Delta$ block is used to represent uncertainty in the dynamical model. If the rigid model with no uncertainty $( \Delta = 0 )$ has a transfer function from input $u + d$ to output y of $G _ { r } ( s )$ , then this uncertain model represents transfer functions of the form $\begin{array} { r } { G _ { r } ( s ) + \frac { 1 } { s } \Delta ( s ) } \end{array}$ where $\| \Delta ( s ) \| \le 0 . 1$ . The value of 0.1 is chosen such that this set includes $G _ { f } ( s )$ , the transfer function from input to output of the flexible model.
