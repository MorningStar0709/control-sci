Conditional Expectation

$$\mathbf {E} \{a | b \} = \frac {\mathbf {E} \{a \cap b \}}{\mathbf {E} \{b \}}$$

This can be extended to the case where b is the σ -algebra $\mathcal { F } _ { t }$ generated by the sequence of random variables up to and including t. The conditional expectation of X with respect to $\mathcal { F } _ { t }$ is written: $\mathbf { E } \{ X | \mathcal { F } _ { t } \}$ . An increasing σ -algebra $\mathcal { F } _ { t }$ has the property: $\mathcal { F } _ { t - 1 } \subset \mathcal { F } _ { t }$ .
