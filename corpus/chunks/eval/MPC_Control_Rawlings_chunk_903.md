# Exercise A.20: Positive definite but not symmetric matrices

Consider redefining the notation $A > 0$ for $A \in \mathbb { R } ^ { n \times n }$ to mean $x ^ { \prime } A x > 0$ for all $_ x \in$ $\mathbb { R } ^ { n } \neq 0$ . In other words, the restriction that A is symmetric in the usual definition of positive definiteness is removed. Consider also $B : = ( A + A ^ { \prime } ) / 2$ . Show the following hold for all A. (a) $A > 0$ if and only if B is positive definite. (b) $\operatorname { t r } ( A ) = \operatorname { t r } ( B )$ . (Johnson, 1970; Johnson and Hillar, 2002)
