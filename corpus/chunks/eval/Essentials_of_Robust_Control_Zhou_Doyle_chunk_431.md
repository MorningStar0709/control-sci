$${ M } { = } { \gamma ^ { 2 } P _ { s } ^ { * } ( I + ( 1 - \gamma ^ { 2 } ) P _ { s } P _ { s } ^ { * } ) ^ { - 1 } }{Y ^ {*} Y} = {(\gamma^ {2} - 1) (I + P _ {s} P _ {s} ^ {*}) (I + (1 - \gamma^ {2}) P _ {s} P _ {s} ^ {*}) ^ {- 1}.}$$
