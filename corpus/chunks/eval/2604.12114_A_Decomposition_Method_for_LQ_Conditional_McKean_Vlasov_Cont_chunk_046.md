where 1 denotes the indicator function. Clearly, $\tilde { \alpha } ^ { \epsilon } \in L _ { \mathcal { F } } ^ { 2 } \left( [ t , T ] , \mathbb { R } ^ { d } \right)$ and $\mathbb { E } \left[ \tilde { \alpha } _ { \tau } ^ { \epsilon } \big | \mathcal { F } _ { \tau } ^ { 0 } \big | = 0 \right.$ , for every $\tau \in [ t , T ]$ , which implies that $\tilde { \alpha } ^ { \epsilon } \in \breve { \mathcal { U } } _ { t }$ . Moreover, the state trajectory $\tilde { z } ^ { \epsilon } \in \mathcal { C } _ { \mathcal { F } } \left( [ t , T ] , \mathbb { R } ^ { n } \right)$ of (B.2) corresponding to the control input $\tilde { \alpha } ^ { \epsilon }$ , can be decomposed as

$$\tilde {z} _ {\tau} ^ {\epsilon} = \mathbb {1} _ {[ t, h)} (\tau) z _ {\tau} + \mathbb {1} _ {[ h, T ]} (\tau) z _ {\tau} ^ {\epsilon}, \quad \forall \tau \in [ t, T ], \quad h \in (t, T). \tag {B.10}$$

Therefore, from (B.7) and by the law of iterated conditional expectation, for any $\alpha \in \breve { \mathcal { U } } _ { t } , \epsilon > 0$ and $h \in ( t , T )$ , we have

$$V (t, \breve {\xi}) \leq \breve {J} _ {t, \breve {\xi}} (\tilde {\alpha} ^ {\epsilon})
\begin{array}{l} = \mathbb {E} \left\{\frac {1}{2} \int_ {t} ^ {h} \left[ z _ {\tau} ^ {\intercal} Q _ {\tau} z _ {\tau} + 2 z _ {\tau} ^ {\intercal} S _ {\tau} \alpha_ {\tau} + \alpha_ {\tau} ^ {\intercal} R _ {\tau} \alpha_ {\tau} \right] d \tau + \breve {J} _ {h, z _ {h}} (\alpha^ {\epsilon})   \bigg |   \mathcal {F} _ {t} \right\} \\ <   \mathbb {E} \left\{\frac {1}{2} \int_ {t} ^ {h} \left[ z _ {\tau} ^ {\intercal} Q _ {\tau} z _ {\tau} + 2 z _ {\tau} ^ {\intercal} S _ {\tau} \alpha_ {\tau} + \alpha_ {\tau} ^ {\intercal} R _ {\tau} \alpha_ {\tau} \right] d \tau + V (h, z _ {h}) \mid \mathcal {F} _ {t} \right\} + \epsilon , \quad d t \otimes d \mathbb {P} - a. s. \tag {B.11} \\ \end{array}
$$

Since the inequality (B.11) holds for arbitrary $\alpha \in \breve { \mathcal { U } } _ { t }$ and $\epsilon > 0 .$ , then $\forall h \in ( t , T )$ , we have

$$V (t, \check {\xi}) \leq \inf _ {\alpha \in \check {\mathcal {U}}} \mathbb {E} \left\{\frac {1}{2} \int_ {t} ^ {h} \left[ z _ {\tau} ^ {\intercal} Q _ {\tau} z _ {\tau} + 2 z _ {\tau} ^ {\intercal} S _ {\tau} \alpha_ {\tau} + \alpha_ {\tau} ^ {\intercal} R _ {\tau} \alpha_ {\tau} \right] d \tau + V (h, z _ {h}) \Bigg | \mathcal {F} _ {t} \right\}, \quad d t \otimes d \mathbb {P} - a. s. \tag {B.12}$$

From (B.6) and (B.12), the equality (B.4) holds. This completes the proof.
