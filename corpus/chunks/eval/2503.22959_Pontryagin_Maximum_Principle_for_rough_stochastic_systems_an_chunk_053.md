$$\mathcal {A} := \{u: [ 0, T ] \times \Omega^ {\prime \prime} \to V, (\mathcal {F} _ {t} ^ {\prime \prime}) _ {t} \text {-adapted and continuous with} \mathbb {E} ^ {\prime \prime} [ \sup _ {t \in [ 0, T ]} | u _ {t} | ^ {p} ] < \infty , \forall p \geq 2 \}$$

and the set of admissible anticipative controls by

$$\bar {\mathcal {A}} := \left\{\bar {u}: [ 0, T ] \times \Omega \rightarrow V, \left(\mathcal {F} _ {T} ^ {\prime} \otimes \mathcal {F} _ {t} ^ {\prime \prime}\right) _ {t} \text {-adapted and continuous with}, \right.\mathbb {E} ^ {\prime \prime} [ \sup _ {t \in [ 0, T ]} | u _ {t} (\omega^ {\prime}) | ^ {p} ] < \infty , \mathbb {P} ^ {\prime} - a. s. \forall p \geq 2 \}. \tag {5.3}$$

which is a subset of $\tilde { \mathcal { A } }$ from Section 1.1. To highlight the connection between rough and pathwise control problems, we also consider the set of “rough controls”

$$\mathcal {A} ^ {o} := \left\{u: [ 0, T ] \times \mathscr {C} _ {g} ^ {0, \alpha} \times \Omega^ {\prime \prime} \rightarrow V, (\mathcal {B} (\mathscr {C} _ {g} ^ {0, \alpha}) \otimes \mathcal {F} _ {t} ^ {\prime \prime}) _ {t} \text {-adapted and continuous with} \right.\mathbb {E} ^ {\prime \prime} [ \sup _ {t \in [ 0, T ]} | u _ {t} (\boldsymbol {\eta}) | ^ {p} ] < \infty , \forall \boldsymbol {\eta} \in \mathscr {C} _ {g} ^ {0, \alpha}, \forall p \geq 2 \Big \}. \tag {5.4}$$

The sets of anticipative and rough controls can be identified. For any anticipative $\bar { u } \in \bar { A }$ and any rough path $\pmb { \eta } = ( \eta , \eta ^ { ( 2 ) } ) \in \mathcal { C } _ { g } ^ { 0 , \alpha }$ , we can define a rough control $u ^ { \eta } \in { \mathcal { A } } ^ { o }$ via

$$
u _ {t} ^ {\eta} (\omega^ {\prime \prime}) := \left\{ \begin{array}{l l} \bar {u} _ {t} (\eta , \omega^ {\prime \prime}), & \text { if } \bar {u} _ {t} (\eta , \omega^ {\prime \prime}) \text { is   well - defined }; \\ 0, & \text { otherwise } \end{array} \right. \tag {5.5}
$$

and for any rough $u \in \mathcal A ^ { o }$ , we can define an anticipative control $\bar { u } \in \bar { A }$ by

$$
\bar {u} _ {t} (\omega^ {\prime}, \omega^ {\prime \prime}) := \left\{ \begin{array}{l l} u _ {t} (\mathbf {B} (\omega^ {\prime}), \omega^ {\prime \prime}), & \text { if } u _ {t} (\mathbf {B} (\omega^ {\prime}), \omega^ {\prime \prime}) \text { is   well - defined }; \\ 0, & \text { otherwise }. \end{array} \right. \tag {5.6}
$$
