$$
\begin{array}{l} \mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\Gamma^ {\varepsilon (k)}) = \mathrm{Tr} [ C \Gamma^ {\varepsilon (k)} ] + \varepsilon   \mathcal {S} [ \Gamma^ {\varepsilon (k)} ] + \tau_ {1} \mathcal {E} [ \Gamma_ {1} ^ {\varepsilon (k)} | \rho ] + \tau_ {2} \mathcal {E} [ \Gamma_ {2} ^ {\varepsilon (k)} | \sigma ] \geq \\ \geq \mathrm{Tr} [ C \Gamma^ {\varepsilon (k)} ] - \varepsilon \mathrm{Tr} [ \mathbb {I} _ {d} ] + \tau_ {1} \mathcal {E} [ \Gamma_ {1} ^ {\varepsilon (k)} | \rho ] + \tau_ {2} \mathcal {E} [ \Gamma_ {2} ^ {\varepsilon (k)} | \sigma ] = \\ = - \varepsilon \operatorname{Tr} [ \mathbb {I} _ {d} ] + \tau_ {1} \operatorname{Tr} [ \rho ] + \tau_ {2} \operatorname{Tr} [ \sigma ] + \sum_ {i, j = 1} ^ {d _ {1}, d _ {2}} \left\langle \phi_ {i} ^ {\varepsilon (k)} \otimes \xi_ {j} ^ {\varepsilon (k)} \right| C \Gamma \left| \phi_ {i} ^ {\varepsilon (k)} \otimes \xi_ {j} ^ {\varepsilon (k)} \right\rangle + \\ + \sum_ {i = 1} ^ {d _ {1}} \alpha_ {i} ^ {\varepsilon (k)} \left(\log \alpha_ {i} ^ {\varepsilon (k)} - \left\langle \phi_ {i} ^ {\varepsilon (k)} \right| \log \rho \left| \phi_ {i} ^ {\varepsilon (k)} \right\rangle - 1\right) + \\ + \sum_ {j = 1} ^ {d _ {2}} \beta_ {j} ^ {\varepsilon (k)} \left( \right.\log \beta_ {j} ^ {\varepsilon (k)} - \left\langle \right. \xi_ {j} ^ {\varepsilon (k)} \left. \right| \log \sigma \left| \right. \xi_ {j} ^ {\varepsilon (k)} \left. \right\rangle - 1\left. \right)\rightarrow + \infty , \\ \end{array}
$$

as the terms $x \mapsto x ( \log x - a )$ are superlinear at infinity.

Whence, we get $\mathcal { F } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } ( \Gamma ^ { \varepsilon ( k ) } ) \to \infty$ and $\mathcal { F } _ { \varepsilon } ^ { \tau } ( \Gamma ^ { \tau ( h ) } ) \to \infty$ , which contradicts the hypothesis that

$$\sup _ {\varepsilon > 0} \{\mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\Gamma^ {\varepsilon}) \} < \infty \qquad \text {and} \qquad \sup _ {\tau > 0} \{\mathcal {F} _ {\varepsilon} ^ {\tau} (\Gamma^ {\tau}) \} < \infty .$$

Consequently, the sequences are bounded, and due to the Banach-Alaoglu theorem, we can extract convergent subsequences. □
