Since M is finite, there is a regime $\ell ^ { \prime }$ that will appear infinitely often in $\{ \ell _ { i } \} _ { i \ge 0 }$ . Let $\ell _ { \tilde { n } } = \ell ^ { \prime }$ , for some $\tilde { n } > 1$ . After ˆn steps, the regime ℓ′ reappears, i.e. $\ell _ { \tilde { n } + \hat { n } } = \ell ^ { \prime }$ . Then, by (A.59), we get

$$v _ {\ell^ {\prime}, \iota_ {0}} ^ {\varepsilon} (x _ {0}) = v _ {\ell^ {\prime}, \iota_ {0}} ^ {\varepsilon} (x _ {0}) + \vartheta_ {\ell^ {\prime}, \ell_ {\tilde {n} + 1}} + \vartheta_ {\ell_ {\tilde {n} + 1}, \ell_ {\tilde {n} + 2}} + \dots + \vartheta_ {\ell_ {\tilde {n} + \hat {n} - 1}, \ell^ {\prime}}. \tag {A.60}$$

Notice that (A.60) contradicts the assumption that there is no loop of zero cost (see Eq. (1.8)). From here we conclude that (A.50) must occur. Taking $v ^ { \varepsilon } - u ^ { \varepsilon }$ and proceeding in the same way as before, it follows that for each $( \ell , \iota ) \in \mathbb { M } \times \mathbb { I } , v _ { \ell , \iota } ^ { \bar { \epsilon } } - u _ { \ell , \iota } ^ { \varepsilon } \leq 0$ in O, and hence we conclude that the solution $u ^ { \varepsilon }$ to the HJB equation (1.15) is unique.
