# 14.8 Concluding Remarks

1. Adaptive regulation is an important practical issue in several classes of applications where one can assume that the plant model is almost invariant, while the model of the disturbance to be attenuated or suppressed is unknown and/or time-varying.   
2. The use of the internal model principle allows us to provide solutions for suppressing the effect of the disturbances upon the output provided that, the model of the disturbance can be estimated.   
3. Indirect and direct adaptive regulation schemes can be built.   
4. The use of the Youla-Kucera controller parametrization allows us to built direct adaptive regulation schemes which are simpler that indirect adaptive regulation schemes.   
5. Direct adaptive regulation schemes using the Youla-Kucera controller parametrization can be extended to the multivariable case (Ficocelli and Amara 2009).
