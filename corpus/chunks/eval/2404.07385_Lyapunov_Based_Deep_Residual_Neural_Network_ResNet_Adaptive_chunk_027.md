# V. CONCLUSION AND FUTURE WORK

This paper provides the first result on Lyapunov-derived adaptation laws for the weights of each layer of a ResNetbased adaptive controller. A nonsmooth Lyapunov-based analysis is provided to guarantee asymptotic tracking error convergence. Comparative Monte Carlo simulations are provided to demonstrate the performance of the developed ResNet-based adaptive controller. The developed ResNet-based adaptive controller provides approximately 64% improvement in the tracking and function approximation performance, in comparison to an equivalent fully-connected DNN-based adaptive controller. Additionally, the ResNet overcomes the vanishing gradient problem present in the fully-connected DNN.

Future work can explore incorporating a long short-term memory component in the ResNet architecture, based on our recent work [37], to model uncertainties with longterm temporal dependencies. Additionally, composite adaptive methods can be explored that incorporate a prediction error of the uncertainty, in addition to the tracking error, in the adaptation law.
