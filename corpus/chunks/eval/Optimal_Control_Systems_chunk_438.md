# 7.7 Problems

Problem 7.9 For a double integral plant

$$\dot {x} _ {1} (t) = x _ {2} (t)\dot {x} _ {2} (t) = u (t)$$

with control constraint $|u(t)| \leq 1$ , find the optimal control which transfers the plant from initial condition $x_{1}(0) = 1, x_{2}(0) = 1$ to the final condition $x_{1}(t_{f}) = x_{2}(t_{f}) = 0$ in such a way so as to minimize the performance measure

$$J = \int_ {0} ^ {4} u ^ {2} (t) d t$$

and calculate the minimum value $J^{*}$ .

Problem 7.10 For a second-order system

$$\dot {x} _ {1} (t) = x _ {2} (t)\dot {x} _ {2} (t) = - 2 x _ {2} (t) + 3 x _ {2} (t) + 5 u (t)$$

with control constraint $|u(t)| \leq 1$ , find the optimal control which transfers the plant from initial condition $x_{1}(0) = 1, x_{2}(0) = 1$ to the final condition $x_{1}(t_{f}) = x_{2}(t_{f}) = 0$ in such a way so as to minimize the performance measure

$$J = \int_ {0} ^ {5} u ^ {2} (t) d t$$

and calculate the minimum value $J^{*}$ .

Problem 7.11 The double integral plant

$$\dot {x} _ {1} (t) = x _ {2} (t)\dot {x} _ {2} (t) = u (t)$$

is to be transferred from any state to the origin in minimum time with the state and control constraints as $|u(t)| \leq 1$ and $|x_{2}(t)| \leq 2$ . Determine the optimal control law.

Problem 7.15 For the automobile suspension system described in Problem 1.5, formulate the minimum-energy problem and find the optimal control.
