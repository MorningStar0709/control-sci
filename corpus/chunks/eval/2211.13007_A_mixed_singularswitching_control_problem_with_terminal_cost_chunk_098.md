# A.5 Proof of Proposition 1.1

Proof of Proposition 1.1. Existence. Now, let $( \ell , \iota ) \in \mathbb { M } \times \mathbb { I }$ be fixed. Since $u _ { \ell , \iota } ^ { \varepsilon _ { n } }$ is the unique strong solution to the HJB equation (1.15) when $\varepsilon = \varepsilon _ { n }$ , which belongs to $\mathrm { C } ^ { 0 } ( { \overline { { \mathcal { O } } } } )$ , it follows that for each $\ell ^ { \prime } \in \mathbb { M } \backslash \{ \ell \} , u _ { \ell , \iota } ^ { \hat { \varepsilon } _ { n } } - ( u _ { \ell ^ { \prime } , \iota } ^ { \hat { \varepsilon } _ { n } } + \mathring { \vartheta } _ { \ell , \ell ^ { \prime } } ) \leq u _ { \ell , \iota } ^ { \varepsilon _ { n } } - \mathcal { M } _ { \ell , \iota } u ^ { \varepsilon _ { n } } \leq 0$ in O. From here and (3.18), it yields that $u _ { \ell , \iota } - u _ { \ell ^ { \prime } , \iota } - \vartheta _ { \ell , \ell ^ { \prime } } \leq 0$ in O. Then, $u _ { \ell , \iota } - \mathcal { M } _ { \ell , \iota } u \leq 0$ , in O. Also, we know that $\begin{array} { r } { [ c _ { \iota } - \mathscr { L } _ { \ell , \iota } ] u _ { \ell , \iota } ^ { \varepsilon _ { n } } + \psi _ { \varepsilon _ { n } } ( \vert \mathrm { D } ^ { 1 } u _ { \ell , \iota } ^ { \varepsilon _ { n } } \vert ^ { 2 } - g _ { \iota } ^ { 2 } ) \leq h _ { \iota } \mathrm { ~ a . e ~ } } \end{array}$ . in O. Then,

$$0 \leq \psi_ {\varepsilon_ {n}} (| \mathrm{D} ^ {1} u _ {\ell , \iota} ^ {\varepsilon_ {n}} | ^ {2} - g _ {\iota} ^ {2}) \leq h _ {\iota} - [ c _ {\iota} - \mathcal {L} _ {\ell , \iota} ] u _ {\ell , \iota} ^ {\varepsilon_ {n}}, \quad \text { a.e. in } \mathcal {O}. \tag {A.61}$$
