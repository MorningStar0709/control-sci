# H.7 Proof of Lemma 39

proof. By construction, when using $C _ { K } ^ { \mathrm { u n c } }$ we have the recursive relationship that $x _ { t } = ( a -$ $b K ) x _ { t - 1 } + w _ { t - 1 }$ . Because we assume that $a - b K = 1 - \epsilon < 1$ , we have that

$$| x _ {t} | \leq | x | + \sum_ {i = 0} ^ {\infty} (a - b K) ^ {i} \bar {w} = | x | + \frac {\bar {w}}{1 - (a - b K)} = | x | + \frac {\bar {w}}{\epsilon} = \beta ,$$

where we define $\begin{array} { r } { \beta = | x | + \frac { \bar { w } } { \epsilon } } \end{array}$ . This proves the first part of the lemma. Furthermore, this implies that the magnitude of the control is never greater than

$$| u _ {t} | = | K | | x _ {t} | \leq | K | \beta .$$

Using controller $C _ { K } ^ { \mathrm { u n c } }$ , let $x _ { 0 } , x _ { 1 } , . . . , x _ { T }$ be the sequence of states starting at $x _ { 0 } = x$ and let $y _ { 0 } , y _ { 1 } , . . . , y _ { T }$ be the series of states starting at $y _ { 0 } = y$ . Define $d _ { t } = | x _ { t } - y _ { t } |$ . Note that $d _ { 0 } = | x - y |$ . Furthermore, for all t,

$$d _ {t} = (a - b K) d _ {t - 1}.$$

and

$$| C _ {K} ^ {\mathrm{unc}} (x _ {t}) - C _ {K} ^ {\mathrm{unc}} (y _ {t}) | = K d _ {t}.$$

Therefore, we have the following bound.

$$
\begin{array}{l} | T J (\theta , C _ {K} ^ {\mathrm{unc}}, T, x, W ^ {\prime}) - T J (\theta , C _ {K} ^ {\mathrm{unc}}, T, y, W ^ {\prime}) | \\ = \left| \left(q x _ {T} ^ {2} - q y _ {T} ^ {2}\right) + \sum_ {t = 0} ^ {T - 1} q x _ {t} ^ {2} - q y _ {t} ^ {2} + r (K x _ {t}) ^ {2} - r (K y _ {t}) ^ {2} \right| \\ \leq \sum_ {t = 0} ^ {T} 2 q | x _ {t} | d _ {t} + q d _ {t} ^ {2} + 2 r | K x _ {t} | | K d _ {t} | + r K ^ {2} d _ {t} ^ {2} \\ \leq (2 q + 2 r K ^ {2}) \beta \sum_ {t = 0} ^ {T} d _ {t} + (q + r K ^ {2}) \sum_ {t = 0} ^ {T} d _ {t} ^ {2} \quad | x _ {t} | \leq \beta \\ \leq (2 q + 2 r K ^ {2}) \beta \sum_ {t = 0} ^ {\infty} (a - b K) ^ {t} d _ {0} + (q + r K ^ {2}) \sum_ {t = 0} ^ {\infty} (a - b K) ^ {2 t} d _ {0} ^ {2} \\ = 2 (q + r K ^ {2}) \beta \frac {| x - y |}{1 - (a - b K)} + \frac {(q + r K ^ {2}) (x - y) ^ {2}}{1 - (a - b K) ^ {2}} \\ \leq 2 (q + r K ^ {2}) \beta \frac {| x - y |}{1 - (a - b K)} + \frac {(q + r K ^ {2}) (x - y) ^ {2}}{1 - (a - b K)} \quad a - b K <   1 \\ \leq \frac {2 (q + r K ^ {2}) \left(| x | + \frac {\bar {w}}{\epsilon}\right) | x - y | + (q + r K ^ {2}) (x - y) ^ {2}}{\epsilon}. \tag {153} \\ \end{array}
$$

This is exactly the desired result of the second equation of Lemma 39.
