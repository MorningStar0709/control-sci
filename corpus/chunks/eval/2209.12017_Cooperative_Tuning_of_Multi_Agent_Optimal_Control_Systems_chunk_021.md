# A. Simulation Result

The other parameters used for the following simulation are: $N = 5 , T = 6 0 , \Delta = 0 . 1 \mathrm { s } , \eta ( k ) = 0 . 1 \forall k \geq 0$ . A periodic time variant graph $\mathbb { G } _ { k }$ is defined in Fig. 4. The weight matrix $W ( k )$ is defined by Metropolis weights [24]. The initial state $\mathbf { \Delta x } _ { i , 0 }$ and parameter $\theta _ { i } ( 0 )$ are generated randomly.

As shown in Fig. 3(a) and 3(b), the tunable parameters $\theta _ { i }$ are initialized as different positions at first iteration. As the iteration k increases, the $\theta _ { i } ( k )$ converge to a common point, resulting in multiple agents rendezvousing with each other. In Fig. $3 ( \mathrm { c } )$ , the loss is decreasing when the parameter error $\begin{array} { r l } { } & { { } \sum _ { i = 1 } ^ { N } \sum _ { j = 1 } ^ { N } | | \pmb { \theta } _ { i } - \pmb { \theta } _ { j } | | ^ { 2 } } \end{array}$ is decreasing significantly, and finally both the loss and the parameter error converge.
