Our solution to Problem 6.3 is based on the PPC error transformation, which converts the constrained error behavior $- \rho _ { p _ { i } } ( t ) > y _ { i } ( t ) - y _ { i , \mathrm { d } } ( t ) >$ $\rho _ { p _ { i } } ( t )$ to an unconstrained one. More specifically, we define the errors

$$e _ {p} := \left[ e _ {p _ {1}}, \dots , e _ {p _ {n}} \right] ^ {\top} := y - y _ {\mathrm{d}}, \tag {6.12}$$

as well as the error transformations $\varepsilon _ { p _ { i } } \in$ R according to:

$$e _ {p _ {i}} = \rho_ {p _ {i}} T (\varepsilon_ {p _ {i}}), \forall i \in \{1, \dots , n \}, \tag {6.13}$$

where $T : \mathbb { R }  ( - 1 , 1 )$ is a smooth, strictly increasing analytic function, with $T ( 0 ) = 0$ . Since T is increasing, the inverse mapping $T ^ { - 1 } : ( - 1 , 1 ) \to$ R is well-defined, and it holds that

$$\lim _ {\zeta \to - \infty} T (\zeta) = - 1, \quad \lim _ {\zeta \to + \infty} T (\zeta) = 1 \tag {6.14a}$$

and hence, if $\varepsilon _ { p _ { i } }$ remains bounded in a compact set, the desired funnel objective $- \rho _ { p _ { i } } ( t ) < e _ { p _ { i } } ( t ) < \rho _ { p _ { i } } ( t )$ is achieved, $\forall i \in \left\{ 1 , \ldots , n \right\}$ . We further require that

$$| \zeta | < \left| \frac {\partial T ^ {- 1} (\zeta)}{\partial \zeta} T ^ {- 1} (\zeta) \right|, \forall \zeta \in (- 1, 1). \tag {6.15}$$

A possible choice that satisfies the aforementioned specifications is $T ( \zeta ) =$ $\frac { \exp ( \zeta ) - 1 } { \exp ( \zeta ) + 1 }$ .

From (6.13), we obtain

$$\varepsilon_ {p _ {i}} = T ^ {- 1} \left(\frac {e _ {p _ {i}}}{\rho_ {p _ {i}}}\right), \tag {6.16}$$

which, after differentiation, becomes

$$\dot {\varepsilon} _ {p _ {i}} = \frac {r _ {p _ {i}}}{\rho_ {p _ {i}}} \left(x _ {2 _ {i}} - \dot {y} _ {i, \mathrm{d}} - \frac {\dot {\rho} _ {p _ {i}} e _ {p _ {i}}}{\rho_ {p _ {i}}}\right),$$

or, in stack vector form,

$$\dot {\varepsilon} _ {p} = r _ {p} \rho_ {p} ^ {- 1} \left(x _ {2} - \dot {y} _ {\mathrm{d}} - \dot {\rho} _ {p} \rho_ {p} ^ {- 1} e _ {p}\right), \tag {6.17}$$
