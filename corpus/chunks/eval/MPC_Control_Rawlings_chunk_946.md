# B.4 Robust Stability

We now turn to the task of obtaining stability conditions for discrete time systems subject to disturbances. There are two separate questions that should be addressed. The first is nominal robustness; is asymptotic stability of a set A for a (nominal) system $x ^ { + } = f ( x )$ maintained in the presence of arbitrarily small disturbances? The second question is the determination of conditions for asymptotic stability of a set A for a system perturbed by disturbances lying in a given compact set.
