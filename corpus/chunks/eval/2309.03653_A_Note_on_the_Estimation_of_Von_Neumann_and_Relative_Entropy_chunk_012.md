$$
\begin{array}{l} | S (\hat {\boldsymbol {\varrho}} | | \boldsymbol {\varrho}) | = | \operatorname{tr} \left(\hat {\boldsymbol {\varrho}} \log \hat {\boldsymbol {\varrho}}\right) + S (\boldsymbol {\varrho}) - S (\boldsymbol {\varrho}) - \operatorname{tr} \left(\hat {\boldsymbol {\varrho}} \log \boldsymbol {\varrho}\right) | \\ \leq | S (\underline {{\boldsymbol {\varrho}}}) - S (\hat {\underline {{\boldsymbol {\varrho}}}}) | + | \underbrace {\operatorname{tr} \left((\underline {{\boldsymbol {\varrho}}} - \hat {\underline {{\boldsymbol {\varrho}}}}) \log \underline {{\boldsymbol {\varrho}}}\right)} _ {(\underline {{\boldsymbol {\varrho}}} - \hat {\underline {{\boldsymbol {\varrho}}}}, \log \underline {{\boldsymbol {\varrho}}}) _ {\mathrm{tr}}} |. \\ \end{array}
$$

Using the Cauchy–Schwarz inequality,

$$| S (\hat {\pmb {\varrho}} | | \pmb {\varrho}) | \leq | S (\pmb {\varrho}) - S (\hat {\pmb {\varrho}}) | + \| \pmb {\varrho} - \hat {\pmb {\varrho}} \| \cdot \| \log \pmb {\varrho} \|.$$

Note that $\pmb { \varrho } ( t ) = \pmb { U } ( t ) \pmb { \varrho } ( 0 ) \pmb { U } ( t ) ^ { \dag }$ where $U ( t ) \triangleq e ^ { - \imath H t }$ is unitary. Applying the definition of the matrix logarithm,

$$\log \pmb {\varrho} (t) = \pmb {U} (t) \log (\pmb {\varrho} (0)) \pmb {U} (t) ^ {\dagger}.$$

Combining this equation with the unitary invariance of the Hilbert-Schmidt norm on $\mathbb { C } ^ { d \times d }$ , we see that $\| \log \pmb { \varrho } ( t ) \| ~ = ~ \| \log \pmb { \varrho } ( 0 ) \|$ for all $t \ \geq \ 0 .$ . Since $\varrho ( 0 )$ is assumed to be positive definite $D \triangleq \Vert \log \varrho ( 0 ) \Vert < \infty$ . Therefore

$$S (\hat {\boldsymbol {\varrho}} \| \boldsymbol {\varrho}) \leq | S (\boldsymbol {\varrho}) - S (\hat {\boldsymbol {\varrho}}) | + D \| \hat {\boldsymbol {\varrho}} - \boldsymbol {\varrho} \|.$$

Theorem 2 proved that $| S ( \pmb { \varrho } ) - S ( \pmb { \underline { { \hat { \varrho } } } } ) |$ is essentially exponentially convergent. Hence $S ( \boldsymbol { \underline { { \theta } } } | | \boldsymbol { \underline { { \varrho } } } )$ is the sum of an essentially exponentially convergent term and an exponentially convergent term. Therefore $S ( \boldsymbol { \underline { { \theta } } } | | \boldsymbol { \underline { { \varrho } } } )$ is itself essentially exponentially convergent. □

Theorem 3 shows that the quantum state observer developed in Section II will also produce essentially exponential convergence of the relative entropy to zero as long as $\varrho ( 0 )$ is full-rank. One would imagine the rank condition is satisfied quite often in nature; however, it is an interesting open research problem to study if this assumption can be relaxed.
