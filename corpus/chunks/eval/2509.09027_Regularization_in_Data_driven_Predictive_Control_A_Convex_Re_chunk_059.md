$$\min _ {\sigma_ {y} \in \Gamma , u \in \mathcal {U}, y \in \mathcal {Y}} \| y \| _ {Q} ^ {2} + \| u \| _ {R} ^ {2} + \lambda_ {y} \| \sigma_ {y} \| _ {2} ^ {2}

(1) When $H _ { 1 } = \mathrm { c o l } ( U _ { \mathrm { P } } , Y _ { \mathrm { P } } , U _ { \mathrm { F } } )$ has full row rank, we show that (21) and (B.1) have the same feasible region: if $\sigma _ { y } , u , y , g$ is feasible to (21), then the same $\sigma _ { y } , u , y$ is also feasible for (B.1).
