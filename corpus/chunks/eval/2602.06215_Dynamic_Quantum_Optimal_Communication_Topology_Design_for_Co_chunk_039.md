# 5.2 Numerical Examples

In all examples, the candidate communication graph is the complete undirected graph on n nodes, and the degree bound is fixed at $\gamma = 2 .$ . The degree penalty coefficients are chosen as $\kappa _ { i } = 0 . 1 , i = 1 , \ldots , n .$ so that highly unbalanced degree profiles are mildly penalized. The ADMM hyperparameters are $\rho = 2 0 , \qquad \mu _ { \mathrm { c a r d } } = 0 . 1$ , while the auxiliary penalty β is chosen as a function of the network size $n = N _ { \mathrm { A G E N T S } } .$

$$
\beta = \left\{ \begin{array}{l l} 2 0 0, & n = 5, \\ 4 0 0, & n = 6, \\ 6 0 0, & n = 7. \end{array} \right.
$$

The maximum number of ADMM iterations at each topology update is set to $k _ { \operatorname* { m a x } } = 5 0 0$ ADMM tolerance $1 0 ^ { - 3 }$ . For the second-order consensus dynamics we fix the control gains $\alpha = 3 . 0 , \qquad \beta = 3 . 0$ , as in (9). The initial positions are sampled uniformly at random in the interval $[ - 5 , 5 ] , x _ { i } ( 0 ) \sim \mathcal { U } ( - 5 , 5 )$ , and the initial velocities in the second-order cases are set to zero, $v _ { i } ( 0 ) = 0$ .

The initial topology is a connected graph that satisfies the degree bound and Assumption 2, and it is held constant until $t = 5 \mathrm { ~ s ~ }$ . After this first update time, the topology is recomputed every $\Delta t = 0 . 5$ s using Algorithm 2. The overall closed-loop simulation is capped at $T _ { \mathrm { m a x } } = 1 0 ~ \mathrm { s } .$ .

To monitor convergence we use a standard consensus error metric. The closed-loop run is terminated as soon as the position disagreement drops below the tolerance

$$e (t) \leq \text { CONS\_TOL } = 1 0 ^ {- 3}$$

or when t reaches $T _ { \mathrm { m a x } } .$ . The quantum binary block uses VarQITE with imaginary-time horizon $\tau \ = \ 1 . 5$ , discretized in 30 steps, an EfficientSU2 ansatz with one repetition and linear entanglement.
