The following theorem states the stability results for NMPC.

Theorem D.1 ([87]). Consider system (D.1) and suppose that

• the NMPC open-loop optimal control problem (D.2) is feasible at $t = 0$   
• the terminal set ${ \mathcal { E } } \subseteq { \mathcal { X } }$ is closed with $0 \in \mathcal { E }$ and the terminal cost $E ( x )$ is cont. differentiable and positive definite   
• the terminal set and terminal cost are chosen such that, $\forall x \in \mathcal { E }$ , there exists an (admissible) input $u \varepsilon : [ 0 , \delta ] \to \mathcal { U }$ such that $x ( s ) \in \mathcal { E }$ , $\forall s \in [ 0 , \delta ]$ , and

$$\frac {\partial E}{\partial x} f (x (s), u _ {\mathcal {E}} (s)) + F (x (s), u _ {\mathcal {E}} (s)) \leq 0, \forall s \in [ 0, \delta ]$$

Then, the closed-loop system is asymptotically stable with the region of attraction being the sets of states for which the an admissible input exists.
