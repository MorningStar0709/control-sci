# 5. SMP for anticipative pathwise stochastic control problems

We are now ready to introduce a probabilistic framework that allows us to define and analyze pathwise control problems with anticipative controls. Specifically, we consider the canonical space $\Omega ^ { \prime }$ equipped with the Wiener probability $\mathbb { P } ^ { \prime } , d _ { B }$ -dimensional canonical process B and the augmented filtration $( \mathcal { F } _ { t } ^ { \prime } ) _ { t }$ . Fix a probability space $\left( \Omega ^ { \prime \prime } , \mathcal { F } ^ { \prime \prime } , \mathbb { P } ^ { \prime \prime } \right)$ that supports a $d _ { W }$ -dimensional Brownian motion $W$ and let $\{ \mathcal { F } _ { t } \} _ { 1 }$ t be the augmented filtration of the filtration generated by W. Consider the following probability space as our model for anticipative pathwise control problems:

$$(\Omega , (\mathcal {F} _ {t}) _ {t}, \mathbb {P}) = (\Omega^ {\prime}, (\mathcal {F} _ {t} ^ {\prime}) _ {t}, \mathbb {P} ^ {\prime}) \otimes (\Omega^ {\prime \prime}, (\mathcal {F} _ {t} ^ {\prime \prime}) _ {t}, \mathbb {P} ^ {\prime \prime}) \tag {5.1}$$

To lift Brownian sample paths $B ( \omega ^ { \prime } )$ to a geometric rough path we define for each $( s , t ) \in \Delta$ , the Stratonovich integral

$$\mathbb {B} _ {s, t} ^ {\text { Strato }} := \int_ {s} ^ {t} \delta B _ {s, r} \circ d B _ {r}$$

It is well known (see [13, Chapter 3]) that for all $\omega ^ { \prime }$ outside a $\mathbb { P } ^ { \prime } .$ -null set $N ^ { \prime }$ of $\Omega ^ { \prime } .$ , the Brownian rough path

$$\mathbf {B} ^ {\text { Strato }} (\omega^ {\prime}) := (B (\omega^ {\prime}), \mathbb {B} ^ {\text { Strato }} (\omega^ {\prime}))$$

belongs to the space ${ \mathcal { C } } _ { g } ^ { 0 , \alpha }$ for all $\alpha \in ( 1 / 3 , 1 / 2 )$ and that the following lifting map is $( \mathcal { F } _ { t } ^ { \prime } )$ −progressively measurable:

$$
\begin{array}{r c l}\mathbf {B} := \mathbf {B} ^ {\text { Strato }}:&[ 0, T ] \times \Omega^ {\prime}&\rightarrow \quad (\mathcal {C} _ {g} ^ {0, \alpha}, \mathcal {B} (\mathcal {C} _ {g} ^ {0, \alpha}))\\&(t, \omega^ {\prime})&\mapsto \quad \mathbf {B} ^ {\text { Strato }} (\omega^ {\prime}). _ {\wedge t}\end{array}\tag {5.2}
$$
