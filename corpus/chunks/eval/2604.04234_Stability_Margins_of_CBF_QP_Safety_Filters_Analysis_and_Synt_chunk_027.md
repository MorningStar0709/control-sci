# B. Aircraft Roll–Yaw Example

We next consider the roll–yaw dynamics of a mid-size aircraft around an operating point from [11, Sec. 14.8]. The plant state is $\boldsymbol { x } _ { p } = [ \beta , p _ { s } , r _ { s } ] ^ { \top }$ , where $\beta$ is the sideslip angle and $p _ { s } , r _ { s }$ are the roll and yaw rates. Using an integrated tracking-error state $e _ { y I }$ , the augmented dynamics are

$$
\left[ \begin{array}{c} \dot {e} _ {y I} \\ \dot {x} _ {p} \end{array} \right] = A \left[ \begin{array}{c} e _ {y I} \\ x _ {p} \end{array} \right] + B u + \left[ \begin{array}{c} - y _ {\mathrm{cmd}} \\ 0 \end{array} \right], \tag {41}
$$

with

$$
A = \left[ \begin{array}{c c} - \kappa & C _ {p} \\ 0 & A _ {p} \end{array} \right], \qquad B = \left[ \begin{array}{c} 0 \\ B _ {p} \end{array} \right], \qquad C _ {p} = [ 0 1 0 ],
$$

where $\kappa = 0 . 5$ , and $A _ { p } , B _ { p }$ are taken from [11]. We use an LQR nominal controller with

$$Q = \operatorname{diag} (1 0. 2 5, 0, 0, 1 6. 0 2), \quad R = \operatorname{diag} (1, 0. 4 9).$$

Safety is imposed through the roll-rate constraint $\mathit { p _ { s } } ~ \leq$ 0.4, which yields the affine barrier function $h ( x ) = c ^ { \top } x +$ d with $c = \big \lceil 0 \quad 0 \quad - 1 \quad 0 \big \rceil ^ { \mathrm { ~ ! ~ } } , d = 0 . 4 .$ . For the baseline controller, the exact upper endpoint of the active-mode gain interval, computed from Cor. 1, is $\kappa _ { \operatorname* { m a x } } \approx 1 . 0 1 8$ . Moreover, the interval [0.1, 1.015] is certified by the LMI condition in Prop. 5. The corresponding exact phase and delay margins, obtained from (21)–(22), are $\phi _ { \mathrm { P M } } \approx 2 . 8 8 ^ { \circ } , \tau _ { \mathrm { D M } } \approx 1 . 0 9 ~ \mathrm { s }$ . For reference, the inactive mode $A _ { 0 }$ is stable for all positive uniform gain scalings and has phase margin approximately $7 3 ^ { \circ }$ , so the robustness bottleneck here is the active mode.
