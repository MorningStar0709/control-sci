# Exercise A.34: Probability density of the inverse function

Consider a scalar random variable $\xi \in \mathbb { R }$ and let the random variable η be defined by the inverse function

$$\eta = \xi^ {- 1}$$

(a) $\operatorname { I f } \xi$ is distributed uniformly on [a, 1] with $0 < a < 1$ , what is the density of $\eta ?$

(b) Is $\eta ^ { \prime } s$ density well defined if we allow a = 0? Explain your answer.
