$$
\begin{array}{l} (R _ {T} (\pi) - R _ {t ^ {\star} (\delta)} (\pi)) _ {\mathcal {G}} \\ \leq \bar {Q} \sum_ {t = t ^ {\star} (\delta)} ^ {T - 1} \| x _ {t} ^ {\pi} - x _ {t} ^ {\star} \| + \bar {R} \sum_ {t = t ^ {\star} (\delta)} ^ {T - 1} \| u _ {t} ^ {\pi} - u _ {t} ^ {\star} \| \\ \leq \bar {Q} \sum_ {t = 0} ^ {T - 1} \| x _ {t} ^ {\pi} - x _ {t} ^ {\star} \| + \bar {R} \sum_ {t = 0} ^ {T - 1} \| v _ {t} ^ {\pi} - v _ {t} ^ {\star} \| + \| \zeta_ {t} \| \\ \leq \bar {Q} \sum_ {t = 0} ^ {T - 1} \| B (\theta^ {\star}) \| e _ {t - 1} \\ + \bar {R} \sum_ {t = 0} ^ {T - 1} (e _ {t} + \| B (\theta^ {\star}) \| \sum_ {i = 0} ^ {t - 1} q _ {1} (t) e _ {t - i - 1}) + 3 \sigma_ {t} \\ \leq (\bar {Q} + \bar {R} C _ {1}) \| B (\theta^ {\star}) \| \sum_ {t = 0} ^ {T - 1} e _ {t} + 3 \sigma_ {t} \\ \leq (\bar {Q} + \bar {R} C _ {1}) \| B (\theta^ {\star}) \| \sum_ {t = 0} ^ {T - 1} q _ {2} (0) \| \theta_ {t} - \theta^ {\star} \| + 3 \sigma_ {t} \\ \leq \bar {C} \sum_ {t = 0} ^ {T - 1} \epsilon_ {t} + 3 \sigma_ {t} \\ \end{array}
\leq \bar {C} \sqrt {c _ {3}} \sum_ {t = 0} ^ {T - 1} \log (t / \delta) ^ {1 / 2} t ^ {- (1 - \alpha) / 2} + 3 d _ {x} ^ {1 / 4} \sigma \sum_ {t = 0} ^ {T - 1} t ^ {- \alpha / 2}\leq \frac {2 \bar {C} \sqrt {c _ {3}}}{\alpha + 1} \log (T / \delta) ^ {1 / 2} T ^ {(\alpha + 1) / 2} + \frac {6 d _ {x} ^ {1 / 4}}{\alpha + 1} \sigma T ^ {(\alpha + 1) / 2},$$

with $\bar { C } = ( \bar { Q } + \bar { R } C _ { 1 } ) \lVert B ( { \theta } ^ { \star } ) \rVert C _ { 1 }$ .

Putting it together, we have under the event $\mathcal { G } \colon$

$$R _ {T} (\pi) _ {\mathcal {G}} = (R _ {T} (\pi) - R _ {t ^ {\star} (\delta)} (\pi)) _ {\mathcal {G}} + R _ {t ^ {\star} (\delta)} (\pi) _ {\mathcal {G}}\leq \frac {2 \bar {C} \sqrt {c _ {3}}}{\alpha + 1} \log (T / \delta) ^ {1 / 2} T ^ {(\alpha + 1) / 2}+ \frac {2 d _ {x} ^ {1 / 4}}{\alpha + 1} \sigma T ^ {(\alpha + 1) / 2} + \bar {C} _ {2} \log (1 / \delta) + \bar {C} _ {1}.$$

To obtain a bound on the expected regret, first note that

$$\mathbb {E} [ R _ {T} (\pi) ] = (1 - \delta) R _ {T} (\pi) _ {\mathcal {G}} + \delta R _ {T} (\pi) _ {\neg \mathcal {G}}$$

where $R _ { T } ( \pi ) _ { \neg { g } }$ is the regret when the event $\mathcal { G }$ does not hold and can be taken to be linear in time with some constant $c .$ This follows again from the recursive feasibility of STT-MPC (Theorem 3.4), which ensures that $\boldsymbol { x } _ { t } ^ { \pi }$ and $\boldsymbol { u } _ { t } ^ { \pi }$ are bounded in the set C. Letting $\delta = 1 / T ^ { 2 }$ leads to

$$\mathbb {E} [ R _ {T} (\pi) ] = (1 - \delta) R _ {T} (\pi) _ {\mathcal {G}} + \delta R _ {T} (\pi) _ {\neg \mathcal {G}}= R _ {T} (\pi) _ {\mathcal {G}} + \frac {1}{T ^ {2}} R _ {T} (\pi) _ {\mathcal {G}} + \frac {c}{T}\lesssim \frac {2 \sqrt {3} \bar {C} \sqrt {c _ {3}}}{\alpha + 1} \log (T) ^ {1 / 2} T ^ {(\alpha + 1) / 2}.$$

Here, the notation $\lesssim$ hides the universal constant such that the inequality holds. □
