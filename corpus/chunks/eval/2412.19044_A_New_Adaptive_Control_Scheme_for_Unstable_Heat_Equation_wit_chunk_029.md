# 8 Numerical simulations

In this section, we present numerical simulations for systems (3.3) and (4.17) to validate the theoretical results. The finite difference scheme is adopted in discretization. The numerical results are programmed in Matlab. The time step and the space step are taken as 0.0001 and 0.02, respectively. The reference signal and the corresponding parameters are chosen as

$$r ^ {*} = 3, c _ {0} = c _ {1} = 5, q = 2, b = - 1 0. \tag {8.1}$$

In both cases, the initial state is chosen as

$$w (x, 0) = q x - 1, \hat {w} (x, 0) = \hat {z} (x, 0) = 0, \zeta (0) = 0. \tag {8.2}$$

The solution of system (3.3) is plotted in Figure 1. From Figure 1 we see that the system state has been stabilized effectively, which shows that our controller works well. Both the state observer and the control coefficient update law are convergent. However, ζ(t) has not converged to $\textstyle { \frac { 1 } { b } }$ . Hence, the numerical simulation is consistent with our theoretical results in Theorem 3.1 and Remark 3.1. The tracking result of system (4.17) is plotted in Figure 2 which shows that the performance output converges to the constant reference smoothly. Since the persistent excitation condition is satisfied, the estimate $\zeta$ converges to $\textstyle { \frac { 1 } { b } }$ as $t \to + \infty$ . All the state of system (4.17) are bounded.
