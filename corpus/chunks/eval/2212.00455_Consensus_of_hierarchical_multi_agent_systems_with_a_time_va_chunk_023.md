# 4.2 Water pumping system

In water pumping systems, it is needed to supply a certain flow $( Q ^ { * } )$ . In this case, the main challenge is to coordinate all the pumps to supply the desired flow while keeping the number of active pumps limited. The motivation for this constraint is to keep a proper balance among the lifespans of the pumps, where the working time of the pumps should be similar among all of them. This is why the switching algorithm is a suitable solution.

In this case, we consider the system presented in (Kusumawardana et al., 2019), where the flow $( Q = K * \omega )$ is proportional to the speed rotation of the pump (ω) and is expressed as

$$\Sigma_ {i}: x _ {i} (k + 1) = x _ {i} (k) - w q _ {i} (k) u _ {i} (k), \tag {42}$$

for all $i = \{ 1 , 2 , \dots , N \}$ , where $k \in \mathbb { Z } _ { + } , T _ { s }$ is the sampling period, $x _ { i } \in \mathbb { R }$ is the state variable describing the $i ^ { t h }$ pump flow (Qi), w = 0.2e3 m2 $\begin{array} { r }  \begin{array} { l } { w } \end{array} = \begin{array} { l } { 0 . 2 e ^ { 3 } \frac { m ^ { 2 } } { \mathrm { s A } } } \end{array} \end{array}$ sA is a constant defined by pump characteristics and $u _ { i } \in \mathbb { R }$ is the control input describing the pump armature current $\left( I a _ { i } \right)$ .
