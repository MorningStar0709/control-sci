where $A _ { c } ( \Theta ) \ : = \ : \Theta ^ { \intercal } H ( \Theta )$ is the closed-loop matrix. We know that $P ( \Theta )$ satisfies the Riccati equation as

$$P - A _ {c} ^ {\intercal} P A _ {c} = Q + K ^ {\intercal} R K \succ 0 \implies \left(P ^ {\frac {1}{2}} A _ {c} P ^ {- \frac {1}{2}}\right) ^ {\intercal} P ^ {\frac {1}{2}} A _ {c} P ^ {- \frac {1}{2}} \prec I$$

where we dropped Θ dependence for simplicity. Therefore, similarity transformation of the closedloop matrix $\bar { A } _ { c } : = P ^ { \frac { 1 } { 2 } } A _ { c } P ^ { - \frac { 1 } { 2 } }$ is a contraction, i.e., $\| P ^ { \frac { 1 } { 2 } } A _ { c } P ^ { - \frac { 1 } { 2 } } \| _ { 2 } = : \sigma _ { \Theta } < 1$ . Multiplying both sides of (132) by $P ^ { - { \frac { 1 } { 2 } } }$ we obtain

$$\delta \bar {P} (\delta \Theta) = \bar {A} _ {c} ^ {\intercal} \delta \bar {P} (\delta \Theta) \bar {A} _ {c} + \bar {A} _ {c} ^ {\intercal} P ^ {\frac {1}{2}} \delta \Theta^ {\intercal} H P ^ {- \frac {1}{2}} + P ^ {- \frac {1}{2}} H ^ {\intercal} \delta \Theta P ^ {\frac {1}{2}} \bar {A} _ {c}$$

where $\delta \bar { P } ( \delta \Theta ) ~ = ~ P ^ { - \frac { 1 } { 2 } } \delta P ( \delta \Theta ) P ^ { - \frac { 1 } { 2 } }$ . Taking the spectral norm of both sides and using submultiplicativity of spectral norm as well as equivalence of matrix norms, we have that

$$
\begin{array}{l} \| \delta \bar {P} (\delta \Theta) \| _ {2} \leq \| A _ {c} \| _ {2} ^ {2} \| \delta \bar {P} (\delta \Theta) \| _ {2} + 2 \| \bar {A} _ {c} \| _ {2} \| P ^ {\frac {1}{2}} \delta \Theta^ {\intercal} H P ^ {- \frac {1}{2}} \| _ {2} \\ \leq \| A _ {c} \| _ {2} ^ {2} \| \delta \bar {P} (\delta \Theta) \| _ {2} + 2 \| \bar {A} _ {c} \| _ {2} \| P ^ {\frac {1}{2}} \delta \Theta^ {\intercal} H P ^ {- \frac {1}{2}} \| _ {F} \\ = \sigma_ {\Theta} ^ {2} \| \delta \bar {P} (\delta \Theta) \| _ {2} + 2 \sigma_ {\Theta} \| \delta \Theta^ {\intercal} H \| _ {F} \\ \end{array}
$$

By rearranging the inequality and using the property $\| \delta \Theta ^ { \intercal } H \| _ { F } \leq \| \delta \Theta \| _ { V ^ { - 1 } } \| H \| _ { V }$ , we obtain

$$\| \delta \bar {P} (\delta \Theta) \| _ {2} \leq \frac {2 \sigma_ {\Theta}}{1 - \sigma_ {\Theta} ^ {2}} \| \delta \Theta \| _ {V ^ {- 1}} \| H \| _ {V}$$
