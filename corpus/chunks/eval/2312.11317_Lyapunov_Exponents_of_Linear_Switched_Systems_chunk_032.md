$$\operatorname{tr} (A B) = - \lambda \mu \left(\frac {1}{\alpha^ {2}} + \alpha^ {2} + \beta^ {2}\right).$$

Since $\textstyle { \frac { 1 } { \alpha ^ { 2 } } } + \alpha ^ { 2 } \geq 2$ , we obtain

$$| \mathrm{tr} (A B) | \geq 2 | \lambda \mu | = \sqrt {\mathrm{tr} (A ^ {2}) \mathrm{tr} (B ^ {2})}.$$

Remark 6.2. From the previous proof, we can give a geometric interpretation of the sign of $\mathrm { t r } ( A B )$ in the case $\operatorname { t r } ( B ^ { 2 } ) \leq \operatorname { t r } ( A ^ { 2 } ) < 0$ . Define the cone of imaginary matrices in ${ \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ :

$$\mathcal {C} = \{M \in \mathfrak {s l} _ {2} (\mathbb {R}) \mid \operatorname{tr} (M ^ {2}) < 0 \}.$$

To see that C is a cone, one can reason as at the beginning of Subsection 3.1. Using the form for A, B introduced in the previous proof, if $\mathrm { t r } ( A B ) < 0$ , then $\lambda \mu > 0 .$ that is A, B are in the same connected component. If instead $\operatorname { t r } ( A B ) > 0$ , then $\lambda \mu < 0$ , hence $A , B$ are in two different connected component.

In particular, if $\mathrm { t r } ( A B ) < 0$ , then the whole segment of velocities

$$\{u A + (1 - u) B \mid u \in [ 0, 1 ] \}$$

is inside the cone C. In this case, if the singular control is admissible, then it has imaginary eigenvalues, hence singular constant control cannot be optimal (the resulting trajectory in $\mathrm { S L _ { 2 } ( \mathbb { R } ) }$ would be periodic, hence bounded). So, in this case the best admissible strategy is the one given in Subsection 4.4.1, and we obtain the third point of Main Theorem.

If instead $\operatorname { t r } ( A B ) > 0$ , then part of the segment of velocities is outside the cone, and in particular singular control is admissible since $\operatorname { t r } ( A ^ { 2 } ) < 0$ .

The following Lemma shows that in this case constant singular control are again optimal.

Lemma 6.3. $I f \operatorname { t r } ( B ^ { 2 } ) \leq \operatorname { t r } ( A ^ { 2 } ) < 0$ and tr $\cdot ( A B ) \geq \sqrt { \mathrm { t r } ( A ^ { 2 } ) \mathrm { t r } ( B ^ { 2 } ) }$ , then

$$\frac {2}{\pi} \frac {1}{\sqrt {\frac {- 2}{\operatorname{tr} (A ^ {2})}} + \sqrt {\frac {- 2}{\operatorname{tr} (B ^ {2})}}} \mathrm{arcosh} \left(\frac {\operatorname{tr} (A B)}{\sqrt {\operatorname{tr} (A ^ {2}) \operatorname{tr} (B ^ {2})}}\right) \leq \sqrt {\frac {1}{2} \frac {\operatorname{tr} (A B) ^ {2} - \operatorname{tr} (A ^ {2}) \operatorname{tr} (B ^ {2})}{2 \operatorname{tr} (A B) - \operatorname{tr} (A ^ {2}) - \operatorname{tr} (B ^ {2})}}.$$
