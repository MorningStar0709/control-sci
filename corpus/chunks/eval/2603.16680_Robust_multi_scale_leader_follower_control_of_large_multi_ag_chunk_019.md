# A. Finite population effects

Our architecture rests on a mean-field assumption that requires populations to be sufficiently large. To quantify the minimum population sizes for the micro-to-macro bridges to operate reliably, we simulated system (4) under the control law of Sec. III, using the same scheme and gains as in Fig. 2. The total masses are held fixed at $M ^ { F } = 1$ and $M ^ { L } = 3 0$ across all runs, so that varying $N ^ { L }$ or $N ^ { F }$ does not change the leader-to-follower mass ratio. $N ^ { L } ~ ( N ^ { F } )$ were sampled in the interval [10,5000] using 30 samples equally spaced in logarithmic scale.

Fig. 4 (orange line) shows the residual error as a function of the number of leaders $N ^ { L }$ , with the number of followers fixed at $N ^ { F } = 1 0 0 0$ . A clear threshold emerges: below the critical value of $N ^ { L } \approx 1 3 0$ , the leaders are too sparse for the macro-to-micro bridge to faithfully reproduce the control field, and the error consistently settles at a high value. Above this threshold, the error norm on the followers density decreases below $1 0 ^ { - 2 }$ , identifying this value as the minimum amount of leaders required to correctly reconstruct densities from agents’ positions. Conversely, Fig. 4 (blue line) reports the error as a function of $N ^ { F }$ with $N ^ { L } = 1 0 0 0$ fixed. We find that, even in the presence of a large number of leaders, a minimum number of followers $( N _ { F } \approx 4 0 0 )$ is required to reduce the norm of the error to values less than $1 0 ^ { - 2 }$ . These observations are in line with the herdability conditions recently reported in [21].
