# 4.3 Shidoku Puzzle

Consider Shidoku is a 4x4 version with constraints expressed as

Integer Value: $\textstyle \prod _ { h = 1 } ^ { 4 } ( x _ { i j } - h ) = 0 $ ,

$\begin{array} { r } { \mathrm { R o w s : } \sum _ { j } x _ { i j } = 1 0 , \prod _ { j } x _ { i j } = 2 4 , } \end{array}$

$\begin{array} { r } { \mathrm { C o l s : } \sum _ { i } x _ { i j } = 1 0 , \prod _ { i } x _ { i j } = 2 4 , } \end{array}$

$\begin{array} { r } { \mathrm { B l o c k s : } \sum _ { \left( k , \ell \right) \in B _ { p } } x _ { k \ell } = 1 0 , \prod _ { \left( k , \ell \right) \in B _ { p } } x _ { k \ell } = 2 4 , } \end{array}$

Fixed position: $x _ { 1 , 2 } = 1 , x _ { 1 , 4 } = 4 , x _ { 3 , 1 } = 2 , x _ { 3 , 4 } = 3$ . (13)

The Shidoku puzzle is solved using the SMC-based optimization framework defined by the closed-loop dynamics $\begin{array} { r } { \dot { x } = - J ^ { \top } ( J _ { h } J _ { h } ^ { \top } ) ^ { - 1 } \left( K \cdot \frac { h } { | h | + \varepsilon } + \alpha h \right) } \end{array}$ , where $K = 5$ $\alpha = 0 . 1$ , and $\varepsilon = 1 0 ^ { - 3 }$ ensures smooth approximation of the discontinuous sgn function to reduce chattering.
