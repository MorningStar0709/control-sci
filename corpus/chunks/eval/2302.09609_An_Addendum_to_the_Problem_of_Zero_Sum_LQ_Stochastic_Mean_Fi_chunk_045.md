\begin{array}{l} \hat {R} _ {\hat {W}} (t) + \sum_ {j = 1} ^ {r} \hat {B} _ {j \hat {W}} ^ {\top} (t) X (t) \hat {B} _ {j \hat {W}} (t) \\ - \left(\hat {R} _ {1 2} (t) + \hat {W} ^ {\top} (t) \hat {R} _ {2 2} (t) + \sum_ {j = 1} ^ {r} \hat {B} _ {j \hat {W}} ^ {\top} (t) X (t) \hat {B} _ {j 2} (t)\right) \\ \times \left(\hat {R} _ {2 2} (t) + \sum_ {j = 1} ^ {r} \hat {B} _ {j 2} ^ {\top} (t) X (t) \hat {B} _ {j 2} (t)\right) ^ {- 1} \star \ll 0, t \in (\tau , T ] \tag {136} \\ \end{array}
$$

The left hand side of (135) is the Schur complements of the (2,2)-block of the matrix:

$$\Theta (t) = \mathbb {W} ^ {\top} (t) \mathbb {R} (t, X (t)) \mathbb {W} (t), t \in \mathcal {I} _ {\mathbf {X}} \tag {137}$$

where $\mathbb { W } ( t ) = \left( \begin{array} { c c } { I _ { m _ { 1 } } } & { 0 } \\ { W ( t ) } & { I _ { m _ { 2 } } } \end{array} \right)$ Using (137) one can show by direct calculation that the left Im2 hand side of (135) is the Schur complement of the (2,2)-block of the matrix R(t, X(t)). This yields that X(t) verify the sign condition (14b) on (τ, T ]. A similar reasoning as above yields to the conclusion that X(t) verify the sign condition (15b) on (τ, T ].

Now by taking the limit for t → τ one obtains that X(·) is define also at t = τ and satisfies also the sign conditions (14)-(15).

Further, again from the general theory of ODEs and based on the continuity of the coefficients of the Riccati equation (7) one deduces that the solution of the equation (7) can be extended to another interval $( \tau _ { 1 } , \tau ]$ and it satisfies the sign conditions $( 1 4 ) - ( 1 5 )$ on this interval. In this way, the maximality of the interval $( \tau , T ]$ is violated. Hence (134) remains true on $\mathcal { T } _ { X }$ . In addition, X(t) being bounded on $\mathcal { T } _ { X } .$ , it can be extended to [0; T ].

It remains now to prove that $\mathcal { T } _ { \hat { X } } = [ 0 ; T ]$ . First, by using similar arguments as the ones used to prove that $\mathcal { T } _ { X } \neq \emptyset$ , we can show that $\mathcal { T } _ { \hat { X } }$ is not empty. Hence, in order to get the desired conclusion it is sufficient to show that $\hat { X } ( \hat { t } )$ is uniformly bounded on $\mathcal { T } _ { \hat { X } }$ . Here also, one can apply a similar procedure as above in order to get the expected result. Indeed, thanks to Lemma 3.11 and Lemma 3.13 one can show that:

$$\Upsilon_ {\Phi} ^ {\hat {\Phi}} (t) \leq \hat {X} (t) \leq Y _ {K W} ^ {\hat {K W}} (t); \quad t \in \mathcal {I} _ {\hat {X}} \tag {138}$$

This ends the proof.
