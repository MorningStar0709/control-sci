# APPENDIX A PROOF OF LEMMA 1

Proof. Applying Adam’s law on equation (2) yields

$$\underset {\pi} {\operatorname{argmax}} \quad \mathbb {E} _ {p (\mathbf {c} (t))} \left\{\mathbb {E} _ {p _ {\pi^ {c}, \pi^ {m}} \left(\{\mathrm{tr} \} _ {t ^ {\prime}} ^ {T ^ {\prime}} | \mathbf {c} (t)\right)} \left\{\mathbf {g} \left(t ^ {\prime}\right) | \mathbf {c} (t) \right\} \right\}, \text { s.t. } | \mathcal {C} | \leq 2 ^ {R} \tag {10}$$

where $\mathbf { c } ( t )$ is generated by the communication policy $\pi ^ { c }$ and the joint pmf of the system’s trajectory $\{ t r \} _ { t ^ { \prime } } ^ { \dot { T } ^ { \prime } }$ is directly influenced by the action policy ${ \bar { \pi } } ^ { m }$ . The conditional pmf $p _ { \pi ^ { c } , \pi ^ { m } } ( \{ \mathbf { t r } \} _ { t ^ { \prime } } ^ { \mathcal { \bar { T } } ^ { \prime } } | \mathbf { c } ( t ) )$ is the joint probability of the trajectory of the system given the received communication $\mathbf { c } ( t )$ when policies $\dot { \pi } ^ { c } ( \cdot )$ and $\pi ^ { m } ( \cdot )$ are followed. We proceed by negating the equation (10) and adding a second term to the objective function which is constant with respect to the decision variables of the problem to have

$$\underset {\pi^ {c}} {\operatorname{argmin}} \quad \mathbb {E} _ {p (\mathfrak {s} (t))} \left\{\mathbb {E} _ {p _ {\pi^ {*}} \left(\{\mathrm{tr} \} _ {t ^ {\prime}} ^ {T ^ {\prime}} | \mathfrak {s} (t)\right)} \left\{\mathbf {g} \left(t ^ {\prime}\right) \mid \mathbf {s} (t) \right\} \right\} - \tag {11}\mathbb {E} _ {p (\mathfrak {c} (t))} \Big \{\mathbb {E} _ {p _ {\pi^ {c}, \pi^ {m}} (\{\mathsf {t r} \} _ {t ^ {\prime}} ^ {T ^ {\prime}} | \mathbf {c} (t))} \big \{\mathbf {g} (t ^ {\prime}) | \mathbf {c} (t) \big \} \Big \}, \text { s.t. } | \mathcal {C} | \leq 2 ^ {R}.$$

We replace the conditional expectation of system return by the value function $V ( \cdot )$ , [40](Ch. 3.5), and we will have

$$
\begin{array}{l} \underset {\pi^ {c}} {\operatorname{argmin}} \quad \mathbb {E} _ {p (\mathbf {s} (t))} \left\{V ^ {\pi^ {*}} (\mathbf {s} (t)) \right\} - \mathbb {E} _ {p (\mathbf {c} (t))} \left\{V ^ {\pi^ {m}} (\mathbf {c} (t)) \right\}, \\ \text { s.t. } \quad | \mathcal {C} | \leq 2 ^ {R}. \tag {12} \\ \end{array}
$$

Note that the empirical joint distribution of $\mathbf { c } ( t )$ can be obtained by following the communication policy $\dot { \pi } ^ { c }$ on the empirical distribution of ${ \bf s } ( t )$ .
