Integrating the above differential inequality, we obtain the following estimate for every $d \in L ^ { \infty } \left( \mathbb { R } _ { + } ; \mathbb { R } \right)$ :

$$
\begin{array}{l} \left| x (t) \right| \leq \exp \left(- \min (c, \sigma) t / 2\right) \sqrt {\max (\Gamma , 1 / \Gamma)} | x (0) | \\ + \sqrt {\frac {\max (\Gamma , 1)}{c \min (c , \sigma)}} \| d \| _ {\infty} + | \theta | \sqrt {\frac {\sigma}{\min (\Gamma , 1) \min (c , \sigma)}} \tag {131} \\ \end{array}
$$

Estimate (131) guarantees the p-ISS property for system (129) with gain function $\gamma ( s ) = s \sqrt { \frac { \operatorname* { m a x } \left( \Gamma , 1 \right) } { c \operatorname* { m i n } \left( c , \sigma \right) } }$ and residual constant $\alpha = \left| \theta \right| \sqrt { \frac { \sigma } { \operatorname* { m i n } \left( \Gamma , 1 \right) \operatorname* { m i n } \left( c , \sigma \right) } }$ . In the disturbance-free case we guarantee p-UGAS with residual constant $\alpha = \left| \theta \right| \sqrt { \frac { \sigma } { \operatorname* { m i n } \left( \Gamma , 1 \right) \operatorname* { m i n } \left( c , \sigma \right) } }$

Although the p-ISS property is a very strong stability property and estimate (131) also guarantees exponential convergence, there is a problem here. The problem is that the residual constant is proportional to the absolute value of the unknown parameter, i.e., $\alpha \sim | \theta |$ . One may say that the stability analysis that we have performed is very conservative and that is why we get such a result. Unfortunately, this is not the case. If we perform an analysis of the equilibrium points of system (129) then we can easily see that there are equilibrium points of the form $x = \left( y , z \right) = \left( \pm \sqrt { \frac { \sigma \left( \theta - c \right) } { \Gamma + \sigma q } } , - \frac { \Gamma c + \theta \sigma q } { \Gamma + \sigma q } \right)$ when $\theta > c$ . Consequently, the residual constant (which must be large enough so that the ball with radius the residual constant contains all equilibrium points) must be approximately proportional to the absolute value of the unknown parameter.
