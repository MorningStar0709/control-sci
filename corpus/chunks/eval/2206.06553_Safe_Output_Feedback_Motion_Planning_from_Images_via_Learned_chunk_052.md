$$
\begin{array}{l} \underset {W _ {e}, \lambda_ {o}, \mu_ {i} ^ {e} (x)} {\text { minimize }} \quad \frac {1}{\lambda_ {c}} \frac {\bar {\lambda} _ {D _ {c}} (W _ {c})}{\sqrt {\underline {{\lambda}} _ {D _ {c}} (W _ {c})}} \rho \\ \underline {{\beta}} \mathbf {I} _ {n _ {x}} \preceq W _ {e} \preceq \bar {\beta} \mathbf {I} _ {n _ {x}} \\ \end{array}
$$

We change the objective function here to account for the known components of the disturbance bound; specifically, from (16) we know that any disturbance from the innovation term in the observer will be pre-multiplied by ${ \scriptstyle \frac { 1 } { 2 } } \rho M _ { e }$ ; this accounts for the extra factors in the objective. As before, we introduce Lagrange multipliers $\mu _ { e } ( x )$ to only enforce (D.13) over the subset $D _ { e } \subseteq { \mathcal { X } }$ , and restrict the eigenvalues of $W _ { e }$ to ensure its inverse is well-defined. To solve (D.14) approximately, we drop the extra square root factor, and do the same line search on $\lambda _ { e }$ and bisection search on the condition number as for the CCM, but instead minimizing $\rho$ instead of solving a feasibility problem in each SoS program. We note that for both CCM and OCM synthesis, if the system is linear, ${ \mathrm { i . e . , ~ } } { \dot { x } } = A x + B u$ , then $W _ { c }$ and $W _ { e }$ can be chosen as constant, simplifying both (D.12) and (D.14) to a standard semidefinite program (SDP), since all constraints which were previously polynomial functions of x simplify to constant linear matrix inequalities (LMIs).
