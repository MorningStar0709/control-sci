# B. The case of undirected topology

The interaction topology is modeled as an connected undirected graph ${ \bar { \mathcal { G } } } ,$ , as shown in Figure 2. The control input $u _ { i } ( t )$ adopts the adaptive control protocol (30). The initial states for each agent are randomly chosen from [−2, 2]. By Theorem IV.4, we can choose $\gamma = 1$ . Let $c _ { i } ( 0 ) ~ = ~ 1 , ~ i ~ = ~ 1 , . . . , 6 .$ The revolutions of adaptive gains $c _ { i } ( t )$ are shown in Figure 4. Considering the relative state of one sample path for each agent $x _ { i 1 } ( t ) - x _ { 1 1 } ( t )$ and $x _ { i 2 } ( t ) - x _ { 1 2 } ( t )$ , we have Figure $^ { 6 , }$ which indicates that states of the six agents tend to be a.s. consensus over time. For m.s. consensus analysis, we generate $1 0 ^ { 2 }$ sample paths. Then, considering the behaviors of the m.s. relative states $E \| x _ { i 1 } ( t ) - x _ { 1 1 } ( t ) \| ^ { 2 }$ and $E \| x _ { i 2 } ( t ) - x _ { 1 2 } ( t ) \| ^ { 2 }$ , we obtain Figure $^ { 6 , }$ which demonstrates that the six agents reach m.s. consensus.
