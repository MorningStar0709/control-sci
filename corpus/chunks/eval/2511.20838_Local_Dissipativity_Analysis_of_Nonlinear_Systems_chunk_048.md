# 6.3 Three dimensional polynomial system

Finally, we implemented Problem 2 to determine the $\mathcal { L } _ { 2 }$ gain of the following three-dimensional polynomial system,

$$
\mathcal {G} _ {\Omega} \left(x _ {0}\right): \left\{ \begin{array}{l l} \dot {x} _ {1} & = - x _ {1} - x _ {3} + x _ {2} - x _ {3} x _ {2} ^ {2} \\ \dot {x} _ {2} & = - x _ {2} x _ {3} ^ {2} - x _ {2} + u \\ \dot {x} _ {3} & = \frac {1}{2} \left(x _ {1} - x _ {3}\right) \\ y & = x _ {2} \end{array} \right. \tag {29}
$$

over $\mathcal { X } \ = \ [ - 0 . 5 , 0 . 5 ] \times [ - 0 . 5 , 0 . 5 ] \times [ - 0 . 5 , 0 . 5 ]$ , where $\Omega \subseteq { \mathcal { X } }$ is a RPI set determined using the SOSAnalysis package [20]. Using 24,576 simplices, the gain was determined to be $\gamma \geq 1 . 6 5$ . In contrast, when applying [20], a viable gain was searched for using the SOSAnalysis package using a simple line search. The gain found by [20] was $\gamma \geq 1 . 0 1$ .
