Remark 6. In the absence of restrictions on discrete and continuous control actions, maximum hands-off hybrid control for the switched systems in Examples 1 and 2 can be attempted to be designed using a convex relaxation of the optimization problem (2.4) [4]. In particular, an ℓ1-norm approximation can be considered. However, for a solution of the ℓ1-optimization problem to match a solution of the ℓ0-optimization problem, we need to rely on restricted isometry property (resp., range space property) of the transition matrix, $\Phi _ { \nu _ { T } } : = \left( \prod _ { t = 1 } ^ { T - 1 } A _ { \nu ( t ) } b _ { \nu ( 0 ) } \right. \quad \cdots \quad A _ { \nu ( T - 1 ) } b _ { \nu ( T - 2 ) } \quad b _ { \nu ( T - 1 ) } \left. \right)$ Φν ∶ and existence of a unique solution to $\Phi _ { \nu _ { T } } \mu _ { T } = - \prod _ { t = 0 } ^ { T - 1 } A _ { \nu ( t ) } \xi ,$ T 1 , as demonstrated in [4]. In general, verifying the above conditions numerically is not a straightforward task. We typically rely on sufficient conditions on the elements of the matrices, $( A _ { i } , b _ { i } ) , i \in \mathcal { N }$ for this purpose, see e.g., [4, Propositions 3 and 4]. We note that in the setting of Examples 1 and 2, [4, Propositions 3 and 4] do not hold. We have, however, been able to solve the optimization problem (2.4) for these examples using the techniques proposed in this paper.
