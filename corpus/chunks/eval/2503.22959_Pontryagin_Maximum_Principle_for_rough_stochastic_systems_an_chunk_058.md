Hence $\boldsymbol { u } ^ { * , \eta }$ is an optimal control for the rough control problem for $\mathbb { P } ^ { \prime } { \mathrm { - a . e } }$ . Brownian rough path $\pmb { \eta } = \mathbf { B } ( \omega ^ { \prime } )$ . 5.3. SMP for generalized pathwise stochastic control problems. The integrability condition on the payoff required in Proposition 5.2 is guaranteed if the functions g and h are bounded or if Assumption 5.1 holds and the randomized solution ${ \bar { X } } ^ { \bar { u } }$ is integrable. Unfortunately, the latter cannot be inferred from the a priori estimate given in Proposition 3.4. The process Xη belongs to $H ^ { 1 } ( \Omega ^ { \prime \prime } ; [ 0 , T ] )$ for any η. However, since controls are anticipative, this does not guarantee that the randomized process $\bar { X } ^ { \mathbf { B } }$ belongs to $H ^ { 1 } ( \Omega ; [ 0 , T ] )$ . Without integrability of the randomized SDE the conditional expectation (5.10) may not be well-defined. In particular, the above proposition may not apply to LQ problems as stated.

To overcome the integrability problem and to extend our analysis to LQ problems we introduce the generalized cost function

$$\tilde {J} (\bar {u}; \omega) := J (u ^ {\eta}; \eta) | _ {\eta = \mathbf {B} (\omega^ {\prime})}, \tag {5.14}$$

for the pathwise stochastic problem, along with the corresponding value function

$$\tilde {\mathcal {V}} (\omega) := \operatorname{essinf} _ {\bar {u} \in \bar {\mathcal {A}}} \tilde {J} (\bar {u}; \omega). \tag {5.15}$$

In what follows, we call the resulting pathwise control problem a generalized pathwise stochastic control problem.

Remark 5.2. (1). Thanks to the joint measurability established in Proposition 3.10, the function $\tilde { J }$ is measurable, and thus the above essential infimum is well-defined.
