# 7.5. Proof of Theorem 4

To prove Theorem 4, we need to make use of the rank condition of $G _ { \theta }$ and R in (16). We first show in Lemma 5 about the null space and rank condition induced from the condition rank $\ d ( \mathcal { H } ) = \ d \ b { n } + T \ d { m }$ .

Lemma 5 If the observability matrix $\mathcal { O } _ { [ 0 , T _ { 0 } - 1 ] }$ is full column rank, then the null space $\mathcal { N } ( G _ { \theta } )$ is the same as $\mathcal { N } ( \mathcal { H } )$ . Moreover, if rank $( \mathcal { H } ) = n \overset { \cdot } { + } T m ,$ , then rank $( G _ { \theta } ) = n + T m$ .

Proof We first prove that the null space $\mathcal { N } ( G _ { \theta } )$ is the same as $\mathcal { N } ( \mathcal { H } )$ from (i) and (ii) :

(i) For all $\pmb q \in \mathcal { N } ( \mathcal { H } )$ ), we have $[ \mathcal { H } _ { y } ] \pmb { q } = \mathbb { 0 } _ { T n }$ and $[ \mathcal { H } _ { u } ] \pmb { q } = \mathbb { 0 } _ { T m }$ . Plugging in the expression of $G _ { \theta }$ yields $G _ { \theta } \mathbf { q } = \mathbb { 0 } _ { T m + n }$ . Namely, $\pmb { q } \in \mathcal { N } ( G _ { \pmb { \theta } } )$ .

(ii) For all ${ \pmb v } \in \mathcal { N } ( G _ { \pmb \theta } )$ , we have

$$
\left[ \begin{array}{c} \mathcal {H} _ {u} ^ {T _ {0} - 1: T - 1} - \left(\boldsymbol {I} _ {T - T _ {0}} \otimes \theta\right) \mathcal {H} _ {y} ^ {T _ {0} - 1: T - 1} \\ \mathcal {H} _ {y} ^ {0: T _ {0} - 1} \\ \mathcal {H} _ {u} ^ {0: T _ {0} - 2} \end{array} \right] \boldsymbol {v} = \mathbb {O} _ {T m + T _ {0} d}, \tag {30}
$$

which gives

$$
\begin{array}{l} \mathcal {H} _ {u} ^ {k} \boldsymbol {v} = \theta \mathcal {H} _ {y} ^ {k} \boldsymbol {v} \text {   for   } k = T _ {0}, \dots , T - 1 \\ \mathcal {H} _ {y} ^ {0: T _ {0} - 1} \boldsymbol {v} = \mathbb {O} _ {T _ {0} d} \tag {31} \\ \mathcal {H} _ {u} ^ {0: T _ {0} - 2} \boldsymbol {v} = \mathbb {O} _ {T _ {0} m}. \\ \end{array}
$$

From (23), we have ,

$$\mathcal {H} _ {y} ^ {k} = \boldsymbol {C} \boldsymbol {A} ^ {T _ {0}} \mathcal {O} _ {[ 0, T _ {0} - 1 ]} ^ {\dagger} \mathcal {H} _ {y} ^ {k - T _ {0}: k - 1} + \left(\mathcal {T} _ {[ T _ {0}, T _ {0} ]} - \boldsymbol {C} \boldsymbol {A} ^ {T _ {0}} \mathcal {O} _ {[ 0, T _ {0} - 1 ]} ^ {\dagger} \mathcal {T} _ {[ 0, T _ {0} - 1 ]}\right) \mathcal {H} _ {u} ^ {k - T _ {0}: k - 2} \tag {32}$$

for $k = T _ { 0 } , \cdots , T - 1$ .

Plugging (31) in (32) induces $\mathcal { H } _ { y } ^ { k } v = \mathbb { 0 } _ { d }$ and $\mathcal { H } _ { u } ^ { k } v \ = \ \mathbb { 0 } _ { m }$ for $k = 0 , \cdots , T - 1$ . Hence, $\mathcal { H } { \boldsymbol { v } } = \mathbb { 0 } _ { T m + T d }$ . Namely, $\pmb { v } \in \mathcal { N } ( \mathcal { H } )$ .
