We collect below a set of useful definitions:

Definition B.7 (Various forms of stability). The closed positive invariant set A is

(a) locally stable if, for each $\varepsilon > 0$ , there exists a $\delta = \delta ( \varepsilon ) > 0$ such that $| x | _ { \mathcal { A } } < \delta$ implies $| \phi ( i ; x ) | _ { \mathcal { A } } < \varepsilon$ for all $i \in \mathbb { I } _ { \geq 0 }$ .   
(b) unstable, if it is not locally stable.   
(c) locally attractive if there exists $\eta > 0$ such that $| x | _ { \mathcal { A } } < \eta$ implies $| \phi ( i ; x ) | _ { \mathcal { A } } \to 0$ as $i  \infty$ .
