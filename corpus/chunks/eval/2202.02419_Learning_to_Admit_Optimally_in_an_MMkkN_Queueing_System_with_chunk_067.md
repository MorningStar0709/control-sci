# A.2. Proof of Lemma 6

Proof of Lemma 6. We first bound the probability term $\begin{array} { r } { \mathbb { P } \left( \sum _ { j = 1 } ^ { i } y _ { j } < n , \sum _ { j = 1 } ^ { i + 1 } y _ { j } \ge n \right) } \end{array}$ using the probability of the first event. We take $p _ { i } = 1 - q _ { i } = \exp \left( - i ^ { 1 - \epsilon } \right)$ and then use the Chernoff bound to get

$$\mathbb {P} \left(y _ {1} + \dots + y _ {i} < n, y _ {1} + \dots + y _ {i + 1} \geq n\right) \leq \mathbb {P} \left(y _ {1} + \dots + y _ {i} \leq n\right) \leq \min _ {t \geq 0} e ^ {t n} \prod_ {j = 1} ^ {i} \frac {p _ {j}}{e ^ {t} - \left(1 - p _ {j}\right)}. \tag {62}$$

Take $b = \lceil ( \log \left( n + 1 \right) ) ^ { \frac { 1 } { 1 - \epsilon } } \rceil$ and $t \geq 0$ such that $\textstyle e ^ { t } = { \frac { n + 1 } { n } } q _ { i }$ . From (62), for $i \geq d \geq b$ we have

$$
\begin{array}{l} \mathbb {P} \left(y _ {1} + \dots + y _ {i} \leq n\right) \leq \left(\frac {n + 1}{n}\right) ^ {n} q _ {i} ^ {n} \prod_ {j = 1} ^ {i} \frac {p _ {j}}{\frac {1}{n} \left(1 - p _ {i}\right) + \left(p _ {j} - p _ {i}\right)} \leq \left(\frac {n + 1}{n}\right) ^ {n} q _ {i} ^ {n} \prod_ {j = 1} ^ {i} p _ {j} \prod_ {j = 1} ^ {b} \frac {1}{p _ {j} - p _ {i}} \prod_ {j = b + 1} ^ {i} \frac {n}{1 - p _ {i}} \\ \leq \left(\frac {n + 1}{n}\right) ^ {n} q _ {i} ^ {n - (i - b)} n ^ {i - b} \prod_ {j = b + 1} ^ {i} p _ {j} \prod_ {j = 1} ^ {b} \frac {1}{1 - \exp \left(- \left(i ^ {1 - \epsilon} - j ^ {1 - \epsilon}\right)\right)}. \tag {63} \\ \end{array}
$$

Since $q _ { i } \leq 1$ and $n \geq i - b .$ , we have $\begin{array} { r } { \left( \frac { n + 1 } { n } \right) ^ { n } q _ { i } ^ { n - ( i - b ) } \le e } \end{array}$ . By concavity and gradient inequality, for $1 \leq j \leq i$ we have $\begin{array} { r } { i ^ { 1 - \epsilon } - j ^ { 1 - \epsilon } \ge \frac { 1 - \epsilon } { i ^ { \epsilon } } \left( i - j \right) } \end{array}$ .
