# C VARIABLE TABLE

A large variety of variables could be defined by BEAR users. Three inputs (Filename, Weatherfile, Location) are required for setting up a basic building environment, while users could modify other variables for better simulation accuracy. Variable U-WAll contains the U values of each wall used in the building model, which could be changed if user would like to replace the material of the wall. Variable Target should be used for self-define control objectives. Variable Time-Reso should be modified if user would like to change the sample time of the simulation. Variable Reward-gamma can change the reward function coefficient. Variable SHGC is based on the materials of window. Both SHGC and Ground temperature could have partial impact on zone temperature depending on the building structure, thus SHGC-weight and Ground-weight are provided for user to tune. Variable Full-Occ is used to set up occupancy of each zone. Variable Activity-sch represents the metabolic heat of different activities. Variable ACmap and Max-power can be used to change the location and the maximum output of the HVAC system.
