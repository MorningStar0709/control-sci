# C.5 Stability of Feedback Interconnected Systems

Theorem C.1 (Landau 1980) Consider a linear time invariant block described by (C.1) and (C.2) belonging to the class L(Λ) in feedback connection with a discrete linear time-varying block described by (C.27) and (C.28) belonging to the class $N ( { \cal { T } } ) . { \cal { I } } f \mathrm { : }$

$$\Lambda - \Gamma (t) \geq 0 \tag {C.39}$$

the following properties hold:

(P1)

$$\lim _ {t \to \infty} x (t) = 0; \quad \lim _ {t \to \infty} y (t) = 0; \quad \lim _ {t \to \infty} u (t) = - \lim _ {t \to \infty} \bar {y} (t) = 0 \tag {C.40}$$

with $\bar { P } ( t ) , Q ( t ) , S ( t ) , R ( t )$ bounded or unbounded for $t > 0$

$$

$$

with Q(t), S(t), R(t) bounded or unbounded for $t > 0$ .

$$\bar {x} (t) ^ {T} P (t) \bar {x} (t) \leq C _ {1} < \infty ; \quad \forall t > 0 \tag {C.42}$$

$$\lim _ {t \to \infty} \bar {x} (t) ^ {T} P (t) \bar {x} (t) = \text { const } \tag {C.43}$$
