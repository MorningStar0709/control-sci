# D Martingales

Proof The proof is based on the use of the near supermartingale convergence theorem (Theorem D.1). The appropriate near supermartingale is:

$$T (t) = \frac {V (t)}{t} + \sum_ {i = 1} ^ {t - 1} \frac {1}{i + 1} \frac {V (i)}{i} \tag {D.18}$$

where:

$$
\begin{array}{l} V (t) = x ^ {T} (t) P x (t) + \bar {x} ^ {T} (t) \bar {P} (t) \bar {x} (t) + \sum_ {i = 0} ^ {t - 1} x ^ {T} (i) M _ {0} x (i) \\ + \sum_ {i = 0} ^ {t - 1} u ^ {T} (i + 1) K u (i + 1) \tag {D.19} \\ \end{array}
$$

Taking into account (D.6) and (D.8), one has:

$$
\begin{array}{l} \sum_ {i = 0} ^ {t} y ^ {T} (i + 1) u (i + 1) = - \sum_ {i = 0} ^ {t} \bar {y} ^ {T} (i + 1) \bar {u} (i + 1) \\ + \sum_ {i = 0} ^ {t} \bar {y} ^ {T} (i + 1) \omega (i + 1) \tag {D.20} \\ \end{array}
$$

The expressions for $\begin{array} { r } { \sum _ { i = 0 } ^ { t } y ^ { T } ( i + 1 ) u ( i + 1 ) } \end{array}$ and $\textstyle \sum _ { i = 0 } ^ { t } { \bar { y } } ^ { T } ( i + 1 ) \bar { u } ( i + 1 )$ can be obtained from Lemmas C.5 and C.7, since the two blocks involved belong to the classes $L ( \varLambda )$ and $N ( T )$ respectively. Combining (C.26) and (C.34) through the use of (D.20) and rearranging the various terms taking into account (D.19), one obtains:

$$
\begin{array}{l} V (t + 1) = x ^ {T} (t + 1) P x (t + 1) + \bar {x} ^ {T} (t + 1) \bar {P} (t + 1) \bar {x} (t + 1) \\ + \sum_ {i = 0} ^ {t} x ^ {T} (i) M _ {0} x (i) + \sum_ {i = 0} ^ {t} u ^ {T} (i + 1) K u (i + 1) \\ = x ^ {T} (0) P x (0) + \bar {x} ^ {T} (0) \bar {P} (0) \bar {x} (0) - \sum_ {i = 0} ^ {t} [ x ^ {T} (i), u ^ {T} (i + 1) ] M {\left[ \begin{array}{c} x (i) \\ u (i + 1) \end{array} \right]} \\ - \sum_ {i = 0} ^ {t} [ \bar {x} ^ {T} (i), \bar {u} ^ {T} (i + 1) ] \bar {M} (t) \left[ \begin{array}{c} \bar {x} (i) \\ \bar {u} (i + 1) \end{array} \right] \\ - \sum_ {i = 0} ^ {t} \bar {y} ^ {T} (i + 1) [ \Lambda - \Gamma (i) ] \bar {y} (i + 1) \\ + 2 \sum_ {i = 0} ^ {t} \bar {y} ^ {T} (i + 1) \omega (i + 1) \tag {D.21} \\ \end{array}
$$

Since $\Lambda - \Gamma ( t ) \geq 0 \left( { \mathrm { f r o m } } \left( \mathrm { D } . 1 4 \right) \right)$ and $\bar { M } ( t ) \geq 0$ (from (C.34)), one concludes from (D.21) that:

$$V (t + 1) \leq V (t) + 2 \bar {y} (t + 1) \omega (t + 1) \tag {D.22}$$

and:

$$\frac {V (t + 1)}{t + 1} \leq \frac {V (t)}{t} - \frac {1}{t + 1} \frac {V (t)}{t} + \frac {2}{t + 1} \bar {y} (t + 1) \omega (t + 1) \tag {D.23}$$
