# B. Comparison with identification-based ETS and STS control

In this part, we compare the data-driven approaches described above to an alternative approach, consisting of least-squares identification of the system matrices and subsequent modelbased ETS and STS as presented in Sections III and IV. In the system identification step, the following least-squares problem is considered

$$[ \hat {A} \hat {B} ] ^ {\top} = \arg \min _ {(A B)} \sum_ {T = 0} ^ {\rho} \| x (T + 1) - A x (T) - B u (T) ] \| _ {2} ^ {2}.$$

The least-squares solution $[ \hat { A } \hat { B } ] ^ { \intercal }$ is given by

$$[ \hat {A} \hat {B} ] ^ {\top} = ([ X ^ {\top} U ^ {\top} ] ^ {\top} [ X ^ {\top} U ^ {\top} ]) ^ {- 1} [ X ^ {\top} U ^ {\top} ] ^ {\top} X _ {+} ^ {\top}$$

which is the estimation of the real system matrices A and B. Next, we will show the the results of identification-based ETS and STS control.

(Identification-based ETS and STS control) Using the leastsquares approach and the same $\rho = 8 0 0$ measurements as in the data-driven control, the system matrices are estimated as

$$
\hat {A} = \left[ \begin{array}{l l} 1. 0 0 0 0 & 0. 0 9 9 5 \\ 0. 0 0 0 0 & 0. 9 9 0 0 \end{array} \right], \hat {B} = \left[ \begin{array}{l} 0. 0 0 0 5 \\ 0. 0 1 0 0 \end{array} \right].
$$

Subsequently, by solving the model-based approach in Remark 5 with the same parameters as in data-driven control, the controller and the triggering matrices were computed as follows

$$
K = \left[ \begin{array}{l l} - 1. 9 7 7 5 & - 8. 0 8 7 3 \end{array} \right], \Omega = \left[ \begin{array}{l l} 0. 0 0 1 1 & 0. 0 0 3 7 \\ 0. 0 0 3 7 & 0. 0 1 6 8 \end{array} \right].
$$
