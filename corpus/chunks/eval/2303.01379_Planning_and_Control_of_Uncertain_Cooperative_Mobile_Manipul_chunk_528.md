# 6.4.3 Simulation Results

We consider here the simulation of two inverted pendulum connected by a spring and a damper [372], with dynamics:

$$
\begin{array}{l} J _ {1} \ddot {x} _ {1 _ {1}} = g _ {s} \sin (x _ {1 _ {1}}) - \frac {1}{4} F _ {s} \cos (x _ {1 _ {1}} - \theta_ {f}) - T _ {f _ {1}} + u _ {1} \\ J _ {2} \ddot {x} _ {1 _ {2}} = 1. 2 5 g _ {s} \sin (x _ {1 _ {2}}) + \frac {1}{4} F _ {s} \cos (x _ {1 _ {2}} - \theta_ {f}) - T _ {f _ {2}} + \sigma_ {f} (t) u _ {2}, \\ \end{array}
$$

where $\begin{array} { r } { F _ { s } : = 1 5 0 ( d _ { s } - \frac { 1 } { 2 } ) + \dot { d } _ { s } } \end{array}$ is the force between the connection points of the spring and damper at the pendulums, and

$$d _ {s} := \sqrt {\frac {1}{4} + \frac {1}{4} (\sin (x _ {1 _ {1}} - x _ {1 _ {2}})) + \frac {1}{8} (1 - \cos (x _ {1 _ {2}} - x _ {1 _ {1}}))}$$

is the distance between these connection points; $\theta _ { f }$ is defined as

$$\theta_ {f} := \tan^ {- 1} \left(\frac {\frac {1}{4} (\cos (x _ {1 _ {2}}) - \cos (x _ {1 _ {1}}))}{\frac {1}{2} + \frac {1}{4} (\sin (x _ {1}) - \sin (x _ {2}))}\right)$$

and $T _ { f _ { 1 } } , T _ { f _ { 2 } }$ are friction terms on the motors evolving according to $T _ { f _ { i } } : =$ $\tau _ { f _ { i } } + \dot { \tau } _ { f _ { i } } + \dot { x } _ { 1 _ { i } }$ , with

$$\dot {\tau} _ {f _ {i}} = \dot {x} _ {1 _ {i}} - \frac {| \dot {x} _ {1 _ {i}} |}{1 + \exp \left(- \left| \frac {\dot {x} _ {1 _ {i}}}{0 . 1} \right| ^ {2}\right)}$$

The time varying signal $\sigma _ { f } ( t )$ is taken as:

$$
\sigma_ {f} (t) = \left\{ \begin{array}{l l} 1 & \text { if } t \in [ 0, 3) \cup [ 3. 5, \infty), \\ 0. 5 & \text { if } t \in [ 3, 3. 5) \end{array} \right.
$$
