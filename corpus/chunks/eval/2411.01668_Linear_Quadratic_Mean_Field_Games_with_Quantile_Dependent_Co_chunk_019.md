Lemma 8.1 For $a > 0$ and $b > 0$ , the following inequality holds

$$| e ^ {\sqrt {a}} - e ^ {\sqrt {b}} | \leq \frac e ^ {\max \{\sqrt {| a |} , \sqrt {| b |} \}}{2 \min \{\sqrt {a} , \sqrt {b} \}} \cdot | a - b |.$$

Proof By the mean value theorem, the exists $c \in ( x , y )$ (assuming $x > y$ without loss of generality) such that $\begin{array} { r } { e ^ { c } = \frac { e ^ { x } - e ^ { y } } { x - y } } \end{array}$ x−y and this implies $| e ^ { \sqrt { a } } - e ^ { \sqrt { b } } | \leq e ^ { \operatorname* { m a x } \{ | \sqrt { a } | , | \sqrt { b } | \} } | \sqrt { a } - \sqrt { b } |$ . We note that $\begin{array} { r } { \sqrt { a } - \sqrt { b } = \frac { 1 } { \sqrt { a } + \sqrt { b } } ( a - b ) } \end{array}$ , and hence $\begin{array} { r } { | \sqrt { a } - \sqrt { b } | = \frac { 1 } { \sqrt { a } + \sqrt { b } } | a - b | \leq \frac { 1 } { 2 M _ { 0 } } | a - b | } \end{array}$ where $M _ { 0 } : = \operatorname* { m i n } \{ { \sqrt { a } } , { \sqrt { b } } \} > 0$ . Combining the observations above leads to the desired result.

An immediate consequence of Lemma 8.1 is the following.

Corollary 8.2 For $a > 0 , b > 0$ and $c > 0$ , the following inequality holds

$$| e ^ {c \sqrt {a}} - e ^ {c \sqrt {b}} | \leq \frac {e ^ {\max \{c \sqrt {| a |} , c \sqrt {| b |} \}}}{2 \min \{c \sqrt {a} , c \sqrt {b} \}} \cdot c ^ {2} | a - b |.$$
