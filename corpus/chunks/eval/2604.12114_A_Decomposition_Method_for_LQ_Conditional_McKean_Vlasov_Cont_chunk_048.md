# Appendix B.2. Stochastic HJB Equation and Optimal Control

Following [35, 36, 37, 38], and temporarily disregarding the constraint on the admissible control set $\breve { U } _ { t }$ , the value function (B.3) satisfies the stochastic HJB equation given by

$$
\left\{ \begin{array}{l} \mathcal {V} _ {t} (t, x) = - \inf _ {a \in \mathbb {R} ^ {d}} H (t, x, a, \mathcal {V} _ {x} (t, x), \mathcal {V} _ {x x} (t, x), \Sigma_ {x} (t, x)) + \Sigma (t, x) d W _ {t} ^ {0}, \quad (t, x) \in [ 0, T) \times \mathbb {R} ^ {n} \\ \mathcal {V} (T, x) = \frac {1}{2} x ^ {\intercal} Q _ {T} x, \quad x \in \mathbb {R} ^ {n} \end{array} \right. \tag {B.14}
$$

where $\nu _ { t }$ denotes the partial derivative of V with respect to t, $\nu _ { x }$ and $\Sigma _ { x }$ respectively represent the gradients of V and Σ with respect to x, and $\mathcal { V } _ { x x }$ denotes the Hessian matrix of V with respect to x. Moreover, H is the stochastic Hamiltonian defined for $( t , x , \alpha , C , G , E ) \in [ 0 , T ] \times \mathbb { R } ^ { n } \times \mathbb { R } ^ { d } \times \mathbb { R } ^ { n } \times \mathbb { S } ^ { n } \times \mathbb { R } ^ { n }$ by

$$H (t, x, a, C, G, E) := \left(A _ {t} x + B _ {t} a\right) ^ {\intercal} C + \frac {1}{2} D _ {t} ^ {\intercal} G D _ {t} + D _ {t} ^ {\intercal} E + \frac {1}{2} \left(x ^ {\intercal} Q _ {t} x + 2 x ^ {\intercal} S _ {t} a + a ^ {\intercal} R _ {t} a\right). \tag {B.15}$$

Under Assumption $2 ,$ the above stochastic HJB equation is well-posed [35]. By considering the verification theorems [35, Subsection 3.2] and [37, Theorem 2] and by following the same procedure as in [37, Subsection 2.3], for any $t \in [ 0 , T ] , \check { \xi } \in \left\{ \check { \xi } \in L _ { \mathcal { F } _ { t } } ^ { 2 } \left( \Omega , \mathbb { R } ^ { n } \right) : \mathbb { E } \big [ \check { \xi } | \mathcal { F } _ { t } ^ { 0 } \big ] = 0 \right\}$ , the value function (B.3) for problem (B.1)-(B.2) has the following analytical form

$$V (t, \check {\xi}) = \frac {1}{2} \check {\xi} ^ {\intercal} P _ {t} \check {\xi}, \quad d t \otimes d \mathbb {P} - a. s. \tag {B.16}$$

Moreover, the optimal control $\alpha ^ { * }$ for problem (B.1)-(B.2) is given by

$$\alpha_ {t} ^ {*} = R _ {t} ^ {- 1} \left(B _ {t} ^ {\intercal} P _ {t} + S ^ {\intercal}\right) z _ {t} ^ {*}, \quad d t \otimes d \mathbb {P} - a. s, \tag {B.17}$$

and the corresponding optimal trajectory $z ^ { * }$ satisfies

$$d z _ {t} ^ {*} = \left(A _ {t} - B _ {t} R _ {t} ^ {- 1} (S _ {t} + P _ {t} B _ {t}) ^ {\intercal}\right) z _ {t} ^ {*} d t + D _ {t} d W _ {t}, \quad z _ {0} ^ {*} = \xi - \mathbb {E} [ \xi ], \tag {B.18}$$
