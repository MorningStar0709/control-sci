Since the inequality (32) is a Bilinear Mattrix Inequality (BMI) the following corollary presents a sufficient condition for static output feedback as an LMI.

Corollary 3.2: Under the same assumptions of theorem 3.1 if there is a solution of inequality (32) with R replaced by $I - F D R ^ { - 1 } - R ^ { - 1 } D ^ { \bar { T } } F ^ { T }$ and P is the unique positive definite solution of the Riccati equation (6), then the control law (30) is a stabilizing static output feedback for system (28).

Proof: The result follows because the term that was removed from R is the matrix $F D R ^ { - 1 } D ^ { T } F ^ { T }$ , which is positive semidefinite. ✷

Remark 3.3: Notice that from the proof of theorem 3.1 the inequality (32) is equivalent to inequality (39), provided that the Riccati equation (6) is satisfied. Furthermore, for $D = 0$ if inequality (39) is satisfied it implies that inequality (18) is also satisfied. ✷
