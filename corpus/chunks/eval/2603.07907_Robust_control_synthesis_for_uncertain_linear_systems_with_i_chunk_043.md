# APPENDIX I J-SPECTRAL FACTORIZATION

Following the procedures in [32], [42], we use the Popov IQC multiplier as an illustrative example to present the Jspectral factorization procedure.

The numerical procedure to construct a J-spectral factorization for a para-Hermitian multiplier $\Pi \ = \ \Pi ^ { \sim }$ can be summarized as follows. We first obtain a minimal state-space realization of $\Pi _ { \hat { P } }$ . The modified Popov IQC (8) is given by

$$
\Pi_ {\bar {P}} = \left[ \begin{array}{c c} \frac {- 0 . 0 1 \beta^ {2}}{(s - \alpha) (s + \alpha)} & \frac {\alpha}{s - \alpha} \\ - \frac {\alpha}{s + \alpha} & 0 \end{array} \right] + \left[ \begin{array}{c c} 0 & 1 \\ 1 & - 0. 0 1 \end{array} \right] = \hat {\Pi} _ {P} + D
$$

where we denote $\begin{array} { r l } { G ( s ) = \bigl \lbrack g _ { 1 } ( s ) } & { { } g _ { 2 } ( s ) \bigr \rbrack } \end{array}$ , with

$$
g _ {1} (s) = \frac {1}{(s - \alpha) (s + \alpha)} \left[ \begin{array}{c} - 0. 0 1 \beta^ {2} \\ - \alpha (s - \alpha) \end{array} \right],

g _ {2} (s) = \frac {1}{s - \alpha} \left[ \begin{array}{c} \alpha \\ 0 \end{array} \right],
d _ {1} (s) = s ^ {2} - \alpha^ {2},d _ {2} (s) = s - \alpha .$$

A controllable canonical-form realization can be written as

$$
A _ {1} = \left[ \begin{array}{c c} 0 & 1 \\ \alpha^ {2} & 0 \end{array} \right], \quad B _ {1} = \left[ \begin{array}{c} 0 \\ 1 \end{array} \right], \quad C _ {1} = \left[ \begin{array}{c c} - 0. 0 1 \beta^ {2} & 0 \\ \alpha^ {2} & - \alpha \end{array} \right],

A _ {2} = \alpha , \quad B _ {2} = 1, \quad C _ {2} = \left[ \begin{array}{c} \alpha \\ 0 \end{array} \right].
$$

After augmentation, we obtain

$$
A = \left[ \begin{array}{c c} A _ {1} & 0 \\ 0 & A _ {2} \end{array} \right] = \left[ \begin{array}{c c c} 0 & 1 & 0 \\ \alpha^ {2} & 0 & 0 \\ - \frac {}{} 0 & - \frac {}{} 0 & \frac {}{} \alpha \end{array} \right],

B = \left[ \begin{array}{c c} B _ {1} & 0 \\ 0 & B _ {2} \end{array} \right] = \left[ \begin{array}{c c} 0 & 0 \\ 1 & 0 \\ - 0 & - 1 \end{array} \right],

C = \left[ \begin{array}{c c} C _ {1} & C _ {2} \end{array} \right] = \left[ \begin{array}{c c c} - 0. 0 1 \beta^ {2} & 0 & \alpha \\ \alpha^ {2} & - \alpha & 0 \end{array} \right].
$$

To obtain a minimal realization, we examine the observability of $( A , C )$ .
