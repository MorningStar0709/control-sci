# A. Special Case: Control-Affine Systems

When the true dynamics (1) is known to have a controlaffine structure, we can learn its Lyapunov control without explicitly learning a feedback controller. Consider a controlaffine system in standard form

$$\dot {x} (t) = f _ {1} (x (t)) + f _ {2} (x (t)) u (t) \tag {20}$$

for $x ( t ) \ \in \ \mathcal { X } \ \subset \ \mathbb { R } ^ { n } , u ( t ) \ \in \ \mathcal { U } \ \subset \ \mathbb { R } ^ { m }$ . Exploiting this structural knowledge, we can also form the projected model $f ^ { * }$ to be control-affine as well, i.e., $f ^ { * } ( x , u ) \ = \ f _ { 1 } ^ { * } ( x ) \ +$ $f _ { 2 } ^ { * } ( x ) u$ . Then, the Lyapunov stability condition (5) becomes

$$\nabla V (x) ^ {\top} (f _ {1} ^ {*} (x) + f _ {2} ^ {*} (x) u ^ {*} (x)) \leq - \alpha V (x) \tag {21}$$

for all $x \in \mathcal { X } \setminus \{ 0 \}$ with a given $\alpha > 0$ . Due to the controlaffine structure, we can easily compute the controller that minimizes the LHS of (21) instead of directly learning it:

$$u ^ {*} (x) = \underset {u \in \mathcal {U}} {\arg \min} \nabla V (x) ^ {\top} f _ {2} ^ {*} (x) u. \tag {22}$$

Considering $\mathcal { U } = \left\{ u \in \mathbb { R } ^ { m } : - u _ { \mathrm { l i m } } \leq u \leq u _ { \mathrm { l i m } } \right\}$ for some $u _ { \mathrm { l i m } } \geq 0 , ( 2 2 )$ is reduced to

$$u ^ {* (i)} (x) = - \operatorname{sign} \left(\left\{\nabla V (x) ^ {\top} f _ {2} ^ {*} (x) \right\} ^ {(i)}\right) u _ {\lim} ^ {(i)} \tag {23}$$

for each $i ~ = ~ 1 , 2 , \ldots , m$ where $( \cdot ) ^ { ( i ) }$ denotes the i-th component. Then, we can just optimize the neural networks for the dynamics model and the Lyapunov function as before with the induced feedback controller (23).

Constructing the projected model $f ^ { * }$ to be control-affine can be simply done by setting the nominal model to be control-affine. Specifically, $\hat { f } ( x , u ) = \hat { f } _ { 1 } ( x ) + \hat { f } _ { 2 } ( x ) \mathrm { ~ }$ u induces $f _ { 2 } ^ { * } = \hat { f } _ { 2 }$ . To ensure that the origin is an equilibrium for this structure as in (8), we can set

$$\hat {f} _ {2} (x) = g _ {f _ {2}} (x), \tag {24}\hat {f} _ {1} (x) = g _ {f _ {1}} (x) - g _ {f _ {1}} (0) - \hat {f} _ {2} (0) u ^ {*} (0),$$

where $g _ { f _ { 1 } } \ : \ \mathbb { R } ^ { n } \ \to \ \mathbb { R } ^ { n }$ and $g _ { f _ { 2 } } \ : \ \mathbb { R } ^ { n } \ \to \ \mathbb { R } ^ { n \times m }$ are unconstrained neural networks.
