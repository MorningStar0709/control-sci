To get a uniform upper bound on $\| \Psi _ { t , 1 } \|$ over $t \geq 1$ , we invoke the optimality of Kalman filtering. Specifically, it is well-known (see, $\mathrm { { e . g . , [ 4 ] ) } }$ that $\Sigma _ { t } = \mathbb { E } \big [ ( x _ { t } - \hat { x } _ { t } ^ { \mathrm { K F } } ) ( x _ { t } - \hat { x } _ { t } ^ { \mathrm { K F } } ) ^ { \top } \big ]$ for all $t \geq 0$ . Thus, the filter $\hat { x } _ { t } ^ { \mathrm { S K } }$ given by the recursion $\hat { x } _ { t + 1 } ^ { \mathrm { s K } } = ( A - L C ) \hat { x } _ { t } ^ { \mathrm { s K } } \bar { + } L y _ { t }$ initialized with $\dot { x } _ { 0 } ^ { \mathtt { S K } } = 0$ is a suboptimal filter. Denoting the error covariance of $\hat { x } _ { t } ^ { \mathrm { S K } }$ as $\tilde { \Sigma } _ { t } = \mathbb { E } \big [ ( x _ { t } - \hat { x } _ { t } ^ { \mathrm { S K } } ) ( x _ { t } - \hat { x } _ { t } ^ { \mathrm { S K } } ) ^ { \top } \big ]$ , we obtain from the suboptimality of $\hat { x } _ { t } ^ { \mathrm { S K } }$ that $\Sigma _ { t } \preceq \tilde { \Sigma } _ { t }$ for all $t \geq 0$ [4, Chapter 4]. Moreover, the error covariance $\tilde { \Sigma } _ { t }$ satisfies the following recursion [4, Chapter 4]:

$$\tilde {\Sigma} _ {t + 1} = (A - L C) \tilde {\Sigma} _ {t} (A - L C) ^ {\top} + W + L V L ^ {\top},$$

initialized with $\tilde { \Sigma } _ { 0 } = 0$ . Unrolling this recursion, we get that for any $t \geq 0$ ,

$$
\begin{array}{l} \tilde {\Sigma} _ {t} = \sum_ {k = 0} ^ {t - 1} (A - L C) ^ {k} (W + L V L ^ {\top}) \big ((A - L C) ^ {\top} \big) ^ {k} \\ \Longrightarrow \tilde {\Sigma} _ {t} \preceq \left\| W + L V L ^ {\top} \right\| \sum_ {k = 0} ^ {t - 1} \left((A - L C) (A - L C) ^ {\top}\right) ^ {k} \\ \Longrightarrow \| \tilde {\Sigma} _ {t} \| \leq \left\| W + L V L ^ {\top} \right\| \sum_ {k = 0} ^ {t - 1} \| A - L C \| ^ {2 k} \leq \left\| W + L V L ^ {\top} \right\| \frac {\kappa_ {F} ^ {2}}{1 - \gamma_ {F} ^ {2}}, \tag {E.6} \\ \end{array}
$$

where the last inequality follows from similar arguments to those for Lemma 2.1. Using (E.6) in (E.5), we obtain that (E.2) holds for all $t \geq 1$ .
