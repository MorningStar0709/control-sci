# Exercise A.15: Convex combination

Suppose $S \subset \mathbb { R } ^ { n }$ is convex. Let $\{ x _ { i } \} _ { i = 1 } ^ { k }$ be points in S and let $\{ \mu ^ { i } \} _ { i = 1 } ^ { k }$ be scalars such that $\mu ^ { i } \geq 0$ for $i = 1 , 2 , \ldots , k$ and $\textstyle \sum _ { i = 1 } ^ { k } \mu ^ { i } = 1$ . Show that

$$\left(\sum_ {i = 1} ^ {k} \mu^ {i} x _ {i}\right) \in S.$$
