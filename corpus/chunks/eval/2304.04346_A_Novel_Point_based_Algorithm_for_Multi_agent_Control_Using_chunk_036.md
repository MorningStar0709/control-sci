In the case that a lower bound for the reward minimization problem can be expressed as a function of the marginal belief, one can use a similar way to obtain the corresponding α-constraints as well. Suppose that $\underline { { v } } ^ { b ^ { 1 } }$ is a lower bound of the minimum total reward at belief b for all b such that $\varphi ( b ) = b ^ { 1 }$ , then we can use an auxiliary vector $\underline { { y } } \in \mathbb { R } ^ { K ^ { 1 } }$ to formulate the α-constraints:

$$y (k ^ {1}, k ^ {2}) \geq \underline {{y}} (k ^ {1}) \quad \forall k ^ {1} \in \mathcal {K} ^ {1}, k ^ {2} \in \mathcal {K} ^ {2}, \tag {21}(b ^ {1}) ^ {T} \underline {{y}} \geq \underline {{v}} ^ {b ^ {1}}.$$
