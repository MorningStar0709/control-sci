$$
\mathbf {A} _ {0} = \left[ \begin{array}{c c c c c c} 0 & 1 & 0 & 0 & 0 & 0 \\ 1 & 0 & 2 & 0 & 0 & 0 \\ 0 & 2 & 0 & 1 & 0 & 0 \\ 0 & 0 & 2 & 0 & 1 & 0 \\ 0 & 0 & 0 & 3 & 0 & 1 \\ 0 & 0 & 1 & 0 & 0 & 0 \end{array} \right], A = \left[ \begin{array}{c c c} - 1 & 2 & 5 \\ 1 & - 1 & 2 \\ - 5 & 0 & - 1 \end{array} \right],

B ^ {T} = \left[ \begin{array}{c c c} 0 & 0 & 1 \end{array} \right], K = \left[ \begin{array}{c c c} - 0. 0 3 6 5 & 0. 4 2 9 5 & 0. 9 2 1 6 \end{array} \right].
$$

It can be verified that the NDS will achieve the constant cooperation pattern in Definition 1. The critical sampling period is ln $2 / \| A _ { c } \| ~ = ~ 0 . 0 7 3 \mathrm { s }$ by Theorem 2. The initial states of each node in each dimension are randomly generated in [0, 1000]. The state trajectory starts from 0s and ends at 50s. Five groups of noise standard deviation are given by $\begin{array} { r } { { \bf G } _ { 1 } = \mathrm { d i a g } \{ 2 , 1 , 0 . 2 \} , { \bf G } _ { 2 } = { \bf G } _ { 1 } / 2 , { \bf G } _ { 3 } = { \bf G } _ { 1 } / 4 , { \bf G } _ { 4 } = { \bf G } _ { 1 } / 2 0 , } \end{array}$ , ${ \bf G } _ { 5 } = { \bf G } _ { 1 } / 4 0$ . The inference error of an interested matrix $M _ { 0 }$ is calculated by

$$\operatorname{Er} \left(\widehat {M _ {0}}, M _ {0}\right) = \left\| \gamma \widehat {M _ {0}} - M _ {0} \right\| _ {F} / \left\| M _ {0} \right\| _ {F}, \tag {94}$$

where $\gamma$ represents the scalar ambiguity for $\widehat { M _ { 0 } }$ (if there is).
