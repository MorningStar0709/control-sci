Hence $\zeta ^ { * } ( \mathsf { X } \times \mathsf { A } ) \geq 1$ . This implies that $\zeta ^ { * } ( \mathsf { X } \times \mathsf { A } ) = 1$ . That is, both $\zeta ^ { * }$ and $\mu ^ { * }$ are probability measures. Therefore, $\left( 1 - \beta \right) \mu ^ { * } + \beta \zeta ^ { * } p _ { \mu ^ { * } }$ ∗ and $\zeta ^ { * } p _ { \mu } { } ^ { }$ ∗ are also probability measures. But it is known that if two probability measures ν and θ satisfy $\nu \leq \theta$ for any Borel set, then $\nu = \theta$ . Hence,

$$\hat {\zeta} ^ {*} = (1 - \beta) \mu^ {*} + \beta \zeta^ {*} p _ {\mu^ {*}}\mu^ {*} = \zeta^ {*} p _ {\mu^ {*}}$$

Note that given $\mu ^ { * }$ , the following optimization problems are equivalent

<table><tr><td>Problem 1</td><td>Problem 2</td></tr><tr><td> $\text{minimize}_{\zeta \in \mathcal{M}_{+}(\mathsf{X} \times \mathsf{A})} \langle \zeta, c_{\mu^{*}} \rangle$ </td><td> $\text{minimize}_{\zeta \in \mathcal{M}_{+}(\mathsf{X} \times \mathsf{A})} \langle \zeta, c_{\mu^{*}} \rangle$ </td></tr><tr><td> $\text{subject to } \hat{\zeta} \geq (1 - \beta)\mu^{*} + \beta \zeta p_{\mu^{*}}$ </td><td> $\text{subject to } \hat{\zeta} = (1 - \beta)\mu^{*} + \beta \zeta p_{\mu^{*}}$ </td></tr></table>

The second problem is a LP formulation of $\mathrm { M D P } _ { \mu ^ { * } }$ and so $\zeta ^ { * }$ is the optimal occupation measure. Hence $\pi ^ { * } \in \Lambda ( \mu ^ { * } )$ . Using the same method as in the proof of Lemma 5.1, we can also prove that $\hat { \zeta } ^ { * } = \mu ^ { * }$ using $\mu ^ { * } = \zeta ^ { * } p _ { \mu ^ { * } }$ ∗ . Hence, by $\hat { \zeta } ^ { * } = \left( 1 - \beta \right) \mu ^ { * } + \beta \zeta ^ { * } p _ { \mu ^ { * } }$ , we have

$$\mu^ {*} (\cdot) = \int_ {\mathsf {X} \times \mathsf {A}} p _ {\mu^ {*}} (\cdot | x, a) \pi^ {*} (d a | x) \mu^ {*} (d x)$$

that is, $\mu ^ { * }$ is an invariant distribution of the transition probability $p _ { \mu ^ { * } } ^ { \pi ^ { * } }$ . Hence, $\mu ^ { * } \in \Phi ( \pi ^ { * } )$ . This implies that $( \mu ^ { * } , \pi ^ { * } )$ is MFE. □
