# Ports and Subsystems Library

Subsystem: provides template for constructing a subsystem. The default is a subsystem with one input and one output. The user can add input or output ports by dragging and dropping In1 (Inport) or Out1 (Outport) from the same library. The user builds the subsystem by adding the desired blocks such as Integrator, Transfer Fcn, Sum, Gain, etc.
