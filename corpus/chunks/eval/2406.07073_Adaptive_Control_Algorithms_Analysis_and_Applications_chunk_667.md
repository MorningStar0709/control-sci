For the convergence of the parameters one has the following result:

Lemma 14.3 In the presence of a disturbance characterized by a model of the form (14.10) with the order of the polynomial $D _ { P }$ equal to $n _ { D _ { P } }$ and an estimated parameter vector $\hat { \theta }$ with $n _ { D _ { P } }$ parameters $\hat { q _ { i } } , i = 0 , 1 , \dotsc , n _ { D _ { P } } - 1$ , (14.44) implies:

$$\lim _ {t \rightarrow \infty} [ q _ {i} - \hat {q} _ {i} (t + 1) ] = 0, \quad i = 0, 1, \dots , n _ {D _ {P}} - 1 \tag {14.51}$$

Proof The proof is similar to that of Theorem 3.5. Since $\varepsilon ( t + 1 )$ goes to zero asymptotically, when $t \to \infty$ one can write from (14.45):

$$\lim _ {t \rightarrow \infty} \varepsilon (t + 1) = \lim _ {t \rightarrow \infty} \left[ \sum_ {i = 0} ^ {n _ {D _ {P}} - 1} \left(q _ {i} - \hat {q} _ {i} (t + 1)\right) q ^ {- i} \right] w _ {2} (t) = 0 \tag {14.52}$$

Since asymptotically $\hat { q _ { i } } , i = 0 , 1 , \dotsc , n _ { D _ { P } } - 1$ will be a constant, (14.52) can hold either if $q _ { i } - \hat { q _ { i } } = 0 , i = 0 , 1 , \dotsc , n _ { D _ { P } } - 1$ or if $w _ { 2 } ( t )$ is a solution of the difference equation $\begin{array} { r } { [ \sum _ { i = 0 } ^ { n _ { D _ { P } } - 1 } ( q _ { i } - \hat { q } _ { i } ) q ^ { - i } ] w _ { 2 } ( t ) = 0 } \end{array}$ . In the presence of the external disturbance, $w _ { 2 } ( t )$ which is a filtered version of the disturbance will be characterized by a difference equation of order $n _ { D _ { P } }$ and it can not be a solution of a difference equation of order $n _ { D _ { P } } - 1$ . Therefore $\begin{array} { r } { \operatorname* { l i m } _ { t \to \infty } \varepsilon ( t + 1 ) = 0 } \end{array}$ implies also the parametric convergence in the presence of the disturbance. -
