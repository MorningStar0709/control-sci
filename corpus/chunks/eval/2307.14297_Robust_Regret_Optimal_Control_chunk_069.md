Thus, Condition A) in Lemma 6 fails to holds with the complex matrix $M ( e ^ { j \theta _ { 0 } } )$ . By contraposition of Condition C) in Lemma $^ { 6 , }$ there exists a matrix $\Delta _ { 0 } \in \mathbb { C } ^ { n _ { w } \times n _ { v } }$ such that: (i) $\bar { \sigma } ( \Delta _ { 0 } ) \leq 1$ and (ii) either $F _ { U } ( M ( e ^ { j \theta _ { 0 } } ) , \Delta _ { 0 } )$ is not well-posed or $\bar { \sigma } ( F _ { U } ( M ( e ^ { j \theta _ { 0 } } ) , \Delta _ { 0 } ) ) \ge 1$ .

The offending matrix $\Delta _ { 0 }$ can be interpolated to construct an offending system $\Delta ( z )$ . Specifically, if the frequency is $\theta _ { 0 } \in ( 0 , \pi ) \cup ( \pi , 2 \pi )$ then we can construct a stable, LTI system $\Delta$ such that $\Delta ( e ^ { j \theta _ { 0 } } ) = \Delta _ { 0 }$ and $\| \Delta \| _ { \infty } \leq 1$ . Theorem 9.1 of [1] gives the construction for continuoustime and a similar construction works in discrete-time. One additional detail is required if the frequency is $\theta _ { 0 } = 0$ , π, or 2π. $( M ( e ^ { j \theta _ { 0 } } )$ is the same for $\theta _ { 0 } = 0$ and $2 \pi . )$ The frequency response $M ( e ^ { j \theta _ { 0 } } )$ is a real matrix at these frequencies. Thus if $\theta _ { 0 } = 0 , \pi$ , or $2 \pi$ then we can define a constant system $\Delta ( z ) = \Delta _ { 0 }$ with $\| \Delta \| _ { \infty } = \bar { \sigma } ( \Delta _ { 0 } ) \le 1$ .
