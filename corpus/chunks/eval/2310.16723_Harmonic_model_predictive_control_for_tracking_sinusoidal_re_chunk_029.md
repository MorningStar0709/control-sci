Proof of Theorem 2. The proof is very similar to the asymptotic stability proof of the original HMPC formulation [15, Theorem 3]. As in [15], the proof is based on finding a Lyapunov function that satisfies the asymptotic stability conditions from [15, Theorem 2]. The difference is that the Lyapunov function is taken for $x ( t ) - \mathring { x } ( t )$ , whereas in [15] the Lyapunov function is taken for $x ( t ) - \mathring { x } _ { e } ,$ since the artificial reference in [15] is a steady-state. This requires two modifications to the proof. The first is that [15, Lemma 2] has to be rewritten for the case of a harmonic reference instead of a steady-state reference. The lemma can be rewritten, with small modifications, to prove that $x ( t ) = x _ { e } ^ { * } + x _ { c } ^ { * }$ if and only if $x ( t ) = \mathring { x } ( t )$ . The second are the modifications due to the time-varying nature of the reference $( x _ { r } ( \cdot ) , u _ { r } ( \cdot ) )$ and to the difference between the offset cost function $V _ { h }$ of (9) with the one used in [15]. These differences require several modifications which are solved using simple algebraic manipulations, resulting in the same lines of reasoning and arguments used in [15, Theorem 3]. ■
