# B. Numerical analysis on the nature of $\tilde { \varepsilon } ^ { T } \tilde { \Pi } \tilde { \varepsilon }$

We recorded the first two gradients of the polynomial $\tilde { \varepsilon } ^ { T } \tilde { \Pi } \tilde { \varepsilon }$ with respect to s for 20,000 combinations of randomly sampled ε˜ and s such that $| \tilde { \varepsilon } | < 1$ and $s \in ( 0 , 1 ]$ . From the results shown in Figure 8, the data points predominantly lie on the first quadrant, suggesting that the function has positive gradients, which is a sign of monotonically increasing function. The absence of any points on the fourth quadrant motivates the conclusion: if $\tilde { \varepsilon } ^ { T } \tilde { \Pi } _ { s } \tilde { \varepsilon } > 0$ then $\mathring { \varepsilon } ^ { T } \tilde { \Pi } _ { s s } \tilde { \varepsilon } > 0$ . The results remain the same for different norm lengths of ε˜.
