Remark 6.7. Since funnel control traditionally guarantees confinement of the state in the desired funnel, a common practice is to tune the funnel to converge to arbitrarily small values, achieving thus “practical stability”, i.e., the state converging arbitrarily close to zero. Note that, in our case, the funnel functions $\rho _ { p _ { i } } , \rho _ { v _ { i } }$ , are not required to decrease to values arbitrarily close to zero, yet asymptotic stability is still achieved. In fact, the proposed control schemes can be used to achieve merely asymptotic stability results without any funnel constraints, if the latter is not required. More specifically, given the initial errors $e _ { p _ { i } } ( t _ { 0 } )$ , we can use the proposed control protocols by employing any constant values $\rho _ { p _ { i } } > | e _ { p _ { i } } ( t _ { 0 } ) | , \forall i \in \{ 1 , \ldots , n \}$ .
