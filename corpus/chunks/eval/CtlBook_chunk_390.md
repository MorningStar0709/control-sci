# A.6 Logarithms Quiz

Take this quiz then check your answers on Page 193. If the problem is numerical, you may use only the following functions on your calculator (or fewer as instructed):

$$
\begin{array}{c c c c} * & \div & + & - \end{array}
$$

It should be easy for you to get exact answers. If not, then you need to review the concepts

1. Find the log of $A * B$   
2. Find the log of $( a ^ { 2 } ) * { \sqrt { ( b ) } }$   
3. Find the log (base 10) of $\textstyle { \frac { 1 , 0 0 0 , 0 0 0 } { R } }$   
4. Find the natural log of $e ^ { 2 7 . 4 }$   
5. Find the base 10 log of $e ^ { 2 7 . 4 }$
