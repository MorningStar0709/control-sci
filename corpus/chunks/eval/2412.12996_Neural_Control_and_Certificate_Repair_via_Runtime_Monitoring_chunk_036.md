Learning-based control with Lyapunov functions. Before showing how to extract new training data and use it for policy and Lyapunov function repair, we first provide an overview of the general framework for jointly learning neural network policies and Lyapunov functions. Two neural networks are learned simultaneously, by minimizing a loss function which captures each of the defining conditions of Lyapunov functions in Proposition 2. That way, the learning process is guided towards learning a neural control policy that admits a Lyapunov function and hence satisfies the stability property.
