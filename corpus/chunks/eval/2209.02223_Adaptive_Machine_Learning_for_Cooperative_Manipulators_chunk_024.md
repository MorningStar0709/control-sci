# Appendix C

It is known the mass matrix is a positive definite matrix with lower and upper bounded limited norms such that the following inequality holds for $c _ { m } , c _ { M } > 0$

$$c _ {m} \boldsymbol {I} \leq \boldsymbol {M} _ {i} (\boldsymbol {q}) \leq c _ {M} \boldsymbol {I} \quad \forall i = 1, 2. \tag {56}$$

The nonlinear vector $h ( q , \dot { q } )$ contains the gravitational plus the Coriolis and centrifugal terms. For revolute joint robots, the function has a periodic dependence of q and quadratic dependence on q˙ [41, p.144]. Therefore, the nonlinear vector can be assumed bounded as

$$\| \boldsymbol {h} _ {i} (\boldsymbol {q}, \dot {\boldsymbol {q}}) \| \leq c _ {g} + c _ {h} \| \dot {\boldsymbol {x}} \| ^ {2}\leq c _ {g} + c _ {h} c _ {v} + c _ {h} \left\| \boldsymbol {z} \right\| ^ {2}. \tag {57}$$

Similar to (56) and (57), the bound limits can be expressed for the interconnected system (29) as

$$\bar {c} _ {m} \boldsymbol {I} \leq \bar {\boldsymbol {M}} (\boldsymbol {q}) \leq \bar {c} _ {M} \boldsymbol {I}, \tag {58}$$

and

$$\left\| \bar {\boldsymbol {h}} (\boldsymbol {q}, \dot {\boldsymbol {q}}) \right\| \leq \bar {c} _ {g} + \bar {c} _ {h} c _ {v} + \bar {c} _ {h} \| \boldsymbol {z} \| ^ {2}. \tag {59}$$

The inverse matrix $\hat { N } ^ { + }$ always exists and hence the matrix must be upper bounded, i.e.,

$$\hat {\boldsymbol {N}} ^ {+} \leq c _ {n} \boldsymbol {I} \quad \forall \boldsymbol {\theta} \in \Theta \tag {60}$$

for $c _ { n } > 0$ . Moreover, since T and D are Lipschitz in terms of the parameter variables, we can say there exist positive scalar $\varepsilon _ { t }$ and $\varepsilon _ { d }$ such that

$$\| \tilde {\boldsymbol {T}} \| = \| \boldsymbol {T} (\boldsymbol {\theta}) - \boldsymbol {T} (\hat {\boldsymbol {\theta}}) \| \leq \varepsilon_ {t} \| \tilde {\boldsymbol {\theta}} \| \tag {61}\| \tilde {\boldsymbol {D}} \| = \| \boldsymbol {D} (\boldsymbol {\theta}) - \boldsymbol {D} (\hat {\boldsymbol {\theta}}) \| \leq \varepsilon_ {d} \| \tilde {\boldsymbol {\theta}} \| \tag {62}$$

From inequalities (60) and (61) and identity (39) we get

$$\| \tilde {\boldsymbol {N}} \| \leq c _ {t} c _ {n} \| \tilde {\boldsymbol {\theta}} \|. \tag {63}$$
