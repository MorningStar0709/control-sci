# Exercise B.6: Continuity of Lyapunov function for asymptotic stability

Let X be a compact subset of $\mathbb { R } ^ { n }$ containing the origin in its interior that is positive invariant for the system $x ^ { + } ~ = ~ f ( x )$ . If $f ( \cdot )$ is continuous on X and the origin is asymptotically stable with a region of attraction X, show that the Lyapunov function suggested in Exercise B.4 is continuous on X.
