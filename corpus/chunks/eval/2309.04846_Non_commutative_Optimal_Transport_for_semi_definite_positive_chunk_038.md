The final result of this section establishes the convergence of the minimizers of the regularized problems $\{ \mathcal { F } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } \} _ { \varepsilon > 0 }$ and $\{ \mathcal { F } _ { \varepsilon } ^ { \tau } \} _ { \tau > 0 }$ , and as a consequence, the existence of the minimizers in the Γ−limits.

Theorem 3.8 (Convergence of the minima when $\varepsilon \to 0 ^ { + } \ \mathrm { o r } \ \tau \to + \infty )$ . Let $\mathcal { H } _ { 1 }$ and $\mathcal { \mathrm { H } } _ { 2 }$ be finite-dimensional Hilbert spaces. Let $C \in \mathrm { H } ( \mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 } )$ be a Hermitian operator on $\mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 }$ , $\rho \in \mathrm { H } _ { \geq } ( \mathcal { H } _ { 1 } ) , \ \sigma \in \mathrm { H } _ { \geq } ( \mathcal { H } _ { 2 } )$ be Hermitian semi-definite positive operators on $\mathcal { H } _ { 1 }$ and $\mathcal { \mathrm { H } } _ { 2 }$ . Consider the families $\{ \mathcal { F } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } \} _ { \varepsilon > 0 } , \{ \mathcal { F } _ { \varepsilon } ^ { \tau } \} _ { \tau > 0 }$ and the respective sequences of their minimizers $\{ \Gamma ^ { \varepsilon } \} _ { \varepsilon } , \{ \Gamma ^ { \tau } \} _ { \tau }$ . Then

(i) There exists $\Gamma ^ { 0 }$ such that, up to taking subsequences, $\Gamma ^ { \varepsilon }  ^ { * } \Gamma ^ { 0 }$ and

$$\lim _ {\varepsilon \to 0} \mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\Gamma^ {\varepsilon}) = \min _ {\Gamma \in \mathrm{H} _ {\geq} (\mathcal {H} _ {1} \otimes \mathcal {H} _ {2})} \mathcal {F} _ {0} ^ {\tau_ {1}, \tau_ {2}} (\Gamma) = \mathcal {F} _ {0} ^ {\tau_ {1}, \tau_ {2}} (\Gamma^ {0}).$$

(ii) $I f \operatorname { T r } [ \rho ] = \operatorname { T r } [ \sigma ]$ , there exists $\Gamma ^ { \infty }$ such that, up to taking subsequences, $\Gamma ^ { \tau }  ^ { * } \Gamma ^ { \infty }$ and

$$\lim _ {\tau \to \infty} \mathcal {F} _ {\varepsilon} ^ {\tau} (\Gamma^ {\tau}) = \min \{\mathcal {F} _ {\varepsilon} ^ {\infty} (\Gamma): \Gamma \in \mathrm{H} _ {\geq} (\mathcal {H} _ {1} \otimes \mathcal {H} _ {2}), \Gamma_ {1} = \rho , \Gamma_ {2} = \sigma \} = \mathcal {F} _ {\varepsilon} ^ {\infty} (\Gamma^ {\infty}),$$

which coincides with the solution of the von Neumann entropy regularized Non-commutative Optimal transport problem, provided that ρ and σ are density matrices (1.1).
