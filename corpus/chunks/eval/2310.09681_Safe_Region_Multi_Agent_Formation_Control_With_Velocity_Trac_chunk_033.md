# Appendix D. Proof of Lemma 3

Proof. Let $e = \hat { v } _ { i j } - ( v _ { i } - v _ { j } )$ , then

$$
\begin{array}{l} \dot {e} = \dot {\hat {v}} _ {i j} - (\dot {v} _ {i} - \dot {v} _ {j}), \\ = - \eta (\dot {\phi} _ {i j} - (v _ {i} - v _ {j})) - (u _ {i} - u _ {j}), \\ = - \eta (\hat {v} _ {i j} - (v _ {i} - v _ {j})) - (u _ {i} - u _ {j}), \\ = - \eta e - (u _ {i} - u _ {j}). \\ \end{array}
$$

Hence we get

$$
\begin{array}{l} e = \exp (- \eta t) e (0) + \int_ {0} ^ {t} \exp (- \eta (t - \tau)) (u _ {i} - u _ {j}) d \tau \\ \| e \| \leq \exp (- \eta t) \| e (0) \| + \int_ {0} ^ {t} \exp (- \eta (t - \tau)) \| u _ {i} - u _ {j} \| d \tau \\ \leq \exp (- \eta t) \| e (0) \| + \frac {2 u _ {\max}}{\eta} \\ \end{array}
$$

Using the assumption that the initial velocity of all the agents is 0 and setting the initial value of $\phi _ { i j } ( 0 ) = ( p _ { i } ( 0 ) -$ $p _ { i } ( 0 ) )$ we get

$$\| e \| \leq \frac {2 u _ {m a x}}{\eta}.$$

This completes the proof.
