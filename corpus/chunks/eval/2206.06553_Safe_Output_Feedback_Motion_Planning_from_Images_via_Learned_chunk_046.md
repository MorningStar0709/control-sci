Proof. Using a vector-valued comparison theorem [14, Corollary 1.7.1], we have that $d _ { c } ( t )$ and $d _ { e } ( t )$ can be upper bounded on [0, T ] by the solution to the upper bound of (19), i.e., $[ \dot { d } _ { c } , \dot { d } _ { e } ] ^ { \top } = \dot { \operatorname { R H S } }$ , provided that the solution to RHS exists until at least $t = T$ . Further prerequisites for applying [14, Corollary 1.7.1] are that RHS is quasimonotone nondecreasing in $[ d _ { c } , d _ { e } ] ^ { \top }$ and that RHS is continuous in t and $[ d _ { c } , d _ { e } ] ^ { \top }$ . In the following, we show that these requirements hold when using $\bar { \epsilon } _ { 1 } , \bar { \epsilon } _ { 2 }$ , and a modified, continuous version of $\bar { \epsilon } _ { 3 }$ .

First, we describe this modification to $\bar { \epsilon } _ { 3 } ( x ^ { * } ( t ) , \theta )$ , which we note is not continuous in t, in general. This is because as t changes, the max term in (B.9) can discontinuously change, based on the datapoints that fall inside $\varOmega _ { c } ( t )$ . However, we can obtain a smooth upper bound to $\bar { \epsilon } _ { 3 } ( x ^ { * } , \theta )$ by anticipating these discontinuous changes during planning (we can do this since we know the nominal dynamics and the dataset), and smoothing out these discontinuities, e.g., as in red in Fig. 11.
