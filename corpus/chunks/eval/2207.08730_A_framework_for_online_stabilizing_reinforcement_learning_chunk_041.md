# F. Bounded noise models

This section briefly reflects the key models from the survey [50]. Some Latin and Greek letters here are to be interpreted separately from the rest of the text and thus not to be confused. The easiest way to achieve bounded noise is to apply a saturation function the standard Brownian motion $B _ { t }$ . Such is the case of the sine-Wiener process $\begin{array} { r } { Z _ { t } ^ { \prime } = \sin \left( \sqrt { \frac { 2 } { \tau _ { a } } } Z _ { t } ^ { \prime } \right) } \end{array}$ with an autocorrelation time parameter $\tau _ { a }$ . Another way is to augment the system description with a dynamical noise model. Thus, the overall description would read, for instance:

$$
\begin{array}{l} \mathrm{d} X _ {t} = \left(f \left(X _ {t}, U _ {t}\right) + \sigma \left(X _ {t}, U _ {t}\right) Z _ {t} ^ {\prime}\right) \mathrm{d} t, \\ \mathrm{d} Z ^ {\prime} = z (Z ^ {\prime}) \mathrm{d} t + z (Z ^ {\prime}) \mathrm{d} P. \end{array} \tag {49}
\mathrm{d} Z _ {t} ^ {\prime} = \zeta (Z _ {t} ^ {\prime}) \mathrm{d} t + \chi (Z _ {t} ^ {\prime}) \mathrm{d} B _ {t},$$

where $\{ Z _ { t } ^ { \prime } \}$ is the noise process with an internal model described by the drift function $\zeta$ and diffusion function Λ. Particular ways to construct the respective stochastic differential equations include the following [50]:

• The Doering-Cai-Lin (DCL) noise

$$\mathrm{d} Z _ {t} ^ {\prime} = - \frac {1}{w} Z _ {t} ^ {\prime} \mathrm{d} t + \sqrt {\frac {1 - Z _ {t} ^ {\prime 2}}{b _ {1} (b _ {2} + 1)}} \mathrm{d} B _ {t}, \tag {50}$$

with parameters $b _ { 2 } > - 1 , b _ { 1 } > 0 ;$

• The Tsallis-Stariolo-Borland (TSB) noise

$$\mathrm{d} Z _ {t} ^ {\prime} = - \frac {1}{w} \frac {Z _ {t} ^ {\prime}}{1 - Z _ {t} ^ {\prime 2}} \mathrm{d} t + \sqrt {\frac {1 - b _ {2}}{b _ {1}}} \mathrm{d} B _ {t}, \tag {51}$$
