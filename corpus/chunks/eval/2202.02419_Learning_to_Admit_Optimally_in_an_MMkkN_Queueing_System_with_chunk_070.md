# A.3. Proof of Corollary 1

Proof. We follow the same arguments as in Theorem 4 to show a $O ( \log ( n ) )$ regret. As a parallel to Lemma 6, we bound $\begin{array} { r } { \sum _ { i = \tilde { d } } ^ { n - 1 } i \mathbb { P } \big ( \sum _ { j = 1 } ^ { i } y _ { j } < n , \sum _ { j = 1 } ^ { i + 1 } y _ { j } \ge n \big ) } \end{array}$ for independent geometric random variables $\{ y _ { i } \} _ { i = 1 } ^ { n }$ 1 with success probability $\{ f ( i ) ^ { - 1 } \} _ { i = 1 } ^ { n }$ 1 following similar arguments to Lemma 6. Denote the smallest i that satisfies $i ^ { 1 - \epsilon _ { i } } \geq \log ( n + 1 )$ as b and let $\tilde { d }$ be the smallest integer i such that $\begin{array} { r } { \log ( n + 1 ) \leq \frac { 1 } { 3 } i ^ { 1 - \epsilon _ { b + 1 } } } \end{array}$ . We note that $i ^ { 1 - \epsilon _ { i } }$ is increasing for $i \geq 1$ as $\epsilon _ { i }$ is a decreasing sequence. Take $p _ { i } = \exp \left( - i ^ { 1 - \epsilon _ { i } } \right)$ and $t \geq 0$ such that $\textstyle e ^ { t } = { \frac { n + 1 } { n } } ( 1 - p _ { i } )$ , which exists for $i > b .$ . From (63), for $i > b ,$

$$\mathbb {P} \left(y _ {1} + \dots + y _ {i} \leq n\right) \leq e n ^ {i - b} \prod_ {j = b + 1} ^ {i} p _ {j} \prod_ {j = 1} ^ {b} \frac {1}{1 - \exp \left(- \left(i ^ {1 - \epsilon_ {i}} - j ^ {1 - \epsilon_ {j}}\right)\right)}. \tag {67}$$

Moreover, for $1 \leq j \leq i ,$ , by concavity and gradient inequality, we have $\epsilon _ { j } \geq \epsilon _ { i }$ and

$$i ^ {1 - \epsilon_ {i}} - j ^ {1 - \epsilon_ {j}} \geq i ^ {1 - \epsilon_ {i}} - j ^ {1 - \epsilon_ {i}} \geq \frac {1 - \epsilon_ {i}}{i ^ {\epsilon_ {i}}} (i - j). \tag {68}$$

We define $\kappa = \lceil i ^ { \epsilon _ { i } } / ( 1 - \epsilon _ { i } ) \rceil$ and using (64), simplify the second product term in the RHS of (67) to get

$$\prod_ {j = 1} ^ {b} \frac {1}{1 - \exp \left(- \left(i ^ {1 - \epsilon_ {i}} - j ^ {1 - \epsilon_ {j}}\right)\right)} \leq \prod_ {j = 1} ^ {b} \frac {1}{1 - \exp \left(- \frac {1 - \epsilon_ {i}}{i ^ {\epsilon_ {i}}} (i - j)\right)} \leq c _ {u} ^ {\kappa} 2 ^ {\kappa - 1} \frac {1}{(\kappa - 1) !} \left(\frac {i ^ {\epsilon_ {i}}}{1 - \epsilon_ {i}}\right) ^ {\kappa - 1}. \tag {69}$$

Furthermore, using an integral lower bound, we find an upper bound for the term $\textstyle \prod _ { j = b + 1 } ^ { i } p _ { j } :$

$$(b + 1) ^ {1 - \epsilon_ {b + 1}} + \dots + i ^ {1 - \epsilon_ {i}} \geq (b + 1) ^ {1 - \epsilon_ {b + 1}} + \dots + i ^ {1 - \epsilon_ {b + 1}} \geq \frac {1}{2 - \epsilon_ {b + 1}} \left(i ^ {2 - \epsilon_ {b + 1}} - b ^ {2 - \epsilon_ {b + 1}}\right). \tag {70}$$
