Robustness. We have investigated the robustness of both schemes with respect to different forms of uncertainty, such as modeling and computational errors, by adding two noise terms to the vehicle dynamics: $\dot { x } _ { i } ( t ) =$ $\nu _ { i } ( t ) + w _ { 1 } ( t )$ , $\dot { \nu } _ { i } ( t ) = u _ { i } ( t ) + w _ { 2 } ( t )$ , where $w _ { 1 } ( t ) , w _ { 2 } ( t )$ denote two random processes defined in an appropriate probability space which, in our simulation, are set to be uniformly distributed over [−2, 2] and [−0.2, 0.2], respectively. We repeated the prior simulation experiment with added noise and results shown in Figs. 5 and 6. We can see that the event-triggered and self-triggered schemes with almost similar performance because of their conservativeness keep the functions well away from the unsafe region (below 0) in contrast to the time-driven approach where we observe constraint violations due to noise, e.g., CAV 8 in Fig. 5 and CAVs 3, 4, and, 9 in Fig. 6.
