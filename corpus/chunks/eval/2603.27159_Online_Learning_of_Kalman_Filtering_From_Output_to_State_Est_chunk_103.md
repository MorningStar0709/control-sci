# E.1 Statistical Inequalities

Lemma E.1. [51, Lemma $1 \%$ Let $\boldsymbol { w } \sim \mathcal { N } ( \boldsymbol { 0 } , \Sigma _ { w } )$ with $w \in \mathbb { R } ^ { d }$ and $\Sigma _ { w } \succeq 0$ . For $a n y \ z \ge 1$ , it holds that

$$\mathbb {P} (\| w \| > \sqrt {5 \operatorname{tr} (\Sigma_ {w}) z}) \leq e ^ {- z}.$$

Moreover, for any $\delta \in ( 0 , 1 / e )$ , it holds with probability at least $1 - \delta$ that

$$\| w \| \leq \sqrt {5 \operatorname{tr} (\Sigma_ {w}) \log \frac {1}{\delta}}.$$

Lemma E.2. Let $w _ { t } \stackrel { i . i . d . } { \sim } \mathcal { N } ( 0 , \Sigma _ { w } )$ t i.i.d. ∼ N (0, Σw) with wt ∈ Rd and Σw ⪰ 0 for t = 0, . . . , T − 1. Then, for any $\boldsymbol { w } _ { t } \in \mathbb { R } ^ { d }$ $\Sigma _ { w } \succeq 0 ~ f o r ~ t = 0 , \dots , T - 1$ $T \geq 1$ ,

$$\mathbb {E} \left[ \max _ {0 \leq t \leq T - 1} \| w _ {t} \| ^ {2} \right] \leq 5 \operatorname{tr} (\Sigma_ {w}) (1 + \log T).$$

Proof. First, we get from Lemma E.1 and a union bound over all $t \in \{ 0 , \ldots , T - 1 \}$ that

$$\mathbb {P} \Big (\max _ {0 \leq t \leq T - 1} \| w _ {t} \| ^ {2} > x \Big) \leq T \exp \Big (- \frac {x}{5 \operatorname{tr} (\Sigma_ {w})} \Big),$$

for any $x \ge 5 \mathrm { t r } ( \Sigma _ { w } )$ log T . It then follows that

$$
\begin{array}{l} \mathbb {E} \left[ \max _ {0 \leq t \leq T - 1} \| w _ {t} \| ^ {2} \right] = \int_ {0} ^ {\infty} \mathbb {P} \left(\max _ {0 \leq t \leq T - 1} \| w _ {t} \| ^ {2} > x\right) d x \\ \leq \int_ {0} ^ {5 \operatorname{tr} (\Sigma_ {w}) \log T} 1 d x + \int_ {5 \operatorname{tr} (\Sigma_ {w}) \log T} ^ {\infty} T \exp \Big (- \frac {x}{5 \operatorname{tr} (\Sigma_ {w})} \Big) d x \\ = 5 \operatorname{tr} (\Sigma_ {w}) \log T + 5 \operatorname{tr} (\Sigma_ {w}) T \int_ {\log T} ^ {\infty} e ^ {- y} d y \quad \left(\text {   Let   } y = \frac {x}{5 \operatorname{tr} (\Sigma_ {w})}\right) \\ = 5 \operatorname{tr} (\Sigma_ {w}) \log T + 5 \operatorname{tr} (\Sigma_ {w}). \\ \end{array}
$$

Lemma E.3. [70, Corollary 3.9] Let $\{ X _ { k } \} _ { k \ge 0 }$ be a martingale difference sequence adapted to a filtration $\{ \mathcal { F } _ { k } \} _ { k \ge 0 }$ . Suppose $| X _ { k } | \le c _ { k }$ for all $k \geq 0$ . Then, for any $s \geq 1$ and any $\epsilon \in \mathbb { R }$ , it holds that

$$\mathbb {P} \Bigl (\sum_ {k = 1} ^ {s} X _ {k} > \epsilon \Bigr) \leq \exp \Bigl (- \frac {\epsilon^ {2}}{2 \sum_ {k = 1} ^ {s} c _ {k} ^ {2}} \Bigr).$$

Equivalently, for any $0 < \delta < 1$ and any $\varepsilon \in \mathbb { R }$ , it holds with probability at least $1 - \delta$ that

$$\sum_ {k = 1} ^ {s} X _ {k} \leq \sqrt {\log \frac {1}{\delta} \sum_ {k = 1} ^ {s} 2 c _ {k} ^ {2}}.$$
