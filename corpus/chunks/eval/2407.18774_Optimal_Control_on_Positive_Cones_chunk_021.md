# A Proof of Lemma 1

Proof. We start by developing the formula for the dual cone. Let $B \left( X \right) \ =$ $A \ : ( \dot { T } X T ^ { \top } )$ . By Farkas’ lemma,

$$\mathcal {P} _ {Q} ^ {*} = \left\{B ^ {*} (Y): Y \succeq 0 \right\}.$$

Since for any symmetric matrices C and $D , \operatorname { t r } \left( C D \right) = \operatorname { v e c } \left( C \right) ^ { \top } \operatorname { v e c } \left( D \right)$ , we see that

$$
\begin{array}{l} \operatorname{tr} (B (X) Y) = \operatorname{tr} \left(A (T X T ^ {\top}) A (Y)\right) \\ = \operatorname{tr} \left(T X T ^ {\top} A (Y)\right) \\ = \operatorname{tr} \left(X T ^ {\top} A (Y) T\right). \\ \end{array}
$$

This shows that $B ^ { * } \left( Y \right) = T ^ { \top } A \left( Y \right) T$ as required. To show the second claim observe that if

$$
\left[ \begin{array}{c c} \Lambda & M _ {1} \\ M _ {1} ^ {\top} & M _ {2} \end{array} \right] \in \mathcal {P} ^ {*},
$$

then there exists a

$$

$$
