# 13.3 System Identification

System identification is the experimental approach to process modeling. System identification includes the following:

• Experimental planning   
- Selection of model structure

- Criteria   
• Parameter estimation   
- Model validation

In practice, the procedure of system identification is iterative. When investigating a process where the a priori knowledge is poor, it is reasonable to start with transient or frequency-response analysis to get crude estimates of the dynamics and the disturbances. The results can be used to plan further experiments. The data obtained are then used to estimate the unknown parameters in the model. Based on the results, the model structure can be improved and new experiments may be necessary.
