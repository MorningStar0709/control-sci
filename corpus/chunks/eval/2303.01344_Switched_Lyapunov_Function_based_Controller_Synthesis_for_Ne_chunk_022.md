Please note, that increasing the value of ¯δ to 5 would result in 46086 optimization variables and 1048576 LMIs necessary to obtain the control law proposed in [5], which greatly increases the complexity for the over-approximation of the NCS and computational effort for the control synthesis. For the proposed control law from e.g. Theorem 1, the value $\bar { \delta } \ = \ 5$ would, however, lead to only 321 optimization variables and 25 LMIs. The complexity of defining the NCS as switched system and the corresponding computational effort in this case would not be affected at all.
