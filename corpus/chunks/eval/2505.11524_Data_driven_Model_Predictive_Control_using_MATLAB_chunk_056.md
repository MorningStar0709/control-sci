$$L _ {2} = \alpha_ {2} \operatorname{trace} \left(\left[ \mathbf {X} _ {\mathrm{D}} - \bar {\mathbf {X}} _ {\mathrm{D}} \right] ^ {\top} \mathbf {W} _ {\mathbf {x}} \left[ \mathbf {X} _ {\mathrm{D}} - \bar {\mathbf {X}} _ {\mathrm{D}} \right]\right) = \alpha_ {2} (\mathrm{D} - 1) \left(w _ {1} V _ {x _ {1}} + \dots + w _ {l} V _ {x _ {l}}\right) \tag {82}$$

which is obtained as a weighted sum of the predicted state variances computed over the training data. Consequently, the variance of the state variables can be ordered by adjusting the weighting parameters $w _ { i } , i = 1 , 2 , \ldots , l .$ Let s denotes the number of state variables in $\mathbf { X } _ { \mathrm { D } }$ satisfying:

$$V _ {x _ {i}} > \delta , \quad \delta > 0 \tag {83}$$

which leads to the partition $\mathbf { x } = \left\lceil \mathbf { x } _ { \mathrm { a } } \right\rceil , \mathbf { x } _ { \mathrm { a } } \in \mathbb { R } ^ { s } , \mathbf { x } _ { \mathrm { b } } \in \mathbb { R } ^ { l - s }$ . The key idea of model order determination/reduction proposed in [31] is to replace $\mathbf { x } _ { \mathrm { b } }$ with the mean vector $\bar { \bf x } _ { \mathrm { b } }$ and reabsorb it in the model as a bias term. This results in $\mathbf { x _ { a } }$ as the only dynamic term in the model. Consequently, the resultant model with $\mathbf { x _ { a } }$ as the state vector will be of order s.
