# APPENDIX I ALGEBRAIC RESULTS

The following result is described in [6, Section 2.6.2]

Lemma 38 (Projection Lemma): Let $U \in \mathbb { R } ^ { m \times p }$ and $V \in$ $\mathbb { R } ^ { n \times p } ;$ , and let $Q = Q ^ { \top } \in \mathbb { R } ^ { p \times p }$ . There exists a matrix $X \in$ Rm×n satisfying,

$$Q + U ^ {\top} X V + V ^ {\top} X U \prec 0 \tag {80}$$

if and only if

$$U _ {\perp} ^ {\top} Q U _ {\perp} \prec 0 \quad \text { and } \quad V _ {\perp} ^ {\top} Q V _ {\perp} \prec 0. \tag {81}$$

Next, we introduce an incremental matrix multiplier for this fixed point map from [10].

Lemma 39: Let the map $\Psi \colon \mathbb { R } ^ { n }  \mathbb { R } ^ { n }$ admit an incremental matrix multiplier M . Let the map $x ^ { \star } \colon \mathbb { R } ^ { m }  \mathbb { R } ^ { n }$ be wellposed and defined as the solution of the implicit equation,

$$x ^ {\star} (u) = \Psi (W x ^ {\star} (u) + B u) \tag {82}$$

where $W \in \mathbb { R } ^ { n \times n } , B \in \mathbb { R } ^ { n \times m }$ . Then the map $x ^ { \star } ( u )$ admits the incremental multiplier matrix

$$
\left[ \begin{array}{c c} B & W \\ 0 & I _ {n} \end{array} \right] ^ {\top} M \left[ \begin{array}{c c} B & W \\ 0 & I _ {n} \end{array} \right]. \tag {83}
$$
