# 4. NUMERICAL EXPERIMENTS

To show the benefits of minimal regret estimation, we will simulate the trajectories of the benchmark system "NN4" of order 4 from Complib (Leibfritz (2004)) under nine different disturbance patterns. In order to assess the performance in practice, we also provide the average performance for the first three fifth order systems in Complib $" \mathrm { A C 1 " , ~ " \mathrm { A C 2 } " }$ , and $" { \mathrm { A C 3 } } "$ , which model aircrafts. For convenience, we consider only LTI systems, even though our framework also encompasses LTV systems, see (1). All systems are discretized with $T _ { s } = 5 m s$ and the prediction horizon is $T = 1 0$ . First, the optimal error maps $\Phi _ { v }$ and $\Phi _ { w }$ are computed for $\mathcal { H } _ { 2 } , \mathcal { H } _ { \infty }$ and minimal regret (denoted by R) methods. Then, the performance is evaluated using (10) and the cost $\| e \| _ { 2 } ^ { 2 }$ for each disturbance pattern. Each element of the noises v and w follow independent uniform or Gaussian distributions in three cases, and deterministic sequences (e.g. a sine wave $w _ { t } = [ \sin ( t ) , \ldots , \sin ( t ) ] ^ { \top } )$ in the other six cases.
