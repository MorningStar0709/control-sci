Moreover, since $x _ { 1 } \in \mathcal { X } , \tilde { x } _ { 1 } \in \mathcal { B } _ { p } ( x _ { 1 } ^ { * } ; r ) \subseteq \mathcal { X }$ , and both are in $\boldsymbol { B } _ { p } ( \boldsymbol { x } _ { 1 } ^ { * } ; r )$ whenever $\left( \mathrm { A . 2 } \right)$ holds (note that $\delta < r$ from construction), the local Lipschitz bound (A.3) gives

$$f (x _ {1}) \geq f (\tilde {x} _ {1}) - \kappa_ {r} \| x _ {1} - \tilde {x} _ {1} \| _ {p} \geq f (x _ {1} ^ {*}) - \kappa_ {r} L \| D x _ {2} \| _ {p},$$

where we used (A.4) in the last step. Therefore, for any $\lambda _ { w } \ge 0$ ,

$$f (x _ {1}) + \lambda_ {w} \| D x _ {2} \| _ {p} \geq f (x _ {1} ^ {*}) + (\lambda_ {w} - \kappa_ {r} L) \| D x _ {2} \| _ {p}.$$

This is exactly (A.1) with $\lambda _ { w } ^ { * } : = \kappa _ { r } L$ . This completes the proof.
