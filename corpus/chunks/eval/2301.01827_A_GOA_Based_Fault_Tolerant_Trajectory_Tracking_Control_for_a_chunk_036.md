# APPENDIX B PROOF OF THE SMC STABILITY

To prove the stability of the SMC, construct a Lyapunov function,

$$\mathbf {V} = \frac {1}{4 \lambda} \mathbf {s} ^ {T} \mathbf {M} \mathbf {s} + \frac {1}{2} \mathbf {Q} ^ {T} \Gamma^ {- 1} \mathbf {Q}, \tag {28}$$

where $\mathbf { Q } = \widetilde { \tau } _ { r } - \widetilde { \tau } _ { e s t }$ and $\widetilde { \boldsymbol { \tau } } _ { r } = \widetilde { \mathbf { M } } \dot { \mathbf { v } } _ { r } + \widetilde { \mathbf { C } } \mathbf { v } _ { r } + \widetilde { \mathbf { D } } \mathbf { v } + \widetilde { \mathbf { g } } .$

Previously we have given $\mathbf { e _ { v } } = \mathbf { v _ { c } } - \mathbf { v _ { \rho } }$ , and ${ \bf s } \dot { \bf e } _ { \bf v } + 2 \lambda { \bf e } _ { \bf v } +$ $\lambda ^ { 2 } \int \mathbf { e } _ { \mathbf { v } } d t$ , such that two equations can be deducted as,

$$\mathbf {v} = \mathbf {v} _ {\mathbf {c}} - \frac {\mathbf {s} - \dot {\mathbf {e}} _ {\mathbf {v}} - \lambda^ {2} \int \mathbf {e} _ {\mathbf {v}} d t}{2 \lambda}, \tag {29}\dot {\mathbf {v}} = \dot {\mathbf {v}} _ {\mathbf {c}} - \frac {\dot {\mathbf {s}} - \ddot {\mathbf {e}} _ {\mathbf {v}} - \lambda^ {2} \mathbf {e} _ {\mathbf {v}}}{2 \lambda}, \tag {30}$$

therefore the following items can be defined,

$$\mathbf {v} _ {\mathbf {r}} = \mathbf {v} _ {\mathbf {c}} + \frac {\dot {\mathbf {e}} _ {\mathbf {v}} + \lambda^ {2} \int \mathbf {e} _ {\mathbf {v}} d t}{2 \lambda}, \tag {31}\dot {\mathbf {v}} _ {\mathbf {r}} = \dot {\mathbf {v}} _ {\mathbf {c}} + \frac {\ddot {\mathbf {e}} _ {\mathbf {v}} + \lambda^ {2} \mathbf {e} _ {\mathbf {v}}}{2 \lambda}. \tag {32}$$

By substituting into Eq. (2),

$$
\begin{array}{l} \mathbf {M} \frac {\dot {\mathbf {s}}}{2 \lambda} + \mathbf {C} \frac {\mathbf {s}}{2 \lambda} \\ = \mathbf {M} (\dot {\mathbf {v}} _ {\mathbf {c}} + \frac {\ddot {\mathbf {e}} _ {\mathbf {v}} + \lambda^ {2} \mathbf {e} _ {\mathbf {v}}}{2 \lambda}) + \mathbf {C} (\mathbf {v} _ {\mathbf {c}} + \frac {\dot {\mathbf {e}} _ {\mathbf {v}} + \lambda^ {2} \int \mathbf {e} _ {\mathbf {v}} d t}{2 \lambda}) \\ + \mathbf {D} \mathbf {v} + g - \tau = \mathbf {M} \dot {\mathbf {v}} _ {\mathbf {r}} + \mathbf {C} \mathbf {v} _ {\mathbf {r}} + \mathbf {D} \mathbf {v} + g - \tau . \tag {33} \\ \end{array}
$$
