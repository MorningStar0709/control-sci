# Neglecting model part D - inhibition through pH and ammonia

The full inhibition function $I _ { \mathrm { a c } }$ is a major source of nonlinearity in the ADM1-R3. Neglecting model part D (inhibition through pH and ammonia) allows to cut out the first and last factor of $I _ { \mathrm { a c } } ,$ see (25). Consequently, $S _ { \mathrm { n h 3 } }$ can be omitted in the state vector.
