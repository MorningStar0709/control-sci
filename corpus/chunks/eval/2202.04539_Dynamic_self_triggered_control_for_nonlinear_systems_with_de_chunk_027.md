$$
U _ {1} (x (r _ {k} ^ {+})) \leq \tilde {\beta} _ {1} \left(\left\| \left[ \begin{array}{l} x (r _ {0}) \\ \eta (r _ {0}) \end{array} \right] \right\|, t _ {k}, k\right) \tag {A.8}

and

$$
\left\| \eta (r _ {k} ^ {+}) \right\| \leq (m - 1) \tilde {\beta} _ {1} \left(\left\| \left[ \begin{array}{l} x (r _ {0}) \\ \eta (r _ {0}) \end{array} \right] \right\|, t _ {k}, k\right) \tag {A.9}

for a class KLL function $\tilde { \beta } _ { 1 }$ that are valid at sampling instants. To show asymptotic stability it thus remains to derive a similar bound between sampling instants, which works again analogous as in the proof of (Hertneck and Allg¨ower, 2021b, Theorem 1). In particular, we first note that

$$\left\| \eta (t, j) \right\| = \left\| \eta (r _ {k} ^ {+}) \right\| \tag {A.10}$$

holds for $r _ { k } ^ { + } ~ \preceq ~ ( t , j ) ~ \preceq ~ r _ { k + 2 }$ . For $x ( t , j ) , e ( t , j )$ and $s ( t , j )$ , we can use again Proposition 1 to derive a bound. Recall that the conditions of Proposition 1 hold for all $k ~ \in ~ \mathbb { N } _ { \epsilon }$ at least for one $\textit { p } \in \{ 1 , \ldots , n _ { p } \}$ , for which $U _ { p } ( \xi ( r _ { k } ^ { + } ) ) \ \leq \ K ( x ( r _ { k } ) , e ( r _ { k } ) , s ( r _ { k } ) , c _ { \mathcal { X } } )$ and $t _ { k + 2 } ~ - ~ t _ { k } ~ \leq$ $\bar { T _ { \mathrm { m a x } } } \left( \bar { \mathcal { P } } _ { p } , \lambda , c _ { U } , \tau _ { \mathrm { m a d } } \right)$ . $$
\begin{array}{l} U _ {p} (\xi (t, j)) \leq e ^ {\max \{\epsilon_ {p}, 1 \} t _ {\max}} U _ {p} (\xi (r _ {k} ^ {+})) \\ \leq e ^ {\max \{\epsilon_ {p}, 1 \} t _ {\max}} K (x (r _ {k}), e (r _ {k}), s (r _ {k}), c _ {\mathcal {X}}). \tag {A.11} \\ \end{array}

$$
