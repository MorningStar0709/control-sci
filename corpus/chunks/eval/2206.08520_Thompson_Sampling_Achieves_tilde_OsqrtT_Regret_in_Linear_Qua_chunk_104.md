# F.5. Bounding $R _ { T } ^ { \mathbf { g a p } }$

Lemma 26 (Bounding $R _ { T } ^ { \mathbf { g a p } }$ for TSAC) Let $R _ { T } ^ { g a p }$ be as defined by (115). Under the event of $E _ { T }$ , we have that

$$\left| R _ {T} ^ {g a p} \right| = \tilde {O} \left(\mathrm{poly} (n, d) \sqrt {T \log (1 / \delta)}\right).$$

with probability at least $1 - 2 \delta$ for large enough $T .$ .

Proof

$$
\begin{array}{l} R _ {T} ^ {\text { gap }} = \sum_ {t = 0} ^ {T} \mathbb {E} \left[ x _ {t + 1} ^ {\intercal} \left(P (\tilde {\Theta} _ {t + 1}) - P (\tilde {\Theta} _ {t})\right) x _ {t + 1} \mathbb {1} _ {E _ {t + 1}} \mid \mathcal {F} _ {t} \right] (134) \\ = \sum_ {t = 0} ^ {K} \mathbb {E} \left[ x _ {t _ {k} + 1} ^ {\intercal} \left(P (\tilde {\Theta} _ {t _ {k} + 1}) - P (\tilde {\Theta} _ {t _ {k}})\right) x _ {t _ {k} + 1} \mathbb {1} _ {E _ {t _ {k} + 1}} \mid \mathcal {F} _ {t _ {k}} \right] (135) \\ \end{array}
$$

Separating the duration of TSAC into two parts at $t \ = \ T _ { r }$ , we obtain two same term achieved in (Abeille and Lazaric, 2018). Note that in Abeille and Lazaric (2018), the authors follow frequent update rule and TSAC updates every τ0 time-steps. The proof of these terms similarly follow Section 5.2 in Abeille and Lazaric (2018) and using Lemma 25 we obtain ${ \tilde { O } } ( ( n + d ) ^ { n + d } { \sqrt { T _ { r } } } +$ $\mathrm { p o l y } ( n , d ) \sqrt { T - T _ { r } } )$ . Note that there is an additional $\tau _ { 0 }$ factor in these bounds, due to “relatively slower” update of TSAC. For large enough T such that the second term dominates the overall upper bound, we obtain the advertised guarantee.
