# D. Proof of Lemma 3

Let $\mathcal { A } ~ = ~ \left\lceil \mathcal { A } _ { 0 } \quad \mathcal { A } _ { 1 } \quad \mathcal { A } _ { 1 } ^ { * } \right\rceil$ with $\mathcal { A } _ { 0 } ~ \in ~ \mathbb { R } ^ { n \times m _ { 0 } } , ~ \mathcal { A } _ { 1 } ~ \in$ $\mathbb { C } ^ { n \times m _ { 1 } }$ and $m \ = \ m _ { 0 } + 2 m _ { 1 }$ . Then, $\boldsymbol { A } \boldsymbol { A } ^ { \sf H }$ is real and symmetric. Thus, there exists $\boldsymbol { \mathcal { U } } \in \mathbb { R } ^ { n \times n }$ such that $u ^ { \top } u = I$ and $\mathcal { A } \mathcal { A } ^ { \sf H } = \mathcal { U } S ^ { 2 } \mathcal { U } ^ { \top }$ with S as in (15), where r = rank A and $S _ { 1 }$ is diagonal and positive definite. Let

$$
\mathcal {V} = \mathcal {A} ^ {\mathsf {H}} \mathcal {U} \left[ \begin{array}{c c} \mathcal {S} _ {1} ^ {- 1} & 0 \\ 0 & I \end{array} \right].
$$

Clearly, $\mathcal { U S V } ^ { \sf H } = \mathcal { U } \left[ \begin{array} { l l } { S _ { 1 } } & { 0 } \\ { 0 } & { I } \end{array} \right] \left[ \begin{array} { l l } { S _ { 1 } ^ { - 1 } } & { 0 } \\ { 0 } & { I } \end{array} \right] \mathcal { U } ^ { \top } \mathcal { A } = \mathcal { A } \mathrm { ~ a n d ~ }$

$$
\mathcal {V} ^ {\mathsf {H}} \mathcal {V} = \left[ \begin{array}{c c} \mathcal {S} _ {1} ^ {- 1} & 0 \\ 0 & I \end{array} \right] \mathcal {U} ^ {\top} \mathcal {A} ^ {\mathsf {H}} \mathcal {A} \mathcal {U} \left[ \begin{array}{c c} \mathcal {S} _ {1} ^ {- 1} & 0 \\ 0 & I \end{array} \right] = I.
$$

Finally, V has the desired structure $\nu = \left[ \mathcal { V } _ { 0 } \quad \mathcal { V } _ { 1 } \quad \mathcal { V } _ { 1 } ^ { * } \right] ^ { \mathsf { H } }$ with

$$
\mathcal {V} _ {0} = \left[ \begin{array}{c c} \mathcal {S} _ {1} ^ {- 1} & 0 \\ 0 & I \end{array} \right] \mathcal {U} ^ {\top} A _ {0} \in \mathbb {R} ^ {m \times m _ {0}},
$$

and

$$
\mathcal {V} _ {1} = \left[ \begin{array}{c c} \mathcal {S} _ {1} ^ {- 1} & 0 \\ 0 & I \end{array} \right] \mathcal {U} ^ {\top} A _ {1} \in \mathbb {C} ^ {m \times m _ {1}}.
$$
