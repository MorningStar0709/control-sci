# VI. CONCLUSIONS

In this letter, we present QSID-MPC—a model predictive control framework that utilizes system identification from dither quantized data. We theoretically analyze the connection between the true system and the system identified from quantized data. The effect of quantization is quantified for both finite and large data regimes. Our analysis shows the quantization resolution  affects the identified system as O( ) ϵin finite data regime and $O ( \epsilon ^ { 2 } )$ ϵ in large data regime. Further analysis of the performance of MPC shows that the true system is uniformly ultimately bounded by QSID-MPC with a bound of $O ( \epsilon ^ { 2 } )$ for the large data regime, thereby proving its robustness ϵagainst quantization. Our analysis is validated via repeated trials of experiments on multiple problems.
