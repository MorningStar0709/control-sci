# B.2 Proof of Theorem 4

Proof: Our proof is based on the implicit function theorem (Theorem 9). Let $\epsilon : = \beta ^ { - 1 }$ . From optimality conditions similar to (20) (without assuming that $R _ { \infty } = \alpha ^ { 2 } R _ { 2 } )$ , we define the following vectorized function

$$
F (K, X, \Gamma , \epsilon) = \left[ \begin{array}{c} F _ {K} (K, X, \Gamma , \epsilon) \\ F _ {X} (K, X, \Gamma , \epsilon) \\ F _ {\Gamma} (K, X, \Gamma , \epsilon) \end{array} \right]

= \left[ \begin{array}{c} \operatorname{vec} \left(R _ {2} K + B ^ {\mathsf {T}} \Gamma + \epsilon^ {2} R _ {\infty} K X \Gamma\right) \\ \operatorname{vec} \left(A _ {K} X + X A _ {K} ^ {\mathsf {T}} + \epsilon^ {2} X S _ {K} X + W\right) \\ \operatorname{vec} \left(\tilde {A} _ {K} ^ {\mathsf {T}} \Gamma + \Gamma \tilde {A} _ {K} + Q _ {2} + K ^ {\mathsf {T}} R _ {2} K\right) \end{array} \right]
$$

where $\tilde { A } _ { K } = A _ { K } + \epsilon ^ { 2 } X S _ { K }$ is Hurwitz. This equation characterizes the minimizer of $J _ { \mathrm { m i x } }$ . Since $\epsilon _ { 0 } = 0$ gives the LQR case, we know that there exists some $( X _ { 0 } , \Gamma _ { 0 } , K _ { 0 } )$ satisfying

$$F (X _ {0}, \Gamma_ {0}, K _ {0}, \epsilon_ {0}) = 0$$

and $A _ { K _ { 0 } } : = A + B K _ { 0 }$ is guaranteed to be stable. Then, it is not very difficult to compute the Jacobian $\mathbf { J } _ { ( K , X , \Gamma ) } F$ of

$F \mathrm { a t } \left( K _ { 0 } , X _ { 0 } , \Gamma _ { 0 } , 0 \right)$

$$
\mathbf {J} _ {(K, X, \Gamma)} F (K _ {0}, X _ {0}, \Gamma_ {0}, 0) = \left[ \begin{array}{c c} I \otimes R _ {2} & [ 0 (I \otimes B ^ {\mathsf {T}}) ] \\ \left[ \begin{array}{c} * \\ 0 \end{array} \right] & \left[ \begin{array}{c c} H _ {1} & 0 \\ 0 & H _ {2} \end{array} \right] \end{array} \right],
$$

where $H _ { 1 }$ and $H _ { 2 }$ are the following invertible matrices

$$H _ {1} = I \otimes A _ {K _ {0}} + A _ {K _ {0}} \otimes I,H _ {2} = I \otimes A _ {K _ {0}} ^ {\mathsf {T}} + A _ {K _ {0}} ^ {\mathsf {T}} \otimes I.$$

From the Schur complement, since

$$
I \otimes R _ {2} - \left[ 0 (I \otimes B ^ {\mathsf {T}}) \right] \left[ \begin{array}{c c} H _ {1} ^ {- 1} & 0 \\ 0 & H _ {2} ^ {- 1} \end{array} \right] \left[ \begin{array}{c} * \\ 0 \end{array} \right] = I \otimes R _ {2}
$$

is invertible, so is $\mathbf { J } _ { ( K , X , \Gamma ) } F ( K _ { 0 } , X _ { 0 } , \Gamma _ { 0 } , 0 )$ . Therefore, the desired result follows from Theorem 9. In particular, there exists a neighborhood $U \subset \mathrm { R }$ of $\epsilon _ { 0 } = 0$ and a real analytic mapping φ : U → Rmn+2n $\phi : U \to \operatorname { R } ^ { m n + 2 n ^ { 2 } }$ such that

$$F (\phi (\epsilon), \epsilon) = 0, \quad \forall \epsilon \in U,$$

and $A _ { K }$ is Hurwitz. Note that the stability of $A _ { K }$ is guaranteed for any sufficiently small $\epsilon > 0$ . Therefore, a solution to (20) exists for any sufficiently small $\epsilon \in U$ , i.e., for any sufficiently large $\beta > ( \operatorname* { s u p } _ { \epsilon \in U } \epsilon ) ^ { - 1 }$ . ✷
