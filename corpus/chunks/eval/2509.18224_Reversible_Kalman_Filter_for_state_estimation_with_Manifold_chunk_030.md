# Algorithm 5 EKF Prediction Step for Odometry sensor

Require: Distance traveled by left and right wheel $d _ { L }$ and $d _ { R }$ , Covariance $P _ { k - 1 }$ , Process noise $Q _ { k }$ , Time step dt

Ensure: Updated $S = \log ( q )$ , position p, $P _ { k }$

1: $[ d , d \theta ] \overset { - } { \longleftarrow } [ \frac { d _ { L } + d _ { R } } { 2 } , \frac { d _ { L } - \dot { d } _ { R } } { 2 d _ { w } } ]$ 2dw   
2: $\theta  S [ 3 ]$ ▷ z-axis rotation angle   
3: $\mathcal { S }  \mathcal { S } + [ 0 , 0 , d \theta ]$   
$4 \colon \mathbf { p } \gets \mathbf { p } + [ d \cdot \cos ( \theta + d \theta / 2 ) , d \cdot \sin ( \theta + d \theta / 2 ) , 0 ]$

6: $J _ { 6 , 1 : 2 }  - d \cdot [ \sin ( \theta ) , \cos ( \theta ) ]$   
7: $K _ { 1 , 1 : 2 } \gets [ \sin ( \theta ) , \cos ( \theta ) ] )$   
8: $K _ { 2 , 1 : 6 } \gets [ - \frac { d } { 2 } \sin ( \theta ) , \frac { d } { 2 } \cos ( \theta ) , 0 , 0 , 0 , 1 ]$   
9: $P _ { k }  J ^ { \top } \cdot \bar { P _ { k - 1 } } \cdot J + \bar { K ^ { \top } } Q _ { k } K$   
10: return ${ \mathcal { S } } , { \mathbf { p } } , P _ { k }$

Let us consider (O, A, P, ⃗n), giving Odometry, accelerometer, pressure input, and the normal. The conditions that are used are different, since we use the fact that

$$\left(\mathbf {p} + \mathbf {s} \cdot d t - \vec {g} \cdot (d t) ^ {2} + R ^ {- 1} \cdot \mathbf {A}\right) \cdot \vec {z} = \mathbf {P} \text { and } R ^ {- 1} \vec {n} = \vec {z}.$$

For having a finite number of solutions, one has to ensure that ⃗n $\neq \vec { z } ,$ which means that the plan cannot be orthogonal to the gravity vector. It is coherent with this set of sensors, because it corresponds to the case where accelerometer and pressure sensors are not able to bring information on the orientation.
