# C.2 MATLAB $^{©}$ for Continuous-Time Tracking System

The following MATLAB© files are used to solve the Example 4.1. The main file is Example4.1(example4\_1.m) which requires the files: Example 4.1 (example4\_1p.m), Example 4.1(example4\_1g.m), and Example 4.1 (example4\_1x.m). The file Example 4.1(example4\_1.m) is for solving the set of first order Riccati differential equations; Example 4.1 (example4\_1g.m) is the set of first order g vector differential equations; and Example 4.1 (example4\_1x.m) is the set of state differential equations.
