# C.1. On the multi-asset variant of the Obizhaeva–Wang model

Recall Setting 4.10 where we consider a multi-asset variant of the Obizhaeva–Wang model [46] (in particular, we take the price impact $\gamma$ and the resilience $\rho$ to be constant in our setting of Section 2.2). In the next example, we illustrate that it is not sufficient for the existence of an optimal execution strategy in Setting 4.10 to assume that $\gamma \in \mathbb { S } _ { > 0 } ^ { n }$ and $\rho \in \mathbb { S } _ { > 0 } ^ { n }$ .

Example C.1. In Setting 4.10 let $T \in ( 0 , \frac { 2 } { 5 } ) , n = 2 , d = 0$ ,

$$
\gamma = \left( \begin{array}{c c} 2 & 1 \\ 1 & 1 \end{array} \right), \quad \text {and} \quad \rho = \left( \begin{array}{c c} 1 & 2 \\ 2 & 5 \end{array} \right).
$$

Note that it holds that $\gamma \in \mathbb { S } _ { > 0 } ^ { 2 }$ and that also $\rho \in \mathbb { S } _ { > 0 } ^ { 2 }$ . We denote $e _ { 1 } = ( 1 , 0 ) ^ { \top } \in \mathbb { R } ^ { 2 }$ . For every $k \in \mathbb N$ we consider the $\mathbb { R } ^ { 2 } .$ -valued function $X ^ { k } \ : = \ : ( X ^ { k } ( s ) ) _ { s \in [ 0 , T ] }$ defined by $X ^ { k } ( 0 - ) = x , X ^ { k } ( T ) = 0$ , and

$$X ^ {k} (s) = x + k \gamma^ {- 1} e _ {1} + s k \gamma^ {- 1} \rho e _ {1} = x + k \left(1 - 1\right) + s k \left(- 1 3\right), \quad s \in [ 0, T).$$

It holds for all $k \in \mathbb N$ that the associated $D ^ { k } \equiv D ^ { X ^ { k } }$ given by $D ^ { k } ( 0 - ) ~ = ~ d = ~ 0$ and (23) (or, equivalently, (1)) satisfies for all $r \in [ 0 , T )$ that $D ^ { k } ( r ) \ = \ ( k , 0 ) ^ { \top }$ and $D ^ { k } ( T ) = - \gamma x - T \rho ( k , 0 ) ^ { \top }$ . In particular, it follows for all $k \in \mathbb N$ that $X ^ { k }$ is an admissible trade execution strategy (in $\mathcal { A } ^ { f v }$ and $\mathcal { A } ^ { p m } )$ . Note that when, for $k \in \mathbb N$ , we use the strategy $X ^ { k }$ , then we at the initial time buy k units in the first asset and concurrently sell k units in the second asset. The initial block trade for all $k \in \mathbb N$ contributes

$$\frac {1}{2} \left(\Delta X ^ {k} (0)\right) ^ {\top} \gamma \Delta X ^ {k} (0) = \frac {k ^ {2}}{2} > 0 \tag {C.19}$$

to the execution costs $C ( X ^ { k } )$ . During the time interval (0, T ) we then for all $k \in \mathbb N$ sell in the first asset with rate k and we buy in the second asset with the faster rate 3k. We thereby exploit the lower price impact and the higher resilience in the second asset with, in some sense, not too adverse side effects on the less liquid first asset.
