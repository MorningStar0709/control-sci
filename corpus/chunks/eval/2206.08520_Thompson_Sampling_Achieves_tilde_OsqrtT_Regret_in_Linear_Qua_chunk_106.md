On the other hand, Theorem 6 puts strict lower bounds on the growth of $T _ { w }$ in order to maintain asymptotically constant optimistic probability. In particular, for singular $A _ { c , * }$ , this condition is stated as $T _ { w } = \omega ( \sqrt { T } \log T )$ . Combined with the required upper bound $O \left( ( { \sqrt { T } } ) ^ { 1 + o ( 1 ) } \right)$ , it must be that $T _ { w } = \operatorname* { m a x } \Big ( T _ { 0 } , c ( \sqrt { T } \log T ) ^ { 1 + o ( 1 ) } \Big )$ for a constant $c > 0$ for large enough T . Inserting this result in (136) gives us

$$R _ {T} = \tilde {O} \left((n + d) ^ {n + d} \sqrt {T}\right), \quad \text { w . p . } 1 - 1 0 \delta$$

for large enough T . Observe that exponential dimension dependence is unavoidable in this case as the system is excited with isotropic noise in every direction long enough to dominate with exponential dimension.

For non-singular $A _ { c , * } ,$ the lower bound is stated as $T _ { w } = \omega ( \log T )$ . For large enough $T ,$ , choosing $T _ { w } = \operatorname* { m a x } \left( T _ { 0 } , c ( \log T ) ^ { 1 + o ( 1 ) } \right)$ for a constant $c > 0$ is sufficient to satisfy both the upper and lower bounds on $T _ { w }$ . Inserting this result in (136) gives us

$$R _ {T} = \tilde {O} \left(\mathrm{poly} (n, d, \log (1 / \delta)) \sqrt {T}\right), \quad \text { w.p. } 1 - 1 0 \delta$$

for large enough T . Observe that the exponential dimension dependence is not dominant anymore since logarithmically large $T _ { w }$ is sufficient to guarantee asymptotically constant optimistic probability.
