# H.3 Proof of Proposition 13

proof.

Lemma 39. Under Assumptions 1–3 and 4, for any $\theta \in \Theta$ and any $K \in [ \frac { a - 1 } { b } , \frac { a } { b } ]$ , when using controller $C _ { K } ^ { \mathrm { u n c } }$ under dynamics θ where $\begin{array} { r } { 1 - ( a - b K ) = \epsilon = \Omega _ { T } ( 1 ) } \end{array}$ and starting at state $x _ { 0 } = x$ , then for all t, the state $x _ { t }$ at time t satisfies

$$| x _ {t} | \leq | x | + \frac {\bar {w}}{\epsilon}.$$

Furthermore, for any $x , y , W ^ { \prime }$ and $\tau \leq T$ ,

$$
\begin{array}{l} | \tau J (\theta , C _ {K} ^ {\mathrm{unc}}, \tau , x, W ^ {\prime}) - \tau J (\theta , C _ {K} ^ {\mathrm{unc}}, \tau , y, W ^ {\prime}) | \leq \frac {(q + r K ^ {2}) (x - y) ^ {2} + 2 (q + r K ^ {2}) \left(| x | + \frac {\bar {w}}{\epsilon}\right) | x - y |}{\epsilon} \\ = O _ {T} \left((x - y) ^ {2} + | x (x - y) |\right). \\ \end{array}
$$

The proof of Lemma 39 can be found in Appendix H.7.

By Lemma 23, under event $E \subseteq E _ { 2 } ^ { 0 }$ , we have $| F _ { \mathrm { o p t } } ( \hat { \theta } _ { \mathrm { w u } } ) - F _ { \mathrm { o p t } } ( \theta ^ { * } ) | \leq O _ { T } ( T ^ { - 1 / 4 } )$ . Therefore, by Lemma 25, under event E and for large enough T ,

$$1 - (a ^ {*} - b ^ {*} F _ {\mathrm{opt}} (\hat {\theta} _ {\mathrm{wu}})) \geq c _ {\mathrm{L25}} - b ^ {*} | F _ {\mathrm{opt}} (\hat {\theta} _ {\mathrm{wu}}) - F _ {\mathrm{opt}} (\theta^ {*}) | \geq c _ {\mathrm{L25}} / 2.$$

Conditional on event $E , C ^ { \mathrm { a l g } }$ is safe for dynamics $\theta ^ { * }$ , and therefore by Lemma 43, the state of $C ^ { \mathrm { a l g } }$ at time $T _ { 0 }$ satisfies $| x _ { T _ { 0 } } ^ { \prime } | \leq B _ { x } = \tilde { O } _ { T } ( 1 )$ conditional on E. Therefore, by Lemma 39, conditional on $E _ { i }$ ,

$$(T - T _ {0}) \cdot J (\theta^ {*}, C _ {F _ {\mathrm{opt}} (\hat {\theta} _ {\mathrm{wu}})} ^ {\mathrm{unc}}, T - T _ {0}, x _ {T _ {0}} ^ {\prime}, \{w _ {t} \} _ {t = T _ {0}} ^ {T - 1}) - (T - T _ {0}) \cdot J (\theta^ {*}, C _ {F _ {\mathrm{opt}} (\hat {\theta} _ {\mathrm{wu}})} ^ {\mathrm{unc}}, T - T _ {0}, 0, \{w _ {t} \} _ {t = T _ {0}} ^ {T - 1}) = \tilde {O} _ {T} (1).$$
