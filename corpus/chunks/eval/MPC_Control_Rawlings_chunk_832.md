# A.11 Continuity

We now summarize some essential properties of continuous functions.

1. A function $f : \mathbb { R } ^ { n }  \mathbb { R } ^ { m }$ is said to be continuous at a point $x \in \mathbb { R } ^ { n }$ , if for every $\delta > 0$ there exists a $\rho > 0$ such that

$$\left| f \left(x ^ {\prime}\right) - f (x) \right| < \delta \forall x ^ {\prime} \in \operatorname{int} (B (x, \rho))$$

A function $f : \mathbb { R } ^ { n }  \mathbb { R } ^ { m }$ is said to be continuous if it is continuous at all $\boldsymbol { x } \in \mathbb { R } ^ { n }$ .

2. Let X be a closed subset of $\mathbb { R } ^ { n }$ . A function $f : X \to \mathbb { R } ^ { m }$ is said to be continuous at a point x in X if for every $\delta > 0$ there exists a $\rho > 0$ such that

$$\left| f \left(x ^ {\prime}\right) - f (x) \right| < \delta \forall x ^ {\prime} \in \operatorname{int} (B (x, \rho)) \cap X$$

A function $f : \mathbb { R } ^ { n }  \mathbb { R } ^ { m }$ is said to be continuous on X if it is continuous at all x in X.
