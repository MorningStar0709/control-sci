# C. SDP variables for the proposed method

Since the computational time can vary depending on solvers, we compare the number of variables involved in the semidefinite programming (SDP) formulations. In the case that the edge set E is chosen as $\mathcal { Q } _ { \mathcal { G } }$ , the orders of the number of SDP variables are $\mathcal { O } ( n d ) , \ O ( n d ) , \ O ( n ^ { 2 } )$ and $\mathcal { O } ( n ^ { 2 } d ^ { 2 } )$ for $\mathcal { K } _ { \mathrm { d i a g } } , \mathcal { K } _ { S } , \mathcal { K } _ { \mathrm { e x t } }$ and $\boldsymbol { \mathcal { K } } _ { S , \mathrm { { e x t } } }$ , respectively, where d denotes the maximum number of neighboring agents; precisely,

$$
\begin{array}{l} \mathcal {K} _ {\text { diag }}: \quad \mathcal {F} (Z) \leq n (d + 1) \\ \mathcal {K} _ {\mathcal {S}}: \mathcal {F} (\tilde {Z}) \leq 4 n d \\ \mathcal {K} _ {\mathrm{ext}}: \quad \mathcal {F} (Q) \leq \frac {1}{2} n (n + 1) \\ \mathcal {K} _ {\mathcal {S}, \mathrm{ext}}: \quad \mathcal {F} (\tilde {Q}) \leq \frac {1}{2} (2 n d) (2 n d + 1) \\ \end{array}
$$

where ${ \mathcal { F } } ( X )$ denotes the number of SDP variables in matrix X. Here, the number of SDP variables of the proposed method $\kappa _ { S , \mathrm { e x t } } , \mathcal { O } ( n ^ { 2 } d ^ { 2 } )$ , is the largest. However, for a large, sparse graph satisfying $n \gg d ,$ the number approximates $\mathcal { O } ( n ^ { 2 } )$ , the same as the number of SDP variables of the extended LMIs, $\mathcal { K } _ { \mathrm { e x t } }$ .
