# 7.8. Voltage control in power distribution networks

To validate the performance of the proposed method on a larger system, we conduct experiments on the voltage control problem in IEEE 33bus test feeder (Baran and Wu, 1989). We adopt the Lindisflow model where the dynamics of voltage is described by a linear transition model (Cui et al., 2022b; Zhu and Liu, 2015). The state ${ \pmb x } ( t ) \in \mathbb { R } ^ { 3 2 }$ is voltage in all the buses apart from the reference bus (the voltage of the reference bus is fixed). The action is the reactive power in each bus. We assume that there is no real-time communication between buses during real-time implementation, so the action at each bus can only change with the local measurement of voltage. The goal is to train a linear decentralized feedback controller to minimize total voltage deviation as well as the control effort in the time horizon $K = 2 0$ , written as $\begin{array} { r } { J ( \pmb { \theta } ) = \sum _ { k = 1 } ^ { K } \| \pmb { y } ( k ) \| _ { 1 } + 0 . 3 \| \pmb { u } _ { \pmb { \theta } } ( k ) \| _ { 1 } } \end{array}$ . The number of training episode is $E = 5 0 0$ .
