# 4.2.1 SSNN based D-NMPC: Numerical example

The SSNN-based D-NMPC is illustrated on a CSTR system example. The CSTR is defined by the discrete-time state equation in Eq. (74). The input-output data is generated by a forward simulation of the state equation over $\mathrm { N _ { T } } = 1 0 0 0$ instants with the sampling period T = 1 sec., and initial condition $\mathbf { x } _ { 0 } = \left[ 0 \begin{array} { l l } { 0 } & { 0 } \end{array} \right] ^ { \top }$ . The control input u is chosen as a PRMS signal. The first 500 samples of the dataset are used for training the SSNN and the remaining samples for validation. The SSNN with ${ \mathrm { L } } _ { \mathrm { f } } = 2 , n _ { 1 } = 5 , { \mathrm { L } } _ { \mathrm { h } } = 2 , p _ { 1 } = 5$ and tanh activation function in the hidden layers is selected. The SSNN is trained using the training data where the loss function is chosen as in Eq. (78) and the unconstrained optimization problem in Eq. (79) is solved for θ using f mincon in Matlab. The output reference is chosen similarly to the SPC example and the parameters for MPC are chosen as with $\begin{array} { r } { \mathrm { ~ N = 1 0 , ~ } \mathbf { Q } _ { \mathbf { x } } = \left\lceil \begin{array} { l l } { 5 } & { 0 } \\ { 0 } & { 4 } \end{array} \right\rceil } \end{array}$ , and $\mathbf { R } _ { \mathbf { u } } = 3 $ . The state reference xr and control reference ur are computed by solving Eq. (80) with each of the output references. The training input and output data, along with the output predicted with the trained SSNN, are plotted in Fig. 8(a). The output tracking with SSNN-based NMPC is given in Fig. 8(b) and the corresponding control input in Fig. 8(c). It can be observed that the output converges to the desired reference with the SSNN-based NMPC scheme.
