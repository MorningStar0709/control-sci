Example C.4. In Setting 4.10 let $n = 2 , d = 0$ ,

$$
\gamma = \left( \begin{array}{c c} 2 & 1 \\ 1 & 1 \end{array} \right), \quad \text {and} \quad \rho = \left( \begin{array}{c c} 4 & 2 \\ 3 & 5 \end{array} \right).
$$

It then holds that $\gamma \in \mathbb { S } _ { > 0 } ^ { n }$ and

$$
\rho \gamma + \gamma \rho^ {\top} = \left( \begin{array}{c c} 2 0 & 1 7 \\ 1 7 & 1 6 \end{array} \right) \in \mathbb {S} _ {> 0} ^ {n},
$$

which implies that $\kappa \in \mathbb { S } _ { > 0 } ^ { n }$ . Now we can apply Corollary 4.11 and obtain the existence of a unique optimal strategy.

In some subsettings of Setting 4.10, the formulas in Corollary 4.11 simplify. For example, consider the condition $\rho \gamma = \gamma \rho ^ { \top }$ , which within Setting 4.10 is equivalent to $\mathcal { B } = \mathcal { B } ^ { \top }$ . It then follows that $\mathcal { R } = - \mathcal { B }$ . We further obtain that $\mathcal { R } ^ { - 1 } \mathcal { B } ^ { \top } = - I _ { n }$ and $\mathcal { B R } ^ { - 1 } \mathcal { B } ^ { \top } = - \mathcal { B }$ . If, in addition, $\rho$ is symmetric, then it holds that $\mathcal { B } = - \rho$ and $\kappa = \rho .$ . Therefore, we have the following corollary.
