$$
\begin{array}{l} \mathbb {P} \left[ \| \frac {1}{M} \sum_ {i = 1} ^ {M} S _ {L} (E, \mathcal {Y} _ {T} ^ {i}) - \mathbb {E} \left[ S _ {L} (E, \mathcal {Y} _ {T} ^ {i}) \right] \| \left\| \mathcal {A} _ {L} ^ {\intercal} H ^ {\intercal} H \mathcal {A} _ {L} \right\| _ {*} \geq t \right] \\ \leq 2 T \exp \left[ \frac {- M t ^ {2} / 2}{\kappa_ {\omega} ^ {2} \bar {\mu} _ {L} ^ {2} T ^ {2} \| E \| ^ {2} + 2 \kappa_ {\omega} \bar {\mu} _ {L} T \| E \| t / 3} \right] \\ \end{array}
$$

because by Lemma 8 we have $\kappa _ { L } ^ { 2 } T ^ { 2 } \lVert \mathbf { \mathcal { A } } _ { L } ^ { \intercal } H ^ { \intercal } H \mathbf { \mathcal { A } } _ { L } \rVert _ { * } \leq \bar { \mu } _ { L } T$ .

Next, by substituting t with $t / \| \mathcal { N } _ { L } [ E ] \|$ ∗ in (37) we have

$$\mathbb {P} \left[ \| \mathcal {Z} _ {L} - \mathbb {E} \left[ \mathcal {Z} _ {L} \right] \| \| \mathcal {N} _ {L} [ E ] \| _ {*} \geq t \right] \leq 2 T \exp \left[ \frac {- M t ^ {2} / 2}{\bar {\nu} _ {L} ^ {2} T ^ {2} \| E \| ^ {2} + 2 \bar {\nu} _ {L} T \| E \| t / 3} \right]$$

because Lemma 8 implies that $\kappa _ { L } ^ { 2 } T ^ { 2 } \lVert \mathcal { N } _ { L } [ E ] \rVert _ { * } \ \leq \ \bar { \nu } _ { L } T \lVert E \rVert$ . Thus, by combining the last two inequalities and using the union bound for (38) we obtain that

$$
\begin{array}{l} \mathbb {P} \left[ | \langle \nabla \widehat {J} _ {T} (L) - \nabla J _ {T} (L), E \rangle | \geq t \right] \leq 2 T \exp \left[ \frac {- M t ^ {2} / 2}{\bar {\nu} _ {L} ^ {2} T ^ {2} \| E \| ^ {2} + 2 \bar {\nu} _ {L} T \| E \| t / 3} \right] \\ + 2 T \exp \left[ \frac {- M t ^ {2} / 2}{\kappa_ {\omega} ^ {2} \bar {\mu} _ {L} ^ {2} T ^ {2} \| E \| ^ {2} + 2 \kappa_ {\omega} \bar {\mu} _ {L} T \| E \| t / 3} \right] \\ \end{array}
$$

Finally, substituting t with $t \| E \|$ and choosing $E = \nabla \widehat { J } _ { T } ( L ) - \nabla J _ { T } ( L )$ proves the second claim.
