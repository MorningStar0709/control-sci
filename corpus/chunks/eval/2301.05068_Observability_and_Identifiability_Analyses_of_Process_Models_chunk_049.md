# Neglecting model part E - computation of pH

Measuring the pH allows to infer $S _ { \mathrm { H } ^ { + } }$ + directly because these two variables are linked via the negative common logarithm, (28d). Measureing the pH hence allows to interpret the variable $S _ { \mathrm { H } ^ { + } }$ as a time-variant parameter (without any associated differential equation). The states $S _ { \mathrm { i o n } } , \ S _ { \mathrm { a c ^ { - } } }$ and $S _ { \mathrm { h c o 3 ^ { - } } }$ (x12 to $x _ { 1 4 }$ in (23) and (24)) only appear in the computation of the charge balance Φ, (27) which is required to calculate $S _ { \mathrm { H } ^ { + } }$ . However, as $S _ { \mathrm { H } ^ { + } }$ can be directly determined from pH measurements, the states $S _ { \mathrm { i o n } } , S _ { \mathrm { a c ^ { - } } }$ and $S _ { \mathrm { h c o 3 ^ { - } } }$ become redundant. Their respective differential equations can be cut out of the system of equations. The resulting model BMR3+ABCD (not shown here) only incorporates dissociation between ammonium and ammonia. Yet, full inhibition through all three factors of $I _ { \mathrm { a c } }$ (pH, nitrogen limitation and ammonia) are considered.
