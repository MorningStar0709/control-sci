Figure 11.57 shows the voltage commands $\delta e _ { \mathrm { i n } } ( t )$ that are produced by the controller $G _ { C } ( s )$ . Note that both the lead-only and lead-plus-integral controllers begin with a large voltage spike $\delta e _ { \mathrm { i n } } ( t ) \approx 5 . 3 \ : \mathrm { V }$ in order to increase the coil current, which in turn increases the electromagnetic force and causes the ball to accelerate upward toward the $\delta z _ { \mathrm { r e f } } = 0 . 0 0 3$ m reference position. However, when the ball approaches the electromagnet tip (reducing the air gap), the attractive electromagnetic force increases even if current remains constant, as shown by Eq. (11.67). When the ball overshoots the new reference position, both feedback controllers reduce the source voltage (and thus current) to values below the nominal voltage $( \mathrm { i } . \mathrm { e } . , \delta e _ { \mathrm { i n } } ( t ) < 0 )$ . Figures 11.55 and 11.57 show that the ball settles at about $\delta z = 0 . 0 0 4 8$ m (4.8 mm) and $\delta e _ { \mathrm { i n } } = - 0 . 8 \ : \mathrm { V }$ for the lead-only controller. This steady-state position can be determined by equating the electromagnetic force (11.67) with the ball’s weight mg.

$$\frac {K _ {F} I _ {\mathrm{ss}} ^ {2}}{(d - z _ {\mathrm{ss}}) ^ {2}} = m g \tag {11.81}$$

For steady-state position $z _ { \mathrm { s s } } = 0 . 0 0 4 8 \mathrm { m }$ , the steady-state current computed using Eq. (11.81) is $I _ { \mathrm { s s } } = 0 . 6 4 \mathrm { A }$ , and the corresponding steady-state source voltage is $e _ { \mathrm { i n } } ( \infty ) = R I _ { \mathrm { s s } } = 3 . 2 \mathrm { V } .$ Hence, the steady-state perturbation voltage is $\delta \boldsymbol { e } _ { \mathrm { i n } } = \boldsymbol { e } _ { \mathrm { i n } } ( \infty ) - \boldsymbol { e } _ { \mathrm { i n } } ^ { * } = - 0 . 8 \ : \mathrm { V } ,$ , as shown in Fig. 11.57. For the desired reference position $z _ { \mathrm { s s } } = 0 . 0 0 3 \ : \mathrm { m }$ , Eq. (11.81) shows that the steady-state current is $I _ { \mathrm { s s } } = 0 . 7 \mathrm { A }$ , and therefore the steady-state voltage is 3.5 V. Hence, the required steady-state perturbation voltage is $\delta e _ { \mathrm { i n } } = - 0 . 5 \mathrm { V } ,$ , which is the steadystate voltage command of the lead-plus-integral controller system, as shown in Fig. 11.57.
