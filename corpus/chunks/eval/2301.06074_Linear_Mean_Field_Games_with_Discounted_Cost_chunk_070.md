Note that $\zeta ^ { * }$ is the optimal occupation measure of the LP formulation of $\mathrm { M D P } _ { \mu ^ { * } }$ , and $\mathbf { S O } _ { : }$ $\pi ^ { * }$ is the optimal policy. Hence, $\pi ^ { * } \in \Lambda ( \mu ^ { * } )$ . Furthermore, since

$$\zeta^ {*} (C) := (1 - \beta) \sum_ {t = 0} ^ {\infty} \beta^ {t} \operatorname * {P r} \left[ (x (t), a (t)) \in C \right], C \in \mathcal {B} (\mathrm{X} \times \mathrm{A})$$

we have

$$
\begin{array}{l} \zeta^ {*} p _ {\mu^ {*}} (\cdot) = \int_ {X \times A} p _ {\mu^ {*}} (\cdot | x, a) \zeta^ {*} (d x, d a) \\ = \int_ {\mathsf {X} \times \mathsf {A}} p _ {\mu^ {*}} (\cdot | x, a) \left\{(1 - \beta) \sum_ {t = 0} ^ {\infty} \beta^ {t} \operatorname * {P r} \left[ (x (t), a (t)) \in d x \times d a \right] \right\} \\ = (1 - \beta) \sum_ {t = 0} ^ {\infty} \beta^ {t} \left\{\int_ {\mathsf {X} \times \mathsf {A}} p _ {\mu^ {*}} (\cdot | x, a) \operatorname * {P r} \left[ (x (t), a (t)) \in d x \times d a \right] \right\} \\ \end{array}

\begin{array}{l} = (1 - \beta) \sum_ {t = 0} ^ {\infty} \beta^ {t} \operatorname * {P r} \left[ x (t + 1) \in \cdot \right] \\ = \frac {1 - \beta}{\beta} \sum_ {t = 1} ^ {\infty} \beta^ {t} \operatorname * {P r} \left[ x (t + 1) \in \cdot \right] + \frac {1 - \beta}{\beta} \operatorname * {P r} \left[ x (0) \in \cdot \right] - \frac {1 - \beta}{\beta} \operatorname * {P r} \left[ x (0) \in \cdot \right] \\ = \frac {1 - \beta}{\beta} \sum_ {t = 0} ^ {\infty} \beta^ {t} \operatorname * {P r} \left[ x (t + 1) \in \cdot \right] - \frac {1 - \beta}{\beta} \mu^ {(\cdot)} (\text {as} x (0) \sim \mu^ {*}) \\ = \frac {\hat {\zeta} ^ {*} (\cdot)}{\beta} - \frac {\mu^ {*} (\cdot)}{\beta} + \mu^ {*} (\cdot) \\ \end{array}
$$

But since $\mu ^ { * } = \zeta ^ { * } p _ { \mu ^ { * } }$ , the last expression implies that $\hat { \zeta } ^ { * } = \mu ^ { * }$ . Hence, $\mu ^ { * }$ satisfies the following in view of property (b)

$$\mu^ {*} (\cdot) = \int_ {\mathsf {X} \times \mathsf {A}} p _ {\mu^ {*}} (\cdot | x, a) \pi^ {*} (d a | x) \mu^ {*} (d x)$$

that is, $\mu ^ { * }$ is an invariant distribution of the transition probability $p _ { \mu ^ { * } } ^ { \pi ^ { * } }$ . Hence, $\mu ^ { * } \in \Phi ( \pi ^ { * } )$ . This means that $( \mu ^ { * } , \pi ^ { * } )$ is MFE. □
