# APPENDIX

In the appendix, an example is given to show the states of the system (1) controlled by (2) are unbounded under bounded disturbance. We observe a simple MAS with only 2 agents under fully connected communication topology. Moreover, let the dynamics of each agent follows (1), which is then controlled by (2). Then, the concatenated controlled system is written as

$$
\left[ \begin{array}{l} \dot {x} _ {1} \\ \dot {x} _ {2} \end{array} \right] = - c \left[ \begin{array}{c c} 1 & - 1 \\ - 1 & 1 \end{array} \right] \left[ \begin{array}{l} x _ {1} \\ x _ {2} \end{array} \right] + \left[ \begin{array}{l} f (x _ {1}) - \mu_ {1} (x _ {1}) \\ f (x _ {2}) - \mu_ {1} (x _ {2}) \end{array} \right] \tag {36}
$$

with $\mu _ { i } ( x _ { i } )$ is sufficiently accurate such that only a small constant prediction error $\varepsilon \in \mathbb { R } _ { + }$ is left, i.e., $f ( x _ { i } ) { - } { \mu } _ { i } ( x _ { i } ) =$ ε for all $i = 1 , 2 .$ . The explicit solution of (36) denotes

$$
\left[ \begin{array}{l} x _ {1} (t) \\ x _ {2} (t) \end{array} \right] = \left[ \begin{array}{l} 1 \\ 1 \end{array} \right] \frac {x _ {1} (0) + x _ {2} (0) + 2 \varepsilon t}{2} - \left[ \begin{array}{l} 1 \\ - 1 \end{array} \right] \frac {x _ {1} (0) - x _ {2} (0)}{2} \exp (- 2 c t) \tag {37}
$$

For non-zero prediction error ε, the unboundness of the state trajectory is obvious to see by considering

$$\lim _ {t \to \infty} x _ {i} (t) = \left(x _ {1} (0) + x _ {2} (0)\right) / 2 + \varepsilon \lim _ {t \to \infty} t \to \infty , \quad (3 8)$$

for $i \ = \ 1 , 2 .$ . Therefore, the conventional control law (2) can only stabilize the MAS with (1) for completely known dynamics, i.e., $f ( x _ { i } ) ~ = ~ \mu _ { i } ( x _ { i } )$ , and not suitable in our setting. There also exists other robust or adaptive control laws for average consensus, which will be regarded as future work due to our main focus on the decentralized event-trigger design for online learning in multi-agent systems.
