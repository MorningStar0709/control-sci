# C Analytical Conic Bound

Lemma 20 Consider (27) over $\Omega \subseteq { \mathcal { X } }$ , where Ω is a RPI set and $\chi = [ - 1 , 1 ]$ . If there exists a $\mathbf { P } > 0 f o r$ which

$$
\left[ \begin{array}{c c} - k _ {2} \mathbf {P} + C ^ {\top} C & \frac {1}{2} \mathbf {P} ^ {\top} B + C ^ {\top} D - C ^ {\top} \frac {a + b}{2} \mathbf {I} \\ \frac {1}{2} B ^ {\top} \mathbf {P} + D ^ {\top} C - \frac {a + b}{2} \mathbf {I} C & a b \mathbf {I} + D ^ {\top} D - (a + b) \mathbf {I} D \end{array} \right] \preceq 0, \tag {C.1}
$$

then $\mathcal { G } _ { \Omega } \in c o n e ( a , b )$

PROOF. Consider the storage function $V ~ = ~ { \textstyle \frac { 1 } { 2 } } { \bf P } x ^ { 2 }$ . For the system to be locally dissipative,

$$
\begin{array}{l} x ^ {\top} \mathbf {P} \dot {x} \leq \\ (C x + D u) ^ {\top} \mathbf {Q} (C x + D u) + 2 (C x + D u) ^ {\top} \mathbf {S} u \\ + u ^ {\top} \mathbf {R} u, \\ \end{array}
$$

must be satisfied for all $x \in \Omega$ . The above inequality is equivalent to

$$
\begin{array}{l} k _ {1} \mathbf {P} x ^ {2} - k _ {1} \mathbf {P} x ^ {4} \geq \\ - k _ {2} \mathbf {P} x ^ {2} + x \mathbf {P} B u \\ - (C x + D u) ^ {\top} \mathbf {Q} (C x + D u) \\ - 2 (C x + D u) ^ {\top} \mathbf {S} u - u ^ {\top} \mathbf {R} u. \\ \end{array}
$$

Because $x ^ { 4 } \leq x ^ { 2 }$ when $x \in [ - 1 , 1 ]$ , the above inequality may be bound above by

$$
\begin{array}{l} 0 \geq - (C x + D u) ^ {\top} \mathbf {Q} (C x + D u) - 2 (C x + D u) ^ {\top} \mathbf {S} u \\ - \boldsymbol {u} ^ {\top} \mathbf {R} \boldsymbol {u} - k _ {2} \mathbf {P} x ^ {2} + x \mathbf {P} B \boldsymbol {u}. \\ \end{array}
$$

Replacing Q, S, and R with the associated values in Table 1 for non-degenerate interior conic systems and reformulating the above inequality as an LMI results in (C.1). ✷
