# E.2 Kalman Filter Properties

Lemma E.4. Consider system (1) and suppose Assumption 2.1 holds. Let $\kappa _ { F }$ and $\gamma _ { F }$ be given by (14). Then, it holds that $0 < \gamma _ { F } < 1$ and $\left\| ( A - L C ) ^ { k } \right\| \leq \kappa _ { F } \gamma _ { F } ^ { k }$ for all $k \in \mathbb { Z } _ { \geq 0 }$ , where L is given by (3) and satisfies $\| L \| \leq \kappa _ { F }$ .

Proof. First, we note from (13) that $W \succeq \alpha _ { 0 } I _ { n }$ and Σ given by (10) satisfies $\Sigma \preceq \bar { \sigma } I _ { n }$ , which also implies via (13) that $W \succeq ( \alpha _ { 0 } / \bar { \sigma } ) \Sigma = 2 ( 1 - \gamma _ { F } ) \Sigma$ . Since we know from [4, Chapter 4] that Σ also satisfies the following equation:

$$\Sigma = (A - L C) \Sigma (A - L C) ^ {\top} + W + L V L ^ {\top}, \tag {E.1}$$

we get

$$\Sigma \succeq (A - L C) \Sigma (A - L C) ^ {\top} + 2 (1 - \gamma_ {F}) \Sigma ,$$

which further implies

$$\Sigma^ {- 1 / 2} (A - L C) \Sigma (A - L C) ^ {\top} \Sigma^ {- 1 / 2} \preceq (2 \gamma_ {F} - 1) I _ {n}.$$

Denoting $H = \Sigma ^ { 1 / 2 }$ and $\Lambda = \Sigma ^ { - 1 / 2 } ( A - L C ) \Sigma ^ { 1 / 2 }$ , we have $A - L C = H \Lambda H ^ { - 1 }$ . In the following, we will provide upper bounds on $\| H \| , \left\| H ^ { - 1 } \right\|$ and ∥Λ∥.

Observing that

$$\Sigma^ {- 1 / 2} (A - L C) \Sigma (A - L C) ^ {\top} \Sigma^ {- 1 / 2} = H ^ {- 1} (H \Lambda H ^ {- 1}) H H (H \Lambda H ^ {- 1}) ^ {\top} H ^ {- 1} = \Lambda \Lambda^ {\top} \preceq (2 \gamma_ {F} - 1) I _ {n},$$

we obtain $\| \Lambda \| \le \sqrt { 2 \gamma _ { F } - 1 } \le \gamma _ { F }$ . Since $H = \Sigma ^ { 1 / 2 }$ , we directly have $\| H \| = \left\| \Sigma ^ { 1 / 2 } \right\| = \sqrt { \bar { \sigma } }$ From (E.1), we also know that $\Sigma \succeq W \succeq \alpha _ { 0 } I _ { n }$ , which implies that $\Sigma ^ { - 1 / 2 } \preceq \sqrt { 1 / \alpha _ { 0 } } I _ { n }$ and thus $\| \Sigma ^ { - 1 / 2 } \| \le \sqrt { 1 / \alpha _ { 0 } }$ . Since $H ^ { - 1 } = \Sigma ^ { - 1 / 2 }$ , we have $\| H ^ { - 1 } \| \le \sqrt { 1 / \alpha _ { 0 } }$ . Combining the above arguments, we see that for any $k \in \mathbb { Z } _ { \geq 0 } , \| ( A - L C ) ^ { k } \| = \| H \| \| H ^ { - 1 } \| \| \Lambda \| ^ { k } \leq \kappa _ { F } \gamma _ { F } ^ { k }$ . Moreover, since $V \succeq \alpha _ { 0 } I _ { p }$ by (13), we get from (E.1) that $\Sigma \succeq L V L ^ { \top } \succeq \alpha _ { 0 } L L ^ { \top }$ which implies $\lVert \bar { \boldsymbol { L } } \rVert \leq \sqrt { \bar { \sigma } / \alpha _ { 0 } } .$ .

Finally, our above arguments readily show that $\bar { \sigma } \geq \alpha _ { 0 }$ , which implies via $\gamma _ { F } = 1 - \alpha _ { 0 } / ( 2 \bar { \sigma } )$ by (13) that $0 < \gamma _ { F } < 1$ .
