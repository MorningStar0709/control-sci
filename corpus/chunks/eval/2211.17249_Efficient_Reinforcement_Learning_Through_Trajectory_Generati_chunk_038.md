# 7.6. Trajectory generation algorithm for output-feedback control

Algorithm 3 Trajectory generation for output-feedback control   
Data collection: Collect historic measurement of the system and stack each T-length input-output trajectory as Hankel matrix H shown in (5) until $\text{rank}(\mathcal{H}) = n + Tm$ Data generation: Input : Hankel matrix H, weights $\theta$ and the distribution D for the control policy, the batchsize Q for the generated trajectories, the set $S_{X}$ of historic initial extended state $\mathcal{X}(T_0 - 1)$ Function TrajectoryGen( $H, \theta, D, Q, S_{X}$ ):
Plug in $\theta$ to compute $G_{\theta}$ in (16).
Conduct eigenvalue decomposition of $(G_{\theta}G_{\theta}^{\top})$ to obtain $P_{\theta} := [p_1 \quad p_2 \quad \cdots \quad p_s]$ and $\Lambda = diag(\lambda_1, \lambda_2, \cdots, \lambda_s)$ with $\lambda_i$ being nonzero eigenvalues and $p_i$ being orthonormal eigenvectors.
for i = 1 to Q do
    Sample $\mathcal{X}(T_0 - 1)$ from $S_{X}$ . Sample $\{w_i(T_0 - 1), \cdots, w_i(T - 1)\}$ from distribution D.
    Compute the coefficient $g_i^* = G_\theta^\top P_\theta \Lambda^{-1} P_\theta^\top [w_i(T_0 - 1)^\top \cdots w_i(T - 1)^\top \mathcal{X}(T_0 - 1)^\top]^{\top}$ .
    Generate the i-th trajectory $\tau_i := [\tilde{u}_i(T_0 - 1)^\top \cdots \tilde{u}_i(T - 1)^\top \tilde{y}_i(T_0 - 1)^\top \cdots \tilde{y}_i(T - 1)^\top]^{\top} = [\mathcal{H}_u^{T_0 - 1:T - 1}, \mathcal{H}_y^{T_0 - 1:T - 1}] g_i^*$ .
end
return $[\tau_1, \cdots, \tau_Q]$
