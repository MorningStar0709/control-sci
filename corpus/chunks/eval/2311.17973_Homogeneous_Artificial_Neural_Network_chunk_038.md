# 5.2 Example 2: Identification of homogeneous system

Let us consider now the rotation dynamics (1) of the rigid body with ${ \boldsymbol { J } } =$ $\begin{array}{c} \left( \begin{array} { c } { 1 } \\ { 0 . 3 } \end{array} \right) 1 . 2 \ 0 . 2  \\ { 0 . 1 \ 0 . 2 \ 0 . 8 } \end{array}$ . First, we approximate the right-hand side of (1)

$$h (x) = - J ^ {- 1} (\omega \times J \omega + \tau), x = (\omega^ {\top}, \tau^ {\top}) ^ {\top} \in \mathbb {R} ^ {6}$$

by means of the conventional ANN with the sigmoid activation function

$$g _ {\epsilon} (x) = C \sigma (A x + b)$$

where $A \in \mathbb { R } ^ { N \times 6 } , b \in \mathbb { R } ^ { N }$ are randomly selected for $N = 5 0 0$ and $C \in \mathbb { R } ^ { 3 \times N }$ is obtained by means of the least square error minimization for $M = 2 0 0 0 0$ measurements $y _ { i } = h ( x _ { i } )$ of the function h at the points $x _ { i } \in \mathbb { R } ^ { 6 }$ randomly uniformly distributed in the set

$$\Omega = \{x \in \mathbb {R} ^ {6}: 0. 9 5 \leq | x | \leq 1. 0 5 \}.$$
