In our SSR, the state vector is $\mathbf { x } = \left[ z _ { 1 } \dot { z } _ { 1 } z _ { 2 } \dot { z } _ { 2 } \right] ^ { T }$ , the input variables are $u _ { 1 } = z _ { 0 } ( t )$ and $u _ { 2 } = \dot { z } _ { 0 } ( t )$ , and the two system outputs (or measurements) are position and acceleration of the driver mass, or $y _ { 1 } = z _ { 2 }$ and $y _ { 2 } = \ddot { z } _ { 2 }$ . Recall that this SSR is relatively easy to derive because we defined two independent input variables when, in reality, the system has one independent input, $z _ { 0 } ( t )$ , because $u _ { 2 } = \dot { u } _ { 1 }$ . Therefore, we must be careful that our second input $u _ { 2 }$ is indeed the time derivative of the first input.
