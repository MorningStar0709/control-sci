$$
\tilde {\Theta} ^ {\top} = \left[ \begin{array}{c c c c c c c} I _ {N - 1} & A _ {K} ^ {\top} & K & 0 & 0 & I _ {N - 1} & K \\ 0 & B _ {1} ^ {\top} & 0 & I _ {N - 1} & 0 & 0 & 0 \\ 0 & I _ {N - 1} & 0 & 0 & I _ {N - 1} & 0 & 0 \end{array} \right],
$$

and the negative definiteness of

$$
\tilde {\Theta} ^ {\top} \mathrm{diag} \left(\left[ \begin{array}{c c} 0 & P ^ {- 1} \\ P ^ {- 1} & 0 \end{array} \right], \frac {1}{\nu} \left[ \begin{array}{c c} c & 0 \\ 0 & - P ^ {- 1} \end{array} \right], \frac {1}{\tau} D _ {r}\right) \tilde {\Theta}. \tag {A.4}
$$

The above matrix $\left( \mathrm { { A . 4 } } \right)$ can be written as the sum of three terms. Multiply it by $\{ z ^ { \top } ~ ( K ^ { \top } z ) z ^ { \top } ~ r ( z , u ) ^ { \top } ]$ from the left and the transpose from the right, then the second term becomes

$$\nu^ {- 1} \left(K ^ {\top} z\right) ^ {2} \left(c - z ^ {\top} P ^ {- 1} z\right) \geq 0 \tag {A.5}$$

from the geometry of $\mathbb { X } _ { R o A }$ as well as the third term satisfies

$$
\begin{array}{l} \tau^ {- 1} \left[ \alpha c _ {1} ^ {2} \| z \| ^ {2} + \beta c _ {2} ^ {2} \| u \| ^ {2} - \| \hat {r} (z, u) \| ^ {2} \right] \\ \geq \tau^ {- 1} \left[ (c _ {1} \| z \| + c _ {2} \| u \|) ^ {2} - \| \hat {r} (z, u) \| ^ {2} \right] \geq 0 \tag {A.6} \\ \end{array}
$$

with probability $1 - \delta$ according to Proposition 1, which is due to $\textstyle { \frac { 1 } { \alpha } } + { \frac { 1 } { \beta } } = 1 \Rightarrow \alpha \beta = \alpha + \beta$ and

$$
\begin{array}{l} (\alpha - 1) c _ {1} ^ {2} \| z \| ^ {2} + (\beta - 1) c _ {2} ^ {2} \| u \| ^ {2} \\ \geq \sqrt {(\alpha - 1) (\beta - 1)} c _ {1} c _ {2} \| z \| \| = c _ {1} c _ {2} \| z \| \| u \|. \\ \end{array}
$$

Then we obtain the negatively definite property of time derivative of $V ( x )$ from the first term after multiplication, i.e.,

$$\frac {\mathrm{d}}{\mathrm{d} t} V (x) = z ^ {\top} P ^ {- 1} \left(\frac {\mathrm{d} z}{\mathrm{d} t}\right) + \left(\frac {\mathrm{d} z}{\mathrm{d} t}\right) ^ {\top} P ^ {- 1} z < 0 \quad (\text {A.7})$$

which shows the forward invariance of $\mathbb { X } _ { R o A }$

Since the dictionary function $\Psi ( x )$ is continuously differentiable, it is also Lipschitz continuous, i.e., $\exists \dot { L } _ { z } ~ >$ $0 , \mathrm { ~ s . t . } \| z \| = \| \Psi ( x ) \| \leq \hat { L _ { z } } \| x \|$ , which leads to

$$V (x) \leq \lambda_ {\max} (P ^ {- 1}) \| z \| ^ {2} \leq \lambda_ {\max} (P ^ {- 1}) L _ {z} ^ {2} \| x \| ^ {2}. \tag {A.8}$$

Furthermore, since there exists a matrix $C \in \mathbb { R } ^ { n \times N }$ such that $x = C z = C \Psi ( x ) \Rightarrow \| x \| \leq \| C \| \| z \|$ , then

$$V (x) \geq \lambda_ {\min} (P ^ {- 1}) \| z \| ^ {2} \geq \frac {\lambda_ {\min} (P ^ {- 1})}{\| C \| ^ {2}} \| x \| ^ {2}. \tag {A.9}$$

Together with (A.7), we conclude the closed-loop exponential stability. ✷
