so by the squeeze theorem,

$$\lim _ {n \to \infty} \frac {1}{n} | (\rho^ {n} (s), \nabla_ {y} ^ {2} \psi) _ {\mathcal {D} ^ {*} (Y; \mathbb {R}), \mathcal {D} (Y; \mathbb {R})} | = 0.$$

So, (29) is satisfied. Finally, we move onto the boundary condition. Define:

$$b _ {n} := (\nabla_ {y} ^ {\top} \rho_ {n}) \mathbf {L} _ {n} \mathbf {n} | _ {\partial Y}, b ^ {*} := \lim _ {n \to \infty} (\nabla_ {y} ^ {\top} \rho^ {n}) \mathbf {L} _ {n} \mathbf {n} | _ {\partial Y} = (\nabla_ {y} ^ {\top} \rho^ {*}) \mathbf {L n} | _ {\partial Y}.$$

From each solution $\rho ^ { n }$ , we know that $b _ { n } ( t , y ) ~ = ~ 0$ almost everywhere on $[ 0 , T ] \times \partial Y$ . So:

$$- e ^ {- n} < | | b _ {n} | | _ {\mathrm{L} ^ {2} ([ 0, T ] \times \partial Y; \mathbb {R})} ^ {2} < e ^ {- n},$$

and by the squeeze theorem, lim $_ { \cdot n \to \infty } | | b _ { n } | | _ { \mathrm { L } ^ { 2 } ( [ 0 , T ] \times \partial Y ; \mathbb { R } ) } ^ { 2 } = 0$ . Now, we can also dominate $b _ { n }$ by $g ( x ) = 1$ , so it follows:

$$\lim _ {n \to \infty} | | b _ {n} | | _ {\mathrm{L} ^ {2} ([ 0, T ] \times \partial Y; \mathbb {R})} ^ {2} = | | b ^ {*} | | _ {\mathrm{L} ^ {2} ([ 0, T ] \times \partial Y; \mathbb {R})} ^ {2} = 0,$$

hence $b ^ { * } = 0 ~ { \mathrm { a . e } }$ . on $[ 0 , T ] \times \partial Y$ . So, (30) is satisfied.

Thus, the limit point $( A , B , \rho ) ^ { * }$ is a weak solution to the ADP system.
