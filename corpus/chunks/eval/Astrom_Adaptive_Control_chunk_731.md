Pre-tuning. The tuning procedure requires reasonable controller parameters to be known so that a closed-loop system with a well-damped response is obtained. There is a pre-tune mode that can be used if the prior information that is needed is not available. A step test is done in which the user specifies the step size. Initial estimates of the controller parameters are determined from the step, and the time scale and the noise level are also determined. The pre-tune mode can be invoked only when the process is in steady state.
