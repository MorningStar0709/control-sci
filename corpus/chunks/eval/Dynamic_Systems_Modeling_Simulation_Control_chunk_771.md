# User-Defined Functions Library

Fcn: general function of input signal u. The user defines the output function in the dialog box using standard MATLAB mathematical operations such as 4\*sin(3\*u), exp(−2\*u), 5\*u - u^3, and so on.

Interpreted MATLAB Fcn: applies the specified MATLAB function to the input(s). Allows the user to write a customized script (M-file) that computes the output(s) as a function of the input(s).
