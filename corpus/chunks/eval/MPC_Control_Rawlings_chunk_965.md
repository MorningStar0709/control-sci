# Exercise B.2: Lyapunov function for exponential stability

Let $V : \mathbb { R } ^ { n } \to \mathbb { R } _ { \geq 0 }$ be a Lyapunov function for the system $x ^ { + } = f ( x )$ with the following properties. For all $x \in \mathbb { R } ^ { n }$

$$a _ {1} | x | ^ {\sigma} \leq V (x) \leq a _ {2} | x | ^ {\sigma}V (f (x)) - V (x) \leq - a _ {3} | x | ^ {\sigma}$$

in which $a _ { 1 } , a _ { 2 } , a _ { 3 } , \sigma > 0$ . Show that the origin of the system $x ^ { + } = f ( x )$ is globally exponentially stable.
