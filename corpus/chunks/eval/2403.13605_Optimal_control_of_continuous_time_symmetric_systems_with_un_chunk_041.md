# A.1 Proof of Theorem 2

In the presence of measurement noise, the update step of Algorithm 1 is written as

$$\tilde {u} _ {k + 1} = (1 - \alpha) \tilde {u} _ {k} + \alpha \tilde {\mathcal {T}} _ {k} (\tilde {u} _ {k})= (1 - \alpha) \tilde {u} _ {k} + \alpha \big (\mathcal {T} (\tilde {u} _ {k}) + \zeta_ {k} \big)= (1 - \alpha) \tilde {u} _ {k} + \alpha \big (\mathcal {T} (u _ {k}) + \mathcal {S} (v _ {k}) + \zeta_ {k} \big), \tag {A.1}$$

where $\tilde { u } _ { k }$ is the input signal in the kth iteration. It is deduced from (A.1) that the propagated noise $v _ { k }$ in (40) satisfies the recurrence

$$
\begin{array}{l} \nu_ {k + 1} = (1 - \alpha) u _ {k} + \alpha \mathcal {T} (u _ {k}) - u _ {k + 1} + (1 - \alpha) \nu_ {k} \\ + \alpha \mathcal {S} (\nu_ {k}) + \alpha \zeta_ {k} \\ = \big ((1 - \alpha) \mathcal {I} + \alpha \mathcal {S} \big) \nu_ {k} + \alpha \zeta_ {k} (A.2) \\ = R ^ {- 1 / 2} \mathcal {M} _ {\alpha} R ^ {1 / 2} \nu_ {k} + \alpha \zeta_ {k}, (A.3) \\ \end{array}
$$

where $\mathcal { M } _ { \alpha }$ is given by (32). As $\nu _ { 0 } = 0$ , it follows from (A.3) and (39) that

$$
\begin{array}{l} \nu_ {k} = \alpha \sum_ {i = 0} ^ {k - 1} R ^ {- 1 / 2} \mathcal {M} _ {\alpha} ^ {i} R ^ {1 / 2} \zeta_ {k - 1 - i} \\ = \sum_ {i = 0} ^ {k - 1} \mathcal {K} _ {1, i} \eta_ {1, k - 1 - i} + \mathcal {K} _ {2, i} \eta_ {2, k - 1 - i}, \quad k \in \mathbb {N} \tag {A.4} \\ \end{array}
$$

where

$$
\begin{array}{l} \mathcal {K} _ {1, i} = - \alpha R ^ {- 1 / 2} \mathcal {M} _ {\alpha} ^ {i} R ^ {- 1 / 2} \Sigma_ {e} \mathcal {J G} \Sigma_ {e} Q \mathcal {J}, \\ \mathcal {K} _ {2, i} = - \alpha R ^ {- 1 / 2} \mathcal {M} _ {\alpha} ^ {i} R ^ {- 1 / 2} \Sigma_ {e} \mathcal {J} \tag {A.5} \\ \end{array}
$$

are bounded linear operators in $\mathcal { L } _ { 2 } ^ { m } ( 0 , t _ { f } )$ . Therefore,

$$\mathbb {E} (\nu_ {k}) = \sum_ {i = 0} ^ {k - 1} \mathcal {K} _ {1, i} \mathbb {E} (\eta_ {1, k - 1 - i}) + \mathcal {K} _ {2, i} \mathbb {E} (\eta_ {2, k - 1 - i}) = 0.$$

Next we show that the covariance $\begin{array} { r l } { \mathrm { C o v } ( \nu _ { k } ; t ) } & { { } = } \end{array}$ $\mathbb { E } \big ( \nu _ { k } ( t ) \nu _ { k } ^ { \prime } ( t ) \big )$ is bounded. Since the measurement noises

$$\eta_ {j, i}, \quad j = 1, 2, i = 0, 1, \dots , k - 1$$

are zero-mean pairwise-independent random variables, one can write from (A.4) that

$$\operatorname{Cov} (\nu_ {k}; t) = \tag {A.6}\sum_ {i = 0} ^ {k - 1} \left(\operatorname{Cov} \left(\mathcal {K} _ {1, i} \eta_ {1, k - 1 - i}; t\right) + \operatorname{Cov} \left(\mathcal {K} _ {2, i} \eta_ {2, k - 1 - i}; t\right)\right).$$
