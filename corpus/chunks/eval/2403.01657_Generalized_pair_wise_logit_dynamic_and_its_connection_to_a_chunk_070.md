# C.2 Convergence study

A convergence study on the GPL dynamic and MFG counterpart is conducted for the identified model with $\eta = 0 . 1$ for the second example in Section 4. For each model, the numerical solution with the resolution $N = 2 ^ { 9 }$ and $M = 2 ^ { 4 } \times 1 0 ^ { 3 }$ is considered as a benchmark solution because no analytical solution has been found. Subsequently, we estimate the convergence rate of numerical solutions through a series of computations with the lower resolutions $\left( N , M \right) = \left( N _ { i } , M _ { i } \right) = \left( 2 ^ { 5 + i } , 2 ^ { i } \times 1 0 ^ { 3 } \right)$ ( i = 1, 2,3 ). The numerical solution with $\left( N , M \right) = \left( N _ { i } , M _ { i } \right)$ is denoted by  . Assume $\left( N , M \right) = \left( N _ { i } , M _ { i } \right)$ . Then, the convergence rate of $\Phi _ { i + 1 }$ is estimated for each $i = 2 , 3$ as $\left( \frac { \left\| \Phi _ { i + 1 } - \Phi _ { 4 } \right\| _ { \infty } } { \left\| \Phi _ { i } - \Phi _ { 4 } \right\| _ { \infty } } \right)$ with the error $\left\| \Phi _ { i } - \Phi _ { 4 } \right\| _ { \alpha }$ being the maximum difference $\left| \Phi _ { i } - \Phi _ { 4 } \right|$ among all space–time grids using the arithmetic average; i.e., for a given $i \in \{ 1 , 2 , 3 \}$ , we set

$$
\left| \Phi_ {i} - \Phi_ {4} \right| = \max_ {\substack {j = 1,2,3,\dots ,N _ {i} \\ k = 1,2,3,\dots ,M _ {i}}} \left| \Phi_ {i} ^ {(k - 1)} \left(x _ {j - 1 / 2}\right) - \hat {\Phi} _ {4} \left(x _ {j - 1 / 2}\right) \right| _ {t = k / M _ {i}} \Bigg |, \tag{125}
$$

where $\left. \hat { \Phi } _ { 4 } \left( x _ { j - 1 / 2 } \right) \right| _ { t = k / M _ { i } }$ is evaluated as the arithmetic mean of the nearest $2 ^ { 4 - i }$ solution values of $\Phi _ { 4 }$ t-k/M defined around $x _ { j - 1 / 2 }$ . The arithmetic mean is obtained here because the grid points are not always overlapping between different numerical resolutions. The first-order convergence of the obtained numerical solutions can be observed from these tables.
