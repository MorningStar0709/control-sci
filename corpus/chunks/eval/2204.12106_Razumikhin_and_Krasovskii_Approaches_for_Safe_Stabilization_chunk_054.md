To guarantee the synchronization objective, the desired reference trajectory for the master robot is given by $\mathbf { X } _ { \mathrm { m } } \in \mathbb { R } ^ { 2 }$ , and accordingly the reference trajectory for the slave robot is given by $\mathbf { X } _ { \mathrm { s } } ~ \in ~ \mathbb { R } ^ { 2 }$ . To this end, the following Lyapunov-Krasovskii functional is introduced:

$$
\begin{array}{l} V _ {\mathrm{k}} (x _ {t}) := e ^ {T} (t) P _ {1} e (t) + \int_ {t - \Delta_ {\mathrm{m}}} ^ {t} e _ {\mathrm{m}} ^ {T} (s) P _ {2} e _ {\mathrm{m}} (s) d s \\ + \int_ {t - \Delta_ {\mathrm{s}}} ^ {t} e _ {\mathrm{s}} ^ {T} (s) P _ {3} e _ {\mathrm{s}} (s) d s, \tag {67} \\ \end{array}
$$

where $P _ { 1 } ~ \in ~ \mathbb { R } ^ { 4 \times 4 } , P _ { 2 } , P _ { 3 } ~ \in ~ \mathbb { R } ^ { 2 \times 2 }$ are positive-definite matrices, and $e ( t ) : = ( e _ { \mathrm { m } } ( t ) , e _ { \mathrm { s } } ( t ) )$ is the synchronization error with $e _ { \mathrm { m } } ( t ) : = x _ { \mathrm { m } } ( t ) - \mathbf { x } _ { \mathrm { m } } ( t ) \in \mathbb { R } ^ { 2 }$ and $e _ { \mathrm { s } } ( t ) : = x _ { \mathrm { s } } ( t ) - \mathbf { x } _ { \mathrm { s } } ( t ) \in \mathbb { R } ^ { 2 }$ . From Definition 8, $V _ { \mathsf { k } }$ is a CLKF if the condition (31) holds.

To avoid the collision between the robots and the obstacle $O \left( \mathrm { i . e . } \right.$ ., the brown region in Figure (3a)), we introduce the functional $h _ { \mathsf { k } } ( \phi ) = ( \phi _ { 1 } ^ { 2 } - \bar { 3 } ) ^ { 2 } + \phi _ { \gamma } ^ { 2 } - \bar { 4 }$ with $\phi = ( \phi _ { 1 } , \phi _ { 2 } )$ , and the safety objective is satisfied only when $h _ { \mathrm { k } } ( \phi ) \geq 0$ .
