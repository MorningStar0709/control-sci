$$
\begin{array}{l} \left\| x (t) - \mathrm{x} _ {\Lambda_ {1}} (t) \right\| = \left\| \left(x (t) - \hat {x} _ {\Lambda_ {1}} (t)\right) + \left(\hat {x} _ {\Lambda_ {1}} (t) - \mathrm{x} _ {\Lambda_ {1}} (t)\right) \right\| \\ \leq \left\| \Phi_ {\Lambda_ {1}} ^ {\dagger} \left(z _ {\Lambda_ {1}} (t) - \hat {z} _ {\Lambda_ {1}} (t)\right) \right\| + \| \hat {x} _ {\Lambda_ {1}} (t) - x _ {\Lambda_ {1}} (t) \| \\ \leq \kappa \tilde {z} _ {\mathrm{ini}} \cdot \mathbf {1} _ {\{t <   l _ {\max} \}} + \epsilon . \tag {B.4} \\ \end{array}
$$

Next, from (12) and (B.1), we have $\| \mathbf { x } _ { \Lambda _ { 1 } } ( t ) \| \le \| \hat { x } _ { \Lambda _ { 1 } } ( t ) \| +$ $\epsilon \leq \kappa M + \epsilon ,$ or equivalently,

$$\left\| \bar {x} _ {\Lambda_ {1}} (t) \right\| \leq \frac {\kappa M + \epsilon}{\mathfrak {s} _ {1} ^ {2} \mathfrak {s} _ {2}}. \tag {B.5}$$

If (14) holds in addition, then

$$\| \bar {x} _ {\Lambda_ {1}} (t) - \bar {x} (t) \bmod q \| \leq \| \bar {r} (t) \| \leq \frac {2 \kappa \tilde {z} _ {\mathrm{ini}} + \epsilon}{\mathsf {s} _ {1} ^ {2} \mathsf {s} _ {2}}.$$

Hence, under (13), Proposition 12 yields

$$\| \bar {x} _ {\Lambda_ {1}} (t) - \bar {x} (t) \| = \| \bar {x} _ {\Lambda_ {1}} (t) - \bar {x} (t) \bmod q \| \leq \| \bar {r} (t) \|. \tag {B.6}$$

Since $\| x ( t ) - x ( t ) \| \leq \| x ( t ) - x _ { \Lambda _ { 1 } } ( t ) \| + \| x _ { \Lambda _ { 1 } } ( t ) - x ( t ) \|$ , combining (B.4) and (14) results in (15), and this concludes the proof.
