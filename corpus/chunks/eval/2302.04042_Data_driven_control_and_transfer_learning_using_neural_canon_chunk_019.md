A detailed derivation of the mathematical model using a variational approach can be found in [5] and [13]. The governing equations include the partial differential equation of the Euler-Bernoulli beam, the momentum equations of the driving unit and tip mass, as well as the boundary constraints. By approximating the beam deformation w(Y, t) with a first-order Rayleigh-Ritz ansatz $w ( Y , t ) = x _ { c } ( t ) +$ $\Psi ( Y ) { \bar { q } } ( t )$ and proper spatial ansatz function $\Psi ( Y )$ , the Euler-Lagrange equations yield a finite-dimensional mechanical system $M \ddot { q } + C q = G u$ with generalized coordinates $\boldsymbol { q } = [ \dot { x } _ { c } , \bar { q } ] ^ { T }$ and input force $u = F$ . The matrices are

$$
M = \left[ \begin{array}{c c} m _ {1 1} & m _ {1 2} \\ m _ {1 2} & m _ {2 2} \end{array} \right], C = \left[ \begin{array}{c} 0 \\ c _ {2} \end{array} \right], \text { and } G = \left[ \begin{array}{c} 1 \\ 0 \end{array} \right]
$$

with $ m _ { 1 1 } \ = \ \rho A L + m _ { c } + m _ { h } , \ m _ { 1 2 } \ = \ m _ { h } \Psi ( L ) \ +$ $\begin{array} { r } { \rho A \int _ { 0 } ^ { L } \Psi ( Y ) \mathrm { d } Y , m _ { 2 2 } = m _ { c } \Psi ( L ) ^ { 2 } + \rho A \int _ { 0 } ^ { L } \Psi ( Y ) ^ { 2 } \mathrm { d } Y } \end{array}$ and $\begin{array} { l } { c _ { 2 } } \end{array} = \ E I \int _ { 0 } ^ { L } \left( \partial ^ { 2 } \Psi ( Y ) / \partial Y ^ { 2 } \right) ^ { 2 } \mathrm { d } Y$ , where L denotes the length of the elastic mast, $\rho A$ the mass density and EI the bending stiffness. This approximation of the single mast stacker crane serves as simulation benchmark example for the proposed data-driven controller design.
