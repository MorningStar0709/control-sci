We also design and test the iFIR controller on a nonlinear passive plant. We replace the linear spring between the carts with a piece-wise linear one. The force-displacement characteristic has slope 1 for small displacements $( | \Delta x | \leq 0 . 5 )$ and 2 for large displacements $( | \Delta x | \ge 0 . 5 )$ . To deal with the nonlinearity, we reduce the aggressiveness of the reference model, by taking the reference model to be $\begin{array} { r } { \frac { 1 + 0 . 1 s } { 1 + 2 \zeta s T + s ^ { 2 } T ^ { 2 } } , \zeta = } \end{array}$ 1, $T \ = \ 1 \mathrm { s } .$ . The iFIR controller is designed using (2) and (14), based on nonlinear plant data. Performance degrades but stability is guaranteed. The iFIR design outperforms the optimal PID, as shown in Figure 7.
