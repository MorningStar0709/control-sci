Lemma E.6. For any positive semi-definite matrix $P , Q \ \in \ \mathbb { R } ^ { n \times n }$ with $P \ \preceq \ Q$ , it holds that $\| P \| \leq \| Q \|$ .

Proof. Since P and Q are symmetric, we know that $\| P \| = \lambda _ { \operatorname* { m a x } } ( P )$ and $\| Q \| = \lambda _ { \operatorname* { m a x } } ( Q )$ . By the relation $P \preceq Q$ , we have $x ^ { \top } P x \preceq x ^ { \top } Q x$ for all $x \in \mathbb { R } ^ { n }$ . Since $\begin{array} { r } { \lambda _ { \operatorname* { m a x } } ( M ) = \operatorname* { m a x } _ { x \in \mathbb { R } ^ { n } , \| x \| = 1 } x ^ { \top } } \end{array}$ M x for any $M \in \mathbb { R } ^ { n \times n }$ , we obtain that $\lambda _ { \operatorname* { m a x } } ( P ) \leq \lambda _ { \operatorname* { m a x } } ( Q )$ .

for all $k _ { 1 } , k _ { 2 } \in \mathbb { Z } _ { \geq 0 }$ with $k _ { 2 } > k _ { 1 }$ .

Proof. The proof follows from that of [36, Lemma 5].

Lemma E.8. For any $\gamma \in ( 0 , 1 )$ , it holds that

$$\sum_ {k = 2 s} ^ {t} (k - s) \gamma^ {k - s - 1} \leq \frac {\gamma^ {s - 1} ((1 - \gamma) s + \gamma)}{(1 - \gamma) ^ {2}}.$$

Proof. We have

$$\sum_ {k = 2 s} ^ {t} (k - s) \gamma^ {k - s - 1} = \sum_ {k = s} ^ {t - s - 1} k \gamma^ {k - 1} \leq \sum_ {k = s} ^ {\infty} k \gamma^ {k - 1}.$$

Similarly, consider the series

$$\sum_ {k = s} ^ {\infty} \gamma^ {k} = \gamma^ {s} \sum_ {k = 0} ^ {\infty} \gamma^ {k} = \frac {\gamma^ {s}}{1 - \gamma}.$$

Differentiating both sides of the above equation with respect to γ, one can show that

$$\sum_ {k = s} ^ {\infty} k \gamma^ {k - 1} = \frac {d}{d \gamma} \left(\frac {\gamma^ {s}}{1 - \gamma}\right) = \frac {\gamma^ {s - 1} ((1 - \gamma) s + \gamma)}{(1 - \gamma) ^ {2}},$$

which proves the second inequality of the lemma.
