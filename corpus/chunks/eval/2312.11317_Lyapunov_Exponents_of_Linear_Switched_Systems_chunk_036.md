In our particular case, that is $M = \operatorname { S L } _ { 2 } ( \mathbb { R } )$ and $\mathcal { M } = \mathfrak { s l } _ { 2 } ( \mathbb { R } )$ , we have that the Killing bilinear form on ${ \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ is non-degenerate, so it provides an intrinsic identification between $\mathcal { M }$ and $\mathcal { M } ^ { \ast }$ . That is, for every $\xi \in \mathcal { M } ^ { * }$ there is an $\eta \in \mathcal { M }$ such that

$$K (\eta , v) = \langle \xi , v \rangle \quad \forall v \in \mathcal {M}.$$

So, one can compute

$$K (\dot {\eta}, v) = \langle \dot {\xi}, v \rangle = \left\langle (\operatorname{ad} \mathrm{d} \mathcal {H}) ^ {*} \xi , v \right\rangle = \left\langle \xi , (\operatorname{ad} \mathrm{d} \mathcal {H}) v \right\rangle == \langle \xi , [ \mathrm{d} \mathcal {H}, v ] \rangle = K (\eta , [ \mathrm{d} \mathcal {H}, v ]) = K ([ \eta , \mathrm{d} \mathcal {H} ], v), \quad \forall v \in T _ {\mathbb {1}} M.$$

So, the final form for the Hamiltonian system is

$$
\left\{ \begin{array}{l} \dot {q} = L _ {q *} \mathrm{d} \mathcal {H}, \\ \dot {\eta} = [ \eta , \mathrm{d} \mathcal {H} ]. \end{array} \right. \tag {A.12}
$$

Notice that the second equation does not involve q.
