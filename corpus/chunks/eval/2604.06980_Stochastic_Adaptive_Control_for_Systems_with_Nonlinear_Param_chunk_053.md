For the last term on the RHS of (F.9), applying (A.8) and, for all k ≥ 0 wk = µ− $k \geq 0$ , setting $\mathbf { w } _ { k } = \mu _ { k } ^ { - 1 / 2 } d _ { k }$ and $\mathbf { A } _ { k } = P _ { k + 1 } ^ { - 1 }$

$$\sum_ {\tau = 0} ^ {t} \mu_ {\tau} ^ {- 1} d _ {\tau} ^ {2} \phi_ {\tau} ^ {\top} P _ {\tau + 1} \phi_ {\tau} = O (\log r _ {t}), a. s. \tag {F.11}$$

Let $D _ { t } = \mu _ { t } ^ { - 1 } d _ { t } ^ { 2 } \phi _ { t } ^ { \top } P _ { t + 1 } \phi _ { t }$ for $t \geq 0$ . By (F.11) and the Abel–Dini–Pringsheim theorem (see, e.g., (Hildebrandt , 1942, Theorem 1a)), it follows almost surely that

$$\sum_ {\tau = 0} ^ {t} \frac {\mu_ {\tau} ^ {- 1} d _ {\tau} ^ {2} \phi_ {\tau} ^ {\top} P _ {\tau + 1} \phi_ {\tau}}{(1 + \log r _ {\tau}) ^ {1 + \delta}} = O \left(\sum_ {\tau = 0} ^ {t} \frac {D _ {\tau}}{(\sum_ {k = 0} ^ {\tau} D _ {k}) ^ {1 + \delta}}\right) = O (1).$$

Since $\mu _ { t } \geq ( 1 + \log r _ { t } ) ^ { 1 + \delta }$ for all $t \geq 0$ , we further have

$$\sum_ {\tau = 0} ^ {t} \mu_ {\tau} ^ {- 2} d _ {\tau} ^ {2} \phi_ {\tau} ^ {\top} P _ {\tau + 1} \phi_ {\tau} = O (1), a. s. \tag {F.12}$$

From (F.12) and $\mathrm { s u p } _ { t \geq 0 } \mathbb { E } [ \| w _ { t + 1 } \| ^ { 2 } ~ | ~ \mathcal { F } _ { t } ] < \infty$ , we have

$$\sum_ {\tau = 0} ^ {t} \mu_ {\tau} ^ {- 2} d _ {\tau} ^ {2} \phi_ {\tau} ^ {\top} P _ {\tau + 1} \phi_ {\tau} \mathbb {E} [ \| w _ {\tau + 1} \| ^ {2} \mid \mathcal {F} _ {\tau} ] = O (1). \tag {F.13}$$

Moreover, the sequence $\{ \| w _ { t + 1 } \| ^ { 2 } - \mathbb { E } [ \| w _ { t + 1 } \| ^ { 2 } \mid \mathcal { F } _ { t } ] \} _ { t \geq 0 }$ is a martingale difference sequence with respect to $\{ \mathcal { F } _ { t } \}$ . Thus, by the martingale convergence theorem (Theorem 2.8 in Chen and Guo (1991)) and (F.12), we obtain

$$\sum_ {\tau = 0} ^ {t} \mu_ {\tau} ^ {- 2} d _ {\tau} ^ {2} \phi_ {\tau} ^ {\top} P _ {\tau + 1} \phi_ {\tau} (\| w _ {\tau + 1} \| ^ {2} - \mathbb {E} [ \| w _ {\tau + 1} \| ^ {2} \mid \mathcal {F} _ {\tau} ]) = O (1), \text { a.s. } \tag {F.14}$$

Substituting (F.10)-(F.14) into (F.9) yields Lemma 2. ✷
