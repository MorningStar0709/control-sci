# APPENDIX A PROOF OF THEOREM 1: INVISIBILITY OF REDISTRIBUTION TERM IN DYNAMIC ALLOCATION

To show that the redistribution term $g _ { \perp } ^ { * } S \zeta$ is invisible to the hybrid high-level controller and the system output, we need to verify that distributed torque components are bounded in the first place and the system states and outputs with input $\tau =$ ${ \bar { \tau } } + g _ { \perp } ^ { * } S \zeta$ are identical to those obtained under the nominal input τ¯.

From (11a), the control input norm satisfies

$$\| \boldsymbol {\tau} (t) \| \leq \| \bar {\boldsymbol {\tau}} (t) \| + \| \boldsymbol {g} _ {\perp} ^ {*} S \| \| \boldsymbol {\zeta} (t) \|,$$

under Assumption 1, $\| \zeta ( t ) \|$ is bounded by (13), therefore

$$\left\| \boldsymbol {\tau} (t) \right\| \leq \left\| \bar {\boldsymbol {\tau}} (t) \right\| + \left\| \boldsymbol {g} _ {\perp} ^ {*} S \right\| \left(\delta (\left\| \boldsymbol {\zeta} (0) \right\|, t) + \mu \left(\left\| \bar {\boldsymbol {\tau}} \right\| _ {[ 0, t ]}\right)\right).$$

by assumption the nominal control input $\bar { \tau } ( t )$ is uniformly bounded for all $t \geq 0$ , i.e., there exists a constant $\bar { \tau } _ { \operatorname* { m a x } } > 0$ such that $\| \bar { \tau } ( t ) \| \leq \bar { \tau } _ { \operatorname* { m a x } }$ , it follows that

$$\| \boldsymbol {\tau} (t) \| \leq \bar {\tau} ^ {\max} + \| \boldsymbol {g} _ {\perp} ^ {*} S \| \left(\delta (\| \boldsymbol {\zeta} (0) \|, t) + \mu (\bar {\tau} ^ {\max})\right).$$

which implies that $\tau ( t )$ is uniformly bounded for all $t \geq 0$ Therefore, all actuator torques remain bounded.

Finally, since $\mathbf { \pmb { g } } _ { \perp } ^ { * }$ lies in the null space of the augmented control matrix of the hybrid system, the redistribution term $g _ { \perp } ^ { * } S \zeta$ does not affect the system dynamics. Therefore, under the given assumption, the redistribution input $g _ { \perp } ^ { * } S \zeta$ is invisible to the system’s high-level controller and system output, which confirms (14).
