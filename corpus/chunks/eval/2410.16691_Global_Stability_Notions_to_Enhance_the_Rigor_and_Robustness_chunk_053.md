Again, an advantage of the adaptive controller (128) is its simplicity. The adaptive controller (128) consists again of two parts: the static part $u = - c y - \hat { \theta } y - q y ^ { 3 }$ and the dynamic part $\frac { d \hat { \theta } } { d t } = \Gamma y ^ { 2 } - \sigma \hat { \theta }$ Due to the σ-modification we cannot expect the dynamic part of the adaptive controller (128) to work as “an identifier” that produces a reliable parameter estimate ${ \hat { \theta } } ( t )$ of the true value of the parameter  . Instead, as noted above, the variable $\hat { \theta }$ works more or less as a dynamic controller gain which cannot increase too much.
