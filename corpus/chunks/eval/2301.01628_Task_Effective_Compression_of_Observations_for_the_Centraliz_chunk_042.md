Proof. We depart from the result of lemma 1 - problem (3). By taking the expectation over the empirical distribution of ${ \bf s } ( t )$ and applying Bellman optimality equation, we obtain

$$\underset {\pi} {\operatorname{argmin}} \quad \frac {1}{n} \sum_ {t = 1} ^ {n} \left| Q ^ {\pi^ {*}} \big (\mathbf {s} (t), \pi^ {*} (\mathbf {s} (t)) \big) - Q ^ {\pi^ {m}} \Big (\mathbf {c} (t), \pi^ {m} \big (\pi^ {c} (\mathbf {s} (t)) \big) \Big) \right|,\text { s.t. } \quad | \mathcal {C} | \leq 2 ^ {R}, \tag {15}$$

where the vector $\pi ^ { c } ( \mathbf { s } ( t ) )$ is of N dimensions and its i-th element is $\mathbf { c } _ { i } ( t )$ . We proceed by plugging $\pi ^ { c , A B S A - 1 } ( \cdot )$ and $\Pi ^ { m ^ { * } }$ , according to the definition 1, into the equation (15) to obtain

$$\frac {1}{n} \sum_ {t = 1} ^ {n} \left| Q ^ {\pi^ {*}} (\mathbf {s} (t), \pi^ {*} (\mathbf {s} (t))) - Q ^ {\pi^ {*}} (\mathbf {c} (t), \pi^ {*} (\mathbf {s} ^ {\prime})) \right|, \tag {16}$$

$$\frac {1}{n} \sum_ {t = 1} ^ {n} \left| Q ^ {\pi^ {*}} (\mathbf {s} (t), \pi^ {*} (\mathbf {s} (t))) - Q ^ {\pi^ {*}} (\mathbf {s} (t), \pi^ {*} (\mathbf {s} (t))) \right| = 0. \tag {17}$$
