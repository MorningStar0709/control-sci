# A.1 Neural Networks Architecture

In this paper, we employ feedforward neural networks with $N \geq 2$ hidden layers (each containing k neurons) to approximate control policies and value functions. We specify the detailed architecture in this appendix. The network architecture consists of one input layer, one output layer, and N hidden layers. The state vector s is encoded into the network through the input layer. More precisely, we can represent it as follows:

$$F (\boldsymbol {s}; \phi) = h _ {o} \circ \sigma_ {N} \circ h _ {N} \circ \dots \circ \sigma_ {1} \circ h _ {1} (\boldsymbol {s}), \tag {25}$$

where $h _ { 1 } : \mathbb { R } ^ { m }  \mathbb { R } ^ { k } , h _ { n } : \mathbb { R } ^ { k }  \mathbb { R } ^ { k } ( 2 \leq n \leq N )$ , and $h _ { o } : \mathbb { R } ^ { k }  \mathbb { R } ^ { d }$ are all affine functions. In particular, $h _ { 1 } ( { \pmb x } ) = { \pmb H } ^ { 1 } { \pmb x } + { \pmb b } ^ { 1 }$ , $h _ { n } ( { \pmb x } ) = H ^ { n } { \pmb x } + { \pmb b } ^ { n } , h _ { o } ( { \pmb x } ) = H ^ { o } { \pmb x } + { \pmb b } ^ { o }$ , where $\pmb { H } ^ { 1 } \in \mathbb { R } ^ { k \times m } ( \mathbf { a } k \times$ m matrix), $H ^ { n } \in \mathbb { R } ^ { k \times k }$ (a k × k matrix), $\pmb { H } ^ { o } \in \mathbb { R } ^ { d \times k }$ (a d × k matrix), and $\pmb { b } ^ { 1 } , \pmb { b } ^ { n } \in \mathbb { R } ^ { k }$ , for $n = 1 , \cdots , N , \pmb { b } ^ { o } \in \mathbb { R } ^ { d } . \pmb { \sigma } _ { n } : \mathbb { R } ^ { k }  \mathbb { R } ^ { k }$ is a component-wise activation function given by $\sigma _ { n } ( x _ { 1 } , \cdots , x _ { k } ) = ( \sigma ( x _ { 1 } ) , \cdots , \sigma ( x _ { k } ) )$ ) for each $n = 1 , \cdots , N .$ . The hyperparameter φ in the body text of the paper refers to the matrices $\{ H ^ { 1 } , \cdots , H ^ { N } , H ^ { o } \}$ and $\{ \pmb { b } ^ { 1 } , \cdots , \pmb { b } ^ { N } , \pmb { b } ^ { o } \}$ used in the construction of network (25).
