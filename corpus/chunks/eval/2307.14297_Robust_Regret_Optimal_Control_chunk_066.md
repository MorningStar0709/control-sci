$$
\begin{array}{l} \det (I _ {n _ {v}} - M _ {1 1} (z) \Delta (z)) \\ = \det \left(\left[ \begin{array}{c c} I _ {n _ {v}} & 0 \\ 0 & I _ {n _ {w}} \end{array} \right] - \left[ \begin{array}{c c} 0 & M _ {1 1} (z) \\ \Delta (z) & 0 \end{array} \right]\right) \\ = \det \left(\left[ \begin{array}{c c} I _ {n _ {v}} & - D _ {v w} \\ - D _ {\Delta} & I _ {n _ {w}} \end{array} \right] - \left[ \begin{array}{c c} C _ {v} & 0 \\ 0 & C _ {\Delta} \end{array} \right] \left[ \begin{array}{c c} z I - A & 0 \\ 0 & z I - A _ {\Delta} \end{array} \right] ^ {- 1} \left[ \begin{array}{c c} 0 & B _ {w} \\ B _ {\Delta} & 0 \end{array} \right]\right). \\ \end{array}
$$

Apply the block-matrix determinant formula again to the last line to obtain:

$$\det (I _ {n _ {v}} - M _ {1 1} (z) \Delta (z)) = \det (z I - \tilde {A}). \tag {54}$$

This equality holds when: (i) $F _ { U } ( M , \Delta )$ is well-posed and (ii) z is not a pole of $M _ { 1 1 }$ or $\Delta$ . These conditions ensure that the various determinants are well-defined. We can now complete the proof.

$A \to B )$ : Assume A) holds and $\Delta$ is any stable system with $\| \Delta \| _ { \infty } \leq 1$ . We will show that B) holds: $F _ { U } ( M , \Delta )$ is well-posed and stable.

Define the complement of the open unit disk as $\mathcal { D } ^ { c } : = \{ z : | z | \geq 1 \}$ . Both $M _ { 1 1 }$ and $\Delta$ are stable systems so the maximum modulus principle holds [43]:

$$\sup _ {z \in \mathcal {D} ^ {c}} \bar {\sigma} \left(M _ {1 1} (z) \Delta (z)\right) = \sup _ {\theta \in [ 0, 2 \pi ]} \bar {\sigma} \left(M _ {1 1} (e ^ {j \theta}) \Delta (e ^ {j \theta})\right).$$

The right-side is equal to $\| M _ { 1 1 } \Delta \| _ { \infty }$ . This can be bounded using the small gain assumptions:

$$\left\| M _ {1 1} \Delta \right\| _ {\infty} \leq \left\| M _ {1 1} \right\| _ {\infty} \cdot \left\| \Delta \right\| _ {\infty} < 1. \tag {55}$$

Hence $\bar { \sigma } \left( M _ { 1 1 } ( z ) \Delta ( z ) \right) < 1$ for all $z \in \mathcal { D } ^ { c }$ . This further implies that that $\| M _ { 1 1 } ( z ) \Delta ( z ) v \| _ { 2 } <$ < $\lVert \boldsymbol { v } \rVert _ { 2 }$ for any nonzero vector v. Hence $( I - M _ { 1 1 } ( z ) \Delta ( z ) ) v \ \ne \ 0$ for any $v \ne 0$ . Therefore det $( I _ { n _ { v } } - M _ { 1 1 } ( z ) \Delta ( z ) ) \neq 0$ for all $z \in \mathcal { D } ^ { c }$ .
