The assertion about the A matrix in the assumption is equivalent to the boundedness of the maximum eigenvalue of the positive semidefinite matrix $E [ A ( \xi _ { 0 } ) ^ { T } A ( \xi _ { 0 } ) ]$ (recall the definition of maximum singular values). Hence, we prove each entry of this matrix is bounded.

$$
