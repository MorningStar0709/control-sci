The Fritz-John condition in Proposition C.22 is known as the Karush-Kuhn-Tucker (KKT) condition if $\mu _ { 0 } > 0 ;$ if this is the case, $\mu _ { 0 }$ may be normalized to $\mu _ { 0 } ~ = ~ 1$ . A constraint qualification is required for the Karush-Kuhn-Tucker condition to be a necessary condition of optimality for the optimization problem considered in this section. A simple constraint qualification is linear independence of $\{ \nabla g _ { i } ( u ) , ~ i ~ \in$ $\begin{array} { r } { \begin{array} { r } { \mathcal { I } ^ { 0 } ( u ) , \nabla h _ { j } ( u ) , j \in \mathcal { F } \} } \end{array} } \end{array}$ at a local minimizer u. For, if u is a local minimizer and $\mu _ { 0 } = 0$ , then the Fritz-John condition implies that $\begin{array} { r } { \sum _ { i \in \mathcal { I } ^ { 0 } ( u ) } \mu _ { i } \nabla g _ { i } ( u ) + \sum _ { j \in \mathcal { F } } \lambda _ { j } \nabla h _ { j } ( u ) = 0 } \end{array}$ which contradicts the linear independence of $\{ \nabla g _ { i } ( u ) , i \in \mathcal { I } ^ { 0 } ( u ) , \nabla h _ { j } ( u ) , j \in \mathcal { F } \}$ since not all the multipliers are zero. Another constraint qualification, used in Propositions C.20 and C.21 for an optimization problem in which the constraint set is $U : = \{ u \mid g _ { i } ( u ) \le 0 , i \in \mathcal { I } \}$ , is the existence of a vector $\bar { h } ( u ) \in \mathcal { F } _ { U } ( u )$ such that $\langle \nabla g _ { i } ( u ) , \bar { h } \rangle < 0$ for all $i \in \mathcal { I } ^ { 0 } ( u )$ ; this condition ensures $\mu _ { 0 } = 1$ in C.25. Many other constraint qualifications exist; see, for example, Bertsekas (1999), Chapter 3.
