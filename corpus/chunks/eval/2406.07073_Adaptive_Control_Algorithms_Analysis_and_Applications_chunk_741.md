If there exists a real scalar function $V ( x , t )$ such that:

1. $V ( 0 , t ) = 0 \ - \infty < t < \infty .$   
2. $V ( x , t ) \geq \alpha ( \| x \| ) > 0 , \forall x \neq 0 , x \in R _ { n }$ and ∀t where α(·) is a continuous nondecreasing scalar function such that $\alpha ( 0 ) = 0 \left( V ( x , t )\right)$ is positive definite).   
3. V (x, t) → ∞ with x → ∞ (V (x, t) is radially unbounded).   
4. $\begin{array} { r } { A V ( x , t ) = V ( x , t + 1 ) - V ( x , t ) \leq - \mu ( \| x \| ) < 0 , \ \forall x \neq 0 , \ x \in R _ { n } } \end{array}$ , ∀t along the trajectories of the system (B.1) where $\mu ( \cdot )$ is a continuous non-decreasing scalar function such that $\mu ( 0 ) = 0 . \left( \varDelta V ( x , t )\right.$ is negative definite.) Then the equilibrium state $x _ { e } = 0$ is globally asymptotically stable and $V ( x , t )$ is a Lyapunov function for the system (B.1).   
If, in addition:

5. $V ( x , t ) \leq \beta ( \| x \| )$ where β(·) is a continuous, non-decreasing scalar function such that $\beta ( 0 ) = 0$ , then $x _ { e } = 0$ is uniformly globally asymptotically stable.

Corollary B.1 If in Theorem B.1, $V ( x , t )$ is replaced by $V ( x )$ , the equilibrium state $x _ { e } = 0$ of the autonomous system:

$$x (t + 1) = f [ x (t) ] \tag {B.3}$$

is globally uniformly asymptotically stable if:

1. $V ( 0 ) = 0 ;$   
2. $V ( x ) > 0 , \forall x \neq 0 , x \in R _ { n }$ ;  
3. V (x) → ∞ with x → ∞;   
4. $\Delta V [ x ( t ) ] = V [ x ( t + 1 ) ] - V [ x ( t ) ] < 0$ along the trajectories of (B.3) for all $x \neq 0 , x \in R _ { n }$ .

Corollary B.2 In Corollary B.1, condition 4 can be replaced by:

1. $\Delta V ( x ) \leq 0 , \forall x \neq 0 , x \in R _ { n }$ ;
