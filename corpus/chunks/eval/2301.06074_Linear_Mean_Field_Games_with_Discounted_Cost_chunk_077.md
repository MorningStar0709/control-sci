Now, let us derive the joint KKT conditions for player 1 and player 2, whose solution gives a Nash equilibrium for GNEP. To this end, we need to define several functions. First we define

$$L _ {1} (\zeta , \mu , \lambda) := \langle \zeta , c _ {\mu} \rangle + \langle h _ {1} (\zeta , \mu), \lambda \rangleL _ {2} (\zeta , \mu , \gamma) := g (\zeta , \mu) + \langle h _ {2} (\zeta , \mu), \gamma \rangle$$

where λ and $\gamma$ are Lagrange multipliers of player 1 and player 2, respectively. Let $\lambda =$ $( \lambda _ { 1 } , \lambda _ { 2 } )$ , where $\lambda _ { 1 } \in \mathbb { R } ^ { | \breve { \mathsf { X } } \times \mathsf { A } | }$ and $\boldsymbol { \lambda } _ { 2 } \in \mathbb { R } ^ { | \times | }$ , and let $\gamma = ( \gamma _ { 1 } , \gamma _ { 2 } , \gamma _ { 3 } )$ , where $\gamma _ { 1 } \in \mathbb { R } ^ { | \mathsf { X } | } , \gamma _ { 2 } \in \mathbb { R }$ , and $\gamma _ { 3 } \in \mathbb { R } ^ { | \times | }$ . Note that for any $( x , a ) \in { \mathsf { X } } \times { \mathsf { A } }$ , we have

$$\partial_ {\zeta (x, a)} L _ {1} (\zeta , \mu , \lambda) = c _ {\mu} (x, a) - \lambda_ {1} (x, a) - \lambda_ {2} (x) + \beta \sum_ {y \in \mathsf {X}} \lambda_ {2} (y) p _ {\mu} (y | x, a)$$

and similarly, for any $x \in \mathsf X$ , we have

$$\partial_ {\mu (x)} L _ {2} (\zeta , \mu , \lambda) = \partial_ {\mu (x)} g (\zeta , \mu) - \gamma_ {1} (x) - \gamma_ {2} - \gamma_ {3} (x) + \sum_ {y \in \mathsf {X}} \gamma_ {3} (y) p _ {\zeta} (y | x)$$
