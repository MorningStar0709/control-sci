# Example A.4 Time-delay process

Many industrial processes can be approximated by first-order dynamics and a time delay. One example is a paper machine (see Fig. A.4). The input is the thick stock flow, that is, the amount of pulp. The output is the basic weight, that is, the thickness of the paper. The equations describing the system can be normalized to the transfer function

$$G (s) = \frac {1}{s + 1} e ^ {- s \Gamma} \tag {A.10}$$

Another physical process that can be described by (A.10) is a mixing system with long pipes. Example 2.8 gives the zero-order-hold sampling of (A.10).
