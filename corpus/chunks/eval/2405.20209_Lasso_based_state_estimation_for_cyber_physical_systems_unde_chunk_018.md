# 5.2 Sparse soft observer

In this section, we test the proposed sparse soft observer for recursive and online SSR. We perform 100 random runs and we show the average results, in terms of time evolution of the state estimation error $\| \hat { x } - \widetilde { x } \| _ { 2 } / \| \widetilde { x } \| _ { 2 }$ and support error, defined as $\begin{array} { r } { \sum _ { j } \vert \mathbf { 1 } ( \hat { \mathbf { a } } _ { j } \neq 0 ) - \mathbf { 1 } ( \widetilde { \boldsymbol { a } } _ { j } \neq 0 ) \vert } \end{array}$ |, where $\mathbf { 1 } ( v ) = 1$ if v is true and 0 otherwise.
