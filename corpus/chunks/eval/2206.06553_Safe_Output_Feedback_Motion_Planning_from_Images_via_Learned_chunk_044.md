In this appendix, we present the proofs of the theoretical results in the main body of the paper; for convenience, we have copied the theorem statements here.

Lemma C.1 $( \dot { d } _ { c } ( t ) )$ . The integral term in (8) can be bounded as

$$\int_ {0} ^ {1} \| R _ {c} (\gamma_ {c} ^ {t} (s)) w _ {c} (t) \| d s \leq \sqrt {\bar {\lambda} _ {D _ {c}} (M _ {c})} \bar {w} _ {x} + L _ {\Delta k} d _ {e}. \tag {C.10}$$

Proof. Recall the form of $w _ { c }$ from (14). First consider the third term in (14) (the $\mathrm { d y } .$ namics error). By using $\lVert w _ { x } \rVert \leq \bar { w } _ { x }$ and $\| R _ { c } ( \cdot ) \| \le \sqrt { \bar { \lambda } _ { D _ { c } } ( M _ { c } ) }$ , we can pull $w _ { x }$ and $R _ { c } ( \cdot )$ out of the integral to form the first term in the lemma statement. The second term follows by performing a max over geodesic parameters $s \in [ 0 , 1 ]$ and substituting the appropriate arguments into (15). Specifically, we have the following chain of relations:

$$
\begin{array}{l} \int_ {0} ^ {1} \left\| R _ {c} (\gamma_ {c} ^ {t} (s)) B (u (\hat {x}, x ^ {*}, u ^ {*}) - u _ {\text { closest }}) \right\| d s \\ \leq \int_ {0} ^ {1} \max _ {s \in [ 0, 1 ]} \| R _ {c} (\gamma_ {c} ^ {t} (s)) B (u (\hat {x}, x ^ {*}, u ^ {*}) - u _ {\mathrm{closest}}) \| d s \\ = \int_ {0} ^ {1} \Delta k (\hat {x}, x, x ^ {*}, u ^ {*}) d s = \Delta k (\hat {x}, x, x ^ {*}, u ^ {*}) \\ = | \varDelta k (\hat {x}, x, x ^ {*}, u ^ {*}) - \varDelta k (x, x, x ^ {*}, u ^ {*}) | \leq L _ {\varDelta k} d _ {e} (\hat {x}, x) = L _ {\varDelta k} d _ {e} \\ \end{array}
$$

The first inequality holds via upper-bounding the norm over the geodesic parameters, the first equality holds by substituting the definition of ∆k, the second equality holds as $\varDelta k$ is not a function of $s ,$ the third equality holds since $\varDelta _ { k } ( x , x , x ^ { * } , u ^ { * } ) = 0$ , and the final inequality holds from the definition of our Lipschitz constant.
