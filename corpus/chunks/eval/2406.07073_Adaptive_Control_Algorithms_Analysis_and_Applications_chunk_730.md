# Strategy 2:

1. sample the output;   
2. send the control signal (computed in the previous sampling period);   
3. update the controller parameters;   
4. compute the control signal;   
5. wait for the next sample.

Using the Strategy 2, a delay of one sample is introduced in the system and this has to be taken into account for controller design.
