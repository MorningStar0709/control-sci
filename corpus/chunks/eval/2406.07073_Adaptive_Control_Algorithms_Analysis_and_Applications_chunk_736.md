# Appendix A Stochastic Processes

Probability Space The outcome of a phenomenon will be denoted by $\xi . \xi$ belongs to a set $\varOmega$ which contains all the possible outcomes of a phenomenon. Therefore, $\varOmega$ is the sure event. One denotes the impossible event by $\phi$ . The set of events is denoted by F. A probability measure P is associated to Ω, i.e., for subsets of $\varOmega$ probability of occurrence is assigned. The triplet $( \varOmega , \mathcal { F } , P )$ is called a probability space.

A σ -algebra $\mathcal { F }$ is a set of subsets of $\varOmega$ which contains the empty set $\phi$ and is closed under complements and countable unions.

A random variable generates a σ -algebra $\mathcal { F }$ since all possible outcomes define subsets in the event space $\varOmega$ .

Axioms of Probability Consider two events a and b belonging to $\varOmega$ . To each event one assigns a probability $P ( \cdot )$ satisfying the following axioms:

1. $P ( a ) \geq 0 ; \forall a \in \varOmega$   
2. $P ( \varOmega ) = 1$   
3. $P ( a \cup b ) = P ( a ) + P ( b ) { \mathrm { ~ i f ~ } } a \cap b = \emptyset$

Random Variable A random variable X is a function from the event space $\varOmega$ to $R ^ { 1 }$ and will be denoted by $X ( \xi )$ with $\xi \in \varOmega$ . (The argument will be omitted in general.)

Induced Probability Since to each subset (event) of $\varOmega$ , a probability has been assigned and since x is a function of $\xi$ , there is a probability that X takes a value in a subset of $R ^ { 1 }$ , i.e.:

$$P \{X (\xi) \leq x _ {1} \} \quad \text { or } \quad P \{x _ {1} \leq X (\xi) \leq x _ {2} \}$$

Distribution Function

$$F _ {X} (x) = P \{X \leq x \}; - \infty < x < \infty$$

is called the distribution function.

Probability Density Function

$$F _ {X} (x) = P \{X \leq x \} = \int_ {- \infty} ^ {x} f _ {X} (x) d x$$

where $f _ { X } ( x )$ is the probability density function (i.e., $F _ { X } ( x )$ is continuous and the derivatives with respect to x exist).

Gaussian (Normal) Distribution

$$f _ {X} (x) = \frac {1}{\sigma \sqrt {2 \pi}} e ^ {- x ^ {2} / 2 \sigma^ {2}}$$

Expectation (Mean Value) of a Random Variable

$$m (X) = \mathbf {E} \{X (\xi) \} = \int_ {- \infty} ^ {\infty} f _ {X} (x) d x$$

Covariance of a Random Variable

$$\operatorname{cov} X = \mathbf {E} [ X - m (x) ] ^ {2} = \mathbf {E} \{[ X (\xi) - m (x) ] ^ {2} \}$$

Independent Random Variables Two random variables X and Y are characterized by the joint distribution function

$$F _ {X Y} (x y) = P \{X \leq x; Y \leq y \}$$
