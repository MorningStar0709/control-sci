# 4. Design of Parameter Update Law for Stable Adaptation

The task is to design the update law that specifies $\dot { \hat { \theta } } \left( t \right)$ to achieve stable convergence of both errors ${ \bf e } _ { I } ^ { l } \left( t \right)$ and $\tilde { \theta } \left( t \right)$ to zero. Continuous-time recursive algorithms based on gradient flow and its variants can be applied for minimisation of the loss function using the information of local loss geometry evaluated at $\hat { \theta } \left( t \right)$ , such as $L , \nabla _ { \boldsymbol { \hat { \theta } } } L ,$ , and $\nabla _ { \boldsymbol { \hat { \theta } } } ^ { 2 } L$ .

Let $\begin{array} { r } { V _ { e } \left( \mathbf { e } _ { I } ^ { l } \left( t \right) \right) = \frac { 1 } { 2 } \mathbf { e } _ { I } ^ { l } ^ { T } \left( t \right) P \mathbf { e } _ { I } ^ { l } \left( t \right) } \end{array}$ be the Lyapunov function certifying global exponential stability of $\Sigma _ { e } ^ { C L \bullet } \mathrm { s }$ origin under the stabilising coupling input $U _ { c } \left( \mathbf { e } _ { I } ^ { l } \left( t \right) \right) = - K _ { e } \mathbf { e } _ { I } ^ { l } \left( t \right)$ in the absence of uncertainty where $P > 0$ is the unique solution satisfying $\begin{array} { r } { \left( A _ { e } - B _ { e } K _ { e } \right) ^ { T } P + P \left( A _ { e } - B _ { e } K _ { e } \right) + Q = 0 } \end{array}$ for any given $Q > 0 .$ . $$
