# 4.1.1 RNN-based D-NMPC: Numerical example

This section illustrates RNN-based D-NMPC on a continuous stirred tank reactor (CSTR) system example. To generate the input-output training data, a second-order state-space model of CSTR is considered [24]:

$$
\mathbf {x} _ {k + 1} = \mathbf {x} _ {k} + T \left[ \begin{array}{c} - x _ {1 _ {k}} + D _ {a} (1 - x _ {1 _ {k}}) e ^ {x _ {2 _ {k}}} \\ x _ {2 _ {k}} + B D _ {a} (1 - x _ {1 _ {k}}) e ^ {x _ {2 _ {k}}} - D _ {b} (x _ {2 _ {k}} - u _ {k}) \end{array} \right] \tag {74}
y _ {k} = x _ {2 _ {k}} + v _ {k}$$

where $x _ { 1 _ { k } }$ is the reactant concentration, $x _ { 2 _ { k } }$ is the reactant temperature, $u _ { k }$ is the coolant temperature, $y _ { k }$ is the measured reactant temperature with $v _ { k }$ as the measurement noise, and T is the sampling period. The dimensionless constants of the system are chosen as $B = 2 2 . 0 , D _ { a } = 0 . 0 8 2 , D _ { b } = 3 . 0$ . The input-output data is generated by a forward simulation of Eq. (74) over $\mathrm { N _ { T } } ~ = ~ 1 0 0 0$ instants with the sampling period $T \stackrel { = } { = } 1$ sec., initial condition $\mathbf { x } _ { 0 } = \left[ 0 \quad 0 \right] ^ { \top }$ , while the control input u is chosen as a PRMS signal. The first 500 samples of the dataset are used for training the RNN and the remaining samples for validating the model. The RNN with $\mathrm { L } = 2 , n _ { 1 } = 5$ and tanh activation function in the hidden layer is trained using the training data, where the loss function is chosen as in Eq. $( 7 1 )$ . The trained RNN model is then used for implementing D-NMPC as in Algorithm 5 with $\Nu = 1 0 , \mathbf Q _ { \mathbf y } = 2$ , and $\mathbf { R } _ { \mathbf { u } } = 3$ . The response with RNN-based NMPC is shown in Fig. 7. The training input and output data, along with the output predicted with the trained RNN, are plotted in Fig. 7(a). The output tracking with RNN-based NMPC is given in Fig. 7(b), which shows that the output follows the desired reference. Fig. 7(c) shows the conprol input.
