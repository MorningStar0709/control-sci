# V. NUMERICAL EXAMPLE

We applied the proposed VEMPC protocol to an inverted pendulum system [20]. By linearizing and discretizing the system with the sampling period of 50 ms, the model of the form (1) is obtained as

$$
A = \left[ \begin{array}{c c} 1. 0 2 4 6 & 0. 0 5 0 4 \\ 0. 9 8 9 0 & 1. 0 2 4 6 \end{array} \right], \quad B = \left[ \begin{array}{c} 0. 0 2 5 1 \\ 1. 0 0 8 2 \end{array} \right],
$$

where the parameters are set as $m = 0 . 2 \mathrm { k g }$ (mass), $l { \bf \Psi } =$ 0.5 m (length), and $g = 9 . 8 1 \mathrm { m } / \mathrm { s } ^ { 2 }$ (gravitational acceleration). The state $\boldsymbol { x } ( t ) = : [ \boldsymbol { \theta } ( t ) ; \dot { \boldsymbol { \theta } } ( t ) ]$ consists of the angular position $\theta ( t )$ and angular velocity $\theta ( t )$ of the pendulum, and the input u(t) represents the applied torque force. The initial state was set to $x ( 0 ) = [ 0 . 3 ; 0 . 1 ]$ .

For the MPC formulation, we set the prediction horizon as $N = 1 0$ and the cost matrices as

$$
Q = \left[ \begin{array}{c c} 5 0, & 0 \\ 0, & 5 \end{array} \right], \quad Q _ {f} = 2 Q, \quad R = 0. 1.
$$

The state and input constraints were defined as $\| \theta ( t ) \| \leq 0 . 5 .$ , $\lVert \dot { { \boldsymbol { \theta } } } ( t ) \rVert \leq 0 . 8$ , and $\| u ( t ) \| \leq 1$ , resulting in a total of $p = 6 0$ constraints. For reference distribution and sampling, we set $\Sigma _ { 0 } = 0 . 2 5 ^ { 2 } I , \lambda = 0 . 1$ , and $K = 2 4 0$ .
