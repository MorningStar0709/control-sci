$$
\begin{array}{l} \dot {\mathbf {V}} = \frac {1}{4 \lambda} (\mathbf {s} ^ {T} \dot {\mathbf {M}} \mathbf {s} + \dot {\mathbf {s}} ^ {T} \mathbf {M} \mathbf {s} + \mathbf {s} ^ {T} \mathbf {M} \dot {\mathbf {s}}) \\ + \frac {1}{2} \dot {\mathbf {Q}} ^ {T} \Gamma^ {- 1} \mathbf {Q} + \frac {1}{2} \mathbf {Q} ^ {T} \Gamma^ {- 1} \dot {\mathbf {Q}} \\ = \frac {1}{2 \lambda} \mathbf {s} ^ {T} (\mathbf {M} \dot {\mathbf {s}} + \mathbf {C s}) + \frac {1}{2} \dot {\mathbf {Q}} ^ {T} \Gamma^ {- 1} \mathbf {Q} + \frac {1}{2} \mathbf {Q} ^ {T} \Gamma^ {- 1} \dot {\mathbf {Q}} \\ = \mathbf {s} ^ {T} \left(\mathbf {M} \dot {\mathbf {v}} _ {\mathbf {r}} + \mathbf {C v} _ {\mathbf {r}} + \mathbf {D v} + g - \tau\right) + \dot {\mathbf {Q}} ^ {T} \Gamma^ {- 1} \mathbf {Q}. \tag {34} \\ \end{array}
$$

By substituting Eq. (19),

$$
\begin{array}{l} \dot {\mathbf {V}} = \mathbf {s} ^ {T} \left(\mathbf {M} \dot {\mathbf {v}} _ {\mathbf {r}} + \mathbf {C v} _ {\mathbf {r}} + \mathbf {D v} + g - \tau\right) \\ + (\dot {\tilde {\tau}} _ {r} - \dot {\tilde {\tau}} _ {e s t}) ^ {T} \Gamma^ {- 1} \mathbf {Q} \\ = - \mathbf {s} ^ {T} (\mathbf {K} _ {1} \mathbf {s} + \mathbf {K} _ {2} | \mathbf {s} | ^ {r} s i g n (\mathbf {s})) + (\dot {\tilde {\tau}} _ {r}) ^ {T} \Gamma^ {- 1} \mathbf {Q}. \tag {35} \\ \end{array}
$$

The dynamic item $\widetilde { \tau } _ { r }$ is bounded due to the slow velocity eof the underwater vehicle and $\mathbf { s } ^ { T } ( \mathbf { K } _ { 1 } \mathbf { s } + \mathbf { K } _ { 2 } | \mathbf { s } | ^ { r } \mathrm { s i g n } ( \mathbf { s } ) ) \geq$ $( \dot { \widetilde { \tau } } _ { \mathrm { r } } ) ^ { \mathrm { T } } \Gamma ^ { - 1 } \mathbf { Q }$ . When ${ \bf K } _ { 1 } , ~ { \bf K } _ { 2 }$ and Γ are assigned with large enough values at the design step, $\dot { \mathbf { V } } \leq 0$ can be achieved and V is ensured to be bounded, thus leading to the conclusion that Q is bounded. Then design a new Lyapunov function as

$$\mathbf {V} _ {2} = \frac {1}{4 \lambda} \mathbf {s} ^ {T} \mathbf {M} \mathbf {s}, \tag {36}$$

whose derivative can be deducted as,

$$\dot {\mathbf {V}} _ {2} = \mathbf {s} ^ {T} (\mathbf {Q} - \mathbf {K} _ {1} \mathbf {s} - \mathbf {K} _ {2} | s | ^ {r} \text { sign } (s)), \tag {37}$$

where $0 < r < 1$ . Suppose $| | Q | | < a$

$$\dot {\mathbf {V}} _ {2} \leq \frac {1}{2} | | s | | ^ {2} + \frac {1}{2} a - \lambda_ {\min} (\mathbf {K} _ {1}) | | s | | ^ {2} - \lambda_ {\min} (\mathbf {K} _ {2}) | | s | | ^ {1 + r}, \tag {38}$$

choose $\mathbf { K } _ { 1 }$ when $\begin{array} { r } { \lambda _ { m i n } ( \mathbf { K } _ { 1 } ) > \frac { 1 } { 2 } + \beta , } \end{array}$ , where $\beta > 0$ ,

$$\dot {\mathbf {V}} _ {2} \leq - \beta | | s | | ^ {2} - \lambda_ {\min} (\mathbf {K} _ {2}) | | s | | ^ {1 + r} + \frac {1}{2} a, \tag {39}$$

which induces that the Lyapunov function converges to a range close to zero in a finite time and s converges to a range close to zero in a finite time. Therefore, the supposed condition of the Lyapunov theorem can be regarded as satisfied, thus proving the stability of the designed SMC.
