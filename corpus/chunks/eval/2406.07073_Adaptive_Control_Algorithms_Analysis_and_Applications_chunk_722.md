• reinitialization of the adaptation gain (adding a positive definite matrix Q at certain instants).

Taking $\lambda _ { 1 } ( t ) = \lambda _ { 2 } ( t )$ the trace of the adaptation gain is kept constant (or at a desired value) by selecting $\lambda _ { 1 } ( t )$ as follows:

$$\lambda_ {1} (t) = 1 - \frac {\operatorname{tr} [ F (t) \phi_ {f} (t) \phi_ {f} ^ {T} (t) F (t) ]}{\operatorname{tr} F (t) [ m ^ {2} (t) + \phi_ {f} ^ {T} (t) F (t) \phi_ {f} (t) ]} \tag {16.28}$$

To start the adaptation algorithm, one generally uses the variable forgetting factor (or decreasing adaptation gain) and one switches to the constant trace algorithm when tr $F ( t ) \leq$ desired trace (see also Sect. 3.2.3). One should also assure that (regularization):

$$f _ {\min} I _ {n _ {p}} \leq F (t) \leq f _ {\max} I _ {n _ {p}}, \quad 0 < f _ {\min} \leq f _ {\max} < \infty \tag {16.29}$$

where $n _ { p }$ is the number of parameters to be adapted.

The scheduling variable $\alpha _ { 2 } ( t )$ is essentially used for the reinitialization of the adaptation gain. When parameter changes occur or are detected $( \alpha _ { 2 } ( t ) = 1 )$ .
