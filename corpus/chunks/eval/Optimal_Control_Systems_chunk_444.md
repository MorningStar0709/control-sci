# Norm of a Vector

The norm of a vector x, written as $||x||$ , is a measure of the size or length of the vector. Further,

1. $||\mathbf{x}|| > 0$ for all $\mathbf{x}$ and $||\mathbf{x}|| = 0$ only if $\mathbf{x} = 0$ .

2. $||\alpha \mathbf{x}|| = a||\mathbf{x}||$ for any scalar $\alpha$ and for all $\mathbf{x}$ .

3. $||x+y|| \leq ||x|| + ||y||$ for all x and y, called the Schwartz inequality.

The norm is calculated by either of the following ways

1.

$$\left| \left| \mathbf {x} \right| \right| ^ {2} = < \mathbf {x}, \mathbf {x} > = \mathbf {x} ^ {\prime} \mathbf {x}, \text { or }\left| \left| \mathbf {x} \right| \right| = \left[ \sum_ {i = 1} ^ {n} x _ {i} ^ {2} \right] ^ {1 / 2}, \text { called the Euclidean norm } (A. 1. 4)$$

2.

$$\left| \left| \mathbf {x} \right| \right| = \max \left| x _ {i} \right| \tag {A.1.5}$$

3.

$$\left| \left| \mathbf {x} \right| \right| = \sum_ {i = 1} ^ {n} \left| x _ {i} \right| \tag {A.1.6}$$
