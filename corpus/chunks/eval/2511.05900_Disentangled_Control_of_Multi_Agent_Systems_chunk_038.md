By summing (36) over all $i \in \mathcal N .$ we have $\begin{array} { r } { \sum _ { i \in \mathcal { N } } \nabla _ { x _ { i } } J _ { i } ( x , t ) \dot { x } _ { i } \le - \frac { \partial } { \partial t } J ( x , t ) - \alpha ( J ( x , t ) ) } \end{array}$ , which does not imply

However, (37) is essential to ensure asymptotic convergence of $\boldsymbol { J } ( \boldsymbol { x } , t )$ to 0 per Lemma III.1.

A further issue is that, even in the special case where $J _ { i } ( x , t )$ could be written as an explicit function of the distance between two agents, i.e.,

$$J _ {i} (x, t) = \sum_ {j \in \mathcal {N} _ {i}} J _ {i j} (\| x _ {i} - x _ {j} \|, t), \quad \forall i \in \mathcal {N}, \tag {38}$$

although the form (38) is in fact not admissible for coverage control, the condition (37) is still not guaranteed by Proposition 4 in [22].

Remark A.2. A special case in which the method proposed in [22] is theoretically guaranteed to converge occurs when

$$\nabla_ {x _ {i}} J (x, t) = \nabla_ {x _ {i}} J _ {i} (x, t), \quad \forall i \in \mathcal {N}. \tag {39}$$

However, the condition (39) is rather restrictive, as it does not hold even in basic multi-agent tasks such as rendezvous.
