# APPENDIX C DATASET AND OUTPUT FILES

The simulation output is automatically logged into three files:

• simulation\_log.csv: contains all time-series values of $z , \ v _ { r } , \ v _ { e } , \ u , \ a _ { l } , \ V ( x )$ , and V˙ (x) for each time step;   
• simulation\_run.mp4: records the visual animation of the Pygame simulation for presentation or analysis;   
• simulation\_plots.png: summarizes the evolution of the main control and Lyapunov variables.

These artifacts provide both numerical and visual evidence supporting the stability analysis of the proposed LQRL approach.
