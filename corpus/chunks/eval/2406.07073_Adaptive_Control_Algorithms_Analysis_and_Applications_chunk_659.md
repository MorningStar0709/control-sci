• Input sensitivity function (the transfer function between the disturbance $p ( t )$ and the input of the system $u ( t ) )$ :

$$S _ {u p} (z ^ {- 1}) = - \frac {A (z ^ {- 1}) R (z ^ {- 1})}{P (z ^ {- 1})} \tag {14.7}$$

Here

$$
\begin{array}{l} P (z ^ {- 1}) = A (z ^ {- 1}) S (z ^ {- 1}) + z ^ {- d} B (z ^ {- 1}) R (z ^ {- 1}) \\ = A (z ^ {- 1}) S ^ {\prime} (z ^ {- 1}) H _ {S} (z ^ {- 1}) + z ^ {- d} B (z ^ {- 1}) R ^ {\prime} (z ^ {- 1}) H _ {R} (z ^ {- 1}) \tag {14.8} \\ \end{array}
$$

defines the poles of the closed loop (roots of $P ( \boldsymbol { z } ^ { - 1 } ) )$ . In pole placement design, $P ( z ^ { - 1 } )$ is the polynomial specifying the desired closed-loop poles and the controller polynomials $R ( z ^ { - 1 } )$ and $S ( z ^ { - 1 } )$ are minimal degree solutions of (14.8) where the degrees of P , R and S are given by: $n _ { P } \le n _ { A } + n _ { B } + d - 1 , n _ { S } = n _ { B } + d - 1$ and $n _ { R } = n _ { A } - 1$ . Using (14.2) and (14.3), one can write the output of the system as:

$$y (t) = \frac {A (q ^ {- 1}) S (q ^ {- 1})}{P (q ^ {- 1})} p (t) = S _ {y p} (q ^ {- 1}) p (t) \tag {14.9}$$

For more details on RS-type controllers and sensitivity functions see Landau and Zito (2005). Suppose that $p ( t )$ is a deterministic disturbance, so it can be written as

$$p (t) = \frac {N _ {p} (q ^ {- 1})}{D _ {p} (q ^ {- 1})} \delta (t) \tag {14.10}$$

where $\delta ( t )$ is a Dirac impulse and $N _ { p } ( z ^ { - 1 } ) , D _ { p } ( z ^ { - 1 } )$ ) are coprime polynomials in $z ^ { - 1 }$ , of degrees $n _ { N _ { p } }$ and $n _ { D _ { p } }$ , respectively. In the case of stationary disturbances the roots of $D _ { p } ( z ^ { - 1 } )$ are on the unit circle. The energy of the disturbance is essentially represented by $D _ { p }$ . The contribution of the terms of $N _ { p }$ is weak compared to the effect of $D _ { p }$ , so one can neglect the effect of $N _ { p }$ .

Internal Model Principle The effect of the disturbance given in (14.10) upon the output:

$$y (t) = \frac {A (q ^ {- 1}) S (q ^ {- 1})}{P (q ^ {- 1})} \frac {N _ {p} (q ^ {- 1})}{D _ {p} (q ^ {- 1})} \delta (t) \tag {14.11}$$

where $D _ { p } ( z ^ { - 1 } )$ is a polynomial with roots on the unit circle and $P ( z ^ { - 1 } )$ is an asymptotically stable polynomial, converges asymptotically towards zero if and only $i \dot { f } ^ { 3 }$ the polynomial $S ( z ^ { - 1 } )$ in the RS controller has the form:

$$S (z ^ {- 1}) = D _ {p} (z ^ {- 1}) S ^ {\prime} (z ^ {- 1}) \tag {14.12}$$

In other terms, the pre-specified part of $S ( z ^ { - 1 } )$ should be chosen as $H _ { S } ( z ^ { - 1 } ) =$ $D _ { p } ( z ^ { - 1 } )$ ) and the controller is computed using (14.8), where $P , D _ { p } , A , B , H _ { R }$ and d are given.4
