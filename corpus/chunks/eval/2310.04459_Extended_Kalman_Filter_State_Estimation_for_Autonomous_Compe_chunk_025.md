# 6.3. Position Accuracy Comparison

Position correction shown in Figure 10, which is a result of data fusion, is both a benefit and drawback to the EKF. It is a benefit because the robot can more accurately localize. It is a drawback because the estimated position will suddenly shift, which can lead to jerky path-following movements.
