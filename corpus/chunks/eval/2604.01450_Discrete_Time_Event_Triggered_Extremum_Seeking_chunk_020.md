$$
\begin{array}{l} | \hat {G} [ k ] | = | \hat {G} _ {\mathrm{av}} [ k ] + \hat {G} [ k ] - \hat {G} _ {\mathrm{av}} [ k ] | \\ \geq \left| | \hat {G} _ {\mathrm{av}} [ k ] | - | \hat {G} [ k ] - \hat {G} _ {\mathrm{av}} [ k ] | \right|, \tag {76} \\ \end{array}
| e [ k ] | = | e _ {\mathrm{av}} [ k ] + e [ k ] - e _ {\mathrm{av}} [ k ] |\leq \left| e _ {\mathrm{av}} [ k ] \right| + \left| e [ k ] - e _ {\mathrm{av}} [ k ] \right|. \tag {77}$$

Using (72), (73), (74) and (75), the following bounds are obtained for (76) and (77), respectively:

$$
\begin{array}{l} | \hat {G} [ k ] | \geq \left| \left| \left(1 - (k - k _ {l}) \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) \hat {G} _ {\mathrm{av}} [ k _ {l} ] \right| - \mathcal {O} (\epsilon) \right| \\ = \underline {{\hat {G}}} [ k ], \tag {78} \\ \end{array}
| e [ k ] | \leq \left| (k - k _ {l}) \frac {\epsilon a ^ {2} H ^ {*} K}{2} \hat {G} _ {\mathrm{av}} [ k _ {l} ] \right| + \mathcal {O} (\epsilon)= \bar {e} [ k ]. \tag {79}$$

At this stage, we combine the upper bound for the error $\bar { e } [ k ]$ in (79) with the lower bound of the gradient estimate ${ \underline { { \hat { G } } } } [ k ]$ in order to determine the minimum number of iterations after which the event-triggering condition is necessarily violated. In other words, if at some iteration k one has ${ \sqrt { \sigma } } { \hat { G } } [ k ] - \alpha { \bar { e } } [ k ] \leq$ 0, then the condition (43) can no longer be ensured, and thus an event must occur. Therefore, the first iteration $k ^ { * }$ such that $\sqrt { \sigma } \hat { \cal G } [ k ] - \alpha \bar { e } [ k ] \leq 0$ provides an estimate of the minimum number of iterations after which the event-triggering condition is necessarily violated, $i . e .$ ,

$$k ^ {*} = \arg \min _ {k \in \mathbb {N}} \left\{\alpha \left(\left| (k - k _ {l}) \frac {\epsilon a ^ {2} H ^ {*} K}{2} \hat {G} _ {\mathrm{av}} [ k _ {l} ] \right| + \mathcal {O} (\epsilon)\right) \geq \right.\left. \sqrt {\sigma} \left| \left| \left(1 - (k - k _ {l}) \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) \hat {G} _ {\mathrm{av}} [ k _ {l} ] \right| - \mathcal {O} (\epsilon) \right| \right\}. \tag {80}$$

Hence, the proof is complete.
