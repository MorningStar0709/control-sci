# B. Proof of Lemma 2

By Lemma 1, there exists a nonsingular matrix $\Phi _ { \lambda }$ such that $\Phi _ { \lambda } \tilde { \Lambda } \Phi _ { \lambda } ^ { - 1 } = I _ { p } \otimes \Lambda$ . Let $\Phi _ { \lambda } \tilde { x } ( 0 ) = \operatorname { c o l } ( \beta _ { 1 } , \dots , \beta _ { p } )$ , with $\beta _ { i } \in \mathbb { R } ^ { n }$ , and let $\bar { V } _ { i } \in \mathbb { R } ^ { n \times n }$ be the solution of

$$V _ {i} \Lambda = \Lambda V _ {i}, \quad V _ {i} \Gamma = \beta_ {i}, \tag {68}$$

which exists and is unique by [20, Appendix II] because (Λ, Γ) is controllable. Define

$$
V := \left[ \begin{array}{c c c} V _ {1} ^ {\top} & \dots & V _ {p} ^ {\top} \end{array} \right] ^ {\top} \in \mathbb {R} ^ {n p \times n}, \tag {69}
$$

and let $\epsilon ( t ) : = V \chi ( t )$ , where χ(t) is given in (9) and satisfies $\dot { \chi } ( t ) = \Lambda \chi ( t )$ , with χ(0) = Γ. We obtain

$$\dot {\epsilon} (t) = V \Lambda \chi (t) = \Phi_ {\lambda} \tilde {\Lambda} \Phi_ {\lambda} ^ {- 1} \epsilon (t), \quad \epsilon (0) = V \Gamma = \Phi_ {\lambda} \tilde {x} (0), \tag {70}$$

which yields

$$C \Phi_ {\lambda} ^ {- 1} V \chi (t) = C \Phi_ {\lambda} ^ {- 1} e ^ {\Phi_ {\lambda} \tilde {\Lambda} \Phi_ {\lambda} ^ {- 1}} \Phi_ {\lambda} \tilde {x} (0) = C e ^ {\tilde {\Lambda} t} \tilde {x} (0), \tag {71}$$

and, thus, $H _ { 0 } : = C \Phi _ { \lambda } ^ { - 1 } V$ .
