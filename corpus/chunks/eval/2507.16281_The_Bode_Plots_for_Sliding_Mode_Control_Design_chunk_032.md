# A. Slow Motions due to Constant Perturbations

Once the EG given in (33) has been determined, the bias component in the sliding variable can be computed using the expression (45), namely

$$S _ {0} ^ {*} = \lim _ {\mathrm{s} \rightarrow 0} \frac {\eta G (\mathrm{s}) G _ {s} (\mathrm{s})}{1 + \frac {K _ {n}}{\mathrm{s}} W (\mathrm{s})} = 0, \tag {47}$$

where $G ( \mathrm { s } ) = 1 / \mathrm { s } , G _ { a } ( \mathrm { s } ) = 1 / ( \mu \mathrm { s } + 1 ) ^ { 2 } , G _ { s } ( \mathrm { s } ) = \mathrm { s } + b ,$ and $K _ { n } = ( 1 - 2 b \mu ) / \mu$ . On the other hand, the bias level in the tracking error can be computed from the sensitivity TF (46), resulting

$$\sigma_ {0} ^ {*} = \lim _ {\mathrm{s} \rightarrow 0} \frac {\eta G (\mathrm{s})}{1 + \frac {K _ {n}}{\mathrm{s}} W (\mathrm{s})} = 0, \tag {48}$$

The constant perturbation (3) satisfy $| \dot { f } | ~ \le ~ \bar { L } ~ = ~ 0$ and $| f | ~ \le ~ L ~ = ~ \eta ,$ uniformly in time. This means that the controller parameter in (40) must be selected $\rho ~ > ~ b \eta$ in ideal conditions $( \mu = 0 )$ . Several simulations are conducted using the parameters $b = 1 , \rho = 5 , \mu = 0 . 0 5$ , and various magnitudes of the constant disturbance (3), specifically $\eta = 1$ , 2, and 3, as depicted from top to bottom in Figure 13. At first glance, the steady-state responses align well with the theoretical predictions (47)–(48); notably, the bias levels in both the sliding variable and the tracking error remain effectively negligible. According to the analytical predictions (27), the chattering parameters are $\omega ^ { * } ~ = ~ 1 8 . 9 7 4$ [rad/s], $A ^ { * } = 0 . 1 7 6 8$ and $a ^ { * } = 0 . 0 0 9 3$ . In comparison, the simulation measurements yield $\omega \ : = \ : 1 8 . 4 7 9$ [rad/s], $A = 0 . 1 8 4 1$ , and $a = 0 . 0 1 0 1$ , resulting in prediction errors of 2.609% for the frequency, 4.129% for the amplitude of the sliding variable and 8.602% for the amplitude of the tracking error.
