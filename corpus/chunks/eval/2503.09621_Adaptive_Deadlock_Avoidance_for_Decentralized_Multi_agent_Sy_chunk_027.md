# C. Real World Experiment

We further use four Husarion ROSbot 2 PRO [22] robots as agents to play the multi-agent position-swapping game to verify the performance of our method. Besides, we put another robot at the center of the scenario as a static obstacle to verify the algorithm’s effectiveness. Several markers are set on the robot’s body for the motion capture system OptiTrack acquiring and broadcasting the real-time robots’ positions. We use the Robot Operating System(ROS) with a local internet with a router to establish connections between the OptiTrack and all four robots. After acquiring their positions, each robot uses the onboard computation resource to calculate the optimal control input for themselves. As shown in Fig. 5, the robots could automatically detour at the necessary positions. For the detailed implementation, we need to map the single-integrator dynamics to unicycle control commands for controlling the Husarion ROSbot 2PRO robots. Similarly to Robotarium [23], we use a nearidentity diffeomorphism (NID) between the single-integrator and unicycle models [24] to enable the mapping from the single-integrator control to the unicycle control.
