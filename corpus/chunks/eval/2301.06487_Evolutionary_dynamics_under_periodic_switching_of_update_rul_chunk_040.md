One thinks of the condition $\bar { \Phi } ( x _ { C } , x _ { C \mid C } , 0 ) \quad = \quad 0$ as determining a set on which the flow is given by $\begin{array} { r c l } { \frac { \mathrm { d } x _ { C } ( \tau ) } { \mathrm { d } \tau } } & { = } & { \bar { \Psi } ( x _ { C } , x _ { C | C } , 0 ) } \end{array}$ . From Fenichel’s Second Theorem [46], only $\mathcal { M } _ { 0 } ^ { 1 }$ is a normally hyperbolic compact manifold with a boundary. Then, for $\varepsilon \ > \ 0$ sufficiently small, there exists a local stable manifold $\mathcal { M } _ { \varepsilon } ^ { 1 }$ . This manifold lies within $\mathcal { O } ( \varepsilon )$ of $\mathcal { M } _ { 0 } ^ { 1 }$ and is diffeomorphic to a stable manifold $\mathcal { M } _ { 0 } ^ { 1 }$ . Moreover, it is smooth and locally invariant under the flow of the system (A39). $$\frac {\mathrm{d} x _ {C} (\tau)}{\mathrm{d} \tau} = \frac {k ^ {2} (k - 2) [ b - (k + 2) c ]}{(k + 1) ^ {2} (k - 1)} x _ {C} (1 - x _ {C}). \tag {A43}$$
