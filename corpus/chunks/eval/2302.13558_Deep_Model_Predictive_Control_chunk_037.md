$$
\begin{array}{l} V \left(x _ {t + 1}, K _ {t + 1}\right) - V \left(x _ {t}, K _ {t}\right) \leqslant - (1 - 2 \gamma^ {2}) V _ {m} ^ {2} \left(x _ {t}\right) + \frac {a _ {0}}{m ^ {2}} \left\| \varepsilon_ {j} \left(x _ {t}\right) \right\| ^ {2} + 2 \hat {c} _ {3} ^ {2} \hat {\delta} _ {g} ^ {2} \| \tilde {u} _ {t} \| ^ {2} - a _ {0} \left(\frac {1 - \theta}{\sigma^ {2}}\right) \| \tilde {u} _ {t} \| ^ {2} \\ = - (1 - 2 \gamma^ {2}) V _ {m} ^ {2} (x _ {t}) + \frac {a _ {0}}{m ^ {2}} \left\| \varepsilon_ {j} (x _ {t}) \right\| ^ {2} \\ \leqslant - (1 - 2 \gamma^ {2}) c _ {1} ^ {2} \| x _ {t} \| ^ {4} + a _ {0} \left(\frac {\beta}{m}\right) ^ {2} \| x _ {t} \| ^ {4} \\ = - \eta \left\| x _ {t} \right\| ^ {4}, \\ \end{array}
$$

where $\eta = ( 1 - 2 \gamma ^ { 2 } ) c _ { 1 } ^ { 2 } - \textstyle { \frac { 2 } { 1 - \theta } } \left( \sigma { \hat { c } } _ { 3 } { \hat { \delta } } _ { g } \textstyle { \frac { \beta } { m } } \right) ^ { 2 } > 0$ because $\begin{array} { r } { \beta < \frac { c _ { 1 } m } { \sqrt { 2 } \sigma \hat { c } _ { 3 } \hat { \delta } _ { g } } \sqrt { ( 1 - 2 \gamma ^ { 2 } ) ( 1 - \theta ) } } \end{array}$ . Therefore,

$$\left\| x _ {t} \right\| ^ {4} \leqslant \frac {1}{\eta} \left(- V \left(x _ {t + 1}, K _ {t + 1}\right) + V \left(x _ {t}, K _ {t}\right)\right).$$

By summing from $t = \hat { N }$ to $k + \hat { N }$ on both sides, we get

$$\sum_ {t = \hat {N}} ^ {\hat {N} + k} \| x _ {t} \| ^ {4} \leqslant \frac {1}{\eta} \left(- V (x _ {\hat {N} + k + 1}, K _ {\hat {N} + k + 1}) + V (x _ {\hat {N}}, K _ {\hat {N}})\right)
\begin{array}{l} \leqslant \frac {1}{\eta} V (x _ {\hat {N}}, K _ {\hat {N}}) = \frac {1}{\eta} \left(V _ {m} ^ {2} (x _ {\hat {N}}) + a _ {0} V _ {a} (K _ {\hat {N}})\right) \\ \leqslant \frac {1}{\eta} \left(\hat {c} ^ {2} + \frac {4 a _ {0}}{\theta} \bar {W}\right), \\ \end{array}
$$

where the last inequality is due to Lemma 1-(i) and the fact that $x _ { \hat { N } } \in \mathcal { X } _ { \hat { c } } ( 0 )$ . Since the right hand side of the above inequality is independent of k, we have $\begin{array} { r } { \sum _ { t = \hat { N } } ^ { \infty } \| x _ { t } \| ^ { 4 } \leqslant \frac { 1 } { \eta } \left( \hat { c } ^ { 2 } + \frac { 4 a _ { 0 } } { \theta } \bar { W } \right) } \end{array}$ , which implies $\left\| x _ { t } \right\| \to 0 \mathrm { a s } t \to \infty$ . □
