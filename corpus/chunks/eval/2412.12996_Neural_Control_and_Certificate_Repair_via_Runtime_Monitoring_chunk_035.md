# Runtime Monitoring of Lyapunov Functions

In this section, we present how to construct $\mathcal { M } _ { \sf C e r t P M }$ in the case of a stability property and a Lyapunov function certificate. Consider a stability property defined by the goal set $\chi _ { g } .$ . Let π be a policy and V be a Lyapunov function. Consider a finite sequence of observed states $x _ { 0 } , x _ { 1 } , \ldots , x _ { n } . \ M$ CertPM checks if all the defining conditions in Proposition 2 are satisfied at xn: $x _ { n } \mathrm { { : } }$

• The certificate violation verdict for the Zero upon goal condition in Proposition 2 is issued whenever $x _ { n } \in \mathcal { X } _ { g }$ is a goal state but $\nu ( x _ { n } ) ~ \neq ~ 0$ . We set $\mathcal { M } _ { \sf C e r t P M } ( x _ { 0 } , x _ { 1 } , \dots , x _ { n } ) = 1$ .   
• The certificate violation verdict for the Strict positivity condition in Proposition 2 is issued whenever whenever $\nu ( x _ { n } ) < 0$ . We set $\mathcal { M } _ { \sf C e r t P M } ( x _ { 0 } , x _ { 1 } , . . . , x _ { n } )$ .   
• The certificate violation verdict for the Decreasing condition in Proposition 2 is issued as follows. Similarly as for barrier functions, we approximate the Decreasing condition by considering the subsequent observed state $x _ { n + 1 }$ and approximating the Lie derivative via

$$\widehat {L _ {f} \mathcal {V}} (x _ {n}) = \frac {\mathcal {V} (x _ {n + 1}) - \mathcal {V} (x _ {n})}{t _ {n + 1} - t _ {n}}.$$

The certificate violation verdict for the Decreasing condition in Proposition 2 is issued whenever $x _ { n } \notin \pmb { \mathscr { X } }$ $\chi _ { g } , \ \mathcal { V } ( x _ { n } ) \quad > \quad 0$ and $\widehat { L _ { f } \mathcal { V } } ( x _ { n } ) \quad \geq \quad 0 .$ . We set $\mathcal { M } _ { \sf C e r t P M } ( x _ { 0 } , x _ { 1 } , \dots , x _ { n } )$ .

• Finally, if none of the three conditions are violated, no violation verdict is issued. We set $\mathcal { M } _ { \sf C e r t P M } ( x _ { 0 } , x _ { 1 } , \dots , x _ { n } ) = 0$ .
