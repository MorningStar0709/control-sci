# A. System Matrices for the POCP Problem

As also shown in [9], the matrices in (3) are given by $H = \hat { B } ^ { \top } \hat { H } \hat { B } + ( I _ { N } \otimes R ) , G = \hat { B } ^ { \top } \hat { H } \hat { A } , W = Q + \hat { A } ^ { \top } \hat { H } \hat { A } ,$

$$
\hat {H} = \left[ \begin{array}{c c} (I _ {N} \otimes Q) & 0 \\ 0 & P \end{array} \right],

\hat {B} = \left[ \begin{array}{c c c} 0 & 0 & 0 \\ B & 0 & 0 \\ \vdots & \ddots & \vdots \\ A ^ {N - 1} B & \dots & B \end{array} \right], \text {and} \hat {A} = \left[ \begin{array}{c} I \\ A \\ \vdots \\ A ^ {N} \end{array} \right].
$$
