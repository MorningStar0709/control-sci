# Exercise A.22: Some matrix differentiation

Derive the following formulas (Bard, 1974). A, $B \in \mathbb { R } ^ { n \times n } , a , x \in \mathbb { R } ^ { n }$

(a)

$$\frac {\partial x ^ {\prime} A x}{\partial x} = A x + A ^ {\prime} x$$

(b)

$$\frac {\partial A x a ^ {\prime} B x}{\partial x ^ {\prime}} = \left(a ^ {\prime} B x\right) A + A x a ^ {\prime} B$$

(c)

$$\frac {\partial a ^ {\prime} A b}{\partial A} = a b ^ {\prime}$$
