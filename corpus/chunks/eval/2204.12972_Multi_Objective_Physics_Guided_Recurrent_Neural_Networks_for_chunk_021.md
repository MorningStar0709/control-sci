We want to emphasize that comparable results can be obtained using a data-driven method like e.g., SINDy by Brunton et al. (2016), given an adequate library of candidate functions is identified. However, in contrast to our method, approaches like SINDy do not account for the physical plausibility of the obtained model.

To illustrate the MOPGRNN’s simulative performance in terms of accuracy and physical plausibility, both the predicted trajectories and the violation $\Delta : = | \bar { \Delta } E _ { \mathcal { M } , t } - \Delta \mathbf { \hat { } { } } E _ { \mathcal { S } , t } |$ | of the physics-based restriction are shown in Fig. 6. Except of small deviations, the $\mathcal { M } _ { M O P G R N N }$ succeeds at capturing the golf robot’s dynamics (in light blue). Despite progressive accumulation, absolute constraint violation (in red) remains reasonably low. Since in practice the obtained model is used in conjunction with a model-based controller with regular feedback, long prediction horizons are usually not relevant anyway.
