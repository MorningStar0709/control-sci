# C. Fixed-Time Stability and Input-to-State Stability

Consider the following system

$$\dot {x} = f (x, u), \quad x (0) = x _ {0}, \tag {1}$$

where $x \in \mathbb { R } ^ { n }$ is the state, $u \in \mathcal { L } _ { \infty } ^ { m }$ is the input, and $f : \mathbb { R } ^ { n } \times$ $\mathbb { R } ^ { m } \to \mathbb { R } ^ { n }$ is a continuous function that satisfies $f ( 0 , 0 ) = 0$ . Given $x _ { 0 } \in \mathbb { R } ^ { n }$ and $u \in \mathcal { L } _ { \infty } ^ { m }$ , a solution to (1) is represented as $x ( t , x _ { 0 } , u )$ for $t \geq 0 .$ , where $x ( 0 , x _ { 0 } , u ) = x _ { 0 }$ . If x0 and u are clear from the context, we will use $x ( t ) : = x ( t , x _ { 0 } , u )$ . When x is defined for all $t \geq 0$ , we say that x is forward complete.

In this paper, we are interested in studying fixed-time (ISS) stability properties for system (1). The following definitions, borrowed from [4], [28], [29], formalize these notions:

Definition 1: The origin of (1) with $u = 0$ is said to be uniformly globally fixed-time stable $( F x T S )$ if there exists $\beta \in$ $\kappa \mathcal { L } _ { \mathrm { F x T } }$ such that for each $x _ { 0 } \in \mathbb { R } ^ { n }$ , every solution of (1) is forward complete and satisfies

$$| x (t) | \leq \beta (| x _ {0} |, t), \quad \forall \quad t \geq 0. \tag {2}$$

Definition 2: The origin of (1) is said to be fixed-time inputto-state stable $( F x T – I S S )$ if there exists $\beta \in \mathcal { K L } _ { \mathrm { F x T } }$ and $\varrho \in \mathcal { K }$ such that for each $x _ { 0 } \in \mathbb { R } ^ { n }$ and $u \in \mathcal { L } _ { \infty } ^ { m }$ , every solution x(t) of (1) is forward complete and satisfies

$$| x (t) | \leq \beta (| x _ {0} |, t) + \varrho (| u | _ {\infty}), \forall t \geq 0.$$

The FxT and FxT-ISS properties of (1) can be studied via Lyapunov functions. In particular, given a locally Lipschitz continuous function $V : \mathbb { R } ^ { n }  \mathbb { R } _ { \ge 0 }$ , we define

$$\dot {V} (x, u) := D V (x; f (x, u)).$$

Note that $\dot { V }$ also satisfies $\dot { V } ( x , u ) = D ^ { + } V ( x ( t , x _ { 0 } , u ) )$ along the trajectories of the system (1) for all $t \ \geq \ 0 .$ . When $u ( t ) \equiv 0$ in (1), we simply use $\dot { V } ( x ) : = \dot { V } ( x , 0 )$ .
