# A. Simulations

When ${ \mathfrak { O } } _ { s } = 0 . 4 ~ { \mathrm { s } } ,$ the controllers exhibit similar behavior. The difference is observed only at the beginning, where ICs and MPCs have quite different paths. The interpolating coefficients are zero except at the very beginning of the simulation in the ${ \vec { y } } -$ axis. Thus, the ICs achieved optimal behavior for the most of time. It Table I, we can see the interpolation-based controllers are worse in optimality criterion because they consumed more energy. However, they followed the trajectory more closely.
