# 7.4. Results: turbulent flow measurement

The results for hot-wire measurements are covered in this subsection. For brevity, we omit the details of the measurement procedure, which are accessible in the dissertation by Wang (2023). According to the Reynolds decomposition given in Pope (2000), we decompose the measured wind velocity u into the sum of a time-averaged velocity u and fluctuating velocity u, such that $u = \overline { { u } } + \widetilde { u } ,$ e ewhere (·) stands for the time-averaged quantity. Based on hotwire measurements, we characterize the FAWT wind field with the following quantities: the time-averaged velocity u, the variance of the fluctuating velocity $\overline { { \widetilde { u } ^ { 2 } } }$ , and the turbulence intensity (TI), which is defined by ${ \sqrt { \overline { { \widetilde { u } ^ { 2 } } } } } / \overline { { u } } .$ . We eitemize the results as follows:

$\bullet \ \overline { { u _ { 3 0 } } } = 5 . 4 7 2 \ \mathrm { m / s } ; \widetilde { u _ { 3 0 } } ^ { 2 } = 0 . 0 6 1 \ \mathrm { m ^ { 2 } / s ^ { 2 } } ; \mathrm { T I _ { 3 0 } } = 0 . 0 4 5 1 ;$   
$\bullet \ \overline { { u _ { 4 0 } } } = 6 . 8 7 6 \ \mathrm { m / s } ; \overline { { \widetilde { u _ { 4 0 } } ^ { 2 } } } = 0 . 0 8 2 \ \mathrm { m ^ { 2 } / s ^ { 2 } } ; \mathrm { T I _ { 4 0 } } = 0 . 0 4 1 7 ;$   
$\bullet \ \overline { { u _ { 5 0 } } } = 8 . 2 1 3 \ \mathrm { m / s } ; \widetilde { u _ { 5 0 } } ^ { 2 } = 0 . 1 1 6 \ \mathrm { m ^ { 2 } / s ^ { 2 } } ; \mathrm { T I _ { 5 0 } } = 0 . 0 4 1 5 ;$   
$\bullet \ \overline { { u _ { 6 0 } } } = 9 . 5 9 0 \ \mathrm { m / s } ; \overline { { \widetilde { u _ { 6 0 } } ^ { 2 } } } = 0 . 1 6 8 \ \mathrm { m ^ { 2 } / s ^ { 2 } } ; \mathrm { T I _ { 6 0 } } = 0 . 0 4 2 7 ;$   
• $\overline { { u _ { 7 0 } } } = 1 0 . 9 2 0 ~ \mathrm { m / s ; ~ } \widetilde { u _ { 7 0 } } ^ { 2 } = 0 . 2 3 7 ~ \mathrm { m ^ { 2 } / s ^ { 2 } ; ~ T I _ { 7 0 } } = 0 . 0 4 4 6 .$

To summarize, for the FAWT in the setup described earlier in this subsection, we observe that u ranges from 5.472 m/s to 10.920 m/s, and TI is around 0.043. Moreover, we observe that $\overline { { \widetilde { u } ^ { 2 } } }$ has positive correlation with u. We assume that higher $\overline { { \widetilde { u } ^ { 2 } } }$ ebrings higher turbulence energy, which causes higher disturbance inputs to the aircraft flying within the wind field.
