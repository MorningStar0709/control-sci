Now, $( x ^ { \star } , a ^ { \star } )$ is the unique minimum of Lasso if $\| \zeta _ { \bar { S } } \| _ { \infty } < 1$ . Therefore, we study the inequality

$$
\left\| I _ {\bar {S}} ^ {\top} \left( \begin{array}{c c} \mathcal {O} & I _ {S} \end{array} \right) \left( \begin{array}{c c} \mathcal {O} ^ {\top} \mathcal {O} & \mathcal {O} _ {S} ^ {\top} \\ \mathcal {O} _ {S} & I _ {h} \end{array} \right) ^ {- 1} \binom {0} {\text { sign } (\widetilde {a} _ {S})} \right\| _ {\infty} <   1. \tag {14}
$$

Since $I _ { \bar { \cal S } } ^ { \top } I _ { \cal S } = 0 .$

$$
I _ {\bar {S}} ^ {\top} \left( \begin{array}{c c} \mathcal {O} & I _ {S} \end{array} \right) = \left( \begin{array}{c c} \mathcal {O} _ {\bar {S}} & 0 \end{array} \right). \tag {15}
$$

From Schur’s complement arguments,

$$
\left( \begin{array}{c c} \mathcal {O} ^ {\top} \mathcal {O} & \mathcal {O} _ {\mathcal {S}} ^ {\top} \\ \mathcal {O} _ {\mathcal {S}} & I _ {h} \end{array} \right) ^ {- 1} = \left( \begin{array}{c c} \Omega_ {1} & \Omega_ {2} \\ \Omega_ {3} & \Omega_ {4} \end{array} \right) \tag {16}
$$

where

$$\Omega_ {2} = - [ \mathcal {O} ^ {\top} \mathcal {O} - \mathcal {O} _ {\mathcal {S}} ^ {\top} \mathcal {O} _ {\mathcal {S}} ] ^ {- 1} \mathcal {O} _ {\mathcal {S}} = - [ \mathcal {O} _ {\bar {\mathcal {S}}} ^ {\top} \mathcal {O} _ {\bar {\mathcal {S}}} ] ^ {- 1} \mathcal {O} _ {\mathcal {S}} ^ {\top}. \tag {17}$$

By applying (15),(16) and (17) to (14),

$$
\left( \begin{array}{c c} \mathcal {O} _ {\bar {\mathcal {S}}} & 0 \end{array} \right) \left( \begin{array}{c c} \Omega_ {1} & \Omega_ {2} \\ \Omega_ {3} & \Omega_ {4} \end{array} \right) \binom {0} {\text {sign} (\widetilde {a} _ {\mathcal {S}})} = - \mathcal {O} _ {\bar {\mathcal {S}}} ^ {\dagger^ {\top}} \mathcal {O} _ {\mathcal {S}} ^ {\top} \text {sign} (\widetilde {a} _ {\mathcal {S}}). \tag {18}
$$

In conclusion, condition (14) is equivalent to (5).

Since $\left. \mathcal { O } _ { \bar { s } } ^ { \dagger ^ { \top } } \mathcal { O } _ { s } ^ { \top } \mathrm { s i g n } ( \widetilde { \boldsymbol { a } } _ { S } ) \right. _ { \infty } \leq \left. \mathcal { O } _ { \bar { s } } ^ { \dagger ^ { \top } } \mathcal { O } _ { s } ^ { \top } \right. _ { \infty }$ then (6) is sufficient for a successful Lasso.
