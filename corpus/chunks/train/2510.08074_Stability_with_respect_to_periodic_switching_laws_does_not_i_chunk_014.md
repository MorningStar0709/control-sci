# 4. An explicit example

We illustrate Theorem 3 with an explicit example. Consider two matrices $A _ { 0 } , A _ { 1 } \in$ $M _ { 2 } ( \mathbb { R } )$ defined by

$$
A _ {0} = \left( \begin{array}{c c} - 1 & \frac {\sqrt {\tau} (\tau - 1)}{\sqrt {2}} \\ \frac {1 - \tau}{\sqrt {2 \tau}} & - \tau \end{array} \right), A _ {1} = \left( \begin{array}{c c} - \tau & \frac {\tau - 1}{\sqrt {2 \tau}} \\ \frac {\sqrt {\tau} (1 - \tau)}{\sqrt {2}} & - 1 \end{array} \right),
$$

where $\tau = 0 . 1 2 9 9 9 9 2 . .$ . is the unique real number which satisfies

$$\tau = \exp \left(\frac {\pi (\tau + 1)}{2 (\tau - 1)}\right), \qquad 0 < \tau < 1.$$

This pair $A _ { 0 } , A _ { 1 }$ can be shown to satisfy the properties described in Theorem 2. We briefly outline a demonstration of this fact which applies the ideas of $[ 8 ]$ to exhibit a non-strict common Lyapunov function. Consider the function $f \colon  { \mathbb { R } } ^ { 2 } \to  { \mathbb { R } }$ which is zero at the origin and which is otherwise given by

$$\left(x _ {1} ^ {2} + \sqrt {2 \tau} x _ {1} x _ {2} + \tau x _ {2} ^ {2}\right) e ^ {\frac {2 (\tau + 1)}{\tau - 1} \arctan \left(\frac {x _ {1}}{x _ {1} + \sqrt {2 \tau} x _ {2}}\right)}$$

when $( x _ { 1 } , x _ { 2 } ) \in \mathbb { R } ^ { 2 }$ satisfies $x _ { 1 } x _ { 2 } \ge 0$ and by

$$\left(\tau x _ {1} ^ {2} - \sqrt {2 \tau} x _ {1} x _ {2} + x _ {2} ^ {2}\right) e ^ {\frac {2 (1 + \tau)}{1 - \tau} \arctan \left(\frac {x _ {2}}{\sqrt {2 \tau} x _ {1} - x _ {2}}\right)}$$

when $x _ { 1 } x _ { 2 } \ \leq \ 0$ . A prolonged but elementary calculation shows that $f$ is a $C ^ { 1 }$ function, is non-negative for all nonzero $( x _ { 1 } , x _ { 2 } )$ , and satisfies

$$
\begin{array}{l} (\nabla f) (x _ {1}, x _ {2}) \cdot A _ {0} (x _ {1}, x _ {2}) ^ {T} = 0 \quad \text {if} x _ {1} x _ {2} \geq 0, \\ (\nabla f) (x _ {1}, x _ {2}) \cdot A _ {0} (x _ {1}, x _ {2}) ^ {T} <   0 \quad \text { if } x _ {1} x _ {2} <   0 \\ \end{array}
$$

and

$$
\begin{array}{l} (\nabla f) (x _ {1}, x _ {2}) \cdot A _ {1} (x _ {1}, x _ {2}) ^ {T} = 0 \quad \text { if } x _ {1} x _ {2} \leq 0, \\ (\nabla f) (x _ {1}, x _ {2}) \cdot A _ {1} (x _ {1}, x _ {2}) ^ {T} <   0 \quad \text {if} x _ {1} x _ {2} > 0. \\ \end{array}
$$

These properties imply that f is a $C ^ { 1 }$ non-strict Lyapunov function for the corresponding switched system: if $x ( t )$ is a nonzero solution to an equation

$$\dot {x} (t) = ((1 - u (t)) A _ {0} + u (t) A _ {1}) x (t)$$
