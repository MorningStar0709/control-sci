$$
K _ {1} = \left[ \begin{array}{c c c} 1 / 4 & 3 / 4 & 0 \\ 3 / 4 & 1 / 4 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \end{array} \right], K _ {2} = \left[ \begin{array}{c c c} 3 / 4 & 1 / 4 & 0 \\ 1 / 4 & 3 / 4 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \end{array} \right]. \tag {7}

Let $\theta ^ { ( 1 ) } = ( 1 , 0 ) ^ { \mathtt { T } }$ and $\theta ^ { ( 2 ) } = ( 0 , 1 ) ^ { \mathtt { T } }$ . Let us fix the initial state to be $s _ { 1 }$ . Since a nonzero reward is only earned during a $s _ { 2 } \to s _ { 4 }$ transition, we note for any policy $\pi : { \mathcal { A } }  S$ that $V ^ { \pi } ( s _ { 1 } ) = \pi ( a _ { 1 } | s _ { 1 } ) \pi ( a _ { 2 } | s _ { 2 } ) \eta$ . We also have,

$$
(K _ {1} + K _ {2}) / 2 = \left[ \begin{array}{c c c} 1 / 2 & 1 / 2 & 0 \\ 1 / 2 & 1 / 2 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \end{array} \right].
$$

We will show that ${ \textstyle \frac { 1 } { 2 } } V ^ { \pi _ { \theta ^ { ( 1 ) } } } + { \textstyle \frac { 1 } { 2 } } V ^ { \pi _ { \theta ^ { ( 2 ) } } } > V ^ { \pi _ { \left( \theta ^ { ( 1 ) } + \theta ^ { ( 2 ) } \right) / 2 } }$ .

$$V ^ {\pi_ {\theta^ {(1)}}} (s _ {1}) = V ^ {K _ {1}} (s _ {1}) = (1 / 4). (1 / 4). r = r / 1 6.V ^ {\pi_ {\theta^ {(2)}}} (s _ {1}) = V ^ {K _ {2}} (s _ {1}) = (3 / 4). (3 / 4). r = 9 r / 1 6.$$

Also, on the other hand we have,

$$V ^ {\pi \left(\theta^ {(1)} + \theta^ {(2)}\right) / 2} = V ^ {\left(K _ {1} + K _ {2}\right) / 2} (s _ {1}) = (1 / 2). (1 / 2). r = r / 4.$$

This shows that $\theta \mapsto V ^ { \pi _ { \theta } }$ is non-concave, which concludes the proof for direct parameterization.
