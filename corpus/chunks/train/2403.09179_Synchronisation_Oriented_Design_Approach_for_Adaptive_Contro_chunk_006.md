# II. Synchronisation-Oriented Approach to Adaptive Control

The proposed design approach proceeds with the insights gained from distributed consensus algorithms; i) both the plant and the reference model in MRAC system can actively participate in the synchronisation process via their inputs, and ii) the input of each system can be defined in various ways without changing the state error dynamics. This section is devoted to the detailed development of the idea into a systematic and generalised approach towards the design of MRAC systems.
