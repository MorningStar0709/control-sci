# A. Parameterization

We first derive an unconstrained parameterization for synaptic matrices of S-contracting FRNNs.

Theorem 35 (Parameterization of weight matrices): The following statements are equivalent:

(i) The synaptic matrix W satisfies the continuous time MONE FRNN certificate (25) for some $P \succ 0$ , diagonal matrix $Q \succ 0 ,$ and $c \in [ 0 , 1 ]$ .

(ii) W can be written as

$$W = 2 \sqrt {1 - c} \mathrm{diag} (\mathrm{e} ^ {d}) S V ^ {\top} V - \mathrm{diag} (\mathrm{e} ^ {2 d}) (V ^ {\top} V) ^ {2}$$

where $S \in \mathbb { R } ^ { n \times n }$ satisfies $S ^ { \top } S \preceq I _ { n } , V$ is a full rank matrix in $\mathbb { R } ^ { n \times n }$ , and $d \in \mathbb { R } ^ { n }$ .

Proof: Using the Schur complement, the inequality (25) is equivalent to

$$(P + W ^ {\top} Q) Q ^ {- 1} (P + Q W) \preceq 4 (1 - c) P. \tag {73}$$

From the Douglas-Fillmore-Williams Lemma [5, Theorem 8.6.2], the inequality (73) is true if and only if there exists a matrix $S \in \mathbb { R } ^ { n \times n }$ satisfying $S ^ { \top } S \preceq I _ { n }$ and

$$Q ^ {- 1 / 2} (P + Q W) = 2 \sqrt {(1 - c)} S P ^ {1 / 2}. \tag {74}$$

Upon rearranging to solve for $W .$ ,

$$W = 2 \sqrt {1 - c} Q ^ {- 1 / 2} S P ^ {1 / 2} - Q ^ {- 1} P. \tag {75}$$

Now, the set of all valid Q can be parameterized using $\mathrm { d i a g ( e ^ { - 2 d } ) }$ , and the set of all $P \succ 0$ can be parameterized as $V ^ { \dagger } V$ for full rank $V \in \mathbb { R } ^ { n \times n }$ .

Remark 36 (Unconstrained Optimization Formulation): In numerical implementations, the set of all matrices S such that $S ^ { \top } S \preceq I _ { n }$ may be parameterized via a free variable $X \in$ $\mathbb { R } ^ { n \times n }$ by setting $S = \overset { \cdot } { X } ( I + X ^ { \top } X ) ^ { - 1 / 2 }$ . Similarly, to ensure $V ^ { \top } V$ does not lose rank, we may approximate $V ^ { \dagger } V$ using a free matrix $Y \in \mathbb { R } ^ { n \times n }$ and a small constant $\epsilon > 0$ by setting $P = Y ^ { \top } Y + \epsilon I _ { n } .$ .

Theorem 35 provides a constructive mechanism to design continuous-time FRNNs that are contracting by construction. By optimizing over the free variables $\{ X , Y , d \}$ , stability is preserved during learning.
