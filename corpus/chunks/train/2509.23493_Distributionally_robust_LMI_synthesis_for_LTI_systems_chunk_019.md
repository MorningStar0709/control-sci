# B. Robust performance analysis for independent disturbances

In this section, we derive the dual problem of (6) for the case of independent disturbances. As in Section V-A, this dual takes the form of a robust performance analysis problem typically found in dissipativity theory.

Theorem 5.6: Let A be stable. Consider the case ${ \mathcal { S } } = { \mathcal { S } } _ { \mathrm { i n d } }$ In this case, (6) is equivalent to the optimization problem

$$\underset {\lambda \geq 0, \mathcal {P}, Q} {\text { minimize }} \quad \operatorname{tr} (Q \Sigma_ {\mathrm{nom}}) + \lambda \gamma^ {2} \tag {13a}\text { s.t. } \quad \mathcal {A} ^ {\top} \mathcal {P} \mathcal {A} - \mathcal {P} + \mathcal {C} ^ {\top} \mathcal {C} < 0, \tag {13b}
\left( \begin{array}{c c} \mathcal {B} & 0 \\ \mathcal {D} & 0 \\ I & 0 \\ 0 & I \end{array} \right) ^ {\top} \left( \begin{array}{c c c c} \mathcal {P} & 0 & 0 & 0 \\ 0 & I & 0 & 0 \\ 0 & 0 & - \lambda I & \lambda I \\ 0 & 0 & \lambda I & - Q - \lambda I \end{array} \right) \left( \begin{array}{c c} \mathcal {B} & 0 \\ \mathcal {D} & 0 \\ I & 0 \\ 0 & I \end{array} \right) <   0. \tag {13c}
$$

The stability of A further guarantees the feasibility of (13).

Proof: By confining Σ to the set ${ \mathcal { S } } _ { \mathrm { i n d } } .$ , we can rewrite the dual optimization problem corresponding to (13) as

$$\sup _ {\Sigma \succeq 0, \Sigma \in \mathcal {S} _ {\mathrm{ind}}} \inf _ {\lambda \geq 0, Q, \mathcal {P}} \operatorname{tr} \left(Q \Sigma_ {\mathrm{nom}}\right) + \lambda \gamma^ {2} +
\operatorname{tr} \Bigg (\underbrace {\left(\bigstar\right) ^ {\top} \left( \begin{array}{c c c c c} - \mathcal {P} & 0 & 0 & 0 & 0 \\ 0 & \mathcal {P} & 0 & 0 & 0 \\ 0 & 0 & I & 0 & 0 \\ 0 & 0 & 0 & - \lambda I & \lambda I \\ 0 & 0 & 0 & \lambda I & - Q - \lambda I \end{array} \right) \left( \begin{array}{c c c} I & 0 & 0 \\ \mathcal {A} & \mathcal {B} & 0 \\ \mathcal {C} & \mathcal {D} & 0 \\ 0 & I & 0 \\ 0 & 0 & I \end{array} \right)} _ {= (\star)} \Sigma \Bigg).
$$

Specifically, the left upper 1 × 1 block of (⋆) corresponds to (13b) and the right lower $2 \times 2$ block corresponds to (13c). The remaining blocks of (⋆) are irrelevant, since the top right $1 \times 2$ and bottom left $2 \times 1$ blocks of Σ are zero. The rest of the proof is analogous to the one of Theorem 5.5. □
