Finally, the number of oscillations (cycles) $N _ { \mathrm { c y c l e s } }$ during the transient response can be estimated by dividing the settling time Eq. (7.79) by the damped period $T _ { \mathrm { p e r i o d } } = 2 \pi / \omega _ { d }$ . After substituting $\omega _ { d } = \omega _ { n } \sqrt { 1 - \zeta ^ { 2 } }$ and some simplification, we obtain

$$N _ {\text { cycles }} = \frac {2 \sqrt {1 - \zeta^ {2}}}{\pi \zeta} \tag {7.80}$$

Equation (7.80) shows that the number of cycles during the transient response is a function of damping ratio only.
