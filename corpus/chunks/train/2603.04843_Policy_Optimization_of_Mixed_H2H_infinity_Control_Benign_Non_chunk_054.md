# A.4 Proof of Theorem 2

To prove that $J _ { \mathrm { m i x } }$ is real analytic on $\displaystyle \kappa _ { \beta } .$ , it suffices to show that the stabilizing solution $X _ { K }$ to (8) depends real analytically on $K \in \mathcal { K } _ { \beta }$ . This will be established using the following version of the implicit function theorem.

Theorem 9 [51, Theorem 2.3.5] Let $F : \mathrm { R } ^ { k } \times \mathrm { R } ^ { m }  \mathrm { R } ^ { m }$ be a mapping such that each of its components, denoted by $F _ { j } \ f o r \ j = 1 , . . . , m$ , is a real analytic function. Suppose $( x _ { 0 } , y _ { 0 } ) \in \mathrm { R } ^ { k } \times \mathrm { R } ^ { m }$ is a solution to the equation $F ( x , y ) = 0 $ . Furthermore, suppose the Jacobian matrix

$$
\mathbf {J} _ {y} F (x, y) = \left[ \begin{array}{c c c} \frac {\partial F _ {1}}{\partial y _ {1}} & \dots & \frac {\partial F _ {1}}{\partial y _ {m}} \\ \vdots & \ddots & \vdots \\ \frac {\partial F _ {m}}{\partial y _ {1}} & \dots & \frac {\partial F _ {m}}{\partial y _ {m}} \end{array} \right]
$$

is invertible at $( x , y ) = ( x _ { 0 } , y _ { 0 } )$ . Then there exists a neighborhood $U \subseteq \mathrm { R } ^ { k }$ of x0 and a mapping $\phi : U \to \mathrm { R } ^ { m }$ such that (1) each component of φ is a real analytic function, and (2) $F ( x , \phi ( x ) { \big ) } = 0$ for all $x \in U$ .

We now proceed to verify the conditions in Theorem 9 for the Riccati equation (8). By vectorizing (8), we define the following mapping $F : { \mathcal { K } } _ { \beta } \times \mathrm { R } ^ { n ^ { 2 } } \to \mathrm { R } ^ { n ^ { 2 } }$ by

$$
\begin{array}{l} F (K, \operatorname{vex} (X)) = \left(I _ {n} \otimes A _ {K} + A _ {K} \otimes I _ {n}\right) \operatorname{vec} (X) \\ + \beta^ {- 2} \mathrm{vec} (X S _ {K} X) + \mathrm{vec} (W), \\ \end{array}
$$

We then compute the Jacobian of $F \operatorname { w . r . t . } \operatorname { v e x } ^ { \mathsf { T } } ( X )$ . Using standard matrix calculus and the symmetry of X and $S _ { K }$ ,

$$
\begin{array}{l} \frac {\partial \mathrm{vec} (X S _ {K} X)}{\partial \mathrm{vec} ^ {\mathsf {T}} X} \\ = (X ^ {\mathsf {T}} \otimes I _ {n}) \frac {\partial \operatorname{vec} (X S _ {K})}{\partial \operatorname{vec} ^ {\mathsf {T}} X} + (I _ {n} \otimes X S _ {K}) \frac {\partial \operatorname{vec} X}{\partial \operatorname{vec} ^ {\mathsf {T}} X} \\ = (X ^ {\mathsf {T}} \otimes I _ {n}) (S _ {K} ^ {\mathsf {T}} \otimes I _ {n}) + I _ {n} \otimes X S _ {K} \\ = X ^ {\mathsf {T}} S _ {K} ^ {\mathsf {T}} \otimes I _ {n} + I _ {n} \otimes X S _ {K}. \\ \end{array}
$$
