# Solution:

An example of an open set S is the area inside a circle, not including its circumference: $\mathcal { S } = \{ x , y \mid x ^ { 2 } + y ^ { 2 } < r ^ { 2 } , r \in \mathbb { R } \}$ . In this case, its border is not included.

If we take the same set but including its border, so: $\mathcal { S } = \{ x , y \mid x ^ { 2 } + y ^ { 2 } \leq r ^ { 2 } , r \in \mathbb { R } \}$ , then this is an example of closed set. 
