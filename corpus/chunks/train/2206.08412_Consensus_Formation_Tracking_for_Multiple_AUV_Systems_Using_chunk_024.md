$$
\begin{array}{l} \dot {V} _ {2} \leq - s ^ {\mathrm{T}} (A _ {s} - S) ^ {- 1} s \\ - x _ {s} ^ {\mathrm{T}} \left[ D _ {s} ^ {- 1} A _ {s} - \beta_ {0} A _ {s} ^ {- 1} (L + B) \right] x _ {s} \\ + \frac {2}{\beta_ {0}} s ^ {\mathrm{T}} \bar {J} (\eta) \bar {B} d \tag {41} \\ - x _ {s} ^ {\mathrm{T}} (A _ {s} - S) ^ {- 1} (L + B) \bar {J} (\eta) \bar {B} d. \\ \end{array}
$$

The term $A _ { s } \mathrm { ~ - ~ } S$ is already positive definite due to the fact that $s < 0$ , and we also need the following matrix inequality to be satisfied

$$D _ {s} ^ {- 1} A _ {s} - \beta_ {0} A _ {s} ^ {- 1} (L + B) \succ 0. \tag {42}$$

Define $P ^ { \prime } = D _ { s } ^ { - 1 } A _ { s } - \beta _ { 0 } { A _ { s } } ^ { - 1 } \left( L + B \right)$ . Similarly, as above $\dot { V } _ { 2 }$ can be rewritten as

$$
\begin{array}{l} \dot {V} _ {2} \leq - (1 - \theta) \zeta_ {1} ^ {\prime} \| s \| ^ {2} - (1 - \vartheta) \zeta_ {2} ^ {\prime} \| x _ {s} \| ^ {2}, \\ \left(\forall \| s \| \geq \frac {2 \tilde {\lambda}}{\theta \beta_ {0} \zeta_ {1} ^ {\prime}}, \forall \| x _ {s} \| \geq \frac {\zeta_ {3} ^ {\prime} \tilde {\lambda}}{\vartheta \zeta_ {2} ^ {\prime}}\right) \tag {43} \\ \end{array}
$$

where $\zeta _ { 1 } ^ { \prime } = \lambda _ { \operatorname* { m i n } } \Big ( ( A _ { s } - S ) ^ { - 1 } \Big ) , \zeta _ { 2 } ^ { \prime } = \lambda _ { \operatorname* { m i n } } ( P ^ { \prime } )$ and $\zeta _ { 3 } ^ { \prime } =$ $\lambda _ { \operatorname* { m a x } } \left( \left( A _ { s } - S \right) ^ { - 1 } \left( L + B \right) \right)$ . θ and ϑ are two numbers in the interval (0, 1).

It follows from (43) that $\dot { V } _ { 2 } < 0$ , when $\begin{array} { r } { \| s \| \ge \frac { 2 \tilde { \lambda } } { \theta \beta _ { 0 } \zeta _ { ~ 1 } ^ { \prime } } } \end{array}$ and $\begin{array} { r } { \| x _ { s } \| \geq \frac { \zeta _ { \ 3 } ^ { \prime } \tilde { \lambda } } { \vartheta \zeta _ { \ 3 } ^ { \prime } } } \end{array}$ , which shows that the interconnected subsystems (26) and (37) are input-to-state stable.

Since the subsystem (25) and interconnected subsystems (26) and (27) are all input-to-state stable, so long as the controller parameters are properly chosen such that the conditions (33), (35), (40) as well as (42) are satisfied, it is readily shown that the overall closed-loop system (25)-(27) is input-to-state stable. This completes the proof.
