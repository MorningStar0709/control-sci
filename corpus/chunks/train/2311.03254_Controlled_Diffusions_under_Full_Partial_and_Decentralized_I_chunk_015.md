In the following, we adopt the latter approach, as it will be much simpler to be generalized to information structures beyond the fully observed model, including decentralized information structures. This analysis is based on the supporting result in Lemma 2.5, which shows that the Radon-Nikodym derivative is continuous as a function of policies (under a suitable topology over the policy space) . Similar continuity result proved in [12, Lemma 1], using $\bar { L ^ { 2 } }$ convergence of the policies, under the assumption that the information structures are not corrupted by any external noise processes . In [11], the authors established existence of a team optimal policy by applying Girsanov’s change of measure argument . In order to do so, as an intermediate step they prove that the Radon-Nikodym derivative is continuous as a function of policy under the weak\* topology (see [11, Lemma 1]) .

Lemma 2.4. The space UWRC under the weak convergence topology is compact.

Proof. Note that for $m \in \mathfrak { U } _ { \mathsf { W R C } }$ the joint distribution of $( \boldsymbol { X } ^ { \prime } , m )$ is an element of $\mathcal { P } \left( C ( [ 0 , T ] ; \mathbb { R } ^ { d } ) \times \mathcal { P } _ { \lambda } ( [ 0 , T ] \times \mathbb { U } ) \right)$ . Since under any $m \in \mathfrak { U } _ { \mathsf { W R C } }$ the marginal of the joint distribution of $( X ^ { ' } , m )$ on $C ( [ 0 , T ] ; \mathbb { R } ^ { d } )$ is fixed and $\mathcal { P } _ { \lambda } ( [ 0 , T ] \times \mathbb { U } )$ is compact (via Prohorov’s theorem), it follows that the law of the elements of UWRC is tight. Thus relatively compact by Prohorov’s theorem. Let $m ^ { n } \in \mathfrak { U } _ { \mathsf { W R C } }$ be such that $m ^ { n }  m$ . Since the marginal of the joint distribution of $( X ^ { n ^ { \prime } } , m ^ { n } )$ on $C ( [ 0 , T ] ; \mathbb { R } ^ { d } )$ is Wiener measure for all $n \in \mathbb { N } .$ , it follows that the the marginal of the joint distribution of the limit $( \boldsymbol { X } ^ { \prime } , m )$ on $C ( [ 0 , T ] ; \mathbb { R } ^ { d } )$ is also Wiener measure .
