# A. Position Control using Adaptive Backstepping

The position control is achieved via adaptive backstepping control design which involves constructing Lyapunov functions to ensure stability and designing virtual control laws through recursive steps to address underactuation issue. Let $\eta _ { d } = [ x _ { d } , \dot { x } _ { d } , y _ { d } , \dot { y } _ { d } , z _ { d } , \dot { z } _ { d } , \phi _ { d } , \dot { \phi } _ { d } , \theta _ { d } , \dot { \theta } _ { d } , \psi _ { d } , \dot { \psi } _ { d } ] ^ { \top } \in \mathbb { R } ^ { 1 2 }$ be the desired reference trajectories of the system, generated by the admittance controller according to the applied forces of the human guidance. Let the position tracking error be:

$$
E _ {p} = \left[ \begin{array}{l} E _ {p x} \\ E _ {p y} \\ E _ {p z} \end{array} \right] = \left[ \begin{array}{l} x _ {d} - x \\ y _ {d} - y \\ z _ {d} - z \end{array} \right] \in \mathbb {R} ^ {3} \tag {13}
$$

The time derivative of (13) is:

$$
\dot {E} _ {p} = \left[ \begin{array}{l} \dot {E} _ {p x} \\ \dot {E} _ {p y} \\ \dot {E} _ {p z} \end{array} \right] = \left[ \begin{array}{l} \dot {x} _ {d} - \dot {x} \\ \dot {y} _ {d} - \dot {y} \\ \dot {z} _ {d} - \dot {z} \end{array} \right] = \left[ \begin{array}{l} \dot {x} _ {d} - \nu_ {s _ {x}} \\ \dot {y} _ {d} - \nu_ {s _ {y}} \\ \dot {z} _ {d} - \nu_ {s _ {z}} \end{array} \right] \tag {14}
$$

Let ⊙ and $\circled{ e}$ be the elementwise multiplication and exponentiation operators, respectively. Based on (13) and (14), let the following vector of scalar functions, $\begin{array} { r } { V _ { p } = \frac { 1 } { 2 } E _ { p } ^ { \mathcal { C } 2 } } \end{array}$ be Lyapunov function candidates where $V _ { p } = [ V _ { p x } , \bar { V _ { p y } } , \bar { V _ { p z } } ] ^ { \top }$ . Taking the time derivative of $V _ { p }$ , such that:

$$
\dot {V} _ {p} = E _ {p} \odot \dot {E} _ {p} = E _ {p} \odot \left[ \begin{array}{l} \dot {x} _ {d} - \nu_ {s _ {x}} \\ \dot {y} _ {d} - \nu_ {s _ {y}} \\ \dot {z} _ {d} - \nu_ {s _ {z}} \end{array} \right] \tag {15}
$$

Consider $X _ { t e m p } = [ \nu _ { s _ { x d } } , \nu _ { s _ { y d } } , \nu _ { s _ { z d } } ] ^ { \top } \in \mathbb { R } ^ { 3 }$ as a temporary desired control input used to ensure asymptotic stability of the position, such that:

$$X _ {t e m p} = K _ {p} E _ {p} + \dot {X} _ {d} \in \mathbb {R} ^ {3} \tag {16}$$
