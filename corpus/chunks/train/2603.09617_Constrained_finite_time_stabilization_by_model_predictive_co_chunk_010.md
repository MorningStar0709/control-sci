# A. Finite-time MPC with infinite control horizon

The key setting in the finite-time MPC with infinite control horizon is that, summation of stage costs starts from the number of system dimension to infinity, i.e., the lower summation index is $\mathbf { \ddot { \boldsymbol { l } } } = \mathbf { \boldsymbol { n } } ^ { \prime } \mathbf { \dot { \boldsymbol { \mathbf { \mathit { \varepsilon } } } } }$ :

$$J (k) = \sum_ {i = n} ^ {\infty} \left(\| x (i | k) \| _ {Q} ^ {2} + \| u (i | k) \| _ {R} ^ {2}\right). \tag {15}$$

The optimization in finite-time MPC is established by

$$[ U ^ {*} (k), X ^ {*} (k) ] = \arg \min _ {U (k), X (k)} J (k), \tag {16}$$

subject to (6)–(8) for infinite $i = 0 , \ldots , \infty ,$ . The finite-time MPC is implemented by

$$u (k) = [ 1, 0, \dots , \dots ] _ {1 \times \infty} U ^ {*} (k), \tag {17}$$

where the optimal predictive control sequence $U ^ { * } ( k )$ is solved from (16).

Remark 3: In (16), the control horizon is infinity, such that the initial feasibility region is comparable with classic MPC techniques.

$$x ^ {*} (i | k), u ^ {*} (i | k), i = 0, 1, \dots , \infty ,$$

for $i = 0 , 1 , \cdots , \infty .$ , satisfying system dynamics and all constraints.
