Note this implies that if $\bar { \theta } ^ { r } \leq \hat { \theta } ^ { r - 1 }$ , then each state $x ^ { H } \in S ^ { H , r }$ represents a node and capacity state from which there is guaranteed to exist a feasible path back to the depot according to ${ \hat { \theta } } ^ { r }$ . The size of ${ \mathcal { S } } ^ { H , r }$ grows with each additional iteration r.

We define a value function $Q ^ { H , r } ( \cdot )$ over ${ \mathcal { S } } ^ { H , r }$ , where

$$
Q ^ {H, r} \left(x ^ {H}\right) = \left\{ \begin{array}{l l} \max _ {(p, k) \in W ^ {r} \left(x ^ {H}\right)} V _ {n} ^ {H} \left(\mathbf {x} _ {k:} ^ {H, p}\right) & x ^ {H} \in \mathcal {S} ^ {H, r} \\ - \infty & \text { else } \end{array} \right. \tag {24}
$$

where

$$
\begin{array}{l} W ^ {r} (x ^ {H}) = \{(p, k): p \in [ 0, r - 1 ], k \geq 0 \\ \text { with } x _ {k} ^ {H, p} = x; \text { for } x _ {k} ^ {H, p} \in \mathcal {S} ^ {H, r} \} \\ \end{array}
$$

which corresponds a state $x ^ { H } \in \mathbf { x } ^ { H , p }$ with the number of nodes the agent visited after being in state $x ^ { H }$ in iteration $p .$
