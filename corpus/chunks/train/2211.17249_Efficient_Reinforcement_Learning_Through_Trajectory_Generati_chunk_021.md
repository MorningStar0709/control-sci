Note that the matrix $G _ { \theta } \ \in \ \mathbb { R } ^ { ( T m + T _ { 0 } d ) \times ( L - T + 1 ) }$ is not a square matrix and there might be multiple solutions to (16). Moreover, $G _ { \theta }$ may not be full row rank and thus $\left( G _ { \theta } G _ { \theta } ^ { \top } \right)$ may not be invertible. Here, we compute the eigenvalue decomposition of $( G _ { \theta } G _ { \theta } ^ { \top } )$ . Let s be the number of nonzero eigenvalue of $\left( G _ { \theta } G _ { \theta } ^ { \top } \right)$ . Let $\lambda _ { i }$ be the i-th non-zero eigenvalue and $\pmb { p } _ { i }$ be the associated eigenvector of $( G _ { \theta } G _ { \theta } ^ { \top } )$ . Denote $P _ { \pmb \theta } : = \left[ \pmb { p } _ { 1 } \quad \pmb { p } _ { 2 } \quad \cdots \quad \pmb { p } _ { s } \right]$ and $\pmb { \Lambda } = \mathrm { d i a g } ( \lambda _ { 1 } , \lambda _ { 2 } , \dotsb , \lambda _ { s } )$ . Then clearly $\left( G _ { \theta } G _ { \theta } ^ { \top } \right) = \dot { P _ { \theta } } \Lambda P _ { \theta } ^ { \top }$ and we compute the solution of (16) given by

$$\boldsymbol {g} ^ {*} = \boldsymbol {G} _ {\boldsymbol {\theta}} ^ {\top} \boldsymbol {P} _ {\boldsymbol {\theta}} \boldsymbol {\Lambda} ^ {- 1} \boldsymbol {P} _ {\boldsymbol {\theta}} ^ {\top} [ \boldsymbol {w} (T _ {0} - 1); \dots ; \boldsymbol {w} (T - 1); \mathcal {X} (T _ {0} - 1) ]. \tag {17}$$

Next, we prove the existence and uniqueness of the trajectory generated by (16). The goal is to show that given $( \pmb { w } ( T _ { 0 } - 1 ) , \cdots , \pmb { w } ( T - 1 ) , \pmb { \chi } ( T _ { 0 } - 1 ) )$ , any g that satisfies (16) will generate the same trajectory using $\mathcal { H } g$ . So it is suffice to choose the closed-form solution in (17).
