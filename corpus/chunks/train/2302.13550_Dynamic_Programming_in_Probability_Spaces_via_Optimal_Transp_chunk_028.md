$$
\begin{array}{l} j _ {N} (x _ {N}, r _ {N}) = g _ {N} (x _ {N}, r _ {N}); \\ j _ {k} (x _ {k}, r _ {N}) = \inf _ {u _ {k} \in U _ {k}} \mathbb {E} ^ {w _ {k} \sim \xi_ {k}} \left[ g _ {k} (x _ {k}, u _ {k}, w _ {k}) + j _ {k + 1} (f _ {k} (x _ {k}, u _ {k}, w _ {k}), r _ {N}) \right], \end{array} \tag {5.1}
$$

where $\xi _ { k } \in \mathcal { P } ( W _ { k } )$ is the probability measure of the noise, and $W _ { k }$ is the space of possible realizations. Since $j _ { k }$ is of the form required for Corollary 4.2, it is tempting to extend our results. Unfortunately, the noisy drift may favor a different allocation of the particles, and the expectation annihilates such an effect.

Counterexample 5.4 (Corollary 4.2 does not readily extend). Consider a horizon $N = 2$ and the setting depicted in Figure 2c. Let $X _ { k } = R _ { k } = U _ { k } = \{ \pm 1 , \pm 2 \}$ , and consider uniformly distributed noise over $W _ { k } = \{ 1 , 2 \}$ . The particle dynamics is $f _ { k } ( x _ { k } , u _ { k } , w _ { k } ) = w _ { k }$ if $x _ { k } < 0$ and $f _ { k } ( x _ { k } , u _ { k } , w _ { k } ) = u _ { k }$ otherwise. The stage cost is $\kappa [ g _ { k } ]$ , where $g _ { k } ( x _ { k } , - x _ { k } , w _ { k } ) = 2 { \mathrm { ~ i f ~ } } x _ { k } > 0$ , and 0 otherwise. The terminal cost enforces the configuration $\rho _ { N } = { \textstyle { \frac { 1 } { 2 } } } \delta _ { - 2 } + { \textstyle { \frac { 1 } { 2 } } } \delta _ { - 1 }$ , namely $\kappa [ g _ { N } ]$ ] with $g _ { N } ( x _ { N } , r _ { N } ) = 0$ if $x _ { N } = r _ { N }$ and +∞ otherwise. The recursion (5.1) yields $j _ { 0 } ( x _ { 0 } , r _ { N } ) = 1$ for $x _ { 0 } , r _ { N } < 0$ (any input at the first stage and $u _ { 1 } = r _ { N }$ at the second stage). Therefore, with the initial configuration and target configuration $\mu _ { 0 } = \rho _ { N } = \textstyle { \frac { 1 } { 2 } } \left( \delta _ { - 2 } + \delta _ { - 1 } \right)$ , Corollary 4.2 yields $\tilde { J } _ { 0 } ( \mu _ { 0 } , \rho _ { N } ) = \mathcal { K } [ j _ { 0 } ] ( \mu _ { 0 } , \rho _ { N } ) = 1$ . Instead, the DPA in the probability space gives $\mu _ { 1 } = { \textstyle \frac { 1 } { 2 } } ( \delta _ { + 1 } + \delta _ { + 2 } )$ with zero cost (regardless of the input). Then, the evolution is deterministic and the cost-to-go amounts to $j _ { 1 } ( x _ { 1 } , r _ { N } ) = 0$ unless $r _ { N } = - x _ { 1 }$ . Thus, Corollary 4.2 applies and yields $J _ { 1 } ( \mu _ { 1 } , \rho _ { N } ) = \dot { K } [ j _ { 1 } ] ( \mu _ { 1 } , \rho _ { N } ) = \textstyle { \frac { 1 } { 2 } } j _ { 1 } ( + 1 , - 2 ) +$ $\textstyle { \frac { 1 } { 7 } } j _ { 1 } ( + 2 , - 1 ) = 0$ . Overall, $J _ { 0 } ( \mu _ { 0 } , \rho _ { N } ) \le 0 + J _ { 1 } ( \mu _ { 1 } , \rho _ { N } ) = 0 < 1 = \tilde { J } _ { 0 } ( \mu _ { 0 } , \rho _ { N } )$ . Thus, the naive application of Corollary 4.2 is suboptimal.
