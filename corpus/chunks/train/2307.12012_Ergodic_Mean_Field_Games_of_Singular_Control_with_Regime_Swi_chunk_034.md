for $\tilde { \eta } : =$ inf $\left\{ t > 0 : Z _ { t } = \alpha _ { i } ( \theta ) \right\}$ , by comparison principle (a slightly version of Proposition B.2) we also have $X _ { t } ^ { \hat { \beta } _ { i } ( \theta ) , \xi ^ { * } ( \theta ) } \leq Z _ { t }$ , for any $t \in [ 0 , \tau \wedge \tilde { \eta } ) , \ \mathbb { P } _ { i } { \mathrm { - } } { \mathrm { a . s . } }$ ., which in turn implies

$$\mathbb {P} _ {(\beta_ {i} (\theta), i)} \big (\eta (\alpha_ {i} (\theta)) < \tau \big) \geq \mathbb {P} _ {(\beta_ {i} (\theta), i)} \big (\tilde {\eta} < \tau \big).$$

Therefore, we find

$$\mathbb {P} _ {(\beta_ {i} (\theta), i)} \big (\eta (\alpha_ {i} (\theta)) < \tau \big) \geq \mathbb {P} _ {(\beta_ {i} (\theta), i)} \big (\tilde {\eta} < \tau \big) = \int_ {0} ^ {\infty} \mathbb {E} [ \mathbf {1} _ {\{\tilde {\eta} < t \}} ] F _ {\tau} (d t) := \rho > 0, \tag {4.7}$$

where $F _ { \tau }$ denotes the distribution function of τ and the last inequality follows from the fact that the SDE (4.6) induces a regular diffusion (cf. Assumption 2.1).

Using (4.7), we have

$$\mathbb {E} _ {(x, i)} \big [ \eta (\alpha_ {i} (\theta)) \big ] = \mathbb {E} _ {(x, i)} \big [ \eta (\alpha_ {i} (\theta)) \mathbf {1} _ {\{\eta (\alpha_ {i} (\theta)) < \tau \}} \big ] + \mathbb {E} _ {(x, i)} \big [ \eta (\alpha_ {i} (\theta)) \mathbf {1} _ {\{\eta (\alpha_ {i} (\theta)) \geq \tau \}} \big ] \tag {4.8}\leq \mathbb {E} _ {i} [ \tau ] + \mathbb {E} _ {(x, i)} \big [ \mathbb {E} _ {(X _ {\tau} ^ {\xi^ {*} (\theta)}, Y _ {\tau})} \big [ \eta (\alpha_ {i} (\theta)) (1 - \mathbf {1} _ {\{\eta (\alpha_ {i} (\theta)) < \tau \}}) \big ] \big ]= \mathbb {E} _ {i} [ \tau ] + \mathbb {E} _ {(x, i)} [ (1 - \mathbf {1} _ {\{\eta (\alpha_ {i} (\theta)) < \tau \}}) ] \mathbb {E} _ {(x, i)} [ \mathbb {E} _ {(X _ {\tau} ^ {\xi^ {*} (\theta)}, Y _ {\tau})} [ \eta (\alpha_ {i} (\theta)) ] ]= \mathbb {E} _ {i} [ \tau ] + (1 - \mathbb {P} _ {(x, i)} (\eta (\alpha_ {i} (\theta)) < \tau)) \mathbb {E} _ {(x, i)} [ \eta (\alpha_ {i} (\theta)) ],\leq \mathbb {E} _ {i} [ \tau ] + (1 - \rho) \mathbb {E} _ {(x, i)} [ \eta (\alpha_ {i} (\theta)) ],$$

where in the second equality we use the strong Markov property of $( X _ { t } ^ { \xi ^ { * } ( \theta ) } , Y _ { t } ) _ { t \geq 0 }$ . Then, since (4.8) holds for any $( x , i ) \in \mathcal { O }$ , we obtain

$$\sup _ {(x, i) \in \mathcal {O}} \mathbb {E} _ {(x, i)} [ \eta (\alpha_ {i} (\theta)) ] \leq \max _ {i \in \mathbb {Y}} \mathbb {E} _ {i} [ \tau ] + (1 - \rho) \sup _ {(x, i) \in \mathcal {O}} \mathbb {E} _ {(x, i)} [ \eta (\alpha_ {i} (\theta)) ].$$
