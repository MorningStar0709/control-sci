Proof of Theorem 2. Recall that for any matrix ${ \textbf { \textit { A } } } \in$ $\mathbb { C } ^ { d \times \dot { d } }$ the trace norm $\| A \| _ { 1 }$ and Hilbert-Schmidt norm√ $\| A \|$ satisfy the inequality $\| A \| _ { 1 } \leq { \sqrt { d } } \| A \|$ . By assumption of the theorem, the error dynamics of the observer are exponentially convergent. Therefore $\epsilon ( t ) \triangleq \| \varrho ( t ) -$ $\hat { \varrho } ( t ) \vert \vert _ { 1 } \le \sqrt { d } M e ^ { - \sigma t }$ . Accordingly, for $\begin{array} { r } { T = { \frac { \ln \left( e M { \sqrt { d } } \right) } { \sigma } } } \end{array}$ the inequality $\sqrt { d } M e ^ { - \sigma t } \leq 1 / e$ holds true for all $t \geq T$ . Past that point in time, Fannes’ inequality holds:

$$| S (\boldsymbol {\varrho}) - S (\hat {\boldsymbol {\varrho}}) | \leq \epsilon (t) \log (d) - \epsilon (t) \log (\epsilon (t)).$$

Note that the function $g ( x ) \ { \stackrel { \triangle } { = } } \ - x \log x$ is increasing on $( 0 , \frac { 1 } { e } )$ . Thus the convergence hypothesis pairs with Fannes’ inequality to give

$$
\begin{array}{l} | S (\boldsymbol {\varrho}) - S (\hat {\boldsymbol {\varrho}}) | \leq \log (d) \sqrt {d} M e ^ {- \sigma t} \\ - \sqrt {d} M e ^ {- \sigma t} \log \left(\sqrt {d} M e ^ {- \sigma t}\right) \\ = \log (d) \sqrt {d} M e ^ {- \sigma t} \\ - \sqrt {d} M \left(\log \Bigl (\sqrt {d} M \Bigr) - \sigma t\right) e ^ {- \sigma t}. \\ \end{array}
$$

Using Lemma 2,

$$
\begin{array}{l} | S (\boldsymbol {\varrho}) - S (\hat {\boldsymbol {\varrho}}) | \leq \log (d) \sqrt {d} M e ^ {- \sigma t} \\ - \sqrt {d} M \log (\sqrt {d} M) e ^ {- \sigma t} + \sqrt {d} M e ^ {- a \sigma t}. \\ \end{array}
$$

This proves the theorem.
