# 2.2. Assumptions and examples. The instantaneous profit function $\pi$ fulfills the following conditions.

Assumption 2.2. The function $\pi : \mathbb { R } \times \mathbb { R } _ { + } \mapsto \mathbb { R } _ { + }$ is such that:

(1) $\pi ( \cdot , \theta ) \in C ^ { 2 } ( { \mathcal { T } } )$ , for any $\theta \in \mathbb { R } _ { + } ,$ ;   
(2) $\pi ( \cdot , \theta )$ is non-decreasing and concave, for any $\theta \in \mathbb { R } _ { + }$ ;   
(3) $\pi _ { x \theta }$ is continuous and it is such that $\pi _ { x \theta } ( x , \theta ) < 0 ,$ , for any $( x , \theta ) \in \mathcal { T } \times \mathbb { R } _ { + }$ ;   
(4) for every $( i , \theta ) \in \mathbb { Y } \times \mathbb { R } _ { + }$ ,

$$\lim _ {x \downarrow \underline {{x}}} \widehat {\mathbb {E}} _ {(x, i)} \left[ \int_ {0} ^ {\infty} e ^ {\int_ {0} ^ {t} b _ {x} (\widehat {X} _ {s}, Y _ {s}) d s} \pi_ {x} (\widehat {X} _ {t}, \theta) d t \right] = \infty ,$$

and,

$$\lim _ {x \uparrow \overline {{x}}} \widehat {\mathbb {E}} _ {(x, i)} \left[ \int_ {0} ^ {\infty} e ^ {\int_ {0} ^ {t} b _ {x} (\widehat {X} _ {s}, Y _ {s}) d s} \pi_ {x} (\widehat {X} _ {t}, \theta) d t \right] = 0;$$

(5) for every $( x , i , \theta ) \in \mathcal { O } \times \mathbb { R } _ { + }$ ,

$$\widehat {\mathbb {E}} _ {(x, i)} \left[ \int_ {0} ^ {\infty} e ^ {- c t} | \pi_ {x} (\widehat {X} _ {t}, \theta) | d t \right] < \infty ,$$

and, for some $\epsilon _ { 0 } : = \epsilon _ { 0 } ( x ) \in ( 0 , 1 )$ ,

$$
\begin{array}{l} \widehat {\mathbb {E}} \left[ \int_ {0} ^ {\infty} e ^ {- c t} \sup _ {x ^ {\prime} \in \mathcal {I} _ {x} ^ {\epsilon_ {0}}} \left(| \pi_ {x x} (\widehat {X} _ {t} ^ {x ^ {\prime}}, \theta) | \partial_ {x} \widehat {X} _ {s} ^ {x} | _ {x = x ^ {\prime}}\right) d t \right] + \\ + \widehat {\mathbb {E}} \bigg [ \int_ {0} ^ {\infty} e ^ {- c t} | \pi_ {x} (\widehat {X} _ {t} ^ {x}, \theta) | \sup _ {x ^ {\prime} \in \mathcal {I} _ {x} ^ {\epsilon_ {0}}} \bigg (\int_ {0} ^ {t} | b _ {x x} (\widehat {X} _ {s} ^ {x ^ {\prime}}, Y _ {s} ^ {i}) | \partial_ {x} \widehat {X} _ {s} ^ {x} | _ {x = x ^ {\prime}} d s \bigg) d t \bigg ] <   \infty , \\ \end{array}
$$

where $\mathcal { T } _ { x } ^ { \epsilon _ { 0 } } : = ( x , x + \varepsilon _ { 0 } ) , c > 0$ is the same constant as in Assumption $2 . I - ( 3 )$ and $\partial _ { x } \widehat { X } _ { t } ^ { x } : =$

$$\exp \big (\int_ {0} ^ {t} \big (b _ {x} + \partial_ {x} (\sigma \sigma_ {x}) - \frac {1}{2} \sigma_ {x} ^ {2} \big) (\widehat {X} _ {s} ^ {x}, Y _ {s}) d s + \int_ {0} ^ {t} \sigma_ {x} (\widehat {X} _ {s} ^ {x}, Y _ {s}) d \widehat {W _ {s}} \big);$$
