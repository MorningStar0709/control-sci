# 2.2 Dissipativity

We model the control performance requirement with the notion of dissipativity. Under assumptions for existence and uniqueness of solutions, the system

$$\dot {\boldsymbol {x}} (t) = f (\boldsymbol {x} (t), \boldsymbol {d} (t)) \quad \boldsymbol {e} (t) = g (\boldsymbol {x} (t), \boldsymbol {d} (t)), \tag {4}$$

where x $\in \mathbb { R } ^ { n }$ is dissipative with respect to a supply rate $s : \mathbb { R } ^ { n _ { d } } \times \mathbb { R } ^ { n _ { e } } \to$ R if there exists a nonnegative storage function $S : \mathbb { R } ^ { n }  \mathbb { R } _ { \geq 0 }$ such that

$$S (\boldsymbol {x} (T)) - S (\boldsymbol {x} (0)) \leq \int_ {0} ^ {T} s (\boldsymbol {d} (t), \boldsymbol {e} (t)) d t$$

for any $T \geq 0$ and trajectory $( x , d , e )$ that satisfies the dynamics in (4). A thorough presentation of dissipativity can be found in [26] and [27].

In particular, we consider a quadratic supply rate

$$
s (d, e) = \left[ \begin{array}{c} d \\ e \end{array} \right] ^ {\top} X \left[ \begin{array}{c} d \\ e \end{array} \right] \tag {5}
$$

where

$$
X = \left[ \begin{array}{c c} X _ {d d} & X _ {d e} \\ X _ {d e} ^ {\top} & X _ {e e} \end{array} \right]
$$

and a quadratic storage function

$$S (x) = x ^ {\top} P x \tag {6}$$

where $P \succeq 0$

Important classes of quadratic supply rates include $s ( \bar { d } , e ) = 0$ , which corresponds to stability (when we also impose $P \succ 0 ) , s ( \dot { d , } e ) = \gamma ^ { 2 } \| d \| ^ { 2 } - \| e \| ^ { 2 }$ , which corresponds an $L _ { 2 }$ gain bound of $\gamma ,$ and $\boldsymbol { s } ( \boldsymbol { \ddot { d } } , \boldsymbol { \ddot { e } } ) = \boldsymbol { d } ^ { \top } \boldsymbol { e } ,$ which corresponds to passivity. Note that, when the system described by f and g is LTI, there is no loss in assuming a quadratic storage function [27].
