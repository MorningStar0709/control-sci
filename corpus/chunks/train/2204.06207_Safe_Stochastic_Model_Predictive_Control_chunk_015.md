# C. Stability

In MPC, stability is often proved by showing that the value function is decreasing for subsequent time steps, also known as the descent property. These proofs are based on the MPC idea of a moving horizon, where the previously planned input sequence remains valid and only one additional input element is added to the input sequence for the next time step. For the proposed safe SMPC algorithm, however, this assumption does not hold. Since switching between different modes is possible, the predicted input and state trajectories may vary at each time step. We tackle this challenge by using an ISS result from [12, Lemma 22], which we repeat in the following lemma.

Lemma 1 (Lipschitz ISS Lyapunov Function [12]): Let $f : \mathcal { X } _ { 0 } \times \mathcal { W }  \mathbb { R } ^ { n _ { x } }$ be Lipschitz continuous on $\mathcal { X } _ { 0 } \times \mathcal { W }$ . Let $\mathcal { X } _ { 0 } \in \mathbb { R } ^ { n _ { x } }$ contain the origin and be a robust positively invariant set for the function ${ f } ( { \pmb x } , { \pmb w } )$ . Let there exist a Lipschitz continuous function $V : \mathcal { X } _ { 0 }  \mathbb { R } _ { \ge 0 }$ such that for all $\pmb { x } \in \mathcal { X } _ { 0 }$

$$\alpha_ {1} (| | \boldsymbol {x} | |) \leq V (\boldsymbol {x}) \leq \alpha_ {2} (| | \boldsymbol {x} | |) \tag {10a}V (\boldsymbol {f} (\boldsymbol {x}, \mathbf {0})) - V (\boldsymbol {x}) \leq - \alpha_ {3} (| | \boldsymbol {x} | |) \tag {10b}$$

with functions $\alpha _ { 1 } , \alpha _ { 2 } , \alpha _ { 3 } \in \mathcal { K } _ { \infty }$ . Then, V is an ISS Lyapunov function and the origin is ISS for system ${ f } ( { \pmb x } , { \pmb w } )$ with region of attraction $\mathcal { X } _ { 0 }$ .

Lemma 1 ensures that the origin of a system subject to uncertainty is ISS if the undisturbed system is asymptotically stable and the system is Lipschitz continuous with respect to state x and uncertainty w.
