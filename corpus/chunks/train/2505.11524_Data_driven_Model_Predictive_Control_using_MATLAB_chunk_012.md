Using $\mathbf { Y } _ { k + 1 } , \mathbf { U } _ { k }$ and defining $\mathbf { Q } _ { \mathbf { Y } } \in \mathbb { R } ^ { p \mathrm { N } \times p \mathrm { N } } , \mathbf { R } _ { \mathbf { U } } \in \mathbb { R } ^ { m \mathrm { N } \times m \mathrm { N } } , \mathbf { Y } _ { \mathrm { r } } \in \mathbb { R } ^ { p \mathrm { N } } , \mathbf { U } _ { \mathrm { r } } \in \mathbb { R } ^ { m \mathrm { N } }$ as follows:

$$
\mathbf {Q} _ {\mathbf {Y}} = \left[ \begin{array}{c c c} \mathbf {Q} _ {\mathbf {y} _ {k + 1 | k}} & \dots & \mathbf {0} \\ \vdots & & \vdots \\ \mathbf {0} & \dots & \mathbf {Q} _ {\mathbf {y} _ {k + N | k}} \end{array} \right] \quad \mathbf {R} _ {\mathbf {U}} = \left[ \begin{array}{c c c} \mathbf {R} _ {\mathbf {u} _ {k | k}} & \dots & \mathbf {0} \\ \vdots & & \vdots \\ \mathbf {0} & \dots & \mathbf {R} _ {\mathbf {u} _ {k + N - 1 | k}} \end{array} \right] \quad \mathbf {Y} _ {\mathrm{r}} = \left[ \begin{array}{c} \mathbf {y} _ {\mathrm{r}} \\ \vdots \\ \mathbf {y} _ {\mathrm{r}} \end{array} \right] \quad \mathbf {U} _ {\mathrm{r}} = \left[ \begin{array}{c} \mathbf {u} _ {\mathrm{r}} \\ \vdots \\ \mathbf {u} _ {\mathrm{r}} \end{array} \right] \tag {13}
$$

the MPC optimization problem in Eq. (12) can be rewritten as:

$$\inf _ {\mathbf {U} _ {k}, \mathbf {Y} _ {k + 1}} \left[ \mathbf {Y} _ {r} - \mathbf {Y} _ {k + 1} \right] ^ {\top} \mathbf {Q} _ {\mathbf {Y}} \left[ \mathbf {Y} _ {r} - \mathbf {Y} _ {k + 1} \right] + \left[ \mathbf {U} _ {r} - \mathbf {U} _ {k} \right] ^ {\top} \mathbf {R} _ {\mathbf {U}} \left[ \mathbf {U} _ {r} - \mathbf {U} _ {k} \right]$$

$\mathrm { s u b j e c t ~ t o ~ } \mathbf { Y } _ { k + 1 } \in \mathbb { Y } ^ { \mathrm { N } } , ~ \mathbf { U } _ { k } \in \mathbb { U } ^ { \mathrm { N } }$ (14)

$$\mathbf {Y} _ {k + 1} = \mathbf {A} _ {\mathbf {Y}} \mathbf {x} _ {k | k} + \mathbf {B} _ {\mathbf {Y}} \mathbf {U} _ {k}.$$

Substituting $\mathbf { Y } _ { k + 1 }$ from Eq. (9) in Eq. (14), the MPC optimization problem can be represented with only $\mathbf { U } _ { k }$ as the decision variable:

$$\inf _ {\mathbf {U} _ {k}} \left[ \mathbf {Y} _ {r} - \left(\mathbf {A} _ {\mathbf {Y}} \mathbf {x} _ {k | k} + \mathbf {B} _ {\mathbf {Y}} \mathbf {U} _ {k}\right) \right] ^ {\top} \mathbf {Q} _ {\mathbf {Y}} \left[ \mathbf {Y} _ {r} - \left(\mathbf {A} _ {\mathbf {Y}} \mathbf {x} _ {k | k} + \mathbf {B} _ {\mathbf {Y}} \mathbf {U} _ {k}\right) \right] + \left[ \mathbf {U} _ {r} - \mathbf {U} _ {k} \right] ^ {\top} \mathbf {R} _ {\mathbf {U}} \left[ \mathbf {U} _ {r} - \mathbf {U} _ {k} \right] \tag {15}$$
