# A Priori Choice of $\pmb { n } _ { A }$

Two cases can be distinguished:

1. Industrial plant (temperature, control, flow rate, concentration, and so on). For this type of plant in general:

$$n _ {A} \leq 3$$

and the value $n _ { A } = 2$ , which is very typical, is a good starting value to choose.

2. Electromechanical systems. $n _ { A }$ results from the structural analysis of the system.

Example Flexible Robot Arm with two vibration modes. In this case, $n _ { A } = 4$ is chosen, since a second-order is required, to model a vibratory mode.
