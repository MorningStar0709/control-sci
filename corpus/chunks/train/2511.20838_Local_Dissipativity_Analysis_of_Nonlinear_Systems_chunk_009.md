Sufficient conditions for global QSR dissipativity of smooth, nonlinear, control affine systems were formatted in [11, Theorem 1], [28, Theorem 2]. Local dissipativity can be established instead by applying the proof strategy from [28, Theorem 2] while restricting the initial state to a set, $x _ { 0 } \in \Omega$ , that is robustly positive invariant to inputs in $u \in \mathcal { U } _ { \Omega } \ne \emptyset$ and using a more general storage function. The resulting conditions are summarized next, providing a MI that mirrors the KYP Lemma.

Theorem 3 [11, Theorem 1], [28, Theorem 2] Suppose that for some $\mathbf { \mathop { \Omega } } ^ { \mathbf { \Sigma } } \subset \mathbb { R } ^ { n }$ and dynamics defined by $( 1 ) , \mathcal { U } _ { \Omega }$ contains a neighborhood of the origin. If there exists a non-negative, smooth function, $V : \bar { \Omega } \to \bar { \mathbb { R } } _ { + }$ , that satisfies

$$
\left[ \begin{array}{c c c} \nabla V ^ {\top} f (x) & \frac {1}{2} \nabla V ^ {\top} \bar {G} (x) - h (x) ^ {\top} \mathbf {S} & h (x) ^ {\top} \\ * & \hat {\mathbf {R}} & \bar {J} (x) ^ {\top} \\ * & * & \mathbf {Q} ^ {- 1} \end{array} \right] \preceq 0, \quad (5)
$$

where $\hat { \mathbf { R } } = - \mathbf { R } - \mathbf { S } ^ { \top } \bar { J } ( x ) - \bar { J } ( x ) ^ { \top } \mathbf { S }$ , then $\mathcal { G } _ { \Omega }$ is locally dissipative on Ω with respect to (3).

PROOF. First note that for each $x _ { 0 } \in \Omega , \mathcal { U } _ { \Omega } ( x _ { 0 } )$ contains a neighborhood of the origin because $\mathcal { U } _ { \Omega } \subseteq \mathcal { U } _ { \Omega } ( x _ { 0 } )$ . For u $\in \mathcal { U } _ { \Omega } ( x _ { 0 } )$ , we can follow the steps proving [11, Theorem 1] with ${ \dot { V } } ( x ) = \phi ( x )$ up to [11, Equation 6], which verifies (4), and consequently local QSR dissipativity.
