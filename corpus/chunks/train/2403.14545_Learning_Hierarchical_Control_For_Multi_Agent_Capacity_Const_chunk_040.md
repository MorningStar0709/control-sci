# A. Agent

We consider a single agent $( M = 1 )$ and a task graph containing $| V | = 7$ nodes, as depicted in Fig. 2. At time step t of iteration r, the agent is modeled with state

$$x _ {t} ^ {r} = (z _ {t} ^ {r}, y _ {t} ^ {r}, \theta_ {t} ^ {r}, v _ {t} ^ {r}, s o c _ {t} ^ {r}, \tau_ {t} ^ {r}), \tag {57}$$

where $\boldsymbol { z } _ { t } ^ { r }$ and $y _ { t } ^ { r }$ denote the agent’s position, ${ \theta } _ { t } ^ { r }$ the steering angle, $v _ { t } ^ { r }$ the velocity, $s o c _ { t } ^ { r }$ the battery state of charge, and $\boldsymbol { \tau } _ { t } ^ { r }$ how much time has passed since the agent left the depot node $\mathcal { N } _ { 1 }$ at the beginning of the iteration. The agent state evolves according to the kinematic bicycle model

$$z _ {t + 1} ^ {r} = z _ {t} ^ {r} + v _ {t} ^ {r} \cos \theta_ {t} ^ {r} d t \tag {58a}y _ {t + 1} ^ {r} = y _ {t} ^ {r} + v _ {t} ^ {r} \sin \theta_ {t} ^ {r} d t \tag {58b}\theta_ {t + 1} ^ {r} = \theta_ {t} ^ {r} + \delta_ {t} ^ {r} d t \tag {58c}v _ {t + 1} ^ {r} = v _ {t} ^ {r} + a _ {t} ^ {r} d t \tag {58d}s o c _ {t + 1} ^ {r} = s o c _ {t} ^ {r} + \alpha v _ {t} ^ {r} d t \tag {58e}\tau_ {t + 1} ^ {r} = \tau_ {t} ^ {r} + d t, \tag {58f}$$

where the parameter α determines the battery discharge rate; here we use $\alpha = 1 . 6$ The two actuation inputs are the steering rate $\delta _ { t } ^ { r }$ and acceleration $a _ { t } ^ { r }$ . We use a time step of $d t = 0 . 1$ . The agent states and inputs are constrained as

$$\theta_ {t} ^ {r} \in [ - \pi , \pi ] \tag {59a}v _ {t} ^ {r} \in [ 0, 5 ] \tag {59b}s o c _ {t} ^ {r} \in [ 0, 1 0 0 ] \tag {59c}\tau_ {t} ^ {r} \in [ 0, 1 2 0 ] \tag {59d}\delta_ {t} ^ {r} \in [ - 2, 2 ] \tag {59e}a _ {t} ^ {r} \in [ - 2, 2 ]. \tag {59f}$$

There are two capacity states: the state of charge of the agent and how much time has passed since the beginning of the iteration

$$c _ {t} ^ {r} = (s o c _ {t} ^ {r}, \tau_ {t} ^ {r}).$$

At the beginning of each iteration r, the agent is located at the depot node $\mathcal { N } _ { 1 }$ and each capacity state is initialized at 0, i.e. $c _ { 0 } ^ { r } = ( 0 , 0 ) , \ x _ { 0 } ^ { r } = ( 1 , 0 , 0 )$ . The capacity state constraints indicate the agent must visit as many nodes as possible without depleting its battery and return to the depot by the allotted latest time $T _ { m a x } = 1 2 0$ . Again, we model the battery depletion as increasing along the route.
