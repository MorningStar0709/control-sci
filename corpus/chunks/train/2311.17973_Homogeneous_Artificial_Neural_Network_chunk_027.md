$$\nu_ {\epsilon} = \frac {1}{M _ {\delta}} \int_ {K _ {\delta}} \frac {1}{\ln \| x \| _ {\mathbf {d}}} \ln \frac {h (x) + o (x , \epsilon)}{h (\pi_ {\mathbf {d}} (x))) + o (\pi_ {\mathbf {d}} (x)) , \epsilon)} d x,$$

where su $\mathrm { p } _ { y \in \Omega } | o ( y , \epsilon ) | \le \epsilon$ (see (10)). Since h is assumed to be nonzero and $\pi _ { \mathbf { d } } ( x ) = \mathbf { d } ( - \ln \| x \| _ { \mathbf { d } } ) x \in S$ then $\begin{array} { r } { | h ( \pi _ { \mathbf { d } } ( x ) ) | \ge \underline { { h } } = \operatorname* { i n f } _ { y \in S \cap K _ { \delta } } h ( y ) > 0 } \end{array}$ . This implies that

$$\frac {1}{\| x \| _ {\mathbf {d}}} \ln \frac {g _ {\epsilon} (x)}{g _ {\epsilon} (\pi_ {\mathbf {d}} (x))} \rightarrow \frac {1}{\| x \| _ {\mathbf {d}}} \ln \frac {h (x)}{h (\pi_ {\mathbf {d}} (x))} \quad \text {as} \quad \epsilon \rightarrow 0 \tag {25}$$

uniformly on any compact from $\Omega \cap K _ { \delta } \backslash \{ \mathbf { 0 } \}$ , so $\nu _ { \epsilon } \to \nu$ as $\epsilon \to 0$ .

The practical homogeneity degree is still well-defined by the formula (22) provided that the practical “necessary” condition of homogeneity (21) is fulfilled. The formula (22) defines the practical homogeneity degree as a mean value of some function. This value can also be approximated as follows

$$\nu_ {\epsilon} \approx \frac {1}{M} \sum_ {i = 1} ^ {M} \frac {1}{\ln \| x _ {i} \| _ {\mathbf {d}}} \ln \frac {g _ {\epsilon} (x _ {i})}{g _ {\epsilon} (\pi_ {\mathbf {d}} (x _ {i}))}, \tag {26}$$

where the points $x _ { i } \in K _ { \delta } \cap \Omega$ are assumed to be uniformly distributed in $K _ { \delta } \cap \Omega$ and $M \in \mathbb { N }$ is a large enough number.
