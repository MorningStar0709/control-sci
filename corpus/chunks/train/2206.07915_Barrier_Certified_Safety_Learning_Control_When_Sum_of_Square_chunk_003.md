# II. PRELIMINARY

A discrete-time nonlinear control-affine system with state $s _ { t } , s _ { t + 1 } \in S$ and control action $a _ { t } \in A$ is considered as:

$$s _ {t + 1} = f (s _ {t}) + g (s _ {t}) a _ {t} + d (s _ {t}), \forall t \in [ \underline {{t}}, \overline {{t}} ], \tag {1}$$

where $s _ { t } , s _ { t + 1 }$ are finished within a bounded time t, while t and t are constants.

Notation: From the infinite-horizon discounted Markov decision process (MDP) theory [28], let $r : \mathcal { S } \times \mathcal { A }  \mathcal { R }$ denote the reward of the current state-action pair $\left( { { s _ { t } } , { a _ { t } } } \right)$ in (1), let $P ( s _ { t } , s _ { t + 1 } , a _ { t } ) = p ( s _ { t + 1 } | s _ { t } , a _ { t } )$ denote the probability distribution to next state $s _ { t + 1 }$ , and let $\gamma \in ( 0 , 1 )$ denote a discount factor. Thus, a MDP tuple $\tau = ( s _ { t } , s _ { t + 1 } , a _ { t } , r _ { t } , P _ { t } , \gamma )$ establishes at each state t.
