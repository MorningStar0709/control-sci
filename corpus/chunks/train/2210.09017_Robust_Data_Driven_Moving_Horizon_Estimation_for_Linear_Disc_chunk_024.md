# B. Nonlinear System

From here on, we consider the original nonlinear system, introduced in [37]

$$\dot {x} _ {1} (t) = - \frac {a _ {1}}{A _ {1}} \sqrt {2 g x _ {1} (t)} + \frac {a _ {3}}{A _ {1}} \sqrt {2 g x _ {3} (t)} + \frac {\gamma_ {1}}{A _ {1}} u _ {1} (t) \tag {18a}\dot {x} _ {2} (t) = - \frac {a _ {2}}{A _ {2}} \sqrt {2 g x _ {2} (t)} + \frac {a _ {4}}{A _ {2}} \sqrt {2 g x _ {4} (t)} + \frac {\gamma_ {2}}{A _ {2}} u _ {2} (t) \tag {18b}\dot {x} _ {3} (t) = - \frac {a _ {3}}{A _ {3}} \sqrt {2 g x _ {3} (t)} + \frac {1 - \gamma_ {2}}{A _ {3}} u _ {2} (t) \tag {18c}\dot {x} _ {4} (t) = - \frac {a _ {4}}{A _ {4}} \sqrt {2 g x _ {4} (t)} + \frac {1 - \gamma_ {1}}{A _ {4}} u _ {1} (t). \tag {18d}$$

The numerical parameter values are given in [37]. Obviously, for this nonlinear case the derived robust stability guarantees do not hold anymore.

In the offline phase, we apply a persistently exciting input to the system which is sampled from a uniform distribution $\mathcal { U } ( 0 , 1 )$ (alternatively (0, 10)) and we collect $N ~ = ~ 1 0 0$ samples from system (18). The measurement noise in the offline phase corrupting the output and state measurements follows a truncated (at ±0.003) normal distribution with $\mu _ { \mathcal { N } } =$ 0 and $\sigma = 0 . 0 0 1$ . Once the data collection is completed, we switch to the online phase (in which we once again apply a sinusoidal signal).

The model-based and the data-driven MHE schemes are applied for the same parameters for the true system dynamics and the prior estimate, i.e., ρ, X, P , R1 (alternatively R2), L, $x ( 0 ) , { \hat { x } } ( 0 )$ remain the same, compared to Section IV-A. Regarding the implementation of the data-driven MHE scheme, the numerical values of $c _ { \alpha } , c _ { \sigma } , \overline { { \varepsilon } } _ { x } , \overline { { \varepsilon } } _ { y }$ remain also unchanged. In order to implement a standard model-based MHE scheme, we use the same noisy offline data collected from the nonlinear system (18) and identify a four-dimensional (discrete-time) linear system, by SID or PEM, such that we encounter a bias error. We simulate the different MHE schemes for 50 different online noise realizations and 50 different random initial conditions sampled from the interval $x ( 0 ) \in [ 0 , 1 0 ] ^ { 4 }$ as in the previous subsection.
