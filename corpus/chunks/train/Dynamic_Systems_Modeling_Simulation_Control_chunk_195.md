$$P = 1. 0 1 3 (1 0) ^ {5} + 3. 6 m \mathrm{N} / \mathrm{m} ^ {2}$$

where fluid mass m is in kg. The tank holds water (density $\rho = 1 0 0 0 \mathrm { k g } / \mathrm { m } ^ { 3 } )$ ) and has a constant circular cross-sectional area A. Compute the fluid capacitance C of the tank.
