where all the derivatives of are evaluated at $w ^ { \star } , p _ { 0 } , ( \lambda ^ { \star } , \mu ^ { \star } ) \in M ( p _ { 0 } )$ , and $g _ { \mathcal { A } }$ and $\mu _ { \mathcal { A } } ^ { \star }$ are vectors with elements $g _ { j }$ and $\mu _ { j } ^ { \star }$ for all $\boldsymbol { j } \in \mathcal { A } ( \boldsymbol { w } ^ { \star } )$ λ , µ, respectively.
