# 2.3 Communication Network for TH-MAS

A directed graph $\mathcal { G } ~ = ~ ( \nu , \mathcal { E } )$ represents the communication network of the TH-MAS. The corresponding set of nodes is $\mathcal { V } = \{ 1 , 2 , \dots , N + 1 \}$ with a set of directed edges $\mathcal { E } \subseteq \mathcal { V } \times$ V. Since the communication network has only one leader, the communication network that only contains the follower agents is represented by the graph

$$\mathcal {G} ^ {\prime} = \left(\mathcal {V} ^ {\prime}, \mathcal {E} ^ {\prime}\right) \text { with } \mathcal {V} ^ {\prime} \subseteq \mathcal {V} \text { and } \mathcal {E} ^ {\prime} \subseteq \mathcal {E}.$$

Notice that the system constraint (7) results in a time-varying graph for the followers network $\boldsymbol { \mathcal { G } ^ { \prime } } ( \boldsymbol { k } )$ where the set of node cardinality changes, such that

$$\left| \mathcal {V} ^ {\prime} (k) \right| = \sigma (m), \forall k = m M \tag {11}$$

and, therefore,

$$\left| \mathcal {V} ^ {\prime} (k + 1) \right| = \left| \mathcal {V} ^ {\prime} (k) \right|, \tag {12}$$

for all $k \in \{ ( m - 1 ) M + 1 , m M \}$ .

Remark 1. A directed graph has a spanning tree if there exists a node (agent), which is called the root node such that there exist directed paths from the root to any node (agent) in the graph. This is a necessary condition for all the agents reaching consensus. Hence, in this paper, both G and $\boldsymbol { \mathcal { G } ^ { \prime } } ( \boldsymbol { k } )$ have a spanning tree with the leader $\Sigma _ { L }$ as the root.
