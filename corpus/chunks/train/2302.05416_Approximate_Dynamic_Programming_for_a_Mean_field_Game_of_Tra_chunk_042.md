# Step 3. Iteration of Solutions.

Now, for $n \geq 2$ , let:

$$
\begin{array}{l} \frac {\mathrm{d}}{\mathrm{d} t} A ^ {n} (t) = \mathfrak {G} (A ^ {n} (t), B ^ {n} (t), \rho^ {n - 1} (t)), A ^ {n} (0) = A _ {0}; \\ \frac {\mathrm{d}}{\mathrm{d} t} B ^ {n} (t) = \mathfrak {H} (A ^ {n} (t), B ^ {n} (t), \rho^ {n - 1} (t)), B ^ {n} (0) = B _ {0}; \tag {52} \\ \frac {\mathrm{d}}{\mathrm{d} t} \rho^ {n} (t) = \mathcal {L} _ {n} (t; \tilde {u} ^ {n} (\cdot), \tilde {w} ^ {n} (\cdot)) \rho^ {n} (t) \text {in} (0, T ] \times \mathcal {Y}; \\ \rho^ {n} (0, \cdot) = \rho_ {0} \in D (\mathcal {L} _ {n}). \\ \end{array}
$$

First, the ODEs are solved and then PDE, as the coupling between the equations is only one-directional. For the ODE, we note that by continuous differentiability in time of $\rho ^ { n - 1 } , \mathfrak { G } ( A ( \cdot ) , B ( \cdot ) , \rho ^ { n - 1 } ( \cdot ) )$ will also be continuously differentiable in time. By Picard-Lindelöf there is a local classical solution on $[ 0 , t ^ { * , n } ( A _ { 0 } , B _ { 0 } ) )$ , and then we apply our approach to rule out finite-time blowup via Grönwall-Bellman from Section 1 to conclude that there is a classical solution $( A , B ) ^ { n } ( t ) \in \mathrm { C } ^ { 1 } ( [ 0 , T ] ; \mathbb { R } ^ { K \times K } \times \mathbb { R } ^ { K \times K } )$ . Then, we employ the argument of Section 2 to conclude that for each $n \geq 2 .$ , there is a classical solution to the PDE in this iterated system of type:

$$(A, B, \rho (\cdot)) ^ {n} (\cdot) \in \mathrm{C} ^ {0} ([ 0, T ]; D (\mathcal {L} _ {n}) \times \mathbb {R} ^ {K \times K} \times \mathbb {R} ^ {K \times K}) \cap \mathrm{C} ^ {1} ([ 0, T ]; \mathbb {H} ^ {0}).$$
