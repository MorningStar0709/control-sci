# C. Morse function on SO(3)

The following Lemma is utilized in the rotational ESO and attitude tracking control schemes for the aircraft.

Lemma 6. [41] Consider attitude kinematics

$$\dot {R} = R \Omega^ {\times}, R \in \mathrm{SO} (3), \Omega \in \mathfrak {s o} (3). \tag {40}$$

Define $K = \mathrm { d i a g } ( K _ { 1 } , K _ { 2 } , K _ { 3 } )$ , where $K _ { 1 } > K _ { 2 } > K _ { 3 } \geq 1$ Define

$$s _ {K} (R) = \sum_ {i = 1} ^ {3} K _ {i} (R ^ {\mathrm{T}} \mathbf {e} _ {i}) \times \mathbf {e} _ {i}, \tag {41}$$

such that $\begin{array} { r } { \frac { \mathrm { d } } { \mathrm { d } t } \langle K , I - R \rangle = \Omega ^ { \mathrm { T } } s _ { K } ( R ) } \end{array}$ . Here $\langle A , B \rangle = \operatorname { t r } ( A ^ { \mathrm { T } } B )$ which makes $\langle K , I - R \rangle$ a Morse function defined on $\mathrm { S O ( 3 ) }$ . Let $S \subset \mathrm { S O } ( 3 )$ be a closed subset containing the identity in its interior, defined by

$$\mathcal {S} = \left\{R \in \mathrm{SO} (3): R _ {i i} \geq 0 \text { and } R _ {i j} R _ {j i} \leq 0, \right. \tag {42}\forall i, j \in \{1, 2, 3 \}, i \neq j \}.$$

$$s _ {K} (R) ^ {\mathrm{T}} s _ {K} (R) \geq \langle K, I - R \rangle . \tag {43}$$

$$C \triangleq \{I, \operatorname{diag} (1, - 1, - 1), \operatorname{diag} (- 1, 1, - 1), \tag {44}\left. \operatorname{diag} (- 1, - 1, 1) \right\} \subset \mathrm{SO} (3).$$
