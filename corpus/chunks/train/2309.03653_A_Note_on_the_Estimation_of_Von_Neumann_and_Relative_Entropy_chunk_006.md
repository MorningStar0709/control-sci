$$\| \boldsymbol {e} (t) \| _ {2} \leq M e ^ {- \lambda t}, \quad \forall t \geq 0. \tag {9}$$

Letting $\tilde { \varrho } ( t ) = \mathrm { v e c } \{ \tilde { \pmb { x } } ( t ) \} ^ { - 1 }$ and recalling that the vectorization operator is an isometry between $\mathbb { C } ^ { d ^ { 2 } }$ and Cd×d, $\mathbb { C } ^ { d \times d }$ one can conclude

$$\left\| \tilde {\boldsymbol {\varrho}} (t) - \boldsymbol {\varrho} (t) \right\| = \left\| \boldsymbol {e} (t) \right\| _ {2} \leq M e ^ {- \lambda t} \tag {10}$$

for all $t \geq 0$ , which achieves the goal of (2).

Linear observability cannot be achieved with an arbitrarily small number of observables. In fact, at least d observables are required. The following result extends that of [9] to the case where the measurement statistics do not come from a POVM.

Theorem 1. If the closed quantum system $( A , C )$ is linearly observable then $N _ { \mathrm { m } } \geq d .$ .

Proof. The nullspace of A has dimension at least d. Let $v _ { 1 } , v _ { 2 } , \ldots , v _ { d }$ be linearly independent vectors from this space, and define

$$\boldsymbol {v} _ {j} ^ {\prime} \triangleq \boldsymbol {C} \boldsymbol {v} _ {j} \in \mathbb {C} ^ {N _ {\mathrm{m}}}, j = 1, 2, \dots , d.$$

If $N _ { \mathrm { m } } ~ < ~ d ,$ then the vectors $\{ \pmb { v } _ { j } ^ { \prime } \} _ { j = 1 } ^ { d }$ are linearly dependent and there exists a set of non-zero coeffecients $\mathsf { \tilde { \{ \beta _ { 1 } , \beta _ { 2 } , \dots , \beta _ { d } \} } }$ such that

$$\sum_ {j = 1} ^ {d} \beta_ {j} \boldsymbol {v} _ {j} ^ {\prime} = 0.$$

Therefore, v $\begin{array} { r } { \triangleq \sum _ { j = 1 } ^ { d } \beta _ { j } \pmb { v } _ { j } \neq \mathbf { 0 } } \end{array}$ is such that $C v = 0$ and $C A ^ { k } v = \mathbf { 0 }$ for $k = 1 , 2 , \ldots , d ^ { 2 } - 1$ . That is the nullspace of $O ( A , C )$ contains a non-zero vector. The rank of the observability matrix is then strictly less than $d ^ { 2 }$ and the system is not linearly observable. □
