# 4 Data-Driven Dilation Symmetry

In this section we continue the study of a possibility of transformation of an existing artificial neural network to a homogeneous one. We assume that an ANN is given by $g _ { \epsilon } : \mathbb { R } ^ { n } \mapsto \mathbb { R } ^ { m }$ such that

$$g _ {\epsilon} (x) = C \sigma (A x + b)$$

approximates a function $g \in C ( \mathbb { R } ^ { n } \backslash \{ \mathbf { 0 } \} , \mathbb { R } )$ on a compact $\Omega \subset \mathbb { R } ^ { n } \backslash \{ \mathbf { 0 } \}$ , where $A \in \mathbb { R } ^ { N \times n } , b \in \mathbb { R } ^ { N } , C \in \mathbb { R } ^ { 1 \times N }$ . The index $\epsilon > 0$ indicates that the approximation precision of ANN (see Theorem 2) is given by

$$\sup _ {x \in \Omega} | g (x) - g _ {\epsilon} (x) | \leq \epsilon , \quad \forall x \in \Omega .$$

The ANN is supposed to be well trained to approximate a homogeneous function $g = h$ on a compact Ω, i.e., $\epsilon > 0$ is assumed to be small enough. But now we consider the case when some (or any) information about dilation symmetry of the function h is unavailable, so, first, the parameters of the dilation group and/or the homogeneity degree have to be identified using $g _ { \epsilon }$ , next, a homogeneous ANN has to be designed.
