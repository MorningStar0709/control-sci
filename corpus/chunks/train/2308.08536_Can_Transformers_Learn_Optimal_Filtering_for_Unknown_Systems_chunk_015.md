# C. Proof of the Main Theorem

In this section, we provide the proof for Theorem 1. Extending Assumption 2, the following lemma tells how the noise $\{ \mathbf { w } , \mathbf { v } \}$ would affect the loss performance.

Lemma 1: Suppose Assumptions 1 and 2 hold. Under the same setup as in Assumption 2, let $\bar { y } : = L _ { g } L _ { \rho } \bar { w } + \bar { v }$ . Then,

$$\mathbb {E} _ {\mathbf {w} _ {t}, \mathbf {v} _ {t}} ^ {\prime} \left[ \left| \ell (\mathbf {y} _ {t}, \mathrm{TF} (\mathcal {Y} _ {t - 1})) - \ell (\mathbf {y} _ {t}, \mathrm{TF} (\mathcal {Y} _ {t - 1} ^ {\prime})) \right| \right] \leq \frac {2 K L _ {\ell} \bar {y}}{t - \tau}.$$
