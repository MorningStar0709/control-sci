Remark 2: Assumption 1 is weaker than in some reinforcement learning schemes with guarantees, such as [39] where L needs actually to satisfy a Hamilton-Bellman-Jacobi inequality, but is still somewhat strong (however, see [33]). This is, as discussed above, the price to pay for ensuring practical stabilizability property of the policy resulting from learning. However, already controllability, as required in convergence analyses of reinforcement learning (see, e. g., [37], [39]), implies stabilizability [27], [42]. If the closed-loop is stable, there is, e. g., a constructive way to find a Lyapunov function (see [43]). Also, there are numerical techniques to build control Lyapunov functions [44]. Furthermore, they can be constructed from physical considerations, e. g., kinetic and potential energy [33]. Then, there are systematic techniques such as augmented Lyapunov functions in adaptive control; backstepping; stabilizing model-predictive control whose finite-horizon cost function can be shown to be a Lyapunov function etc.

Remark 3: Also note that the existence of such $\alpha _ { \mathrm { l o w } } ( \cdot )$ and $\alpha _ { \mathrm { { u p } } } ( \cdot )$ are equivalent to radial unboundedness and positivedefiniteness respectively [45]. Furthermore, a constructive proof of the latter properties would have to involve a construction of such functions.
