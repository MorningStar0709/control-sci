# 2.6 Is a Terminal Constraint Set $\mathbb { X } _ { f }$ Necessary?

While addition of a terminal cost $V _ { f } ( \cdot )$ does not materially affect the optimal control problem, addition of a terminal constraint $x ( N ) \in \mathbb { X } _ { f }$ , which is a state constraint, may have a significant effect. In particular, problems with only control constraints are usually easier to solve. So if state constraints are not present or if they are handled by penalty functions (soft constraints), it is highly desirable to avoid the addition of a terminal constraint. Moreover, it is possible to establish continuity of the value function for a range of optimal control problems $i f$ there are no state constraints; continuity of the value function ensures a degree of robustness (see Chapter 3). It is therefore natural to ask if the terminal constraint can be omitted without affecting stability. There are several answers to this question.
