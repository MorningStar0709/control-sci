$$\hat {\boldsymbol {\Theta}} ^ {*} \in \underset {\hat {\boldsymbol {\Theta}}} {\operatorname{argmin}} F (\hat {\boldsymbol {\Theta}}) := \frac {1}{D} \sum_ {s = 1} ^ {D} \left| \left| \mathbf {u} _ {s} ^ {*} - \mathbf {u} ^ {\text { HTNN }} (x _ {0, s}; \hat {\boldsymbol {\Theta}}) \right| \right| ^ {2}.$$

Given the structure of the HTNN network, we obtain again a minmax representation for the learned function $\mathbf { u } ^ { \mathrm { H T N N } } ( \bar { { x } } _ { 0 } )$ , which depends on the (optimal) parameters $\hat { \Theta } ^ { * }$ of the HTNN:

$$\mathbf {u} ^ {\mathrm{HTNN}} (x _ {0}) = \mathcal {M} _ {i n} ^ {\mathrm{HTNN}} \mathcal {M} _ {a x} ^ {\mathrm{HTNN}} (x _ {0}; \hat {\Theta} ^ {*}). \tag {19}$$

From previous theorems it follows that $\mathbf { u } ^ { \mathrm { H T N N } } ( x _ { 0 } ) = \mathbf { u } ^ { * } ( x _ { 0 } )$ , provided that the number of layers and hidden neurons are chosen as in Theorem 2.
