# CANONICAL FORM FOR THE NOMINAL SYSTEM:

Canonical Form of representation of a standard Continuous-Time state space system is as follows

$$\dot {X} _ {1} (t) = A _ {1 1} x _ {1} (t) + A _ {1 2} y (t) + B _ {1} u (t) \tag {5.2}\dot {Y} (t) = A _ {2 1} x _ {1} (t) + A _ {2 2} y (t) + B _ {2} u (t)$$

Similarly, Adaptive Sliding Mode Observer (ASMO) is represented as follows in State space form is as follows

$$\stackrel {\bullet} {\hat {X}} _ {1} (t) = A _ {1 1} x _ {1} (t) + A _ {1 2} y (t) + B _ {1} u (t) + L v \tag {5.3}\stackrel {\bullet} {\hat {Y}} (t) = A _ {2 1} x _ {1} (t) + A _ {2 2} y (t) + B _ {2} u (t) - v$$

Where $( { \overset { \land } { x } } _ { 1 } , y )$ represent the state estimates, $L \in { \cal R } ^ { ( n - p ) \times p }$ is a gain matrix and $\nu _ { i } = M \mathrm { s g n } ( y _ { i } - y _ { i } )$ where $M \in R$ Error of the system along with ASMO is

$$
\begin{array}{l} \bullet \\ e _ {x} = A _ {1 1} e _ {x} (t) + A _ {1 2} e _ {y} (t) + L v \\ \bullet \end{array} \tag {5.4}
e _ {y} = A _ {2 1} e _ {x} (t) + A _ {2 2} e _ {y} (t) - v$$

A Lyapunov stability function can be examined to define the stability of the Induction Machine along with the state estimators is represented as follows,

$$V = \frac {1}{2} e _ {x} ^ {T} P e _ {x} + \frac {d}{2} \Delta \omega_ {r} ^ {2} + a ^ {T} \alpha a + b ^ {T} \beta b \tag {5.5}$$

The Time derivate of above function to guarantee the stability of the system will be $\overset { \bullet } { V } < 0$ . Controller gains are adjusted in such a way that $\dot { \nu }$ is a Negative Definite function. The main aim of ASMO controller parameters adjustements are done to ensure that the state trajectory of the switching function should lies within the specified switching surface.
