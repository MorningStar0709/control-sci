For time interval $t ~ \le ~ t _ { q } , ~ \dot { V } ~ \le ~ 0 .$ , inferring $V \left( t \right) \ \leq$ $V \left( t _ { 0 } \right)$ , hence $V \left( t _ { q } \right) ~ \leq ~ \dot { V } \left( t _ { 0 } \right) .$ the Lyapunov function satisfies the following, min $\left\{ \lambda _ { \operatorname* { m i n } } \left( \mathbf { P } \right) , | k _ { p } | \right\} \left\| \boldsymbol { x } \right\| ^ { 2 } \leq V \leq$ max $\left\{ \lambda _ { \operatorname* { m a x } } \left( \mathbf { P } \right) , \left| k _ { p } \right| \right\} \left\| x \right\| ^ { 2 }$ , we can write,

$$\| \pmb {\chi} (t) \| ^ {2} \leq \frac {1}{\min \{\lambda_ {\min} (\mathbf {P}) , | k _ {p} | \}} V (t) \quad \forall t \geq t _ {0},\leq \frac {1}{\min \left\{\lambda_ {\min} (\mathbf {P}) , | k _ {p} | \right\}} V (t _ {q}) \quad \forall t > t _ {q},\leq \frac {e ^ {- \overline {{\kappa}} (t - t _ {q})}}{\min \left\{\lambda_ {\min} (\mathbf {P}) , | k _ {p} | \right\}} V (t _ {0}) \quad \forall t > t _ {q},\| \pmb {\chi} (t) \| \leq e ^ {- \kappa (t - t _ {q})} \alpha \| \pmb {\chi} (t _ {0}) \| \quad \forall t > t _ {q},$$

Remark 3.
