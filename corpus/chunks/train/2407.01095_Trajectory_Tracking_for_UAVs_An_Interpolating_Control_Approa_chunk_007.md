In eIC, there are two distinct LPs. If $\pmb { x } \in \Omega ^ { l } \setminus \Omega ^ { m }$ , the interpolation is done between $\pmb { u } ^ { l }$ and $\pmb { u } ^ { m }$ and the IC control law takes the form of

$$\boldsymbol {u} (\boldsymbol {x} _ {k}, \bar {\boldsymbol {x}} _ {k} ^ {N}) = c _ {k} \boldsymbol {u} ^ {l} \left(\boldsymbol {x} _ {k} ^ {l}, c _ {k} \bar {\boldsymbol {x}} _ {k} ^ {N}\right) + (1 - c) \boldsymbol {u} ^ {m} \left(\boldsymbol {x} ^ {m}, (1 - c _ {k}) \bar {\boldsymbol {x}} _ {k}\right). \tag {8}$$

If the $\pmb { x } \in \pmb { \varOmega } ^ { m }$ , the interpolation is performed for both $\pmb { u } ^ { m }$ and $\pmb { u } ^ { h }$ , which results in control law

$$\boldsymbol {u} \left(\boldsymbol {x} _ {k}, \bar {\boldsymbol {x}} _ {k} ^ {N}\right) = c _ {k} \boldsymbol {u} ^ {m} \left(\boldsymbol {x} _ {k} ^ {m}, c _ {k} \bar {\boldsymbol {x}} _ {k}\right) + (1 - c _ {k}) \boldsymbol {u} ^ {h} \left(\boldsymbol {x} _ {k} ^ {h}, (1 - c _ {k}) \bar {\boldsymbol {x}} _ {k} ^ {N}\right). \tag {9}$$
