# 5.3 Data-Driven Pinning Control for Cell Fate Dynamics

Indeed, our frameworks can be extended to the model-free version via a combination with existing data reconstruction method. To be concrete, we show that our framework can combine with Neural ODEs (NODEs) (Chen et al., 2018) to learn the control policy from time series data for the Cell Fate system (Sun et al., 2017; Laslo et al., 2006), which describes the interaction between two suppressors during cellular differentiation for neutrophil and macrophage cell fate choices. The system ${ \dot { \pmb x } } = f ( { \pmb x } ) , { \pmb x } = ( x _ { 1 } , . . . , x _ { 6 } )$ has three steady states: $P _ { 1 , 2 , 3 } ,$ , where $P _ { 2 , 3 }$ correspond to different cell fates and are stable and $P _ { 1 }$ represents a critical expression level connecting the two fates and is unstable. The network structure of this 6-D system is a treemap, where one root node $x _ { 1 }$ can stabilize itself under the original dynamic. Hence, we choose root node $x _ { 2 }$ with maximum our degree and add pinning control on it to stabilize the system to unstable state $P _ { 1 }$ . The original trajectory that converges to $P _ { 2 }$ (left) and the controlled trajectory that converges to $P _ { 1 }$ (right) are shown in Figure 9. The original trajectory is used to train the NODE to reconstruct the vector field ${ \hat { f } } ,$ then we use the sample of $\hat { f }$ as training data to learn our stochastic pinning control.
