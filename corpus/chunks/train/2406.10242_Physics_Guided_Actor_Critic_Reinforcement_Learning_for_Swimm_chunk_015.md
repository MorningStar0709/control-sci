$$P (s \mid t) \propto \frac {1}{s} \exp \Big [ - t S _ {1} \big (t ^ {- 1} \log \big (s / s _ {d} \big) + \phi \big) \Big ] \to \frac {1}{s}\times \exp \left[ - t \Big (S _ {1} (\bar {\lambda} _ {1}) + \Big (\frac {1}{t} \log \big (s / s _ {d} \big) + \phi - \bar {\lambda} _ {1} \Big) ^ {2} S _ {1} ^ {\prime \prime} (\bar {\lambda} _ {1}) \Big) \right].$$

Taking the limit $t \to \infty$ and $s \gg s _ { d } ,$ leads to

$$P _ {\mathrm{st}} (s) \propto \frac {1}{s} \left(\frac {s _ {d}}{s}\right) ^ {2 (\phi - \bar {\lambda} _ {1}) S _ {1} ^ {\prime \prime} (\bar {\lambda} _ {1})}, \tag {13}$$

where $s _ { d } , \bar { \lambda } _ { 1 }$ and $S _ { 1 } ^ { \prime \prime }$ are dependent of the system’s dynamics.

Notice, that the stationary version, given in Eq. (13), settles if $\phi > \bar { \lambda } _ { 1 }$ , and that it is fully consistent with Eq. (15), derived for the short δ-correlated velocity gradient, then $S _ { 1 } ^ { \prime \prime } ( \bar { \lambda } _ { 1 } ) = 1 / ( ( d - 1 ) D )$ and $\bar { \lambda } _ { 1 } { = } d ( d - \bar { 1 } ) \bar { D } / 2$ .
