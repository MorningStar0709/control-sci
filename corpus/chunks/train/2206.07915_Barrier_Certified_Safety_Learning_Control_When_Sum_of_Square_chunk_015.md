# IV. NUMERICAL EXAMPLE

A swing-up pendulum model from OpenAI Gym environment (Pendulum-v1) is used to verify the above theorems,

$$m l ^ {2} \ddot {\theta} = m g l \sin (\theta) + a. \tag {20}$$

Let $s _ { 1 } = \theta , s _ { 2 } = \dot { \theta } .$ The dynamics of (20) are defined as

$$
\left[ \begin{array}{c} \dot {s} _ {1} \\ \dot {s} _ {2} \end{array} \right] = \left[ \begin{array}{c} s _ {2} \\ - \frac {g}{l} \sin (s _ {1}) + \frac {a}{m l ^ {2}} + d \end{array} \right], \tag {21}
$$

where d denotes unknown dynamics that generate by inaccurate parameters $\bar { m } = 1 . 4 , \bar { l } = 1 . 4$ , as [23] introduced. When we try to construct a polynomial system (8), the $\sin ( s _ { 1 } )$ in (21) will be approximated by Chebyshev interpolants within $s _ { 1 } ~ \in ~ [ - 3 , 3 ]$ . Then, with SOSOPT Toolbox [34], Mosek solver [35] and Tensorflow, we implement Algorithm 1 in the pendulum model.
