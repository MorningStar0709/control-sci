# B. Model Predictive Control

In this work, we use a centralized model predictive controller to minimize the cost function. We define $s ^ { t : k }$ and $\mathbf { a } ^ { t : k }$ as the global system state and input of all agents from time t to k, respectively. The model predictive controller finds a set of optimal inputs $\mathbf { a } ^ { ( t + 1 : t + T ) ^ { * } }$ which minimize the cost function $\hat { C } ( \hat { s } ^ { t + 1 : t + T } , a ^ { t + 1 : t + T } )$ over predicted state ˆs and control inputs a for some finite time horizon T :

$$\mathbf {a} ^ {(t + 1: t + T) ^ {*}} = \underset {\mathbf {a} ^ {t + 1: t + T}} {\arg \min} C (\hat {s} ^ {t + 1: t + T}, a ^ {t + 1: t + T}). \tag {1}$$

As the dynamics of multi-agent system is implicit in our setting, so we use a nonlinear function f to predict future implicit states:

$$\hat {s} ^ {t + 1} = f (s ^ {t}, \mathbf {a} ^ {t}). \tag {2}$$

We can apply this formula recurrently to predict further states until the end of the horizon T .
