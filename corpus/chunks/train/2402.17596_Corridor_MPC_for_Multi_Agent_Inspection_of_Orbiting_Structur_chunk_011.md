# IV. SAMPLED DATA HOCBF

To solve Problem 1, we define the sampled-data HOCBF, which will be applied inside the control scheme presented in Section V. For a zero order hold (ZOH) sampled-data system, the state measurements are only available at discrete time steps k∆t. Assuming that a control input $\mathbf { \Gamma } \mathbf { \mathbf { \mathbf { u } } } ( k \Delta t ) \mathbf { \Sigma } \in \mathbb { \Gamma } \mathbb { U }$ is applied to (2) such that $\tilde { \zeta } ( \tilde { \mathbf { \widetilde { x } } } ( k \Delta t ) , \mathbf { \widetilde { u } } ( k \Delta t ) , \delta \pmb { d } ( k \Delta t ) , k \Delta t ) \ \ge \ 0$ is satisfied, this condition is not sufficient to guarantee safety throughout the sampling interval $[ k \Delta t , ( k { + } 1 ) \Delta t ]$ .
