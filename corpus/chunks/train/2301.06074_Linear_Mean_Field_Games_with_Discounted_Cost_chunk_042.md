In the remainder of this section, $\{ ( \mathbf { x } ( t ) , \mathbf { a } ( t ) ) \} _ { t \geq 0 }$ denotes the state-action vectors under the joint policy $\pi ^ { * } : = ( \pi ^ { * } , \ldots , \pi ^ { * } )$ and initial distribution $\mu ^ { * , \otimes ^ { N } }$ for N -agent linear meanfield game. Similarly, $\{ ( x ( t ) , a ( t ) ) \} _ { t \geq 0 }$ denotes the state-action pairs under the policy $\pi ^ { * }$ and initial distribution $\mu ^ { * }$ of a generic agent in the infinite population limit.

For $t = 0 , ( x _ { 1 } ( 0 ) , \ldots , x _ { N } ( 0 ) ) \sim \bar { \mu } ^ { * , \otimes ^ { N } }$ , and therefore, there exists, by Glivenko-Cantelli’s theorem, a function $\alpha ( N )$ such that

$$\mathbb {E} [ W _ {1} (\mathsf {e} [ \cdot | \mathbf {x} (0) ], \mu^ {*}) ] \leq \alpha (N) \tag {16}$$

and $\alpha ( N )  0$ as $N \to \infty$ . Indeed, if X is a compact subset of some finite dimensional Euclidean space, then we have an explicit expression for $\alpha ( N )$ which is due to [14, Theorem 1]: for any $q > 1$ , there exists a constant C depending on q and dim(X) such that

$$
\alpha (N) := C \left(\int_ {\mathsf {X}} \| x \| ^ {q} \mu^ {*} (d x)\right) ^ {\frac {1}{q}} \left\{ \begin{array}{l l} \frac {1}{\sqrt {N}} + \frac {\sqrt [ q ]{N}}{N} & \text {if} \dim (\mathsf {A}) <   2 \\ \frac {\log (1 + N)}{\sqrt {N}} + \frac {\sqrt [ q ]{N}}{N} & \text {if} \dim (\mathsf {A}) = 2 \\ \frac {1}{\sqrt [ d ]{N}} + \frac {\sqrt [ q ]{N}}{N} & \text {if} \dim (\mathsf {A}) > 2 \end{array} \right.
$$
