# 3 Existence and uniqueness of the solutions to the HJB equations (1.10) and (1.15)

To study the existence and uniqueness of the solutions u and $u ^ { \varepsilon }$ to the variational inequalities (1.10) and (1.15), respectively, we will proceed in the same way as in [13], i.e., we will first verify that the sequence $\{ u ^ { \varepsilon , \breve { \delta } } \} _ { ( \varepsilon , \delta ) \in ( 0 , 1 ) ^ { 2 } }$ is bounded, uniformly in $( \varepsilon , \delta )$ , with respect to the norms $\| \cdot \| _ { \mathrm { C } ^ { 0 } ( \overline { { \mathcal { O } } } ) } , \| \cdot \| _ { \mathrm { C } _ { \mathrm { l o c } } ^ { 1 } ( \mathcal { O } ) }$ and $\begin{array} { r l } { \| \cdot \| _ { \mathrm { C } _ { \mathrm { l o c } } ^ { 2 } ( \mathcal { O } ) } ; } \end{array}$ see Lemmas 3.1–3.4; and then, for each $\varepsilon \in ( 0 , 1 )$ fixed, $u ^ { \varepsilon }$ will be taken as limit of $u ^ { \varepsilon , \delta }$ when δ goes to zero.
