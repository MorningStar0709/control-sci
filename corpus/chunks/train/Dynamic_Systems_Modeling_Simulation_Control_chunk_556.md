A proportional-derivative controller is needed to add damping to the closed-loop system and improve the transient response. The PD controller transfer function is

$$G _ {C} (s) = K _ {P} + K _ {D} s \tag {10.22}$$

which has one zero at $s = - K _ { P } / K _ { D } .$ . After including the PD controller (again with fixed actuator gain $K _ { A } = 2 \mathrm { N } / \mathrm { V } )$ , the closed-loop transfer function (10.19) becomes

$$T (s) = \frac {\frac {2 (K _ {P} + K _ {D} s)}{s ^ {2} + 0 . 3 s}}{1 + \frac {2 (K _ {P} + K _ {D} s)}{s ^ {2} + 0 . 3 s}} = \frac {2 (K _ {P} + K _ {D} s)}{s ^ {2} + (0 . 3 + 2 K _ {D}) s + 2 K _ {P}} \tag {10.23}$$

Equation (10.23) shows that the resulting closed-loop system using PD control has a unity DC gain (i.e., $2 K _ { P } / 2 K _ { P } )$ for any nonzero gain $K _ { P }$ and hence it will also perfectly track a step input at steady state.
