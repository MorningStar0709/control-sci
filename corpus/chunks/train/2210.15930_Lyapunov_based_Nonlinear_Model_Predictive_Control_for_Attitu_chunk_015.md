# 5 Results

In this section, the performance of the proposed LNMPC controller is evaluated and compared with two state-of-the-art nonlinear controllers, the backstepping controller (BSC) and the sliding mode controller (SMC). The UAV model used is the AscTec Pelican as shown in Figure 1 with parameters shown in Table 1.

Constraints on the UAV are defined as

$$- \frac {\pi}{2} \leq \phi , \theta \leq \frac {\pi}{2},- \frac {\pi}{2} \leq \dot {\phi}, \dot {\theta}, \dot {\psi} \leq \frac {\pi}{2}, \tag {35}- 0. 1 \leq \tau_ {\phi}, \tau_ {\theta}, \tau_ {\psi} \leq 0. 1.$$

For the LNMPC, the sampling period is chosen as $d t = 0 . 0 2 { \mathrm { s } } .$ , the predictive horizon is chosen as $T = 3 0 d t$ , and the weight matrices are chosen as

$$
\begin{array}{l} \boldsymbol {P} = \boldsymbol {Q} = \operatorname{diag} (3 0. 0, 3 0. 0, 3 0. 0, 1. 0, 1. 0, 1. 0), \\ \boldsymbol {P} = \operatorname{diag} (1. 0, 1. 0, 1. 0) \end{array} \tag {36}
\boldsymbol {R} = \mathrm{diag} (1. 0, 1. 0, 1. 0).$$

Since the BSC and SMC do not consider the UAV constraints, their control signals are limited to $u \in \{ u _ { m a x } , u _ { m i n } \}$ as follow:

$$u = \min \left\{u _ {\max}, \max \left\{u, u _ {\min} \right\} \right\}. \tag {37}$$

Comparisons are then conducted in four scenarios with different input references.
