Example 4 (continued-verification of UGAOS): Consider again system (24) with $\sigma , k > 0$ . We next show that system (24) with $k \geq \frac { 1 } { 4 \sigma } + \theta$ and output $y = \xi$ is UGAOS. Using the radially unbounded function defined by (25) that satisfies for all $\boldsymbol { x } \in \mathbb { R } ^ { 2 }$ equation (26), we notice that (26) implies (16) with $W ( x ) = \xi ^ { 2 } / 2$ and $\rho ( s ) = 2 k s$ . It is also clear that (12) holds with $a ( s ) = s ^ { 2 } / 2$ Finally, we obtain for all $\boldsymbol { x } \in \mathbb { R } ^ { 2 }$ :

$$\dot {W} (x) = - \left(z + k + \sigma (\theta + z) ^ {2}\right) \xi^ {2} \tag {29}$$

Since the inequality $z + k + \sigma { \big ( } \theta + z { \big ) } ^ { 2 } \geq k - { \frac { 1 } { 4 \sigma } } - \theta$ holds for all $z \in \mathbb { R }$ , we conclude from (29) that 4 (13) holds when $k \geq \frac { 1 } { 4 \sigma } + \theta$ . Therefore, Theorem 3 implies that system (24) is UGAOS when $k \geq \frac { 1 } { 4 \sigma } + \theta$ .

We also notice that equation (29) implies the differential inequality $\dot { W } ( x ) \leq - 2 \biggl ( k - \frac { 1 } { 4 \sigma } - \theta \biggr ) W ( x )$ for all $\boldsymbol { x } \in \mathbb { R } ^ { 2 }$ . Integrating the above differential inequality, we obtain the estimate

$$\left| y (t, x _ {0}) \right| = \left| \xi (t) \right| \leq \exp \left(- \left(k - \frac {1}{4 \sigma} - \theta\right) t\right) \left| \xi_ {0} \right|$$

that holds for all $x _ { 0 } = \left( \xi _ { 0 } , z _ { 0 } \right) ^ { \prime } \in \mathbb { R } ^ { 2 } , \ t \geq 0$ . Therefore, when $k > \frac { 1 } { 4 \sigma } + \theta$ we are able to conclude that the output converges to zero at an exponential rate (notice that uniform attractivity does not necessarily imply an exponential convergence rate). The above estimate is independent of $z _ { 0 }$ and that is the reason that such a strong stability property is termed in [24] as “State-Independent Uniform Output Stability”. However, the reader should notice that a state-independent stability estimate (i.e., an estimate independent of $z _ { 0 } )$ cannot be easily shown in the case $k = \frac { 1 } { 4 \sigma } + \theta$ , where we know that UGAOS holds.
