# 2.2 Dual problem

Let X, Y be complete separable metric spaces, $\mu \in \mathcal { M } _ { + } ( X ) , \nu \in \mathcal { M } _ { + } ( Y )$ be positive measures and $c \in L ^ { \infty } ( X \times Y )$ be a cost function. For given $\varepsilon , \tau _ { 1 } , \tau _ { 2 } > 0$ positive numbers, we define the unbalanced dual functional $D _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } : L ^ { \infty } ( X ) \times L ^ { \infty } ( Y )  \mathbb { R }$ as

$$D _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (u, v) = - \tau_ {1} \int_ {X} (e ^ {- \frac {u}{\tau_ {1}}} - 1) \mathrm{d} \mu - \tau_ {2} \int_ {Y} (e ^ {- \frac {v}{\tau_ {2}}} - 1) \mathrm{d} \nu - \varepsilon \int_ {X \times Y} e ^ {\frac {u + v - c}{\varepsilon}} \mathrm{d} \mu \otimes \nu .$$

The corresponding dual (Kantorovich) problem takes the form

$$\sup \{D _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (u, v): u \in L ^ {\infty} (X), v \in L ^ {\infty} (Y) \}. \tag {2.6}$$

In the following, we will prove the existence of a maximizer in (2.6) via the direct method of Calculus of Variations. We follow the approach introduced in [12] and define the Unbalanced Entropic c−transform.

Definition 2.1 (Unbalanced Entropic c-transform or $( c , \tau , \varepsilon )$ -transform). Let $\varepsilon , \tau _ { 1 } , \tau _ { 2 } > 0$ be the positive parameters, X and Y be complete separable metric spaces, $\mu \in { \mathcal { M } } _ { + } ( X )$ , ν ∈ ${ \mathfrak { M } } _ { + } ( Y )$ be positive measures, and let $c \in L ^ { \infty } ( X \times Y )$ be a cost function. Given $u \in L ^ { \infty } ( X )$ and $v \in L ^ { \infty } ( Y )$ , the entropic $( c , \tau _ { 2 } , \varepsilon )$ -transform of a function u is defined by a map $( \cdot ) ^ { ( c , \tau _ { 2 } , \varepsilon ) }$ : $L ^ { \infty } ( X ) \to L ^ { \infty } ( Y )$ a s

$$u ^ {(c, \tau_ {2}, \varepsilon)} (y) = - \frac {\tau_ {2} \varepsilon}{\tau_ {2} + \varepsilon} \log \int_ {X} \exp \left(\frac {u (x) - c (x , y)}{\varepsilon}\right) \mathrm{d} \mu (x), \tag {2.7}$$

and analogously for v, we define the map $( \cdot ) ^ { ( c , \tau _ { 1 } , \varepsilon ) } : L ^ { \infty } ( Y ) \to L ^ { \infty } ( X )$ a s

$$v ^ {(c, \tau_ {1}, \varepsilon)} (x) = - \frac {\tau_ {1} \varepsilon}{\tau_ {1} + \varepsilon} \log \int_ {Y} \exp \left(\frac {v (y) - c (x , y)}{\varepsilon}\right) \mathrm{d} \nu (y). \tag {2.8}$$
