# A.5 Proof of Lemma 4

We begin with an auxiliary lemma that characterizes the derivative of the stabilizing solution to a Riccati equation. The key idea is to linearize the Riccati equation around the solution, yielding a Lyapunov equation for the derivative.

Lemma 10 Le ${ \mathrm { ~  ~ : ~ } } ( - \delta , \delta )  \mathrm { R } ^ { n \times n } , G : ( - \delta , \delta )  \mathrm { S } _ { + } ^ { n }$ , and $H : ( - \delta , \delta ) \to \mathrm { S } _ { + } ^ { n }$ be real analytic matrix-valued functions for some $\delta > 0$ . $$\mathcal {R} (X _ {t}, t) := M _ {t} X _ {t} + X _ {t} M _ {t} ^ {\mathsf {T}} + G _ {t} + X _ {t} H _ {t} X _ {t} = 0 \quad (\mathrm{A.5})$$

$$
\begin{array}{l} \left(M _ {0} + X _ {0} H _ {0}\right) X _ {0} ^ {\prime} + X _ {0} ^ {\prime} \left(M _ {0} + X _ {0} H _ {0}\right) ^ {\top} \tag {A.6} \\ + \left(M _ {0} ^ {\prime} X _ {0} + X _ {0} M _ {0} ^ {\prime \mathsf {T}} + G _ {0} ^ {\prime} + X _ {0} H _ {0} ^ {\prime} X _ {0}\right) = 0. \\ \end{array}
