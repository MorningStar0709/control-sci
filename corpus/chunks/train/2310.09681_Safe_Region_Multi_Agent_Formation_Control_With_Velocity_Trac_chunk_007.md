Definition 1. Desired formation: Consider a network of N agents given by $\mathcal { P } ^ { * } = \mathbf { \bar { \{ } } { }  p _ { 1 } ^ { * } , \dots , p _ { N } ^ { * } \} , p _ { i } ^ { * } \in \mathbb { R } ^ { 2 }$ representing a feasible target shape, such that graph formed by ${ \mathcal { P } } ^ { * } ; { \mathcal { G } } ^ { * } =$ $( \dot { \mathcal { V } } , \mathcal { E } ^ { * } )$ , where

$$\mathcal {E} ^ {*} = \left\{(i, j) | \| p _ {i} ^ {*} - p _ {j} ^ {*} \| < r - \epsilon , i, j \in \mathcal {V} \right\}.$$

The desired formation is defined as:

$$\mathcal {P} _ {d} = \{p _ {i} | (p _ {i} - p _ {j}) = (p _ {i} ^ {*} - p _ {j} ^ {*}) \forall j \in \mathcal {N} _ {i}; i \in \mathcal {V} \}.$$

Therefore, $\mathcal { P } _ { d }$ represents the collection of all formations that are identical to ${ \mathcal { P } } ^ { * }$ in terms of shape and size but may differ in translation.

Besides maintaining a desired geometric shape, the multi-agent system will be steered to a target region, assumed convex, denoted by $\Omega \subset \mathbb { R } ^ { 2 }$ while avoiding K unsafe regions denoted using $\overline { { \Omega } } = \{ \overline { { \Omega } } _ { k } \ | \ \overline { { \Omega } } _ { k } \subset \mathbb { R } ^ { 2 } \ \forall \ \overline { { k } } \ =$ $1 , 2 , \ldots , { \bar { K } } \}$ . A collision involving agent i and obstacle k is defined as occurring when the position of agent i lies within the boundary of the obstacle $\overline { { \Omega } } _ { k } ,$ , i.e. $p _ { i } ~ \in \overline { { \Omega } } _ { k }$ . On the other hand, an inter-agent collision between two agents is defined as occurring when the distance between them is less than a specified threshold value, denoted as $\delta _ { i n }$ . Similar to [44, 45], we let $v _ { d } ( t )$ denote the reference velocity for all agents to converge to after entering the target region, although only some agents know $v _ { d } ( t )$ , which are referred as stealthy leaders.

Definition 2. Stealthy leader/leaders: The set of stealthy leaders $\mathcal { L } \subseteq \mathcal { V }$ is defined as the agents having the information about the final target region and the reference tracking velocity signal.
