# A.3 Proof of Lemma 3

We aim to show that for any $K _ { 0 } \in \partial { \cal K } _ { \beta }$

$$\lim _ {K \to K _ {0}, K \in \mathcal {K} _ {\beta}} J _ {\mathrm{mix}} (K) = \tilde {J} _ {\mathrm{mix}} (K _ {0}),$$

i.e., the extended cost function $\tilde { J } _ { \mathrm { m i x } }$ is continuous on the boundary $\partial { \cal K } _ { \beta }$ . To this end, it suffices to establish

$$\lim _ {K \to K _ {0}, K \in \mathcal {K} _ {\beta}} \| X _ {K} - X _ {K _ {0}} \| = 0. \tag {A.4}$$

That is, the minimal solution $X _ { K }$ to the Riccati equation (8) depends continuously on $K \in \mathrm { c l } ( \mathscr { K } _ { \beta } )$ .

The proof relies on the following nontrivial theorem that establishes the continuous dependence of minimal solutions on the coefficients of Riccati equations.

Theorem 8 ([44, Theorem 11.2.1]) The minimal solution X to the Riccati equation $A X + \dot { X } A ^ { \mathsf { T } } + X D X + C = 0$ is a continuous function of the coefficient $( A , C , D )$ on the set of all $( A , C , \dot { D } )$ such that $D \succeq \ddot { 0 } , C = \dot { C } ^ { \mathsf { T } }$ and $( \cdot \cal { A } , \cal { D } )$ is stabilizable.

To show (A.4), we apply Theorem 8 to the Riccati equation (8) corresponding to any $K \in \mathrm { c l } ( \mathscr { K } _ { \beta } )$ . It is not difficult to check that the coefficient matrices ${ \bf \Delta } \cdot { \bf \Delta } \cdot { \bf \Delta } W , \beta ^ { - 2 } S _ { K }$ depend continuously on K and satisfy all the conditions required by Theorem 8. Consequently, the minimal solution $X _ { K }$ depends continuously on $\bar { K } \in \mathrm { { c l } } ( \mathcal { K } _ { \beta } )$ ), proving (A.4).
