# A. Tracking Problem of Networked MAS

Consider the nonlinear MAS with $N \in  { \mathbb { N } } _ { + }$ agents, whose dynamics is described as

$$\dot {x} _ {\mathrm{p}} ^ {i} = f _ {\mathrm{p}} ^ {i} (x _ {\mathrm{p}}, u _ {i}, w _ {\mathrm{p}}), \quad y _ {\mathrm{p}} ^ {i} = g _ {\mathrm{p}} ^ {i} (x _ {\mathrm{p}} ^ {i}), \tag {2}$$

where $i \in \mathcal { N } : = \{ 1 , \dots , N \}$ ; for the i-th agent, $\boldsymbol { x } _ { \mathrm { p } } ^ { i } \in \mathbb { R } ^ { n _ { \mathrm { p } } ^ { i } }$ is the agent state, $u _ { i } \in \mathbb { R } ^ { n _ { u } ^ { 2 } }$ is the control input, $w _ { \mathfrak { p } } \in \mathbb { R } ^ { n _ { 1 } }$ is the external disturbance, and $y _ { \mathrm { p } } ^ { i } \in \mathbb { R } ^ { n _ { y } ^ { i } }$ is the output. For the i-th agent, its reference system to be tracked is given by

$$\dot {x} _ {\mathrm{r}} ^ {i} = f _ {\mathrm{p}} ^ {i} (x _ {\mathrm{r}}, u _ {\mathrm{f}} ^ {i}, w _ {\mathrm{r}}), \quad y _ {\mathrm{r}} ^ {i} = g _ {\mathrm{p}} ^ {i} (x _ {\mathrm{r}} ^ {i}), \tag {3}$$

where $x _ { \mathrm { r } } ^ { i } ~ \in ~ \mathbb { R } ^ { n _ { \mathrm { p } } ^ { i } }$ is the reference state, $u _ { \mathrm { f } } ^ { i } ~ \in ~ \mathbb { R } ^ { n _ { u } ^ { i } }$ is the feedforward control input of the reference system, $w _ { \mathrm { r } } \in \mathbb { R } ^ { n _ { 2 } }$ is the external disturbance, and $y _ { \mathrm { r } } ^ { i } \in \mathbb { R } ^ { n _ { y } ^ { 2 } }$ is the reference output.

Let $x _ { \mathfrak { p } } : = ( x _ { \mathfrak { p } } ^ { 1 } , \ldots , x _ { \mathfrak { p } } ^ { N } ) \in \mathbb { R } ^ { n _ { \mathfrak { p } } }$ and $x _ { \mathrm { r } } : = ( x _ { \mathrm { r } } ^ { 1 } , \ldots , x _ { \mathrm { r } } ^ { N } ) \in$ Rnp with $\begin{array} { r } { n _ { \mathrm { p } } : = \sum _ { i = 1 } ^ { N } n _ { \mathrm { p } } ^ { i } } \end{array}$ . In (2)-(3), f ip (or $f _ { \mathrm r } ^ { i } )$ is written as a function related to the overall vector $x _ { \mathrm { p } } \ \left( \mathrm { o r } \ x _ { \mathrm { r } } \right)$ , but it depends only on the states of the i-th agent (or reference) and its neighbor agents (or references); see Fig. 1. The physical coupling among all agents is characterized by a graph $\mathcal { G } _ { \mathrm { p } } : =$ $( \mathcal { N } , \mathcal { E } _ { \mathfrak { p } } )$ , whereas the physical coupling among all references is characterized by a graph $\mathcal { G } _ { \mathrm { r } } : = ( \mathcal { N } , \mathcal { E } _ { \mathrm { r } } )$ . For these two graphs, the following assumption is made; see also [8]–[10].

Assumption $I \colon { \mathcal { G } } _ { \mathfrak { p } }$ and $\mathcal { G } _ { \mathrm { r } }$ are undirected and connected.
