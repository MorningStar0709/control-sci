$$d \widehat {X} _ {t} = r \widehat {X} _ {t} d t + \lambda \widehat {\pi} _ {t} \mathcal {V} _ {t} d t + \widehat {\pi} _ {t} \sqrt {\mathcal {V} _ {t}} d W _ {t} ^ {1} + c (\eta - \vartheta) d t + c \vartheta \widehat {q} _ {t} d t + b \widehat {q} _ {t} d W _ {t}, \tag {3.13}$$

with $\widehat { X } _ { t } = \widehat { x }$ . Here $( \widehat { q } _ { t } , \widehat { \pi } _ { t } )$ denotes admissible reinsurance-investment strategy. A reinsuranceinvestment strategy $\widehat { u } _ { t } , t \in [ 0 , T ]$ is said to be admissible if it satisfies that $( \widehat { q } _ { t } , \widehat { \pi } _ { t } )$ is progressively measurable in regards to $\mathcal { F } _ { t }$ which satisfies $\begin{array} { r } { E \Big \lceil \int _ { 0 } ^ { T } b ^ { 2 } \widehat { q } _ { t } ^ { 2 } d t \Big \rceil < \infty } \end{array}$ and $\begin{array} { r } { E \left\lceil \int _ { 0 } ^ { T } \widehat { \pi } _ { t } ^ { 2 } \mathcal { V } _ { t } d t \right\rceil < } \end{array}$ $\infty _ { \mathrm { : } }$ , and there exists a unique strong solution to (3.13). The set of all admissible strategies is denoted by Π.

The aim of the insurer is to find the best strategy to maximize the expected utility of the terminal wealth with minimum guaranteed threshold $L \geq 0 .$ , that is

$$\sup _ {\widehat {u} \in \widehat {\Pi}} E \big [ U (\widehat {X} _ {T} - L) \big ], \tag {3.14}$$

where $\widehat { X } _ { t }$ is subject to (3.13) and $\widehat { X } _ { T } \ \geq \ L$ . To solve utility maximization (3.14), using $\begin{array} { r } { D ( t ) : = c ( \eta - \vartheta ) \int _ { t } ^ { T } e ^ { - r ( s - t ) } d s , X _ { t } : = \widehat { X } _ { t } + D ( t ) , \pi _ { t } : = \widehat { \pi } _ { t } / X _ { t } } \end{array}$ and $q _ { t } : = { \widehat { q } } _ { t } / X _ { t }$ , the wealth process (3.13) is rewritten as

$$d X _ {t} = X _ {t} \left[ r d t + \lambda \pi_ {t} \mathcal {V} _ {t} d t + \pi_ {t} \sqrt {\mathcal {V} _ {t}} d W _ {t} ^ {1} + c \vartheta q _ {t} d t + b q _ {t} d W _ {t} \right]. \tag {3.15}$$

Obviously, the reinsurance-investment strategy $( q _ { t } , \pi _ { t } )$ in (3.15) is also the admissible strategy and its admissible set is denoted by Π. Note that $X _ { T } = \widehat { X } _ { T }$ . Thus, the equivalent form of utility maximization problem (3.14) is

$$\sup _ {(q, \pi) \in \Pi} E \big [ U (X _ {T} - L) \big ], \tag {3.16}$$
