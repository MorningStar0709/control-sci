# A. Transforming Plant Model

We transform the uncertain plant $\tilde { P } _ { \Delta _ { p } }$ into the form of an LTI system interconnected with an uncertainty block. This form mirrors the form of the RINN controller, and enables the entire closed-loop system to be modeled as an LTI system interconnected with an uncertainty block (grouping the controller’s activation functions into the uncertainty block).

We simplify the LTI part of the uncertain plant (2), making an LTI system with inputs $w _ { p }$ and u˜ and outputs $v _ { p }$ and y. Combining $\begin{array} { r } { u = v _ { p } + \frac { 1 - \sigma } { 2 } \dot { w _ { p } } } \end{array}$ 2 and $\begin{array} { r } { v _ { p } = \tilde { u } + \frac { 1 + \sigma } { 2 } \dot { w } _ { p } } \end{array}$ 1+σ2 wp gives $\boldsymbol { u } = \boldsymbol { w _ { p } } + \boldsymbol { \tilde { u } }$ . Plugging this into (1):

$$
\begin{array}{l} \left[ \begin{array}{c} \dot {x} _ {p} (t) \\ y (t) \end{array} \right] = \left[ \begin{array}{c c} A _ {p} & B _ {p} \\ C _ {p} & 0 \end{array} \right] \left[ \begin{array}{c} x _ {p} (t) \\ u (t) \end{array} \right] \\ = \left[ \begin{array}{c c c} A _ {p} & B _ {p} & B _ {p} \\ C _ {p} & 0 & 0 \end{array} \right] \left[ \begin{array}{c} x _ {p} (t) \\ w _ {p} (t) \\ \tilde {u} (t) \end{array} \right]. \\ \end{array}
$$

We next add $v _ { p }$ as an output using $v _ { p } ( t )$ from (2):

$$
\left[ \begin{array}{c} \dot {x} _ {p} (t) \\ v _ {p} (t) \\ y (t) \end{array} \right] = \left[ \begin{array}{c c c} A _ {p} & B _ {p} & B _ {p} \\ 0 & \frac {1 + \sigma}{2} I & I \\ C _ {p} & 0 & 0 \end{array} \right] \left[ \begin{array}{c} x _ {p} (t) \\ w _ {p} (t) \\ \tilde {u} (t) \end{array} \right] \tag {5}
w _ {p} (t) = \Delta_ {p} (v _ {p}) (t).$$

Equation (5) gives the plant $\tilde { P } _ { \Delta _ { p } }$ in the form of an LTI system interconnected with an uncertainty, similar to the controller diagram in Figure 2, as opposed to the structured interconnection in Figure 1.
