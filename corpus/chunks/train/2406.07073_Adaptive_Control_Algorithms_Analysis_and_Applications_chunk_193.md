Example We will illustrate the application of this technique for the case of a recursive estimation scheme using a parallel adjustable predictor given by (4.23) (output error), and using the parameter adaptation algorithm of (4.30) where we will take $\nu ( t + 1 ) = \varepsilon ( t + 1 )$ .

In order to apply the ODE of (4.36) for convergence analysis, one needs to calculate $f ( { \hat { \theta } } )$ for this scheme as defined in (4.34).

From (4.26) one has:

$$
\begin{array}{l} \nu (t + 1) = \varepsilon (t + 1) \\ = - \sum_ {i = 1} ^ {n} a _ {i} v (t + 1 - i) + [ \theta - \hat {\theta} (t + 1) ] ^ {T} \phi (t) + w (t + 1) \tag {4.40} \\ \end{array}
$$

For analysis purposes only, making $ { \hat { \theta } } ( t ) =  { \hat { \theta } }$ , one gets (after passing the term $\textstyle - \sum _ { i = 1 } ^ { n } a _ { i } \nu ( t + 1 - i )$ in the right hand side):

$$A (q ^ {- 1}) \nu (t + 1, \hat {\theta}) = (\theta - \hat {\theta}) ^ {T} \phi (t, \hat {\theta}) + w (t + 1) \tag {4.41}$$

where:

$$A (q ^ {- 1}) = 1 + a _ {1} q ^ {- 1} + \dots + a _ {n} q ^ {- n} = \frac {1}{H (q ^ {- 1})} \tag {4.42}$$

Passing both sides of (4.41) through $H ( q ^ { - 1 } )$ , one gets:

$$\nu (t + 1, \hat {\theta}) = H (q ^ {- 1}) (\theta - \hat {\theta}) ^ {T} \phi (t, \hat {\theta}) + w ^ {\prime} (t + 1) \tag {4.43}$$

where:

$$w ^ {\prime} (t + 1) = H (q ^ {- 1}) w (t + 1) \tag {4.44}$$

Using (4.43), $f ( { \hat { \theta } } )$ will be given by:

$$
\begin{array}{l} f (\hat {\theta}) = \mathbf {E} \left\{\phi (t, \hat {\theta}) v (t + 1, \hat {\theta}) \right\} \\ = \mathbf {E} \{\phi (t, \hat {\theta}) H (q ^ {- 1}) \phi^ {T} (t, \hat {\theta}) \} (\theta - \hat {\theta}) + \mathbf {E} \{\phi (t, \hat {\theta}) w ^ {\prime} (t + 1) \} \tag {4.45} \\ \end{array}
$$

and the associated ODE takes the form:

$$\frac {d \hat {\theta}}{d \tau} = - \mathbf {E} \{\phi (t, \hat {\theta}) H (q ^ {- 1}) \phi^ {T} (t, \hat {\theta}) \} (\hat {\theta} - \theta) + \mathbf {E} \{\phi (t, \hat {\theta}) w ^ {\prime} (t + 1) \} \tag {4.46}$$

Since u(t) and $w ( t + 1 )$ are assumed to be independent, one concludes that ${ \bf E } \{ \phi ( t , \hat { \theta } ) w ^ { \prime } ( t + 1 ) \} = 0 ( \phi ( t , \hat { \theta } )$ ) depends only on $u ( t ) )$ and the equilibrium points of (4.46) are given by:

$$\phi^ {T} (t, \hat {\theta}) [ \hat {\theta} - \theta ] = 0 \tag {4.47}$$
