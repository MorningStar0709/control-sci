Classically [4], an average state-space model can be obtained by applying Kirchhoff voltage law (KVL) and Kirchhoff current law (KCL) during the two operation modes. The state vector is $\mathbf { x } \overset { \cdot } { = } [ I _ { L _ { 1 } } ^ { ' } I _ { L _ { 2 } } \overset {  } { V } _ { C _ { 1 } } V _ { C _ { 2 } } \hat { V } _ { C _ { I N } } V _ { C _ { o u t } } ] ^ { T }$ , the output is $y =$ $V _ { C _ { o u t } } ,$ the controlled input is the duty-cycle $d ( t ) \ \bar { \in } \ [ 0 . 1 , \bar { 0 . 9 } ]$ and the disturbance is $\begin{array} { r } { \dot { \mathbf { w } } = [ V _ { I N } \quad \delta _ { R _ { v a r } } ] ^ { T } } \end{array}$ . The average statespace model can be expressed as :

$$
\begin{array}{r l} \mathbf {K} \dot {\mathbf {x}} (t) & = (\mathbf {A} _ {0} + d (t) \mathbf {A} _ {1}) \mathbf {x} (t) + (\mathbf {B} _ {0} + d (t) \mathbf {B} _ {1}) \mathbf {w} (t) \\ y (t) & = \mathbf {C x} (t) \end{array} \tag {1}
$$

The matrices K, A0, $\mathbf { A } _ { 1 } \in \mathbb { R } ^ { 6 \times 6 }$ , B0, $\mathbf { B } _ { 1 } \in \mathbb { R } ^ { 6 \times 2 }$ , $\mathbf { C } \in \mathbb { R } ^ { 1 \times 6 }$ , are given in Table II.
