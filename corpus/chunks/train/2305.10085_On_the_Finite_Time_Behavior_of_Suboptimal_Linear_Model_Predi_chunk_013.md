# IV. FINITE-TIME ANALYSIS

The exponential stability of the combined dynamics (5) allows one to study the finite-time performance of the suboptimal LQMPC. In this section, we quantify the incurred suboptimality (7) of suboptimal dynamics (5) both for varying and for fixed number of optimization iterations ℓ. Before proceeding with the suboptimality analysis we introduce some auxiliary lemmas.

Lemma 4: Given the dynamics (5), for all $s _ { 0 } \in \Sigma _ { N } , k \ge 0$ and $\ell _ { k } > \ell ^ { \star }$

$$\| \Delta \mu_ {k} \| := \| z _ {k} - \mu^ {\star} (x _ {k} ^ {\star}) \| \leq b _ {0} \| x _ {0} \| _ {W} \prod_ {i = - 1} ^ {k - 1} \varepsilon_ {i} + c \| x _ {0} \| _ {W} \beta^ {k},$$

where $c = \operatorname* { m a x } \{ \tau ^ { - 1 } , \| H ^ { - \frac { 1 } { 2 } } \| \| H ^ { - \frac { 1 } { 2 } } G P ^ { - \frac { 1 } { 2 } } \| \} , b _ { 0 } = c \cdot h _ { 0 }$ and $h _ { 0 }$ is defined as in Theorem 3.

Proof: The proof follows by showing that $\| \Delta \mu _ { k } \|$ is upper bounded by the the sum of two Lyapunov functions and enjoys the same rate of decay as those. In particular, for any $k \geq 0$ and $s _ { k } \in \Sigma _ { N }$

$$
\begin{array}{l} \| \Delta \mu_ {k} \| = \| z _ {k} - \mu^ {\star} (x _ {k}) + \mu^ {\star} (x _ {k}) - \mu^ {\star} (x _ {k} ^ {\star}) \| \\ \leq \left\| z _ {k} - \mu^ {\star} (x _ {k}) \right\| + \left\| \mu^ {\star} (x _ {k}) - \mu^ {\star} (x _ {k} ^ {\star}) \right\| \\ \leq \tau^ {- 1} \tau \| d (z _ {k}, x _ {k}) \| + \| H ^ {- \frac {1}{2}} \| \| H ^ {- \frac {1}{2}} G (x _ {k} - x _ {k} ^ {\star}) \| \\ \leq \tau^ {- 1} \tau \| d (z _ {k}, x _ {k}) \| \\ + \| H ^ {- \frac {1}{2}} \| \| H ^ {- \frac {1}{2}} G P ^ {- \frac {1}{2}} \| (\| x _ {k} \| _ {P} + \| x _ {k} ^ {\star} \| _ {P}) \\ \leq c \left(\underbrace {\tau \| d (z _ {k} , x _ {k}) \| + \| x _ {k} \| _ {P}} _ {:= l (x _ {k}, z _ {k})} + \underbrace {\| x _ {k} ^ {\star} \| _ {P}} _ {:= g (x _ {k} ^ {\star})}\right), \\ \end{array}
$$

where the first and third inequalities make use of the triangle inequality, and the second one follows from Lemma 1. From Lemma 2 and (10), we note that $l ( x _ { k } , z _ { k } ) \le \mathcal { L } ( x _ { k } , z _ { k } )$ and $g ( x _ { k } ^ { \star } ) \leq \psi ( x _ { k } ^ { \star } )$ , and it holds that

$$\left\| \Delta \mu_ {k} \right\| \leq c \left(\mathcal {L} (x _ {k}, z _ {k}) + \psi (x _ {k} ^ {\star})\right).$$

The result follows by a repeated application of (9) and (12).
