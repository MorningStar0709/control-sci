# A. Implementation

We developed a custom training environment based on the equations of motion (7)–(10) using OpenAI Gym library, as shown in Figure 3, where the equations of motion are numerically integrated. Our proposed frameworks can be trained with any SARL and MARL algorithm, and in this study, we selected TD3 and MATD3, which are model-free, off-policy, actor-critic algorithms. After completing training in our simplified custom environment designed to prioritize computational efficiency, we evaluated the trained policies in a more realistic PyBullet environment for sim-to-sim verification. This evaluation allows us to assess how well the policies generalize to a more complex and realistic simulation setting. The code of our experiments can be accessed at https: //github.com/fdcl-gwu/gym-rotor.

Network Structures. In our experiments, both critic and actor networks were implemented as MLPs with two hidden layers with ReLU activations. The critic networks employed 512 nodes per hidden layer for all frameworks. In contrast, the actor networks utilized a variable number of hidden nodes depending on the agent: 16 nodes for SARL and Agent#1 in MARL frameworks, while 8 nodes for Agent#2 due to its smaller input dimension. Lastly, the actor network’s output layer used the hyperbolic tangent activations to ensure control signals remain within the range of [−1, 1].

Hyperparameters. The reward coefficients $k _ { x }$ and $k _ { b _ { 1 } }$ in Section IV were set to 7.0 and 2.0, respectively, to minimize tracking errors in position and heading. To eliminate steadystate errors, $k _ { I _ { 3 } }$ and $k _ { I _ { b _ { 1 } } }$ were chosen as 0.04 and 0.06, with $\alpha = 0 . 0 1$ in (22). Also, to achieve smooth and stable control, the penalizing terms were set to $k _ { v } = k _ { \Omega } = k _ { \Omega _ { 3 } } = k _ { \omega _ { 1 2 } } =$ 0.25 and $k _ { b _ { 3 } } ~ = ~ 3 . 5$ . Note that excessively large penalties prioritized stabilization over position tracking and prevented the quadrotor from moving toward its target.
