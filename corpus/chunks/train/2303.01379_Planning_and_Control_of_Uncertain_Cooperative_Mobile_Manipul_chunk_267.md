also known as a sphere world [84]. We consider the following common feasibility assumption [84] for ${ \mathcal { F } } \colon$ :

Assumption 4.2. The workspace W and the obstacles ${ \mathcal { O } } _ { j }$ satisfy:

$$\left\| c _ {i} - c _ {j} \right\| > r _ {o _ {i}} + r _ {o _ {j}} + 2 rr _ {\mathcal {W}} - \left\| c _ {j} \right\| > r _ {o _ {j}} + 2 r.$$

The aforementioned assumption implies essentially that there is enough space among the obstacles and the workspace boundary and the obstacles for the robot to navigate, or equivalently, $\bar { \mathcal { O } } _ { j } \subset \bar { \mathcal { W } }$ and $\bar { \mathcal { O } } _ { i } \cap \bar { \mathcal { O } } _ { j } = \emptyset , \forall i , j \in \mathcal { I }$ with $i \neq j$ .

Moreover, Assumption 4.2 implies that we can find some $\bar { r } > 0$ such that

$$\left\| c _ {i} - c _ {j} \right\| > r _ {o _ {i}} + r _ {o _ {j}} + 2 r + 2 \bar {r}, \forall i, j \in \mathcal {J}, i \neq j, \tag {4.5a}r _ {\mathcal {W}} - \left\| c _ {j} \right\| > r _ {o _ {j}} + 2 r + 2 \bar {r}, \quad \forall j \in \mathcal {J} \tag {4.5b}$$

This section treats the problem of navigating the robot to a destination $x _ { \mathrm { d } }$ while avoiding the obstacles and the workspace boundary, formally stated as follows:

Problem 4.1. Consider a robot subject to the uncertain dynamics (4.1), operating in the aforementioned sphere world, with $( x ( t _ { 0 } ) , v ( t _ { 0 } ) ) \in \mathcal { F } \times \mathbb { R } ^ { n }$ Given a destination $x _ { \mathrm { d } } \in { \mathcal { F } }$ , design a control protocol u such that

$$
\begin{array}{l} x (t) \in \mathcal {F}, t \geq t _ {0} \\ \lim _ {t \to \infty} (x (t), v (t)) = (x _ {\mathrm{d}}, 0) \\ \end{array}
$$
