$$
\begin{array}{l} \| A _ {d} - I \| = \| e ^ {A _ {c} \tau} - I \| = \left\| \sum_ {k = 1} ^ {\infty} \frac {(A _ {c} \tau) ^ {k}}{k !} \right\| \\ \leq \sum_ {k = 1} ^ {\infty} \frac {\left\| A _ {c} \tau \right\| ^ {k}}{k !} = e ^ {\left\| A _ {c} \tau \right\|} - 1 <   e ^ {\ln 2} - 1 = 1. \tag {34} \\ \end{array}
$$

Substituting (34) into $\begin{array} { r } { \| \log ( A _ { d } ) \| \leq \sum _ { k = 1 } ^ { \infty } \frac { \| A _ { d } - I \| ^ { k } } { k } } \end{array}$ , it follows that $\left\| \log ( A _ { d } ) \right\|$ will converge.

Next, we show that log $( A _ { d } ) \ = \ A _ { c } \tau$ holds. For simplicity, we suppose that $A _ { c }$ is diagonalizable (when $A _ { c }$ is not diagonalizable, it can be approximated by a sequence of diagonalizable matrices). Then, $A _ { c }$ can be expressed as $A _ { c } = C J C ^ { - 1 } = C \mathrm { d i a g } \{ \lambda _ { 1 } , \cdot \cdot \cdot , \lambda _ { N n } \} C ^ { - 1 }$ , and it holds that

$$(A _ {d} - I) ^ {k} = C \operatorname{diag} \left\{\left(\tilde {\lambda} _ {i} - 1\right) ^ {k}, \dots , \left(\tilde {\lambda} _ {N n} - 1\right) ^ {k} \right\} C ^ {- 1}. \tag {35}$$

Since $\| A _ { c } \tau \| ~ < ~ \ln 2$ indicates $| \lambda _ { i } \tau | \ < \ \ln 2 .$ , it follows by holomorphicity that $\log ( e ^ { \lambda _ { i } \tau } ) ~ { \stackrel { . } { = } } ~ \lambda _ { i } \tau$ exactly even if $\lambda _ { i }$ is a complex number (see [44, Lemma 2.6]). Finally, we have

$$
\begin{array}{l} \sum_ {k = 1} ^ {\infty} (- 1) ^ {k + 1} \frac {(A _ {d} - I) ^ {k}}{k} = C \operatorname{diag} \left\{\log \tilde {\lambda} _ {i}, \dots , \log \tilde {\lambda} _ {N n} \right\} C ^ {- 1} \\ = C \operatorname{diag} \left\{\lambda_ {1} \tau , \dots , \lambda_ {N n} \tau \right\} C ^ {- 1} = A _ {c} \tau , \tag {36} \\ \end{array}
$$

where log $\begin{array} { r } { \tilde { \lambda } _ { i } = \sum _ { k = 1 } ^ { \infty } ( - 1 ) ^ { k + 1 } \frac { ( \tilde { \lambda } _ { i } - 1 ) ^ { k } } { k } } \end{array}$ is used in the first row. Therefore, $A _ { c }$ can be reconstructed by $\textstyle { \frac { 1 } { \tau } } \log ( A _ { d } )$ . The proof is completed. □
