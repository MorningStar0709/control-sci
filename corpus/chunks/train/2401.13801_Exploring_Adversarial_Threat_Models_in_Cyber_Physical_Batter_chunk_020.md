To solve these two co-state equations (9) and (10), λ is chosen of the form $\lambda = S X - V .$ , where S and V are variables to be determined [30], [31]. Upon taking the derivative of this equation with respect to time, we get the following form:

$$\dot {\lambda} = \dot {S} X + S \dot {X} - \dot {V}\Longrightarrow \dot {\lambda} = \dot {S} X + S [ A X + B U _ {a} + B U _ {n o m} ] - \dot {V} \tag {11}$$

Upon substituting the expression of $\dot { \lambda }$ and λ into (10), we get

$$
\begin{array}{l} Q _ {2} [ X - X _ {r e f} ] + A ^ {T} [ S X - V ] \\ = - \dot {S} X - S \left[ A X - B R ^ {- 1} B ^ {T} (S X - V) + B U _ {\text {nom}} \right] + \dot {V}. \tag {12} \\ \end{array}
$$

The equation (12) can be decoupled as two Riccati equations:

$$\dot {V} + A ^ {T} V - S B R ^ {- 1} B ^ {T} V - S B U _ {\text {nom}} + Q X _ {\text {ref}} = 0, \tag {13}\dot {S} + S A - S B R ^ {- 1} B ^ {T} S + A ^ {T} S + Q = 0 \tag {14}$$

The boundary conditions are computed to be:

$$S (T) = Q _ {1}, V (T) = Q _ {1} X _ {r e f} (T) \tag {15}$$

Solving (13)-(14) will give us S and V , and subsequently, we can compute $U _ { a }$ as $U _ { a } = - R ^ { - 1 } B ^ { T } [ S X - V ]$ .

D. Open-loop model and real-time feedback-based output attack generation

Once the input current attack is generated, the next focus is on achieving stealthiness. That is, the adversary’s goal is not to trigger any alarm when the input attack is injected. To achieve such stealthiness, the adversary designs an output attack signal $Y _ { a }$ which corrupts the measured terminal voltage, as shown in (6). This attack signal $Y _ { a }$ is generated in a way such that the effect of $U _ { a }$ is cancelled by $Y _ { a }$ at the terminal voltage output.

For this output attack generation, the adversary utilizes the knowledge of system model (5)-(6). To this end, $Y _ { a }$ is generated such that the following condition is satisfied:

$$
\begin{array}{l} Y = Y _ {n o m}, \\ \Longrightarrow g (X, U _ {n o m} + U _ {a}) + Y _ {a} = g \left(X _ {n o m}, U _ {n o m}\right), \\ \Longrightarrow Y _ {a} = g (X _ {\text { nom }}, U _ {\text { nom }}) - g (X, U _ {\text { nom }} + U _ {a}), \tag {16} \\ \end{array}
$$

where X is generated by solving $\dot { X } = A X + B ( U _ { n o m } + U _ { a } )$ , $X _ { n o m }$ is generated by solving $\dot { X } _ { n o m } = A X _ { n o m } + B U _ { n o m } ,$ and $Y _ { n o m } = g ( X _ { n o m } , U _ { n o m } )$ is the output under no attack condition. The condition (16) ensures that the measured output Y under the attack should resemble to the output $Y _ { n o m }$ under user-applied nominal input $U _ { n o m }$ .
