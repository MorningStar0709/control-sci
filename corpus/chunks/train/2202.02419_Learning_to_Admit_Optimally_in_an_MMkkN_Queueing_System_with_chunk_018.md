If $M _ { i } = 0$ for all $i ,$ the maximum of $l \left( \mathcal { H } _ { n } ; \mu \right)$ in $[ 0 , + \infty )$ is obtained at $\mu = 0$ , and if $N _ { i } = 0$ for all $i ,$ the maximum is reached as $\mu \to + \infty$ . The derivative of $l \left( \mathcal { H } _ { n } ; \mu \right)$ is given by

$$l ^ {\prime} \left(\mathcal {H} _ {n}; \mu\right) = \sum_ {i = 1} ^ {n} \frac {M _ {i} T _ {i} \exp (- \mu T _ {i})}{1 - \exp (- \mu T _ {i})} - \sum_ {i = 1} ^ {n} N _ {i} T _ {i}. \tag {8}$$
