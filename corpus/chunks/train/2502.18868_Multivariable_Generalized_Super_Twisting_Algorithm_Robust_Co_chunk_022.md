# 6 The realization of second-order sliding modes

To conclude the theoretical analysis, in this section, we show that a second-order sliding mode is achieved after some finite time. To accomplish this, let us redefine the variable $z ( t )$ in a more convenient manner for this purpose as

$$\bar {z} = \eta + (B K _ {2}) ^ {- 1} \left(f + C _ {0} \zeta\right) \tag {61}$$

where, from (9), $\bar { z } \ = \ z + ( B K _ { 2 } ) ^ { - 1 } C _ { 0 } \zeta$ and since we have already proven that ζ and z converge to zero, it is straightforward to see that ¯z also converges to zero.
