# C. Control Barrier Functions

The converted nonlinear system (3) can still be written in a nonlinear affine form as (1) with the total disturbances added. Without loss of generality, the following discussion about CBF-QP is based on a nonlinear affine form given by

$$\dot {x} = f (x) + g (x) u + d, \tag {4}$$

where $d \in \mathbb { R } ^ { n }$ includes all total disturbances corresponding to all measured outputs.

The definition of CBF in the context of uncertainty is given in [6]. A continuous and differentiable function $h : \mathbb { R } ^ { n } $ R is called a CBF for the system (4), if

$$\sup _ {u \in U} (L _ {f} h (x) + L _ {g} h (x) u + h _ {x} (x) d) \geq - \beta (h (x)) \tag {5}$$

for all $t \geq 0$ and $x \in X$ , where $\begin{array} { r } { h _ { x } ( x ) = \frac { \partial h ( x ) } { \partial x } } \end{array}$ , and $\beta ( s )$ is an extended class K function. In particular, we usually choose $\beta ( s ) = \gamma s$ for a constant $\gamma > 0$ . From (5), we have a set of control signals ensuring that the states of the system (4) are always in a safe set C if $h _ { x } ( x ) \neq 0$ for all $x \in \partial \mathcal { C }$ , and the initial states are in the safe set:

$$
\begin{array}{l} K _ {\mathrm{cbf}} (t, x, d) \triangleq \left\{u \in U: L _ {f} h (x) + L _ {g} h (x) u \right. \tag {6} \\ \left. + h _ {x} (x) d \geq - \gamma h (x) \right\} \\ \end{array}
$$

where $\partial \mathcal { C }$ represents the boundary of C.
