# Some Special Cases:

Case 1: $D _ { 1 2 } = I$

In this case

1. In part (a), (i) becomes $\gamma > \overline { { \sigma } } ( D _ { 1 1 2 1 } )$ .

2. In part (b)

$${\hat {D} _ {1 1}} = {- D _ {1 1 2 2}}\hat {D} _ {1 2} \hat {D} _ {1 2} ^ {*} = I - \gamma^ {- 2} D _ {1 1 2 1} D _ {1 1 2 1} ^ {*}\hat {D} _ {2 1} ^ {*} \hat {D} _ {2 1} = I.$$

Case 2: $D _ { 2 1 } = I$

In this case

1. In part (a), (i) becomes $\gamma > \overline { { \sigma } } ( D _ { 1 1 1 2 } )$ .   
2. In part (b)

$$\hat {D} _ {1 1} = - D _ {1 1 2 2}\hat {D} _ {1 2} \hat {D} _ {1 2} ^ {*} = I\hat {D} _ {2 1} ^ {*} \hat {D} _ {2 1} = I - \gamma^ {- 2} D _ {1 1 1 2} ^ {*} D _ {1 1 1 2}.$$

Case 3: ${ \cal D } _ { 1 2 } = I \ \& \ { \cal D } _ { 2 1 } = I$

In this case

1. In part (a), (i) drops out.   
2. In part (b)

$$\hat {D} _ {1 1} = - D _ {1 1 2 2}\hat {D} _ {1 2} \hat {D} _ {1 2} ^ {*} = I\hat {D} _ {2 1} ^ {*} \hat {D} _ {2 1} = I.$$
