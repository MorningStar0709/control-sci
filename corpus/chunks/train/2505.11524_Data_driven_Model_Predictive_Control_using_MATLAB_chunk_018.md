# 2.2 Conventional NMPC

Conventional NMPC uses a nonlinear model (in state-space form) of the system for designing MPC control law where the cost function is quadratic as in Eq. (11) or (22). Consider a discrete-time nonlinear system:

$$
\begin{array}{r l} \mathbf {x} _ {k + 1} & = \mathbf {f} (\mathbf {x} _ {k}, \mathbf {u} _ {k}) \\ \mathbf {y} _ {k} & = \mathbf {h} (\mathbf {x} _ {k}) \end{array} \tag {26}
$$

where $k \in \mathbb { T } , \mathbf { x } _ { k } \in \mathbb { X } \subseteq \mathbb { R } ^ { n } , \mathbf { u } _ { k } \in \mathbb { U } \subseteq \mathbb { R } ^ { m } , \mathbf { f } : \mathbb { X } \times \mathbb { U } \to \mathbb { X }$ is the state function that maps the current state $\mathbf { x } _ { k }$ to the next state $\mathbf { x } _ { k + 1 }$ for the control action ${ \bf u } _ { k } .$ , and h : $\colon \mathbb { X }  \mathbb { Y }$ is the output function. The input, state, and output constraint sets are defined as in (6). For a given initial condition ${ \bf x } _ { k \left| k \right. }$ and control sequence ${ \bf U } _ { k }$ , the predicted state and output sequence can be computed as:

$$
\mathbf {X} _ {k + 1} = \left[ \begin{array}{c} \mathbf {x} _ {k + 1 | k} \\ \vdots \\ \mathbf {x} _ {k + N | k} \end{array} \right] = \left[ \begin{array}{c} \mathbf {f} (\mathbf {x} _ {k | k}, \mathbf {u} _ {k | k}) \\ \vdots \\ \mathbf {f} (\dots \mathbf {f} (\mathbf {x} _ {k | k}, \mathbf {u} _ {k | k}), \mathbf {u} _ {k + N - 1 | k}) \end{array} \right] = \mathbf {f} _ {\mathrm{xp}} (\mathbf {x} _ {k | k}, \mathbf {U} _ {k}) \tag {27}

\mathbf {Y} _ {k + 1} = \left[ \begin{array}{c} \mathbf {y} _ {k + 1 | k} \\ \vdots \\ \mathbf {y} _ {k + N | k} \end{array} \right] = \left[ \begin{array}{c} \mathbf {h} (\mathbf {x} _ {k + 1 | k}) \\ \vdots \\ \mathbf {h} (\mathbf {x} _ {k + N | k}) \end{array} \right] = \mathbf {f} _ {\mathrm{yp}} (\mathbf {x} _ {k | k}, \mathbf {U} _ {k}) \tag {28}
$$

which are used to construct model-based equality constraints in output-based and state-based NMPC as discussed next.
