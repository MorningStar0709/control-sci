$$
\begin{array}{l} \left\| \zeta (x) - \sum_ {j = 0} ^ {n} \lambda_ {j} \zeta (x _ {j}) \right\| _ {\infty} = \frac {1}{2} \sum_ {j = 0} ^ {n} \lambda_ {j} \left\| \overline {{\mathbf {H}}} _ {\zeta} (x, x _ {0}) - \overline {{\mathbf {H}}} _ {\zeta} (x _ {j}, x _ {0}) \right\| _ {\infty} \\ \leq \frac {1}{2} n \beta \sum_ {j = 0} ^ {n} \lambda_ {j} \bar {c} _ {j}, \tag {7} \\ \end{array}
$$

where $\{ \lambda _ { j } \} _ { j = 0 } ^ { n } \in \mathbb { R } _ { 0 } ^ { 1 }$ is the set of unique coefficients satisfying $\textstyle x = \sum _ { j = 0 } ^ { n } \lambda _ { j } x _ { j }$ with $\textstyle \sum _ { j = 0 } ^ { n } \lambda _ { j } = 1$ ,

$$\beta \geq \max _ {p, q, r \in \mathbb {Z} _ {1} ^ {n}} \max _ {\xi \in \sigma} \left| \frac {\partial^ {2} \zeta^ {(p)} (x)}{\partial x ^ {(q)} \partial x ^ {(r)}} \right| _ {x = \xi}, a n d \tag {8}\bar {c} _ {j} = \| x _ {j} - x _ {0} \| \left(\max _ {k \in \mathbb {Z} _ {1} ^ {n}} \| x _ {k} - x _ {0} \| + \| x _ {j} - x _ {0} \|\right). \tag {9}$$

In Appendix A, Lemma 18 establishes a similar, but tighter, error bound than the one presented above. The proof also uses Taylor’s Theorem to bound the change in a function across a simplex. But while Lemma 9 performs a Taylor expansion about a vertex point of the simplex, $x _ { 0 } ,$ Lemma 18 (Appendix A) performs the expansion about an arbitrary point within the simplex. Importantly, the behavior of these error bounds differs at the origin. When $x _ { j } = 0 , c _ { j } = 0$ in Lemma 9, while $c _ { j } \neq 0$ in Lemma 18. Thus, we will use the tighter bound from Lemma 18 throughout, except when simplices contain the origin. Then, we will revert to using Lemma 9.
