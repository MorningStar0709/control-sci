# C. Mixed-integer programming (MIP) formulations

In order to obtain formal guarantees on the behaviour of an NN controller, we wish to solve optimization problems over NNs to global optimality. In particular, ReLU NNs can be represented exactly using MIP [9]. Additionally, the solutions sets of parametric quadratic programs (including MPC policies) can be formulated exactly as MIPs [6]. As such, optimization problems that include both NNs and MPC policies can be formulated as combined MIPs and solved to global optimality using off-the-shelf MIP solvers.

Big-M formulation for ReLU. A simple way to represent the ReLU activation function, max $\left( W _ { i } ^ { ( l ) } z ^ { ( l - 1 ) } + b _ { i } ^ { ( l ) } , 0 \right)$ , is to use the so-called ‘Big-M’ formulation [10]:

$$z _ {i} ^ {(l)} \geq W _ {i} ^ {(l)} z ^ {(l - 1)} + b _ {i} ^ {(l)} \tag {5a}z _ {i} ^ {(l)} \leq \left(W _ {i} ^ {(l)} z ^ {(l - 1)} + b _ {i} ^ {(l)}\right) - \underline {{{M}}} _ {i} ^ {(l)} (1 - s) \tag {5b}0 \leq z _ {i} ^ {(l)} \leq \overline {{M}} _ {i} ^ {(l)} s \tag {5c}s \in \{0, 1 \} \tag {5d}$$

wher e W (l) $W _ { i } ^ { ( l ) }$ is the i-th row of the weight matrix, s is an auxiliary binary variable denoting the on/off state of the neuron, and M (l)i , $\underline { M } _ { i } ^ { ( l ) } , \overline { M } _ { i } ^ { ( l ) }$ are ‘Big-M constants’ that satisfy

$$\underline {{M}} _ {i} ^ {(l)} \leq \min _ {z ^ {(l - 1)}} W _ {i} ^ {(l)} z ^ {(l - 1)} + b _ {i} ^ {(l)} \leq \overline {{M}} _ {i} ^ {(l)} \tag {6}$$

From this we can observe that the big-M constants must be valid bounds of the pre-activation term. We call any function with such a mixed-integer formulation ‘MIP-representable.’ Since the composition of MIP-representable functions is also MIP-representable [6], we have that $f _ { N N } \left( z ^ { ( 0 ) } \right) = \hat { z } ^ { ( L ) }$ can be represented in mixed-integer form by adding constraints (5) for every neuron in the network.
