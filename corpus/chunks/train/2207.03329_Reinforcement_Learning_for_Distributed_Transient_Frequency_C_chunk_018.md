# 4.3. Distributed dynamic budget assignment

Before proceeding to the synthesis of distributed control policy, here we focus on the assignment of budgets introduced previously. Notice that both sufficient conditions (6) and (11) obtained above are naturally distributed, except for the non-sparse requirement $\begin{array} { r } { \sum _ { i = 1 } ^ { n } b _ { i } \ = \ 0 . } \end{array}$ . Indeed, the satisfaction of this equality requires coordination across the buses. Interestingly, a static, a priori budget assignment in general does not work. This is because, if $b _ { i } \neq 0$ , then (11) might require the control input to be infinitely large (instead of vanishing) when $\omega _ { i }$ approaches $\omega ^ { \infty }$ . Instead, the following result details a state-dependent budget assignment mechanism that ensures $b _ { i }$ approaches zero as ωi approaches $\omega ^ { \infty }$ and enforces $\textstyle \sum _ { i = 1 } ^ { n } b _ { i } = 0$ in a distributed way.

Proposition 4.3. (Distributed dynamic budget assignment): For $x \ \in \ \mathbb { R } ^ { \bar { m } + n } , \ l e t \ \mathcal { T } ^ { \mathrm { t h } } \ \subseteq \ \mathcal { I }$ denote the set of buses satisfying ωi $\notin \ : \left[ \underline { { \omega } } _ { i } ^ { \mathrm { t h } } , \overline { { \omega } } _ { i } ^ { \mathrm { t h } } \right]$ and ${ \mathcal { E } } ^ { \mathrm { t h } } \subseteq { \mathcal { E } }$ the set of edges between any pair of nodes in ${ \mathcal { T } } ^ { \mathrm { t h } }$ . Define the (possibly unconnected) state-dependent subgraph $\tilde { \mathcal { G } } = ( \mathcal { T } , \mathcal { E } ^ { \mathrm { t h } } )$ of $\mathcal { G } = ( \mathcal { T } , \mathcal { E } )$ and let $\tilde { \mathcal { L } }$ be its Laplacian matrix. Given $\xi \in \mathbb { R } ^ { n }$ , let the budgets be assigned as $b _ { i } = [ \tilde { \mathcal { L } } ] _ { i } \xi$ for each $i \in \mathcal { T }$ . Then the following holds:

1. $\textstyle \sum _ { i = 1 } ^ { n } b _ { i } = 0$ always holds;   
2. For each $i \in \mathcal { I } , b _ { i } = 0$ whenever $\omega _ { i } ( t ) \in [ \underline { { \omega } } _ { i } ^ { \mathrm { t h } } , \overline { { \omega } } _ { i } ^ { \mathrm { t h } } ]$ .
