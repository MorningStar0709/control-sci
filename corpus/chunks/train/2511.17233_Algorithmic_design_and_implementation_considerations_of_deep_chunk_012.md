4: Constraint tightening according to [3, §7]
5: Generate the reference trajectory ( $x_{t}^{r}, u_{t}^{r}$ ) by Algorithm 2
6: for each t do
7: measure state $x_{t}$ 8: if $t = T_{k}$ then
9: activate the training thread
10: take random samples from the replay buffer
11: freeze output layer weights to be $K_{T_{k}}$ 12: train the network
13: $k \leftarrow k + 1$ 14: Compute $K_{t}$ by Algorithm 3
15: if $t = t_{j}$ then
16: update $\phi_{j}$ in the main loop
17: $j \leftarrow j + 1$ 18: $u_{t}^{a} = -K_{t}^{\top} \phi_{j}(x_{t})$ 19: if ( $x_{t}, u_{t}^{a}$ ) satisfies the experience selection criterion then
20: Store ( $x_{t}, u_{t}^{a}$ ) in the replay buffer
21: Solve (10) to get $u_{t}^{m}$ 22: return $u_{t} = u_{t}^{a} + u_{t}^{m}$ 23: $t \leftarrow t + 1$
