# 3.1 Duality in steady-state regime

Using the duality relationship (9), the MSE in prediction (5) is expressed as:

$$J _ {T} ^ {\mathrm{est}} (L) = \operatorname{tr} \left[ X _ {T} (L) H ^ {\intercal} H \right] + \operatorname{tr} [ R ],$$

where $\begin{array} { r } { X _ { T } ( L ) : = A _ { L } ^ { T } P _ { 0 } ( A _ { L } ^ { \sf T } ) ^ { T } + \sum _ { t = 0 } ^ { T - 1 } A _ { L } ^ { t } ( Q + L R L ^ { \sf T } ) ( A _ { L } ^ { \sf T } ) ^ { t } } \end{array}$ . Define the set of Schur stabilizing gains

$$\mathcal {S} := \{L \in \mathbb {R} ^ {n \times m}: \rho (A _ {L}) < 1 \}.$$

For any $L \in S$ , in the steady-state, the mean-squared prediction error assumes the form,

$$\lim _ {T \to \infty} J _ {T} ^ {\text { est }} (L) = \operatorname{tr} \left[ X _ {\infty} (L) H ^ {\intercal} H \right] + \operatorname{tr} \left[ R \right],$$

where $X _ { \infty } ( L ) : = \operatorname* { l i m } _ { T \to \infty } X _ { T } ( L )$ and coincides with the unique solution X of the discrete Lyapunov equation $X = A _ { L } X A _ { \iota } ^ { \intercal } + Q + L R L ^ { \intercal }$ (existence of unique solution follows from $\rho ( A _ { L } ) < 1 )$ . Given the steady-state limit, we formally analyze the following constrained optimization problem:

$$\min _ {L \in \mathcal {S}} \leftarrow J (L) := \operatorname{tr} \left[ X _ {(L)} H ^ {\intercal} H \right], \quad \text { s.t. } \quad X _ {(L)} = A _ {L} X _ {(L)} A _ {L} ^ {\intercal} + Q + L R L ^ {\intercal}. \tag {10}$$

Remark 3. Note that the latter problem is technically the dual of the optimal LQR problem as formulated in [6] by relating ${ \bar { A ^ { ' } } }  A ^ { \intercal } , - H  B ^ { \intercal } , L  K ^ { \intercal }$ , and $H ^ { \bar { \tau } } H  \Sigma$ . However, the main difference here is that the product H⊺H may not be positive definite, for example, due to rank deficiency in H specially when m $< n$ (whereas $\underline { { \lambda } } ( \Sigma ) > 0$ appears in all of the bounds in $[ 6 , 1 4 ] )$ Thus, in general, the cost function $J ( L )$ is not necessarily coercive in $L ,$ which can drastically effect the optimization landscape. For the same reason, in contrast to the LQR case [6, 14], the gradient dominant property of $J ( \bar { L } )$ is not clear in the filtering setup. In the next section, we show that such issues can be avoided as long as the pair $( A , H )$ is observable. Also, the learning problem posed here is distinct from its LQR counterpart (see Table 1).
