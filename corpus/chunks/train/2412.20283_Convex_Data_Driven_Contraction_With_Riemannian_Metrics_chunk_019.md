# A. Linear System

First, we validate our results in a linear system. Consider the system given by:

$$
\dot {x} = \left[ \begin{array}{l l} 0. 4 2 8 5 & - 0. 4 2 9 8 \\ 0. 4 0 1 8 & 1. 3 0 3 6 \end{array} \right] x + \left[ \begin{array}{l l} - 0. 7 8 2 6 & 0. 7 7 3 1 \\ - 0. 5 1 1 0 & 0. 0 3 3 9 \end{array} \right] u. \tag {25}
$$

If $u ~ = ~ [ 0 , 0 ] ^ { \top }$ the above is unstable with eigenvalues at $0 . 7 2 9 1 , 1 . 0 0 3 0$ The objective is to find a controller u and metric tensor W that ensures the system is contractive from data observations. Solving (19) produces the following constant matrix $W \in S _ { n } ^ { + + }$ and scalar ρ

$$
W = \left[ \begin{array}{l l} 0. 1 0 1 9 & 0. 0 1 5 4 \\ 0. 0 1 5 4 & 0. 0 0 2 8 \end{array} \right], \rho = 9. 9 6 8 6. \tag {26}
$$

In order to close the loop in the underlying system, one needs to define $u ( x )$ from $\begin{array} { r } { \frac { \partial \boldsymbol { u } ( \boldsymbol { x } ) } { \partial \boldsymbol { x } } = - \frac { 1 } { 2 } \rho \boldsymbol { G } ^ { \dagger } \boldsymbol { W } ^ { - 1 } } \end{array}$ . Following [25], but simplifying the control input to track the origin:

$$u (x) = \int_ {0} ^ {1} - \frac {1}{2} \rho (\gamma (s)) G ^ {\top} W (\gamma (s)) ^ {- 1} \frac {\partial \gamma (s)}{\partial s} d s \tag {27}$$

where $\gamma \ : \ [ 0 , 1 ] \  \ \mathbb { R } ^ { 2 } , \gamma ( 0 ) \ = \ [ 0 , 0 ] ^ { \top } , \gamma ( 1 ) \ = \ x$ is the geodesic under the norm induced by $W ^ { - 1 }$ , which if constant implies $\gamma ( s ) = s x .$ . Since both $\rho , W$ are constant we have that the feedback controller that makes the system contractive is $\begin{array} { r } { u ( x ) = - \frac { 1 } { 2 } \rho G ^ { \top } W ^ { - 1 } x } \end{array}$ . Note that in this case a similar result can be obtained using Krasovskii’s criteria [7].
