# A. Structure of the algorithm

In the case of IMU sensors, the Multiplicative Extended Kalman Filter (MEKF) takes as input a sequence of sensor measurements $\omega _ { k } , A _ { k } ^ { \mathcal { R } }$ , and $\boldsymbol { M } _ { k } ^ { \mathcal { R } }$ .

At each sample k, MEKF updates an inner state vector xˆ in two steps: a prediction step and an update step. The structure of xˆ is a 6-dimensional vector, where the first three components correspond to the logarithm of a unit quaternion (axis–angle representation), and the last three components correspond to the gyroscope bias.

• Prediction step: compute q using gyroscope data, and update

$$\hat {x} [ 1: 3 ] \leftarrow \log (q),\hat {x} [ 4: 6 ] \leftarrow \hat {x} [ 4: 6 ].$$

• Update step: in this step, MEKF corrects the inner state vector using accelerometer and magnetometer information. In static conditions, one can recover a unique quaternion q from $\boldsymbol { A } _ { k } ^ { \mathcal { R } }$ and $\boldsymbol { M } _ { k } ^ { \mathcal { R } }$ , using the identity $q _ { k } ^ { \mathcal { G } } \mathbf { \Xi } \cdot \left( \mathbf { A } _ { k } ^ { \mathcal { R } } \otimes \mathbf { M } _ { k } ^ { \mathcal { R } } \right) \mathbf { \Xi } \cdot \left( q _ { k } ^ { \mathcal { G } } \right) ^ { - 1 } \mathbf { \Xi } = \mathbf { \Xi } \left( q _ { k } ^ { \mathcal { G } } \cdot \mathbf { A } _ { k } ^ { \mathcal { R } } \left( q _ { k } ^ { \mathcal { G } } \right) ^ { - 1 } \right)$ ⊗ $\left( q _ { k } ^ { \mathcal { G } } \cdot \mathbf { M } _ { k } ^ { \mathcal { R } } \left( q _ { k } ^ { \mathcal { G } } \right) ^ { - 1 } \right)$ , where $\otimes$ denotes the cross product. In other words, the knowledge of the rotation of two orthogonal vectors is sufficient to determine a third one, and thus uniquely defines a quaternion.
