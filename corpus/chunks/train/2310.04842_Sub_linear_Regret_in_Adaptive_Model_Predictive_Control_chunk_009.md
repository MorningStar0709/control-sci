# 3.2. Polytopic tubes and associated linear constraints

With the considered control inputs, the system can be rewriten as

$$x _ {t + 1} = \Phi (\theta^ {\star}) x _ {t} + B (\theta^ {\star}) (v _ {0 | t} + \zeta_ {t}) + w _ {t}.$$

We apply a tube-based approach, and at time t, we build a polytopic tube based on:

(i) $\Theta _ { t } ,$ encoding the uncertainty about $\theta ^ { \star }$ . We denote by m the number of vertices of $\Theta _ { t }$ , and the vertices themselves by $\theta _ { t } ^ { ( j ) } , j = 1 , \dots , m .$ .

if $t < t^{\star}(\delta)$ then

Compute matrices $H_t^{(j)}$ Solve $\mathcal{P}_N(x_t, \theta_{\rho(t)})$ and obtain $v_{0|t}^\pi$ Apply $u_t = Kx_t + v_{0|t}^\pi + \zeta_t$ end for

To define $\bar { \mathcal { W } } _ { t }$ , let $\bar { B } _ { t } = \operatorname* { m a x } _ { \theta \in \Theta _ { t } } \| B ( \theta ) \| _ { 2 }$ .
