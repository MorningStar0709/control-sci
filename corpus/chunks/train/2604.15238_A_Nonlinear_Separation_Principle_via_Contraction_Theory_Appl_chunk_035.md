# B. Contracting observer design

We now consider the dual problem of state estimation. We seek to estimate $x \in \mathbb { R } ^ { n }$ , given the output $y \in \mathbb { R } ^ { p }$ , and we propose the following observer architecture:

$$\dot {\hat {x}} = - \hat {x} + \Psi (W \hat {x} + B u + L (y - \hat {y})), \quad \hat {y} = C \hat {x}, \tag {55}$$

where $\hat { x } \in \mathbb { R } ^ { n }$ is the estimated state, and L is the observer gain matrix.

Proposition 30 (S-Contracting observer design): Consider the observer (55) with exogenous inputs u and $y .$

(i) The observer is an FRNN with state xˆ, and a weight matrix $W _ { o b s } = W - L C$ .   
(ii) The observer is S-contracting for a given contraction rate $c \in \mathsf { \Gamma } ( 0 , 1 ]$ if and only if there exist $P \ \succ \ 0 .$ , diagonal $Q \succ 0$ and a matrix M such that,

$$
\left[ \begin{array}{c c} - 2 (1 - c) P & P + W ^ {\top} Q - C ^ {\top} M ^ {\top} \\ P + Q W - M C & - 2 Q \end{array} \right] \preceq 0. \tag {56}
$$

If (56) holds, the observer gain guaranteeing contraction is given by $L = Q ^ { - 1 } M$ .

Proof: The proof is presented in Appendix III-B

Corollary 31 (Necessity of linear detectability): An FRNN with synaptic matrix W , and output matrix C can be observed by an S-contracting FRNN only if the pair $( W - I _ { n } , C )$ is detectable.

Proof: Pre- and post-multiplying (56) by $\begin{array} { r l } { [ I _ { n }  } & { { } I _ { n } ] } \end{array}$ and its transpose, respectively, yields

$$Q W + W ^ {\top} Q - 2 Q - M C - C ^ {\top} M ^ {\top} + 2 c P \preceq 0.$$

Substituting $M = Q L$ and using the fact that $- 2 c P \prec 0 ,$ ,

$$Q (W - I _ {n} - L C) + (W - I _ {n} - L C) ^ {\top} Q \prec 0$$

which is the standard continuous-time Lyapunov condition for the pair $( W - I _ { n } , C )$ to be detectable.

Corollary 32 (Necessary and sufficient conditions $f o r \ S -$ contracting observer design): Let $\Pi _ { C }$ denote a matrix whose columns form a basis for the null space of C, such that $C \Pi _ { C } = 0$ . An FRNN with synaptic matrix W , and output matrix $C$ can be observed by an S-contracting FRNN if and only if the following strict inequality holds for some $c > 0 :$

$$
\left[ \begin{array}{c c} \Pi_ {C} ^ {\top} & 0 \\ 0 & I _ {n} \end{array} \right] \left[ \begin{array}{c c} - 2 P & P + W ^ {\top} Q \\ P + Q W & - 2 Q \end{array} \right] \left[ \begin{array}{c c} \Pi_ {C} & 0 \\ 0 & I _ {n} \end{array} \right] \prec 0. \tag {57}
$$
