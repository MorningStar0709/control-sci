# C. Consensus Performance Analysis

To analyze the control performance under distributed control law (3) with auxiliary dynamics (4) and event-triggered online learning (11), the error between the state $x _ { i }$ of agent i and their ideal average ${ \bar { x } } ^ { * }$ defined in Definition 1 is studied, which is shown as follows.

Theorem 1: Consider a MAS containing N agents under a connected undirected topology  with individual dynamics (1) satisfying Assumption 1, which is controlled by (3) with (4). To infer the unknown function $f ( \cdot )$ in (3), a GP model satisfying Assumption 3 is employed for each agent with a data set under Assumption 2. Moreover, each agent adopts the decentralized event-trigger (5) and (11) for online learning by adding new data pairs into the individual data set. Choose $\delta \in ( 0 , 1 / N )$ , then the system achieves probabilistic ǫ-average consensus with $\epsilon = 2 c ^ { - 1 } N \underline { { \eta } }$ , i.e.,

$$\operatorname * {P r} \left\{\lim _ {t \rightarrow \infty} \| \boldsymbol {x} (t) - \mathbf {1} _ {N} \bar {x} ^ {*} \| \leq \epsilon \right\} \geq 1 - N \delta . \tag {17}$$

The proof of Theorem 1 is divided into two parts, namely the consensus of the auxiliary state (4) in Lemma 2 and the ultimate boundness of the consensus error in Lemma 3.

Lemma 2: Consider a multi-agent system with N agents under a connected undirected graph ${ \mathcal { G } } ,$ whose auxiliary dynamics is designed as in (4), such that $\dot { \bar { \pmb { x } } } = - \bar { c } \pmb { \mathcal { L } } \bar { \pmb { x } }$ with $\bar { \pmb x } =$ $[ \bar { x } _ { 1 } , \cdots , \bar { x } _ { N } ] ^ { T }$ . Then, all the observed auxiliary states $\bar { x } _ { i }$ for $\forall i \in \mathcal { V }$ converge to the average state ${ \bar { x } } ^ { * }$ recalled from Definition 1 as $\bar { \boldsymbol { x } } ^ { * } = N ^ { - 1 } \mathbf { 1 } _ { N } ^ { T } \boldsymbol { x } ( 0 )$ , i.e., lim $\mathfrak { l } _ { t  \infty } \bar { \pmb x } ( t ) = { \mathbf 1 } _ { N } \bar { x } ^ { * } / N$ .
