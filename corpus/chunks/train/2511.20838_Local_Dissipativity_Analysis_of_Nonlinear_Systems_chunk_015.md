# Theorem 10 Consider

$$
\mathbf {M} (x) = \left[ \begin{array}{c c} \phi (x) & \mathbf {0} \\ \mathbf {0} & - \mathbf {I} \end{array} \right] + H e \left(\left[ \begin{array}{c c} 0 & \zeta (x) ^ {\top} \\ \mathbf {0} & \mathbf {0} \end{array} \right] \left[ \begin{array}{c c} 0 & \mathbf {0} \\ \mathbf {0} & \mathbf {I} \end{array} \right]\right), \tag {10}
$$

where $x \in \mathbb { R } ^ { n } , \phi : \mathbb { R } ^ { n } \to \mathbb { R } , \zeta : \mathbb { R } ^ { n } \to \mathbb { R } ^ { m } , \phi , \zeta \in$ $\mathcal { C } ^ { 2 }$ . Let $\sigma : = c o \{ x _ { j } \} _ { j = 0 } ^ { n }$ be an n-simplex in $\mathbb { R } ^ { n } . ~ I f ~ x =$ $\textstyle \sum _ { j = 0 } ^ { n } \lambda _ { j } x _ { j } \in \sigma ,$ then

$$
\begin{array}{l} \mathbf {M} (x) - \sum_ {j = 0} ^ {n} \lambda_ {j} \mathbf {M} (x _ {j}) \preceq \\ \sum_ {j = 0} ^ {n} \lambda_ {j} \left[ \begin{array}{c c c} \frac {1}{2} \hat {\phi} _ {j} & * & * \\ \mathbf {0} & \frac {1}{2} \Pi^ {- 1} & * \\ \hat {\zeta} & \mathbf {0} & - 2 \Pi^ {- 1} \end{array} \right] =: \mathbf {E} (x), \quad (1 1) \\ \end{array}
$$

where $\begin{array} { r } { \hat { \phi } _ { j } \ge \operatorname* { m a x } _ { j \in \mathbb { Z } _ { 0 } ^ { n } } \left| \overline { { \mathbf { H } } } _ { \phi } ( x _ { j } , x ) \right| , \hat { \zeta } \in \mathbb { R } ^ { m \times 1 } } \end{array}$ where each element $\begin{array} { r } { \hat { \zeta } ^ { ( k ) } \geq \operatorname* { m a x } _ { j \in \mathbb { Z } _ { 0 } ^ { n } } \left| \overline { { \mathbf { H } } } _ { \zeta ^ { ( k ) } } ( x _ { j } , x ) \right| f o r k \in \mathbb { Z } _ { 1 } ^ { m } } \end{array}$ , and $\Pi \in \mathbb { R } ^ { m \times m }$ is a diagonal, positive definite matrix with elements $\pi _ { k } , k \in \mathbb { Z } _ { 1 } ^ { m }$ .

Moreover, $i f \mathbf { M } ( x _ { j } ) { + } \mathbf { E } ( x _ { j } ) \preceq 0$ holds for all vertex points of σ, then $\dot { \mathbf { M } } ( x ) \preceq 0$ holds for all x ∈ σ.

Theorem 10 bounds the difference between $\mathbf { M } ( x )$ at any convex combination of vertex points, $\textstyle \mathbf { M } ( \sum _ { i = 0 } ^ { n } \lambda _ { i } \mathbf { x } _ { i } )$ , and the convex combination of it evaluated at each vertex point, $\textstyle \sum _ { j = 0 } ^ { n } \lambda _ { j } \mathbf { M } ( x _ { j } )$ . The proof parallels [7, Proposition 2.2] by developing remainder terms using Taylor’s Theorem, but exploits the structure of $\mathbf { M } ( x )$ to establish an LMI error bound. Note the negative identity matrix in (10) is essential to enforce negative semi-definiteness.
