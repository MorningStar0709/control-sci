# 3.2 Finite difference scheme for continuous flow

We first present a computational scheme to solve the governing equation at each section of the continuous flow between switchings. For brevity, we do not introduce an additional notational convention to distinguish the variables over a specific section from other sections. It is understood that all of the developments in this subsection are devoted to a single section of continuous flow.

We begin by considering 1-D spatial domain for derivation of the scheme. This can be extended later to higher spatial dimensions. Suppose the time duration of each section and the space variables are discretized into N + 1 and $N _ { X } + 1$ points respectively. Then the time step is $\Delta t = T / N _ { T }$ and the spacial grid size is $h = ( x _ { m a x } - x _ { m i n } ) / N _ { X }$ .

The value function and the density are discretized by the matrices $U , M ~ \in ~ \mathbb { R } ^ { ( N _ { T } + 1 ) \times ( N _ { X } ^ { \times } + 1 ) } ~ \mathrm { a s } ~ u ( t _ { n } , x _ { i } ) ~ \overset { \times } { \approx }$ $U _ { i } ^ { n } , m ( t _ { n } , x _ { i } ) \approx M _ { i } ^ { n } { \mathrm { ~ f o r ~ } } n \in \{ 0 , \ldots , N _ { T } \} , i \in \{ 0 , \ldots , \dot { N } _ { X } \} .$ .

To evaluate the terms in the differential equations, we use the following relations:

$\bullet ( D _ { t } W ) ^ { n } = ( W ^ { n + 1 } - W ^ { n } ) / \Delta t , \quad W \in \mathbb { R } ^ { N _ { T } + 1 }$   
$\bullet \ ( D W ) _ { i } = ( W _ { i + 1 } - W _ { i } ) / h , \quad W \in \mathbb { R } ^ { N _ { X } + 1 }$   
$\bullet ( \Delta _ { h } W ) _ { i } = ( W _ { i + 1 } - 2 W _ { i } + W _ { i - 1 } ) / h ^ { 2 } , \quad W \in \mathbb { R } ^ { N _ { X } + 1 }$   
$\bullet [ \nabla _ { h } W ] _ { i } = ( ( D W ) _ { i } , ( D W ) _ { i - 1 } ) .$

The discrete version of the Hamiltonian, which satisfies key numerical properties, is obtained from Godunov’s scheme [Laney, 1998]. For instance, the discrete Hamiltonian could have the following quadratic structure in 1-D:

$$\tilde {H} (x, m, [ \nabla_ {h} W ] _ {i}) = \tilde {H} (x, m, p _ {1}, p _ {2}) = \frac {1}{2} c _ {K} ((p _ {1} ^ {-}) ^ {2} + (p _ {2} ^ {+}) ^ {2}),$$

and its derivatives with respect to the p-variables are

$$\partial_ {p _ {1}} \tilde {H} = - c _ {K} p _ {1} ^ {-}, \partial_ {p _ {2}} \tilde {H} = c _ {K} p _ {2} ^ {+}$$

where

$$x ^ {-} = \max (- x, 0), x ^ {+} = \max (x, 0).$$

Then the discrete version of the HJB equation (15) for MFG can be formulated as [Achdou et al., 2020]

$$- (D _ {t} U _ {i}) ^ {n} - \nu (\Delta_ {h} U ^ {n}) _ {i} + \tilde {H} (x _ {i}, M ^ {n + 1}, [ \nabla_ {h} U ] _ {i}) = 0, (2 3)$$
