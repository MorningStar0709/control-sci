# V. REMARKS ON THE NOISE BOUND COMPUTATION

Note that the bound ∆ in Assumption 2 depends both on the noise signals w(·), v(·) and the parameters of the plant (3) and the observer (16). Specifically, $d ( t )$ in (25) can be split as $d ( t ) = d _ { w } ( t ) + d _ { v } ( t )$ , where $d _ { w } ( t )$ and $d _ { v } ( t )$ are the outputs of the following dynamical systems:

$$\dot {\eta} _ {w} = \tilde {\Lambda} \eta_ {w} + E w, \quad d _ {w} = C \eta_ {w} \tag {46}\dot {\eta} _ {v} = \tilde {\Lambda} \eta_ {v} - \Pi L v, \quad d _ {v} = C \eta_ {v} + v, \tag {47}$$

with zero initial conditions. It is convenient to report the input–output description of these systems via their transfer function matrices:

$$\mathcal {G} _ {w} (s) := \frac {\mathcal {N} _ {w} (s)}{\mathcal {D} _ {\lambda} (s)}, \quad \mathcal {G} _ {v} (s) := \frac {\mathcal {D} (s)}{\mathcal {D} _ {\lambda} (s)}, \tag {48}$$

where $\mathcal { D } _ { \lambda }$ is given in (12). $\mathcal { G } _ { w } ( s )$ and $\mathcal { G } _ { v } ( s )$ follow from similar computations to those in Appendix I after noticing that, due to (3) and (14), $\Pi L = [ \lambda _ { 0 } I _ { p } { - } A _ { 0 } ^ { \top } ~ . . . ~ \lambda _ { n - 1 } I _ { p } { - } A _ { n - 1 } ^ { \top } ] ^ { \top }$ . Note that only $\mathcal { G } _ { v } ( s )$ depends on unknown parameters.

Using the property $d ( t ) d ( t ) ^ { \top } \preceq | d ( t ) | ^ { 2 } I _ { p }$ and the triangle inequality in $\mathcal { L } _ { 2 } [ 0 , T ]$ , we obtain

$$
\begin{array}{l} \int_ {0} ^ {T} d (\tau) d ^ {\top} (\tau) \mathrm{d} \tau \preceq I _ {p} \| d (\cdot) \| _ {2, [ 0, T ]} ^ {2} \tag {49} \\ \preceq I _ {p} \big (\| d _ {w} (\cdot) \| _ {2, [ 0, T ]} + \| d _ {v} (\cdot) \| _ {2, [ 0, T ]} \big) ^ {2}. \\ \end{array}
$$

Thus, ∆ in (26) can be computed using (49) if upper bounds of $\| d _ { w } ( \cdot ) \| _ { 2 , [ 0 , T ] }$ and $\| d _ { v } ( \cdot ) \| _ { 2 , [ 0 , T ] }$ are known. Below, we show how to compute them under certain prior knowledge of $w ( \cdot )$ and $v ( \cdot )$ and suitable properties of $\mathcal { G } _ { w } ( s )$ and $\mathcal { G } _ { v } ( s )$ .
