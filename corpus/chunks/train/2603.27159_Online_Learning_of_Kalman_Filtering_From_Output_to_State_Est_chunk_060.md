# B.1 Proof of Lemma 2.1

Recalling that $M _ { s } ^ { \star } = ( A - L C ) ^ { s - 1 } L \in \mathbb { R } ^ { n \times p }$ for all $s \in [ h ]$ , we have

$$
\begin{array}{l} \left\| M _ {s} ^ {\star} \right\| = \left\| (A - L C) ^ {s - 1} L \right\| \\ \leq \left\| (A - L C) ^ {s - 1} \right\| \| L \| \\ \leq \kappa_ {F} ^ {2} \gamma_ {F} ^ {s - 1}, \\ \end{array}
$$

where the last relation uses Lemma E.4. Noting the basic inequality $\left\| P \right\| _ { F } ^ { 2 } \leq \operatorname* { m i n } \{ m , n \} \left\| P \right\| ^ { 2 }$ for any $P \in \mathbb { R } ^ { m \times n }$ , we have $\| M _ { s } ^ { \star } \| _ { F } \leq \sqrt { \operatorname* { m i n } \{ p , n \} } \bar { \kappa _ { F } ^ { 2 } } \gamma _ { F } ^ { s - 1 }$ for all $s \in [ h ]$ . Moreover, we have

$$
\begin{array}{l} \left\| M ^ {\star} \right\| _ {F} = \left\| \left[ M _ {s} ^ {\star} \right] _ {s \in [ h ]} \right\| _ {F} \\ \leq \left(\sum_ {s = 1} ^ {h} \| M _ {s} ^ {\star} \| _ {F} ^ {2}\right) ^ {1 / 2} \\ \leq \big (\min \{p, n \} \kappa_ {F} ^ {4} \sum_ {s = 1} ^ {h} \gamma_ {F} ^ {2 (s - 1)} \big) ^ {1 / 2} \\ \leq \sqrt {\min \{p , n \}} \frac {\kappa_ {F} ^ {2}}{1 - \gamma_ {F}}. \\ \end{array}
$$
