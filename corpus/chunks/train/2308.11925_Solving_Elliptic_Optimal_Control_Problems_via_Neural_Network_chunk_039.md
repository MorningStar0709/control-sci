Since $L = c \log ( d + s ) , \pmb { n } _ { L } = c ( d , s ) \epsilon ^ { - \frac { d } { s - 2 - \mu } }$ and $R = c ( d , s ) \epsilon ^ { - \frac { 9 d + 4 s } { 2 ( s - 2 - \mu ) } }$ , the choices of the numbers $n _ { d }$ and $n _ { b }$ imply that the statistical error $\mathcal { E } _ { s t a t }$ is also bounded by $c \epsilon ^ { 2 }$ . This completes the proof of the theorem.

Remark 3.7. The error bound on the OSNN approximation in Theorem 3.6 is given in terms of the $L ^ { 2 } ( \Omega )$ norm, due to the presence of the consistency error induced by the $L ^ { 2 } ( \partial \Omega )$ boundary penalty, which precludes obtaining $H ^ { 1 } ( \Omega )$ error estimates. In contrast, for the conventional Galerkin FEM discretization, one can obtain error estimates in $H ^ { 1 } ( \Omega )$ ; see [17] for an early work, and [38] for many further results. See also $\it { \Omega } / \mathrm { { 2 } } 1 \it { \mathrm { ] } }$ for error estimates of discretizing optimal control problems with matrix-valued positive definite kernels.
