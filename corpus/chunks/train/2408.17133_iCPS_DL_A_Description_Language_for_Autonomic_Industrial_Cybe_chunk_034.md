The message pass action, written as $\mathsf { p } \to \mathsf { q } : \mathsf { U }$ instead of $( \mathsf { p } , \to$ $, \mathsf { q } , \mathsf { U } )$ , denotes the passing of message U from participant p to participant q. A triple $S _ { 1 } \xrightarrow { \sigma } S _ { 2 }$ denotes that the agents in local configuration $S _ { 1 }$ interact to observe action, $\sigma \in \Sigma _ { \mathsf { C } }$ , and proceed to $S _ { 2 }$ . It is desirable for a local configuration to satisfy the following properties: i) The deadlock-freedom property requires that a local context can either perform an observable action or ensure all its roles remain inactive. ii) The liveness property expects that every active role in a local configuration can eventually participate in an action. The following definitions formally express the two properties:

• A local context S is deadlock-free whenever:

– Either there exist $S ^ { \prime }$ and $\sigma \in \Sigma _ { \mathsf { C } }$ such that $S \longrightarrow S ^ { \prime } .$ or $\mathsf { a p } ( S ) = \emptyset$ .

– For all $S ^ { \prime }$ , such that $S \longrightarrow S ^ { \prime }$ for some $\sigma \in \Sigma _ { \mathsf { C } } , S ^ { \prime }$ is deadlock-free.

• A local configuration S is live whenever:

– For all $p ~ \in ~ \mathsf { a p } ( S )$ , there exist $S _ { 1 } , \ldots , S _ { n } , S ^ { \prime }$ and $\sigma _ { 1 } , . . . , \sigma _ { n } , \sigma \in \Sigma _ { \mathsf { C } }$ , with $n \geq 0 .$ , such that $S \xrightarrow { \sigma _ { 1 } }$ $S _ { 1 } , . . . , S \xrightarrow { \sigma _ { n } } S _ { n } ,$ and $S _ { n } \xrightarrow { \sigma } S ^ { \prime }$ , where $\sigma$ is of the form $\mathsf { p } \to \mathsf { q } : \mathsf { U }$ or ${ \mathsf { q } }  { \mathsf { p } } : { \mathsf { U } }$ .
