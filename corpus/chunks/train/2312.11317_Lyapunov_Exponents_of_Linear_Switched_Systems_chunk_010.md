# 3.1 Invariant Hyperboloid

Equation (2.5) is invariant after a dilation, in the sense that for any $c \in \mathbb { R } , \mathrm { i f } \ \eta ( \cdot )$ is a solution with initial value $\eta ( 0 ) = \eta _ { 0 }$ , then $\tilde { \eta } ( t ) = c \eta ( t )$ is a solution with initial value $\tilde { \eta } ( 0 ) = c \eta _ { 0 }$ .

$\mathrm { S o }$ , it is not restrictive to suppose that $\mathrm { t r } ( \eta ( t ) ^ { 2 } ) = \mathrm { t r } ( \eta _ { 0 } ^ { 2 } ) = 1$ . Hence, the trajectory η is contained in the set

$$H := \{M \in \mathfrak {s l} _ {2} (\mathbb {R}) \mid \operatorname{tr} (M ^ {2}) = 1 \},$$

which is a (connected) hyperboloid in the three-dimensional vector space ${ \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ . To see this, just take

$$
M = \left( \begin{array}{c c} m _ {1} & m _ {2} \\ m _ {3} & - m _ {1} \end{array} \right).
$$

Then

$$\mathrm{tr} (M ^ {2}) = 2 (m _ {1} ^ {2} + m _ {2} m _ {3}) = 2 \left(m _ {1} ^ {2} + \left(\frac {m _ {2} + m _ {3}}{2}\right) ^ {2} - \left(\frac {m _ {2} - m _ {3}}{2}\right) ^ {2}\right).$$

Since $A , B , [ A , B ]$ are linearly independent, we can write

$$M = \alpha A + \beta B + \gamma [ A, B ],$$

and the equation defining H reads

$$\alpha^ {2} \mathrm{tr} (A ^ {2}) + \beta^ {2} \mathrm{tr} (B ^ {2}) + \gamma^ {2} \mathrm{tr} ([ A, B ] ^ {2}) + 2 \alpha \beta \mathrm{tr} (A B) = 1.$$

To simplify the previous expression, we will use the following result.

Lemma 3.2. Let A, $B \in { \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ . Then

$$\operatorname{tr} ([ A, B ] ^ {2}) = 2 \operatorname{tr} (A B) ^ {2} - 2 \operatorname{tr} (A ^ {2}) \operatorname{tr} (B ^ {2}). \tag {3.3}$$

Proof. From the definition of bracket, we have

$$\mathrm{tr} ([ A, B ] ^ {2}) = 2 \mathrm{tr} ((A B) ^ {2}) - 2 \mathrm{tr} (A ^ {2} B ^ {2}).$$

Since A, $B \in { \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ , $A ^ { 2 }$ and $B ^ { 2 }$ are scalar, that is $A ^ { 2 } = \lambda ^ { 2 } \mathrm { I d }$ and $B ^ { 2 } = \mu ^ { 2 } \mathrm { I d }$ . So

$$\mathrm{tr} (A ^ {2} B ^ {2}) = 2 \lambda^ {2} \mu^ {2} = \frac {1}{2} \mathrm{tr} (A ^ {2}) \mathrm{tr} (B ^ {2}).$$

On the other hand, the minimal polynomial of AB is

$$(A B) ^ {2} - \operatorname{tr} (A B) A B + \det (A B) \mathrm{Id} = 0,$$

and computing the trace of the expression above leads to

$$\operatorname{tr} ((A B) ^ {2}) = \operatorname{tr} (A B) ^ {2} - 2 \det A \det B = \operatorname{tr} (A B) ^ {2} - \frac {1}{2} \operatorname{tr} (A ^ {2}) \operatorname{tr} (B ^ {2}).$$

and so

$$\mathrm{tr} ([ A, B ] ^ {2}) = 2 \mathrm{tr} ((A B) ^ {2}) - 2 \mathrm{tr} (A ^ {2} B ^ {2}) = 2 \mathrm{tr} (A B) ^ {2} - 2 \mathrm{tr} (A ^ {2}) \mathrm{tr} (B ^ {2}).$$

as we wanted to prove.

Using the previous Lemma, we can rewrite the equation for H as

$$\alpha^ {2} \mathrm{tr} (A ^ {2}) + \beta^ {2} \mathrm{tr} (B ^ {2}) + 2 \gamma^ {2} \big (\mathrm{tr} (A B) ^ {2} - \mathrm{tr} (A ^ {2}) \mathrm{tr} (B ^ {2}) \big) + 2 \alpha \beta \mathrm{tr} (A B) = 1.$$
