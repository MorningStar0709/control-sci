Case $( \mathrm { a } ) \colon \pi _ { \theta _ { t } } ( m ^ { * } ) \geqslant \pi _ { \theta _ { t } } ( m )$ . This implies, by the softmax property, that $\theta _ { t } ( m ^ { * } ) \geqslant \theta _ { t } ( m )$ . After gradient ascent update step we have:

$$
