# C. Asymptotic Inference Performance and Improved Design Under Limited Observations

Note that in the inference setting, only the estimate $\widehat { A } _ { d }$ is available and thus $A _ { c }$ is reconstructed by

$$\widehat {A} _ {c} = \frac {1}{\tau} \log (\widehat {A} _ {d}). \tag {38}$$

The feasibility of (38) is based on two conditions. First, $\widehat { A } _ { d }$ needs to be invertible such that it can be represented by the exponential of a matrix [44, Theorem 2.10]. As we have analyzed in Remark 1, both the matrices $Y _ { T } ^ { + }$ and $Y _ { T } ^ { - }$ have full rank. Consequently, it holds that the inverse of $\widehat { A } _ { d } ( T )$ exists as shown in the following expression

$$
\widehat {A} _ {d} ^ {- 1} = \left\{ \begin{array}{l} (\Sigma_ {0} (T) - I _ {N} \otimes \Gamma) \Sigma_ {1} (T) ^ {- 1}, \text {   if   } A _ {d} \in \mathcal {S} _ {d}, \\ \Sigma_ {0} (T) (\Sigma_ {1} (T)) ^ {- 1}, \text {   otherwise }, \end{array} \right. \tag {39}
$$

where $\Sigma _ { 1 } ( T )$ is invertible in each case. Second, the convergence of (38) requires $\| \widehat { A } _ { d } - I \| < 1$ . Based on Theorem 1, we point out when $A _ { d } \in S _ { d } ,$ , this norm condition holds in high probability with sufficient observations since the inference error will decay to zero. When $A _ { d } \notin { \cal S } _ { d } .$ , the inference error will also decay to zero if $\sigma _ { \mathrm { m i n } } ( Y _ { T } ^ { - } )$ grows faster than $O ( \sqrt { T } )$ .

Next, we present the asymptotic inference performance of $\widehat { A } _ { c } ( T )$ . For simplicity of notation, we define

$$\tilde {A} _ {d} = A _ {d} - I, E _ {d} (T) = \widehat {A} _ {d} (T) - A _ {d},\tilde {\delta} = \| \tilde {A} _ {d} \|, \delta_ {T} = \| E _ {d} (T) \|,$$

where $\tilde { \delta } < 1$ holds due to (48). Recall that if $A _ { d } \in S _ { d }$ , then the constructed $\widehat { A } _ { d } ( T )$ is asymptotically unbiased with probability one. Based on this property, it is further deduced that there always exists a $T _ { d } ~ > ~ 0$ such that the following inequality holds with high probability

$$\delta_ {T} < 1 - \tilde {\delta}, \forall T \geq T _ {d}. \tag {40}$$

Since we focus on characterizing the inference performance of $\widehat { A } _ { c } ( T )$ as $T$ grows, we directly consider $T ~ \geq ~ T _ { d }$ and demonstrate that how δT will influence $\| \widehat { A } _ { c } ( T ) - A _ { c } \|$ .
