A simple way to solve this issue is to impose a blockdiagonal structure on the Lyapunov matrices [7], [10]. With this structural constraint, (4) is convexly relaxed into the following solution set, denoted as $\kappa _ { \mathrm { d i a g } } ;$

$$
\begin{array}{l} \mathcal {K} _ {\mathrm{diag}} = \left\{Z Q ^ {- 1}: \exists Z \in \mathcal {S}, Q = \operatorname{blkdiag} \left(Q _ {1}, \dots , Q _ {N}\right) \succ O, \right. \\ \text { s.t. } \left[ \begin{array}{c c} Q & * \\ A Q + B Z & Q \end{array} \right] \succ O \}. \tag {5} \\ \end{array}
$$

The non-convex constraint $K = Z Q ^ { - 1 } \in { \mathcal { S } }$ is relaxed into a convex constraint by imposing structural constraints $Z \in { }$ $s$ and $Q = \mathrm { b l k d i a g } ( Q _ { 1 } , \dots , Q _ { N } )$ . However, this introduces conservatism, because there are $Z \not \in { \mathcal { S } }$ and non-block-diagonal matrices $Q$ producing $Z Q ^ { - 1 } \in \dot { S }$ . Note that the Lyapunov matrix $P = Q ^ { - 1 }$ is constrained to a block-diagonal structure in this case.

To mitigate this conservatism, extended LMIs [12]–[14] have been proposed. The following lemma shows that $\mathcal { K } _ { \mathrm { e x t } }$ , the solution set of an extended LMI, is a convex relaxation of (4) and less conservative than ${ \ K } _ { \mathrm { d i a g } }$ .

Lemma 1 (Distributed version of [12]): Let

$$\mathcal {K} _ {\text { ext }} = \{K = Z G ^ {- 1} \in \mathcal {S}: \exists Z \in \mathcal {S}, Q \succ O,G = \operatorname{blkdiag} (G _ {1}, \dots , G _ {N}),
\text { s.t. } \left[ \begin{array}{c c} G + G ^ {\top} - Q & * \\ A G + B Z & Q \end{array} \right] \succ O \}. \tag {6}
$$

Then, ${ \mathcal { K } } _ { \mathrm { d i a g } } \subset { \mathcal { K } } _ { \mathrm { e x t } } \subset { \mathcal { K } } _ { \mathrm { a l l } }$ holds.

Proof: $\mathcal { K } _ { \mathrm { d i a g } } \subset \mathcal { K } _ { \mathrm { e x t } }$ is shown by setting $G = Q . \kappa _ { \mathrm { e x t } } \subset$ $\kappa _ { \mathrm { a l l } }$ is shown by transforming Thm. 3 in [12].

This formulation removes the block-diagonal constraint on the Lyapunov matrix $P \ = \ Q ^ { - 1 }$ from ${ \ K } _ { \mathrm { d i a g } }$ in (1) by introducing an auxiliary matrix G, while ensuring $Z G ^ { - 1 } \in { \mathcal { S } }$ by the restrictions $Z \in S$ and $G = \mathrm { b l k d i a g } ( Q _ { 1 } , \dots , Q _ { N } )$ . However, $\mathcal { K } _ { \mathrm { e x t } }$ still brings some degree of conservatism, that is, there is a gap between $\mathcal { K } _ { \mathrm { e x t } }$ and $\kappa _ { \mathrm { a l l } }$ .
