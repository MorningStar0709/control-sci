# 1.3 Preliminaries

Graph Theory [15]: A MAS is modeled as an undirected graph $\mathcal { G } = ( \dot { \mathcal { V } } , \dot { \mathcal { E } } )$ , where $\mathcal { V } = \{ 1 , 2 , \hdots , n \}$ is the set of nodes (agents), and $\mathcal { E } \subseteq \mathcal { V } \times \mathcal { V }$ is the set of communication links. A path is a sequence of edges and nodes between two distinct nodes with no duplicated edges or nodes. A graph is connected if a path exists between any pair of nodes. The adjacency matrix $A \in \mathbb { R } ^ { n \times n }$ is defined by $a _ { i j } = 1$ if $( i , j ) \in \mathcal { E }$ , and 0 otherwise. The degree matrix $D$ is diagonal with $\begin{array} { r } { d _ { i i } = \sum _ { j } a _ { i j } } \end{array}$ . The Laplacian matrix is given by $L = D - A$ . The algebraic connectivity of the graph is the second smallest eigenvalue $\lambda _ { 2 }$ of $L ,$ which reflects the graph’s connectivity and affects consensus convergence.
