Let $D ( q )$ and $C ( q , \dot { q } )$ defined in (8) and (9), respectively, then the following statements hold (Esp´ındola & Tang 2023a):

(1) Matrix $D ( q )$ is symmetric and positive definite:

$$\lambda_ {m i n} (M) I _ {4} \leq D (q) \leq \lambda_ {m a x} (M) I _ {4}, \forall q \in \mathcal {S} ^ {3}. \tag {11}$$

(2) Matrix $\dot { D } ( q ) - 2 C ( q , \dot { q } )$ is skew-symmetric:

$$x ^ {T} (\dot {D} (q) - 2 C (q, \dot {q})) x = 0, \forall q \in \mathcal {S} ^ {3}, \forall x \in R ^ {4}. \tag {12}$$
