# A. The hard-constrained minimum principle-based IOC method

Due to the intuitive nature of minimum-principle based methods, we will use the hard-constrained minimum principle based method [39] in this work. The minimum principle methods solve the IOC problem by assuming that the considered system is only approximately optimal and not strictly optimal. Optimality, here, is defined based on the necessary conditions of optimality or the Karush-Kuhn-Tucker conditions. In the next two sections, this method will be extended for the swarm inverse optimal control problem. The hard-constrained method is a minimum principle based IOC approach [39]. The Hamiltonian function [40] for the problem (4) is:

$$H (x (t), u (t), p (t)) = c ^ {T} \phi (x (t), u (t)) + p ^ {T} (t) f (x (t), u (t)), \tag {11}$$

where $p ( t )$ is the costate trajectory. If $( x ( t ) , u ( t ) )$ are optimal, that is, $( x ( t ) , u ( t ) ) \ = \ ( x ^ { * } ( t ) , u ^ { * } ( t ) )$ , where $( x ^ { * } ( t ) , u ^ { * } ( t ) )$ ) is the optimal solution of problem (10) for a given c, then there exists a costate trajectory, $\begin{array} { r } { p ( t ) \ : \ \mathbb { R } \ \to \ \mathbb { R } ^ { n } } \end{array}$ (here, $p ( t ) = p ^ { * } ( t )$ , where $p ^ { * } ( t )$ is the optimal costate trajectory) and it is necessary that [19], [41]:

$$\dot {p} ^ {T} (t) + \nabla_ {x} H (x (t), u (t), p (t)) = \mathbf {0}, \tag {12}\nabla_ {u} H (x (t), u (t), p (t)) = \mathbf {0}. \tag {13}$$

The following condition together with equations (12) and (13) is sufficient and necessary for optimality [19]:

$$\frac {\partial^ {2} H (t , x (t) , u (t) , p (t))}{\partial u ^ {2}} > \mathbf {0}. \tag {14}$$

For the optimal control problem (10), the necessary conditions of optimality, equations (12) and (13) can be written as [19], [41]

$$\dot {p} ^ {T} (t) + c ^ {T} \nabla_ {x} \phi (x (t), u (t)) + p ^ {T} (t) \nabla_ {x} f (x (t), u (t)) = \mathbf {0}, \tag {15}c ^ {T} \nabla_ {u} \phi (x (t), u (t)) + p ^ {T} (t) \nabla_ {u} f (x (t), u (t)) = \mathbf {0}. \tag {16}$$
