The problem of listing all walks of length less than equal to T between the two vertices u and v, where $\ell ( u ) = \mathcal { X } _ { s }$ and $\ell ( v ) = \mathcal { X } _ { 0 }$ on a directed graph (employed in Step I. of Algorithm 2) is a standard problem in graph algorithms and can be solved by using minor logical modifications in off-the-shelf graph algorithms, e.g., depth-first search [3], breadthfirst search [3], etc.

Our results so far rely on the admissibility of a state-space abstraction by the switched system (2.1).
