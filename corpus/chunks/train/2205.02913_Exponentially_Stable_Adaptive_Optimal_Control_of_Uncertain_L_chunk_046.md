# 5 CONCLUSION

The adaptive control system is proposed to estimate the unknown parameters of the control law, which are optimal in terms of a linear-quadratic cost function. The global stability of the closed-loop adaptive control system and the convergence of the tracking error to the set with adjustable bound are proved. Recommendations on the choice of the adaptive system arbitrary parameters are given. The conducted numerical experiments confirmed all theoretical results and conclusions.

Scope of the further research and improvements of the proposed adaptive suboptimal control system can be structured as follows (in descending order of importance): 1) to relax the requirement $\Psi ( t ) \in \mathrm { F E }$ to a weaker semi-persistent excitation $( \Psi \left( t \right) \in \mathsf { s } \mathbf { - } \mathsf { P E } )$ or semi-finite excitation $\left( \Psi \left( t \right) \in \mathrm { s } { \mathrm { - } } \mathrm { F E } \right)$ , which are simpler to be satisfied by real world signals in comparison with FE; 2) to expand these results to systems with piecewise constant parameters with guarantee of exponential stability and convergence of both tracking and parameter errors to a compact set with adjustable bound; 3) not to use the analytical solution (9) of the Riccati equation (8) in (16) and derive a more reliable parameterization of (16) type in order to prevent possible singularity of solutions in (9) in case of $\lambda _ { i } \left( D \right) \gg \lambda _ { i } \left( D \right)$ and wrong choice of $\tau _ { \infty }$ value (the solution of the first problem in this list will be of great importance to achieve the goal under consideration); 4) to approximate the fundamental matrix $\Phi \left( \tau _ { \infty } \right)$ by more computationally stable method in comparison with the applied Taylor series expansion (for example, with the help of the Padé approximation47).
