with the barred terms as introduced in the beginning of this section. We use Gauss’ principle [175] to derive closed form expressions for $J ^ { \top } h$ and $h _ { O }$ . Let the unconstrained coupled object-robots system be

$$\bar {B} \alpha := \bar {\tau} - \bar {C} _ {q} \widetilde {x} - \bar {g},$$

where $\alpha : = \alpha ( \bar { x } ) : \mathbb { R } ^ { 2 n + 6 } \times \mathbb { T }  \mathbb { R } ^ { n + 6 }$ is the unconstrained acceleration, i.e., the acceleration the system would have if the agents did not grasp the object. According to Gauss’s principle [175], the actual accelerations $\dot { \widetilde { x } }$ of the system are the closest ones to $\alpha ( \bar { x } )$ e, while satisfying the rigidity constraints. More rigorously, $\dot { \widetilde { x } }$ is the solution of the constrained minimization problem

$$\min _ {z} \left(z - \alpha (\bar {x})\right) ^ {\top} \bar {B} (\bar {x}) \big (z - \alpha (\bar {x}) \big)\mathrm{s.t.} A (\bar {x}, t) z = b (\bar {x}, t).$$

The solution to this problem is obtained by using the Karush-Kuhn-Tucker conditions [176] and has a closed-form expression. It can be shown that it satisfies

$$\bar {B} z = \alpha + A ^ {\top} \big (A \bar {B} ^ {- 1} A ^ {\top} \big) ^ {\dagger} \big (b - A \alpha \big),$$

where $^ \dagger$ denotes the Moore-Penrose inverse. The aforementioned expression is compliant with the one in [177],

$$\bar {B} z = \alpha + \bar {B} ^ {\frac {1}{2}} \left(A \bar {B} ^ {- \frac {1}{2}}\right) ^ {\dagger} (b - A \alpha),$$

since it holds that $A ^ { \top } ( A { \bar { B } } ^ { - 1 } A ^ { \top } ) ^ { \dagger } = { \bar { B } } ^ { \frac { 1 } { 2 } } ( A { \bar { B } } ^ { - \frac { 1 } { 2 } } ) ^ { \dagger }$ . Indeed, according to Theorem 3.8 of [178], it holds that $H ^ { \dagger } = H ^ { \top } ( H H ^ { \top } ) ^ { \dagger }$ , for any $H \in \mathbb { R } ^ { x \times y }$ . Then the aforementioned equality is obtained by setting $H = A \bar { B } ^ { - \frac { 1 } { 2 } }$ .

Therefore, the forces, projected onto the joint-space of the agents, have the form

$$
\left[ \begin{array}{c} - J ^ {\top} h \\ h _ {o} \end{array} \right] = A ^ {\top} (A \bar {B} ^ {- 1} A ^ {\top}) ^ {\dagger} (b - A \alpha) \tag {3.49a}
= \bar {B} ^ {\frac {1}{2}} \left(A \bar {B} ^ {- \frac {1}{2}}\right) ^ {\dagger} (b - A \alpha). \tag {3.49b}$$
