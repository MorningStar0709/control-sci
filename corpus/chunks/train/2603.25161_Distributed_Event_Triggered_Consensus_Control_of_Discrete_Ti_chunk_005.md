# III. CONSENSUS AND LQ COST UNDER ALL-TIME COMMUNICATION

In this section, we consider consensus control of multiagent systems under all-time communication, and derive the corresponding LQ cost.

With the distributed controller (3), the overall closed-loop system is given by

$$x [ k + 1 ] = (I _ {N} \otimes A - c L \otimes B F) x [ k ]. \tag {8}$$

Since $\mathcal { G }$ is weighted undirected and connected, the Laplacian matrix L is symmetric positive semidefinite and has a simple zero eigenvalue. Let $\dot { \boldsymbol { U } } \in \mathbb { R } ^ { N \times N }$ be an orthogonal matrix such that

$$U ^ {\top} L U = \Lambda := \operatorname{diag} (0, \lambda_ {2} (L), \dots , \lambda_ {N} (L)),$$

where $0 ~ < ~ \lambda _ { 2 } ( L ) ~ \leq ~ \cdots ~ \leq ~ \lambda _ { N } ( L )$ are the nonzero eigenvalues of $L .$ We partition $U$ as $\begin{array} { r } { U = \big [ \frac { 1 } { \sqrt { N } } \mathbf { 1 } _ { N } U _ { 2 } \big ] } \end{array}$ . Define $\tilde { x } [ k ] : = ( U ^ { \top } \otimes I _ { n } ) x [ k ]$ with $\tilde { x } [ k ] = [ \tilde { x } _ { 1 } ^ { \top } [ k ] \enspace \dots \enspace \tilde { x } _ { N } ^ { \top } [ k ] ] ^ { \top }$ . Then, the closed-loop system (8) is transformed into

$$\tilde {x} [ k + 1 ] = (I _ {N} \otimes A - c \Lambda \otimes B F) \tilde {x} [ k ], \tag {9}$$

that is,

$$\tilde {x} _ {1} [ k + 1 ] = A \tilde {x} _ {1} [ k ], \tag {10}\tilde {x} _ {i} [ k + 1 ] = (A - c \lambda_ {i} (L) B F) \tilde {x} _ {i} [ k ], \quad i = 2, \dots , N. (1 1)$$

From (10), we can see that $\tilde { x } _ { 1 } [ k ]$ represents the component of $x [ k ]$ along the consensus subspace since

$$\tilde {x} _ {1} [ k ] = \left(\frac {1}{\sqrt {N}} \mathbf {1} _ {N} ^ {\top} \otimes I _ {n}\right) x [ k ]$$

implies that the corresponding component in $\mathbb { R } ^ { N n }$ is

$$\left(\frac {1}{\sqrt {N}} \mathbf {1} _ {N} \otimes I _ {n}\right) \tilde {x} _ {1} [ k ] \in \operatorname{Im} (\mathbf {1} _ {N} \otimes I _ {n}).$$

In contrast, (11) describes the disagreement modes among the agents since

$$
\begin{array}{l} x [ k ] = (U \otimes I _ {n}) \tilde {x} [ k ] \\ = \left(\frac {1}{\sqrt {N}} \mathbf {1} _ {N} \otimes I _ {n}\right) \tilde {x} _ {1} [ k ] + \sum_ {i = 2} ^ {N} (v _ {i} \otimes I _ {n}) \tilde {x} _ {i} [ k ], \\ \end{array}
$$
