# B. Event-Triggering Mechanism

In what follows, Definition 2 illustrates how the small design parameter $\sigma _ { \mathrm { ~ \normalfont ~  ~ } } \in \mathrm { ~ \normalfont ~ ( 0 , 1 ) ~ }$ and the error signal $e [ k ]$ —representing the deviation between the gradient estimate ${ \hat { G } } [ k ]$ and its last transmitted value—are leveraged to construct a static event-triggered mechanism for scalar gradient-based extremum seeking in discrete-time. The resulting strategy triggers the recalculation of the tuning law, as defined in (23) only when needed. The closed-loop system preserves asymptotic stability and guaranteed convergence, as supported by the theoretical results in [8].

Before stating Definition 2, we introduce the following assumption which will be used in the sequence.

Assumption 1. There exist ϵ, a, and K such that

$$0 < \left| 1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2} \right| < 1, \tag {29}$$

and a known positive constant α satisfying

$$\alpha > \frac {\epsilon a ^ {2} | H ^ {*} | | K |}{\sqrt {2}} \frac {\sqrt {1 + 7 \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}}}{\left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right)}. \tag {30}$$

Definition 2 (Event-Triggering Condition). The triggering mechanism for discrete-time gradient-based extremum seeking controller with static event-triggered condition consists of two components:

1) A sequence of increasing triggering iteration step K such that $\mathcal { K } = \{ k _ { 0 } , k _ { 1 } , k _ { 2 } , . . . \}$ with $k _ { 0 } = 0 ,$ , generated according to:

$\bullet I f \left\{ k \in \mathbb { Z } : \ k > k _ { l } \ \wedge \ \sqrt { \sigma } | \hat { G } [ k ] | - \alpha | e [ k ] | < 0 \right\} = 0 ,$ , then the set of triggering instants $\begin{array} { r l } { i s } & { { } \kappa ^ { \prime } } \end{array} =$ $\{ k _ { 0 } , k _ { 1 } , \ldots , k _ { l } \}$ .   
$\bullet I f \left\{ k \in \mathbb { Z } : \ k > k _ { l } \ \wedge \ \sqrt { \sigma } | \hat { G } [ k ] | - \alpha | e [ k ] | < 0 \right\} \neq 0 ,$ the next triggering instant is given by

$$k _ {l + 1} = \inf \left\{k \in \mathbb {Z}: k > k _ {l} \wedge \sqrt {\sigma} | \hat {G} [ k ] | - \alpha | e [ k ] | < 0 \right\}. \tag {31}$$

2) A feedback control action (23) updated at the triggering instants kl and held constant for all $k \in [ k _ { l } , k _ { l + 1 } )$ .
