# BACKGROUND: INVERSE OPTIMAL CONTROL PROBLEM

The inverse optimal control problem is an approach of recovering the unknown cost function of an existing optimal control problem by measuring its state and control trajectories. Natural movements and locomotion have been hypothesized to be optimal [16], [17]. Since the cost function has been identified to be the most succinct, robust and transferrable definition of a task [20], the IOC problem can be seen as an approach for biomimicry [38].

For a general nonlinear optimal control problem

$$
\begin{array}{l} \underset {u} {\text { minimize }} \quad \int_ {t _ {0}} ^ {t _ {f}} \Phi (x (t), u (t)) d t, \\ \text { subject   to } \quad \dot {x} (t) = f (x (t), u (t)), \tag {7} \\ x (t _ {0}) = x _ {0}, \\ \end{array}
$$

$\Phi ( x ( t ) , u ( t ) )$ is the cost function and $f ( x ( t ) , u ( t ) )$ represents the system dynamics of a general nonlinear system.

Assumption 2: It is assumed that $f \in C ^ { 1 }$ , that is, continuously differentiable [19].

Assumption 3: It is assumed that the system in problem (7) is controllable [38].

Assumption 4: It is assumed that $\Phi ( x ( t ) , u ) ( t ) )$ is positive semi-definite in $x ( t )$ and $u ( t )$ , for all $t \geq 0 \ [ 3 8 ]$ .

Assumption 5: It is assumed that problem (7) is a nonsingular problem [38].

At this stage, we will define the IOC problem statement.

Problem Statement 1: Given the system dynamics ${ \dot { x } } ( t ) =$ $f ( x ( t ) , u ( t ) )$ , and a specific system trajectory $x ( t )$ and control signal $u ( t ) , \forall t _ { 0 } \leq t \leq t _ { f }$ , measured from the primary optimal control problem $( 1 0 )$ , the objective of the IOC problem is to find the unknown cost function, $\Phi ( { \boldsymbol x } ( t ) , { \boldsymbol u } ( t ) )$ .
