# 3. THE ERGODIC OPTIMAL CONTROL PROBLEM

Recalling (2.8), in this section we fix $\theta \in \mathbb { R } _ { + }$ and solve the ergodic control problem. In particular, we want to find

$$\bar {\lambda} (\theta) := \sup _ {\xi \in \mathcal {A} _ {e}} J (x, i; \xi , \theta). \tag {3.1}$$

To the best of our knowledge, this is the first paper that solves a singular stochastic control problem with regime switching and ergodic performance criterion. In order to solve (3.1), we let $V : \mathcal { O } \times \mathbb { R } _ { + } \to$ R and $\lambda : \mathbb { R } _ { + } \times \mathbb { Y }  \mathbb { R }$ to be determined such that $V ( \cdot , i ; \theta ) \in C ^ { 2 } ( \mathcal { I } )$ , for any $( i , \theta ) \in \mathbb { Y } \times \mathbb { R } _ { + }$ , and the pair (V, λ) solves the variational inequality

$$\max \left\{\mathcal {L} _ {(X, Y)} V (x, i; \theta) + \pi (x, \theta) - \lambda (\theta , i), V _ {x} (x, i; \theta) - k _ {1}, k _ {2} - V _ {x} (x, i; \theta) \right\} = 0. \tag {3.2}$$

It will be shown in Proposition 3.4 below that a solution (V, λ) to (3.2) allows to obtain $\bar { \lambda }$ in the sense that

$$\bar {\lambda} (\theta) = \sum_ {i = 1} ^ {d} p (i) \lambda (\theta , i), \tag {3.3}$$

where $( p ( i ) ) _ { i \in \mathbb { Y } }$ denotes the stationary distribution of the Markov chain Y . The following additional assumption on π holds throughout the rest of this paper.

Assumption 3.1. For $x _ { - } ( \theta ) : = ( x _ { - } ( i , \theta ) ) _ { i \in \mathbb { Y } }$ and $x _ { + } ( \theta ) : = ( x _ { + } ( i , \theta ) ) _ { i \in \mathbb { Y } }$ , with $x _ { - } ( i , \theta ) < x _ { + } ( i , \theta )$ , i ∈ Y, it holds:

$$
\pi_ {x} (x, \theta) + k _ {1} b _ {x} (x, i) \left\{ \begin{array}{l l} > 0, & x \in (\underline {{x}}, x _ {-} (i, \theta)), \\ = 0, & x = x _ {-} (i, \theta), \\ <   0, & x \in (x _ {-} (i, \theta), \overline {{x}}), \end{array} \right.

\pi_ {x} (x, \theta) + k _ {2} b _ {x} (x, i) \left\{ \begin{array}{l l} > 0, & x \in (\underline {{x}}, x _ {+} (i, \theta)), \\ = 0, & x = x _ {+} (i, \theta), \\ <   0, & x \in (x _ {+} (i, \theta), \overline {{x}}). \end{array} \right.
$$
