# 10.7 SCM Representation

We define an SCM with variables:

• $X _ { t }$ : exogenous drivers (economic activity, shocks),   
• $\Theta _ { i } \colon$ agent attributes $( \theta _ { i } , \eta _ { i } )$ ,   
• $P _ { t } \colon$ policy parameters,   
• $E _ { t } \mathrm { : }$ aggregate emissions,   
• $Y _ { t } \colon$ welfare outcomes (cost, compliance, volatility).

Directed edges include $( P _ { t } , \Theta , X _ { t } ) \to E _ { t }$ and $E _ { t } \to Y _ { t }$ , while policy adaptation introduces $E _ { t } \to P _ { t + 1 }$ . Interventions do $( P _ { t } = p )$ formalize counterfactuals about alternate tax or cap trajectories.
