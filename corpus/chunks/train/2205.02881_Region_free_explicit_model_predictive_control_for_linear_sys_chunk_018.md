We fix some arbitrary pair $( \mathbf { z } , \mathbf { x } )$ with the above properties, and we will construct a feasible arc C with the required initial conditions. Since our constraints are affine, we choose $C ( t ) : =$ ${ \bf z } + t { \bf x } , t \in [ 0 , \tau )$ , as suggested on p. 11 of [27]. Then

$$f _ {k} (C (t), \theta_ {n}) = \mathcal {I} _ {k} (W + S \theta_ {n} - G (\mathbf {z} + t \mathbf {x})) f _ {k} (- \theta_ {n}) + t ^ {\prime} (- f ^ {\prime} (- \theta_ {n})) > 0 \tag {20}= f _ {k} (\mathbf {z}, \theta_ {n}) + t \left\langle \mathbf {x}, f _ {k} ^ {\prime} (\mathbf {z}, \theta_ {n}) \right\rangle \geq 0$$

for all $t \geq 0$ and $k \in \mathbf { B }$ with

$$\mathbf {B} := \left\{k \in \mathbf {P} \mid \langle \mathbf {x}, f _ {k} ^ {\prime} (\mathbf {z}, \theta_ {n}) \rangle \geq 0 \right\}.$$

Now let

$$a := \min _ {k \in \mathbf {P} \backslash \mathbf {B}} f _ {k} (\mathbf {z}, \theta_ {n}) > 0$$

(because $\mathbf { B } \supset \mathbf { A } _ { \mathbf { z } }$ by the choice of z and x) and

$$b := \max _ {k \in \mathbf {P} \backslash \mathbf {B}} - \langle \mathbf {x}, f _ {k} ^ {\prime} (\mathbf {z}, \theta_ {n}) \rangle > 0$$

(because of the definition of B). Then (20) is satisfied also for all $k \in \mathbf { P } \setminus \mathbf { B }$ and for all $t \in [ 0 , \tau )$ , with $\tau : = a / b > 0$ . Thus the constraint qualification holds.

Let $\mathbf { z } _ { \ast }$ be a minimizer of (13) with the corresponding active set A. By [27, Thm 2.4], the equivalent conditions in [27, Prop. 2.2] hold, i.e., there exists a finite measure $u ^ { * }$ on P, such that,

$$f ^ {\prime} (\mathbf {z} _ {*}) = \int_ {\mathbf {P}} f _ {k} ^ {\prime} (\mathbf {z} _ {*}, \theta_ {n}) d u ^ {*}, \tag {21}$$

2) $u ^ { * } ( \mathbf { P } ^ { \prime } ) = 0$ for all $\mathbf { P } ^ { \prime } \subset \mathbf { A } ^ { c }$ and   
3) $u ^ { * } ( \mathbf { P } ^ { \prime } ) \geq 0$ for all $\mathbf { P } ^ { \prime } \subset \mathbf { P } .$ .
