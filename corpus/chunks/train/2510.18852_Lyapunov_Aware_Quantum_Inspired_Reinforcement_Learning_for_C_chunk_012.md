# F. Training Algorithm

The parameters θ of the policy are updated via a finitedifference approximation of the policy gradient:

$$\theta_ {k + 1} = \theta_ {k} - \alpha \nabla_ {\theta} (- r _ {t} + \mathcal {L} _ {\mathrm{stab}}), \tag {16}$$

where α is the learning rate.

The agent interacts with the environment, collecting state transitions $( x _ { t } , u _ { t } , r _ { t } , x _ { t + 1 } )$ , and computes gradient estimates from perturbations in θ. This method avoids explicit backpropagation and maintains compatibility with the quantum-inspired representation.
