# A General Structure and Stability of PAA

One can consider as a general structure for the PAA (integral type):

$$\boxed {\hat {\theta} (t + 1) = \hat {\theta} (t) + F (t) \phi (t) v (t + 1)} \tag {3.231}\boxed {\nu (t + 1) = \frac {\nu^ {0} (t + 1)}{1 + \phi^ {T} (t) F (t) \phi (t)}} \tag {3.232}
\begin{array}{l} F (t + 1) ^ {- 1} = \lambda_ {1} (t) F (t) ^ {- 1} + \lambda_ {2} (t) \phi (t) \phi^ {T} (t) \\ 0 <   \lambda_ {1} (t) \leq 1; 0 \leq \lambda_ {2} (t) <   2 \tag {3.233} \\ F (0) > 0; F (t) ^ {- 1} > \alpha F (0) ^ {- 1}; \infty > \alpha > 0 \\ \end{array}
$$

where ${ \hat { \theta } } ( t )$ is the adjustable parameter vector, $F ( t )$ is the adaptation gain, $\phi ( t )$ is the observation vector, $\nu ^ { 0 } ( t + 1 )$ is the a priori adaptation error and $\nu ( t + 1 )$ is the a posteriori adaptation error. The a priori adaptation error $\nu ^ { 0 } ( t + 1 )$ depends only on the adjustable parameter vector $\hat { \theta } ( i )$ up to and including $i = t . \ \nu ^ { 0 } ( t + 1 )$ is in fact the prediction of $\nu ( t + 1 )$ based on these $\hat { \theta } ( i )$ ), i.e.:

$$\nu^ {0} (t + 1) = \nu (t + 1 / \hat {\theta} (t), \hat {\theta} (t - 1), \dots)$$

The adaptation gain matrix $F ( t )$ is computed recursively using the matrix inversion lemma, and (3.3.124) becomes:

$$F (t + 1) = \frac {1}{\lambda_ {1} (t)} \left[ F (t) - \frac {F (t) \phi (t) \phi^ {T} (t) F (t)}{\frac {\lambda_ {1} (t)}{\lambda_ {2} (t)} + \phi^ {T} (t) F (t) \phi (t)} \right] \tag {3.234}$$

Associated with the PAA of (3.231) through (3.233) one considers the class of adaptive systems for which the a posteriori adaptation error satisfies an equation of the form:

$$\boxed {\nu (t + 1) = H \left(q ^ {- 1}\right) \left[ \theta - \hat {\theta} (t + 1) \right] ^ {T} \phi (t)} \tag {3.235}$$

where:

$$H (q ^ {- 1}) = \frac {H _ {1} (q ^ {- 1})}{H _ {2} (q ^ {- 1})} \tag {3.236}$$

with:

$$H _ {j} (q ^ {- 1}) = 1 + q ^ {- 1} H _ {j} ^ {*} (q ^ {- 1}) = 1 + \sum_ {i = 1} ^ {n _ {j}} h _ {i} ^ {j} q ^ {- i}; \quad j = 1, 2 \tag {3.237}$$

and θ is a fixed value of the unknown parameter vector.

The relationship between a priori and a posteriori adaptation errors given in (3.232), can be alternatively expressed using (3.231) as:

$$\nu (t + 1) = [ \hat {\theta} (t) - \hat {\theta} (t + 1) ] ^ {T} \phi (t) + \nu^ {0} (t + 1) \tag {3.238}$$

From (3.235) and (3.236) one gets:

$$
\begin{array}{l} \nu (t + 1) = [ \theta - \hat {\theta} (t + 1) ] ^ {T} \phi (t) - H _ {2} ^ {*} (q ^ {- 1}) \nu (t) \\ + H _ {1} ^ {*} (q ^ {- 1}) [ \theta - \hat {\theta} (t) ] ^ {T} \phi (t - 1) \tag {3.239} \\ \end{array}
$$
