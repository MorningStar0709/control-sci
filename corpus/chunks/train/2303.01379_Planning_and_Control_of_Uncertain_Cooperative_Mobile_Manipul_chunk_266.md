# 4.2.1 Problem Statement

Consider a spherical robot operating in a bounded workspace , characterized by its position vector $x \in \mathbb { R } ^ { n } , n \in \{ 2 , 3 \}$ and radius $r > 0$ , and subject to the dynamics:

$$\dot {x} = v \tag {4.1a}m \dot {v} + f (x, v) + m g = u, \tag {4.1b}$$

where $m > 0$ is the unknown mass, $g \in \mathbb { R } ^ { n }$ is the constant gravity vector, $u \in \mathbb { R } ^ { n }$ is the input vector, and $f : \mathbb { R } ^ { 2 n }  \mathbb { R } ^ { n }$ is a friction-like function, satisfying the following assumption:

Assumption 4.1. The function $f : \mathbb { R } ^ { 2 n }  \mathbb { R } ^ { n }$ is analytic and satisfies

$$\| f (x, v) \| \leq \alpha \| v \|,$$

$\forall x , v \in \mathbb { R } ^ { 2 n }$ , where $\alpha \in \mathbb { R } _ { \geq 0 }$ is an unknown positive constant.

The aforementioned assumption is a standard condition concerning friction-like terms, which are bounded by the robot velocity [252, 253]. Constant unknown friction terms could be also included in the dynamics (e.g., incorporated in the gravity vector). Note also that $\| f ( x , v ) \| \leq \alpha \| v \|$ implies $f ( x , 0 ) = 0$ , and $\left. \frac { \partial f ( x , v ) } { \partial x } \right| _ { v = 0 } = 0$ . The workspace is assumed to be an open ball centered at the origin

$$\mathcal {W} := \mathcal {B} (0, r _ {\mathcal {W}}) = \{z \in \mathbb {R} ^ {n}: \| z \| < r _ {\mathcal {W}} \}, \tag {4.2}$$

where $r _ { \mathcal { W } } > 0$ is the workspace radius. The workspace contains $M \in \mathbb { N }$ closed sets $\mathcal { O } _ { j } , \ j \in \mathcal { I } : = \{ 1 , \ldots , M \}$ , corresponding to obstacles. Each obstacle is a closed ball centered at $c _ { j } \in \mathbb { R } ^ { 3 }$ , with radius $r _ { o _ { j } } > 0 \colon$ :

$$\mathcal {O} _ {j} := \bar {\mathcal {B}} (c _ {j}, r _ {o _ {j}}) = \{z \in \mathcal {W}: \| z - c _ {j} \| \leq r _ {o _ {j}} \}, \forall j \in \mathcal {J}.$$

The analysis that follows will be based on the transformed workspace:

$$\bar {\mathcal {W}} := \{z \in \mathbb {R} ^ {n}: \| z \| < \bar {r} _ {\mathcal {W}} := r _ {\mathcal {W}} - r \}, \tag {4.3}$$

and the set of obstacles

$$\bar {\mathcal {O}} _ {j} := \{z \in \mathcal {W}: \| z - c _ {j} \| \leq \bar {r} _ {o _ {j}} := r _ {o _ {j}} + r \}, \forall j \in \mathcal {J}.$$

and the robot is reduced to the point x. The free space is defined as

$$\mathcal {F} := \bar {\mathcal {W}} \backslash \bigcup_ {j \in \mathcal {J}} \bar {\mathcal {O}} _ {j}, \tag {4.4}$$
