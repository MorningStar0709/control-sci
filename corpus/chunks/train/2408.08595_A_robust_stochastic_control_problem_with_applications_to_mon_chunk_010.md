# 3.1 BSDEs with unbounded coefficients

In this subsection, we will give the solvability of two BSDEs with unbounded coefficients, which plays a key role in solving the robust control problem.

First, we consider the following backward stochastic differential equations:

$$
\left\{ \begin{array}{l} d h _ {t} = \left[ \left(- A _ {t} + (D _ {t} ^ {- 1} B _ {t}) ^ {\prime} C _ {t}\right) h _ {t} + (D _ {t} ^ {- 1} B _ {t}) ^ {\prime} L _ {t} + \frac {L _ {t} ^ {\prime} L _ {t}}{h _ {t}} \right] d t + L _ {t} ^ {\prime} d W _ {t}, \\ h _ {T} = 1. \end{array} \right. \tag {3.6}
$$

It can be observed that equation (3.6) is a backward stochastic Riccati equation. Although this class of BSDEs with bounded coefficients are well studied in the literature, the solvability of (3.6) cannot be obtained from existing results since A is unbounded.

Lemma 3.1 Under assumption (A), the BSDE (3.6) admits a unique solution $( h , L ) \in L _ { \mathbb { F } , \mathbb { P } } ^ { \infty } ( 0 , T ; \mathbb { R } ) \times$ $L _ { \mathbb { F } , \mathbb { P } } ^ { 2 , \mathrm { B M O } } ( 0 , T ; \mathbb { R } ^ { n } )$ . In addition, the first component of the solution satisfies $h _ { t } \geqslant \delta ,$ , for all $t \in [ 0 , T ]$ and some $\delta > 0$ . Moreover, there exists a constant $K > 0$ such that for any $p > 1$ and $t \in [ 0 , T ]$ , we have

$$\mathbb {E} _ {t} \left[ \mathcal {E} \left(\int_ {t} ^ {T} - h _ {s} ^ {- 1} L _ {s} ^ {\prime} d W _ {s}\right) ^ {p} \right] \leq K.$$

Proof. Under the assumption (A), it follows from BMO martingale theory that the process

$$N _ {t} := \mathcal {E} \left\{\int_ {0} ^ {t} (- D _ {s} ^ {- 1} B _ {s}) ^ {\prime} d W _ {s} \right\},$$

$$\left. \frac {d \overline {{\mathbb {P}}}}{d \mathbb {P}} \right| _ {\mathcal {F} _ {T}} = N _ {T}.$$

Denote by $\overline { { \mathbb { E } } } [ \cdot ]$ the expectation under $\overline { { \mathbb { P } } }$ and by $\overline { { \mathbb { E } } } _ { t } [ \cdot ]$ the conditional expectation under $\overline { { \mathbb { P } } }$ given that $\mathcal { F } _ { t }$ . Set
