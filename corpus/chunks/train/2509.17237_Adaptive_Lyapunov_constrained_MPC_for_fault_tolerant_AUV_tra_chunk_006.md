# B. Thruster Mapping and Fault Parameterization

The generalized force is produced by four thrusters via the allocation model

$$\boldsymbol {\tau} = \boldsymbol {T} (\boldsymbol {\theta}) \boldsymbol {\Gamma} \boldsymbol {u}, \tag {5}$$

where $\ b { u } \in \mathbb { R } ^ { 4 \times 1 }$ stacks the individual thruster commands, $\pmb { T } ( \pmb \theta ) = [ t _ { 1 } \ t _ { 2 } \ t _ { 3 } \ t _ { 4 } ] \ \in \ \mathbb { R } ^ { 3 \times 4 }$ encodes the horizontal-plane geometry, and $\mathbf { \Gamma } \Gamma = \operatorname { d i a g } ( \gamma _ { 1 } , \gamma _ { 2 } , \gamma _ { 3 } , \gamma _ { 4 } ) \in \mathbb { R } ^ { 4 \times 4 }$ models perthruster effectiveness with $\gamma _ { i } ~ \in ~ [ 0 , 1 ]$ . A misalignment of thruster i by angle $\theta _ { i }$ is captured column-wise by

$$
\boldsymbol {T} (\boldsymbol {\theta}) = \left[ \begin{array}{c c c c} \boldsymbol {t} _ {1}, & \dots , & \boldsymbol {R} (\theta_ {i}) \boldsymbol {t} _ {i}, & \dots , & \boldsymbol {t} _ {4} \end{array} \right],
$$

with planar rotation $\mathbf { \mathit { R } } ( \theta _ { i } )$ . Nominal operation corresponds to $\gamma _ { i } = 1 , \theta _ { i } = 0 ;$ a blocked thruster has $\gamma _ { i } = 0$ .
