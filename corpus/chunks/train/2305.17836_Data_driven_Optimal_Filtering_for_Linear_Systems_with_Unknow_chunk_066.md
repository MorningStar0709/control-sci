$$
\begin{array}{l} \| H ^ {\intercal} H \mathcal {A} _ {L} \| _ {*} = \mathrm{tr} \left[ \sqrt {H ^ {\intercal} H \mathcal {A} _ {L} \mathcal {A} _ {L} ^ {\intercal} H ^ {\intercal} H} \right] \\ \leq \left\| \mathcal {A} _ {L} \mathcal {A} _ {L} ^ {\intercal} \right\| ^ {1 / 2} \left\| H ^ {\intercal} H \right\| _ {*} \\ \leq \left[ \sum_ {i = 0} ^ {\infty} \| A _ {L} ^ {i} \| ^ {2} \right] ^ {1 / 2} \| H ^ {\intercal} H \| _ {*} \\ \leq \left[ \frac {C _ {L} ^ {2}}{1 - \rho (A _ {L})} \right] ^ {1 / 2} \| H ^ {\intercal} H \| _ {*}, \\ \end{array}
$$

where the last inequality follows by Lemma 6. Finally, notice that

$$
\begin{array}{l} \left\| \mathcal {N} _ {L} [ E ] \right\| _ {*} \leq 2 \left\| \mathcal {A} _ {L} ^ {\intercal} H ^ {\intercal} H \mathcal {M} _ {L} [ E ] \right\| _ {*} \\ \leq 2 \| \mathcal {M} _ {L} [ E ] \| \| H ^ {\intercal} H \mathcal {A} _ {L} \| _ {*} \\ \end{array}
$$

and thus combining the last three bounds implies the second claim. This completes the proof.

The next tool we will be using is the following famous bound on random matrices which is a variant of Bernstein inequality:

Lemma 9 (Matrix Bernstein Inequality [46, Corollary 6.2.1]). Let Z be a $d \sb { 1 } \times d \sb { 2 }$ random matrices such that $\mathbb { E } \left[ Z \right] = \bar { Z } a n d \| \dot { Z } \| \dot { \le K }$ almost surely. Consider M independent copy of Z as $Z _ { 1 } , \cdots , Z _ { M }$ , then for every $t \geq 0 ,$ , we have

$$\mathbb {P} \left[ \left\| \frac {1}{M} \sum_ {i} Z _ {i} - \bar {Z} \right\| \geq t \right] \leq (d _ {1} + d _ {2}) \exp \left\{\frac {- M t ^ {2} / 2}{\sigma^ {2} + 2 K t / 3} \right\}$$

where $\sigma ^ { 2 } = \operatorname* { m a x } \{ \| \mathbb { E } \left[ Z Z ^ { \mathsf { T } } \right] \| , \ \| \mathbb { E } \left[ Z ^ { \mathsf { T } } Z \right] \| \}$ is the per-sample second moment. This bound can be expressed as the mixture of sub-gaussian and sub-exponential tail as $\begin{array} { r } { ( d _ { 1 } + d _ { 2 } ) \exp \left\{ - c \operatorname* { m i n } \{ \frac { t ^ { 2 } } { \sigma ^ { 2 } } , \frac { t } { 2 K } \} \right\} } \end{array}$ for some c.

We are now well-equipped to provide the main proofs.
