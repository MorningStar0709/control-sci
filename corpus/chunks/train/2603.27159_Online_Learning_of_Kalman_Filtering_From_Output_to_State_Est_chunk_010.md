# 2.1 Steady-State Kalman Filter and Truncated Filters

In this subsection, we introduce the class of filters considered in the online optimization algorithms that will be proposed in later sections. We first observe from (2) and (5) that the idea of Kalman filtering is to compute the current state estimate $\hat { x } _ { t } ^ { \mathrm { K F } }$ based on the output history $y _ { 0 } , \ldots , y _ { t - 1 }$ Specifically, initialized with $\hat { x } _ { 0 } ^ { \mathtt { K F } } = 0$ , we unroll (2) to express $\hat { x } ^ { \mathrm { K F } }$ as

$$\hat {x} _ {t} ^ {\mathrm{KF}} = \sum_ {s = 1} ^ {t} \Psi_ {t, s} L _ {s - 1} y _ {s - 1}, \tag {7}$$

where for any $k , l \geq \mathbb { Z } _ { \geq 0 } , \Psi _ { k , l } \triangleq ( A - L _ { k - 1 } C ) ( A - L _ { k - 2 } C ) \times \cdots \times ( A - L _ { l } C ) { \mathrm { ~ i f ~ } } k > l$ and $\Psi _ { k , l } \triangleq I _ { n }$ if $k \leq l .$ . Thus, we may view $\left[ \Psi _ { t , s } L _ { s - 1 } \right] _ { s \in [ t ] }$ as the (optimal) parameter associated with the Kalman filter $\hat { x } _ { t } ^ { \\mathrm { K F } }$ . However, the parameter $\left[ \Psi _ { t , s } L _ { s - 1 } \right] _ { s \in [ t ] }$ is time varying over t and its dimension grows linearly as t. To overcome these drawbacks, we adopt the steady-state version of $\hat { x } _ { t } ^ { \mathrm { K F } }$ (see, e.g., [4, Chapter 4]) and truncate it to a fixed-length interval of the past outputs. Formally, denote

$$M _ {s} ^ {\star} \triangleq (A - L C) ^ {s - 1} L \in \mathbb {R} ^ {n \times p}, \forall s \geq 1, \tag {8}$$

where

$$L = A \Sigma C ^ {\top} (C \Sigma C ^ {\top} + V) ^ {- 1}, \tag {9}$$

and Σ is the unique positive semidefinite solution to the following discrete algebraic Ricatti equation:1

$$\Sigma = A \Sigma A ^ {\top} - A \Sigma C ^ {\top} (C \Sigma C ^ {\top} + V) ^ {- 1} C \Sigma A ^ {\top} + W. \tag {10}$$

Defining $y _ { t } = 0$ for all $t < 0$ , the steady-state Kalman filter $\hat { x } _ { t } ^ { \mathrm { S K } }$ is given by

$$\hat {x} _ {t} ^ {\mathrm{SK}} = \sum_ {s = 1} ^ {t} M _ {s} ^ {\star} y _ {t - s}, \forall t \geq 0, \tag {11}$$

and a truncated steady-state Kalman filter $\hat { x } _ { t } ^ { \mathrm { T K } }$ is given by

$$\hat {x} _ {t} ^ {\mathrm{TK}} = \sum_ {s = 1} ^ {h} M _ {s} ^ {\star} y _ {t - s}, \forall t \geq 0, \tag {12}$$

where $h \in \mathbb { Z } _ { \geq 1 }$ will be a tuning parameter in our algorithm that is kept fixed for all t. It follows from (12) that the parameter associated with $\hat { x } _ { t } ^ { \mathrm { T K } }$ satisfies that $[ M _ { s } ^ { \star } ] _ { s \in [ h ] } \in \mathbb { R } ^ { n \times ( p h ) }$ and does not change over time t.
