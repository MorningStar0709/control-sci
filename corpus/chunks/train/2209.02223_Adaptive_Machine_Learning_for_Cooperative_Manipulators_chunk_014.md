# 2.2 Parametric Design of Coordination Control

Fig.2 schematically illustrates the self-tuning cooperative controller which incorporates the updated kinematic parameters from the estimator. The concern is the stability of the entire estimation and control process and the tracking performance of the controller, given the timevarying kinematic parameters which starts from an initial guess at the parameters.

Assume that vectors $\mathbf { \Delta } _ { \mathbf { \mathcal { X } } _ { i } }$ is the minimal representation of the position and orientation of the ith robot end-effector. Then, the following mapping is in order

$$
\dot {\boldsymbol {x}} _ {i} \triangleq \boldsymbol {L} _ {i} \left[ \begin{array}{l} \boldsymbol {v} _ {i} \\ \boldsymbol {\omega} _ {i} \end{array} \right] \quad \forall i = 1, 2 \tag {23}
$$

where $\pmb { { \cal L } } _ { i } = \mathrm { d i a g } ( { \pmb I } , { \pmb L } _ { o } ( { \pmb q } _ { i } ) )$ , and transformation matrix $\scriptstyle { \pmb { L } } _ { o }$ depends on a particular set of parameters used to represent the orientation [41]. Also, assume that the unknown kinematic parameters are placed in a single vector

$$
\boldsymbol {\theta} = \left[ \begin{array}{c} \boldsymbol {\rho} \\ \boldsymbol {\eta} \end{array} \right].
$$

Then, in view of (3) and (23), one can show that the generalized velocities of the two manipulators are related by

$$\dot {\boldsymbol {x}} _ {2} = \boldsymbol {L} _ {2} \boldsymbol {T} (\boldsymbol {\theta}) \boldsymbol {L} _ {1} ^ {- 1} \dot {\boldsymbol {x}} _ {1} \tag {24}$$

where

$$
\boldsymbol {T} (\boldsymbol {\theta}) = \left[ \begin{array}{c c} \boldsymbol {A} (\boldsymbol {\eta}) ^ {T} & \boldsymbol {A} (\boldsymbol {\eta}) ^ {T} [ \boldsymbol {\rho} \times ] \\ \boldsymbol {0} & \boldsymbol {A} (\boldsymbol {\eta}) ^ {T} \end{array} \right] \tag {25}
$$

is the velocity transformation matrix. Time-derivative of (24) yields

$$\ddot {\boldsymbol {x}} _ {2} = \boldsymbol {L} _ {2} \boldsymbol {T} \boldsymbol {L} _ {1} ^ {- 1} \ddot {\boldsymbol {x}} _ {1} + \boldsymbol {D} \dot {\boldsymbol {x}} _ {1}, \tag {26}$$

where ${ \cal D } = \dot { L } _ { \mathrm { 2 } } T - L _ { \mathrm { 2 } } T L _ { \mathrm { 1 } } ^ { - 1 } \dot { L } _ { \mathrm { 1 } } L _ { \mathrm { 1 } } ^ { - 1 }$ .
