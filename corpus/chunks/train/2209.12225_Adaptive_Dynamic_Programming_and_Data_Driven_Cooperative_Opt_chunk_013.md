Now we are ready to introduce the variables $\bar { x } _ { i j } = x _ { i } - X _ { i j } v ,$ $j = 0 , 1 , 2 , . . . , h _ { i } + 1$ with $h _ { i } = ( n _ { i } + p _ { i } ) q$ being the dimension of the null space of $\left( I _ { q } \otimes C _ { i } \right)$ , where the following is met: $X _ { i 0 } =$ $0 _ { n _ { i } \times q } , \ { X _ { i 1 } } ^ { - } \in \ \mathbb { R } ^ { n _ { i } \times q ^ { - } }$ such that $C _ { i } X _ { i 1 } + F _ { i } = 0$ , and $X _ { i j } \in$ $\mathbb { R } ^ { n _ { i } \times \mathbf { \dot { q } } }$ , $\forall j \in 2 , 3 , . . . , h _ { i } + 1$ , such that all $\mathrm { v e c } ( X _ { i j } )$ form a basis for ker $\left( I _ { q } \otimes C _ { i } \right)$ , where ker(·) denotes the null space.

The definition of $\bar { x } _ { i j }$ enbales us to solve the Sylvester map of trail matrices $X _ { i j }$ which is in itself crucial for solving the regulator equations to finally approximate $L ^ { \star }$ and $K ^ { \star }$ without previous knowledge of the systems’ matrices $( A _ { i } , B _ { i }$ , and $D _ { i } )$ . By taking the time derivative along the trajectories of $\bar { x } _ { i j }$ we have

$$
\begin{array}{l} \dot {\bar {x}} _ {i j} = \dot {x} _ {i} - X _ {i j} \dot {v} \\ = A _ {i} x _ {i} + B _ {i} u _ {i} + D _ {i} v - X _ {i j} E v \\ = A _ {i, k} \bar {x} _ {i j} + B _ {i} (K _ {i, k} \bar {x} _ {i j} + u _ {i}) + (D _ {i} - S _ {i} (X _ {i j})) v, \tag {23} \\ \end{array}
$$

where $A _ { i . k } = A _ { i } - B _ { i } K _ { i , k }$ and $S _ { i } ( X ) = X E \mathrm { - } A _ { i } X$ is a Sylvester map, $S _ { i } : \mathbb { R } ^ { n _ { i } \times q }  \mathbb { R } ^ { n _ { i } \times q }$ . By taking the integration over the time inerval $[ t , t + \delta t ] , \delta t > 0$ , we obtain

$$
\begin{array}{l} | \bar {x} _ {i j} (t + \delta t) | _ {P _ {i, k}} - | \bar {x} _ {i j} (t) | _ {P _ {i, k}} \\ = \int_ {t} ^ {t + \delta t} \left(| \bar {x} _ {i j} | _ {\left(A _ {i, k} ^ {\mathrm{T}} P _ {i, k} + P _ {i, k} A _ {i, k}\right)} + 2 \left(u _ {i} + K _ {i, k} \bar {x} _ {i j}\right) ^ {\mathrm{T}} B _ {i} ^ {\mathrm{T}} P _ {i, k} \bar {x} _ {i j} \right. \\ \left. + 2 v ^ {\mathrm{T}} \left(D _ {i} - \mathcal {S} _ {i} \left(X _ {i j}\right)\right) ^ {\mathrm{T}} P _ {i, k} \bar {x} _ {i j}\right) \mathrm{d} \tau \\ = \int_ {t} ^ {t + \delta t} \left(- | \bar {x} _ {i j} | _ {(Q _ {i} + K _ {i, k} ^ {\mathrm{T}} R _ {i} K _ {i, k})} + 2 (u _ {i} + K _ {i, k} \bar {x} _ {i j}) ^ {\mathrm{T}} R _ {i} K _ {i, k + 1} \bar {x} _ {i j} \right. \\ \left. + 2 v ^ {\mathrm{T}} \left(D _ {i} - \mathcal {S} _ {i} \left(X _ {i j}\right)\right) ^ {\mathrm{T}} P _ {i, k} \bar {x} _ {i j}\right) \mathrm{d} \tau . \tag {24} \\ \end{array}
$$
