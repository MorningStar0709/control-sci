A system G is said to be causal if $P _ { T } G d = P _ { T } G ( P _ { T } d )$ , i.e. the output up to time T only depends on the input up to time T . The system is said to be non-causal if it is not causal, i.e. the output can possibly on future values of the input.

The matrix $A \in \mathbb { R } ^ { n \times n }$ is said to be Schur stable if the spectral radius is $< 1$ . If A is Schur stable then G is a causal, stable system. Hence G maps an input $d \in \ell _ { 2 }$ to output $e \in \ell _ { 2 }$ starting from the initial condition $x _ { - \infty } = 0$ . $$\| G \| _ {2 \to 2} := \max _ {0 \neq d \in \ell_ {2}} \frac {\| G d \| _ {2}}{\| d \| _ {2}}.$$
