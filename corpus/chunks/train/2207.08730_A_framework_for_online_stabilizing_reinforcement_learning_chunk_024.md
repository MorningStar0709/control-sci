$$
\begin{array}{l l} w ^ {*} \leftarrow & \underset {w \in \mathbb {W}} {\arg \min} \left(\hat {J} ^ {w} (x _ {k - 1}) - r _ {\text { prev }} - \hat {J} ^ {w _ {\text { prev }}} (x _ {k})\right) ^ {2} \\ & \text { s.t. } \quad \begin{array}{c} \hat {J} ^ {w} (x _ {k}) - \hat {J} ^ {w _ {\text { prev }}} (x _ {\text { prev }}) \leq - \bar {\nu} \delta , \\ \hat {\alpha} _ {\text { low }} (\| x _ {k} \|) \leq \hat {J} ^ {w} (x _ {k}) \leq \hat {\alpha} _ {\text { up }} (\| x _ {k} \|); \end{array} \end{array}
$$

if solution $w ^ { * }$ found then   
$w_{k}\gets w^{*}$ Update the actor: $U_{k\delta}\gets \arg \min_{u\in \mathbb{U}}\left\{r(x_{k},u) + \hat{J}^{w_{k}}(\Lambda_{u}(x_{k}))\right\};$ $x_{\mathrm{prev}}\gets x_k$ $w_{\mathrm{prev}}\gets w_k$ $r_{\mathrm{prev}}\gets r(x_k,u_k)$

Invoke a stabilizing action: $U_{k\delta} \leftarrow \eta(x_k)$ $r_{\mathrm{prev}} \leftarrow r(x_k, u_k) + r_{\mathrm{prev}};$
