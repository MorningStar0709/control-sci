Then, making use of the Cauchy-Schwarz inequality, the second term in (18) can be upper bounded as

$$\left| \boldsymbol {\zeta} ^ {\top} S \boldsymbol {g} _ {\perp} ^ {* ^ {\top}} W \bar {\boldsymbol {\tau}} \right| \leq \left\| \boldsymbol {g} _ {\perp} ^ {* ^ {\top}} W \right\| \| S \boldsymbol {\zeta} \| \bar {\tau} _ {\max}. \tag {20}$$

Substituting into (18) gives

$$\dot {V} (\boldsymbol {\zeta}) \leq - \alpha \| S \boldsymbol {\zeta} \| ^ {2} + \beta \| S \boldsymbol {\zeta} \|. \tag {21}$$

where $\begin{array} { r l } { \alpha } & { { } : = \phantom { + } \operatorname* { i n f } _ { t \geq 0 } ( \lambda _ { \operatorname* { m i n } } ( \pmb { g } _ { \perp } ^ { * \mathrm { ~ T ~ } } W ( t ) \pmb { g } _ { \perp } ^ { * } ) ) } \end{array}$ and $\beta \quad : =$ $\begin{array} { r } { \operatorname* { s u p } _ { t \geq 0 } ( { \bar { \tau } _ { \operatorname* { m a x } } } \| g _ { \bot } ^ { * } \ W ( t ) \| ) } \end{array}$ . $$\dot {V} (\boldsymbol {\zeta}) < 0 \quad \forall \| S \boldsymbol {\zeta} \| > \frac {\beta}{\alpha}, \tag {22}$$
