# Pulse Input

A pulse input consists of a constant (step) input that lasts for a finite duration, and then instantly steps to zero. Therefore, a pulse input with magnitude A can be described as

$$
\text { Pulse   input } \quad u (t) = \left\{ \begin{array}{l l} 0 & \text { for } \quad t \leq 0 \\ A & \text { for } \quad 0 <   t <   t _ {1} \\ 0 & \text { for } \quad t \geq t _ {1} \end{array} \right. \tag {5.125}
$$

In Eq. (5.125), the pulse input steps up to magnitude A at time t = 0, and then steps down to zero at time $t = t _ { 1 }$ .
