$$
\begin{array}{l} \mathbf {x} _ {\mathrm{N}} = \mathbf {A} ^ {\mathrm{N}} \mathbf {x} _ {0} + \mathbf {A} ^ {\mathrm{N} - 1} \mathbf {B u} _ {0} + \mathbf {A} ^ {\mathrm{N} - 2} \mathbf {B u} _ {1} + \dots + \mathbf {B u} _ {\mathrm{N} - 1} = \mathbf {A} ^ {\mathrm{N}} \mathbf {x} _ {0} + \mathbf {C} _ {\mathbf {u}} \mathbf {U} _ {0} \\ \vdots \end{array} \tag {48}
\mathbf {x} _ {\mathrm{N} + \mathrm{H} - 1} = \mathbf {A} ^ {\mathrm{N}} \mathbf {x} _ {\mathrm{H} - 1} + \mathbf {A} ^ {\mathrm{N} - 1} \mathbf {B} \mathbf {u} _ {\mathrm{H} - 1} + \mathbf {A} ^ {\mathrm{N} - 2} \mathbf {B} \mathbf {u} _ {\mathrm{H}} + \dots + \mathbf {B} \mathbf {u} _ {\mathrm{N} + \mathrm{H} - 2} = \mathbf {A} ^ {\mathrm{N}} \mathbf {x} _ {\mathrm{H} - 1} + \mathbf {C} _ {\mathbf {u}} \mathbf {U} _ {\mathrm{H} - 1}.$$

which can be compactly written as:

$$
\mathbf {X} _ {\mathrm{f}} = \mathbf {A} ^ {N} \mathbf {X} _ {\mathrm{p}} + \underbrace {\left[ \begin{array}{l l l l} \mathbf {A} ^ {N - 1} \mathbf {B} & \mathbf {A} ^ {N - 2} \mathbf {B} & \dots & \mathbf {B} \end{array} \right]} _ {\mathbf {C} _ {\mathrm{u}}} \mathbf {U} _ {\mathrm{p}}. \tag {49}
$$

In [16], it is shown that, for any observable system and $p \mathrm { N } \geq n$ , there exists a matrix L such that:

$$\mathbf {A} ^ {\mathrm{N}} + \mathbf {L A} _ {\mathbf {Y}} = 0. \tag {50}$$

Now, using Eqs. (47,49,50), we can represent $\mathbf { Y } _ { \mathrm { f } }$ as a linear function of $\mathbf { Y } _ { \mathrm { p } } , \mathbf { U } _ { \mathrm { p } }$ and $\mathbf { U } _ { \mathrm { f } }$ which is the key idea in SPC.
