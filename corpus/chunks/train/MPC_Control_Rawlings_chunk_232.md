These properties are almost identical to those required in Theorems B.11 and B.13 in Appendix B to establish asymptotic stability of the origin with a region of attraction $x _ { N }$ . The second property falls short because the upper bound holds for all x in $\mathbb { X } _ { f }$ rather than for all x in $x _ { N }$ . Despite this, asymptotic stability of the origin with a region of attraction $x _ { N }$ can still be established when $\mathbb { X } _ { f }$ contains the origin in its interior as we show subsequently. Alternatively, the second inequality is sometimes assumed to hold for all $x \in \mathcal { X } _ { N }$ , in which case asymptotic stability of the origin can be established using standard theorems in Appendix B; see the subsequent Assumption 2.23. Finally, as we show next, $V _ { N } ^ { 0 } ( x ) \leq \alpha _ { 2 } ( | x | )$ for all $x \in \mathbb { X } _ { f }$ implies, under some mild assumptions, that $V _ { N } ^ { 0 } ( x ) \leq \alpha _ { 2 } ( | x | )$ for all $\boldsymbol { x } \in \mathcal { X } _ { N }$ so that, under these assumptions, the results of Appendix B can again be used to prove asymptotic stability of the origin.
