Taking supremum of both sides with respect to b yields $H \geq H ^ { * * }$ .

To show $H \leq H ^ { * * }$ , we first observe that

$$H ^ {* *} (t, \boldsymbol {x}, p) = \sup _ {b \in \mathbb {R} ^ {n}} \left\{p \cdot b - H ^ {*} (t, \boldsymbol {x}, b) \right\}= \sup _ {b \in \mathbb {R} ^ {n}} \left[ p \cdot b - \sup _ {p ^ {\prime}} \left\{p ^ {\prime} \cdot b - H (t, \boldsymbol {x}, p ^ {\prime}) \right\} \right]= \sup _ {b \in \mathbb {R} ^ {n}} \inf _ {p ^ {\prime}} \left\{(p - p ^ {\prime}) \cdot b + H (t, \boldsymbol {x}, p ^ {\prime}) \right\}.$$

Therefore,

$$H ^ {* *} (t, \pmb {x}, p) \geq \inf _ {p ^ {\prime}} \left\{H (t, \pmb {x}, p ^ {\prime}) - (p ^ {\prime} - p) \cdot b \right\}$$

for all $b \in \mathbb { R } ^ { n }$ . Choosing $b = D _ { p } H ( t , \pmb { x } , p )$ , it follows from the convexity of $H ( t , { \boldsymbol { x } } , \cdot )$ that

$$
\begin{array}{l} H (t, \pmb {x}, p ^ {\prime}) - (p - p ^ {\prime}) \cdot b = H (t, \pmb {x}, p ^ {\prime}) - (p ^ {\prime} - p) \cdot D _ {p} H (t, \pmb {x}, p) \\ \geq H (t, \boldsymbol {x}, p), \\ \end{array}
$$

which implies that $H ^ { * * } \geq H$ . Therefore, we have $H ^ { * * } = H$ . By the uniqueness of the viscosity solution, we conclude that $V \equiv W$ .

To show that the second formula (2.7) is valid, we recall Proposition 1 implying that $H ^ { * } ( t , { \boldsymbol { x } } , b ) <$ < ∞ if and only if $b \in \mathrm { C o n v } ( B ( { \pmb x } , t ) )$ . The second formula directly follows from the definition of $B ( x ( s ) , s )$ and Proposition 1. □

Our representation formula is equivalent to the one in [1]. However, departing from their DP approach, our simple derivation makes direct use of the Lagrangian formula, taking the convex conjugate of the Hamiltonian as the Lagrangian. Another important observation is that the Lagrangian framework provides a natural representation of the value function with a convex function $H ^ { * }$ even if the original stage-cost function L is nonconvex. Exploiting this property, we propose a controller synthesis scheme via convex optimization in Section 4.

Now, our main question is whether the optimal solution of (2.7) yields to obtain a feasible control $u ( t ) \in U$ . In general, the convex conjugate of the Hamiltonian can be represented as

$$H ^ {*} (t, \boldsymbol {x}, b) = \sum_ {i = 1} ^ {N} \gamma_ {i} \tilde {L} (t, \boldsymbol {x}, b _ {i})$$

with $\begin{array} { r } { b = \sum _ { i = 1 } ^ { N } \gamma _ { i } b _ { i } \in \operatorname { C o n v } ( B ( \pmb { x } , t ) ) } \end{array}$ , where $b _ { i } \in B ( { \pmb x } , t ) , \gamma _ { i } \geq 0$ and $\textstyle \sum _ { i = 1 } ^ { N } \gamma _ { i } = 1$ . This representation comes from the duality property of the Legendre transform [21]. However, it is unclear whether an optimal $\beta ( t ) \in \operatorname { C o n v } ( B ( x ( t ) , t ) )$ induces an admissible control $u ( t ) \in U$ . We tackle this issue in Section 3.
