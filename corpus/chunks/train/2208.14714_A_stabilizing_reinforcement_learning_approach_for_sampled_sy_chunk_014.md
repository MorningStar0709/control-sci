Assumption 2: For any bounded $\mathbb { X } \subset \mathbb { R } ^ { n }$ containing the origin, i) there exist $q _ { 1 } , q _ { 2 } \in \mathcal { K } _ { \infty }$ such that for all $x \in \mathbb { X }$ and all $w \in \mathbb { W } , q _ { 1 } ( \| x \| ) \leq \hat { J } ( x , w ) \leq q _ { 2 } ( \| x \| )$ , ii) there exist $\bar { \varphi } _ { 1 } , \bar { \varphi } _ { 2 } > 0$ such that

$$
\sup_{\substack{x\in \mathbb{X}\\ w\in \mathbb{W}}}\| \nabla_{x}\hat{J} (x,w)\| \leq \bar{\varphi}_{1},\sup_{\substack{x\in \mathbb{X}\\ w\in \mathbb{W}}}\| \nabla_{x}^{2}\hat{J} (x,w)\| \leq \bar{\varphi}_{2},
$$

iii) given an adaptive control Lyapunov function $V ( x )$ , there exists $w ^ { \# } \in \mathbb { W }$ such that

$$V (x) \equiv \hat {J} (x, w ^ {\#}), \tag {19}$$

in other words, $\forall x \in \mathbb { X } V ( x ) = \hat { J } ( x , w )$ , and, iv), there exists a subset $\mathbb { W } ^ { \prime } \subset \mathbb { W }$ such that for all $w \in \mathbb { W } ^ { \prime } , V ( x ) \leq \hat { J } ( x , w )$ for all $x \in \mathbb { X }$ .

Remark 5: All requirements of Assumption 2 can be met by suitable design of $\varphi .$ For instance, iii) implies that the activation function be constructed so as to allow structural matching with V , which is thus required in analytical form, through a particular $w .$ This assumption may be relaxed in principle to approximate structure matching without disrupting the subsequent analysis substantially. By construction of $\varphi ,$ the set W may be decomposed into those weights $\mathbb { W } ^ { \prime }$ that hold the critic not smaller than the adaptive control Lyapunov function over the state space. A somewhat similar approach was used, e. g., in [17]. For instance, take V as a basis function of the form

$$\hat {J} (x, w) = w _ {1} V (x) + \sum_ {j = 2} ^ {l} w _ {j} \varphi_ {j} (x)$$

to which $\boldsymbol { w } ^ { \# } = [ 1 , 0 \ldots 0 ] ^ { \intercal }$ is the recovering weight as per the above assumption. With, $\mathbf { e . g . } , \varphi _ { j } ( x ) \geq V ( x ) , \mathbb { W } ^ { \prime } = \{ w \in$ $\mathbb { W } : w _ { 1 } \geq 1 , w _ { j } \geq 0 , j = 2 , \dots , l \}$ yields a suitable constraint set.

Positive definiteness of the approximant can be guaranteed by using, e. g., [37, Thm. 2], in which the Cauchy-Schwarz inequality and W ensure an upper bound and the lower bound is provided by the adaptive control Lyapunov function. That is,

$$
\begin{array}{l} q _ {1} (\| x \|) := \alpha_ {1} (\| x \|) \leq V (x) \\ \leq \hat {J} (x, w) \leq \overline {{w}} L _ {\varphi} (z, r) \| x \| =: q _ {2} (\| x \|). \\ \end{array}
$$

Property ii) is a bounding condition on the gradient and Hessian of the regressor, of which the former is a common assumption (see, e. g., [8], [49]) and the latter is required in establishing a sampling time.
