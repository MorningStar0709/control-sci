# 2 Problem statement

Consider an uncertain MIMO system in regular form, described by

$$
\begin{array}{l} \dot {\zeta} = A \zeta + E \sigma \\ \therefore C \zeta + D _ {1} + D _ {2} + f (t) \end{array} \tag {1}
\dot {\sigma} = C \zeta + D \sigma + B u + f (t)$$

where $\zeta \in \mathbb { R } ^ { r } , \sigma \in \mathbb { R } ^ { n }$ are the state variables, $u \in \mathbb { R } ^ { m }$ is the control input, $B \in \mathbb { R } ^ { n \times m }$ is the input matrix, and $f ( t ) : \mathbb { R } _ { + }  \mathbf { \bar { \mathbb { R } } ^ { n } }$ is the uncertain disturbance. With respect to it, the following assumptions are made:

(A1) The constant matrices (A, E, C, D, B) of compatible dimensions are uncertain. They are elements of the compact convex polyhedral set M generated by the convex combination of the vertices, that is $\mathbb { M } =$ co $\{ ( A , E , C , D , B ) _ { i } \} _ { i \in \mathbb { K } }$ .   
(A2) Each uncertain matrix $A \in \mathbb { M }$ is Hurwitz.   
(A3) Each uncertain matrix $B \in \mathbb { B } = \mathrm { c o } ( B _ { i } ) _ { i \in \mathbb { K } }$ with $\dot { B _ { i } } \in \mathbb { R } ^ { n \times m } , \forall i \in \mathbb { K }$ is full row rank.   
(A4) The disturbance f (t) is a smooth function, satisfying the following inequality

$$\| \dot {f} (t) \| \leq \delta , \quad \forall t \geq 0 \tag {2}$$

where δ is a positive constant.

Assumption (A1) implies that any element of M is given by $\begin{array} { r } { ( A , \mathsf { \tilde { E } } , C , D , B ) \stackrel { } { = } \sum _ { i \in \mathbb { K } } \lambda _ { i } ( A , E , C , D , B ) . } \end{array}$ i for some $\lambda \in \Lambda$ . Assumption $\left( \mathrm { A 2 } \right)$ ensures that the zero dynamics given by $\dot { \zeta } = A \zeta$ is globally exponentially stable. Notice that this condition is usual since the sliding surface can be appropriated defined to ensure that this assumption is fulfilled, see [20] and the example given in Section 7. Assumption (A3) leads to rank $( B ) = n , \forall B \in \mathbb { B }$ . The last assumption is usually adopted in the framework of MSTA control design, see [4].
