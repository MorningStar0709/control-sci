# 2.1 Feed-Forward Neural Network

The most prevalent form of Neural Networks (NNs) are feed-forward NNs, therefore usually simply referred to as neural networks. They consist of several layers of neurons and forward-facing connections between them, functionally transforming an input $\pmb { x } \in \mathbb { X } \subseteq \mathbb { R } ^ { n }$ into an output signal $\hat { \pmb y } \in \mathbb { Y } \subseteq \mathbb { R } ^ { l }$ , often used to recognize and model patterns in data. A NN f is defined by

$$\hat {\boldsymbol {y}} = \boldsymbol {\ell} (\boldsymbol {x}; \boldsymbol {\theta}), \quad \boldsymbol {\ell}: \mathbb {X} \mapsto \mathbb {Y}, \tag {1}$$

and its learnables $\pmb \theta : = [ ( \pmb w _ { i } , \pmb b _ { i } ) ; i = 1 , \varrho . . . ] \in \mathbb { R } ^ { q }$ with weight parameters ${ \pmb w } _ { i }$ of the connections and biases $\mathbf { } _ { b _ { i } }$ of the neurons for each layer i. Through an iterative training scheme, the learnables are adjusted to fit the network’s output yˆ to a given target $y \in \mathbb { Y }$ by error backpropagation, c.f. Bishop (2006). Besides the mentioned learnable parameters, a NN is determined by a comprehensive set of hyperparameters. These include, e.g., the number of hidden neurons, characterizing the complexity of the networks modeling capability.

Let the dynamics and output functions f and g of a timevariant, non-autonomous system S with state $\pmb { x } _ { t } \in \mathbb { X } \subseteq$ $\mathbb { R } ^ { n }$ , control input $\pmb { u } _ { t } \in \mathbb { U } \subseteq \mathbb { R } ^ { m }$ , output $\pmb { y } _ { t } \in \mathbb { Y } \subseteq \mathbb { R } ^ { l }$ , evolving in time $t \in \mathbb { T } \subseteq \mathbb { R }$ , as well as a set of parameters $\pmb { p } \in \mathbb { R } ^ { q }$ be given by

$$
\begin{array}{l} \dot {\boldsymbol {x}} _ {t} = \boldsymbol {f} \left(\boldsymbol {x} _ {t}, \boldsymbol {u} _ {t}, t; \boldsymbol {p}\right), \quad \boldsymbol {f}: \mathbb {X} \times \mathbb {U} \times \mathbb {T} \mapsto \mathbb {X}, \tag {2} \\ \boldsymbol {y} _ {t} = \boldsymbol {g} \left(\boldsymbol {x} _ {t}, \boldsymbol {u} _ {t}, t; \boldsymbol {p}\right), \quad \boldsymbol {g}: \mathbb {X} \times \mathbb {U} \times \mathbb {T} \mapsto \mathbb {Y}. \\ \end{array}
$$
