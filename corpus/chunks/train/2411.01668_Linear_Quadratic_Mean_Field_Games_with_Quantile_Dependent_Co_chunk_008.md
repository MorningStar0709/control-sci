# 4 Special Case with Variance Dependent Costs

If the cost coefficient is defined by

$$q _ {t} ^ {\alpha} = q \left(1 + e ^ {\sqrt {V (t)} \Phi_ {\alpha} ^ {- 1}}\right) \tag {23}$$

such that the mean does not appear in $q _ { t } ^ { \alpha }$ , then one can decouple the dynamics of $s ( t )$ and $\overline { { x } } ( t )$ via a decoupling Riccati equation (see e.g., [4, 13, 21]). If we introduce $s _ { t } = P _ { t } \overline { { x } } ( t )$ , then

$$\dot {s} _ {t} = \dot {P} _ {t} \overline {{{x}}} (t) + P _ {t} \dot {\overline {{{x}}}} (t). \tag {24}$$

Replacing $\dot { s } _ { t }$ and $\dot { \overline { { x } } } ( t )$ using (14) and (13) yields

$$- \dot {P} _ {t} \overline {{x}} = \left[ 2 (a - \frac {b ^ {2}}{r} \Pi_ {t}) P _ {t} - \frac {b ^ {2}}{r} P _ {t} ^ {2} - q _ {t} ^ {\alpha} \right] \overline {{x}}.$$

Since this should hold for all ${ \overline { { x } } } ,$ we obtain the following Riccati equation that decouples the dynamics for x and s:

$$- \dot {P} _ {t} = 2 (a - \frac {b ^ {2}}{r} \Pi_ {t}) P _ {t} - \frac {b ^ {2}}{r} P _ {t} ^ {2} - q _ {t} ^ {\alpha}, \quad P _ {T} = 0.$$

Remark 4.1 Such an analysis for decoupling s and x would not work when $q _ { t } ^ { \alpha }$ depends on the mean x. This is because $P ( \overline { { x } } , t )$ depends on x through the coefficient $q _ { t } ^ { \alpha }$ of the Riccati equation for $P$ and in this case one should introduce $s _ { t } = P ( \overline { { x } } ( t ) , t ) \overline { { x } } ( t )$ . Hence one should use

$$\dot {s} _ {t} = \left[ \frac {\partial}{\partial t} \dot {P} + \frac {\partial P}{\partial \overline {{x}}} \cdot \dot {\overline {{x}}} (t) \right] \overline {{x}} (t) + P \dot {\overline {{x}}} (t)$$

to replace (24), and it may not lead to a decoupling Riccati equation for $P .$ .

Thus, the solution for the MFG problem defined by (5), (6) and (23), if exists is given by

$$u _ {i} (t) = - \frac {b}{r} (\Pi_ {t} x _ {i} (t) + P _ {t} \overline {{x}}) \tag {25}$$

with $\Pi _ { t }$ and $V ( t )$ given by (12) and (15), and x and $P _ { t }$ given by

$$\dot {\overline {{{x}}}} = (a - \frac {b ^ {2}}{r} (\Pi_ {t} + P _ {t})) \overline {{{x}}}, \quad \overline {{{x}}} (0) = \mu_ {0} \tag {26}- \dot {P} _ {t} = 2 (a - \frac {b ^ {2}}{r} \Pi_ {t}) P _ {t} - \frac {b ^ {2}}{r} P _ {t} ^ {2} - q _ {t} ^ {\alpha}, \quad P _ {T} = 0 \tag {27}$$

with $q _ { t } ^ { \alpha } = q ( 1 + e ^ { \sqrt { V ( t ) } \Phi _ { \alpha } ^ { - 1 } } ) \geq 0$ . We note that the last equation (27) can be solved after the two equations (12) and (15) for Πt and $V ( t )$ are solved.
