# 例6.1 电容的频率响应特性

考虑如下式所描述的电容：

$$i = C \frac {\mathrm{d} v}{\mathrm{d} t}$$

其中：v 是电压输入；i 是电流输出。确定电容的正弦稳态响应。

解答。这个电路的传递函数为

$$\frac {I (s)}{V (s)} = G (s) = C s$$

于是，

$$G (\mathrm{j} \omega) = C \mathrm{j} \omega$$

计算幅值和相位，我们得到

$$M = | C \mathrm{j} \omega | = C \omega , \quad \phi = \angle (C \mathrm{j} \omega) = 9 0 ^ {\circ}$$

对于单位幅值的正弦输入 v，输出 i 也是正弦曲线，其幅值为 $C\omega$ ，且输出相位将超前输入 $90^{\circ}$ 。注意，对于这个例子，幅值与输入频率成比例，而相位与频率无关。
