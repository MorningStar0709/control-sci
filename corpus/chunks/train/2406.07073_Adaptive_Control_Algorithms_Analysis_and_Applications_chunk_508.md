# Lemma 11.1

(1) Assume that:

$$\| \phi (t) \| _ {F (t)} = [ \phi^ {T} (t) F (t) \phi (t) ] ^ {1 / 2} \leq C _ {1} + C _ {2} \max _ {0 \leq k \leq t + d + 1} | \varepsilon^ {0} (k) |0 < C _ {1}, C _ {2} < \infty , F (t) > 0 \tag {11.24}$$

(2) Assume that:

$$\lim _ {t \to \infty} \frac {[ \varepsilon^ {0} (t + d + 1) ] ^ {2}}{1 + \| \phi (t) \| _ {F (t)} ^ {2}} = 0 \tag {11.25}$$

then:

$$\| \phi (t) \| \text { is bounded } \tag {11.26}\lim _ {t \to \infty} \varepsilon^ {0} (t + d + 1) = 0 \tag {11.27}$$

Proof The proof is trivial if $\varepsilon ^ { 0 } ( t )$ is bounded for all t. Assume now that $\varepsilon ^ { 0 } ( t +$ $d + 1 )$ is asymptotically unbounded. Then there is a particular subsequence such that: $\begin{array} { r } { \operatorname* { l i m } _ { t _ { n } \to \infty } \varepsilon ^ { 0 } ( t _ { n } ) = \infty } \end{array}$ and $| \varepsilon ^ { 0 } ( t + d + 1 ) | < | \varepsilon ^ { 0 } ( t _ { n } ) | ; \ t + d + 1 \le t _ { n }$ . For this sequence one has:

$$\frac {| \varepsilon^ {0} (t + d + 1) |}{(1 + \| \phi (t) \| _ {F} ^ {2}) ^ {1 / 2}} \geq \frac {\varepsilon^ {0} (t + d + 1)}{1 + \| \phi (t) \| _ {F (t)}} \geq \frac {| \varepsilon^ {0} (t + d + 1) |}{1 + C _ {1} + C _ {2} | \varepsilon^ {0} (t + d + 1) |}$$

but:

$$\lim _ {t \rightarrow \infty} \frac {\left| \varepsilon^ {0} (t + d + 1) \right|}{1 + C _ {1} + C _ {2} \left| \varepsilon^ {0} (t + d + 1) \right|} = \frac {1}{C _ {2}} > 0$$

which contradicts Assumption 2 and proves that neither $\varepsilon ^ { 0 } ( t + d + 1 )$ nor $\| \phi ( t ) \|$ can become unbounded and that (11.27) is true. -
