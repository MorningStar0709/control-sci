$$\operatorname{tr} \left(\left(u _ {*} A + (1 - u _ {*}) B\right) ^ {2}\right) \geq \operatorname{tr} (A ^ {2}) \quad \text { iff } \quad \operatorname{tr} (A B) \geq \frac {\operatorname{tr} (A ^ {2}) + \operatorname{tr} (B ^ {2})}{2}. \tag {5.9}$$

${ \mathrm { S o } } ,$ taking into account that singular control is admissible if $\operatorname { t r } ( A B ) \geq \operatorname { t r } ( A ^ { 2 } )$ , we obtain that singular constant control are better than constant control if $\operatorname { t r } ( A B ) \geq \operatorname { t r } ( A ^ { 2 } )$ . On the other hand, if $\mathrm { t r } ( A B ) \leq$ $\operatorname { t r } ( B ^ { 2 } )$ then constant control is better than constant singular control.

The following result ends the discussion about the singular case.

Proposition 5.10. There are no solution to the adjoint system (2.5) containing a bang-bang piece and a singular piece.

Proof. We already noticed that for $\eta _ { * }$ we have $\gamma = 0$ in (4.9) and the intersection line between the planes $\Pi _ { A , c _ { * } }$ and $\Pi _ { B , c _ { * } }$ is tangent to the hyperboloid H, where $c _ { * } = \mathrm { t r } ( A \eta _ { * } )$ . Recall from Section 4 that as the switching time in a bang-bang trajectory in $\mathrm { S L _ { 2 } ( \mathbb { R } ) }$ tends to zero, the resulting trajectory tends to a singular trajectory. The convergence in $\operatorname { S L _ { 2 } } ( \mathbb { R } )$ implies that also the sequence of the corresponding adjoint trajectories tends to the singular adjoint trajectory, which is constant. Thus, in a neighborhood of $\eta _ { * }$ sufficiently small and contained in the hyperboloid H, the phase portrait is as in Figure 4.

So, to resume, we found that if an extremal contains a singular piece, then the whole extremal is singular, and the corresponding Lyapunov exponent is

$$\ell_ {\text { sing }} (A, B) = \frac {1}{2} \sqrt {\frac {\text { tr } ([ A , B ] ^ {2})}{2 \text { tr } (A B) - \text { tr } (A ^ {2}) - \text { tr } (B ^ {2})}}, \tag {5.11}$$

and this is the optimal solution if $\operatorname { t r } ( A B ) \geq \operatorname { t r } ( A ^ { 2 } )$ .
