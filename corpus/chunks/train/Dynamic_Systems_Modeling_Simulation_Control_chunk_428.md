# Differentiation

$\mathbf { a } . { \mathcal { L } } \{ { \dot { f } } ( t ) \} = s F ( s ) - f ( 0 )$ (8.5)   
$\mathbf { b } . \mathcal { L } \{ \ddot { f } ( t ) \} = s ^ { 2 } F ( s ) - s f ( 0 ) - \dot { f } ( 0 )$ (8.6)   
$\mathbf { c . } ~ { \mathcal { L } } \Big \{ \frac { d ^ { n } } { d t ^ { n } } f ( t ) \Big \} = s ^ { n } F ( s ) - s ^ { n - 1 } f ( 0 ) - s ^ { n - 2 } { \dot { f } } ( 0 ) - \cdot \cdot \cdot - f ^ { ( n - 1 ) } ( 0 )$ (8.7)

In the above equations $f ( 0 )$ is the initial value of $f ( t )$ evaluated at $t = 0 , \dot { f } ( 0 )$ is the initial value of the first time derivative, $f ^ { ( n - 1 ) } ( 0 )$ ) is the initial value of the $n - 1$ time derivative, and so on. If all initial conditions are zero then the Laplace transform of the nth time derivative of $f ( t )$ is $s ^ { n } F ( s )$ . In general, differentiation in the time domain is equivalent to multiplication by s in the Laplace domain.
