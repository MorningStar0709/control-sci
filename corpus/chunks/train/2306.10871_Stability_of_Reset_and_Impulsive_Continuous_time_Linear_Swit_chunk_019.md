Consequently, if we have a compact set of possible resets M not containing O and any signal $\sigma \in \mathcal { S } ^ { \mathcal { G } } [ \tau , \eta ]$ with $\tau \geq \tau ^ { * }$ and $\eta \leq \eta ^ { * }$ , there exists a collection of nonzero reset maps $\mathcal { R } \subseteq \mathbb { M }$ so that the reset switched system $\Sigma _ { \sigma } ^ { \mathcal { R } }$ is stable. Also note that $\eta ^ { * }$ is dependent on the initial choice of $\epsilon _ { \mathrm { : } }$ , hence can be made arbitrary large or small, and yields a corresponding dwell time $\tau ^ { * }$ . So there is a relation (through the choice of Jordan basis matrices $P _ { p } )$ between $\tau ^ { * }$ and $\eta ^ { * }$ . Such relations exist in the literature in the context of stability of switched systems with both stable and unstable subsystems, and are referred to as dwell-flee relations [24]. The preceding theorem, thus, states that switched systems satisfying the above dwell-flee relations (dwell time constraints, when there are no unstable subsystems) allow us to choose resets from the given compact set of matrices so that the reset switched system is stable.
