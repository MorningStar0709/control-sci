# IV. MAIN RESULT

In this section, we present solutions to Problem 1. Consider system (1) with distributed state feedback (2). In the following, for the communication graph ${ \mathcal { G } } ,$ , we suppose that Assumption 1 holds. First, we present $\hat { \mathcal { K } } _ { S , \mathrm { e x t } }$ , a solution set to Problem 1, characterized by non-convex optimization. In this formulation, the block diagonal restriction is imposed on the auxiliary matrix $\tilde { G }$ rather than on ${ \tilde { Q } } .$ thereby removing the structural constraint on the Lyapunov matrix.

Lemma 6: Let

$$\hat {\mathcal {K}} _ {\mathcal {S}, \text { ext }} = \left\{K = (E ^ {\top} E) ^ {- 1} E ^ {\top} \tilde {Z} \tilde {G} ^ {- 1} E \in \mathcal {S}: \exists \tilde {Q} \succ O \right.\tilde {Z} = \operatorname{blkdiag} \left(\tilde {Z} _ {1}, \dots \tilde {Z} _ {q}\right), \rho \in \mathbb {R}\tilde {G} = \operatorname{blkdiag} \left(\tilde {G} _ {1}, \dots \tilde {G} _ {q}\right)\text { s.t. } \Psi (\tilde {G}, \tilde {Q}, \tilde {Z}) + \rho \tilde {H} ^ {\top} W \tilde {H} \succ O \} \tag {11}$$

with W in (9) and

$$
\Psi (\tilde {G}, \tilde {Q}, \tilde {Z}) = \left[ \begin{array}{c c} \tilde {G} + \tilde {G} ^ {\top} - \tilde {Q} & * \\ \tilde {A} \tilde {G} + \tilde {B} \tilde {Z} & \tilde {Q} \end{array} \right], \tag {12}

Proof: First, we prove that $\hat { \kappa } _ { s , \mathrm { e x t } } \subset \kappa _ { \mathrm { a l l } }$ . Consider $K ~ = ~ ( E ^ { \top } E ) ^ { - 1 } E ^ { \top } \tilde { Z } \tilde { G } ^ { - 1 } E ~ \in ~ \hat { \mathcal { K } } _ { S , \mathrm { e x t } }$ .
