The primary purpose of this article is to deal with the robust PTCC problems of the CAN (2). To this end, we are first devoted to proposing a distributed control protocol to address the PTCC problems of the disturbance-free nominal CAN described by (3). We then contribute to developing a prescribed-time sliding mode control protocol to tackle the robust PTCC problems of the CAN (2) based on the control protocol for the nominal CAN (3).

Definition 2. (Prescribed-Time Stability) We say the CAN (2) attains prescribed-time stability in a preassigned finite time $T \in \mathbb { R } _ { \geq 0 }$ if there is a suitable control protocol $u _ { k } , k \in \mathcal I _ { N }$ so that $\begin{array} { r } { \operatorname* { l i m } _ { t  T } x _ { k } ( t ) = 0 } \end{array}$ and $x _ { k } \left( t \right) = 0 , \forall t \geq T$ hold for $\forall k \in \mathcal { I } _ { N }$ .

Definition 3. (Prescribed-Time Bipartite Consensus) We say the CAN (2) accomplishes prescribed-time bipartite consensus in a preassigned finite time $T \in \mathbb { R } _ { > 0 }$ if there exist a suitable control protocol $u _ { k } , k \in \mathcal I _ { N }$ so that lim $_ { t  T } | x _ { k } ( t ) | = x ^ { * }$ and $| x _ { k } \left( t \right) | = x ^ { * } , \forall t \ge T$ hold for $\forall k \in \mathcal { I } _ { N } ,$ , where $x ^ { * } > 0$ .

(Prescribed-Time Interval Bipartite Consensus) The CAN (2) under a quasi-strongly connected signed digraph G is said to attain prescribed-time interval bipartite consensus within a preassigned finite time $T \in \mathbb { R } _ { > 0 }$ if there exist $x _ { f } >$ 0 and a proper control protocol $u _ { k } , k \in \mathcal I _ { N }$ such that the following hold:

1) $\operatorname* { l i m } _ { t  T } | x _ { k } ( t ) | = x _ { f }$ and $| x _ { k } \left( t \right) | = x _ { f } , \forall t \geq T , k \in \mathcal { L } ;$
