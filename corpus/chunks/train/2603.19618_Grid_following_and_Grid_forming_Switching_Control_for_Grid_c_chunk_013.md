# A. Definition of the SSSR for Switched System

In general, a power system can be mathematically modeled as a set of parameterized differential-algebraic equations (DAEs). By performing small-signal linearization around a steady-state equilibrium, the system eigenvalues can be computed to assess small-signal stability. Consequently, the SSSR of a power system is defined as the set of operating points within the parameter space where small-signal stability is strictly preserved, while the boundary of this region is delineated by the operating points corresponding to the system’s marginal stability limits [30].

According to Lyapunov’s indirect method, when all eigenvalues of the state matrix A have negative real parts, the system is asymptotically stable about its steady-state operating point $\scriptstyle { \pmb x } _ { 0 } .$ . Consequently, the system is deemed to be smallsignal stable. Both the state matrix A and the corresponding eigenvalue vector $\lambda = [ \lambda _ { 1 } , \lambda _ { 2 } , \cdots , \lambda _ { n } ]$ are functions of the system parameter vector $\pmb { p } \in \mathbb { R } ^ { l }$ . Considering an l-dimensional parameter space S := span $\{ p _ { 1 } , p _ { 2 } , \cdots , p _ { l } \}$ spanned from p, the SSSR is mathematically defined as

$$\boldsymbol {\Omega} := \left\{\boldsymbol {p} \in \mathcal {S} \mid \max _ {i} \left[ \operatorname{Re} \left(\lambda_ {i} (\boldsymbol {p})\right) \right] < 0 \right\} \tag {12}$$

where $\operatorname { R e } \left( \lambda _ { i } ( \pmb { p } ) \right)$ denotes the real part of the i-th eigenvalue. Accordingly, the boundary of SSSR is characterized as

$$\partial \boldsymbol {\Omega} := \left\{\boldsymbol {p} \in \mathcal {S} \mid - \varepsilon \leq \max _ {i} \left[ \operatorname{Re} \left(\lambda_ {i} (\boldsymbol {p})\right) \right] \leq 0 \right\} \tag {13}$$

where $\varepsilon \ > \ 0$ represents a predefined boundary tolerance, indicating that the rightmost eigenvalue is located critically close to the imaginary axis.
