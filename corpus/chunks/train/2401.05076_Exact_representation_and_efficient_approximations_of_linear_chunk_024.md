# 4.1. Structured U-HTNN architecture

Notice that in both H and $\mathbf { q } ( x _ { 0 } )$ the term $\mathbf { A } ^ { T } \mathbf { Q }$ appears, which we will denote with $Q _ { 1 1 } ^ { [ k ] }$ . The remaining terms are denoted $Q _ { 1 2 } ^ { [ k ] } = \mathbf { A }$ and $Q _ { 2 1 } ^ { [ k ] } = { \bf A } _ { N }$ .
