# Appendix B

Dynamics equations of every manipulator in the joint space is described by

$$\boldsymbol {M} _ {i} ^ {\prime} \ddot {\boldsymbol {q}} _ {i} + \boldsymbol {h} _ {i} ^ {\prime} (\boldsymbol {q} _ {i}, \dot {\boldsymbol {q}} _ {i}) = \boldsymbol {\tau} _ {i} + \boldsymbol {J} _ {i} ^ {T} \boldsymbol {f} _ {i} \tag {55}$$

Substituting the joint acceleration from $\ddot { \pmb q } _ { i } = { \pmb J } _ { i } ^ { - 1 } \ddot { \pmb x } _ { i } - { \pmb J } _ { i } ^ { - 1 } \dot { \pmb J } _ { i } \dot { \pmb q } _ { i }$ into (55) and then multiply the resultant equation by $J _ { i } ^ { - T }$ yields (27), in which the dynamics parameters in the task space are related to those in the joint space by

$$\boldsymbol {M} _ {i} \triangleq \boldsymbol {J} _ {i} ^ {- T} \boldsymbol {M} _ {i} ^ {\prime} \boldsymbol {J} _ {i} ^ {- 1}\pmb {h} _ {i} \triangleq \pmb {J} _ {i} ^ {- T} \pmb {h} _ {i} ^ {\prime} - M _ {i} \dot {\pmb {J}} _ {i} \dot {\pmb {q}}$$
