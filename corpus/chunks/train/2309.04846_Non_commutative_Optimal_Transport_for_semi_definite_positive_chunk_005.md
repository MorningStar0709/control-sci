# 2 Unbalanced Optimal Transport

This section aims to provide a brief, self-contained introduction of (static) Unbalanced Optimal Transport between positive measures, to draw an analogy to the Non-commutative case in section 3. A similar approach has also been considered in [2], including proof of convergence for the Unbalanced Sinkhorn algorithm and the Unbalanced multi-marginal optimal transport theory.

Let X and Y be complete separable metric spaces, $\mu \in { \mathcal { M } } _ { + } ( X )$ and $\nu \in \mathcal { M } _ { + } ( Y )$ be positive measures, and let $c : X \times Y $ R be a measurable function on $X \times Y$ . The Unbalanced Optimal Transport (UOT) problem is defined by

$$\mathcal {O T} ^ {\tau_ {1}, \tau_ {2}} (\mu , \nu) = \inf _ {\gamma \in \mathcal {M} _ {+} (X \times Y)} \int_ {X \times Y} c (x, y) d \gamma + \tau_ {1} \mathrm{KL} ((e _ {1}) _ {\sharp} \gamma | \mu) + \tau_ {2} \mathrm{KL} ((e _ {2}) _ {\sharp} \gamma | \nu), \tag {2.1}$$

where $e _ { 1 } : X \times Y \to X$ and $e _ { 2 } : X \times Y \to Y$ are the projection operators, i.e. $e _ { 1 } ( x , y ) = x $ , $e _ { 2 } ( x , y ) = y$ , and $( e _ { i } ) _ { \sharp } \gamma$ , for $i = 1 , 2$ denotes the push-forward of the measure γ via the projector operator $e _ { i }$ . The functional KL is the Kullback-Leibler divergence between two positive measures α and $\beta$

$$
\operatorname{KL} (\alpha | \beta) = \left\{ \begin{array}{l l} \int_ {+ \infty} \left[ \frac {d \alpha}{d \beta} \left(\log \left(\frac {d \alpha}{d \beta}\right) - 1\right) + 1 \right] d \beta , & \text {if} \alpha \ll \beta , \\ & \text {otherwise}, \end{array} \right. \tag {2.2}
$$

where $\textstyle { \frac { d \alpha } { d \beta } }$ denotes the Radon-Nikodym derivative of α with respect to $\beta$ .

The main idea behind UOT is to introduce a so-called slack variable that represents the discrepancy between the masses of the source and target distributions. The UOT functional (2.1) extends classical Optimal Transport theory to handle scenarios where the mass (or density) of the data $\mu , \nu$ are positive measures rather than probability ones.

Therefore, the problem (2.1) relaxes the constraint that the total mass of the source and target distributions must be equal, allowing for imbalanced mass distributions, in the sense that if $\mu , \nu$ are probability distributions, the UOT problem (2.1) reduces to the optimal transport problem when $\tau _ { 1 } , \tau _ { 2 } \to + \infty$

$$\mathcal {O T} (\mu , \nu) = \inf _ {\gamma \in \mathcal {M} _ {+} (X \times Y)} \int_ {X \times Y} c (x, y) d \gamma + i ((e _ {1}) _ {\sharp} \gamma | \mu) + i ((e _ {2}) _ {\sharp} \gamma | \nu), \tag {2.3}$$

where i denotes the indicator function, i.e. $i ( \alpha | \beta ) = 0 \ \mathrm { i f } \ \alpha = \beta$ and $i ( \alpha | \beta ) = + \infty$ otherwise.
