# A. Preliminaries of experimental validation

Experiments were performed with a mobile robot under CALF in Algorithm 1 variation with the critic loss defined via temporal difference over an experience replay as per (10). The noiseless dynamics part of the robot were described as:

$$\mathrm{d} X _ {t} = f _ {\text { Cart }} (X _ {t}, U _ {t}) \mathrm{d} t, \tag {14}$$

where $T _ { \mathrm { C a r t } } ( x _ { 1 } , x _ { 2 } , x _ { 3 } , u _ { 1 } , u _ { 2 } ) \ = \ \left( u _ { 1 } \cos x _ { 3 } , u _ { 1 } \sin x _ { 3 } , u _ { 2 } \right) ^ { \top }$ , $x _ { 1 } , x _ { 2 } , x _ { 3 }$ are the Cartesian coordinates and angle, and $u _ { 1 } , u _ { 2 }$ are the linear and angular speed, respectively. Both actor and critic were implemented as neural networks. The running cost was considered in the following form:

$$\rho = \chi^ {\top} H \chi , \tag {15}$$

where $\chi = [ x - x ^ { * } , u ] ,$ , H is a diagonal, positive-definite, running cost matrix and $x ^ { * }$ is the target position (see details in the next section). The running cost matrices were taken in several variants, to differently prioritize the state and action components of the running cost, namely:

$$H _ {1} = \operatorname{diag} \left\{1 0, 1 0, 1 0, 0, 0 \right\}, H _ {2} = \operatorname{diag} \left\{1 0, 1 0, 1 0, 1, 1 \right\},H _ {3} = \operatorname{diag} \left\{1 0, 1 0, 1 0 0, 0, 0 \right\}, H _ {4} = \operatorname{diag} \left\{1 0, 1 0 0, 1 0, 0, 0 \right\},H _ {5} = \operatorname{diag} \left\{1 0 0, 1 0, 1 0, 0, 0 \right\}. \tag {16}$$

The model of the mobile robot is also known as the nonholonomic integrator (NI) $\begin{array} { r l } { ( \dot { x } _ { \mathrm { N I 1 } } , \dot { x } _ { \mathrm { N I 2 } } , \dot { x } _ { \mathrm { N I 3 } } ) ^ { \top } } & { { } = } \end{array}$ $( u _ { \mathrm { N I 1 } } , u _ { \mathrm { N I 2 } } , x _ { \mathrm { N I 1 } } u _ { \mathrm { N I 2 } }  \ \mathrm { ~ - ~ } \ x _ { \mathrm { N I 2 } } u _ { \mathrm { N I 1 } } ) ^ { \top }$ , for which various stabilization methods exist. We took $\begin{array} { r l } { L ( x _ { \mathrm { N I } } ) _ { \mathrm { . } } } & { { } = } \end{array}$ $\operatorname* { m i n } _ { \zeta \in [ - \rho , \rho ] } \left\{ x _ { \mathrm { N I 1 } } ^ { 4 } + x _ { \mathrm { N I 2 } } ^ { 4 } + \frac { | x _ { \mathrm { N I 3 } } | ^ { 3 } } { ( x _ { \mathrm { N I 1 } } \cos ( \zeta ) + x _ { \mathrm { N I 2 } } \sin ( \zeta ) + \sqrt { | x _ { \mathrm { N 3 } } | } ) ^ { 2 } } \right\}$ |xNI3|3 and the corresponding nominal parking policy as per [46], while using a transformation between the NI and Cartesian coordinates of (14) was used.
