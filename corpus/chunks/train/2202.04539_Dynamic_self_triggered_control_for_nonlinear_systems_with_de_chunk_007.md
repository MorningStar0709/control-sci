# 3. HYBRID LYAPUNOV FUNCTIONS FOR SYSTEMS WITH DELAYS

In this section, we present how a bound on a hybrid Lyapunov function similar to the one used in Hertneck and Allg¨ower (2021b) can be obtained for the setup with delays that is considered in this paper. The bound will be useful to determine sampling instants. We adapt the following condition from (Heemels et al., 2010, Condition IV.1) to our setup.

Condition 1. Consider some sets $\mathcal { X } \subseteq \mathbb { R } ^ { n _ { x } }$ and $\mathcal { E } \subseteq \mathbb { R } ^ { n _ { x } }$ . There exist a function $\tilde { W } ~ : ~ \{ 0 , 1 \} \times \mathcal { E } \times \mathcal { E } ~ \to ~ \mathbb { R } _ { \ge 0 }$ with $\tilde { W } ( \ell , . , . )$ locally Lipschitz for all $\ell ~ \in ~ \{ 0 , 1 \}$ , a locally Lipschitz function $\tilde { V } : \mathcal { X } \to \mathbb { R } _ { \ge 0 } , \mathcal { K } _ { \infty }$ functions $\underline { { \beta } } _ { \tilde { V } } , \overline { { \beta } } _ { \tilde { V } } , \underline { { \beta } } _ { \tilde { W } }$ and $\overline { { \beta } } _ { \tilde { W } }$ , continuous functions $H _ { i } : \mathcal { X } \times \mathcal { E } $ $\mathbb { R } _ { \geq 0 } ,$ constants $L _ { i } \in \mathbb { R } , \gamma _ { i } > 0$ for $i \in \{ 0 , 1 \} , \epsilon \in \mathbb { R }$ and $\lambda \in [ 0 , 1 )$ such that

$$\tilde {W} (1, e, - e) \leq \lambda \tilde {W} (0, e, s) \tag {3a}\tilde {W} (0, s + e, - s - e) \leq \tilde {W} (1, e, s) \tag {3b}$$

and $\beta _ { _ { \tilde { W } } } ( \| e , s \| ) \ \leq \ \tilde { W } ( \ell , e , s ) \ \leq \ \overline { { { \beta } } } _ { \tilde { W } } ( \| e , s \| )$ hold for all $e \in \mathcal { E } , \overset { \cdot \cdot } { s } \in \mathcal { E }$ and $\ell \in \{ 0 , 1 \}$ ,

$$\left\langle \frac {\partial \tilde {W} (\ell , e , s)}{\partial e}, g (x, e) \right\rangle \leq L _ {\ell} \tilde {W} (\ell , e, s) + H _ {\ell} (x, e) \tag {4}$$

holds for all $x \in \mathcal { X } , s \in \mathcal { E } , \ell \in \{ 0 , 1 \}$ and almost all $e \in { \mathcal { E } }$ ,

$$\left\langle \nabla \tilde {V} (x), f (x, e) \right\rangle \leq - \epsilon \tilde {V} (x) - H _ {\ell} ^ {2} (x, e) + \gamma_ {\ell} ^ {2} \tilde {W} (\ell , e, s) \tag {5}$$

holds for all $e \in \mathcal { E } , s \in \mathcal { E } , \ell \in \{ 0 , 1 \}$ and almost all $x \in \mathcal { X }$ and $\underline { { \beta } } _ { \tilde { V } } ( \| x \| ) \leq \tilde { V } ( x ) \leq \overline { { \beta } } _ { \tilde { V } } ( \| x \| )$ holds for all $x \in \mathcal { X }$ .
