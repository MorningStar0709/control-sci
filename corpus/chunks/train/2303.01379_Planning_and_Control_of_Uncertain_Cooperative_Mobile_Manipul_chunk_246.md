Let a desired position trajectory for the object center of mass be $p _ { \mathrm { d } } :$ $\mathbb { R } _ { \geq 0 }  \mathbb { R } ^ { 3 }$ , and $e _ { p } : = p _ { O } - p _ { \mathrm { d } }$ . Let also a desired object orientation be expressed in terms of a desired rotation matrix $R _ { \mathrm { d } } : \mathbb { R } _ { \geq 0 }  \mathbb { S } \mathbb { O } ( 3 )$ , with $\dot { R } _ { \mathrm { d } } = S ( \omega _ { \mathrm { d } } ) R _ { \mathrm { d } }$ , where $\omega _ { \mathrm { d } } : \mathbb { R } _ { \ge 0 }  \mathbb { R } ^ { 3 }$ is the desired angular velocity. Then an orientation error metric that was also used in the previous formationcontrol section is

$$e _ {o} := \frac {1}{2} \mathrm{tr} \left(I _ {3} - R _ {\mathrm{d}} ^ {\top} R _ {o}\right) \in [ 0, 2 ], \tag {3.60}$$

which, after differentiation and by using (3.34a) becomes (see also (3.6b))

$$\dot {e} _ {o} = \frac {1}{2} e _ {R} ^ {\top} R _ {o} ^ {\top} \left(\omega_ {o} - \omega_ {\mathrm{d}}\right), \tag {3.61}$$

where $e _ { R } : = S ^ { - 1 } \left( R _ { \mathrm { d } } ^ { \top } R _ { o } - R _ { o } ^ { \top } R _ { \mathrm { d } } \right) \in \mathbb { R } ^ { 3 }$ . It holds that

$$
e _ {R} = 0 \Leftrightarrow \left\{ \begin{array}{l} e _ {o} = 0 \Leftrightarrow \operatorname{tr} (R _ {\mathrm{d}} ^ {\top} R _ {o}) = 3 \Leftrightarrow R _ {o} = R _ {\mathrm{d}} \\ e _ {o} = 2 \Leftrightarrow \operatorname{tr} (R _ {\mathrm{d}} ^ {\top} R _ {o}) = - 1 \Leftrightarrow R _ {o} \neq R _ {\mathrm{d}} \end{array} \right..
$$

The second case represents an undesired equilibrium, where the desired and the actual orientation differ by 180 degrees. This issue is caused by topological obstructions on SO(3) and it has been proven that no continuous controller can achieve global stabilization [114]. The following control design guarantees that $e _ { O } ( t ) < 2 , \forall t \in \mathbb { R } _ { \geq 0 }$ , from all initial conditions satisfying $e _ { o } ( 0 ) < 2$ .

The next corollary shows that a standard inverse-dynamics control protocol guarantees convergence of $p ( t ) - p _ { \mathrm { d } } ( t ) , e _ { o } ( t )$ to zero while avoiding internal forces, provided that the right inverse $\dot { G } ^ { \ast } = M G ^ { \top } ( G M G ^ { \top } ) ^ { - 1 }$ is used.
