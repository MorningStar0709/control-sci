where the parameters ${ \bf x } _ { h } \doteq ( x _ { e } , x _ { s } , x _ { c } )$ and ${ \bf u } _ { h } \doteq ( u _ { e } , u _ { s } , u _ { c } )$ are decision variables of the HMPC’s optimization problem, which, at each sample time t and for a given choice of the prediction horizon $\bar { N }$ and a reference parameterized by ${ \bf x } _ { r } ( t )$ and ${ \bf u } _ { r } ( t )$ , is given by

$$
\min_ {\substack {\mathbf {x}, \mathbf {u}, \\ \mathbf {x} _ {h}, \mathbf {u} _ {h}}} V _ {h} (\mathbf {x} _ {h}, \mathbf {u} _ {h}; \mathbf {x} _ {r} (t), \mathbf {u} _ {r} (t)) + \sum_ {k = 0} ^ {N - 1} \ell_ {h} (x ^ {k}, u ^ {k}, x _ {h} ^ {k}, u _ {h} ^ {k}) \tag{9a}
\text { s.t. } x ^ {0} = x (t) \tag {9b}x ^ {k + 1} = A x ^ {k} + B u ^ {k}, k \in \mathbb {Z} _ {0} ^ {N - 1} \tag {9c}\underline {{{y}}} \leq E x ^ {k} + F u ^ {k} \leq \overline {{{y}}}, k \in \mathbb {Z} _ {0} ^ {N - 1} \tag {9d}x ^ {N} = x _ {e} + x _ {s} \sin (w N) + x _ {c} \cos (w N) \tag {9e}\left(\mathbf {x} _ {h}, \mathbf {u} _ {h}\right) \in \mathcal {D} \tag {9f}\left(\mathbf {x} _ {h}, \mathbf {u} _ {h}\right) \in \mathcal {C} _ {\sigma}, \tag {9g}$$

where $\mathbf { x } = ( x ^ { 0 } , \ldots , x ^ { N - 1 } ) , \mathbf { u } = ( u ^ { 0 } , \ldots , u ^ { N - 1 } )$ , the two terms of the cost function are given by the stage cost function

$$\ell_ {h} (x, u, x _ {h}, u _ {h}) = \| x - x _ {h} \| _ {Q} ^ {2} + \| u - u _ {h} \| _ {R} ^ {2},$$

with $Q \in \mathbb { S } _ { + } ^ { n _ { x } } , R \in \mathbb { S } _ { + } ^ { n _ { u } }$ , and the offset cost function

$$V _ {h} (\cdot) = \| x _ {e} - x _ {r e} \| _ {T _ {e}} ^ {2} + \| x _ {s} - x _ {r s} (t) \| _ {T _ {h}} ^ {2} + \| x _ {c} - x _ {r c} (t) \| _ {T _ {h}} ^ {2}+ \| u _ {e} - u _ {r e} \| _ {S _ {e}} ^ {2} + \| u _ {s} - u _ {r s} (t) \| _ {S _ {h}} ^ {2} + \| u _ {c} - u _ {r c} (t) \| _ {S _ {h}} ^ {2},$$

with $T _ { e } \in \mathbb { S } _ { + } ^ { n _ { x } } , T _ { h } \in \mathbb { D } _ { + } ^ { n _ { x } } , S _ { e } \in \mathbb { S } _ { + } ^ { n _ { u } }$ , and $S _ { h } \in \mathbb { D } _ { + } ^ { n _ { u } }$ ; and $\sigma > 0$ is taken as an arbitrarily small scalar to avoid a possible controllability loss in the presence of active constraints at an equilibrium point [7].
