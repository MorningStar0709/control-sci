# 3 Connection with stochastic optimal transport

In this section, we give a connection between (1.4) and stochastic optimal transport such as the Schr¨odinger problem.

We shall impose the following condition on D:

(B1) The class D is represented as

$$\mathcal {D} = (f _ {0}, g _ {0}) + \bigcup_ {\lambda > 0} \lambda \tilde {\mathcal {D}}$$

where $f _ { 0 } \in C ( [ 0 , T ] \times \mathbb { R } ^ { d } \times U )$ with $\begin{array} { r } { \operatorname* { i n f } _ { ( t , x , u ) } f _ { 0 } ( t , x , u ) > - \infty , g _ { 0 } \in C _ { b } ( \mathbb R ^ { d } ) } \end{array}$ , and the class $\{ \phi _ { f , g }$ : $( f , g ) \in \tilde { \mathcal { D } } \}$ is non-empty, convex, symmetric at origin, and compact in $C _ { b } ( [ 0 , T ] \times \mathbb { R } ^ { d } \times U , \mathbb { R } ^ { 2 } )$ .

Example 3.1. A class of neural networks satisfies (B1). More precisely, let $f _ { \theta } ( t , x , u )$ and $g _ { \boldsymbol { \theta } } ( x )$ be neural networks with a fixed architecture, parameterized by weights $\theta \in \Theta$ , where $\Theta \subset \mathbb { R } ^ { m }$ is a compact, convex, and symmetric set. If the map $\theta \mapsto \phi _ { f _ { \theta } , g _ { \theta } }$ is affine (as in the case of a single hidden layer with linear output), then the class $\big \{ \phi _ { f _ { \theta } , g _ { \theta } } : \theta \in \Theta \big \}$ is a compact, convex, and symmetric subset of $C _ { b } ( [ 0 , T ] \times \mathbb { R } ^ { d } \times U , \mathbb { R } ^ { 2 } )$ q, so (B1) is satisfied with $\tilde { \mathcal { D } } = \{ ( f _ { \theta } , g _ { \theta } ) : \theta \in \Theta \}$ .

For any $\mathcal { P } ( \mathbb { R } ^ { d } \times U )$ -valued processes $\pi = \{ \pi _ { t } \} _ { 0 \leqslant t \leqslant T }$ and $\nu = \{ \nu _ { t } \} _ { 0 \leqslant t \leqslant T }$ , put

$$\rho (\pi , \nu) = \sup _ {(f, g) \in \tilde {\mathcal {D}}} \bigg \{\int_ {0} ^ {T} \langle f (t, \cdot , \cdot), \pi_ {t} - \nu_ {t} \rangle d t + \langle g, \tilde {\pi} _ {T} - \tilde {\nu} _ {T} \rangle \bigg \}.$$

Here, $\tilde { \pi } _ { t } ( d x ) = \pi _ { t } ( d x \times U )$ and ${ \tilde { \nu } } _ { t }$ is similarly defined.
