$$
\mathcal {L} _ {\mathbb {S E} (3)} ^ {\circledcirc} (\mathcal {G}) = \text { span } \bigg \{\left[ \begin{array}{c} (I _ {3} \otimes S (\mathbf {e} _ {h})) p _ {\mathcal {G}} \\ \mathbb {1} _ {n} \otimes \mathbf {e} _ {h} \end{array} \right], h = 1, 2, 3 \bigg \}. \tag {3.45}
$$

It is also known that $S _ { b } \subseteq \mathrm { n u l l } ( \mathcal { R } _ { \mathcal { G } , b } )$ . Thus $\boldsymbol { S _ { b } }$ describes the trivial motions of an SE(3) bearing framework [163].

The above discussion immediately leads to the following proposition.

Proposition 3.1. The trivial motions of a D&B framework are characterized by the set

$$
\mathcal {S} _ {d b} := \bar {\mathcal {S}} _ {d} \cap \mathcal {S} _ {b} = \operatorname{span} \left\{\left[ \begin{array}{c} \mathbb {1} _ {N} \otimes I _ {3} \\ 0 _ {3 N \times 3} \end{array} \right], \mathcal {L} _ {\mathbb {S E} (3)} ^ {\circledcirc} (\mathcal {G}) \right\}.
$$

Furthermore, it follows that ${ \cal S } _ { d b } \subseteq \mathrm { n u l l } ( \widetilde { \mathcal { R } } _ { \mathcal { G } } )$ .

Having characterized the trivial motions, it now follows from Definition 3.2 that for infinitesimal rigidity, we require that null $( \mathcal { \tilde { R } } _ { \mathcal { G } } ) = \mathcal { S } _ { d b }$ . This is summarized in the following proposition.

Proposition 3.2. The framework $( \mathcal { G } , p _ { \mathcal { G } } , R _ { \mathcal { G } } )$ is D&B infinitesimally rigid in SE(3) if and only if

$$
\begin{array}{l} \operatorname{null} \left(\widetilde {\mathcal {R}} _ {\mathcal {G}}\right) = \operatorname{null} \left(\bar {\mathcal {R}} _ {\mathcal {G}, d}\right) \cap \operatorname{null} \left(\mathcal {R} _ {\mathcal {G}, b}\right) \\ = \text { span } \left\{\left[ \begin{array}{c} \mathbb {1} _ {N} \otimes I _ {3} \\ 0 _ {3 N \times 3}, \end{array} \right], \mathcal {L} _ {\mathbb {S E} (3)} ^ {\circledcirc} (\mathcal {G}) \right\} = \mathcal {S} _ {d b}. \\ \end{array}
$$

Equivalently, the D&B framework is infinitesimally rigid in SE(3) if and only if

$$\operatorname{rank} \left(\widetilde {\mathcal {R}} _ {\mathcal {G}}\right) = \dim \left(\widetilde {\mathcal {R}} _ {\mathcal {G}}\right) - \dim (\operatorname{null} \left(\widetilde {\mathcal {R}} _ {\mathcal {G}}\right)) = 6 N - 6.$$
