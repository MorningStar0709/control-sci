# 5.2 Decision Unbounded Continuous Control Environments

We evaluate TLA on a suite of 8 continuous control environments using the OpenAI gym library (Brockman et al., 2016): two classic control problems and six MuJoCo environments (Todorov, Erez, & Tassa, 2012). We selected tasks on which TD3 has demonstrated state-of-the-art performance to highlight the inefficiency of the policies learned by current state-of-the-art algorithms. We set the time step of the fast-network to be equal to the default step size of the environment so that TLA has the same step size as TD3 and TempoRL. For the TempoRL algorithm, the max skip length J was set to be equal to τ , so the longest action repetition possible is the same for both TLA and TempoRL. Additionally, for reference, we also present the TD3-EA results. However, continuous control tasks are especially difficult to learn using extended actions, and we find that TLA utilizes both the fast and the slow layer for almost every task. Therefore, for a fair comparison, we set the time step of TD3-EA to be approximately equal to the average decisions-step size of TLA after training.

The algorithms’ hyperparameters and neural network sizes were kept the same as in previous work (Fujimoto et al., 2018). The maximum training steps were set to 30,000
