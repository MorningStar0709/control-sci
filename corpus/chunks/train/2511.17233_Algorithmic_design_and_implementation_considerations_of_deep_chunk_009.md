As discussed in [3, §7], constraints of the system (1) are satisfied by the tightening the constraints of the nominal dynamics $\overrightharpoon { \bar { f } }$ used in the design of the reference governor and then forcing the states and control of the online MPC in a tube. The online MPC has constraints only on control, so the satisfaction of state constraints solely depend on the constraint tightening. Theoretical results guarantee that the states of the controlled system lie in a tube defined by level sets, which is defined in terms of the value function of the online MPC [2, Proposition 1].

Constraint tightening plays a very important role in tubebased MPC formulation and the intuition is that when the nominal dynamics satisfy the tightened constraints, the uncertain dynamics will satisfy the constraints given in (1). Since deep MPC has to distribute control authority between MPC component and learning component, we define $\bar { \mathcal { U } } : =$ $\{ u \in \mathbb { R } ^ { m } \mid \| u \| _ { \infty } \leqslant u _ { \mathrm { m a x } } - u _ { \mathrm { m a x } } ^ { a } \}$ and compute the tightened control set $\mathcal { U } _ { r } \subset \bar { \mathcal { U } }$ according to [3, §7].

Once we have tightened sets $\chi _ { r } \subset \chi$ and $\mathcal { U } _ { r } \subset \bar { \mathcal { U } }$ , we can solve a finite horizon problem by using the nominal dynamics $\bar { f } ( x _ { t } , u _ { t } ^ { m } ) = f ( x _ { t } ) + g ( x _ { t } ) u _ { t } ^ { m }$ as follows:

$$
\begin{array}{l} \min _ {(u _ {i} ^ {r}) _ {i = 0} ^ {N - 1}} \quad \sum_ {i = 0} ^ {N - 1} \left\| x _ {i} ^ {r} \right\| _ {Q} ^ {2} + \left\| u _ {i} ^ {r} \right\| _ {R} ^ {2} \\ \text { s.   t. } \quad x _ {0} ^ {r} = x _ {0}, x _ {N} ^ {r} = \mathbf {0}, \tag {2} \\ x _ {i + 1} ^ {r} = \bar {f} (x _ {i} ^ {r}, u _ {i} ^ {r}) \in \mathcal {X} _ {r} \subset \mathcal {X}, \\ x _ {i} ^ {r} \in \mathcal {U} _ {r} \subset \bar {\mathcal {U}}; i = 0, \dots , N - 1, \\ \end{array}
$$

to generate a reference trajectory $( x _ { t } ^ { r } , u _ { t } ^ { r } ) _ { t = 0 } ^ { N } ,$ where ?? is the optimization horizon. The design of reference governor is popular for designing hierarchical MPC to satisfy constraints [19]. We can use CasADi [20] to design a reference governor by using the algorithm 2.

Algorithm 2 Reference governor   
1: Input: $N, Q, R, x_0, x^s, u^s, \mathcal{X}_r, \mathcal{U}_r, \bar{f}$ 2: Output: $(x_t^r, u_t^r)_{t=0}^N$ 3: $x_0 = x_0 - x^s$ 4: Solve (2)  
5: $x_t^r \leftarrow x_t^r + x^s, u_t^r \leftarrow u_t^r + u^s$ 6: return $(x_t^r, u_t^r)_{t=0}^N$
