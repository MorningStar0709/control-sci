# 3.2 The linear subsystem $S _ { L }$

Since $S _ { L }$ is a time-invariant linear system, defined by the state space equations (3) and (13), it is natural to consider the quadratic Lyapunov function candidate (14) whose time derivative with respect to an arbitrary trajectory is given by

$$
\begin{array}{l} \dot {v} (\zeta) = 2 \zeta^ {T} S (A \zeta + E \sigma) \\ = \zeta^ {T} (A ^ {T} S + S A) \zeta + 2 \zeta^ {T} S E G _ {0} x, \tag {16} \\ \end{array}
$$

where we have used (11). The next lemma is an instrumental result to be used in the sequel.

Lemma 1 If the matrix inequality

$$
\left[ \begin{array}{c c} A ^ {T} S + S A + \rho S & \bullet \\ \alpha^ {- 1} G _ {0} ^ {T} E ^ {T} S & - \rho P \end{array} \right] <   0 \tag {17}
$$

is satisfied for scalars $\alpha > 0 , \rho > 0$ , and matrices $S > 0$ $P > 0$ of appropriate dimensions, then the inequality

$$\dot {v} < \rho (V - v) \tag {18}$$

holds, for all ${ \mathbf { \xi } } ) \neq \zeta \in \mathbb { R } ^ { r }$ and 0 = $x \in \mathbb { R } ^ { 2 n }$ .

Proof: The Schur Complement with respect to the second row and column of (17) together with $0 < c ( \sigma ) ^ { - 2 } <$ $\alpha ^ { - 2 }$ shows that

$$
\left[ \begin{array}{c c} A ^ {T} S + S A + \rho S & \bullet \\ c (\sigma) ^ {- 1} G _ {0} ^ {T} E ^ {T} S & - \rho P \end{array} \right] <   0 \tag {19}
$$

holds as well. Multiplying it to the left by $[ \zeta ^ { T } x ^ { T } R ( \sigma ) ]$ and to the right by its transpose, taking into account that $c ( \sigma ) ^ { - 1 } { \tilde { G _ { 0 } } } R ( \sigma ) x = G _ { 0 } x _ { : }$ , (19) yields

$$
\begin{array}{l} \dot {v} = \zeta^ {T} (A ^ {T} S + S A) \zeta + 2 \zeta^ {T} S E G _ {0} x \\ = \zeta^ {T} (A ^ {T} S + S A) \zeta + 2 c (\sigma) ^ {- 1} \zeta^ {T} S E G _ {0} R (\sigma) x \\ <   - \rho \zeta^ {T} S \zeta + \rho x ^ {T} R (\sigma) P R (\sigma) x \tag {20} \\ \end{array}
$$

which reproduces (18). The proof is concluded.

It is interesting to observe that a necessary condition for the feasibility of the inequality (17) is matrix A to be Hurwitz stable. This means that the isolated (with $\sigma =$ 0) subsystem $ { \boldsymbol { S } } _ { L }$ must be asymptotically stable. Or more generally, in the presence of parameter uncertainty, it must be robustly stable. These conditions are ensured by

Assumption (A3). In a practical setting, such conditions can be achieved through a proper sliding surface design, as illustrated in Section 7. Finally, it is also important to point out that (18) imposes that $\dot { v } ( \zeta ) < 0$ only in a subset and not for all $( \zeta , \overset { \cdot } { x } ) \in \mathbb { R } ^ { r } \times \mathbb { R } ^ { 2 \overset { \cdot } { n } }$ .
