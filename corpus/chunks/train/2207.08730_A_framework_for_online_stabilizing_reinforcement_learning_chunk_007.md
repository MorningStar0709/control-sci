# III. BACKGROUND

Consider an agent-environment loop that assumes a state $X _ { t } \in \mathbb { X } \subset \mathbb { R } ^ { n }$ at a time $t \in \mathcal { T } = \mathbb { R } _ { + }$ , whose dynamics are modeled by the following transition law:

$$\mathrm{d} X _ {t} = f \left(X _ {t}, U _ {t}\right) \mathrm{d} t + \sigma \left(X _ {t}, U _ {t}\right) \mathrm{d} Z _ {t}, \tag {1}\mathbb {P} \left[ X _ {0} = \text { const } \right] = 1$$

where an unknown noise $Z _ { t } \in \mathbb { R } ^ { d }$ is modelled by an arbitrary Lipschitz-continuous random process with a fixed Lipschitz constant $\mathsf { L i p } _ { Z } ,$ , and $\sigma : \mathbb { X } \times \mathbb { U }  \mathbb { R } ^ { n \times d }$ is a bounded mixing matrix, i. e., $\| \sigma ( x , u ) \| _ { \mathrm { o p } } \le \sigma _ { \operatorname* { m a x } } , \sigma _ { \operatorname* { m a x } } > 0$ . Here and after, when claims are made about sections of random processes, the respective conditions are meant to hold almost surely. We assume that $f$ and σ are locally Lipschitz continuous with respect to $X _ { t } .$ . Particular models for such a bounded noise as in the above description are given in Section F of the appendix.

Let a policy $\rho : \mathbb { X } \times \mathcal { T }  \mathbb { U }$ determine the agent’s action $U _ { t } \in \mathbb { U }$ at time t in the following way:

$$U _ {t} = \rho (X _ {t - t \bmod \delta}), \tag {2}$$

where δ is the digital controller sampling time of the present (such a setting is also known as “sample-and-hold” or S&H). Under such a digital control, at best only practical stability in the sense of the following definition can be achieved.

Definition 1: A policy $\rho [ X _ { 0 } ] ( \cdot )$ is said to practically semiglobally stabilize (1) from $X _ { 0 }$ until $s ^ { * }$ if, given $s ^ { * } < r < \infty ,$ , there exists a $\bar { \delta } > 0 \ \mathrm { s . t }$ . under (2) a trajectory $x ( t )$ with a sampling time $\delta \leq \bar { \delta }$ , starting at $X _ { 0 }$ is bounded, enters $B _ { r }$ after a time $t ^ { \prime } ,$ which depends uniformly on $\| X _ { 0 } \| , r$ , and remains there for all $t \geq t ^ { \prime }$ .

In the following, for brevity, the wording “semi-globally” is omitted.
