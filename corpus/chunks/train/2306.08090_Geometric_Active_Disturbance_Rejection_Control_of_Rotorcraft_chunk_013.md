$$\lambda_ {\min} \left\{\mathcal {Q D} (e _ {1}) \right\} \geq \lambda_ {\min} \left\{\mathcal {Q} \right\} \lambda_ {\min} \left\{\mathcal {D} (e _ {1}) \right\} > 0. \tag {28}$$

With $\lambda _ { \operatorname* { m i n } } \left\{ \mathcal { D } ( e _ { 1 } ) \right\} = \lambda _ { \operatorname* { m i n } } \left\{ \phi _ { 1 } ^ { \prime } ( e _ { 1 } ) \right\}$ , substituting (17) and (22) into (29), we obtain,
