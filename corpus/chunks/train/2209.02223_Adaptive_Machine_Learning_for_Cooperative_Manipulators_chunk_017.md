\boldsymbol {u} := \left\{ \begin{array}{l} \boldsymbol {u} _ {1} = \hat {\boldsymbol {Q}} ^ {- 1} \left[ \hat {\boldsymbol {M}} \left(\ddot {\boldsymbol {x}} _ {d} - \boldsymbol {G} _ {d} \dot {\boldsymbol {e}} - \boldsymbol {G} _ {p} \boldsymbol {e}\right) + \hat {\boldsymbol {h}} \right] \\ \boldsymbol {u} _ {2} = \hat {\boldsymbol {T}} \boldsymbol {u} _ {1}, \end{array} \right. \tag {36}
$$

where $\mathbfit { \Delta } \mathbfit { x } _ { d }$ is the desired position trajectory, $e = \pmb { x } - \pmb { x } _ { d }$ is the position error, and $G _ { p } > 0$ and $G _ { d } > 0$ are the feedback gains. Notice that despite the control law (36) involves the matrix inversion computed at the estimated parameters, the matrix is always invertible. In other words, the adaptive controller is singularity-free.
