# Algorithm 1 Sparse soft observer

Input: $\tau \leq n , \lambda > 0 , \nu > 0 , A , \mathcal { O } , \mathbf { y } ( k )$

Output: $\binom { \hat { x } ( k ) } { \hat { \mathbf { a } } ( k ) }$ $\binom { x ( k - \tau + 1 ) } { \mathbf { a } ( k ) }$

1: for all $\dot { k } = \stackrel { \prime } { \tau } - 1 , \tau , \ldots$ do

2: Measurements and estimated measurements update

$$\mathbf {y} (k) = \mathcal {O} x (k - \tau + 1) + \mathbf {a} (k) \tag {21}\hat {\mathbf {y}} (k) = \mathcal {O} \hat {x} (k) + \hat {\mathbf {a}} (k)$$

3: ISTA step: gradient step + soft thresholding

$$\binom {\hat {x} ^ {+}} {\hat {\mathbf {a}} ^ {+}} = \binom {\hat {x} (k)} {\hat {\mathbf {a}} (k)} - \nu (\mathcal {O} I) ^ {\top} [ \hat {\mathbf {y}} (k) - \mathbf {y} (k) ] \tag {22}\hat {\mathbf {a}} (k + 1) = \mathrm{S} _ {\nu \lambda} \left[ \hat {\mathbf {a}} ^ {+} \right] \tag {23}$$

4: State update

$$\hat {x} (k + 1) = A \hat {x} ^ {+} \tag {24}$$

5: end for
