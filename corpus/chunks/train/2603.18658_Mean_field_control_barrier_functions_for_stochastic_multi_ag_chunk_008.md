\begin{array}{l} \int_ {\Omega} e \nabla \cdot (e \mathbf {w} (\mathbf {u} ^ {*})) d \mathbf {x} = - \int_ {\Omega} \nabla e \cdot e \mathbf {w} (\mathbf {u} ^ {*}) d \mathbf {x} \\ = - \frac {1}{2} \int_ {\Omega} \nabla (e ^ {2}) \cdot \mathbf {w} (\mathbf {u} ^ {*}) d \mathbf {x} \\ = \frac {1}{2} \int_ {\Omega} e ^ {2} \nabla \cdot \mathbf {w} \left(\mathbf {u} ^ {*}\right) d \mathbf {x} \tag {19b} \\ \leq \frac {1}{2} \left\| e ^ {2} \nabla \cdot \mathbf {w} (\mathbf {u} ^ {*}) \right\| _ {1} \\ \leq \left\| \nabla \cdot \mathbf {w} \left(\mathbf {u} ^ {*}\right) \right\| _ {\infty} V, \\ \end{array}

where we use Holder’s inequality [20] and integration by ¨ parts in higher dimensions (boundary terms vanish due to boundary conditions), and the Poincare-Wirtinger inequal- \` ity [21].

Substituting (19) into (18) yields

$$\frac {\mathrm{d}}{\mathrm{d} t} V \leq - a V + b \sqrt {V}, \tag {20}$$

where $a = 2 D - \| \nabla \cdot \mathbf { w } ( \mathbf { u } ^ { * } ) \| _ { \infty }$ (which is positive under the theorem hypothesis) and $b = \sqrt { 2 } ( \| \nabla \cdot \left( \bar { \rho } \mathbf { w } ( \mathbf { u } ^ { * } ) \right) \| _ { 2 } + \| \Delta \bar { \rho } \| _ { 2 } )$ . The scalar dynamical system bounding ${ \frac { \mathrm { d } } { \mathrm { d } t } } V$ has a globally stable equilibrium point at $( b / a ) ^ { 2 }$ , proving the claim by the comparison principle. Note that in the absence of dangerous zones, this equilibrium converges to 0. □
