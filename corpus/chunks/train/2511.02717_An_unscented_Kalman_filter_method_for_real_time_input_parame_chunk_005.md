For the mathematical implementation of the standard UKF, consider the nonlinear process equation in the continuous time-domain and in a state space format:

$$\dot {\mathbf {z}} (t) = f (\mathbf {z} (t), \mathbf {u} (t)) + \boldsymbol {\nu} (t) \tag {1}$$

and the nonlinear observation equation:

$$\mathbf {y} (t) = h (\mathbf {z} (t), \mathbf {u} (t)) + \boldsymbol {\eta} (t) \tag {2}$$

The state vector ${ \bf z } ( t ) = [ { \bf x } ( t ) , \dot { \bf x } ( t ) , \pmb \theta ] ^ { T }$ includes the parameters of the system θ apart from the dynamic states, while ${ \bf y } ( t )$ is the observation vector. Also, $f ( \bullet )$ and $h ( \bullet )$ are the state transition function and the observation function, respectively, which take into account the input vector ${ \bf \delta u } ( t )$ . Lastly, $\pmb { \nu } ( t )$ and $\pmb { \eta } ( t )$ are the process and the measurement noise, of covariance matrices $\mathbf Q ( t )$ and ${ \bf R } ( t )$ , respectively. Eqs. 1 and 2 can be discretized as:

$$\mathbf {z} _ {\mathbf {k}} = F (\mathbf {z} _ {\mathbf {k - 1}}, \mathbf {u} _ {\mathbf {k - 1}}) + \boldsymbol {\nu} _ {\mathbf {k - 1}} \tag {3}$$

and,

$$\mathbf {y} _ {\mathbf {k}} = h \left(\mathbf {z} _ {\mathbf {k}}, \mathbf {u} _ {\mathbf {k}}\right) + \boldsymbol {\eta} _ {\mathbf {k}} \tag {4}$$

where,

$$F (\mathbf {z} _ {\mathbf {k} - \mathbf {1}}, \mathbf {u} _ {\mathbf {k} - \mathbf {1}}) = \mathbf {z} _ {\mathbf {k} - \mathbf {1}} + \int_ {(k - 1) \Delta t} ^ {(k) \Delta t} f (\mathbf {z} (t), \mathbf {u} (t)) d t \tag {5}$$

Note that k refers to $k \Delta t$ time instant, where $\Delta t$ is the sampling period. The discretized process and observation covariance matrices are:

$$\mathbf {Q} _ {\mathbf {k} - 1} \approx \frac {\mathbf {Q} ((k - 1) \Delta t)}{\Delta t}, \quad \mathbf {R} _ {\mathbf {k}} = \frac {\mathbf {R} (k \Delta t)}{\Delta t} \tag {6}$$
