# 3 Error Bounds for Multitask System Identification

We are now ready to establish error bounds for the clustered multitask identification procedure described in Algorithm 2. This section characterizes how collaboration, heterogeneity, and adversarial behavior influence the estimation error of the local system models. Complete proofs and constant definitions are provided in Appendix E. For clarity of presentation, we introduce the following key quantities: $\begin{array} { r } { r ^ { \vee } \triangleq \operatorname* { m a x } _ { t \in [ T ] , \ell \in \mathcal C _ { j } } \operatorname { t r } \left( \Sigma _ { z _ { t } } ^ { ( \ell ) } \right) / \Vert \Sigma _ { z _ { t } } ^ { ( \ell ) } \Vert , P _ { \star } ^ { \wedge } \triangleq \operatorname* { m i n } _ { i \in [ M ] } \Vert P _ { \star } ^ { ( i ) } \Vert , C _ { \tau } = \mathtt { p o l y } ( d _ { x } , d _ { u } ) } \end{array}$ , and the contraction rate $\rho = 1 - \eta$ , for some step-size $\eta \in ( 0 , 1 )$ .

Lemma 3.1 (Intra-cluster homogeneity). Consider a cluster $\mathcal { C } _ { j }$ composed of $M _ { j }$ identical systems. Suppose that Assumption 2 holds and that the initial epoch length is set sufficiently large, such that

$$\tau_ {1} \geq C _ {\tau} \max \left\{\frac {d ^ {\prime} \Delta_ {\mathrm{min}} ^ {2} \sigma_ {w} \log \left(\frac {(2 d _ {x} + d _ {u}) M _ {j}}{\delta}\right)}{d M _ {j}}, r ^ {\vee} + \log \left(\frac {2 M _ {j}}{\delta}\right), \log (4 T ^ {2}), \frac {\log \frac {1}{P _ {\star} ^ {\wedge}}}{\log \left(1 - \frac {1}{P _ {\star} ^ {\wedge}}\right)} \right\},$$

for a small $\delta \in ( 0 , 1 )$ . Then, after N ≥ log $\left( C _ { \alpha } \Delta _ { m i n } \tau _ { 1 } ^ { 2 } \right) / \log ( 1 / \rho )$ iterations of RCSI, with probability $1 - \delta$ , the model estimate $\widehat { \Theta } ^ { ( i ) }$ at epoch k satisfies:

$$\left\| \widehat {\Theta} _ {k} ^ {(i)} - \Theta_ {\star} ^ {(i)} \right\| _ {F} ^ {2} \leq C _ {\mathrm{stat}} \frac {\sigma_ {w} ^ {2} (d _ {x} ^ {2} + d _ {x} d _ {u}) \log (M _ {j} / \delta)}{\sigma_ {k} ^ {2} M _ {j} \tau_ {k}} + C _ {\mathrm{mis,1}} e ^ {- C _ {\mathrm{mis,2}} \sigma_ {k} ^ {2} \tau_ {k}}.$$
