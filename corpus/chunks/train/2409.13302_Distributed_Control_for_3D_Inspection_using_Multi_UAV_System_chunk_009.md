# A. Cost Function

Consider a multi-UAV system consisting of N quadrotor UAVs. Let $\mathcal { Q } ~ \subset ~ \mathbb { R } ^ { 3 }$ be a convex region in which the UAVs move and the position of the $i ^ { \mathrm { { t h } } }$ UAV is denoted by p . The set of all UAV positions is also defined as $\mathcal { P } = \{ \mathbf { p } _ { 1 } , . . . , \mathbf { p } _ { N } \}$ . The sensing quality at the point $\mathbf { q } \in \mathcal { Q }$ measured by the $i ^ { \mathrm { { t h } } }$ UAV located at $\mathbf { p } _ { i }$ decreases proportionally with distance $\left\| \mathbf { q } - \mathbf { p } _ { i } \right\|$ . This measure can be expressed by an isotropic, strictly increasing, and convex function $f ( \left| \left| \mathbf { q } - \mathbf { p } _ { i } \right| \right| ) : \mathbb { R } _ { + } \to \mathbb { R } _ { + }$ , called the sensing unreliability function, in which the sensing quality deteriorates as large values are reached. The function $\varphi : \mathcal { Q }  \mathbb { R } _ { + }$ is a density function on $\mathcal { Q }$ that allocates weight to each point $\mathbf { q } \in \mathcal { Q }$

$$
\begin{array}{r}\square \partial \mathcal{W}\quad \square \kappa \in \mathcal{K}\bullet q_c\in \mathcal{Q}_c\bullet q_\kappa \in \mathcal{Q}_\kappa \bullet q_T\in \mathcal{Q}_T\bullet \bar{q}_T\in \bar{\mathcal{Q}}_T \end{array}
$$
