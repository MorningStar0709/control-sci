The minimal distance between the Nyquist plot of $H _ { O L } ( z ^ { - 1 } )$ ) and the critical point will define a stability margin. This minimal distance according to (8.7) will depend upon the maximum of the modulus of the output sensitivity function. This stability margin which we will call subsequently the modulus margin could be linked to the uncertainties upon the plant model.

The following indicators serve for characterizing the distance between the Nyquist plot of $H _ { O L } ( z ^ { - 1 } )$ and the critical point [−1, j 0] (see Fig. 8.4):

• modulus margin (ΔM)   
• delay margin (Δτ )   
• phase margin (Δφ)   
• gain margin (ΔG)
