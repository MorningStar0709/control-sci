Finally, combining (A.6), (A.7) with Lemma 2 gives property (ii) directly.

To prove property (iii), from (10) and the Sherman– Morrison formula (Hager, 1989), we have

$$P _ {t + 1} ^ {- 1} = P _ {t} ^ {- 1} + \frac {d _ {t} ^ {2}}{\mu_ {t}} \phi_ {t} \phi_ {t} ^ {\top} = P _ {0} ^ {- 1} + \sum_ {\tau = 0} ^ {t} \frac {d _ {\tau} ^ {2}}{\mu_ {\tau}} \phi_ {\tau} \phi_ {\tau} ^ {\top}. \tag {A.8}$$

Using (A.6) and the fact that $\| P _ { t } \| ~ \le ~ \| P _ { 0 } \|$ for all $\textit { t } \geq \textit { 0 }$ , there exists a real-valued random variable $C _ { 2 } > 0$ such that $\begin{array} { r } { | \mu _ { t } | = | ( \log r _ { t } ) ^ { 1 + \delta } + \frac { 1 } { 2 } d _ { t } \overline { { g } } _ { t } ^ { 2 } \phi _ { t } ^ { \top } P _ { t } \phi _ { t } | \leq } \end{array}$ $C _ { 2 } [ ( \log r _ { t } ) ^ { 1 + \delta } ( 1 + \| \phi _ { t } \| ^ { 2 } ) ]$ ,a.s. Substituting this bound into (A.8), we have

$$\lambda_ {\min} \{P _ {t} ^ {- 1} \} \geq \frac {d ^ {2}}{C _ {2} (\log r _ {t}) ^ {1 + \delta}} \lambda_ {\min} \left\{\sum_ {\tau = 0} ^ {t - 1} \frac {\phi_ {\tau} \phi_ {\tau} ^ {\top}}{1 + \| \phi_ {\tau} \| ^ {2}} \right\}, \text { a.s. } \tag {A.9}$$

Moreover, we know that

$$\| \theta^ {*} - \hat {\theta} _ {t} \| ^ {2} \leq \frac {\mathrm{tr} [ (\theta^ {*} - \hat {\theta} _ {t}) ^ {\top} P _ {t} ^ {- 1} (\theta^ {*} - \hat {\theta} _ {t}) ]}{\lambda_ {\min} \{P _ {t} ^ {- 1} \}}.$$

Finally, by combining this inequality with Lemma 2, (A.9), and (A.6), we conclude property (iii) holds. ✷
