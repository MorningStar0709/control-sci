Here, A(t), B(t), C(t) are system matrices with compatible dimensions. We say this system is (internally) positive if for any nonnegative initial condition $x ( 0 ) \geq \mathbf { 0 }$ and $u ( t ) \geq \mathbf { 0 }$ , it holds that $x ( t ) \geq \mathbf { 0 }$ and $y ( t ) \geq \mathbf { 0 }$ for $t \geq 0$ . It is said to be a Metzler system if $A ( t )$ is Metzler and $B ( t ) , C ( t ) \geq \mathbf { 0 }$ for almost all $t \geq 0$ .

Here is a lemma modified from [27].

Lemma 1 If (1) is a Metzler system then it is positive. Conversely, if (1) is positive and $A ( t ) , B ( t )$ , $C ( t )$ are continuous, then (1) is a Metzler system.
