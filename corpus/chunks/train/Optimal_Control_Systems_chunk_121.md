# 2.7.4 Summary of Pontryagin Procedure

Consider a free-final time and free-final state problem with general cost function (Bolza problem), where we want to minimize the performance index

$$J = S \left(\mathbf {x} \left(t _ {f}\right), t _ {f}\right) + \int_ {t _ {0}} ^ {t _ {f}} V (\mathbf {x} (t), \mathbf {u} (t), t) d t \tag {2.7.43}$$

for the plant described by

$$\dot {\mathbf {x}} (t) = \mathbf {f} (\mathbf {x} (t), \mathbf {u} (t), t) \tag {2.7.44}$$

with the boundary conditions as

$$\mathbf {x} (t = t _ {0}) = \mathbf {x} _ {0}; \quad t = t _ {f} \text { is free and } \mathbf {x} (t _ {f}) \text { is free. } \tag {2.7.45}$$
