# 例6.7 计算 $K_{\mathrm{v}}$

作为一个确定稳态误差的例子，一个开环系统的伯德图如图6.13所示。假设有如图6.4所示的单位反馈系统，确定速度误差常数 $K_{v}$ 。

图6.13 由系统 $KG(s) = \frac{10}{s(s + 1)}$ 的伯德图确定 $K_{\mathrm{v}}$

解答。因为低频处的斜率为-1，我们知道系统为1型系统。低频渐近线的延长线在幅值为10处穿过 $\omega = 1\mathrm{rad / s}$ 。因此， $K_{\mathrm{v}} = 10$ ，并且单位反馈系统对单位斜坡输入的稳态误差将为0.1。或者，在 $\omega = 0.01$ 处，我们有 $|A(\omega)| = 1000$ ；因此，根据式(6.23)我们得到

$$K _ {0} = K _ {\mathrm{v}} \approx \omega | A (\omega) | = 0. 0 1 (1 0 0 0) = 1 0$$
