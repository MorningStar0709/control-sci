# 4.3. Morse function on SO(3)

The following Lemma is used for the rotational ESO scheme designed for the vehicle.

Lemma 6. Bohn and Sanyal (2016) Consider attitude kinematics

$$\dot {R} = R \Omega^ {\times}, R \in \mathrm{SO} (3), \Omega \in \mathfrak {s o} (3). \tag {40}$$

Define $K = \mathrm { d i a g } ( [ K _ { 1 } , K _ { 2 } , K _ { 3 } ] )$ , where $K _ { 1 } > K _ { 2 } > K _ { 3 } \geq 1$ . Define

$$s _ {K} (R) = \sum_ {i = 1} ^ {3} K _ {i} (R ^ {\mathrm{T}} \mathbf {e} _ {i}) \times \mathbf {e} _ {i}, \tag {41}$$

such that $\begin{array} { r } { \frac { \mathrm { d } } { \mathrm { d } t } \langle K , I - R \rangle = \Omega ^ { \mathrm { T } } s _ { K } ( R ) } \end{array}$ . Here $\langle A , B \rangle = \operatorname { t r } ( A ^ { \mathrm { T } } B )$ , which makes $\langle K , I - R \rangle$ a Morse function defined on SO(3). Let $S \subset \mathrm { S O ( 3 ) }$ be a closed subset containing the identity in its interior, defined by

$$\mathcal {S} = \left\{R \in \mathrm{SO} (3): R _ {i i} \geq 0 \text { and } R _ {i j} R _ {j i} \leq 0, \forall i, j \in \{1, 2, 3 \}, i \neq j \right\}. \tag {42}$$

Then for $\forall R \in S$ , we have

$$s _ {K} (R) ^ {\mathrm{T}} s _ {K} (R) \geq \langle K, I - R \rangle . \tag {43}$$

Remark 1 (Almost global domain of attraction). Sanyal et al. (2010) We know that the subset of SO(3) where $s _ { K } ( R ) = 0 , R \in \mathrm { S O } ( 3 )$ , which is also the set of critical points for $\langle I - R , K \rangle$ , is

$$C \triangleq \{I, \operatorname{diag} (1, - 1, - 1), \operatorname{diag} (- 1, 1, - 1), \operatorname{diag} (- 1, - 1, 1) \} \subset \mathrm{SO} (3). \tag {44}$$

In addition, the global minimum of this Morse function is $R = I$ .
