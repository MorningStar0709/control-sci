The adaptive feedback law proposed in this paper introduces a historical integral term $\begin{array} { r } { ( \mathbf { e . g . } , \int _ { 0 } ^ { t } | r ( s ) x ( s ) | ^ { p } d s } \end{array}$ in [28]), resulting in a closed-loop system described by a path-dependent stochastic differential equation. Since the drift coefficient depends on the entire historical path of the solution, standard existence and uniqueness theorems under Lipschitz conditions no longer apply. Although deterministic counterparts can be treated with well-established methods, the stochastic pathdependent case remains theoretically underdeveloped. Therefore, as pointed in [28], it is necessary to establish the existence and uniqueness of solutions for this class of pathdependent stochastic systems.
