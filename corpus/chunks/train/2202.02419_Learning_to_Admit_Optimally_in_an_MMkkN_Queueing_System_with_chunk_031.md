We next state the strong law of large numbers for martingale sequences in Theorem 1 and then, using this result, prove Lemma 4.

Theorem 1. (Shiryaev 1996, Corollary 7.3.2) let $\{ M _ { n } \} _ { n = 0 } ^ { \infty }$ be a martingale sequence with $M _ { 0 } =$ 0 and $\mathbb { E } \big [ | M _ { n } | ^ { 2 r } \big ] < \infty$ for some $r \geq 1$ , and it satisfies $\begin{array} { r } { \sum _ { n = 1 } ^ { \infty } n ^ { - ( 1 + r ) } \mathbb { E } \left[ | M _ { n } - M _ { n - 1 } | ^ { 2 r } \right] < \infty } \end{array}$ . Then,

$$\lim _ {n \to \infty} \frac {M _ {n}}{n} = 0. a. s.$$

Lemma 4. The martingale process $\{ Y _ { n } ^ { M } \} _ { n = 0 } ^ { \infty }$ found by Doob’s decomposition of $\{ Y _ { n } \} _ { n = 0 } ^ { \infty }$ satisfies

$$\lim _ {n \to \infty} \frac {Y _ {n} ^ {M}}{n} = 0. a. s.$$
