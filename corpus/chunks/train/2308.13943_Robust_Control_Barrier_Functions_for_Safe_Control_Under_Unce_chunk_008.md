# D. QP-CBF Formulation

In order to combine CBF to guarantee safety, the control problem is formulated as a quadratic program with a CBF as a hard constraint. The $\mathrm { Q P }$ formulation is as follows [1]:

$$u ^ {*} (x) = \underset {u \in U} {\arg \min} \| u - k (x) \| ^ {2} \tag {7}\text { s.t. } \quad L _ {f} h (x) + L _ {g} h (x) u + h _ {x} (x) d + \beta (h (x)) > 0$$

where $k ( x )$ is a nominal control law.
