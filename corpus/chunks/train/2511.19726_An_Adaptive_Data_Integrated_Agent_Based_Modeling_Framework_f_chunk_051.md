# 11.7 SCM Representation

An SCM captures the causal structure:

• $X _ { t } \colon$ exogenous factors (weather, baseline demand);   
• $P _ { t } \mathbf { : }$ : tariffs and congestion thresholds;   
• Θ: consumer attributes $( \theta _ { i } , \eta _ { i } )$ ;   
• $s _ { t } \colon$ node loads and congestion;   
• $Y _ { t } \colon$ performance outcomes (overload, demand, voltage).

Arrows represent relationships such as $( P _ { t } , \Theta , X _ { t } ) \to s _ { t }$ and $s _ { t }  Y _ { t }$ , while adaptive control adds $s _ { t }  P _ { t + 1 }$ . Interventions $\mathrm { d o } ( P _ { t } = p )$ capture counterfactual comparisons between adaptive and static frameworks.
