# 4.1 Properties of bang-bang extremals

In order to determine the structure of Pontryagin bang-bang extremals, it will be useful to study the plane of switching points:

$$\Pi = \{M \in \mathfrak {s l} _ {2} (\mathbb {R}) \mid \operatorname{tr} (M A) = \operatorname{tr} (M B) \}. \tag {4.1}$$

This is the set of matrices in ${ \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ for which the function $\varphi$ is zero.

Since the matrices $A , B , [ A , B ]$ form a bases of ${ \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ by hypothesis, we can write the equation $\operatorname { t r } ( M A ) = \operatorname { t r } ( M B )$ a s

$$\alpha \mathrm{tr} (A ^ {2}) + \beta \mathrm{tr} (A B) + \gamma \mathrm{tr} ([ A, B ] A) = \alpha \mathrm{tr} (A B) + \beta \mathrm{tr} (B ^ {2}) + \gamma \mathrm{tr} ([ A, B ] B),$$

that is

$$\beta = \frac {\operatorname{tr} (A B) - \operatorname{tr} (A ^ {2})}{\operatorname{tr} (A B) - \operatorname{tr} (B ^ {2})} \alpha . \tag {4.2}$$

So, if we impose $\mathrm { t r } ( M ^ { 2 } ) = 1$ , that is if we intersect the hyperboloid H with the plane Π we obtain the equation of a conic

$$\frac {\big (2 \mathrm{tr} (A B) - \mathrm{tr} (A ^ {2}) - \mathrm{tr} (B ^ {2}) \big) \mathrm{tr} \big ([ A , B ] ^ {2} \big)}{\big (\mathrm{tr} (A B) - \mathrm{tr} (B ^ {2}) \big) ^ {2}} \alpha^ {2} + \mathrm{tr} \big ([ A, B ] ^ {2} \big) \gamma^ {2} = 1. \tag {4.3}$$
