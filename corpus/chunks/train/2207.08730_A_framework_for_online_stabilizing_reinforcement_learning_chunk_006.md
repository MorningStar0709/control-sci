# A. Notation

The set of real numbers with zero is denoted $\mathbb { R } _ { + }$ . A Lipschitz constant of a locally Liphscitz continuous function $\varphi$ is denoted as $\operatorname { L i p } _ { \varphi } ,$ whereas the concrete compact set on which this constant is taken is defined in the each specific context. Spaces of class kappa functions are denoted $\kappa , \kappa _ { \infty }$ . A diagonal matrix formed from a vector x is denoted diag {x}.
