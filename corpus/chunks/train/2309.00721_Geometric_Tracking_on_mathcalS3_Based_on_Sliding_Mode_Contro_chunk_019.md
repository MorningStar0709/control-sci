# 4.2 Controller Design

The intrinsic error $g _ { e } \in T S ^ { 3 }$ can be expressed in the state manifold of the Lagrangian system (24) as $\bar { g } _ { e } =$ $( q _ { e } , \dot { q } _ { e } ) \in \mathcal { M }$ . The sliding variable therefore, according to (14), is given by

$$s \left(\bar {g} _ {e}\right) = \dot {q} _ {e} + \lambda (q _ {0, e} q _ {e} - \bar {1}), \tag {30}$$

$\lambda > 0$ being a design parameter. The following control law will render the sliding subgroup H¯ attractive

$$\bar {\tau} _ {c} = - \lambda (D (q _ {e}) (q _ {0, e} \dot {q} _ {e} + \dot {q} _ {0, e} q _ {e}) + C (q _ {e}, \dot {q} _ {e}) (q _ {0, e} q _ {e} - \bar {1}))- K _ {r} s \left(\bar {g} _ {e}\right), \tag {31}$$

where $K _ { r } = K _ { r } ^ { T } \in \mathrm { R } ^ { 4 \times 4 }$ is a positive definite gain matrix.

Theorem 7 (Almost-global asymptotic stability) The control law (31) in closed loop with the Lagrangian dynamics (24) stabilizes the equilibrium point $\bar { g } _ { e } ~ = ~ e$ almost globally asymptotically.

PROOF. It follows directly of the sliding variable $s ( \bar { g } _ { e } )$ as a consequence of Lemma 3, Lemma 4 and Lemma 5.
