In the arguments following (48), we showed that for fixed $m , n , t ,$ the ratio $\frac { p ^ { \prime } ( m , n , t ; \mu ) } { p ( m , n , t ; \mu ) }$ is nonincreasing with respect to $\mu .$ . Thus, if $\mu > \mu ^ { * }$ , we have that

$$\frac {p ^ {\prime} (m , n , t ; \mu)}{p (m , n , t ; \mu)} \leq \frac {p ^ {\prime} (m , n , t ; \mu^ {*})}{p (m , n , t ; \mu^ {*})}.$$

and the martingale property of $\{ W _ { n , m } \} _ { m = 0 } ^ { \infty }$ is established. □
