Assumption 2 (Transformer Robustness): Consider a trajectory $\left\{ \mathbf { y } _ { 0 : t } \right\}$ generated by some system i under the noise sequence $\left\{ \mathbf { w } _ { 0 : t } , \mathbf { v } _ { 0 : t } \right\}$ . Let $\left\{ \mathbf { y } _ { 0 : t } ^ { \prime } \right\}$ denote another trajectory under the same noise except that at time $\tau , \ \{ \mathbf { w } _ { \tau } , \mathbf { v } _ { \tau } \}$ is replaced by $\{ \mathbf { w } _ { \tau } ^ { \prime } , \mathbf { v } _ { \tau } ^ { \prime } \}$ . Let $\mathcal { D } _ { t } \ : = \ \mathbf { y } _ { 0 : t }$ and $y _ { t } ^ { \prime } : = \mathbf { y } _ { 0 : t } ^ { \prime } .$ Suppose the loss function $\ell ( \mathbf { y } , \cdot )$ is Lℓ-Lipschitz. Let $\boldsymbol { B } : =$ $\cap _ { j = 0 } ^ { t } \{ \| \mathbf { w } _ { j } \| ~ \leq ~ \bar { w } , \| \mathbf { v } _ { j } \| ~ \leq ~ \bar { v } \}$ for some $\bar { w } , \bar { v } \geq 0$ and $\mathbb { E } ^ { \prime } [ \cdot ] : = \mathbb { E } [ \cdot ~ | ~ B ]$ . Then, there exist constants $K \geq 0$ such that for any system i, any t and $\left\{ \mathbf { w } _ { 0 : t - 1 } , \mathbf { v } _ { 0 : t - 1 } \right\}$ , any $\tau < t$ and $\{ \mathbf { w } _ { \tau } ^ { \prime } , \mathbf { v } _ { \tau } ^ { \prime } \}$ , and any $\mathbb { T } \mathbb { E } \in { \cal A } .$ , we have

$$
\begin{array}{l} \mathbb {E} _ {\mathbf {w} _ {t}, \mathbf {v} _ {t}} ^ {\prime} \left[ \left| \ell (\mathbf {y} _ {t}, \mathrm{TF} (\mathcal {Y} _ {t - 1})) - \ell (\mathbf {y} _ {t}, \mathrm{TF} (\mathcal {Y} _ {t - 1} ^ {\prime})) \right| \right] \\ \leq \frac {K L _ {\ell}}{t - \tau} \sum_ {j = \tau} ^ {t - 1} \| \mathbf {y} _ {j} - \mathbf {y} _ {j} ^ {\prime} \|. \\ \end{array}
$$

In this assumption, trajectories $\mathbf { y } _ { j } ^ { \prime } = \mathbf { y } _ { j } ^ { \prime }$ for $j ~ < ~ \tau$ and possibly differ afterward due to the perturbation at time τ , which explains the summation term in the upper bound. It is shown in [6, Lemma B.5] that this assumption holds for a wide class of transformers.
