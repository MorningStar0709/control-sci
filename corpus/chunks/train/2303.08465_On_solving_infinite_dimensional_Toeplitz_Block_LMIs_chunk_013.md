# A. Truncation operator Π

Definition 4: Consider infinite-dimensional TB matrices A and B of compatible size. The truncation operator $\Pi _ { m }$ at order m is determined by:

$$\Pi_ {m} (\mathcal {A}) = \mathcal {T} _ {m} (A),\Pi_ {m} (\mathcal {A} + \mathcal {B}) = \Pi_ {m} (\mathcal {A}) + \Pi_ {m} (\mathcal {B})
\begin{array}{l} \Pi_ {m} (\mathcal {A B}) = \Pi_ {m} (\mathcal {A}) \Pi_ {m} (\mathcal {B}) + \mathcal {H} _ {(m, \eta)} (A ^ {+}) \mathcal {H} _ {(\eta , m)} (B ^ {-}) \\ + \mathcal {J} _ {n, m} \mathcal {H} _ {(m, \eta)} (A ^ {-}) \mathcal {H} _ {(\eta , m)} (B ^ {+}) \mathcal {J} _ {n, m} \tag {13} \\ \end{array}
$$

where $\eta \in \mathbb { Z } ^ { + } \cup \left\{ + \infty \right\}$ is such that 2η ≥ min $\cdot$ .

For a given $m ,$ the m−truncated TBLMI of (12) is:

$$\Pi_ {m} \big (\mathcal {L} (\mathcal {P}; \mathcal {A} _ {s}, s \in \mathbb {S}) \big) < 0 \tag {14}$$

For example, the m−truncated TBLMI associated to (10) is:

$$
\begin{array}{l} \Pi_ {m} ((\mathcal {A} - \mathcal {N}) ^ {*}) \Pi_ {m} (\mathcal {P}) + \Pi_ {m} (\mathcal {P}) \Pi_ {m} (\mathcal {A} - \mathcal {N}) \\ + \mathcal {H} _ {(m, \eta)} (A ^ {* +}) \mathcal {H} _ {(\eta , m)} (P ^ {-}) + \mathcal {H} _ {(m, \eta)} (P ^ {+}) \mathcal {H} _ {(\eta , m)} (A ^ {-}) \\ + \mathcal {J} _ {n, m} (\mathcal {H} _ {(m, \eta)} (A ^ {* -}) \mathcal {H} _ {(\eta , m)} (P ^ {+}) \\ + \mathcal {H} _ {(m, \eta)} (P ^ {-}) \mathcal {H} _ {(\eta , m)} (A ^ {+})) \mathcal {J} _ {n, m} <   0 \\ \end{array}
$$

with $\eta \in \mathbb { Z } ^ { + } \cup \left\{ + \infty \right\}$ is such that 2η ≥ min $( d ^ { o } A , d ^ { o } P )$ .

Theorem 3: If P solves the infinite-dimensional TBLMI (12) then P solves the truncated TBLMI (14) at any order m.

Proof: Consider a solution P to (12) then for any $m \ > \ 0 ,$ the principal submatrix $\Pi _ { m } \big ( \mathcal { L } ( \mathcal { P } ; \mathcal { \qquad } ) \big )$ of $\mathcal { L } ( \mathcal { P } ; \mathcal { A } _ { s } , s \in \mathbb { S } )$ is necessarily negative definite. Moreover, thanks to (2) and to Definition $4 , \Pi _ { m } ( \mathcal { L } ( \mathcal { P } ; \mathcal { A } _ { s } , s \in \mathbb { S } ) )$ can be explicitly developed without any approximation. Therefore, a solution to the obtained m−truncated TBLMI can be deduced from $\mathcal { P }$ itself.

Consequently, if the infinite-dimensional TBLMI (12) is feasible then there always exists a solution to the truncated TBLMI (14) at any order m. Unfortunately, (14) may contain terms involving infinite-dimensional Hankel matrices (when η = +∞ in (13)). The next section shows how this infinite-dimensional problem can be reduced to a finite one by considering banded approximation of the entries.
