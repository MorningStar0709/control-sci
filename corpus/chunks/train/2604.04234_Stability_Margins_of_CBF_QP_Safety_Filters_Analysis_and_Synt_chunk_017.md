By (25), the right-hand side is negative definite, so $\tilde { A } ( \kappa )$ is Hurwitz for every $\kappa \in [ \underline { { \kappa } } , \bar { \kappa } ]$ . Since $A _ { 0 }$ is Hurwitz, Prop. 2 implies that the origin is GES. □

Prop. 5 gives a conservative but convex certificate for a prescribed gain interval. We next use it to define a computable inner approximation of the exact interval in Definition 1.

Definition 2 (LMI-certified gain interval). Let $\tilde { A } ( \kappa )$ be defined by (23), and assume $\tilde { A } ( 1 )$ is Hurwitz. An interval $[ \kappa _ { 1 } , \kappa _ { 2 } ] \subset \mathbb { R } _ { > 0 }$ is called LMI-certifiable if there exists $P =$ $P ^ { \top } \succ 0$ such that (24)–(25) hold with $( \underline { { { \kappa } } } , \bar { \kappa } ) = ( \kappa _ { 1 } , \kappa _ { 2 } )$ . Define

$$\overline {{\kappa}} _ {\mathrm{LMI}} := \sup \{\kappa \geq 1: [ 1, \kappa ] \text { is LMI - certifiable } \}, \tag {28}\underline {{\kappa}} _ {\mathrm{LMI}} := \inf \{\kappa \in (0, 1 ]: [ \kappa , 1 ] \text { is LMI - certifiable } \}. \tag {29}$$

The interval $\left[ \underline { { \kappa } } _ { \mathrm { L M I } } , \overline { { \kappa } } _ { \mathrm { L M I } } \right]$ is called the LMI-certified gain interval.

The next result shows that the interval in Definition 2 is a conservative inner approximation of the exact stabilitypreserving gain interval.

Proposition 6. Let $\left[ \kappa _ { \mathrm { m i n } } , \kappa _ { \mathrm { m a x } } \right]$ denote the stabilitypreserving gain interval from Definition 1. Then

$$\left[ \underline {{\kappa}} _ {\mathrm{LMI}}, \overline {{\kappa}} _ {\mathrm{LMI}} \right] \subseteq \left[ \kappa_ {\min}, \kappa_ {\max} \right]. \tag {30}$$

Proof. If [1, κ] is LMI-certifiable, then by Prop. 5, the matrix $\tilde { A } ( \kappa )$ is Hurwitz throughout that interval, so $[ 1 , \kappa ] \subseteq$ $[ \kappa _ { \mathrm { m i n } } , \kappa _ { \mathrm { m a x } } ]$ . Taking the supremum over all such $\kappa \geq 1$ gives $\overline { { \kappa } } _ { \mathrm { L M I } } \leq \kappa _ { \mathrm { m a x } }$ . The argument for $\underline { { \kappa } } _ { \mathrm { L M I } } \geq \kappa _ { \mathrm { m i n } }$ is analogous on $( 0 , 1 ]$ . Therefore, (30) holds. □
