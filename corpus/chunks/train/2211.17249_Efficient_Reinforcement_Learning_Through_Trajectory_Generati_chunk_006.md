# 2.2. Problem formulation

We consider a discrete-time linear time-invariant system

$$
\begin{array}{l} \boldsymbol {x} (k + 1) = \boldsymbol {A x} (k) + \boldsymbol {B u} (k) \\ \text {(1)} \end{array}
\pmb {y} (k) = \pmb {C x} (k)$$

with state $\ b { x } \in \mathbb { R } ^ { n }$ , action $\pmb { { u } } \in \mathbb { R } ^ { m }$ , and output $\textbf { \em y } \in \mathbb { R } ^ { q }$ (sometimes called observations in RL literature). If C is not of full (column) rank, we say that the states are not directly observed. Otherwise, we assume $C = { \cal I } _ { n }$ and the states are directly observed. We assume that A and B are not known. The matrix C is also unknown if the state is not directly observed. We also make the standard assumption that (A, B) is stabilizable and $( A , C )$ observable (Hespanha, 2009).

A trajectory is a sequence of observations and actions of length T , given by $\pmb { \tau } = \{ \pmb { y } ( 0 ) , \pmb { u } ( 0 ) , \cdots$ · , $, \pmb { y } ( T - 1 ) , \pmb { u } ( T - 1 ) \}$ . The control action ${ \pmb u } ( k )$ most commonly comes from the control policy conditioned on the observation at time $k ,$ , written as $\pi _ { \pmb { \theta } } \left( \pmb { u } ( k ) ~ | ~ \pmb { y } ( k ) \right)$ with $\pmb \theta$ being the parameter for the control policy. Let $c ( \tau )$ be the cost defined over the trajectory $\tau$ . The goal is to optimize the control paramater $\pmb \theta$ to minimize the expected cost over trajectories, written as:

$$J (\boldsymbol {\theta}) = \mathbb {E} _ {\boldsymbol {\tau} \sim p _ {\pi_ {\boldsymbol {\theta}}}} c (\boldsymbol {\tau}), \tag {2}$$

where $p _ { \pi _ { \theta } }$ is the probability distribution of trajectory subject to the policy $\pi _ { \theta }$ . The definition of $c ( \tau )$ varies for different problems. For linear control policy, quadratic costs are most commonly utilized to convert the optimization into classical linear quadratic problems (Fazel et al., 2018; Zheng et al., 2021). There are typically not closed-form solutions for (2) for other cost functions, e.g., $\begin{array} { r } { c ( \pmb { \tau } ) = \sum _ { i = 1 } ^ { n } \operatorname* { m a x } _ { k = 0 , \cdots , K - 1 } | x _ { i } ( k ) | } \end{array}$ for the frequency control problem in power systems (Cui et al., 2022a). In this case, gradient-based methods can be utilized to update θ, but the lack of system parameters makes it difficult to compute the gradient $\nabla _ { \pmb { \theta } } J ( \pmb { \theta } )$ .
