Current across the second resistor $R _ { 2 }$ is $I _ { 2 } = 3 \mathrm { V } / 4 \Omega = 0 . 7 5 \mathrm { A }$ as current flows from the higher potential $( e _ { A } = 0 )$ to the lower potential $( e _ { O } = - 3 \mathrm { V } )$ .
