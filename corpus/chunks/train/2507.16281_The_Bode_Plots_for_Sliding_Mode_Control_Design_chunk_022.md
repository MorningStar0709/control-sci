1) Linearized Model of Propagation: The EG provides a basis for linearizing the closed-loop dynamics in the presence of slowly varying disturbances and is thus instrumental in constructing analytical propagation models [37]. Consider a perturbation signal of the form $f ( t ) = \eta \cos ( \Omega t )$ . At steady state, the corresponding bias component in the sliding variable (5) is assumed to take the form $\sigma _ { 0 } ( t ) = \sigma _ { 0 } \cos ( \Omega t + \phi )$ . This result is justified by the notion of equivalent control, according to which the slow-motions dynamics can be approximated by the cycle-averaged effect of the fast-motions [24]. Under Assumptions 2-3 and in accordance with Fig. 7, the sensitivity transfer function (see more details in [38]) allowing to compute the propagation of slow motions is given by

$$\frac {\sigma_ {0} (\mathrm{s})}{f _ {0} (\mathrm{s})} = \frac {G (\mathrm{s})}{1 + K _ {n} W (\mathrm{s})}, \tag {34}$$

where $\mathrm { s } = j \Omega$ denotes the Laplace complex frequency, G(s) is the TF of the plant, and $W ( \mathrm { s } ) = G _ { a } ( \mathrm { s } ) G ( \mathrm { s } )$ corresponds to the overall actuator–plant cascade. The parameter $K _ { n }$ is the EG of the relay-type controller computed in (32).
