# 3. Stability results using the expression of the flow

In this section, we consider the expression of the flow of a reset or an impulsive switched system and discuss the stability issues raised in Section 2.4.1. We derive inspiration from the techniques used in [2, 17] to obtain the results. The bounds given in the following remark will be used throughout the paper.

Remark 3.1. For each $p \in \mathcal P$ , let $J _ { p }$ be the complex Jordan form of the subsystem matrix $A _ { p }$ , which is unique up to permutation of Jordan blocks. Then

(1) if $A _ { p }$ is unstable, then there exist $c _ { p } , \mu _ { p } ^ { * } > 0$ satisfying $\|  { \mathrm { e } } ^ { J _ { p } t } \| \le c _ { p }  { \mathrm { e } } ^ { \mu _ { p } ^ { * } t }$ , for all $t > 0$ . Clearly $\mu _ { p } ^ { * } \geq \mu _ { p }$ , where $\mu _ { p } > 0$ is the real part of the eigenvalue of $A _ { p }$ farthest to the imaginary axis on its right.   
(2) If $A _ { p }$ is stable, then there exist $c _ { p } , \lambda _ { p } ^ { * } > 0$ satisfying $\|  { \mathrm { e } } ^ { J _ { p } t } \| \leq c _ { p }  { \mathrm { e } } ^ { - \lambda _ { p } ^ { * } t }$ , for all $t > 0$ Clearly $- \lambda _ { p } ^ { * } \geq - \lambda _ { p } ,$ where $- \lambda _ { p } < 0$ is the real part of the eigenvalue of $A _ { p }$ closest to the imaginary axis on its left.
