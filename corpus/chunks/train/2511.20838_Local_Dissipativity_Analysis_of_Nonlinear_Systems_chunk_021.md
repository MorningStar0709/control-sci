# 4 Local dissipativity analysis

The error bounds developed in Section 3 are now leveraged to formulate conditions for which a CPA function or combined CPA and quadratic function is the storage function of a dissipative system. Section 4.2 then uses these to develop convex optimization problems that simultaneously synthesize a system’s storage function and analyze common QSR characterizations – such as gain, passivity, and conic bounds.
