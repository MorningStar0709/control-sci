# 2.2 Estimation problem

We are interested in computing the state predictions $\hat { x } _ { t + 1 }$ of $x _ { t + 1 }$ for $t = 0 , \ldots , T$ , when only knowing yt and $u _ { t }$ but not $v _ { t }$ and $w _ { t } ,$ and minimizing the total cost

$$\mathrm{cost} (\hat {x} _ {1} - x _ {1}, \dots , \hat {x} _ {T + 1} - x _ {T + 1}) = \sum_ {t = 1} ^ {T + 1} \ell_ {t} (\hat {x} _ {t} - x _ {t}), \tag {2}$$

where $\ell _ { t }$ is a given loss function (e.g. least squares). To do so, we design a dynamic Luenberger-type observer defined by the gains $L _ { \tau | t } \in \mathbb { R } ^ { n \times m } { \mathrm { ~ f o r ~ } } \tau = 0 , \ldots , t ,$ and given by

$$\hat {x} _ {t + 1} = A _ {t} \hat {x} _ {t} + B _ {t} u _ {t} - \sum_ {\tau = 0} ^ {t} L _ {\tau | t} (C _ {\tau} \hat {x} _ {\tau} - y _ {\tau}). \tag {3}$$

The gains are chosen to stabilize the prediction error $e _ { t } = { \hat { x } } _ { t } - x _ { t }$ folloing the dynamics

$$e _ {t + 1} = A _ {t} e _ {t} - w _ {t} - \sum_ {\tau = 0} ^ {t} L _ {\tau | t} (C _ {\tau} e _ {\tau} - v _ {\tau}), \tag {4}$$

while minimizing the cost (2).

Remark. We highlight that the error on the initial state is embedded without loss of generality in the first disturbance, i.e. $w _ { 0 } = A _ { 0 } ( \hat { x } _ { 0 } - x _ { 0 } )$ and $e _ { 0 } = 0$ .

Substituting (4) in the cost (2) gives the final problem

$$\underset {\boldsymbol {v}, \boldsymbol {w}, \{L _ {\tau | t}: t \in [ 0, T ], \tau \in [ 0, t ] \}, e} {\arg \min} \operatorname{cost} (\boldsymbol {e}), \tag {5}\text { s.t. } e _ {t + 1} = A _ {t} e _ {t} - w _ {t} - \sum_ {\tau = 0} ^ {t} L _ {\tau | t} (C _ {\tau} e _ {\tau} - v _ {\tau}),e _ {0} = 0,$$

where

$$
\boldsymbol {v} = \left[ \begin{array}{c} v _ {0} \\ v _ {1} \\ \vdots \\ v _ {T} \end{array} \right], \boldsymbol {w} = - \left[ \begin{array}{c} A _ {0} e _ {0} \\ w _ {1} \\ \vdots \\ w _ {T} \end{array} \right], \boldsymbol {e} = \left[ \begin{array}{c} e _ {1} \\ \vdots \\ e _ {T} \\ e _ {T + 1} \end{array} \right]. \tag {6}
$$

Remark. While in the sequel we focus on the finite horizon estimation problem (5), we notice that it provides the building block for defining MHE methods that are applicable for $t = 0 , \ldots , \infty$ (Alessandri et al., 2008; Ferrari-Trecate et al., 2002; Farina et al., 2010).

We also define the set V such that each element ${ \mathbf { } } v _ { i }$ of v $\in \mathbb { V }$ is in v, and similarly, the set W. From Assumption 1, it follows that V and W are non-empty full-dimensional ellipsoids, and can therefore be written as $\mathbb { V } = \{ \pmb { v } | \Vert \mathscr { H } _ { v } \pmb { v } \Vert _ { 2 } \leq \hat { 1 } \}$ and $\mathbb { W } = \{ \pmb { w } | \| \mathcal { H } _ { w } \pmb { w } \| _ { 2 } \le 1 \}$ }, where $\mathcal { H } _ { v }$ and $\mathcal { H } _ { w }$ are square invertible block-diagonal matrices.
