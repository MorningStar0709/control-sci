The proposed switching algorithm defines a time-varying graph, i.e.

$$
\boldsymbol {\mathcal {G}} (k) = \left\{ \begin{array}{l l} \mathcal {G} _ {0} ^ {\prime} (k) \in \Phi (m), & \text { if } \sigma (m) \neq \sigma (m - 1) \\ & \text { and } k \bmod m = 0, \\ \mathcal {G} _ {i (k)} ^ {\prime} (k) \in \Phi (m) & \text { else   if } k \bmod m \neq 0 \end{array} \right. \tag {17}
$$

where

$$
i (k) = \left\{ \begin{array}{l l} i (k - 1) + 1 & \text { if   } i (k - 1) <   p (m), \\ 0 & \text { if   } i (k - 1) = p (m) \end{array} \right. \tag {18}
$$

with $i ( k ) \in \{ 0 , 1 , \ldots , p ( m ) - 1 \}$ .
