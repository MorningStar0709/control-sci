Then by Arzela-Ascoli Theorem [6, p.175], the set $\{ \mathcal { T } ( \Pi ) : \Pi \in K \}$ is totally bounded and hence the closure of $\{ \mathcal { T } ( \Pi ) : \Pi \in K \}$ is compact [20, Appendix A4]. By definition [6, Def. $4 . 1 ] , \mathcal { T } : K  K$ is a compact operator. Since K be a closed, bounded, and convex subset of the normed space $C ( [ 0 , T ] ; \mathbb { R } )$ (endowed with the uniform norm) and $\tau : K \to C ( [ 0 , T ] ; \mathbb { R } )$ is a compact map such that $\{ \mathcal { T } ( \Pi ) : \Pi \in K \} \subseteq K$ (under the condition (33)), an application of the Schauder fixed-point theorem [6, p. 150] implies the equation (34) has at least one a solution $\Pi \in K$ . Therefore the coupled differential equations (19) and (20) have at least one joint solution $( V , \Pi )$ with $\Pi \in K$ and $V \in C ( [ 0 , T ] , \mathbb { R } )$ .
