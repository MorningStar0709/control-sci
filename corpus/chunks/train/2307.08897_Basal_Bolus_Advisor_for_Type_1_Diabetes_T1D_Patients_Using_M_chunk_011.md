The goal was to train the bolus agent using the pre-trained basal agent in the loop such that the action taken by the basal agent could affect the state of the environment, i.e. the BG simulator which consequently required the bolus agent to adapt its action accordingly. The training objective for this step was similar to the previous step, having the effect of both meals and the action taken by the basal agent on the current state of the environment.

Training of the bolus agent involves minimizing the following objective function:

$$J (\theta) = \mathbb {E} _ {\tau \sim p _ {\theta}} \left[ \sum_ {t = 0} ^ {T} \gamma^ {t} (r _ {t} + \alpha H (\pi_ {\theta} (a _ {t}, b _ {t} | s _ {t}))) \right], \tag {3}$$

where $a _ { t }$ is the action taken by the bolus agents at time t, and $\pi _ { \boldsymbol { \theta } } ( a _ { t } | \boldsymbol { s } _ { t } )$ is the policy of the two agents. $r _ { t }$ is the reward reπθceived at time t obtained from the bolus agent’s reward function explained in Algorithm 2 in section 2.3.

Finally, the performance of the obtained multi-agent basalbolus RL model was evaluated by exposing it to real-time new sets of data generated by the BG simulator, described in section 3.
