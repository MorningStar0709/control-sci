# 3 Fishing quotas

We consider the fishing quota problem introduced in [LPP21]. Let $X _ { t } \in \mathbb { R } ^ { d _ { 1 } }$ be the fish biomass for every fish species; we wish to keep it close to an ideal state $\boldsymbol { \mathcal { X } } _ { t } \in \mathbb { R } ^ { d _ { 1 } }$ . The dynamics of X are given by

$$d X _ {t} = X _ {t} * \left((r - u _ {t} - \kappa X _ {t}) d t + \eta d W _ {t}\right), t \in [ 0, T ], \tag {3.1}$$

where $r \in \mathbb { R } ^ { d _ { 1 } }$ is the growth rate for each species, $u _ { t } \in \mathbb { R } ^ { d _ { 1 } }$ is the controlled fishing (with $d _ { 3 } = d _ { 1 } ) , \kappa \in \mathcal { M } _ { d _ { 1 } , d _ { 1 } } ( \mathbb { R } )$ （2 is the interaction matrix between the fish species, $\eta \in \mathcal { M } _ { d _ { 1 } , d _ { 2 } } ( \mathbb { R } )$ , W is a Rd2-valued Brownian motion. $$J (u) = \mathbb {E} \left[ \int_ {0} ^ {T} (| X _ {t} - \mathcal {X} _ {t} | ^ {2} - \langle \alpha , u _ {t} \rangle) d t + \beta [ u ] ^ {0, T} \right], \tag {3.2}$$
