# B. Graph Theory

Consider a graph $\mathcal { G } = ( \nu , \mathcal { E } , \mathcal { A } )$ , where V denotes the set of nodes indexed by $\mathcal { V } = \{ 1 , \dots , N \} , \mathcal { E } \subseteq \mathcal { V } \times \mathcal { V }$ represents the set of edges and $\mathcal { A } = [ \bar { a } _ { i j } ] \in \mathbb { R } ^ { N \times N }$ is the adjacency matrix, where $a _ { i j } = 1$ if there exists an edge from node i to node $j ,$ and $a _ { i j } ~ = ~ 0$ otherwise. The graph’s Laplacian matrix is denoted by $\mathcal { L } = [ l _ { i j } ] \in \mathbb { R } ^ { N \times N }$ and is defined as $l _ { i j } = - a _ { i j }$ , $\forall i \ne j$ and $\begin{array} { r } { l _ { i i } = \mathbf { \bar { \sum } } _ { j = 1 } ^ { N } a _ { i j } . } \end{array}$ N . Let $\mathcal { N } _ { i } = \{ j \in \mathcal { V } | ( \bar { j } , i ) \in \bar { \mathcal { E } } \}$ represent the set of neighboring nodes to node i. A graph is considered connected if a sequence of edges exists that allows one to move from any one node to any other. For connected undirected graphs, the Laplacian matrix L is positive semidefinite. The incidence matrix, denoted by $\mathcal { D } \doteq [ d _ { i j } ] \in \mathbb { R } ^ { N \times | \mathcal { E } | }$ is defined such that $d _ { i k } = - 1$ if the $k ^ { t h }$ edge starts from node i, $d _ { i k } = 1$ if it ends at node i, and $d _ { i k } = 0$ otherwise. Notably, for an undirected graph, the conditions $\begin{array} { r } { \mathcal { L } \mathbf { 1 } _ { N } = \mathbf { 0 } _ { N } , \mathcal { L } ^ { \top } = \mathcal { L } . } \end{array}$ , and $\begin{array} { r } { \mathcal { L } = \mathcal { D } \mathcal { D } ^ { \top } } \end{array}$ hold [16].
