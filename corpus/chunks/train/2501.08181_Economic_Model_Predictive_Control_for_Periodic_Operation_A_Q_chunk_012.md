# V. STABILITY AND CONVERGENCE ANALYSIS

Algorithm 1 implements (13) in a receding horizon fashion. In the remainder of this section, we show that its solution achieves the same desirable closed-loop properties of the original (non-QP) E-MPC P, namely, (i) for any feasible initial state the optimization problem is recursively feasible, and (ii) the optimal trajectory of the system is asymptotically stable for the controlled system. These properties are formally presented next.

Assumption 4 (Controllability): For integers $0 \leq i < j$ let

$$\Psi (j, i) = \left(A _ {j} \cdot A _ {j - 1} \dots A _ {i + 1}\right) \cdot B _ {i}.$$

Then, let the controllability matrix Γ be defined as

$$
\begin{array}{l} \Gamma_ {k} (c) = [ \Psi (k + c, k), \Psi (k + c, k + 1), \dots \\ \dots , \Psi (k + c, k + c - 1), B _ {k + c} ]. \\ \end{array}
$$

We assume that

$$\exists c \in [ 0, T - 1 ]: \operatorname{rank} \left(\Gamma_ {k} (c)\right) = n, \forall k.$$
