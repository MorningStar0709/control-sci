# Algorithm 1 Three-block ADMM topology solver at update time $t _ { k }$

1: Initialize: $( z ^ { ( 0 ) } , f ^ { ( 0 ) } , r ^ { ( 0 ) } , s ^ { ( 0 ) } , \lambda ^ { ( 0 ) } )$ , penalty parameters $\rho > 0 , \beta \geq 0$ , and stopping tolerances. Set $k \gets 1$ .   
2: while stopping criterion not met do   
3: Block 1: Compute $( z ^ { ( k ) } , f ^ { ( k ) } )$ by solving(22).   
4: Block 2: Update $r ^ { ( \dot { k } ) }$ by solving (23).   
5: Block 3: Update $s ^ { ( k ) }$ componentwise using (25).   
6: Dual update: Update according to (27).   
7: $k \gets k + 1$   
8: end while   
9: Return: final edge vector $z ^ { ( k ) }$ and Laplacian $L _ { k }$ constructed from the active edges.

Therefore

$$s _ {e} ^ {(k)} = - \frac {\lambda_ {e} ^ {(k - 1)} + \rho (z _ {e} ^ {(k)} - r _ {e} ^ {(k)})}{\rho + \beta}, \quad e = 1, \dots , m. \tag {25}$$

This update is fully explicit and independent across edges, so Block 3 is computationally cheap.

After updating $( z ^ { ( k ) } , f ^ { ( k ) } , r ^ { ( k ) } , s ^ { ( k ) } )$ , we update the dual variables. The residual of the coupling constraint (19) at iteration k is

$$d ^ {(k)} := z ^ {(k)} - r ^ {(k)} + s ^ {(k)}. \tag {26}$$

The dual variables are updated by

$$\lambda^ {(k)} = \lambda^ {(k - 1)} + \rho d ^ {(k)}. \tag {27}$$

If the method converges, then $d ^ { ( k ) }$ tends to zero, so z, r, and s agree asymptotically.

Algorithm 1 summarizes the three-block ADMM scheme for a single topology update time. In the closed-loop setting, this algorithm is called at each $t _ { k } ,$ and the resulting edge vector $\overset { \mathcal { O } } { \boldsymbol { z } } { \left( K \right) }$ is used to build the Laplacian $L _ { k }$ .
