If we further assume that $\tilde { A } = \tilde { C } = \tilde { b } = \tilde { \sigma } = 0$ , and that $F , f$ are smooth deterministic functions of time (thus $F ^ { \prime } = f ^ { \prime } = 0 )$ , then we can apply the standard ansatz $\tilde { Y } _ { t } = P _ { t } \tilde { X } _ { t } + q _ { t }$ for smooth functions $( P , q )$ , and apply Itˆo’s formula to $P \tilde { X } + q$ to obtain that

$$d \tilde {Y} _ {t} = P \hat {B} u ^ {*} d t + P \hat {D} u ^ {*} d W _ {t} + \dot {P} X d t + \dot {q} _ {t} d t,$$

where $\hat { B } : = A ^ { - 1 } \tilde { B } , \hat { D } : = A ^ { - 1 } \tilde { D } , \hat { G } : = A _ { T } \tilde { G }$ and $\hat { M } : = A \tilde { M }$ . It then follows from (4.16), (4.17) that

$$\tilde {Z} = P \hat {D} u, \quad u ^ {*} = - (\tilde {N} + \hat {D} ^ {2} P) ^ {- 1} (\hat {B} P \tilde {X} + \hat {B} q), \quad P \hat {B} u ^ {*} + \dot {P} \tilde {X} + \dot {q} = - \hat {M} (A \tilde {X} + \zeta). \tag {4.18}$$

In view of the representation of the optimal control $u ^ { * }$ the pair $( P , q )$ satisfies the (deterministic) Riccati equation

$$
\left\{ \begin{array}{l} \dot {P} _ {t} = \hat {B} _ {t} ^ {2} \left(\tilde {N} + \hat {D} _ {t} ^ {2} P _ {t}\right) ^ {- 1} P _ {t} ^ {2} - \hat {M} _ {t} A _ {t} \\ P _ {T} = 2 \hat {G} A _ {T}, \end{array} \right. \tag {4.19}

\left\{ \begin{array}{l} \dot {q} _ {t} = P _ {t} \hat {B} _ {t} ^ {2} (\tilde {N} + \hat {D} _ {t} ^ {2} P _ {t}) ^ {- 1} q _ {t} - \hat {M} _ {t} \zeta_ {t} \\ q _ {T} = 2 \hat {G} \zeta_ {T}. \end{array} \right. \tag {4.20}
$$
