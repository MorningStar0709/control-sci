Problem 10.14 Consider $M \in \mathbb { C } ^ { 2 n \times 2 n }$ to be given. Let $\Delta _ { 2 }$ be a n × n block structure, and suppose that $\mu _ { 2 } ( M _ { 2 2 } ) < 1$ . Suppose also that $\mathcal { F } _ { l } ( M , \Delta _ { 2 } )$ is invertible for all $\Delta _ { 2 } \in$ $\mathbf { B } \Delta _ { 2 }$ . For each $\alpha > 1$ , find a matrix $W _ { \alpha }$ and a block structure $\hat { \Delta }$ such that

$$\max _ {\Delta_ {2} \in \mathbf {B} \boldsymbol {\Delta} _ {2}} \kappa (\mathcal {F} _ {l} (M, \Delta_ {2})) < \alpha$$

if and only if

$$\mu_ {\hat {\Delta}} (W _ {\alpha}) < 1$$

where κ is the condition number.
