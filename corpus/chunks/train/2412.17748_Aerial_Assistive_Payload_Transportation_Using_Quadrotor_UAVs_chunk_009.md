# A. System Model

For the modeling purposes, we assign a global world inertial reference frame ${ \mathcal { T } } _ { G } = \{ O , X , Y , Z \}$ and a body-fixed reference frame $L _ { F } = \{ o , x _ { L } , y _ { L } , z _ { L } \}$ attached to the CoM of the payload as well as a reference frame ${ { B } _ { i } } = \left\{ { { o } _ { i } } , { { x } _ { B _ { i } } } , y _ { B _ { i } } , { { z } _ { B _ { i } } } \right\}$ attached to the CoM of each quadrotor where $i = 1 . . . N$ $( N = 2$ for the system in this work). All reference frames have a positive z axis pointing upward (gravity is negative). Let $p _ { i } = [ x _ { i } , y _ { i } , z _ { i } ] ^ { \top }$ and $v _ { i } = \dot { p }$ be the translational position and velocity of the quadrotor, respectively, defined in $\mathcal { T } _ { G }$ , $\omega = [ p , q , r ] ^ { \top }$ be the vector of angular rates defined in the body-fixed frame, and $\Theta = [ \phi , \theta , \bar { \psi } ] ^ { \top }$ be the vector of Euler angles, roll, pitch, and yaw that represent the attitude of the aerial vehicle in $\mathcal { T } _ { G } . R \in S O ( 3 )$ represents the rotation matrix from the body-fixed frame to $\mathcal { T } _ { G }$ described with respect to the Special Orthogonal Group $S O ( 3 )$ where R has a determinant of +1 and $R R ^ { \top } \ : = \ : { \bf I } _ { 3 }$ with $\mathbf { I } _ { 3 }$ being an identity matrix with dimension 3-by-3 [38], [45], [46]. The rotation matrix R follows the convention of $Z - X - Y$ of Euler angles and is given by [45]:

$$
\mathbf {R} = \left[ \begin{array}{c c c} c \theta c \psi - s \phi s \theta s \psi & - c \phi s \psi & s \theta c \psi + s \phi c \theta s \psi \\ c \theta s \psi + s \phi s \theta c \psi & c \phi c \psi & s \theta s \psi - s \phi c \theta c \psi \\ - c \phi s \theta & s \phi & c \phi c \theta \end{array} \right] (1)
$$
