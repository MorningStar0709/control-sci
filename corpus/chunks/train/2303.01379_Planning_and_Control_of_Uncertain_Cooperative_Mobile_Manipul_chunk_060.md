# 2.2.1 System Model

We derive in this section the model of the system object-robots.

We denote by $q _ { i } , \dot { q } _ { i } \in \mathbb { R } ^ { n _ { i } }$ , with ${ n _ { i } } \in \mathbb { N } , \forall i \in \mathcal { N } : = \{ 1 , . . . , N \}$ , the generalized joint-space variables and their time derivatives of agent i, with $q _ { i } : =$ $[ q _ { i _ { 1 } } , \ldots , q _ { i _ { n _ { i } } } ]$ . The overall joint configuration is then $q : = [ q _ { 1 } ^ { \top } , \ldots , q _ { N } ^ { \top } ] ^ { \top } , \dot { q } : =$ $[ { \dot { q } } _ { 1 } ^ { \top } , \dots , { \dot { q } } _ { N } ^ { \top } ] ^ { \top } \in \mathbb { R } ^ { n }$ , with $\textstyle n : = \sum _ { i \in { \mathcal { N } } } n _ { i }$ . In addition, the inertial position and orientation of the ith end-effector, denoted by $p _ { E _ { i } }$ and $\eta _ { E _ { i } }$ , respectively, can be derived by the forward kinematics and are smooth functions of $q _ { i } , \mathrm { i . e }$ . $p _ { E _ { i } } : = p _ { E _ { i } } ( q _ { i } ) : \mathbb { R } ^ { n _ { i } }  \mathbb { R } ^ { 3 } , \eta _ { E _ { i } } : = \eta _ { E _ { i } } ( q _ { i } ) : \mathbb { R } ^ { n _ { i } }  \mathbb { T }$ , where T is an appropriate orientation space. The differential equation describing the dynamics of each agent is [85]:

$$B _ {i} \ddot {q} _ {i} + C _ {q _ {i}} \dot {q} _ {i} + g _ {q _ {i}} + d _ {q _ {i}} = \tau_ {i} - J _ {i} ^ {\top} h _ {i}, \tag {2.1}$$

where $B _ { i } : = B _ { i } ( q _ { i } ) : \mathbb { R } ^ { n _ { i } }  \mathbb { R } ^ { n _ { i } \times n _ { i } }$ is the positive definite inertia matrix, $C _ { q _ { i } } : = C _ { q _ { i } } ( q _ { i } , \dot { q } _ { i } ) : \mathbb { R } ^ { 2 n _ { i } }  \mathbb { R } ^ { n _ { i } \times n _ { i } }$ is the Coriolis matrix, $g _ { q _ { i } } : = g _ { q _ { i } } ( q _ { i } )$ : Rni → Rni is the joint-space gravity term, $d _ { q _ { i } } : = d _ { q _ { i } } ( q _ { i } , \dot { q } _ { i } , t ) : \mathbb { R } ^ { 2 n _ { i } } \times \mathbb { R } _ { \ge 0 } $ $\mathbb { R } ^ { n _ { i } }$ is a bounded vector representing unmodeled friction, uncertainties and external disturbances, $h _ { i } \in \mathbb { R } ^ { 6 }$ is the vector of generalized forces that agent i exerts on the grasping point with the object and $\tau _ { i } = [ \tau _ { i , 1 } , \dots , \tau _ { i , n _ { i } } ] ^ { \top } \in \mathbb { R } ^ { n _ { i } }$ is the vector of joint torques, acting as control inputs, $\forall i \in { \mathcal { N } }$ .
