From the strong duality (7), we know $V _ { 0 } ( \pmb { s } ) \leq \overline { { V } } _ { 0 } ( \pmb { s } )$ , which implies $0 \leq \overline { { V } } _ { 0 } ( \pmb { s } ) - V _ { 0 } ( \pmb { s } ) \leq \overline { { V } } _ { 0 } ( \pmb { s } ) - \underline { { V } } _ { 0 } ( \pmb { s } )$ . When the dual gap $\overline { { V } } _ { 0 } - \underline { { V } } _ { 0 }$ is sufficiently tight, we can conclude that the performance of policy π must be very close to the optimality. This line of work has sparked a rapidly expanding literature that leverages information-relaxation duality to certify the near-optimality of a wide range of heuristic policies. More recently, Chen et al. [2024] develop a supersolution-based approach to improve the optimality for a given policy if we find that its duality gap is loose.
