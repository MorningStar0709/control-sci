# A. Robotic Navigation

Our first example involves a robotic navigation task for a system modeled as a double integrator with uncertain friction effects. The system is in the form of (4):

$$
\underbrace {\left[ \begin{array}{c} \dot {x} _ {1} \\ \dot {x} _ {2} \\ \dot {x} _ {3} \\ \dot {x} _ {4} \end{array} \right]} _ {\dot {x}} = \underbrace {\left[ \begin{array}{c} x _ {3} \\ x _ {4} \\ 0 \\ 0 \end{array} \right]} _ {f (x)} + \underbrace {\left[ \begin{array}{c c} 0 & 0 \\ 0 & 0 \\ - \frac {x _ {3}}{\mathrm{m}} & 0 \\ 0 & - \frac {x _ {4}}{\mathrm{m}} \end{array} \right]} _ {Y (x)} \underbrace {\left[ \begin{array}{c} \mu_ {1} \\ \mu_ {2} \end{array} \right]} _ {\theta} + \underbrace {\left[ \begin{array}{c c} 0 & 0 \\ 0 & 0 \\ \frac {1}{\mathrm{m}} & 0 \\ 0 & \frac {1}{\mathrm{m}} \end{array} \right]} _ {g (x)} \underbrace {\left[ \begin{array}{c} u _ {1} \\ u _ {2} \end{array} \right]} _ {u},
$$

where m $\in \mathbb { R } _ { > 0 }$ is a known mass and $\mu _ { 1 } , \mu _ { 2 } \ \in \ \mathbb { R } _ { > 0 }$ are uncertain viscous friction coefficients. For simplicity, we set ${ \mathrm { ~ m ~ } = \ \mu _ { 1 } \ = \ \mu _ { 2 } \ = \ 1 }$ . The objective is to stabilize the system to the origin while avoiding a set of static obstacles in the state space. To achieve the stabilization task, we construct an ES-aCLF as $V ( x ) : = x ^ { \top } P x$ with $P = [ 2 , 0 , 1 , 0 ; 0 , 2 , 0 , 1 ; 1 , 0 , 1 , 0 ; 0 , 1 , 0 , 1 ]$ and $c _ { 3 } = 1$ . The safety objective is achieved by considering a collection of safe sets of the form (2) with $h ^ { ( i ) } ( x ) : = ( x _ { 1 } -$ $p _ { 1 , i } ) ^ { 2 } + ( x _ { 2 } - p _ { 2 , i } ) ^ { 2 } - R ^ { 2 }$ , where $( p _ { 1 , i } , p _ { 2 , i } ) \in \mathbb { R } ^ { 2 }$ denotes the center of a circular disk and $R _ { i } ~ \in ~ \mathbb { R } _ { > 0 }$ its radius. With straightforward calculations one can verify that this CBF candidate has relative degree $r \ = \ 2$ with respect to the dynamics and that Assumption 1 is satisfied. Although the dynamics and safe set are relatively simple, the aCBF techniques from [16], [18]–[20] cannot be applied to solve the problem as currently constructed since the relative degree of h is larger than $\mathrm { o n e ^ { 6 } }$ . To illustrate the efficacy of the developed approach, we simulate the system under the HO-RaCBF/ES-aCLF control architecture presented herein and compare the results to controllers take a purely robust approach.
