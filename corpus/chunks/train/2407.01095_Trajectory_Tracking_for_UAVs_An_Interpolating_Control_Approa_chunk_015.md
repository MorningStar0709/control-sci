# V. RESULTS AND DISCUSSION

The controllers are evaluated for tracking a lemniscate reference trajectory in terms of control quality and computational complexity. Control quality is assessed based on the quadratic criterion, the integral square error (ISE), and the energy required to control the UAV.

The quadratic criterion value in Equation (1) is weighted by the inverse of $T _ { s } .$ In contrast, the ISE evaluates the accuracy of the reference position tracking. For example, for position x the ISE is calculated as

$$I S E = \frac {1}{T _ {s}} \sum_ {k = 0} e _ {k} ^ {2}, \tag {13}$$

where $e _ { k } \ = \ ( { \bar { x } } _ { k } - x _ { k } )$ . The presented ISE value is the sum of the ISE from all axes. The energy is directly dependent on the controller output, which is by design given as the desired acceleration. To illustrate, for the ⃗z-axis controller, energy is assessed using the following equation

$$E = \frac {1}{T _ {s}} \sum_ {k = 0} ^ {\infty} z _ {k} ^ {2}. \tag {14}$$

In laboratory experiments, the criteria cannot be weighted solely by $T _ { s }$ due to asynchronous communication and delays resulting from the control action calculation. Thus, the difference between time instants $t _ { k } \mathrm { ~ - ~ } t _ { k - 1 }$ is used to weight the individual elements. The evaluation of controller time demands is based on total, average, and maximum computation time. For laboratory experiments, the number of calculated control actions is also included.
