Definition 1: A function $\rho : \mathbb { R } _ { \geq 0 } $ R is a prescribed performance function if it satisfies, for all $t \in \mathbb { R } _ { > 0 } \colon ( \mathrm { i } ) \rho ( t ) \in$ $\mathcal { C } ^ { 1 } ; ( \mathrm { i i } ) 0 < \rho ( t ) \le \overline { { \rho } }$ for some $\overline { { \rho } } < \infty$ and (iii) $| \dot { \rho } ( t ) | \leq \dot { \bar { \rho } }$ for some $\dot { \overline { { \rho } } } < \infty$ .

One conventional choice of prescribed performance function is the decreasing exponential function

$$\rho (t) = (\rho (0) - \rho (\infty)) e ^ {- l t} + \rho (\infty), \tag {6}$$

where $\rho ( 0 )$ and $\rho ( \infty )$ denote the initial and steady-state values, and $l > 0$ specifies the decay rate.
