For an undirected graph $\mathcal { G } ( t )$ , both $\mathcal { L } ( t )$ and $\mathcal { Q } ( t )$ are positive semi-definite matrices. A union graph denoted $\cup _ { \tau \in [ t , t + T ] } \mathcal { G } ( \tau )$ is the graph formed by Adjacency matrix elements, $\bar { a } _ { i j } ( t ) \triangleq$ $\int _ { t } ^ { t + T } a _ { i j } ( \tau )$ dτ . The union graph, as defined, is the graph obtained by collecting all the edges in the sub-graphs appearing over a time-interval $[ t , t + T ]$ .

Definition 1. [38] For a time-varying graph $\mathcal { G } ( t )$ with adjacency matrix elements $a _ { i j } ( t )$ , the weighted incidence matrix H : R+ → Rn× n(n−1)2 $\mathcal { H } : \dot { \mathbb { R } } ^ { + }  \mathbb { R } ^ { n \times \frac { n ( n - 1 ) } { 2 } }$ is defined as,

$$
\mathcal {H} (t) \triangleq h _ {i j} (t) \triangleq \left\{ \begin{array}{l l} \sqrt {a _ {i j} (t)}, & \text { if } e _ {j} = (i, j) \\ - \sqrt {a _ {i j} (t)}, & \text { if } e _ {j} = (j, i) \\ 0, & \text { otherwise } \end{array} \right.
$$

Remark II.1. In the aforementioned definition, for undirected graphs, it is standard practice to choose an arbitrary orientation (information flow direction). This has no effect on the graph Laplacian and can always be computed as, $\mathcal { L } ( t ) = \mathcal { H } ( t ) \mathcal { H } ^ { T } ( t )$ .

Definition 2. [39] At any given time $^ { \cdot } t ^ { \cdot } ,$ an undirected graph $\mathcal { G } ( t )$ is called bipartite if there exists a disjoint partition of the node set denoted as $\mathcal { V } = \mathcal { V } _ { + } ( t ) \cup \mathcal { V } _ { - } ( t )$ such that all edges in $\mathcal { G } ( t )$ are between the node sets, and there are no edges within the node set. Mathematically, for all $( i , j ) \in \mathcal { E } ( t )$ , $i \in \mathcal { V } _ { k } ( t ) \implies j \in \mathcal { V } \setminus \mathcal { V } _ { k } ( t )$ for $k \in \{ + , - \}$ . A graph is called jointly (non-)bipartite over [t, t+T ] if the corresponding union graph $\cup _ { \tau \in [ t , t + T ] } \mathcal { G } ( \tau )$ is (non-)bipartite.

Remark II.2. The above definition implies that the graph need not necessarily be (non-)bipartite for all time instants between $[ t , t + T ]$ , but the graph obtained by collecting all the edges in the sub-graphs appearing over the time interval is (non-)bipartite.
