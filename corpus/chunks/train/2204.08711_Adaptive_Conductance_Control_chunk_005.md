# 2.1. Conductance-based model of a neuron

A conductance-based model of a neuron is a one-port electrical circuit with the architecture shown in Figure 1: a capacitor of capacitance $c > 0$ in parallel with a leak current $I _ { \mathrm { l e a k } }$ and several intrinsic ionic currents. The capacitance and leak current model the neuron’s cell membrane, and the ionic currents model the flow of ions across this membrane. Additional extrinsic currents model the synaptic currents $I _ { \mathrm { s y n , p } } ,$ due to interconnections with other neurons, as well as an input (or applied ) current u. The latter represents the current injected into the circuit by an intracellular electrode. The capacitor voltage v, which models the neuronal membrane potential, evolves according to Kirchhoff’s law, that is

$$c \dot {v} = - I _ {\text { leak }} - \sum_ {\text { ion } \in \mathcal {I}} I _ {\text { ion }} - \sum_ {\text { syn } \in \mathcal {S}} \sum_ {\text { p } \in \mathcal {P}} I _ {\text { syn,p }} + u \tag {1a}= - I _ {\text { leak }} - I _ {\text { int }} - I _ {\text { ext }} + u \tag {1b}$$

where $I _ { \mathrm { i n t } }$ is the parallel combination of intrinsic ionic currents, $I _ { \mathrm { e x t } }$ is the parallel combination of synaptic currents, I is the index set of intrinsic ionic currents, S is the index set of synaptic neurotransmitter types, and $\mathcal { P }$ is the index set of presynaptic neurons.

Each current in the circuit is ohmic in nature, but with a conductance that can be nonlinear and voltage-dependent. The leak current has a constant conductance and is given by

$$I _ {\mathrm{leak}} = \mu_ {\mathrm{leak}} (v - E _ {\mathrm{leak}}),$$

whereas the intrinsic ionic currents are modelled by

$$I _ {\mathrm{ion}} = \mu_ {\mathrm{ion}} m _ {\mathrm{ion}} ^ {p _ {\mathrm{ion}}} h _ {\mathrm{ion}} ^ {q _ {\mathrm{ion}}} (v - E _ {\mathrm{ion}}) \tag {2a}\tau_ {m, \text { ion }} (v) \dot {m} _ {\text { ion }} = - m _ {\text { ion }} + \sigma_ {m, \text { ion }} (v) \tag {2b}\tau_ {h, \text { ion }} (v) \dot {h} _ {\text { ion }} = - h _ {\text { ion }} + \sigma_ {h, \text { ion }} (v) \tag {2c}$$
