Let $\gamma _ { i } : = \alpha _ { i } ^ { - 1 } \circ \sigma _ { i }$ and $\chi _ { i } : = \alpha _ { i } ^ { - 1 } \circ \hat { \chi } _ { i }$ for $i = 1 , 2$ . Then, we obtain:

$$V (x) \geq \max \left\{\gamma_ {1} (W (y)), \chi_ {1} (| u _ {1} |) \right\} \Rightarrow \dot {V} (x) \leq - \alpha_ {1} (V (x))W (y) \geq \max \left\{\gamma_ {2} (V (x)), \chi_ {2} (| u _ {2} |) \right\} \Rightarrow \dot {W} (y) \leq - \alpha_ {2} (W (y)).$$

By Lemma 10 in the Appendix, there exists $C > 0$ such that the inequality

$$\frac {(n _ {1} + 1) \kappa_ {1} ^ {2} \varepsilon_ {1} ^ {2}}{a _ {1}} \max _ {j} \left(\frac {(n _ {2} + 1) \kappa_ {2} ^ {2} \varepsilon_ {2} ^ {2}}{a _ {2}}\right) ^ {\eta_ {2, j}} < C, \tag {18}$$

implies the existence of $\hat { \gamma } _ { 1 } , \hat { \gamma } _ { 2 } \in \mathcal { K } _ { \infty } ^ { P } \cup \mathcal { K } _ { \infty } ^ { - P }$ such that $\hat { \gamma } _ { i } ( s ) >$ $\gamma _ { i } ( s )$ and $\hat { \gamma } _ { i } \circ \hat { \gamma } _ { 3 - i } ( s ) < s$ for all $s > 0$ and $i = 1 , 2$ . Inequality (18) holds if $\varepsilon _ { 1 } , \varepsilon _ { 2 }$ satisfy the following condition:

$$| \varepsilon_ {1} | \max _ {j} | \varepsilon_ {2} | ^ {\eta_ {2, j}} < \sqrt {\frac {C a _ {1}}{(n _ {1} + 1) \kappa_ {1} ^ {2}} \min _ {j} \left(\frac {a _ {2}}{(n _ {2} + 1) \kappa_ {2} ^ {2}}\right) ^ {\eta_ {2 , j}}} := \varepsilon .$$
