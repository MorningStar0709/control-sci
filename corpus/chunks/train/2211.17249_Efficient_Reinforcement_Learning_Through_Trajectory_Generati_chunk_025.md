Output-feedback control in a power distribution network. We further conduct experiments on the voltage control problem in IEEE 33bus test feeder (Baran and Wu, 1989; Cui et al., 2022b), where ${ \pmb x } ( t ) \in \mathbb { R } ^ { 3 2 }$ . We assume only 20 buses are measured and controlled, so $\pmb { y } ( t ) , \pmb { u } ( t ) \in \mathbb { R } ^ { 2 0 }$ . The observability matrix $\mathcal { O } _ { [ 0 , T _ { 0 } ] } ( A , C )$ becomes full column rank when $T _ { 0 } = 3 .$ . The time horizon of trajectory is $K = 2 0$ . According to the trajectory generation algorithm developed in Subsection 4.2, we setup $T = K { + } T _ { 0 } { - } 1 = 2 2$ and collect historic trajectory of length $L = ( m { + } 1 ) T { - } 1 { + } n = 4 9 3$ . With a sampling period of 1s, data collection takes 493s. Figure $2 ( \mathrm { a } ) \mathrm { - } ( \mathrm { c } )$ compares the training loss, testing loss and the number of samples, respectively. PG-TrajectoryGen achieves the same training and testing loss as PG-Sample-1000 with much smaller number of samples.
