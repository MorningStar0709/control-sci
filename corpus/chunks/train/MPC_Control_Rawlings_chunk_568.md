# Exercise 4.10: Relationships between observability, FSO, MHE detectability and i-IOSS

Show that for the nonlinear system $x ^ { + } = f ( x , w ) , y = h ( x )$ with Lipschitz continuous $f$ and h, the following relationships hold between observability, FSO, MHE detectability, and i-IOSS (detectability).

$$\text { observable } \implies \text { FSO } \implies \text { MHE detectable } \implies \text { i - IOSS }\text { observable } \Leftarrow \text { FSO } \Leftarrow \text { MHE detectable } \Leftarrow \text { i - IOSS }$$
