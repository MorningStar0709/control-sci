# Algorithm 1. Computation of a maximum hands-off hybrid control sequence for the switched system (2.1)

Step I. Let $\begin{array} { r } { W _ { T } ^ { * } \ = \ \overline { { v } } _ { 0 } , ( \overline { { v } } _ { 0 } , \overline { { v } } _ { 1 } ) , \ \overline { { v } } _ { 1 } , ( \overline { { v } } _ { 1 } , \overline { { v } } _ { 2 } ) , \overline { { v } } _ { 2 } , \ldots , \overline { { v } } _ { r - 1 } , ( \overline { { v } } _ { r - 1 } , \overline { { v } } _ { r } ) , \overline { { v } } _ { r } \ \in \ \mathcal { W } _ { T } ( \mathcal { X } _ { s } , \mathcal { X } _ { 0 } ) } \end{array}$ b e a ξ−hands-off walk on $G ( V , E )$ .

Step II. Construct a hybrid control sequence, $( \overline { { \nu } } _ { T } , \overline { { \mu } } _ { T } )$ , as follows:

Case I. $\begin{array} { r } { \mathrm { I f } \ r = T . } \end{array}$ , then construct the discrete control sequence as:

$$\overline {{\nu}} (t) = \alpha (\overline {{v}} _ {t}, \overline {{v}} _ {t + 1}), t = 0, 1, \ldots , r - 1,$$

and construct the continuous control sequence as:

$$\overline {{\mu}} (t) = \beta (\overline {{v}} _ {t}, \overline {{v}} _ {t + 1}), t = 0, 1, \ldots , r - 1.$$

Case II. I $\ : r < T \ :$ , then construct the discrete control sequence as:

$$\overline {{\nu}} (t) = \alpha (\overline {{v}} _ {t}, \overline {{v}} _ {t + 1}), t = 0, 1, \dots , r - 1$$

and

$$
\overline {{\nu}} (t) = \left\{ \begin{array}{l l} \overline {{\nu}} (t - 1), & \text {if (\overline {{\nu}} (t - 1), \overline {{\nu}} (t - 1)) \in \mathcal {E} (\mathcal {N}),} \\ i \in \mathcal {N} _ {s}, & \text if (\overline {{\nu}} (t - 1), \overline {{\nu}} (t - 1)) \notin \mathcal {E} (\mathcal {N}), i \in \mathcal {N} _ {s} \text {and (\overline {{\nu}} (t - 1), i) \in \mathcal {E} (\mathcal {N}),} \\ j \in \mathcal {N}, & \text {if (\overline {{\nu}} (t - 1), \overline {{\nu}} (t - 1)) \notin \mathcal {E} (\mathcal {N}), (\overline {{\nu}} (t - 1), i) \notin \mathcal {E} (\mathcal {N}) \text {for all} i \in \mathcal {N} _ {s}} \\ & \text {and (\overline {{\nu}} (t - 1), j) \in \mathcal {E} (\mathcal {N}),} \end{array} \right.
$$

$t = r , r + 1 , \ldots , T - 1$ , and construct the continuous control sequence as:

$$\overline {{\mu}} (t) = \beta (\overline {{v}} _ {t}, \overline {{v}} _ {t + 1}), t = 0, 1, \dots , r - 1$$

and

$$\overline {{\mu}} (t) = 0, t = r, r + 1, \dots , T - 1.$$
