# 2. 模糊逻辑控制的位置跟踪

被控对象为

$$G (s) = \frac {4 0 0}{s ^ {2} + 5 0 0 s}$$

按模糊控制器的设计步骤进行设计,模糊系统输入取跟踪误差及其变化率,模糊系统的输出取控制输入,其中输入变量和输出变量的变化范围是模糊系统设计的关键。

位置指令取正弦信号 $0.5\sin (10t)$ ，根据实际的输入、输出值的变化范围，跟踪误差及其变化率变化范围取 $[-0.3, 0.3]$ ，控制输入的变化范围取 $[-30, 30]$ 。

首先运行模糊控制器程序 chap4\_2.m, 可得到模糊系统输入和输出的隶属函数, 如图 4-10 和图 4-11 所示, 同时将模糊控制系统保存在文件 fuzz.fis 中。然后运行模糊控制的 Simulink 仿真程序, 仿真结果如图 4-12 所示。模糊控制位置跟踪的 Simulink 仿真程序见 chap4\_3sim.mdl。
