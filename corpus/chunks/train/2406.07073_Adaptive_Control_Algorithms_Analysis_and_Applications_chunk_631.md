# 13.3.2 Stability of the Injected System

It is shown that the injected system is a switching system containing only the stable systems constructed by the members of the multi-estimator block and their corresponding controllers in the multi-controller block. In this subsection we study the stability of a switching system containing n stable models in the state space. Consider the following models:

$$x (t + 1) = A _ {1} x (t), \qquad x (t + 1) = A _ {2} x (t), \qquad \ldots , \qquad x (t + 1) = A _ {n} x (t)$$
