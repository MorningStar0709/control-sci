# III. EDMD ON NOISY DATA

For the current work, we assume that the noisy measurement of state is $\hat { x } _ { t } = x _ { t } + n _ { t }$ with small perturbations $n _ { t } .$ , while the measurement errors of control inputs are not considered. The Koopman operator $\psi ( \cdot )$ operates on the data of the perturbed states, rendering the perturbed lifted data-matrix $\hat { \mathbf { X } } _ { \mathrm { l i f t } }$ . Based on the corrupted measurement data, the expressions for the lifted state matrix can be written as:

$$\hat {\mathbf {X}} _ {\text { lift }} ^ {1} = \left[ \psi (x _ {0} + n _ {0}), \dots , \psi (x _ {N - 1} + n _ {N - 1}) \right] \tag {13}\hat {\mathbf {X}} _ {\text {lift}} ^ {2} = \left[ \psi (x _ {1} + n _ {1}), \dots , \psi (x _ {N} + n _ {N}) \right] \tag {14}\hat {\mathbf {Y}} = \left[ y _ {1} + m _ {1}, y _ {2} + m _ {2}, \dots , y _ {N} + m _ {N} \right]. \tag {15}$$

Thus, the perturbed least-square regression for Koopman model identification produces $( A , B _ { 2 } , C _ { 2 } )$ , which will be used for controller design. Note that, $( A , B _ { 2 } )$ are the perturbed transition matrices from the nominal matrices $( A _ { p } , B _ { p } , C _ { 2 } )$ . The least-square problem formulation can be written as

$$\min _ {A, B _ {2}} \left| \left| \hat {\mathbf {X}} _ {\text { lift }} ^ {2} - A \hat {\mathbf {X}} _ {\text { lift }} ^ {1} - B _ {2} \mathbf {U} \right| \right| _ {F} \tag {16}\min _ {C _ {2}} \left| \left| \hat {\mathbf {Y}} - C _ {2} \mathbf {X} _ {\text { lift }} ^ {1} \right| \right| _ {F} \tag {17}$$

Assuming the selected basis function is continuously differentiable (for instance, polynomial, Gaussian, radial basis function kernel), then, using Taylor expansion, the observables with perturbed data can be approximated by its first-order equation as

$$\psi (x _ {k} + n _ {k}) \approx \psi (x _ {k}) + \frac {\partial \psi}{\partial x _ {k}} n _ {k} \tag {18}$$
