# C. System Description

In this work, we consider a multi-agent system with n first-order stochastic agents modeled by following stochastic differential equation:

$$\mathrm{d} x _ {i} = u _ {i} \mathrm{d} t + g (x _ {i}) \mathrm{d} W _ {t}, \quad i \in [ 1; n ], \tag {1}$$

where $x _ { i } \in \mathbb { R } , u _ { i } \in \mathbb { R }$ are the position and control input of the ith agent, respectively and $g : \mathbb { R }  \mathbb { R }$ is a Lipschitz continuous diffusion function with Lipschitz constant ${ \sf k } _ { g } \in$ $\mathbb { R } _ { 0 } ^ { + }$ such that: $\| g ( x ) - g ( x ^ { \prime } ) \| \leq \mathsf { k } _ { g } \| x - x ^ { \prime } \|$ for all $x , x ^ { \prime } \in \mathbb { R }$ . Note that for the sake of simplicity, here we consider onedimensional agents and the result can be extended to higher dimensions with the appropriate use of Kronecker product.

Let us consider that agents represent the vertices $\nu =$ $\{ 1 , 2 , \ldots , n \}$ of an undirected graph G. We assume that the communication graph is static, i.e., neighbouring agents ${ \mathcal { N } } _ { i }$ of agent $i \in [ 1 ; n ]$ do not vary over time and each agent is driven by a first-order consensus protocol with an external input $v _ { i } ~ \in ~ \mathbb { R }$ , i.e., $\begin{array} { r } { u _ { i } \ = \ - \sum _ { j \in \mathcal { N } _ { i } } ( x _ { i } - x _ { j } ) + v _ { i } } \end{array}$ , with corresponding stochastic differential equation given as:

$$\mathrm{d} x _ {i} = \left(- \sum_ {j \in \mathcal {N} _ {i}} (x _ {i} - x _ {j}) + v _ {i}\right) \mathrm{d} t + g (x _ {i}) \mathrm{d} W _ {t}, \quad i \in \mathcal {V}. \tag {2}$$

Let $x = [ x _ { 1 } , \ldots , x _ { n } ] ^ { T } \in \mathbb { R } ^ { n }$ and $v = [ v _ { 1 } , \ldots , v _ { n } ] ^ { T } \in \mathbb { R } ^ { n }$ be stack vectors of absolute positions and external inputs of all agents, respectively. Denote $\overline { { x } } = [ \overline { { x } } _ { 1 } , \ldots \overline { { x } } _ { m } ] ^ { T } \in \mathbb { R } ^ { m }$ the stack vector of relative positions between the pair of communicating agents $( i , j ) = e _ { k } \in \mathcal { E } .$ , where $\overline { { x } } _ { k } \triangleq x _ { i j } =$ $x _ { i } - x _ { j }$ such that $( i , j ) \ = \ e _ { k } \ \in \ \mathcal { E } , k \ \in \ [ 1 ; m ]$ . We also mention some interesting properties which are useful in the paper: $L x = D \overline { { x } } , \overline { { x } } = D ^ { T } x ,$ and if ${ \overline { { x } } } = 0 .$ we have $L x = 0$ . By stacking agents in (2), the dynamics of the stochastic multi-agent system is rewritten as:

$$\mathrm{d} x = (- L x + v) \mathrm{d} t + G (x) \mathrm{d} W _ {t}, \tag {3}$$
