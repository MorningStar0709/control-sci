Since unit quaternions do not form a vector space, they cannot be subtracted to form an orientation error; instead, we should use the properties of the quaternion group algebra. Let $\boldsymbol { e } _ { \zeta } = [ e _ { \varphi } , e _ { \epsilon } ^ { \top } ] ^ { \top } \in \mathbb { S } ^ { 3 }$ be the unit quaternion describing the orientation error. Then, it holds that [85],

$$
e _ {\zeta} := \zeta_ {\mathrm{d}} \cdot \zeta_ {o} ^ {+} = \left[ \begin{array}{c} \varphi_ {\mathrm{d}} \\ \epsilon_ {\mathrm{d}} \end{array} \right] \cdot \left[ \begin{array}{c} \varphi_ {o} \\ - \epsilon_ {o} \end{array} \right],
$$

and yields

$$
e _ {\zeta} = \left[ \begin{array}{c} e _ {\varphi} \\ e _ {\epsilon} \end{array} \right] := \left[ \begin{array}{c} \varphi_ {o} \varphi_ {\mathrm{d}} + \epsilon_ {o} ^ {\top} \epsilon_ {\mathrm{d}} \\ \varphi_ {o} \epsilon_ {\mathrm{d}} - \varphi_ {\mathrm{d}} \epsilon_ {o} + S (\epsilon_ {o}) \epsilon_ {\mathrm{d}} \end{array} \right].
$$

By employing the quaternion dynamics (see (2.7a)) and certain properties of skew-symmetric matrices [111], it can be shown that the error dynamics of $e _ { p } , e _ { \varphi }$ are:

$$\dot {e} _ {p} = \dot {p} _ {O} - \dot {p} _ {\mathrm{d}} \tag {2.17a}\dot {e} _ {\varphi} = \frac {1}{2} e _ {\epsilon} ^ {\top} e _ {\omega} \tag {2.17b}\dot {e} _ {\epsilon} = - \frac {1}{2} \left[ e _ {\varphi} I _ {3} + S (e _ {\epsilon}) \right] e _ {\omega} - S (e _ {\epsilon}) \omega_ {\mathrm{d}}, \tag {2.17c}$$

where $e _ { \omega } : = \omega _ { o } - \omega _ { \mathrm { d } }$ is the angular velocity error, with $\omega _ { \mathrm { d } } = 2 E ( \zeta _ { \mathrm { d } } ) ^ { \top } \dot { \zeta } _ { \mathrm { d } }$ , as indicated by (2.7b).

Due to the ambiguity of unit quaternions, when $\zeta _ { O } = \zeta _ { \mathrm { d } }$ , then $e _ { \zeta } =$ $[ 1 , 0 _ { 3 } ^ { \top } ] ^ { \top } \in \mathbb { S } ^ { 3 }$ . If $\zeta _ { o } = - \zeta _ { \mathrm { d } }$ , then $e _ { \zeta } = [ - 1 , 0 _ { 3 } ^ { \top } ] ^ { \top } \in \mathbb { S } ^ { 3 }$ , which, however, represents the same orientation. Therefore, the control objective established in Problem 2.1 is equivalent to

$$
\lim _ {t \to \infty} \left[ \begin{array}{c} e _ {p} (t) \\ | e _ {\varphi} (t) | \\ e _ {\epsilon} (t) \end{array} \right] = \left[ \begin{array}{c} 0 \\ 1 \\ 0 \end{array} \right].
$$
