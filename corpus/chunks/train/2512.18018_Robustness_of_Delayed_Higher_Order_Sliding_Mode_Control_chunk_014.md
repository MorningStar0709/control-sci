Theorem 9. Under conditions of Theorem 8, the system (2.1), (4.1) is ISS with respect to the input w, $\delta \in \mathcal { L } _ { \infty } ^ { n } , \delta ^ { \top } b \equiv 0$ uniformly in $d \in \mathcal { D }$ provided that $\chi > 1$ is chosen sufficiently close to 1.

Proof. Following Theorem 1, the function Q in (3.3) verifies all conditions to define an ILF. Indeed, the function Q is continuously differentiable for all $( V , x ) \in$ $\mathbb { R } _ { + } ^ { \star } \times \mathbb { R } ^ { n }$ , and for any $x \in \mathbb { R } ^ { n } \backslash \{ 0 \}$ there exists a solution $V ( x )$ such that $Q ( V ( x ) , x ) = 0$ [35] (as usual we complement this statement with $V ( 0 ) = 0 )$ . Moreover, the following chain of inequalities can be established:

$$\sqrt {\lambda_ {\min} (X)} \min \{V, V ^ {n} \} \leq | x | \leq \sqrt {\lambda_ {\max} (X)} \max \{V, V ^ {n} \}$$

which are satisfied for all $( V , x ) \in \mathbb { R } _ { + } ^ { \star } \times \mathbb { R } ^ { n }$ with $Q ( V , x ) = 0$ , and $X$ is a positive definite matrix due to LMIs (3.2). We can express the partial derivatives as follows:

$$
\begin{array}{l} \frac {\partial Q (V , x)}{\partial V} \\ = - 2 x ^ {\top} \left[ \operatorname{diag} \{(n - i + 1) V ^ {- n + i - 2} \} _ {i = 1} ^ {n} \right] X ^ {- 1} D _ {\mathbf {r}} (V ^ {- 1}) x \\ = - V ^ {- 1} x ^ {\top} D _ {\mathbf {r}} \left(V ^ {- 1}\right) \left(X ^ {- 1} G _ {\mathbf {r}} + G _ {\mathbf {r}} X ^ {- 1}\right) D _ {\mathbf {r}} \left(V ^ {- 1}\right) x \\ <   0 \\ \end{array}
$$

due to the imposed LMI $X G _ { \mathbf { r } } + G _ { \mathbf { r } } X \succ 0$ , and

$$\frac {\partial Q (V , x)}{\partial x} = 2 x ^ {\top} D _ {\mathbf {r}} (V ^ {- 1}) X ^ {- 1} D _ {\mathbf {r}} (V ^ {- 1}).$$

Recall the useful properties $D _ { \mathbf { r } } ( s ) A = s A D _ { \mathbf { r } } ( s )$ and $D _ { \mathbf { r } } ( s ) b = s b$ that are satisfied by construction.
