# C.5 Proof of Lemma 4.5

Consider a single term $\left\| y _ { t } - { \hat { y } } _ { t } ^ { \mathrm { T K } } \right\| ^ { 2 } - \left\| y _ { t } - { \hat { y } } _ { t } ^ { \mathrm { S K } } \right\| ^ { 2 }$ for any $t \in \{ 0 , \ldots , T - 1 \}$ . We first recall from Lemma 3.2 that $\| y _ { t } \| \le \dot { R } _ { y }$ (under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v } )$ . We then get from (17) that

$$\left| \left| y _ {t} ^ {\mathrm{TK}} \right| \right| = \left\| \sum_ {s = 1} ^ {h} N _ {s} ^ {\star} y _ {t - s} \right\| \leq \left\| Y _ {t - 1: t - h} \right\| \left\| N ^ {\star} \right\|\leq \sqrt {h} R _ {y} R _ {N},$$

where we used the upper bounds $\| Y _ { t - 1 : t - h } \| \leq \sqrt { h } R _ { y }$ and $\lVert N ^ { \star } \rVert \leq R _ { N }$ shown in Lemmas 3.2 and 3.4, respectively. Moreover, recalling (16), we also have

$$\left\| \hat {y} _ {t} ^ {\mathrm{SK}} \right\| = \left\| \sum_ {s = 1} ^ {t} N _ {s} ^ {\star} y _ {t - s} \right\| \leq R _ {y} \| L \| \| C \| \frac {\kappa_ {F}}{1 - \gamma_ {F}}, \tag {C.9}$$

where the inequality follows from Lemma 2.2. Furthermore, we obtain that

$$\left\| \hat {y} _ {t} ^ {\mathsf {S K}} - \hat {y} _ {t} ^ {\mathsf {T K}} \right\| = \left\| \sum_ {s = h + 1} ^ {t} N _ {s} ^ {\star} y _ {t - s} \right\|\leq R _ {y} \left\| L \right\| \left\| C \right\| \frac {\kappa_ {F} \gamma_ {F} ^ {h}}{1 - \gamma_ {F}}.$$

Combining the above arguments, we deduce that

$$
\begin{array}{l} \left\| y _ {t} - \hat {y} _ {t} ^ {\mathrm{TK}} \right\| ^ {2} - \left\| y _ {t} - \hat {y} _ {t} ^ {\mathrm{SK}} \right\| ^ {2} = \left(y _ {t} - \hat {y} _ {t} ^ {\mathrm{TK}} - y _ {t} + \hat {y} _ {t} ^ {\mathrm{SK}}\right) ^ {\top} \left(2 y _ {t} - \hat {y} _ {t} ^ {\mathrm{TK}} - \hat {y} _ {t} ^ {\mathrm{SK}}\right) \\ \leq \left\| \hat {y} _ {t} ^ {\mathrm{SK}} - \hat {y} _ {t} ^ {\mathrm{TK}} \right\| \left\| 2 y _ {t} - \hat {y} _ {t} ^ {\mathrm{TK}} - \hat {y} _ {t} ^ {\mathrm{SK}} \right\| \\ \leq \left(2 R _ {y} + \sqrt {h} R _ {y} R _ {N} + R _ {y} \| L \| \| C \| \frac {\kappa_ {F}}{1 - \gamma_ {F}}\right) \left\| \hat {y} _ {t} ^ {\mathrm{SK}} - \hat {y} _ {t} ^ {\mathrm{TK}} \right\| \\ \leq \Big (2 + \sqrt {h} R _ {N} + \| L \| \| C \| \frac {\kappa_ {F}}{1 - \gamma_ {F}} \Big) \kappa_ {F} \| L \| \| C \| \frac {R _ {y} ^ {2} \gamma_ {F} ^ {h}}{1 - \gamma_ {F}}. \\ \end{array}
$$

Now, since we set $h = \lfloor \log T / \log ( 1 / \gamma _ { F } ) \rfloor$ , it holds that $\gamma _ { F } ^ { h } \leq 1 / T$ . Summing over $t = 0 , \ldots , T - 1$ yields (34). ■
