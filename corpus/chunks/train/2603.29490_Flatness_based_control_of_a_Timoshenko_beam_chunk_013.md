# IV. HYPERBOLIC CONTROLLER FORM (HCF)

To obtain the HCF for the Timoshenko beam based on the input parametrization (17), we define the HCF state

$$\eta_ {1} (\tau , t) = y _ {1} (t + \tau), \quad \tau \in [ 0, 2 \tau_ {1} ] \tag {18a}\eta_ {2} (\tau , t) = y _ {2} (t + \tau), \quad \tau \in [ 0, 2 \tau_ {2} ] \tag {18b}$$

in view of (16) and (17). Then, the components $\eta _ { i } ( \tau , t ) , i =$ 1, 2, satisfy homogeneous transport equations $\partial _ { t } \eta _ { i } ( \tau , t ) =$ $\partial _ { \tau } \eta _ { i } ( \tau , t )$ with normalized, negative transport velocities on domains $\tau \in [ 0 , 2 \tau _ { i } )$ . The respective outputs constitute the flat output components, i.e., $\eta _ { i } ( 0 , t ) = y _ { i } ( t )$ . The BCs of the HCF are obtained from the input parametrization (17), where expressions $y _ { i } ( t + \tau ) , i = 1 , 2$ , for $\tau \in [ 0 , 2 \tau _ { i } ]$ can be replaced by the HCF state in (18). However, for values $y _ { 1 } ( t +$ τ) with prediction amplitudes $\tau > 2 \tau _ { 1 }$ , this substitution is no longer valid. Instead, applying a forward time shift to (17a) reveals that these values can be replaced by predictions of $u _ { 1 } ( t )$ . Hence, using the substitution

$$
y _ {1} (t + \tau) = \left\{ \begin{array}{l l} \eta_ {1} (\tau , t), & \tau \in [ 0, 2 \tau_ {1} ] \\ - \bar {u} _ {1} (t + \tau - 2 \tau_ {1}), & \tau > 2 \tau_ {1} \end{array} \right. \tag {19a}
y _ {2} (t + \tau) = \eta_ {2} (\tau , t), \quad \tau \in [ 0, 2 \tau_ {2} ] \tag {19b}$$

in (17) and subsequently solving for the boundary values $\eta _ { i } ( 2 \tau _ { i } , t )$ provides the BCs of the HCF
