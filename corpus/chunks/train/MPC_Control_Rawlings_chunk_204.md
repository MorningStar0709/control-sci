Assumption 2.2 (Continuity of system and cost). The functions $f :$ $\mathbb { X } \times \mathbb { U } \to \mathbb { R } ^ { n } , \ell : \mathbb { X } \times \mathbb { U } \to \mathbb { R } _ { \ge 0 }$ and $V _ { f } : \mathbb { X } _ { f } \ \to \ \mathbb { R } _ { \ge 0 }$ are continuous, $f ( 0 , 0 ) = 0 , \ell ( 0 , 0 ) = 0$ and $V _ { f } ( 0 ) = 0$ .

Assumption 2.3 (Properties of constraint sets). The sets X and $\mathbb { X } _ { f }$ are closed, $\mathbb { X } _ { f } \subseteq \mathbb { X }$ and U are compact; each set contains the origin.

The sets U, X and $\mathbb { X } _ { f }$ are assumed to contain the origin because the first problem we tackle is regulation to the origin. This assumption is modified when we consider the tracking problem.

Proposition 2.4 (Existence of solution to optimal control problem). Suppose Assumptions 2.2 and 2.3 hold. Then

(a) The function $V _ { N } ( \cdot )$ is continuous in $\mathbb { Z } _ { N }$ .

(b) For each $\boldsymbol { x } \in \mathcal { X } _ { N }$ , the control constraint set $\mathcal { U } _ { N } ( x )$ is compact.

(c) For each $\boldsymbol { x } \in \mathcal { X } _ { N }$ , a solution to $\mathbb { P } _ { N } ( x )$ exists.
