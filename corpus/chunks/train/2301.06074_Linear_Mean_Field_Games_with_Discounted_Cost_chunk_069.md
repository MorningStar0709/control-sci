$$\zeta^ {\pi} (C) := (1 - \beta) \sum_ {t = 0} ^ {\infty} \beta^ {t} \operatorname * {P r} \left[ (x (t), a (t)) \in C \right], C \in \mathcal {B} (\mathrm{X} \times \mathrm{A})$$

Note that $\zeta ^ { \pi }$ is a probability measure on $\mathsf { X } \times \mathsf { A }$ as a result of the normalizing constant $( 1 - \beta )$ . One can prove that $\zeta ^ { \pi }$ satisfies

$$\hat {\zeta} ^ {\pi} (\cdot) = (1 - \beta) \mu (\cdot) + \beta \int_ {X \times A} p _ {\mu} (\cdot | x, a) \zeta^ {\pi} (d x, d a) \tag {58}$$

Conversely, if any finite measure $\zeta$ satisfies (58), then it is a β-discount expected occupation measure of some policy π [19, Lemma 3.3]. Using the β-discount expected occupation measure, we can write

$$J (\mu ; \pi) = \frac {1}{(1 - \beta)} \left\langle \zeta^ {\pi}, c _ {\mu} \right\rangle$$

Therefore, since $( 1 - \beta )$ is just a constant, $\mathrm { M D P } _ { \mu }$ is equivalent to the above linear program. Using LP formulation, we first establish the following result.

LEMMA 5.1. Let $( \zeta ^ { * } , \mu ^ { * } ) \in \mathcal { M } ( \mathsf { X } \times \mathsf { A } ) _ { + } \times \mathcal { M } ( \mathsf { X } ) _ { + }$ be a pair with the following properties

(a) Given $\mu ^ { * } , \zeta ^ { * }$ is the optimal solution to the above LP formulation of $M D P _ { \mu ^ { * } }$   
(b) Given $\zeta ^ { * } , \mu ^ { * }$ satisfies the following linear equation

$$\mu^ {*} (\cdot) = \int_ {X \times A} p _ {\mu^ {*}} (\cdot | x, a) \zeta^ {*} (d x, d a)$$

If we disintegrate $\zeta ^ { * }$ as follows $\zeta ^ { * } ( d x , d a ) = \pi ^ { * } ( d a | x ) \hat { \zeta } ^ { * } ( d x )$ , then $( \mu ^ { * } , \pi ^ { * } )$ is $M F E .$ .

PROOF. Note that since $\mu ^ { * }$ and $\zeta ^ { * }$ are not assumed to be probability measures at the beginning, we need to establish this first. Since

$$\hat {\zeta} ^ {*} = (1 - \beta) \mu^ {*} + \beta \zeta^ {*} p _ {\mu^ {*}}$$

we have $\varsigma ^ { * } ( \mathsf { X } \times \mathsf { A } ) = ( 1 - \beta ) \mu ^ { * } ( \mathsf { X } ) + \beta \zeta ^ { * } ( \mathsf { X } \times \mathsf { A } ) \mu ^ { * } ( \mathsf { X } )$ . Similarly, since

$$\mu^ {*} = \zeta^ {*} p _ {\mu^ {*}}$$

we have $\mu ^ { * } ( \mathsf { X } ) = \zeta ^ { * } ( \mathsf { X } \times \mathsf { A } ) \mu ^ { * } ( \mathsf { X } )$ , which implies that $\zeta ^ { * }$ is a probability measure. In view of this and using the first identity, we obtain the following

$$1 = (1 - \beta) \mu^ {*} (\mathsf {X}) + \beta \mu^ {*} (\mathsf {X}) = \mu^ {*} (\mathsf {X})$$

that is, $\mu ^ { * }$ is also a probability measure.
