With this translational motion control scheme, a desired control force vector $\varphi = f R \mathbf { e } _ { 3 } \in \mathbb { R } ^ { 3 }$ is generated. With $f R { \bf { e } } _ { 3 }$ obtained in this manner, the methodology utilized in §3.3 of [27] can be employed to generate the desired (reference) attitude profile to be tracked by the attitude control system, which is described in the following subsection. The term $r _ { 3 d }$ in [27], which denotes the third column of the rotation matrix, is re-defined here as $r _ { 3 d } : = \varphi / | | \varphi | |$ .
