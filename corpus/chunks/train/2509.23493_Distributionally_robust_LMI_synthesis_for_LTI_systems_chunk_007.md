# IV. PRELIMINARIES

In this article, we make use of well-known results on the 2-Wasserstein metric and its relation to the second moment matrices of random variables.

Definition $4 . l \colon$ The 2-Wasserstein metric between probability distributions $\mathbb { P } _ { 1 }$ and $\mathbb { P } _ { 2 }$ on $\mathbb { R } ^ { d }$ is defined as

$$\mathbb {W} \left(\mathbb {P} _ {1}, \mathbb {P} _ {2}\right) := \inf _ {\pi \in \Pi \left(\mathbb {P} _ {1}, \mathbb {P} _ {2}\right)} \left(\int_ {\mathbb {R} ^ {d} \times \mathbb {R} ^ {d}} \| \xi_ {1} - \xi_ {2} \| ^ {2} d \pi \left(\xi_ {1}, \xi_ {2}\right)\right) ^ {\frac {1}{2}},$$

where $\Pi ( \mathbb { P } _ { 1 } , \mathbb { P } _ { 2 } )$ is the set of all joint distributions of the random variables $\xi _ { 1 } , \xi _ { 2 }$ with marginal distributions $\mathbb { P } _ { 1 } , \mathbb { P } _ { 2 }$ .

In [4], the Gelbrich distance between probability distributions is introduced. It is a lower bound on the 2-Wasserstein metric and enjoys the following properties.

Theorem $4 . 2 \ ( I 4 ,$ Theorem $2 . I$ and Proposition 2.2]): For two distributions $\mathbb { P } _ { 1 } , \ \mathbb { P } _ { 2 }$ on $\mathbb { R } ^ { d }$ with mean vectors $\mu _ { 1 }$ and µ2 and covariance matrices $\Sigma _ { 1 }$ and $\Sigma _ { 2 }$ , respectively, the following holds:

1) $\mathbb { G } ( \mathbb { P } _ { 1 } , \mathbb { P } _ { 2 } ) \le \mathbb { W } ( \mathbb { P } _ { 1 } , \mathbb { P } _ { 2 } )$ , where $\mathbb { G } ( \mathbb { P } _ { 1 } , \mathbb { P } _ { 2 } )$ denotes the Gelbrich distance between $\mathbb { P } _ { 1 } , \mathbb { P } _ { 2 }$ and is defined as

$$\mathbb {G} \big (\mathbb {P} _ {1}, \mathbb {P} _ {2} \big) ^ {2} := \left\| \mu_ {1} - \mu_ {2} \right\| ^ {2} + \mathrm{tr} \Big (\Sigma_ {1} + \Sigma_ {2} - 2 \Big (\Sigma_ {1} ^ {\frac {1}{2}} \Sigma_ {2} \Sigma_ {1} ^ {\frac {1}{2}} \Big) ^ {\frac {1}{2}} \Big).$$

This bound is exact if $\mathbb { P } _ { 1 } , \mathbb { P } _ { 2 }$ are normal distributions.

2) If $\mu _ { 1 } = \mu _ { 2 }$ , the Gelbrich distance $\mathbb { G } ( \mathbb { P } _ { 1 } , \mathbb { P } _ { 2 } )$ is equal to the optimal value of the optimization problem

$$
\begin{array}{l} \underset {\overline {{\Sigma}} \in \mathbb {R} ^ {d \times d}} {\text { minimize }} \operatorname{tr} \left(\Sigma_ {1} + \Sigma_ {2} - 2 \overline {{\Sigma}}\right) ^ {\frac {1}{2}} \\ \text { s.t. } \left( \begin{array}{c c} \Sigma_ {1} & \overline {{\Sigma}} \\ \overline {{\Sigma}} ^ {\top} & \Sigma_ {2} \end{array} \right) \succeq 0. \\ \end{array}
$$

3) If $\Sigma _ { 1 } \succ 0$ and $\mathbb { P } _ { 1 } , \mathbb { P } _ { 2 }$ are normal distributions with zero mean, then there exists an optimal linear transport plan $\Delta \in \mathbb { R } ^ { d \times d }$ and two random variables $\xi _ { 1 } \sim \mathbb { P } _ { 1 } , \xi _ { 2 } \sim \mathbb { P } _ { 2 }$ such that $\xi _ { 2 } = \Delta \xi _ { 1 }$ and

$$\mathbb {W} \left(\mathbb {P} _ {1}, \mathbb {P} _ {2}\right) = \mathbb {G} \left(\mathbb {P} _ {1}, \mathbb {P} _ {2}\right) = \sqrt {\operatorname{tr} \left((I - \Delta) \Sigma_ {1} (I - \Delta) ^ {\top}\right)}.$$
