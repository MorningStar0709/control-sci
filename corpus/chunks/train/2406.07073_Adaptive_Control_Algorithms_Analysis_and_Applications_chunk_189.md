$$
\begin{array}{l} \hat {\theta} (t + N + 1) = \hat {\theta} (t) + \sum_ {i = 0} ^ {N} \frac {1}{t + i} \phi (t + i) \underline {{\varepsilon}} (t + 1 + i) \\ + \sum_ {i = 0} ^ {N} \frac {1}{t + i} \phi (t + i) w (t + 1 + i) \tag {4.20} \\ \end{array}
$$

Since $t \gg N$ and $t \gg 1$ , the various gains $\textstyle { \frac { 1 } { t + i } }$ for $i = 0 , 1 , \ldots , N$ , can be approximated by their mean value:

$$\frac {1}{N + 1} \left(\sum_ {i = 0} ^ {N} \frac {1}{t + i}\right)$$

and the above expression for $\hat { \theta } ( t + N + 1 )$ can be approached by:

$$
\begin{array}{l} \hat {\theta} (t + N + 1) \approx \hat {\theta} (t) + \left(\sum_ {i = 0} ^ {N} \frac {1}{t + i}\right) \frac {1}{N + 1} \sum_ {i = 0} ^ {N} \phi (t + i) \underline {{\varepsilon}} (t + 1 + i) \\ + \left(\sum_ {i = 0} ^ {N} \frac {1}{t + i}\right) \frac {1}{N + 1} \sum_ {i = 0} ^ {N} \phi (t + i) w (t + 1 + i) \tag {4.21} \\ \end{array}
$$

Now, since $N \gg 1$ and taking into account that in this example $\phi ( t )$ and $w ( t + 1 )$ do not depend upon ${ \hat { \theta } } ( t )$ , one has:

$$\frac {1}{N + 1} \sum_ {i = 0} ^ {N} \phi (t + i) w (t + 1 + i) \approx \mathbf {E} \{\phi (t + i) w (t + 1 + i) \} \tag {4.22}$$

(under the ergodicity assumption). From (4.22), one concludes that in order to obtain unbiased estimates, one should have ${ \bf E } \{ \phi ( t ) w ( t + 1 ) \} = 0$ . Taking into account how $y ( t )$ entering in $\phi ( t )$ is generated through (4.1) it results that in the case of an equation error recursive identification scheme (like RLS) unbiased estimates will be obtained only if $w ( t + 1 )$ is a stationary white noise sequence.

Let us now consider an output error recursive identification scheme using a parallel adjustable predictor and a parameter adaptation algorithm of the form of (4.17). The a posteriori output of the adjustable predictor will be given in this case by:

$$
\begin{array}{l} \hat {y} (t + 1) = - \sum_ {i = 1} ^ {n _ {A}} \hat {a} _ {i} (t + 1) \hat {y} (t + 1 - i) + \sum_ {i = 1} ^ {n _ {B}} \hat {b} _ {i} (t + 1) u (t + 1 - i) \\ = \hat {\theta} ^ {T} (t + 1) \phi (t) \tag {4.23} \\ \end{array}
$$

where:

$$\hat {\theta} (t) ^ {T} = [ \hat {a} _ {1} (t), \dots , \hat {a} _ {n _ {A}} (t), \hat {b} _ {1} (t), \dots , \hat {b} _ {n _ {B}} (t) ] \tag {4.24}$$

and:

$$\phi (t) = [ - \hat {y} (t), \dots , - \hat {y} (t - n _ {A} + 1), u (t), \dots , u (t - n _ {B} + 1) ] \tag {4.25}$$
