1) Loeb’s Stability Criterion: It is said that a limit cycle is stable if it returns to its original equilibrium state [28], while if its amplitude or frequency grows or decays until reaching another equilibrium state, it is considered unstable. Let $( A ^ { * } , \omega ^ { * } )$ be a solution to the HB equation (23), that is,

$$\mathbf {U} (A ^ {*}, \omega^ {*}) + j \mathbf {V} (A ^ {*}, \omega^ {*}) = 0, \tag {28}$$

where $\textbf { U } = \mathrm { \ R e } \left\{ N W + 1 \right\}$ and $\textbf { V } = \mathrm { I m } \left\{ N W + 1 \right\}$ . The stability of a predicted limit cycle in (27), under quasi-static perturbations4 around the equilibrium $( A ^ { * } , \omega ^ { * } )$ , is characterized by the following inequality

$$
\left. \left(\frac {\partial \mathbf {U}}{\partial A} \frac {\partial \mathbf {V}}{\partial \omega} - \frac {\partial \mathbf {U}}{\partial \omega} \frac {\partial \mathbf {V}}{\partial A}\right) \right| _ {\substack {A = A ^ {*} \\ \omega = \omega^ {*}}} > 0. \tag{29}
$$

This condition, known as Loeb’s criterion [43], provides a necessary requirement for orbital stability. A more recent study based on the dynamic harmonic balance [44] (DHB), has introduced a sufficient condition for the same purpose.

The functions $\mathbf { U } ( A , \omega )$ and $\mathbf { V } ( A , \omega )$ that define the HB equation (24) satisfy the following partial derivative relations

$$
\begin{array}{l} \frac {\partial \mathbf {U}}{\partial A} = - \frac {4 \rho}{\pi A ^ {2}}, \quad \frac {\partial \mathbf {V}}{\partial A} = 0, \\ \frac {\partial \mathbf {U}}{\partial \omega} = \frac {\mathrm{d}}{\mathrm{d} \omega} \operatorname{Re} \left\{\frac {1}{W (j \omega)} \right\}, \quad \frac {\partial \mathbf {V}}{\partial \omega} = \frac {\mathrm{d}}{\mathrm{d} \omega} \operatorname{Im} \left\{\frac {1}{W (j \omega)} \right\}. \\ \end{array}
$$
