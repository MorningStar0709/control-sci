# 5. Union graph and its controllablity

In this section, the controllability of the system is discussed from the perspective of the union graph, more specifically the relationship of controllability between union graph and system (2) is discussed, and a new criterion to identify the controllability of the union graph is obtained by using the concept of EP.

Definition 4. The union graph of a collection of graphs $G _ { m } = \left\{ V , E _ { m } , A _ { m } \right\} , m = 1 , \ldots , t$ , is denoted by $\tilde { G } \ =$ $\{ V , \tilde { E } , A \}$ , where $\textstyle { \tilde { E } } = \bigcup _ { m = 1 } ^ { t } E _ { m }$ , and $\begin{array} { r } { \tilde { \mathcal { A } } = \sum _ { m = 1 } ^ { t } \mathcal { A } _ { m } . } \end{array}$ .

Accordingly, union graphs can be described as

$$\dot {x} (t) = \tilde {L} ^ {*} x (k) + \tilde {M} y (k) \tag {4}$$

where

$$
\tilde {L} ^ {*} = \left(t \left[ \begin{array}{c c c c} A & 0 & \dots & 0 \\ 0 & A & \dots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & A \end{array} \right] - \left[ \begin{array}{c c c c} B K & 0 & \dots & 0 \\ 0 & B K & \dots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & B K \end{array} \right] L ^ {*}\right), \tilde {M} = \left(M \left[ \begin{array}{c c c c} C & 0 & \dots & 0 \\ 0 & C & \dots & 0 \\ \vdots & \vdots & \ddots & \vdots \\ 0 & 0 & \dots & C \end{array} \right]\right)
$$

$L ^ { * }$ is the Laplacian matrix of the union graph.

In order to discuss the relationship between the controllability of system (2) and (4), the following lemma is introduced

Lemma 4. System (2) is controllable if and only if the matrix

$$\tilde {W} _ {c} = \left[ \tilde {M}, \tilde {L} _ {1} \tilde {M}, \dots , \tilde {L} _ {t} \tilde {M}, \tilde {L} _ {1} ^ {2} \tilde {M}, \dots , \tilde {L} _ {1} ^ {d N - 1} \tilde {M}, \tilde {L} _ {1} ^ {d N - 1} \tilde {L} _ {2} \tilde {M}, \dots , \tilde {L} _ {t} ^ {d N - 1} \tilde {M} \right]$$

is full row rank.

PROOF. The proof is similar to Lemma 1 in Guan et al. (2019), thus there is omitted.

Theorem 4. System (2) is controllable if the system (4) is controllable

PROOF. From the Kalman rank criterion, Tthe following matrix is full row rank

$$\left[ \tilde {M}, \tilde {L} ^ {*} \tilde {M}, \dots , \tilde {L} ^ {* d N - 1} \tilde {M} \right]$$

which is expanded by using the properties of the union graph

$$\left[ \tilde {M}, \left(\tilde {L} _ {1} + \ldots + \tilde {L} _ {t}\right) ^ {*} \tilde {M}, \ldots , \left(\tilde {L} _ {1} + \ldots + \tilde {L} _ {t}\right) ^ {* d N - 1} \tilde {M} \right]$$
