# C.7 Doubling Trick to Remove the Knowledge of T in Algorithm

Theorem 4.1 shows that Algorithm 1 achieves $\mathcal { O } ( \log ^ { 4 } T )$ regret when the horizon length $T$ is known a priori. We now argue that using a doubling trick, one can achieve the same regret without the knowledge of $T .$ The arguments follow by dividing the total horizon $T$ into M consecutive intervals with length $L _ { i }$ for $i ~ \geq ~ 0$ , where $L _ { i } ~ = ~ T _ { i } - T _ { i - 1 }$ with $T _ { i } ~ = ~ 2 ^ { 2 ^ { i } }$ and $T _ { - 1 } = 0$ , and $\begin{array} { r } { M = \lceil \log ( \frac { \log T } { \log 2 } ) / \log 2 \rceil = \mathcal { O } ( \log \log T ) } \end{array}$ . For each interval of length $L _ { i }$ , we apply Algorithm 1 with the horizon length set to be $L _ { i } .$ , which yields the regret $\mathtt { R } _ { \mathtt { y } } ( L _ { i } ) = \mathcal { O } ( \log ^ { 4 } L _ { i } )$ . Summing over all $i \in \{ 0 , \ldots , M \}$ , we obtain

$$
\begin{array}{l} \mathbb {R} _ {\mathbf {y}} (T) = \sum_ {i = 0} ^ {M} \mathbb {R} _ {\mathbf {y}} (T _ {i} - T _ {i - 1}) \leq \sum_ {i = 0} ^ {M} \mathcal {O} (\log^ {4} T _ {i}) \\ = \sum_ {i = 0} ^ {M} \mathcal {O} (2 ^ {4 i}) \\ = \mathcal {O} \left(\frac {1 - 2 ^ {4 (M + 1)}}{1 - 2 ^ {4}}\right) \\ = \mathcal {O} (2 ^ {4 (M + 1)}) \\ = \mathcal {O} \left(2 ^ {4 \log \log T}\right) = \mathcal {O} (\log^ {4} T). \\ \end{array}
$$
