# 1.2.4 Intelligent data storage

Most data-driven algorithms used for infinite-horizon problems need a so-called intelligent data storage to ensure the recorded data is informative enough for convergence [2, 16, 39, 12, 33]. This is often expressed as a rank condition similar to the persistence of excitation. However, verifying these conditions is non-trivial and the number of data points needed for this purpose is not known in advance. The proposed algorithm in this paper also needs informative data to find the optimal solution to infinite-horizon problems. However, this requirement is met generically (in the measure-theoretic sense) by using n equally distanced data points.
