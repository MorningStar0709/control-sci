$$
\hat {J} = \sum_ {t = - \infty} ^ {\infty} \hat {x} _ {t} ^ {\top} \hat {X} \hat {x} _ {t} - \hat {x} _ {t + 1} ^ {\top} \hat {X} \hat {x} _ {t + 1} + \left[ \begin{array}{l} \hat {x} _ {t} \\ d _ {t} \end{array} \right] ^ {\top} \left[ \begin{array}{l} \hat {K} _ {x} ^ {\top} \\ I \end{array} \right] \hat {H} \left[ \begin{array}{l l} \hat {\kappa} _ {x} & I \end{array} \right] \left[ \begin{array}{l} \hat {x} _ {t} \\ d _ {t} \end{array} \right].
$$

The first two terms are a telescoping sum. These terms sum to zero by the boundary conditions $\hat { x } _ { - \infty } = \hat { x } _ { \infty } = 0$ . Thus the cost is equal to $\hat { J } = \langle \hat { F } d , \hat { F } d \rangle$ where $\hat { F }$ is the following intermediate system:

$$\hat {x} _ {t + 1} = \hat {A} \hat {x} _ {t} + \hat {B} d _ {t} \tag {40}\tilde {e} _ {t} = \hat {H} ^ {\frac {1}{2}} \hat {K} _ {x} \hat {x} _ {t} + \hat {H} ^ {\frac {1}{2}} d _ {t}.$$

Note that $\hat { F }$ is a square $n _ { d } \times n _ { d }$ system and invertible. The state matrix for $\hat { F } ^ { - 1 }$ is $\hat { A } - \hat { B } \hat { K } _ { x }$ which is a Schur matrix by Statement 1. Hence $\hat { F } ^ { - 1 }$ is stable and causal. However, $\hat { A }$ may not necessarily be a Schur matrix so one additional step is required to obtain the spectral factor F .

By Lemma 3, the cost is equal to $\hat { J } = \langle d , \hat { F } ^ { \sim } \hat { F } d \rangle$ . The matrix $\hat { A }$ is assumed to be nonsingular. This can be used to express the para-Hermitian conjugate $\hat { F } ^ { \sim }$ as:

$$\bar {x} _ {t + 1} = \hat {A} ^ {- \top} \bar {x} _ {t} - \hat {A} ^ {- \top} \hat {K} _ {x} ^ {\top} \hat {H} ^ {\frac {1}{2}} \bar {e} _ {t} \tag {41}\bar {d} _ {t} = (\hat {B} ^ {\top} \hat {A} ^ {- \top}) \bar {x} _ {t} + (I - \hat {B} ^ {\top} \hat {A} ^ {- \top} \hat {K} _ {x} ^ {\top}) \hat {H} ^ {\frac {1}{2}} \bar {e} _ {t}.$$

The system $\hat { F } ^ { \sim } \hat { F }$ corresponds to the serial connection of $\hat { F }$ and $\hat { F } ^ { \sim }$ , i.e. combine (40) and (41) with $\bar { e } _ { t } = \tilde { e } _ { t }$ . This yields a state-space model for $\hat { F } ^ { \sim } \hat { F }$ :
