# 3 Mean Field Game Solution

We follow the standard idea in [12] to derive the mean field game solutions. First, assuming $( \overline { { x } } ( t ) ) _ { t \in [ 0 , T ] }$ is given, the optimal tracking solution via Dynamic Programming is given by

$$u _ {i} ^ {*} (t) = - \frac {b}{r} (\Pi_ {t} x _ {i} (t) + s _ {t}) \tag {8}$$

where

$$
\begin{array}{l} - \dot {\Pi} _ {t} = 2 a \Pi_ {t} - \frac {b ^ {2}}{r} \Pi_ {t} ^ {2} + q _ {t} ^ {\alpha}, \quad \Pi_ {t} \geq 0, \quad \Pi_ {T} = 0, \tag {9} \\ - \dot {s} _ {t} = \left(a - \frac {b ^ {2}}{r} \Pi_ {t}\right) s _ {t} - q _ {t} ^ {\alpha} \overline {{x}} (t), \qquad s _ {T} = 0. \\ \end{array}
$$

The control law above corresponds to the best response for the representative agent i when the mean trajectory $( \overline { { x } } ( t ) ) _ { t \in [ 0 , T ] }$ is given. Then the closed-loop dynamics for a representative agent i that implements the best response are given by

$$d x _ {i} (t) = (a - \frac {b ^ {2}}{r} \Pi_ {t}) x _ {i} (t) d t - \frac {b ^ {2}}{r} s _ {t} d t + \sigma d w _ {i} (t)$$

and if all the agents follows the best response, the evolution of the mean field, if exists, satisfies

$$d \overline {{x}} (t) = \left((a - \frac {b ^ {2}}{r} \Pi_ {t}) \overline {{x}} (t) - \frac {b ^ {2}}{r} s _ {t}\right) d t$$

with $\overline { { x } } ( t ) = \mathbb { E } x _ { i } ( t )$ .

Let $e _ { i } ( t ) = x _ { i } ( t ) - { \overline { { x } } } ( t )$ and then the evolution $e _ { i } ( t )$ satisfies

$$d e _ {i} (t) = (a - \frac {b ^ {2}}{r} \Pi_ {t}) e _ {i} (t) d t + \sigma d w _ {i} (t).$$

Applying Itˆo’s lemma to $e _ { i } ^ { 2 }$ yields the following

$$
\begin{array}{l} d e _ {i} ^ {2} = 2 e _ {i} d e _ {i} + \sigma^ {2} d t \\ = 2 (a - \frac {b ^ {2}}{r} \Pi_ {t}) e _ {i} ^ {2} d t + 2 \sigma e _ {i} d w _ {i} + \sigma^ {2} d t. \\ \end{array}
$$

Let $V ( t ) : = \mathbb { E } [ e _ { i } ^ { 2 } ]$ denote the variance of the state $x _ { i } ( t )$ of a representative agent i. Then taking the expectation on both sides (of the corresponding integral equation) yields the dynamics for the variance $V ( t )$ as follows:

$$\dot {V} (t) = 2 (a - \frac {b ^ {2}}{r} \Pi_ {t}) V (t) + \sigma^ {2}$$

with the initial condition (given by the variance of the initial condition of the generic agent) $V ( 0 ) = V _ { 0 } : = \mathbb { E } ( x _ { i } ( 0 ) - \mathbb { E } x _ { i } ( 0 ) ) ^ { 2 }$ . See e.g., [1] for the covariance evolution for linear stochastic differential equations.
