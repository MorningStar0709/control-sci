$$\bar {J} \left(v ^ {*}\right) \leq \bar {J} (v), \quad \forall v \in \bar {\mathcal {U}} \quad \text { and } \quad \check {J} (\alpha^ {*}) \leq \check {J} (\alpha), \quad \forall \alpha \in \check {\mathcal {U}}. \tag {11}$$

According to Proposition 1, for any admissible pair $( x , u )$ of Problem MFT the corresponding pairs $( { \bar { x } } , { \bar { u } } )$ and $( \breve { x } , \breve { u } )$ , as defined in (7), are admissible for Problem 1 and Problem 2, respectively. Hence, from (11), we have

$$\bar {J} \left(v ^ {*}\right) \leq \bar {J} (\bar {u}) \quad \text { and } \quad \check {J} \left(\alpha^ {*}\right) \leq \check {J} (\check {u}). \tag {12}$$

From Corollary 1 and (12), for any admissible pair (x, u) of Problem MFT, we have

$$J (u) = \bar {J} (\bar {u}) + \check {J} (\check {u}) \geq \bar {J} \left(v ^ {*}\right) + \check {J} \left(\alpha^ {*}\right) = J \left(v ^ {*} + \alpha^ {*}\right), \quad \forall u \in \mathcal {U}, \tag {13}$$

where the last equality holds because $\mathbb { E } [ v _ { t } ^ { * } + \alpha _ { t } ^ { * } | \mathcal { F } _ { t } ^ { 0 } ] = v _ { t } ^ { * }$ and $\mathbb { E } [ y _ { t } ^ { * } + z _ { t } ^ { * } | \mathcal { F } _ { t } ^ { 0 } ] = y _ { t } ^ { * }$ for every $t \in [ 0 , T ]$ under Assumption 1. Hence, $\left( y ^ { * } + z ^ { * } , v ^ { * } + \alpha ^ { * } \right)$ is an optimal pair for J(·), and therefore, Problem MFT is solvable.

We now show that if Problem MFT is solvable, then both Problem 1 and Problem 2 are solvable and statement (b) holds. Suppose that Problem MFT is solvable; that is, there exists an optimal pair $( x ^ { * } , u ^ { * } )$ such that

$$J \left(u ^ {*}\right) \leq J (u), \quad \forall u \in \mathcal {U}. \tag {14}$$

According to Proposition 1, for any admissible pair $( y , v )$ for Problem 1, $( y + \breve { x } ^ { \ast } , v + \breve { u } ^ { \ast } )$ is an admissible pair for Problem MFT. Hence, from Corollary 1, we have

$$J (v + \breve {u} ^ {*}) = \bar {J} (v) + \breve {J} (\breve {u} ^ {*}), \tag {15}$$

since $\mathbb { E } [ v _ { t } + \breve { u } _ { t } ^ { * } | \mathcal { F } _ { t } ^ { 0 } ] = v _ { t }$ and $\mathbb { E } [ y _ { t } + \breve { x } _ { t } ^ { * } | \mathcal { F } _ { t } ^ { 0 } ] = y _ { t }$ for every $t \in [ 0 , T ]$ under Assumption 1. Moreover, since $u ^ { * } = \bar { u } ^ { * } + \breve { u } ^ { * }$ , we have

$$\bar {J} \left(\bar {u} ^ {*}\right) = J \left(\bar {u} ^ {*} + \breve {u} ^ {*}\right) - \breve {J} \left(\breve {u} ^ {*}\right) \leq J \left(v + \breve {u} ^ {*}\right) - \breve {J} \left(\breve {u} ^ {*}\right) = \bar {J} (v), \quad \forall v \in \bar {\mathcal {U}}. \tag {16}$$
