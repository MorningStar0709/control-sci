# A. Model and Algorithm

We consider an N-player noncooperative game, where each player $i \in [ N ]$ ] controls their action $u _ { i } \in \mathbb { R }$ , which maps to their payoff, $J _ { i } ( x , u ) \in \mathbb { R }$ , via a dynamic plant of the form:

$$\dot {x} = f (x, u), \tag {41}$$

where $x \in \mathbb { R } ^ { n }$ is the state, $\boldsymbol { u } = [ u _ { 1 } , . . . , u _ { N } ] ^ { \top }$ is the vector of players’ actions, and $f : \mathbb { R } ^ { n } \times \mathbb { R } ^ { N } \to \mathbb { R } ^ { n }$ is a continuous function. We make the following assumption on the plant (41), which is similar to Assumption 3 from Section IV:

Assumption 7: For system (41), there exists an ℓ-Lipschitz, $\mathcal { C } ^ { 1 }$ mapping $h : \mathbb { R } ^ { N }  \mathbb { R } ^ { n }$ such that $f ( h ( u ) , u ) = 0$ for all $u \in \mathbb { R } ^ { \bar { N } }$ . Moreover, there exists $\gamma > 0$ such that for each $\boldsymbol { \hat { u } } \in \mathbb { R } ^ { N }$ , there exists $a > 0 , p \in ( 0 , 1 )$ , and $q > 1$ such that

$$
\begin{array}{l} | x - h (\hat {u}) | \geq \gamma | u - \hat {u} | \\ \Rightarrow 2 (x - h (\hat {u})) ^ {\top} f (x, u) \leq - a | x - h (\hat {u}) | ^ {2 p} - a | x - h (\hat {u}) | ^ {2 q}, \tag {42} \\ \end{array}
$$

for all $x \in \mathbb { R } ^ { n }$ and $u \in \mathbb { R } ^ { N }$ .

The goal of player $i \in [ N ]$ is to design an update law on $u _ { i }$ in a way that optimizes their payoff $J _ { i } ( x , u )$ . Since the convergence of (41) requires $x = h ( u )$ , the players essentially want to seek the Nash equilibrium of the game parameterized by the cost functions $\{ \mathcal { T } _ { i } ( u ) \} _ { i = 1 } ^ { N }$ , where $\mathcal { T } _ { i } ( u ) = J _ { i } ( h ( u ) , u )$ . We will refer to this game as the quasi-steady state game. The Nash equilibrium of this game is defined as the following:

Definition 5: Consider an N-player game, where player $i \in$ $[ N ]$ has action $u _ { i } \in \mathbb { R }$ and cost function $\mathcal { I } _ { i } ( u )$ , where $u =$ $[ u _ { 1 } , . . . , u _ { N } ] ^ { \top }$ . An action profile $\boldsymbol { u } ^ { * } \in \mathbb { R } ^ { N }$ is said to be a Nash Equilibrium (NE) if, for each $i \in [ N ]$ , the following holds

$$\mathcal {J} _ {i} (u _ {i}, u _ {- i} ^ {*}) \geq \mathcal {J} _ {i} (u ^ {*}), \quad \forall u _ {i} \in \mathbb {R},$$

where $\boldsymbol { u } _ { - i } ^ { * } = [ u _ { 1 } ^ { * } , . . . , u _ { i - 1 } ^ { * } , u _ { i + 1 } ^ { * } , . . . , u _ { N } ^ { * } ] ^ { \top } .$
