# 3.2. Analysis

In this section, we focus on the Erlang-B queueing system to provide a simpler pathway to the queueing system with a non-zero waiting room. In Section 3.2.1, the convergence of $\pi _ { \mathrm { A l g 1 } }$ to the optimal policy is shown by a martingale-based analysis coupled with the SLLN for martingale sequences. Then, in Section 3.2.2, we evaluate the finite-time performance of our proposed policy in terms of the expected regret defined in Definition 1 using martingale concentration inequalities.

3.2.1. Asymptotic Optimality First, we describe a stochastic process whose limiting behavior will determine the performance of our learning scheme. Define $\{ { \tilde { X } } _ { n } \} _ { n = 0 } ^ { \infty }$ as

$$\tilde {X} _ {n} = (X _ {n}, N _ {n}, \alpha_ {n}) = \Big (\sum_ {i = 1} ^ {n} \Big (g \left(T _ {i}, M _ {i}, c / R\right) - h \left(T _ {i}, N _ {i}, c / R\right) \Big), N _ {n}, \alpha_ {n} \Big). \tag {11}$$

We note that the action at arrival n defined by $\pi _ { \mathrm { A l g 1 } }$ is uniquely determined by ${ \tilde { X } } _ { S ( n ) } \colon$ if a server is available and $X _ { S ( n ) } > 0$ , the arrival will be accepted. Otherwise, if $X _ { S ( n ) } \leq 0$ , the arrival will be admitted with probability $p _ { \alpha _ { n } }$ . To prove asymptotic optimality, we show that eventually, $X _ { n }$ will always be positive for $\textstyle \mu > { \frac { c } { R } }$ , and negative for $\textstyle \mu < { \frac { c } { R } }$ . In the process $\{ \tilde { X } _ { n } \} _ { n = 0 } ^ { \infty } , X _ { n }$ is updated as

$$X _ {n} - X _ {n - 1} = g \left(T _ {n}, M _ {n}, c / R\right) - h \left(T _ {n}, N _ {n}, c / R\right). \tag {12}$$
