$$\frac {1}{2} \| \nabla J (L _ {0}) \| _ {F} ^ {2} \left[ (\gamma + 1) ^ {2} \ell (\beta) \eta^ {2} + 2 (\gamma - 1) \eta \right] \leq \beta - \alpha ,$$

or equivalently,

$$\left[ (\gamma + 1) \ell (\beta) \eta + \frac {\gamma - 1}{\gamma + 1} \right] ^ {2} \leq \left(\frac {\gamma - 1}{\gamma + 1}\right) ^ {2} + \frac {2 \ell (\beta) [ \beta - \alpha ]}{\| \nabla J (L _ {0}) \| _ {F} ^ {2}}.$$

But then it suffices to have

$$(\gamma + 1) \ell (\beta) \eta + \frac {\gamma - 1}{\gamma + 1} \leq \frac {\sqrt {2 \ell (\beta) [ \beta - \alpha ]}}{\| \nabla J (L _ {0}) \| _ {F}}.$$

Finally, note that by Lemma 2 and (11c) we have

$$\| \nabla J (L _ {0}) \| _ {F} \leq \frac {\ell (\alpha)}{c _ {3} (\alpha)} [ J (L _ {0}) - J (L ^ {*}) ] \leq \frac {\ell (\alpha)}{c _ {3} (\alpha)} [ \alpha - \alpha^ {*} ]. \tag {32}$$

Using this uniform bound of gradient on sublevel set $S _ { \alpha }$ and noting that $\gamma \in [ 0 , 1 ]$ , we can obtain the sufficient condition for η to be feasible. This completes the proof. □
