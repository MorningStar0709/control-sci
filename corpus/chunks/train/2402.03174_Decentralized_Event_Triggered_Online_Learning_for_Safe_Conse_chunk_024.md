# A. Simulation Setting

To evaluate the designed control law (3), (4) with eventtriggered online learning with (5), (11), we consider a multiagent system with $N = 4$ agents under dynamics (1) with $h ( x _ { i } ) = 0 , g ( x _ { i } ) = 1$ satisfying Assumption 1 and

$$f (x _ {i}) = \sin (1 0 x _ {i}) + \frac {1}{2 \exp (- x _ {i} / 1 0)} + 5 \tag {33}$$

for each agent $i \in \mathcal { V } = \{ 1 , \cdots , 4 \}$ with the compact domain $\mathbb { X } = [ - 1 . 5 , 1 . 5 ]$ . The communication topology among the agents is defined by a connected undirected graph $\mathcal { G }$ with the edge set $\mathcal { E } ~ = ~ \{ ( 1 , 2 ) , ( 2 , 3 ) , ( 3 , 4 ) , ( 4 , 1 ) \}$ . To infer the unknown function $f ( \cdot )$ , each agent $\textit { i } \in \mathcal { V }$ employs a Gaussian process model using the kernel function satisfying Assumption 3 as

$$\kappa (x, x ^ {\prime}) = \sigma_ {f} ^ {2} \exp \left(- \frac {| x - x ^ {\prime} | ^ {2}}{2 l ^ {2}}\right) \tag {34}$$

with $\sigma _ { f } = 1 , l = 0 . 0 5$ and the individual data set $\mathbb { D } _ { i }$ satisfying Assumption 2 with the variance of the measurement noise as $\sigma _ { n } = 0 . 0 1$ . The simulation time is set as 10.
