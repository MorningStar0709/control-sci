# Remarks

• The controller $R _ { 0 } , S _ { 0 } , T _ { 0 } = P _ { D } ( q ^ { - 1 } )$ corresponds to the tracking and regulation with independent objectives for the plant model (7.1) with $d = 0$ and with the desired closed-loop poles defined by $P _ { D } ( q ^ { - 1 } )$ .

• The controller of (7.125) and (7.126) shows for ${ \cal P } _ { F } ( q ^ { - 1 } ) = 1$ that in order to control a system with delay, the controller for the case without delay can be used by replacing the measured output $y ( t )$ by its d-steps ahead prediction ${ \hat { y } } ( t + d | t )$ ).

• The other remarks from Sect. 7.3.3 for the pole placement are also applicable in this case.
