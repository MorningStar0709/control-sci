# B. Uncertainty quantification

The following theorem, which is inspired by [31, Prop. 7], provides an upper and a lower bound on the $\scriptstyle L - \mathrm { g a p }$ in case the restricted behaviors have a specific form.

Theorem 1. Let $\boldsymbol { B } \in \mathcal { L } ^ { q , c }$ and $\tilde { B } \in \mathcal { L } ^ { q , c }$ . Given $L \in \mathbb { Z } _ { + }$ , with $L > l ,$ assume

$$
\mathcal {B} | _ {L} = \mathrm{im} \left[ \begin{array}{c} I \\ F \end{array} \right] \text {   and   } \tilde {\mathcal {B}} | _ {L} = \mathrm{im} \left[ \begin{array}{c} I \\ \tilde {F} \end{array} \right]. \tag {7}
$$

Then

$$\frac {\| F - \tilde {F} \| _ {2}}{\sqrt {1 + \| F \| _ {2} ^ {2}} \sqrt {1 + \| \tilde {F} \| _ {2} ^ {2}}} \leq \operatorname{gap} _ {L} (\mathcal {B}, \tilde {\mathcal {B}}) \leq \| F - \tilde {F} \| _ {2}. \tag {8}$$

Proof. We first prove the left inequality, i.e., the lower bound on the L-gap. By (5),

$$
\begin{array}{l} \operatorname{gap} _ {L} (\mathcal {B}, \tilde {\mathcal {B}}) = \| (I + \tilde {F} \tilde {F} ^ {\top}) ^ {- \frac {1}{2}} [ - \tilde {F} I ] \left[ \begin{array}{c} I \\ F \end{array} \right] (I + F ^ {\top} F) ^ {- \frac {1}{2}} \| _ {2} \\ \geq \sigma_ {\min} ((I + \tilde {F} \tilde {F} ^ {\top}) ^ {- \frac {1}{2}}) \sigma_ {\min} ((I + F ^ {\top} F) ^ {- \frac {1}{2}}) \| F - \tilde {F} \| _ {2}, \\ \end{array}
$$

where $\sigma _ { \mathrm { m i n } } ( X )$ denotes the smallest singular value of a given matrix X . Next, we will work out $\sigma _ { \mathrm { m i n } } ( ( I + \tilde { F } \tilde { F } ^ { \top } ) ^ { - \frac { 1 } { 2 } } )$ ). Note that

$$
\begin{array}{l} \sigma_ {\min} ((I + \tilde {F} \tilde {F} ^ {\top}) ^ {- \frac {1}{2}}) = \frac {1}{\sigma_ {\max} ((I + \tilde {F} \tilde {F} ^ {\top}) ^ {\frac {1}{2}})} \\ = \frac {1}{\sqrt {\sigma_ {\mathrm{max}} (I + \tilde {F} \tilde {F} ^ {\top})}} \\ = \frac {1}{\sqrt {\| (I + \tilde {F} \tilde {F} ^ {\top}) \| _ {2}}}, \\ \end{array}
$$

where $\sigma _ { \mathrm { m a x } } ( X )$ denotes the largest singular value of $X$ Finally, note that for any eigenvalue λ and corresponding eigenvector v of $\tilde { F } \tilde { F } ^ { \top }$ , we have that

$$(I + \tilde {F} \tilde {F} ^ {\top}) (I + \tilde {F} \tilde {F} ^ {\top}) v = (1 + \lambda) ^ {2} v.$$
