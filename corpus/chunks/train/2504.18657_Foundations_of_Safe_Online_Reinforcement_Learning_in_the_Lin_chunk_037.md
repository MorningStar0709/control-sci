# A.2 Problem Specifications

Below is a (non-exhaustive) list of notation used throughout the appendix.

• $q , r :$ coefficients for the cost at time t of $q x _ { t } ^ { 2 } + r u _ { t } ^ { 2 }$ .   
• $W = \{ w _ { t } \} _ { t = 0 } ^ { T - 1 }$ : The noise random variables for the T -length trajectory.   
• D : Distribution of $w _ { t }$   
– $B _ { P }$ : Upper bound on the density of D   
– $F _ { \mathcal { D } }$ : Cumulative Density Function (CDF) of D   
$- \textit { w } \mathrm { : }$ the bound of D when the distribution is bounded.

• $\Theta = [ \underline { { a } } , \bar { a } ] \times [ \underline { { b } } , \bar { b } ]$ : The given initial set of dynamics such that $\theta ^ { \ast } \in \Theta$ and $\mathrm { s i z e } ( \Theta ) =$ $\operatorname* { m i n } ( { \bar { a } } - \underline { { a } } , { \bar { b } } - \underline { { b } } )$   
• $\theta ^ { * } = ( a ^ { * } , b ^ { * } )$ : The true (unknown) dynamics.   
• $C ^ { \mathrm { i n i t } }$ : The initial safe controller satisfying Assumption 2.

• $D = \left( D _ { \mathrm { { L } } } , D _ { \mathrm { { U } } } \right)$ : the expected-state boundary for the safety constraint.   
• A set of controls $\{ u _ { t } \}$ are safe for dynamics $\{ \theta _ { t } \}$ if for all t, $D _ { \mathrm { L } } \leq a _ { t } x _ { t } + b _ { t } u _ { t } \leq D _ { \mathrm { U } }$ .   
• $H _ { t } = ( x _ { 0 } , u _ { 0 } , x _ { 1 } , u _ { 1 } , . . . , u _ { t - 1 } , x _ { t } )$ and $\mathcal { F } _ { t } = \sigma ( H _ { t } )$ .   
• $J ( \theta , C , T , x , W )$ : The random variable cost of using controller C starting at state $x _ { 0 } = x$ for T time steps under dynamics θ with noise random variables $W$ .

• $J ^ { * } ( \theta , C , T ) = J ^ { * } ( \theta , C , T , 0 ) = \mathbb { E } [ J ( \theta , C , T , x , W ) \mid \theta , C , T , x ]$ and $J ^ { * } ( \theta , C , T ) = J ^ { * } ( \theta , C , T , 0 )$ .

• $J ^ { * } ( \theta , C ) = J ^ { * } ( \theta , C , 0 ) = \operatorname* { l i m } _ { T \to \infty } J ^ { * } ( \theta , C , T , 0 )$ .

• $\mathcal { C } ^ { \theta } ~ = ~ \{ C _ { K } ^ { \theta } \} _ { K \in [ K _ { \mathrm { L } } ^ { \theta } , K _ { \mathrm { U } } ^ { \theta } ] }$ : a class of controllers that are safe for dynamics θ that are parameterized by $\breve { K } \in [ K _ { \mathrm { L } } ^ { \theta } , K _ { \mathrm { U } } ^ { \theta } ]$

• $K _ { \mathrm { o p t } } ( \theta , T )$ : The K that maximizes $J ^ { * } ( \theta , C _ { K } ^ { \theta } , T , 0 )$ for $K \in [ K _ { \mathrm { L } } ^ { \theta } , K _ { \mathrm { U } } ^ { \theta } ]$ .

• $K _ { \mathrm { o p t } } ( \theta )$ : The K that maximizes $J ^ { * } ( \theta , C _ { K } ^ { \theta } )$ for $K \in [ K _ { \mathrm { L } } ^ { \theta } , K _ { \mathrm { U } } ^ { \theta } ]$ .

• $C _ { K } ^ { \mathrm { u n c } }$ : The unconstrained linear controller with parameter K, i.e. such that $C _ { K } ^ { \mathrm { u n c } } ( x ) =$ $- K x$ .

• $F _ { \mathrm { o p t } } ( \theta )$ : The K that maximizes $J ^ { * } ( \theta , C _ { K } ^ { \mathrm { u n c } } )$ .
