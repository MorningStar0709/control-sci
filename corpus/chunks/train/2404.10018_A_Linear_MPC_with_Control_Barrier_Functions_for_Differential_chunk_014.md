# A. Feedback Linearization

The unicycle model in (9) can be represented as MIMO affine in control system as follows:

$$
f (x) = \left[ \begin{array}{l} 0 \\ 0 \\ 0 \end{array} \right], g _ {1} (x) = \left[ \begin{array}{c} \cos (x _ {3}) \\ \sin (x _ {3}) \\ 0 \end{array} \right], g _ {2} (x) = \left[ \begin{array}{l} 0 \\ 0 \\ 1 \end{array} \right] \tag {17}
$$

where $h _ { 1 } ( x ) = x _ { 1 }$ and $h _ { 2 } ( x ) = x _ { 2 }$ . To calculate the relative degree vector $\boldsymbol { r } ~ = ~ [ r _ { 1 } , r _ { 2 } ] ^ { \top }$ of (17), recall (1) and consider $r _ { 1 } = 1$ . One finds

$$
\mathcal {L} _ {g _ {1}} h _ {1} = \left[ \begin{array}{c c c} 1 & 0 & 0 \end{array} \right] \left[ \begin{array}{c} \cos (x _ {3}) \\ \sin (x _ {3}) \\ 0 \end{array} \right] = \cos (x _ {3}) \tag {18}

\mathcal {L} _ {g _ {2}} h _ {1} = \left[ \begin{array}{l l l} 1 & 0 & 0 \end{array} \right] \left[ \begin{array}{l} 0 \\ 0 \\ 1 \end{array} \right] = 0 \tag {19}
$$

Since $\mathcal { L } _ { g _ { 1 } } h _ { 1 } \neq 0$ for $x _ { 3 } \neq \{ 9 0 , 2 7 0 \} \mathrm { d e g }$ , one concludes that $r _ { 1 } = 1$ . The control input $u _ { 1 }$ appears in the first output expect for $x _ { 3 } = \{ 9 0 , 2 7 0 \}$ , which is intuitive since the linear velocity will not contribute to movement in the x direction when the unicycle has $\theta \ = \ \{ 9 0 , 2 7 0 \}$ (visit Fig. 1.(b)) and recall the nonholonomic constraints in (7). Consider $r _ { 2 } = 1$ , one has

$$
\mathcal {L} _ {g _ {1}} h _ {2} = \left[ \begin{array}{c c c} 0 & 1 & 0 \end{array} \right] \left[ \begin{array}{c} 0 \\ \sin (x _ {3}) \\ 0 \end{array} \right] = \sin (x _ {3}) \tag {20}

\mathcal {L} _ {g _ {2}} h _ {2} = \left[ \begin{array}{l l l} 0 & 1 & 0 \end{array} \right] \left[ \begin{array}{l} 0 \\ 0 \\ 1 \end{array} \right] = 0 \tag {21}
$$

Since $\mathcal { L } _ { g _ { 1 } } h _ { 2 } \neq 0$ for $x _ { 3 } ~ \neq ~ \{ 0 , 1 8 0 \}$ deg, one concludes that $r _ { 2 } ~ = ~ 1$ . The control input $u _ { 2 }$ appears in the second output expect for $x _ { 3 } = \{ 0 , 1 8 0 \}$ , which is intuitive since the linear velocity will not contribute to movement in the y direction when the unicycle has $\theta = \{ 0 , 1 8 0 \}$ (visit Fig. 1.(b)) and recall the nonholonomic constraints in (7)). In the view of (3), the decoupling input matrix $A ( x )$ can be rewritten as follows:

$$
A (x) = \left[ \begin{array}{l l} \mathcal {L} _ {g _ {1}} h _ {1} & \mathcal {L} _ {g _ {2}} h _ {1} \\ \mathcal {L} _ {g _ {1}} h _ {2} & \mathcal {L} _ {g _ {2}} h _ {2} \end{array} \right] = \left[ \begin{array}{l l} \cos (x _ {3}) & 0 \\ \sin (x _ {3}) & 0 \end{array} \right] \tag {22}
$$

It becomes apparent that $A ( x )$ is singular and in view of (2) the full input-to-state feedback linearization using the control input

$$
u = A ^ {- 1} (x) \left(\left[ \begin{array}{l} \mathcal {L} _ {f} ^ {r _ {1}} h _ {1} (x) \\ \mathcal {L} _ {f} ^ {r _ {2}} h _ {2} (x) \end{array} \right] + \left[ \begin{array}{l} v _ {1} \\ v _ {2} \end{array} \right]\right)
$$

is unsolvable. As such, some modifications are necessary to render the nonlinear model dynamics in (9) accounting for the full input-to-state feedback linearizable form which is the focus of the next subsections.
