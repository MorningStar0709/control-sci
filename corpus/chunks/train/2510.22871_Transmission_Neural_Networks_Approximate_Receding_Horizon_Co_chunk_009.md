# IV. INFECTION PROBABILITY UPPER BOUND BY TRANSNNS

In this section, we investigate further the relation between the state evolution in (1) and that in (7); more specifically, we show that without the assumption (A4) the states of the dynamics in (7) (and equivalently those of TransNNs in (8)) provide upper bounds for the infection probabilities from the stochastic virus spread model in (1).

Proposition 2 Assume (A1) and (A3) hold. Let $X _ { i } ( k ) , i \in$ [n] be the binary state in dynamics (1) and let $p _ { i } ( k )$ denote the probability state in dynamics (7). Given the same physical contact network $( V , E ^ { k } )$ for all $k \in \mathbb { N }$ , the same conditional transmission probabilities $\{ w _ { i j } ^ { k } , i , j \in [ n ] , k \ge 0 \}$ and the same initial infection probability (i.e. $p _ { i } ( 0 ) = \mathrm { P r } ( X _ { i } ( 0 ) = 1 )$ for all $i \in [ n ] ,$ , the following inequality holds

$$p _ {i} (k) \geq \operatorname * {P r} (X _ {i} (k) = 1), \quad \forall i \in [ n ], \quad \forall k \in \mathbb {N}; \tag {10}$$

if, furthermore, (A4) is satisfied for the dynamics (1), then the inequality in (10) becomes an equality for all $k \in$ $\{ 0 , 1 , \ldots , T \}$ and all $i \in [ n ]$ . ✷

PROOF Define the binary random variable $Z _ { i j } ( k ) \triangleq 1 -$ $W _ { i j } ^ { k } X _ { j } ( k )$ . Then $Z _ { i j } ( k ) = 1$ represents the event that node i does not successfully receive virus transmission from node $j$ at time k. We observe that for any $i \in [ n ]$ ,

$$
\begin{array}{l} \mathbb {E} (1 - X _ {i} (k + 1)) \\ = \mathbb {E} \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - W _ {i j} ^ {k} X _ {j} (k)\right) \triangleq \mathbb {E} \prod_ {j \in N _ {i} ^ {\circ k}} Z _ {i j} (k) \\ = \operatorname * {P r} \left(\left(Z _ {i j} = 1\right) _ {j \in N _ {i} ^ {\circ k}}\right) \geq \prod_ {j \in N _ {i} ^ {\circ k}} \operatorname * {P r} \left(Z _ {i j} (k) = 1\right) \\ \end{array}
$$

since the following holds

$$\operatorname * {P r} (Z _ {i j} (k) = 1 | (Z _ {i \ell} (k) = 1) _ {\ell \in S \subset [ n ], \ell \neq j}) \geq \operatorname * {P r} (Z _ {i j} (k) = 1),$$

for all subset $S \in [ n ]$ and all $i , j \in [ n ]$ . That is, conditioning on node i not successfully receiving virus transmissions from neighbours other than node $j ,$ does not reduce the probability that node i does not successfully receive virus transmission from node $j ^ { 1 }$ .
