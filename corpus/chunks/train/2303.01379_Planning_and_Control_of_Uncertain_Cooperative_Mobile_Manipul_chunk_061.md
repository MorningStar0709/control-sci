The generalized velocity of each agent’s end-effector $v _ { i } : = [ \boldsymbol { \dot { p } } _ { E _ { i } } ^ { \top } , \boldsymbol { \omega } _ { E _ { i } } ^ { \top } ] ^ { \top } \in$ $\mathbb { R } ^ { 6 }$ , where $\boldsymbol { \omega } _ { E _ { i } } \in { \mathbb { R } } ^ { 3 }$ is the respective angular velocity, can be considered as a transformed state through the differential kinematics $v _ { i } = J _ { i } \dot { q } _ { i } \ [ 8 5 ]$ , where $J _ { i } : = J _ { i } ( q _ { i } ) : \mathbb { R } ^ { n _ { i } }  \mathbb { R } ^ { 6 \times n _ { i } }$ is a smooth function representing the geometric Jacobian matrix, $\forall i \in \mathcal { N } \ [ 8 5 ]$ . The latter leads also to

$$\dot {v} _ {i} = J _ {i} \ddot {q} _ {i} + \dot {J} _ {i} \dot {q} _ {i}. \tag {2.2}$$

We define also the sets $\mathsf { S } _ { i } : = \{ q _ { i } \in \mathbb { R } ^ { n _ { i } } : \operatorname* { d e t } ( J _ { i } ( q _ { i } ) J _ { i } ( q _ { i } ) ^ { \top } ) > 0 \}$ , which contains all the singularity-free configurations. By employing the differential kinematics as well as (2.2), we obtain from (2.1) the transformed task space dynamics [85]:

$$M _ {i} \dot {v} _ {i} + C _ {i} v _ {i} + g _ {i} + d _ {i} = u _ {i} - h _ {i}, \tag {2.3}$$

with the corresponding task space terms $M _ { i } : = M _ { i } ( q _ { i } ) : \mathsf { S } _ { i } \to \mathbb { R } ^ { 6 \times 6 } , C _ { i } : =$ $C _ { i } ( q _ { i } , { \dot { q } } _ { i } ) : \mathsf { S } _ { i } \times \mathbb { R } ^ { n _ { i } } \to \mathbb { R } ^ { 6 \times 6 } , \ g _ { i } : = g _ { i } ( q _ { i } ) : \mathsf { S } _ { i } \to \mathbb { R } ^ { 6 } , \ d _ { i } : = d _ { i } ( q _ { i } , { \dot { q } } _ { i } , t ) : .$ $\mathsf { S } _ { i } \times \mathbb { R } ^ { n _ { i } } \times \mathbb { R } _ { \ge 0 } \to \mathbb { R } ^ { 6 }$ and $u _ { i } = [ u _ { i , 1 } , \ldots , u _ { i , 6 } ] ^ { \top } \in \mathbb { R } ^ { 6 }$ being the task space wrench, related to $\tau _ { i }$ via $\tau _ { i } = J _ { i } ^ { \top } u _ { i } + ( I _ { n _ { i } } - J _ { i } ^ { \top } \widetilde { J } _ { i } ^ { \top } ) \tau _ { i 0 }$ , where $\widetilde { J } _ { i }$ is a generalized inverse of $J _ { i } \ [ 8 5 ] ; \tau _ { i 0 }$ concerns redundant agents $( n _ { i } > 6 )$ and does not contribute to end-effector forces.

The agent task-space dynamics (2.3) can be written in vector form as:

$$M \dot {v} + C v + g + d = u - h, \tag {2.4}$$
