# 5.2 Evaluating the Adversarial Game

Agent $c _ { 0 } \mathrm { { ^ { \circ } s } }$ goal is to satisfy a formula φ , and agent $c _ { 1 } \mathrm { { ' } s }$ goal is to satisfy a formula ψ, given by:

$$
\phi : \left(\{\bar {a} \bar {b} \bar {c} \} \rightarrow \mathbf {X} \{\bar {a} \bar {b} c \} \rightarrow \mathbf {X} \{a \bar {b} c \}\right) \cup \left(\{\bar {a} b \bar {c} \} \rightarrow \mathbf {X} \{\bar {a} b c \} \rightarrow \mathbf {X} \{a \bar {b} c \}\right) \cup \left(\{a b \bar {c} \} \rightarrow \mathbf {X} \{a b c \} \rightarrow \mathbf {X} \{\bar {a} b c \}\right) \\ ((\{a b c \}) = (Y (\bar {\overline {{a}}}) - (Y (\overline {{a}})) = Y (\overline {{\overline {{a}}}}) \end{array} \tag {7}
\psi : \left(\{\bar {a} b \bar {c} \} \rightarrow \mathbf {X} \{\bar {a} \bar {b} \bar {c} \} \cup \left(\{\bar {a} b c \} \rightarrow \mathbf {X} \{\bar {a} \bar {b} c \} \right.\right.$$

In this scenario, there are more options for φ satisfiability than for ψ; φ can be satisfied by three different sub-runs, but all require two successive successful state transitions. ψ can only be satisfied by two different sub-runs, but each just requires one successful state transition.
