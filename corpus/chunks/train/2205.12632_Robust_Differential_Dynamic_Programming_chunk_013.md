# A. A simple case

If in (14), the matrices $\overline { { q } } _ { t } ^ { 3 1 } , \overline { { Q } } _ { t } ^ { 3 2 } , \overline { { Q } } _ { t } ^ { 3 3 } , \overline { { Q } } _ { t } ^ { 3 4 }$ Qt do not contain decision variables and $\overline { { \boldsymbol { Q } } } _ { t } ^ { s { \scriptscriptstyle 3 } }$ is positive definite, then a convexification of this matrix inequality can be achieved by a single Schur complement. The considered independence of $\check { q } _ { t } ^ { 3 1 } , \overline { { Q } } _ { t } ^ { 3 2 } , \overline { { Q } } _ { t } ^ { 3 3 } , \dot { \overline { { Q } } } _ { t } ^ { 3 4 }$ of decision variables is often obtained, when in $( 6 ) ~ g ( \cdot )$ is not a function of u.

Consider the matrix $\scriptstyle { \mathbf { L } } _ { t }$ equal to

$$
(\star) ^ {\top} \left( \begin{array}{c c c c} \bar {q} _ {t} ^ {1 1} - p _ {t} ^ {1 1} & \bar {q} _ {t} ^ {1 2} - p _ {t} ^ {1 2} & \bar {q} _ {t} ^ {1 3} & \bar {q} _ {t} ^ {1 4} \\ \bar {q} _ {t} ^ {2 1} - p _ {t} ^ {2 1} & \overline {{\boldsymbol {Q}}} _ {t} ^ {2 2} - \boldsymbol {P} _ {t} ^ {2 2} & \overline {{\boldsymbol {Q}}} _ {t} ^ {2 3} & \overline {{\boldsymbol {Q}}} _ {t} ^ {2 4} \\ \bar {q} _ {t} ^ {3 1} & \overline {{\boldsymbol {Q}}} _ {t} ^ {3 2} & 0 & \overline {{\boldsymbol {Q}}} _ {t} ^ {3 4} \\ \bar {q} _ {t} ^ {4 1} & \overline {{\boldsymbol {Q}}} _ {t} ^ {4 2} & \overline {{\boldsymbol {Q}}} _ {t} ^ {4 3} & \overline {{\boldsymbol {Q}}} _ {t} ^ {4 4} \end{array} \right) \left( \begin{array}{c c c} 1 & 0 & 0 \\ 0 & \boldsymbol {I} & 0 \\ k _ {t} ^ {1} & \boldsymbol {K} _ {t} ^ {2} & 0 \\ 0 & 0 & \boldsymbol {I} \end{array} \right).
$$

Then, $\scriptstyle { \mathbf { L } } _ { t }$ is affine linear in $\mathbf { \boldsymbol { K } } _ { t } , \mathbf { \boldsymbol { P } } _ { t }$ and possible decision variables in $\overline { { \boldsymbol { Q } } } _ { t }$ . In this case, (13) can be expressed as

$$
\boldsymbol {L} _ {t} + \left( \begin{array}{c c c} k _ {t} ^ {1} & \boldsymbol {K} _ {t} ^ {2} & 0 \end{array} \right) ^ {\top} \overline {{\boldsymbol {Q}}} _ {t} ^ {3 3} \left( \begin{array}{c c c} k _ {t} ^ {1} & \boldsymbol {K} _ {t} ^ {2} & 0 \end{array} \right) \prec 0.
$$

Due to $\overline { { \boldsymbol { Q } } } _ { t } ^ { 3 3 } \succ 0$ , a Schur complement yields

$$
\left( \begin{array}{c c c c} l _ {t} ^ {1 1} & l _ {t} ^ {1 2} & l _ {t} ^ {1 3} & (k _ {t} ^ {1}) ^ {\top} \\ l _ {t} ^ {2 1} & \boldsymbol {L} _ {t} ^ {2 2} & \boldsymbol {L} _ {t} ^ {2 3} & (\boldsymbol {K} _ {t} ^ {2}) ^ {\top} \\ l _ {t} ^ {3 1} & \boldsymbol {L} _ {t} ^ {3 2} & \boldsymbol {L} _ {t} ^ {3 3} & 0 \\ k _ {t} ^ {1} & \boldsymbol {K} _ {t} ^ {2} & 0 & - (\overline {{\boldsymbol {Q}}} _ {t} ^ {3 3}) ^ {- 1} \end{array} \right) \prec 0.
$$

This matrix inequality is linear in all decision variables.
