# 3.1 Hyperparameter Estimation

To apply the MAP estimator (10) or (11), the hyperparameter $\textbf { g } \in \ \mathbb { R } ^ { M }$ needs to be estimated as well. In this regard, we adopt the maximum marginal likelihood method. Consider the equation $\zeta = \Phi H \bar { \bf g } - \Phi \Delta _ { H } { \bf g } + \epsilon$ . The hyperparameter g is estimated by maximizing the marginal likelihood: $\hat { \mathbf { g } } = \operatorname { a r g m a x } _ { \mathbf { g } } p ( \zeta | \mathbf { g } )$ , where $\zeta | \mathbf { g } \sim$ $\mathcal { E } _ { f } \left( \Phi H \mathbf { g } , \Phi \Sigma _ { g } ( \mathbf { g } ) \Phi ^ { \top } + \Sigma _ { \epsilon } \right)$ .

Let $\Psi ( { \bf g } ) : = \Phi \Sigma _ { g } ( { \bf g } ) \Phi ^ { \top } + \Sigma _ { \epsilon }$ and $\delta _ { \zeta } ( { \bf g } ) \ = \ \zeta \mathrm { ~ - ~ } \Phi H { \bf g }$ . If there exists g such that $\Psi ( \mathbf { g } )$ is positive definite, the hyperparameter estimation problem is given by

$$\hat {\mathbf {g}} = \underset {\mathbf {g}} {\operatorname{argmax}} \det \left(\Psi (\mathbf {g})\right) ^ {- \frac {1}{2}} f \left(\left\| \boldsymbol {\delta} _ {\zeta} (\mathbf {g}) \right\| _ {\Psi^ {- 1} (\mathbf {g})} ^ {2}\right). \tag {15}$$

When Ψ(g) is singular for all g, similar to (11), let $( U _ { \Psi , 1 } , U _ { \Psi , 2 } , \Psi _ { 1 } ( \mathbf { g } ) )$ be the decomposition of $\Psi ( \mathbf { g } )$ according to (2). We have

$$\hat {\mathbf {g}} = \underset {\mathbf {g}} {\operatorname{argmax}} \det \left(\Psi_ {1} (\mathbf {g})\right) ^ {- \frac {1}{2}} f \left(\left\| U _ {\Psi , 1} ^ {\top} \boldsymbol {\delta} _ {\zeta} (\mathbf {g}) \right\| _ {\Psi_ {1} ^ {- 1} (\mathbf {g})} ^ {2}\right) \tag {16}\mathrm{s.t.} U _ {\Psi , 2} ^ {\top} \pmb {\delta} _ {\zeta} (\mathbf {g}) = \mathbf {0}.$$

The estimated hyperparameter $\hat { \bf g }$ is used in (10) or (11) to obtain the MAP estimate. This assumes certainty equivalence without considering the hyperparameter estimation error, which is common for Bayesian methods with hyperparameters. The algorithm is summarized in Algorithm 1.
