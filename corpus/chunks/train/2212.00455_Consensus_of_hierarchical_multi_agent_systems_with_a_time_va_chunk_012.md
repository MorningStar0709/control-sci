# 3.2 Consensus of TH-MAS: Lower Layer

Consider a state feedback controller, such that, for the TH-MAS defined by (4) and (5), the control inputs $( u _ { i } )$ are calculated as

$$C _ {i}: u _ {i} (k) = - k _ {f b} \sum_ {j = 1, j \neq i} ^ {N + 1} a _ {i j} \left(x _ {i} (k) - x _ {j} (k)\right), \tag {19}$$

for all $i , j = \{ 1 , \dots , N + 1 \}$ , where $k _ { f b } \in \mathbb { R }$ is the feedback gain and $a _ { i j }$ are the coefficients of the adjacency matrix A. Then, by implementing the switching algorithm (17) and state feedback controller (19), also called consensus protocol, in the above mentioned TH-MAS, the resulting closed-loop system is expressed by

$$\bar {\boldsymbol {\Sigma}} _ {d} := \boldsymbol {x} (k + 1) = \boldsymbol {\mathcal {P}} (k) \cdot \boldsymbol {x} (k), \tag {20}$$

where $\mathfrak { s } ( k ) = [ x _ { 1 } , \dots , x _ { N + 1 } ] ^ { \top } \in \mathbb { R } ^ { N + 1 }$ is a vector containing the state of all the agents, and for all $k \in [ ( m - 1 ) M + 1 , m M )$ ), i.e. for $q _ { L } ( k ) = 0$ ,

$$\boldsymbol {\mathcal {P}} (k) = \mathbf {I} _ {N + 1} - w \cdot k _ {f b} \cdot \mathcal {L} (k) \tag {21}$$

is a primitive row-stochastic matrix with $\begin{array} { r } { \mathcal { L } ( k ) = f _ { \mathcal { L } } ( \mathcal { G } ( k ) ) } \end{array}$ , as in (17) and $k _ { f b }$ as in (19). Note that $a _ { i j } = 0$ if the agents are not active; hence, the Laplacian matrix also describes the behaviour represented by $B ( k )$ . In this paper, we focus on solving the consensus problem for a TH-MAS represented by (20).

The consensus problem stands for the challenge of steering all the followers to a common state, i.e.

$$\lim _ {k \to \infty} \boldsymbol {x} (k) = \mathbf {1} _ {N + 1} \bar {x} \tag {22}$$

where $\pmb { x } ( k ) = [ x _ { 1 } , \dots , x _ { N + 1 } ] ^ { \top } \in \mathbb { R } ^ { N + 1 }$ is a vector containing the state of all the agents and x¯ ∈ R is the consensus value. It is important to remark that x¯ is determined by the Laplacian matrix and the initial states of the agents in the system as the following lemma explains.
