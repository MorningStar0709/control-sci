# 5. EXAMPLE

In this section, we illustrate the proposed dynamic STC mechanism with a numerical example from the literature. The system that we consider is the same as in (Theodosis and Dimarogonas, 2018, Example 2). The dynamics of the plant are

$$\dot {x} = - x \sin^ {2} (x ^ {2}) + \hat {u} \cos (x ^ {2})$$

and we consider the state feedback $u \ = \ - x \cos ( x ^ { 2 } )$ . As in (Theodosis and Dimarogonas, 2018, Example 2), we consider $\tau _ { \mathrm { m a d } } ~ = ~ 0 . 0 0 0 4 \mathrm { s }$ . Using cos $\begin{array} { l l } { \displaystyle \big ( ( x + e ) ^ { 2 } \big ) } & { = } \end{array}$ $\cos \left( x ^ { 2 } \right) \ - \ a _ { 1 } \left( e ^ { 2 } \ + \ 2 x e \right) \ \mathrm { a n d } \ e \cos \left( \left( x + e \right) ^ { 2 } \right) \ = \ a _ { 2 } e$ for varying parameters $a _ { 1 } , \dot { a } _ { 2 } \in [ - 1 , 1 ]$ , we note that

$$
\begin{array}{l} \hat {u} = - \hat {x} \cos (\hat {x} ^ {2}) = (x + e) \cos ((x + e) ^ {2}) \\ = - x \cos (x ^ {2}) - e (a _ {1} (x e + 2 x ^ {2}) + a _ {2} e) \\ \end{array}
$$

leading to $f ( x , e ) = - x - e a _ { 3 } ( x , e )$ , where $a _ { 3 } ( x , e ) = a _ { 2 } +$ $a _ { 1 } \ : \big ( e x + 2 x ^ { 2 } \big )$ and to $g ( x , e ) = - \dot { f } ( x , e )$ . We consider the function $V ( x ) = 0 . 5 0 5 x ^ { 2 }$ and choose $c _ { \mathcal { X } } = 4 . 5 5$ which leads to the sets $\dot { \mathcal { X } } = [ - 3 , 3 ]$ and $\mathcal { E } = [ - 6 , 6 ]$ .
