We also allow the control policy to be randomized, that is ${ \mathcal { P } } ( \mathbb { U } )$ - valued, where ${ \mathcal { P } } ( \mathbb { U } )$ denotes the space of probability measures on U under the topology of weak convergence. An admissible control is a ${ \mathcal { P } } ( \mathbb { U } )$ -valued process $U _ { t }$ which satisfies the following non-anticipativity condition: for $s < t , W _ { t } - W _ { s }$ is independent of

$$\mathfrak {F} _ {s} := \text { the completion of } \sigma (X _ {0}, U _ {r}, W _ {r}: r \leq s) \text { relative to } (\mathfrak {F}, \mathbb {P}) .$$

Let U denote the space of all admissible controls. An admissible control is said to be Markov if $U _ { t } = v ( t , X _ { t } )$ for some measurable map $v : \mathbb { R } _ { + } \times \mathbb { R } ^ { d } \to \mathcal { P } ( \mathbb { U } )$ . Let $\mathfrak { U } _ { \mathfrak { m } }$ denote the space of all Markov controls .

To ensure existence and uniqueness of weak solutions of (2.1), we impose the following assumptions on the drift b and the diffusion matrix σ .

(A1) Continuity & Boundedness condition: The function b is jointly continuous in $( x , \zeta )$ and σ is locally Lipschitz continuous, i.e., for some constant $C _ { R } > 0$ depending on $R > 0$ , we have

$$\| \sigma (x _ {1}) - \sigma (x _ {2}) \| ^ {2} \leq C _ {R} | x _ {1} - x _ {2} | ^ {2} \tag {2.2}$$

for all $x _ { 1 } , x _ { 2 } \in \mathcal { B } _ { R }$ and $\zeta \in \mathbb { U }$ , where $\| \sigma \| : = \sqrt { \mathrm { T r } ( \sigma \sigma ^ { \mathsf { T } } ) }$ . Also, we assume that $b , \sigma$ are uniformly bounded, i.e., for some positive constant $\hat { C } _ { 0 }$ , we have

$$\sup _ {\zeta \in \mathbb {U}} | b (x, \zeta) | + \| \sigma (x) \| \leq \hat {C} _ {0} \quad \forall x \in \mathbb {R} ^ {d}.$$

(A2) Nondegeneracy: For some $\hat { C } _ { 1 } > 0$ , it holds that

$$\sum_ {i, j = 1} ^ {d} a ^ {i j} (x) z _ {i} z _ {j} \geq \hat {C} _ {1} | z | ^ {2} \quad \forall x \in \mathbb {R} ^ {d},$$

and for all $z = ( z _ { 1 } , \ldots , z _ { d } ) ^ { \mathsf { T } } \in \mathbb { R } ^ { d }$ , where $\begin{array} { r } { a : = \frac { 1 } { 2 } \sigma \sigma ^ { \mathsf { T } } } \end{array}$ .

In view of (A2), it is easy to see that $\sigma ^ { - 1 }$ exists and it is bounded . Using the Girsanov’s change of measure technique as in [1, Theorem 2.2.11] we have that under the hypotheses (A1)–(A2), for each U ∈ U there exists a unique weak solution of (2.1) (see also [23, Section 4, pp. 190-197], [30]) .
