# B. Quadratic Constraints (QCs) on $\Delta _ { p }$ and ϕ

With both the plant and the controller now in the form of an LTI system interconnected with an uncertainty or a nonlinearity, we characterize the non-LTI components—that is, $\Delta _ { p }$ and ϕ—by describing the relationship between the inputs and outputs of each block with quadratic constraints [16]–[19]. For simplicity and computational tractability, we use static quadratic constraints for both $\Delta _ { p }$ and ϕ.

Since $\Delta _ { p }$ is diagonal and has $\mathcal { L } _ { 2 }$ gain less than or equal to α, it satisfies the following integral quadratic constraint (IQC) for all $\Lambda _ { p } \in \mathbb { R } ^ { n _ { u } \times n _ { u } }$ where $\Lambda _ { p } \succeq 0$ and diagonal, $v _ { p } \in \mathcal { L } _ { 2 e } ^ { n _ { u } } , w _ { p } = \Delta _ { p } ( v _ { p } )$ , and $T \geq 0 ~ [ \bar { 1 } 6 ] $

$$
\int_ {0} ^ {T} \left[ \begin{array}{c} v _ {p} (t) \\ w _ {p} (t) \end{array} \right] ^ {\top} \left[ \begin{array}{c c} \alpha^ {2} \Lambda_ {p} & 0 \\ 0 & - \Lambda_ {p} \end{array} \right] \left[ \begin{array}{c} v _ {p} (t) \\ w _ {p} (t) \end{array} \right] \geq 0. \tag {6}
$$

To restrict $\Delta _ { p }$ to be LTI, one can include dynamic filters into (6). The class of filters for uncertain LTI dynamics is described in [16].

Since ϕ is applied elementwise and sector-bounded in [0, 1], it satisfies the following quadratic constraint for all $\Lambda _ { k } \in \mathbb { R } ^ { n _ { \phi } \times n _ { \phi } }$ where $\Lambda _ { k } \succeq 0$ and diagonal, $v _ { k } \in \mathbb { R } ^ { n _ { \phi } }$ , and $w _ { k } = \phi ( v _ { k } ) \ [ 2 0 ]$ :

$$
\left[ \begin{array}{l} v _ {k} \\ w _ {k} \end{array} \right] ^ {\top} \left[ \begin{array}{c c} 0 & \Lambda_ {k} \\ \Lambda_ {k} & - 2 \Lambda_ {k} \end{array} \right] \left[ \begin{array}{l} v _ {k} \\ w _ {k} \end{array} \right] \geq 0. \tag {7}
$$
