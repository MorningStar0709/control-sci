# 4.4.3 Simulation Results

We consider a simulation example with N = 8 rigid bodies in SE(3), described by ellipsoids with axes lengths $l _ { x , i } = 0 . 5 \mathrm { m } , l _ { y , i } = 0 . 3 \mathrm { m } , l _ { z , i } = 0 . 2 \mathrm { m } , \forall i \in \mathcal { N }$ The initial poses are (in m)

$$
\begin{array}{l} p _ {1} = [ 3, 3, 0 ] ^ {\top}, \quad p _ {2} = - [ 3, 3, 0 ] ^ {\top} \\ p _ {3} = [ 3, - 3, 0 ] ^ {\top}, \quad p _ {4} = [ - 3, 3, 0 ] ^ {\top} \\ p _ {5} = [ 3, 3, 3 ] ^ {\top}, \quad p _ {6} = - [ 3, 3, 3 ] ^ {\top} \\ p _ {7} = [ 3, - 3, 3 ] ^ {\top}, \quad p _ {8} = [ - 3, 3, - 3 ] ^ {\top} \\ \end{array}
\zeta_ {1} = \zeta_ {8} = [ 0. 7 6 9, 0. 1 6 9 6, 0. 6 1 5 3, 0. 0 3 5 8 ] ^ {\top}\zeta_ {2} = \zeta_ {6} = [ 0. 8 4 8 8, - 0. 3 9 1 3, - 0. 0 5 9 8, - 0. 3 5 0 5 ] ^ {\top}\zeta_ {3} = \zeta_ {5} = [ 0. 7 6 3 8, - 0. 5 2 8 3, - 0. 3 2 7 5, - 0. 1 7 3 8 ] ^ {\top}\zeta_ {4} = \zeta_ {7} = [ 0. 7 2 5 7, 0. 3 0 8 1, 0. 3 7 1 4, 0. 4 9 0 4 ] ^ {\top}$$

We consider that $\phi _ { s } ( x _ { s } )$ describes an independent multi-agent navigation objective, with desired configurations as

$$p _ {1 _ {\mathrm{d}}} = p _ {2}, p _ {2 _ {\mathrm{d}}} = p _ {1}, p _ {3 _ {\mathrm{d}}} = p _ {4}, p _ {4 _ {\mathrm{d}}} = p _ {3},p _ {5 _ {\mathrm{d}}} = p _ {6}, p _ {6 _ {\mathrm{d}}} = p _ {5}, p _ {7 _ {\mathrm{d}}} = p _ {8}, p _ {8 _ {\mathrm{d}}} = p _ {7},$$

and $\zeta _ { i _ { \mathrm { d } } } ~ = ~ [ 1 , 0 , 0 , 0 ] ^ { \top } , ~ \forall i ~ \in ~ \mathcal { N }$ . We set the errors $e _ { p _ { i } } : = p _ { i } - p _ { i _ { \mathrm { d } } }$ and $e _ { \zeta _ { i } } : = [ e _ { \varphi _ { i } } , e _ { \epsilon _ { i } } ^ { \top } ] ^ { \top } : = \zeta _ { i _ { \mathrm { d } } } \cdot \zeta _ { i } ^ { + }$ , and $e _ { \varphi _ { i } } , e _ { \epsilon _ { i } }$ are the scalar and vector parts, respectively, of the quaternion error (see Section 2.2 ). The desired quaternion configuration is achieved when $e _ { \zeta _ { i } } = [ \pm 1 , 0 , 0 , 0 ] ^ { \top }$ and hence the function $\phi _ { s } ( x _ { s } )$ is chosen as

$$\phi_ {s} = \sum_ {i \in \mathcal {N}} \left(\frac {1}{2} \| p _ {i} - p _ {i _ {\mathrm{d}}} \| ^ {2} + 1 - e _ {\varphi_ {i}} ^ {2}\right),$$

with

$$\dot {\phi} _ {s} = \sum_ {i \in \mathcal {N}} \left((p _ {i} - p _ {i _ {\mathrm{d}}}) ^ {\top} \dot {p} _ {i} - e _ {\varphi_ {i}} e _ {\epsilon_ {i}} ^ {\top} \omega_ {i}\right).$$
