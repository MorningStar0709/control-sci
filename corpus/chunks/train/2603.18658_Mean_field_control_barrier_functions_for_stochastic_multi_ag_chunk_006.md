$$D \int_ {\Omega} \delta_ {\rho} \mathcal {H} \Delta \rho d \mathbf {x} = D \int_ {\Omega} \Delta (\delta_ {\rho} \mathcal {H}) \rho d \mathbf {x}. \tag {12}$$

Combining (11) and (12) with (10), we obtain

$$
\begin{array}{l} \frac {\mathrm{d}}{\mathrm{d} t} \mathscr {H} (t, \rho (t, \mathbf {x})) = \int_ {\Omega} \nabla \delta_ {\rho} \mathscr {H} (t, \rho) \cdot \rho (t, \mathbf {x}) \mathbf {w} (t, \mathbf {x}, \mathbf {u}) \mathrm{d} \mathbf {x} \\ + D \int_ {\Omega} \Delta \left(\delta_ {\rho} \mathcal {H} (t, \rho)\right) \rho (t, \mathbf {x}) d \mathbf {x} + \partial_ {t} \mathcal {H} (t, \rho) \tag {13} \\ \end{array}
$$

Enforcing (7) then yields the MF-CBF constraint

$$
\begin{array}{l} \int_ {\Omega} \nabla \delta_ {\rho} \mathscr {H} (t, \rho) \cdot \rho (t, \mathbf {x}) \mathbf {w} (t, \mathbf {x}, \mathbf {u}) d \mathbf {x} \geq \\ - D \int_ {\Omega} \Delta \left(\delta_ {\rho} \mathcal {H} (t, \rho)\right) \rho (t, \mathbf {x}) d \mathbf {x} - \partial_ {t} \mathcal {H} (t, \rho) - \alpha (\mathcal {H} (t, \rho)), \tag {14} \\ \end{array}
$$

which defines the set of safe control inputs in (8), concluding the proof. □

Finally, given a nominal control input $\mathbf { u } ( t , \mathbf { x } )$ , its safetyadjusted counterpart $\mathbf { u } ^ { * } ( t , \mathbf { x } )$ can be computed by solving:

$$\mathbf {u} ^ {*} (t, \mathbf {x}) = \underset {\mathbf {u} ^ {\prime}} {\arg \min} \mathcal {L} [ \mathbf {u} ^ {\prime} (t, \mathbf {x}) - \mathbf {u} (t, \mathbf {x}) ] \tag {15}\text { s.t. } \quad \mathbf {u} ^ {\prime} \in \mathscr {K} _ {\mathrm{CBF}} (t, \rho),$$

where $\mathcal { L }$ is a quadratic functional cost such as the squared ${ \mathcal { L } } ^ { 2 } ( \Omega ) { \mathrm { - n o r m } }$ .

We note that (8) closely resembles its microscopic counterpart (3). Specifically, both the constraints involve a term related to the deterministic drift and a term related to diffusion arising from stochasticity.
