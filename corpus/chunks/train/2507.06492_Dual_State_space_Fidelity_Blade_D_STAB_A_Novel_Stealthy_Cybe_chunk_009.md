# A. High-fidelity Advesarial MPC

To execute the D-STAB attack, the attacker first defines a specific adversarial objective and designs corresponding constraints in the high-fidelity state-space model, while ensuring that the constraints perceived by the defender reflect normal system operations. These high-fidelity adversarial constraints are crafted based on domain-specific knowledge, leveraging the attacker’s information advantage over the defender.

The attacker then solves the high-fidelity adversarial MPC problem to generate a sequence of adversarial control inputs. The high-fidelity adversarial MPC problem is formulated as

follows:

$$
\begin{array}{l} \min _ {u _ {0: N - 1 | t}} J (x _ {H, t}; \theta_ {H}) := \sum_ {k = 0} ^ {N - 1} g _ {H} (x _ {H, k | t}, u _ {k | t}; \theta_ {H}) \\ + g _ {H, N} (x _ {H, N | t}; \theta_ {H}) \\ \end{array}
$$

subject to: $x _ { H , k + 1 | t } = f _ { H , k | t } ( x _ { H , k | t } , u _ { k | t } ) , \ x _ { H , 0 | t } = x _ { H , t } ,$

$$w _ {A} (x _ {H, k | t}) \geq 0,w _ {H, k \mid t} (x _ {H, k \mid t}, u _ {k \mid t}) \leq 0 \text { for } k = 0, 1, \dots , N - 1w _ {H, N \mid t} (x _ {H, N \mid t}) \leq 0, \tag {1}$$

where $x _ { H , k | t } \in \mathbb { R } ^ { n _ { H } }$ represents the high-fidelity system state at time step t, and prediction step $k ,$ and $u _ { k \mid t } \in \mathbb { R } ^ { m }$ is the control input. The function $f _ { H , k | t } : \mathbb { R } ^ { n _ { H } } \times \dot { \mathbb { R } } ^ { m _ { H } } \to \mathbb { R } ^ { n _ { H } }$ denotes the high-fidelity system dynamics, while $g _ { H } : \mathbb { R } ^ { n _ { H } } \times$ $\mathbb { R } ^ { m _ { H } }  \mathbb { R }$ and $g _ { H , N } : \mathbb { R } ^ { n _ { H } }  \mathbb { R }$ are the instantaneous and terminal cost functions parameterized by $\theta _ { H } \in \mathbb { R } ^ { r _ { H } }$ . The constraint $w _ { A } : \mathbb { R } ^ { n _ { H } }  \mathbb { R }$ represents the attacker-designed conditions that reflect specific adversarial goals, and $w _ { H , k \mid t } :$ : $\mathbb { R } ^ { n _ { H } } \times \mathbb { R } ^ { m _ { H } } \to \mathbb { R }$ and $w _ { H , N | t } : \mathbb { R } ^ { n _ { H } }  \mathbb { R }$ are constraints ensuring stealth by maintaining normal operations in the lowfidelity state-space representation.
