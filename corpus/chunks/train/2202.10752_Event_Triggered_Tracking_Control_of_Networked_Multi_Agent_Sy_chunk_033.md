# C. Tracking Performance Analysis

Now we are ready to state the main results of this section.

Theorem 1: Consider the system (18) and let Assumptions 1-6 hold. If the MASP $T _ { i }$ and the MAD $\Delta _ { i }$ satisfy

$$\gamma_ {i 0} \phi_ {i 0} (\tau_ {i}) \geq (1 + \varrho_ {i 1}) \bar {\lambda} _ {i} ^ {2} \gamma_ {i 1} \phi_ {i 1} (0), \quad \tau_ {i} \in [ 0, T _ {i} ], \tag {36a}\gamma_ {i 1} \phi_ {i 1} (\tau_ {i}) \geq (1 + \varrho_ {i 0}) \gamma_ {i 0} \phi_ {i 0} (\tau_ {i}), \quad \tau_ {i} \in [ 0, \Delta_ {i} ], \tag {36b}$$

where $\phi _ { i b _ { i } }$ is the solution to (35) satisfying $\phi _ { i b _ { i } } ( 0 ) , \phi _ { i b _ { i } } ( T _ { i } ) >$ 0, then the system (18) is ISS from $( e _ { \mathrm { r } } , e _ { \mathrm { f } } , w )$ to $( \eta , e _ { \mathrm { a } } )$ . That is, there exist $\beta \in \mathcal { K L L }$ and $\varphi _ { 1 } ~ \in ~ \mathcal { K } _ { \infty }$ such that for all $( t , j ) \in \mathbb { R } _ { \geq 0 } \times \mathbb { N } .$ ,

$$
\begin{array}{l} \left| \left(\eta (t, j), e _ {\mathrm{a}} (t, j)\right) \right| \leq \beta \left(\left| \mathfrak {X} (0, 0) \right|, t, j\right) + \varphi_ {1} \left(\left\| e _ {\mathrm{f}} \right\| _ {(t, j)}\right) \\ + \varphi_ {2} (\| e _ {\mathrm{r}} \| _ {(t, j)}) + \varphi_ {3} (\| w \| _ {(t, j)}). \tag {37} \\ \end{array}
$$

The proof is presented in Appendix B. In particular, a novel Lyapunov function is proposed to investigate the effects on the sampling, the designed ETMs and time delays on the tracking performance. Comparing with existing works [4], [16], [17], [39] on NCS and [18], [19] on MAS, the event-triggered tracking control problem is studied here for networked MAS under both decentralized ETMs and network constraints. Theorem 1 implies the convergence of the tracking error to a region around the origin, and the size of the convergence region depends on the network-induced error $( e _ { \mathrm { r } } , e _ { \mathrm { f } } )$ and the external disturbance w. If the external disturbance is not considered here, then the convergence region is only related to $( e _ { \mathrm { r } } , e _ { \mathrm { f } } )$ . If the feedforward control inputs are transmitted directly to the agents and reference system, then $e _ { \mathrm { f } } = 0$ and $\varphi _ { 1 } \equiv 0$ , and thus the convergence region can be further smaller.
