Compiling the three previous propositions, we obtain that the infinite horizon LQG-GMFGs under study is solvable with explicit costs at equilibrium, whenever there exists processes $\{ z _ { t } ^ { \alpha } , s _ { t } ^ { \alpha } , q _ { t } ^ { \alpha } , \alpha ~ \in ~ [ 0 , 1 ] , ~ t ~ \in ~ [ 0 , \infty ) \}$ ⊂ $C _ { b } \left( \left[ 0 , \mathrm { \tilde { \infty } } \right) \right) \times L ^ { 2 } \left( \mathrm { \tilde { [ 0 , 1 ] } } \right)$ that are solutions to the following ODEs:
