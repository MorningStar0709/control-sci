# Step 4. Energy Estimates for ϕ.

Following the proof of Theorem 6.2.2. of [14], we have that:

$$| | \nabla_ {y} \phi | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} ^ {2} \leq B [ \phi , \phi ; \lambda ] + C | | \phi | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} ^ {2}.$$

for some constant $C > 0$ . If from before $\omega > C ,$ , then:

$$| | \nabla_ {y} \phi | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} ^ {2} \leq B [ \phi , \phi ; \lambda ] + \omega | | \phi | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} ^ {2}.$$

If not, simply re-select $\omega = C ,$ , and $\mathcal { R } ( \lambda ; \mathcal { L } _ { n } ( t ; u , w ) )$ will be invertible for $\lambda \geq \omega$ , and the same conclusion as above follows.
