# Lemma 1 (Bounded real lemma [39, Corollary 13.24])

Consider the transfer function $\mathbf { G } ( s ) \ : = \ : { \cal C } ( s I - \dot { A } ) ^ { - 1 } B ,$ , with $A \in \mathrm { R } ^ { n \times n } , B \in \mathrm { R } ^ { \bar { n } \times m }$ , and $C \in \mathrm { R } ^ { p \times n } . \ T h e n , f o r$ any $\beta > 0$ , the following statements are equivalent.

(1) The matrix A is Hurwitz and $\| \mathbf { G } ( s ) \| _ { \mathcal { H } _ { \infty } } < \beta .$   
(2) There exists an $X \succ 0$ satisfying the Riccati inequality

$$A X + X A ^ {\mathsf {T}} + \beta^ {- 2} X C ^ {\mathsf {T}} C X + B B ^ {\mathsf {T}} \prec 0. \tag {12}$$

(3) The Riccati equation

$$A X + X A ^ {\mathsf {T}} + \beta^ {- 2} X C ^ {\mathsf {T}} C X + B B ^ {\mathsf {T}} = 0 \tag {13}$$

admits a unique stabilizing solution X $( i . e . , \ A +$ $\beta ^ { - 2 } X C ^ { \mathsf { T } } C$ is Hurwitz) satisfying $X \succeq 0 . \ I f \left( A , B \right)$ is further controllable, we have $X \succ 0$ .

From [39, Corollary 13.13], the stabilizing solution X to (13) is also minimal, i.e., $X \preceq { \hat { X } }$ for any other symmetric solution $\hat { X }$ to (13).
