# 5.1 Lower Bound Results

We first characterize the fundamental limitations of any algorithm for the problem of learning Kalman filtering for state estimation; the proof of Theorem 5.1 is included in Appendix D.

Theorem 5.1. Consider any (potentially randomized) algorithm for learning the Kalman filter for state estimation that only has access to the outputs $y _ { 0 } , \ldots , y _ { t - 1 }$ when designing the state estimate $\hat { x } _ { t }$ for each time step $t \geq 0$ . Let $\sigma _ { w } \in \mathbb { R } _ { > 0 }$ . Then, for any $T \geq 2$ , there exists a scalar and stable LTI system such that $\mathtt { R _ { x } } ( T )$ defined in (21) satisfies

$$\mathbb {E} \left[ \mathrm{R} _ {\mathrm{x}} (T) \right] \geq 6 T \sigma_ {w} - \frac {5 6 + 5 0 \sigma_ {w}}{1 8 7 5 \sigma_ {w}}, \tag {36}$$

where $\mathbb { E } [ \cdot ]$ is taken with respect to the potential randomness of the algorithm and $\{ w _ { t } \} _ { t \ge 0 } , \{ v _ { t } \} _ { t \ge 0 }$ . Additionally, consider any algorithm that returns $\hat { x } _ { t }$ with $\| \hat { x } _ { t } \| \le c _ { A }$ for all $t \geq 0$ almost surely. Then, for any $T \geq 3$ and any $\delta \in ( 0 , 1 )$ , it holds with probability at least $1 - \delta$ that

$$\mathbb {R} _ {\mathbf {x}} (T) \geq 6 T \sigma_ {w} - \frac {5 6 + 5 0 \sigma_ {w}}{1 8 7 5 \sigma_ {w}} - \mathcal {O} \left(\sqrt {T (c _ {\mathcal {A}} + \log^ {1 . 5} T) \log \frac {3}{\delta}}\right). \tag {37}$$

Proof idea. The proof hinges on the fact that for partially observed systems, there can be multiple systems with different system matrices $A , C$ that yield the same output history $y _ { 0 } , y _ { 1 } , \ldots$ (given the same realization of the noise sequences $\{ w _ { t } \} _ { t \ge 0 }$ and $\{ v _ { t } \} _ { t \geq 0 } )$ . Hence, any algorithm (given only access to $y _ { 0 } , y _ { 1 } , \dotsc \big )$ cannot perform well simultaneously on all such systems, i.e., any algorithm must incur large regret for at least one of such systems.
