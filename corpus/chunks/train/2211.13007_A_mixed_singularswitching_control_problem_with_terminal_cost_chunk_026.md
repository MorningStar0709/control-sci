Lemma 3.2. If (H1), (H3) and (H4) hold, there exists a unique non-negative solution u to the Dirichlet problem (3.4) such that $\overline { { u } } _ { \ell , \iota } \in \mathrm { C } ^ { 4 , \alpha } ( \overline { { \mathcal { O } } } )$ for each $( \ell , \iota ) \in \mathbb { M } \times \mathbb { I }$ .

Though due to the results of Sweers [17], under weak assumptions in $h _ { \iota }$ and the parameters of $\mathcal { L } _ { \ell , \iota }$ , that guaranteed the existence of the non-negative solution v to (3.4) in a strong sense, it is possible to verify that, by fixed point arguments, (3.4) has a classical solution under the assumptions imposed above; see (H1), (H3) and (H4). The reader can find the proof of Lemma 3.2 in the appendix; see Subsection A.1, since this is similar to the proof of Proposition 1.4.

Proof of Lemma 3.1. By Lemma 3.2, let us consider $\overline { { u } } \in \mathcal { C } _ { m , n } ^ { 4 , \alpha }$ as the unique non-negative solution to the Dirichlet problem (3.4), which is a super-solution to the NPDS (1.13). Then, from Lemma 2.4, we get that $u _ { \ell , \iota } \leq \overline { { u } } _ { \ell , \iota }$ ι for $( \ell , \iota ) \in \mathbb { M } \times \mathbb { I } .$ . Let us assume (H6) holds. It implies that u is a sub-solution to the NPDS (1.13). Therefore $\underline { { u } } _ { \ell , \iota } \le u _ { \ell , \iota }$ for $( \ell , \iota ) \in \mathbb { M } \times \mathbb { I }$ Now, we take a point x in ∂O and a unit vector $\mathbb { n } _ { x }$ outside of O such that it is not tangent to O. Defining $\mathbb { v } = - \mathbb { n } _ { x }$ , we have that

$$\partial_ {\mathbb {v}} u _ {\ell , \iota} (x) = \lim _ {\varrho \rightarrow 0} \frac {u _ {\ell , \iota} (x + \varrho \mathbb {v}) - f _ {\iota} (x)}{\varrho} \geq \lim _ {\varrho \rightarrow 0} \frac {\underline {{u}} _ {\ell , \iota} (x + \varrho \mathbb {v}) - f _ {\iota} (x)}{\varrho} = \partial_ {\mathbb {v}} \underline {{u}} _ {\ell , \iota} (x).$$

In the same way it gives $\partial _ { \mathbb { v } } u _ { \ell , \iota } ( x ) \leq \partial _ { \mathbb { v } } \overline { { u } } _ { \ell , \iota } ( x )$ . Since, $\mathbb { v } = - \mathbb { n } _ { x }$ and $u _ { \ell , \iota } , \underline { { u } } _ { \ell , \iota } , \overline { { u } } _ { \ell , \iota } \in \mathrm { C } ^ { 1 } ( \overline { { \mathcal { O } } } )$ , it yields

$$\langle \mathbb {n} _ {x}, \mathrm{D} ^ {1} \overline {{u}} (x) \rangle \leq \langle \mathbb {n} _ {x}, \mathrm{D} ^ {1} u (x) \rangle \leq \langle \mathbb {n} _ {x}, \mathrm{D} ^ {1} \underline {{u}} (x) \rangle . \tag {3.5}$$
