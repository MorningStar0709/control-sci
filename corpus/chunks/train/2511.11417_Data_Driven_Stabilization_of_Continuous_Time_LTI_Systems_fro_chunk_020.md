# A. Bound on the Filtered Process Noise

Suppose that a scalar $\delta _ { w } \ge 0$ is known such that

$$\| w (\cdot) \| _ {2, [ 0, T ]} ^ {2} \leq \delta_ {w}. \tag {50}$$

Then, a bound on $\| d _ { w } ( \cdot ) \| _ { \mathcal { L } _ { 2 } , [ 0 , T ] }$ is obtained by finding an upper bound $\gamma > 0$ of the $\dot { \mathcal { L } } _ { 2 } [ \dot { 0 } , T ]$ gain of system (46), which ensures:

$$\left\| d _ {w} (\cdot) \right\| _ {2, [ 0, T ]} \leq \gamma \| w (\cdot) \| _ {2, [ 0, T ]} \leq \gamma \sqrt {\delta_ {w}}. \tag {51}$$

By [23, Thm. 3.7.4], γ is such that the following differential Riccati equation:

$$\dot {W} = - \tilde {\Lambda} ^ {\top} W - W \tilde {\Lambda} - \gamma^ {- 2} W E E ^ {\top} W - C ^ {\top} C, \quad W (T) = 0 _ {n p}, \tag {52}$$

has a solution over [0, T ]. Since all matrices in (46) are known, (52) can be used to test $\gamma .$ Let $\gamma _ { \infty } > 0$ be such that $\| \mathcal { G } _ { w } \| _ { \infty } < \gamma _ { \infty }$ . Then, a conservative choice for $\gamma$ is $\gamma _ { \infty }$ , because by [23, Lemma 3.7.7] (52) with $\gamma = \gamma _ { \infty }$ has a solution over [0, T ] for all finite T . Otherwise, a tight bound can be found by searching in the interval $[ 0 , \gamma _ { \infty } ]$ for $\gamma > 0$ as small as possible such that (52) has a solution over [0, T ].
