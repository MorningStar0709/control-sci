Assumption 1 ensures that g(φ) is not tangential to the level set of the sliding surface functional U(φ). If Assumption 1 does not hold, then higher-order sliding surface functionals can be introduced [30, 31], and then the following analysis can be proceeded similarly. This will be further discussed in Section 5.3.

With Assumption 1, we define the matrix functional:

$$\mathbf {M} (\phi) := I - \frac {g (\phi) \mathbf {G} ^ {\top} (\phi) \mathbf {H} (\phi)}{\| \mathbf {G} (\phi) \| ^ {2}}. \tag {45}$$

The next proposition shows the properties of $\mathbf { M } ( \phi )$ , which extends the results in [28] into the time-delay case and lays a foundation for the control design.

Proposition 7. The following statements are valid:

$$\mathbf {M} (\phi) f (\phi) = \mathbf {J} _ {1} (\phi) \mathbf {H} ^ {\top} (\phi), \tag {46}(I - \mathbf {M} (\phi)) f (\phi) = \mathbf {J} _ {2} (\phi) \mathbf {H} ^ {\top} (\phi) + \mathbf {J} _ {3} (\phi) \mathbf {H} ^ {\top} (\phi), \tag {47}$$

where $\mathbf { J } _ { 1 } ( \phi ) = - 2 \mathbf { J } _ { 2 } ( \phi )$ , and

$$\mathbf {J} _ {2} (\phi) = \frac {g (\phi) \mathbf {G} ^ {\top} (\phi) f ^ {\top} (\phi) - f (\phi) \mathbf {G} (\phi) g ^ {\top} (\phi)}{2 \| \mathbf {G} (\phi) \| ^ {2}},\mathbf {J} _ {3} (\phi) = \frac {f (\phi) \mathbf {G} (\phi) g ^ {\top} (\phi) + g (\phi) \mathbf {G} ^ {\top} (\phi) f ^ {\top} (\phi)}{2 \| \mathbf {G} (\phi) \| ^ {2}}.$$

Proof. Since ${ \bf H } ( \phi ) f ( \phi ) \in$ R and $\mathbf { G } ( \phi ) = \mathbf { H } ( \phi ) g ( \phi ) \in \mathbb { R } ^ { 1 \times m } ,$ , we have that $\mathbf { H } ( \phi ) f ( \phi ) = f ^ { \top } ( \phi ) \mathbf { H } ^ { \top } ( \phi )$ and $\| \mathbf { G } ( \phi ) \| ^ { 2 } = \mathbf { G } ( \phi ) \mathbf { G } ^ { \top } ( \phi )$ .

Define $\mathbf { J } ( \phi ) : = f ( \phi ) \mathbf { G } ( \phi ) g ^ { \top } ( \phi )$ , and we have from (45) that

$$\mathbf {M} (\phi) f (\phi) = f (\phi) - \frac {g (\phi) \mathbf {G} ^ {\top} (\phi) \mathbf {H} (\phi) f (\phi)}{\| \mathbf {G} (\phi) \| ^ {2}}= \frac {f (\phi) \| \mathbf {G} (\phi) \| ^ {2} - g (\phi) \mathbf {G} ^ {\top} (\phi) \mathbf {H} (\phi) f (\phi)}{\| \mathbf {G} (\phi) \| ^ {2}}= \frac {f (\phi) \mathbf {G} (\phi) \mathbf {G} ^ {\top} (\phi) - \mathbf {J} ^ {\top} (\phi) \mathbf {H} ^ {\top} (\phi)}{\| \mathbf {G} (\phi) \| ^ {2}}= \frac {(\mathbf {J} (\phi) - \mathbf {J} ^ {\top} (\phi)) \mathbf {H} ^ {\top} (\phi)}{\| \mathbf {G} (\phi) \| ^ {2}}.$$
