In G , a collection of pairwise distinct nodes $v _ { k _ { 0 } } , v _ { k _ { 1 } } , \ldots , v _ { k _ { m } }$ so that $\left( v _ { k _ { \ i - 1 } } , v _ { k _ { \ i } } \right) \ \in \ \mathcal { E } , \ i \ = \ 1 , \ldots , m$ is called a (directed) path from node $\boldsymbol { v } _ { k _ { 0 } }$ to node $v _ { k _ { m } }$ . If G admits paths between every two distinct nodes, we say that it is strongly connected. Note that strong connectivity degenerates into connectivity when G is undirected. $\mathcal { G }$ is termed quasi-strongly connected, if there is at least one vertex, called root, having paths to all other vertices. $\mathcal { G }$ is weakly connected provided the undirected graph, induced by replacing each edge of G with an undirected edge, is connected. Notice that the weak connectivity is a rather general connectivity condition, including the strong and quasi-strong connectivity as special cases. A maximal strongly connected subgraph of G with no incoming edges from nodes outside is called a closed strong component (CSC) of G .
