# 2. PROBABILISTIC SETTING

Let $( \Omega , \mathcal { F } , \mathbb { F } : = \{ \mathcal { F } _ { t } \} _ { t \geq 0 } , \mathbb { P } )$ be a filtered probability space which satisfies the usual conditions, on which it is defined a one-dimensional F-Brownian motion $\{ W _ { t } \} _ { t \ge 0 }$ and an independent irreducible Fadapted continuous-time Markov chain $\{ Y _ { t } \} _ { t \ge 0 }$ . The Markov chain Y has state space $\mathbb { Y } : = \{ 1 , . . . , d \}$ and transition matrix $\mathbb { Q } : = \{ q _ { i j } \} _ { 1 \leq i , j \leq d }$ . The transition rates are such that $\kappa _ { i } : = - q _ { i i } > 0$ and the condition $\textstyle \sum _ { i \in \mathbb { Y } } q _ { i j } = 0$ holds for every $i \in \mathbb { Y }$ . Accordingly, setting $P _ { i j } ( t ) : = \mathbb { P } ( Y _ { t } = j | Y _ { 0 } = i )$ , we have $\begin{array} { r } { P _ { i j } ^ { \prime } ( t ) = \sum _ { k \in \mathbb { Y } } q _ { i \ell } P _ { \ell j } ( t ) } \end{array}$ , for $i , j \in \mathbb { Y }$ . It also follows that Y admits a stationary distribution,

whose i-th component is given by

$$p (i) := \frac {\kappa_ {i}}{\sum_ {j = 1} ^ {d} \kappa_ {j}}. \tag {2.1}$$

For given Borel-measurable functions $b : \mathbb { R } \times \mathbb { Y } \to \mathbb { R } , \sigma : \mathbb { R } \times \mathbb { Y } \to ( 0 , \infty )$ , we introduce the process $X ^ { 0 }$ with state space $\mathcal { T } : = ( \underline { { x } } , \overline { { x } } )$ and dynamics

$$d X _ {t} ^ {0} = b (X _ {t} ^ {0}, Y _ {t}) d t + \sigma (X _ {t} ^ {0}, Y _ {t}) d W _ {t}, \quad (X _ {0} ^ {0}, Y _ {0}) = (x, i) \in \mathcal {I} \times \mathbb {Y}. \tag {2.2}$$

Here, $- \infty \leq \underline { { x } } < \overline { { x } } \leq + \infty$ are the boundary points of the state space of $X ^ { 0 }$ . Let then

$$\mathcal {A} := \{\{\xi_ {t} \} _ {t \geq 0}, \text { F - adapted, with bounded - variation, left - continuous, } \xi_ {0} = 0, \text { a.s. } \}$$

and notice that any $\xi \in { \mathcal { A } }$ admits the Jordan decomposition $\xi = \xi ^ { + } - \xi$ −, for $\xi ^ { \pm }$ nondecreasing. $\mathrm { A l s o , }$ let $\{ | \xi | _ { t } \} _ { t \ge 0 } : = \xi ^ { + } + \xi ^ { - }$ denote the variation of $\xi \in { \mathcal { A } }$ . Then, for given $\xi \in { \mathcal { A } }$ and $x \in \mathcal { Z }$ , we introduce the process $X ^ { \xi }$ with dynamics

$$d X _ {t} ^ {\xi} = b (X _ {t} ^ {\xi}, Y _ {t}) d t + \sigma (X _ {t} ^ {\xi}, Y _ {t}) d W _ {t} + d \xi_ {t} ^ {+} - d \xi_ {t} ^ {-}, \quad (X _ {0} ^ {\xi}, Y _ {0}) = (x, i) \in \mathcal {I} \times \mathbb {Y}. \tag {2.3}$$

The following assumption in particular ensures that there exists a unique strong solution to (2.3), for every $\xi \in { \mathcal { A } }$ and $( x , i ) \in \mathcal { I } \times \mathbb { Y }$ (hence, in particular, for $\xi \equiv 0 ;$ see Theorem 7 Chapter V in [48]). In the following, we shall denote such strong solutions by $( X ^ { x , \xi } , Y ^ { i } )$ and $( X ^ { x , 0 } , Y ^ { i } )$ , when needed.
