Algorithm 1: Open-loop trajectory optimization via model-free ILQR   
Input: Initial State - $\Phi_0$ , System parameters - $\mathcal{P}$ ; $m \leftarrow 1$ . /* Initialize the iteration number $m$ to 1.*/
forward_pass_flag = true.
/* Run until the difference in costs between subsequent iterations is less an $\epsilon$ fraction of the former cost.*/
while $m == 1$ or $(cost(\mathbb{T}_{nom}^m) / cost(\mathbb{T}_{nom}^{m-1})) < 1 + \epsilon$ do
    /*Each iteration has a backward pass followed by a forward pass.*/ $\{k_{0:N-1}^m, K_{0:N-1}^m\}$ , backward_pass_success_flag =
    Backward Pass( $\mathbb{T}_{nom}^m$ , $\mathcal{P}$ ).
    if backward_pass_success_flag == true then $\mathbb{T}_{nom}^{m+1}$ , forward_pass_flag = Forward
    Pass( $\mathbb{T}_{nom}^m, \{k_{0:N-1}^m, K_{0:N-1}^m\}$ , $\mathcal{P}$ ).
    while forward_pass_flag == false do $\mathbb{T}_{nom}^{m+1}$ , forward_pass_flag = Forward
    Pass( $\mathbb{T}_{nom}^m, \{k_{0:N-1}^m, K_{0:N-1}^m\}$ , $\mathcal{P}$ ).
    Reduce $\alpha$ from $\mathcal{P}$ .
    end while
    end if
    else
    | Increase $\mu$ from $\mathcal{P}$ . /* Regularization step */
    end if $m \leftarrow m + 1$ . $\mathbb{T}_{nom}^*\leftarrow \mathbb{T}_{nom}^{m+1}$ .
end while
return $\mathbb{T}_{nom}^*$

Algorithm 2: D2C Algorithm   
1) Solve the deterministic open-loop optimization problem for optimal open-loop control sequence and state trajectory $\left(\{\bar{U}_{t}^{*}\}_{t=0}^{T-1},\{\bar{\Phi}_{t}^{*}\}_{t=0}^{T}\right)$ using ILQR (Section III-A).
2) Linearize and identify the LTV system parameters $(\hat{A}_{t}, \hat{B}_{t})$ via least square (Section III-B).
3) Solve the Riccati equations for each step along the nominal trajectory for feedback gain $\{K_{t}\}_{t=0}^{T-1}$ .
4) Apply the closed-loop control policy,
while t < T do

$$U _ {t} = \bar {U} _ {t} ^ {*} + K _ {t} \delta \Phi_ {t},\Phi_ {t + 1} = f (\Phi_ {t}) + B _ {t} (U _ {t} + \epsilon w _ {t}),\delta \Phi_ {t + 1} = \Phi_ {t + 1} - \bar {\Phi} _ {t + 1} ^ {*} \tag {12}t = t + 1.$$

end while
