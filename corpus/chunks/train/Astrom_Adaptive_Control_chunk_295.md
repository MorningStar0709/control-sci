# DEFINITION 5.4 Class K function

A continuous function $\alpha: [0, a) \to [0, \infty)$ is said to belong to class $K$ if it is strictly increasing and $\alpha(0) = 0$ . It is said to belong to class $K_{\infty}$ if $a = \infty$ and $\alpha(r) \to \infty$ as $r \to \infty$ .
