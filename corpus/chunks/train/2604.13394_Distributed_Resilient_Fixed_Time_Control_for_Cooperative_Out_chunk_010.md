# D. Problem Statement

Consider a MAS composed of an exosystem and N agents. The dynamics of agent i are given by

$$\dot {x} _ {i} (t) = A _ {i} x _ {i} (t) + B _ {i} u _ {i} (t) + E _ {i} v (t), \tag {2}e _ {i} (t) = C _ {i} x _ {i} (t) + F _ {i} v (t),$$

where $x _ { i } ( t ) \in \mathbb { R } ^ { n _ { i } }$ and $u _ { i } ( t ) \in \mathbb { R } ^ { m _ { i } }$ i are its state and input, respectively, $e _ { i } ( t ) \in \mathbb { R } ^ { p _ { i } }$ is its regulated output, $n _ { i }$ and mi are two positive integers with $m _ { i } \leq n _ { i } , A _ { i } , B _ { i } , C _ { i } , E _ { i }$ and $F _ { i }$ are real matrices, $( A _ { i } , B _ { i } )$ is controllable with $B _ { i }$ being of full column rank; $v ( t ) \in \mathbb { R } ^ { q }$ denotes the reference input to be followed and/or the disturbance to be rejected, which is generated by the following exosystem,

$$\dot {v} (t) = S v (t), \tag {3}$$

where $S \in \mathbb { R } ^ { q \times q }$ .

This work aims to tackle the resilient fixed-time COR problem, formally defined below:

Resilient Fixed-Time COR Problem: Given the MAS (2)– (3) over directed graphs under DoS attacks, design a distributed controller such that the resulting closed-loop system is fixed-time stable at the origin when $v ( t ) \equiv 0 ;$ ; and the regulated output of each agent approaches zero in a fixed time, i.e.,

$$e _ {i} (t) = 0, \forall t \geq T, \tag {4}$$

where $T \in \mathbb { R } _ { + }$ + is independent of the initial states.

To tackle this problem, we introduce some assumptions below.

Assumption 1: The duration of DoS attacks satisfies

$$| \Pi_ {D} (t _ {0}, t) | \leq \nu_ {d} + \frac {t - t _ {0}}{p _ {d}}, \forall t \geq t _ {0},$$

where $\nu _ { d } > 0$ and $p _ { d } > 1 .$

Assumption 2: For directed graph ${ \mathcal { G } } ,$ there exists a spanning tree with node 0 as its root.

Assumption 3: The regulator equations

$$A _ {i} \Pi_ {i} + B _ {i} \Gamma_ {i} + E _ {i} = \Pi_ {i} S, \tag {5a}C _ {i} \Pi_ {i} + F _ {i} = 0, i \in \mathcal {V} _ {a}, \tag {5b}$$

have solutions $( \Pi _ { i } , \Gamma _ { i } )$ .

Remark 1: Assumption 1 ensures that DoS attack duration remains bounded by a proportion of the overall time interval [12, 19, 21]. Assumptions 2–3 are made as in [6, 19, 20, 37], which are necessary and standard in solving the problem of COR for MASs.

Before proceeding, we present the following lemma, which describes essential properties of matrix H related to the directed graph G.

Lemma 1: [42, 43] Let H be the matrix corresponding to the graph G under Assumption 2. The following statements are true:

(i) Each eigenvalue of H has a positive real part.

(ii) There is a matrix $K \triangleq \mathrm { d i a g } \{ k _ { 1 } , k _ { 2 } , \cdot \cdot \cdot , k _ { N } \} > 0$ such that $\mathcal { H } ^ { T } K + K \mathcal { H } - 2 I _ { N } \geq 0$ .
