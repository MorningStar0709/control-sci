Each RNN consists of two components, a Gated Recurrent Unit (GRU), c.f. Cho et al. (2014), and a single layer of neurons. The GRU is used to model the information flow between subsequent hidden states $h _ { k } , k = 0 , \ldots , N _ { t } ,$ , taking the inputs $\mathbf { \boldsymbol { s } } _ { k } = \left[ \hat { \mathbf { x } } _ { t _ { k } } , \mathbf { \boldsymbol { u } } _ { t _ { k } } , t _ { k } \right] ^ { T }$ into account. It is defined by learnable network parameters, comprising input weights W , recurrent weights R and biases b:

$$
\boldsymbol {W} := \left[ \begin{array}{c} \boldsymbol {W} _ {z} \\ \boldsymbol {W} _ {r} \\ \boldsymbol {W} _ {\tilde {h}} \end{array} \right], \quad \boldsymbol {R} := \left[ \begin{array}{c} \boldsymbol {R} _ {z} \\ \boldsymbol {R} _ {r} \\ \boldsymbol {R} _ {\tilde {h}} \end{array} \right], \quad \boldsymbol {b} := \left[ \begin{array}{c} \boldsymbol {b} _ {z} \\ \boldsymbol {b} _ {r} \\ \boldsymbol {b} _ {\tilde {h}} \end{array} \right]. \tag {5}
$$

An update gate $z _ { k }$ is used to add new information to the hidden state $h _ { k }$ , which acts like a temporal memory, while a reset gate $\mathbfit { r } _ { k }$ is used for successive removal of information, interpretable as an act of forgetting. Using these, a candidate state $\tilde { h } _ { k }$ is calculated:

$$\boldsymbol {z} _ {k} = \sigma_ {g} \left(\boldsymbol {W} _ {\boldsymbol {z}} \boldsymbol {s} _ {k} + \boldsymbol {b} _ {\boldsymbol {z}} + \boldsymbol {R} _ {\boldsymbol {z}} \boldsymbol {h} _ {k - 1}\right),\boldsymbol {r} _ {k} = \sigma_ {g} \left(\boldsymbol {W} _ {\boldsymbol {r}} \boldsymbol {s} _ {k} + \boldsymbol {b} _ {\boldsymbol {r}} + \boldsymbol {R} _ {\boldsymbol {r}} \boldsymbol {h} _ {k - 1}\right), \tag {6}\tilde {\boldsymbol {h}} _ {k} = \sigma_ {s} \left(\boldsymbol {W} _ {\tilde {\boldsymbol {h}}} \boldsymbol {s} _ {k} + \boldsymbol {b} _ {\tilde {\boldsymbol {h}}} + \boldsymbol {r} _ {k} \odot \left(\boldsymbol {R} _ {\tilde {\boldsymbol {h}}} \boldsymbol {h} _ {k - 1}\right)\right).$$
