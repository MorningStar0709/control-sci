Corollary 12.9 Suppose $N = [ { \frac { A | ~ B ~ ] } { C | ~ D ~ ] } }$ is stable and minimal, and X is the observability Gramian. Then N is an inner if and only if

(a) $D ^ { * } C + B ^ { * } X = 0$   
(b) $D ^ { * } D = I .$

A transfer matrix $N _ { \perp }$ is called a complementary inner factor (CIF) of $\textit { N i f } [ N { } N _ { \perp { } } ]$ is square and is an inner. The dual notion of the complementary co-inner factor is defined in the obvious way. Given an inner N, the following lemma gives a construction of its CIF. The proof of this lemma follows from straightforward calculation and from the fact that $C X ^ { + } X = C$ since Im $( I - X ^ { + } X ) \subset \ker ( X ) \subset \operatorname { K e r } ( C )$ .

Lemma 12.10 Let $N = [ { \frac { A | \ B \  } { C | \ D \ } } ]$ be an inner and X be the observability Gramian. Then a $C I F \ : N _ { \perp }$ is given by

$$
N _ {\perp} = \left[ \begin{array}{c c} A & - X ^ {+} C ^ {*} D _ {\perp} \\ \hline C & D _ {\perp} \end{array} \right]
$$

where $D _ { \perp }$ is an orthogonal complement of D such that $[ D \ D _ { \perp } ]$ is square and orthogonal.
