# III. MAIN RESULTS

The challenge in solving the cooperative tuning problem of multi-agent optimal control systems in (2) comes from two parts: first, each $L _ { i }$ here is a function of the trajectory of a dynamical system $\pmb { S } _ { i } ( \pmb \theta _ { i } )$ , which makes the whole optimization problem at least a bi-level optimization; second, the optimization goal is not just the minimization of each system’s own loss but the team-average loss, for which all tunable parameters need to be adjusted cooperatively. Motivated by these two challenges, in this section we will develop a method to solve (2) by a combination of consensus-based distributed optimization in [17] and a gradient generator in [7].
