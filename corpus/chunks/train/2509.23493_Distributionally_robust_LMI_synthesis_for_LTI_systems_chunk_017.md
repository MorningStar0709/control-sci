# A. Robust performance analysis for correlated disturbances

In this section, we derive the dual problem of problem (6) for the case of correlated disturbances. This dual takes the form of a robust performance analysis problem typically obtained from dissipativity theory.

Theorem 5.5: Let A be stable. Then, for the case $\mathcal { S } = \mathcal { S } _ { \mathrm { c o r } } .$ , (6) is equivalent to the optimization problem

$$\underset {\lambda \geq 0, \mathcal {P}, Q} {\text { minimize }} \quad \operatorname{tr} (Q \Sigma_ {\mathrm{nom}}) + \lambda \gamma^ {2} \tag {12a}
\text { s.t. } \quad (\bullet) ^ {\top} \left( \begin{array}{c c c c c} - \mathcal {P} & 0 & 0 & 0 & 0 \\ 0 & \mathcal {P} & 0 & 0 & 0 \\ 0 & 0 & I & 0 & 0 \\ 0 & 0 & 0 & - \lambda I & \lambda I \\ 0 & 0 & 0 & \lambda I & - Q - \lambda I \end{array} \right) \left( \begin{array}{c c c} I & 0 & 0 \\ \mathcal {A} & \mathcal {B} & 0 \\ \mathcal {C} & \mathcal {D} & 0 \\ 0 & I & 0 \\ 0 & 0 & I \end{array} \right) <   0. \tag {12b}
$$
