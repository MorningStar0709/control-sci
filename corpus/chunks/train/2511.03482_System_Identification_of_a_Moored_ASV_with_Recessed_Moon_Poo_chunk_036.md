# 4. HDMDc operator estimation.

• Define $\widehat { \mathbf { Y } } = \left[ \frac { \widehat { \mathbf { X } } } { \widehat { \mathbf { U } } } \right] .$

• estimate the augmented operator by solving the Tikhonov-regularized least-square problem:

$$[ \widehat {\mathbf {A}} \widehat {\mathbf {B}} ] = \widehat {\mathbf {X}} ^ {\prime} \widehat {\mathbf {Y}} ^ {T} (\widehat {\mathbf {Y}} \widehat {\mathbf {Y}} ^ {T} + \lambda \mathbf {I}) ^ {- 1}.$$
