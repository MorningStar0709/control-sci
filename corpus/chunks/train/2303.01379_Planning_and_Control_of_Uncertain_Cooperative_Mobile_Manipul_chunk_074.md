$$\dot {\hat {\vartheta}} _ {i} = - \gamma_ {i} Y _ {f _ {i}} ^ {\top} J _ {O _ {i}} e _ {v _ {f}}, \tag {2.25a}\dot {\hat {d}} _ {i} = - \beta_ {i} \delta_ {i} ^ {\top} J _ {O _ {i}} e _ {v _ {f}} \tag {2.25b}\dot {\hat {\vartheta}} _ {O} = - \gamma_ {O} Y _ {f O} ^ {\top} e _ {v f} \tag {2.25c}\dot {\hat {d}} _ {O _ {i}} = - \beta_ {O} \delta_ {O} ^ {\top} e _ {v _ {f}}, \tag {2.25d}$$

with arbitrary bounded initial conditions, where $\beta _ { i } , \beta _ { o } , \gamma _ { i } , \gamma _ { o } \in \mathbb { R } _ { > 0 }$ are positive gains, $\forall i \in \mathcal N$ .

The control and adaptation laws can be written in vector form

$$u = \widetilde {Y} _ {f} \hat {\vartheta} + \widetilde {\delta} \hat {d} + G _ {M} ^ {+} \left[ Y _ {f _ {O}} \hat {\vartheta} _ {O} - e + \delta_ {O} \hat {d} _ {O} - K _ {v} e _ {v _ {f}} \right] \tag {2.26a}\dot {\hat {\vartheta}} = - \Gamma \widetilde {Y} _ {f} ^ {\top} G ^ {\top} e _ {v _ {f}} \tag {2.26b}\dot {\hat {d}} = - B _ {g} \widetilde {\delta} ^ {\top} G ^ {\top} e _ {v _ {f}} \tag {2.26c}\dot {\hat {\vartheta}} _ {O} = - \gamma_ {O} Y _ {f O} ^ {\top} e _ {v f} \tag {2.26d}\dot {\hat {d}} _ {O} = - \beta_ {O} \delta_ {O} ^ {\top} e _ {v _ {f}}, \tag {2.26e}$$

where $\widetilde { Y } _ { f } : = \widetilde { Y } ( q , \dot { q } , v _ { f } , \dot { v } _ { f } ) , G _ { M } ^ { + } : = G _ { M } ^ { + } ( q ) : = [ J _ { M _ { 1 } } ^ { \top } , \ldots , J _ { M _ { N } } ^ { \top } ] ^ { \top } \in \mathbb { R } ^ { 6 N \times 6 }$ , $B _ { g } : = \mathrm { d i a g } \{ [ \beta _ { i } I _ { \mu } ] _ { i \in \mathcal { N } } \}$ , and $\Gamma : = \mathrm { d i a g } \{ [ \gamma _ { i } I _ { \ell } ] _ { i \in \mathcal { N } } \}$ . The matrix $G _ { M } ^ { + } ( \boldsymbol { q } )$ was introduced in [73], where it was proved that it yields a load distribution that is free of internal forces. The parameters $m _ { o } ^ { \star } , m _ { i } ^ { \star }$ are used to distribute the object’s needed effort (the term that right multiplies $G _ { M } ^ { + } ( \boldsymbol { q } )$ in (2.26a)) to the agents.
