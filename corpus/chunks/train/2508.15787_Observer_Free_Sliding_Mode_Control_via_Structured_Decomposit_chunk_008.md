# A. Stability Analysis via Lyapunov Method

To assess the stability of the proposed observer-free control strategy, we consider the Lyapunov candidate function:

$$V (s) = \frac {1}{2} s ^ {2}, \tag {8}$$

which is positive definite and radially unbounded with respect to the sliding variable s.

Recall that the sliding variable is defined structurally as:

$$s = \alpha (x, \dot {x}) - \beta (x, u), \tag {9}$$

where the function $\alpha ( x , { \dot { x } } ) = { \dot { x } } + k _ { 1 } x$ captures the system’s internal dynamics, and β(x, u) = ug(x) $\begin{array} { r } { \beta ( x , u ) = \frac { u } { g ( x ) } } \end{array}$ reflects the normalized control input, with $g ( x )$ denoting the control effectiveness.

Taking the time derivative of s, we obtain:

$$\dot {s} = \dot {\alpha} - \dot {\beta}. \tag {10}$$

We make the following regularity assumptions:

• The function $g ( x )$ is smooth and strictly non-zero in the domain of interest.   
• The mappings $\alpha ( x , \dot { x } )$ and $\beta ( x , u )$ are continuously differentiable.   
• The control law is defined as:

$$u = - \lambda \tanh (\alpha), \quad \lambda > 0, \tag {11}$$

which ensures smooth and bounded actuation.

Substituting the control law into $\beta ( x , u )$ , we observe that:

$$\beta (x, u) = - \lambda \frac {\tanh (\alpha)}{g (x)}.$$

Therefore, as $\alpha \  \ 0 .$ , we have $\beta \  \ 0 .$ , and consequently $s  0 ,$ which is consistent with convergence to the sliding surface.

The time derivative of the Lyapunov function then becomes:

$$\dot {V} (s) = s \dot {s}. \tag {12}$$

Due to the structure of the control input, the closed-loop dynamics of s induce a damping behavior. While an explicit expansion of s˙ is not required for our qualitative analysis, we assert that the control structure ensures:

$$\dot {V} (s) \leq - \epsilon s ^ {2} < 0, \quad \forall s \neq 0, \tag {13}$$

for some constant $\epsilon > 0$ depending on λ and the gain distribution in $g ( x )$ . This inequality implies that $V ( t )$ decreases monotonically and that $s ( t ) \to 0$ as $t \to \infty$ .

Hence, the sliding surface is globally asymptotically stable. Moreover, the use of a smooth hyperbolic tangent function eliminates discontinuities in the control input and avoids chattering, while the absence of an observer simplifies the implementation and improves robustness against modeling uncertainties.
