The reader certainly noted the fact that the bound on the time derivative (32) depends explicitly on the state variable x trough the positive function $c ( \sigma )$ , a feature that needs to be faced with care. To make this point clearer, from (18) and (32) it is readily verified that

$$\dot {v} (\zeta) + c (\sigma) ^ {- 1} \dot {V} (x) < 0 \tag {34}$$

for all vectors $( 0 , 0 ) \ \not = \ ( \zeta , x ) \ \in \ \mathbb { R } ^ { r } \times \mathbb { R } ^ { 2 n }$ . Unfortunately, the state-dependent term $c ( \sigma )$ in (34) prevents the determination of a Lyapunov function associated to the closed-loop system. A natural candidate would be $\nu ( \zeta , x )$ such that $\dot { \nu } ( \zeta , x ) = \dot { v } ( \zeta ) + c ( \sigma ) ^ { - 1 } \dot { V } ( x ) < 0$ which results by solving the equations $\nabla _ { \zeta } \nu = \nabla _ { \zeta } v$ and $\nabla _ { \boldsymbol { x } } \nu = c ( \sigma ) ^ { - 1 } \nabla _ { \boldsymbol { x } } V$ . The first is simple but the second one does not appear to exhibit a known solution that yields $\nu ( \zeta , x ) > 0$ as required to be a Lyapunov function associated to the whole feedback system under analysis. On the other hand, the positive function $\nu ( \zeta , x ) \stackrel { \circ } { = } v ( \zeta ) + c ( \sigma ) ^ { - 1 } V ( x ) > 0$ does not satisfy the second gradient condition and consequently it is not possible to be sure that $\dot { \nu } ( \zeta , x ) < 0 .$ A simple and in some sense unexpected way, based on the results of Lemma 1 and Lemma 2, to circumvent this difficulty is stated in the next theorem.

Theorem 1 If there exist positive definite radially unbounded Lyapunov functions $v ( \zeta )$ and $V ( x )$ such that

$$\dot {v} < \rho_ {L} (V - v) \tag {35}\dot {V} < \rho_ {N} (v - V) \tag {36}$$

where $\rho _ { L } ( \zeta ) > 0 , \rho _ { N } ( x ) > 0$ , for all $0 \neq \zeta \in \mathbb { R } ^ { r }$ and all $0 \neq x \in \mathbb { R } ^ { 2 n }$ , along arbitrary trajectories of the subsystems $ { \boldsymbol { S } } _ { L }$ and $\mathcal { S } _ { N }$ , respectively. $\dot { T h e n } ,$ the closed-loop system is globally asymptotically stable.

$$\nu (\zeta , x) = \max \{v (\zeta), V (x) \} \tag {37}$$
