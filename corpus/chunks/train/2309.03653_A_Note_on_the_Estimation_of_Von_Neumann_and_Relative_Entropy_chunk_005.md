$$
\left\{ \begin{array}{l} \dot {\boldsymbol {x}} (t) = \boldsymbol {A} \boldsymbol {x} (t) \\ \boldsymbol {y} (t) = \boldsymbol {C} \boldsymbol {x} (t) \end{array} \right. \tag {4}
$$

where A is given by

$$\boldsymbol {A} \triangleq - \imath (\boldsymbol {I} \otimes \boldsymbol {H} - \boldsymbol {H} ^ {T} \otimes \boldsymbol {I}) \tag {5}$$

and C is given by

$$
\boldsymbol {C} \triangleq \left[ \begin{array}{c} \operatorname{vec} \left\{\boldsymbol {M} _ {1} \right\} ^ {\dagger} \\ \operatorname{vec} \left\{\boldsymbol {M} _ {2} \right\} ^ {\dagger} \\ \vdots \\ \operatorname{vec} \left\{\boldsymbol {M} _ {N _ {\mathrm{m}}} \right\} ^ {\dagger} \end{array} \right] \in \mathbb {C} ^ {N _ {\mathrm{m}} \times d ^ {2}}. \tag {6}
$$

The system is linearly observable if and only if the observability matrix

$$
\boldsymbol {O} (\boldsymbol {A}, \boldsymbol {C}) \triangleq \left[ \begin{array}{c} \boldsymbol {C} \\ \boldsymbol {C A} \\ \boldsymbol {C A ^ {2}} \\ \vdots \\ \boldsymbol {C A ^ {d ^ {2} - 1}} \end{array} \right] \in \mathbb {C} ^ {N _ {\mathrm{m}} d ^ {2} \times d ^ {2}}
$$

is of rank $d ^ { 2 } { \mathrm { ~ ( i . e . , } } O ( A , C )$ has a trivial nullspace) [15, pg. 221]. An equivalent definition of observability is the ability to choose $\boldsymbol { K } \in \mathbb { C } ^ { d ^ { 2 } \times N _ { \mathrm { m } } }$ such that the matrix $\left( A - \right.$ $K C )$ has eigenvalues in any desired location [16]. The matrix K is called the observer gain. A linear quantum state observer dynamically updates its estimate $\ b { \tilde { x } } \in \mathbb { C } ^ { d ^ { 2 } }$ of the true state x as follows:

$$
\left\{ \begin{array}{l} \dot {\tilde {\boldsymbol {x}}} (t) = \boldsymbol {A} \tilde {\boldsymbol {x}} (t) + \boldsymbol {K} (\boldsymbol {y} (t) - \tilde {\boldsymbol {y}} (t)) \\ \tilde {\boldsymbol {y}} (t) = \boldsymbol {C} \tilde {\boldsymbol {x}} (t) \end{array} . \right. \tag {7}
$$

$$\dot {\boldsymbol {e}} (t) = (\boldsymbol {A} - \boldsymbol {K C}) \boldsymbol {e} (t). \tag {8}$$
