Furthermore, given a causal formula $[ { \vec { X } }  { \vec { x } } ] ( Y = y )$ , the satisfaction relation between the causal model, the context and the causal formula, denoted by $( M , { \vec { u } } ) \models [ { \vec { X } }  { \vec { x } } ] ( Y = y )$ , holds if, and only if, in the causal model resulting by the intervention $M _ { \vec { X }  \vec { x } } = ( \mathcal { S } , \mathcal { F } _ { \vec { X }  \vec { x } } )$ , we have that $( \mathcal { F } _ { Y } \in \mathcal { F } _ { \vec { X }  \vec { x } } \wedge U _ { 1 } = u _ { 1 } , U _ { 2 } = u _ { 2 } , . . . , U _ { i } = u _ { i } ) \implies \mathcal { F } _ { Y } = y$ .

Thus, given a context $\vec { u } \in \mathcal { R } ( \mathcal { U } )$ , we write $( M , { \vec { u } } ) \models [ { \vec { X } }  { \vec { x } } ] ( Y = y )$ if the variable $Y \in \nu$ has the value y in a causal model M where $X _ { i }$ is set to $x _ { i }$ for $i = 1 , . . . , k$ . The notation can also be used in the presence of a Boolean combination of primitive events: $( M , { \vec { u } } ) \models [ { \vec { X } }  { \vec { x } } ] \Phi$ . Note that $( M , \vec { u } ) \models [ \vec { X }  \vec { x } ] \Phi \iff ( M _ { \vec { X }  \vec { x } } , \vec { u } ) \models \Phi$ .

Furthermore, in the special case where no assignments are performed $( \mathrm { i . e . , } \ k = 0 )$ , we write $( M , { \vec { u } } ) \models ( Y = y )$ , if and only if the variable $Y \in \nu$ has the value y given the context ${ \vec { u } } \in \mathcal { R } ( { \mathcal { U } } )$ and the causal model M. This notation can also be used in the presence of a Boolean combination of primitive events: $( M , \vec { u } ) \models \Phi$ .
