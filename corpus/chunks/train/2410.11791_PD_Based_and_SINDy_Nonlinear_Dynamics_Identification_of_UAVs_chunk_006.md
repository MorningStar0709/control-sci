The dynamic model of the unmanned aerial vehicle (UAV) can be derived using Euler-Lagrange formalism [18], which allows us to describe the equations of motion based on kinetic energy, potential energy, and the generalized forces acting on the system. Specifically, the system’s state is represented by $\mathbf { x } ~ = ~ \left\lceil \mathbf { q } \dot { } \dot { } \mathbf { q } \right\rceil ^ { \mathbf { \hat { \mathsf { T } } } } ~ \in ~ \mathbb { R } ^ { 1 \dot { 2 } }$ , with $\dot { \textbf { q } } = \ \lvert \pmb { \eta } \mathclose \rvert \ \mathfrak { a } \rvert ^ { \tau } \ \stackrel { \triangledown } { \in } \ \mathbb { R } ^ { 6 }$ and $\dot { \mathbf { q } } = \left\lceil \dot { \pmb { \eta } } \dot { \pmb { \Omega } } \right\rceil ^ { \intercal } \bar { \mathbf { \Omega } } \in \mathbb { R } ^ { 6 }$ , where $\begin{array} { r } { \pmb { \eta } = \left[ \eta _ { x } \quad \eta _ { y } \quad \eta _ { z } \right] ^ { \intercal } } \end{array}$ denotes the position coordinates, $\dot { \pmb { \eta } } = \left\lceil \dot { \eta } _ { x } \quad \mathbf { \bar { \eta } } _ { y } \quad \dot { \eta } _ { z } \right\rceil$ are the linear velocities in inertial frame, $\Omega = { \bar { \left. \phi \quad \Omega \quad \psi \right\rceil } } ^ { \textsf { \bar { T } } }$ represents the Euler angles (roll, pitch, yaw), and $\dot { \pmb { \Omega } } = \left\lceil \dot { \phi } \dot { \mathrm { ~ \bf ~ \ " ~ } } \dot { \boldsymbol { \Omega } } \dot { \mathrm { ~ \bf ~ \it ~ \psi ~ } } \right\rceil ^ { \intercal }$ are the corresponding angular velocities in body frame.
