\quad \text { Suff large } T$$

Finally, for $j \geq \log ^ { 3 3 } ( T )$ , conditional on event $E _ { 1 } ^ { t } \cap E _ { \mathrm { L } 8 } ( x , K , \theta , W ^ { \prime } ) \cap E ^ { * } ( x , K , \theta , W ^ { \prime } ) \cap A _ { j }$ and for sufficiently large T,

$$
\begin{array}{l} d _ {j + 1} \leq \left(1 + O _ {T} \left(\frac {1}{\log^ {1 0} (T)}\right)\right) ^ {j + 1} \cdot 2 ^ {\kappa (j)} \cdot O _ {T} (\epsilon) ^ {\lambda (j)} \cdot d \tag {22} \\ \leq \left(1 + O _ {T} \left(\frac {1}{\log^ {1 0} (T)}\right)\right) ^ {j + 1 - 0} \cdot O _ {T} (\epsilon) ^ {\lambda (j) - \lambda (0)} \cdot 2 ^ {\kappa (j)} \cdot d \\ \leq \left(1 - \frac {1}{2 \log^ {9} (T)}\right) ^ {j + 1} 2 ^ {\kappa (j)} \cdot d \quad \text { Lemma   8 } \\ \leq \left(1 - \frac {1}{\log^ {1 0} (T)}\right) ^ {j + 1} \cdot d. \quad \text { As   in   Equation   (19) } \\ \end{array}
$$

Combining all three cases, we have that for all $j \geq 0$ , conditional on $E _ { 1 } ^ { t } \cap E _ { \mathrm { L 8 } } ( x , K , \theta , W ^ { \prime } ) \cap$ $E ^ { * } ( x , K , \theta , W ^ { \prime } ) \cap A _ { j } , A _ { j + 1 }$ holds. As in Case 1, we can conclude by induction using $A _ { 0 }$ as the base case to get that conditional on $E _ { \mathrm { L 8 } } ( x , K , \theta , W ^ { \prime } ) \cap E ^ { * } ( x , K , \theta , W ^ { \prime } ) \cap E _ { 1 } ^ { t }$ , the event $A _ { t }$ holds, which implies that

$$
d _ {j} \leq \left\{ \begin{array}{l l} \left(1 - \frac {1}{\log^ {1 0} (T)}\right) ^ {j} \cdot d, & \text { if } j > \log^ {3 3} (T) \\ 2 d + \tilde {O} _ {T} (\epsilon), & \text { if } j \leq \log^ {3 3} (T). \end{array} \right. \tag {26}
$$

Taking $\begin{array} { r } { E _ { \mathrm { L 5 } } ( x , K , \theta , W ^ { \prime } ) = E _ { \mathrm { L 8 } } ( x , K , \theta , W ^ { \prime } ) \cap E ^ { * } ( x , K , \theta , W ^ { \prime } ) \cap E _ { 1 } ^ { t } } \end{array}$ , we have by a union bound that $\mathbb { P } ( E _ { \mathrm { L 5 } } ( x , K , \theta , W ^ { \prime } ) ) = 1 - o _ { T } ( 1 / T ^ { 2 0 } )$ . This completes the proof of Lemma 5 for Case 2.
