# 4.2 Implementation of the dynamic STC mechanism and stability result

Recall that the idea for the dynamic STC mechanism is to maximize $\tau _ { \mathrm { m a x } } ( r _ { k } ^ { + } ) = t _ { k + 2 } - t _ { k }$ such that (18) and thus (14) hold. We will use a similar approach as in Hertneck and Allg¨ower (2021c) to realize this. For any parameter set $p \in \{ 1 , \ldots , n _ { p } \}$ , we note that if

$$e ^ {- \epsilon_ {p} \tau_ {\mathrm{max}} (r _ {k} ^ {+})} U _ {p} (\xi (r _ {k} ^ {+}))\leq e ^ {- \epsilon_ {1} \tau_ {\max} (r _ {k} ^ {+})} K (x (r _ {k}), e (r _ {k}), \eta (r _ {k}), c _ {\mathcal {X}})$$

and

$$\tau_ {\max} (r _ {k} ^ {+}) \leq T _ {\max} (\mathcal {P} _ {p}, \lambda , c _ {U}, \tau_ {\mathrm{mad}}) \tag {20}$$

hold with $c _ { U } = \gamma _ { 1 , 1 } \phi _ { 1 , 1 } ( 0 )$ , then it follows from Proposition 1 due to (13) that (14) holds for $r _ { k + 2 } ^ { + } .$ . The next step is thus to maximize $\tau _ { \operatorname* { m a x } } ( r _ { k + 1 } )$ such that (19) and (20)

hold at least for one $p \in \{ 2 , \ldots , n _ { p } \}$ . Note that (19) can be rewritten as

$$\left(- \epsilon_ {p} + \epsilon_ {1}\right) \tau_ {\max} (r _ {k} ^ {+}) \leq \log \left(\frac {K (x (r _ {k}) , e (r _ {k}) , \eta (r _ {k}) , c _ {\mathcal {X}})}{U _ {p} (\xi (r _ {k} ^ {+}))}\right).$$

Otherwise, i.e., $\mathrm { i f } \ - \epsilon _ { p } + \epsilon _ { 1 } \le 0$ , we can directly use the maximum value $\tau _ { \operatorname* { m a x } } ( r _ { k } ^ { + } ) = T _ { \operatorname* { m a x } } ( \mathcal { P } _ { p } , \lambda , c _ { U } , \tau _ { \operatorname* { m a d } } )$ .

In this case, the respective parameter set will be discarded.
