$$
\left[ \begin{array}{c c} \frac {1}{2} (\mathbf {B} + \mathbf {B} ^ {\top}) & \mathbf {A} _ {1} ^ {\top} \\ \mathbf {A} _ {1} & - \mathbf {I} \end{array} \right] + \text {He} \left(\left[ \begin{array}{c c} \mathbf {0} & \mathbf {0} \\ \mathbf {0} & \mathbf {I} \end{array} \right] ^ {\top} \left[ \begin{array}{c c} \mathbf {0} & \mathbf {0} \\ \frac {1}{2} \mathbf {A} _ {2} & \mathbf {0} \end{array} \right]\right) \preceq 0.
$$

Both ${ \bf A } _ { 2 }$ and $\mathbf { B } _ { 2 }$ contain x – meaning that an infinite number of constraints would be needed to enforce the LMI for all $x \in B _ { \epsilon } ( 0 )$ . Therefore, the constraint can instead be enforced by a singular LMI that bounds it above for all $x \in B _ { \epsilon } ( 0 )$ .

First, Young’s relation, with scaling matrix $\bar { \boldsymbol { \Pi } } = \left[ \begin{array} { l l } { \pi _ { 0 } } & { \mathbf { 0 } } \\ { \mathbf { 0 } } & { \boldsymbol { \Pi } } \end{array} \right]$ $\left( \pi _ { 0 } > 0 \right)$ , bounds the Hermition above to produce the LMI

$$
\left[ \begin{array}{c c} \frac {1}{2} (\mathbf {B} + \mathbf {B} ^ {\top} + \mathbf {A} _ {2} ^ {\top} \Pi \mathbf {A} _ {2}) & \mathbf {A} _ {1} ^ {\top} \\ \mathbf {A} _ {1} & - \mathbf {I} + \frac {1}{2} \Pi^ {- 1} \end{array} \right] \preceq 0.
$$

Then, the definition of matrix two norms and the Cauchy Schwarz inequality are used to produce the bounds $\mathbf { A } _ { 2 } ^ { \top } \Pi \mathbf { A } _ { 2 } ~ \preceq ~ \| \mathbf { A } _ { 2 } \| _ { 2 } ^ { 2 } \left\| \Pi \right\| _ { 2 } \mathbf { I } ~ \preceq ~ \hat { \zeta } ^ { 2 } \epsilon ^ { 2 } \left\| \Pi \right\| _ { 2 } \mathbf { I }$ and $\mathbf { B } _ { 2 } \preceq \hat { \theta } \epsilon \mathbf { I }$ . Finally, perform a Schur complement [1] on $\frac { 1 } { 2 } \hat { \zeta } ^ { 2 } \epsilon ^ { 2 } \left\| \Pi \right\| _ { 2 } \mathbf { I }$ , which results in the term $- \frac { 2 } { \left. \Pi \right. }$ 2 on the diagonal of the matrix. Let $\pi _ { M }$ be the maximum term of Π. For a diagonal, square matrix $\| \Pi \| _ { 2 } \le \pi _ { M }$ and therefore, − 2∥Π∥ $\begin{array} { r } { - \frac { 2 } { \| \Pi \| _ { 2 } } \le - \frac { 2 } { \pi _ { M } } } \end{array}$ πM . Recall that $\frac { 1 } { \pi _ { M } } ^ { - }$ πM is the minimum, nonzero term of $\Pi ^ { - 1 }$ to get (13). Thus, (13) implies (12). ✷
