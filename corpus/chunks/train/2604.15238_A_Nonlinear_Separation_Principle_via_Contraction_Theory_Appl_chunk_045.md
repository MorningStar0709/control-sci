# B. Applications to implicit neural networks

Recent literature [26] suggests that increasing the number of iterations of a DEQ improves its expressivity. This is due to the fact that one can model locally Lipschitz maps via fixed point equations. In order to canonically integrate this insight into a DEQ, we propose the following architecture.

Proposition 37 (Local Lipschitz continuity of the equilibrium map): Let $W \colon \mathcal { U } \to \mathbb { R } ^ { n \times n }$ and $B \colon \mathcal { U } \ \to \ \mathbb { R } ^ { n }$ be globally Lipschitz with constants $\ell _ { W }$ and $\ell _ { B }$ . Let $\Psi : \mathbb { R } ^ { n }  \mathbb { R } ^ { n }$ be MONE. Define the map $x ^ { \star } \colon \mathcal { U } \to \mathbb { R } ^ { n }$ by the implicit equation

$$x ^ {\star} (u) = \Psi (W (u) x ^ {\star} (u) + B (u)). \tag {76}$$

If $W ( u ) - I _ { n }$ is Lyapunov diagonally stable uniformly in u, then the map $x ^ { \star } ( u )$ is locally Lipschitz for each $u \in \mathcal { U }$ .

Proof: Let $u , u ^ { \prime } \in \mathcal { U } ,$ and let $x : = x ^ { \star } ( u ) , x ^ { \prime } : = x ^ { \star } ( u ^ { \prime } )$ . Set $\varepsilon : = ( W ( u ) – W ( u ^ { \prime } ) ) x ^ { \prime } + B ( u ) – B ( u ^ { \prime } )$ . Since $W - I _ { n }$ is LDS, there must exist a positive diagonal $Q \succ 0$ and $\delta > 0$ such that $Q W ( u ) + W ( u ) ^ { \top } Q \leq 2 Q - 2 \delta I _ { n }$ . Using the MONE incremental matrix multiplier,

$$\left\| x - x ^ {\prime} \right\| _ {Q} ^ {2} \leq \left(x - x ^ {\prime}\right) ^ {\top} Q (W (u) \left(x - x ^ {\prime}\right) + \varepsilon). \tag {77}$$

Applying the LDS bound to the first term on the right cancels $\| x - x ^ { \prime } \| _ { Q } ^ { 2 }$ from both sides, yielding,

$$\delta \| x - x ^ {\prime} \| _ {2} ^ {2} \leq (x - x ^ {\prime}) ^ {\top} Q \varepsilon . \tag {78}$$

Through Cauchy Schwarz and global Lipschitz continuity of W, B we obtain the Lipschitz bound,

$$\left\| x ^ {\star} (u) - x ^ {\star} \left(u ^ {\prime}\right) \right\| \leq \frac {\| D \|}{\delta} \left(\ell_ {W} \left\| x ^ {\star} \left(u ^ {\prime}\right) \right\| + \ell_ {B}\right) \| u - u ^ {\prime} \|. \tag {79}$$

Fixing $u _ { 0 } \in \mathcal { U }$ and $r > 0 ,$ , applying (79) with $u ^ { \prime } = u _ { 0 }$ yields a uniform bound $\| x ^ { \star } ( u ^ { \prime } ) \| \le R ( u _ { 0 } , r ) < \infty$ for all $u ^ { \prime } \in$ $B ( u _ { 0 } , r )$ . Substituting back into (79) gives Lipschitz constant $\begin{array} { r } { L ( u _ { 0 } , r ) = \frac { \| D \| } { \delta } ( \ell _ { W } R ( u _ { 0 } , r ) + \ell _ { B } ) < \infty } \end{array}$ on $B ( u _ { 0 } , r )$ .
