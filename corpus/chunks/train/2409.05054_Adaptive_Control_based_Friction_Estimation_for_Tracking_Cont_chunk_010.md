# A. Problem Statement

In this paper we estimate only the friction parameters $\pi _ { f }$ assuming the estimated rigid body dynamic parameters πˆ are known, the target is to design an adaptive output feedback tracking controller that ensures limt ${ \bf \nabla } \to \infty \tilde { \bf X } = 0 .$ , where $\tilde { \mathbf { x } } = \left[ \tilde { \mathbf { q } } \quad \dot { \tilde { \mathbf { q } } } \right] , \tilde { \mathbf { q } } = \mathbf { q } - \mathbf { q } _ { d }$ and $\dot { \tilde { \mathbf { q } } } = \dot { \mathbf { q } } - \dot { \mathbf { q } } _ { d } . \mathbf { q } _ { d } ( t )$ represent the tracking error of the trajectory, with q, q˙ the current joint positions and velocities and $\mathbf { q } _ { d } , \dot { \mathbf { q } } _ { d }$ the desired joint positions and velocities. The proposed controller supposed to be robust to the model mismatch of the rigid body dynamics π˜ and friction model $\tilde { \pi } _ { f }$ , where $\tilde { \pi } = \hat { \pi } - \pi$ and $\tilde { \pi } _ { f } = \hat { \pi } _ { f } - \pi _ { f }$ .
