Points on this circle correspond to values of $x ,$ $y ,$ and z such that the matrix in (6) is positive semi-definite with unit trace, and the function $f \left( \cdot \right)$ in (5) maps points on this set back onto itself. Since the set is compact and convex, the Brouwer fixed-point theorem guarantees the existence of at least one fixed-point for this function (that is, a point on the circle that is mapped back onto itself by f (·)).

where $A \ \geq \ 0$ and $c \geq 0$ (all the entries of A and c are non-negative), c is a row vector (the output has dimension 1), and the initial condition is assumed to also be non-negative $( x _ { 0 } ~ \ge ~ 0 )$ . Such systems are typically called internally positive systems in the literature [7]. This is because they preserve positivity in the sense that the state trajectories $x _ { 0 } , x _ { 1 } , x _ { 2 } , \ldots$ . and output trajectories $y _ { 0 } , y _ { 1 } , y _ { 2 } , . . .$ . remain nonnegative. Since the non-negative orthant is itself a cone, this system property motivates replacing C in (4) with a slice of non-negative orthant, namely the unit simplex $\mathcal { C } = \{ x \geq 0 : \overline { { \textstyle \sum _ { k = 1 } ^ { n } x _ { k } = 1 } } \}$ , and $f \left( \cdot \right)$ in (5) with

$$f (x) = \frac {x A + \alpha c}{\sum_ {k = 1} ^ {n} (x A) _ {k} + \alpha c _ {k}}.$$

The ideas behind the presented proof work under these substitutions more or less without change [some of the algebra needs to be adjusted; for example, recalculating (7), where in addition $\lambda _ { \alpha }$ is set to $\textstyle \sum _ { k = 1 } ^ { n } { \bigl ( } x A { \bigr ) } _ { k } + \alpha c _ { k } J$ . Doing so leads to the conclusion that asymptotic stability and observability imply the existence of a $q > 0$ (meaning that q is entrywise positive) such that

$$q = c + q A.$$

This means that when the dynamical system is positive, condition (iii) can be $r e \mathrm { - }$ placed with the condition $( i i i ) ^ { \prime \prime } c ( I - A ) ^ { - 1 } > 0$ [note the other implications can also be proved as hinted at in Remark 3, but using the Lyapunov candidate $V \left( x \right) =$ $c \left( I - A \right) ^ { - 1 } x ,$ so we really can replace (iii) with $( i i i ) ^ { \prime \prime }$ in Theorem 1 in the internally positive system case].
