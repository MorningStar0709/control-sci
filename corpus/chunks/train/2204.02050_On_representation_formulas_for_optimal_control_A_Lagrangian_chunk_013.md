Proof. By Lemma 1, for any $\varepsilon > 0$ , we can select $\tilde { u } \in \mathcal { U }$ such that $| x ( s ) - \tilde { x } ( s ) | < \varepsilon$ for $s \in [ t , T ]$ . Since $g$ is Lipschitz continuous, $| g ( x ( T ) ) - g ( \tilde { x } ( T ) ) | \leq C \varepsilon$ for some positive constant C.

It follows from the expression (3.1) of $H ^ { * }$ that

$$
\begin{array}{l} H ^ {*} (s, x (s), \beta (s)) - H ^ {*} (s, \tilde {x} (s), \tilde {\beta} (s)) \\ = S (s, x (s)) - S (s, \tilde {x} (s)) + H _ {f} ^ {*} (s, \beta (s) + A x (s)) - H _ {f} ^ {*} (s, \tilde {\beta} (s) + A \tilde {x} (s)) \\ \leq C \varepsilon + A (\beta (s) - \tilde {\beta} (s)) \\ \end{array}
$$

for some positive constant C and some matrix $A \in \mathbb { R } ^ { n \times n }$ since ${ \pmb x } \mapsto { \pmb S } ( s , { \pmb x } )$ is Lipschitz continuous and $b \mapsto H _ { f } ^ { * } ( s , \pmb { x } , b )$ is linear.

We now observe that

$$
\begin{array}{l} \left| \int_ {t} ^ {T} \beta (s) - \tilde {\beta} (s) \mathrm{d} s \right| = \left| \int_ {t} ^ {T} - \dot {x} (s) + \dot {\tilde {x}} (s) \mathrm{d} s \right| \\ = | x (t) - x (T) - \tilde {x} (t) + \tilde {x} (T) | \\ \leq C \varepsilon \\ \end{array}
$$

for some positive constant C. Therefore, the result follows.
