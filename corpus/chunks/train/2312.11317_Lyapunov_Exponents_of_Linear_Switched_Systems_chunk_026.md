$$X \left(\frac {t}{2}, \frac {s}{2}\right) ^ {2} = X (t, s) + e ^ {\frac {t}{2} A} \left[ e ^ {\frac {s}{2} B}, e ^ {\frac {t}{2} A} \right] e ^ {\frac {s}{2} B}.$$

So, in the end, it amounts just to compute tr $\left( e ^ { \frac { t } { 2 } A } \left[ e ^ { \frac { s } { 2 } B } , e ^ { \frac { t } { 2 } A } \right] e ^ { \frac { s } { 2 } B } \right)$ , which can be done using formula (4.19). One obtains

$$\mathrm{tr} \left(e ^ {\frac {t}{2} A} \big [ e ^ {\frac {s}{2} B}, e ^ {\frac {t}{2} A} \big ] e ^ {\frac {s}{2} B}\right) = \frac {\sinh (\lambda \frac {t}{2}) ^ {2} \sinh (\mu \frac {s}{2}) ^ {2}}{2 \lambda^ {2} \mu^ {2}} \mathrm{tr} \big ([ A, B ] ^ {2} \big).$$

So, if $\operatorname { t r } \left( [ A , B ] ^ { 2 } \right) > 0$ , then we obtained point (1) of the Lemma. If instead $\operatorname { t r } \left( [ A , B ] ^ { 2 } \right) < 0$ , then all inequalities are reversed and we obtain point (2). □

Remark 4.32. Notice that this is a completely general result about product of matrix exponentials. Indeed, it holds for a generic control $u _ { t , s } ,$ even if it do not correspond to a Pontryagin extremal.
