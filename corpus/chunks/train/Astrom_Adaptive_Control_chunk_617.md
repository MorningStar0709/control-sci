# 10.4 VARIABLE-STRUCTURE SYSTEMS

In Section 10.2 we showed how fixed robust controllers can be obtained by increasing the complexity of the controller. Another way to obtain a robust controller is to use a special version of on-off control called a variable-structure system (VSS). The key idea is to apply strong control action when the system deviates from the desired behavior. The name “variable-structure” alludes to the fact that the controller structure may be changed.
