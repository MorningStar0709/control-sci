# 2.2 Representation Formulas for Constrained Optimal Control

We now extend the representation formula (2.7) to the case of state-constrained optimal control. Specifically, the state trajectory of (2.1) must lie in a convex compact set Ω, namely, ¯ $x ( t ) \in \bar { \Omega }$ for any t and for some bounded open set $\Omega \subset \mathbb { R } ^ { n }$ . The value function for the state-constrained optimal control problem is defined as

$$V (\boldsymbol {x}, t) := \inf _ {u \in \mathcal {U}} \left\{\int_ {t} ^ {T} L (s, x (s), u (s)) \mathrm{d} s + g (x (T)): x (t) = \boldsymbol {x}, x (s) \in \bar {\Omega} \right\}. \tag {2.8}$$

We further assume that the state trajectory can be driven into $\bar { \Omega }$ with some control $u \in \mathcal { U }$ at the boundary of Ω, which can be stated as follows:

Assumption 1. There exists a positive constant ν such that for all $\mathbf { \boldsymbol { x } } \in \partial \Omega$ and $t \in [ 0 , T ]$ , there exists $a \in U$ satisfying $f ( t , { \pmb x } , a ) \cdot n ( { \pmb x } ) \le - \nu _ { 1 }$ , where the vector $\ b { n } ( \ b { x } ) \in \mathbb { R } ^ { n }$ is normal to ∂Ω at x.

Under this assumption, by the classical result on state-constrained HJ equation [17], V is the unique viscosity solution in $\Omega \times ( 0 , T )$ and the viscosity supersolution on ${ \bar { \Omega } } \times ( 0 , T )$ of

$$
\left\{ \begin{array}{l l} - V _ {t} + H (t, \boldsymbol {x}, D V) = 0 & \quad \text {in} \quad \Omega \times (0, T), \\ V (\boldsymbol {x}, T) = 0 & \quad \text {on} \quad \bar {\Omega}, \end{array} \right.
$$

where the Hamiltonian H is the same as before. Such V is called the state-constraint viscosity solution, proposed by [18]. Taking the convex conjugate $H ^ { * }$ as the Lagrangian, we can derive representation formulas for the state-constrained value function V , similar to (2.6) and (2.7).
