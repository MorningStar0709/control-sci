(H2) The switching costs sequence $\{ \vartheta _ { \ell , \ell ^ { \prime } } \} _ { \ell , \kappa \in \mathbb { I } }$ is such that $\vartheta _ { \ell , \ell ^ { \prime } } \geq 0$ and satisfies

$$\text { no family of regimes } \{\ell_ {0}, \ell_ {1}, \dots , \ell_ {n}, \ell_ {0} \}
\begin{array}{l} \text {   no   family   of   regimes   } [ x _ {0}, x _ {1}, \dots , x _ {n}, x _ {0} ] \\ \text {   such   that   } \vartheta_ {\ell_ {0}, \ell_ {1}} = \vartheta_ {\ell_ {1}, \ell_ {2}} = \dots = \vartheta_ {\ell_ {n}, \ell_ {0}} = 0. \end{array} \tag {1.8}
$$
