The control action $U ( k + 1 )$ is computed at discrete time k minimizing a cost function J consisting of several terms. Penalties are placed of predicted errors in $\alpha - \beta , x - y$ and the number of switch changes are typically used, resulting in

$$J = \left\| \hat {E} _ {\alpha \beta} (k + 2) \right\| ^ {2} + \lambda_ {x y} \left\| \hat {E} _ {x y} (k + 2) \right\| ^ {2} + \lambda_ {s c} S C (k + 1) \tag {6}$$

where $\| \hat { E } _ { \alpha \beta } \| ^ { 2 }$ is the quadratic deviation of predictions from reference in $\alpha - \beta$ plane, $\| \hat { E } _ { x y } \| ^ { 2 }$ is the quadratic deviation of predictions from reference in $x - y$ plane and SC is the number of switch changes in the VSI:

$$S C (k + 1) = \sum_ {h = 1} ^ {5} | u _ {h} (k + 1) - u _ {h} (k) | \tag {7}$$

The weighting factors of the cost function are parameters $\lambda _ { x y }$ and $\lambda _ { s c } .$ . These parameters have an influence on the selected control action on each sampling period, as a result they influence the performance indices obtained as will be shown in Section 2.1.
