# 2.1.1 Output-based LMPC

Output-based MPC uses the predicted output and control input sequence to construct the cost function. The objective is to find the optimal control sequence to regulate or track the output towards its reference. The output reference is denoted by $\mathbf { y } _ { \mathrm { r } } ,$ , which can be a constant (set point tracking or regulation problem) or time-varying (trajectory tracking problem). The following derivation is for regulation problems and can be extended to trajectory tracking problems by replacing yr with $\mathbf { y } _ { \mathrm { r } _ { k } }$ . For a prediction horizon N and control horizon $\mathrm { N _ { c } , }$ the cost function for an output-based MPC at time instant k is defined as:

$$J _ {y} = \sum_ {j = k + 1} ^ {k + N} \left[ \mathbf {y} _ {\mathrm{r}} - \mathbf {y} _ {j | k} \right] ^ {\top} \mathbf {Q} _ {\mathbf {y} _ {j | k}} \left[ \mathbf {y} _ {\mathrm{r}} - \mathbf {y} _ {j | k} \right] + \sum_ {i = k} ^ {k + N _ {\mathrm{c}} - 1} \left[ \mathbf {u} _ {\mathrm{r}} - \mathbf {u} _ {i | k} \right] ^ {\top} \mathbf {R} _ {\mathbf {u} _ {i | k}} \left[ \mathbf {u} _ {\mathrm{r}} - \mathbf {u} _ {i | k} \right] \tag {10}$$

where $\mathbf { Q } _ { \mathbf { y } _ { i \mid k } } \in \mathbb { R } ^ { p \times p } , { \bf R } _ { { \bf u } _ { i \mid k } } \in \mathbb { R } ^ { m \times m }$ are the output and input weighting matrices, respectively. In the rest of the paper, we consider $\mathrm { N } _ { \mathrm { c } } = \mathrm { N }$ which modifies the above cost function as:

$$J _ {y} = \sum_ {i = k} ^ {k + N - 1} \left[ \mathbf {y} _ {\mathrm{r}} - \mathbf {y} _ {i + 1 | k} \right] ^ {\top} \mathbf {Q} _ {\mathbf {y} _ {i + 1 | k}} \left[ \mathbf {y} _ {\mathrm{r}} - \mathbf {y} _ {i + 1 | k} \right] + \left[ \mathbf {u} _ {\mathrm{r}} - \mathbf {u} _ {i | k} \right] ^ {\top} \mathbf {R} _ {\mathbf {u} _ {i | k}} \left[ \mathbf {u} _ {\mathrm{r}} - \mathbf {u} _ {i | k} \right] \tag {11}$$

using which the output-based LMPC problem is defined as follows:

Problem 1 (Output-based LMPC). Given the current output ${ \bf y } _ { k \left| k \right. }$ , compute the control input sequence ${ \bf U } _ { k }$ for the LTI system in Eq. (38) by solving:

$$\inf _ {\mathbf {U} _ {k}, \mathbf {Y} _ {k + 1}} J _ {y}$$

$\mathrm { s u b j e c t ~ t o ~ } \mathbf { Y } _ { k + 1 } \in \mathbb { Y } ^ { \mathrm { N } } , ~ \mathbf { U } _ { k } \in \mathbb { U } ^ { \mathrm { N } }$ (12)

$$\mathbf {x} _ {i + 1 | k} = \mathbf {A x} _ {i | k} + \mathbf {B u} _ {i | k}\mathbf {y} _ {i | k} = \mathbf {C x} _ {i | k}, \quad k \in \mathbb {T}, i = k, \dots , k + N - 1.$$
