Fig. 1 illustrates the behavior of the controllers in the ideal situation. Fig. 1(a) shows the norm $| | q _ { e } - \bar { 1 } | |$ . Note that both controllers achieved the convergence at 50 (s). Fig. 1(b) shows the norm of the sliding variables $\left\| s \left( \dot { \bar { g } } \right) \right\|$ and $\| s _ { 0 } \|$ . A large transient response in $s _ { 0 }$ of the nongeometric controller is observed, which caused it to take a longer time (40 (s)) to converge, while the geometric controller stabilizes $\dot { s } \left( \bar { g } \right)$ at 15 (s). Fig. 1(c) draws the norm of the torque $\lVert \dot { \boldsymbol { \tau } } \rVert$ . Note that the non-geometric controller requires slightly higher torque input than the geometric controller at the transient. Finally, Fig. 1(d) illustrates the energy consumption of each controller in terms of $\sqrt { \int _ { 0 } ^ { t } \tau ^ { T } \tau d \mathrm { t } }$ . The lower energy consumption in the geometric controller can be noticed.

To better appreciate the convergence behavior of these controllers, the sliding variable of both controllers is shown in Fig. 2. Note that the behavior of the sliding surface $s \left( \bar { g } _ { e } \right)$ of the geometric controller first exhibits the convergence to the siding surface $s = 0 _ { 4 \times 1 }$ and later to the equilibrium $( s ( \bar { g } _ { e } ) , \bar { q _ { e } ) } = ( 0 _ { 4 \times 1 } , \bar { 1 } )$ ). On the other hand, the sliding surface $s _ { 0 }$ of the non-geometric controller oscillates around the surface $s _ { 0 } = 0 _ { 4 \times 1 }$ and then convergences to the equilibrium $( s _ { 0 } , \ q _ { e } ) \ = \ ( 0 _ { 4 \times 1 } , \bar { 1 } )$ with an oscillating behavior, leading to more energy consumption than the geometric controller.
