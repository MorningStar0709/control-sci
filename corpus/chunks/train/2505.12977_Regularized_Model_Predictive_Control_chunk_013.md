# IV. CONVERGENCE AND STABILITY OF RE-MPC

To meet Objective II, we derive sufficient conditions for stability and convergence of the Re-MPC algorithm for a large value of $\mu .$ To do so, we analyze the Riccati equation (20) along with the obtained feedback gains (13). Using Lemma 1 and some algebraic simplifications, we obtain

$$
\left[ \begin{array}{c} X _ {k} ^ {*} \\ U _ {k} ^ {*} \end{array} \right] = \left[ \begin{array}{c c} \bar {Q} _ {k + l} & 0 \\ 0 & \bar {R} \end{array} \right] ^ {- 1} \left[ \begin{array}{c c} \bar {B} _ {1} & - \bar {B} _ {2} \end{array} \right] ^ {\mathsf {T}} O _ {k + l} ^ {- 1} \bar {A} x _ {k | k}, \tag {21}
$$

where $O _ { k + l } = \bar { B } _ { 1 } \bar { Q } _ { k + l } ^ { - 1 } \bar { B } _ { 1 } ^ { \mathsf { T } } + \bar { B } _ { 2 } \bar { R } ^ { - 1 } \bar { B } _ { 2 } ^ { \mathsf { T } } .$

Remark 3: To ensure the feasibility of the Re-MPC algorithm, it is crucial that the matrix $O _ { k + l }$ is invertible. Given that $\bar { Q } _ { k + l } \ > \ 0$ and $\bar { R } \succ 0 .$ , this requirement is satisfied if rank $\begin{array} { r l } { \big ( \left[ \bar { B } _ { 1 } \ } & { { } - \bar { B } _ { 2 } \right] \big ) = \bar { n } } \end{array}$ . Given the structure of ${ \bar { B } } _ { 1 }$ as a lower triangular matrix with an identity matrix on its diagonal, it is always full-rank. As a result, the rank condition is always satisfied as long as ${ \bar { B } } _ { 2 }$ does not introduce rank deficiency.

Remark 4: By applying equation (21) and defining the augmented matrices in Theorem 1 as index matrices, i.e., ${ { \bar { A } } _ { l } } ~ : = ~ { \bar { A } } , ~ { { \bar { B } } _ { i , l } } ~ : = ~ { { \bar { B } } _ { i } }$ for $i \ = \ 1 , 2 ,$ , and $\begin{array} { r } { \bar { \cal R } _ { l } : = \bar { \cal R } , } \end{array}$ , within a prediction horizon of size l, and after some algebraic manipulations, the Riccati equation (20) can be rewritten as

$$P _ {k + l - 1} = \bar {A} _ {l} ^ {\mathsf {T}} \left(\bar {B} _ {1, l} \bar {Q} _ {k + l} ^ {- 1} \bar {B} _ {1, l} ^ {\mathsf {T}} + \bar {B} _ {2, l} \bar {R} _ {l} ^ {- 1} \bar {B} _ {2, l} ^ {\mathsf {T}}\right) ^ {- 1} \bar {A} _ {l} + Q, \tag {22}$$

or alternatively, in a steady-state scenario,

$$P _ {l} = \bar {A} _ {l} ^ {\mathsf {T}} \big (\bar {B} _ {1, l} \bar {Q} _ {l} ^ {- 1} \bar {B} _ {1, l} ^ {\mathsf {T}} + \bar {B} _ {2, l} \bar {R} _ {l} ^ {- 1} \bar {B} _ {2, l} ^ {\mathsf {T}} \big) ^ {- 1} \bar {A} _ {l} + Q. \tag {23}$$

Theorem 2: Under Assumptions 2 and 3, the Riccati equation (23) admits a unique solution $P _ { l } ~ \succ ~ 0$ for the entire process as the prediction horizon shifts within the total time horizon $t _ { f }$ .
