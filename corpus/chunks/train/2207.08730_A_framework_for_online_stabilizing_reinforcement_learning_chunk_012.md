# IV. PRELIMINARIES

Let $\Lambda _ { U _ { t } } ( X _ { t } )$ be the Euler estimate of $X _ { t + \delta }$ for the noiseless system:

$$\Lambda_ {U _ {t}} (X _ {t}) = X _ {t} + \delta f (X _ {t}, U _ {t}).$$

Remark 6: The Euler prediction scheme is chosen for the ease of exposition of the forthcoming analyses, but other choices, including Runge-Kutta methods would work.

Online reinforcement learning implies that at each time step k an update is being carried out for both the control input u and the critic weights w:

$$u _ {k}, w _ {k} \leftarrow F (x _ {k}, w _ {k - 1}, \dots) \tag {7}$$

The present paper discusses how online stability can be ensured by enforcing certain stabilzing constraints when performing the latter update. Let the stabilizing constraint function $G _ { \varepsilon } ^ { \bar { \nu } } ( \cdot , \cdot , \cdot , \cdot )$ be defined as follows:

$$
G _ {\varepsilon} ^ {\bar {\nu}} (w, w _ {\text { prev }}, x _ {\text { next }}, x) := \left[ \begin{array}{c} \hat {J} ^ {w} (x) - \hat {J} ^ {w _ {\text { prev }}} (x) - \delta \varepsilon \\ L (x _ {\text { next }}) - \hat {J} ^ {w} (x _ {\text { next }}) \\ \hat {J} ^ {w} (x _ {\text { next }}) - \hat {J} ^ {w} (x) + \delta \bar {\nu} \\ \hat {\alpha} _ {\text { low }} (\| x \|) - \hat {J} ^ {w} (x) \\ \hat {J} ^ {w} (x) - \hat {\alpha} _ {\text { up }} (\| x \|) \end{array} \right] \tag {8}
$$

where $\hat { \alpha } _ { 1 , 2 } ~ \in ~ K _ { \infty }$ are chosen s. t. ∀x $\begin{array} { r l } { \hat { \alpha } _ { \mathrm { l o w } } ( \| x \| ) } & { { } \leq } \end{array}$ $\alpha _ { \mathrm { l o w } } ( \| x \| ) , \alpha _ { \mathrm { u p } } ( \| x \| ) \leq \hat { \alpha } _ { \mathrm { u p } } ( \| x \| )$ .

Note that ensuring $G _ { \varepsilon } ^ { \bar { \nu } } ( w , w _ { \mathrm { p r e v } } , x _ { \mathrm { n e x t } } , x ) \leq 0$ is equivalent to imposing the following four scalar constraints:

$$\hat {J} ^ {w} (x) \leq \hat {J} ^ {w _ {\text { prev}}} (x) + \delta \varepsilon , \tag {C1}L (x _ {\text { next }}) \leq \hat {J} ^ {w} (x _ {\text { next }}), \tag {C2}\hat {J} ^ {w} (x _ {\text { next }}) - \hat {J} ^ {w} (x) \leq - \delta \bar {\nu}, \tag {C3}\hat {\alpha} _ {\text { low }} (\| x \|) \leq \hat {J} ^ {w} (x) \leq \hat {\alpha} _ {\text { up }} (\| x \|), \tag {C4}$$
