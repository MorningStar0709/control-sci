# 2.1 Local Dissipativity

Hill and Moylan informally defined a dissipative system as a “system that absorbs more energy from the external world than it supplies” [11]. Essentially, any change in the system’s internal energy (represented by the positive semi-definite “storage function”) is bounded above by a function of the input signal to the system and the resulting output, as shown below.

Definition 1 [32, Definition 2]: A dynamical system (1) is dissipative with respect to a locally integrable supply rate $w ( t ) = w ( u , y )$ if there exists a non-negative function, $\dot { V : \mathbb { R } ^ { n } } \to \dot { \mathbb { R } _ { + } }$ , called the storage function, such that for all $( t _ { 0 } , t _ { 1 } ) \in \mathbb { R } _ { + } , x \in \mathbb { R } ^ { n }$ , and $\mathbf { u } \in \mathcal { L } _ { 2 } ^ { m }$

$$V (x _ {0}) + \int_ {t _ {0}} ^ {t _ {1}} w (t) d t \geq V (x _ {1}), \tag {2}$$

where $x _ { 1 }$ and y(t) are the result of applying (1) from time $t _ { 0 }$ to $t _ { 1 }$ with input u(t).

While dissipativity was originally defined globally, there are many nonlinear systems for which dissipativity is a local property about an attractor [10,30]. Different definitions of local dissipativity have been used throughout the literature with various input signals, attractors, and system definitions [10,30,23]. Below, Definition 2 gives a local QSR dissipativity that brings together several concepts in the literature [10,23].

Definition 2 The mapping (1) is locally QSR dissipative on $\Omega \in \Re ^ { n }$ , if the following conditions are satisfied:

(1) The set $\boldsymbol { \mathcal { U } } _ { \Omega } ( \boldsymbol { x } _ { 0 } )$ contains a neighborhood of the origin.

(2) Given the supply rate function

$$w (u, y) = y ^ {\top} \mathbf {Q} y + 2 y ^ {\top} \mathbf {S} u + u ^ {\top} \mathbf {R} u, \tag {3}$$

the inequality

$$\int_ {0} ^ {T} w (u, y) \geq \beta \tag {4}$$

is satisfied for all $\mathbf { u } \in \mathcal { U } _ { \Omega } ( x _ { 0 } )$ and $T \geq 0$ , where $\beta \in \mathbb { R } , \ \mathbf { Q } = \mathbf { Q } ^ { \intercal } \in \mathbb { R } ^ { p \times p } , \ \mathbf { S } \in \mathbb { R } ^ { p \times m }$ , and ${ \bf R } =$ $\mathbf { R } ^ { \top } \in \dot { \mathbb { R } } ^ { m \times m }$ .

The family of mappings $\mathcal { G } _ { \Omega }$ is locally QSR dissipative on Ω with inputs in UΩ if all of its elements are dissipative with a shared supply rate and $\boldsymbol { \mathcal { U } } _ { \Omega }$ contains a neighborhood of the origin.
