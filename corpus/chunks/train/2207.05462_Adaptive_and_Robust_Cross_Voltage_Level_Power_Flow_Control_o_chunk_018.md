# Reference Simulation Model

The reference simulation model is based on the LTI modeling from Section III-B. The input parameters of the DERs are shown in Table I. The parameters $P _ { \mathrm { i n s t } , d } ^ { \mathrm { D E R } }$ are given in the Simbench dataset, whereas $T _ { \mathrm { d e l a y } , d } ^ { \mathrm { D E R } }$ an d $T _ { \mathrm { r i s e } , d } ^ { \mathrm { D E R } }$ were assumed by using the DER models of the detailed simulation model. For each DER-type, $T _ { \mathrm { r i s e } , d } ^ { \mathrm { D E R } }$ is determined by the medium rise time of the positive and negative step response of the detailed model of the corresponding DER-type. In order to demonstrate the dynamic behavior of the reference DER models and the detailed DER models, the step responses (at t = 1) are contrasted in Fig. 8. The detailed CHP plant model is characterized by comparatively large time constants $T _ { \mathrm { d e l a y } , d } ^ { \mathrm { D E R } }$ and $T _ { \mathrm { r i s e } , d } ^ { \mathrm { D E R } }$ . The peculiarity of the detailed WT model is that the negative and positive step responses have a different rise time. In addition, a clearly recognizable deviation from the response of the first-order transfer function can be observed in the step responses of the detailed CHP plant and WT model. The ICT-communication and the feedback sensor delays are assumed with $T _ { \mathrm { c o m } } = T _ { \mathrm { m e a s } } = 0 . 1 \mathrm { s }$ .
