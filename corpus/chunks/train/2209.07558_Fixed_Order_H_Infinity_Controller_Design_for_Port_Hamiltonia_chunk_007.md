If the interconnection between P and K is well-posed, i. e., −D22 Ip2 Im2 −DK∗ is invertible, then the closed-loop transfer function exists and is given by the lower linear fractional transformation of $P$ and K defined as

$$
\begin{array}{l} (P \star K) (s) := P _ {1 1} (s) \\ + P _ {1 2} (s) K (s) \left(I _ {p _ {2}} - P _ {2 2} (s) K (s)\right) ^ {- 1} P _ {2 1} (s). \\ \end{array}
$$

An optimal controller $\mathbf { K } _ { * } = ( A _ { \mathbf { K } _ { * } } , B _ { \mathbf { K } _ { * } } , C _ { \mathbf { K } _ { * } } , D _ { \mathbf { K } _ { * } } )$ of order k then satisfies

$$\mathbf {K} _ {*} = \underset {(A _ {\mathbf {K}}, B _ {\mathbf {K}}, C _ {\mathbf {K}}, D _ {\mathbf {K}})} {\arg \min} \| P \star K \| _ {\mathcal {H} _ {\infty}},$$

s. t. A(K) is asymptotically stable,

where K is as in (1), the $\mathcal { H } _ { \infty }$ norm of $P \star K$ is defined as

$$
\begin{array}{l} \| P \star K \| _ {\mathcal {H} _ {\infty}} := \sup _ {s \in \mathbb {C}, \operatorname{Re} (s) > 0} \| (P \star K) (s) \| _ {2} \\ = \sup _ {\omega \in \mathbb {R}} \| (P \star K) (\mathrm{i} \omega) \| _ {2}, \\ \end{array}
$$

and $\mathcal { A } ( \mathbf { K } )$ is the closed-loop system matrix, defined as

$$
\left[ \begin{array}{c c} A & 0 \\ 0 & A _ {\mathbf {K}} \end{array} \right] + \left[ \begin{array}{c c} B _ {2} & 0 \\ 0 & B _ {\mathbf {K}} \end{array} \right] \left[ \begin{array}{c c} I _ {m _ {2}} & - D _ {\mathbf {K}} \\ - D _ {2 2} & I _ {p _ {2}} \end{array} \right] ^ {- 1} \left[ \begin{array}{c c} 0 & C _ {\mathbf {K}} \\ C _ {2} & 0 \end{array} \right]. \tag {2}
$$

This matrix (resp. the closed-loop system) is called asymptotically stable, if all its eigenvalues have strictly negative real part.
