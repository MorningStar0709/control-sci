$$\| x _ {t} ^ {\pi} - x _ {t} ^ {\star} \| \leq \| B (\theta^ {\star}) \| \frac {R}{\| B (\theta^ {\star}) \| C _ {1} ^ {2}} \sum_ {i = 0} ^ {t - 1} q _ {1} (t) \leq \frac {R}{C _ {1}},$$

which proves the induction step.

Proceeding similarly, we obtain the following bound,

$$
\begin{array}{l} \left\| v _ {t} ^ {\pi} - v _ {t} ^ {\star} \right\| \\ = \left\| v _ {t} ^ {\pi} - y _ {t} ^ {\star} \left(x _ {0}\right) \right\| \\ \leq \left\| v _ {t} ^ {\pi} - y _ {t} ^ {\star} \left(x _ {t - 1}\right) \right\| + \sum_ {i = 0} ^ {t - 1} \left\| v _ {t} ^ {\star} \left(x _ {t - i}\right) - y _ {t} ^ {\star} \left(x _ {t - i - 1}\right) \right\| \\ \leq \| v _ {t} ^ {\pi} - v _ {t} ^ {\star} (x _ {t - 1}) \| + \sum_ {i = 0} ^ {t - 1} q _ {1} (i) \| x _ {t - i} - z _ {t - i} ^ {\star} (x _ {t - i - 1}) \| \\ \leq e _ {t} + \| B (\theta^ {\star}) \| \sum_ {i = 0} ^ {t - 1} q _ {1} (t) e _ {t - i - 1}, \\ \end{array}
$$

where again, we have used the triangle inequality for the first inequality, and the bound (14) for the second. Finally, we have

$$\| u _ {t} ^ {\pi} - u _ {t} ^ {\star} \| = \| v _ {t} ^ {\pi} + \zeta_ {t} - v _ {t} ^ {\star} \|\leq \left\| v _ {t} ^ {\pi} - v _ {t} ^ {\star} \right\| + \left\| \zeta_ {t} \right\|.$$

With these bounds in hand, we first account for the regret up to time $t ^ { \star } ( \delta )$ while the event G holds. Note that a crude bound is sufficient to show that it is finite:

$$R _ {t ^ {\star} (\delta)} (\pi) _ {\mathcal {G}} \leq \bar {Q} \sum_ {t = 0} ^ {t ^ {\star} (\delta)} \| x _ {t} ^ {\pi} - x _ {t} ^ {\star} \| + \bar {R} \sum_ {t = 0} ^ {t ^ {\star} (\delta)} \| u _ {t} ^ {\pi} - u _ {t} ^ {\star} \|\leq (2 \bar {Q} \bar {x} + 2 \bar {R} \bar {u}) t ^ {\star} (\delta)\leq \bar {C} _ {2} + \bar {C} _ {3} \log (1 / \delta).$$

Where $\bar { C } _ { 2 } = ( 2 \bar { Q } \bar { x } + 2 \bar { R } \bar { u } ) c _ { 2 }$ and $\bar { C } _ { 3 } = ( 2 \bar { Q } \bar { x } + 2 \bar { R } \bar { u } ) c _ { 3 }$
