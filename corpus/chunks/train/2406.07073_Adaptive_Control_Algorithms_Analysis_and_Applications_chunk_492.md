# 10.6.3 Combining Data Normalization with Dead Zone

Replacing the PAA of (10.121) through (10.123) by the PAA given in (10.36) through (10.39), with $\phi ( t )$ and $\nu ( t + 1 )$ replaced by their normalized counterpart $\bar { \phi } ( t )$ and $\bar { \nu } ( t + 1 )$ , and with Δ replaced by a time-varying bound, one has the following result.

Theorem 10.5 Under the same hypotheses as in Theorem 10.4, but using the PAA with dead zone:

$$\hat {\theta} (t + 1) = \hat {\theta} (t) + \alpha (t) F (t) \bar {\phi} (t) \bar {\nu} (t + 1) \tag {10.137}F (t + 1) ^ {- 1} = F (t) ^ {- 1} + \alpha (t) \bar {\phi} (t) \bar {\phi} ^ {T} (t); \quad F (0) > 0 \tag {10.138}\bar {\nu} (t + 1) = \frac {\bar {y} (t + 1) - \hat {y} ^ {0} (t + 1)}{1 + \bar {\phi} ^ {T} (t) F (t) \bar {\phi} (t)} \tag {10.139}
\alpha (t) = \left\{ \begin{array}{l l} 1 & | \bar {\nu} (t + 1) | > \bar {\delta} (t + 1) \\ 0 & | \bar {\nu} (t + 1) | \leq \bar {\delta} (t + 1) \end{array} \right. \tag {10.140}
$$

where $\bar { \delta } ( t )$ is given by:

$$\bar {\delta} ^ {2} (t + 1) = \delta^ {2} (t + 1) + \sigma \delta (t + 1); \quad \delta (t) > 0; \sigma > 0 \tag {10.141}$$

and

$$\delta (t + 1) = d _ {2} + \frac {d _ {1}}{m (t)} \geq | \bar {w} (t + 1) | \tag {10.142}$$

one has:

$$\text { (i) } \quad \sum_ {t = 0} ^ {\infty} \alpha (t) [ \bar {\nu} ^ {2} (t + 1) - \bar {\delta} ^ {2} (t + 1) ] < \infty \tag {10.143}\text {(ii)} \quad \lim _ {t \rightarrow \infty} \alpha (t) [ \bar {\nu} ^ {2} (t + 1) - \bar {\delta} ^ {2} (t + 1) ] = 0 \tag {10.144}\text {(iii)} \quad \lim _ {t \to \infty} \sup | \bar {\nu} (t + 1) | \leq \bar {\delta} (t + 1) \tag {10.145}\text {(iv)} \quad \tilde {\theta} ^ {T} (t) F (t) ^ {- 1} \tilde {\theta} (t) \leq \tilde {\theta} ^ {T} (0) F (0) ^ {- 1} \tilde {\theta} (0) < \infty \tag {10.146}\text { (v) } \quad \lim _ {t \to \infty} \sup | \varepsilon (t + 1) | \leq \bar {\delta} (t + 1) m (t) \tag {10.147}w h e r e \varepsilon (t + 1) i s g i v e n b y (1 0. 1 2 8)\text {(vi)} \quad F (t), \tilde {\theta} (t) \text { converge } \tag {10.148}$$

The interest of this result is that it assures that even if $\| \phi ( t ) \|$ becomes unbounded, $\| \hat { \theta } ( t ) \|$ will be bounded and the unnormalized a posteriori prediction error is bounded by $\| \phi ( t ) \|$ (i.e., does not grow faster than $\phi ( t ) )$ and that $\tilde { \theta } ( t )$ (respectively $\hat { \theta } ( t ) )$ converges. The convergence of $\tilde { \theta } ( t )$ is a direct consequence of using $\sigma > 0$ in the definition of $\bar { \delta } ( t )$ .
