# B. Design of cost signal

As explained in Subsection II-D, to reduce the vertical body acceleration $\ddot { z } _ { s }$ (ride comfort), and vertical tire deflection $\Delta l _ { t }$ (road holding), along with minimizing the input command $\dot { z } _ { l i n }$ (actuator energy consumption), the cost signal in (8) is defined as:

$$
\begin{array}{l} z _ {k} = \left[ w _ {1} \ddot {z} _ {s}, w _ {2} \Delta l _ {t}, w _ {3} \dot {z} _ {\text {lin}} \right] ^ {T} \tag {31} \\ = C _ {z} x _ {k} + D _ {z u} u _ {k} + D _ {z p} p _ {k} + D _ {z d} d _ {k}, \\ \end{array}
$$

with the reference cost in (9) given by $\bar { z } _ { k } = [ 0 0 0 ] ^ { T }$ , where $w _ { 1 } , w _ { 2 }$ and w3 are tuneable weights to enhance performance (see Section V). Rewriting $z _ { k }$ in state space form the coefficient matrices are defined as follows:

$$
C _ {z} = \left[ \begin{array}{c c c c c} - \frac {c _ {e q}}{m _ {s}} w _ {1} & \frac {c _ {e q}}{m _ {s}} w _ {1} & \frac {k _ {e q}}{m _ {s}} w _ {1} & 0 & - \frac {k _ {e q}}{m _ {s}} w _ {1} \\ 0 & 0 & 0 & w _ {2} & 0 \\ 0 & 0 & 0 & 0 & 0 \end{array} \right], \tag {32}

D _ {z u} = \left[ \begin{array}{c} - \frac {c _ {e q}}{m _ {s}} w _ {1} \\ 0 \\ w _ {3} \end{array} \right].
$$

For $k = N$ only the terminal state $x _ { N }$ is predicted, thus the terminal cost signal is defined as follows:

$$
\begin{array}{l} z _ {N} = \left[ w _ {4} \ddot {z} _ {s}, w _ {5} \Delta l _ {t} \right] ^ {T} \tag {33} \\ = \hat {C} _ {z} x _ {N} + \hat {D} _ {z p} p _ {N}, \\ \end{array}
$$

where again $w _ { 4 }$ and $w _ { 5 }$ are predefined weights. The coefficient matrix $\breve { \hat { C } } _ { z }$ is defined as follows:

$$
\hat {C} _ {z} = \left[ \begin{array}{c c c c c} - \frac {c _ {e q}}{m _ {s}} w _ {4} & \frac {c _ {e q}}{m _ {s}} w _ {4} & \frac {k _ {e q}}{m _ {s}} w _ {4} & 0 & - \frac {k _ {e q}}{m _ {s}} w _ {4} \\ 0 & 0 & 0 & w _ {5} & 0 \end{array} \right]. \tag {34}
$$

Since the uncertainties and additive disturbances are allowed only in state dynamics, $D _ { z p } , D _ { z d }$ and $\hat { D } _ { z p }$ are zero matrices with appropriate dimensions. The dimension of the cost signal does not change but the coefficient matrices change due to augmentation of the PI controller. The cost signal is thereby redefined for the augmented system as follows:

$$\tilde {z} _ {k} = \tilde {C} _ {z} \tilde {x} _ {k} + \tilde {D} _ {z u} u _ {k}, \tag {35}\tilde {z} _ {N} = \tilde {C} _ {z N} \tilde {x} _ {N}$$

where $\tilde { C } _ { z } = [ C _ { z } \ 0 ] , \tilde { C } _ { z N } = [ \hat { C } _ { z } \ 0 ]$ and $\tilde { D } _ { z u } = D _ { z u } .$

The robust model predictive control detailed in Section III is synthesized given the derived augmented system in (30) to achieve the (higher frequency) suspension performance objectives, as captured by (35), whilst guaranteeing the (low frequency) tracking property of $\Delta z _ { l i n }$ .
