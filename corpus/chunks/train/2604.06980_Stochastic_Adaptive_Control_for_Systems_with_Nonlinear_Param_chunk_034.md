# A Proof of Theorem 1

For notational simplicity, define

$$\psi_ {t} = f (\theta^ {* \top} \phi_ {t}) - f (\hat {\theta} _ {t} ^ {\top} \phi_ {t}), \quad t \geq 0. \tag {A.1}$$

Since both $\phi _ { t }$ and $\widehat { \theta } _ { t }$ are $\mathcal { F } _ { t ^ { - } }$ −measurable, it follows that ψt is also Ft−measurable. As a key step toward Theorem 1, we first establish the following lemma.

Lemma 2 Under the conditions of Theorem 1, the parameter estimates produced by Algorithm 1 have the following property as $t \to \infty ;$

$$\operatorname{tr} \left[ \left(\theta^ {*} - \hat {\theta} _ {t}\right) ^ {\top} P _ {t} ^ {- 1} \left(\theta^ {*} - \hat {\theta} _ {t}\right) \right] + \sum_ {\tau = 0} ^ {t} a _ {\tau} \| \psi_ {\tau} \| ^ {2} = O (1), a. s.,$$

where $\{ a _ { \tau } \} _ { \tau \geq 0 }$ is defined in (11).

The proof of Lemma 2 is provided in Appendix F.

Proof. [Proof of Theorem 1.] We first prove property (i). Taking trace on both sides of (10) and summing up, we have

$$\sum_ {\tau = 0} ^ {\infty} a _ {\tau} d _ {\tau} ^ {2} \phi_ {\tau} ^ {\top} P _ {\tau} ^ {2} \phi_ {\tau} \leq \sum_ {\tau = 1} ^ {t} [ t r (P _ {\tau}) - t r (P _ {\tau + 1}) ] < \infty . \tag {A.2}$$

Besides, according to Cauchy-Schwarz inequality,

$$\sum_ {\tau = 0} ^ {\infty} \left\| a _ {\tau} d _ {\tau} P _ {t} \phi_ {\tau} \psi_ {\tau} ^ {\top} \right\| \leq \sum_ {\tau = 0} ^ {\infty} a _ {\tau} d _ {\tau} ^ {2} \phi_ {\tau} ^ {\top} P _ {t} ^ {2} \phi_ {\tau} + \sum_ {\tau = 0} ^ {\infty} a _ {\tau} \| \psi_ {\tau} \| ^ {2}.$$

Therefore, by (A.2) and Lemma 2, we have

$$\sum_ {\tau = 0} ^ {\infty} \left\| a _ {\tau} d _ {\tau} P _ {\tau} \phi_ {\tau} \psi_ {\tau} ^ {\top} \right\| < \infty , \text { a.s. } \tag {A.3}$$

As for the noise term, note that $a _ { t } \leq 1$ by (11). Using (A.2) again and by Assumption 2, we know that

$$\sum_ {\tau = 0} ^ {\infty} \mathbb {E} [ \| a _ {\tau} d _ {\tau} P _ {\tau} \phi_ {\tau} w _ {t + 1} \| ^ {2} \mid \mathcal {F} _ {t} ] < \infty , \text { a.s. }$$

So by Chow’s martingale convergence theorem (See Theorem 2.7 in Chen and Guo (1991)), we know that $\begin{array} { r } { \big \| \sum _ { \tau = 0 } ^ { \infty } a _ { \tau } d _ { \tau } P _ { \tau } \phi _ { \tau } w _ { \tau + 1 } \big \| < \infty } \end{array}$ almost surely. Hence, combining with (A.3), one can obtain

$$\sum_ {\tau = 0} ^ {\infty} \left\| a _ {\tau} d _ {\tau} P _ {\tau} \phi_ {\tau} \left(\psi_ {\tau} + w _ {\tau + 1}\right) ^ {\top} \right\| < \infty , \text { a.s. } \tag {A.4}$$
