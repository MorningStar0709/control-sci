# Initial-value theorem

The initial-value theorem relates the initial value of the time function $f ( 0 + )$ with its Laplace transform $F ( s )$

$$f (0 +) = \lim _ {t \to 0 +} f (t) = \lim _ {s \to \infty} s F (s) \tag {8.11}$$

Note that $f ( 0 + )$ is the function evaluated at time $t = 0 +$ , which is a small incremental amount greater than zero. Unlike the final-value theorem, the initial-value theorem does not have restrictions on the poles of $F ( s )$ and, therefore, it can be used to find $f ( 0 + )$ for a sinusoidal function, for example.
