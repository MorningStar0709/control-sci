# B. LMI based Controller for Switched NCS

In this section we propose a static state feedback control law of the form

$$
\boldsymbol {u} _ {k} = - \boldsymbol {K} _ {x} \boldsymbol {x} _ {k} = - \hat {\boldsymbol {K}} \boldsymbol {\xi} _ {k} = - \left[ \begin{array}{l l} \boldsymbol {K} _ {x} & \boldsymbol {0} \end{array} \right] \boldsymbol {\xi} _ {k}, \tag {10}
$$

which ensures asymptotic stability of the switched hybrid system (9) leading to the closed loop system

$$\boldsymbol {\xi} _ {k + 1} = \left(\sum_ {i = 1} ^ {\bar {\delta}} \alpha_ {i, k} \hat {\boldsymbol {A}} _ {i} - \hat {\boldsymbol {B}} \hat {\boldsymbol {K}}\right) \boldsymbol {\xi} _ {k}. \tag {11}$$

The motivation to define the controller (10) as a linear combination of system states only, is based on the need to avoid more restrictive assumptions and possible deadlocks which could occur when using a dynamic controller, see [5].
