$$\mathbf {J} \left(\mathbf {x} _ {k}, \mathbf {u} _ {k}, \delta \mathbf {u} _ {k}\right) \approx \left(\mathbf {x} _ {d, k + 1} - \bar {\mathbf {x}} _ {k + 1} - \mathbf {H} ^ {\prime} \delta \mathbf {u} _ {k}\right) ^ {\top} \mathbf {Q} \left(\mathbf {x} _ {d, k + 1} - \bar {\mathbf {x}} _ {k + 1} - \mathbf {H} ^ {\prime} \delta \mathbf {u} _ {k}\right) + \left(\bar {\mathbf {u}} _ {k} + \delta \mathbf {u} _ {k}\right) ^ {\top} \mathbf {R} \left(\bar {\mathbf {u}} _ {k} + \delta \mathbf {u} _ {k}\right). \tag {39}$$

To find the optimal $\delta { \mathbf { u } } _ { k }$ that minimizes the cost function, we set its derivative with respect to $\delta { \bf { u } } _ { k }$ to zero. The solution, detailed in the Appendix 9.2, is given by:

$$\delta \mathbf {u} _ {k} = \left(\mathbf {H} ^ {\prime \top} \mathbf {Q} \mathbf {H} ^ {\prime} + \mathbf {R}\right) ^ {- 1} \left(\mathbf {H} ^ {\prime \top} \mathbf {Q} \tilde {\mathbf {x}} _ {k + 1} - \mathbf {R} \bar {\mathbf {u}} _ {k}\right). \tag {40}$$

where $\tilde { \mathbf { x } } _ { k + 1 }$ represents the desired state trajectory deviation from the predicted mean as $\tilde { \mathbf { x } } _ { k + 1 } : =$ $\mathbf { x } _ { d , k + 1 } - \bar { \mathbf { x } } _ { k + 1 }$ . This iterative update process for $\delta { \bf { u } } _ { k } .$ , with $\bar { \mathbf { u } } _ { k } \gets \bar { \mathbf { u } } _ { k } + \delta \mathbf { u } _ { k }$ until it converges, ensures dynamic optimization of control actions, with only the first control input implemented at each step. This cycle repeats, recalculating optimal controls at each new timestep, thereby adapting to evolving system dynamics and optimizing control strategies over time.
