$$
\left[ \begin{array}{c} z _ {\Delta 1, k} \\ z _ {\Delta 2, k} \end{array} \right] = \left[ \begin{array}{c c} D _ {\Delta \bar {\Psi} 1, k} & D _ {\Delta \bar {\Psi} 2, k} \\ 0 & I _ {n _ {q}} \end{array} \right] \left[ \begin{array}{c} q \\ p \end{array} \right]. \tag {13}
$$

For each bounded time-varying uncertainty block $\Delta _ { k } \in \Delta$ , the corresponding multipliers (11) and (12) are used to characterize its input-output behavior for $k \in \mathbf { I } [ 1 , \nu ]$ and $k \in$ ${ \bf I } [ \nu + 1 , n _ { q } ]$ , respectively. Owing to the special structure of Ψ¯ described above, the output $\begin{array} { r } { \dot { z } _ { \Delta , k } = \dot { \left[ z _ { \Delta 1 , k } ^ { \mathrm { T } } \quad z _ { \Delta 2 , k } ^ { \mathrm { T } } \right] } ^ { \mathrm { T } } } \end{array}$ is given by (13).

Similarly, the IQC-induced LTI system $\Psi _ { \mathcal { N } , l }$ associated with the sector-bounded nonlinearity $\dot { \mathcal { N } }$ in (6) can be represented in state-space form, for $l \in { \bf I } [ 1 , N _ { n _ { \psi } } ] \ ^ { 1 }$ , as

$$
\left[ \begin{array}{c} \dot {x} _ {\mathcal {N} \bar {\Psi}} \\ z _ {\mathcal {N} 1, l} \\ z _ {\mathcal {N} 2, l} \end{array} \right] = \left[ \begin{array}{c c c} A _ {\mathcal {N} \bar {\Psi}} & B _ {\mathcal {N} \bar {\Psi} 1} & B _ {\mathcal {N} \bar {\Psi} 2} \\ C _ {\mathcal {N} \bar {\Psi}, l} & D _ {\mathcal {N} \bar {\Psi} 1, l} & D _ {\mathcal {N} \bar {\Psi} 2, l} \\ 0 & 0 & I _ {n _ {u}} \end{array} \right] \left[ \begin{array}{c} x _ {\mathcal {N} \bar {\Psi}} \\ v \\ w \end{array} \right], \tag {14}
