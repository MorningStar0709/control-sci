The Softmax Policy Class. We assign weights $\theta _ { m } \in \mathbb { R }$ , to each controller $K _ { m } \in \mathcal { C }$ and define $\theta : = [ \theta _ { 1 } , \cdot \cdot \cdot , \theta _ { M } ]$ . The improper class $\mathcal { T } _ { s o f t }$ is parameterized by $\theta$ as follows. In each round, the policy $\pi _ { \theta } \in \mathcal { I } _ { s o f t } ( \mathcal { C } )$ chooses a controller drawn from softmax $( \theta )$ , i.e., the probability of choosing Controller $K _ { m }$ is given by, $\pi _ { \theta } ( m ) \ : = \quad$ $\begin{array} { r l } { \left. \boldsymbol { e } ^ { \theta _ { m } } / \left( \sum _ { m ^ { \prime } = 1 } ^ { M } e ^ { \theta _ { m ^ { \prime } } } \right) \right. } & { { } } \end{array}$ . Note, therefore, that in every round, our algorithm interacts with the MDP only through the controller sampled in that round.
