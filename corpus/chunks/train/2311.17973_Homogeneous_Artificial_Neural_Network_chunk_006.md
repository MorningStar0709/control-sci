# 1.2 Basic idea of homogeneous approximation

Following the classical model of a controlled rotational motion of the rigid body in the 3-D space, the dynamics of the angular velocity of the body is governed by

$$\dot {\omega} (t) = J ^ {- 1} (- \omega \times J \omega + \tau), \quad t \geq 0, \tag {1}$$

where $\omega ( t ) = ( \omega _ { 1 } ( t ) , \omega _ { 2 } ( t ) , \omega _ { 3 } ( t ) ) ^ { \top } \in \mathbb { R } ^ { n }$ is the angular velocity in the body frame, $J = J ^ { \top } \in \mathbb { R } ^ { n \times n }$ is the inertia matrix, $\tau ( t ) \in \mathbb { R } ^ { 3 }$ is the external momentum (e.g., control input) and × denotes the vector product, i.e., $\omega \times J \omega =$ $\left( \begin{array} { c c c } { 0 } & { - \omega _ { 3 } } & { \omega _ { 2 } } \\ { \omega _ { 3 } } & { 0 } & { - \omega _ { 1 } } \\ { - \omega _ { 2 } } & { \omega _ { 2 } } & { 0 } \end{array} \right) J \omega$ −ω1 . The function $h : \mathbb { R } ^ { 6 } \mapsto \mathbb { R } ^ { 3 }$ given by $h ( \omega , \tau ) = - J ^ { - 1 } ( \omega \times $ 0 $J \omega + \tau )$ is homogeneous of degree 2 with respect to a generalized dilation:

$$h (e ^ {s} \omega , e ^ {2 s} \tau) = e ^ {2 s} h (\omega , \tau), \quad \forall s \in \mathbb {R}, \forall \omega , \tau \in \mathbb {R} ^ {3}.$$

This means that the right-hand side of the differential equation (1) is homogeneous. Considering the problem of data-driven approximation the dynamic equation, if $h _ { \epsilon }$ is an ANN approximation of h on a compact $\Omega \subset \mathbb { R } ^ { 6 }$ such that $| h ( \omega , \tau ) - h _ { \epsilon } ( \omega , \tau ) | \le \epsilon$ for all $( \omega , \tau ) \in \Omega$ , then $h _ { \epsilon } ^ { s } ( \omega , \tau ) : = e ^ { 2 s } h _ { \epsilon } ( e ^ { - s } \omega , e ^ { - 2 s } \tau )$ is an approximation of h on the set $\mathbf { d } ( s ) \Omega$ , where ${ \bf d } ( s ) = \mathrm { d i a g } ( e ^ { s } I _ { 3 } , e ^ { 2 s } I _ { 3 } )$ is the generalized dilation. Indeed, due to d-homogeneity of $h ,$ the approximation error for $h _ { \epsilon } ^ { s }$ admits the estimate $| h ( \omega , \tau ) - h _ { \epsilon } ^ { s } ( \omega , \tau ) | = e ^ { 2 s } | h ( e ^ { - s } \omega , e ^ { - 2 s } \tau ) -$ $h _ { \epsilon } ( e ^ { - s } \omega , e ^ { - 2 s } \tau ) \vert \le e ^ { 2 s } \epsilon$ for all $( \boldsymbol { \omega } , \tau ) \in \mathbf { d } ( s ) \Omega$ . Since $s \in \mathbb { R } ^ { n }$ is an arbitrary number, then the mapping $h _ { \epsilon } ^ { s }$ can be interpreted as as a homogeneous extrapolation of $h _ { \epsilon }$ away from Ω. This paper utilizes such an extrapolation for a global ANN approximator design.
