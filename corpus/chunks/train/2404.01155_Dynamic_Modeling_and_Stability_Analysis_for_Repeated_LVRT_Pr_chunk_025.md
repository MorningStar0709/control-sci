# A. Parameter Sensitivity Analysis Based on the Sobol’ Method

The Sobol’ method is a quantitative global sensitivity analysis technique based on variance decomposition, capable of handling nonlinear and non-additive systems [41]. The basic principle of the Sobol’ method is introduced in this subsection.

The stability index calculation can be viewed as a black box described by a function $Y = g \left( X \right)$ , where the input vector $\boldsymbol { X } \ : = \ : [ \dot { X _ { 1 } } , X _ { 2 } , \cdot \cdot \cdot , X _ { k } ]$ considers k of multiple influencing parameters, and the output scalar $Y$ represents the stability index $\mu .$ . All the input variables $X _ { i }$ are independent and uniformly distributed in a k-dimensional unit hypercube $\Omega ^ { k } \ = \ \{ X | \bar { 0 } \ \leq \ X _ { i } \ \leq \ 1 , i \ = \ 1 , \cdots , k \}$ . Since there exist limited outputs for any points inside $\Omega ^ { k }$ , the function $Y = g \left( X \right)$ is continuous and integrable.

The high-dimensional model representation (HDMR) of $g \left( X \right)$ with $2 ^ { k }$ decomposition terms can be expressed as

$$
\begin{array}{l} g (\boldsymbol {X}) = g _ {0} + \sum_ {i = 1} ^ {k} g _ {i} (X _ {i}) + \sum_ {1 \leq i <   j \leq k} ^ {k} g _ {i, j} (X _ {i}, X _ {j}) \tag {29} \\ + \dots + g _ {1, 2, \dots , k} \left(X _ {1}, X _ {2}, \dots , X _ {k}\right) \\ \end{array}
$$

where $g _ { 0 }$ is a constant, and other terms are functions only of the input variables in their own indices. Besides, $g \left( X \right)$ and all the decomposition terms are integrable [42].

Since the integrable function $Y = g \left( X \right)$ is also squareintegrable over $\bar { \Omega } ^ { k }$ , (29) can be squared and integrated to obtain the decomposed variance expression of $Y$ as

$$\operatorname{Var} (Y) = \sum_ {i = 1} ^ {k} \operatorname{Var} _ {i} + \sum_ {1 \leq i < j \leq k} ^ {k} \operatorname{Var} _ {i, j} + \dots + \operatorname{Var} _ {1, 2, \dots , k} \tag {30}$$

where each term represents the partial variance contribution from the corresponding input variable or the interaction of multiple variables.
