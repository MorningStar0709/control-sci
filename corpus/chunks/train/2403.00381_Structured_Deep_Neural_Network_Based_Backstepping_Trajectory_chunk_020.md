# B. Controller Design and Performance Analysis

With the approximate model parameters, the NBS tracking controller (9) is modified to

$$
\begin{array}{l} \boldsymbol {u} = \hat {\boldsymbol {G}} + \hat {\boldsymbol {M}} \dot {\phi} + \hat {\boldsymbol {C}} \phi - \frac {\partial \Phi (\boldsymbol {z} _ {1} ; \boldsymbol {\theta} _ {1} , \boldsymbol {S})}{\partial \boldsymbol {z} _ {1}} - D (\boldsymbol {z} _ {2}; \boldsymbol {\theta} _ {2}) \boldsymbol {z} _ {2}, \\ \phi = \dot {\boldsymbol {q}} ^ {d} - \frac {\partial \Phi (\boldsymbol {z} _ {1} ; \boldsymbol {\theta} _ {1} , \boldsymbol {S})}{\partial \boldsymbol {z} _ {1}}. \tag {16} \\ \end{array}
$$

In the following theorem, we demonstrate that the modified NBS tracking controller can achieve bounded tracking error in the presence of modeling uncertainties.

Theorem 4. Consider the system (14) with a bounded disturbance $\| \tau ^ { d } \| ^ { 2 } \leq d ,$ and the model uncertainty satisfies (15). If the controller is designed as (16), with $\begin{array} { r } { D \ge ( b + \frac { b ^ { 2 } } { 2 } + } \end{array}$ $\begin{array} { r } { \frac { a } { 2 } + 1 ) \pmb { I } , \frac { \partial ^ { 2 } \Phi } { \partial z _ { 1 } ^ { 2 } } \Big | _ { z _ { 1 } = 0 } \geq } \end{array}$ αI and  to the set $\alpha > { \sqrt { a } } ,$ tracking, where $z _ { 1 }$ $\begin{array} { r } { \{ z _ { 1 } | \| z _ { 1 } \| ^ { 2 } \le \frac { k ^ { 2 } + d } { \alpha ^ { 2 } - a } \} } \end{array}$ $k = \operatorname* { m a x } ( c + a \| \pmb { q } ^ { d } \| + b \| \pmb { \dot { q } } ^ { d } \| )$ .

Proof. From (8) we have

$$\| \boldsymbol {q} \| \leq \| \boldsymbol {z _ {1}} \| + \| \boldsymbol {q} ^ {d} \|, \quad \| \dot {\boldsymbol {q}} \| \leq \| \boldsymbol {z _ {2}} \| + \| \dot {\boldsymbol {q}} ^ {d} \| + \| \frac {\partial \Phi (\boldsymbol {z _ {1}})}{\partial \boldsymbol {z _ {1}}} \|.$$

According to (15), one can obtain $\| \delta ( \pmb q , \pmb { \dot { q } } ) \| \ \leq \ a \| \pmb { z _ { 1 } } \| \ +$ $\begin{array} { r } { b \| z _ { 2 } \| + b \| \frac { \partial \Phi ( z _ { 1 } ) } { \partial z _ { 1 } } \| + c + a \| \pmb { q } ^ { d } \| + b \| \pmb { \dot { q } ^ { d } } \| } \end{array}$ . Since $\boldsymbol q ^ { d } , \dot { \boldsymbol q } ^ { d }$ are bounded reference states, we can define $k = \operatorname* { m a x } ( c + a \| q ^ { d } \| +$ $b \Vert \dot { \pmb q } ^ { d } \Vert )$ . Then we have the following.

$$\| \delta (\boldsymbol {q}, \dot {\boldsymbol {q}}) \| \leq a \| \boldsymbol {z _ {1}} \| + b \| \boldsymbol {z _ {2}} \| + b \| \frac {\partial \Phi (\boldsymbol {z _ {1}})}{\partial \boldsymbol {z _ {1}}} \| + k.$$
