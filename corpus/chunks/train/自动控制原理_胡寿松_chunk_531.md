# (1) 线性一阶系统的相轨迹

描述线性一阶系统自由运动的微分方程为

$$T \ddot {c} + c = 0$$

相轨迹方程为

$$\dot {c} = - \frac {1}{T} c \tag {8-23}$$

设系统初始条件为 $c(0) = c_0$ ，则 $\dot{c}(0) = \dot{c}_0 = -\frac{1}{T} c_0$ ，相轨迹如图8-14所示。
