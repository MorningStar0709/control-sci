Lemma 3.1: Let $\alpha ~ \geq ~ 0 , ~ \sigma ~ \in ~ \mathbb { R }$ , the plant in (1), and the controller in (3) be given. If there exist $X \in \mathbf { \Sigma }$ $\mathbb { R } ^ { \left( n _ { p } + n _ { k } \right) \times \left( n _ { p } + n _ { k } \right) }$ with $\boldsymbol { X } \succ 0 , \boldsymbol { \Lambda } _ { p } \in \mathbb { R } ^ { n _ { u } \times n _ { u } }$ with $\Lambda _ { p } \succeq 0$ and diagonal, and $\Lambda _ { k } \in \mathbb { R } ^ { n _ { \phi } \times n _ { \phi } }$ with $\Lambda _ { k } \succeq 0$ and diagonal, such that

$$
\left[ \begin{array}{c c} A ^ {\top} X + X A & X B _ {w} \\ B _ {w} ^ {\top} X & 0 \end{array} \right] + (\star) ^ {\top} M \left[ \begin{array}{c c} C _ {v} & D _ {v w} \\ 0 & I \end{array} \right] \preceq 0, \tag {10}
$$

then the controller satisfies the disk margin $D ( \alpha , \sigma )$ .

Proof: Since the closed-loop system is in the form of an LTI system interconnected with an uncertainty described by an integral quadratic constraint, we apply Lemma 1 of [8]. ■

Remark 1: This condition can also be used to find the largest α the closed-loop system satisfies by bisection on α.
