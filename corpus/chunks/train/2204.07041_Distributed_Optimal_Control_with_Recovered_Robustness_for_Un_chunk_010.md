# III. FORMULATION OF $H _ { 2 }$ AND $H _ { \infty }$ CONSENSUS PROBLEM

Consider a network of N identical linear agents subject to different noises and external disturbances. The dynamics of the i-th agent are described by

$$
\begin{array}{l} \dot {x} _ {i} = A x _ {i} + B _ {0} w _ {0 i} + B _ {1} w _ {i} + B _ {2} u _ {i}, \tag {2} \\ y _ {i} = C _ {2} x _ {i} + D _ {0} w _ {0 i} + D _ {1} w _ {i}, \quad i = 1, \dots , N, \\ \end{array}
$$

where $x _ { i } ~ \in ~ \mathbf { R } ^ { n }$ is the state, $u _ { i } ~ \in ~ \mathbf { R } ^ { p }$ is the control input, and $y _ { i } \in \mathbf { R } ^ { m _ { 1 } }$ is the measurement output of the i-th agent, respectively. In $( 2 ) , w _ { i } \in \mathbf { R } ^ { q _ { 1 } }$ denotes the external disturbance signal, representing the modeling uncertainty and/or unmodeled dynamics of the i-th agent, w0i $\mathbf { \Sigma } \in \mathbf { R } ^ { q _ { 2 } }$ is a white noise signal with ${ \bf E } \{ w _ { 0 i } ( t ) \} = 0$ and ${ \bf E } \{ w _ { 0 i } ( t ) w _ { 0 i } ( \tau ) ^ { T } \} = \delta ( t - \tau ) I$ . The matrices $A , B _ { 0 } , B _ { 1 } , B _ { 2 } , C _ { 2 } ,$ , $D _ { 0 }$ and $D _ { 1 }$ are of suitable dimensions. The pair $( A , B _ { 2 } )$ is assumed to be stabilizable and the pair $( C _ { 2 } , A )$ is assumed to be detectable.

The communication graph among the N agents is represented by an undirected graph G.

The agents in (2) are said to achieve consensus if there exist control laws ui such that, given $w _ { 0 i } = 0$ and $w _ { i } = 0 , x _ { i } - x _ { j } \to 0$ as $t \to \infty$ for all $i , j = 1 , \dots , N$ . In this paper, output variables $z _ { i } ,$ $i = 1 , \cdots , N$ , as defined in (3) (see [14], [16], [2] also), are adopted to quantify the consensus performance,

$$z _ {i} = \frac {1}{N} \sum_ {j = 1} ^ {N} C _ {1} (x _ {i} - x _ {j}), \quad i = 1, \dots , N, \tag {3}$$

where $z _ { i } \in \mathbf { R } ^ { m _ { 2 } }$ , and $C _ { 1 } \in \mathbf { R } ^ { m _ { 2 } \times n }$ is a given constant weighting matrix. Note that other candidate performance variables could also be applied, for example, those depending on the specific network topology G as in [15], [34].
