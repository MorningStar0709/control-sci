# 3.3.2. Type II: A New Observation at Every Step

In Type II, a new observation is obtained at every step. Therefore, the belief $\bar { W } _ { b , k }$ is updated according to the new observation $O _ { k }$ at every step. Each belief is still about the disturbance sequence from the current step to the terminal step, as shown in Fig. 4 (b). The belief is updated at every step using the new observation and the predictor,

$$\bar {W} _ {b, k} = \mathcal {P} _ {k} (o _ {k} (s)), \tag {15}$$

where the predictor P may be dependent on k. We denote the sequence of beliefs $[ \bar { W } _ { b , 0 } , \bar { W } _ { b , 1 } , . . . , \bar { W } _ { b , N - 1 } ]$ as $\bar { \bar { W } } _ { b }$ , and we write

$$\bar {\bar {W}} _ {b} = \bar {\mathcal {P}} (\bar {o} (s)), \tag {16}$$

In predictive optimal control, the control policy at step k is determined using the belief $\bar { W } _ { b , k }$ , just like at any step of Type I. So the at every step, the feedback control policy is in the form

$$u _ {k} = \pi_ {\bar {W} _ {b, k}} (k, x _ {k}). \tag {17}$$
