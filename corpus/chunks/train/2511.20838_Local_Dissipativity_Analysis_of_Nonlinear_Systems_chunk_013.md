# 3 Error bounds for function synthesis

Lemma 9 is an invaluable tool for ensuring that CPA function adheres to a scalar inequality condition. In this section, we expand this tool to more complex MI conditions, which often codify complex control analysis and synthesis requirements. We develop novel error bounds for an MI on an n-simplex and later leverage them for dissipativity analysis.

In Section 3.2, an additional LMI is developed to enforce a scalar inequality constraint over a closed ball about the origin. This is later used to combine CPA and quadratic storage functions for dissipativity analysis.
