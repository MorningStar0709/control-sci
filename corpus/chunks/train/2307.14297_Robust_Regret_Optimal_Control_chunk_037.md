# B. Boeing 747

A simplified (linearized) model for the longitudinal dynamics of a Boeing 747 at one steady, level flight condition are given by (Problem 17.6 in [34]):

$$x _ {t + 1} = A x _ {t} + B \tilde {u} _ {t} + d _ {t}, \tag {30}$$

where the state matrices are:

$$
A = \left[ \begin{array}{c c c c} 0. 9 9 & 0. 0 3 & - 0. 0 2 & - 0. 3 2 \\ 0. 0 1 & 0. 4 7 & 4. 7 & 0 \\ 0. 0 2 & - 0. 0 6 & 0. 4 & 0 \\ 0. 0 1 & - 0. 0 4 & 0. 7 2 & 0. 9 9 \end{array} \right] \quad \text { and } \quad B = \left[ \begin{array}{c c} 0. 0 1 & 0. 9 9 \\ - 3. 4 4 & 1. 6 6 \\ - 0. 8 3 & 0. 4 4 \\ - 0. 4 7 & 0. 2 5 \end{array} \right].
$$

We will assume there is a multiplicative uncertainty at the plant input:

$$\tilde {u} = (I _ {2} + 0. 6 \Delta) u, \tag {31}$$

where $u _ { t } \in \mathbb { R } ^ { 2 }$ is the (nominal) command from the controller and $\Delta$ is a $2 \times 2$ , stable, LTI uncertainty such that $\| \Delta \| _ { \infty } \leq 1$ . Multiplicative uncertainty is a commonly used to account for non-parametric (dynamic) modeling errors [1]. No assumptions are made regarding the statedimension of $\Delta .$ The constant factor of 0.6 implies that the effect of the uncertainty can be up to 60% of the size of the nominal control at each frequency. This factor could be made frequency-dependent to account for increased non-parametric errors at high frequencies. We’ll use the constant factor for simplicity.

Define the generalized error as $e _ { t } = \left[ { \begin{array} { l } { x _ { t } } \\ { u _ { t } } \end{array} } \right]$ . The corresponding per-step cost is $e _ { t } ^ { \top } e _ { t } = x _ { t } ^ { \top } x _ { t } +$ $u _ { t } ^ { \top } u _ { t }$ . We’ll also assume the controller has access to full information measurements, i.e. $y _ { t } = \left[ { \begin{array} { l } { x _ { t } } \\ { d _ { t } } \end{array} } \right]$ . Based on this information, we can formulate the robust regret optimal control design as in Figure 4. Specifically, define $( v , w )$ as the input/output signals of the uncertainty in (31), i.e. $w = \Delta v$ with v = u. Then substitute $\tilde { u } = u + 0 . 6 w$ into (30):

$$x _ {t + 1} = A x _ {t} + (0. 6 B) w _ {t} + d _ {t} + B u _ {t}. \tag {32}$$

This has the form of the state update equation for the uncertain plant $P$ in (17) with $B _ { w } = 0 . 6 B$ , $B _ { d } = I _ { 4 }$ , and $B _ { u } = B$ . The output equations for $( v , e , y )$ in (17) can be derived similarly. Note that the nominal dynamics are given by $\Delta = 0$ and neglecting the $( v , w )$ channels.
