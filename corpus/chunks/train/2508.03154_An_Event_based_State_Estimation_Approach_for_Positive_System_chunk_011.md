Let $D = \left[ { \begin{array} { c c } { P } & { 0 } \\ { 0 } & { Q } \end{array} } \right]$ Q 0 where $P$ $Q$ are diagonal positive definite matrices, then $D \bar { L } = \left[ { \begin{array} { c c } { P } & { 0 } \\ { 0 } & { Q } \end{array} } \right] \left[ { \begin{array} { c } { 0 } \\ { L } \end{array} } \right] =$ $\left[ \begin{array} { l } { 0 } \\ { Q L } \end{array} \right]$ .

The 1st term $\xi ^ { \top } ( t ) ( D \bar { A } + \bar { A } ^ { \top } D ) \xi ( t )$ in (16) can be represented as,

$$
\left[ \begin{array}{c c} P A + A ^ {\top} P & (\alpha \beta + \beta - 1) C ^ {\top} L ^ {\top} Q \\ * & Q A + A ^ {\top} Q - Q L C - C ^ {\top} L ^ {\top} Q \end{array} \right] \prec 0. \tag {20}
