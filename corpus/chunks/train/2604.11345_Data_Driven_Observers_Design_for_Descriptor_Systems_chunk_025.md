Remark 9 Condition (48), known as the observer matching condition, guarantees instantaneous unknown input isolation via the measurement y(k) and the algebraic constraints of E [27,28]. When the descriptor system (34) reduces to a standard LTI system $( i . e .$ , $E = I _ { n } ) $ , condition (48) simplifies to $r k ( C \bar { F } ) = r k ( F )$ , which exactly coincides with the classical UIO matching condition (see [29]).

Under Assumption 4, since $r k \left[ { \begin{array} { l l } { E } & { F } \\ { C } & { 0 } \\ { I _ { n } } & { 0 } \end{array} } \right] \ = n + r k ( F ) \ =$ $r k \left[ { \begin{array} { l } { E } \\ { C } \end{array} } F \right]$ , there exist matrices $\bar { T } \in \mathbb { R } ^ { n \times n }$ and $\bar { N } \in \mathbb { R } ^ { n \times p }$ such that

$$\bar {T} E + \bar {N} C = I _ {n}, \quad \bar {T} F = 0. \tag {49}$$

By using (49), the system (34) is transformed into:

$$x (k + 1) = \bar {T} A x (k) + \bar {T} B u (k) + \bar {N} y (k + 1). \tag {50}$$

Theorem 4 (Equivalence of model-based and data-driven UIO conditions) Suppose that Assumptions 3 and 4 hold. Then, the kernel inclusion condition (42) is always satisfied. Moreover, the following statements are equivalent:

(i) There exists a full-order UIO of the form (11) for system (34).   
(ii) There exist parameter matrices $\bar { \Sigma } _ { X _ { p } } , \bar { \Sigma } _ { U _ { p } } , \bar { \Sigma } _ { Y _ { p } }$ , and $\bar { \Sigma } _ { Y _ { f } }$ satisfying (43), with $\bar { \Sigma } _ { X _ { p } }$ being Schur stable. (iii)

$$
