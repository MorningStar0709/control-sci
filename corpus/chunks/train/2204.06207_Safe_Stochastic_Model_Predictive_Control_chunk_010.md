3) Safe SMPC Modes: We are now able to propose two different modes within the safe SMPC algorithm, evaluated at time step t.

Stochastic Mode: The control law ${ \textbf { \em u } } _ { t } ~ = ~ { \textbf { \em u } } ^ { \mathrm { s } } ( { \textbf { \em x } } _ { t } )$ is applied if the SMPC OCP is feasible and if (9) is fulfilled, $\mathrm { i . e . , } U _ { t } ^ { \mathrm { s * } } \neq \emptyset$ and $\overline { { \boldsymbol { x } } } _ { t + 1 } ^ { \mathrm { s } } \in \overline { { \boldsymbol { \mathcal { X } } } } _ { 0 }$ .

Backup Mode: If the SMPC OCP is infeasible or if (9) is not satisfied, i.e., $U _ { t } ^ { \mathrm { s } * } = \varnothing$ or $\overline { { \boldsymbol { x } } } _ { t + 1 } ^ { \mathrm { s } } ~ \notin ~ \overline { { \boldsymbol { \mathcal { X } } } } _ { 0 }$ , the backup MPC OCP is solved and the control law ${ \pmb u } _ { t } = { \pmb u } ^ { \mathrm { b } } ( { \pmb x } _ { t } )$ is applied.

Note that in the stochastic mode, only one OCP is solved, while the backup mode requires solving two OCPs.
