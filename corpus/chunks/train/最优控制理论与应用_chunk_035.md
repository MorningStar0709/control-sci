# 2.4 习题

1. 求使 $f(x) = \ln x - x^2$ 最小的 $x$ 。  
2. 求使 $f(X) = -10x_1^2 - 12x_1x_2 - 4x_2^2$ 为极值的极值点 $X$ 。  
3. 求使 $f(X) = 5x_{1}^{2} + x_{2}^{2} + 5x_{3}^{2} + 4x_{1}x_{2} - 8x_{1}x_{3} - 4x_{2}x_{3}$ 为极值的极值点 $X$ 。  
4. 求使 $f(X) = 4x_{1}^{2} + 5x_{2}^{2}$ 最小，且 $g(X) = 2x_{1} + 3x_{2} - 6 = 0$ 的 $X$ 。  
5. 求原点到曲线 $y^{2} - (x - 1)^{3} = 0$ 的距离为最小的点。  
6. 求函数 $f(X) = \sqrt{x_1^2 + x_2^2 + x_3^2}$ 的极值，若 $(x_{1} - x_{2})^{2} - x_{3}^{2} = 1$ 。  
7. 在第一象限内作椭球面 $\frac{x^2}{a^2} + \frac{y^2}{b^2} + \frac{z^2}{c^2} = 1$ 的切平面，使切平面与三坐标面所围成的四面体体积最小，求切点的坐标。
