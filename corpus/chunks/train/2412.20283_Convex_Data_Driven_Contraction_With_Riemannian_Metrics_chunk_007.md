# C. Sum-of-Squares

Here we briefly review some concepts from semialgebraic optimization that will be used to recast the data-driven contraction problem into a tractable convex optimization. A polynomial $p ( x )$ is an SOS iff it can be written as $\textstyle p ( x ) = \sum _ { i } q _ { i } ( x ) ^ { 2 }$ , for some polynomials $q _ { i } .$ . Alternatively, $p ( x )$ is SOS iff there exists a polynomial vector $v ( x )$ and a matrix $Q \succeq 0$ such that $p ( x ) = v ( x ) ^ { \top } Q v ( x )$ .

A semialgebraic set  isbounded degree polynomials $\{ g _ { i } ( x ) \} _ { i = 1 } ^ { N _ { g } }$ y a fi and $\{ h _ { j } ( x ) \} _ { j = 1 } ^ { N _ { h } }$ r of:

$$
\begin{array}{l} \mathbb {K} = \{x \in \mathbb {R} ^ {n} \mid g _ {i} (x) \geq 0, h _ {j} (x) = 0, \\ i = 1, \dots , N _ {g}, j = 1, \dots , N _ {h} \}. \tag {2} \\ \end{array}
$$

The set K is Archimedean if there exists an $R > 0$ such that the polynomial $R - \| x \| _ { 2 } ^ { 2 }$ satisfies

$$R - \| x \| _ {2} ^ {2} = \sigma_ {0} (x) + \sum_ {i} \sigma_ {i} (x) g _ {i} (x) + \sum_ {j} \phi_ {j} (x) h _ {j} (x)$$

for some polynomials $\phi _ { j } ( x )$ and SOS polynomials $\sigma _ { i } ( x )$ . A matrix-valued polynomial ${ \dot { P } } \in S _ { n } ^ { + + } [ x ]$ if $\forall x \in \mathbb { R } ^ { n } : P ( x ) \in$ $S _ { n } ^ { + + }$ . A matrix-valued polynomial is SOS if there exists a vector of polynomials $v ( x )$ and a Gram matrix $Q \succeq 0$ such that $P ( x ) = ( v ( x ) \otimes I ) ^ { \top } Q ( v ( x ) \otimes I )$ . Every positive definite matrix-valued polynomial $P ( x )$ over an Archimedean set K satisfies (Theorem 2 of [21], Scherer Positivstellensatz)

$$P (x) = \sigma_ {0} (x) + \sum_ {i} \sigma_ {i} (x) g _ {i} (x) + \sum_ {j} \phi_ {j} (x) h _ {j} (x). \tag {3}$$

for $\sigma _ { i } ( x )$ SOS polynomial matrices and $\phi _ { j }$ symmetric polynomial matrices. The Putinar Positivstellensatz is the restriction of the Scherer Positivstellensatz to n = 1 [22].
