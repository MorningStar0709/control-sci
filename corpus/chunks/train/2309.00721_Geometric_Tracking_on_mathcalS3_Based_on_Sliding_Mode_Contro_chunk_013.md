# 3.2 Sliding Subgroup

The sliding subgroup, defined below, inherits the structure of the Lie group $T \mathcal { S } ^ { 3 }$ , allowing us to design sliding mode controllers on the Lie group.

Lemma 3 (Sliding subgroup) For $g = ( q , p ) \in T S ^ { 3 }$ , let the sliding variable $s ( g )$ be defined as

$$s (g) = p + \lambda \left(q _ {0} q - \bar {1}\right) \tag {14}$$

where $\lambda > 0$ is a scalar.

Then the set

$$H = \left\{g \in T \mathcal {S} ^ {3} \mid s (g) = 0 _ {4 \times 1} \right\} \tag {15}$$

is a Lie subgroup.

PROOF. Its clear that (14) defines a smooth map. The associativity of H is inherited from $T S ^ { 3 }$ . Therefore, it remains to verify the following:

• Identity $e \in H \colon s ( e ) = s ( \bar { 1 } , 0 _ { 4 \times 1 } ) = 0 .$ .

• Inverse: Let $g = ( q , p ) \in H$ . Then $s ( g ) = 0 \implies p =$ $- \lambda \left( q _ { 0 } q - \bar { 1 } \right)$ . Then

$$s \left(g ^ {- 1}\right) = p ^ {- 1} + \lambda \left(q _ {0} q ^ {- 1} - \bar {1}\right)= - \lambda (q _ {0} q ^ {- 1} - \bar {1}) + \lambda (q _ {0} q ^ {- 1} - \bar {1}) = 0 _ {4 \times 1}.$$

This shows that $g ^ { - 1 } \in H .$ .

• Closure of the group operation: $\forall g _ { 1 } \ = \ ( q _ { 1 } , p _ { 1 } )$ and $g _ { 2 } = ( q _ { 2 } , p _ { 2 } ) \in \boldsymbol { H }$ , it has that $p _ { 1 } = - \lambda \left( q _ { 0 , 1 } q _ { 1 } - \hat { 1 } \right)$ and $p _ { 2 } = - \lambda \left( q _ { 0 , 2 } q _ { 2 } - \bar { 1 } \right)$ . Thus,

$$s \left(g _ {1} \cdot g _ {2}\right) = s \Big (Q (q _ {1}) q _ {2}, Q (q _ {1}) p _ {2} + W (q _ {2}) p _ {1} \Big)+ \lambda Q (q _ {1}) (q _ {0, 2} q _ {2} - \bar {1}) + \lambda W (q _ {2}) (q _ {0, 1} q _ {1} - \bar {1})\left. - \lambda (q _ {0, 1 2} Q (q _ {1}) q _ {2} - \bar {1})\right)= Q \left(q _ {1}\right) p _ {2} + W \left(q _ {2}\right) p _ {1} + \lambda Q \left(q _ {1}\right) \left(q _ {0, 2} q _ {2} - \bar {1}\right)+ \lambda W (q _ {2}) (q _ {0, 1} q _ {1} - \bar {1}) - \lambda (q _ {0, 1 2} Q (q _ {1}) q _ {2} - \bar {1})+ \lambda (q _ {0, 1 2} Q (q _ {1}) q _ {2} - \bar {1})= Q (q _ {1}) (- \lambda (q _ {0, 2} q _ {2} - \bar {1}))+ W (q _ {2}) (- \lambda (q _ {0, 1} q _ {1} - \bar {1}))+ \lambda Q (q _ {1}) (q _ {0, 2} q _ {2} - \bar {1}) + \lambda W (q _ {2}) (q _ {0, 1} q _ {1} - \bar {1})= 0 _ {4 \times 1}.$$

Therefore, $g _ { 1 } \cdot g _ { 2 } \in H .$
