# 2.1 System dynamics and robustness

Consider the continuous-time linear dynamical system

$$\dot {x} (t) = A x (t) + B u (t) + B _ {w} w (t), \tag {1}$$

where $x ( t ) \in \mathrm { R } ^ { n }$ is the state, $u ( t ) \in \mathrm { R } ^ { m }$ is the control input, $w ( t ) \in \operatorname { R } ^ { n }$ is the disturbance. We focus on static state feedback policies of the form $u ( t ) = K x ( t )$ , where $K \in \mathrm { R } ^ { m \times n }$ . The set of stabilizing policies is defined as

$$\mathcal {K} = \left\{K \in \mathrm{R} ^ {m \times n} \mid A + B K \text { is Hurwitz } \right\}. \tag {2}$$

We henceforth denote $A _ { K } : = A + B K$ and $W : = B _ { w } B _ { w } ^ { \sf T }$

In standard LQR, the disturbance $w ( t )$ is modeled as zero-mean white Gaussian noise with identity covariance, i.e., $\mathrm { E } [ w ( t ) w ( \tau ) ] = \delta ( t - \tau ) I _ { n }$ . The objective is to find a stabilizing policy $K \in { \mathcal { K } }$ that minimizes the averaged cost lim $\begin{array} { r } { \mathbf { \Omega } ^ { _ { 1 } } . . . _ { \infty } \operatorname { E } \left[ \frac { 1 } { T } \int _ { 0 } ^ { T } x ( t ) ^ { \mathsf { T } } Q _ { 2 } x ( t ) + u ( t ) ^ { \mathsf { T } } R _ { 2 } u ( t ) d t \right] } \end{array}$ , where $Q _ { 2 } \succeq 0$ and $R _ { 2 } ~ \succ ~ 0$ are performance weight matrices.
