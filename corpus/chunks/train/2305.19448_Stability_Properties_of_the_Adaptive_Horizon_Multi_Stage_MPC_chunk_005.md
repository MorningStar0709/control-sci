# 2. Standard MPC

We define the standard MPC as follows. For system (1) the controller receives the current state $x _ { k }$ and assumes a nominal parameter $d ^ { 0 }$ . The optimal control problem (OCP) for standard MPC is a parametric NLP in $x _ { k }$ and $d ^ { 0 }$ written as follows:

$$V _ {N} ^ {\mathrm{nom}} (x _ {k}, d ^ {0}) = \min _ {z _ {i}, v _ {i}} \quad \psi (z _ {N}, d ^ {0}) + \sum_ {i = 0} ^ {N - 1} \ell (z _ {i}, v _ {i}, d ^ {0}) \tag {2a}\text { s.t. } \quad z _ {i + 1} = f (z _ {i}, v _ {i}, d ^ {0}), \quad i = 0, \dots , N - 1 \tag {2b}z _ {0} = x _ {k}, \tag {2c}z _ {i} \in \mathbb {X}, \nu_ {i} \in \mathbb {U}, \tag {2d}z _ {N} \in \mathbb {X} _ {f} \tag {2e}$$

where $N$ is the prediction horizon, $V _ { N } ^ { \mathrm { n o m } }$ is the optimal cost, $z _ { i }$ and $\nu _ { i }$ are the state and input variable vectors, respectively at time $t _ { k + i }$ . The objective (2a) consists of a stage cost $\ell \colon \mathbb { R } ^ { n _ { x } + n _ { u } + n _ { d } } \mapsto \mathbb { R }$ , and a terminal cost $\psi \colon \mathbb { R } ^ { n _ { x } + n _ { d } } \mapsto \mathbb { R } .$ 1
