# 2.1 Certainty-Equivalent Linear Quadratic Control

We fix an honest system $i \in \mathcal { C } _ { j }$ with $u _ { \mathrm { e x p } } ^ { ( i ) } = 0$ (for now). The local control objective is to design a linear feedback controller that minimizes the infinite-horizon quadratic cost

$$J ^ {(i)} (K) \triangleq \operatorname * {l i m s u p} _ {T \to \infty} \frac {1}{T} \mathbb {E} _ {K} \left[ \sum_ {t = 0} ^ {T - 1} c _ {t} ^ {(i)} \right], \text {with} c _ {t} ^ {(i)} \triangleq x _ {t} ^ {(i) \top} (Q + K ^ {\top} R K) x _ {t} ^ {(i)},$$

where $Q \succeq 0$ and $R \succ 0$ . When the ground-truth dynamics $( A _ { \star } ^ { ( i ) } , B _ { \star } ^ { ( i ) } )$ are known, the optimal controller $K _ { \star } ^ { ( i ) } = K ( A _ { \star } ^ { ( i ) } , B _ { \star } ^ { ( i ) } ) \triangleq - \left( R + B _ { \star } ^ { ( i ) \top } P _ { \star } ^ { ( i ) } B _ { \star } ^ { ( i ) } \right) ^ { - 1 } B _ { \star } ^ { ( i ) \top } P _ { \star } ^ { ( i ) } A _ { \star } ^ { ( i ) }$ is obtained by solving the discrete algebraic Riccati equation (DARE), $P _ { \star } ^ { ( i ) } = \tt D A R E ( A  _ { \star } ^ { ( i ) } , B _ { \star } ^ { ( i ) } , Q , R )$ .

In practice, the true model parameters $( A _ { \star } ^ { ( i ) } , B _ { \star } ^ { ( i ) } )$ are typically unknown and must be inferred from data. A standard approach to designing a near-optimal controller is certainty-equivalent control [Mania et al., 2019], which first estimates the system model and then computes the controller as if the estimate were exact. In particular, by using trajectory data $\mathcal { D } _ { \tau } ^ { ( i ) }$ , each system performs ordinary least-squares (OLS) estimation as

$$\widehat {\Theta} ^ {(i)} \triangleq [ \hat {A} ^ {(i)} \hat {B} ^ {(i)} ] = \underset {\Theta \in \mathbb {R} ^ {d _ {x} \times d ^ {\prime}}} {\operatorname{argmin}} \left\| X _ {\tau} ^ {(i)} - \Theta Z _ {\tau} ^ {(i)} \right\| _ {F} ^ {2}, \tag {3}$$

and designs a controller $\hat { K } ^ { ( i ) } = K ( \hat { A } ^ { ( i ) } , \hat { B } ^ { ( i ) } )$ with the estimated model.
