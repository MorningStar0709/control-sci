The constant $\alpha > 0$ is called the residual constant while the function is called the gain function of the input $d \in D$ to the output  . It is clear that the gain function is a measure of the sensitivity of system (36), (2) to the disturbance $d \in D$ . Notice that p-IOS with residual constant $\alpha > 0$ is equivalent to IOS for system (36) with output $y = \left( \left| h ( x ) \right| - \alpha \right) ^ { + }$ . In the disturbance-free case, the p-IOS property coincides with the p-UGAOS property.

We say that system (36), (2) satisfies the practical Output Asymptotic Gain $( p { \mathord { - } } O A G )$ property if there exists a non-decreasing, continuous function $\tilde { \gamma } : \mathbb { R } _ { + } \to \mathbb { R } .$ with $\tilde { \gamma } ( 0 ) = 0$ and a constant $\tilde { \alpha } > 0$ such that the following estimate holds for all $x _ { 0 } \in \mathbb { R } ^ { n }$ and for every $d \in L ^ { \infty } \left( \mathbb { R } _ { + } ; D \right)$ :

$$\operatorname * {l i m s u p} _ {t \to + \infty} \left(\left| y (t, x _ {0}; d) \right|\right) \leq \tilde {\gamma} \left(\left\| d \right\| _ {\infty}\right) + \tilde {\alpha} \tag {39}$$

The constant $\tilde { \alpha } > 0$ is called the asymptotic residual constant while the non-decreasing, continuous function $\tilde { \gamma }$ is called the asymptotic gain function of the input $d \in D$ to the output the asymptotic gain function and the asymptotic residual constant are additional measures of the sensitivity of system (36), (2) to the disturbance $d \in D$ . It is a consequence of the semigroup property that if system (36), (2) satisfies the p-IOS property then system (36), (2) also satisfies the p-OAG property with asymptotic gain function being less than or equal to the gain function (i.e., $\tilde { \gamma } ( s ) \leq \gamma ( s )$ for all $s \geq 0 )$ and asymptotic residual constant being less than or equal to the residual constant $( \mathrm { i } . \mathrm { e } . , \ \tilde { \alpha } \leq \alpha \ )$ . In the disturbance-free case, the p-OAG property coincides with the p-GOA property.

When $\tilde { \gamma } \equiv 0$ we say that system (36), (2) satisfies the zero practical Output Asymptotic Gain property (zero p-OAG).

When $h ( x ) = x$ then the word “output” in the above properties is either replaced by the word “state” (e.g., ISS, p-ISS) or is omitted (e.g., p-AG, zero p-AG).
