# Solution:

We can devise the following sequence $( x _ { k } )$ whose upper limit exists:

$$\frac {x}{2 x + 1} = \frac {1}{3}, \frac {2}{5}, \frac {3}{7}, \dots \tag {1}$$

We can easily find its upper limit by:

$$\lim _ {k \to \infty} s u p (x _ {k}) = \frac {1}{2} \tag {2}$$

As for the infimum, we can devise another sequence $( x _ { k } )$ as:

$$\frac {2 x + 1}{x} = 3, \frac {5}{2}, \frac {7}{3}, \dots \tag {3}$$

We can easily find its infimum by:

$$\lim _ {k \to \infty} i n f (x _ {k}) = 2 \tag {4}$$
