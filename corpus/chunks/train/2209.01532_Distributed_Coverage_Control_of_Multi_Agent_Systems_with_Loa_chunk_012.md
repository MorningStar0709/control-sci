# Lemma 3.5.

$$\lim _ {t \to \infty} \dot {c} _ {E _ {i}} (t) = 0, \quad i \in I _ {N}.$$

Proof. It follows from Lemma 3.1 and Equations (2) that $\begin{array} { r } { \operatorname* { l i m } _ { t  \infty } \dot { \varphi } _ { i } ( t ) = 0 , i \in I _ { N } } \end{array}$ . In light of Equation (1), one has

$$\dot {m} _ {i} (t) = \omega (\varphi_ {i + 1}) \dot {\varphi} _ {i + 1} (t) - \omega (\varphi_ {i}) \dot {\varphi} _ {i} (t),$$

which leads to lim $\mathsf { i } _ { t  \infty } \dot { m } _ { i } ( t ) = 0$ . For simplicity, introduce the following notation

$$
m _ {i, x} = \left\{ \begin{array}{l l} \int_ {\varphi_ {i}} ^ {2 \pi} \omega_ {x} (\theta) d \theta + \int_ {0} ^ {\varphi_ {i + 1}} \omega_ {x} (\theta) d \theta , & \text {   if   } \varphi_ {i + 1} <   \varphi_ {i} \\ \int_ {\varphi_ {i}} ^ {\varphi_ {i + 1}} \omega_ {x} (\theta) d \theta . & \text {   otherwise   } \end{array} \right.
$$

and one has

$$\dot {m} _ {i, x} (t) = \omega_ {x} (\varphi_ {i + 1}) \dot {\varphi} _ {i + 1} (t) - \omega_ {x} (\varphi_ {i}) \dot {\varphi} _ {i} (t),$$

which leads to lim $\mathbf { \boldsymbol { 1 } } _ { t  \infty } \dot { m } _ { i , x } ( t ) = 0$ . It follows from li $\begin{array} { r } { \mathsf { 1 } _ { t  \infty } \dot { m } _ { i } ( t ) = \operatorname* { l i m } _ { t  \infty } \dot { m } _ { i , x } ( t ) = 0 } \end{array}$ that

$$\lim _ {t \to \infty} \dot {c} _ {E _ {i}} ^ {x} (t) = \lim _ {t \to \infty} \frac {\dot {m} _ {i , x} (t) \cdot m _ {i} (t) - m _ {i , x} (t) \cdot \dot {m} _ {i} (t)}{(m _ {i} (t)) ^ {2}} = 0.$$
