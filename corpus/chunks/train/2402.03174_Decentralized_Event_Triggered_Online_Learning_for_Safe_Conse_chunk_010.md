# A. Gaussian Process Regression

To infer the unknown $f ( \cdot )$ , Gaussian process regression is adopted, which induces a distribution of f (·) by the prior mean $m ( \cdot ) : \mathbb { X }  \mathbb { I }$ R and kernel function $\kappa ( \cdot , \cdot ) : \mathbb { X } \times \mathbb { X } $ $\mathbb { R } _ { 0 , + } , \mathrm { i . e . , ~ } f ( \cdot ) \sim \mathcal { G P } ( m ( \cdot ) , \kappa ( \cdot , \cdot ) )$ . The prior mean m(·) reflects the known part of $f ( \cdot )$ , which for the dynamics in (1) is encoded in $h ( \cdot )$ . Therefore, $m ( \cdot ) = 0$ is set. The kernel function $\kappa ( \cdot )$ indicates the covariance between two sample points and satisfies the following assumption.

Assumption 3: The continuous function $f ( \cdot )$ is a sample from a Gaussian process $\mathcal { G P } ( 0 , \kappa ( \cdot , \cdot ) )$ ) with a Lipschitz kernel $\kappa ( | x - x ^ { \prime } | ) = \kappa ( x , x ^ { \prime } )$ w.r.t $\vert x - x ^ { \prime } \vert$ .

From this assumption, the prior distribution of the unknown function $f ( \cdot )$ is defined. Moreover, the assumption on the kernel $\kappa ( \cdot , \cdot )$ can be achieved by choosing any continuous function reflecting the continuity of $f ( \cdot )$ , and then its Lipschitz continuity is derived by considering the compact input domain X. And the commonly used kernels, such as square exponential kernel, satisfy the Lipschitz condition. Therefore, this assumption is practically not restrictive.

Consider an individual data set $\mathbb { D } _ { i }$ on agent $i \in \nu$ containing $M \in \mathbb { N }$ training samples and satisfying Assumption 2, the posterior mean $\mu _ { i } ( \cdot )$ and variance $\sigma _ { i } ( \cdot )$ are obtained by using the Gaussian process model with Assumption 3 as

$$\mu_ {i} (x _ {i}) = \boldsymbol {k} _ {X, i} ^ {T} (\boldsymbol {K} _ {i} + \sigma_ {n} ^ {2} \boldsymbol {I} _ {M}) ^ {- 1} \boldsymbol {y}, \tag {6}\sigma_ {i} ^ {2} (x _ {i}) = \kappa (0) - \boldsymbol {k} _ {X, i} ^ {T} (\boldsymbol {K} _ {i} + \sigma_ {n} ^ {2} \boldsymbol {I} _ {M}) ^ {- 1} \boldsymbol {k} _ {X, i}, \tag {7}$$
