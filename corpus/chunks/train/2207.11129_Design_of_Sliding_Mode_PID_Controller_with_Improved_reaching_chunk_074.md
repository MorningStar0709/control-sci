# 3.3 Stability

3.3.1 Lyapunov stability criteria’s origin dates back to $1 9 7 0 ^ { \circ } \mathrm { s }$ by Leitmannn and Gutman [38, 67-68]. It is the most popular approach to evaluate and to prove the stable convergence property of the sliding mode controller. Second Lyapunov approach is used to prove the stability. Candidate lyapunov function is given as:

$$V (t) = \frac {1}{2} s ^ {2} (t) \text { with } \mathrm{V} (0) = 0 \text { and } \mathrm{V} (\mathrm{t}) > 0 \text { for } \mathrm{s} (\mathrm{t}) \neq 0. \tag {3.20}$$

A sufficient condition or reaching condition is to select the control law that forces the trajectory of error to the sliding phase from reaching phase, given as

$$\dot {\mathcal {V}} (s) = - s ^ {T} \operatorname{diag} \left\{\operatorname{sign} \left(s _ {i}\right) \right\} | \dot {s} | = - \left| s ^ {T} \right| | \dot {s} | < 0 \tag {3.21}$$

Substituting (16) in (21), we get

$$\dot {\mathcal {V}} (t) = s (t) (- k _ {s c} \text {sign} (s (t)) - d (t)) \tag {3.22}$$

which is derived to prove the Lyapunov criteria.

$$\dot {\mathcal {V}} (t) = - k _ {s c} | s (t) | - s (t) d (t) \tag {3.23}\dot {\mathcal {V}} (t) \leq - k _ {s c} | s (t) | + s (t) d _ {\max} \tag {3.24}\dot {\mathcal {V}} (t) \leq - | s (t) | (k _ {s c} - d _ {m a x}) \tag {3.25}\dot {\mathcal {V}} (t) \leq 0. \tag {3.26}$$

In the reaching phase with $s ( t ) \neq 0$ , we get $k _ { s c } > d _ { m a x }$ and $| s ( t ) | > 0$ which lead to $\dot { \mathcal { V } } ( t ) \leq$ 0 i.e. a negative definite condition, satisfying the direct Lyapunov stability criteria.
