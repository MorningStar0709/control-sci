Instead of bounding the error over the whole domain (as in $\bar { \epsilon } _ { 1 } ) , \bar { \epsilon } _ { 2 } ( x ^ { * } , \theta )$ is tighter as it only bounds the error over the tube, $\varOmega _ { c } ( t )$ . However, due to the leading term in $( \mathbf { B } . 3 ) , \bar { \epsilon } _ { 2 } ( x ^ { * } , \theta )$ scales linearly with $d _ { c } ,$ even if the true error $\epsilon ( x , \theta )$ is relatively constant, making it loose for large $d _ { c }$ . Thus, we derive a third error bound to mitigate this, $\bar { \epsilon } _ { 3 } ( x ^ { * } , \theta )$ (cf. Fig. 4.C). Overall, this third error bound $\bar { \epsilon } _ { 3 } ( x ^ { * } , \theta )$ is useful for large $d _ { c }$ , as there is no direct scaling with $d _ { c } ;$ however, it can be conservative if the data has high dispersion.
