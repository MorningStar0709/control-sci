# 3.2 Design of non-overshooting sliding mode system

Theorem 3.1 (Robust non-overshooting 2-sliding mode): The 2-sliding mode system is as follows:

$$\dot {e} _ {1} (t) = e _ {2} (t)
\dot {e} _ {2} (t) = \left\{ \begin{array}{l} - k _ {c} \operatorname{sign} \left[ e _ {2} (t) + e _ {2 c} \operatorname{sign} \left(e _ {1} (t)\right) \right] + d (t), \text {if} | e _ {1} (t) | > e _ {1 c}; \\ - k _ {2} \operatorname{sign} \left[ e _ {2} (t) + k _ {1} e _ {1} (t) \right] + d (t), \text {if} | e _ {1} (t) | \leq e _ {1 c} \end{array} \right. \tag {18}
$$

where, $e _ { 1 } \left( t \right)$ and $e _ { 2 } \left( t \right)$ are the sliding variables; the bounded unknown disturbance $d ( t )$ satisfies $\begin{array} { r } { \operatorname* { s u p } _ { t \in [ 0 , \infty ) } | d ( t ) | \leq L _ { d } < \infty ; k _ { c } > L _ { d } ; e _ { 1 c } \in ( 0 , k _ { 2 M } - L _ { d } ) , e _ { 2 c } \in ( e _ { 1 c } , \sqrt { ( k _ { 2 M } - L _ { d } ) e _ { 1 c } } |  } \end{array}$ , and $k _ { 2 M } > L _ { d }$ is the up-bound of $k _ { 2 }$ from the system gain limitation; $e _ { 1 } \left( \dot { t _ { c } } \right)$ and $e _ { 2 } \left( t _ { c } \right)$ are the initial values of $e _ { 1 } \left( t \right)$ and $e _ { 2 } \left( t \right)$ respectively when $| e _ { 1 } \left( t \right) | \le e _ { 1 c } ;$ and

$$
k _ {1} \in \left\{ \begin{array}{l} \left(0, \frac {\left| e _ {2} (t _ {c}) \right|}{\left| e _ {1} (t _ {c}) \right|}\right), \text {   if   } e _ {1} (t _ {c}) e _ {2} (t _ {c}) <   0 \text {   and   } | e _ {1} (t _ {c}) | <   | e _ {2} (t _ {c}) |; \\ \left(\frac {| e _ {2} (t _ {c}) |}{| e _ {1} (t _ {c}) |}, \infty\right), \text {   if   } e _ {1} (t _ {c}) e _ {2} (t _ {c}) <   0 \text {   and   } | e _ {1} (t _ {c}) | \geq | e _ {2} (t _ {c}) |; \\ (0, \infty), \text {   if   others } \end{array} \right. \tag {19}

k _ {2} > \left\{ \begin{array}{l} \max \left\{k _ {1} \left| e _ {2} \left(t _ {c}\right) \right| + L _ {d}, \frac {e _ {2} ^ {2} \left(t _ {c}\right)}{2 \left| e _ {1} \left(t _ {c}\right) \right|} + L _ {d} \right\}, \text {   if   } e _ {1} \left(t _ {c}\right) e _ {2} \left(t _ {c}\right) <   0 \text {   and   } \left| e _ {1} \left(t _ {c}\right) \right| <   \left| e _ {2} \left(t _ {c}\right) \right|; \\ \max \left\{k _ {1} \left| e _ {2} \left(t _ {c}\right) \right| + L _ {d}, \frac {k _ {1} ^ {2}}{3} \left(\left| e _ {1} \left(t _ {c}\right) \right| + \sqrt {e _ {1} ^ {2} \left(t _ {c}\right) + 3 \left(\frac {e _ {2} \left(t _ {c}\right)}{k _ {1}}\right) ^ {2}}\right) + L _ {d} \right\}, \text {   if   others } \end{array} \right. \tag {20}
$$
