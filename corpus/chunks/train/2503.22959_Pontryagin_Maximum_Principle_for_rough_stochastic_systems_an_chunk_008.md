# 2. Preliminaries on rough paths and rough stochastic integrals

We start recalling basic results on rough paths, rough stochastic integrals and related rough analysis results that will be used throughout. We refer to the textbook [14] for more details on rough analysis.

In what follows, $\alpha \in \left( \frac { 1 } { 3 } , \frac { 1 } { 2 } \right)$ is a H¨older coefficient, [0, T ] is a fixed time interval, and $( V , | \cdot | )$ is a Euclidean space. $\mathcal { L } ( V ^ { 1 } ; V ^ { 2 } )$ denotes the space of linear functions between the Euclidean spaces $V ^ { 1 }$ to $V ^ { 2 }$ .

All random variables and stochastic processes are defined on a common complete filtered probability space $( \Omega , \mathcal { F } , ( \mathcal { F } _ { t } ) _ { t } , \mathbb { P } )$ that supports a dW -dimensional Brownian motion $W$ . For any m < ∞ we denote by $L ^ { m }$ the space of all V -valued random variables ξ such that

$$\| \xi \| _ {m} := \mathbb {E} [ | \xi | ^ {m} ] ^ {\frac {1}{m}} < \infty .$$

The space of all a.s. finite V -valued random variables is denoted $L ^ { \infty }$ and

$$\| \xi \| _ {\infty} := \operatorname * {e s s s u p} _ {\omega} | \xi (\omega) |$$

For $2 \leq m \leq n$ we denote by $H ^ { m , n } ( [ s , t ] ; V )$ with $H ^ { m } ( [ s , t ] ; V ) : = H ^ { m , m } ( [ s , t ] ; V )$ and $S ^ { m } ( [ s , t ] ; V )$ , the spaces of adapted, V -valued progressively measurable processes X, equipped with the respective norms

$$\| X \| _ {H ^ {m, n}, [ s, t ]} := \left[ \mathbb {E} [ \int_ {s} ^ {t} | X _ {r} | ^ {n} d r ] ^ {\frac {m}{n}} \right] ^ {\frac {1}{m}}, \quad \| X \| _ {S ^ {m}, [ s, t ]} := \| \sup _ {r \in [ s, t ]} | X _ {r} | \| _ {m} < \infty .$$

For any function $f : \mathbb { R } ^ { d }  V$ that has continuous bounded derivatives up to order n, we write

$$| f | _ {n} := \max _ {i = 0, \dots , n} \sup _ {x \in \mathbb {R} ^ {d}} | D ^ {i} f (x) |.$$

For a one-parameter function/path $\eta :  0 , T   V$ we set $\delta \eta _ { s , t } : = \eta _ { t } - \eta _ { s }$ . The space of V -valued α-H¨older continuous functions is denoted $C ^ { \alpha } ( [ 0 , T ] ; V )$ . For any two-parameter function $A : \Delta _ { [ s , t ] } \to V$ where

$$\Delta_ {[ s, t ]} := \{(u, v) \in [ s, t ] ^ {2} \mid s \leq u < v \leq t \}$$

the difference operator $\delta A _ { s , u , t }$ is defined by

$$\delta A _ {s, u, t} := A _ {s, t} - A _ {s, u} - A _ {u, t}, \quad s \leq u \leq t,$$

and the α-H¨older norm is denoted

$$| A | _ {\alpha , [ s, t ]} := \sup _ {(u, v) \in \Delta_ {[ s, t ]}} \frac {| A _ {u , v} |}{| v - u | ^ {\alpha}}.$$
