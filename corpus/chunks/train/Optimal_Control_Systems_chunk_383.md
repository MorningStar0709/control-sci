# THEOREM 7.2

The necessary and sufficient conditions for the time-optimal control system to be singular is that the matrix $G_{j}, j = 1, 2, \ldots, r$ , is singular or that the system is uncontrollable.

Thus, for a singular interval to exist, it is necessary that the system is uncontrollable, conversely, if the system is completely controllable, a singular interval cannot exist.
