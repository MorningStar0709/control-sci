PROOF. According to (15) there are l minima to be computed which are the maximum of $\lvert \mathcal { C } _ { i } \rvert$ real numbers with $i = 1 : l .$ . Then, the value of $\boldsymbol { \mathcal U }$ can be computed by taking the minimum of the resulting l maxima. Below we prove that these operations can be achieved by a HTNN. From Lemma 3 one can compute the maximum of ¯m numbers given by ¯m affine functions with a HT network with $r ( \bar { m } )$ hidden neurons, $\zeta ( \bar { m } )$ layers and $w ( { \bar { m } } )$ maximum width. We construct the HT network for $\mathcal { M } ( x )$ as follows: we create l HTNNs and arrange them in parallel each one corresponding to a convex max affine function max $\mathbf { \Sigma } _ { j \in \mathcal { C } _ { i } } \mathcal { M } _ { j } ( x )$ for $i = 1 : l ,$ using Lemma 3. If we denote $\left| \mathcal { C } _ { i } \right| = \bar { m } _ { i } .$ then each individual network has $r ( \bar { m } _ { i } )$ hidden neurons, $\zeta ( \bar { m } _ { i } )$ layers and $w \big ( \bar { m } _ { i } \big )$ maximum width, for all $i = 1 : l .$ .

These l concatenated HTNNs have the same input x and outputs l real numbers. Finally, we create a HTNN with $r ( l )$ hidden neurons, $\zeta ( l )$ layers and $w ( l )$ maximum width, that takes the minimum of l real numbers using again Lemma 3 since min $\operatorname { \rho } _ { 1 } ( \cdot ) = - \operatorname* { m a x } ( - \cdot )$ . Combining Lemma 4 and 5, the number of layers of this extended HTNN is:

$$\zeta = \zeta (l) + \max _ {i = 1: l} \zeta (\bar {m} _ {i}) - 1.$$

The last term appears due to the fact that the first layer $\zeta ( l )$ of HTNN is absorbed into the last layer of the other one, since their dimensions are compatible. Further, the number of hidden neurons is

$$r = r (l) + \sum_ {i = 1} ^ {l} r (\bar {m} _ {i}) + 2 \left(\max _ {j = 1: l} \zeta (\bar {m} _ {j}) - \zeta (\bar {m} _ {i})\right), \tag {16}$$

$$w = \max \left(\sum_ {i = 1} ^ {l} \max (w (\bar {m} _ {i}), 2), w (l)\right).$$

$$\lceil \log_ {2} l \rceil + \lceil \log_ {2} m \rceil + 1.$$

Similarly we can derive an upper bound for the maximum width:

$$\max (l \max (w (m), 2), w (l)) \leq \max (l \max (m, 2), l) = l \max (m, 2).$$
