The following proposition states that the SVD-like decomposition of $P _ { \beta } A _ { f , g } | _ { d }$ can be computed using matrices in the matrix representation $[ P _ { \beta } A _ { f , g } ] _ { d } ^ { \beta }$ derived in the previous section.

Proposition 3: $\mathrm { I f } \ ( W , \Sigma , V )$ is the SVD of $G _ { \beta } ^ { + } I ^ { \top } G _ { d } ^ { + }$ with $W = \big \lceil w _ { 1 } , \quad \ldots , \quad w _ { M } \big \rceil , V = \big \lceil v _ { 1 } , \quad \ldots , \quad v _ { M } \big \rceil .$ , and $\Sigma =$ diag $\left( \left[ \overleftarrow { \boldsymbol { \sigma } } _ { 1 } , \quad \ldots , \quad \boldsymbol { \sigma } _ { M } \right] \right)$ , then for all $i = 1 , \ldots { \dot { , } } M , \sigma _ { i }$ are pseudo-singular values of $P _ { \beta } A _ { f , g } | _ { d }$ with left pseudo-singular functions $\bar { \phi } _ { i } : = v _ { i } ^ { \top } d$ and right pseudo-singular functions $\psi _ { i } : = w _ { i } ^ { \top } \beta .$ .

Proof: Let $\phi _ { i } = v _ { i } ^ { \top } d$ and $\psi _ { i } = w _ { i } ^ { \top } \beta$ and $h = \delta ^ { \top } d .$ Then,

$$
\begin{array}{l} P _ {\beta} A _ {f, g} h = \sum_ {i = 1} ^ {M} \sigma_ {i} \psi_ {i} \left<   h, \phi_ {i} \right> _ {\tilde {H} _ {d}} \\ \Longleftrightarrow P _ {\beta} A _ {f, g} \delta^ {\top} d = \sum_ {i = 1} ^ {M} \sigma_ {i} w _ {i} ^ {\top} \beta \left\langle \delta^ {\top} d, v _ {i} ^ {\top} d \right\rangle_ {\tilde {H} _ {d}} \\ \end{array}
$$
