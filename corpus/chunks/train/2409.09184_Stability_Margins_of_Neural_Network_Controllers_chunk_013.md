# C. Stability Condition

We form the closed-loop system of (5) and (3):

$$
\left[ \begin{array}{l} \dot {x} (t) \\ v (t) \end{array} \right] = \left[ \begin{array}{c c} A & B _ {w} \\ C _ {v} & D _ {v w} \end{array} \right] \left[ \begin{array}{l} x (t) \\ w (t) \end{array} \right] \tag {8}
w (t) = \Delta (v) (t),$$

where $\begin{array} { r l r l r l r } { x ( t ) } & { { } } & { = } & { { } } & { \left[ x _ { p } ( t ) ^ { \top } } & { x _ { k } ( t ) ^ { \top } \right] ^ { \top } , } & { v ( t ) } & { { } } & { = } \end{array}$ $\begin{array} { r l r } { \left[ v _ { p } ( t ) ^ { \top } \quad v _ { k } ( t ) ^ { \top } \right] ^ { \top } , w ( \bar { t } ) } & { = } & { \left[ w _ { \underline { { p } } } ( t ) ^ { \top } \quad w _ { k } ( t ) ^ { \top } \right] ^ { \top } } \end{array}$ , and $\begin{array} { r } { \Delta \textrm {  { \Sigma } } : \boldsymbol { v } \mapsto \left[ \Delta _ { p } ( v _ { p } ) ^ { \top } \right. \left. \phi ( v _ { k } ) ^ { \top } \right] ^ { 1 } } \end{array}$ . The expansions of $A , B _ { w } , C _ { v } , D _ { v w }$ matrices are:

$$
\begin{array}{l} A = \left[ \begin{array}{c c} A _ {p} + B _ {p} D _ {k u y} C _ {p} & B _ {p} C _ {k u} \\ B _ {k y} C _ {p} & A _ {k} \end{array} \right] \qquad B _ {w} = \left[ \begin{array}{c c} B _ {p} & B _ {p} D _ {k u w} \\ 0 & B _ {k w} \end{array} \right] \\ C _ {v} = \left[ \begin{array}{c c} D _ {k u y} C _ {p} & C _ {k u} \\ D _ {k v y} C _ {p} & C _ {k v} \end{array} \right] \qquad D _ {v w} = \left[ \begin{array}{c c} \frac {1 + \sigma}{2} I & D _ {k u w} \\ 0 & D _ {k v w} \end{array} \right]. \\ \end{array}
$$

Since $\Delta$ is formed by stacking $\Delta _ { p }$ and $\phi ,$ it satisfies an integral quadratic constraint formed by interleaving the IQCs satisfied by $\Delta _ { p }$ and $\Delta _ { k }$ : for all $\Lambda _ { p } \succeq 0$ and diagonal, $\Lambda _ { k } \succeq 0$ and diagonal, $v \in \mathcal { L } _ { 2 e } , w = \Delta ( v )$ , and $T \geq 0$ ,

$$
\int_ {0} ^ {T} \left[ \begin{array}{l} v (t) \\ w (t) \end{array} \right] ^ {\top} M \left[ \begin{array}{l} v (t) \\ w (t) \end{array} \right] \geq 0 \tag {9}
$$

where

$$
M \triangleq \left[ \begin{array}{c c} M _ {v v} & M _ {v w} \\ M _ {v w} ^ {\top} & M _ {w w} \end{array} \right] = \left[ \begin{array}{c c c c} \alpha^ {2} \Lambda_ {p} & 0 & 0 & 0 \\ 0 & 0 & 0 & \Lambda_ {k} \\ \hline 0 & 0 & - \Lambda_ {p} & 0 \\ 0 & \Lambda_ {k} & 0 & - 2 \Lambda_ {k} \end{array} \right].
$$

We now present the stability condition for the closed-loop system (8).
