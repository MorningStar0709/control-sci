# A. Preliminaries

Consider a decentralized multi-agent system, where N agents working in a shared k-dimension workspace and the dynamics of each agent $i \in \mathcal { T } = \{ 1 , \cdots , N \}$ in control affine form can be described as follows.

$$\dot {\mathbf {x}} _ {i} = f (\mathbf {x} _ {i}) + g (\mathbf {x} _ {i}) \mathbf {u} _ {i} \tag {1}$$

where $\mathbf { x } _ { i } \in \mathcal { X } _ { i } \subset \mathbb { R } ^ { d }$ and $\mathbf { u } _ { i } \in \mathcal { U } _ { i } \subset \mathbb { R } ^ { m }$ are the agent’s system state and control input, respectively. $f : \mathbb { R } ^ { d } \mapsto \mathbb { \mapsto }$ Rd and $g : \mathbb { R } ^ { d } \mapsto \mathbb { R } ^ { d \times m }$ are assumed to be locally Lipschitz continuous and full rank [14]. Given the control affine system in Eq. (1) for every agent, we can define a set of stabilizing controls and safety states using the following lemmas.

Lemma 1. [Summarized from [17]] A positive definite function $V ( \bar { \mathbf { x } _ { i } } ) : \mathbb { R } ^ { d } \mapsto \mathbb { R } _ { > 0 }$ is a control Lyapunov function (CLF), if there exists an extended class-K function $\gamma ( \cdot )$ such that in $\mathfrak { f } _ { \mathbf { u } _ { i } \in \mathbb { R } ^ { m } } \{ \dot { V } ( \mathbf { x } _ { i } , \mathbf { u } _ { i } ) \} \ \leq \ - \gamma ( V ( \mathbf { x } _ { i } ) )$ . The admissible control space for any Lipschitz continuous controller $\mathbf { u } _ { i } \in$ $\mathbb { R } ^ { m }$ enforcing system’s stability to $V ( \cdot ) = 0$ thus becomes:

$$K _ {\mathrm{clf}} (\mathbf {x} _ {i}) = \{\mathbf {u} _ {i} \in \mathbb {R} ^ {m} | L _ {f} V (\mathbf {x} _ {i}) + L _ {g} V (\mathbf {x} _ {i}) \mathbf {u} _ {i} \leq - \gamma (V (\mathbf {x} _ {i})) \} \tag {2}$$

where $\begin{array} { r c l } { L _ { f } V ( \mathbf { x } _ { i } ) } & { = } & { \nabla V ^ { T } ( \mathbf { x } _ { i } ) f ( \mathbf { x } _ { i } ) } \end{array}$ and $\begin{array} { r l } { L _ { g } V ( \mathbf { x } _ { i } ) } & { { } = } \end{array}$ $\nabla V ^ { T } ( \mathbf { x } _ { i } ) g \dot { ( } \mathbf { x } _ { i } )$ , respectively.

This definition allows us to devise a set of controls to ensure the closed-loop control system’s stability prescribed by $V ( \cdot )$ for each agent in the multi-agent system.
