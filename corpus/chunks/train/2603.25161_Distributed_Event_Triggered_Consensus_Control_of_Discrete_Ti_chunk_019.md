Since $( ( 1 - a ) \eta - a ) ^ { 2 } > 0$ for η satisfying (47), the sign of $d \beta _ { \sigma , \varepsilon } ( \eta ) / d \eta$ is determined by the sign of $h ( \eta )$ . By solving $h ( \eta ) = 0$ , we obtain

$$\eta = \frac {2 a \pm \sqrt {4 a ^ {2} + 4 a (1 - a)}}{2 (1 - a)} = \frac {a \pm \sqrt {a}}{1 - a}.$$

Hence, the two roots are given by

$$\eta_ {-} = \frac {a - \sqrt {a}}{1 - a} = - \frac {\sqrt {a}}{1 + \sqrt {a}} < 0,\eta_ {+} = \frac {a + \sqrt {a}}{1 - a} = \frac {\sqrt {a}}{1 - \sqrt {a}}.$$

Therefore, η− does not satisfy (47). Moreover,

$$\eta_ {+} - \frac {a}{1 - a} = \frac {\sqrt {a}}{1 - \sqrt {a}} - \frac {a}{1 - a} = \frac {\sqrt {a}}{1 - a} > 0,$$

which means that $\eta _ { + }$ satisfies (47).

Since $1 - a > 0$ , the quadratic function $h ( \eta )$ is convex, and since its roots are $\eta _ { - } < 0$ and $\begin{array} { r } { \eta _ { + } > \frac { a } { 1 - a } } \end{array}$ , it follows that

$$
\left\{ \begin{array}{l l} h (\eta) <   0, & \frac {a}{1 - a} <   \eta <   \eta_ {+}, \\ h (\eta) > 0, & \eta > \eta_ {+}. \end{array} \right.
$$

Hence, the sign of $d \beta _ { \sigma , \varepsilon } ( \eta ) / d \eta$ is

$$
\left\{ \begin{array}{l l} \frac {d}{d \eta} \beta_ {\sigma , \varepsilon} (\eta) <   0, & \frac {a}{1 - a} <   \eta <   \eta_ {+}, \\ \frac {d}{d \eta} \beta_ {\sigma , \varepsilon} (\eta) > 0, & \eta > \eta_ {+}. \end{array} \right.
$$

Therefore, $\beta _ { \sigma , \varepsilon } ( \eta )$ is strictly decreasing on $( a / ( 1 - a ) , \eta _ { + } )$ and strictly increasing on $( \eta _ { + } , \infty )$ . Consequently,

$$\eta = \frac {\sqrt {\sigma \alpha_ {S} (\varepsilon)}}{1 - \sqrt {\sigma \alpha_ {S} (\varepsilon)}}$$

is the unique minimizer of $\beta _ { \sigma , \varepsilon } ( \eta )$ over (47).

Finally, we derive the corresponding minimum value of $\beta .$ Since

$$1 + \eta^ {\star} (\sigma ; \varepsilon) = 1 + \frac {\sqrt {a}}{1 - \sqrt {a}} = \frac {1}{1 - \sqrt {a}},$$

and

$$
\begin{array}{l} (1 - a) \eta^ {\star} (\sigma ; \varepsilon) - a = (1 - a) \frac {\sqrt {a}}{1 - \sqrt {a}} - a \\ = (1 + \sqrt {a}) \sqrt {a} - a = \sqrt {a}, \\ \end{array}
$$

it follows that

$$\beta_ {\min} (\sigma ; \varepsilon) = \beta_ {\sigma , \varepsilon} \left(\eta^ {\star} (\sigma ; \varepsilon)\right) = \frac {\sigma \eta^ {\star} (\sigma ; \varepsilon) \left(1 + \eta^ {\star} (\sigma ; \varepsilon)\right)}{(1 - a) \eta^ {\star} (\sigma ; \varepsilon) - a}= \frac {\sigma}{(1 - \sqrt {a}) ^ {2}} = \frac {\sigma}{\left(1 - \sqrt {\sigma \alpha_ {S} (\varepsilon)}\right) ^ {2}}.$$

This completes the proof.
