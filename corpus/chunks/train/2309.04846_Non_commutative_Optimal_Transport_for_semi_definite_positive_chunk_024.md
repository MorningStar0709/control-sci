Notice that if Γ is not semi-definite positive, then there exists $\lambda _ { k } \in \operatorname { S p } ( \Gamma )$ that is negative. Consider the diagonal decomposition $\begin{array} { r } { \Gamma = \sum _ { i } \lambda _ { i } \vert \psi _ { i } \rangle \langle \psi _ { i } \vert } \end{array}$ and take $Z _ { n } = n | \psi _ { k } \rangle \langle \psi _ { k } |$ , then

$$\mathrm{Tr} [ - Z _ {n} \Gamma ] = - n \lambda_ {k} \underset {n \to \infty} {\longrightarrow} + \infty ,$$

on the other hand, using the (strict) convexity of the trace function $A \mapsto \operatorname { T r } [ \exp ( A ) ]$ (see e.g., Theorem 2.10 in [6]), we get

$$- \varepsilon \operatorname{Tr} [ e ^ {\frac {- Z _ {n} - C}{\varepsilon}} ] \geq - \frac {\mathcal {E}}{2} \operatorname{Tr} [ e ^ {- \frac {2 Z _ {n}}{\varepsilon}} ] - \frac {\mathcal {E}}{2} \operatorname{Tr} [ e ^ {- \frac {2 C}{\varepsilon}} ] = - \frac {\mathcal {E}}{2} e ^ {- \frac {2 n}{\varepsilon}} - \frac {\mathcal {E}}{2} \operatorname{Tr} [ e ^ {- \frac {2 C}{\varepsilon}} ] \xrightarrow [ n \to \infty ]{} - \frac {\mathcal {E}}{2} \operatorname{Tr} [ e ^ {- \frac {2 C}{\varepsilon}} ],$$

which is a finite value, and whence $\Theta ^ { * } ( - \Gamma ) = + \infty$ .

For Γ semi-definite positive, invoking the variational principle, and using strict convexity of

Θ, we can obtain the unique maximizer $\tilde { Z } = - C - \varepsilon \log \Gamma$ , which yields

$$
\begin{array}{l} \Theta^ {*} (- \Gamma) = \sup _ {Z \in \mathrm{H} (\mathcal {H} _ {1} \otimes \mathcal {H} _ {2})} \left\{\mathrm{Tr} [ - Z \Gamma - \varepsilon \exp (\frac {- Z - C}{\varepsilon}) ] \right\} = \mathrm{Tr} [ - \tilde {Z} \Gamma - \varepsilon \exp (\frac {- \tilde {Z} - C}{\varepsilon}) ] = \\ = \operatorname{Tr} [ - (- C - \varepsilon \log \Gamma) \Gamma - \varepsilon \Gamma ] = \operatorname{Tr} [ C \Gamma ] + \varepsilon \operatorname{Tr} [ \Gamma (\log \Gamma - \mathbb {I} _ {d}) ] = \operatorname{Tr} [ C \Gamma ] + \varepsilon \mathcal {S} [ \Gamma ], \tag {3.10} \\ \end{array}
$$

where we use the fact the trace of a product of two operators commutes.
