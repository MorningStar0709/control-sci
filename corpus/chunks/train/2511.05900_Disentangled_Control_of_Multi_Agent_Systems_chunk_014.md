Intuitively, the SE-NS-TV-CLFs arise in scenarios where there exist groupwise shared objectives. For this type of CLFs, inspired by the idea of [41], we propose the following theorem to achieve the decentralized realization of Theorem III.1 in multi-agent systems.

Theorem IV.1. (Constraint Allocation) For any $\mathcal { N } ^ { \mathrm { S E } }$ per Definition IV.1, if each agent is executing the time-varying state-feedback controller $u _ { i } ( \{ x _ { i } \} _ { i \in \mathcal { N } ^ { \mathrm { S E } } } , t )$ such that

$$
\begin{array}{l} \sup _ {\zeta_ {i} \in \partial_ {x _ {i}} ^ {\circ} V _ {\mathcal {N} ^ {\mathrm{SE}}}} \zeta_ {i}   \dot {x} _ {i} + w _ {\mathcal {N} ^ {\mathrm{SE}}} ^ {i} \bigg (\frac {\partial}{\partial t} V _ {\mathcal {N} ^ {\mathrm{SE}}} (\{x _ {i} \} _ {i \in \mathcal {N} ^ {\mathrm{SE}}}, t) \\ \left. \right. + \alpha \left(V _ {\mathcal {N} ^ {\mathrm{SE}}} \left(\left\{x _ {i} \right\} _ {i \in \mathcal {N} ^ {\mathrm{SE}}}, t)\right)\right) \leq 0, \quad \forall t \geq 0, \tag {13} \\ \end{array}
$$

$\forall i \in \mathcal { N } ^ { \mathrm { S E } }$ , where

$$\sum_ {i \in \mathcal {N} ^ {\mathrm{SE}}} w _ {\mathcal {N} ^ {\mathrm{SE}}} ^ {i} = 1, \quad w _ {\mathcal {N} ^ {\mathrm{SE}}} ^ {i} \in \mathbb {R}, \tag {14}$$

and α is a locally Lipschitz class K function, then $V _ { \mathcal { N } ^ { \mathrm { S E } } } ( \{ x _ { i } \} _ { i \in \mathcal { N } ^ { \mathrm { S E } } } , t )$ asymptotically converges to 0.

Proof. Summing (13) over $i \in \mathcal { N } ^ { \mathrm { S E } }$ yields

$$
\begin{array}{l} \sum_ {i \in \mathcal {N} ^ {\mathrm{SE}}} \sup _ {\zeta_ {i} \in \partial_ {x _ {i}} ^ {\circ} V _ {\mathcal {N} ^ {\mathrm{SE}}}} \zeta_ {i}   \dot {x} _ {i} + \frac {\partial}{\partial t} V _ {\mathcal {N} ^ {\mathrm{SE}}} (\{x _ {i} \} _ {i \in \mathcal {N} ^ {\mathrm{SE}}}, t) \\ + \alpha (V _ {\mathcal {N} ^ {\mathrm{SE}}} (\{x _ {i} \} _ {i \in \mathcal {N} ^ {\mathrm{SE}}}, t)) \leq 0, \quad \forall t \geq 0. \\ \end{array}
$$
