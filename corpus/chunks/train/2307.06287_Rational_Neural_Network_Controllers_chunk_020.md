# 3.2 Rational Approximation of Sigmoid Activation Function

It is also possible to approximate the sigmoid function as a rational function. We define the function

$$\operatorname{Rsig} (x) = \phi (x) = \frac {(x + 4) ^ {2}}{2 (x ^ {2} + 1 6)}.$$

As shown in Figure 7a, this is a good approximation to the sigmoid function. Figure 7b shows the error with the sigmoid function. Rsig can be expressed as the equality constraint

$$2 (x ^ {2} + 1 6) \phi (x) - (x + 4) ^ {2} = 0.$$
