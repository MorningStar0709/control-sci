$$N _ {i} (A, \omega) = \overline {{{{N _ {1}}}}} (A, \omega , 0) + \frac {A}{2} \frac {\mathrm{d}}{\mathrm{d} A} \left\{\overline {{{{N _ {1}}}}} (A, \omega , 0) \right\}. \tag {16}$$

The justification for using the IIDF (16) lies in its ability to approximate the results of the integrals in (15) near of the origin $\sigma _ { 0 } \equiv 0 { \mathrm { . } }$ , as follows: $\widetilde { N _ { 0 } } \approx N _ { i } \widetilde { ( A , \omega ) }$ and $\widetilde { N _ { 1 } } \approx \overline { { N _ { 1 } } } ( A , \omega )$ , respectively. Then, the fast and slow components at the output of the nonlinearity can be predicted by

$$u _ {0} ^ {*} (t) = \overline {{{N _ {i}}}} \sigma_ {0} \cos (\Omega t), \tag {17}u _ {1} ^ {*} (t) = \overline {{{N _ {1}}}} A \sin (\omega t), \tag {18}$$

whenever the conditions $\Omega \ll \omega$ and $| \sigma _ { 0 } | \ll A$ are satisfied.

Consider the relay-type controller operating under a twosinusoid input signal as defined in (14). The corresponding SIDF is given in (12), then, a local approximation around $\sigma _ { 0 } =$ 0 yields

$$\overline {{N _ {1}}} (A) = \frac {4 \rho}{\pi A} \left. \sqrt {1 - \left(\frac {\sigma_ {0}}{A}\right) ^ {2}} \right| _ {\sigma_ {0} = 0} = \frac {4 \rho}{\pi A}. \tag {19}$$

Therefore, the IIDF (16) of the relay takes the form

$$N _ {i} (A) = \frac {4 \rho}{\pi A} + \frac {A}{2} \frac {\mathrm{d}}{\mathrm{d} A} \left\{\frac {4 \rho}{\pi A} \right\} = \frac {2 \rho}{\pi A}. \tag {20}$$
