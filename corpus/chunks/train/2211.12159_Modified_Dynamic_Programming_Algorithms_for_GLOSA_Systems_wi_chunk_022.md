For the solution of the resulting Quadratic Programming (QP) problem, Fletcher’s active set algorithm is used (Fletcher, 1971). The critical issue here is that the minimization in the recursive Bellman equation should be executed in a parametrized way, i.e., deliver 𝛿𝒂 as a function of 𝛿𝒙, which is not compatible with a numerical solution procedure. $$\min Q (\bar {x}, \bar {a} + \delta a)$$
