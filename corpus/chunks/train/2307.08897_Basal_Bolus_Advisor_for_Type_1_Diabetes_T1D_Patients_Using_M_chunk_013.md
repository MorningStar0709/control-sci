Algorithm 1 Basal Agent Reward Function   
1: Initialize parameters $BG^{buffer}$ , $BG^{bounds}$ 2: Inputs:
3: $BG^{buffer}$ : A buffer of CGM readings for the past 7 hours.
4: $BG^{bounds}$ : Bounds of the BG readings, i.e. (105, 115) [mg/dL]
5: Output:
6: $r^{episode}$ : Calculated reward for each episode
7: $nnz \leftarrow$ number of nonzero elementss
8: $n \leftarrow$ number of samples in the $BG^{buffer}$ array
9: for each episode do
10:    for each step, $t = 1, \ldots, N$ do
11: $r_{t}^{(105,115)} \leftarrow \frac{10}{n} \times nnz(BG^{buffer} < \max(BG^{bounds}) \land BG^{buffer} > \min(BG^{bounds}))$ 12: $r_{t}^{(100,120)} \leftarrow \frac{10}{n} \times nnz(BG^{buffer} < (\max(BG^{bounds}) + 5) \land BG^{buffer} > (\min(BG^{bounds}) - 5))$ 13: $r_{t}^{(70,180)} \leftarrow \frac{10}{n} \times nnz(BG^{buffer} < (\max(BG^{bounds}) + 65) \land BG^{buffer} > (\min(BG^{bounds}) - 35))$ 14:    end for
15: $r^{episode} \leftarrow \sum_{t=1}^{N} \exp\left(\frac{r_{t}^{(105,115)}}{2}\right) + \exp\left(\frac{r_{t}^{(100,120)}}{2}\right) + \exp\left(\frac{r_{t}^{(70,180)}}{2}\right)$ 16: end for

The bolus agent considers several factors, including the patient’s previous BG value, meal information generated by the simulator, and a history of actions taken in the last three hours. Its primary objective is to maintain the BG level within the euglycemia range of (70, 180) mg/dL.

The underlying principle for the bolus agent is to receive a positive reward when it takes action immediately after mealtimes and successfully keeps the BG level within the euglycemia range. On the other hand, if the bolus agent takes action at any other time or fails to take action after meal intake, it receives a negative reward as a penalty. These rewards are computed every 15 minutes and accumulated at the end of each episode, which corresponds to a one-week duration.

The total reward for the bolus agent per episode is calculated using the equation:

$$r _ {B o l u s} ^ {\text { episode }} = \sum_ {i = 1} ^ {M} r ^ {B G} (i) + r ^ {\text { Action }} (i), \tag {5}$$
