# 4.4.2. Predictor Performance Measures

If we compute the average of multiple one-time prediction performance measures from one predictor’s result, we obtain a predictor performance measure, whose expectation is

$$
\begin{array}{l} \mathcal {E} _ {m} (\mathcal {P}) = \mathbb {E} m [ \bar {w} (S), \mathcal {P} (o (S)) ] \\ = \mathbb {E} \left[ \mathbb {E} \left[ m [ \bar {w} (S), \mathcal {P} (o (S)) ] \mid o (S) \right] \right]. \tag {27} \\ \end{array}
$$

Based on our definition of the observable true distribution $\bar { W } _ { t o } ( o )$ , for a given, fixed observation $^ { O , }$ the expectation of the one-time prediction performance measure of a belief $\bar { W } _ { b }$ can be computed using $\bar { W } _ { t o } ( o )$ ,

$$\mathbb {E} \left[ m [ \bar {w} (S), \bar {W} _ {b} ] \Big | o (S) = o \right] = \underset {\bar {w} \sim \bar {W} _ {t o} (o)} {\hat {\mathbb {E}}} m (\bar {w}, \bar {W} _ {b}). \tag {28}$$

The right-hand side of the (28) can be computed just using two probability distributions $\bar { W } _ { t o } ( o )$ and $\bar { W } _ { b }$ . Therefore, we define

$$E _ {m} (\bar {W} _ {t o} (o), \bar {W} _ {b}) = \underset {\bar {w} \sim \bar {W} _ {t o} (o)} {\hat {\mathbb {E}}} m (\bar {w}, \bar {W} _ {b}). \tag {29}$$

According to (27), (28) and (29), given a one-time prediction performance measure m, the expected predictor performance measure can also be written as

$$\mathcal {E} _ {m} (\mathcal {P}) = \mathbb {E} E _ {m} [ \bar {W} _ {t o} (o (S)), \mathcal {P} (o (S)) ]. \tag {30}$$
