The next lemma shows that HTNNs yield min-max functions.

Lemma 2. Any neural network $f ( x ; { \hat { \Theta } } )$ defined as in (11) with input $x \in \mathbb { R } ^ { n _ { x } }$ and HardTanh activation functions represents a min-max affine function.

PROOF. The HTNN $f ( x ; { \hat { \Theta } } )$ is a min-max affine function because it only contains compositions of affine transformations with min-max affine activation functions (HardTanhs) and minmax functions are closed under addition, multiplication and composition of such functions. □
