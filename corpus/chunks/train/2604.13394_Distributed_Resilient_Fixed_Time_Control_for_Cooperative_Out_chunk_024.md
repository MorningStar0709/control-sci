Lemma 2: [44, 45] Consider the linear system

$$\dot {x} (t) = A x (t) + B u (t), \tag {23}$$

where $\ b { x } ( t ) \in \mathbb { R } ^ { n }$ and $u ( t ) \in \mathbb { R } ^ { m }$ are its state and input, respectively, (A, B) is controllable and B is of full column rank.

Moreover, det $( X ) \neq 0$ , and $R _ { i } A ^ { l } B \stackrel { = } { = } 0 ,$ for $l ~ = ~ 0 , 1 , \cdots , q _ { j } ~ - ~ 2 , ~ j ~ = ~ 1 , 2 , \cdots , m _ { \ i }$ , where $\boldsymbol { X } \ =$

$$
\left[ \begin{array}{c} R _ {1} A ^ {q _ {1} - 1} B \\ \vdots \\ R _ {m} A ^ {q _ {m} - 1} B \end{array} \right] \text {   and   } R _ {j} \text {   is   the   row   indexed   by   } j \text {   in   } R.
$$

(ii) There are nonsingular transformations $\bar { \boldsymbol { u } } ( t ) = G \boldsymbol { u } ( t )$ and $\bar { \boldsymbol { x } } ( t ) = T \boldsymbol { x } ( t )$ such that system (23) is transformed into

$$\dot {\bar {x}} (t) = \bar {A} \bar {x} (t) + \bar {B} \bar {u} (t), \tag {24}$$

where $\bar { A } \ = \ [ \bar { A } _ { l j } ] , \ \bar { B } \ = \ b l o c { k } { d i a g } \{ \Theta _ { q _ { 1 } } , \Theta _ { q _ { 2 } } , \cdot \cdot \cdot \ , \Theta _ { q _ { m } } \} ,$ ,

$$
