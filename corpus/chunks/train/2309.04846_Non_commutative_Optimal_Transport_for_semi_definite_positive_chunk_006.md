# 2.1 Shannon-Entropy regularized Unbalanced Optimal Transport

Let $\varepsilon , \tau _ { 1 } , \tau _ { 2 } > 0$ be positive numbers, X and Y be complete separable metric spaces. The Shannon entropy regularization of the Unbalanced Optimal Transport between positive measures $\mu \in { \mathcal { M } } _ { + } ( X )$ and $\nu \in \mathbb { M } _ { + } ( Y )$ is given by

$$\mathcal {O T} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\mu , \nu) = \inf _ {\gamma \in \mathcal {M} _ {+} (X \times Y)} \int_ {X \times Y} c d \gamma + \varepsilon S (\gamma) + \tau_ {1} \mathrm{KL} ((e _ {1}) _ {\sharp} \gamma | \mu) + \tau_ {2} \mathrm{KL} ((e _ {2}) _ {\sharp} \gamma | \nu), \tag {2.4}$$

where $c : X \times Y  \mathbb { R }$ is a measurable cost function and KL is the Kullback-Leibler divergence (2.2) between two positive measures α and $\beta$ . The functional $S : \mathbb { M } _ { + } ( X \times Y ) \to \mathbb { R }$ is the Shannon-entropy

$$
S (\gamma) = \left\{ \begin{array}{l l} \int_ {X \times Y} \frac {d \gamma}{d (\mu \otimes \nu)} \left(\log \left(\frac {d \gamma}{d (\mu \otimes \nu)}\right) - 1\right) \mu \otimes \nu & \text { if } \gamma \ll \mu \otimes \nu \\ + \infty & \text { otherwise } \end{array} , \right. \tag {2.5}
$$

where $\frac { d \gamma } { d ( \mu \otimes \nu ) }$ denotes the Radon-Nikodym derivative of $\gamma$ with respect to $\mu \otimes \nu .$ .

The existence of a minimizer in (2.4) is guaranteed, for instance, when the cost c is nonnegative and lower semicontinuous [7].
