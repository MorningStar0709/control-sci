# 2.1 Representation Formulas for Unconstrained Optimal Control

Before stating our main observation, let us recall the Lagrangian framework [20]. It is known that for a Lagrangian $\textstyle { \mathcal { L } } ( t , \pmb { x } , b )$ which is convex in b, the value function

$$\phi (x, t) := \inf \left\{\int_ {0} ^ {t} \mathcal {L} (s, \xi (s), \dot {\xi} (s)) \mathrm{d} s + g (\xi (0)): \xi (t) = \boldsymbol {x}, \dot {\xi} (s) \in L ^ {1} ([ 0, t ]) \right\} \tag {2.5}$$

is the unique viscosity solution of the following HJ PDE:

$$
\left\{ \begin{array}{l l} \phi_ {t} + H (t, \boldsymbol {x}, D \phi) = 0 & \quad \text {in} \quad \mathbb {R} ^ {n} \times (0, \infty), \\ \phi (\boldsymbol {x}, 0) = g (\boldsymbol {x}) & \quad \text {on} \quad \mathbb {R} ^ {n}, \end{array} \right.
$$

where $H ( t , \mathbf { x } , p ) : = \operatorname* { s u p } _ { b } \{ p \cdot b - \mathcal { L } ( t , \mathbf { x } , b ) \}$ .

Now regarding $H ^ { * }$ as the Lagrangian, we obtain the following representation of the value function (2.2).

Theorem 1. For $( \pmb { x } , t ) \in \mathbb { R } ^ { n } \times [ 0 , T ]$ , the value function (2.2) can be represented as

$$
\begin{array}{l} V (\boldsymbol {x}, t) = \inf \left\{\int_ {t} ^ {T} H ^ {*} (s, x (s), - \dot {x} (s)) \mathrm{d} s + g (x (T)): x (t) = \boldsymbol {x} \right\} (2.6) \\ = \inf \left\{\int_ {t} ^ {T} H ^ {*} (s, x (s), \beta (s)) \mathrm{d} s + g (x (T)): x (t) = \boldsymbol {x}, \beta (s) \in \operatorname{Conv} (B (x (s), s)) \right\}. (2.7) \\ \end{array}
$$

Proof. Let us define

$$W (\boldsymbol {x}, t) := \inf \left\{\int_ {t} ^ {T} H ^ {*} (s, x (s), - \dot {x} (s)) d s + g (x (T)): x (t) = \boldsymbol {x} \right\}.$$

It follows form the Lagrangian formula (2.5) that W solves the following HJ equation in the viscosity sense:

$$
\left\{ \begin{array}{l l} - W _ {t} + H ^ {* *} (t, \pmb {x}, D W) = 0 & \quad \text {in} \quad \mathbb {R} ^ {n} \times (0, T), \\ W (\pmb {x}, T) = g (\pmb {x}) & \quad \text {on} \quad \mathbb {R} ^ {n}, \end{array} \right.
$$

where $\begin{array} { r } { H ^ { * * } ( t , \pmb { x } , p ) : = \operatorname* { s u p } _ { b \in \mathbb { R } ^ { n } } \{ b \cdot p - H ^ { * } ( t , \pmb { x } , b ) \} } \end{array}$ is the convex conjugate of $H ^ { * } ( t , { \boldsymbol { x } } , \cdot )$

By the definition of $H ^ { * }$ , we first notice that for any $p , b \in \mathbb { R } ^ { n }$

$$H ^ {*} (t, \pmb {x}, b) \geq p \cdot b - H (t, \pmb {x}, p),$$

which implies that

$$H (t, \boldsymbol {x}, p) \geq p \cdot b - H ^ {*} (t, \boldsymbol {x}, b).$$
