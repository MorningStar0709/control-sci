# 4.2 Observation model for the estimation problem

Herein, first we show that the estimation error and its differential can be characterized as a “simple norm” of the concatenated noise (Proposition 3). This norm is induced by a metric that encapsulates the system dynamics which is explained below. Before proceeding to the results of this section, we assume that both the process and measurement noise are bounded:

Assumption 4. Assume that (almost surely) $\| x _ { 0 } \| , \| \xi ( t ) \| \le \kappa _ { \xi }$ and $\| \omega ( t ) \| \le \kappa _ { \omega }$ for all t. Also, for simplicity, suppose the initial state has zero mean, $i . e . , m _ { 0 } = 0 _ { n }$ .

For two vectors ${ \vec { v } } ,$ ⃗w $\in \mathbb { R } ^ { ( T + 1 ) n }$ , we define

$$\langle \vec {v}, \vec {w} \rangle_ {\mathcal {A} _ {L}} := \mathrm{tr} \left[ \vec {v} \vec {w} ^ {\intercal} \mathcal {A} _ {L} ^ {\intercal} H ^ {\intercal} H \mathcal {A} _ {L} \right],$$

where $\begin{array} { r } { \small A _ { L } : = \left( A _ { L } ^ { 0 } \quad A _ { L } ^ { 1 } \quad \ldots \quad A _ { L } ^ { T } \right) } \end{array}$ . Also, define $\begin{array} { r } { \mathcal { M } _ { L } [ E ] : = ( M _ { 0 } [ E ] \quad M _ { 1 } [ E ] \quad \cdots \quad M _ { T } [ E ] ) } \end{array}$ with $M _ { 0 } [ E ] = 0 , M _ { 1 } [ E ] = E H$ and $\begin{array} { r } { M _ { i + 1 } [ E ] = \sum _ { k = 0 } ^ { i } A _ { L } ^ { i - k } E H A _ { L } ^ { k } } \end{array}$ for $i = 1 , \cdots , T - 1$ .
