In general with a nonlinear nominal model $f ( \cdot )$ , we can linearize (31) around a point $( \bar { x } _ { k } , \bar { u } _ { k } )$ as:

$$
\begin{array}{l} x _ {k + 1} \approx f (\bar {x} _ {k}, \bar {u} _ {k}) + \frac {\partial f (x , u)}{\partial x} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} (x _ {k} - \bar {x} _ {k}) + \frac {\partial f (x , u)}{\partial u} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} (u _ {k} - \bar {u} _ {k}) \\ + \mu^ {d} (\bar {x} _ {k}, \bar {u} _ {k}) + \frac {\partial \mu^ {d} (x , u)}{\partial x} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} (x _ {k} - \bar {x} _ {k}) + \frac {\partial \mu^ {d} (x , u)}{\partial u} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} (u _ {k} - \bar {u} _ {k}). \tag {32} \\ \end{array}
$$

By defining $\delta x _ { k } = x _ { k } - \bar { x } _ { k }$ and $\delta u _ { k } = u _ { k } - \bar { u } _ { k }$ , we obtain:

$$
\begin{array}{l} \delta x _ {k + 1} + \bar {x} _ {k + 1} \approx f (\bar {x} _ {k}, \bar {u} _ {k}) + \frac {\partial f (x , u)}{\partial x} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta x _ {k} + \frac {\partial f (x , u)}{\partial u} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta u _ {k} \\ + \mu^ {d} (\bar {x} _ {k}, \bar {u} _ {k}) + \frac {\partial \mu^ {d} (x , u)}{\partial x} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta x _ {k} + \frac {\partial \mu^ {d} (x , u)}{\partial u} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta u _ {k}. \tag {33} \\ \end{array}
$$

If the linearization is conducted at the mean points for each time step, i.e. $\bar { x } _ { k + 1 } = \mu _ { k + 1 } ^ { x } , \bar { x } _ { k } = \mu _ { k } ^ { x }$ and $\bar { u } _ { k } = \mu _ { k } ^ { u }$ , and according to (25a), we have ${ \bar { x } } _ { k + 1 } \approx f \left( { \bar { x } } _ { k } , { \bar { u } } _ { k } \right) + \mu ^ { d } \left( { \bar { x } } _ { k } , { \bar { u } } _ { k } \right)$ . The linearized prediction model (32) becomes:

$$\delta x _ {k + 1} \approx \frac {\partial f (x , u)}{\partial x} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta x _ {k} + \frac {\partial f (x , u)}{\partial u} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta u _ {k} + \frac {\partial \mu^ {d} (x , u)}{\partial x} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta x _ {k} + \frac {\partial \mu^ {d} (x , u)}{\partial u} \Bigg | _ {\bar {x} _ {k}, \bar {u} _ {k}} \delta u _ {k}. \tag {34}$$
