A trajectory (t) may be related to an input-output representaωtion via a permutation matrix $\Pi \in \mathbb { R } ^ { q \times q }$ with $q = n _ { u } + n _ { y }$ and $\omega ( t ) = \Pi { \binom { u ( t ) } { y ( t ) } } .$ The behavioral framework describes a system as the span of all possible input-output trajectories generated by its dynamics. The complexity of a system is defined by its structure indices (Willems, 1986a): the number of inputs $n _ { u } ,$ , the order $n _ { x } ,$ , and the lag  (the smallest integer  for which the matrix $\left\lceil C \quad C A \quad \cdots \quad C A ^ { \ell - 1 } \right\rceil$ has rank $n _ { x } )$ .

The system models described so far can also be formulated in discrete time k ∈ N with input $u _ { k } \ \in \ \mathbb { R } ^ { n _ { u } }$ , state $x _ { k } \ \in \ \mathbb { R } ^ { n _ { x } }$ , and output $y _ { k } \in \mathbb R ^ { n _ { \mathrm { y } } }$ . Discrete-time versions of the nonlinear model in (3), control-affine model in (4), and LTI model in (5) are defined using difference equations with equivalent notation. Specifically, a discrete-time nonlinear system can be written as:

$$x _ {k + 1} = f (x _ {k}, u _ {k}), \tag {11}y _ {k} = h (x _ {k}, u _ {k}),$$

and a discrete-time LTI model can be formulated as:

$$x _ {k + 1} = A x _ {k} + B u _ {k}, \tag {12}y _ {k} = C x _ {k} + D u _ {k},$$

with given initial conditions x0. Instead of using the Laplace transform, discrete-time LTI systems are converted to the complex domain using the Z transform and are associated with a discrete-time transfer function $G ( z )$ such that $Y ( z ) = G ( z ) U ( z )$ , where U(z) and Y(z) are the Z transforms of the input $u _ { k }$ and output yk.
