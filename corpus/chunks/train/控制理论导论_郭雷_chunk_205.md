# 习题 3.2

3.2.1 证明以下各个集合均为实数环 $\mathbb{R}$ 的子环：

(a) $\{a + \sqrt{3} b \mid a, b \in \mathbb{Z}\}$ ;   
(b) $\{a + 2^{1/3}b + 2^{2/3}c \mid a, b, c \in \mathbb{Z}\}$ ;

(c) $\{a2^{b} \mid a, b \in \mathbb{Z}\}$ .

3.2.2 $M_{n}$ 为 $n \times n$ 矩阵集合在普通矩阵加法和乘法下所构成的环， $A \subset M_{n}$ ，当 $A$ 满足以下条件时，它是否为 $M_{n}$ 的一个子环？

(a) A 为上三角阵;  
(b) $\det(A)=1;$   
(c) A 为对角阵;  
(d) $A^{T}A = I.$

3.2.3 设 $V$ 为一Lie代数. 证明Jacobi等式也可以表示为

$$[ [ X, Y ], Z ] + [ [ Y, Z ], X ] + [ [ Z, X ], Y ] = 0, \quad \forall X, Y, Z \in V.$$

3.2.4 证明 3.2.4 中的三个集合均为 Lie 代数.

3.2.5 考虑一个双线性控制系统

$$\dot {x} = f (x) + g (x) u = A x + u B x.$$

这里

$$
\boldsymbol {A} = \left[ \begin{array}{l l} 0 & 1 \\ 0 & 0 \end{array} \right], \quad \boldsymbol {B} = \left[ \begin{array}{l l} 0 & - 1 \\ 1 & 0 \end{array} \right].
$$

求由 $f(x), g(x)$ 生成的 Lie 代数.

3.2.6 给定一个环 $R$ , 设 $f: R \to R$ 为一环同态, 且 $S = \{x \in R \mid f(x) = x\}$ , 则 $S$ 为 $R$ 的一个子环.

3.2.7 一个环 $R$ 的中心定义为

$$C (R) = \{c \in R \mid r c = c r, \forall r \in R \}.$$

证明 $C(R)$ 是 $R$ 的一个子环.

3.2.8 设 $I$ 和 $J$ 为 Lie 代数 $V$ 的两个理想. 证明

$$I + J \stackrel {\text { def }} {=} \{X + Y \mid X \in I, Y \in J \}$$

和

$$I J \stackrel {\text { def }} {=} \{[ X, Y ] \mid X \in I, Y \in J \}$$

均为 V 的理想.

3.2.9 如果 $I$ 是环 $R$ 的一个理想且 $I$ 包含 $R$ 的一个可逆元，则 $I = R$ .

3.2.10 证明：(i) 在 $\mathbb{R}^2$ 中定义普通加法及以下乘法： $(x_1, x_2) \odot (y_1, y_2) = (x_1y_1, x_2y_2)$ ，则 $(\mathbb{R}^2, +, \odot)$ 为一环；(ii) 令 $f: \mathbb{R}^2 \to M_2$ 定义为 $(a, b) \mapsto \begin{bmatrix} a & 0 \\ 0 & b \end{bmatrix}$ ， $f$ 则为一环同态。

3.2.11 检验 $\operatorname{sl}(2, \mathbb{R})$ 是否可解，是否为半单 Lie 代数.
