# Answers:

1. $G _ { 1 } ( s )$ Takes longer because the envelope time constant is determined by the magnitude of the real part of the CC poles. $e ^ { - 1 0 t }$ dies down faster than $e ^ { - 5 t }$ .   
2. Trick question! For CC poles: $\sigma \pm j \omega _ { : }$ , the oscillation frequency depends on $\omega _ { n } = \sqrt { \sigma ^ { 2 } + \omega ^ { 2 } }$ which is the same for both systems! (see Eqn. 5.2)   
3. $G _ { 5 } ( s )$ Overshoot depends on the angle between the CC poles and the negative real axis $( \theta = \cos ^ { - 1 } \zeta )$ . The greater the angle, the higher the overshoot (See Chapter 9).
