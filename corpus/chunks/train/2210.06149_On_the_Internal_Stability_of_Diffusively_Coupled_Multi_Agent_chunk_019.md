# 3.2.4. Time-varying 𝐾

The main result also extends to the case of time-varying controllers. This is particularly relevant for varying interconnection topologies, i.e. those where $E _ { \mathcal { G } ( t ) } = E ( t )$ is the incidence matrix of the time-varying graph $\mathcal { G } ( t )$ . Still, the condition $\mathbb { T } ^ { \top } E ( t )$ holds for any topology, rendering the denominator in (9) not stably invertible. We can then use (Verma, 1988, Theorem (i)) to show that under no choice of $K _ { \mathrm { e } }$ the system is stabilizable, at least in the finite-dimensional case, whenever either one of the conditions in (7) holds.
