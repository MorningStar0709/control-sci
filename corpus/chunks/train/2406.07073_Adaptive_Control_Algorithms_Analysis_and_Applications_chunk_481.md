$$\lim _ {t \to \infty} \nu (t + 1) = 0 \tag {10.69}$$

In other terms, we recover the same type of properties as for the case without projection. The key technical fact which allows this result to be obtained is (10.60).

Therefore, in the case of a general type adaptation gain, we should have the property that:

$$\tilde {\theta} _ {p} ^ {T} (t + 1) F (t + 1) ^ {- 1} \tilde {\theta} _ {p} (t + 1) \leq \tilde {\theta} ^ {T} (t + 1) F (t + 1) ^ {- 1} \tilde {\theta} (t + 1) \tag {10.70}$$

The interpretation of this condition in the case of an adaptation gain $F ( t + 1 )$ is that a transformation of coordinates has to be considered by defining:

$$\hat {\theta} ^ {\prime} (t + 1) = F (t + 1) ^ {- 1 / 2} \hat {\theta} (t + 1) \tag {10.71}$$

where:

$$F (t + 1) ^ {- 1} = [ F (t + 1) ^ {- 1 / 2} ] ^ {T} F (t + 1) ^ {- 1 / 2} \tag {10.72}$$

and therefore, the domain $\mathcal { D }$ should also be transformed in $\mathcal { D } ^ { \prime }$ such that if:

$$x \in \mathscr {D}; \quad x ^ {\prime} = F (t + 1) ^ {- 1 / 2} x \in \mathscr {D} ^ {\prime}$$
