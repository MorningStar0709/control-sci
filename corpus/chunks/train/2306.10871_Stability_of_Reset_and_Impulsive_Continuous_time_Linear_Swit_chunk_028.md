# 5. Convex hull of resets

The dwell time constraints and dwell-flee relations for stability under arbitrary impulses obtained thus far have been in terms of infinitely many linear matrix inequalities when the collection of impulses is infinite (but compact), and standard LMI tools can not be applied here. However, these infinitely many inequalities reduce to only finitely many when the collection of impulses is a convex hull of finitely many matrices. Suppose we have a switched system (1) and the collection of impulses is given by a convex hull of k matrices $M _ { 1 } , \dots , M _ { k } \ \in \ M _ { n } ( \mathbb { R } )$ , that is, $\mathcal { T } = \mathrm { c o n v } \{ M _ { 1 } , . . . , M _ { k } \} =$ $\textstyle \left\{ \sum _ { i = 1 } ^ { k } \gamma _ { i } M _ { i } \colon \gamma _ { i } \geq 0 \right.$ for all i, and $\textstyle \sum _ { i = 1 } ^ { k } \gamma _ { i } = 1 \biggr \}$ , Since the obtained LMIs are convex with respect to the impulse matrix variable, the LMIs can be reduced to finitely many by using the fact that the maximum value of a convex function over this convex hull is obtained at one of the vertices Mi, [14]. Specifically,

(1) In Theorem 4.4, condition (10) consists of infinitely many inequalities if $\mathcal { T }$ is an infinite set. Suppose $\mathcal { T } = \operatorname { c o n v } \{ M _ { 1 } , \dots , M _ { k } \}$ . Denoting $\| x \| _ { Q _ { p } } = \sqrt { x ^ { \top } Q _ { p } x }$ , the inequality (10) can be rewritten as

$$\| M \mathrm{e} ^ {A _ {q} \tau} x \| _ {Q _ {p}} \leq \| x \| _ {Q _ {q}} \forall x \in \mathbb {R} ^ {n}, \forall p, q \in \mathcal {P}, \forall M \in \mathcal {I}.$$

For each $x \in \mathbb { R } ^ { n }$ , the function $f _ { q } ( M ) = \lVert M \mathrm { e } ^ { A _ { q } \tau } \boldsymbol { x } \rVert _ { Q _ { 1 } }$ is convex and hence attains its maximum at one of the vertices of the convex hull $\mathcal { T } .$ . Thus the above set of inequalities are satisfied for all $M \in \mathcal { T }$ if and only if these are satisfied for $M = M _ { 1 } , \dots , M _ { k }$ only. Thus, the inequalities (10) reduce to $k | \mathcal { P } | \ ( | \mathcal { P } | - 1 )$ many

$$\| M _ {i} \mathrm{e} ^ {A _ {q} \tau} x \| _ {Q _ {p}} \leq \| x \| _ {Q _ {q}} \forall x \in \mathbb {R} ^ {n}, \forall p, q \in \mathcal {P}, \forall i = 1, \ldots , k.$$

Thus in Theorem 4.4, condition (10) can be replaced by

$$\mathrm{e} ^ {A _ {q} ^ {\top} \tau} M _ {i} ^ {\top} Q _ {p} M _ {i} \mathrm{e} ^ {A _ {q} \tau} \leq Q _ {q} \text {for all} p, q \in \mathcal {P} \text {and} i = 1, \ldots , k.$$
