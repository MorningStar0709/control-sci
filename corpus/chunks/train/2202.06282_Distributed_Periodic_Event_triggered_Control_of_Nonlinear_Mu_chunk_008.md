# 3.1 Multi-agent systems

We consider the setting where multiple agents, each with a local digital platform, communicate with each other via a packet-based network to achieve a common goal such as stabilization, consensus, $\mathcal { L } _ { p ^ { - } }$ performance, etc., which will be captured by a dissipativity property as in Definition 1, as explained further below. To be precise, we consider a collection of $N \in  { \mathbb { N } } _ { > 0 }$ heterogeneous agents $\mathcal { A } _ { 1 } , \mathcal { A } _ { 2 } , \ldots , \mathcal { A } _ { N }$ , which are interconnected according to a digraph $\mathcal { G } ( \nu , \mathcal { E } )$ where $\mathcal { V } : = \ \{ 1 , 2 , \ldots , N \}$ . The digital platform of each agent is used for the implementation of algorithms and control computations. Due to the digital nature, measurements are not continuously available, but only on specific sampling times, as explained in more detail in Section 3.3. The dynamics of the $i ^ { \mathrm { t h } }$ agent, $i \in \mathcal { N } : =$ $\{ 1 , 2 , \ldots , N \}$ , are given by

$$
\mathcal {A} _ {i}: \left\{ \begin{array}{l} \dot {x} _ {i} = f _ {i} (x, \widehat {y} _ {i} ^ {\text { in }}, v _ {i}), \\ y _ {i} = h _ {i} (x _ {i}), \end{array} \right. \tag {2}
$$

where $\begin{array} { r l r } { x _ { i } } & { { } \in } & { \mathbb { R } ^ { n _ { x , i } } } \end{array}$ is the local state vector, x := $( x _ { 1 } , x _ { 2 } , \ldots , x _ { N } ) \in \mathbb { R } ^ { n _ { x } }$ with $\textstyle n _ { x } : = \sum _ { i \in { \mathcal { N } } } n _ { x , i }$ is the global state vector, $v _ { i } \in \mathbb R ^ { n _ { v , i } }$ is a local exogenous disturbance or input, $y _ { i } \in \dot { \mathbb { R } } ^ { n _ { y , i } }$ is the local output, $\bar { y } : = ( y _ { 1 } , y _ { 2 } , \dotsc , y _ { N } ) \in$ $\mathbb { R } ^ { n _ { y } }$ with $\textstyle n _ { y } : = \sum _ { i \in N } n _ { y , i }$ is the global output and $\widehat { y } _ { i } ^ { \mathrm { i n } } \in$ $\mathbb { R } ^ { n _ { y } }$ is agent $A _ { i } \mathrm { { ' } s }$ estimate of the outputs of agents $A _ { m } .$ , $m \in \mathcal { V } _ { i } ^ { \mathrm { i n } } ,$ , where $\mathcal { V } _ { i } ^ { \mathrm { i n } }$ denotes the collection of all agents that transmit information to agent $\mathcal { A } _ { i } .$ , as will be explained in more detail in Section 3.2. We assume that the functions $f _ { i }$ are continuous and that the functions $h _ { i }$ are continuously differentiable. The maps $f _ { i }$ may depend on the entire vector x, implying that we can allow physical couplings between agents, for example in an interconnected physical plant, see Fig. 1 for an illustration.
