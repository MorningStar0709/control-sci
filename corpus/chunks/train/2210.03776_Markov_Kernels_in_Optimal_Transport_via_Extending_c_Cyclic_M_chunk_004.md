# 2. Background on Optimal Transport

We now provide the essential background on optimal transport that is need for the work that follows. General background material far beyond what we present here can be found in [1, 20, 29].

Optimal transport considers probability measures on Polish spaces $( X , { \mathcal { B } } _ { X } )$ and $( Y , { \mathcal { B } } _ { Y } )$ . Let the set of probability measures on X be denoted by ${ \mathcal { P } } ( X )$ . If $\mu \in { \mathcal { P } } ( X )$ and $\nu \in { \mathcal { P } } ( Y )$ , then the optimal transport cost is the infimum of the functional $C ( \gamma )$ . The functional $C ( \gamma )$ is defined by

$$C (\gamma) = \int_ {X \times Y} c (x, y) d \gamma (x, y) \tag {1}$$

where $\gamma \in { \mathcal { P } } ( X \times Y )$ is a coupling of $\mu$ and ν. A measure $\gamma \in { \mathcal { P } } ( X \times Y )$ is a coupling of $\mu$ and ν if $( \mathrm { p r o j } _ { X } ) _ { \# } \gamma = \mu$ and $( \mathrm { p r o j } _ { Y } ) _ { \# } \gamma = \nu ,$ with $\mathrm { p r o j } _ { X }$ and $\mathrm { p r o j } _ { Y }$ being the projection maps from $X \times Y$ to X and Y respectively. The set of couplings in ${ \mathcal { P } } ( X \times Y )$ is denoted $\Pi ( \mu , \nu )$ . Any $\gamma \in \Pi ( \mu , \nu )$ that obtains the infimal value is called an optimal coupling of $\mu$ and ν. The cost function $c ( x , y )$ is assumed to be continuous and non-negative for reasons discussed in Section 3. So long as $c ( x , y )$ is lower semi-continuous and bounded from below, then there exists an optimal coupling, as stated in Theorem 2. Allow $C ( \mu , \nu )$ to refer to the optimal transport cost between $\mu$ and ν.
