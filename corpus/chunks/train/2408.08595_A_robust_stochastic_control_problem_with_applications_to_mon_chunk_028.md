# 5.2 Application to MMV and MV investment-reinsurance problem

In this subsection, we slightly modify the setup of former subsection. Let $( \Omega , \mathcal { F } , \mathbb { P } )$ be a probability space on which defined a standard n-dimensional Brownian motion. We further assume the Poisson random measure $\bar { \gamma }$ on $[ 0 , T ] \times \Omega \times \mathbb { R } _ { + }$ is independent with the Brownian motion W under the probability measure P. We denote the intensity $\lambda > 0$ and the distribution function $\nu : \mathbb { R } ^ { + }  [ 0 , 1 ]$ , and the compensated Poisson random measure can be write as

$$\tilde {\gamma} (d t, d y) = \bar {\gamma} (d t, d y) - \lambda \nu (d y) d t.$$

Define the filtration $\mathbb { F } = \{ \mathcal { F } _ { t } \} _ { t > 0 }$ as the augmented natural filtration generated by W and $\bar { \gamma }$ . Define the filtration $\mathbb { F } ^ { W } = \{ \mathcal { F } _ { t } ^ { W } \} _ { t \ge 0 }$ as the augmented natural filtration generated by W .

Let $B ( \mathbb { R } _ { + } )$ be the Borel σ-algebra of $\mathbb { R } _ { + } , \mathcal { P }$ the F-predictable σ-algebra on $[ 0 , T ] \times \Omega$ . We will introduce some further notations. Let $L ^ { 2 , \nu }$ be the set of $B \left( \mathbb { R } _ { + } \right)$ -measurable functions $\varphi : \mathbb { R } _ { + } \to \mathbb { \quad }$ R such that $\begin{array} { r } { \int _ { \mathbb { R } _ { + } } \varphi ( y ) ^ { 2 } \nu ( d y ) < \infty } \end{array}$ . Let $L _ { \mathcal { P } } ^ { 2 , \nu } ( 0 , T ; \mathbb { R } )$ be the set of $\mathcal { P } \otimes B ( \mathbb { R } _ { + } )$ )-measurable functions $\varphi : [ 0 , T ] \times \Omega \times \mathbb { R } _ { + } \to \mathbb { R }$ such that $\begin{array} { r } { \mathbb { E } \int _ { 0 } ^ { T } \int _ { \mathbb { R } _ { + } } | \varphi | ^ { 2 } \lambda \nu ( d y ) d t < \infty } \end{array}$ . Let $L _ { \mathcal { P } } ^ { \infty , \nu } ( 0 , T ; \mathbb { R } )$ be the set of functions $\varphi \in L _ { \mathcal { P } } ^ { 2 , \nu } ( 0 , T ; \mathbb { R } )$ which are essentially bounded w.r.t. dt $\otimes d \mathbb { P } \otimes \nu ( d y )$ .
