# 3.2 Linearity of $H _ { f } ^ { * } ( t , b )$ in b

We discuss a class of optimal control problems in which $H _ { f } ^ { * } ( t , b )$ is linear in b while $f ( s , \pmb { x } , a )$ is nonlinear in $a .$ The simplest case is $R ( t , a ) = 0$ so that $\check { L ( t , x , a ) }$ is independent of the control input. It is straightforward to check that $H ^ { * } = 0$ in that case. However, it is a too restrictive case. Thus, we aim to find a class of problems with control-dependent cost functions.

Consider a dynamical system of the form

$$\dot {x} (t) = (\psi (u (t)), A x (t)) \in \mathbb {R} ^ {n}$$

where $A x ( t ) \in \mathbb { R } ^ { n - 2 } , \psi ( u ( t ) ) \in \mathbb { R } ^ { 2 } , u ( t ) \in U \subset \mathbb { R } ^ { 2 }$ and $\ b { x } ( t ) \in \mathbb { R } ^ { n }$ . For simplicity, let $U = U _ { 1 } \times U _ { 2 }$ and $U _ { 1 } = \{ q _ { 1 } , q _ { 2 } \}$ with $q _ { i } \in \mathbb { R }$ and $U _ { 2 }$ represents an arbitrary interval in R. It is a natural class of control sets as we often consider switched systems where both discrete and continuous control inputs are used. For $a = ( a _ { 1 } , a _ { 2 } ) \in U _ { 1 } \times U _ { 2 }$ , we further assume

$$\psi (a) = (\psi_ {1} (a), \psi_ {2} (a)) = (f _ {1} (a _ {1}) a _ {2}, f _ {2} (a _ {1}) a _ {2})$$

and $L ( s , { \pmb x } , { a } ) = S ( { \pmb x } ) + c { a } _ { 2 }$ for some $c \in \mathbb { R }$ , where $f _ { i } ( \cdot ) \mathrm { \bar { s } }$ are functions defined on $U _ { i }$ . In this setting, $H _ { f } ^ { * }$ is linear in b. To see this, let $b _ { 1 } : = f _ { 1 } ( a _ { 1 } ) a _ { 2 }$ and $b _ { 2 } : = f _ { 2 } ( a _ { 1 } ) a _ { 2 }$ . Then, $b _ { 1 }$ and $b _ { 2 }$ form a line segment for each $a _ { 1 } \in \{ q _ { 1 } , q _ { 2 } \}$ . As a result, we have two line segments represented by $( b _ { 1 } , b _ { 2 } )$ that are parametrized over $U = U _ { 1 } \times U _ { 2 }$ . Since the Lagrangian $L ( t , \pmb { x } , a ) = V ( \pmb { x } ) + c a _ { 2 }$ is maximized at the end point of each line segment, $\tilde { L } ( t , { \pmb x } , b ) = S ( { \pmb x } ) + c ^ { \prime } \cdot b$ for some $c ^ { \prime } \in \mathbb { R } ^ { 2 }$ . Therefore, $H _ { f } ^ { * } ( t , b )$ is linear in b.
