$$
\hat {\mathbf {x}} _ {j} = \left[ \begin{array}{c} \mathbf {x} _ {j} \\ \mathbf {x} _ {j - 1} \\ \dots \\ \mathbf {x} _ {j - s} \end{array} \right] \in \mathbb {R} ^ {N (s + 1)}, \quad \hat {\mathbf {u}} _ {j} = \left[ \begin{array}{c} \mathbf {u} _ {j} \\ \mathbf {u} _ {j - 1} \\ \dots \\ \mathbf {u} _ {j - z} \end{array} \right] \in \mathbb {R} ^ {Q (z + 1)}. \tag {5}
$$

As a consequence, the extended state matrix and the extended system input matrix are defined as $\widehat { \mathbf { A } } \in$ $\mathbb { R } ^ { N ( s + 1 ) \times N ( \hat { s } + 1 ) }$ and $\widehat { \mathbf { B } } \in \mathbb { R } ^ { N ( s + 1 ) \times Q ( z + 1 ) }$ , respectively.

The procedure to extract the extended matrices from data starts by introducing the vector $\hat { \bf y } _ { j } ~ \in { }$ $\mathbb { R } ^ { N ( s + 1 ) + Q ( z + 1 ) }$

$$
\hat {\mathbf {y}} _ {j} = \left[ \begin{array}{l} \hat {\mathbf {x}} _ {j} \\ \hat {\mathbf {u}} _ {j} \end{array} \right], \tag {6}
$$

such that Eq. (3) can be rewritten as:

$$
\hat {\mathbf {x}} _ {j + 1} = \widehat {\mathbf {G}} \hat {\mathbf {y}} _ {j}, \quad \text { with } \quad \widehat {\mathbf {G}} = \left[ \begin{array}{l l} \widehat {\mathbf {A}} & \widehat {\mathbf {B}} \end{array} \right]. \tag {7}
$$

Data are collected for model building from m snapshots and are arranged in two augmented data matrices $\widehat { \mathbf { Y } } \in \mathbb { R } ^ { ( N ( s + 1 ) + Q ( z + 1 ) ) \times ( m - 1 ) }$ and $\widehat { \mathbf { X } } ^ { \prime } \in \mathbb { R } ^ { N ( s + 1 ) \times ( m ^ { \star } - 1 ) }$ , which are built as:

$$
\widehat {\mathbf {Y}} = \left[ \begin{array}{l} \mathbf {X} \\ \mathbf {S} \\ \mathbf {U} \\ \mathbf {Z} \end{array} \right], \quad \widehat {\mathbf {X}} ^ {\prime} = \left[ \begin{array}{l} \mathbf {X} ^ {\prime} \\ \mathbf {S} ^ {\prime} \end{array} \right]. \tag {8}
$$

The matrices $\mathbf { X } \in \mathbb { R } ^ { N \times ( m - 1 ) } , \mathbf { X } ^ { \prime } \in \mathbb { R } ^ { N \times ( m - 1 ) }$ , and $\mathbf { U } \in \mathbb { R } ^ { Q \times ( m - 1 ) }$ contain the extended state and input snapshots at the m considered time instants:

$$
\mathbf {X} = \left[ \begin{array}{c c c c} \mathbf {x} _ {j} & \mathbf {x} _ {j + 1} & \ldots & \mathbf {x} _ {m - 1} \end{array} \right],

\mathbf {X} ^ {\prime} = \left[ \begin{array}{l l l l} \mathbf {x} _ {j + 1} & \mathbf {x} _ {j + 2} & \dots & \mathbf {x} _ {m} \end{array} \right], \tag {9}
