Now we define another set-valued map for any policy $\pi .$ . To this end, given $\pi \in \Pi _ { l }$ , let us define the following transition probability for any $\mu \in { \mathcal { P } } ( \mathbb { X } )$

$$p _ {\mu} ^ {\pi} (\cdot | x) := \int_ {\mathsf {A}} p (\cdot | x, a, \mu) \pi (d a | x)$$

In view of this, let us define the following set-valued map

$$\Phi (\pi) := \left\{\mu \in \mathcal {P} (X): \mu \text { is an invariant distribution of } p _ {\mu} ^ {\pi} \right\}$$

Again, under mild regularity conditions on ${ \bar { p } } ,$ it is possible to prove that $\Phi ( \pi )$ is non-empty for all π or at least for all continuous π under weak convergence topology.

Now it is time to introduce the equilibrium notion adapted in the infinite population limit, which is called stationary mean-field equilibrium (MFE).

DEFINITION 2 (Stationary Mean-field equilibrium). A pair $( \mu ^ { * } , \pi ^ { * } )$ is a stationary meanfield equilibrium if $\pi ^ { * } \in \Lambda ( \mu ^ { * } )$ and $\mu ^ { * } \in \Phi ( \pi ^ { * } )$ .

Under mild regularity conditions on $\bar { p }$ and c¯, it is possible to prove that MFE exists [34, Theorem 3.3]. For instance, if $\bar { p }$ and c¯ are continuous, and X is compact, then there exists a MFE. This can be established via the method that is used to prove [34, Theorem 3.3]. As the conditions for the existence of MFE are weaker than the assumptions that are introduced in the next section, in the remainder of this paper, we suppose that there exists at least one MFE $( \mu ^ { * } , \pi ^ { * } )$ .

3.1.1. Assumptions. We now state the assumptions that are used throughout Section 3 and Section 4. To avoid potential compactness issues, we only consider games with compact state spaces.

(a) X is a compact Borel space and A is a convex and compact subset of some finitedimensional Euclidean space.   
(b) The one-stage cost function $\bar { c } : \mathsf { X } \times \mathsf { A } \times \mathsf { X } \to [ 0 , \infty )$ is Lipschitz continuous with Lipschitz constants $( \mathsf { L } _ { 1 } , \mathsf { L } _ { 2 } , \mathsf { L } _ { 3 } )$ ; that is, for any $( x , a , y ) , ( z , b , r ) \in \mathsf { X } \times \mathsf { A } \times \mathsf { X }$ , we have

$$| \bar {c} (x, a, y) - \bar {c} (z, b, r) | \leq \mathsf {L} _ {1} d _ {\mathsf {X}} (x, z) + \mathsf {L} _ {2} \| a - b \| + \mathsf {L} _ {3} d _ {\mathsf {X}} (y, r)$$
