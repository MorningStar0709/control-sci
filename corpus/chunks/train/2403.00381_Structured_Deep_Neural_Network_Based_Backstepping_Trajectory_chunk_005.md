# A. Euler-Lagrange Equation

The Euler-Lagrange equation describes the motion of a mechanical system, which is

$$\frac {d}{d t} \left(\frac {\partial L}{\partial \dot {\boldsymbol {q}}}\right) - \frac {\partial L}{\partial \boldsymbol {q}} = \boldsymbol {u} + \boldsymbol {\tau} ^ {\boldsymbol {d}}, L (\boldsymbol {q}, \dot {\boldsymbol {q}}) = T (\boldsymbol {q}, \dot {\boldsymbol {q}}) - V (\boldsymbol {q}), (1)$$

where L is the Lagrangian, T is the kinetic energy, V is the potential energy, q is the generalized coordinates, u is the generalized non-conservative force, and $\tau ^ { d }$ is the disturbance acting on the system. Equation (1) can be equivalently represented as

$$M (\boldsymbol {q}) \ddot {\boldsymbol {q}} + C (\boldsymbol {q}, \dot {\boldsymbol {q}}) \dot {\boldsymbol {q}} + G (\boldsymbol {q}) = \boldsymbol {u} + \tau^ {d}, \tag {2}$$

where $\begin{array} { r } { M ( \pmb q ) = \frac { \partial ^ { 2 } ( T ( \pmb q , \dot { \pmb q } ) ) } { \partial \dot { \pmb q } \partial \dot { \pmb q } } } \end{array}$ is the inertia matrix, $C ( q , \dot { q } ) =$ $\dot { M } ( \pmb q ) - \frac { 1 } { 2 } \dot { M } ( \pmb q ) ^ { \top }$ is the Coriolis matrix, and $G ( \pmb { q } ) ~ =$ $- { \frac { \partial ( V ( q ) ) } { \partial q } }$ ∂(V (q)) is the gravitational force vector. The Euler-Lagrange equation has the following properties [32].

Property 1. M is positive definite and is bounded by

$$a _ {1} \| \boldsymbol {x} \| ^ {2} \leq \boldsymbol {x} ^ {\top} \boldsymbol {M} \boldsymbol {x} \leq a _ {2} \| \boldsymbol {x} \| ^ {2}, \quad \forall \boldsymbol {x} \in \mathbb {R} ^ {n}$$

where $a _ { 1 } , a _ { 2 } \in \mathbb { R } ^ { + }$ are positive constants.

Property 2. $\dot { M } - 2 C$ is skew symmetric satisfying $x ^ { \top } ( { \dot { M } } -$ $2 C ) { \boldsymbol { x } } = 0 , \quad \forall { \boldsymbol { x } } \in \mathbb { R } ^ { n }$ .
