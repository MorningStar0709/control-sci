Next, introduce the error estimate between the functionals of the N-player game and the MFG. For $N \in  { \mathbb { N } }$ and $1 \leq n \leq N$ , set

$$R _ {N} (\xi^ {n}) := J ^ {n} (\xi^ {n}, \bar {\xi} ^ {- n}) - J ^ {n} (\xi^ {n}, \theta^ {*}), \quad \xi^ {n} \in \widehat {\mathcal {A}} _ {b}.$$

Since the control $\xi ( \theta ^ { * } )$ is optimal for $\theta ^ { * }$ (cf. Proposition 3.5), using (5.5), we have that

$$J ^ {n} (\bar {\xi} ^ {n}, \bar {\xi} ^ {- n}) - J ^ {n} (\xi^ {n}, \bar {\xi} ^ {- n}) = R _ {N} (\bar {\xi} ^ {n}) - R _ {N} (\xi^ {n}) + J ^ {n} (\bar {\xi} ^ {n}, \theta^ {*}) - J ^ {n} (\xi^ {n}, \theta^ {*})= R _ {N} (\bar {\xi} ^ {n}) - R _ {N} (\xi^ {n}) + J (\xi (\theta^ {*}), \theta^ {*}) - J (\tilde {\xi} ^ {n}, \theta^ {*})\geq R _ {N} (\bar {\xi} ^ {n}) - R _ {N} (\xi^ {n}), \text { for any } \xi^ {n} \in \widehat {\mathcal {A}} _ {b}.$$

Therefore, in order to complete the proof it is sufficient to show that

$$\lim _ {N \uparrow \infty} \sup _ {\xi^ {n} \in \widehat {\mathcal {A}} _ {b}} R _ {N} (\xi^ {n}) = 0. \tag {5.6}$$

By using that $\begin{array} { r } { \operatorname* { l i m } _ { T \uparrow \infty } \left( \alpha _ { T } + \beta _ { T } \right) \geq \varlimsup _ { T \uparrow \infty } ( \alpha _ { T } ) + \varliminf _ { T \uparrow \infty } ( \beta _ { T } ) } \end{array}$ , write

$$R _ {N} (\xi^ {n}) \leq \operatorname * {l i m s u p} _ {T \uparrow \infty} \frac {1}{T} \mathbb {E} \left[ \int_ {0} ^ {T} \left(\pi (X _ {t} ^ {n, \xi^ {n}}, \theta_ {\bar {\xi} - n} ^ {N} (t)) - \pi (X _ {t} ^ {n, \xi^ {n}}, \theta^ {*})\right) d t \right]. \tag {5.7}$$

Next, by Proposition 4.2, there exists a stationary distribution $\nu ^ { \xi ^ { n } }$ for the process $( X _ { t } ^ { n , \xi ^ { n } } , Y _ { t } ) _ { t \geq 0 }$ . Moreover, Proposition 5.1 ensures that the process $\left( ( X ^ { n , \xi ^ { n } } , Y ^ { n } ) , \{ ( X ^ { \ell , \bar { \xi } ^ { \ell } } , Y ^ { \ell } ) _ { t \geq 0 } \} _ { \ell \neq n } \right)$ admits an ergodic distribution $\otimes _ { \ell \neq n } \nu ^ { \ell } \otimes \nu ^ { \xi ^ { n } }$ with support in the compact set $( { \widehat { K } } \times \mathbb { Y } ) ^ { N }$ . Thus, by continuity of the functions $f , F$ and π, Proposition 5.1 also allows to rewrite (5.7) as

$$R _ {N} (\xi^ {n}) \leq \int_ {\mathcal {O}} \int_ {\mathcal {O} ^ {N - 1}} \Bigl (\pi \Bigl (x _ {n}, F \bigl (\sum_ {\ell \neq n} \frac {f (x _ {\ell})}{N - 1} \bigr) \Bigr) - \pi (x _ {n}, \theta^ {*}) \Bigr) \otimes_ {\ell \neq n} \nu^ {\ell} (d x _ {\ell}, i _ {\ell}) \otimes \nu^ {\xi^ {n}} (d x _ {n}, i _ {n}).$$
