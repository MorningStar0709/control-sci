# C. Proof of Theorem 2

Let $\bar { \nu } : = 4 \mathrm { L i p } _ { J } \sigma _ { \operatorname* { m a x } } \mathrm { L i p } _ { Z } .$ .

Consider a current state $X _ { k \delta } \in B _ { S ^ { * } } \backslash B _ { s }$ ∗ at some time step $k \in \mathbb { N } _ { 0 } .$ . If $\delta \leq \bar { \delta }$ and $U _ { k \delta } : = \eta ( X _ { k \delta } )$ , then by Assumption 1-2, it holds that

$\mathrm { i } ) \hat { J } ^ { w ^ { \# } } ( X _ { ( k + 1 ) \delta } ) - \hat { J } ^ { w ^ { \# } } ( X _ { k \delta } ) \leq - \delta \nu ( X _ { k \delta } ) ,$   
$\mathrm { i i } ) \alpha _ { \mathsf { l o w } } \big ( \| X _ { k \delta } \| \big ) \leq \hat { J } ^ { w ^ { \# } } ( X _ { k \delta } ) \leq \alpha _ { \mathsf { u p } } \big ( \| X _ { k \delta } \| \big ) .$

Given these properties, it needs to be shown that (C1)–(C4) are well posed to mean that these constraints are feasible for all times where $X _ { k \delta } \notin \mathcal { B } _ { s ^ { * } }$ .

First let’s ensure that $\delta > 0$ is sufficiently small and select and adequate ε.

$$
\begin{array}{l} \delta <   \frac {\sigma_ {\max} \mathrm{Lip} _ {Z}}{3 \mathrm{Lip} _ {f} (\bar {f} + \sigma_ {\max} \mathrm{Lip} _ {Z})}, \\ \varepsilon := (2 + \frac {2}{3}) \mathrm{Lip} _ {J} \sigma_ {\max} \mathrm{Lip} _ {Z} = \frac {\bar {\nu}}{2} + \frac {\bar {\nu}}{6}. \tag {34} \\ \end{array}
$$

Note, that the latter implies

$$\operatorname{Lip} _ {\hat {J}} \operatorname{Lip} _ {f} (\bar {f} + \sigma_ {\max} \operatorname{Lip} _ {Z}) \delta < \frac {\operatorname{Lip} _ {\hat {J}} \sigma_ {\max} \operatorname{Lip} _ {Z}}{3} < \frac {\bar {\nu}}{1 2}. \tag {35}$$

From i), it follows that

$$
\begin{array}{l} \hat {J} ^ {w ^ {\#}} \left(\Lambda_ {U _ {k \delta}} \left(X _ {k \delta}\right)\right) - \hat {J} ^ {w ^ {\#}} \left(X _ {k \delta}\right) \\ \leq - \delta \nu (X _ {k \delta}) + \\ \mathrm{Lip} _ {\hat {J}} \mathrm{Lip} _ {f} (\bar {f} + \sigma_ {\max} \mathrm{Lip} _ {Z}) \delta^ {2} + \mathrm{Lip} _ {\hat {J}} \sigma_ {\max} \mathrm{Lip} _ {Z} \delta \\ \leq - \delta (\nu (X _ {k \delta}) - \frac {\bar {\nu}}{1 2} - \frac {\bar {\nu}}{4}) \leq - \delta \bar {\nu}. \\ \end{array}
$$

Hence, (C3) is satisfied.

Now, note that

$$\hat {J} ^ {w ^ {\#}} (\cdot) := L (\cdot), \tag {36}$$

which directly implies that (C2) is satisfied.
