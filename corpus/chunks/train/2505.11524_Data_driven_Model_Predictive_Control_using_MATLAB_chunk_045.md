# 3.5.1 PEM-based D-LMPC: Numerical example

To illustrate PEM-based D-LMPC, the input-output training data is generated using a fourth-order LTI system for which the parameters are:

$$
\mathbf {A} = \left[ \begin{array}{c c c c} 0. 5 & 0 & 0. 0 5 & 0. 1 \\ 0 & 0. 7 & 0 & 0. 0 4 \\ 0 & 0 & 0. 5 5 & 0. 1 \\ 0. 2 & 0. 1 & 0 & 0. 1 \end{array} \right] \quad \mathbf {B} = \left[ \begin{array}{c} 0. 5 \\ 0 \\ 0. 1 \\ 0. 7 \end{array} \right] \quad \mathbf {C} = \left[ \begin{array}{l l l l} 1 & 0 & 0 & 0 \end{array} \right]. \tag {66}
$$

The LTI system is simulated for 1000 instants with PRMS input, and the first 500 samples are used for training the model and the rest for validating the model. The system parameters identified with PEM is then used for implementing MPC for which the simulation parameters are chosen as $\mathrm { { N _ { T } ~ = ~ 2 0 0 , ~ N ~ = ~ 2 0 , ~ Q _ { x } ~ = ~ 2 I _ { 4 } , R _ { u } ~ = ~ 3 ~ a n d ~ x _ { 0 } ~ = ~ } }$ $\left[ 0 \quad 0 \quad 0 \quad 0 \quad 0 \right] ^ { T }$ . The constraint set is defined as $u _ { m i n } = - 5 , u _ { m a x } = 5$ . The response of the LTI system with the PEM-based D-LMPC scheme is given in Fig. 5. The output reference is chosen similarly to the SPC example, with $\mathbf { y } _ { \mathrm { r } }$ taking values as 1, 0.7, 0.5. The training input and output data, along with the output predicted with the PEM model, are plotted in Fig. 5(a). The output tracking with MPC is given in Fig. 5(b), which shows that the output follows the desired reference. The control input plot is given in Fig. 5(c).
