# IV. LEARNING-BASED FIXED-TIME CONTROL WITH GAUSSIAN PROCESS REGRESSION

To address the problem of uncertainty approximation, we employ Gaussian Process (GP) regression as a datadriven method to model the unknown function $f _ { i } ( x )$ and its associated uncertainty. Using GP regression, we propose a learning-based fixed-time controller that guarantees fixedtime convergence.

An important assumption imposes a restriction on the complexity of the map $f _ { i } ( x )$ through the reproducing kernel Hilbert space (RKHS) norm, as described below.

Assumption 1: For the map $f _ { i } : X \to \mathbb { R } ^ { n }$ in , the RKHS norm with respect to the kernel κ is bounded, i.e., $\| f _ { i } \| _ { \kappa } \leq$ $\infty$ for all $i \in \{ 1 , \ldots , n \}$ .

This assumption is satisfied by most commonly used kernels, as all continuous functions defined over a compact state-space meet this condition [21]. For further details on the RKHS norm, we refer the interested reader to [28].

We assume access to measurements $x \in X$ and $y _ { i } =$ $f _ { i } ( x ) + \mathcal { W }$ , where $\mathscr { W } \sim \mathcal { N } ( 0 _ { n } , \rho _ { F } ^ { 2 } { I } _ { n } )$ represents additive noise with ${ \rho } _ { F } \in \mathbb { R } _ { 0 } ^ { + }$ , as stated in [19].

Assumption 2: Measurements $x \in X$ and $y _ { i } = f _ { i } ( x ) + \mathcal { W }$ are available, where ${ \mathcal { W } } \sim { \mathcal { N } } ( 0 _ { n } , \rho _ { F } ^ { 2 } I _ { n } )$ is the noise.
