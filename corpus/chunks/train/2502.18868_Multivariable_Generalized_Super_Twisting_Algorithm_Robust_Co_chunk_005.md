# 1.1 Notation

For real vectors or matrices, $( ^ { T } )$ refers to their transpose. For symmetric matrices, $( \bullet )$ denotes the symmetric block. The symbols $\mathbb { R } , \mathbb { R } _ { + }$ , N and K denote the sets of real, real non-negative, natural numbers and $\mathbb { K } =$ $\{ 1 , 2 , \cdots , N \}$ with $\bar { N } \in \mathbb { N } \backslash \{ 0 \}$ , respectively. For any symmetric matrix, $X > 0 \ ( \dot { X } \stackrel { . } { \geq } 0 )$ denotes a positive (semi-) definite matrix. As usual, for real matrices the symbol  denotes the maximum singular value and for real vectors, the same symbol denotes the Euclidean norm. The unit simplex $\boldsymbol { \Lambda } \doteq \mathbb { R } ^ { N }$ stands for the set of all nonnegative vectors with the sum of components equal to one. The convex hull of matrices $\{ B _ { i } \} _ { i \in \mathbb { K } } ,$ with the same dimensions, is denoted by $\mathrm { c o } \{ B _ { i } \} _ { i \in \mathbb { K } }$ . Finally, $I _ { n }$ denotes the $n \times n$ identity matrix. Filippov’s definition for the solution of discontinuous differential equations is adopted [2].
