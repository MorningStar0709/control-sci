To distinguish the Jacobian notation from the gradient notation, for a vectorvalued function $G : \mathbb { R } ^ { n } \times \mathbb { R } _ { > 0 } \to \mathbb { R } ^ { p }$ , we denote its Jacobian with respect to x by $\frac { \partial G } { \partial x } \ \in \ \mathbb { R } ^ { p \times n }$ . Throughout this paper, for notational compactness, we sometimes also use the dot notation ˙(·) to represent the total time derivative of a quantity.

If $F : \mathbb { R } ^ { n } \times \mathbb { R } _ { \geq 0 } $ R is locally Lipschitz in x and $C ^ { 1 }$ in t, the Clarke generalized gradient [30] is defined as

$$\partial_ {x} ^ {\circ} F (x, t) := \operatorname{co} \left\{\lim _ {k \rightarrow \infty} \nabla_ {x} F (\tilde {x} _ {k}, t) \Bigg | \tilde {x} _ {k} \rightarrow x, \tilde {x} _ {k} \notin \mathcal {E} _ {t} \right\},$$

in which co denotes the convex hull of a set, and $\mathcal { E } _ { t }$ of Lebesgue measure zero is the set where F is nondifferentiable with respect to x at time t. Correspondingly, the nonsmooth2 Lie derivative, e.g., [31], [32], is calculated as

$$L _ {X} ^ {\circ} F (x, t) = \sup _ {\nu \in \partial_ {x} ^ {\circ} F (x, t)} \nu X (x, t).$$
