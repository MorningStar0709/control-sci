# A. Controllability of dynamic network

A dynamic network can typically be modeled as a linear time-varying (LTV) system, and following [44], its dynamics can be represented as

$$\frac {d \boldsymbol {x} (t)}{d t} = \boldsymbol {A} (t) \boldsymbol {x} (t) + \boldsymbol {B} (t) \boldsymbol {u} (t)$$

where state vector $\pmb { x } ( t ) = ( x _ { 1 } ( t ) , \ldots , x _ { N _ { t } } ( t ) ) ^ { \mathsf { T } } \in \mathbb { R } ^ { N _ { t } }$ represents the states of all $N _ { t }$ nodes at time ??, with $x _ { \mathrm { n } } ( t )$ denoting the state of node ??. The number of nodes $N _ { t }$ is not necessarily fixed but can change at certain time instants. $\pmb { A } ( t ) \in \mathbb { R } ^ { N _ { t } \times N _ { t } }$ denotes the adjacency matrix of the dynamic network at $t , B ( t ) \in \mathbb { R } ^ { N _ { t } \times M _ { t } }$ , where $M _ { t } \le N _ { t }$ specifies the nodes to be controlled by the external controller at ??, and $\pmb { u } ( t ) = ( u _ { 1 } ( t ) , \ldots , u _ { M _ { t } } ( t ) ) ^ { \mathnormal { \mathrm { T } } } \in \mathbb { R } ^ { M _ { t } }$ defines the input on $M _ { t }$ nodes. The controller uses input ${ \pmb u } ( t )$ to control the system, where each control signal $u _ { \mathrm { m } } ( t )$ can typically influence multiple nodes, depending on the system’s control structure. The set of nodes controlled by the external signals is determined by the input matrix $\pmb { B } ( t )$ , which maps the control signals to the corresponding network nodes.
