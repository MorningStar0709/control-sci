$$\lim _ {h \downarrow 0} \int_ {[ 0, T ]} (D ^ {h, N} (s -)) ^ {\top} d X ^ {h, N} (s) + \frac {1}{2} \int_ {[ 0, T ]} (\Delta X ^ {h, N} (s)) ^ {\top} \widetilde {\gamma} d X ^ {h, N} (s) \tag {A.5}= N a \left(- e _ {2} ^ {\top} \widetilde {\gamma} ^ {\top} e _ {1} + e _ {1} ^ {\top} \widetilde {\gamma} ^ {\top} e _ {2}\right) = - N a \left(e _ {1} ^ {\top} (\widetilde {\gamma} - \widetilde {\gamma} ^ {\top}) e _ {2}\right).$$

Note that for all $p \in [ 1 , \infty )$ it holds $\begin{array} { r } { E [ \operatorname* { s u p } _ { s \in [ 0 , T ] } \| \nu _ { s } \| _ { F } ^ { p } ] < \infty } \end{array}$ and $\begin{array} { r } { E [ \mathrm { s u p } _ { s \in [ 0 , T ] } \Vert \nu _ { s } ^ { - 1 } \Vert _ { F } ^ { p } ] < } \end{array}$ $\infty$ . This, $( \mathrm { A . 4 } ) , \ ( \mathrm { A . 5 } )$ , Hölder’s inequality, and the dominated convergence theorem show for all $N \in  { \mathbb { N } }$ that

$$\lim _ {h \downarrow 0} E _ {0} \biggl [ \int_ {[ 0, T ]} (D ^ {h, N} (s -)) ^ {\top} d X ^ {h, N} (s) + \frac {1}{2} \int_ {[ 0, T ]} (\Delta X ^ {h, N} (s)) ^ {\top} \widetilde {\gamma} d X ^ {h, N} (s) \biggr ] = - N a e _ {1} ^ {\top} (\widetilde {\gamma} - \widetilde {\gamma} ^ {\top}) e _ {2}.$$

The fact that $a e _ { 1 } ^ { \top } ( \widetilde { \gamma } - \widetilde { \gamma } ^ { \top } ) e _ { 2 } > 0$ demonstrates that the right-hand side of this equation tends to −∞ as $N \to \infty$ . Thus, by trading faster $( h \downarrow 0 )$ and larger volumes $( N \to \infty )$ , we in this example can generate arbitrarily large negative costs.
