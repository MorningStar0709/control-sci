# 2.2 Continuous Piecewise Affine Functions

The storage function V is non-unique and has no set functional form [32]. Further, the QSR dissipativity condition is a function of $x \in \Omega$ . CPA functions will help us in the nontrivial task of searching for a storage function which satisfies (5) across Ω. Inspired by the CPA Lyapunov function search techniques of [7,8], we will use this flexible functional form for candidate storage functions.

Definition 4 [8] Affine independence: A collection of m vectors $\{ x _ { 0 } , x _ { 1 } , \dotsc , x _ { m } \} \subset \bar { \mathbb { R } } ^ { n }$ is affinely independent if $x _ { 1 } - x _ { 0 } , \ldots , x _ { m } - x _ { 0 }$ are linearly independent.

Definition 5 [8] n - simplex: A simplex, σ, is defined the convex hull of $n + 1$ affinely independent vectors, $c o \{ x _ { j } \} _ { j = 0 } ^ { n } ,$ , where each vector, $x _ { j } \in \mathbb { R } ^ { n }$ , is a vertex.

Definition 6 [8] Triangulation: Let ${ \mathcal { T } } { = } \{ { \sigma } _ { i } \} _ { i = 1 } ^ { m _ { \mathcal { T } } } \in { \mathfrak { R } } ^ { n }$ represent a finite collection of mT simplexes, where the intersection of any two simplexes is a face or an empty set.

Let ${ \mathcal T } = \{ \sigma _ { i } \} _ { i = 1 } ^ { m \tau }$ . Further, let $\{ x _ { i , j } \} _ { j = 0 } ^ { n }$ be a simplex’s $\left( \sigma _ { i } \right)$ vertices. The choice of $x _ { i , 0 }$ in $\sigma _ { i }$ is arbitrary unless $0 \in \sigma _ { i }$ , in which case $x _ { i , 0 } = \mathrm { ~ 0 ~ } [ 8 ]$ . The vertices of the triangulation $\tau$ in a set Ω are denoted as $\mathbb { E } _ { \Omega }$ . Let $\mathcal { T } _ { 0 }$ denote the simplexes in T containing 0 and $\tau _ { \Omega \backslash \{ 0 \} }$ denotes simplices that do not contain 0.
