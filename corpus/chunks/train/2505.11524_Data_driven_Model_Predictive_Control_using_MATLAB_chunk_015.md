$$\inf _ {\mathbf {U} _ {k}} \mathbf {U} _ {k} ^ {\top} \mathbf {H} _ {\mathbf {Y}} \mathbf {U} _ {k} + \mathbf {U} _ {k} ^ {\top} \mathbf {q} _ {\mathbf {Y}} \tag {20}$$

$$\mathbf {u} _ {k \mid k} = \mathbf {U} _ {k (1: m)} ^ {*}. \tag {21}$$

Here, one of the challenges associated with the implementation of output-based MPC is the requirement of the current state ${ \bf x } _ { k \left| k \right. }$ (which is normally unknown) in computing the term $\mathbf { q } \mathbf { \mathbf { \mathbf { v } } }$ in the cost function $J _ { y }$ . Therefore, we need to estimate the current state from the input-output data for which state estimators such as Kalman filters can be used. Next, we move on to the state-based LMPC.
