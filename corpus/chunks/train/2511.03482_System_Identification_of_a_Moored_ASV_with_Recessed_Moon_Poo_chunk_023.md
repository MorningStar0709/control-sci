$$
\mathbf {S} = \left[ \begin{array}{c c c c} \mathbf {x} _ {j - 1} & \mathbf {x} _ {j} & \dots & \mathbf {x} _ {m - 2} \\ \mathbf {x} _ {j - 2} & \mathbf {x} _ {j - 1} & \dots & \mathbf {x} _ {m - 3} \\ \vdots & \vdots & \vdots & \vdots \\ \mathbf {x} _ {j - s - 1} & \mathbf {x} _ {j - s} & \dots & \mathbf {x} _ {m - s - 1} \end{array} \right], \tag {10}

The augmented matrix $\widehat { \mathbf { G } } = [ \widehat { \mathbf { A } } \widehat { \mathbf { B } } ] \in \mathbb { R } ^ { N ( s + 1 ) \times ( N ( s + 1 ) + Q ( z + 1 ) ) }$ is approximated by HDMDc by solving the b b bfollowing regularized least-square minimization:

$$\min _ {\widehat {A}, \widehat {B}} \quad | | \widehat {\mathbf {X}} ^ {\prime} - [ \widehat {\mathbf {A}} \widehat {\mathbf {B}} ] \widehat {\mathbf {Y}} | | _ {F} ^ {2} + \lambda [ \widehat {\mathbf {A}} \widehat {\mathbf {B}} ] | | _ {F} ^ {2}, \tag {13}$$

which solution is given by:

$$[ \widehat {\mathbf {A}} \widehat {\mathbf {B}} ] = \widehat {\mathbf {X}} ^ {\prime} \widehat {\mathbf {Y}} ^ {\mathrm{T}} \left[ \widehat {\mathbf {Y}} \widehat {\mathbf {Y}} ^ {\mathrm{T}} + \lambda \mathbf {I} \right] ^ {- 1}. \tag {14}$$
