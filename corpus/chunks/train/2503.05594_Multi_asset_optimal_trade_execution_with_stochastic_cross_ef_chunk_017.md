# 2. The multi-asset trade execution problem for finite-variation strategies

Throughout the whole article, we let $T \in ( 0 , \infty )$ be the time horizon, $n \in \mathbb { N }$ the number of assets, $x \in \mathbb { R } ^ { n }$ the initial position, $d \in \mathbb { R } ^ { n }$ the initial price deviation, and $m \in \mathbb { N }$ . We fix a filtered probability space $( \Omega , \mathcal { F } _ { T } , ( \mathcal { F } _ { s } ) _ { s \in [ 0 , T ] } , P )$ that satisfies the usual conditions and supports an m-dimensional Brownian motion $W = ( W _ { 1 } , \dots , W _ { m } ) ^ { \top }$ .
