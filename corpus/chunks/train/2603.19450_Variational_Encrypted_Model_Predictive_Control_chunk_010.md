# B. Exponential Tilting and Efficient Encrypted Sampling

The estimator (12) involves evaluating the extended-value cost ${ \mathcal { I } } ( U ; x _ { 0 } )$ for K trajectory samples. Homomorphically computing this trajectory cost online is expensive because evaluating the quadratic term $U ^ { \top } H U$ , for example, requires at least N m calls of $\otimes$ for encrypted matrix-vector multiplication with the diagonal-based method of [18, Section 4.3].

In what follows, we propose a method to circumvent the computation of this expensive quadratic term. This is achieved by applying a closed-form transformation to the reference sampling distribution, allowing it to naturally absorb the quadratic term. Consequently, the resulting estimator can be evaluated without costly encrypted matrix-vector multiplications, leading to a highly efficient online protocol.

To facilitate the closed-form transformation, we set the reference distribution $\kappa _ { 0 }$ to be a zero-mean Gaussian with covariance $\Sigma _ { 0 } \succ 0$

$$\kappa_ {0} = \mathcal {N} (0, \Sigma_ {0}). \tag {13}$$

The zero-mean assumption is adopted on $\kappa _ { 0 }$ for simplicity but the same derivation extends to a general Gaussian $\mathcal { N } ( m _ { 0 } , \Sigma _ { 0 } )$ .

Theorem 1: The exponentially tilted distribution defined by $\begin{array} { r } { \tilde { \kappa } ( d U ) \ \propto \ \exp ( - J _ { Q } ( U ; x _ { 0 } ) / \lambda ) \kappa _ { 0 } ( d U ) } \end{array}$ is a Gaussian distribution $\tilde { \kappa } = \mathcal { N } ( m _ { U } ( x _ { 0 } ) , \Sigma _ { U } )$ with

$$m _ {U} (x _ {0}) := - \frac {1}{\lambda} \Sigma_ {U} S ^ {\top} x _ {0}, \tag {14a}\Sigma_ {U} := \left(\Sigma_ {0} ^ {- 1} + \frac {1}{\lambda} H\right) ^ {- 1}. \tag {14b}$$

Proof: Multiplying exp $\dot { \mathopen { } \mathclose \bgroup \left( - J _ { Q } ( U ; x _ { 0 } ) / \lambda \aftergroup \egroup \right) }$ with the Gaussian density $\kappa _ { 0 } = { \mathcal { N } } ( 0 , \Sigma _ { 0 } )$ ) yields an exponent of the form

$$- \frac {J _ {Q} (U ; x _ {0})}{\lambda} - \frac {1}{2} U ^ {\top} \Sigma_ {0} ^ {- 1} U= - \frac {1}{2} U ^ {\top} \left(\Sigma_ {0} ^ {- 1} + \frac {1}{\lambda} H\right) U - \frac {1}{\lambda} (S ^ {\top} x _ {0}) ^ {\top} U.$$

Completing the square for U reveals that this exponent corresponds to $\mathcal { N } ( m _ { U } ( x _ { 0 } ) , \Sigma _ { U } )$ , with all terms independent of U absorbed into the normalization constant. This concludes the proof.
