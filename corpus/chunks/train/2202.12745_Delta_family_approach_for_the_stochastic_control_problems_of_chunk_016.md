# 3.1 Optimal investment under the Heston model

Consider the Heston model:

$$\frac {d S _ {t}}{S _ {t}} = (r + \lambda \mathcal {V} _ {t}) d t + \sqrt {\mathcal {V} _ {t}} d W _ {t} ^ {(1)}, \tag {3.1}d \mathcal {V} _ {t} = \kappa (\theta - \mathcal {V} _ {t}) d t + \sigma \sqrt {\mathcal {V} _ {t}} d W _ {t} ^ {(2)}, \tag {3.2}$$

where $E [ d W _ { t } ^ { ( 1 ) } d W _ { t } ^ { ( 2 ) } ] = \rho d t$ . By definition, the wealth process is given by

$$\frac {d X _ {t}}{X _ {t}} = \pi_ {t} \frac {d S _ {t}}{S _ {t}} + (1 - \pi_ {t}) r d t = \lambda \pi_ {t} \mathcal {V} _ {t} d t + \pi_ {t} \sqrt {\mathcal {V} _ {t}} d W _ {t} ^ {(1)}, \tag {3.3}$$

where V is given by (3.2). The infinitesimal generator of (3.3) is given by

$$\mathcal {L} ^ {\pi} f (x, v) = \lambda \pi v x \frac {\partial f}{\partial x} + \kappa (\theta - v) \frac {\partial f}{\partial v} + \frac {\pi^ {2} v x ^ {2}}{2} \frac {\partial^ {2} f}{\partial x ^ {2}} + \frac {\sigma^ {2} v}{2} \frac {\partial^ {2} f}{\partial v ^ {2}} + \rho \pi \sigma x v \frac {\partial^ {2} f}{\partial x \partial v}. \tag {3.4}$$

A key observation is that the above infinitesimal generator is still a quadratic function in π. This is very convenient in working out the explicit optimizers, as seen in the onedimensional case. To use the Legendre functions and the Gauss-Legendre quadrature rules, we set $( x , v ) \in [ x _ { \operatorname* { m i n } } , x _ { \operatorname* { m a x } } ] \times [ v _ { \operatorname* { m i n } } , v _ { \operatorname* { m a x } } ]$ and transform it into $( y _ { 1 } , y _ { 2 } ) \in \Omega ^ { 2 } : = [ - 1 , 1 ] ^ { 2 }$ using the following transformation

$$\hat {x} (y _ {1}) = (x _ {\max} - x _ {\min}) (y _ {1} + 1) / 2 + x _ {\min}, \tag {3.5}\hat {v} (y _ {2}) = (v _ {\max} - v _ {\min}) (y _ {2} + 1) / 2 + v _ {\min}, \tag {3.6}\tilde {x} (x) = \frac {2 (x - x _ {\min})}{x _ {\max} - x _ {\min}} - 1, \tag {3.7}\tilde {v} (v) = \frac {2 (v - v _ {\min})}{v _ {\max} - v _ {\min}} - 1. \tag {3.8}$$

By combining (2.19) and (3.4), we obtain that

$$V (t, x, v) \approx \sup _ {\pi \in \mathcal {A}} \sum_ {k _ {1}, k _ {2} = 0} ^ {\infty} \left(\int_ {\Omega^ {2}} V (t + h, \hat {x} (y _ {1}), \hat {v} (y _ {2})) g _ {k _ {1}} ^ {(1)} (y _ {1}) g _ {k _ {2}} ^ {(2)} (y _ {2}) d \mathbf {y}\right)\cdot \left(g _ {k _ {1}} ^ {(1)} (\tilde {x} (x)) g _ {k _ {2}} ^ {(2)} (\tilde {v} (v)) + h \cdot \mathcal {L} ^ {\pi} \left(g _ {k _ {1}} ^ {(1)} (\tilde {x} (x)) g _ {k _ {2}} ^ {(2)} (\tilde {v} (v))\right)\right), \tag {3.9}$$
