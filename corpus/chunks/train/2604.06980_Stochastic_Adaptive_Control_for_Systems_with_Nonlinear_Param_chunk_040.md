$$\sum_ {\tau = 0} ^ {t} \left\| \left(\theta^ {*} - \hat {\theta} _ {\tau}\right) ^ {\top} \phi_ {\tau} \right\| ^ {\gamma} I \left(\bar {\mu} _ {\tau} \geq (1 + \log r _ {\tau}) ^ {1 + \delta}\right) \tag {B.6}= \sum_ {\tau = 0} ^ {t} | \bar {\mu} _ {\tau} h _ {\tau} | ^ {\frac {\gamma}{2}} = o (\bar {r} _ {t}), \mathrm{a.s.}, t \to \infty .$$

Then, from (B.4), we also have

$$
\begin{array}{l} \sum_ {\tau = 0} ^ {t} \| (\theta^ {*} - \hat {\theta} _ {\tau}) ^ {\top} \phi_ {\tau} \| ^ {\gamma} I \left(\bar {\mu} _ {\tau} <   (1 + \log r _ {\tau}) ^ {1 + \delta}\right) \\ = O ((1 + \log r _ {t}) ^ {\frac {(1 + \delta) \gamma}{2}}) = O ((\log t + \log \bar {r} _ {t}) ^ {\frac {(1 + \delta) \gamma}{2}}), \text { a.s. } \tag {B.7} \\ \end{array}
$$

Combining (B.6) and (B.7) gives

$$\sum_ {\tau = 0} ^ {t} \| (\theta^ {*} - \hat {\theta} _ {\tau}) ^ {\top} \phi_ {\tau} \| ^ {\gamma} = o (\bar {r} _ {t}) + O ((\log t) ^ {\frac {(1 + \delta) \gamma}{2}}), \text {a.s.}$$

Using this together with (B.1), we obtain

$$\sum_ {\tau = 0} ^ {t} \| (\theta^ {*} - \bar {\theta}) ^ {\top} \phi_ {\tau} \| ^ {\gamma} = o (\bar {r} _ {t}) + O ((\log t) ^ {\frac {(1 + \delta) \gamma}{2}}), \text { a.s. } \tag {B.8}$$

Additionally, under Assumption (26), the controller mapping satisfies $\lVert \pi _ { \hat { \theta } _ { \tau - 1 } } ( x _ { \tau } ) - \pi _ { \bar { \theta } } ( x _ { \tau } ) \rVert \leq L _ { 1 } \lVert \hat { \theta } _ { \tau - 1 } -$ ${ \bar { \theta } } \| \| x _ { \tau } \| + L _ { 2 }$ , and therefore,

$$\sum_ {\tau = 0} ^ {t} \| \pi_ {\hat {\theta} _ {\tau - 1}} (x _ {\tau}) - \pi_ {\bar {\theta}} (x _ {\tau}) \| ^ {\gamma} = o (\bar {r} _ {t}) + O (1), a. s. \tag {B.9}$$

For each $t \geq 0$ , define $\bar { v } _ { t } = ( A ^ { * } - \bar { A } ) x _ { t } + B ^ { * } \pi _ { \hat { \theta } _ { t - 1 } } ( x _ { t } ) -$ $\bar { B } \pi _ { \bar { \theta } } ( x _ { t } ) + B ^ { * } v _ { t }$ . Then, for each $t \geq 0$ ,

$$\bar {v} _ {t} = [ \theta^ {*} - \bar {\theta} ] ^ {\top} \phi_ {t} + \bar {B} \pi_ {\hat {\theta} _ {t - 1}} (x _ {t}) - \bar {B} \pi_ {\bar {\theta}} (x _ {t}) + \bar {B} v _ {t}.$$

Then, from Assumption 2 and (24), the noise and input sequences satisfy, almost surely, $\textstyle \sum _ { \tau = 1 } ^ { t } \| w _ { \tau } \| ^ { \gamma } = O ( t )$ , and Pt−1 $\smash { \sum _ { \tau = 0 } ^ { t - 1 } \| v _ { \tau } \| ^ { \gamma } = O ( t ) }$ . Combining these with (B.8) and $\overline { { ( \mathrm { B } } . 9 ) }$ , it follows that

$$\sum_ {\tau = 0} ^ {t - 1} \| \bar {v} _ {\tau} \| ^ {\gamma} + \sum_ {\tau = 1} ^ {t} \| w _ {\tau} \| ^ {\gamma} = o (\bar {r} _ {t}) + O (t), a. s. \tag {B.10}$$
