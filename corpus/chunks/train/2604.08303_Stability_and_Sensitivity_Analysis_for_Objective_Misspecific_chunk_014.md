# III. STABILITY CONDITIONS

In order to derive the conditions for closed-loop stability of the multi-agent dynamical system with heterogeneous MPG controllers (7), the following assumptions are made.

Assumption 1. The following hold for the model predictive game with misspecifications described in (5):

(i) The open-loop dynamics (3) are stable, $i . e . , \ \rho ( A ) < 1$ .   
(ii) The coupling constraint set C and the local sets $\mathcal { U } _ { i } ,$ are closed and convex. The set Z is compact and non-empty.   
(iii) The function $g _ { i } ^ { ( j ) } ( \mathbf { x } , \cdot , u _ { - i } )$ is strongly convex and continuously-differentiable, for any fixed $u _ { - i } , \ \forall i \in N$ .   
(iv) The pseudo-gradient $\mathbf { F } ^ { ( j ) } ( \cdot , \mathbf { x } )$ in (6) is $\rho _ { j } { - } s t r o n g l y$ monotone for any fixed x and $\forall \rho _ { j } > 1 , \forall j \in N$ .

These ensure the existence and uniqueness of an equilibrium point for each $S ^ { ( j ) }$ as given by Proposition 2. The following Theorem 1 provides a stability condition for a multi-agent dynamical system consisting of MPG controllers with objective misspecifications.
