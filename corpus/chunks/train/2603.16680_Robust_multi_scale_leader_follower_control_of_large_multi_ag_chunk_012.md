# B. Deconvolution and choice of α(t)

Since $\nu ^ { F L } = f * \rho ^ { L }$ , we now recover a reference density $\bar { \rho } ^ { L }$ for the leaders that produces $\nu ^ { F L }$ as defined in (8). Using the kernel (2), any velocity field v can be deconvolved to retrieve the generating density as

$$\rho (x, t) = \frac {v _ {x} (x , t)}{2} - \frac {1}{2 \ell^ {2}} \int v (x, t) d x + I (t), \tag {20}$$

where $I : \mathbb { R }  \mathbb { R }$ is an arbitrary function of time; see [13, App. B]. Since $\nu ^ { F L }$ is a convex combination of feedforward and feedback contributions and convolution is linear, the generating density takes the form

$$\bar {\rho} ^ {L} (x, t) = [ 1 - \alpha (t) ] \rho^ {F F} (x) + \alpha (t) \rho^ {F B} (x, t) + G (t), \tag {21}$$

where $G : \mathbb { R } _ { \geq 0 }  \mathbb { R }$ is an arbitrary function of time, and $\rho ^ { F F }$ , $\rho ^ { F B }$ are obtained by deconvolving $\nu ^ { F F }$ and $\nu ^ { F B }$ , respectively.

As noted in [13], the deconvolution operation does not guarantee that the obtained densities are physically meaningful (i.e. that they are non negative and that they sum to the mass of available leaders). Instead, to guarantee that the density integrates to the available mass of leaders $M ^ { L }$ , we choose suitable values of α and C. Specifically, it holds that

$$\int_ {\Omega} \bar {\rho} ^ {L} \mathrm{d} \mathbf {x} = [ 1 - \alpha ] M ^ {F F} + \alpha M ^ {F B} + | \Omega | C, \tag {22}$$

where $\begin{array} { r } { M ^ { F B } = \int _ { \Omega } \rho ^ { F B } } \end{array}$ dx and we recall $\begin{array} { r } { M ^ { F F } = \int _ { \Omega } \rho ^ { F F } } \end{array}$ dx. We need to choose α that guarantees

$$[ 1 - \alpha ] M ^ {F F} + \alpha M ^ {F B} (t) + | \Omega | C \leq M ^ {L}. \tag {23}$$

This allows choosing a non-negative C such that $\begin{array} { r } { \int _ { \Omega } \hat { \rho } ^ { L } \mathrm { d } x = } \end{array}$ $M ^ { L }$ . Note that (23) is equivalent to

$$\alpha [ M ^ {F B} - M ^ {F F} ] + | \Omega | C \leq M ^ {L} - M ^ {F F}. \tag {24}$$

Assuming $M ^ { L } - M ^ { F F } > 0 .$ , which is guaranteed under the feasibility conditions reported in [13], we choose

$$
\alpha = \left\{ \begin{array}{l l} \frac {M ^ {L} - M ^ {F F}}{M ^ {F B} (t) - M ^ {F F}} & \text { if   } M ^ {F B} (t) > M ^ {F F} \\ 1, & \text { otherwise } \end{array} \right. \tag {25}
$$

and

$$C = \max \left[ 0, M ^ {L} - M ^ {F F} - \alpha (M ^ {F B} - M ^ {F F}) \right]. \tag {26}$$

This ensures that $\bar { \rho } ^ { L }$ is a physically meaningful density.
