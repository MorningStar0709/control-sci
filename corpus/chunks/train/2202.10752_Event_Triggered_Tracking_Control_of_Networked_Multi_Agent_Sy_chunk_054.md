# B. Robust Distributed Estimation

Consider the following linear plant, which is borrowed from [31] and can be treated as the leader agent,

$$\dot {x} _ {\mathrm{p}} = A x _ {\mathrm{p}} + B w, \tag {63}$$

where $x _ { \mathrm { p } } = ( x _ { \mathrm { p } } ^ { 1 } , x _ { \mathrm { p } } ^ { 2 } , x _ { \mathrm { p } } ^ { 3 } ) \in \mathbb { R } ^ { 3 }$ is the plant state, and w ∈ $\mathbb { R } ^ { 3 }$ is the external disturbance. Assume that the plant (63) has the oscillatory dynamics for $( x _ { \mathsf { p } } ^ { 1 } , x _ { \mathsf { p } } ^ { 2 } ) \in \mathbb { R } ^ { 2 }$ and the trivial dynamics for $x _ { \mathrm { p } } ^ { 3 } \in \mathbb { R }$ . Therefore, we take $A = { \left[ \begin{array} { l l l } { 0 } & { - 1 } & { 0 } \\ { 1 } & { 0 } & { 0 } \\ { 0 } & { 0 } & { 0 } \end{array} \right] }$ and 0 $B = \operatorname { d i a g } \{ 1 , 0 . 5 , - 1 \}$ . The outputs of (63) are $y = ( y _ { \mathrm { p } } ^ { 1 } , \vec { y } _ { \mathrm { p } } ^ { 2 } ) : =$ $( C _ { 1 } x _ { \mathrm { p } } , C _ { 2 } x _ { \mathrm { p } } )$ with $C _ { 1 } = ( 1 , 1 , 0 )$ and $C _ { 2 } = ( 0 , 0 , 1 )$ .

According the outputs of the plant (63), we design two observers, which can be treated as the follower agents and assumed to be all-to-all connected, i.e., $\mathcal { G } _ { \mathrm { o } } ~ = ~ \left\lceil \begin{array} { l l } { 0 } & { 1 } \\ { 1 } & { 0 } \end{array} \right\rceil$ . These two observers are design as the following form $( i = 1 , 2 )$

$$\dot {x} _ {\mathrm{o}} ^ {i} = A x _ {\mathrm{o}} ^ {i} + \vartheta_ {i}, \quad \dot {\vartheta} _ {i} = 0, \quad y _ {\mathrm{o}} ^ {i} = C _ {i} x _ {\mathrm{o}} ^ {i}, \tag {64}$$

with the update of $\vartheta _ { i } \in \mathbb { R } ^ { 3 }$ given by $( k \in \mathcal { N } _ { i } )$

$$\vartheta_ {i} ^ {+} = J _ {i} (y _ {\mathrm{o}} ^ {i} - y _ {\mathrm{p}} ^ {i}) + J _ {i k} (y _ {\mathrm{o}} ^ {k} - y _ {\mathrm{p}} ^ {k}) + \chi_ {i} (x _ {\mathrm{o}} ^ {i} - x _ {\mathrm{o}} ^ {k}). \tag {65}$$

Since $( C _ { 1 } , A )$ and $( C _ { 2 } , A )$ are not detectable, each follower agent cannot estimate the full state of the plant by using an observer without using the information from the other follower agent. Hence, two observers in (64) are allowed to communicate with each other to reconstruct the state $x _ { \mathrm { p } }$ .
