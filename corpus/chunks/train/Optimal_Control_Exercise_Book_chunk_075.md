# Solution:

We can prove this by contradiction. We assume $a < b$ and arbitrarily choose $\begin{array} { r } { \varepsilon = \frac { b - a } { 2 } } \end{array}$ . By hypothesis, we have

$$a + \frac {b - a}{2} \geq b \iff \frac {a}{2} \geq \frac {b}{2} \iff a \geq b \tag {331}$$

which contradicts our assumption. Hence, $a \geq b .$ .
