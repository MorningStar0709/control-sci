$$f _ {\text { power }} := \sum_ {i \in \mathcal {N}} \sum_ {g \in \mathcal {G} _ {i}} \left(c _ {i, g} ^ {2} \left(P _ {i, g} ^ {\text { gen }}\right) ^ {2} + c _ {i, g} ^ {1} P _ {i, g} ^ {\text { gen }} + c _ {i, g} ^ {0}\right) \tag {2}$$

where $c _ { i , g } ^ { 2 } , c _ { i , g } ^ { 1 } , c _ { i , g } ^ { 0 }$ c2i,g , c1i,g , c0i,g are cost parameters of the gth generator at bus i.

2) Physical Constraints: The physical constraints include the generator’s generation limits, described by:

$$\underline {{{P}}} _ {i, g} \leq P _ {i, g} \leq \overline {{{P}}} _ {i, g}, \quad \forall i \in \mathcal {N}, g \in \mathcal {G} \tag {3}$$

where $\underline { { P } } _ { i , g }$ and $\overline { { P } } _ { i , g }$ denote the lower and upper bounds of the gth generator at bus i.

The voltage constraint can be expressed as:

$$\underline {{{v}}} V _ {0} \leq V _ {i} \leq \overline {{{v}}} V _ {0}, \quad \forall i \in \mathcal {N} \tag {4}$$

which requires that the voltage magnitudes of all buses must be constrained within the range of $[ \underline { { v } } V _ { 0 } , \bar { v } V _ { 0 } ]$ ], v and v represent the lower and upper bounds, respectively.

3) Physical Power System Operation: The physical side power system optimization problem is formulated as:

$$\min _ {P _ {i, g} ^ {\text { gen }}, Q _ {i, g} ^ {\text { gen }}} f _ {\text { power }} \tag {5a}\text { s. t. } \quad (1), (3), (4). \tag {5b}$$
