# 2.2 System Transformation

In this section, we transform the Euler-Lagrangian system (1) to a linearized model by defining two auxiliary variables $\zeta ( q ) = M ( q ) q$ and $\xi ( q , \dot { q } ) = M ( q ) \dot { q }$ . In previous work, $\xi ( q , \dot { q } )$ is also referred to as the general momentum [30]. For brevity, we represent these variables as the time-dependent form ζ(t) and ξ(t). From Assumption 1, we know that $\xi ( t )$ is bounded, i.e.,

$$\| \xi (t) \| \leq \sup _ {q \in \mathbb {R} ^ {n}} \| M (q) \| _ {2} \sup \| \dot {q} \| = \sigma_ {M} \alpha_ {1},$$

where $\sigma _ { M } \in \mathbb { R } ^ { + }$ is the spectral radius of $M ( q )$ , for which the boundedness of the inertia matrix $M ( q )$ is applied (See [35], Page 134, Property 6.1). Notice that ζ(t) is measurable since it depends only on position measurement $q ( t )$ . Then, the time-derivatives of ζ(t) and ξ(t) are

$$\dot {\zeta} = \dot {\hat {M}} (q) q + \hat {M} (q) \dot {q}, \dot {\xi} = \dot {\hat {M}} (q) \dot {q} + \hat {M} (q) \ddot {q},$$

where $\begin{array} { r } { \dot { \hat { M } } ( q ) = \frac { \partial \hat { M } ( q ) } { \partial q } \dot { q } } \end{array}$ is the time derivative of the identified inertia matrix $\hat { M } ( q )$ , and $\dot { \xi } ( t )$ further leads to

$$
\begin{array}{l} \dot {\xi} = \dot {\hat {M}} (q) \dot {q} + M (q) \ddot {q} + \tilde {M} (q) \ddot {q} \\ = \dot {\hat {M}} (q) \bar {\dot {q}} + \tau + \tau_ {\mathrm{d}} - \hat {N} (q, \bar {\dot {q}}) \bar {\dot {q}} - \hat {G} (q) - \hat {F} (\bar {\dot {q}}) + \eta (q, \dot {q}, \bar {\dot {q}}), \\ \end{array}
$$

where the uncertainty term η reads

$$\eta (q, \dot {q}, \bar {\dot {q}}) = \left(\frac {\partial \hat {M} (q)}{\partial q} \dot {q}\right) \varepsilon_ {\dot {q}} + \tilde {M} (q) \ddot {q} - \tilde {N} (q, \dot {q}, \bar {\dot {q}}) - \tilde {G} (q) - \tilde {F} (\dot {\hat {q}}).$$

For brevity, we represent it as $\eta ( t )$ . From Assumption 1 and 2 we know that $\eta ( t )$ is also bounded, i.e., $\| \eta ( t ) \| \le \epsilon _ { \eta }$ , where

$$\epsilon_ {\eta} = \epsilon_ {\dot {q}} \alpha_ {1} \sup _ {q \in \mathbb {R} ^ {n}} \| \partial \hat {M} (q) / \partial q \| _ {2} + \epsilon_ {M} \alpha_ {2} + \epsilon_ {N} + \epsilon_ {G} + \epsilon_ {F}.$$

Thus, we formulate system (1) as the following linear form,

$$
\begin{array}{l} \dot {x} (t) = A x (t) + u (t) + E d (t) \\ \zeta (t) = C x (t), \end{array} \tag {2}
$$
