# B. Decentralized Event-Triggered Mechanisms

With Assumptions 4-6, we next show how to design the ETM for each network. For this purpose, the function $\Gamma _ { i }$ in (13) is defined as a mapping from $\bar { \mathbb { R } } ^ { n _ { z } ^ { i ^ { - } } } \times \mathbb { R } ^ { n _ { e } ^ { i } } \times \mathbb { R } ^ { n _ { e } ^ { i } } \times \mathbb { N } \times \{ 0 , 1 \}$ to $\mathbb { R }$ , and given explicitly by

$$
\begin{array}{l} \Gamma_ {i} (z _ {i}, e _ {i}, m _ {i}, \kappa_ {i}, b _ {i}) := (1 - 2 b _ {i}) \gamma_ {i b _ {i}} W _ {i} ^ {2} (e _ {i}, m _ {i}, \kappa_ {i}, b _ {i}) \\ - (1 - b _ {i}) \rho_ {i} \bar {\lambda} _ {i} \varphi_ {i b _ {i}} (z _ {i}), \tag {32} \\ \end{array}
$$

where $W _ { i }$ and $\varphi _ { i b _ { i } }$ are from in Assumptions 4 and $^ { 6 , }$ respectively. $\rho _ { i } \in [ 0 , \bar { \rho } _ { i } )$ , and

$$\bar {\lambda} _ {i} := \max \left\{\lambda_ {i}, \frac {\rho_ {i} \gamma_ {i 0}}{1 - \rho_ {i} \bar {L} _ {i 0}} \right\}, \tag {33}
\bar {\rho} _ {i} := \left\{ \begin{array}{l l} 1, & \bar {L} _ {i 0} \leq - \gamma_ {i 0}, \\ \min \{1, (\bar {L} _ {i 0} + \gamma_ {i 0}) ^ {- 1} \}, & \bar {L} _ {i 0} > - \gamma_ {i 0}, \end{array} \right. \tag {34}
$$

with $\lambda _ { i }$ from Assumption 4 and $\gamma _ { i 0 } , \bar { L } _ { i 0 }$ from Assumption 6.
