# 2.2 Preconditioned stochastic gradient Langevin dynamics

We consider preconditioned stochastic gradient algorithms i.e. for $( P _ { n } )$ a preconditioner rule the update reads

$$\theta_ {n + 1} = \theta_ {n} - \gamma_ {n + 1} P _ {n + 1} \cdot g _ {n + 1} \tag {2.9}$$

where $g _ { n + 1 }$ is defined in (2.8). We use the Adam [KB15], the RMSprop [TH12] and the Adadelta [Zei12] preconditioners, which are detailed in Algorithms 1, 2 and 3 respectively. For some algorithm name, the corresponding Langevin algorithm denoted L-name reads

$$\theta_ {n + 1} = \theta_ {n} - \gamma_ {n + 1} P _ {n + 1} \cdot g _ {n + 1} + \sigma_ {n + 1} \sqrt {\gamma_ {n + 1}} \mathcal {N} (0, P _ {n + 1}) \tag {2.10}$$

where $\left( \sigma _ { n } \right)$ is a constant or non-decreasing sequence controlling the amount of injected noise.
