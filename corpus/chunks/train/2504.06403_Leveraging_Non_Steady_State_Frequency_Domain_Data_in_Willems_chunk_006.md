Observe that Definition 1 exploits the symmetry of $S ( \hat { \omega } _ { k } ^ { M } )$ by including also $F _ { L } ^ { * } ( S _ { [ 1 , M - 1 ] } )$ in $\Psi _ { L } \big ( S _ { [ 0 , M - 1 ] } \big )$ . Consequently, using M frequencies, we can achieve up to (2M − $1 ) / n _ { s }$ orders of PE, i.e., $L \leqslant ( 2 M - 1 ) / n _ { s }$ .

Remark 1. As detailed in [23], the conjugate symmetry in $\Psi _ { L } \big ( S _ { [ 0 , M - 1 ] } \big )$ allows it to be transformed to the real-valued matrix

$$\left[ \operatorname{Re} (F _ {L} (S _ {[ 0, M - 1 ]})) \quad \operatorname{Im} (F _ {L} (S _ {[ 1, M - 1 ]})) \right],$$

which can be beneficial for certain numerical solvers. Hence, Definition 1 and other results in the sequel can be equivalently formulated in terms of real-valued matrices.
