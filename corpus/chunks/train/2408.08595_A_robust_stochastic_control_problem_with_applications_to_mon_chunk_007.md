For any process $\eta \in L _ { \mathbb { F } } ^ { 0 } \left( 0 , T ; \mathbb { R } ^ { n } \right)$ such that

$$\Lambda_ {t} ^ {\eta} := \mathcal {E} \left(\int_ {0} ^ {t} \eta^ {\prime} d W\right) \tag {2.2}$$

is a martingale, define Pη by

$$\left. \frac {d \mathbb {P} ^ {\eta}}{d \mathbb {P}} \right| _ {\mathcal {F} _ {t}} = \Lambda_ {t} ^ {\eta}.$$

By Girsanov’s theorem,

$$W _ {t} ^ {\eta} := W _ {t} - \int_ {0} ^ {t} \eta_ {s} d s \tag {2.3}$$

is a Brownian motion under Pη. Denote

$$\mathcal {A} := \left\{\eta \in L _ {\mathbb {F}} ^ {0} (0, T; \mathbb {R} ^ {n}) \mid \mathbb {E} \left[ \Lambda_ {t} ^ {\eta} \right] = 1 \text { and } \mathbb {E} \left[ \left(\Lambda_ {t} ^ {\eta}\right) ^ {2} \right] < \infty \text { for all } t \in [ 0, T ] \right\}.$$

Let us now state our robust control problem as follows:

$$\sup _ {u \in \Pi} \inf _ {\eta \in \mathcal {A}} \mathbb {E} ^ {\mathbb {P} ^ {\eta}} \left[ X _ {T} ^ {u} + \frac {1}{2 \theta} \left(\Lambda_ {T} ^ {\eta} - 1\right) \right], \tag {2.4}$$

where θ is a given positive constant. The robust control problem is well-defined, according to the following lemma which give the solvability of state equation (2.1).

Lemma 2.1 Under assumption (A), for any $u \in L _ { \mathbb { F } , \mathbb { P } } ^ { 2 } ( 0 , T ; \mathbb { R } ^ { n } )$ , the equation (2.1) admits a unique solution $X \in L _ { \mathbb { F } , \mathbb { P } } ^ { 2 } ( \Omega ; C ( [ 0 , T ] ; \mathbb { R } ) )$ .

$$
\left\{ \begin{array}{l} d X _ {t} = \left(A _ {t} \bar {X} _ {t} + u _ {t} ^ {\prime} B _ {t}\right) d t + \left(\bar {X} _ {t} C _ {t} ^ {\prime} + u _ {t} ^ {\prime} D _ {t}\right) d W _ {t}, \\ X _ {0} = x. \end{array} \right. \tag {2.5}
$$

From [1, Lemma 1.6], we know $A _ { t } \bar { X } _ { t } \in L _ { \mathbb { F } . \mathbb { P } } ^ { 2 } \left( \Omega ; L ^ { 1 } ( 0 , T ; \mathbb { R } ) \right)$ .
