Similar to (11), the governing equations of the hybrid FESexoskeleton dynamic allocator can now be expressed as

$$\boldsymbol {\tau} ^ {+} = \bar {\boldsymbol {\tau}} ^ {+} + \boldsymbol {g} _ {\perp} ^ {* +} S ^ {+} \boldsymbol {\zeta} ^ {+}, \tag {32a}\dot {\zeta} ^ {+} = \phi^ {+} (\zeta^ {+}, \bar {\tau} ^ {+}, \eta^ {+}), \tag {32b}
\text { s.t. } \quad \left\{ \begin{array}{l} \tau^ {m _ {f _ {i}}} \in \mathbb {A} _ {m _ {f _ {i}}}, \\ \tau^ {m _ {e _ {i}}} \in \mathbb {A} _ {m _ {e _ {i}}}, \\ \tau^ {E} \in \mathbb {A} _ {E}, \end{array} \right.
$$

where $\dot { \zeta ^ { + } } = \phi ^ { + } ( . )$ is input-to-state stable with respect to ${ { \bar { \tau } } ^ { + } } ( t )$ and

$$
\boldsymbol {g} _ {\perp} ^ {* +} = \left[ \begin{array}{c} \mathbf {I} _ {n _ {m}} \\ - \mathbf {1} _ {n _ {m}} ^ {\top} \end{array} \right] \in \mathbb {R} ^ {(n _ {m} + 1) \times n _ {m}},
$$

$n _ { m } = n _ { f } + n _ { e }$ denotes the total number of muscle groups and

$$
S ^ {+} = S \left[ \begin{array}{c c} \mathbf {1} _ {n _ {f}} ^ {\top} & \mathbf {0} _ {n _ {e}} ^ {\top} \\ \mathbf {0} _ {n _ {f}} ^ {\top} & \mathbf {1} _ {n _ {e}} ^ {\top} \end{array} \right], \text {and} \boldsymbol {\zeta} ^ {+} = [ \zeta_ {1} ^ {+}, \zeta_ {2} ^ {+}, \ldots , \zeta_ {n _ {m}} ^ {+} ] ^ {\top} \in \mathbb {R} ^ {n _ {m}}
$$

is the dynamic input allocator, with $l ^ { + } = \mathrm { r a n k } ( \pmb { g } _ { 1 } ^ { * + } ) = n _ { m }$ . Moreover, $\mathbf { I } _ { \rho } \in \mathbb { R } ^ { \rho \times \rho }$ and ${ \mathbf 0 } _ { \rho } \in \mathbb { R } ^ { \rho }$ denote the identity matrix and the column vector of zeros of appropriate dimension $\rho ,$ respectively.

Similar to (16) and Theorem 2, it is straightforward to show that the allocation dynamics

$$\dot {\boldsymbol {\zeta}} ^ {+} = - K ^ {+} S ^ {+} \boldsymbol {g} _ {\perp} ^ {* + ^ {\top}} W ^ {+} \boldsymbol {\tau} ^ {+}, \tag {33}$$

is a valid candidate for the dynamic allocation structure in (32). Here, $K ^ { + } \ = \ \mathrm { d i a g } ( [ k _ { 1 } ^ { + } , \ldots , k _ { n _ { m } } ^ { + } ] ) \ \in \ \mathbb { R } ^ { n _ { m } \times n _ { m } }$ and $W ^ { + } = \mathrm { d i a g } ( [ w _ { 1 } ^ { + } , \ldots , w _ { n _ { m } + 1 } ^ { + } ] ) ^ { - } \in \mathbb { R } ^ { ( n _ { m } + 1 ) \times ( n _ { m } + 1 ) }$ are positive diagonal matrices that determine the convergence speed and distribution ratio of the dynamic allocation control.

Remark 11 (Limitation of Dynamic Allocation within Muscle Groups). Although dynamic allocation provides a theoretical solution for addressing input redundancy within muscle groups II-A4, its practical implementation would require isolated observation of individual muscle responses, which remains impractical with current experimental techniques. Nevertheless, this formulation, (32) and (33), provides valuable insight into redundancy within muscle groups and a principled framework for resolving redundancy in musculoskeletal models (e.g., in OpenSim and MuJoCo). □
