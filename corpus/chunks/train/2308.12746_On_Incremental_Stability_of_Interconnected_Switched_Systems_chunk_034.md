# 6.2 Cascade Connection

Consider the special case of a generalised feedback interconnection, i.e. cascaded interconnection, which is representative of many applications. Only the simulation results are presented, as the theoretical results can be directly derived from the feedback connection. The example considered is

$$
\mathcal {G}: \left\{ \begin{array}{l l} \dot {\xi} _ {1} = - 4 \xi_ {1} - 9 \xi_ {1} ^ {2} + u _ {1} & \text { for   } i = 1, \text {   when   } \xi_ {1} > 0, \\ \dot {\xi} _ {1} = - 4 \xi_ {1} + 9 \xi_ {1} ^ {2} + u _ {1} & \text { for   } i = 2, \text {   when   } \xi_ {1} <   0. \end{array} \right. \tag {50}

\mathcal {H}: \left\{ \begin{array}{l l} \dot {\xi_ {2}} = - 8 \xi_ {2} - 3 \xi_ {2} ^ {2} + 4 \xi_ {1} + u _ {2} & \text { for } i = 1, \text { when } \xi_ {2} > 0, \\ \dot {\xi_ {2}} = - 8 \xi_ {2} + 3 \xi_ {2} ^ {2} + 4 \xi_ {1} + u _ {2} & \text { for } i = 2, \text { when } \xi_ {2} <   0, \end{array} \right. \tag {51}
$$
