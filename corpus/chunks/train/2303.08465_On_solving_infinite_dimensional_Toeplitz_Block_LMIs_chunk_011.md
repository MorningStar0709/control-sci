Now, we define what we call a TBLMI.

Definition 3: A TBLMI is defined by:

$$\mathcal {L} (\mathcal {P}; \mathcal {A} _ {s}, s \in \mathbb {S}) < 0 \tag {12}$$

where $\mathcal { P }$ is the unknown TB operator, $\mathcal { A } _ { s } , s \in \mathbb { S }$ are given TB operators and S is a finite set of subscripts. Both $\cdot$ and $\mathcal { L } ( \mathcal { P } ; \mathcal { A } _ { s } , s \in \mathbb { S } )$ are assumed to be bounded operator on $\cdot$ . For instance, in (10), we have two given operators $\mathcal { A } _ { 1 } ~ = ~ \mathcal { A }$ and $A _ { 2 } = \mathcal { N }$ .

Assumption 1: All the entries $\mathcal { A } _ { s } , \ s \ \in \ \mathbb { S }$ are bounded operators on $\ell ^ { 2 }$ (equivalently $A _ { s } \in L ^ { \infty }$ where $\mathcal { A } _ { s } = \mathcal { T } ( A _ { s } ) )$ , except $\mathcal { N }$ given by (5) which is not. $\mathcal { N }$ appears with the following TB form: $\mathcal { N } ^ { \ast } \mathcal { P } + \mathcal { P } \mathcal { N }$ which corresponds to the Toeplitz transformation ${ \mathcal T } ( \dot { P } ) = - \mathcal N ^ { * } \mathcal P - \mathcal P \mathcal N$ where $\mathcal { P } \ =$ $\cdot$ and where $P$ is a T −periodic and absolutely continuous matrix function [2].

The problem we tackle in this paper is to determine a solution of the following Convex Optimisation Problem (COP):

$$\mathbf {C O P}: \min _ {\mathcal {P} ^ {*} = \mathcal {P} > 0} t r (\mathcal {P}) \text { subject to: }\mathcal {L} (\mathcal {P}; \mathcal {A} _ {s}, s \in \mathbb {S}) \leq 0$$

We assume that this convex optimization problem is feasible and that the optimal solution is unique, bounded on $\cdot$ and continuous with respect to the entries $\cdot$ .
