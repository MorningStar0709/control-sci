# B. State-based event-triggered mechanism

In the S-ETM approach, the measurement error depends on the agent’s state, given by

$$e _ {i} (t) = x _ {i} (t _ {k} ^ {i}) - x _ {i} (t), \tag {8}$$

where $t _ { k } ^ { i }$ denotes the k-th triggering event of agent i and $x _ { i } ( t _ { k } ^ { i } )$ is the state of agent i at the triggering event $t _ { k } ^ { i } .$ Now, let $\ddot { \boldsymbol { A } } = 0 , \boldsymbol { B } = 1 , m = 1$ , and $n = 1$ . Then, the triggering condition is given by [2]

$$(e _ {i} (t)) ^ {2} \geq \eta_ {i} (\sum_ {j \in N _ {i} ^ {I}} (x _ {i} (t) - x _ {j} (t))) ^ {2}), \tag {9}$$

where $0 < \eta _ { i } < 1$ . The condition (9) is only based on the < η <relative information of i-th agent’s neighbors. In this case, the control protocol for each agent is

$$u _ {i} (t) = - K \sum_ {j \in N _ {i} ^ {I}} (x _ {i} (t _ {k} ^ {i}) - x _ {j} (t _ {k ^ {\prime}} ^ {j})), \quad t \in [ t _ {k} ^ {i}, t _ {k + 1} ^ {i}), \tag {10}$$

where $\textbf { \textit { K } } \in \mathbb { \mathbb { R } } ^ { m \times n }$ is the feedback control gain matrix, and K is designed such that $A \ - \ \lambda _ { i } B K , \ \forall i \ = \ 2 , . . . , N$ become Hurwitz where $\lambda _ { i } , \ \forall i \ = \ 2 , . . . , N$ , ...,are the nonzero λ , ...,eigenvalues of the graph Laplacian matrix $\mathcal { L } .$ Moreover, $\begin{array} { r } { k ^ { ' } : = \arg \operatorname* { m i n } _ { l \in \mathbb { N } : t \geq t _ { l } ^ { j } } \Big \{ t - t _ { l } ^ { j } \Big \} , ~ t _ { k ^ { \prime } } ^ { j } } \end{array}$ denotes the k0-th triggering event of agent $j$ and $x _ { j } ( t _ { k ^ { \prime } } ^ { j } )$ is the state of agent $j$ at the triggering event $t _ { k ^ { \prime } } ^ { j }$ .
