# A. Multi-agent systems

Consider a heterogeneous MAS consisting of a set of N interacting agents $\mathcal { V } = \{ 1 , \ldots , N \}$ . Denote with x and u the global state and input of the system and suppose each agent $i \in \nu$ evolves as:

$$\dot {x} _ {i} (t) = f _ {i} (x _ {i} (t)) + g _ {i} (u _ {i} (t)) + w _ {i} (\boldsymbol {x}, t), \tag {1}$$

where $x _ { i } \in \mathbb { R } ^ { n _ { i } }$ and $u _ { i } \in \mathbb { R } ^ { m }$ i are the state and input of agent i, respectively, $f _ { i } : \mathbb { R } ^ { n _ { i } }  \mathbb { R } ^ { n _ { i } }$ is the flow drift, $g _ { i } :$ $\mathbb { R } ^ { m _ { i } } \to \mathbb { R } ^ { n _ { i } }$ is an input function, and $w _ { i } : \mathbb { X } _ { i = 1 } ^ { N } \mathbb { R } ^ { n _ { i } } \times$ $\mathbb { R } _ { > 0 } \to \mathbb { R } ^ { n _ { \mathrm { : } } }$ i represents external disturbances acting on i.

$\begin{array} { r } { n = \sum _ { i = 1 } ^ { N } n _ { i } } \end{array}$ and $\begin{array} { r } { m = \sum _ { i = 1 } ^ { N } m _ { i } } \end{array}$ N be the dimensions of the global state and input. Then, $\begin{array} { r } { \pmb { x } = \left[ x _ { 1 } ^ { \top } , \ldots , x _ { N } ^ { \top } \right] ^ { \top } \in \mathbb { R } ^ { n } } \end{array}$ and $\pmb { u } = \left[ u _ { 1 } ^ { \top } , \ldots , u _ { N } ^ { \top } \right] ^ { \top } \in \mathbb { R } ^ { m }$ .

Assumption $\mathbf { 1 } \colon \mathrm { ( i ) ~ } \ \overline { { f } } _ { i } : \mathbb { R } ^ { n _ { i } } \to \mathbb { R } ^ { n _ { i } }$ is locally Lipschitz; (ii) $g _ { i } : \mathbb { R } ^ { m _ { i } }  \mathbb { R } ^ { n _ { i } }$ is measurable and essentially locally bounded; (iii) $w _ { i } : \texttimes _ { i = 1 } ^ { N } \mathbb { R } ^ { n _ { i } } \times \mathbb { R } _ { \ge 0 }  \mathbb { R } ^ { n _ { i } }$ is continuous and uniformly bounded in Ni=1 $\mathsf { X } _ { i = 1 } ^ { N } \mathbb { R } ^ { n _ { i } } \times \mathbb { R } _ { \ge 0 }$ .

Assumption 2: One of the following holds: $( \mathrm { i } ) ~ g _ { i }$ is bounded; (ii) ${ \dot { g } } _ { i }$ is bounded.

Remark 1: To ensure convergence of the input observer in Section V without requiring Assumption $2 \mathrm { - } ( \mathrm { i } )$ to hold, $g _ { i } ( u _ { i } )$ in (1) is assumed independent of $x _ { i }$ . Yet, as stated in Remark 7, under Assumption $2 \mathrm { - } ( \mathrm { i } ) , g _ { i }$ can be extended to $g _ { i } ( x _ { i } , u _ { i } )$ while preserving the state observer convergence.
