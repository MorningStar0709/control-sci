$$\mathbf {H} _ {\left(\mathrm{N}, \mathrm{H}, y _ {1}\right)} = \underbrace {\mathbf {W} \boldsymbol {\Sigma} ^ {\frac {1}{2}} \mathbf {P}} _ {\mathbf {O} _ {\mathrm{N}}} \underbrace {\mathbf {P} ^ {- 1} \boldsymbol {\Sigma} ^ {\frac {1}{2}} \mathbf {V} ^ {\top}} _ {\mathbf {C} _ {\mathrm{H}}} \approx \underbrace {\mathbf {W} _ {\mathrm{s}} \boldsymbol {\Sigma} _ {\mathrm{s}} ^ {\frac {1}{2}} \mathbf {P} _ {\mathrm{s}}} _ {\mathbf {O} _ {\mathrm{N} _ {s}}} \underbrace {\mathbf {P} _ {\mathrm{s}} ^ {- 1} \boldsymbol {\Sigma} _ {\mathrm{s}} ^ {\frac {1}{2}} \mathbf {V} _ {\mathrm{s}} ^ {\top}} _ {\mathbf {C} _ {\mathrm{H} _ {s}}} \tag {40}$$

where $\mathbf { W } _ { \mathrm { s } } = \mathbf { W } _ { ( : , 1 : s ) } , \mathbf { V } _ { \mathrm { s } } = \mathbf { V } _ { ( : , 1 : s ) } , \boldsymbol { \Sigma } _ { \mathrm { s } } = \boldsymbol { \Sigma } _ { ( 1 : s , 1 : s ) } , \mathbf { P } _ { \mathrm { s } } = \mathbf { P } _ { ( 1 : s , 1 : s ) }$ , and $\mathbf { O } _ { \mathrm { N } _ { s } } , \mathbf { C } _ { \mathrm { H } _ { i } }$ s are rank−s approximations to extended controllability and observability matrices. Further, from the definition of ON in Eq. (36) it can be observed that:

$$\mathbf {O} _ {\mathrm{N} _ {s (2: \mathrm{N},:)}} = \mathbf {O} _ {\mathrm{N} _ {s (1: \mathrm{N} - 1,:)}} \mathbf {A} _ {\mathrm{s}} \tag {41}$$

which can be solved for $s ^ { t h }$ order system matrix. This gives the system parameters for an $s ^ { t h }$ order state-space realization as:

$$\mathbf {A} _ {\mathrm{s}} = \mathbf {O} _ {\mathrm{N} _ {s (1: \mathrm{N} - 1,:)}} ^ {\dagger} \mathbf {O} _ {\mathrm{N} _ {s (2: \mathrm{N},:)}} \quad \mathbf {B} _ {\mathrm{s}} = \mathbf {C} _ {\mathrm{H} _ {s (:, 1)}} \quad \mathbf {C} _ {\mathrm{s}} = \mathbf {O} _ {\mathrm{N} _ {s (1,:)}}. \tag {42}$$

Now, using the ${ \bf A } _ { \mathrm { s } } , { \bf B } _ { \mathrm { s } } , { \bf C } _ { \mathrm { s } }$ matrices, output or state-based LMPC can be designed as in Section 2.1 with $\mathbf { A } = \mathbf { A } _ { \mathrm { s } }$ , $\mathbf { B } = \mathbf { B } _ { \mathrm { s } } , \bar { \mathbf { C } } = \mathbf { C } _ { \mathrm { s } }$ . The steps for the D-LMPC based on the Ho-Kalman-Kung algorithm are summarized below.

Algorithm 1 : Ho-Kalman-Kung based D-LMPC   
1: Require the impulse response data $y_{0}, y_{1}, \ldots, y_{D}$ .
2: Select N and H such that $N + H - 1 \leq D$ .
3: Compute $\mathbf{H}_{(\mathrm{N},\mathrm{H},y_{1})}$ as in Eq. (36).
4: Find SVD of $\mathbf{H}_{(\mathrm{N},\mathrm{H},y_{1})}$ as in Eq. (37)
5: Find an approximate model order s as in Eq. (39).
6: Compute $O_{N_{s}} = and C_{H_{s}} = as in Eq. (40)$ .
7: Use $O_{N_{s}}$ and $C_{H_{s}}$ to find $A_{s}$ , $B_{s}$ , and $C_{s}$ as in Eq. (42).
8: Use $A_{s}$ , $B_{s}$ , and $C_{s}$ to implement output or state based LMPC as in Section 2.1.
