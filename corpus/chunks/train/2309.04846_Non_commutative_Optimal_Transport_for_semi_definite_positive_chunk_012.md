$$A \geq \varepsilon \int_ {X \times Y} e ^ {\frac {u + v - c}{\varepsilon}} \mathrm{d} \mu \otimes \nu = \varepsilon \int_ {X} e ^ {\frac {u}{\varepsilon}} \left[ \int_ {Y} e ^ {\frac {v - c}{\varepsilon}} \mathrm{d} \nu \right] \mathrm{d} \mu = \varepsilon \int_ {X} e ^ {\frac {u}{\varepsilon}} \nu (Y) \left[ \int_ {Y} e ^ {\frac {v - c}{\varepsilon}} \mathrm{d} \sigma \right] \mathrm{d} \mu .$$

Then by the monotonicity and convexity of the exponential, see that

$$A \geq \varepsilon \nu (Y) \int_ {X} e ^ {\frac {u}{\varepsilon}} \left[ \int_ {Y} e ^ {\frac {v - | | c | | _ {\infty}}{\varepsilon}} \mathrm{d} \sigma \right] \mathrm{d} \mu \geq \varepsilon \nu (Y) \int_ {X} e ^ {\frac {u}{\varepsilon}} \exp \left(\frac {\int_ {Y} v \mathrm{d} \sigma - | | c | | _ {\infty}}{\varepsilon}\right) \mathrm{d} \mu ,$$

and applying (2.16) to the latter expression, by monotonicity we obtain

$$A \geq \varepsilon \nu (Y) \int_ {X} e ^ {\frac {u}{\varepsilon}} \exp \left(\frac {\omega_ {2} - | | c | | _ {\infty}}{\varepsilon}\right) \mathrm{d} \mu ,$$

or, equivalently,

$$\int_ {X} e ^ {\frac {u}{\varepsilon}} \mathrm{d} \mu \leq \frac {A}{\varepsilon \nu (Y)} e ^ {- \frac {\omega_ {2} - | | c | | _ {\infty}}{\varepsilon}}. \tag {2.17}$$

By symmetry, we also get similar bounds for v

$$\int_ {Y} e ^ {\frac {v}{\varepsilon}} \mathrm{d} \nu \leq \frac {A}{\varepsilon \mu (X)} e ^ {- \frac {\omega_ {1} - | | c | | _ {\infty}}{\varepsilon}}. \tag {2.18}$$

Finally, we can now find the upper and lower bounds for the integral parts of $u ^ { ( c , \tau _ { 2 } , { \varepsilon } ) }$ and

$\boldsymbol { v } ^ { ( c , \tau _ { 1 } , \varepsilon ) }$ that are independent of u and v.

Using (2.17), see that

$$\int_ {X} e ^ {\frac {u - c}{\varepsilon}} \mathrm{d} \mu \leq \int_ {X} e ^ {\frac {u + | | c | | _ {\infty}}{\varepsilon}} \mathrm{d} \mu \leq \frac {A}{\varepsilon \nu (Y)} \exp \left(- \frac {\omega_ {2} - 2 | | c | | _ {\infty}}{\varepsilon}\right).$$

Conversely, applying Jensen’s inequality once more, along with (2.15)

$$\int_ {X} e ^ {\frac {u - c}{\varepsilon}} \mathrm{d} \mu \geq \mu (X) \int_ {X} e ^ {\frac {u - | | c | | _ {\infty}}{\varepsilon}} \mathrm{d} \rho \geq \mu (X) \exp \left(\frac {\int_ {X} u \mathrm{d} \rho - | | c | | _ {\infty}}{\varepsilon}\right) \geq \mu (X) \exp \left(\frac {\omega_ {1} - | | c | | _ {\infty}}{\varepsilon}\right).$$

By identical computations for v, we can finally estimate the bounds $\mu$ and $\nu { - } \mathrm { a . e . }$ .
