Figure 2. A virtual camera with configurable parameters is used to render images $\left( I _ { 0 t } \right)$ from scenes under dynamic illumination. In forward mode (standard mode for training and inference), the render is adjusted by dynamic parameters predicted by the ACC algorithm and augmented with physically realistic noise to produce the adjusted image $\left( I _ { t } \right)$ , which is then evaluated by the perception task and used to predict dynamic parameters for the next frame. During training, camera parameters $\Phi _ { \mathrm { c a m } }$ are optimised by GA using a task fitness function $F _ { \mathrm { t a s k } }$ (blue arrow), while task model parameters $\Phi _ { \mathrm { t a s k } }$ are updated by gradient descent on the task loss $l _ { \mathrm { t a s k } }$ (red arrow). The ACC network parameters $\Phi _ { \mathrm { A C C } }$ are trained using the proposed DF-Grad method, which combines the task loss $l _ { \mathrm { t a s k } }$ with a GA supervision loss $l _ { \mathrm { G A } }$ . The latter measures the difference between ACC predictions and GA-perturbed parameters. The combined loss is used to update $\Phi _ { \mathrm { A C C } }$ via a gradient-based method. In perturbation mode (only for perturbation optimisation), the render is adjusted with GA-perturbed parameters and evaluated in the same way as in forward mode.
