The function Area Search starts with the solution found by Path Search and increases $K _ { \mathrm { I } }$ according to Eq. 17 in the beginning. Subsequently, the function searches the whole rectangle until $K _ { \mathrm { P } }$ reaches zero and $K _ { \mathrm { I } }$ reaches $K _ { \mathrm { I } } ^ { \mathrm { I n i t } }$ . If Area Search finds new solutions, the solutions are saved. After the search is finished, the solution with the lowest rise time $t _ { \mathrm { r } }$ is taken as the output of Area Search. If Area Search does not find any new solution, the output of Area Search is the solution of Path Search.

After $K _ { \mathrm { I } }$ and $K _ { \mathrm { P } }$ are determined for the combination n, the main function calls the sub-function Find W to find a set of scaling factors $W _ { n }$ for each combination $n$ . For this purpose, the closed-loop transfer function $G _ { a } ^ { \mathrm { C L } } ( s )$ is determined for each DER $a \in A _ { n }$ according to Eq. 6 to Eq. 14, whereby Eq. 9 can be ignored. In Eq. 11, the previously determined controller parameters $K _ { \mathrm { P } , n }$ and $K _ { \mathrm { I } , n }$ are used. Afterward, the function checks if $G _ { a } ^ { \mathrm { C L } } ( s )$ fulfills the disk margin and overshooting requirements. If the requirements are not met, $w _ { a }$ is reduced stepwise by 0.1 % of $\bar { w } _ { a } ^ { \mathrm { I n i t } }$ . The individual scaling factors $w _ { a }$ reduce the influence of DERs, which are not conducive to the robustness of the control loop. In this way, the robustness of the control loop is further increased.
