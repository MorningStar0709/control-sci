# 1.1 Model formulation

Let $W = \{ W _ { t } : t \geq 0 \}$ be a k-dimensional standard Brownian motion and let $I ^ { ( \ell ) } = \{ I _ { t } ^ { ( \ell ) }$ : $t \geq 0 \}$ , with $\ell \in \mathbb { M } .$ , be a continuous-time Markov chain with finite state space I and generator matrix $Q _ { \ell } = ( q _ { \ell } ( \iota , \kappa ) ) _ { \iota , \kappa \in \mathbb { I } } , \mathrm { i . e . }$ .,

$$
\mathbb {P} \left[ I _ {t + \Delta t} ^ {(\ell)} = \kappa \mid I _ {t} ^ {(\ell)} = \iota , I _ {s} ^ {(\ell)}, s \leq t \right] = \left\{ \begin{array}{l l} q _ {\ell} (\iota , \kappa) \Delta t + o (\Delta t), & \text { if } \kappa \neq \iota , \\ 1 + q _ {\ell} (\iota , \kappa) \Delta t + o (\Delta t) & \text { if } \kappa = \iota . \end{array} \right. \tag {1.1}
$$

The entries of the generator matrix $Q _ { \ell }$ satisfy

$$q _ {\ell} (\iota , \kappa) \geq 0 \text {for} \iota , \kappa \in \mathbb {I}, \text {with} \kappa \neq \iota ,q _ {\ell} (\iota , \iota) = - \sum_ {\kappa \in \mathbb {I} \backslash \{\iota \}} q _ {\ell} (\iota , \kappa) \text {for} \iota \in \mathbb {I}.$$
