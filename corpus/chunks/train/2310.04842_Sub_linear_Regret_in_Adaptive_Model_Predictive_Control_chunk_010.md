where the matrix $T \in \mathbb { R } ^ { d _ { \alpha } \times d _ { x } }$ is chosen such that, for some $\lambda \in [ 0 , 1 )$ , the set $\{ x : T x \leq 1 \}$ is λ-contractive with respect to the system $x _ { t + 1 } = \Phi ( \theta ) x _ { t }$ for all $\theta \in \Theta _ { 0 }$ . This property is needed to ensure the robust positive invariance of $X _ { N \mid t }$ (see Lemma 5.7 in (Kouvaritakis & Cannon, 2016)). To derive the associated linear constraints, we apply a standard result to ensure inclusion of polyhedra (see Proposition 3.31 in (Blanchini & Miani, 2008)). More precisely, for any $j = 1 , \ldots , m$ , we have $X _ { k \mid t } \subseteq \{ x :$ $\Phi ( \theta _ { t } ^ { ( j ) } ) x + B ( \theta _ { t } ^ { ( j ) } ) v _ { k | t } + w \in X _ { k + 1 | t } \}$ for all $w \in \bar { \mathcal { W } } _ { t }$ if there exists $H _ { t } ^ { ( j ) } \geq 0$ such that:

$$H _ {t} ^ {(j)} T = T \Phi (\theta_ {t} ^ {(j)}),H _ {t} ^ {(j)} \alpha_ {k | t} + T B (\theta_ {t} ^ {(j)}) v _ {k | t} + \bar {w} _ {t} \leq \alpha_ {k + 1 | t},$$

where $\bar { w } _ { t }$ is such that $\begin{array} { r } { [ \bar { w } _ { t } ] _ { i } ~ = ~ \operatorname* { m a x } _ { w \in \bar { \mathcal { W } } _ { t } } [ T w ] _ { i } } \end{array}$ for $i \in$ $\{ 1 , \ldots , d _ { \alpha } \}$ . Similarly, we have $X _ { k | t } \subseteq \{ x : ( F + G K ) x +$ $G ( v _ { k | t } + \zeta ) \leq \mathbf { 1 } \}$ for all $\zeta \in \mathcal { Z } _ { t }$ , if there exists $H _ { c } \geq 0$ , such that:

$$H _ {c} T = F + G KH _ {c} \alpha_ {k | t} + G v _ {k | t} + \bar {\zeta} _ {t} \leq \mathbf {1},$$

where $\bar { \zeta } _ { t }$ is such that $\begin{array} { r } { [ \bar { \zeta } _ { t } ] _ { i } ~ = ~ \operatorname* { m a x } _ { \zeta \in \mathcal { Z } _ { t } } [ G \zeta ] _ { i } } \end{array}$ for $i \in$ $\{ 1 , \ldots , d _ { c } \}$ .

Remark 3.2. For simplicity, we have chosen $\bar { w } _ { t }$ and $\bar { \zeta } _ { t }$ so as to be conservative with respect to the noise we apply. However, notice that we could well pre-sample the noise sequence $( \zeta _ { t } ) _ { t \geq 0 }$ and use it as part of our predictions.

The matrices $H _ { t } ^ { ( j ) }$ , and $H _ { c }$ are chosen such that for all $i \in \{ 1 , \ldots , d _ { \alpha } \}$ and for all $j \in \{ 1 , \ldots , m \}$ ,

$$\left(H _ {t} ^ {(j)}\right) _ {i} = \arg \min _ {h} \{\mathbf {1} ^ {\top} h: h ^ {\top} T = (T) _ {i} \Phi (\theta_ {t} ^ {(j)}), h \geq 0 \},$$

$$\left(H _ {c}\right) _ {i} = \arg \min _ {h} \{\mathbf {1} ^ {\top} h: h ^ {\top} T = (F + G K) _ {i}, h \geq 0 \}.$$

Finally, we have the terminal conditions

$$H _ {t} ^ {(j)} \alpha_ {N} + \bar {w} _ {t} \leq \alpha_ {N},H _ {c} \alpha_ {N} + \bar {\zeta} _ {t} \leq \mathbf {1}.$$
