The simulation shown in Fig. 6.3 represents a very complex behavior. Although essentially the same phenomenon repeats itself, the solution is not periodic. This is seen more clearly if the system is simulated for a longer time. Figure 6.4 shows a phase plane when the simulation time is extended to 10,000 time units. The solution is very irregular. There is, however, some pattern in the motion, as is indicated in the figure. For example, the state moves close to the curve given by Eq. (6.12) for part of the motion. The behavior shown is in fact an example of chaotic behavior. The pattern shown in Fig. 6.4 is called a strange attractor.
