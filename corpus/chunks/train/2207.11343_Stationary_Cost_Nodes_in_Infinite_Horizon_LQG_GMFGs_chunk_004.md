For non-uniform networks, Graphon Mean Field Games model asymptotic limits of population games in the double limit, $n \to \infty$ and mi $1 _ { l = 1 : n } | C _ { l } |  \infty$ (observe this implies that the number of agents, denoted by $\begin{array} { r } { N = \sum _ { l = 1 } ^ { n } \mathsf { \bar { | } } C _ { l } \vert } \end{array}$ , goes to infinity).

We assume that the sequence of dense graphs (characterizing the networks) denoted by $\{ ( g _ { i , j } ^ { n } ) _ { i , j = 1 : n } \} _ { n = } ^ { \infty }$ 1 converges, in the cut-metric (see Lovasz (2012)), to a unique limit graphon denoted

$$
\begin{array}{l} g: [ 0, 1 ] \times [ 0, 1 ] \to [ 0, 1 ] \\ (\alpha , \beta) \mapsto g (\alpha , \beta). \\ \end{array}
$$

Graphons are bounded symmetric Lebesgue measurable functions $g : [ 0 , 1 ] \times [ 0 , 1 ] \stackrel { \left. } { \right. } [ 0 , 1 ]$ which can be interpreted as specifying weighted links on the set of nodes [0, 1] (see Lovasz (2012)).

With the network interaction within a cluster being uniform, we deduce that in the infinite cluster size case, at any graphon node $\alpha \in [ 0 , 1 ]$ , there exists a representative (or typical) agent, denoted $A _ { \alpha } ,$ whose state’s evolution is given by the SDE, $\forall t \in [ 0 , \infty )$ ,

$$d x _ {t} ^ {\alpha} = b u _ {t} ^ {\alpha} d t + \sigma d w _ {t} ^ {\alpha}, \quad x _ {0} ^ {\alpha} \sim \mathcal {N} (m ^ {\alpha}, \nu^ {2}). \tag {4}$$

Each representative agent Aα aims at minimizing an infinite horizon quadratic cost given by

$$J (u ^ {\alpha}, z ^ {\alpha}) := \mathbb {E} \int_ {0} ^ {\infty} e ^ {- \rho t} \big [ r (u _ {t} ^ {\alpha}) ^ {2} + \big (x _ {t} ^ {\alpha} - z _ {t} ^ {\alpha} \big) ^ {2} \big ] d t, \tag {5}$$

where $r , \rho > 0$ and, the global mean field denoted by $z _ { t } ^ { \alpha }$ , are given by,

$$z _ {t} ^ {\alpha} := \int_ {0} ^ {1} g (\alpha , \beta) \mathbb {E} [ x _ {t} ^ {\beta} ] d \beta , \forall t \in [ 0, \infty), \forall \alpha \in [ 0, 1 ]. \tag {6}$$
