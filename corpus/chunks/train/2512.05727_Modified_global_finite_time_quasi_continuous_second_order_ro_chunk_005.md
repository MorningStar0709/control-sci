# 3. UNPERTURBED BEHAVIOR

In the following, we analyze first the behavior of an unperturbed control system (1)–(3), i.e. for $d = 0$ , and that separately for I. and III. and then for II. and IV. quadrants. More rigorously, we define the quadrants as

$$x \in \mathrm{I.} \cup \mathrm{III.} \equiv U = \left\{x \in \mathbb {R} ^ {2}: x _ {1} x _ {2} \geq 0 \right\}, \tag {4}x \in \text { II. } \cup \text { IV. } \equiv C = \{x \in \mathbb {R} ^ {2}: x _ {1} x _ {2} < 0 \}, \tag {5}$$

respectively, where U refers to an undamped subset and C to a convergence subset of trajectories in the state-space. Recall that due to ${ \dot { x } } _ { 1 } = x _ { 2 }$ and, thus, clockwise evolvement of the state trajectories, the convergence towards zero equilibrium appears only when $x ( t ) \in C ,$ cf. Fig. 1. Also it is worth noting that zero equilibrium itself belongs to $U _ { ; }$ , so that a control action with zero damping, cf. (3), appears also at $x ( t ) = ( 0 , 0 )$ , namely in the sliding-mode. Respectively, no control singularities can occur at x2-axis if $x _ { 2 } \neq 0$ , cf. Ruderman and Efimov (2025). For $U _ { : }$ , we provide the analytic solutions in section 3.1. For $C ,$ the analytic solution (section 3.2) is obtained for the subset

$$C _ {a} = \left\{x \in C: x _ {2} ^ {2} < 2 \gamma | x _ {1} | \right\} \tag {6}$$

of initial conditions, see Fig. 2 (cyan shadowed regions).

Due to an obvious symmetry of the trajectory solutions and, therefore, without loss of generality, the solutions will be developed below in the I. quadrant for U and in the IV. quadrant for $C _ { a } ,$ , cf. the dashed lines in Fig. 1.

Further we note that since a transition from $U$ to $C _ { a }$ is possible at $x _ { 2 } = 0$ only, no control jumps appear when switching δ in the control law (2), (3). Respectively, the state trajectories and so the control value remain smooth outside zero equilibrium.
