# 3.1. Systems with Imperfect State Information

We work with problems where a hidden state $\mathbf { z } _ { t } \in \mathbb { R } ^ { N _ { \mathbf { z } } }$ changes over time according to some transition function f given controls $\mathbf { u } _ { t } \in \mathbb { R } ^ { N _ { \mathbf { u } } }$ and process noise $\mathbf { w } _ { t }$ . The hidden state can only be estimated through observations $\mathbf { x } _ { t } \in \mathbb { R } ^ { N _ { \mathbf { x } } }$ that are produced by an emission function g subject to measurement noise variables $\mathbf { v } _ { t } .$ . The process and the observation noise might be state and control dependent with known distributions and dependency. Using some control ut in some state $\mathbf { z } _ { t }$ leads to a cost $\mathbf { c } _ { t } \in \mathbb { R } ^ { + }$ which is determined by the cost function c. This setup can be summarised:

$$\mathbf {z} _ {t + 1} = f \left(\mathbf {z} _ {t}, \mathbf {u} _ {t}, \mathbf {w} _ {t}\right) \quad \mathbf {c} _ {t} = c \left(\mathbf {z} _ {t}, \mathbf {u} _ {t}\right) \quad \mathbf {x} _ {t} = g \left(\mathbf {z} _ {t}, \mathbf {v} _ {t}\right), \tag {1}$$

with initial state distribution $p ( \mathbf { z } _ { 1 } )$ . The controls are given by a stationary policy $\mathbf { u } _ { t } = \pi ( \mathbf { x } _ { 1 : t } , \mathbf { u } _ { 1 : t - 1 } )$ that is conditioned on all previous observations and controls. The expected total cost of a policy π is:

$$J ^ {\pi} (\mathbf {z}) = \mathbb {E} \Big [ \sum_ {t = 1} ^ {\infty} \beta^ {t - 1} c (\mathbf {z} _ {t}, \mathbf {u} _ {t}) | \mathbf {z} _ {1} = \mathbf {z} \Big ],$$

where $\beta \in ( 0 , 1 )$ . An optimal policy attains minimal expected cost, i.e. $\pi ^ { \star } = \arg \operatorname* { m i n } _ { \pi } \mathbb { E } \left[ J ^ { \pi } ( \mathbf { z } ) \right]$ .
