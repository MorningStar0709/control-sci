# 3.1 LMI Error Bounds

A positive definite error bound matrix is developed for a MI constraint applied to an n−simplex. Enforcing the MI constraint plus its error bound on the vertex points of an n-simplex $( \sigma = \cos \{ x _ { j } \} _ { j = 0 } ^ { n } )$ implies that the MI holds for all points within that simplex $( x \in \sigma )$ . While [7] established Lemma 9 as a general error bound for $\mathcal { C } ^ { 2 }$ vector-valued functions, it does not translate automatically to a MI that contains $\mathcal { C } ^ { 2 }$ vector-valued functions because the structure of a MI affects its definiteness.
