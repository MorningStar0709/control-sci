Proposition 1 (First-order necessary condition for constrained optimality): Suppose that f is a ${ \mathcal { C } } ^ { 1 }$ (continuously differentiable) function and $x ^ { * }$ is its local minimum. Then,

$$\langle \nabla f (x ^ {*}), d \rangle \geq 0 \tag {5}$$

for all feasible directions d.
