Since a and b have the same integral,

$$\int_ {X} d a = \| a ^ {+} \| - \| a ^ {-} \| = \| b ^ {+} \| - \| b ^ {-} \| = \int_ {X} d b,$$

and the same mass,

$$\int_ {X} d \alpha = \| a ^ {+} \| + \| a ^ {-} \| = \| b ^ {+} \| + \| b ^ {-} \| = \int_ {X} d \beta ,$$

$$\| a ^ {+} \| = \| b ^ {+} \| \mathrm{and} \| a ^ {-} \| = \| b ^ {-} \|.$$

Since K is a Markov kernel, we have $\| a ^ { + } K \| ~ = ~ \| a ^ { + } \|$ , implying that $\| a ^ { + } K \| ~ = ~ \| b ^ { + } \|$ . Hence $( a K ) ^ { + } \ =$ $( a ^ { + } K ) ^ { + }$ and $b ^ { + } = a ^ { + } K$ , that is, when b is the image of a from the kernel K, then $b ^ { + } = a ^ { + } K$ .
