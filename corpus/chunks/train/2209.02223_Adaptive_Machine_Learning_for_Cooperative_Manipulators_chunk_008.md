Here, I is the identity matrix with adequate dimension, $\eta _ { v }$ is the vector part of the quaternion and $\eta _ { o }$ is the scalar part, i.e., $\pmb { \eta } = [ \pmb { \eta } _ { v } ^ { T } \ \eta _ { o } ] ^ { T }$ . Then, due to the closed-loop kinematics, the velocities of the end-effectors are related to one other by

$$\omega_ {2} = A (\eta) \omega_ {1}, \tag {3a}\boldsymbol {v} _ {2} = \boldsymbol {A} (\boldsymbol {\eta}) \boldsymbol {v} _ {1} - \boldsymbol {\rho} \times \boldsymbol {\omega} _ {2}. \tag {3b}$$

Our estimation strategy is to solve (3) for parameters $\rho$ and $\eta$ such that the residual errors are minimized in the least-squares sense. That is, given the measurements of the linear and angular velocities ${ \pmb v } _ { 1 _ { i } } , { \pmb v } _ { 2 _ { i } } , { \pmb \omega } _ { 1 _ { i } }$ , and $\omega _ { 2 _ { i } }$ at time $t _ { i } .$ , we have

$$
\min _ {\boldsymbol {\eta}, \rho} \frac {1}{2} \sum_ {i = 1} ^ {k} a _ {i} \left\| \left[ \begin{array}{c} \boldsymbol {\epsilon} _ {\omega_ {i}} \\ \boldsymbol {\epsilon} _ {v _ {i}} \end{array} \right] \right\| ^ {2} \tag {4}
\mathrm{s.t.} \| \boldsymbol {\eta} \| = 1
\mathbf {\Lambda} ^ {\mathcal {A}} \boldsymbol {J} = \left[ \begin{array}{c c} \boldsymbol {R} ^ {T} & \boldsymbol {0} \\ \boldsymbol {0} & \boldsymbol {R} ^ {T} \end{array} \right] ^ {\mathcal {B}} \boldsymbol {J},
$$

where R is the rotation matrix representing the end-effector orientation. Here, $\mathbf { \nabla } \cdot \mathbf { \vec { J } } = \mathbf { \nabla } J$ for simplicity of notation.

where

$$\boldsymbol {\epsilon} _ {\omega_ {i}} (\boldsymbol {\eta}) = \boldsymbol {\omega} _ {2 _ {i}} - \boldsymbol {A} (\boldsymbol {\eta}) \boldsymbol {\omega} _ {1 _ {i}} \tag {5a}\boldsymbol {\epsilon} _ {v _ {i}} (\boldsymbol {\eta}, \boldsymbol {\rho}) = \boldsymbol {v} _ {2 _ {i}} - \boldsymbol {A} (\boldsymbol {\eta}) \boldsymbol {v} _ {1 _ {i}} + \boldsymbol {\rho} \times \boldsymbol {\omega} _ {2 _ {i}} \tag {5b}$$

are the residual errors, and $a _ { i }$ are non-negative weights, which can be selected according to the time varying forgetting function

$$a _ {i} = e ^ {- \mu (t _ {k} - t _ {i})} \tag {6}$$
