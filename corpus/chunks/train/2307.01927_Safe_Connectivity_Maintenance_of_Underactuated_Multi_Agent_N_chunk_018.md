# IV. THEORETICAL ANALYSIS

This section analyzes under which conditions our safe interaction controller can maintain connectivity and avoid collisions [26]. Our analysis follows the common approach to demonstrate that a flock converges to a lattice structure while preventing inter-agent collisions using energy-based analysis and LaSalle’s invariance principle [12]. We start by introducing a moving referential frame for the structural collective dynamics [12] with respect to the flock centroid $\pmb q _ { c } .$ . The relative coordinates are given by $\tilde { \pmb q } _ { i } = { \pmb q } _ { i } - { \pmb q } _ { c }$ and $\tilde { \pmb { q } } _ { i j } = \tilde { \pmb { q } } _ { i } - \tilde { \pmb { q } } _ { j } = \pmb { q } _ { i j }$ . Therefore, ψ $\mathbf { \sigma } \cdot ( | | \mathbf { q } _ { i j } | | _ { 2 } ) = \psi ( | | | \tilde { \mathbf { q } } _ { i j } | | _ { 2 } )$ , and the total tension energy or potential energy for the structural dynamics in the relative coordinates yields

$$
H (\tilde {\boldsymbol {q}}) = \frac {1}{2} \sum_ {i = 1} ^ {N} \sum_ {\substack {j = 1 \\ j \neq i}} ^ {N} \psi \left(\left\| \tilde {\boldsymbol {q}} _ {i j} \right\| _ {2}\right) \tag{10}
$$

A possible approach, although conservative, is to show that a global tension energy decrease of the system $\dot { H } =$ $\textstyle \sum _ { i = 1 } ^ { n } { \bar { \dot { H } } } _ { i } \leq 0$ can be achieved by guaranteeing local tension energy decrease $\forall i \in \nu$ . Assume that $G ( t )$ switches at time $t _ { l }$ for $l = 0 , 1 , 2 . .$ . and $\dot { H } \le 0$ on each $[ t _ { l } , t _ { l + 1 } )$ . Then, $H ( t _ { k } ) = H ( t _ { k } ^ { - } ) + m _ { k } \psi ( \| R _ { c o m } - \epsilon \| )$ [23] with $m _ { k }$ the number of edges added at switching time k. The energy can be bound for any subsequent time as the graph topology becomes fixed after a certain time, and only a finite number of maximum edges can be added. The time-derivative of $H _ { i }$ along the trajectory of agent i yields

$$
\dot {H} _ {i} = \dot {\tilde {\boldsymbol {q}}} _ {i} ^ {\top} \sum_ {\substack {j = 1 \\ j \neq i}} ^ {N} \nabla_ {\tilde {\boldsymbol {q}} _ {i}} \psi \left(\left\| \tilde {\boldsymbol {q}} _ {i j} \right\| _ {2}\right) \tag{11}
$$
