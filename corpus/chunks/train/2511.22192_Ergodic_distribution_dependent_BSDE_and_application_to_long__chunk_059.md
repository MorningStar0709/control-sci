# 4 Long-time behaviour of McKean-Vlasov BSDEs

The main goal of this section is to establish long-time behaviour of the solution of the (decoupled) finite horizon BSDE towards the solution to the (decoupled) McKean-Vlasov infinite horizon EBSDE. Let us denote $( Y ^ { \hat { T , } x , [ \theta ] } , Z ^ { T , x , [ \theta ] } )$ the solution of the decoupled McKean-Vlasov finite horizon BSDE, $( Y ^ { T , [ \theta ] } , Z ^ { T , [ \theta ] } )$ the solution of the coupled finite horizon BSDE and $( Y ^ { x , [ \theta ] } , Z ^ { x , [ \hat { \theta ] } } , \lambda )$ the solution to the decoupled EBSDE. Namely, we set:

$$Y _ {t} ^ {T, x, [ \theta ]} = g (X _ {T} ^ {x, [ \theta ]}, [ X _ {T} ^ {[ \theta ]} ]) + \int_ {t} ^ {T} f (X _ {s} ^ {x, [ \theta ]}, [ X _ {s} ^ {[ \theta ]} ], Z _ {s} ^ {T, x, [ \theta ]}) \mathrm{d} s - \int_ {t} ^ {T} Z _ {s} ^ {T, x, [ \theta ]} \mathrm{d} W _ {s}, \quad \forall t \in [ 0, T ], \quad (\text {Decoupled BSDE})Y _ {t} ^ {x, [ \theta ]} = Y _ {T} ^ {x, [ \theta ]} + \int_ {t} ^ {T} \left(f (X _ {s} ^ {x, [ \theta ]}, [ X _ {s} ^ {[ \theta ]} ], Z _ {s} ^ {x, [ \theta ]}) - \lambda\right) \mathrm{d} s - \int_ {t} ^ {T} Z _ {s} ^ {x, [ \theta ]} \mathrm{d} W _ {s}, \quad \forall 0 \leq t \leq T < \infty , \quad (\mathrm{DecoupledEBSDE})$$

where we recall that $X ^ { x , [ \theta ] }$ and $X ^ { [ \theta ] }$ are, respectively, the solutions of (Decoupled SDE) and (MV-SDE) and for $g : \mathbb { R } ^ { d } \times$ $\mathcal { P } _ { 2 q + 2 } \to \mathbb { R } ^ { d }$ being such that

Assumption $( \mathcal { H } _ { g } )$ . There exist $q \geq 0 , c > 0$ and $\epsilon \in ( 0 , 1 ]$ such that, for all $x , x ^ { \prime } \in \mathbb { R } ^ { d } , \theta , \theta ^ { \prime } \in \mathrm { L } ^ { 2 q + 2 } ( \Omega , \mathbb { R } ^ { d } )$ ,

$1 . | g ( x , [ \theta ] ) | \leq c ( 1 + | x | ^ { q + 1 } + \| \theta \| _ { 2 q + 2 } ^ { q + 1 } ) ,$   
$\begin{array} { r l } { { 2 . } } & { | g ( x , [ \theta ] ) - g ( x ^ { \prime } , [ \theta ^ { \prime } ] ) | \leq c ( 1 + | x | ^ { q } + | x ^ { \prime } | ^ { q } + \| \theta \| _ { 2 \sigma + 2 } ^ { q } + \| \theta ^ { \prime } \| _ { 2 \sigma + 2 } ^ { q } ) \left( | x - x ^ { \prime } | ^ { \epsilon } + \mathcal { W } _ { 1 } ( [ \theta ] , [ \theta ^ { \prime } ] ) ^ { \epsilon } \right) . } \end{array}$
