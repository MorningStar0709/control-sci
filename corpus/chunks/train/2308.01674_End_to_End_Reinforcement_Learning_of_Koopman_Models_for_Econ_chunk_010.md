# 2.3 Post-optimal sensitivity analysis of convex problems

Obtaining post-optimal sensitivities of solution mappings of optimization problems has been a topic of study for several decades $( \mathrm { e . g . }$ , Fiacco and Kyparisis (1985); Ralph and Dempe (1995); Agrawal et al. (2019b)). Given a parametric nonlinear program with parameters $\theta ,$ and a solution point $\ b { a } ^ { * }$ for a given $\pmb { \theta } ^ { * }$ , the solution point is differentiable with respect to the parameters in a neighborhood of $\pmb { \theta } ^ { * }$ under the following assumptions (Pistikopoulos et al. (2020); Fiacco and Ishizuka (1990)): (i) The objective function and constraints of the problem are twice continuously differentiable in a. (ii) The gradients of objective function and constraints with respect to a and the constraints are once continuously differentiable in θ. (iii) The second-order sufficient conditions (SOSC) for a local minimum hold at $\ b { a } ^ { * }$ . (iv) The linear independence constraint qualification (LICQ) is satisfied at $\ b { a } ^ { * }$ . (v) Strict complementary slackness (SCS) holds at $\ b { a } ^ { * }$ . Note that the above conditions also imply that the KKT conditions hold (Karush (1939); Boyd et al. (2004)), and that $\ b { a } ^ { * }$ is an isolated local minimum (Fiacco and Ishizuka (1990)). For a convex parametric problem this means that the solution map of the optimization problem becomes single-valued, i.e., the solution map is locally an implicit function of the problem parameters.
