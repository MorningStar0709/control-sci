# Exercise 1.39: Normal distributions under linear transformations

Given the normally distributed random variable, $\boldsymbol { \xi } \in \mathbb { R } ^ { n }$ , consider the random variable, $\eta \in \mathbb { R } ^ { n }$ , obtained by the linear transformation

$$\eta = A \xi$$

in which A is a nonsingular matrix. Using the result on transforming probability densities, show that if $\xi \sim N ( m , P )$ , then $\eta \sim N ( A m , A P A ^ { \prime } )$ . This result basically says that linear transformations of normal random variables are normal.
