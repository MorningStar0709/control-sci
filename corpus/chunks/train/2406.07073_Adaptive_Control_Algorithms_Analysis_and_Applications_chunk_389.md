# 8.5.1 Output Sensitivity Function

Using an RST controller, the output sensitivity function is given by:

$$S _ {y p} (z ^ {- 1}) = \frac {A (z ^ {- 1}) S (z ^ {- 1})}{A (z ^ {- 1}) S (z ^ {- 1}) + z ^ {- d} B (z ^ {- 1}) R (z ^ {- 1})} \tag {8.41}$$

where

$$R (z ^ {- 1}) = H _ {R} (z ^ {- 1}) R ^ {\prime} (z ^ {- 1}) \tag {8.42}S (z ^ {- 1}) = H _ {S} (z ^ {- 1}) S ^ {\prime} (z ^ {- 1}) \tag {8.43}$$

and

$$A (z ^ {- 1}) S (z ^ {- 1}) + z ^ {- d} B (z ^ {- 1}) R (z ^ {- 1}) = P _ {D} (z ^ {- 1}) P _ {F} (z ^ {- 1}) = P (z ^ {- 1}) \tag {8.44}$$

In (8.42) and (8.43), $H _ { R } ( z ^ { - 1 } )$ and $H _ { S } ( z ^ { - 1 } )$ correspond to the pre-specified parts of $R ( z ^ { - 1 } )$ and $S ( z ^ { - 1 } )$ respectively. $S ( z ^ { - 1 } )$ and $R ( \bar { z } ^ { - 1 } )$ are the solutions of (8.44) where $P ( z ^ { - 1 } )$ represents the desired closed-loop poles in pole placement control strategy. The polynomial $P ( z ^ { - 1 } )$ is factorized in order to emphasize the dominant poles defined by $P _ { D } ( z ^ { - 1 } )$ ) and the auxiliary poles defined by $P _ { F } ( z ^ { - 1 } )$ .
