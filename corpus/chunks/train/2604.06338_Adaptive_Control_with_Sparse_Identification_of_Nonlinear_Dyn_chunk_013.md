The closed-loop system is described by a differential inclusion. Since the closed-loop system is locally essentially bounded and measurable, the differential inclusion meets the conditions of [37, Thm. 2.2] and as a result, there exists $\delta > 0$ such that starting from the initial condition $( t _ { s } , z ( t _ { s } ) )$ ), the closed loop admits a maximal absolutely continuous solutions on $[ t _ { s } , t _ { s } + \delta )$ . Let $t \mapsto z ( t )$ be one such solution. Let $\vartheta : [ t _ { s } , t _ { s } + \delta )  \mathbb { R } ^ { p }$ and $\sigma : [ t _ { s } , t _ { s } + \delta )  \mathbb { R } ^ { p }$ be measurable selections that occur due to discontinuous points in the closed loop error system, such that $\varrho ( t ) \in \mathrm { { P r o j } } _ { \Theta } \left( \widehat { \theta } , \phi , \Gamma \right)$ and $\sigma ( t ) \ \in \ \operatorname { S G N } ( { \hat { \theta } } )$ , and $\hat { \theta } ( t ) = \vartheta ( t ) - \lambda \gamma \Gamma \sigma ( t )$ , for almost all $t \in [ t _ { s } , t _ { s } + \delta )$ . Since θ is constant, we have $\dot { \tilde { \theta } } = - \dot { \tilde { \theta } }$ . Whenever the time derivative of $t \mapsto V ( z ( t ) )$ exists, it satisfies

$$
\begin{array}{l} \dot {V} (z (t)) = - e ^ {\top} (t) K e (t) + e ^ {\top} (t) Y (x (t)) \tilde {\theta} (t) \\ - \tilde {\theta} ^ {\top} (t) \Gamma^ {- 1} (\vartheta (t) - \lambda \gamma \Gamma \sigma (t)). \tag {29} \\ \end{array}
$$

Applying the fundamental property of the projection operator, $\bar { \tilde { \theta } } ^ { \top } \bar { \Gamma } ^ { - 1 } \vartheta \leq \tilde { \theta } ^ { \top } \Gamma ^ { - 1 } \phi$ . Substituting the definition of $\phi$ in (11), we have

$$
\begin{array}{l} \tilde {\theta} ^ {\top} \Gamma^ {- 1} \phi = \tilde {\theta} ^ {\top} \Gamma^ {- 1} \Gamma (Y (x) ^ {\top} e + \gamma (\mathcal {U} + \mathcal {Y} \hat {\theta})) \\ = \tilde {\theta} ^ {\top} Y (x) ^ {\top} e + \gamma \tilde {\theta} ^ {\top} (\mathcal {U} - \mathcal {Y} \hat {\theta}). \tag {30} \\ \end{array}
$$

By the design of the ICL memory stack, $\mathcal { U } = \mathcal { V } \theta$ . Consequently, $\mathcal { U } - \mathcal { y } \hat { \theta } = \mathcal { y } ( \theta - \hat { \theta } ) = \mathcal { y } \tilde { \theta }$ . Substituting these relations back into (31) yields

$$\dot {V} (z (t)) \leq - e ^ {\top} (t) K e (t) - \gamma \tilde {\theta} ^ {\top} (t) \mathcal {Y} \tilde {\theta} (t) + \lambda \gamma \tilde {\theta} ^ {\top} (t) \sigma (t), \tag {31}$$
