# V. ROBUST PERFORMANCE ANALYSIS

As is customary in robust control, we begin with the robust performance analysis problem, i.e., with the study of (4) for fixed controller parameters.

We first note that the worst-case disturbance problem (4) can be rewritten as

$$
\underset {\begin{array}{c}w: \mathbb {N} _ {0} \rightarrow L _ {2} (\Omega , \mathbb {R} ^ {n _ {w}}),\\\hat {w}: \mathbb {N} _ {0} \rightarrow L _ {2} (\Omega , \mathbb {R} ^ {n _ {w}})\end{array}} {\text { maximize }} \quad \underset {T \rightarrow \infty} {\limsup} \frac {1}{T} \sum_ {t = 0} ^ {T - 1} \mathbb {E} \| z (t) \| ^ {2}, \tag {5a}

\binom {\chi (t + 1)} {z (t)} = \left( \begin{array}{c c} \mathcal {A} & \mathcal {B} \\ \mathcal {C} & \mathcal {D} \end{array} \right) \binom {\chi (t)} {w (t)}, \tag {5b}
\gamma^ {2} \geq \mathbb {E} \| w (t) - \hat {w} (t) \| ^ {2}, \tag {5c}\mathbb {P} _ {\hat {w} (t)} = \mathbb {P} _ {\text { nom }}, \tag {5d}\chi (0) = 0, \tag {5e}$$

with potentially additional constraints for independence and zero mean assumptions on the random variable w.

Problem (5) introduces an auxiliary stochastic process wˆ with marginal distributions $\mathbb { P } _ { \hat { w } ( t ) } ~ = ~ \mathbb { P } _ { \mathrm { n o m } }$ for all $t ~ \in ~ \mathbb { N } _ { 0 }$ . The purpose of wˆ is to encode the Wasserstein constraint $\mathbb { W } ( \mathbb { P } _ { w ( t ) } , \mathbb { P } _ { \mathrm { n o m } } ) ~ \le ~ \gamma$ in the conditions (5c) and (5d). This reformulation exploits the variational definition of the Wasserstein distance: by requiring $\mathbb { E } \Vert w ( t ) - \hat { w } ( t ) \Vert ^ { 2 } \leq \gamma ^ { 2 }$ , the joint distribution of $( w ( t ) , \hat { w } ( t ) )$ provides an explicit coupling that certifies the Wasserstein bound. Since any feasible coupling yields an upper bound on the Wasserstein distance, this reformulation captures distributions within the Wasserstein ball. This is made precise in the following result.

Theorem 5.1: Consider a fixed controller (2) and a fixed system (4b). Then the problems (4) and (5) are equivalent in the sense that every feasible point w ∈ W of (4) is in one-toone correspondence with a feasible point $( w , \hat { w } )$ of (5) and the objective values of these solutions are equal, respectively. The proof is deferred to Appendix A.
