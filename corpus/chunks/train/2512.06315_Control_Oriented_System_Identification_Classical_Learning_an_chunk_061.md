# 4.2.5. Gaussian Process Models

Nonlinear system identification can also be approached using Gaussian process (GP) regression (Rasmussen and Williams, 2006). The book by Kocijan (2016) presents techniques for learning and validating GP dynamics models from measurement data, for embedding prior knowledge into the GP models, and for designing optimal control, model predictive control, and adaptive control based on the GP dynamics models.

A GP is a stochastic process, that is, a collection of random variables indexed by time or space, such that any finite subset of the random variables has a joint Gaussian distribution.
