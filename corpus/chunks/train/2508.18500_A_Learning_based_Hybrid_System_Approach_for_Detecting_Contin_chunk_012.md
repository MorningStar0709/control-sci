Each encoder layer contains h such attention heads operating in parallel, allowing the model to capture diverse aspects of the input sequence. The outputs of the attention heads are concatenated and passed through another learned linear layer. Residual connections and layer normalization follow both the multi-head attention and feedforward sublayers to promote stable and efficient training. After the final encoder layer, global average pooling is applied across the time dimension to obtain a fixed-length latent feature vector $\bar { \textbf { h } } \in \mathbb { R } ^ { d }$ , where d is the Transformer embedding dimension. This vector summarizes the temporal information from the entire input sequence.

The pooled representation h¯ is then fed into a fully connected output layer that produces the predicted contingency class probabilities:

$$\hat {p} = \operatorname{softmax} (\mathbf {W} _ {\text { class }} \bar {\mathbf {h}} + \mathbf {b} _ {\text { class }}), \tag {13}$$

where $\hat { p } \in \mathbb { R } ^ { N _ { c } }$ represents the probability distribution over $N _ { c }$ scenario classes. The predicted scenario class label αˆ is obtained by selecting the class with the highest probability:

$$\hat {\alpha} = \arg \max _ {i} \hat {p} _ {i}, \tag {14}$$

The model is trained end-to-end using the categorical crossentropy loss, enabling it to learn temporal patterns associated with different contingency types directly from raw SHS state and output data. This provides a powerful framework for realtime contingency classification.
