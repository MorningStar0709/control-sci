# 4. Using Multiple Lyapunov Functions

A standard way of proving stability of an n-dimensional system ${ \dot { x } } \ = \ f ( x )$ , where $f : \mathbb { R } ^ { n }  \mathbb { R } ^ { n }$ , is to find a positive definite ${ \mathcal { C } } ^ { 1 } .$ -function $V : \mathbb { R } ^ { n }  \mathbb { R }$ , known as a Lyapunov function, which decreases along any solution trajectory of the system, that is, $\dot { V } ( \boldsymbol { x } ( t ) ) < 0$ , for each initial condition $\boldsymbol { x } ( 0 ) \in \mathbb { R } ^ { n }$ . The linear system ${ \dot { x } } = A x$ is stable if and only if A is a Hurwitz matrix. For a symmetric positive definite matrix $Q$ , the function $V ( x ) \stackrel { \textstyle \cdot } { = } x ^ { \top } Q x$ is a Lyapunov function for the linear system ${ \dot { x } } = A x$ if and only if $Q A + A ^ { \top } Q$ is negative definite. A known extension of this method to address the stability of switched systems is known as Multiple Lyapunov functions (MLFs) approach, which will be used in this section.

4.1. When all subsystems are stable. For a given switched system, if all the subsystems share a common Lyapunov function, then it serves as a Lyapunov function for the switched system as well, and stability of the switched system under arbitrary switching is guaranteed. Existence of a common Lyapunov function is not always possible. The MLF approach for stability of the switched system (1) is a two-step process:

(1) For each $p \in \mathcal { P }$ , obtain a quadratic Lyapunov function $V _ { p } ( x ) = x ^ { \top } Q _ { p } x$ for each subsystem ${ \dot { x } } \ = \ A _ { p } x$ . For a given signal σ, construct the piecewise Lyapunov function along the trajectory as $V ( x ( t ) ) : = V _ { \sigma ( t ) } ( x ( t ) )$ , for $t \geq 0$ , which switches between to the Lyapunov function $V _ { p }$ in accordance with the active subsystem ${ \dot { x } } = A _ { p } x$ . By construction, V decreases along any solution trajectory in between any two consecutive switching instances, that is, for all $k \geq 1$ and for any $x ( 0 ) \in$ R n , $\dot { V } ( x ( t ) ) < 0$ when $t \in ( t _ { k - 1 } , t _ { k } )$ . However, the function V is, in general, discontinuous, see [20].

(2) Relax the condition of V decreasing along any solution trajectory by $V ( x ( t ) )$ decreasing along the sequence of switching instances $\{ t _ { k } \} _ { k \geq 0 }$ for any initial condition $\boldsymbol { x } ( 0 ) \in \mathbb { R } ^ { n }$ .
