Then $\zeta$ is approximated by a finite sum of Dirac measures $\zeta ^ { \ell } = \textstyle \sum _ { i } ^ { \ell } c _ { i } ^ { \ell } \delta _ { \gamma _ { i } ^ { \ell } }$ with positive weights $( c _ { i } ^ { \ell } ) _ { 1 \leq i \leq \ell }$ and mean reversions $( \gamma _ { i } ^ { \ell } ) _ { 1 \leq i \leq \ell }$ for $\ell \geq 1$ . This in turn yields an approximation of the fractional kernel by a sequence of smoothed kernels $( K ^ { \ell } ) _ { \ell \geq 1 }$ given by

$$K ^ {\ell} (t) = \sum_ {i = 1} ^ {\ell} c _ {i} ^ {\ell} e ^ {- \gamma_ {i} ^ {\ell} t}, \quad \ell \geq 1.$$

Therefore, the stock price process can be well approximated by the multi-factor stochastic volatility model (see Jaber and El Euch (2019), Jaber (2019)) as follows

$$\frac {d S _ {t} ^ {\ell}}{S _ {t} ^ {\ell}} = (r + \lambda \mathcal {V} _ {t} ^ {\ell}) d t + \sqrt {\mathcal {V} _ {t} ^ {\ell}} d W _ {t} ^ {1},$$

and the variance process $\mathcal { V } _ { t } ^ { \ell }$ is given by a fractional square-root process

$$\mathcal {V} _ {t} ^ {\ell} = \xi (t) + \sum_ {i = 1} ^ {\ell} c _ {i} ^ {\ell} \mathcal {V} _ {t} ^ {\ell , i},$$

and

$$d \mathcal {V} _ {t} ^ {\ell , i} = (- \gamma_ {i} ^ {\ell} \mathcal {V} _ {t} ^ {\ell , i} - \kappa \mathcal {V} _ {t} ^ {\ell}) d t + \sigma \sqrt {\mathcal {V} _ {t} ^ {\ell}} d W _ {t} ^ {2},$$

where $\xi ( t ) = \mathcal { V } _ { 0 } + \kappa \theta \int _ { 0 } ^ { t } K ( t - s ) d s$ . These inspire us to approximate the transformed wealth process as follows

$$d X _ {t} ^ {\ell} = X _ {t} ^ {\ell} \left[ (r + \lambda \pi_ {t} \mathcal {V} _ {t} ^ {\ell}) d t + \pi_ {t} \sqrt {\mathcal {V} _ {t} ^ {\ell}} d W _ {t} ^ {1} + c \vartheta q _ {t} d t + b q _ {t} d W _ {t} \right]. \tag {3.17}$$

The utility maximization problem (3.16) is approximated by

$$\sup _ {(q, \pi) \in \Pi^ {\ell}} \mathbb {E} \left[ U (X _ {T} ^ {\ell} - L) \right], \tag {3.18}$$

where $X _ { t } ^ { \ell }$ is subject to (3.17). Therefore, the utility maximization problem (3.18) can be solved by principle of dynamic programming. Define the value function as follows

$$V (t, x, v _ {1}, \ldots , v _ {\ell}) = \sup _ {(q, \pi) \in \Pi^ {\ell}} \mathbb {E} \left[ U \big (X _ {T} ^ {\ell} - L \big) | X _ {t} = x, \mathcal {V} _ {t} ^ {\ell , 1} = v _ {1}, \ldots , \mathcal {V} _ {t} ^ {\ell , \ell} = v _ {\ell} \right],$$

with terminal condition

$$V (T, x, v _ {1}, \dots , v _ {\ell}) = U (x - L).$$
