Fig. 15. Altitude reference tracking simulation of the controller with rotor failure.

Table 3 gives the time response (rise time, overshoot and settling time) values of the controller with rotor failure. When the time response data in Table 3 is examined, it is seen that the rotor error increases the overshoot. However, it is not high enough to prevent the quadrotor from flying stably. The settling time is slightly longer than when there is no rotor error. However, the reference value is captured at around 0.6 seconds for roll, pitch and altitude references. The yaw reference can be captured in about 1 second. The system has a short settling time despite rotor failure.
