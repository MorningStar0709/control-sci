Theorem 1. Consider a density distribution $\rho ( t , { \mathbf { x } } )$ evolving according to the advection–diffusion equation (5) over a domain Ω subject to boundary conditions ensuring mass conservation, that is $\begin{array} { r } { \partial _ { t } \left( \int _ { \Omega } \rho \mathrm { d } \mathbf { x } \right) = 0 . } \end{array}$ . Let $\mathcal { H } ( t , \rho )$ be a continuously differentiable functional defining the safe set $\mathcal { C }$ as in (6), and denote by $\delta _ { \rho }$ H its Frechet derivative ´ with respect to ρ. Let α be a class-K function. Then, if $\rho \left( t _ { 0 } , \mathbf { x } \right) \in \mathcal { C }$ at the initial time $t _ { 0 } ,$ any control input u selected from the set

$$
\begin{array}{l} \mathcal {K} _ {C B F} (t, \rho) = \left\{\mathbf {u}: \int_ {\Omega} \nabla \delta_ {\rho} \mathcal {H} (t, \rho) \cdot \rho \mathbf {w} (t, \mathbf {x}, \mathbf {u}) d \mathbf {x} \geq \right. \\ - D \int_ {\Omega} \Delta \left(\delta_ {\rho} \mathcal {H} (t, \rho)\right) \rho d x \tag {8} \\ \left. \right.\left. - \partial_ {t} \mathcal {H} (t, \rho) - \alpha (\mathcal {H} (t, \rho)) \right\} \\ \end{array}
$$

guarantees forward invariance of the safe set $\mathcal { C } .$

Proof. We compute the total time derivative of the functional $\mathcal { H } ( t , \rho )$ appearing in (7).

Using the chain rule for time-dependent functionals,

$$\frac {\mathrm{d}}{\mathrm{d} t} \mathcal {H} (t, \rho (t, \mathbf {x})) = \int_ {\Omega} \delta_ {\rho} \mathcal {H} (t, \rho) \partial_ {t} \rho (t, \mathbf {x}) \mathrm{d} \mathbf {x} + \partial_ {t} \mathcal {H} (t, \rho). \tag {9}$$

Substituting the dynamics (5) into (9) yields

$$
\begin{array}{l} \frac {d}{d t} \mathcal {H} (t, \rho (t, \mathbf {x})) = - \int_ {\Omega} \delta_ {\rho} \mathcal {H} (t, \rho) \nabla \cdot (\rho \mathbf {w}) d \mathbf {x} \\ + D \int_ {\Omega} \delta_ {\rho} \mathcal {H} (t, \rho) \Delta \rho d x + \partial_ {t} \mathcal {H} (t, \rho). \tag {10} \\ \end{array}
$$

For the advection term in (10), integration by parts together with boundary conditions gives

$$- \int_ {\Omega} \delta_ {\rho} \mathcal {H} \nabla \cdot (\rho \mathbf {w}) d \mathbf {x} = \int_ {\Omega} \nabla \delta_ {\rho} \mathcal {H} \cdot \rho \mathbf {w} d \mathbf {x}. \tag {11}$$
