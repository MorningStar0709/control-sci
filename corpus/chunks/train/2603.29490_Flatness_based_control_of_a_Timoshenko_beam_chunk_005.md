The objective is to design a state feedback controller that stabilizes the transition of the Timoshenko beam (3)–(4) from an initial stationary configuration $( w ( z , t _ { 0 } ) , \varphi ( z , t _ { 0 } ) ) = ( w _ { 0 } ( z ) , \varphi _ { 0 } ( z ) )$ to a desired stationary configuration $( w ( z , t _ { T } ) , \varphi ( z , t _ { T } ) ) \ = \ ( w _ { T } ( z ) , \varphi _ { T } ( z ) )$ in finite time $T = t _ { T } - t _ { 0 } \in \mathbb { R } ^ { + }$ . The control design proceeds in three steps, as illustrated in Figure 1:

1 The HCF for the Timoshenko beam is derived from a flatness-based parametrization of (2)–(4). To this end, a sequence of transformations is applied to the hyperbolic MI system to map it into a form beneficial for this purpose.   
2 For the system in HCF, the subsequent design of a stabilizing tracking controller is straightforward.   
3 The inverse HCF transformation is used to express the control law as a state feedback of the beam variables.
