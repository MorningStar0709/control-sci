Throughout this proof, we assume that the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ holds. First, relating $f _ { t } ( \cdot )$ to $f _ { t ; h } ( \cdot )$ , we know from the upper bound $\lVert \nabla f _ { t } ( N _ { t } ) \rVert \leq l _ { y }$ from Lemma 3.4 that

$$\sum_ {t = h} ^ {T - 1} \left(f _ {t} (N _ {t}) - f _ {t} (N ^ {\star})\right) = \sum_ {t = h} ^ {T - 1} \left(f _ {t; h} (N _ {t}) - f _ {t; h} (N ^ {\star})\right) + \sum_ {t = h} ^ {T - 1} \left((f _ {t} - f _ {t; h}) (N _ {t}) + (f _ {t; h} - f _ {t}) (N ^ {\star})\right),$$

$$
