# 4.4.2 General case

We suppose now cosh $( \lambda t ) , \cosh ( \mu s ) \neq 0$ (we already saw the case cosh(λt), cosh(µs) = 0, which can happen if $\lambda , \mu \in i \mathbb { R } )$ . With this assumption, equation (4.14) becomes

$$\tanh (\mu s) = \frac {\mu}{\lambda} \frac {\operatorname{tr} (A B) - \operatorname{tr} (A ^ {2})}{\operatorname{tr} (A B) - \operatorname{tr} (B ^ {2})} \tanh (\lambda t). \tag {4.26}$$

This equation has always a positive solution s = s(t) if tr(AB)−tr(A2)tr(AB)−tr(B2) $s = s ( t )$ ${ \frac { \operatorname { t r } ( A B ) - \operatorname { t r } ( A ^ { 2 } ) } { \operatorname { t r } ( A B ) - \operatorname { t r } ( B ^ { 2 } ) } } > 0$ . In particular

$$s (t) = \frac {\operatorname{tr} (A B) - \operatorname{tr} (A ^ {2})}{\operatorname{tr} (A B) - \operatorname{tr} (B ^ {2})} t + o (t) \quad \text { as } t \to 0. \tag {4.27}$$

Remark 4.28. The value

$$s ^ {\prime} (0) = \frac {\operatorname{tr} (A B) - \operatorname{tr} (A ^ {2})}{\operatorname{tr} (A B) - \operatorname{tr} (B ^ {2})}$$

is intrinsic and depends only on the equation

$$\frac {\partial \Phi}{\partial t} (t, s (t)) = \frac {\partial \Phi}{\partial s} (t, s (t)).$$

Indeed, if we differenciate this equation with respect to t, we obtain

$$\frac {\partial^ {2} \Phi}{\partial t ^ {2}} (t, s (t)) + \frac {\partial^ {2} \Phi}{\partial t \partial s} (t, s (t)) s ^ {\prime} (t) = \frac {\partial^ {2} \Phi}{\partial t \partial s} (t, s (t)) + \frac {\partial^ {2} \Phi}{\partial s ^ {2}} (t, s (t)) s ^ {\prime} (t).$$

Thus

$$s ^ {\prime} (0) = \frac {\frac {\partial^ {2} \Phi}{\partial t \partial s} (0 , 0) - \frac {\partial^ {2} \Phi}{\partial t ^ {2}} (0 , 0)}{\frac {\partial^ {2} \Phi}{\partial t \partial s} (0 , 0) - \frac {\partial^ {2} \Phi}{\partial s ^ {2}} (0 , 0)} = \frac {\mathrm{tr} (A B) - \mathrm{tr} (A ^ {2})}{\mathrm{tr} (A B) - \mathrm{tr} (B ^ {2})}.$$

So, for every $t > 0$ we obtained an s(t) such that the bang-bang control

$$
u (\tau) = \left\{ \begin{array}{l l} 1 & \text {if} \tau \in [ 0, t), \\ 0 & \text {if} \tau \in [ t, t + s (t)), \end{array} \right.
$$

and extended by periodicity for $\tau \in [ t + s ( t ) , + \infty )$ , satisfies all conditions of Pontryagin Maximum Principle.
