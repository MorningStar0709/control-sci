# 2.1 The Koopman Operator and Nonlinear System $R e \mathrm { - }$ formulation

We start with the unactuated nonlinear system

$$\dot {x} = f (x) \tag {1}$$

where $x \in \mathbb { X } \subseteq \mathbb { R } ^ { n }$ denotes the state and $f ( 0 ) = 0 , { \mathrm { i . e . } }$ , the origin is an equilibrium of the unactuated system. We define a Banach space $\mathcal { F }$ of observables $\varphi : \mathbb { X }  \mathbb { R }$ , and the Koopman operator is defined as follows.

Definition 1 The continuous time Koopman operator $\mathcal { K } ^ { t } : \mathcal { F }  \mathcal { F }$ is defined as

$$(\mathcal {K} ^ {t} \varphi) (x _ {0}) = \varphi \circ S (t, x _ {0}) \tag {2}$$

where ◦ denotes the function composition and $S ( t , x _ { 0 } )$ denotes the flow map (solution) of system (1) at time $t > 0$ with an initial state $x _ { 0 }$ . Furthermore, under $t h e$ assumption that $\varphi ( x )$ is continuously differentiable, it satisfies

$$\frac {\mathrm{d} \varphi (x)}{\mathrm{d} t} = \mathcal {L} _ {f} \varphi \triangleq \lim _ {t \to 0} \frac {(\mathcal {K} ^ {t} \varphi - \varphi)}{t} = \nabla \varphi \cdot f \tag {3}$$

where $\mathcal { L } _ { f }$ is defined as the infinitesimal generator, also equals to the Lie derivative with respect to $f$ .

We note that $\mathcal { K } ^ { t } ( \alpha \varphi _ { 1 } + \beta \varphi _ { 2 } ) = \alpha \mathcal { K } ^ { t } \varphi _ { 1 } + \beta \mathcal { K } ^ { t } \varphi _ { 2 }$ , which indicates the linearity of Koopman operator even if the dynamics is nonlinear.

Definition 2 A Koopman eigenfunction corresponding to the unactuated system (1) is an observable $\phi _ { \lambda } \in { \mathcal { F } }$ such that

$$\mathcal {K} ^ {t} \phi_ {\lambda} = e ^ {\lambda t} \phi_ {\lambda} \tag {4}$$

for some $\lambda \in \mathbb { C }$ , which is the associated Koopman eigenvalue. Additionally with (3), we obtain

$$\mathcal {L} _ {f} \phi_ {\lambda} = \nabla \phi_ {\lambda} \cdot f = \lambda \phi_ {\lambda}. \tag {5}$$

It is known that if $\phi _ { \lambda _ { 1 } }$ and $\phi _ { \lambda _ { 2 } }$ are Koopman eigenfunctions with eigenvalues $\lambda _ { 1 }$ and $\lambda _ { 2 }$ respectively, $\bar { \phi } _ { \lambda _ { 1 } } ^ { k _ { 1 } } \phi _ { \lambda _ { 2 } } ^ { k _ { 2 } }$ k2 is also an eigenfunction with eigenvalue $k _ { 1 } \lambda _ { 1 } + k _ { 2 } \lambda _ { 2 } ^ { \cdots } ,$ which means there are perhaps infinitely many eigenfunctions.
