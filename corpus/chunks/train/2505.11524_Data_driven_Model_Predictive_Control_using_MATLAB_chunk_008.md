# 2.1 Conventional LMPC

Consider a linear time-invariant (LTI) system in discrete-time:

$$
\begin{array}{r l} \mathbf {x} _ {k + 1} & = \mathbf {A x} _ {k} + \mathbf {B u} _ {k} \\ \mathbf {y} _ {k} & = \mathbf {C x} _ {k} \end{array} \tag {5}
$$

where $k \in \mathbb { T }$ is the discrete time instant, $\mathbf { y } _ { k } \in \mathbb { Y } \subseteq \mathbb { R } ^ { p }$ is the output (controlled) vector, $\mathbf { u } _ { k } \in \mathbb { U } \subseteq \mathbb { R } ^ { m }$ is the input (control or manipulated) vector, $\mathbf { x } _ { k } \in \mathbb { X } \subseteq \mathbb { R } ^ { n }$ is the state vector, $\mathbf { \bar { A } } \in \mathbb { R } ^ { n \times n }$ is the system matrix, $\mathbf { B } \in \mathbb { R } ^ { n \times m }$ is the input matrix, and $\mathbf { C } \in \dot { \mathbb { R } } ^ { p \times n }$ is the output matrix. The sets Y, U, and X are the constraint sets for the outputs, inputs, and states, which can be defined as [9]:

$$\mathbb {Y} = \{\mathbf {y} \in \mathbb {R} ^ {p}: \mathbf {F _ {y}} \mathbf {y} \leq \mathbf {g _ {y}} \}, \qquad \mathbf {F _ {y}} \in \mathbb {R} ^ {c _ {y} \times p}, \mathbf {g _ {y}} \in \mathbb {R} ^ {c _ {y}}\mathbb {U} = \left\{\mathbf {u} \in \mathbb {R} ^ {m}: \mathbf {F} _ {\mathbf {u}} \mathbf {u} \leq \mathbf {g} _ {\mathbf {u}} \right\}, \quad \mathbf {F} _ {\mathbf {u}} \in \mathbb {R} ^ {c _ {u} \times m}, \mathbf {g} _ {\mathbf {u}} \in \mathbb {R} ^ {c _ {u}} \tag {6}\mathbb {X} = \left\{\mathbf {x} \in \mathbb {R} ^ {n}: \mathbf {F} _ {\mathbf {x}} \mathbf {x} \leq \mathbf {g} _ {\mathbf {x}} \right\}, \quad \mathbf {F} _ {\mathbf {x}} \in \mathbb {R} ^ {c _ {x} \times n}, \mathbf {g} _ {\mathbf {x}} \in \mathbb {R} ^ {c _ {x}}.$$

For the linear system in Eq. (38), the controllability and observability matrices are defined as:

$$
\mathbf {C} _ {n} = \left[ \begin{array}{l l l l} \mathbf {B} & \mathbf {A B} & \dots & \mathbf {A} ^ {n - 1} \mathbf {B} \end{array} \right] \quad \mathbf {O} _ {n} = \left[ \begin{array}{c} \mathbf {C} \\ \mathbf {C A} \\ \vdots \\ \mathbf {C A} ^ {n - 1} \end{array} \right] \tag {7}
$$

A linear system in Eq. (38) is said to be controllable if the controllability matrix is of full row rank, i.e., rank $( \mathbf { C } _ { n } )$ $= n$ . Similarly, the system is said to be observable if rank $( \mathbf { O } _ { n } ) = n$ . In linear MPC design, we normally assume that the system for which the MPC control law is designed is both controllable and observable.
