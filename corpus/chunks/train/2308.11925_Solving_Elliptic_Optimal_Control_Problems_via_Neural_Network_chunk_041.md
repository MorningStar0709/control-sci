First, we consider the linear elliptic control problem:

$$
\min _ {y, u} \Big \{J (y, u) = \frac {1}{2} \| y - y _ {d} \| _ {L ^ {2} (\Omega)} ^ {2} + \frac {\lambda}{2} \| u \| _ {L ^ {2} (\Omega)} ^ {2} \Big \}, \quad \text {subject to} \left\{ \begin{array}{c l} - \Delta y = f + u, & \text {in} \Omega , \\ y = g, & \text {on} \partial \Omega , \end{array} \right.
$$

where f and g are the known problem data and λ is the penalty weight.

Example 4.1. The domain $\Omega = ( 0 , 1 ) ^ { 4 }$ , and $\lambda = 0 . 0 1$ . The data $f \equiv 0 , y _ { d } ( x ) = ( 1 + 1 6 \lambda \pi ^ { 4 } ) \prod _ { i = 1 } ^ { 4 } \sin ( \pi x _ { i } )$ , and $g = 0$ . The exact state y¯ and control u¯ are given by $\begin{array} { r } { \bar { y } ( x ) = \prod _ { i = 1 } ^ { 4 } \sin ( \pi x _ { i } ) } \end{array}$ , and $\begin{array} { r } { \bar { u } ( x ) = 4 \pi ^ { 2 } \prod _ { i = 1 } ^ { 4 } \sin ( \pi x _ { i } ) } \end{array}$ .
