Corollary 3.4 ([15]). Suppose that $\tilde { \mu } _ { T }$ has a positive density ϕT such that

$$\int_ {\mathbb {R} ^ {d}} (| y | ^ {2} + \log \phi_ {T} (y)) \phi_ {T} (y) d y < \infty .$$

Then we have

$$\sup _ {g \in C _ {b} (\mathbb {R} ^ {d})} \left\{\inf _ {u \in \mathcal {U}} \mathbb {E} \left[ \int_ {0} ^ {T} | u _ {t} | ^ {2} d t + g (X _ {T} ^ {u}) \right] - \langle g, \tilde {\mu} _ {T} \rangle \right\} = \inf \left\{\mathbb {E} \int_ {0} ^ {T} | u _ {t} | ^ {2} d t: X _ {T} ^ {u} \sim \tilde {\mu} _ {T}, u \in \mathcal {U} \right\}.$$

Proof. Let $p ( t , x , s , y )$ be the transition density of d-dimensional Brownian motion. Then, there exists a σ-finite product measure $\nu _ { 0 } ( d x ) \nu _ { T } ( d y )$ such that

$$\pi (E) := \int_ {E} p (0, x, 1, y) \nu_ {0} (d x) \nu_ {T} (d y), \quad E \in \mathcal {B} (\mathbb {R} ^ {d} \times \mathbb {R} ^ {d}),$$

satisfies $\pi ( d x \times \mathbb { R } ^ { d } ) = \tilde { \mu } _ { 0 } ( d x )$ and $\pi ( \mathbb { R } ^ { d } \times d y ) = \tilde { \mu } _ { T } ( d y )$ and

$$H (\pi \mid p (0, x, 1, y) \tilde {\mu} _ {0} (d x) d y) < \infty ,$$

where $H ( \pi | R )$ denotes the relative entropy of π with respect to R. See $\mathrm { e . g . }$ , Theorem 2.1 in Nutz [20]. Then, by Theorem 2 in Jamison [9], there exists a weak solution of

$$d X _ {t} ^ {*} = \nabla h (t, X _ {t} ^ {*}) d t + d B _ {t}, \quad X _ {0} ^ {*} \sim \tilde {\mu} _ {0},$$

where

$$h (t, x) = \int_ {\mathbb {R} ^ {d}} p (t, x, 1, y) \nu_ {T} (d y), \quad (t, x) \in [ 0, T) \times \mathbb {R} ^ {d}.$$

We assume that our probability space, filtration, and Brownian motion are identical to those used to construct this weak solution. Then, the control process $\{ u _ { t } ^ { * } \} _ { 0 \leqslant t \leqslant T }$ defined by $u _ { t } ^ { * } = \nabla h ( t , X _ { t } ^ { * } )$ is in U and satisfies

$$\inf _ {u \in \mathcal {U} _ {\rho , 0}} J (u) = J (u ^ {*}) = 2 H (\pi \mid p (0, x, 1, y) \tilde {\mu} _ {0} (d x) d y),$$

whence $u ^ { * }$ is optimal to the Schr¨odinger’s problem (3.3). In particular, the finiteness condition (3.1) is satisfied.

Let us show

$$\lim _ {\varepsilon \searrow 0} \inf _ {u \in \mathcal {U} _ {\rho , \varepsilon}} J (u) = \inf _ {u \in \mathcal {U} _ {\rho , 0}} J (u). \tag {3.4}$$

To this end, let $P ^ { W } , P ^ { * } \in \mathcal { P } ( \mathbb { W } ^ { d } )$ be the law of $X _ { 0 } + B$ and $X ^ { * }$ , respectively. Then it is well-known that

$$\inf H (Q \mid P ^ {W}) = H (P ^ {*} \mid P ^ {W}), \tag {3.5}$$
