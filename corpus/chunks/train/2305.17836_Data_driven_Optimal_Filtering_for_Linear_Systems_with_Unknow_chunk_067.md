# F.2 Proof of Proposition 3: The Observation model

Proof. Recall that

$$\varepsilon (L, \mathcal {Y} _ {T}) = \| H x (T) - H \hat {x} (T) \| ^ {2} = \sum_ {i = 1} ^ {m} | H _ {i} ^ {\intercal} x (T) - H _ {i} ^ {\intercal} \hat {x} (T) | ^ {2}$$

where $H _ { i } ^ { \mathsf { T } }$ is the i-th row of H . Also, by duality, if $z ( t ) = ( A _ { t } ^ { \top } ) ^ { T - t } H _ { i }$ is the adjoint dynamics’ closed-loop trajectory with control signal $u ( t ) = L ^ { \mathsf { T } } z ( t )$ then

$$H _ {i} ^ {\intercal} x (T) - H _ {i} ^ {\intercal} \hat {x} (T) = \vec {z} _ {i} ^ {\intercal} \vec {\xi} - \vec {u} ^ {\intercal} \vec {\omega}$$

where

$$\vec {z} _ {i} ^ {\intercal} = (z (T) ^ {\intercal} \quad z (T - 1) ^ {\intercal} \quad \ldots \quad z (1) ^ {\intercal} \quad z (0) ^ {\intercal}) ,\vec {u} ^ {\intercal} = (u (T) ^ {\intercal} \quad u (T - 1) ^ {\intercal} \quad \ldots \quad u (1) ^ {\intercal} \quad 0 _ {m} ^ {\intercal}).$$

But $\vec { u } = ( I \otimes L ^ { \boldsymbol { \mathsf { T } } } ) \vec { z } _ { i }$ i and then $\vec { z } _ { i } = \mathcal { A } _ { L } ^ { \top } H _ { i }$ . Therefore,

$$
\begin{array}{l} \varepsilon (L, \mathcal {Y} _ {T}) = \sum_ {i = 1} ^ {m} | H _ {i} ^ {\intercal} x (T) - H _ {i} ^ {\intercal} \hat {x} (T) | ^ {2} \\ = \sum_ {i = 1} ^ {m} \mathrm{tr} \left[ \vec {\xi} \vec {\xi} ^ {\intercal} \vec {z} _ {i} \vec {z} _ {i} ^ {\intercal} \right] - \mathrm{tr} \left[ \left(\vec {\xi} \vec {\omega} ^ {\intercal} (I \otimes L ^ {\intercal}) + (I \otimes L) \vec {\omega} \vec {\xi} ^ {\intercal}\right) \vec {z} _ {i} \vec {z} _ {i} ^ {\intercal} \right] \\ + \operatorname{tr} \left[ \vec {\omega} \vec {\omega} ^ {\intercal} (I \otimes L ^ {\intercal}) \vec {z} _ {i} \vec {z} _ {i} ^ {\intercal} (I \otimes L) \right] \\ \end{array}
$$

Then, by using the fact that $\begin{array} { r } { \sum _ { i = 1 } ^ { m } H _ { i } H _ { i } ^ { \intercal } = H ^ { \intercal } H } \end{array}$ , we obtain that

$$\varepsilon (L, \mathcal {Y} _ {T}) = \operatorname{tr} \left[ \mathcal {X} _ {L} \mathcal {A} _ {L} ^ {\intercal} H ^ {\intercal} H \mathcal {A} _ {L} \right].$$

Thus, we can rewrite the estimation error as

$$\varepsilon (L, \mathcal {Y} _ {T}) = \left\langle \vec {\xi}, \vec {\xi} \right\rangle_ {\mathcal {A} _ {L}} + \left\langle (I \otimes L) \vec {\omega}, (I \otimes L) \vec {\omega} \right\rangle_ {\mathcal {A} _ {L}} - 2 \left\langle \vec {\xi}, (I \otimes L) \vec {\omega} \right\rangle_ {\mathcal {A} _ {L}} = \| \vec {\eta} _ {L} \| _ {\mathcal {A} _ {L}} ^ {2}$$
