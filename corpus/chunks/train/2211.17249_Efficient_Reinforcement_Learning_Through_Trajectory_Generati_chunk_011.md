# 3.2. Trajectory generation

For generic RL algorithms, a trajectory is sampled from the system that starts from an initial state $\pmb { x } ( 0 )$ and subsequently implements the control policy $\begin{array} { r } { \pmb { u } ( k ) = \pmb { \theta } \pmb { x } ( k ) + \pmb { w } ( k ) , \pmb { w } ( k ) \sim \mathcal { D } } \end{array}$ for $k = 0 , \cdots , T - 1$ . Given θ, the probability density function of a trajectory is

$$p _ {\pi_ {\boldsymbol {\theta}}} (\boldsymbol {\tau}) = p \left(\boldsymbol {x} (0)\right) \prod_ {k = 0} ^ {T - 1} p \left(\boldsymbol {\theta} \boldsymbol {x} (k) + \boldsymbol {w} (k) | \boldsymbol {x} (k)\right) p \left(\boldsymbol {x} (k + 1) \mid \boldsymbol {x} (k), \boldsymbol {\theta} \boldsymbol {x} (k) + \boldsymbol {w} (k)\right), \tag {8}$$

which is uniquely determined by ${ \pmb x } ( 0 )$ and the sequence of perturbations ${ \pmb w } ( 0 ) , { \pmb w } ( 1 ) , \cdots , { \pmb w } ( T - 1 )$ .

In the following, we generate trajectories for each updated θ as if they truly come from the system starting from ${ \pmb x } ( 0 )$ under perturbations on actions given by ${ \pmb w } ( 0 ) , \dots , { \pmb w } ( T - 1 )$ . Importantly, we use fixed historic trajectories $\{ \boldsymbol { u } _ { d } , \boldsymbol { y } _ { d } \}$ where the Hankel matrix H in (5) satisfies rank $( \mathcal { H } ) = n + T m$ , and $\mathbf { \pmb { u } } _ { d }$ can come from controllers different from the control policy in (4).

The key is to use $\pmb { x } ( 0 )$ and $( \pmb { w } ( 0 ) , \dots , \pmb { w } ( T - 1 ) )$ as extra constraints to find the g in (6) that generates the trajectory which follows the same distribution as (8). From the control policy ${ \pmb u } ( k ) = { \pmb \theta } { \pmb x } ( k ) + { \pmb w } ( k )$ , the trajectory subject to the perturbations ${ \pmb w } ( 0 ) , { \pmb w } ( 1 ) , \cdots , { \pmb w } ( T - 1 )$ should satisfy

$$
\left[ \begin{array}{c} \tilde {\boldsymbol {u}} (0) \\ \vdots \\ \tilde {\boldsymbol {u}} (T - 1) \end{array} \right] = \underbrace {\left[ \begin{array}{c c c} \boldsymbol {\theta} & & \\ & \ddots & \\ & & \boldsymbol {\theta} \end{array} \right]} _ {\boldsymbol {I} _ {T} \otimes \boldsymbol {\theta}} \left[ \begin{array}{c} \tilde {\boldsymbol {x}} (0) \\ \vdots \\ \tilde {\boldsymbol {x}} (T - 1) \end{array} \right] + \left[ \begin{array}{c} \boldsymbol {w} (0) \\ \vdots \\ \boldsymbol {w} (T - 1) \end{array} \right]. \tag {9}
$$
