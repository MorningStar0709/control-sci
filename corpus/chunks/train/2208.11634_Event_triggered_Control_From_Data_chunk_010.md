# A. Assumption on set D

We consider throughout this section system (1) with $d \equiv 0$ and a noise-free dataset D in (3). We define

$$U _ {0} := \left[ u (0) \quad u (T _ {s}) \quad \dots \quad u ((T - 1) T _ {s}) \right] \in \mathbb {R} ^ {m \times T}, \tag {4a}
X _ {0} := \left[ \begin{array}{c c c c} x (0) & x (T _ {s}) & \dots & x ((T - 1) T _ {s}) \end{array} \right] \in \mathbb {R} ^ {n \times T}, \tag {4b}

X _ {1} := \left[ \begin{array}{c c c c} \dot {x} (0) & \dot {x} (T _ {s}) & \dots & \dot {x} ((T - 1) T _ {s}) \end{array} \right] \in \mathbb {R} ^ {n \times T}. \quad (4 \mathrm{c})
$$

We assume the next condition holds on the richness of the data in D, which is discussed later on after Theorem 1.“ ‰

Assumption 1: The matrix $\left[ { \begin{array} { l } { U _ { 0 } } \\ { X _ { 0 } } \end{array} } \right]$ has full row rank.

Assumption 1 can be easily checked for a given set D. For discrete-time systems, when $( u , d )$ is persistently exciting then Assumption 1 holds [16]. For an extension of this result to continuous-time systems see [31]. We are ready to proceed with the design of the controller gain K in (2).
