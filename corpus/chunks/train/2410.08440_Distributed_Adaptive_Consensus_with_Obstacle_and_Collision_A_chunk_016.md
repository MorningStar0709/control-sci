# A. Proposed Distributed Control Law

The proposed distributed control law is presented for each agent i as:

$$u _ {i} = u _ {i} ^ {d} - u _ {i} ^ {c} - u _ {i} ^ {0}$$

where:

$$u _ {i} ^ {d} = \frac {\rho}{d _ {i} + b _ {i} ^ {0}} - \hat {\theta} _ {i} ^ {\top} \phi_ {i} - \hat {\theta} _ {i w} ^ {\top} \phi_ {i w} + \hat {\theta} _ {0} ^ {\top} \phi_ {0} + r _ {i} - c _ {i} E _ {i 0} ^ {\top}, (3 2)u _ {i} ^ {0} = \Gamma_ {b} ^ {0} \sum_ {b = 1} ^ {\xi} m _ {i \bar {b}}, \tag {33}u _ {i} ^ {c} = \Gamma_ {i j} ^ {1} \sum_ {j = 1} ^ {N} m _ {i j} + \Gamma_ {i 0} ^ {2} \sum_ {j = 1} ^ {N} m _ {i 0}. \tag {34}$$

This yields the global form of the control law U as:

$$U = U ^ {d} - U ^ {0} - U ^ {c} \tag {35}$$

where:

$$U ^ {d} = (D + B) ^ {- 1} \rho - \hat {\theta} ^ {\top} \phi - \hat {\theta} _ {w} ^ {\top} \phi_ {w} + \hat {\theta} _ {0} ^ {\top} \phi_ {0} + r - c E _ {0} ^ {\top}, \tag {36}U ^ {0} = \Gamma^ {0} M ^ {0}, \tag {37}U ^ {c} = \Gamma^ {1} M _ {I} ^ {c} + \Gamma^ {2} M _ {0} ^ {c}. \tag {38}$$

$\Gamma ^ { 0 }$ and $\Gamma ^ { 1 }$ are gain matrices that modulate the influence of obstacle avoidance $( U ^ { 0 } )$ and collision avoidance $( U ^ { c } )$ . The overall control law is:

$$U = (D + B) ^ {- 1} \rho - \hat {\theta} ^ {\top} \phi - \hat {\theta} _ {w} ^ {\top} \phi_ {w} + \hat {\theta} _ {0} ^ {\top} \phi_ {0} + r - c E _ {0} ^ {\top}- \Gamma^ {0} M ^ {0} - \Gamma^ {1} M _ {I} ^ {c} - \Gamma^ {2} M _ {0} ^ {c} \tag {39}$$

Here, $U = [ u _ { 1 } , u _ { 2 } , \cdots , u _ { N } ]$ is the control input vector for all agents, and $( D + B ) ^ { - 1 }$ is the inverse of the diagonal matrix diag $[ ( d _ { 1 } + b _ { 1 0 } ) , ( d _ { 2 } + b _ { 2 0 } ) , . . . , ( d _ { N } + b _ { N 0 } ) ]$ ]. The global neural network weight matrices $\hat { \theta } , \hat { \theta } _ { w } , \hat { \theta } _ { 0 }$ correspond to the agents’ dynamics, disturbances, and leader’s dynamics, while the global basis function matrices $\phi , \phi _ { w } , \phi _ { 0 }$ relate to these respective dynamics. $c _ { i } = [ k ^ { 1 } , k ^ { 2 } , \cdot \cdot \cdot , k ^ { n } ] \in \mathbb { R } ^ { 1 \times n }$ is

a gain matrix and $E _ { i 0 } ^ { \top } = \left[ \begin{array} { c } { { \bar { x } _ { i } ^ { 1 } - \bar { \bar { x } } _ { 0 } ^ { 1 } } } \\ { { \bar { x } _ { i } ^ { 2 } - \bar { x } _ { 0 } ^ { 2 } } } \\ { { \vdots } } \\ { { \bar { x } _ { i } ^ { n } - \bar { x } _ { 0 } ^ { n } } } \end{array} \right] \in \mathbb { R } ^ { n \times 1 }$ is the relative

error between the follower and leader. In addition, c and $E _ { 0 }$ are the global forms of $c _ { i }$ and $E _ { i 0 }$ , respectively.
