# A. AUV Kinematics and Dynamics

The planar kinematics are

$$\dot {\boldsymbol {\eta}} = \boldsymbol {J} (\psi) \boldsymbol {\nu}, \tag {1}$$

where $\pmb { \eta } = [ x , y , \psi ] ^ { \top }$ is the Earth-fixed position and heading, $\pmb { \nu } = [ u , v , r ] ^ { \top }$ is the body-fixed velocity, and

$$
\boldsymbol {J} (\psi) = \left[ \begin{array}{c c c} \cos \psi & - \sin \psi & 0 \\ \sin \psi & \cos \psi & 0 \\ 0 & 0 & 1 \end{array} \right] = \left[ \begin{array}{c c} \boldsymbol {R} (\psi) & \boldsymbol {0} \\ 0 & 1 \end{array} \right]. \tag {2}
$$

The 3-DOF Newton–Euler dynamics in the body frame are

$$M \dot {\nu} + C (\nu) \nu + D (\nu) \nu = \tau + w, \tag {3}$$

where $M = M ^ { \top } \succ 0$ is the inertia matrix (including added mass), $\begin{array} { r } { C ( \nu ) = - C ^ { \top } ( \nu ) } \end{array}$ is the Coriolis–centripetal matrix, $D ( \nu ) \succ 0$ is the hydrodynamic damping, ${ \boldsymbol { \tau } } = [ F _ { u } , F _ { v } , F _ { r } ] ^ { \top }$ is the generalized force/moment, and w collects bounded external disturbances.

Define the full state $\pmb { x } = [ \pmb { \eta } ^ { \top } , \pmb { \nu } ^ { \top } ] ^ { \top }$ . The state-space form follows from (1)–(3):

$$
\dot {\boldsymbol {x}} = \left[ \begin{array}{c} \boldsymbol {J} (\psi) \boldsymbol {\nu} \\ \boldsymbol {M} ^ {- 1} (\boldsymbol {\tau} - \boldsymbol {C} (\boldsymbol {\nu}) \boldsymbol {\nu} - \boldsymbol {D} (\boldsymbol {\nu}) \boldsymbol {\nu}) \end{array} \right] = \boldsymbol {f} (\boldsymbol {x}, \boldsymbol {\tau}). \tag {4}
$$
