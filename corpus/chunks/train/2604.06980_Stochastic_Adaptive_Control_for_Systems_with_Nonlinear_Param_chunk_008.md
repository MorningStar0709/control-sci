# 2 Preliminaries and Problem Formulation

In this paper, we consider the stochastic nonlinear system (1). The matrices $A ^ { * }$ and $B ^ { * }$ are unknown, while the nonlinear function $f ( \cdot )$ is known and the trajectory data $\{ x _ { \tau } , u _ { \tau } \} _ { 0 \leq \tau \leq t }$ are observable at each time $t \geq 0$ . Let $\{ \dot { \mathcal { F } } _ { t } \}$ be a filtration, i.e., a nondecreasing family of σ-algebras, defined by $\mathcal { F } _ { t } = \sigma ( x _ { 0 } , u _ { i } , w _ { i } , 0 \leq i \leq t )$ . We assume that the nonlinear function $f ( \cdot )$ and noise process $\{ w _ { t } \} _ { t \ge 0 }$ satisfy the following two assumptions.

Assumption 1 There exist a nonincreasing function $\alpha : \mathbb { R } _ { \geq 0 } \ \to \ \mathbb { R } _ { > 0 }$ and a nondecreasing function $\beta ~ : ~ \mathbb { R } _ { \geq 0 } ~ \bar { \to } ~ \mathbb { R } _ { > 0 }$ with $\operatorname* { i n f } _ { z \in \mathbb { R } ^ { n } } \alpha ( \Vert f ( z ) \Vert ) \ > \ 0$ and $\begin{array} { r } { \operatorname* { s u p } _ { z \in \mathbb { R } ^ { n } } \bar { \beta } ( \| f ( z ) \| ) < \infty , } \end{array}$ , such that, for any $c \geq 0$ and al l $\| { \bar { x } } \| , \| y \| \leq c ,$

$$\left\| f (x) - f (y) \right\| \leq \beta (c) \| x - y \|, a n d \tag {2}(x - y) ^ {\top} (f (x) - f (y)) \geq \alpha (c) \| x - y \| ^ {2}.$$

Assumption 2 The sequence $\{ w _ { t } , \mathcal { F } _ { t } \} _ { t \ge 0 }$ is a martingale difference sequence, and there exist constants $\gamma > 2$ and $\sigma > 0$ such that, for all $t \geq 0$ ,

$$\sup _ {t \geq 0} \mathbb {E} \left[ \| w _ {t + 1} \| ^ {\gamma} \mid \mathcal {F} _ {t} \right] < \infty , \mathbb {E} \left[ w _ {t + 1} w _ {t + 1} ^ {\top} \mid \mathcal {F} _ {t} \right] \geq \sigma I _ {n}.$$

Moreover, $\begin{array} { r } { i f \operatorname* { s u p } _ { x \in \mathbb { R } ^ { n } } \| f ( x ) \| < \infty } \end{array}$ , we additionally assume that $\{ w _ { t } \} _ { t \ge 0 }$ is almost surely bounded.

Next, two sufficient conditions for Assumption 1 are given in Lemma 1 and further illustrated by the examples given in Subsection 2.1, which shows that this assumption holds in a wide range of stochastic nonlinear systems.
