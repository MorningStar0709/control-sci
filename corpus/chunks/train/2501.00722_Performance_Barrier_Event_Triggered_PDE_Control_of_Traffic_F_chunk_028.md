# A. Performance-barrier Continuous-time Event-triggered Control (P-CETC)

Let $I ^ { p } = \{ t _ { 0 } ^ { p } , t _ { 1 } ^ { p } , t _ { 2 } ^ { p } , . . . \}$ denote the sequence of event-times associated with P-CETC. Let the event-trigger parameters be selected as in Assumption 2, and $c > 0$ be an additional design parameter. The proposed P-CETC strategy consists of two parts:

1) An ETC input $U _ { k } ^ { p }$

$$U _ {k} ^ {p} := U (t _ {k} ^ {p}), \tag {74}$$

for $t \in \left[ t _ { k } ^ { p } , t _ { k + 1 } ^ { p } \right) , k \in \mathbb { N }$ , where $U ( t )$ is given by (24). Then, the boundary condition (33) becomes

$$\bar {v} (\ell , t) = r _ {1} U _ {k} ^ {p}. \tag {75}$$

2) A continuous-time event-trigger determining event-times

$$t _ {k + 1} ^ {p} = \inf \left\{t \in \mathbb {R} _ {+} \mid t > t _ {k} ^ {p}, \Gamma^ {p} (t) > 0, k \in \mathbb {N} \right\}, \tag {76}$$

with $t _ { 0 } ^ { p } = 0$ . The triggering function $\Gamma ^ { p } ( t )$ is defined as

$$\Gamma^ {p} (t) := d ^ {2} (t) - \theta m ^ {p} (t) - \frac {c}{\theta_ {m}} W ^ {p} (t), \tag {77}$$

where d(t) is given by (34), mp(t) satisfies the ODE

$$
\begin{array}{l} \dot {m} ^ {p} (t) = - \eta m ^ {p} (t) - \theta_ {m} d ^ {2} (t) + \kappa_ {1} \| \alpha [ t ] \| ^ {2} + \kappa_ {2} \| \beta [ t ] \| ^ {2} \\ + \kappa_ {3} \alpha^ {2} (\ell , t) + c W ^ {p} (t), \tag {78} \\ \end{array}
$$

for $t \in \left( t _ { k } ^ { p } , t _ { k + 1 } ^ { p } \right) , k \in \mathbb { N }$ with $m ^ { p } \left( t _ { 0 } ^ { p } \right) = m ^ { p } ( 0 ) >$ 0, and $m ^ { p } \left( t _ { k } ^ { p - } \right) = m ^ { p } \left( t _ { k } ^ { p } \right) = m ^ { p } \left( t _ { k } ^ { p + } \right)$ . Here, $W ^ { p } ( t )$ is the performance residual, defined as the difference between the value of the performance-barrier $e ^ { - b ^ { \star } t } V _ { 0 } ^ { p }$ and the Lyapunov function

$$W ^ {p} (t) := e ^ {- b ^ {\star} t} V _ {0} ^ {p} - V ^ {p} (t), \tag {79}$$

where $b ^ { \star }$ is given by (59), and $V ^ { p } ( t )$ is the Lyapunov candidate defined as

$$V ^ {p} (t) = V _ {1} (t) + m ^ {p} (t), \tag {80}$$

with V1(t) given by (56) and

$$V _ {0} = V _ {0} ^ {p} = V ^ {p} (0) = V ^ {r} (0). \tag {81}$$
