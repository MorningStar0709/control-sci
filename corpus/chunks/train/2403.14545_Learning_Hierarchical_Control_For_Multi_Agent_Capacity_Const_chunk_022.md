# B. Policy Formulation

At each iteration $r \geq 1$ , the data-driven policies $\pi ^ { H , r }$ and $\pi ^ { L , r }$ are formulated using $S ^ { L , r } , S ^ { H , r } , \hat { \theta } ^ { r }$ , and $\hat { \Theta } ^ { r }$ as follows.

$I )$ High-level policy $\pi ^ { H , r } .$ : At event k of the rth iteration, the high-level controller searches for the path of $N$ edges beginning at the current state $x _ { k } ^ { H , r } = ( n _ { k } ^ { r } , c _ { k } ^ { r } )$ that maximizes the number of unique visited nodes. We introduce the optimal control problem by first listing the necessary constraints.

The binary variable $u \in \{ 0 , 1 \} ^ { | V | \times | V | }$ describes how the agent will be routed, with $u _ { i , j } = 1$ indicating that the agent shall travel from node $v _ { i }$ to node $v _ { j }$ . The constraints

$$\sum_ {j \in \mathcal {V}} u _ {n _ {k} ^ {r}, j} = 1 \tag {26a}c _ {n _ {k} ^ {r}, j, l} = c _ {l, k} ^ {r} \quad \forall j \in \mathcal {V}, l \in [ 1, L ] \tag {26b}u _ {i, j} = 0 \quad \forall i \in \mathcal {V}, j \notin N (i) \tag {26c}c _ {i, j, l} = 0 \quad \forall i \in \mathcal {V}, j \notin N (i), l \in [ 1, L ] \tag {26d}$$

ensure the path starts at node $n _ { k } ^ { r } .$ , and initialize the values of the capacity state accumulation variables $c _ { i , j , l }$ to the current capacity states at event k. Each variable $c _ { i , j , l }$ tracks the value of the lth capacity state $c _ { l }$ as the agent departs node $v _ { i }$ towards node $v _ { j }$ . Paths can only be planned between neighboring nodes. The constraints

$$\sum_ {i \in \mathcal {V}} u _ {i, j} \leq 1 \quad \forall j \in \mathcal {V} \backslash D \tag {27a}\sum_ {j \in \mathcal {V}} u _ {i, j} \leq 1 \quad \forall i \in \mathcal {V} \backslash D \tag {27b}\sum_ {i \in \mathcal {V}} u _ {i, j} \geq \sum_ {i \in \mathcal {V}} u _ {j, i} \quad \forall j \in \mathcal {V} \tag {27c}$$

ensure that a single path is formed, and that no paths begin from any node other than $n _ { k } ^ { r }$ . The constraints
