Let $g : \mathbb { R } $ R be a function defined as $\textstyle g ( x ) = x - { \frac { 1 } { \varphi _ { t } } } x ^ { 2 }$ . One can verify easily that g is monotonically increasing in $\textstyle [ 0 , { \frac { \varphi _ { t } } { 2 } } ]$ . Next with equation 19, we have
