# C. Loss Function

We jointly learn the three components by minimizing the following loss function:

$$\mathcal {L} (\theta) = \frac {1}{N} \sum_ {(x, u, \dot {x}) \in \mathcal {D}} k (u, u ^ {*} (x)) \| \dot {x} - f ^ {*} (x, u) \| ^ {2} + \lambda \| \theta \| _ {2} ^ {2} \tag {13}$$

where $\theta , k ( \cdot , \cdot )$ , and λ denote the parameters for the neural networks, a kernel function, and the regularization coefficient, respectively. The kernel function $k : \mathcal { U } \times \mathcal { U } $ R attains a larger value as the two arguments are getting closer. By weighting the $\ell ^ { 2 }$ difference, we encourage $u ^ { * }$ to learn a policy that generates trajectories with smaller model error. Conversely, we also encourage $f ^ { * }$ to learn better around feasible trajectories, i.e., $( x , u ^ { * } ( x ) )$ . In this work, we use the following kernel function:

$$k (u, u ^ {\prime}) = 1 + \exp (- \beta \| u - u ^ {\prime} \| _ {2} ^ {2}). \tag {14}$$
