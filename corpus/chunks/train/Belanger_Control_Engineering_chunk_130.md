# 3.7.1 Introduction

The transfer function corresponding to a given set of state equations is unique; however, a given transfer function (scalar or matrix) can be generated by an infinite number of state equations, each of which is called a realization of the transfer function. For example, the state equations

$$
\begin{array}{l} \dot {x} = - x + u \\ y = x \\ \end{array}
$$

and

$$
\begin{array}{l} \dot {x} = - x + a u \\ y = \frac {1}{a} x, \quad a \text {   real } \\ \end{array}
$$

have the same transfer function, $1/(s + 1)$ ; both are realizations of that transfer function.
