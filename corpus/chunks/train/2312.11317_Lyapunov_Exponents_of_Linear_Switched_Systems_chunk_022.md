# 4.4.1 Imaginary eigenvalues

We are now going to examine a special case. Suppose that $\operatorname { t r } ( A ^ { 2 } ) , \operatorname { t r } ( B ^ { 2 } ) < 0$ , i.e. A, B have imaginary eigenvalues.

Proposition 4.23. If both A, B have imaginary eigenvalues, the functional Φ is

$$\Phi (t, s) = \mathrm{tr} \big (\exp (t A) \exp (s B) \big) = 2 \cos (\lambda t) \cos (\mu s) - 2 \gamma \sin (\lambda t) \sin (\mu s),W h e r e \gamma = \frac {| \mathrm{tr} (A B) |}{\sqrt {\mathrm{tr} (A ^ {2}) \mathrm{tr} (B ^ {2})}}.$$

This is simply a consequence of formula (4.17) and $\cosh ( i \lambda t ) = \cos ( \lambda t ) , \sinh ( i \mu s ) = i \sin ( \mu s )$ . With this formula, the switching equation (4.14) reads

$$(\gamma \lambda - \mu) \cos (\lambda t) \sin (\mu s) = (\gamma \mu - \lambda) \cos (\mu s) \sin (\lambda t),$$

from which we obtain immediately that there are always solutions for $\cos ( \lambda t ) , \cos ( \mu s ) = 0$ , that is

$$(t, s) \in \left\{\left(\frac {\pi}{2 \lambda}, \frac {3 \pi}{2 \mu}\right), \left(\frac {\pi}{2 \lambda}, \frac {\pi}{2 \mu}\right), \left(\frac {3 \pi}{2 \lambda}, \frac {\pi}{2 \mu}\right) \right\}.$$

From these choices the value of this functional is $\Phi ( t , s ) = \pm 2 \gamma$ . Using formula (4.3) one obtains

$$\ell_ {\mathrm{per,im}} (A, B) = \frac {1}{t + s} \mathrm{arcosh} (\gamma).$$

If $\mathrm { t r } ( A B ) < 0$ , then we have

$$\ell_ {\text { per,im }} (A, B) = \frac {2}{\pi \left(\sqrt {\frac {- 2}{\operatorname{tr} (A ^ {2})}} + 3 \sqrt {\frac {- 2}{\operatorname{tr} (B ^ {2})}}\right)} \operatorname{arcosh} \left(\frac {- \operatorname{tr} (A B)}{\sqrt {\operatorname{tr} (A ^ {2}) \operatorname{tr} (B ^ {2})}}\right). \tag {4.24}$$

Notice that we have chosen the couple (t, s) so to maximize $\frac { 1 } { t + s }$ under the hypothesis $\mathrm { t r } ( B ^ { 2 } ) < \mathrm { t r } ( A ^ { 2 } ) <$ 0.

If instead $\operatorname { t r } ( A B ) > 0$ , then

$$\ell_ {\text { per,im }} (A, B) = \frac {2}{\pi \left(\sqrt {\frac {- 2}{\text { tr } (A ^ {2})}} + \sqrt {\frac {- 2}{\text { tr } (B ^ {2})}}\right)} \text { arcosh } \left(\frac {\text { tr } (A B)}{\sqrt {\text { tr } (A ^ {2}) \text { tr } (B ^ {2})}}\right). \tag {4.25}$$

We will see in the next subsection how to deal with the extremals with $\cos ( \lambda t ) , \cos ( \mu s ) \neq 0$ .
