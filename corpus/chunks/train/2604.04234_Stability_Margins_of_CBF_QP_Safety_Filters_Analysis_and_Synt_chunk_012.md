# B. Classical Robustness Margins

We now specialize the uncertainty model (14) to real gain perturbations. In this case, Thm. 1 characterizes exactly when the perturbed active mode remains Hurwitz and therefore when GES of the origin for the closed loop is guaranteed.

For real gain perturbations, we set $\Delta = \kappa , \kappa > 0$ , so that

$$\tilde {A} (\kappa) := A _ {0} - \kappa U V ^ {\top}. \tag {16}$$

This motivates the following definition, which captures the notion of gain margin for (7).

Definition 1 (Stability-preserving gain interval). Assume $\tilde { A } = \tilde { A } ( 1 )$ is Hurwitz and define

$$\mathcal {K} := \{\kappa > 0: \tilde {A} (\kappa) \text { is Hurwitz } \}. \tag {17}$$

The stability-preserving gain interval is the connected component of K that contains $\kappa = 1$ .

Remark 2. In classical single-loop control, gain margin is often summarized by one or two boundary values relative to the nominal gain. In the present setting, the natural object is the full interval of gain perturbations for which the active mode remains Hurwitz, and the origin is guaranteed to be GES for the closed loop (7).

The next corollary gives the frequency-domain characterization of the boundary of the interval in Definition 1. Intuitively, loss of admissibility occurs when the Nyquist curve of $L ( j \omega )$ reaches the negative real axis at the critical value $- 1 / \kappa$ .

Corollary 1. Let $\left( \underline { { \kappa } } , \bar { \kappa } \right)$ denote the stability-preserving gain interval, and let $L ( s )$ be defined by (11). If $\kappa _ { \star } \in \{ \underline { { \kappa } } , \bar { \kappa } \} \cap$ $( 0 , \infty )$ , then there exists $\omega \ge 0$ such that

$$1 + \kappa_ {\star} L (j \omega) = 0. \tag {18}$$

Equivalently, $\begin{array} { r } { ~ \kappa _ { \star } ~ = ~ \frac { 1 } { | L ( j \omega ) | } } \end{array}$ for some $\omega ~ \geq ~ 0$ such that $L ( j \omega ) \in \mathbb { R } _ { < 0 } .$ .
