Remark 7: With the definition of ǫ in Theorem 1, the trigger condition in (11) can be relaxed as

$$\eta_ {i} (x _ {i}) > c ^ {- 1} \max \left\{| \tilde {x} _ {i} | - \epsilon / \sqrt {N}, 0 \right\} + \underline {{\eta}}, \tag {32}$$

which is composed of individual consensus error ${ \tilde { x } } _ { i } ,$ best prediction performance η and overall consensus error bound ǫ. This formulation indicates a general event-trigger design for online learning in multi-agent system, namely comparison of the current prediction error $\eta _ { i } ( x _ { i } )$ , the deviation of the control error to its bound $| \tilde { x } _ { i } | - \epsilon / \sqrt { N }$ , and the guaranteed best prediction performance η, which is also suitable for cooperative learning with different hyperparameters for each Gaussian process model as in [12].
