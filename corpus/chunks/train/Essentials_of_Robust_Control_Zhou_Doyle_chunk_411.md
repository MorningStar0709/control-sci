# $\mathcal { H } _ { \infty }$ Loop Shaping

This chapter introduces a design technique that incorporates loop-shaping methods to obtain performance/robust stability tradeoffs, and a particular $\mathcal { H } _ { \infty }$ optimization problem to guarantee closed-loop stability and a level of robust stability at all frequencies. The proposed technique uses only the basic concept of loop-shaping methods, and then a robust stabilization controller for the normalized coprime factor perturbed system is used to construct the final controller. This chapter is arranged as follows: The $\mathcal { H } _ { \infty }$ theory is applied to solve the stabilization problem of a normalized coprime factor perturbed system in Section 16.1. The loop-shaping design procedure is described in Section 16.2. The theoretical justification for the loop-shaping design procedure is given in Section 16.3. Some further loop-shaping guidelines are given in Section 16.4.
