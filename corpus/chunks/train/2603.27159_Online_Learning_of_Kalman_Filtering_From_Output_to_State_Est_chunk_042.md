# 6.1 Unstable System with Closed-Loop Input

While the Kalman filter can be applied to estimate the state of potentially unstable system [4], the unboundedness of the state of unstable system could make the task of state estimation vacuous, and closed-loop inputs can thus be applied to make the system stable beforehand [32]. Existing work on learning the Kalman filter can only handle open-loop (marginally) stable system without any external inputs [13, 31] or with only open-loop bounded inputs [12, 14, 15]. Note that when there are inputs $u _ { t }$ to system (1), the state equation is given by $x _ { t + 1 } = A x _ { t } + B u _ { t } + w _ { t }$ . In contrast with the aforementioned work, our algorithm design and regret analysis extend gracefully to open-loop unstable systems that can be stabilized by a static or dynamic output feedback controller [39,59–62].5 Specifically, our current regret analysis for Theorems 4.1 and 5.2 holds under an $\mathcal { O } ( \sqrt { \log T } )$ bound on the system state norm $\| x _ { t } \|$ for all $t \in \{ 0 , \ldots , T - 1 \}$ , which is a consequence of the assumption that A is stable. One can show that such an $\mathcal { O } ( \sqrt { \log T } )$ bound on $\| x _ { t } \|$ still holds when the closed-loop system is stable under the output feedback controls described above. Finally, extending to open-loop unstable system with closed-loop input is generally not possible for the existing works that use least squares methods to learn the filter parameter, since the analysis of these methods typically relies on certain persistency excitation conditions that may not hold under closed-loop inputs [13].
