# 2.1 Optimal control without control constraint

Let $\Omega \subset ( - 1 , 1 ) ^ { d } \subset \mathbb { R } ^ { d } ( d \geq 1 )$ be a bounded domain with a smooth boundary ∂Ω. Consider the following distributed optimal control problem

$$\min J (y, u) := \frac {1}{2} \| y - y _ {d} \| _ {L ^ {2} (\Omega)} ^ {2} + \frac {\lambda}{2} \| u \| _ {L ^ {2} (\Omega)} ^ {2}, \tag {2.1}$$

subject to the following elliptic PDE constraint

$$- \Delta y = f + u \text { in } \Omega \quad \text { with } \quad y = 0 \text { on } \partial \Omega , \tag {2.2}$$

where $u \in L ^ { 2 } ( \Omega )$ is the control, $f \in L ^ { 2 } ( \Omega )$ and $y _ { d } \in L ^ { 2 } ( \Omega )$ are the given source and target function, and the scalar $\lambda > 0$ balances the two terms in $J ( y , u )$ . Problem (2.1)–(2.2) has a unique solution (¯y, u¯) [49, Theorem 2.17]. The goal is to approximate the solution (¯y, u¯) by an NN pair $( \bar { y } _ { \theta } , \bar { u } _ { \kappa } )$ .
