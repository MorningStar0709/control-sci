Theorem 4 indicates that the remainder term $\varepsilon ( k , t _ { f } , t _ { 1 }$ : $t _ { n } )$ vanishes as $k , t _ { f } \to + \infty$ with an exponential decay rate in both k and $t _ { f } .$ Therefore, the optimal infinitehorizon gain $K _ { \infty }$ can be approximated by $K _ { k , t _ { f } }$ , where

$$\left(X _ {t _ {1}: t _ {n}} ^ {\prime} \otimes I _ {m}\right) \operatorname{vec} (K _ {k, t _ {f}}) = - \operatorname{vec} (U _ {t _ {1}: t _ {n}}). \tag {69}$$

Hence, the optimal gain is found by increasing k (the number of iterations that Algorithm 1 is run for) and increasing tf (the horizon length) as

$$\lim _ {k, t _ {f} \to + \infty} K _ {k, t _ {f}} = K _ {\infty}.$$

Remark 7. Equation (69) has a unique solution if and only if $\therefore X _ { t _ { 1 } : t _ { \tau } }$ n is full-rank.

The above rank condition is similar to the one used for solving discrete-time optimal control problems [6]. Assuming $k , t _ { f } $ ∞ and choosing equidistant time samples as $t _ { i } = \bar { ( i - 1 ) t } / n$ , matrix $X _ { t _ { 1 } : t _ { n } }$ can be written as

$$X _ {t _ {1}: t _ {n}} = \left[ x _ {0} \exp (A _ {\mathrm{cl}} \bar {t} / n) x _ {0} \dots \exp ((n - 1) A _ {\mathrm{cl}} \bar {t} / n) x _ {0} \right],$$

where $A _ { \mathrm { c l } } = A - B K _ { \infty }$ . Hence, $X _ { t _ { 1 } : t _ { n } }$ is full-rank if and only if the pair $( \exp ( A _ { \mathrm { c l } } \bar { t } / n ) , x _ { 0 } )$ is controllable, which is true if $x _ { 0 }$ is not orthogonal to any of the left eigenvectors of $\exp ( A _ { \mathrm { c l } } \bar { t } / n )$ . In this case, the matrix $X _ { t _ { 1 } : t _ { n } }$ is full rank generically. Therefore, with a random initial condition (drawn from any continuous probability distribution), equation (69) has a unique solution almost surely.
