Assume that the highest order HOCBF constraint (5) conflicts with control input constraints (2) at $t = t _ { b }$ , i.e., we can not find a feasible controller $\boldsymbol { u } ^ { * } ( t _ { b } )$ to satisfy (5) and (2). Relying on AVCBFs, we can discretize the whole time period [0, T ] into several small time intervals like $[ t _ { a } , t _ { b } ]$ to maximize the feasibility of solving $\mathrm { Q P }$ under safety constraints.
