We next choose η for the fixed ε. For fixed $\hat { \Omega } ^ { \star } ( \varepsilon )$ and $\sigma \in \mathsf { \Gamma } ( 0 , 1 / \alpha _ { S } ( \varepsilon ) )$ , we seek $\eta$ that minimizes $\hat { \rho } ,$ which is achieved by minimizing $\beta .$ Indeed, by differentiating $\hat { \rho }$ with respect to $\beta ,$ we obtain

$$\frac {d \hat {\rho}}{d \beta} = \frac {(1 + \delta^ {- 1}) \alpha_ {S _ {u}} (\varepsilon) (1 - \varepsilon) + \alpha_ {\Gamma_ {U}} (\varepsilon) (1 + \delta)}{(1 - \varepsilon - \alpha_ {\Gamma_ {U}} (\varepsilon) \beta) ^ {2}} \geq 0.$$

This means that, for fixed $\varepsilon , { \hat { \Omega } } ^ { \star } ( \varepsilon ) , \sigma$ , and $\delta > 0 , \hat { \rho }$ is a nondecreasing function of $\beta$ on $1 - \varepsilon - \alpha _ { \Gamma _ { U } } ( \varepsilon ) \beta > 0$ . Hence, it suffices to minimize $\beta$ with respect to η for minimizing ${ \hat { \rho } } .$

The following lemma provides such a choice of $\eta .$

Lemma $4 { : }$ Fix $\varepsilon$ and $\sigma \in ( 0 , 1 / \alpha _ { S } ( \varepsilon ) )$ . Then

$$\eta = \eta^ {\star} (\sigma ; \varepsilon) := \frac {\sqrt {\sigma \alpha_ {S} (\varepsilon)}}{1 - \sqrt {\sigma \alpha_ {S} (\varepsilon)}}, \tag {45}$$

is the unique minimizer of $\beta$ over

$$\eta > \frac {\sigma \alpha_ {S} (\varepsilon)}{1 - \sigma \alpha_ {S} (\varepsilon)}, \tag {46}$$

and the corresponding minimum value is given by

$$\beta_ {\min} (\sigma ; \varepsilon) = \frac {\sigma}{\left(1 - \sqrt {\sigma \alpha_ {S} (\varepsilon)}\right) ^ {2}}.$$

Proof: Define $a : = \sigma \alpha _ { S } ( \varepsilon )$ . Since $\sigma \in ( 0 , 1 / \alpha _ { S } ( \varepsilon ) )$ , it follows that $0 < a < 1$ . Then, (46) can be written as

$$\eta > \frac {a}{1 - a}. \tag {47}$$

For fixed σ and ε, we define

$$\beta_ {\sigma , \varepsilon} (\eta) := \frac {\sigma (1 + \eta)}{1 - \sigma (1 + \eta^ {- 1}) \alpha_ {S} (\varepsilon)}.$$

Multiplying the numerator and denominator by $\eta ,$ we can rewrite it as

$$\beta_ {\sigma , \varepsilon} (\eta) = \frac {\sigma \eta (1 + \eta)}{(1 - a) \eta - a}.$$

Since $\eta > a / ( 1 - a )$ , we have $( 1 - a ) \eta - a > 0$ on the feasible set (47).

Differentiating $\beta _ { \sigma , \varepsilon } ( \eta )$ with respect to η yields

$$\frac {d}{d \eta} \beta_ {\sigma , \varepsilon} (\eta) = \sigma \frac {h (\eta)}{((1 - a) \eta - a) ^ {2}},$$

where

$$h (\eta) := (1 - a) \eta^ {2} - 2 a \eta - a.$$
