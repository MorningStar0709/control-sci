# 3.1.5 Difference Inclusion Description of Uncertain Systems

Here we introduce some notation that will be useful in the sequel. A deterministic discrete time system is usually described by a difference equation

$$x ^ {+} = f (x, u) \tag {3.1}$$

We use $\phi ( k ; x , i , { \mathbf { u } } )$ to denote the solution of (3.1) at time k when the initial state at time i is x and the control sequence is ${ \bf u } = \{ u ( 0 ) , u ( 1 ) , \ldots \}$ ; if the initial time $i = 0$ , we write $\phi ( k ; x , { \mathbf { u } } )$ in place of $\phi ( k ; ( x , 0 ) , { \mathbf { u } } )$ .
