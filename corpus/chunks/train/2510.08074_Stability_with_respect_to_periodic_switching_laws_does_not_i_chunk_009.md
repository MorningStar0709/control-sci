We are now in a position to state our main result, which is the following more precise formulation of Theorem 1:

Theorem 3. Let $A _ { 0 } , A _ { 1 } \in M _ { 2 } ( \mathbb { R } )$ have the properties described in Theorem 2. Let $\alpha > 0$ be irrational, and define four matrices $B _ { 0 } , B _ { 1 } , B _ { 2 } , B _ { 3 } \in M _ { 4 } ( \mathbb { R } )$ by

$$
\begin{array}{l} B _ {0} := A _ {0} \otimes I + \alpha \cdot I \otimes A _ {0}, \\ B _ {1} := A _ {1} \otimes I + \alpha \cdot I \otimes A _ {0}, \\ B _ {2} := A _ {0} \otimes I + \alpha \cdot I \otimes A _ {1}, \\ B _ {3} := A _ {1} \otimes I + \alpha \cdot I \otimes A _ {1}, \\ \end{array}
$$

where I denotes the 2 2 identity matrix. Let B denote the convex hull of $\{ B _ { 0 } , \ldots , B _ { 3 } \}$ . Then:

(i) There exists a constant $K > 0$ such that for every Lebesgue measurable switching law $B \colon [ 0 , \infty ) \to { \mathsf { B } }$ and every $w \in \mathbb { R } ^ { 4 }$ the solution to

(3) ${ \dot { x } } ( t ) = B ( t ) x ( t )$

(4) $x ( 0 ) = w$

satisfies $\| x ( t ) \| \leq K \| x ( 0 ) \|$ for all $t \geq 0$ .

(ii) $I f B \colon [ 0 , \infty )  { \mathsf { B } }$ is Lebesgue measurable and periodic then every solution to (3) converges exponentially to the origin.

(iii) There exist a piecewise constant switching law $B \colon [ 0 , \infty )  \{ B _ { 0 } , B _ { 1 } , B _ { 2 } , B _ { 3 } \}$ and vector $w \in \mathbb { R } ^ { 4 }$ such that the solution to (3), (4) does not accumulate at the origin.

The three clauses of Theorem 3 in turn imply that the linear switched system defined by B is Lyapunov stable, exponentially stable with respect to every periodic switching law, and not globally asymptotically stable with respect to piecewise constant switching signals (hence also with respect to arbitrary switching).

The switching law $B ( t )$ constructed in Theorem 3(iii) is in fact quasi-periodic in the following sense: if u0 is as described in Theorem 2 then $B ( t ) = B _ { 0 }$ when $u _ { 0 } ( t ) = u _ { 0 } ( \alpha t ) = 0 , B ( t ) = B _ { 1 }$ when $1 - u _ { 0 } ( t ) = u _ { 0 } ( \alpha t ) = 0 , B ( t ) = B _ { 2 }$ when $u _ { 0 } ( t ) = 1 - u _ { 0 } ( \alpha t ) = 0$ , and $B ( t ) = B _ { 3 }$ when $1 - u _ { 0 } ( t ) = 1 - u _ { 0 } ( \alpha t ) = 0$ . It is possible to show that pairs of successive switching times of this law $B ( t )$ can occur arbitrarily close together, but that three switching times cannot occur arbitrarily close together.
