# C. Additional Parameters and Metrics

The upper connectivity bound $R _ { c o m }$ in Eq. (2c) and (7) is set to 9km, which corresponds approximately to radio communication capabilities for ASVs. The collision lower threshold from Eq. (2b) is set to $R _ { c o l l } = 1 0 0 \mathrm { m }$ , providing a practical margin, as one would typically do in a realworld implementation. Moreover, we define ϵ = 300m for the edge hysteresis parameter from Eq. (7). We use the Euclidean norm to measure inter-agent distances $d ( q _ { i } , q _ { j } )$ and the minimum flock center distance to target $d _ { m i n } ( \bar { T } )$ .
