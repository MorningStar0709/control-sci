# 9.5 APPLICATIONS OF GAIN SCHEDULING

Gain scheduling is a very useful method. It requires good knowledge about the process and that some auxiliary variables can be measured. A great advantage with the method is that the controller adapts quickly to changing conditions.

This section contains examples of some cases in which it is advantageous to use gain scheduling in some of the forms that have been presented above. The examples include ship steering, pH control, combustion control, engine control, and flight control.
