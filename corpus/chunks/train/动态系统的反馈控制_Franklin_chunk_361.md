根据图6.66所示PID控制器的相频曲线，判断随着 $T_{\mathrm{D}}$ 取值变化 $D_{\mathrm{c}}(s)G(s)$ 会有怎样的变化。当 $1 / T_{\mathrm{D}}\geqslant 2\mathrm{rad / s}$ 时，PID所提供的超前相位可以抵消传感器的相位延迟， $D_{\mathrm{c}}(s)G(s)$ 的相位不会超过 $-180^{\circ}$ ，系统仍然是不稳定的。当 $1 / T_{\mathrm{D}}\leqslant 0.01$ 时， $D_{\mathrm{c}}(s)G(s)$ 的相位在一定的频率范围内接近 $-90^{\circ}$ ，当频率范围更大一些时，相位将会超过 $-115^{\circ}$ ，也就是说，系统的相位裕度的阈值是 $65^{\circ}$ 。图6.68所示的是在 $1 / T_{\mathrm{D}}$ 取0.1时的特性曲线，可以看到要保证相位裕度为 $65^{\circ}$ ， $1 / T_{\mathrm{D}}$ 的取值不能比0.1再大了，对于任意大于0.1的 $1 / T_{\mathrm{D}}$ 取值，相位在任何频率处都不会达到 $-115^{\circ}(\mathrm{PM} = 65^{\circ})$ 。当 $1 / T_{\mathrm{D}}$ 取0.1时，系统的相位裕度为 $65^{\circ}$ ，穿越频率为 $0.5\mathrm{rad / s}$ 。当 $1 / T_{\mathrm{D}} \ll 0.05$ 时，相频曲线如图6.68的虚线所示。如果 $1 / T_{\mathrm{D}}$ 取为0.05，则穿越频率最大可近似取到 $1\mathrm{rad / s}$ 。因此， $0.05 < 1 / T_{\mathrm{D}} < 0.1$ 是 $1 / T_{\mathrm{D}}$ 较理想的取值范围，小于0.05，系统的带宽不会再有明显的增大，而大于0.1就无法得到满意的相位裕度，在此不妨取 $1 / T_{\mathrm{D}} = 0.1$ 。

取 $1/T_{1}$ 为 $1/T_{D}$ 的 1/20，即 $1/T_{1}=0.005$ 。如果 $1/T_{1}$ 与 $1/T_{D}$ 的比值再大一些，就会对穿越频率处的相位产生负面影响，从而降低相位裕度。此外，我们总是希望在低于穿越频率 $\omega_{c}$ 的频段上，系统有较大的幅值，这样可以保证系统有较快的暂态响应速度和较小的误差。因此在设计中 $1/T_{1}$ 与 $1/T_{D}$ 的取值应尽量大一些。为了使相位增加较大，取 $1/T_{D}=0.05$ ，这将允许 $1/T_{1}$ 有较大的值，可以使控制器的积分部分提供较快的响应。注意这个系统中传感器的分离点在 2rad/s 处，它会限制 $1/T_{D}$ 可以选择的最大值。习题 6.63 对这个系统使用了交替设计的方法。

剩下的任务就是确定PID控制器的比例系数 $K$ 了。例6.18中选择 $K$ 值是从满足系统的稳态误差的角度考虑的，而这里是为了得到期望的穿越频率，使系统的相位裕度为期望的 $65^{\circ}$ 。在6.6节讨论过确定 $K$ 的取值的过程。首先取 $K = 1$ ，画出补偿后系统的幅频曲线，求出在期望穿越频率处的幅值，就是 $1 / K$ 。从图6.68中，可以看出 $K = 1$ 时，在 $\omega_{\mathrm{c}} = 0.5\mathrm{rad / s}$ 处，有 $\left|D(s)G(s)\right| = 20$ ，因此 $1 / K = 20$ ， $K = 0.05$ 。现在能够满足所有性能指标要求的PID控制器的表达式为

$$D _ {\mathrm{c}} (s) = \frac {0 . 0 5}{s} [ (1 0 s + 1) (s + 0. 0 0 5) ]$$
