$\begin{array} { r } { \mathrm { w h e r e } , \quad Q _ { u f _ { i } } = \mathbb { E } [ w _ { u f _ { i } } w _ { u f _ { i } } ^ { \iota } ] , \quad Q _ { v } = \mathbb { E } \vert w _ { v } w _ { v } ^ { \iota } \vert } \end{array}$

The predicted error covariance $P ^ { - }$ is predicted with foot model noise $Q _ { f _ { i } } = \mathbb { E } [ w _ { f _ { i } } w _ { f _ { i } } ^ { T } ]$ . The covariance of innovation can be approximated using a moving window as follows:

$$E [ e _ {i} e _ {i} ^ {T} ] \approx \frac {1}{m} \sum_ {n = 0} ^ {m - 1} e _ {i} (k - n) e _ {i} (k - n) ^ {T} = U _ {i} (k) \tag {29}$$

In this case, a small window size (5-10) is used to respond quickly to slips. If the number of data points is less than the size of the moving window, the past innovation is assumed to be zero. The estimated unknown covariance of foot model noise $\hat { Q } _ { u f _ { i } }$ is derived as follows:

$$\hat {Q} _ {u f _ {i}} = R _ {B} ^ {T} (U _ {i} - H _ {v} P _ {k} ^ {-} H _ {v} ^ {T}) R _ {B} - Q _ {v} \tag {30}$$

Here, $Q _ { v }$ also serves to adjust the zero point in the covariance estimation. Therefore, $Q _ { v }$ can be tuned such that the $\hat { Q } _ { u f _ { i } }$ becomes close to zero in the static state. We assume that the $\hat { Q } _ { u f _ { i } }$ is α times the constant covariance $Q _ { f _ { i } }$ . The scaling factor matrix α is diagonal. Its each diagonal element $\alpha _ { j j }$ computed as the ratio of the corresponding diagonal elements of $\hat { Q } _ { u f _ { i } }$ and $Q _ { f _ { i } }$ :

$$\alpha_ {j j} = \frac {\hat {Q} _ {u f _ {i} , j j}}{Q _ {f _ {i} , j j}} \tag {31}\alpha_ {j j} \leftarrow \min \left(\max (\alpha_ {j j}, 1), \alpha_ {\max}\right) \quad \text { for } \quad j = x, y, z.$$

To prevent filter divergence, the scaling factor matrix α is clipped to have maximum and minimum values. Adjust the foot noise covariance as:

$$\hat {Q} _ {f _ {i}} \leftarrow \alpha Q _ {f _ {i}}, \tag {32}$$

The covariance prediction step (14a) is performed again with $\hat { Q } _ { f _ { i } }$ .
