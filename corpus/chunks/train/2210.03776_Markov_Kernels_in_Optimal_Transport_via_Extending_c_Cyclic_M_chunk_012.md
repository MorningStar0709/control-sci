# 4.1. Discrete Optimal Transport and Stochastic Matrices

Consider optimal transport between probability vectors $\mu ,$ and $\nu ,$ and a cost function $c ( x , y ) \leq c _ { X } ( x ) + c _ { Y } ( y )$ . The optimal coupling between $\mu = \left( \mu _ { i } \right)$ , and $\nu = \left( \nu _ { j } \right)$ is then $\mathrm { a \ c { - } C M }$ array $\gamma = \left( \gamma _ { i , j } \right)$ such that

$$\mu_ {i} = \sum_ {j} \gamma_ {i, j}, \text { and } \nu_ {j} = \sum_ {i} \gamma_ {i, j}. \tag {3}$$

Equation (3) is called the marginalization condition and is often written as $\mu = \gamma \mathbb { 1 }$ , and $\nu ^ { T } = \mathbb { 1 } ^ { T } \gamma$ , although $\gamma$ is not used as a matrix in any other context. Note that $\mu$ is the sum of the columns of $\gamma _ { : }$ while ν is the sum of the rows.

If we assume $\mu$ and ν have all non-zero entries (since the corresponding row or

column of $\gamma$ would otherwise be all zeros), then we can form the array

$$S _ {i, j} = \frac {\gamma_ {i , j}}{\nu_ {j}}. \tag {4}$$

Summing the entries of each column of S, we have

$$
\begin{array}{l} \sum_ {i} S _ {i, j} = \sum_ {i} \frac {\gamma_ {i , j}}{\nu_ {j}} \\ = \frac {1}{\nu_ {j}} \sum_ {i} \gamma_ {i, j} \\ \end{array}
= \frac {1}{\nu_ {j}} \nu_ {j} = 1. \tag {5}$$

This shows that $S$ is a stochastic matrix, and by construction we have

$$S \nu = \sum_ {j} \frac {\gamma_ {i , j}}{\nu_ {j}} \nu_ {j} = \mu .$$

We can form a stochastic matrix that sends $\mu$ to ν similarly. Because $\mu$ and ν represent measures, it is more suitable to treat them as row vectors moving forward and to view it as $\tilde { S } = ( \gamma _ { i , j } / \mu _ { i } )$ , and $\mu \tilde { S } = \nu$ . The vectors $\mu$ and ν were chosen to be non-zero to avoid dividing by $0 ,$ , however by restricting to the support of the measures, we avoid any difficulty. In the row (column) corresponding to a zero value of $\mu \ \left( \nu \right)$ , all entries will be zero since we are summing non-negative values to 0. This point will be revisited when we extend this concept from vector measures to the more general case.

The construction of the stochastic matrix $\tilde { S }$ becomes useful when considering a measure $\eta$ that is absolutely continuous with respect to $\mu .$ Every measure on the same space as $\mu$ will be absolutely continuous with respect to $\mu$ since it is nonzero everywhere. Additionally, the Radon-Nikodym derivative will be bounded by the inverse of the smallest entry in $\mu .$ .
