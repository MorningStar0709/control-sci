# 2.2. The stochastic liquidity processes

In the following we introduce the stochastic resilience and price impact processes and some related quantities that we need to set up and solve the optimal trade execution problem.

Let $\rho ~ = ~ ( \rho ( s ) ) _ { s \in [ 0 , T ] }$ be an $\mathbb { R } ^ { n \times n } .$ -valued progressively measurable process which is $d P \times d s { \mathrm { - a . e } }$ . bounded. We call $\rho$ the resilience process. Let $\nu = ( \nu ( s ) ) _ { s \in [ 0 , T ] }$ be the unique solution of

$$d \nu (s) = \nu (s) \rho (s) d s, \quad s \in [ 0, T ], \quad \nu (0) = I _ {n}, \tag {4}$$

and observe that the inverse $\nu ^ { - 1 } = ( \nu ^ { - 1 } ( s ) ) _ { s \in [ 0 , T ] }$ is the unique solution of

$$d \nu^ {- 1} (s) = - \rho (s) \nu^ {- 1} (s) d s, \quad s \in [ 0, T ], \quad \nu^ {- 1} (0) = I _ {n}. \tag {5}$$

Next, we construct the price impact process. To this end let $\mu = ( \mu ( s ) ) _ { s \in [ 0 , T ] }$ be an $\mathbb { R } ^ { n }$ -valued progressively measurable process and let $\sigma = ( \sigma ( s ) ) _ { s \in [ 0 , T ] }$ be an $\mathbb { R } ^ { n \times m } .$ -valued progressively measurable process. Assume that $\mu$ and σ are $d P \times d s { \mathrm { - a . e } }$ . bounded. For each $j \in \{ 1 , \ldots , n \}$ let $\lambda _ { j } = ( \lambda _ { j } ( s ) ) _ { s \in [ 0 , T ] }$ be the $( 0 , \infty )$ -valued stochastic process given as the unique solution of

$$d \lambda_ {j} (s) = \lambda_ {j} (s) \mu_ {j} (s) d s + \sum_ {k = 1} ^ {m} \lambda_ {j} (s) \sigma_ {j, k} (s) d W _ {k} (s), \quad s \in [ 0, T ], \quad \lambda_ {j} (0) \in (0, \infty). \tag {6}$$

For every $\alpha \in \mathbb { R }$ let $\lambda ^ { \alpha } = ( \lambda ^ { \alpha } ( s ) ) _ { s \in [ 0 , T ] }$ be the $\mathbb { S } _ { > 0 } ^ { n }$ -valued stochastic process given by

$$\boldsymbol {\lambda} ^ {\alpha} (s) = \mathrm{diag} ((\lambda_ {1} (s)) ^ {\alpha}, \dots , (\lambda_ {n} (s)) ^ {\alpha}), \quad s \in [ 0, T ]. \tag {7}$$

Furthermore, let $\pmb { \mu } = ( \pmb { \mu } ( s ) ) _ { s \in [ 0 , T ] }$ be defined by

$$\pmb {\mu} (s) = \mathrm{diag} (\mu_ {1} (s), \dots , \mu_ {n} (s)), \quad s \in [ 0, T ],$$
