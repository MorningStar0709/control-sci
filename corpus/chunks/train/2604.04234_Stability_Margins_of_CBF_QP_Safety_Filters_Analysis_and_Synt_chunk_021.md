# C. Certified Phase- and Delay-Robustness Bounds

The gain case is especially simple because the perturbation enters affinely in κ. Phase robustness can also be certified, but now through a conservative sufficient condition. The phase-uncertainty set is an arc of the unit circle, which is not directly convenient for LMI-based certification. Enclosing this arc in a disk yields a conservative but tractable uncertainty description, reducing the problem to robust stability with respect to a bounded scalar complex perturbation.

Lemma 3. Fix $\bar { \phi } \in [ 0 , \pi / 2 ]$ , and define $c _ { \phi } : = \cos \bar { \phi } , r _ { \phi } : =$ sin ϕ¯. Then,

$$\{e ^ {- j \phi}: | \phi | \leq \bar {\phi} \} \subseteq \{\Delta \in \mathbb {C}: | \Delta - c _ {\phi} | \leq r _ {\phi} \}. \tag {36}$$

Proof. Let $\Delta = e ^ { - j \phi } = \cos \phi - j$ sin ϕ with $| \phi | \leq \bar { \phi }$ . Then,

$$| \Delta - c _ {\phi} | ^ {2} = (\cos \phi - c _ {\phi}) ^ {2} + \sin^ {2} \phi = 1 - 2 c _ {\phi} \cos \phi + c _ {\phi} ^ {2}.$$

Since $| \phi | \leq \bar { \phi }$ and $\bar { \phi } \in [ 0 , \pi / 2 ]$ , we have cos $\phi \ge \cos \bar { \phi } = c _ { \phi }$ Hence

$$| \Delta - c _ {\phi} | ^ {2} \leq 1 - c _ {\phi} ^ {2} = \sin^ {2} \bar {\phi} = r _ {\phi} ^ {2},$$

which proves (36).

Now, define

$$A _ {\phi} := A _ {0} - c _ {\phi} U V ^ {\top}. \tag {37}$$

For every complex scalar ∆ in the enclosing disk, one can write $\Delta = c _ { \phi } + \delta _ { \phi }$ with $\delta \in \mathbb { C }$ satisfying $| \delta | \leq r _ { \phi }$ . Therefore $\tilde { A } ( \Delta ) = A _ { \phi } - \delta U V ^ { \top }$ . Thus the phase-certification problem reduces to robust stability with respect to the bounded complex scalar perturbation δ.

The next result gives a tractable sufficient condition for certifying a phase range. The key idea is to use the disk enclosure from Lem. 3, rewrite the perturbed active mode as a feedback interconnection with a bounded complex scalar uncertainty, and then apply the bounded real lemma.

Proposition 8. Let $A _ { \phi }$ be defined by (37). Suppose there exists $P = P ^ { \top } \succ 0$ such that

$$
\left[ \begin{array}{c c c} A _ {\phi} ^ {\top} P + P A _ {\phi} & P U & r _ {\phi} V \\ U ^ {\top} P & - 1 & 0 \\ r _ {\phi} V ^ {\top} & 0 & - 1 \end{array} \right] \prec 0. \tag {38}
$$
