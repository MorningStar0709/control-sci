# D. Agent Dynamics

We consider a linear time-invariant (LTI) of the form

$$\dot {x} = A (x - x _ {s p}) \tag {2}$$

where $x \in \mathbb { R } ^ { n }$ is the state of the system, and the origin is translated to the setpoint $x _ { s p } \in \mathbb { R } ^ { n }$ . The matrix $A \in \mathbb { R } ^ { n \times n }$ is Hurwitz $( \mathrm { i . e . , ~ R e } ( \lambda _ { i } ) ~ < ~ 0 )$ , where $\lambda _ { i } ~ \in \ \mathbb { C }$ is the i-th eigenvalue of A. The matrix A can be seen as the closedloop system matrix obtained by applying a state-feedback controller $( { \mathrm { i . e . , ~ } } u = - K ( x - x _ { s p } ) )$ ), where u is the control input.

For the asymptotically stable LTI system (2), there exists a quadratic Lyapunov function of the form

$$V (x) = \left(x - x _ {s p}\right) ^ {T} P \left(x - x _ {s p}\right) \tag {3}$$

where $P$ is a symmetric positive definite matrix that satisfies the Lyapunov equation

$$A ^ {T} P + P A = - Q \tag {4}$$

for a given symmetric positive definite matrix $Q > 0$ .

Defining the P -norm as

$$\left\| x - x _ {s p} \right\| _ {P} \triangleq \sqrt {(x - x _ {s p}) ^ {T} P (x - x _ {s p})}, \tag {5}$$

for a fixed value $c > 0 .$ , the Lyapunov function (3) defines an ellipsoid, which can be written in terms of (5) as follows:

$$\mathcal {E} _ {c} (x _ {s p}) = \{x \in \mathbb {R} ^ {n} \mid \| x - x _ {s p} \| _ {P} ^ {2} \leq c \}. \tag {6}$$

From (4), $V ( x ( t ) )$ is a monotonically decreasing function; therefore, for any x(t) starting within the ellipsoid (6), the trajectory never leaves it. Hence, $\mathcal { E } _ { c }$ is called a positively invariant set [13]. For the rest of the paper, we will refer to the state of the agent i as $x _ { i }$ and for the related setpoint as $x _ { s p , i } .$ . In this context, the tracking algorithm updates the setpoint of the agent to reach a desired position. This formulation can be extended to more general nonlinear systems by using linearization [13], or Lyapunov-like methods such as quadratic boundedness [14], as used in the context of UAVs in [7], where ellipsoidal positively invariant sets are provided. We use this last case in our experiments.
