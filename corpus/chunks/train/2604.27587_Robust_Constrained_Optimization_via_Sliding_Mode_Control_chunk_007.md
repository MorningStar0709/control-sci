# 2.2 Preliminaries

Consider an optimization problem with equality constraints:

$$\min _ {x \in \mathbb {R} ^ {n}} \phi (x) \quad \text { subject to } \quad h (x) = 0, \tag {1}$$

where $\phi : \mathbb { R } ^ { n }  \mathbb { R }$ and $h : \mathbb { R } ^ { n }  \mathbb { R } ^ { m }$ are continuously differentiable (possibly non-convex) functions. The Lagrangian is defined as: $\mathcal { L } ( x , \lambda ) = \dot { \phi } ( x ) + \lambda ^ { \top } h ( x )$ , with Lagrange multipliers $\boldsymbol { \lambda } \in \mathbb { R } ^ { m }$ .

Lemma 1 Let $x ^ { * }$ be a local minimum of φ, where h is continuously differentiable and the constraint qualification ran $k ( J _ { h } ( x ^ { * } ) ) = m h o l d s$ . Then, there exists $\bar { \lambda } ^ { * } \in \mathbb { R } ^ { m }$ such that:

First-order necessary condition (FONC):

$$\nabla_ {x} \mathcal {L} (x ^ {*}, \lambda^ {*}) = \nabla \phi (x ^ {*}) + J _ {h} (x ^ {*}) ^ {\top} \lambda^ {*} = 0, h (x ^ {*}) = 0, \tag {2}$$

where $J _ { h } ( x ) \in \mathbb { R } ^ { m \times n }$ is the Jacobian of the constraint function $\dot { h } ( x )$ .

Second-order necessary condition $( S O N C )$ :

For non-convex problems, the first-order conditions identify critical points that may be minima, maxima, or saddle points. The second-order condition uses the Hessian $\hat { \nabla } _ { x x } ^ { 2 } \mathcal { L } ( x ^ { * } , \lambda ^ { * } )$ to further classify the point: $d ^ { \top } \nabla _ { x x } ^ { 2 } \mathcal { L } ( x ^ { * } , \lambda ^ { * } ) d \geq 0 \quad \forall \bar { d } \in \ker ( J _ { h } ( x ^ { * } ) )$ .

The FONC and SONC for equality-constrained optimization problems arise naturally from the KKT framework, which generalizes the method of Lagrange multipliers [13].
