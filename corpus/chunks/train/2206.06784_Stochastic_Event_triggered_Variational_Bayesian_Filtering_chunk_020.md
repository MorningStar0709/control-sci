$$E ^ {i} \left\{\log \mu_ {j, k} \right\} = \psi \left(\alpha_ {j, k | k} ^ {i}\right) - \psi \left(\sum_ {j = 1} ^ {M} \alpha_ {j, k | k} ^ {i}\right),$$

in which ψ(·) is the digamma function.

Define a new variable

$$\chi_ {j, k} ^ {i + 1} = \exp (\tau_ {j, k} ^ {i + 1} + E ^ {i} \{\log \mu_ {j, k} \}). \tag {40}$$

Based on the categorical distribution, $\hat { \chi } _ { k } ^ { i + 1 }$ is updated by

$$\hat {\chi} _ {k} ^ {i + 1} = \chi_ {k} ^ {i + 1} / \sum_ {j = 1} ^ {M} \chi_ {j, k} ^ {i + 1}.$$

Similarly, according to the Dirichlet distribution, $\alpha _ { k | k } ^ { i + 1 }$ is updated by

$$\alpha_ {k | k} ^ {i + 1} = \alpha_ {k | k - 1} + E ^ {i + 1} \{\lambda_ {k} \},$$

where $E ^ { i + 1 } \{ \lambda _ { k } \} = \hat { \chi } _ { k } ^ { i + 1 }$
