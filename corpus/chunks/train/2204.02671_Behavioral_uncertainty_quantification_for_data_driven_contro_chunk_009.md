Remark 3 (Data-based computation). The L-gap between behaviors can be directly computed from the knowledge of sufficiently excited trajectories. Let $w ^ { d } \in B | _ { T }$ and $\tilde { w } ^ { d } \in \tilde { B } | _ { T }$ be sufficiently excited T -length trajectories of order $L ,$ with $L > l ,$ . Let

$$
H _ {L} (w ^ {d}) = \left[ U _ {1}   U _ {2} \right] \left[ \begin{array}{c c} S & 0 \\ 0 & 0 \end{array} \right] \left[ \begin{array}{c} V _ {1} \\ V _ {2} \end{array} \right],

H _ {L} (\tilde {w} ^ {d}) = [ \tilde {U} _ {1} \tilde {U} _ {2} ] \left[ \begin{array}{c c} \tilde {S} & 0 \\ 0 & 0 \end{array} \right] \left[ \begin{array}{c} \tilde {V} _ {1} \\ \tilde {V} _ {2} \end{array} \right]
$$

be the singular value decomposition (SVD) of the Hankel matrices $H _ { L } ( w ^ { d } )$ and $H _ { L } ( \tilde { w } ^ { d } )$ with $U _ { 1 } \in \mathbb { R } ^ { q L \times ( m L + n ) }$ a n d $\tilde { U } _ { 1 } \in \mathbb { R } ^ { q L \times ( \grave { m } L + \ ' { n } ) }$ , respectively. Then

$$\operatorname{gap} _ {L} (\mathcal {B}, \tilde {\mathcal {B}}) = \| U _ {1} U _ {1} ^ {\mathsf {T}} - \tilde {U} _ {1} \tilde {U} _ {1} ^ {\mathsf {T}} \| _ {2} = \| \tilde {U} _ {2} ^ {\mathsf {T}} U _ {1} \| _ {2} \tag {5}$$

where the first equality comes from the fact that $\mathrm { g a p } _ { L } ( \boldsymbol { \mathcal { B } } , \tilde { \boldsymbol { B } } ) = \| P _ { \boldsymbol { \mathcal { B } } | _ { L } } - P _ { \tilde { \boldsymbol { B } } | _ { L } } \| _ { \mathcal { I } }$ and since $P _ { B | _ { L } } = U _ { 1 } U _ { 1 } ^ { \mathsf { T } }$ and $P _ { \tilde { B } | _ { L } } = \tilde { U } _ { 1 } \tilde { U } _ { 1 } ^ { \top } \ [ 2 9 , \mathrm { p } . 8 2 ]$ . The second identity is due to [29, Thm 2.5.1]. △

Remark 4 (Interpretation in terms of trajectories). Given $\boldsymbol { B } \in \mathcal { L } ^ { q , c }$ , consider the problem of estimating the closest trajectory $w \in B | _ { L }$ to a given measured trajectory $\tilde { w } \in \mathbb { R } ^ { q L }$ which belongs to a possibly distinct behavior $\tilde { B } \in \mathcal { L } ^ { q , c }$ , i.e.,

$$\underset {w \in \mathcal {B} | _ {L}} {\text { minimize }} \quad \| w - \tilde {w} \| _ {2} ^ {2},\text { subject to } \quad \tilde {w} \in \tilde {\mathcal {B}} | _ {L}.$$

By Lemma 1, $\boldsymbol { B } | _ { L }$ is a subspace and the estimation error is

$$\inf _ {w \in \mathcal {B} | _ {L}} \| w - \tilde {w} \| _ {2} = \left\| (I - P _ {\mathcal {B} | _ {L}}) \tilde {w} \right\| _ {2}. \tag {6}$$

Now suppose $\tilde { B }$ is known to be such that $\mathrm { g a p } _ { L } ( B , \tilde { B } ) \le \epsilon$ Then

$$
\sup_{\substack{\tilde{w}\in \tilde{\mathcal{B}}|_{L}\\ \| \tilde{w}\|_{2}\neq 0}}\inf_{w\in \mathcal{B}|_{L}}\frac{\|w - \tilde{w}\|_{2}}{\|\tilde{w}\|_{2}}\leq \epsilon .
$$

In other words, $\mathrm { g a p } _ { L } ( B , \tilde { B } )$ is an upper bound for the worst case relative estimation error. The domain of the $L _ { - }$ gap metric may be extended to measure distances between subspaces of different dimension [30], so these results may be used, e.g., for smoothing of a noisy trajectory w˜.
