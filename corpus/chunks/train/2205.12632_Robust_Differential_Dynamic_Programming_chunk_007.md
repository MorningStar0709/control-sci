This overestimation is now used to compute the approximate value function $\widetilde { V } _ { t } ( \cdot )$ by demanding

$$\widetilde {V} _ {t} (x) \geq \underset {\boldsymbol {M} _ {t} \in \mathcal {M}, \delta_ {t} ^ {u}} {\text { minimize }} \underset {\delta_ {t} ^ {w}} {\text { maximize }} \overline {{Q}} _ {t} (x, u, w), \tag {10}$$
