# 2) Global Stability Notions for Input-Free Systems

Let $f : \mathbb { R } ^ { n } \to \mathbb { R } ^ { n }$ be a locally Lipschitz vector field with $f ( 0 ) = 0$ and $h : \mathbb { R } ^ { n }  \mathbb { R } ^ { p }$ be a continuous mapping with $h ( 0 ) = 0$ . Consider the dynamical system

$$\dot {x} = f (x), x \in \mathbb {R} ^ {n} \tag {1}$$

with output

$$y = h (x) \tag {2}$$

We assume that the dynamical system (1) is forward complete, i.e., for every $x _ { 0 } \in \mathbb { R } ^ { n }$ the unique solution $x ( t ) = \phi ( t , x _ { 0 } )$ of the initial-value problem (1) with initial condition $x ( 0 ) = x _ { 0 }$ exists for all $t \geq 0$ . We use the notation $y ( t , x _ { 0 } ) = h ( \phi ( t , x _ { 0 } ) )$ for all $t \geq 0 , x _ { 0 } \in \mathbb { R } ^ { n }$ and $B _ { R } = \left\{ \left. x \in \mathbb { R } ^ { n } : \middle | x \right| < R \right\}$ for all $R > 0$ .We say that system (1), (2) is

i) Lagrange output stable if for every $R > 0$ the set $\left\{ \big | y ( t , x _ { 0 } ) \big | { : } x _ { 0 } \in B _ { R } , t \geq 0 \right\}$ is bounded.   
ii) Lyapunov output stable if for every $\varepsilon > 0$ there exists $\delta ( \varepsilon ) > 0$ such that for all $x _ { 0 } \in B _ { \delta ( \varepsilon ) }$ , it holds that $\left| y ( t , x _ { 0 } ) \right| \leq \varepsilon$ for all $t \geq 0$   
iii) Globally Asymptotically Output Stable (GAOS) if system (1), (2) is Lagrange and Lyapunov output stable and $\operatorname* { l i m } _ { t \to + \infty } \bigl ( y ( t , x _ { 0 } ) \bigr ) = 0$ for all $\boldsymbol { x } _ { 0 } \in \mathbb { R } ^ { n }$   
iv) Uniformly Globally Asymptotically Output Stable (UGAOS) if system (1), (2) is Lagrange and Lyapunov output stable and for every $\varepsilon , R > 0$ there exists $T ( \varepsilon , R ) > 0$ such that for all $x _ { 0 } \in B _ { R }$ it holds that $\left| y ( t , x _ { 0 } ) \right| \leq \varepsilon$ for all $t \geq T ( \varepsilon , R )$

We say that system (1), (2) is Globally Output Attractive (GOA) if $\operatorname* { l i m } _ { t  + \infty } \big ( y ( t , x _ { 0 } ) \big ) = 0$ for all $\boldsymbol { x _ { 0 } } \in \mathbb { R } ^ { n }$ . We say that system (1), (2) is practically Globally Output Attractive $( p { - } G O A )$ if there exists a constant $\tilde { \alpha } > 0$ such that $\operatorname* { l i m } _ { t \to + \infty } \operatorname* { s u p } _ { } \bigl ( | y ( t , x _ { 0 } ) | \bigr ) \leq \tilde { \alpha }$ for all $\boldsymbol { x _ { 0 } } \in \mathbb { R } ^ { n }$ .The constant $\tilde { \alpha } > 0$ is called the asymptotic residual constant.
