# B. Adaptive Control Lyapunov Function

The adaptive control literature offers a great variety of tackling partially or completely unknown system dynamics; one particular approach involves an adaptive control Lyapunov function as an extension of the classical control Lyapunov functions [41].

Definition 1 (Adaptive CLF [23]): A smooth function $V$ : $\mathbb { R } ^ { n } \times \mathbb { R } ^ { p } \to \mathbb { R } _ { \geq 0 } ,$ , p. d. and proper in x and θ is called an adaptive control Lyapunov function for (1) if there exists a p. d. matrix $\Gamma \in \mathbb { R } ^ { p \times p }$ such that for each $\theta \in \mathbb { R } ^ { p } , V ( x , \theta )$ is a CLF for the modified system

$$\dot {x} = \underbrace {f (x) + F (x) (\theta + \Gamma \nabla_ {\theta} V (x , \theta)) + g (x) u} _ {=: G (x, u, \theta)}. \tag {4}$$

That is, for any bounded subset $\mathbb { X } \subset \mathbb { R } ^ { n }$ containing the origin there exists a compact set $\mathbb { U } _ { \mathbb { X } } \subset \mathbb { R } ^ { m }$ and a continuous function $\nu : \mathbb { R } ^ { n } \times \mathbb { R } ^ { p } \to \mathbb { R } _ { \ge 0 } , \ p . { \mathrm { d } }$ . in x for all θ, such that

$$\forall \theta \in \mathbb {R} ^ {p}, \forall x \in \mathbb {X}, \exists u \in \mathbb {U} _ {\mathbb {X}}, \tag {5}\langle \nabla_ {x} V (x, \theta), G (x, u, \theta) \rangle \leq - \nu (x, \theta).$$

Given V , a policy $u = \mu ( x , \theta ) : \mathbb { R } ^ { n } \times \mathbb { R } ^ { p }  \mathbb { R } ^ { m }$ can be constructed according to, e. g., Sontag’s formula [41].

Furthermore, for $\Theta \in \mathbb { R } ^ { p }$ , there exists $\alpha _ { 1 , 2 } ^ { \theta } \in { \mathcal { K } } _ { \infty }$ such that $\alpha _ { 1 } ^ { \theta } ( \| x \| ) \leq V ( x , \theta ) \leq \alpha _ { 2 } ^ { \theta } ( \| x \| )$ for all $\theta \in { \dot { \Theta } } .$ .

Along the control process only an estimate $ { \hat { \theta } } =  { \hat { \theta } } ( t )$ of the unknown θ is available. Within the adaptive control Lyapunov function framework it is suggested to use

$$\dot {\hat {\theta}} = \Gamma \tau (x, \hat {\theta}), \tau (x, \hat {\theta}) = \left(\nabla_ {x} ^ {\top} V (x, \hat {\theta}) F (x)\right) ^ {\top}. \tag {6}$$
