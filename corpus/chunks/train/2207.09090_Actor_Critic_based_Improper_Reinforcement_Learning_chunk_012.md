# 2.2. Scheduling in Constrained Queueing Networks

We consider a system that comprises two queues fed by independent, stochastic arrival processes $A _ { i } ( t ) , i \in$ $\{ 1 , 2 \} , t \in \mathbb { N } .$ . The length of queue $i ,$ measured at the beginning of time slot $t ,$ is denoted by $Q _ { i } ( t ) \ \in \ \mathbb { Z } _ { + }$ . A common server serves both queues and can drain at most one packet from the system in a time slot. The server, therefore, needs to decide which of the two queues it intends to serve in a given slot (we assume that once the server chooses to serve a packet, service succeeds with probability 1). The server’s decision is denoted by the vector $\mathbf { D } ( t ) \in \mathcal { A } : = \{ [ 0 , 0 ] , [ 1 , 0 ] , [ 0 , 1 ] \}$ , where $\mathrm { ~ a ~ } ^ { 6 6 } 1 ^ { \circ }$ denotes service and a $\cdot \ ^ { 6 6 } 0 ^ { 9 }$ denotes lack thereof. Let $\mathbb { E } A _ { i } ( t ) = \lambda _ { i } .$ , and note that the arrival rate $\pmb { \lambda } = [ \lambda _ { 1 } , \lambda _ { 2 } ]$ is unknown to the learner. We aim to find a (potentially randomized) policy π to minimize the discounted system backlog given by $J _ { \pi } ( \mathbf { Q } ( 0 ) ) : = \mathbb { E } _ { \mathbf { Q } ( 0 ) } ^ { \pi } \sum _ { t = 0 } ^ { \infty } \gamma ^ { t } \left( { Q } _ { 1 } ( t ) + { Q } _ { 2 } ( t ) \right)$ .
