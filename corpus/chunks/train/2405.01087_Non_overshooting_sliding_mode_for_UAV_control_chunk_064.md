# A. Robust non-overshooting reachability of the first subsystem

Case one: If $e _ { 1 } \left( t \right) > e _ { 1 c } ,$ for system (18), we get

$$\frac {d \left(e _ {2} (t) + e _ {2 c}\right)}{d t} = - k _ {c} \text { sign } \left[ e _ {2} (t) + e _ {2 c} \right] + d (t) \tag {78}$$

A Lyapunov function candidate is selected as

$$V _ {c} = \frac {1}{2} \left(e _ {2} (t) + e _ {2 c}\right) ^ {2} \tag {79}$$

Then, taking the derivative for $V _ { c } ,$ we get

$$\dot {V} _ {c} = \left(e _ {2} (t) + e _ {2 c}\right) \left\{- k _ {c} \mathrm{sign} [ e _ {2} (t) + e _ {2 c} ] + d (t) \right\}\leq - \left(k _ {c} - L _ {d}\right) \left| e _ {2} (t) + e _ {2 c} \right| = - \sqrt {2} \left(k _ {c} - L _ {d}\right) V _ {c} ^ {\frac {1}{2}} \tag {80}$$

We know that $k _ { c } > L _ { d }$ . Therefore, there exists a finite time $t _ { c 1 } > 0$ , for $\textit { t } \geq t _ { c 1 }$ , such that $e _ { 2 } \left( t \right) = - e _ { 2 c }$ . According to ${ \dot { e } } _ { 1 } = e _ { 2 }$ , we get that $\dot { e } _ { 1 } \left( t \right) = - e _ { 2 c }$ for $t \geq t _ { c 1 }$ . Therefore, there exists a finite time $t _ { c } > 0$ , for $t \geq t _ { c } .$ , such that $e _ { 1 } \left( t \right) \leq e _ { 1 c }$ .

Case two: If $e _ { 1 } \left( t \right) < - e _ { 1 c } ,$ for system (18), we get

$$\frac {d \left(e _ {2} (t) - e _ {2 c}\right)}{d t} = - k _ {c} \text {sign} \left[ e _ {2} (t) - e _ {2 c} \right] + d (t) \tag {81}$$

Similar method to case $e _ { 1 } \left( t \right) > e _ { 1 c }$ , when we select the Lyapunov function candidate as $V _ { c } =$ $\begin{array} { r } { \frac { 1 } { 2 } \left( e _ { 2 } \left( t \right) - e _ { 2 c } \right) ^ { 2 } } \end{array}$ , we can get that $e _ { 1 } \left( t \right) \geq - e _ { 1 c }$ .

Combining cases one and two, we can get that $| e _ { 1 } \left( t \right) | \le e _ { 1 c }$ for $t \geq t _ { c } .$ .
