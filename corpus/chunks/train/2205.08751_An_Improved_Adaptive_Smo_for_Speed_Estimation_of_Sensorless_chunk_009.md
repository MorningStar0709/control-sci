# REFERENCE FRAMES

The required transformation in voltages, currents, or flux linkages is derived in a generalized way. The reference frames are chosen to be arbitrarily such as stationary, rotor and synchronous reference frames. R.H. Park, in the 1920s, proposed a new theory of electrical machine analysis to represent the machine in $\mathrm { ~ d ~ - ~ q ~ }$ model. He transformed the stator variables to a synchronously rotating reference frame fixed in the rotor, which is called Park’s transformation.[2]- [3]

He showed that all the time varying inductances that occur due to an electric circuit in relative motion and electric circuits with varying magnetic reluctances could be eliminated. In 1930s, H.C Stanley showed that time varying Inductances in the voltage equations of an induction machine due to electric circuits in relative motion can be eliminated by transforming the rotor variables to a stationary reference frame fixed on the stator. Later, G. Kron proposed a transformation of both stator and rotor variables to a synchronously rotating reference that moves with the rotating magnetic field.
