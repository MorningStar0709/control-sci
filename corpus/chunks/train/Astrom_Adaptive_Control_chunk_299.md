# 5.5 DESIGN OF MRAS USING LYAPUNOV THEORY

We will now show how Lyapunov's stability theory can be used to construct algorithms for adjusting parameters in adaptive systems. To do this, we first derive a differential equation for the error, $e = y - y_{m}$ . This differential equation contains the adjustable parameters. We then attempt to find a Lyapunov function and an adaptation mechanism such that the error will go to zero. When using the Lyaponov theory for adaptive systems, we find that dV/dt is usually only negative semidefinite. The procedure is to determine the error equation and a Lyapunov function with a bounded second derivative. Theorem 5.4 is then used to show boundedness and that the error goes to zero. To show parameter convergence, it is necessary to impose further conditions, such as persistently excitation and uniform observability, on the reference signal and the system.
