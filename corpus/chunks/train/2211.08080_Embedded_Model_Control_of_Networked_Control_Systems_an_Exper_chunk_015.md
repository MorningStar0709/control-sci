# 3.2 EMC mobile robot test - Critical asynchronous sampling times

Figure 5 shows the estimated output of the EMC internal model ${ \bf y } _ { m }$ , for several asynchronous sampling times $T _ { s } ;$ spanning from a fixed minimum value of 10 ms, to a variable maximum value of [50, 100, 150, 200] ms. For this experimental test, the reference motor speed is imposed to a fixed value of $\overline { { \mathbf { y } } } = 6$ rad/sec. The estimation of ${ \bf y } _ { m }$ is effective and characterized by a negligible model error $\mathbf { e } _ { m }$ , until a maximum value $T _ { s } = 1 5 0$ ms, which is sufficiently high to cover a wide range of NCS delay and package loss variations. Only for higher sampling times, $T _ { s } > 1 5 0$ ms, the model output ${ \bf y } _ { m }$ starts to show a degradation in estimation performance.
