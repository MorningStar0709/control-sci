# D.5. Proof of Theorem 6

Defining by $p _ { t } ^ { \mathrm { o p t } } : = \mathbb { P } \left\{ \tilde { \Theta } _ { t } \in S ^ { \mathrm { o p t } } \vert \mathcal { F } _ { t } ^ { \mathrm { c n t } } , \hat { E } _ { t } \right\}$ the optimistic probability, and by $\mathbb { P } _ { t } \{ \cdot \} : = \mathbb { P } \{ \cdot | \mathcal { F } _ { t } ^ { \mathrm { c n t } } \}$ conditional probability measure, we can write

$$p _ {t} ^ {\text { opt }} \geq \mathbb {P} \left\{\tilde {\Theta} _ {t} \in \mathcal {S} ^ {\text { surr }} \mid \mathcal {F} _ {t} ^ {\text { cnt }}, \hat {E} _ {t} \right\} \tag {71}= \mathbb {P} \left\{L (\tilde {\Theta} _ {t} ^ {\intercal} H _ {*}) \leq L (\Theta_ {*} ^ {\intercal} H _ {*}) | \mathcal {F} _ {t} ^ {\mathrm{cnt}}, \hat {E} _ {t} \right\} \tag {72}\geq \min _ {\hat {\Theta} \in \mathcal {E} _ {t} ^ {\mathrm{RLS}}} \mathbb {P} _ {t} \left\{L \left(\hat {\Theta} ^ {\intercal} H _ {*} + \eta^ {\intercal} \beta_ {t} V _ {t} ^ {- \frac {1}{2}} H _ {*}\right) \leq L \left(\Theta_ {*} ^ {\intercal} H _ {*}\right) \right\} \tag {73}= \min _ {\hat {\Theta} \in \mathcal {E} _ {t} ^ {\mathrm{RLS}}} \mathbb {P} _ {t} \left\{L \left(\hat {\Theta} ^ {\intercal} H _ {*} + \Xi \sqrt {F _ {t}}\right) \leq L \left(\Theta_ {*} ^ {\intercal} H _ {*}\right) \right\} \tag {74}$$

where (71) is by Lemma 7, (73) is a worst-case estimation bound within high-probability confidence region, and (74) is because $\eta ^ { \intercal } \beta _ { t } V _ { t } ^ { - \frac { 1 } { 2 } } H _ { \ast }$ and $\Xi \sqrt { F _ { t } }$ have the same distributions with $\boldsymbol { \eta } \in \mathbb { R } ^ { ( n + d ) }$ ×n and $\Xi \in \mathbb { R } ^ { n \times n }$ being i.i.d. standard normal random matrices.

The bound in (74) can be further lower bounded by minimizing over a larger confidence set as

$$p _ {t} ^ {\text { opt }} \geq \min _ {\hat {\Theta} \in \mathcal {E} _ {t} ^ {\mathrm{cl}}} \mathbb {P} _ {t} \left\{L \left(\hat {\Theta} ^ {\intercal} H _ {*} + \Xi \sqrt {F _ {t}}\right) \leq L \left(A _ {c, *}\right) \right\} \tag {75}= \min _ {\hat {\Upsilon}: \| \hat {\Upsilon} \| _ {F} \leq 1} \mathbb {P} _ {t} \left\{L \left(A _ {c, *} + (\Xi + \hat {\Upsilon}) \sqrt {F _ {t}}\right) \leq L \left(A _ {c, *}\right) \right\}, \tag {76}$$

where (75) is by Lemma (8) and (76) is because $H _ { * }$ is full column rank and therefore we can minimize over closed-loop matrices instead of open-loop system parameters.
