# Solution:

A simple example of a set with a supremum is the set $S \ = \ \{ x \ | \ | x | \leq \ 4 2 \}$ . In this set, the supremum $\operatorname* { s u p } s = 4 2$ . In this case, its maximum is also 42.

As for the infimum, if we take the same set S then infS = 0, because of the definition of absolute value. Also in this case, the minimum corresponds to the infimum which is 0. 
