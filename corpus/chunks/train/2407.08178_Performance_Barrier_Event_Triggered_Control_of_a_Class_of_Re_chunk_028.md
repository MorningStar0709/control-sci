# 3.2.3 P-PETC

In this subsection, we present the P-PETC approach derived from the P-CETC scheme (46)-(55), redesigning the triggering function $\Gamma ^ { p } ( t )$ from (49) to $\tilde { \Gamma } ^ { p } ( t )$ for periodic evaluation. This adjustment, coupled with event-triggered control updates, ensures $\Gamma ^ { p } ( t )$ remains non-positive and $m ^ { p } ( t )$ remains positive along the P-PETC closed-loop system solution. As a result, convergence properties equivalent to R4 and R5 in Theorem 1 holds for the P-PETC closedloop system as well.

Let the parameters $\gamma , c , \beta _ { 1 } , \beta _ { 2 } , \rho > 0$ be selected as outlined in Section 3.2.1, and let $\eta > 0$ be selected later.
