# C. Control Barrier Function

The super-level set of the control barrier function (CBF) $h : S  \mathcal { R }$ could validate an safety set $\mathcal { C }$ as

$$\mathcal {C} = \{s _ {t} \in \mathcal {S}: h (s _ {t}) \geq 0 \}. \tag {9}$$

Throughout this paper, we refer to C as a safe region and

$$\mathcal {U} = \mathcal {S} - \mathcal {C} = \{s _ {t} \in \mathcal {S}: h (s _ {t}) < 0 \}, \tag {10}$$

as unsafe regions of the dynamical system (8). Then, the state $s _ { t } \in \mathcal C$ will not enter into U by satisfying the forward invariant constraint below,

$$\forall s _ {t} \in \mathcal {C}: h (s _ {t}) \geq 0, \Delta h (s _ {t}, a _ {t}) \geq 0, \tag {11}$$

where $\Delta { h } ( s _ { t } , a _ { t } ) = h ( s _ { t + 1 } ) - h ( s _ { t } )$ . Before we demonstrate the computation of h of the system (8), the Positivestellensatz (P-satz) needs to be introduced first [33].

Let P be the set of polynomials and $\mathcal { P } ^ { \mathrm { S O S } }$ be the set of sum of squares polynomials, e.g., $\begin{array} { r } { P ( x ) = \sum _ { i = 1 } ^ { k } p _ { i } ^ { 2 } ( x ) } \end{array}$ , where $p _ { i } ( x ) \in \mathcal P$ and $P ( x ) \in { \mathcal { P } } ^ { S O S }$ .

Lemma 2. For polynomials $\{ a _ { i } \} _ { i = 1 } ^ { m } , \{ b _ { j } \} _ { j = 1 } ^ { n }$ and p, define a set $\mathcal { B } = \{ s \in \mathcal { S } : \{ a _ { i } ( s ) \} _ { i = 1 } ^ { m } = 0 , \{ b _ { j } ( \check { s } ) \} _ { j = 1 } ^ { n } \geq 0 \}$ . Let B be compact. The condition $\forall x \in S , p ( s ) \geq 0$ holds if the following condition is satisfied,

$$
\left\{ \begin{array}{l} \exists r _ {1}, \ldots , r _ {m} \in \mathcal {P}, s _ {1}, \ldots , s _ {n} \in \mathcal {P} ^ {S O S}, \\ p - \sum_ {i = 1} ^ {m} r _ {i} a _ {i} - \sum_ {j = 1} ^ {n} s _ {j} b _ {j} \in \mathcal {P} ^ {S O S}. \end{array} \right.
$$

Lemma 2 points out that any strictly positive polynomial p lies in the cone that generated by non-negative polynomials $\{ b _ { j } \} _ { j = 1 } ^ { n }$ in the set of B. Based on the definition of C and $u ,$ P-satz will be adequately used in the safety verification.
