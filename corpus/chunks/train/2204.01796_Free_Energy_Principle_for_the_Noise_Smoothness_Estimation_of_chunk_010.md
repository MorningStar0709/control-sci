$$F = - \frac {1}{2} \tilde {\varepsilon} ^ {T} \tilde {\Pi} \tilde {\varepsilon} + \frac {1}{2} \ln | \tilde {\Pi} | - \frac {1}{2} \varepsilon^ {s T} \Pi^ {s} \varepsilon^ {s} + \frac {1}{2} \ln | \Pi^ {s} |, \tag {7}$$

where ε˜ is the combined prediction error for outputs and states, and $\varepsilon ^ { s }$ is the prediction error for s, given by:

$$
\tilde {\varepsilon} = \left[ \begin{array}{c} \tilde {\mathbf {y}} - \tilde {C} \tilde {x} \\ D \tilde {x} - \tilde {A} \tilde {x} - \tilde {B} \tilde {v} \end{array} \right], \text {   and   } \varepsilon^ {s} = s - \eta^ {s}. \tag {8}
$$

Here $\eta ^ { s }$ and $\Pi ^ { s }$ are the prior smoothness and its prior precision (confidence). For this work, we start the estimation from a low prior $\eta ^ { s } \approx 0$ with a very low confidence $( \Pi ^ { s } = 1 )$ ). Therefore, Equation 7 reduces to:

$$F = - \frac {1}{2} \tilde {\varepsilon} ^ {T} \tilde {\Pi} \tilde {\varepsilon} + \frac {1}{2} \ln | \tilde {\Pi} | - \frac {1}{2} s ^ {2}. \tag {9}$$

The last term in Equation 9 is the novel term that we have introduced for optimizing smoothness, and doesn’t appear in FEP literature. Using this, we propose an online noise smoothness estimation algorithm which estimates s through the gradient ascend (maximization) of F, where ∂ F∂ s |s=so = 0 $\frac { \partial F } { \partial s } | _ { s = s _ { o } } = 0$ and ∂ 2F∂ s2 |s=so < 0, with so being the smoothness value that $\frac { \partial ^ { 2 } F } { \partial s ^ { 2 } } | _ { s = s _ { o } } < 0 .$ $s _ { o }$ maximizes $F .$ The free energy gradients necessary for this scheme are obtained by differentiating Equation 9:

$$\frac {\partial F}{\partial s} = - \frac {1}{2} \tilde {\varepsilon} ^ {T} \frac {\partial \tilde {\Pi}}{\partial s} \tilde {\varepsilon} + \frac {1}{2} \frac {\partial \ln | \tilde {\Pi} |}{\partial s} - s \tag {10}\frac {\partial^ {2} F}{\partial s ^ {2}} = - \frac {1}{2} \tilde {\varepsilon} ^ {T} \frac {\partial^ {2} \tilde {\Pi}}{\partial s ^ {2}} \tilde {\varepsilon} + \frac {1}{2} \frac {\partial^ {2} \ln | \tilde {\Pi} |}{\partial s ^ {2}} - 1,$$

where the gradients of $\ln \left| { \tilde { \Pi } } \right|$ can be computed as (refer Appendix A) :

$$\frac {\partial \ln | \tilde {\Pi} |}{\partial s} = 4 2 (n + m) \frac {1}{s}, \frac {\partial^ {2} \ln | \tilde {\Pi} |}{\partial s ^ {2}} = - 4 2 (n + m) \frac {1}{s ^ {2}}. \tag {11}$$
