3.1. ILF-based HOSMC. The regulators obtained in [35, 36] via the ILF approach can be chosen based on the following conditions1:

Theorem 1. [36] Take the control in the form

$$u (y) = Y X ^ {- 1} D _ {\mathbf {r}} \left(V _ {y} ^ {- 1} (y)\right) y \tag {3.1}$$

$w i t h \ D _ { \mathbf { r } } \left( \lambda \right) = \mathrm { d i a g } \{ \lambda ^ { r _ { i } } \} _ { i = 1 } ^ { n } \ f o r \ \mathbf { r } = \left( r _ { 1 } , \ldots , r _ { n } \right) = \left( n , \ldots , 1 \right) \ a n d \ X \in \mathbb { R } ^ { n \times n } , \ Y \in \mathbb { R } ^ { 1 \times n } .$

solving the following LMIs

$$
\left\{ \begin{array}{l} X \succ 0, \left[ \begin{array}{c c} \Upsilon & b \\ b ^ {\top} & - \varrho_ {2} \end{array} \right] \preceq 0, \\ \left[ \begin{array}{c c} X G _ {\mathbf {r}} + G _ {\mathbf {r}} X & X \\ X & I _ {n} \end{array} \right] \succeq 0, \end{array} \right. \tag {3.2}
\Upsilon = X A ^ {\top} + A X + Y ^ {\top} b ^ {\top} + b Y + \varrho_ {1} (X G _ {\mathbf {r}} + G _ {\mathbf {r}} X),$$

where $\varrho _ { 1 } > \varrho _ { 2 } \Delta ^ { 2 } > 0$ and $G _ { \mathbf { r } } = d i a g \{ r _ { i } \} _ { i = 1 } ^ { n }$ , while the ILF $V _ { y } : \mathbb { R } ^ { n } \to \mathbb { R } _ { + }$ is defined as

$$
\begin{array}{c} Q (V _ {y} (y), y) = 0, \\ Q (V, y) = y ^ {\top} D _ {\mathbf {r}} \left(V ^ {- 1}\right) X ^ {- 1} D _ {\mathbf {r}} \left(V ^ {- 1}\right) y - 1 \end{array} \tag {3.3}
$$

for $y \ne 0$ and $V _ { y } ( 0 ) = 0$ otherwise. Then on all trajectories of the system (2.1), (3.1) with $w \equiv 0$ and $\delta \equiv 0$ :

$$\frac {d}{d t} V _ {y} (x (t)) \leq - (\varrho_ {1} - \varrho_ {2} \Delta^ {2})$$

for almost all $t \geq 0$ while $V _ { y } ( x ( t ) ) \neq 0$ .

Thus, the control (3.1) guarantees the global finite-time stabilization of the origin independently on the disturbances $d \in \mathcal { D }$ in the absence of the measurement noise and mismatched terms. Since the closed-loop system $( 2 . 1 ) , ( 3 . 1 )$ is homogeneous of negative degree, the ISS property with respect to $w , \delta \in \mathcal { L } _ { \infty } ^ { n }$ with $\delta ^ { \top } b \equiv 0$ follows [4]. The drawback of this regulator is the chattering of the control signal appearing in a finite time once the trajectories settled at the origin. If the pair of matrices $( A , b )$ is controllable and the parameters $\varrho _ { 1 }$ and $\varrho _ { 2 }$ are properly chosen, then the LMI (3.2) is guaranteed to be feasible, see Proposition 12 of [36] for more details.
