On the other hand, due to the fact that $f \geq 0$ and $\rho \geq 0$ , the operator $f ( \cdot ) \left| \phi _ { k } \right. \left. \phi _ { k } \right| \rho$ is semi-definite positive for any $k = 1 , \dots , d .$ , and in particular, since $\begin{array} { r } { \sum _ { k = 1 } ^ { d } \left| \phi _ { k } \right. \left. \phi _ { k } \right| = \mathbb { I } _ { d } } \end{array}$ , we have

$$\langle \phi_ {i} | f (\cdot) | \phi_ {i} \rangle \langle \phi_ {i} | \rho | \phi_ {i} \rangle \leq \langle \phi_ {i} | f (\cdot) \left(\sum_ {k = 1} ^ {d} | \phi_ {k} \rangle \langle \phi_ {k} |\right) \rho | \phi_ {i} \rangle = \langle \phi_ {i} | f (\cdot) \rho | \phi_ {i} \rangle ,$$

which finally yields

$$
\begin{array}{l} \operatorname{Tr} [ f (t A + (1 - t) B) \rho ] \leq t \sum_ {i = 1} ^ {d} \left\langle \phi_ {i} \right| f (A) \rho \left| \phi_ {i} \right\rangle + (1 - t) \sum_ {i = 1} ^ {d} \left\langle \phi_ {i} \right| f (B) \rho \left| \phi_ {i} \right\rangle = \\ = t \operatorname{Tr} [ f (A) \rho ] + (1 - t) \operatorname{Tr} [ f (B) \rho ], \\ \end{array}
$$

where, as mentioned before, the inequality is strict for f strictly convex and $\rho \in \operatorname { H } _ { > } ( \operatorname { \mathcal { H } } )$ .
