# 2.1 Consider the function

$$V (x) = x ^ {T} A x + b ^ {T} x + c$$

where $x$ and $b$ are column vectors, $A$ is a matrix, and $c$ is a scalar. Show that the gradient of function $V$ with respect to $x$ is given by

$$\operatorname{grad} _ {x} V = (A + A ^ {T}) x + b$$

This can be used to find the minimum of Eq. (2.7).
