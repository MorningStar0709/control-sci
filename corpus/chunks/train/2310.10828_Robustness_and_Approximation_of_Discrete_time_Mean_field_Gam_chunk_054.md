$$\hat {\mathcal {C}} (n) = \{Q \circ (T _ {n}, I) \in B (X \times A): Q \in \mathcal {C} (n) \}.$$

Since $Q \in { \mathcal { C } } ( n )$ is Borel measurable, for any $Q \in { \mathcal { C } } ( n )$ , we have $Q \circ ( T _ { n } , I ) \in \hat { \mathcal { C } } ( n )$ . By $\hat { H } _ { n } | _ { \hat { \mathcal { C } } ( n ) \times \mathcal { P } ( X ) }$ we denote the restriction of $\hat { H } _ { n }$ to the domain $\hat { \mathcal { C } } ( n ) \times \mathcal { P } ( X ) \subset B ( X \times A ) \times \mathcal { P } ( X )$ . Then, for $\mu ^ { \mathcal { C } } \in \mathcal { P } ( X )$ that is obtained from some $\boldsymbol { \mu } ^ { \mathcal { D } } \in \mathcal { P } ( \boldsymbol { X } _ { n } )$ as in (9), we have

$$\hat {H} _ {1, n} | _ {\hat {\mathcal {C}} (n) \times \mathcal {P} (X)} (Q, \mu^ {\mathcal {C}}) = H _ {1, n} (\tilde {Q}, \mu^ {\mathcal {D}}) \circ (T _ {n}, I)$$

and

$$\hat {H} _ {2, n} | _ {\hat {\mathcal {C}} (n) \times \mathcal {P} (X)} (Q, \mu^ {\mathcal {C}}) = H _ {2, n} (\tilde {Q}, \mu^ {\mathcal {D}}),$$

where ${ \tilde { Q } } \circ ( T _ { n } , I ) = Q$ . By applying Proposition 3 on $\hat { H } _ { n } | _ { \hat { \mathcal { C } } ( n ) \times \mathcal { P } ( X ) }$ , we see that there exists a unique fixed point for the operator $\hat { H } _ { n } | _ { \hat { \mathcal { C } } ( n ) \times \mathcal { P } ( X ) }$ , as under our assumptions the operator $H _ { n }$ has a unique fixed point by Lemma 6.
