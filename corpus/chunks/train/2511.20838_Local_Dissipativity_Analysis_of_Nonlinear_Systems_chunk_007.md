# 2 Problem Statement and Background

This paper aims to characterize the local dissipativity properties of the mapping $\mathcal { G } ( x _ { 0 } ) : \mathcal { L } _ { 2 e } ^ { m } \to \mathcal { L } _ { 2 e } ^ { p }$ , where $\mathbf { y } = \mathcal { G } ( x _ { 0 } ) \mathbf { u }$ is defined by $x ( 0 ) = \overset { \cdot } { x } _ { 0 } \in \mathbb { R } ^ { n }$ and

$$
\mathcal {G} (x _ {0}): \left\{ \begin{array}{l l} \dot {x} & = f (x) + \bar {G} (x) u, \\ y & = h (x) + \bar {J} (x) u. \end{array} \right. \tag {1}
$$

Here, $\bar { G } ( x ) = \mathbf { B } + G ( x )$ and $\bar { J } ( x ) = \mathbf { D } + J ( x )$ . Let $f ( 0 ) = 0 , G ( 0 ) = 0 , h ( 0 ) = 0 , \mathrm { a n d } J ( 0 ) = 0 .$

We will establish local dissipativity by developing constraints on CPA storage functions implying QSR dissipativity. Local dissipativity will be sought for a family of mappings, $\mathcal { G } _ { \Omega } : = \{ \mathcal { G } ( \boldsymbol { x } _ { 0 } ) \ | \ x _ { 0 } \in \Omega \}$ , defined over a robustly positive invariant subset of the state space, $\Omega \subset \mathbb { R } ^ { n }$ , for inputs in a non-empty set $\boldsymbol { \mathcal { U } } _ { \Omega } \subseteq \mathcal { L } _ { 2 } ^ { m }$ defined by

$$
\begin{array}{l} \mathcal {U} _ {\Omega} := \bigcap_ {x _ {0} \in \Omega} \mathcal {U} _ {\Omega} (x _ {0}) \\ \mathcal {U} _ {\Omega} (x _ {0}) := \{\mathbf {u} \in \mathcal {L} _ {2 e} ^ {m} \mid x (t) \in \Omega \forall t \geq 0 \}. \\ \end{array}
$$
