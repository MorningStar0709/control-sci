# C. The canonical dualization

The dualization approach from Section IV-B convexifies the robust dynamic programming step (14) for arbitrary affine linear dependence of $M _ { t }$ and $Q _ { t }$ on decision variables. However, it in turn requires a convex parametrization of the inverse matrices $\breve { M } _ { t }$ and $\bar { Q } _ { t } .$ , which might not always be achievable. For this reason, we explore a general approach for dualization in this section. This general result is based on a factorization approach for $\overline { { \pmb { Q } } } _ { t }$ that allows for efficient parametrization of the inverses. Assume that $\overline { { \pmb { Q } } } _ { t }$ can be written in the form

$$
\overline {{\boldsymbol {Q}}} _ {t} = \left( \begin{array}{c} \boldsymbol {\Pi} _ {\overline {{\boldsymbol {Q}}}} ^ {0} \\ \boldsymbol {\Pi} _ {\overline {{\boldsymbol {Q}}}} ^ {1} \\ \boldsymbol {\Pi} _ {\overline {{\boldsymbol {Q}}}} ^ {2} \end{array} \right) ^ {\top} \left( \begin{array}{c c c} \boldsymbol {S} _ {0} & & \\ & \boldsymbol {S} _ {1} & \\ & & - \boldsymbol {S} _ {2} \end{array} \right) \left( \begin{array}{c} \boldsymbol {\Pi} _ {\overline {{\boldsymbol {Q}}}} ^ {0} \\ \boldsymbol {\Pi} _ {\overline {{\boldsymbol {Q}}}} ^ {1} \\ \boldsymbol {\Pi} _ {\overline {{\boldsymbol {Q}}}} ^ {2} \end{array} \right)
$$

where $S _ { 0 }$ is a constant matrix and $S _ { 1 } , S _ { 2 }$ have positive definite matrix decision variables on their diagonals, i.e., ${ \cal S } _ { 1 } = \mathrm { d i a g } \left( \Lambda _ { 1 } , \Lambda _ { 2 } , \ldots , \Lambda _ { s _ { 1 } } \right) , { \cal S } _ { 2 } = \mathrm { d i a g } \left( \Gamma _ { 1 } , \Gamma _ { 2 } , \ldots \Gamma _ { s _ { 2 } } \right)$ , where repeated blocks are explicitly allowed. An example for this structure of $\overline { { Q } } _ { t } = Q _ { t } + M _ { t }$ is obtained if $Q _ { t }$ is constant and $M _ { t }$ is of the form (8). In this case, we have

$$\overline {{Q}} _ {t} = Q _ {t} + \sum_ {i = 1} ^ {s} \lambda_ {t} ^ {(i)} M ^ {(i)},$$

where $M ^ { ( i ) }$ are symmetric matrices and $\lambda _ { t } ^ { ( i ) } \geq 0$ are nonnegative parameters. Here, the M (i) $M ^ { ( i ) }$ can be decomposed with full rank matrices $M _ { p } ^ { ( i ) }$ and $\boldsymbol { M } _ { m } ^ { ( i ) }$ according to

$$\boldsymbol {M} ^ {(i)} = (\boldsymbol {M} _ {p} ^ {(i)}) ^ {\top} \boldsymbol {M} _ {p} ^ {(i)} - (\boldsymbol {M} _ {m} ^ {(i)}) ^ {\top} \boldsymbol {M} _ {m} ^ {(i)}.$$
