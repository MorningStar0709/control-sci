$$
\begin{array}{l} \mathrm{de} (t - j T) = \Big ((A - \mathcal {G} (t - j T) C) e (t - j T) \\ \left. + \left(A _ {d} - \mathcal {G} (t - j T) C _ {d}\right) e (t - (j + 1) T)\right) \mathrm{d} t \\ + B \mathrm{d} w (t - j T) + \left(B _ {d} - \mathcal {G} (t - j T) D _ {d}\right) \\ \times \mathrm{d} w (t - (j + 1) T). \tag {40} \\ \end{array}
$$

Taking conditional expectation and using $\mathbb { E } [ \mathrm { d } w ( \cdot )$ | $\mathcal { F } _ { t } ^ { y } ] = 0$ again,

$$
\begin{array}{l} \mathrm{d} S _ {j} (t) = \left(P _ {j} (t) \left(A - \mathcal {G} (t - j T) C\right) ^ {T} \right. \\ + \mathbb {E} [ e (t) e ^ {T} (t - (j + 1) T) \mid \mathcal {F} _ {t} ^ {y} ] \\ \left. \times \left(A _ {d} - \mathcal {G} (t - j T) C _ {d}\right) ^ {T}\right) \mathrm{d} t \\ = \left(P _ {j} (t) (A - \mathcal {G} (t - j T) C) ^ {T} \right. \\ \left. + P _ {j + 1} (t) \left(A _ {d} - \mathcal {G} (t - j T) C _ {d}\right) ^ {T}\right) \mathrm{d} t. \tag {41} \\ \end{array}
$$

By the Itˆo rule, $\mathrm { d } Q _ { j } ( t )$ is determined by the quadratic covariation between the diffusion terms in de(t) and $\mathrm { d } e ( t - j T )$ . Since Wiener increments at distinct times are independent, dw(t) is independent of dw $( t - j T )$ and $\mathrm { d } w ( t - ( \bar { j } + 1 ) T )$ , and $\mathrm { { d } } w ( t - T )$ coincides with dw $\mathsf { \Omega } ( t - j T )$ if and only if $j = 1$ . Therefore,

$$\mathrm{d} Q _ {j} (t) = \left(B _ {d} - \mathcal {G} (t) D _ {d}\right) B ^ {T} \mathbb {I} _ {j = 1} \mathrm{d} t. \tag {42}$$

Combining (39)–(42) in (38) gives (36).

Based on the above, we present the optimal filter design algorithm for system (20).
