# B. Theoretical analysis

Let the time-varying switching interface be given by $m ^ { T } x + \mu t + c = 0 ^ { \setminus } 2$ . We change this system into a timeinvariant (TIV) system by conducting a variable augmentation. Define the new augmented variable $\bar { x } = [ x , t ] ^ { \bar { \top } }$ . Then the original system (16) is turned into the following timeinvariant system:

$$
\dot {\bar {x}} = \left[ \begin{array}{c} f (\bar {x}, u) \\ 1 \end{array} \right] = F (\bar {x}, u). \tag {18}
$$

The stage cost is $\bar { L } ( x , u , t ) = L ( \bar { x } , u )$ . The new Hamiltonian is then $\bar { H } ( \bar { x } , u , \bar { \lambda } ) = \bar { L } ( \bar { x } , u ) + \bar { \lambda } ^ { \top } F ( \bar { x } , u )$ . In this case, the optimal control condition gives us

$$
\begin{array}{l} \frac {\partial \bar {H}}{\partial u} = 0 \\ \Rightarrow \quad \frac {\partial \bar {L}}{\partial u} + \bar {\lambda} ^ {\top} \frac {\partial F}{\partial u} = 0 \\ \Rightarrow \frac {\partial L}{\partial u} + [ \lambda^ {\top}, \lambda_ {t} ] \left[ \begin{array}{c} \frac {\partial f}{\partial u} \\ 0 \end{array} \right] = \frac {\partial L}{\partial u} + \lambda^ {\top} \frac {\partial f}{\partial u} = 0. \\ \end{array}
$$

The Euler-Lagrangian equation gives us

$$
\dot {\bar {\lambda}} = \left[ \begin{array}{c} \dot {\lambda} \\ \dot {\lambda} _ {t} \end{array} \right] = - \left(\frac {\partial \bar {H}}{\partial x}\right) ^ {\top} = \left[ \begin{array}{c} - \left(\frac {\partial \bar {H}}{\partial \bar {x}}\right) ^ {\top} \\ - \frac {\partial \bar {H}}{\partial t} \end{array} \right] = \left[ \begin{array}{c} - \left(\frac {\partial H}{\partial x}\right) ^ {\top} \\ - \left(\frac {\partial L}{\partial t} + \lambda^ {\top} \frac {\partial f}{\partial t}\right) \end{array} \right]. \tag {19}
$$

That is

$$\dot {\lambda} = - \left(\frac {\partial H}{\partial x}\right) ^ {\top},\dot {\lambda} _ {t} = - \left(\frac {\partial L}{\partial t} + \lambda^ {\top} \frac {\partial f}{\partial t}\right).$$

With this augmented scheme and using the jump law in (15), we obtain the following law:

$$\Delta \lambda = - \frac {(\Delta L + \langle \lambda \rangle \Delta f) m}{\langle m ^ {T} f + \mu \rangle}, \tag {20}$$

where the notation h·i means the average; µ is the partial derivative of the time-varying switching interface $m ^ { \top } x +$ $\mu t + c = 0$ with respect to time t.
