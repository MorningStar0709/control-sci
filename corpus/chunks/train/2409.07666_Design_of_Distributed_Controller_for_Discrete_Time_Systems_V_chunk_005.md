Another possible way to mitigate the conservatism brought by the structural constraint $Z { \bar { Q ^ { - 1 } } } \in { \mathcal { S } }$ is by exploiting the graph structure. Our prior work on continuous-time systems [15] derived a new convex solution set by decomposing the Lyapunov matrix and system matrices in a clique-wise manner. A discrete-time version of this formulation can also be derived, as shown in Section III. We denote its solution set as $\kappa _ { s }$ . Although $\kappa _ { s }$ is less conservative than $\kappa _ { \mathrm { d i a g } } .$ , it still exhibits some degree of conservatism, as the Lyapunov matrices are structurally restricted in sparsity.

Note that less conservative solution sets $\kappa _ { s }$ and $\mathcal { K } _ { \mathrm { e x t } }$ do not include one another, and we cannot determine which is better. Our goal in this paper is to find a less conservative convex relaxation of (4) compared to both sets, denoted by $\boldsymbol { \mathcal { K } } _ { S , \mathrm { e x t } }$ , as shown in Fig. 1. In other words, we aim to formulate a convex optimization problem that yields a solution to the following:
