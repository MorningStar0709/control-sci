# 3. PROBLEM FORMULATION

In this section, we define the system dynamics, the control objectives, and the assumptions taken. Consider a class of uncertain nonlinear single-input systems,

$$\dot {\mathbf {x}} = \mathbf {A} \mathbf {x} + \mathbf {b} k _ {p} (u + \Delta (\mathbf {x})) \tag {3}$$

where $\mathbf { x } \in \mathbb { R } ^ { n }$ and $u \in \mathbb { R }$ denote the state vector and the control input of the system, respectively. The state matrix $\mathbf { A } \in \mathbb { R } ^ { \bar { n } \times n }$ is unknown, the control vector $\mathbf { b } k _ { p } \in \mathbb { R } ^ { n }$ contains known vector b $\in ~ \mathbb { R } ^ { n }$ , and unknown constant $k _ { p } ~ \in ~ \mathbb { R }$ , however, sign of $k _ { p }$ is known. The nonlinear uncertain term $\Delta \left( \mathbf { x } \right) \bar { \in } \mathbb { R }$ is a matched uncertainty.

Assumption 1. The nonlinear matched uncertain term $\Delta \left( \mathbf { x } \right)$ , is linearly parameterizable as $\Delta \left( \mathbf { x } \right) ~ = ~ \pmb { \theta } ^ { T } \phi \left( \mathbf { x } \right)$ , where $\phi ( \mathbf { x } ) : \mathbb { R } ^ { n } \overset { \cdot } {  } \mathbb { R } ^ { p }$ is a known mapping, and called the regressor vector, and $\pmb { \theta } \in \mathbb { R } ^ { p }$ contains unknown constants.

Assumption 2. The given system (3) is controllable that is, $( \mathbf { A } , \bar { \mathbf { b } } k _ { p } )$ is a controllable pair hence $\mathbf { b } ^ { T } \mathbf { b } > 0$ . The sign of the unknown constant $k _ { p }$ is known and is denoted by $k _ { p } ^ { \prime }$ .

The primary control objective is to design a control law $u ,$ such that the state vector x of the system tracks a reference state trajectory $\mathbf { x } _ { r } ,$ generated online from the following reference dynamics,

$$\dot {\mathbf {x}} _ {r} = \mathbf {A} _ {r} \mathbf {x} _ {r} + \mathbf {b} _ {r} r, \tag {4}$$

where $\mathbf { A } _ { r } \in \mathbb { R } ^ { n \times n }$ is a Hurwitz matrix, $\mathbf { x } _ { r } \in \mathbf { R } ^ { n }$ is the reference state trajectory, and $r \in \mathbb { R }$ denotes a bounded, piece-wise continuous reference signal. Ar is a Hurwitz matrix hence satisfies,
