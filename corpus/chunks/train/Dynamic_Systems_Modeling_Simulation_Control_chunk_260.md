# Ramped Step Input

A ramped step input function exhibits a linear “ramp up” (or “ramp down”) from zero to a constant (or step) value. Hence, the ramped step function is characterized by the slope a and step magnitude A

$$
\text { Ramped   step   input } \quad u (t) = \left\{ \begin{array}{l l} a t & \text { for } \quad 0 \leq t \leq A / a \\ A & \text { for } \quad t > A / a \end{array} \right. \tag {5.124}
$$

Clearly the end of the ramp time is $t _ { 1 } = A / a .$ . In some cases a ramped step input might be a more realistic representation of a step input as many physical inputs require a finite time interval to change to a new constant value.
