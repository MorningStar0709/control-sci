# Exercise 1.40: More on normals and linear transformations

Consider a normally distributed random variable $x \in \mathbb { R } ^ { n } , x \sim N ( m _ { x } , P _ { x } )$ . You showed in Exercise 1.39 for $C \in \mathbb { R } ^ { n \times n }$ invertible, that the random variable y defined by the linear transformation $y = C x$ is also normal and is distributed as

$$y \sim N (C m _ {x}, C P _ {x} C ^ {\prime})$$

Does this result hold for all C? If yes, prove it; if no, provide a counterexample.
