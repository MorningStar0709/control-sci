# 2.2.7 Centralized NMPC

In this section, a centralized systematic solution to Problem 2.2 is introduced. Our overall approach builds on designing a Nonlinear Model Predictive control scheme for the system of the manipulators and the object. Nonlinear Model Predictive Control (see e.g. [86–94]) has been proven suitable for dealing with nonlinearities and state and input constraints.

The coupled agents-object nonlinear dynamics can be written in compact form as follows:

$$
\dot {x} _ {c} = f _ {c} (x _ {c}, u) := \left[ \begin{array}{l} f _ {c _ {1}} (x _ {c}, u) \\ f _ {c _ {2}} (x _ {c}, u) \\ f _ {c _ {3}} (x _ {c}, u) \end{array} \right], x _ {c 0} := x _ {c} (0), \tag {2.52}
$$

where $\boldsymbol { x } _ { c } : = [ \boldsymbol { x } _ { o } ^ { \top } , \boldsymbol { v } _ { o } ^ { \top } , \boldsymbol { q } ^ { \top } ] ^ { \top } \in \mathbb { M } \times \mathbb { R } ^ { n + 6 } , \boldsymbol { u } \in \mathbb { R } ^ { 6 N }$ and $f _ { c } : \mathbb { M } \times { \mathsf { S } } \times \mathbb { R } ^ { 6 N + 6 } \to$ $\mathbb { R } ^ { n + 1 2 }$ , with

$$
\begin{array}{l} f _ {c _ {1}} \left(x _ {c}, u\right) := J _ {O} \left(\eta_ {O}\right) v _ {O}, \\ f _ {c _ {2}} (x _ {c}, u) := \widetilde {M} (x) ^ {- 1} \left[ G (q) u - \widetilde {C} (x) v _ {o} - \widetilde {g} (x) \right], \\ f _ {c _ {3}} (x _ {c}, u) := \widetilde {J} (q) G (q) ^ {\top} v _ {o}, \\ \end{array}
$$

where we have used the first equation of (2.41). Note that $f _ { c }$ is locally Lipschitz continuous in its domain since it is continuously differentiable there. Next, we define the respective errors:

$$
e _ {c} := x _ {c} - x _ {\mathrm{d}} = \left[ \begin{array}{c} x _ {O} \\ v _ {O} \\ q \end{array} \right] - \left[ \begin{array}{c} x _ {\mathrm{d}} \\ \dot {x} _ {\mathrm{d}} \\ q _ {\mathrm{d}} \end{array} \right] = \left[ \begin{array}{c} x _ {O} - x _ {\mathrm{d}} \\ v _ {O} \\ q - q _ {\mathrm{d}} \end{array} \right] \in \mathbb {M} \times \mathbb {R} ^ {6} \times \mathsf {S}, \tag {2.53}
$$

where $q _ { \mathrm { d } } : = [ q _ { 1 , \mathrm { d } } ^ { \top } , \ldots , q _ { N , \mathrm { d } } ^ { \top } ] ^ { \top } \in \mathbb { R } ^ { n }$ is appropriately chosen to comply with the coupled kinematics (2.8) and $x _ { \mathrm { d } }$ . The error dynamics are then ${ { \dot { e } } _ { c } } ( t ) =$ $f _ { c } ( x _ { c } ( t ) , u ( t ) )$ ), which can be appropriately transformed to:

$$\dot {e} _ {c} = f _ {e} (e _ {c}, u), \quad e _ {c 0} := e _ {c} (0) = x _ {c} (0) - x _ {\mathrm{d}}. \tag {2.54}$$
