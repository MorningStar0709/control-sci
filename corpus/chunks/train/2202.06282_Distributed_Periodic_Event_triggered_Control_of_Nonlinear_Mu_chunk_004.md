# 2.1 Notation

The sets of all non-negative and positive integers are denoted N and $\mathbb { N } _ { > 0 } .$ , respectively. The fields of all reals, all non-negative reals and all non-positive reals are indicated by $\mathbb { R } , \breve { \mathbb { R } } _ { \geqslant 0 }$ and $\mathbb { R } _ { \leqslant 0 }$ , respectively. The identity matrix of size $N \times N$ is denoted by $I _ { N }$ , and the vectors in $\mathbb { R } ^ { N }$ whose elements are all ones or zeros are denoted by $\mathbf { 1 } _ { N }$ and ${ \mathbf { 0 } } _ { N }$ , respectively. For N vectors $x _ { i } ~ \in ~ \mathbb { R } ^ { n _ { i } }$ , we use the notation $( x _ { 1 } , x _ { 2 } , \ldots , x _ { N } )$ to denote $\left[ x _ { 1 } ^ { \top } ~ x _ { 2 } ^ { \top } ~ . . . ~ x _ { N } ^ { \top } \right] ^ { \top }$ . Given matrices $A _ { 1 } , \ldots , A _ { n }$ , we denote by diag $\left( A _ { 1 } , \ldots , A _ { n } \right)$ the block-diagonal matrix where the main diagonal blocks consist of the matrices $A _ { 1 }$ to $A _ { n }$ and all other blocks are zero matrices. By $\langle \cdot , \cdot \rangle$ and $| \cdot |$ we denote the usual inner product of real vectors and the Euclidean norm, respectively. We denote the logical and and or operators as $\wedge$ and $\vee ,$ respectively. For two matrices $A \in \mathbb { R } ^ { m }$ ×n and $B \ \in \ \mathbb { R } ^ { j \times q }$ , the Kronecker product of A with B is denoted $A \otimes B \in \mathbb { R } ^ { m p \times n q }$ . The cardinality of a finite set is denoted $| S | .$ The notation $F : X \Longrightarrow Y$ , indicates that F is a set-valued mapping from X to Y with $F ( x ) \subseteq Y$ for all $x \in X$ . For any $x \in \mathbb { R } ^ { n }$ , the distance to a closed non-empty set A is denoted by $\begin{array} { r } { | x | _ { A } : = \operatorname* { m i n } _ { y \in A } | x - y | } \end{array}$ |. We use $U ^ { \circ } ( x ; v )$ to denote the generalized directional derivative of Clarke of a locally Lipschitz function U at x in the direction $v , { \mathrm { i . e . } } ,$ , $\begin{array} { r } { U ^ { \circ } ( x ; v ) : = \operatorname* { l i m } \operatorname* { s u p } _ { h \to 0 ^ { + } , y \to x } ( U ( y + h v ) - U ( y ) ) / h . } \end{array}$ , which reduces to the standard directional derivative $\langle \nabla U ( { \boldsymbol { x } } ) , { \boldsymbol { v } } \rangle$ when U is continuously differentiable.
