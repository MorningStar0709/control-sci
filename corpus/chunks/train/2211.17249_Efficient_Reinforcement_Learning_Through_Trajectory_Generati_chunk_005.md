# 2.1. Notations

Throughout this manuscript, vectors are denoted in lower case bold and matrices are denoted in upper case bold, unless otherwise specified. Vectors of all ones and zeros are denoted by $\mathbb { 1 } _ { n } , \mathbb { 0 } _ { n } \in \mathbb { R } ^ { n }$ , respectively. The identity matrix is denoted by $\ b { I _ { n } } \in \mathbb { R } ^ { n \times n }$ . We use $\mathcal { N } ( A )$ to denote the null space of matrix A. Given n matrices $M _ { i } , i = 1 , \cdots , n$ , we denote $[ M _ { 1 } ; \cdot \cdot \cdot ; M _ { n } ] : = [ M _ { 1 } ^ { \top } \cdot \cdot \cdot M _ { n } ^ { \top } ] ^ { \top }$ .

Given a discrete-time signal $ { \boldsymbol { z } } ( t ) \in  { \mathbb { R } } ^ { d }$ for $t = 0 , 1 , \cdots$ , we use $z _ { [ k , k + T ] } \in \mathbb { R } ^ { T d }$ to denote the vector form of the sequence $\{ z ( k ) , \ldots , z ( k + T ) \}$ }, and the Hankel matrix $\bar { Z _ { i , t , N } } \in \mathbb { R } ^ { t d \times N }$ as

$$
\mathbf {Z} _ {[ k, k + T ]} = \left[ \begin{array}{c} \boldsymbol {z} (k) \\ \vdots \\ \boldsymbol {z} (k + T) \end{array} \right], \mathbf {Z} _ {i, t, N} = \left[ \begin{array}{c c c c} \boldsymbol {z} (i) & \boldsymbol {z} (i + 1) & \dots & \boldsymbol {z} (i + N - 1) \\ \boldsymbol {z} (i + 1) & \boldsymbol {z} (i + 2) & \dots & \boldsymbol {z} (i + N) \\ \vdots & \vdots & \ddots & \vdots \\ \boldsymbol {z} (i + t - 1) & \boldsymbol {z} (i + t) & \dots & \boldsymbol {z} (i + t + N - 2) \end{array} \right],
$$

where $k , i$ and are integers, and $t , N , T$ are natural numbers. The first subscript of $Z _ { i , t , N }$ denotes the time at which the first sample of the signal is taken, the second one the number of samples per each column, and the last one the number of signal samples per each row.
