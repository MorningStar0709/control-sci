# 2.4 Existing NN based approaches to elliptic optimal control

In the literature, three methods have been proposed, i.e., penalty method (PM) [41], augmented Lagrangian method (ALM) [35], and adjoint oriented neural network (AONN) [51]. We describe the them briefly below.

PM transforms (2.1)–(2.2) into an unconstrained problem by including the loss ${ \mathcal { L } } _ { \mathrm { n n } }$ :

$$
\begin{array}{l} \mathcal {L} _ {\mathrm{pm}} (y _ {\theta}, u _ {\kappa}) := J (y _ {\theta}, u _ {\kappa}) + \mu \mathcal {L} _ {\mathrm{nn}} (y _ {\theta}, u _ {\kappa}) \\ = \big (\frac {1}{2} \| y _ {\theta} - y _ {d} \| _ {L ^ {2} (\Omega)} ^ {2} + \frac {\lambda}{2} \| u _ {\kappa} \| _ {L ^ {2} (\Omega)} ^ {2} \big) + \mu \big (\frac {1}{2} \| F (y _ {\theta}, u _ {\kappa}) \| _ {L ^ {2} (\Omega)} ^ {2} + \frac {\alpha}{2} \| y _ {\theta} \| _ {L ^ {2} (\partial \Omega)} ^ {2} \big), \\ \end{array}
$$

where $\mu > 0$ is the penalty weight for the loss $\begin{array} { r } { \mathcal { L } _ { \mathrm { n n } } ( y _ { \theta } , u _ { \kappa } ) = \frac { 1 } { 2 } \| F ( y _ { \theta } , u _ { \kappa } ) \| _ { L ^ { 2 } ( \Omega ) } ^ { 2 } + \frac { \alpha } { 2 } \| y _ { \theta } \| _ { L ^ { 2 } ( \partial \Omega ) } ^ { 2 } } \end{array}$ , and $F ( y _ { \theta } , u _ { \kappa } ) =$ $\Delta y _ { \theta } + f + u _ { \kappa }$ . The weight $\mu$ can be fixed a priori or determined by numerical continuation [35, 22]. The loss ${ \mathcal { L } } _ { \mathrm { p m } }$ becomes more ill-conditioned as $\mu \to \infty [ 7 , 3 5 ]$ , and the case of multiple constraints requires lengthy and time-consuming tuning of weights [29, 51].

ALM mitigates ill-conditioning using Lagrangian multipliers [7].
