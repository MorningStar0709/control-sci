# E. Quantum-Inspired Policy Network

The policy network $\pi _ { \boldsymbol { \theta } } ( \boldsymbol { x } )$ maps states to control actions using a two-layer variational quantum circuit (VQC) surrogate. Each layer comprises parameterized single-qubit rotation gates:

$$
R _ {x} (\theta) = \left[ \begin{array}{c c} \cos (\theta / 2) & - i \sin (\theta / 2) \\ - i \sin (\theta / 2) & \cos (\theta / 2) \end{array} \right], \tag {11}

R _ {y} (\theta) = \left[ \begin{array}{c c} \cos (\theta / 2) & - \sin (\theta / 2) \\ \sin (\theta / 2) & \cos (\theta / 2) \end{array} \right], \tag {12}

R _ {z} (\theta) = \left[ \begin{array}{c c} e ^ {- i \theta / 2} & 0 \\ 0 & e ^ {i \theta / 2} \end{array} \right]. \tag {13}
$$

The state vector is encoded into rotation angles $\phi _ { z } , \phi _ { v _ { r } } , \phi _ { v _ { \epsilon } }$ e via normalization and non-linear squashing (tanh). The expectation value of the Pauli-Z operator yields a bounded scalar:

$$\langle Z \rangle = | 0 | ^ {2} - | 1 | ^ {2}, \tag {14}$$

which is mapped to a continuous control action:

$$u = \tanh (s \langle Z \rangle + b), \tag {15}$$

where s and b are learnable scaling and bias parameters. The output is linearly rescaled to $u \in [ u _ { \mathrm { m i n } } , u _ { \mathrm { m a x } } ]$ .
