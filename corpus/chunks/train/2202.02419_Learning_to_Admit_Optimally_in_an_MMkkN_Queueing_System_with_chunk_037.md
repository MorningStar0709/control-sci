In the second line, we used the fact that given $Y _ { i } \le 0 , K _ { i }$ is geometric with $\mathbb { E } [ K _ { i } ] \le f ( i )$ . The last inequality follows from (35) and Lemma 5. In Lemma 1, we argued that $\mathbb { E } [ \tau _ { i } | Y _ { i - 1 } \le 0 ]$ is bounded by a constant. Hence, for any function $f$ with $\log ( f ) = o ( n )$ , the expected regret is finite. □

Next, we present the finite-time performance guarantee when $\mu < c / R$ . In this regime, the expected regret consists of two terms. The first term arises from the arrivals for which $Y _ { n } > 0$ , and we use the exponentially decaying probability of Lemma 5 to bound this term. The second term results from the arrivals accepted with a given probability when $Y _ { n } \leq 0$ . We will use Lemma 6 presented below to address this term; proof is given in Section A.2. In conclusion, Theorem 4 proves a polynomial in $\log ( n )$ upper bound for the expected regret in the case of $\mu \in ( 0 , c / R )$ .
