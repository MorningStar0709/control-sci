$$\left\| \mathbf {h} _ {t + 1} \right\| = \left\| \mathbf {n} _ {t + 1} \right\| = \left\| \phi \left(\mathbf {U} _ {n} \left(\mathbf {r} _ {t + 1} \odot \mathbf {h} _ {t}\right)\right) \right\|. \tag {16}$$

Again, using Eq. (3) and (4), we have an upper bound for Eq. (16):

$$\left\| \phi (\mathbf {U} _ {n} (\mathbf {r} _ {t + 1} \odot \mathbf {h} _ {t})) \right\| < \| \mathbf {U} _ {n} \mathbf {h} _ {t} \|. \tag {17}$$

If we then restrict the matrix ${ \mathbf { U } } _ { n }$ such that

$$\left\| \mathbf {U} _ {n} \mathbf {h} _ {t} \right\| < \left\| \mathbf {h} _ {t} \right\|, \quad \left\| \mathbf {h} _ {t} \right\| \neq 0, \tag {18}$$

then from Eq. (17) and (16) we get that

$$\left\| \mathbf {h} _ {t + 1} \right\| < \left\| \mathbf {h} _ {t} \right\| \Rightarrow \lim _ {t \rightarrow \infty} \left\| \mathbf {h} _ {\mathbf {t}} \right\| = 0. \tag {19}$$

Using Eq. (5) and Eq. (19) under the L2 norm, we get a constraint for the spectral norm of the matrix ${ \mathbf { U } } _ { n }$ from Eq. (18):

$$\left\| \mathbf {U} _ {n} \right\| _ {2} < 1. \tag {20}$$

Now, we consider the case of an arbitrary update gate $\mathbf { z } _ { t }$ with the restrictions of Eq. (15) and (20). Since the spectral norm of a matrix determines the maximum amount by which a matrix lengthens a vector, with the restriction of Eq. (20), we have that

$$
\begin{array}{l} \left\| \mathbf {h} _ {t + 1} \right\| = \left\| \left(\mathbf {1} - \mathbf {z} _ {t + 1}\right) \odot \mathbf {n} _ {t + 1} + \mathbf {z} _ {t + 1} \odot \mathbf {h} _ {t} \right\| \tag {21} \\ <   \left\| \left(\mathbf {1} - \mathbf {z} _ {t + 1}\right) \odot \mathbf {h} _ {t} + \mathbf {z} _ {t + 1} \odot \mathbf {h} _ {t} \right\| = \| \mathbf {h} _ {t} \|. \\ \end{array}
$$

Since Eq. (21) implies Eq. (10), the system is asymptotically stable if the constraints in Eq. (15) and (20) apply.
