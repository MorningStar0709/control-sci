# 6.3. Simulation results

The simulations are done using JuMP v.0.21.10 [29] as the NLP modeler in a Julia [30] environment. The NLP solver used is IPOPT 3.13.4 [31], and the linear solver is HSL-MA57 [32] on a 2.6 GHz Intel Core-i7 with 16 GB memory.

An initial prediction horizon of $N _ { 0 } = 8$ is chosen with $N _ { \mathrm { m i n } } = 2$ . The scenario tree is fully branched at each MPC iteration making a total of 6561 scenarios at the first iteration.
