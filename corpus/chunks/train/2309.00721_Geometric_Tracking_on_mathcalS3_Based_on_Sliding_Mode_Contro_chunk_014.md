# 3.3 Sliding Property of the Sliding Subgroup

One of the salient features of sliding mode control is that the behavior of a system on the sliding subgroup is independent of the system dynamics. Define $\mathcal { M } : =$ $\left\{ g = ( \overset { \cdot } { q } , p ) \in T S ^ { 3 } \ | \ p = \overset { \cdot } { q } \right\}$ , then $\mathcal { M } \subset T \mathcal { S } ^ { 3 }$ . The following lemma establishes the sliding property of the subgroup $\bar { H } : = H \cap \mathcal { M }$ , where H is defined in (15), that is, $\bar { g } \to e$ asymptotically provided that $\bar { g } \in \bar { H }$ .

Lemma 4 (Sliding property) Let $\bar { g } ~ = ~ ( q , \dot { q } ) ~ \in ~ \bar { H }$ then $\bar { g } \to e \overset { \cdot } { = } ( \bar { 1 } , 0 _ { 4 \times 1 } )$ asymptotically.

PROOF. Since $( q , \dot { q } ) \in H \quad \Longrightarrow \quad s ( \bar { g } ) = 0 _ { 4 \times 1 }$ , it follows that

$$\frac {d}{d t} (q - \bar {1}) = \dot {q} = - \lambda (q _ {0} q - \bar {1}).$$

Consider the candidate Lyapunov function $\begin{array} { r } { V = \frac { 1 } { 2 } \lVert q - { } \rVert } \end{array}$ ${ \bar { 1 } } \| ^ { 2 } = 1 - q _ { 0 }$ . Its time derivative on the sliding subgroup is

$$\dot {V} = (q - \bar {1}) ^ {T} \dot {q} = (q - \bar {1}) ^ {T} (- \lambda (q _ {0} q - \bar {1}))= \lambda \bar {1} ^ {T} (q _ {0} q - \bar {1}) = - \lambda (1 - q _ {0} ^ {2}) = - \lambda (1 + q _ {0}) V.$$

For all $q \in { \mathcal { S } } ^ { 3 } \backslash \{ - { \bar { 1 } } \}$ , it has $1 + q _ { 0 } < 0$ . Therefore, $\dot { V } <$ 0. This shows the almost global asymptotic stability of $q = \bar { 1 }$ .
