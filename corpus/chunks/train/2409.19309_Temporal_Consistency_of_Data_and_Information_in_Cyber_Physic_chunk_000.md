# Temporal Consistency of Data and Information in Cyber-Physical Systems

Hermann Kopetz, TU Wien, Austria

Wilfried Steiner, TTTech, Austria

Abstract—In a large cyber-physical system, a temporal inconsistency of an output value can arise if there is a nonnegligible delay between the instant when a sensor value is acquired from the environment and the instant when a setpoint, based on this sensor value, is used in the environment. Such a temporal inconsistency can be the cause of a critical malfunction of the cyber-physical system. This paper presents a solution of this temporal consistency problem that can best be implemented in a time-triggered architecture (TTA). In a TTA, the instants of sensor value acquisition, setpoint calculation, and actuation on the environment are statically configured, and the cyberphysical system implements software and hardware mechanisms to execute the respective actions tightly at these configured instants.
