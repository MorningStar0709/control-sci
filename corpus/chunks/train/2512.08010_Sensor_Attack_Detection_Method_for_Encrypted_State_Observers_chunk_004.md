# 2.1 LWE based Homomorphic Encryption Scheme

We briefly introduce the LWE based encryption scheme of Regev (2009), focusing on its additively homomorphic property. For $q \in \mathbb { N }$ , we consider $\mathbb { Z } _ { q } : = \bar { \mathbb { Z } } \cap [ - q / 2 , \bar { q } / 2 )$ as the space of messages to be encrypted. The modulo operation that maps Z onto $\mathbb { Z } _ { q }$ is defined as a mod $q : =$ $a - \lfloor ( a + q / 2 ) / q \rfloor \bar { q } \in \mathbb { Z } _ { q }$ for all $a \in \mathbb { Z }$ , which applies componentwisely to vectors and matrices. Given a secret key sk $\in \mathbb { Z } _ { q } ^ { N }$ of length $N \in \mathbb N$ , an h-dimensional message $m \in \mathbb { Z } _ { q } ^ { h }$ is encrypted as

$$\mathsf {E n c} (m) := [ m + b \quad A ] \mod q \in \mathbb {Z} _ {q} ^ {h \times (N + 1)}, \tag {1}$$

where $A \in \mathbb { Z } _ { q } ^ { h \times N }$ is a randomly generated matrix, and $b : = A \cdot s \mathbf { k } + e$ mod $q \in \mathbb { Z } _ { q } ^ { h }$ is the “masking term” that conceals the message. Here, $\bar { \boldsymbol { e } } \in \mathbb { Z } ^ { h }$ is a small “error term” bounded as $\| e \| \le \Delta$ for some $\Delta > 0$ . The decryption of an h-dimensional ciphertext (encrypted message) ${ \textbf { c } } \in$ $\mathbb { Z } _ { q } ^ { h \times ( N + 1 ) }$ is performed as

$$
\operatorname{Dec} (\mathbf {c}) := \mathbf {c} \left[ \begin{array}{c} 1 \\ - \mathsf {s k} \end{array} \right] \bmod q \in \mathbb {Z} _ {q} ^ {h},
$$

so that the original message can be approximately recovered as Dec $( \mathsf { E n c } ( m ) ) \ = \ m + \ e$ mod q. For the sake of simplicity, we omit the modulo operation in the arguments of Enc and Dec throughout the paper.

The described scheme is additively homomprhic, that is,

$$\operatorname{Dec} \left(\mathbf {c} _ {1} + \mathbf {c} _ {2}\right) = \operatorname{Dec} \left(\mathbf {c} _ {1}\right) + \operatorname{Dec} \left(\mathbf {c} _ {2}\right) \mod q \tag {2}$$

holds for any $\mathbf { c } _ { 1 } \in \mathbb { Z } _ { q } ^ { h \times ( N + 1 ) }$ and $\mathbf { c } _ { 2 } \in \mathbb { Z } _ { q } ^ { h \times ( N + 1 ) }$ . From this property, it follows that for any integer matrix K ∈ Zd×h, $K \in \mathbb { Z } ^ { d \times h }$

$$K \cdot \mathsf {E n c} (m) := [ K m + K b K A ] \bmod q \in \mathbb {Z} _ {q} ^ {d \times (N + 1)}$$

is a d-dimensional ciphertext, which can be decrypted as

$$\operatorname{Dec} (K \cdot \operatorname{Enc} (m)) = K \cdot \operatorname{Dec} (\operatorname{Enc} (m)) \bmod q \tag {3}= K m + K e \bmod q.$$

Thus, matrix-vector multiplication can be carried out over encrypted data without decryption. We refer to Km, $K b ,$ , and $\bar { K } e$ as the message, masking term, and error term of $K \cdot \mathsf { E n c } ( m )$ , respectively.
