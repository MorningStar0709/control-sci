# 4.2.3 Dynamic Disturbance Addition

Except for the already considered dynamic uncertainties, we can add to the right-hand side of (4.1) an unknown disturbance vector d $\mathbf { \boldsymbol { \cdot } } = d ( \boldsymbol { x } , \boldsymbol { v } , t )$ :

$$\mathbb {R} ^ {2 n} \times \mathbb {R} _ {\geq 0} \to \mathbb {R} ^ {n}, \text { i.e., }\dot {x} = vm \dot {v} + f (x, v) + m g + d (x, v, t) = u,$$

subject to a uniform boundedness condition $\| d ( x , v , t ) \| ~ \leq \bar { d } , \forall x , v , t \in$ $\mathbb { R } ^ { 2 n } \times \mathbb { R } _ { \geq 0 }$ . In this case, by slightly modifying the control scheme, we still guarantee collision avoidance with the workspace obstacles and boundary. In addition, we achieve uniform ultimate boundedness of the error signals as well as the gradient of $\phi ,$ as the analysis in this section shows.

The control scheme of the previous section is appropriately enhanced to incorporate the $\sigma -$ modification [110], a common technique in adaptive control. More specifically, the adaptation laws (4.13) are modified according to

$$\dot {\hat {m}} := - k _ {m} e _ {v} ^ {\top} (\dot {v} _ {\mathrm{d}} + g) - \sigma_ {m} \hat {m}\dot {\hat {\alpha}} := k _ {\alpha} \| e _ {v} \| ^ {2} - \sigma_ {\alpha} \hat {\alpha},$$

where $\sigma _ { m } , \sigma _ { \alpha }$ are positive gain constants, to be appropriately tuned as per the analysis below.

Consider now the function V as defined (4.14). In view of the analysis of the previous section, the incorporation of $d ( x , v , t )$ , as well as the modification of the adaptation laws, the derivative of V becomes
