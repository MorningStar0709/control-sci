# Improved Gradient Algorithm

In order to assure the stability of the PAA for any value of the adaptation gain α (or of the eigenvalues of the gain matrix F ) the same gradient approach is used but a different criterion is considered:

$$\min _ {\hat {\theta} (t + 1)} J (t + 1) = [ \varepsilon (t + 1) ] ^ {2} \tag {3.20}$$

The equation:

$$\frac {1}{2} \frac {\delta J (t + 1)}{\delta \hat {\theta} (t + 1)} = \frac {\delta \varepsilon (t + 1)}{\delta \hat {\theta} (t + 1)} \varepsilon (t + 1) \tag {3.21}$$

is then obtained. From (3.6) and (3.8) it results that:

$$\varepsilon (t + 1) = y (t + 1) - \hat {y} (t + 1) = y (t + 1) - \hat {\theta} ^ {T} (t + 1) \phi (t) \tag {3.22}$$

and, respectively:

$$\frac {\delta \varepsilon (t + 1)}{\delta \hat {\theta} (t + 1)} = - \phi (t) \tag {3.23}$$
