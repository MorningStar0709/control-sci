# 1.1 Preliminaries and notations

Time indices are denoted by subscripts $\left( \mathrm { e . g . , ~ } x _ { t } \right)$ , and boldface letters denote the stacked vectors at all times, e.g. $\mathbf { \pmb { x } } ~ = ~ [ x _ { 0 } ^ { \top } , \ldots , x _ { T } ^ { \top } ] ^ { \top }$ for a window [0, T ]. Similarly, calligraphic letters denote linear operators applying to such stacked vectors $( \mathrm { e . g . ~ } A$ such that $\boldsymbol { y } = \boldsymbol { A } \boldsymbol { x } )$ . The $\ell _ { 2 ^ { - } }$ norm of a vector is denoted by k · k2, which also denotes the spectral norm of a matrix (its largest eigenvalue). The Frobenius norm of a matrix is denoted by $\lVert \cdot \rVert _ { F }$ . Blackboard bold letters denote sets (e.g. R for all real numbers), except E that denotes the expectation of a random variable.
