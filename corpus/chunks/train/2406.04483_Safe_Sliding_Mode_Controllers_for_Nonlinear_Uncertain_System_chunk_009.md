where $\pmb { f } _ { a } : \mathbb { R } ^ { n - p } \times \mathbb { R } ^ { p }  \mathbb { R } ^ { n - p }$ and $\pmb { f } _ { b } : \mathbb { R } ^ { n - p } \times \mathbb { R } ^ { p }  \mathbb { R } ^ { p }$ are two smooth functions. By viewing ⇣ as the control input for the reduced-order system (2), we assume that the origin of (2) is asymptotically stabilized using $\zeta = \phi ( \eta )$ , where $\phi : \mathbb { R } ^ { n - p } \to \mathbb { R } ^ { p }$ is a continuously differentiable function and satisfies the condition $\phi ( \mathbf { 0 } ) = \mathbf { 0 } .$ . Using this function, the sliding variable $\boldsymbol { s } \in \mathbb { R } ^ { p }$ is defined as $s = \zeta - \phi ( \eta )$ . Using (2) and (3), the dynamics of the corresponding sliding manifold $\mathbf { \boldsymbol { s } } = \mathbf { 0 }$ can be written as follows:

$$\dot {\boldsymbol {s}} = \boldsymbol {f} _ {b} (\boldsymbol {\eta}, \boldsymbol {\zeta}) - \frac {\partial \phi}{\partial \boldsymbol {\eta}} \boldsymbol {f} _ {a} (\boldsymbol {\eta}, \boldsymbol {\zeta}) + \boldsymbol {G} (\boldsymbol {x}) \boldsymbol {E} (\boldsymbol {x}) \boldsymbol {u} + \delta (t, \boldsymbol {x}). \tag {4}$$

Assume the following control law is selected:

$$\boldsymbol {u} = \boldsymbol {E} ^ {- 1} (\boldsymbol {x}) \left(- \hat {\boldsymbol {G}} ^ {- 1} (\boldsymbol {x}) \left(\boldsymbol {f} _ {b} (\boldsymbol {\eta}, \boldsymbol {\zeta}) - \frac {\partial \phi}{\partial \boldsymbol {\eta}} \boldsymbol {f} _ {a} (\boldsymbol {\eta}, \boldsymbol {\zeta})\right) + \boldsymbol {v}\right), \tag {5}$$

where $\hat { \boldsymbol G } : \mathbb { R } ^ { n }  \mathbb { R } ^ { p \times p }$ is a known estimation of G which is assumed to be invertible, and $v : \mathbb { R } ^ { n }  \mathbb { R } ^ { p }$ is an additional control component that needs to be determined. By substituting (5) into (4), we obtain

$$\dot {\boldsymbol {s}} = \boldsymbol {G} (\boldsymbol {x}) \boldsymbol {v} + \boldsymbol {\Delta} (t, \boldsymbol {x}), \tag {6}$$

where

$$\boldsymbol {\Delta} = \boldsymbol {\delta} + \left(\boldsymbol {I} - \boldsymbol {G} (\boldsymbol {x}) \hat {\boldsymbol {G}} ^ {- 1} (\boldsymbol {x})\right) \left(\boldsymbol {f} _ {b} (\boldsymbol {\eta}, \boldsymbol {\zeta}) - \frac {\partial \phi}{\partial \boldsymbol {\eta}} \boldsymbol {f} _ {a} (\boldsymbol {\eta}, \boldsymbol {\zeta})\right).$$
