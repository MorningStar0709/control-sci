# 3.2.1. The A Posteriori Truth

One may argue that we will be able to know the realized disturbance sequence ¯w at step N, so the true probability of the realized disturbance sequence is one, and the probabilities of all other disturbance sequence are zero. So for the belief $\bar { W } _ { b } = \mathcal { P } ( o ( s ) )$ , ¯w(s) is the a posteriori truth.
