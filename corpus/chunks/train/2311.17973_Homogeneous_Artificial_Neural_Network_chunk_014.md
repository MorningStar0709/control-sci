In Section 5 we present an ANN-based approximation of $\| \cdot \| _ { \mathbf { d } }$ , which uses the so-called d-homogeneous projector on the unit sphere

$$\pi_ {\mathbf {d}} (x) = \mathbf {d} (- \ln \| x \| _ {\mathbf {d}}) x, \quad x \neq \mathbf {0}.$$

Obviously, $\| \pi _ { \mathbf { d } } ( x ) \| _ { \mathbf { d } } = 1$ and $\pi _ { \bf d } ( \pi _ { \bf d } ( x ) ) = \pi _ { \bf d } ( x ) , \forall x \neq { \bf 0 } .$ so π is, indeed, a projector on the unit sphere $\{ x \in \mathbb { R } ^ { n } : \| x \| _ { \mathbf { d } } = 1 \}$ . In the case of the canonical homogeneous norm this sphere coincides with $S = \{ x \in \mathbb { R } ^ { n } : \| x \| = 1 \}$ , where $\| \cdot \|$ is the conventional norm in $\mathbb { R } ^ { n }$ that induces $\| \cdot \| _ { \mathbf { d } }$ (see Definition 1).
