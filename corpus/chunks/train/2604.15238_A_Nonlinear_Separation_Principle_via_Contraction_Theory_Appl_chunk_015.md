Extending these results to parametric systems allows one to translate the robustness properties of contracting dynamics into robust stability of the closed-loop system with explicit convergence rate estimates. We formalize this in two corollaries. The first establishes error bounds when the controller and observer design relies on an imprecise parameter’s estimate. The second bounds the tracking error relative to the instantaneous equilibrium $x ^ { \star } ( \theta ( t ) )$ when the parameter varies in time.

Corollary 8 (Robustness to model error): Let Θ be a subset of a normed space with norm ∥·∥Θ. Given maps $F \colon \mathbb { R } ^ { n } \times \mathbb { R } ^ { m } \times$ $\Theta \to \mathbb { R } ^ { n }$ and $h \colon \mathbb { R } ^ { n } \times \Theta  \mathbb { R } ^ { p }$ , consider the control system $\dot { x } = F ( x , u , \theta )$ with output $y = h ( x , \theta )$ . Let $L \colon \mathbb { R } ^ { n } \times \mathbb { R } ^ { m } \times$ $\mathbb { R } ^ { p } \times \Theta \to \mathbb { R } ^ { n }$ be an observer, and let $K \colon \mathbb { R } ^ { n } \times \Theta  \mathbb { R } ^ { m }$ be a controller. Let Assumptions $( \mathbf { A } 1 ) – ( \mathbf { A } 4 )$ hold uniformly for $\theta \in \Theta$ . Further, let K be ℓK,θ-Lipschitz in θ, uniformly in $x ,$ and let L be $\ell _ { L , \theta ^ { - 1 } }$ ipschitz in θ, uniformly in its other arguments. Consider the closed-loop system where the plant corresponds to the true parameter θ, while the controller and the observer use the estimate $\hat { \theta } { : }$

$$\dot {x} = F (x, K (\xi , \hat {\theta}), \theta), \tag {12a}\dot {\xi} = L (\xi , K (\xi , \hat {\theta}), h (x, \theta), \hat {\theta}). \tag {12b}$$

Then, the solutions satisfy the differential inequalities

$$D ^ {+} \| \xi - x \| _ {\mathcal {O}} \leq - c _ {O} \| \xi - x \| _ {\mathcal {O}} + \ell_ {L, \theta} \| \hat {\theta} - \theta \| _ {\Theta}, \tag {13}
\begin{array}{l} D ^ {+} \| x - x ^ {\star} (\theta) \| _ {\mathcal {X}} \leq - c _ {K} \| x - x ^ {\star} (\theta) \| _ {\mathcal {X}} + \ell_ {u} \ell_ {K} \| \xi - x \| _ {\mathcal {O}} \\ + \ell_ {u} \ell_ {K, \theta} \| \hat {\theta} - \theta \| _ {\Theta}. \tag {14} \\ \end{array}
$$
