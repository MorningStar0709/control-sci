# 2.1 Multi-agent Systems

Consider a multi-agent system (MAS) of N agents with a discrete time single integrator model:

$$x _ {i} (t + 1) = x _ {i} (t) + u _ {i} (t), i = 1, 2, \dots , N, \tag {1}$$

where $u _ { i } ( t ) \ \in \ \mathbb { R } ^ { n }$ and $x _ { i } ( t ) \ \in \ \mathbb { R } ^ { n }$ represent the input and state (position), respectively. The initial state $x _ { i } ( 0 )$ is deterministic for all $i \doteq 1 , 2 , \ldots , N$ .

The MAS is connected through an undirected communication graph denoted by $\mathcal { G } ( \mathcal { V } , \mathcal { E } , \mathcal { A } )$ . The graph  is characterized by a node set $\mathcal { V } = 1 , 2 , \dotsc , N$ , an edge set $\mathcal { E } \subseteq \mathcal { V } \times \mathcal { V }$ and an adjacency matrix $\mathcal { A } = [ \stackrel { \prime } { a } _ { i j } ] \in \mathbb { R } ^ { N \times N }$ . Let $N _ { \mathcal { E } } = | \mathcal { E } |$ denote the number of edges and label the edges as $l _ { 1 } , \dot { l } _ { 2 } , \ldots , l _ { N _ { \varepsilon } }$ following the node index order. For $( j , { \bar { i } } ) \in \mathcal E ,$ there exists a communication channel between node j and node i, indicating that node j is a neighbour of node i. The set of neighbours of node i is denoted by $\mathcal { N } _ { i } = \{ j \in \mathcal { V } \mid ( j , i ) \in \mathcal { E } \}$ . Moreover, if ${ \bf { j } } \in \mathcal { N } _ { i } ,$ , then $a _ { i j } = \mathrm { \ i }$ and $a _ { i i } = 0$ otherwise. The edge weight matrix W is then defined as $W \triangleq$ diag $\left\{ a _ { i j } \right\} = \mathbf { I } _ { N \varepsilon } , ( i , j ) = l _ { 1 } , \ldots , l _ { N \varepsilon }$ . An orientation on $\mathcal { G }$ is the assignment of a direction to each edge. Then, an incidence matrix $B = [ b _ { i j } ] \in \mathbb { R } ^ { N \varepsilon \times N }$ of G is defined as $b _ { i j } = 1$ if edge i enters node $\mathbf { \bar { \rho } } _ { j } , b _ { i j } = - 1$ if it leaves node $j ,$ , and $b _ { i j } = 0$ otherwise, $i = l _ { 1 } , l _ { 2 } , \ldots , l _ { N \varepsilon }$ .

The primary cooperative objective is to achieve

$$\lim _ {t \to \infty} \| x _ {i} (t) - x _ {j} (t) - d _ {i j} \| \to 0, \forall (j, i) \in \mathcal {E}, \tag {2}$$

where $d _ { i j } ~ \in ~ \mathbb { R } ^ { n }$ is the desired relative state, and let $d _ { i j } = - \bar { d } _ { j i } \mathrm { i f } ( j , i ) \in \mathcal { E }$ . The following assumption and lemma related to the communication graph are introduced and will be employed later.
