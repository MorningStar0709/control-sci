Lemma A.1. If $w \in \mathcal { C } _ { m , n } ^ { 0 , \alpha }$ is such that $\begin{array} { r } { T [ w ] = \frac { 1 } { \varrho } w = \big ( \frac { 1 } { \varrho } w _ { \ell , \iota } \big ) ( \ell , \iota ) \in \mathbb { M } \times \mathbb { I } } \end{array}$ for some $\varrho \in ( 0 , 1 ]$ , then (2.8) holds.

Proof. Considering $( x _ { \circ } , \ell _ { \circ } , \iota _ { \circ } ) , ( x ^ { \circ } , \ell ^ { \circ } , \iota ^ { \circ } ) \in \overline { { \mathcal { O } } } \times \mathbb { M } \times \mathbb { I }$ be such that

$$w _ {\ell_ {\circ}, \iota_ {\circ}} (x _ {\circ}) = \min _ {(x, \ell , \iota) \in \overline {{\mathcal {O}}} \times \mathbb {M} \times \mathbb {I}} w _ {\ell , \iota} (x) \quad \text {and} \quad w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) = \max _ {(x, \ell , \iota) \in \overline {{\mathcal {O}}} \times \mathbb {M} \times \mathbb {I}} w _ {\ell , \iota} (x),$$

$$
\begin{array}{l} \mathrm{D} ^ {1} w _ {\ell_ {\circ}, \iota_ {\circ}} (x _ {\circ}) = \mathrm{D} ^ {1} w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) = 0, \operatorname{tr} [ a _ {\iota^ {\circ}} \mathrm{D} ^ {2} w _ {\ell^ {\circ}, \iota^ {\circ}} ] (x ^ {\circ}) \leq 0 \leq \operatorname{tr} [ a _ {\iota_ {\circ}} \mathrm{D} ^ {2} w _ {\ell_ {\circ}, \iota_ {\circ}} ] (x _ {\circ}), \\ w _ {\ell_ {\circ}, \iota_ {\circ}} (x _ {\circ}) - w _ {\ell_ {\circ}, \kappa} (x _ {\circ}) \leq 0 \text {for} \kappa \in \mathbb {I} \setminus \{\iota_ {\circ} \}, w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) - w _ {\ell_ {\circ}, \kappa} (x ^ {\circ}) \geq 0 \text {for} \kappa \in \mathbb {I} \setminus \{\iota^ {\circ} \}. \\ \end{array}
