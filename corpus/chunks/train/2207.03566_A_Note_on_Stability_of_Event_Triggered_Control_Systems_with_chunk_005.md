To introduce the triggering condition for determining the time sequence $\{ t _ { i } \} _ { i \in \mathbb { N } }$ , we recall some concepts related to the Lyapunov functional candidate for time-delay systems. A function $V : \mathbb { R } ^ { + } \times \mathbb { R } ^ { n } \mapsto \mathbb { R } ^ { + }$ is said to be of class $\mathcal { V } _ { 0 }$ and we write $V \in \mathcal { V } _ { 0 } ,$ if, for each $x \in C ( \mathbb { R } ^ { + } , \mathbb { R } ^ { n } )$ , the composite function $t \mapsto V ( t , x ( t ) )$ is continuous.
