# B. Stability and convergence:

From instant k, assume a feasible periodic trajectory $\hat { \zeta } _ { k } .$ From the feasible state $x _ { k }$ and considering $\hat { \zeta } _ { k }$ for the linearization of $\widetilde { \mathbb { P } } ,$ the solution to $\widetilde { \mathbb { P } } _ { k } ( x _ { k } , \hat { \zeta _ { k } } , \bar { p } )$ would be $( z _ { k } ^ { * } , \zeta _ { k } ^ { * } )$ . Thus, the cost associated to the system at instant k would be $\widetilde { V } _ { k } ( z _ { k } ^ { * } , \zeta _ { k } ^ { * } , \hat { \zeta } _ { k } , p )$ . First it is proved that this is a decreasing function.

For instant instant, denote $k + 1$ a solution for this new, such that1 $\boldsymbol { z } _ { k } ^ { + } : = ( x _ { k , 0 } ^ { + } , \mathbf { u } _ { k } ^ { + } )$

$$x _ {k, 0} ^ {+} = x _ {k, 1} ^ {*}, \quad \mathbf {u} _ {k} ^ {+} = \left[ u _ {k, 1} ^ {*}, \ldots , u _ {k, N - 1} ^ {*}, \nu_ {k, N} ^ {*} \right].$$

This solution is feasible, as proven in [18, Theorem 3]. Next, according to step 4 in Algorithm 1, the feasible trajectory for the linearization in this instant k + 1, denoted as $\hat { \zeta } _ { k + 1 } ,$ is obtained by shifting $\zeta _ { k } ^ { * }$ as:

$$\hat {\xi} _ {k + 1, 0} = \xi_ {k, 1} ^ {*}, \quad \hat {\boldsymbol {\nu}} _ {k + 1} = \left[ \nu_ {k, 1} ^ {*}, \dots , \nu_ {k, T - 1} ^ {*}, \nu_ {k, 0} ^ {*} \right]. \tag {14}$$

Denote

$$\Delta \widetilde {V} = \widetilde {V} _ {k + 1} (\boldsymbol {z} _ {k} ^ {+}, \hat {\boldsymbol {\zeta}} _ {k + 1}, \hat {\boldsymbol {\zeta}} _ {k + 1}, p) - \widetilde {V} _ {k} (\boldsymbol {z} _ {k} ^ {*}, \boldsymbol {\zeta} _ {k} ^ {*}, \hat {\boldsymbol {\zeta}} _ {k}, p).$$

1Here, we denote by $x _ { k , i }$ the value of x at time $k + i$ predicted from the initial condition $x _ { k }$ .

$$
