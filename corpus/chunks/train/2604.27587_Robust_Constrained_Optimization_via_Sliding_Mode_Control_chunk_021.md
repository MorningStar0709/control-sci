# PROOF. The proof is divided into three steps.

(1) Finite-Time Reaching of the Sliding Manifold: Substituting the control law (10) into the S-dynamics yields $\dot { S } = - K _ { 1 } S - K _ { 2 } | S | ^ { \rho } \mathrm { s g n } ( S ) - \eta \mathrm { s g n } ( S ) + \xi ( t )$ , where $\xi ( t )$ is a matched disturbance bounded by $\| \xi ( t ) \| \overset { } { \leq } \dot { D }$ . $$\dot {V} _ {S} \leq - \lambda_ {\mathrm{min}} (K _ {1}) \| S \| ^ {2} - \lambda_ {\mathrm{min}} (K _ {2}) \| S \| ^ {1 + \rho},$$

since $\eta > D , S$ reaches the origin in finite time $T _ { 1 }$   
(2) Finite-Time Constraint Satisfaction: On the sliding manifold $S ~ = ~ 0$ , the nonsingular terminal constraint implies $\begin{array} { r } { z _ { 1 } ~ = ~ - \frac { 1 } { \beta } | z _ { 2 } | ^ { \gamma } \mathrm { { s g n } } ( z _ { 2 } ) } \end{array}$ . Therefore, $z _ { 1 } ~ = ~ h ( x )$ and $z _ { 2 } = \dot { h } ( x )$ converge to zero in finite time $T _ { 2 }$ .

on $h ( x ) = 0$ with $\dot { h } ( x ) = 0 .$ .
