Lastly, we show the uniqueness of the generated trajectory. Suppose there exists $\pmb { g } _ { 1 }$ and $\mathbf { \delta } \mathbf { \mathcal { g } } _ { 2 }$ , which are both solutions of (10) and $\mathcal { H } g _ { 1 } \ne \mathcal { H } g _ { 2 }$ . Since $\pmb { g } _ { 1 }$ and $\mathbf { \delta } \mathbf { \mathcal { g } } _ { 2 }$ are both solution of (10), then $G _ { \theta } g _ { 1 } \ = \ G _ { \theta } g _ { 1 }$ and thus $( \pmb { g } _ { 1 } - \pmb { g } _ { 2 } ) \in \mathcal { N } ( \pmb { G } _ { \pmb { \theta } } )$ . On the other hand, $\mathcal { H } g _ { 1 } \ \ne \ \mathcal { H } g _ { 2 }$ yields $\mathscr { H } \left( { \pmb g } _ { 1 } - { \pmb g } _ { 2 } \right) \neq 0$ and thus $( \pmb { g } _ { 1 } - \pmb { g } _ { 2 } ) \notin \mathcal { N } ( \mathcal { H } )$ . This contradicts that $\mathcal { N } ( G _ { \theta } )$ is the same as $\mathcal { N } ( \mathcal { H } )$ . Hence, $\mathcal { H } g _ { 1 } = \mathcal { H } g _ { 2 }$ , namely, the generated trajectories are identical.
