Theorem 3.12. The impulsive switched system (3) governed by the graph G and having a compact set of impulses $\mathcal { T }$ is stable under arbitrary impulses for all switching signals $\sigma \in { \cal S } ^ { \mathcal { G } } [ \tau _ { I } , \eta _ { I } ]$ , where

$$\tau_ {I} = \max _ {(p, q) \in \mathcal {E} (\mathcal {G} _ {s})} \max _ {M \in \mathcal {I}} \frac {\ln (c _ {p} \| P _ {q} ^ {- 1} M P _ {p} \|)}{\lambda_ {p} ^ {*}}, \quad \eta_ {I} = - \max _ {(p, q) \in \mathcal {E} (\mathcal {G} _ {u})} \max _ {M \in \mathcal {I}} \frac {\ln (c _ {p} \| P _ {q} ^ {- 1} M P _ {p} \|)}{\mu_ {p} ^ {*}}.$$

Proof. Let $\sigma \in \mathcal { S } [ \tau _ { I } , \eta _ { I } ]$ . If $p \in { \mathfrak { u } }$ , the time spent in the subsystem $A _ { p } ,$ , say $\eta ,$ satisfies $\eta \leq \eta _ { I }$ . Thus, we have $\| P _ { q } ^ { - 1 } M P _ { p } \| \leq c _ { p } ^ { - 1 } \mathrm { e } ^ { - \mu _ { p } ^ { * } \eta }$ for all $M \in \mathcal { T }$ . Similarly, if $p \in { \mathfrak { s } } .$ , the time spent in the subsystem $A _ { p } ,$ say τ , satisfies $\tau \geq \tau _ { I }$ . Then $\| P _ { q } ^ { - 1 } M P _ { p } \| \leq c _ { p } ^ { - 1 } \mathrm { e } ^ { \lambda _ { p } ^ { * } \tau }$ for all $M \in \mathcal { T }$ . Hence the result follows. Note that for each $p , q \in { \mathcal { P } } .$ , the functions $f _ { p , q } ( M ) \ : = \ : \ln \left( c _ { p } \| P _ { q } ^ { - 1 } M P _ { p } \| \right) / \lambda _ { p } ^ { * }$ and $g _ { p , q } ( M ) \ : = \ : - \ln \left( c _ { p } \left\| P _ { q } ^ { - 1 } M P _ { p } \right\| \right) / \mu _ { p } ^ { * }$ are continuous, hence attain maximum and minimum, respectively, in compact domains.
