the right-invariant error $\eta _ { t } ^ { R }$ have a state-trajectory independence propagation[18], which can be expressed differential equation as follow:

$$\frac {d}{d t} \eta_ {t} ^ {R} = g (\eta_ {t} ^ {R}, u _ {t}) - (A d _ {\chi} w _ {t}) ^ {\wedge} \eta_ {t} ^ {R}, \tag {10}$$

where, $g ( \eta _ { t } ^ { R } , u _ { t } ) = f ( \eta _ { t } ^ { R } , u _ { t } ) - \eta _ { t } ^ { R } f ( I _ { d } , u _ { t } )$ . If the error is assumed to be small, we can define a vector error variable $\xi _ { t }$ as

$$\eta_ {t} ^ {R} = \exp (\xi_ {t} ^ {\wedge}) \tag {11}$$

From the Eq. (10) and the first-order approximation $\exp ( \xi _ { t } ^ { \wedge } ) \approx I + \xi ^ { \wedge }$ , we can find a linear approximation of the error dynamics

$$\frac {d}{d t} \xi_ {t} = A _ {u _ {t}} \xi_ {t} - A d _ {\hat {\chi}} w _ {t} \tag {12}$$

The propagation matrix $A _ { u _ { t } }$ is defined by $\begin{array} { r l } {  { g ( \exp ( \xi _ { t } ^ { \wedge } ) , u _ { t } ) = } } \end{array}$ $( A _ { u _ { t } } \bar { \xi } ) ^ { \wedge } + { \cal O } ( \| \xi _ { t } \| ^ { 2 } )$ .

From Eq. (7b), (8), and (11), the linearized error update is as follows:

$$
\xi_ {k} ^ {+} = \xi_ {k} + K _ {k} \left( \begin{array}{c} s ^ {1} (\xi_ {k} ^ {\wedge} b _ {k} ^ {1} + \hat {\chi} _ {k} V _ {k} ^ {1}) \\ \vdots \\ s ^ {m} (\xi_ {k} ^ {\wedge} b _ {k} ^ {m} + \hat {\chi} _ {k} V _ {k} ^ {m}) \end{array} \right) = \xi_ {k} - K _ {k} (H _ {k} \xi_ {k} - M _ {k} v _ {k}) \tag {13}
$$
