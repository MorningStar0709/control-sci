Note that both $c _ { \mu }$ and $p _ { \mu }$ are Lipschitz continuous with Lipschitz constants $( \mathsf { L } _ { 1 } , \mathsf { L } _ { 2 } )$ and $( \mathsf { K } _ { 1 } , \mathsf { K } _ { 2 } )$ , respectively, under assumptions (b) and (c). We now prove that the optimal policy of $\mathrm { M D P } _ { \mu } .$ , denoted as $\pi _ { \mu } .$ , is Lipschitz continuous.

LEMMA 3.1. Given any $\mu ,$ , there exits a single element $\pi _ { \mu }$ in the set $\Lambda ( \mu )$ , which is the unique optimal policy of $M D P _ { \mu }$ . Moreover, $\pi _ { \mu }$ is Lipschitz continuous with Lipschitz constant $\textstyle \frac { 2 \mathsf { L } _ { F } } { \rho } = : \mathsf { L } ^ { * }$ .

PROOF. Define

$$F _ {\mu} (x, a) := c _ {\mu} (x, a) + \beta \int_ {\mathsf {X}} J _ {\mu} (y) p _ {\mu} (d y | x, a).$$

For any $x \in \mathsf { X } , F _ { \mu } ( x , \cdot )$ is $\rho \mathbf { \cdot }$ strongly convex by assumption (d); that is, for any $a , { \hat { a } } \in \mathsf { A } .$ , we have

$$F _ {\mu} (x, \hat {a}) \geq F _ {\mu} (x, a) + \langle \nabla_ {a} F _ {\mu} (x, a), \hat {a} - a \rangle + \frac {\rho}{2} \| a - \hat {a} \| ^ {2}.$$

By strong convexity of $F _ { \mu } ( x , \cdot )$ , there exists an unique optimal policy $\pi _ { \mu } : \mathsf { X } \to \mathsf { A }$ as the following optimality equation admits unique solutions for all $x \in \mathsf X$

$$\min _ {a \in \mathsf {A}} \left[ c _ {\mu} (x, a) + \beta \int_ {\mathsf {X}} J _ {\mu} (y) p _ {\mu} (d y | x, a) \right] =: \min _ {a \in \mathsf {A}} F _ {\mu} (x, a)$$

To prove the Lipschitz continuity of $\pi _ { \mu } .$ , which is the unique minimizer of the above optimality equation for all $x \in \mathsf { X } ,$ fix any $x , y \in \mathsf { X }$ . Then, we have

$$
\begin{array}{l} 0 \geq F _ {\mu} (x, \pi_ {\mu} (x)) - F _ {\mu} (x, \pi_ {\mu} (y)) \\ \geq \langle \nabla_ {a} F _ {\mu} (x, \pi_ {\mu} (y)), \pi_ {\mu} (x) - \pi_ {\mu} (y) \rangle + \frac {\rho}{2} \| \pi_ {\mu} (x) - \pi_ {\mu} (y) \| ^ {2} \\ \end{array}
$$

$$\| \pi_ {\mu} (x) - \pi_ {\mu} (y) \| ^ {2} \leq \frac {2}{\rho} \langle - \nabla_ {a} F _ {\mu} (x, \pi_ {\mu} (y)), \pi_ {\mu} (x) - \pi_ {\mu} (y) \rangle$$
