# VI. SIMULATION RESULTS

For illustrating the proposed observer design, we consider the SIDARTHE-V model [30] described by

$$
\begin{array}{l} \dot {\mathrm{S}} = - \mathrm{S} (\alpha \mathrm{I} + \beta \mathrm{D} + \gamma \mathrm{A} + \delta \mathrm{R}) - \varphi \mathrm{S} \\ \dot {\mathbf {I}} = \mathbf {S} (\alpha \mathbf {I} + \beta \mathbf {D} + \gamma \mathbf {A} + \delta \mathbf {R}) - (\varepsilon + \zeta + \lambda) \mathbf {I} \\ \dot {\mathrm{D}} = \varepsilon \mathrm{I} - (\eta + \rho) \mathrm{D} \\ \dot {\mathrm{A}} = \zeta \mathrm{I} - (\theta + \mu + \kappa) \mathrm{A} \\ \dot {\mathrm{R}} = \eta \mathrm{D} + \theta \mathrm{A} - (\nu + \xi + \tau_ {1}) \mathrm{R} \tag {24} \\ {\dot {\mathrm{T}}} = {\mu \mathrm{A} + \nu \mathrm{R} - (\sigma + \tau_ {2}) \mathrm{T}} \\ \dot {\mathrm{H}} = \lambda \mathrm{I} + \rho \mathrm{D} + \kappa \mathrm{A} + \xi \mathrm{R} + \sigma \mathrm{T} \\ {\dot {\mathrm{E}}} = {\tau_ {1} \mathrm{R} + \tau_ {2} \mathrm{T}} \\ \dot {\mathrm{V}} = \varphi \mathrm{S} \\ \end{array}
$$

where the states correspond to the proportion of Susceptible, Infected (asymptomatic, undetected), Diagnosed (asymptomatic, detected), Ailing (symptomatic, undetected), Recognized (symptomatic, detected), Threatened (acutely symptomatic, detected), Healed (recovered naturally or with treatment), Extinct (died due to the disease). That is, the state space $\mathcal { X } = [ 0 , 1 ] ^ { n _ { x } }$ with $n _ { x } = 9$ . For the definition and interpretation of all the parameters, please refer to [30].
