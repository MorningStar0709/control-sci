# 3. BATTERY MODEL

This section summarizes the reduced-order electrochemical model used in the SG-DKF. The model follows the SPMe-type framework (Li et al., 2018, 2019) and is implemented in discrete time with sampling period ∆t.

Solid-phase diffusion in each electrode is approximated using the parabolic method (Guo and Couto, 2025a; Li et al., 2018). The surface stoichiometries are

$$x _ {s, p, \text { surf }} (k) = x _ {s, p, \text { avg }} (k) + \Delta x _ {s, p} (k), \tag {34}x _ {s, n, \text { surf }} (k) = x _ {s, n, \text { avg }} (k) + \Delta x _ {s, n} (k), \tag {35}$$

where $x _ { s , p , \mathrm { a v g } } ( k )$ and $x _ { s , n , \mathrm { a v g } } ( k )$ denote the averaged stoichiometries and $\Delta x _ { s , p } ( k ) , \tilde { \Delta x } _ { s , n } ( k )$ are diffusion states.

Charge balance gives the averaged stoichiometry dynamics

$$x _ {s, p, \mathrm{avg}} (k + 1) = x _ {s, p, \mathrm{avg}} (k) + \frac {D _ {p}}{Q _ {\mathrm{eff}}} I (k) \Delta t, \tag {36}
\begin{array}{l} x _ {s, n, \text { avg }} (k + 1) = x _ {s, n, \text { avg }} (k) \\ - \frac {D _ {n}}{D _ {p}} \big [ x _ {s, p, \mathrm{avg}} (k + 1) - x _ {s, p, \mathrm{avg}} (k) \big ], \tag {37} \\ \end{array}
$$

where $D _ { p }$ and $D _ { n }$ are capacity-related coefficients. The parabolic approximation yields first-order diffusion dynamics

$$\Delta x _ {s, p} (k + 1) = a _ {p} \Delta x _ {s, p} (k) + b _ {p} I (k), \tag {38}\Delta x _ {s, n} (k + 1) = a _ {n} \Delta x _ {s, n} (k) + b _ {n} I (k), \tag {39}$$

with $a _ { p } , a _ { n } , b _ { p } , b _ { n }$ functions of $\tau _ { p } ^ { s } , \tau _ { n } ^ { s } , D _ { p } , D _ { n } , Q _ { \mathrm { e f f } }$ as detailed in (Li et al., 2018).
