$$- 2 (1 - c) P ^ {- 1} + P ^ {- 1} W ^ {\top} Q W P ^ {- 1} + Q ^ {- 1} \preceq 0, \tag {31}$$

which is equivalent, by the Schur complement, to the inequality (27) with $W ^ { \prime } = \dot { W } ^ { \top } , P ^ { \prime } = P ^ { - 1 }$ , and $Q ^ { \prime } = Q ^ { - 1 }$ .

A similar trick, based on the Schur complement, applies to the discrete-time MONE condition: upon rewriting it as

$$P - 2 Q + \rho^ {- 2} Q W P ^ {- 1} W ^ {\top} Q \preceq 0 \tag {32}$$

and pre- and post-multiplying by $Q ^ { - 1 }$ , one obtains

$$Q ^ {- 1} P Q ^ {- 1} - 2 Q ^ {- 1} + \rho^ {- 2} W P ^ {- 1} W ^ {\top} \preceq 0, \tag {33}$$

which is equivalent, by the Schur complement, to the Hopfield matrix inequality (28) with parameters $W ^ { \prime } = W ^ { \top } , \ P ^ { \prime } =$ $\rho ^ { - 2 } P ^ { - 1 }$ , and $Q ^ { \prime } = Q ^ { - 1 }$ . Finally, for the discrete time CONE case, both conditions are reducible to Schur diagonal stability as shown in Lemma 13. We show that $W ^ { \top } Q W \preceq \rho ^ { 2 } Q$ if and only if $W Q ^ { - 1 } W ^ { \top } \preceq \rho ^ { 2 } Q ^ { - 1 }$ . The former inequality can be written as $X ^ { \top } X \leq I .$ , where $X = \rho ^ { - 1 } Q ^ { 1 / \bar { 2 } } W \bar { Q ^ { - 1 / 2 } }$ , and the latter as $X X ^ { \top } \leq I .$ The equivalence follows since X and $X ^ { \top }$ have same singular values. A more direct proof can be obtained by applying Schur complement to the LMI $\left[ \begin{array} { c c } { { I } } & { { X } } \\ { { X ^ { \top } } } & { { I } } \end{array} \right] \succeq 0$ with respect to each diagonal block.

Corollary 15 (Equivalence to LMI): For a fixed $c > 0$ or $\rho \in [ 0 , 1 )$ , each certificate in Table I is equivalent to a Linear Matrix Inequality (LMI) under a bijective change of variables. Consequently, for fixed rate, the set of admissible triplets $( W , P , Q )$ is characterized by a convex feasibility problem.
