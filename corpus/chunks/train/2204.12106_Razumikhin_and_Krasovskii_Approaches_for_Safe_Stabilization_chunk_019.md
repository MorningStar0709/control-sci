$$L _ {f} V _ {\mathrm{r}} (\phi) + \gamma_ {\mathrm{r}} (V _ {\mathrm{r}} (x)) < \varepsilon \| L _ {g} V _ {\mathrm{r}} (\phi) \| + \eta_ {\mathrm{r}} (\| V _ {\mathrm{r}} (\phi) \|).$$

which implies a $\mathrm { \Phi } _ { \mathrm { r } } ( \phi ) | < \varepsilon | | L _ { g } V _ { \mathrm { r } } ( \phi ) |$ . Furthermore, from (12) and (15), we have that for any nonzero $\phi \in C ( [ - \Delta , 0 ] , \mathbf { B } ( \delta ) )$ ,

$$
\begin{array}{l} \| u (\phi) \| = \left\| \frac {\mathfrak {a} _ {\mathrm{r}} (\phi) + \sqrt {\mathfrak {a} _ {\mathrm{r}} ^ {2} (\phi) + \lambda \| L _ {g} V _ {\mathrm{r}} (\phi) \| ^ {4}}}{\| L _ {g} V _ {\mathrm{r}} (\phi) \| ^ {2}} L _ {g} V _ {\mathrm{r}} (\phi) \right\| \\ \leq \frac {2 | \mathfrak {a} _ {\mathrm{r}} (\phi) | + \sqrt {\lambda} \| L _ {g} V _ {\mathrm{r}} (\phi) \| ^ {2}}{\| L _ {g} V _ {\mathrm{r}} (\phi) \|} \\ \leq (2 + \sqrt {\lambda}) \varepsilon =: \bar {\varepsilon}. \\ \end{array}

Therefore, if the R-SCP is satisfied, then the control input is bounded and the bound is a function of $\varepsilon \in \mathbb { R } ^ { + }$ . Note that the controller (11) is continuous for both the case of nonzero φ and the case where $\phi ( \theta ) = 0$ for all $\theta \in [ - \Delta , 0 ]$ . In addition, if $g ( 0 ) = 0$ , then $L _ { g } V _ { \Gamma } ( 0 ) = 0$ and thus $u ( 0 ) = 0$ from (12), which implies that the case $\phi = 0$ can be removed from (11). Finally, the controller (11) follows Sontag’s formula [3] and extends the results in [3, 9, 37] from the delay-free case to the time-delay case.
