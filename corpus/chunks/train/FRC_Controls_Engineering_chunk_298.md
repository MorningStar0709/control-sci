# 9.6.1 Derivations

Given the a posteriori update equation $\hat { \mathbf { x } } _ { k + 1 } ^ { + } = \hat { \mathbf { x } } _ { k + 1 } ^ { - } + \mathbf { K } _ { k + 1 } ( \mathbf { y } _ { k + 1 } - \hat { \mathbf { y } } _ { k + 1 } )$ , we want to find the value of ${ \bf K } _ { k + 1 }$ that minimizes the a posteriori estimate covariance (the error covariance) because this minimizes the estimation error.
