where $\mathcal { C } _ { k }$ measures closeness between $\mu _ { k }$ and $\rho _ { N }$ , akin to $G _ { k }$ and $G _ { N }$ in (1.2).

Example 1.3 (Large-scale multi-agent systems). The optimal steering of a fleet of M identical agents, with dynamics $x _ { k + 1 } ^ { ( i ) } = f _ { k } ( x _ { k } ^ { ( i ) } , u _ { k } ( x _ { k } ^ { ( i ) } ) )$ , from an initial configuration $\{ x _ { k } ^ { ( i ) } \} _ { i = 1 } ^ { M }$ to a desired one $\rho _ { N }$ k+1 kcan be cast as

$$\inf _ {u _ {k}: X _ {k} \to U _ {k}} \sum_ {k = 0} ^ {N} \mathcal {C} _ {k} (\{x _ {k} ^ {(j)} \} _ {j = 1} ^ {M}, \rho_ {N}) + \frac {1}{M} \sum_ {i = 1} ^ {M} g _ {N} (x _ {N} ^ {(i)}) + \sum_ {k = 0} ^ {N - 1} g _ {k} (x _ {k} ^ {(i)}, u _ {k} (x _ {k} ^ {(i)})), \tag {1.4}$$

where $\mathcal { C } _ { k }$ is a fleet-specific cost $( \mathrm { e . g . } ,$ a cohesion or formation cost), and $g _ { N }$ and $g _ { k }$ are agent-specific costs (e.g., input effort). Often, the interest lies in the macroscopic behavior of the fleet. Hence, it is customary to capture the state of the fleet by a probability measure $\mu _ { k } \in \mathcal P ( X _ { k } )$ and the input by a map $u _ { k } : X _ { k } \to U _ { k } \ [ 3 9 , 4 1 ]$ . The optimization problem in (1.4) can then be written as an optimal control problem, with state $\mu _ { k }$ , input $u _ { k } .$ , and dynamics $\mu _ { k + 1 } = f _ { k } ( \cdot , u _ { k } ( \cdot ) ) _ { \# } \mu _ { k }$ . Overall, (1.5)

$$\inf _ {u _ {k}: X _ {k} \to U _ {k}} \int_ {X _ {N}} g _ {N} (x _ {N}) \mathrm{d} \mu_ {N} (x _ {N}) + \sum_ {k = 0} ^ {N - 1} \mathcal {C} _ {k} (\mu_ {k}, \rho_ {N}) + \int_ {X _ {k}} g _ {k} (x _ {k}, u _ {k} (x _ {k})) \mathrm{d} \mu_ {k} (x _ {k}).$$

Such a modeling approach suits robotics [52], mobility [56], and social networks [2, 38].

$$\inf _ {u _ {k}: X _ {k} \to U _ {k}} G _ {N} (\mu_ {N}, \rho_ {N}) + \sum_ {k = 0} ^ {N - 1} G _ {k} (\mu_ {k}, u _ {k}, \rho_ {k}), \tag {1.6}$$
