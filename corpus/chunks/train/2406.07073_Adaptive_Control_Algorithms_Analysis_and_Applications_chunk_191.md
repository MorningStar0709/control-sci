# 4.2 The Averaging Method for the Analysis of Adaptation Algorithms in a Stochastic Environment

Consider again an algorithm with a scalar decreasing adaptation gain of the form:

$$\hat {\theta} (t + 1) = \hat {\theta} (t) + \frac {1}{t} \phi (t) v (t + 1) \tag {4.30}$$

where $\phi ( t )$ is the observation (or measurement vector) and $\nu ( t + 1 )$ is the a posteriori adaptation error.

We will try next to describe the behavior of the algorithm for large $t \ ( t \gg 1 )$ . Taking a number of steps N such that $1 \ll N \ll t$ , one obtains from (4.30):

$$\hat {\theta} (t + N + 1) = \hat {\theta} (t) + \sum_ {i = 0} ^ {N} \frac {1}{t + i} \phi (t + i) v (t + 1 + i) \tag {4.31}$$

Taking again into account the fact that $t \gg 1$ and $t \gg N$ , the gains 1+ , i = $\begin{array} { r } { \frac { 1 } { t + i } , i = } \end{array}$ $0 , 1 , \ldots , N$ can be approximated by their mean value and one can write:

$$\hat {\theta} (t + N + 1) \approx \hat {\theta} (t) + \left(\sum_ {i = 0} ^ {N} \frac {1}{t + i}\right) \frac {1}{N + 1} \sum_ {i = 0} ^ {N} \phi (t + i) v (t + 1 + i) \tag {4.32}$$

In general, $\phi ( t + i )$ and $\nu ( t + 1 + i )$ will depend on the values of $\widehat { \theta } ( t + i )$ . However, for $t \gg 1$ and $1 \ll N \ll t$ since $\textstyle { \frac { 1 } { t + i } }$ is very small, $\widehat { \theta } ( t + i )$ is slowly varying and one can write:

$$
\begin{array}{l} \frac {1}{N + 1} \sum_ {i = 0} ^ {N} \phi (t + i) \nu (t + 1 + i) \\ \approx \frac {1}{N + 1} \sum_ {i = 0} ^ {N} \phi (t + i, \hat {\theta} (t)) v (t + 1 + i, \hat {\theta} (t)) \tag {4.33} \\ \end{array}
$$

where $\phi ( t + i , \hat { \theta } ( t ) )$ and $\nu ( t + 1 + i , \hat { \theta } ( t ) )$ are stationary quantities generated when for $i \geq 0$ , one fixes ${ \hat { \theta } } ( t + i ) = { \hat { \theta } } ( t )$ . Now if N is sufficiently large (but still $N \ll t )$ :

$$
\begin{array}{l} \frac {1}{N + 1} \sum_ {i = 0} ^ {N} \phi (t + i, \hat {\theta} (t)) v (t + 1 + i, \hat {\theta} (t)) \approx \mathbf {E} \{\phi (t + i, \hat {\theta} (t)) v (t + 1 + i, \hat {\theta} (t)) \} \\ = f (\hat {\theta} (t)) \tag {4.34} \\ \end{array}
$$

and (4.32) becomes:

$$\hat {\theta} (t + N + 1) - \hat {\theta} (t) \approx \left(\sum_ {i = 0} ^ {N} \frac {1}{t + i}\right) f (\hat {\theta} (t)) \tag {4.35}$$
