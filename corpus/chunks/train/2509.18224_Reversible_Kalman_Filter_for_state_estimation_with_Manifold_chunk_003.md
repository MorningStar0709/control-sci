The paper is structured as follows, using 9-axis IMU sensors as the main example. Section II introduces the notation and provides a theoretical overview of quaternion properties. Section III presents the Multiplicative Extended Kalman Filter (MEKF), used as a reference throughout this work, with an emphasis on its numerical properties. Section IV details the theoretical approach we propose, while Section V evaluates its performance on both synthetic and real data. Section VI discusses the main applications and limitations of our method. Finally, the Appendix outlines how our approach can be extended to sensor configurations combining odometry, accelerometer and pressure sensors instead of IMUs. The code for this project is available at the following repository: https://github.com/knov836/revmekf.
