$$
\begin{array}{l} \frac {d}{d t} \sum_ {i = 1} ^ {N} m _ {i} \| p _ {i} - c _ {E _ {i}} \| ^ {2} = - 2 \kappa_ {p} \sum_ {i = 1} ^ {N} m _ {i} \| p _ {i} - c _ {E _ {i}} \| ^ {2} + \sum_ {i = 1} ^ {N} \dot {\varphi} _ {i} [ \eta (\varphi_ {i}, p _ {i - 1}) - \eta (\varphi_ {i}, p _ {i}) ] \\ - \sum_ {i = 1} ^ {N} \dot {\varphi} _ {i} \left[ \eta (\varphi_ {i}, c _ {E _ {i - 1}}) - \eta (\varphi_ {i}, c _ {E _ {i}}) \right] \\ \end{array}
$$

Let $\begin{array} { r } { H ( t ) = \sum _ { i = 1 } ^ { N } m _ { i } \| p _ { i } - c _ { E _ { i } } \| ^ { 2 } \mathrm { ~ a n d ~ } e _ { \eta } ( \varphi _ { i } , p _ { i } , p _ { i - 1 } ) = \eta ( \varphi _ { i } , p _ { i - 1 } ) - \eta ( \varphi _ { i } , p _ { i } ) - \eta ( \varphi _ { i } , c _ { E _ { i - 1 } } ) + \eta ( \varphi _ { i } , c _ { E _ { i } } ) } \end{array}$ . Then it follows that

$$
\begin{array}{l} \dot {H} (t) = - 2 \kappa_ {p} H (t) + \sum_ {i = 1} ^ {N} \dot {\varphi} _ {i} \cdot e _ {\eta} (\varphi_ {i}, p _ {i}, p _ {i - 1}) \\ \leq - 2 \kappa_ {p} H (t) + \| \dot {\varphi} \| \cdot \| \mathbf {e} _ {\eta} (\varphi , p) \| \\ \leq - 2 \kappa_ {p} H (t) + \| \dot {\varphi} (t) \| \cdot \sup _ {\varphi \in [ 0, 2 \pi ], p \in \Omega} \| \mathbf {e} _ {\eta} (\varphi , p) \| \\ \end{array}
$$

with ${ \dot { \pmb { \varphi } } } = ( \dot { \varphi } _ { 1 } , \dot { \varphi } _ { 2 } , . . . , \dot { \varphi } _ { N } ) ^ { T }$ and $\mathbf { e } _ { \eta } ( \varphi , p ) = ( e _ { \eta } ( \varphi _ { 1 } , p _ { 1 } , p _ { N } ) , e _ { \eta } ( \varphi _ { 2 } , p _ { 2 } , p _ { 1 } ) , . . . , e _ { \eta } ( \varphi _ { N } , p _ { N } , p _ { N - 1 } ) ) ^ { T }$ . Solving the above differential inequality yields

$$
\begin{array}{l} H (t) \leq H (t _ {0}) e ^ {- 2 \kappa_ {p} (t - t _ {0})} + \sup _ {\varphi \in [ 0, 2 \pi ], p \in \Omega} \| \mathbf {e} _ {\eta} (\varphi , p) \| \cdot \int_ {t _ {0}} ^ {t} e ^ {2 \kappa_ {p} (\tau - t)} \| \dot {\varphi} (\tau) \| d \tau \\ \leq H (t _ {0}) e ^ {- 2 \kappa_ {p} (t - t _ {0})} + \sup _ {\varphi \in [ 0, 2 \pi ], p \in \Omega} \| \mathbf {e} _ {\eta} (\varphi , p) \| \cdot \frac {\sup _ {\tau \in [ t _ {0} , t ]} \| \dot {\varphi} (\tau) \|}{2 \kappa_ {p}}, \\ \end{array}
$$

which implies input-to-state stability of multi-agent dynamics according to the definition in [22].
