To establish the ultimate boundedness of the closed-loop state, we again proceed by contradiction.

For the sake of contradiction, assume that there exists some time $t ^ { * } > 0$ such that $z ( t _ { s } + t ^ { * } ) \not \in \overline { { B } } _ { z } ( r )$ . Since $z ( \cdot )$ is continuous and our selection of r ensures that $z ( t _ { s } ) \in B _ { z } ( r )$ , by the Intermediate Value Theorem, there exist time instances $\rho _ { 1 } \in ( 0 , t ^ { * } )$ and $\rho _ { 2 } \in ( \rho _ { 1 } , t ^ { * } ]$ such that $z ( t ) \in \overline { { B } } _ { z } ( r )$ for all $t \in [ t _ { s } , t _ { s } + \rho _ { 1 } ]$ and $z ( t ) \not \in \overline { { B } } _ { z } ( r )$ for all $t \in ( t _ { s } + \rho _ { 1 } , t _ { s } + \rho _ { 2 } )$ .
