# 3.3 Computational algorithm for switching

Now we integrate the above numerical solution with an algorithm to address the switching events. The proposed framework is summarized in Algorithm 1, where the solution to the full system is expressed as the root of $\varPhi ( Z ) =$ 0. The expanded function Φ concurrently computes the value of each individual term in (26) for every partition $( t _ { e _ { s } } , t _ { e _ { s + 1 } } )$ , under the constraint that initial and terminal conditions of $U ^ { e _ { s + 1 } } , M ^ { e _ { s } }$ s match. The domain of the oneto-one function Φ is $\mathbb { R } ^ { 2 \times ( N _ { T } + 1 ) \times ( N _ { X } + 1 ) }$ , whereas the domain of $\varphi _ { s }$ is $\mathbb { R } ^ { 2 \times ( N _ { t _ { s } } + 1 ) \times ( N _ { X } + 1 ) }$ , where $N _ { t _ { s } }$ represents the number of time steps in each continuous flow section.

Algorithm 1 Solver for switching systems   
Input: Initial and terminal condition : $U^{N_{T}}, M^{0}$ 1: Initial guess $U^{(0)}, M^{(0)}$ 2: for k = 0 to $N_{iter} - 1$ do

3: def $\Phi(Z)$ 4: $U^{e_{N_{s}+1}} \leftarrow U^{N_{T}}, M^{e_{0}} \leftarrow M^{0}$ 5: for s = 0 to $N_{s}$ do

6: $t_{s} \leftarrow \{t_{e_{s}}, \ldots, t_{e_{s+1}}\}$ 7: $U^{e_{s+1}} = U[t_{e_{s+1}}], M^{e_{s}} = M[t_{e_{s}}]$ 8: $X[t_{s}] \leftarrow \varphi_{s}((U[t_{s}], M[t_{s}]))$ 9: end for

10: return X

11: $\Delta Z^{(k)} \leftarrow \text{solve}\left\{J_{\Phi}(Z^{(k)}) Y = \Phi(Z^{(k)})\right\} \text{ for } Y$ 12: $Z^{(k+1)} \leftarrow Z^{(k)} - \Delta Z^{(k)}$ 13: end for

14: return $(U^{(N_{iter})}, M^{(N_{iter})}) = Z^{(N_{iter})}$

In numerical implementation, evaluating the inverse of the Jacobian of Φ is computationally expensive and unnecessary. Instead, the solution in Step 11 of Algorithm 1 can be computed by solving the linear system of equations

$$J _ {\Phi} (Z ^ {(k)}) Y = \Phi (Z ^ {(k)}), \tag {28}$$

to obtain $Y ~ = ~ J _ { \phi } ( Z ^ { ( k ) } ) ^ { - 1 } \varPhi ( Z ^ { ( k ) } )$ . Moreover, iterative solvers are available for this linear system that only require the Jacobian-vector product, thereby avoiding the need to construct the full Jacobian matrix explicitly.

The choice of the initial value $Z ^ { ( 0 ) }$ is crucial in the first step. Since the solution converges efficiently for large values of viscosity $\nu ,$ the system is first solved with a large ν, which then serves as the initial guess for the case with smaller ν. Thus a continuation method is applied to gradually reach to the required level of ν.
