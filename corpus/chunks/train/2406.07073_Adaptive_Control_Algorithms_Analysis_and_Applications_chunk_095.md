# Rapprochement with the Deterministic Case

Comparing Theorem 2.4 with Theorem 2.3, one can see immediately that the predictors for deterministic case and the stochastic case are the same if $P ( q ^ { - 1 } ) = C ( q ^ { - 1 } )$ ).

In other words, a deterministic predictor is an optimal predictor in the stochastic environment for an ARMAX type model with $C ( q ^ { - 1 } ) = P ( q ^ { - 1 } )$ . Conversely, the polynomial $C ( q ^ { - 1 } )$ will define the decaying dynamics of the prediction error in the deterministic case (and of the initial conditions in the stochastic case).
