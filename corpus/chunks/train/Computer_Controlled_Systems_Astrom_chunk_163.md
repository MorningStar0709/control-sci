# Input-Output Stability

DEFINITION 3.3 BOUNDED-INPUT BOUNDED-OUTPUT STABILITY A linear time-invariant system is defined as bounded-input-bounded-output (BIBO) stable if a bounded input gives a bounded output for every initial value.

From the definition it follows that asymptotic stability is the strongest concept. The following theorem is a result.

THEOREM 3.2 RELATION BETWEEN STABILITY CONCEPTS Asymptotic stability implies stability and BIBO stability.

When the word stable is used without further qualification in this text, it normally means asymptotic stability.

It is easy to give examples showing that stability does not imply BIBO stability, and vice versa.
