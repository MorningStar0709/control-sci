Proposition 10. Consider the system (1) with the safe set $\mathbb { C } \subset$ Rn and the initial state $\xi \in C ( [ - \Delta , 0 ] , \operatorname { I n t } ( \mathbb { C } ) )$ . If

(i) the control Lyapunov functional is $\mathcal { V } ( \phi )$   
(ii) the control barrier functional is ${ \mathcal { B } } ( \phi ) ,$ ,   
(iii) the sliding surface functional is

$$\mathbf {U} (\phi) := \alpha (\mathcal {V} (\phi)) + \beta (\mathcal {B} (\phi)), \tag {57}$$

with the functions α, ${ \mathcal { B } } : \mathbb { R } \to \mathbb { R } ^ { + }$ satisfying

$$\mathbf {U} (\phi) \geq \mathbf {U} (\xi), \quad \forall \phi \in \mathcal {C} ([ - \Delta , 0 ], \partial \mathbb {C}), \tag {58}$$

then the controller (50) guarantees simultaneously the stabilization and safety objectives of the system (1).

Proof. From Theorem 8, the stabilization objective is guaranteed via the controller (50), and we next show the satisfaction of the safety objective. From the construction of U(φ), $\mathbf { U } ( x _ { t } ) \geq 0$ for all $t > 0 .$ . Since $D ^ { + } \mathbf { W } ( \phi ) < 0$ , we have $\mathbf { U } ( x _ { t } ) < \mathbf { U } ( x _ { 0 } )$ for all $t > 0$ . From (58), $\mathbf { U } ( \phi ) \geq \mathbf { U } ( \xi )$ for all $\phi \in C ( [ - \Delta , 0 ] , \partial \mathbb { C } )$ , the state trajectory cannot reach the boundary of the safe set. Note that $\xi \in C ( [ - \Delta , 0 ] , \operatorname { I n t } ( \mathbb { C } ) )$ , and thus the state trajectory will stay in the safe set, which in turn ensures the safety objective. 

Remark 3. The condition (58) can be strengthened to be satisfied for $\phi \in C ( [ - \Delta , 0 ] , \partial \mathbb { C } _ { 1 } )$ with $\mathbb { C } _ { 1 } = \mathbb { C } - \mathbf { B } ( \varepsilon )$ and arbitrarily small $\varepsilon > 0 .$ . That is, the condition (58) is satisfied when the state is closed to ∂C instead of reaches ∂C. In such case, the reach of the boundary of the safe set is avoided to attenuate the possibility of unsafety due to chattering phenomena. 
