# II. UNKNOWN SYSTEM DYNAMICS AND CONTROL DESIGN

Consider a control-affine nonlinear dynamic system modeled as

$$\dot {x} = f (x) + u, \tag {3}$$

where $x : \mathbb { R } _ { > 0 } \to \mathbb { R } ^ { n }$ denotes a Filippov solution to (3), $f : \mathbb { R } ^ { n }  \mathbb { R } ^ { n }$ denotes an unknown differentiable drift vector field, and $u : \mathbb { R } _ { > 0 } \to \mathbb { R } ^ { n }$ denotes a control input.1 Let the tracking error $e : \mathbb { R } _ { \geq 0 } \to \mathbb { R } ^ { n }$ be defined as

$$e \triangleq x - x _ {d}, \tag {4}$$

where $x _ { d } : \mathbb { R } _ { \geq 0 } \to \mathbb { R } ^ { n }$ denotes a continuously differentiable reference trajectory. The reference trajectory is designed such that $\| x _ { d } ( t ) \| \leq \overline { { x _ { d } } } \forall t \in \mathbb { R } _ { \geq 0 }$ and $\dot { x } _ { d } \in { \mathcal { L } } _ { \infty }$ , where $\overline { { x _ { d } } } \in \mathbb { R } _ { > 0 }$ is a constant. The control objective is to design a ResNetbased adaptive controller that achieves asymptotic tracking error convergence.
