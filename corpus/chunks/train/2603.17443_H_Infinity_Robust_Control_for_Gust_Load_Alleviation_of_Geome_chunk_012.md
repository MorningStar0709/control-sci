# 3.2 Augmented Plant

The ROM is augmented with the actuator dynamics to form the design plant:

$$\dot {\mathbf {x}} = \mathbf {A} _ {p} \mathbf {x} + \mathbf {B} _ {w} \mathbf {w} _ {\text {ext}} + \mathbf {B} _ {u} u \tag {5}$$

where $\mathbf { x } = \{ \mathbf { z } , \delta , \dot { \delta } \} ^ { T }$ is the augmented state, $\mathbf { w } _ { \mathrm { e x t } }$ is the exogenous input (gust), and the plant matrices are:

$$
\mathbf {A} _ {p} = \left[ \begin{array}{c c c} \boldsymbol {\Lambda} & \boldsymbol {\Psi} ^ {H} \mathbf {B} _ {c c} & \boldsymbol {\Psi} ^ {H} \mathbf {B} _ {c 1} \\ \mathbf {0} & 0 & 1 \\ \mathbf {0} & 0 & 0 \end{array} \right], \quad \mathbf {B} _ {w} = \left[ \begin{array}{c} \boldsymbol {\Psi} ^ {H} \mathbf {B} _ {g} \\ 0 \\ 0 \end{array} \right], \quad \mathbf {B} _ {u} = \left[ \begin{array}{c} \boldsymbol {\Psi} ^ {H} \mathbf {B} _ {c 2} \\ 0 \\ 1 \end{array} \right] \tag {6}
$$

where $\pmb { \Lambda } = \mathrm { d i a g } ( \lambda _ { 1 } , \dots , \lambda _ { m } )$ is the diagonal matrix of retained eigenvalues, and $\mathbf { B } _ { c } , \mathbf { B } _ { c 1 } , \mathbf { B } _ { c 2 }$ are the control derivatives corresponding to flap rotation, angular velocity, and angular acceleration, respectively.
