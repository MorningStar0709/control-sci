# 6.2. Comparison

We compare our method against existing approaches. These include using TaCOS [53] to jointly optimise camera hardware and task, combined with a non-trainable averagebased AE algorithm (AverageAE [32]), as well as using the neural auto-exposure method (NeuralAE [36]) with offthe-shelf cameras, the Basler Dart DaA1280-54uc [3] and FLIR Flea3 [14] cameras, using human-designed placements from nuScenes [6]. For fairness, TaCOS is run with the same GA configuration as our method (10 candidates per generation, 35 generations). The RL-based DISeR [25] is excluded, as it was developed and demonstrated for single-frame tasks [25, 53]. In our work, each camera configuration needs to be evaluated over multiple frames to determine its performance over changing light conditions, resulting in impractically long optimisation times.
