# Remark

1. Slightly stronger results can in fact be obtained from (10.49) by taking advantage of the term $\alpha ( t ) \phi ( t ) F ( t ) \phi ( t ) \nu ^ { 2 } ( t + 1 )$ . In this case, one considers instead of (10.39) the following stopping rule:

$$
\alpha (t) = \left\{ \begin{array}{l l} 1 & [ 1 + \phi^ {T} (t) F (t) \phi (t) ] ^ {1 / 2} | v (t + 1) | > \varDelta \\ 0 & [ 1 + \phi^ {T} (t) F (t) \phi (t) ] ^ {1 / 2} | v (t + 1) | \leq \varDelta \end{array} \right.
$$

and the results of Theorem 10.2 will hold for $[ 1 + \phi ( t ) F ( t ) \phi ( t ) ] \nu ^ { 2 } ( t + 1 )$ instead of $\nu ^ { 2 } ( t + 1 )$ .

2. The fixed disturbance bound Δ, can be replaced by a time-varying bound $\varDelta ( t )$ $( | w ( t + 1 ) | < \Delta ( t ) < \infty )$ .
