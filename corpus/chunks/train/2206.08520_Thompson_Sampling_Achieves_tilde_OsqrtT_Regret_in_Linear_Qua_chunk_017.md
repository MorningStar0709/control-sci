# 4.1. Prior Work on TS-based Adaptive Control and Challenges

For the frequentist regret minimization problem given in Section 2.2, the state-of-the-art adaptive control algorithm that uses TS is Abeille and Lazaric (2018). They consider the “contractible” LQR systems, i.e. $| A _ { * } + B _ { * } K ( \Theta _ { * } ) | < 1$ , and provide $\tilde { O } ( \sqrt { T } )$ regret upper bound for scalar LQRs, i.e. $n = d = 1$ . Notice that the set of contractible systems is a small subset of the set defined in Assumption 1 and they are only equivalent for scalar systems since $\rho ( A _ { * } - B _ { * } K ( \Theta _ { * } ) ) =$ $\left| A _ { * } - B _ { * } K ( \Theta _ { * } ) \right|$ . This simplified setting allow them to reduce the regret analysis into the trade-off | ∗ −between $\begin{array} { r } { R _ { T } ^ { \mathrm { T S } } = \sum _ { t = 0 } ^ { T } \{ J ( \tilde { \Theta } _ { t } ) - J ( \Theta _ { * } ) \} } \end{array}$ and $\begin{array} { r } { R _ { T } ^ { \mathrm { g a p } } = \sum _ { t = 0 } ^ { T } \mathbb { E } [ x _ { t _ { 1 } } ^ { \intercal } ( P ( \tilde { \Theta } _ { t + 1 } ) - P ( \tilde { \Theta } _ { t } ) x _ { t + 1 } \mid \mathcal { F } _ { t } ] } \end{array}$ .

These regret terms are central in the analysis of several adaptive control algorithms. In the certainty equivalent control approaches, $R _ { T } ^ { \mathrm { T S } }$ is bounded by the quadratic scaling of model estimation error after a significantly long exploration with a known stabilizing controller (Mania et al., 2019; Simchowitz and Foster, 2020). In the optimism-based algorithms, $R _ { T } ^ { \mathrm { T S } }$ is bounded by 0 by design (Abbasi-Yadkori and Szepesv´ari, 2011; Faradonbeh et al., 2017). Similarly, in Bayesian regret setting, (Ouyang et al., 2017) assume that the underlying parameter $\Theta _ { * }$ comes from a known prior that the expected regret is computed with respect to. This true prior yields $\mathbb { E } [ R _ { T } ^ { \mathrm { T S } } ] = 0$ in certain restrictive LQRs. The conventional approach in the analysis of $R _ { T } ^ { \mathrm { g a p } }$ is to have lazy policy updates, i.e., O(log T ) policy changes, via doubling the determinant of $V _ { t }$ (Abeille and Lazaric, 2017; Lale et al., 2022) or exponentially increasing epoch durations (Faradonbeh et al., 2020; Cassel et al., 2020).
