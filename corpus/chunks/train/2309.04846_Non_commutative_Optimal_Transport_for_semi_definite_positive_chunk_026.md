$$\sup _ {V \in \mathrm{H} (\mathcal {H} _ {2})} \left\{- \operatorname{Tr} [ V \Gamma_ {2} ] - \tau_ {2} \operatorname{Tr} [ (\exp (- \frac {V}{\tau_ {2}}) - \mathbb {I} _ {d}) \sigma ] \right\} = \tau_ {2} \mathcal {E} [ \Gamma_ {2} | \sigma ],$$

and thus for Γ semi-definite positive

$$\Xi^ {*} (A ^ {*} \Gamma) = \tau_ {1} \mathcal {E} [ \Gamma_ {1} | \rho ] + \tau_ {2} \mathcal {E} [ \Gamma_ {2} | \sigma ], \tag {3.11}$$

and combining (3.10) with (3.11) yields $\mathcal { F } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } ( \Gamma ) = \Theta ^ { * } ( - \Gamma ) + \Xi ^ { * } ( A ^ { * } \Gamma )$ .

Finally, see that for $\Gamma \in \mathrm { H } _ { \geq } ( \mathcal { \mathcal { H } } _ { 1 } \otimes \mathcal { H } _ { 2 } ) , U \in \mathrm { H } ( \mathcal { H } _ { 1 } ) , V \in \mathrm { H } ( \mathcal { H } _ { 2 } )$ and choosing $Z = A ( U , V ) =$

−U ⊕ V , we get

$$\mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\Gamma) \geq \mathrm{Tr} [ (- U \oplus V) (- \Gamma) ] - \Theta (- U \oplus V) + \mathrm{Tr} [ (- U \oplus V) \Gamma ] - \Xi (U, V) = \mathcal {D} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (U, V),$$

and taking respective infimum and supremum, we conclude the proof.

Remark 3.1. One can notice that in the proof above we show the one-sided inequality related to the Fenchel-Rockafellar Theorem, restricted to the real subspace of bounded operators in finite dimensions

$$\inf _ {(U, V) \in \mathrm{H} (\mathcal {H} _ {1}) \times \mathrm{H} (\mathcal {H} _ {2})} \{\Theta (A (U, V)) + \Xi (U, V) \} \geq \sup _ {\Gamma \in \mathrm{H} (\mathcal {H} _ {1} \otimes \mathcal {H} _ {2})} \{- \Theta^ {*} (- \Gamma) - \Xi^ {*} (A ^ {*} \Gamma) \}.$$

Remark 3.2. Notice that by the properties of the Legendre-Fenchel transform, it follows that both $\Theta ^ { * }$ and $\Xi ^ { * }$ are convex and lower semicontinuous and, as a consequence of (3.10) and (3.11), so are the von Neumann entropy, the relative entropy, and the primal functional $\mathcal { F } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } }$ .
