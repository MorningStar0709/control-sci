# B Bounding estimation error (expanded)

How can we bound the learned perception module error $\boldsymbol { \epsilon } ( \boldsymbol { x } , \theta ) \dot { = } \| \hat { h } ^ { - 1 } \big ( h ( \boldsymbol { x } , \theta ) , \theta \big ) -$ $C _ { r } x \|$ over $D _ { r } \times D _ { \theta } ?$ We describe three options, each with their own strengths/drawbacks. The first and simplest option is to estimate a uniform bound $\bar { \epsilon } _ { 1 }$ on the error:

$$\epsilon (x, \theta) \leq \bar {\epsilon} _ {1}, \quad \forall (x, \theta) \in D _ {r} \times D _ {\theta}. \tag {B.1}$$

This works well if the error is consistent across $D _ { r } \times D _ { \theta } ;$ however, it will be conservative if there are any error spikes. A second option is to derive a spatially-varying bound on $\epsilon ( x , \theta )$ . To do so, we first estimate the Lipschitz constant of $\epsilon , L _ { p }$ , over $D _ { r } \times D _ { \theta } ;$

$$\| \epsilon (\tilde {x}, \tilde {\theta}) - \epsilon (\check {x}, \check {\theta}) \| \leq L _ {p} \| (\tilde {x}, \tilde {\theta}) - (\check {x}, \check {\theta}) \|, \quad \forall (\tilde {x}, \tilde {\theta}), (\check {x}, \check {\theta}) \in D _ {r} \times D _ {\theta}. \tag {B.2}$$

Denote $\mathcal { S } _ { x \theta } = \{ ( x _ { i } , \theta _ { i } ) \} _ { i = 1 } ^ { N _ { \mathrm { d a t a } } }$ . We can then write this bound $\bar { \epsilon } _ { 2 } ( x ^ { * } , \theta )$ (cf. Fig. 4.B for visuals), which crucially is an explicit function of the plan $x ^ { * }$ , and can thus directly guide our planner Alg. 1:

Theorem B.1 $( \bar { \epsilon } _ { 2 } ( x ^ { * } , \theta ) )$ . : Recall that $d _ { c } ( t ) = d _ { c } ( x ^ { * } ( t ) , x ( t ) )$ is the Riemannian distance between the nominal and true state at some time t. An upper bound on the perception error $\epsilon ( x , \theta )$ that scales linearly with $d _ { c } ( t )$ can be written as:

$$\epsilon (x, \theta) \leq L _ {p} d _ {c} / \sqrt {\underline {{\lambda}} _ {D _ {c}} (M _ {c})} + \min _ {1 \leq i \leq N _ {d a t a}} \{L _ {p} (\| (x ^ {*}, \theta) - (x _ {i}, \theta_ {i}) \|) + \epsilon_ {i} \} \doteq \bar {\epsilon} _ {2} (x ^ {*}, \theta) \tag {B.3}$$
