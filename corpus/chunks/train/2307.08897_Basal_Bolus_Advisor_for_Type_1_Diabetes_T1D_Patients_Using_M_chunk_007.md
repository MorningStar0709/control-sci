# 2. Materials and Methods

This section describes the development of a framework for MDI therapy. It involves integrating a multi-agent RL model into a metabolic model and training it with real-time data. The pretrained multi-agent RL model is then utilized to create an automated closed-loop insulin administration system for MDI therapy. Additionally, this section covers the generation of the training dataset as well as the design of the reward function, for building the RL model.
