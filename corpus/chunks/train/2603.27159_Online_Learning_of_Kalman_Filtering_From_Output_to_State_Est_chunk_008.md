# 2 Problem Formulation and Preliminaries

Consider a discrete-time linear time-invariant (LTI) system given by

$$
\begin{array}{l} x _ {t + 1} = A x _ {t} + w _ {t} \tag {1} \\ y _ {t} = C x _ {t} + v _ {t}, \\ \end{array}
$$

where $\boldsymbol { x } _ { t } \in \mathbb { R } ^ { n }$ and $y _ { t } \in \mathbb { R } ^ { p }$ are the state and output of the system, respectively, and $A \in \mathbb { R } ^ { n \times n }$ and $C \in \mathbb { R } ^ { p \times n }$ are the system matrices. The noise $w _ { t } \stackrel { i . i . d . } { \sim } \mathcal { N } ( 0 , W )$ and $v _ { t } \stackrel { i . i . d . } { \sim } \mathcal { N } ( 0 , V )$ with $W , V \succeq 0$ are the process noise and measurement noise, respectively, which are assumed to be independent. For simplicity, we assume that the initial state satisfies $x _ { 0 } = 0$ . We also make the following assumption throughout the paper.

Assumption 2.1. The pair $( A , W ^ { 1 / 2 } )$ is stabilizable and the pair $( A , C )$ is detectable.

Kalman filter for state estimation. Let $\hat { x } _ { t } ^ { \mathrm { K F } }$ be the estimator of the state $x _ { t }$ given by the Kalman filter. The standard form of the Kalman filter is given by the following recursion [4]:

$$\hat {x} _ {t + 1} ^ {\mathrm{KF}} = (A - L _ {t} C) \hat {x} _ {t} ^ {\mathrm{KF}} + L _ {t} y _ {t} \tag {2}$$

initialized with $\hat { x } _ { 0 } ^ { \mathtt { K F } } = 0$ . The matrix $L _ { t } \in \mathbb { R } ^ { n \times p }$ is known as the Kalman gain given by

$$L _ {t} = A \Sigma_ {t} C ^ {\top} (C \Sigma_ {t} C ^ {\top} + V) ^ {- 1}, \tag {3}$$

and $\Sigma _ { t }$ is given by the following recursion initialized with $\Sigma _ { 0 } = 0 \Sigma$

$$\Sigma_ {t} = A \Sigma_ {t} A ^ {\top} - A \Sigma_ {t - 1} C ^ {\top} (C \Sigma_ {t - 1} C ^ {\top} + V) ^ {- 1} C \Sigma_ {t - 1} A ^ {\top} + W. \tag {4}$$

It is well-known (see, e.g. [4]) that for any $t \geq 0$ , the Kalman filter $\hat { x } _ { t } ^ { \mathrm { K F } }$ achieves the minimum mean square estimation error (MSEE), i.e.,

$$\hat {x} _ {t} ^ {\mathrm{KF}} \in \underset {\hat {x} _ {t} \in \sigma (y _ {0}, \dots , y _ {t - 1})} {\arg \min} \mathbb {E} \left[ \| x _ {t} - \hat {x} _ {t} \| ^ {2} \right]. \tag {5}$$
