# IV. EXPLICIT SOLUTION OF THE PQP

We will now solve explicitly the minimization of f in (11) subject to the constraints (12), i.e., the following $\mathrm { p Q P } ,$ which is to be solved by the MPC controller at every time step:

$$\operatorname{argmin} _ {\mathbf {z} \in U ^ {N}} \frac {1}{2} \langle H \mathbf {z}, \mathbf {z} \rangle \quad \text { s.t. } \quad W + S \theta_ {n} - G \mathbf {z} \geq 0, \tag {13}$$

where $W \in \mathbb { R } ^ { \widetilde { p } }$ and $H , S , G$ are all bounded operators.

We start with a simple result on existence and uniqueness, which is contained in [25, Prop. 2.3.3, Thm 3.3.4].

Lemma 3. Assume that the optimization problem (13) is feasible, $i . e . ,$ , that the feasible set $\mathcal { S } ( \boldsymbol { \theta } _ { n } )$ , which consists of all z satisfying the constraint in (13), is nonempty. Moreover, assume that the operators G, H are bounded and that H is coercive. Then (13) has a unique solution $\mathbf { z } _ { \ast }$ .

We next work towards a Hilbert-space extension of the KKT optimality conditions used in the explicit MPC literature. The first step is taken in the following lemma, whose second part is also contained in [25, Thm 9.6.1].

$$f ^ {\prime} (z _ {*}) = \sum_ {k = 1} ^ {\widetilde {p}} \lambda_ {k} f _ {k} ^ {\prime} (z _ {*}).$$
