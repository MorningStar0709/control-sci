Next we show that the MHE cost function converges to zero as $T \to \infty$ for all $x _ { 0 }$ and converging disturbances $( \boldsymbol { w } ( i ) , \nu ( i ) )$ . Since $( \boldsymbol { w } \left( \boldsymbol { i } \right) , \boldsymbol { \nu } \left( \boldsymbol { i } \right) )$ converges to zero, (4.15) implies that $\hat { V } _ { T }$ converges to zero as $T \to \infty$ . The optimal cost at $T , \hat { V } _ { T } ^ { 0 }$ , is bounded above by $\hat { V } _ { T }$ so $\hat { V } _ { T } ^ { 0 }$ also converges to zero. The optimal cost is

$$\hat {V} _ {T} ^ {0} = \sum_ {i = T - N} ^ {T - 1} \ell_ {i} (\hat {w} (i | T), y (i) - h (\hat {x} (i | T)))$$

in which $( \hat { x } ( i | T ) , \hat { w } ( i | T ) )$ are the optimal decisions for $T - N \leq i \leq$ $T - 1 , T \geq N$ . Since $\hat { V } _ { T } ^ { 0 }$ converges to zero, we have

$$y (i) - h (\hat {x} (i | T)) \rightarrow 0 \quad \hat {w} (i | T) \rightarrow 0$$

as $T \to \infty$ . Since $y = h ( x ) + \nu$ and $\nu ( i )$ converges to zero, and $\boldsymbol { w } \left( i \right)$ converges to zero, we also have

$$h (x (i)) - h (\hat {x} (i | T)) \rightarrow 0 \quad w (i) - \hat {w} (i | T) \rightarrow 0 \tag {4.16}$$

for $T - N \leq i \leq T - 1 , T \geq N .$
