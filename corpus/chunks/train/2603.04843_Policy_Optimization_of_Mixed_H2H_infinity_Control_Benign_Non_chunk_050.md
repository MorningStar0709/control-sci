# A.1 Proof of the cost equivalence in (11)

Let $\tilde { S } _ { K } : = Q { + } K ^ { \mathsf { T } } R K$ . By Lemma 1, the Riccati equations

$$A _ {K} X _ {K} + X _ {K} A _ {K} ^ {\mathsf {T}} + \beta^ {- 2} X _ {K} \tilde {S} _ {K} X _ {K} + W = 0, \tag {A.1}A _ {K} ^ {\mathsf {T}} P _ {K} + P _ {K} A _ {K} + \beta^ {- 2} P _ {K} W P _ {K} + \tilde {S} _ {K} = 0 \tag {A.2}$$

admit unique stabilizing solutions $X _ { K }$ and $P _ { K }$ respectively for any $K \in \mathcal { K } _ { \beta }$ . To verify the cost equivalence in (11a), define $\tilde { X } _ { K } : = - \beta ^ { - 2 } X _ { K }$ . Substituting into (A.1) gives

$$A _ {K} \tilde {X} _ {K} + \tilde {X} _ {K} A _ {K} ^ {\mathsf {T}} - \tilde {X} _ {K} \tilde {S} _ {K} \tilde {X} _ {K} - \beta^ {- 2} W = 0, \quad (\mathrm{A.3})$$

where the associated closed-loop matrix is $A _ { K } - \tilde { X } _ { K } \tilde { S } _ { K } =$ $A _ { K } + \beta ^ { - 2 } X _ { K } \tilde { S } _ { K }$ , which is Hurwitz by assumption. Thus, $\tilde { X } _ { K }$ is the stabilizing solution to (A.3). Now consider the Hamiltonian matrices associated with (A.2) and (A.3):

$$
H (K) = \left[ \begin{array}{c c} A _ {K} & \beta^ {- 2} W \\ - \tilde {S} _ {K} & - A _ {K} ^ {\mathsf {T}} \end{array} \right],   \tilde {H} (K) = \left[ \begin{array}{c c} A _ {K} ^ {\mathsf {T}} & - \tilde {S} _ {K} \\ \beta^ {- 2} W & - A _ {K} \end{array} \right].
$$

Since the two Hamiltonians have identical eigenvalues (as $H ( K ) = \tilde { H } ( K ) ^ { \top } )$ , and their associated closed-loop matrices $A _ { K } + \beta ^ { - 2 } W P _ { K }$ and $A _ { K } - \tilde { X } _ { K } \tilde { S } _ { K }$ are both Hurwitz, they must share the same stable eigenvalues. Therefore, we have

$$\mathrm{tr} (A _ {K} + \beta^ {- 2} W P _ {K}) = \mathrm{tr} (A _ {K} + \beta^ {- 2} X _ {K} \tilde {S} _ {K})$$

which implies $\mathrm { t r } ( W P _ { K } ) = \mathrm { t r } ( X _ { K } \tilde { S } _ { K } )$ .
