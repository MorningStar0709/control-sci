# Algorithm 1: FRF and transient estimation.

Data: Input-output spectrum $( \hat { U } _ { [ 0 , M - 1 ] } , \hat { Y } _ { [ 0 , M - 1 ] } )$ , frequencies $\hat { \omega } _ { [ 0 , M - 1 ] } ^ { M }$ and model order $n _ { x } .$ .

Input: $U _ { z } \in \mathbb { C } ^ { n _ { u } }$ and $z \in \mathbb { C } .$

1 Let $L _ { 0 } = n _ { x }$   
2 Compute singular value decomposition

$$
\left[ \begin{array}{c} \Psi_ {L _ {0} + 1} (\hat {U} _ {[ 0, M - 1 ]}) \\ \Psi_ {L _ {0} + 1} (\hat {\Omega} _ {[ 0, M - 1 ]}) \\ \hline \Psi_ {L _ {0} + 1} (\hat {Y} _ {[ 0, M - 1 ]}) \end{array} \right] = \left[ \begin{array}{c c} \mathcal {U} _ {1} & \mathcal {U} _ {2} \end{array} \right] \mathcal {S V} ^ {\mathrm{H}},
$$

with $\mathcal { U } _ { 1 } \in \mathbb { R } ^ { ( L _ { 0 } + 1 ) ( n _ { u } + 1 + n _ { y } ) \times ( n _ { u } + 1 ) ( L _ { 0 } + 1 ) + n _ { x } }$ .

3 Solve the system

$$
\begin{array}{l} \left[ \left[ \begin{array}{c} 0 \\ - W _ {L _ {0} + 1} (z) \otimes I _ {n _ {y}} \end{array} \right] \mid \mathcal {U} _ {1} \right] \left[ \begin{array}{c c} Y _ {z} & T _ {z} \\ \hdashline G _ {Y} & G _ {T} \end{array} \right] \\ = \left[ \begin{array}{c c} W _ {L _ {0} + 1} (z) \otimes U _ {z} & 0 \\ 0 & W _ {L _ {0} + 1} (z) \otimes z \\ \hdashline 0 & 0 \end{array} \right]. \\ \end{array}
$$

Output: $Y _ { z } = H ( z ) U _ { z }$ and $T _ { z } = T ( z )$

such that $\mathcal { U } ^ { \top } \mathcal { U } = I , \mathcal { V } = \left[ \mathcal { V } _ { 0 } \quad \mathcal { V } _ { 1 } \quad \mathcal { V } _ { 1 } ^ { * } \right] ^ { \mathsf { H } } , \mathcal { V } _ { 0 } \in \mathbb { R } ^ { m \times m _ { 0 } }$ $\mathcal { V } _ { 1 } \in \mathbb { C } ^ { m \times m _ { 1 } }$ , such that $\mathcal { V } ^ { \mathsf { H } } \bar { \mathcal { V } } = I ,$ and

$$
\mathcal {S} = \left[ \begin{array}{c c} \mathcal {S} _ {1} & 0 _ {r \times (m - r)} \\ 0 _ {(n - r) \times r} & 0 _ {(n - r) \times (m - r)} \end{array} \right] \in \mathbb {R} ^ {(n - r) \times (m - r)}, \tag {15}
$$

where r = rank A and $S _ { 1 }$ is diagonal and positive definite.
