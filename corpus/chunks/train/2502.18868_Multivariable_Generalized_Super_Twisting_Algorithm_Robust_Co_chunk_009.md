# 3.1 Lyapunov function candidates

The linear subsystem $S _ { L }$ as well as the nonlinear subsystem $\mathcal { S } _ { N }$ are analysed from the viewpoint of two different Lyapunov functions. The difficulty of obtained, from them, a Lyapunov function candidate to their feedback connection is investigated and discussed afterwards. For the moment, let us first introduce the Lyapunov function candidates, namely

$$v (\zeta) = \zeta^ {T} S \zeta \tag {14}$$

with $S \in \mathbb { R } ^ { r \times r }$ being a symmetric, positive definite matrix associated to $S _ { L }$ and

$$V (x) = x ^ {T} R (\sigma) P R (\sigma) x \tag {15}$$

with $P \in \mathbb { R } ^ { 2 n \times 2 n }$ being a symmetric, positive definite matrix, associated to $\mathcal { S } _ { N }$ . It is readily seen that there exist positive scalars $\varepsilon _ { L }$ and $\theta _ { L }$ such that $\varepsilon _ { L } \| \zeta \| ^ { 2 } \leq v ( \zeta ) \leq$ $\theta _ { L } \| \zeta \| ^ { 2 }$ , for all $\zeta \in \mathbb { R } ^ { r }$ . Likewise, there exist positive scalars $\varepsilon _ { N }$ and $\theta _ { N }$ such that $\varepsilon _ { N } \| R ( \sigma ) x \| ^ { 2 } \leq \mathrm { \hat { V } } ( x ) \leq$ $\theta _ { N } \lVert R ( \sigma ) x \rVert ^ { 2 }$ , for all $x \in \mathbb { R } ^ { n }$ . Together, these bounds im-$\mathrm { p l y }$ that both functions are radially unbounded.
