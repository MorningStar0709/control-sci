$$J (u ^ {\alpha}, z) = \pi \left(\nu^ {2} + m ^ {2} + \frac {\sigma^ {2}}{\rho}\right) - 2 m ^ {2} \sum_ {l = 1} ^ {L} f _ {\ell} (\alpha) \bar {\lambda} _ {\ell} \langle 1, f _ {\ell} \rangle- m ^ {2} \sum_ {k = 1} ^ {L} \tilde {\lambda} _ {k} ^ {\alpha} f _ {k} (\alpha) \langle 1, f _ {k} \rangle .$$

Interpreting the quantities $\bar { \lambda } _ { k } , \ \tilde { \lambda } _ { k } ^ { \alpha }$ , for all $\alpha \in [ 0 , 1 ]$ as eigenvalues, we deduce that the cost can be written as a function of newly introduced graphons $\bar { g } ( \cdot , \cdot )$ and $\tilde { g } ( \cdot , \cdot )$ built from the original graphon $g ( \cdot , \cdot )$ . ✷

Assumption (A4) Assume that the orthonormal eigenfunctions $\{ f _ { \ell } \} _ { \ell = 1 } ^ { L }$ of $g$ satisfy the condition

$$\langle 1, f _ {\ell} \rangle = \int_ {0} ^ {1} f _ {\ell} (\beta) d \beta < 0, \quad \forall \ell \in \{1,..., L \}. \tag {47}$$

Given any orthonormal eigenfunction set $\{ f _ { \ell } \} _ { \ell = 1 } ^ { L }$ of graphon $^ { g , }$ we can construct $\{ \tilde { f } _ { \ell } \} _ { \ell = 1 } ^ { L }$ as follows:

$$
\tilde {f} _ {\ell} = \left\{ \begin{array}{l l} f _ {\ell}, & \text {if} \langle 1, f _ {\ell} \rangle <   0, \\ - f _ {\ell}, & \text {if} \langle 1, f _ {\ell} \rangle \geq 0, \end{array} \right. \quad \forall \ell \in \{1,..., L \}.
$$

Then $\{ \tilde { f } _ { \ell } \} _ { \ell = 1 } ^ { L }$ is also an orthonormal eigenfunction set of the same graphon g and it satisfies $\langle 1 , \tilde { f } _ { \ell } \rangle \le 0$ for any $\ell \in \{ 1 , . . . , \overset { \vartriangle } L \}$ . Thus, if all the eigenfunctions (associated with nonzero eigenvalues) of g are not orthogonal to $\mathrm { ~ 1 ~ } \in \ L ^ { 2 } ( [ 0 , 1 ] )$ , there is always a choice of orthonormal eigenfunctions of g that satisfies (A4); in other words, the only restriction that (A4) poses on the graphon g is that all its eigenfunctions (associated with nonzero eigenvalues) should not be orthogonal to $1 \in L ^ { 2 } ( [ 0 , 1 ] )$ .

Assumption (A5) Assume that the eigenvalues $\lambda _ { \ell }$ of $g _ { \colon }$ satisfy, for all $\dot { \ell } , k \in \{ 1 , . . . , L \}$ ,

$$\theta (\lambda_ {\ell}) + \theta (\lambda_ {k}) = \frac {\rho}{2}. \tag {48}$$

Given a graphon g and some node $\alpha \in [ 0 , 1 ]$ , the degree of node $\alpha \in [ 0 , 1 ]$ , is defined by

$$\delta (\alpha) := \int_ {0} ^ {1} g (\alpha , \beta) d \beta = \sum_ {\ell = 1} ^ {L} \lambda_ {\ell} \langle 1, f _ {\ell} \rangle f _ {\ell} (\alpha). \tag {49}$$

Proposition 8. Let the assumptions (A1) to (A5) be in force. Assume that the eigenfunctions are twice differentiable and there exists a node $\alpha ^ { * } \in ( 0 , 1 )$ such that

$$\partial_ {\alpha , \alpha} ^ {2} f _ {\ell} (\alpha^ {*}) > 0, \quad \forall \ell \in \{1,..., L \}. \tag {50}$$

Then, the node $\alpha ^ { * } \in \mathsf { \Gamma } ( 0 , 1 )$ is a node with strict local maximal degree if and only if it is a node with strict local minimal cost.
