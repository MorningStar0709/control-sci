# 2 Existence

First, we collect some notation used in this paper. Denote by $\mathcal { P } ( \mathcal { X } )$ the set of all Borel probability measures ν on a Polish space X . We write $a ^ { \mathsf { T } }$ for the transpose of a vector or matrix a. For Euclidean spaces X and Y, denote by $C ( \mathcal X , \mathcal Y )$ the space of all continuous functions $\varphi : \mathcal { X }  \mathcal { Y }$ . In particular, we denote $C ( \mathcal { X } ) = C ( \mathcal { X } , \mathbb { R } )$ . We also denote by $C _ { b } ( \mathcal { X } )$ the space of bounded continuous functions on X . Given a Polish space X , for any $\pi \in { \mathcal { P } } ( { \mathcal { X } } )$ and any Borel measurable function $\varphi : \mathcal { X }  \mathbb { R }$ that is integrable with respect to π, we put

$$\langle \varphi , \pi \rangle := \int_ {\mathcal {X}} \varphi (x) \pi (d x).$$

Let $\{ B _ { t } \} _ { 0 \leqslant t \leqslant T }$ be an m-dimensional standard Brownian motion on a complete probability space $( \Omega , \mathcal { F } , \mathbb { P } )$ . Further, let $X _ { 0 }$ be a random variable that is independent of $\{ B _ { t } \} _ { 0 \leqslant t \leqslant T }$ such that

$$\mathbb {E} | X _ {0} | ^ {2} < \infty .$$

Denote by $\{ \mathcal { F } _ { t } \} _ { 0 \leqslant t \leqslant T }$ the augmented filtration generated by $\{ B _ { t } \} _ { 0 \leqslant t \leqslant T }$ and $X _ { 0 }$ . For simplicity, we assume $U \ni 0$ . The class $\mathcal { U }$ of controls is then defined by the set of all U-valued $\{ \mathcal { F } _ { t } \}$ -adapted processes $\{ u _ { t } \} _ { 0 \leqslant t \leqslant T }$ satisfying

$$\mathbb {E} \int_ {0} ^ {T} | u _ {t} | ^ {2} d t < \infty .$$

To discuss the uniqueness, we impose the following:

(A1) There exists a constant $C _ { 0 } > 0$ and a modulus of continuity $\rho : [ 0 , \infty )  [ 0 , \infty )$ such that for $\varphi ( t , x , u ) = b ( t , x , u ) , \sigma ( t , x , u )$ we have

$$| \varphi (t, x, u) - \varphi (t, x ^ {\prime}, u ^ {\prime}) | \leqslant C _ {0} | x - x ^ {\prime} | + \rho (| u - u ^ {\prime} |),| \varphi (t, 0, 0) | \leqslant C _ {0},$$

for $t \in [ 0 , T ] , x , x ^ { \prime } \in \mathbb { R } ^ { d }$ , and $u , u ^ { \prime } \in U$ .

Under (A1), for any given $u \in \mathcal { U }$ , there exists a unique solution $\{ X _ { t } ^ { u } \} _ { 0 \leqslant t \leqslant T }$ of the controlled stochastic differential equation (1.2) with initial condition $X _ { 0 }$ such that

$$\mathbb {E} \left[ \sup _ {0 \leqslant t \leqslant T} | X _ {t} ^ {u} | ^ {2} \right] < \infty . \tag {2.1}$$
