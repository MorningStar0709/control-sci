# III. PROBLEM STATEMENT

We assume that the system (1) is detectable in a sense defined in Section II-C, which is necessary for the existence of $\phi , \psi$ such that an asymptotic observer (6) converges, i.e.,

$$\lim _ {t \to \infty} \| x (t; x _ {0}) - \hat {x} (t; \psi (z _ {0}, C x _ {0})) \| = 0.$$

Note that if (1) is observable, differentially observable, or differentially detectable, then it is also detectable. Thus, detectability is the least restrictive assumption to ensure the wellposedness of the observation problem.

Secondly, we assume that the pair $( A , C )$ is a detectable pair. That is, if there exist $v \in \mathbb { R } ^ { n _ { x } }$ and $\lambda \in \mathbb { C }$ such that $A v = \lambda v$ and $C v = 0$ , then the real part $\mathrm { R e } ( \lambda ) < 0$ .

Under the above assumptions, we aim to design an observer of the form (6) that asymptotically estimates the state of (1), i.e., $\begin{array} { r } { \operatorname* { l i m } _ { t  \infty } \Vert x ( t ) - \hat { x } ( t ) \Vert = 0 } \end{array}$ , given the model $A , G , H , C$ and the output measurements y(t).
