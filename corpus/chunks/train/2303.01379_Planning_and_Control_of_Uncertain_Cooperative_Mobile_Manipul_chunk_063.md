Hence, the term $\eta _ { o }$ can be transformed to the unit quaternion $\zeta _ { o } = [ \varphi _ { o } , \epsilon _ { o } ^ { \top } ] ^ { \top } \in \mathbb { S } ^ { 3 }$ where $\varphi _ { o } \in [ - 1 , 1 ]$ and $\epsilon _ { o } \in \mathbb { R } ^ { 3 }$ are the scalar and vector parts, respectively [85].
