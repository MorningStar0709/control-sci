# B. Rotational Dynamic

The rotational motion of quadrotors is defined by a highly nonlinear set of equations which are challenging in terms of control. $\mathcal { T } = [ \tau _ { 1 } , \bar { \tau _ { 2 } } , \tau _ { 3 } ] ^ { \top } \in \mathbb { R } ^ { 3 }$ denotes torque as an attitude control input. Desired trajectories are produced by a total thrust ℑ. The rotational dynamics are as follows [1]:

$$
\text { Rotation } \left\{ \begin{array}{l l} \dot {R} _ {Q} & = - [ \omega ] _ {\times} R _ {Q} \\ J _ {m} \dot {\omega} & = [ J _ {m} \omega ] _ {\times} \omega + \mathcal {T}, \end{array} \right. \tag {4}
$$

where $\omega \in \mathbb { R } ^ { 3 }$ is angular velocity and $R _ { Q } ~ \in ~ S O ( 3 )$ . Let $R _ { Q } , \omega , \mathcal { T } \in \{ B \}$ , and $J _ { m } = [ J _ { i j } ] _ { 3 \times 3 } \in \mathbb { R } ^ { 5 \times 3 }$ is the inertia matrix which is diagonal. By revisiting (4) and substituting (2), the UAV rotational motion is rewritten as [7], [25]:

$$
\text { Rotation } \left\{ \begin{array}{l l} \dot {q} _ {0} & = - \frac {1}{2} \boldsymbol {q} ^ {\top} \omega \\ \dot {\boldsymbol {q}} & = \frac {1}{2} (q _ {0} \omega + [ \boldsymbol {q} ] _ {\times} \omega) \\ J _ {m} \dot {\omega} & = [ J _ {m} \omega ] _ {\times} \omega + \mathcal {T} \end{array} \right. \tag {5}
$$

where $Q = [ q _ { 0 } , q _ { 1 } , q _ { 2 } , q _ { 3 } ] ^ { \top } = [ q _ { 0 } , \pmb { q } ^ { \top } ] ^ { \top } \in \mathbb { R } ^ { 4 }$ . In the light of (4), and subsitituting q, $J _ { m } = d i a g ( J _ { 1 1 } , J _ { 2 2 } , J _ { 3 3 } ) \in \mathbb { R } ^ { 3 \times 3 }$ , the state space expanded attitude model is expressed as:

$$
\text { Rotation } \left\{ \begin{array}{l l} \dot {\omega} _ {1} & = J _ {1 1} ^ {- 1} ((J _ {2 2} - J _ {3 3})   \omega_ {2} \omega_ {3} + \tau_ {1}) \\ \dot {\omega} _ {2} & = J _ {2 2} ^ {- 1} ((J _ {3 3} - J _ {1 1})   \omega_ {3} \omega_ {1} + \tau_ {2}) \\ \dot {\omega} _ {3} & = J _ {3 3} ^ {- 1} ((J _ {1 1} - J _ {2 2})   \omega_ {1} \omega_ {2} + \tau_ {3}) \\ \dot {q} _ {0} & = - \frac {1}{2} (q _ {1} \omega_ {1} + q _ {2} \omega_ {2} + q _ {3} \omega_ {3}) \\ \dot {q} _ {1} & = \frac {1}{2} (q _ {0} \omega_ {1} - q _ {3} \omega_ {2} + q _ {2} \omega_ {3}) \\ \dot {q} _ {2} & = \frac {1}{2} (q _ {3} \omega_ {1} + q _ {0} \omega_ {2} - q _ {1} \omega_ {3}) \\ \dot {q} _ {3} & = \frac {1}{2} (- q _ {2} \omega_ {1} + q _ {1} \omega_ {2} + q _ {0} \omega_ {3}) \end{array} \right. \tag {6}
$$
