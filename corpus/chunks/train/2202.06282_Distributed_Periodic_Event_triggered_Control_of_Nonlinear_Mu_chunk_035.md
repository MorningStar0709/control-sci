# 6. CASE STUDY

We apply the results to the single-integrator consensus problem, where we have a multi-agent system with $N \in$ $\stackrel { \star } { \mathbb { N } } _ { > 0 }$ agents. All agents have state $\mathbf { \bar { \Psi } } _ { x _ { i } } \in \bar { \mathbb { R } } , i \in \mathcal { N }$ , whose dynamics evolve according to ${ \dot { x } } _ { i } ~ = ~ u _ { i }$ with $u _ { i } \in \mathbb { R }$ the control input. The output of the system is the state $x _ { i } , \mathrm { i . e . }$ , $y _ { i } = x _ { i }$ . We assume that the graph $\mathcal G ( \nu , \mathcal { E } )$ with Laplacian matrix L is connected and undirected, i.e., $L ^ { \top } ~ = ~ L$ . The control objective is for the states of all agents to asymptotically converge, i.e., $\begin{array} { r } { \operatorname* { l i m } _ { t  \infty } | x _ { i } ( t ) - x _ { m } \mathbf { \check { \eta } } ( t ) | = 0 } \end{array}$ for all $i , m \in \mathcal { N }$ . To achieve consensus, we implement the control law

$$u _ {i} = - \sum_ {m \in \mathcal {V} _ {i} ^ {\mathrm{in}}} (x _ {i} + e _ {i} ^ {i} - x _ {m} - e _ {m} ^ {i}). \tag {34}$$

We consider the Lyapunov candidate $V ( x ) = x ^ { \top } L x$ where $x : = ( x _ { 1 } , x _ { 2 } , . . . , \bar { x _ { N } } )$ . According to Dolk et al. (2019), the derivative of this Lyapunov function can be upperbounded as $\begin{array} { r } { \langle \nabla V ( x ) , - \dot { L } \dot { x } - L e \rangle \leqslant \sum _ { i \in \mathcal { N } } \big ( - d _ { i } z _ { i } ^ { 2 } - \dot { c _ { i } } \dot { u } _ { i } ^ { 2 } + } \end{array}$ $( \gamma _ { i } ^ { 2 } - \alpha _ { i } ) | e _ { i } ^ { i } | ^ { 2 } )$ with $d _ { i } : = \delta ( 1 - a N _ { i } ) , c _ { i } : = ( 1 - \delta ) ( 1 - a N _ { i } )$ and $\gamma _ { i } = \sqrt { a ^ { - 1 } N _ { i } + \alpha _ { i } }$ , and where $\delta \in ( 0 , 1 ) , a \in ( 0 , \frac { 1 } { N _ { i } } )$ and $\alpha _ { i } ~ > ~ 0$ are tuning parameters. The theorem below shows the exact expressions for all the required conditions. Its proof is omitted for space reasons.

Theorem 2. The system with local dynamics $\dot { x } _ { i } ~ = ~ u _ { i }$ and local controller (34) satisfies Conditions $1 , \ 2$ and 4 with $\begin{array} { r } { H _ { i } = | u _ { i } | , L _ { i } = 0 , s ( x , e ) = \sum _ { i \in \mathcal { N } } \left( - d _ { i } z _ { i } ^ { 2 } - \mu _ { i } e _ { i } ^ { 2 } \right) } \end{array}$ , $\begin{array} { r } { \mathcal { X } = \{ x \in \mathbb { R } ^ { N } \mid x _ { 1 } = x _ { 2 } = . . . = x _ { N } \} , \varsigma _ { i } = 0 , \mu _ { i } = c _ { i } \frac { 1 } { N } , } \end{array}$ , $\gamma _ { i } = \sqrt { a ^ { - 1 } N _ { i } + \alpha _ { i } } ,$ and $\underline { { H } } _ { i } = | u _ { i } |$ . ✷
