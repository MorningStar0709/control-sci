# A. Simulation Setup

We consider the discrete-time system

$$
\boldsymbol {x} _ {t + 1} = \left[ \begin{array}{c c} 1 & 0. 0 0 7 5 \\ - 0. 1 4 3 & 0. 9 9 6 \end{array} \right] \boldsymbol {x} _ {t} + \left[ \begin{array}{l} 4. 7 9 8 \\ 0. 1 1 5 \end{array} \right] u _ {t} + \left[ \begin{array}{l l} 1 & 0 \\ 0 & 1 \end{array} \right] \boldsymbol {w} _ {t} \tag {13}
$$

with ${ \pmb x } = ( x _ { 1 } , x _ { 2 } ) ^ { \top }$ and the normally distributed uncertainty ${ \pmb w } \sim \mathcal { N } ( { \pmb 0 } , { \pmb \Sigma } _ { \pmb w } ) , { \pmb w } _ { t } \in \mathcal { W }$ where $\Sigma _ { w } = \mathrm { d i a g } ( 0 . 0 6 , 0 . 0 6 )$ and ${ \mathcal { W } } = \{ { \pmb w } _ { t } \ | \ | { \pmb w } _ { t } | | _ { \infty } \leq 0 . 0 7 \}$ . The input is bounded by $| u _ { t } | \le 0 . 2$ and we employ the state constraint $x _ { 1 } \ \leq \ 2 . 8$ . Additionally, we define $| x _ { 1 } | \le 1 0$ and $| x _ { 2 } | \le 1 0$ to obtain a bounded set X , however, in the following simulation only $x _ { 1 } \leq 2 . 8$ is regarded. The initial state is $\pmb { x } _ { 0 } = ( - 1 . 3 , 3 . 5 ) ^ { \top }$ .

For the SMPC OCP, we approximate the uncertainty with the non-truncated normal distribution $\pmb { w } \sim \mathcal { N } ( \mathbf { 0 } , \pmb { \Sigma _ { w } } )$ and split the state into a deterministic and a probabilistic part ${ \pmb x } _ { t } = { \pmb z } _ { t } + { \pmb e } _ { t }$ , yielding an adapted input $u _ { t } = K { \pmb x } _ { t } + \nu _ { t }$ with a stabilizing feedback matrix K and the new input decision variable $\nu _ { t }$ . The state constraint is considered as the chance constraint Pr $( x _ { 1 } \in 2 . 8 ) \ \geq \ \beta$ with $\beta \ : = \ : 0 . 8$ . The normal distribution w allows for the chance constraint reformulation

$$x _ {1, k} \leq 2. 8 - \gamma_ {\mathrm{cc}, k} \tag {14a}\gamma_ {\mathrm{cc}, k} = \sqrt {2 [ 1 , 0 ] ^ {\top} \boldsymbol {\Sigma} _ {k} ^ {e} [ 1 , 0 ]} \operatorname{erf} ^ {- 1} (2 \beta - 1) \tag {14b}$$

with the inverse error function $\mathrm { e r f } ^ { - 1 } ( . )$ and the error covariance matrix $\Sigma _ { k + 1 } ^ { e } = \Phi \Sigma _ { k } ^ { e } \Phi ^ { \top } + \Sigma _ { w }$ with $\Sigma _ { 0 } ^ { e } = \mathrm { d i a g } ( 0 , 0 )$ and $\Phi = A + B K$ .

For the backup MPC, we use an RMPC approach according to [28], satisfying Assumption 2. This approach yields the tightened state constraint $\overline { { x } } _ { 1 } \leq 1 . 7 2$ and tightened input constraint $- 0 . 0 1 8 \leq \overline { { u } } _ { t } \leq 0 . 0 2 5$ . The terminal constraint $\mathcal { X } _ { \mathrm { f } }$ is chosen to be a maximal robust control invariant set.
