# B. Common agent approach

We recall that in general, the leader and the followers generate their actions at time t as follows, $a _ { t } ^ { l } \sim$ $\sigma _ { t } ^ { l } ( \cdot | z _ { 1 : t } , a _ { 1 : t - 1 } ^ { l } , x _ { 1 : t } ^ { l } )$ and $a _ { t } ^ { f } \sim \sigma _ { t } ^ { f } ( \cdot | z _ { 1 : t } , a _ { 1 : t - 1 } ^ { l } , x _ { 1 : t } ^ { f } )$ . An alternative way to view the problem is as follows. As is done in the common information approach [31], at time t, a fictitious common agent observes the common information $z _ { 1 : t } , a _ { 1 : t - 1 } ^ { l }$ and generates prescription functions $\gamma _ { t } = ( \gamma _ { t } ^ { l } , \gamma _ { t } ^ { f } ) = \psi _ { t } [ z _ { 1 : t } , a _ { 1 : t - 1 } ^ { l } ]$ . Follower i uses its prescription function $\gamma _ { t } ^ { f , i }$ to operate on its private information $x _ { t } ^ { f , i }$ to produce its action $a _ { t } ^ { f , i }$ , i.e. $\gamma _ { t } ^ { f , i } : ( \mathcal { X } ^ { f , i } ) ^ { t }  \mathcal { P } ( \mathcal { A } ^ { f , i } )$ and $a _ { t } ^ { f , i } \sim \gamma _ { t } ^ { f , i } ( \cdot | x _ { 1 : t } ^ { f , i } )$ . Similarly, leader uses its prescription function $\gamma _ { t } ^ { l }$ to operate on its private information $x _ { t } ^ { l }$ to produce its action $a _ { t } ^ { l } ,$ i.e. $\gamma _ { t } ^ { l } : ( \mathcal { X } ^ { l } ) ^ { t }  \mathcal { P } ( \mathcal { A } ^ { l } )$ and $a _ { t } ^ { l } \sim \gamma _ { t } ^ { l } ( \cdot | x _ { 1 : t } ^ { l } )$ It is easy to see that for any σ policy profile of the players, there exists an equivalent $\psi$ profile of the common agent (and vice versa) that generates the same control actions for every realization of the information of the players.

Here, we will consider Markovian common agent’s policy as follows. We call a common agent’s policy be of “type $\theta "$ if the common agent observes the mean field population state $z _ { t }$ and common belief $\pi _ { t } .$ , and generates prescription functions $\gamma _ { t } = ( \gamma _ { t } ^ { l } , \gamma _ { t } ^ { f } ) = \theta _ { t } [ \pi _ { t } , z _ { t } ]$ . The follower i uses prescription function $\gamma _ { t } ^ { f , i }$ to operate on its current private type $x _ { t } ^ { f , i }$ to produce its action $a _ { t } ^ { f , i }$ , i.e. $\gamma _ { t } ^ { f , i } : \mathcal { X } ^ { f , i }  \mathcal { P } ( \mathcal { A } ^ { f , i } )$ and $a _ { t } ^ { f , i } \sim \gamma _ { t } ^ { f , i } ( \cdot | x _ { t } ^ { f , i } )$ . The leader uses prescription function $\gamma _ { t } ^ { l }$ to operate on its current private type $x _ { t } ^ { l }$ to produce its action $a _ { t } ^ { l } , \mathrm { i . e . } \ \gamma _ { t } ^ { l } : \mathcal { X } ^ { l }  \mathcal { P } ( \mathcal { A } ^ { l } )$ and $a _ { t } ^ { l } \sim \gamma _ { t } ^ { l } ( \cdot | x _ { t } ^ { l } )$ .
