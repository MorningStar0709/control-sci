# 1 Introduction

Kalman filtering is a foundational approach to estimation and prediction of time-series data corrupted by stochastic noise [1, 2]. When applied to estimate the state of partially observed linear dynamical systems with Gaussian process and/or measurement noise, the Kalman filter achieves the minimum mean square estimation error among all state estimators of the system [3, 4]. However, constructing the optimal Kalman filter requires complete and exact knowledge of the underlying system model, which is generally absent in many real-world applications [5, 6]. A natural idea is to use the so-called certainty equivalence approach: Identifying the system model from measurement data and then using the identified system model in the Kalman filter design as if it was the true model [7]. Unfortunately, for partially observed linear dynamical system, one can only identify the system model up to some similarity transformation using system data [6,8,9]. This makes the approach of learning the Kalman filter using identified system model not applicable [10].
