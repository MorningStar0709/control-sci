Remark 5.2. Limiting players to use restricted deviations (see (5.3)) is a way to get a uniform bound in the error term $\epsilon _ { N } ;$ this is used in (5.8) in the proof of Theorem 5.1. It seems possible to extend Theorem 5.1 also to the case in which players’ admissible deviations are not assumed to be bounded a priori; that is, to the case case in which $\widehat { A } _ { b } ^ { N }$ in Definition 5.1 is replaced with $\widehat { A } ^ { N }$ . In order to do so, one has to show that, when player n’s opponents choose reflection strategies $\bar { \pmb { \xi } } ^ { - n } \in \widehat { A } _ { b } ^ { N - 1 }$ of the MFG equilibrium, player n’s optimal strategy over $\widehat { A } _ { e }$ is an element of $\widehat { A } _ { b } \ ( i . e .$ ., without restricting to it ex ante). However this intermediate result comes with a certain technical cost, since one has to: (1) Repeat the analysis of Section 3 by replacing the constant parameter θ by the process $\{ \theta _ { \bar { \pm } ^ { - n } } ^ { N } ( t ) \} _ { \bar { \pm } ^ { - n } } ( t ) \} _ { \bar { \pm } ^ { } }$ t defined in $( 5 . 2 ) ; ( 2 )$ Show a monotonicity of optimal reflection boundaries with respect the stochastic parameter $\theta _ { \bar { \pmb { \xi } } ^ { - n } } ^ { N } ; ( 3 )$ Obtain bounds for the optimal reflection strategies in the spirit of (4.18) and (4.19).
