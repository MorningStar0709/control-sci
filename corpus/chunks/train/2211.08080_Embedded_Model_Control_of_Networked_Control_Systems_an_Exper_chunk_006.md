# 2.2 Plant stability proof

The plant of the DC motor, i.e. the mathematical description of the DC motor physical behaviour, must guarantee BIBO stability. Equation (3) define the state equations for the DC motor

$$
\left\{ \begin{array}{l l} & \omega (k + 1) = \left(- \frac {1}{\tau_ {m}} T _ {s} + 1\right) \omega (k) + \frac {T _ {s}}{\tau_ {m} k _ {v} ^ {\prime}} V _ {a} (k) + d (k), \\ & x _ {d 1} (k + 1) = (T _ {s} + 1) x _ {d 1} (k) + T _ {s} \left[ x _ {d 2} (k) + \overline {{w}} _ {2} (t) \right], \\ & x _ {d 2} (k + 1) = (T _ {s} + 1) x _ {d 2} (k) + T _ {s} \overline {{w}} _ {3} (k), \end{array} \right.
d (k) = T _ {s} \left[ \overline {{w}} _ {1} (k) + x _ {d 1} (k) \right],\omega (0) = \omega_ {0}, \quad x _ {d 1} (0) = x _ {d 1 0}, \quad x _ {d 2} (0) = x _ {d 2 0}, \tag {3}$$

where $\omega$ is the DC motor output speed, $\tau _ { m }$ is the DC motor mechanical time constant, $k _ { v } ^ { \prime }$ is the back electromotive force constant specifically found for the studied motor, $[ \overline { { w } } _ { 1 } , \overline { { w } } _ { 2 } , \overline { { w } } _ { 3 } ]$ are the components of the noise vector $\overline { { \mathbf { w } } } .$ , [xd1, xd2] are the components of the disturbance state $\mathbf { x } _ { d } , T _ { s }$ is the sampling time.

Equation (3) can be translated in frequency domain and reshaped in a matrix form, i.e.

$$W _ {M} (z) = \frac {Y (z)}{U (z)} = \frac {\beta_ {0} (z + 1)}{z ^ {2} + \alpha_ {1} z + \alpha_ {0}},\beta_ {0} = \frac {T ^ {2}}{k _ {v} \left(\tau_ {m} T + 2 \tau_ {m} \tau_ {a}\right)}, \tag {4}\alpha_ {0} = \frac {T ^ {2} - \tau_ {m} T + 2 \tau_ {m} \tau_ {a}}{\tau_ {m} T + 2 \tau_ {m} \tau_ {a}},\alpha_ {1} = \frac {T ^ {2} - 4 \tau_ {m} \tau_ {a}}{\tau_ {m} T + 2 \tau_ {m} \tau_ {a}}$$

The worst case for the stability corresponds to the lowest sampling time possible. For example, by considering for $T _ { s }$ a range of [0.01, 0.03] s, the lowest sampling time corresponds to $T _ { s } = 0 . 0 1 ~ s \colon$ hence the parameters become $\beta _ { 0 } = 0 . 1 3 1 3 , \alpha _ { 0 } = - 0 . 1 4 7 1 , \alpha _ { 1 } =$ −0.4835.

The motor transfer function can be alternatively expressed as

$$W _ {M} (z) = \frac {\beta_ {0} (z + 1)}{(z - d _ {0}) (z - d _ {1})} \tag {5}$$

where the two poles are $d _ { 0 } = 0 . 6 9 5 , d _ { 1 } = - 0 . 2 1 2$ , both inside the unitary circle, leading to asymptotic BIBO stability of the motor plant transfer-function.
