The interpretation of the projection is as follows. For all $x \in \mathcal { X } \backslash \{ 0 \}$ , if the nominal model $\hat { f }$ satisfies the condition (5), it needs no modification so that $f ^ { * } ( x , u ) = f ( x , u )$ for all $u \in \mathcal { U } ;$ on the other hand, if it violates the condition, we minimally adjust it (in the sense of $\ell ^ { 2 }$ norm) so that the projected model $f ^ { * }$ barely satisfies the condition (5) with the equality.
