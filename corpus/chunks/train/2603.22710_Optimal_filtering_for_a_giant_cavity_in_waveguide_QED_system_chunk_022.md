# Algorithm 1: Optimal Filter Design for the cavity coupled to the waveguide at two distant points

Input: Initial state x(0), time delay T .

Output: Stochastic state estimate ${ \hat { x } } ( t )$ , which represents the conditional expectation of the system state.

Step 1. Initialize the filter state $\hat { x } ( 0 ) ~ = ~ \hat { x } _ { 0 }$ for $t \in$ $[ - T , 0 ]$ . Set $\begin{array} { r } { m = \left\lfloor \frac { t } { T } \right\rfloor ; } \end{array}$

Step 2. A terminal condition $m = 0$ is given. If this condition is satisfied, go to Step 4; if not, go to Step 3;

Step 3. Compute the delayed state covariance matrices $\mathrm { d } P _ { m } ( t )$ using (36). Set $m = m - 1$ . Return to Step 2;

Step 4. Derive the differential equation of the covariance matrix $\mathrm { d } P ( t )$ with (28). Compute the optimal filter (22).

Algorithm 1 presents a recursive procedure for the optimal filter design of a cavity coupled to a waveguide at two distant points. The algorithm proceeds by partitioning the time axis into delay intervals of length $\mathbf { \dot { \rho } } _ { T }$ and recursively propagating the delayed cross-covariance matrices $\dot { P _ { j } } ( \boldsymbol { \bar { t } } )$ . This interval-wise backward recursive structure explicitly captures the effect of mixed delays on the estimation error dynamics.
