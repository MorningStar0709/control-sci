# E. Implementing Serial consensus

The following proposition ensures that the serial consensus system can be achieved by controlling the $n ^ { \mathrm { t h } }$ order integrator system (3) with an n-step implementable relative feedback controller as defined in Definition 2.

Proposition 1: Consider the $n ^ { \mathrm { t h } }$ -order serial consensus as defined in (6). If each $L _ { k } \in \mathcal { A } ^ { 1 } ( W , c )$ for some constant c and adjacency matrix W , then the controller in (5) is an nstep implementable relative feedback controller with respect to W and a finite gain $c ^ { \prime } .$ .

To prove this proposition we first need the following two lemmas whose proofs are provided in the appendix.

Lemma 2: If $A _ { 1 } \in \mathcal { A } ^ { q _ { 1 } } ( W , c _ { 1 } )$ and $A _ { 2 } \in \mathcal { A } ^ { q _ { 2 } } \left( W , c _ { 2 } \right)$ then the sum $( A _ { 1 } + A _ { 2 } ) \in \mathcal { A } ^ { \operatorname* { m a x } ( q _ { 1 } , q _ { 2 } ) } \ ( W , c _ { 1 } + c _ { 2 } )$

Lemma 3: Let $A _ { 1 } \in \mathcal { A } ^ { q _ { 1 } } ( W , c _ { 1 } )$ and $A _ { 2 } \in \mathcal { A } ^ { q _ { 2 } } ( W , c _ { 2 } )$ then the product $( A _ { 1 } A _ { 2 } ) \in \mathcal { A } ^ { q _ { 1 } + q _ { 2 } } ( W , c _ { 1 } c _ { 2 } )$

Now we can prove Proposition 1.

Proof: The serial consensus controller can be expanded to the matrix polynomial

$$

$$A _ {k} = \sum_ {\alpha \in \mathcal {I} _ {k}} \prod_ {j \in \alpha} L _ {j}, \text { for all } k \in [ 0, n - 1 ].$$
