Let H be a separable Hilbert space with norm $\| \cdot \| _ { \mathcal { H } }$ and inner product $\langle \cdot , \cdot \rangle _ { \mathscr { H } }$ . Assume that $U$ is a closed convex subset of $\mathcal { H } .$ , and put

$$\mathcal {U} ^ {\beta} [ 0, T ] := \left\{u: [ 0, T ] \to U | \| u \| _ {L ^ {\beta} (0, T; \mathcal {H})} < \infty \right\}.$$

This paper investigates the second order necessary condition for optimal control of the following quantum stochastic control system in noncommutative space $L ^ { 2 } ( \mathcal { C } )$ :

$$
\left\{ \begin{array}{l} d x (t) = D (t, x (t), u (t)) d t + F (t, x (t), u (t)) d W (t) + d W (t) G (t, x (t), u (t)), \quad \text { in } [ t _ {0}, T ], \\ x (t _ {0}) = x _ {0}, \end{array} \right. \tag {1.3}
$$

with the cost functional

$$\mathcal {J} (u (\cdot)) = \int_ {t _ {0}} ^ {T} L (t, x (t), u (t)) d t + g (x (T)), \quad u (\cdot) \in \mathcal {U} ^ {\beta} [ t _ {0}, T ]. \tag {1.4}$$

Here the initial condition $x _ { 0 } \in L ^ { 2 } ( \mathcal { C } _ { t _ { 0 } } )$ , the maps $D ( \cdot , \cdot , \cdot ) , F ( \cdot , \cdot , \cdot ) , G ( \cdot , \cdot , \cdot ) : [ t _ { 0 } , T ] \times L ^ { 2 } ( \mathcal { C } ) \times U \to L ^ { 2 } ( \mathcal { C } )$ are adapted, continuous operator-valued functions, $W ( t )$ is fermion Brownian motion defined by (1.1) of quantum (noncommutative) probability space $( \Lambda ( \mathcal { H } ) , \mathcal { C } , m )$ , the cost density $L ( \cdot , \cdot , \cdot ) : [ t _ { 0 } , T ] \times L ^ { 2 } ( \mathcal { C } ) \times$ $U \to \mathbb { R }$ , and the terminal cost $g ( \cdot ) : L ^ { 2 } ( \mathcal { C } _ { T } ) \to \mathbb { R }$ .
