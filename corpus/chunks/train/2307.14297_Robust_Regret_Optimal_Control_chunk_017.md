# D. Output Feedback Control Design

We now return to the $( \gamma _ { d } , \gamma _ { J } ) \mathrm { - r e g r e t }$ feasibility problem: Given $( \gamma _ { d } , \gamma _ { J } )$ , find a causal, outputfeedback controller K that achieves $( \gamma _ { d } , \gamma _ { J } )$ -regret relative to $K ^ { o }$ or verify that this level of regret cannot be achieved. This section provides a solution to this feasibility problem. Again, we follow the basic procedure in [9], [3], [5], [6], [7], [8], [10] and use a spectral factorization to reduce the problem to an equivalent $H _ { \infty }$ feasibility problem.

The regret in Definition 1 involves bounding the cost $J ( K , d )$ achieved by a causal K by the following:

$$\gamma_ {d} ^ {2} \left\| d \right\| _ {2} ^ {2} + \gamma_ {J} ^ {2} J (K ^ {o}, d) = \sum_ {t = - \infty} ^ {\infty} \gamma_ {d} ^ {2} d _ {t} ^ {\top} d _ {t} + \gamma_ {J} ^ {2} (e _ {t} ^ {o}) ^ {\top} e _ {t} ^ {o}.$$

Here $e ^ { o }$ is the error generated by the closed-loop dynamics with the non-causal controller $K ^ { o }$ in (8). These closed-loop dynamics are given by:

$$x _ {t + 1} = \hat {A} _ {1 1} x _ {t} - B _ {u} K _ {v} v _ {t + 1} + (B _ {d} - B _ {u} K _ {d}) d _ {t}, x _ {- \infty} = 0,\hat {A} _ {1 1} ^ {\top} v _ {t + 1} = v _ {t} - \hat {A} _ {1 1} ^ {\top} X B _ {d} d _ {t}, v _ {\infty} = 0,e _ {t} ^ {o} = \left(C _ {e} - D _ {e u} K _ {x}\right) x _ {t} - D _ {e u} K _ {v} v _ {t + 1} - D _ {e u} K _ {d} d _ {t},$$

where $\hat { A } _ { 1 1 } : = A - B _ { u } K _ { x }$ is a Schur, nonsingular matrix by Lemma 1.

We can explicitly solve for $v _ { t + 1 }$ in terms of $( v _ { t } , d _ { t } )$ by inverting $\hat { A } _ { 1 1 } ^ { \top }$ in the second equation. This allows the closed-loop dynamics with $K ^ { o }$ to be expressed in a simpler form.
