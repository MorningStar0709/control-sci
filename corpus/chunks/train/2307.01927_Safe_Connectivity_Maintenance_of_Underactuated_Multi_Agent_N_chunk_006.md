# A. System Dynamics

We consider a swarm of N agents and use V to describe the set of all agents. Let the actuation signal of each agent i be denoted by $\mathbf { \delta } \mathbf { u } _ { i }$ from a bounded set $\mathbb { U } \in \mathbb { R } ^ { n _ { u } }$ where $n _ { u }$ is the dimensionality of the control. Then, the dynamics for each agent $i \in \nu$ are given by:

$$\dot {\boldsymbol {q}} _ {i} = v (\boldsymbol {q} _ {i}, t) + g (\boldsymbol {q} _ {i}, \boldsymbol {u} _ {i}, t), t \in [ 0, T ] \tag {1}$$

$\mathbf { \Delta } \mathbf { q } _ { i } \in \mathbb { R } ^ { n }$ denotes the position of agent i in the n dimensional state space, where $ { n _ { \mathrm { ~  ~ } } } = \mathrm { ~  { ~ 2 ~ } }$ for a surface vessel on the ocean. The movement of agent i depends on the timevarying non-linear flow field $v ( q _ { i } , t ) \to \mathbb { R } ^ { n }$ and its control $g ( \pmb { q } _ { i } , \pmb { u } _ { i } , t )$ . Although our method works for arbitrary g, we will focus on situations where the agent can directly actuate in each dimension, i.e., $g ( \pmb q _ { i } , \pmb u _ { i } , t ) = \pmb u _ { i } ,$ , in line with our experiments discussed in Section V. Let the agent trajectory resulting from Eq. (1) be described by $\pmb { \xi } _ { i }$ with $\pmb { \xi } _ { i } ( t )$ the state $\pmb q _ { i }$ at t. For the global system of all N agents, we use $\begin{array} { r } { \pmb q = [ \bar { \pmb q } _ { 1 } ^ { \top } , \pmb q _ { 2 } ^ { \top } , \dots , \pmb q _ { N } ^ { \top } ] ^ { \top } , \ \pmb u = [ \pmb { u } _ { 1 } ^ { \top } , \pmb { u } _ { 2 } ^ { \top } , \dots , \bar { \pmb u } _ { N } ^ { \top } ] ^ { \top } } \end{array}$ , and ξ respectively to describe the state, control, and trajectory.

Remark 1. While our method also works in known currents, we focus on realistic settings, where only coarse forecasts vˆ are available to the planner, which differ significantly from the true flows v.
