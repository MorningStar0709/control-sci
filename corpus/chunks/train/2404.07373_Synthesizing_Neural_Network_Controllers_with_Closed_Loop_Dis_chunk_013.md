# 3.1 Static IQCs

We begin with the simplified case where the uncertainty $\Delta$ satisfies a static IQC defined by M, and generalize this in the next subsection.

Lemma 1 The system defined in (10), with the uncertainty satisfying a static IQC defined by M , is dissipative with respect to the quadratic supply rate in (5) if there exist $P \succeq 0$ and $\lambda \geq 0$ such that

$$
\begin{array}{l} \left[ \begin{array}{c c c} A ^ {\top} P + P A & P B _ {w} & P B _ {d} \\ B _ {w} ^ {\top} P & 0 & 0 \\ B _ {d} ^ {\top} P & 0 & 0 \end{array} \right] + \lambda (\star) ^ {\top} M \left[ \begin{array}{c c c} C _ {v} & D _ {v w} & D _ {v d} \\ 0 & I & 0 \end{array} \right] \\ - (\star) ^ {\top} X \left[ \begin{array}{c c c} 0 & 0 & I \\ C _ {e} & D _ {e w} & D _ {e d} \end{array} \right] \preceq 0 \tag {11} \\ \end{array}
$$

PROOF. Consider a trajectory of the system. Multiply (11) on the left and right by $\left\lceil \mathbf { \boldsymbol { x } } ( t ) ^ { \top } \mathbf { \boldsymbol { w } } ( t ) ^ { \top } \mathbf { \boldsymbol { d } } ( t ) ^ { \top } \right\rceil ^ { \top }$ and its transpose. Note that the first term is then equal to $\pmb { x } ( t ) ^ { \top } P \pmb { \dot { x } } ( t ) + \dot { \pmb { x } } ( t ) ^ { \top } P \pmb { x } ( t )$ , the second term is equal to $\lambda ( \star ) ^ { \top } M \left[ \pmb { v } ( t ) ^ { \top } \ \pmb { w } ( t ) ^ { \top } \right] ^ { \top }$ , and the third term is equal to $- ( { \star } ) ^ { \top } X \left[ { \pmb { d } } ( t ) ^ { \top } \ { \pmb { e } } ( t ) ^ { \top } \right] ^ { \top }$ . Integrating from 0 to T shows

$$
\begin{array}{l} \boldsymbol {x} (T) ^ {\top} P \boldsymbol {x} (T) - \boldsymbol {x} (0) ^ {\top} P \boldsymbol {x} (0) + \lambda \int_ {0} ^ {T} \left[ \begin{array}{c} \boldsymbol {v} (t) \\ \boldsymbol {w} (t) \end{array} \right] ^ {\top} M \left[ \begin{array}{c} \boldsymbol {v} (t) \\ \boldsymbol {w} (t) \end{array} \right] d t \\ \leq \int_ {0} ^ {T} \left[ \begin{array}{c} \boldsymbol {d} (t) \\ \boldsymbol {e} (t) \end{array} \right] ^ {\top} X \left[ \begin{array}{c} \boldsymbol {d} (t) \\ \boldsymbol {e} (t) \end{array} \right] d t \\ \end{array}
$$

where the left hand side can be lower bounded by $\pmb { x } ( T ) ^ { \top } P \pmb { x } ( T ) - \pmb { x } ( 0 ) ^ { \top } P \pmb { x } ( 0 )$ due to the signal pair (v, w) satisfying the static IQC defined by M, and thus the integral involving M being nonnegative. This certifies dissipativity with the storage function $S ( x ) = x ^ { \top } P x$ .
