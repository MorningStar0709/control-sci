$$
\begin{array}{l} \Delta V [ \hat {G} _ {\mathrm{av}} [ k ] ] \leq \\ - \frac {1}{2} \left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right) \left(| \hat {G} _ {\mathrm{av}} [ k ] | + \sqrt {\sigma} | \hat {G} _ {\mathrm{av}} [ k ] |\right) \\ \times \left(\left| \hat {G} _ {\mathrm{av}} [ k ] \right| - \sqrt {\sigma} \left| \hat {G} _ {\mathrm{av}} [ k ] \right|\right) \\ = - \frac {1}{2} \left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right) (1 - \sigma) | \hat {G} _ {\mathrm{av}} [ k ] | ^ {2} \\ = - \frac {1}{2} \left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right) (1 - \sigma) V [ \hat {G} _ {\mathrm{av}} [ k ] ]. \tag {54} \\ \end{array}
$$

From (48), $V [ \hat { G } _ { \mathrm { a v } } [ k + 1 ] ] = V [ \hat { G } _ { \mathrm { a v } } [ k ] ] + \Delta V [ \hat { G } _ { \mathrm { a v } } [ k ] ]$ , and by using (54), one has

$$
\begin{array}{l} V [ \hat {G} _ {\mathrm{av}} [ k + 1 ] ] = V [ \hat {G} _ {\mathrm{av}} [ k ] ] + \Delta V [ \hat {G} _ {\mathrm{av}} [ k ] ] \\ \leq \left(1 - \left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right) \frac {(1 - \sigma)}{2}\right) V [ \hat {G} _ {\mathrm{av}} [ k ] ]. \tag {55} \\ \end{array}
$$

The recursive structure of (55) allows us to derive an upper bound for the evolution of the Lyapunov function (47) over successive intervals such that we can iteratively relate the Lyapunov function at the current iteration step k to its value at earlier triggering times. This approach lead us to

$$
\begin{array}{l} V [ \hat {G} _ {\mathrm{av}} [ k ] ] \leq \left(1 - \left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right) \frac {(1 - \sigma)}{2}\right) ^ {k} \\ \times V \left[ \hat {G} _ {\mathrm{av}} [ 0 ] \right]. \tag {56} \\ \end{array}
$$

Therefore, by using (47), we obtain

$$\left. \left| \hat {G} _ {\mathrm{av}} [ k ] \right| \leq \left(1 - \left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right) \frac {(1 - \sigma)}{2}\right) ^ {\frac {k}{2}} \left| \hat {G} _ {\mathrm{av}} [ 0 ] \right|. \right. \tag {57}$$

Moreover, from (41), $| \hat { G } _ { \mathrm { a v } } [ k ] | = | H ^ { * } | | \tilde { \theta } _ { \mathrm { a v } } [ k ] |$ , and, therefore, from (57), we can state

$$\left. \left| \tilde {\theta} _ {\mathrm{av}} [ k ] \right| \leq \left(1 - \left(1 - \left(1 - \frac {\epsilon a ^ {2} H ^ {*} K}{2}\right) ^ {2}\right) \frac {(1 - \sigma)}{2}\right) ^ {\frac {k}{2}} \left| \tilde {\theta} _ {\mathrm{av}} [ 0 ] \right|. \right. \tag {58}$$
