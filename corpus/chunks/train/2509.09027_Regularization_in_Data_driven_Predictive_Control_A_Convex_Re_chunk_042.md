# 6.2 Nonlinear system

We here consider a nonlinear Lotka-Volterra system [1]

$$
\dot {x} = \left[ \begin{array}{c} \dot {x} _ {1} \\ \dot {x} _ {2} \end{array} \right] = \left[ \begin{array}{c} a x _ {1} - b x _ {1} x _ {2} \\ d x _ {1} x _ {2} - c x _ {2} + u \end{array} \right], \tag {39}
$$

where $x _ { 1 } , x _ { 2 }$ denote prey and predator populations and u is the control input. We use $\bar { a } = c = 0 . 5 , \bar { b } = 0 . 0 2 5 , d =$ 0.005 and the discretization time step $\Delta t = 0 . 1$ in our experiments. We denote the discretized form of the nonlinear system (39) and its linearization around the equilibrium $( \bar { u } , \bar { x } _ { 1 } , \bar { x _ { 2 } } ) = ( 0 , \frac { c } { d } , \frac { a } { b } )$ in the error state space as $\hat { x } ( k + 1 ) \ = \ \dot { f } _ { \mathrm { n o n l i n e a r } } \bar { ( } \hat { x } ( k ) , \hat { u } ( k ) )$ and $\hat { x } ( k + 1 ) =$ $f _ { \mathrm { l i n e a r } } ( \hat { x } ( \boldsymbol k ) , \hat { u } ( \boldsymbol k ) )$ , respectively.

We then construct systems with various nonlinearity, which gradually depart from the linear regime and the violation of the LTI assumption becomes more significant. We aim to evaluate how the performance of different DDPC approaches evolves as the level of model mismatch increases. Specifically, we interpolate between fnonlinear and $f _ { \mathrm { l i n e a r } }$ that is

$$
\begin{array}{l} \hat {x} (k + 1) = \epsilon \cdot f _ {\text { linear }} (\hat {x} (k), \hat {u} (k)) \\ + (1 - \epsilon) \cdot f _ {\text {nonlinear}} (\hat {x} (k), \hat {u} (k)), \\ \end{array}
$$

and further collect the input-output trajectories of these systems with different ϵ ranging from 0 to 1 to formulate the associated trajectory libraries. The length of the precollected trajectory is $\dot { T } = 4 0 0$ . The prediction horizon and initial sequence are set as $N = 6 0$ and $T _ { \mathrm { i n i } } = 4$ . We choose $Q = I , R = 0 . 5 I$ and $\mathcal { U } = [ - 2 0 , 2 0 ]$ .
