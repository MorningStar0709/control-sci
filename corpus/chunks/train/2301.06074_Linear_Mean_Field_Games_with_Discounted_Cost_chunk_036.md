$$\int_ {\mathsf {X} ^ {N - 1}} V ^ {*} (y, \mathbf {y} _ {\{2: N \}}) \prod_ {i = 2} ^ {N} p (d y _ {i} | x _ {i}, \pi^ {*} (x _ {i}), \mu) = \int_ {\mathsf {X} ^ {N - 1}} V ^ {*} (y, \mathbf {y} _ {\{2: N \}}) \prod_ {i = 2} ^ {N} p (d y _ {i} | \hat {x} _ {i}, \pi^ {*} (\hat {x} _ {i}), \mu)$$

for any

$$\mathbf {x} _ {\{2: N \}}, \hat {\mathbf {x}} _ {\{2: N \}} \in \mathrm{e} ^ {- 1} \left(\frac {N}{N - 1} \mu - \frac {1}{N - 1} \delta_ {x}\right)$$

as these two vectors are related via some permutation; that is, for some $\sigma \in S _ { \{ 2 : N \} }$ , we have $\mathbf { x } _ { \{ 2 : N \} } = \hat { \mathbf { x } } _ { \sigma ( 2 : N ) }$ . Hence the definition of $T ^ { N }$ does not depend on a particular choice of the vector $\begin{array} { r } { ( x _ { 2 } , \ldots , x _ { N } ) \in \mathtt { e } ^ { - 1 } \left( \frac { N } { N - 1 } \mu - \frac { 1 } { N - 1 } \delta _ { x } \right) } \end{array}$ . Therefore, $T ^ { N }$ is well-defined. In view of this definition, one can re-write the right side of the optimality equation as follows

$$\min _ {a \in \mathsf {A}} \left[ c (x, a, \mu) + \beta \int_ {\mathsf {X}} T ^ {N} (y, \mu , x) p (d y | x, a, \mu) \right]$$

for any $( \mu , x ) \in \mathsf { S } ^ { N }$ ; that is, the right side of the optimality equation depends only on meanfield term and the state variable of agent 1. Since $\gamma _ { \mathrm { o p t } } ^ { N }$ is the minimizer of this re-written optimality equation, we can immediately conclude that $\gamma _ { \mathrm { o p t } } ^ { N }$ depends only on $( \mu , x ) \in \mathsf { S } ^ { N }$ . Let us state this as a separate lemma.

LEMMA 3.4. There exists an optimal policy $\gamma _ { \mathrm { o p t } } ^ { N }$ of $M D P _ { N }$ that is a function of the mean-field term $\mathsf { e } [ \cdot | \mathbf { x } ( t ) ]$ and the state $x _ { 1 } ( t )$ of agent 1; that is, $\gamma _ { \mathrm { o p t } } ^ { N } : \mathsf { S } ^ { N } \to \mathsf { A }$ .

To establish Lipschitz continuity of $\gamma _ { \mathrm { o p t } } ^ { N }$ via assumption (d), we need to first prove that $T ^ { N }$ is Lipschitz continuous.

$$\left| T ^ {N} (y, \mu , x) - T ^ {N} (z, \nu , q) \right| \leq \mathrm{R} _ {1} ^ {N} d _ {\mathrm{X}} (y, z) + \mathrm{R} _ {2} ^ {N} W _ {1} (\mu , \nu) + \mathrm{R} _ {3} ^ {N} d _ {\mathrm{X}} (x, q)$$
