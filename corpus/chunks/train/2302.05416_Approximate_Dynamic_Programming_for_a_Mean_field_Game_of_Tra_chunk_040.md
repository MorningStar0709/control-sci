# Step 1. First Pass for the Weights.

Let $( A , B ) ^ { 1 } ( \cdot )$ be subject to the ODEs:

$$\frac {\mathrm{d}}{\mathrm{d} t} A ^ {1} (t) = \mathfrak {G} (A ^ {1} (t), B ^ {1} (t), \rho^ {0} (t)), A ^ {1} (0) = A _ {0}, \tag {50}\frac {\mathrm{d}}{\mathrm{d} t} B ^ {1} (t) = \mathfrak {H} (A ^ {1} (t), B ^ {1} (t), \rho^ {0} (t)), B ^ {1} (0) = B _ {0}.$$

Lemma 1 gives local Lipschitzianity of the vector fields $\mathfrak { G } ( \cdot , \rho ^ { 0 } ( t ) ) , \mathfrak { H } ( \cdot , \rho ^ { 0 } ( t ) )$ in A, B. So, by Picard-Lindelöf, there is a unique local classical solution to this ODE system of type $\mathrm { C } ^ { 1 } ( [ 0 , t ^ { * } ( A _ { 0 } , B _ { 0 } ) ) ; \mathbb { R } ^ { K \times \bar { K } } \times \mathbb { R } ^ { K \times K } )$ . Note from (39), we have that (using real constant C generically):

$$\mathfrak {G} (A, B, \rho) \leq C (1 + \frac {1}{C} (| | A | | _ {F} + | | B | | _ {F}))$$

for $C > 0$ , and similarly for H. So, from the Grönwall-Bellman inequality, for fixed $s , t \in \mathbb { R } _ { 0 } ^ { + } , 0 \leq s \leq t \colon$

$$
\begin{array}{l} | | A ^ {1} (t) | | _ {F} + | | B ^ {1} (t) | | _ {F} \leq | | A ^ {1} (s) | | _ {F} + | | B ^ {1} (s) | | + \dots \\ \int_ {s} ^ {t} | | \mathfrak {G} (A ^ {1} (\tau), B ^ {1} (\tau), \rho^ {0} (t)) | | _ {F} + | | \mathfrak {H} (A ^ {1} (\tau), B ^ {1} (\tau), \rho^ {0} (t)) | | _ {F} d \tau \leq \dots \\ | | A ^ {1} (s) | | _ {F} + | | B ^ {1} (s) | | + 2 \int_ {s} ^ {t} C (1 + \frac {1}{C} (| | A ^ {1} (\tau) | | _ {F} + | | B ^ {1} (\tau) | | _ {F})) d \tau , \\ \end{array}
$$

so (using constant C generically):

$$| | A ^ {1} (t) | | _ {F} + | | B ^ {1} (t) | | _ {F} \leq (| | A ^ {1} (s) | | _ {F} + | | B ^ {1} (s) | | _ {F} + C (t - s)) \exp (t - s).$$

This rules out finite-time blowup, so there is a unique global classical solution to the ODE system of type $( A , \hat { B } ) ^ { 1 } \in \mathrm { C } ^ { 1 } ( [ 0 , T ] ; \mathbb { R } ^ { \hat { K } \times \tilde { K } } \times \mathbb { R } ^ { K \times K } )$ . Define for $n \geq 1 { : }$ :

$$\tilde {\mathcal {V}} ^ {A ^ {n}, B ^ {n}} (t, y) := \sum_ {i = 0} ^ {K - 1} \sum_ {j = 0} ^ {K - 1} \left(a _ {i j} ^ {n} (t) \sin (\frac {2 \pi i y _ {1}}{\mathfrak {L}}) + b _ {i j} ^ {n} (t) \cos (\frac {2 \pi i y _ {1}}{\mathfrak {L}})\right) \cos (\frac {2 \pi j y _ {2}}{s _ {m a x}}),\tilde {u} ^ {n} (\cdot) := \tilde {u} (\partial_ {y _ {2}} \tilde {\mathcal {V}} ^ {A ^ {n}, B ^ {n}} (\cdot)), \tilde {w} ^ {n} (\cdot) := \tilde {w} (\partial_ {y _ {2}} \tilde {\mathcal {V}} ^ {A ^ {n}, B ^ {n}} (\cdot)).$$

Note that our choice of $\tilde { u } ^ { n } , \tilde { w } ^ { n }$ satisfy the assumptions we needed for $u , w$ in Lemma 3.
