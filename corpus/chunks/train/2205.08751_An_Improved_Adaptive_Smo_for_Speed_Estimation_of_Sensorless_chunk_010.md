# AXES TRANSFORMATION

The per phase equivalent circuit of the induction motor is only valid in steady state condition. It doesn’t hold good while dealing with the transient response of the motor. In transient response condition the voltages and currents in three phases are not in balance condition. It is too much difficult to study the machine performance by analyzing the three phases. In order to reduce this complexity the transformation of axes from 3 –Φ to 2 –Φ is needed. Another reason for the transformation is to analyze any machine with $\cdot _ { \mathrm { n } } \cdot$ number of phases. Thus, an equivalent per phase model of Induction Motor is accepted and adopted universally, that is called as ‘Dynamic d– q model’.

Consider a symmetrical 3-phase induction machine with stationary $\left( a _ { s } b _ { s } c _ { s } \right)$ axis at 120 Degree angle apart. So we needs to transform the 3-phase stationary reference frame $\left( a _ { s } b _ { s } c _ { s } \right)$ variables into 2-phase stationary reference frame $( \mathrm { d ^ { s } } { \cdot } \mathrm { q ^ { s } } )$ variables. Assume that $\mathrm { d ^ { s } } - \mathrm { q ^ { s } }$ axes are oriented at angle of 90 Degree(Since it is a 2-Phase) which is as shown in $\mathrm { F i g }$ 3.1

$$
\left[ \begin{array}{l} V _ {a s} \\ V _ {b s} \\ V _ {c s} \end{array} \right] = \left[ \begin{array}{c c c} \cos \theta & \sin \theta & 1 \\ \cos (1 2 0 ^ {\circ} - \theta) & - \sin (1 2 0 ^ {\circ} - \theta) & 1 \\ \cos (1 2 0 ^ {\circ} + \theta) & \sin (1 2 0 ^ {\circ} + \theta) & 1 \end{array} \right] \left[ \begin{array}{l} V _ {q s} ^ {s} \\ V _ {d s} ^ {s} \\ V _ {o s} ^ {s} \end{array} \right] \tag {3.1}
$$
