# V. SIMULATIONS

In this section, in order to validate our method, we apply the NBS controller to the tracking control of manipulators. We first perform the sin and cos signal tracking task for the two-link planar robot arm with known model information. Our experiments encompass scenarios with and without disturbances and verify the controller performance. Subsequently, using MuJoCo [35], we extend our experimentation to a threelink system, employing LNN to learn the dynamic model of the system and execute trajectory tracking control. We use pytorch with Adam [36] optimizer throughout the experiments. The code is available from: https://github.com/jiajqian/ Neural-Backstepping-tracking-controller.
