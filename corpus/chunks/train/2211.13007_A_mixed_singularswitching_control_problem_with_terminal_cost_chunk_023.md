$$
\begin{array}{l} \mathrm{D} ^ {1} [ \varphi - \eta ] _ {\ell_ {\circ}, \iota_ {\circ}} = 0, \mathrm{tr} [ a _ {\iota_ {\circ}} \mathrm{D} ^ {2} [ \varphi - \eta ] _ {\ell_ {\circ}, \iota_ {\circ}} ] \leq 0, \\ [ \varphi - \eta ] _ {\ell_ {0}, \kappa} - [ \varphi - \eta ] _ {\ell_ {0}, \iota_ {0}} \leq 0, \text {   for   } \kappa \in \mathbb {I} \setminus \{\iota_ {0} \}, \tag {2.10} \\ \eta_ {\ell_ {0}, \iota_ {0}} - \eta_ {\ell^ {\prime}, \iota_ {0}} \leq \varphi_ {\ell_ {0}, \iota_ {0}} - \varphi_ {\ell^ {\prime}, \iota_ {0}}, \text {for} \ell^ {\prime} \in \mathbb {M} \setminus \{\ell_ {0} \}. \\ \end{array}
$$

Since $\psi _ { \delta }$ is increasing, we get that $\psi _ { \delta } ( \eta _ { \ell _ { \circ } , \iota _ { \circ } } - \eta _ { \ell ^ { \prime } , \iota _ { \circ } } - \vartheta _ { \ell _ { \circ } , \ell ^ { \prime } } ) \leq \psi _ { \delta } ( \varphi _ { \ell _ { \circ } , \iota _ { \circ } } - \varphi _ { \ell ^ { \prime } , \iota _ { \circ } } - \vartheta _ { \ell _ { \circ } , \ell ^ { \prime } } )$ at $x _ { \mathrm { o } }$ for each $\ell ^ { \prime } \in \mathbb { M } \setminus \{ \ell _ { \circ } \}$ . From here and by (2.9) and (2.10), it implies

$$
\begin{array}{l} 0 \geq \mathrm{tr} [ a _ {\iota_ {\circ}} \mathrm{D} ^ {2} [ \varphi - \eta ] _ {\ell_ {\circ}, \iota_ {\circ}} ] \\ \geq c _ {\iota_ {0}} [ \varphi - \eta ] _ {\ell_ {0}, \iota_ {0}} - \sum_ {\kappa \in \mathbb {I} \backslash \{\iota_ {0} \}} q _ {\ell_ {0}} (\iota_ {0}, \kappa) [ [ \varphi - \eta ] _ {\ell_ {0}, \kappa} - [ \varphi - \eta ] _ {\ell_ {0}, \iota_ {0}} ] \\ + \sum_ {\ell^ {\prime} \in \mathbb {M} \backslash \{\ell_ {\circ} \}} \left[ \psi_ {\delta} \left(\varphi_ {\ell_ {\circ}, \iota_ {\circ}} - \varphi_ {\ell^ {\prime}, \iota_ {\circ}} - \vartheta_ {\ell_ {\circ}, \ell^ {\prime}}\right) - \psi_ {\delta} \left(\eta_ {\ell_ {\circ}, \iota_ {\circ}} - \eta_ {\ell^ {\prime}, \iota_ {\circ}} - \vartheta_ {\ell_ {\circ}, \ell^ {\prime}}\right) \right] \\ \geq c _ {\iota_ {\circ}} [ \varphi - \eta ] _ {\ell_ {\circ}, \iota_ {\circ}} \quad \text { at } x _ {\circ}. \tag {2.11} \\ \end{array}
$$

Then, $[ \varphi - \eta ] _ { \ell , \iota } ( x ) \leq [ \varphi - \eta ] _ { \ell _ { \circ } , \iota _ { \circ } } ( x _ { \circ } ) \leq 0$ for $( x , \ell , \iota ) \in \overline { { \mathcal { O } } } \times \mathbb { M } \times \mathbb { I }$ , because of $c _ { \iota _ { \circ } } > 0$ on ${ \overline { { \mathcal { O } } } } .$ . Therefore, $\varphi \leq \eta$ for all $x \in { \overline { { \mathcal { O } } } }$ .
