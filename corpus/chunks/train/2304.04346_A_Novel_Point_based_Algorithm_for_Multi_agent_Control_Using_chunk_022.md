# C. Upper Bound Update

We maintain three upper bound functions corresponding to the three stages. Each upper bound function is represented through a set of α-constraints as described in the previous section. Let $\mathbf { M } ^ { \ell } y \le w ^ { \ell }$ represent the α-constraints (on $y \in$ $\scriptstyle { { \mathcal { S } } ^ { \ell } } )$ associated with stage \`. Let $T ^ { \ell }$ be the Bellman operator for stage \`. We now describe the update steps in detail:

Stage $\ell { \bf \Phi } = { \bf \Phi } 0 , 1 ;$ In this stage, the coordinator picks a prescription for agent $i = \ell + 1$ . For $b \in \Delta (  { \mathcal { S } } ^ { \ell } )$ , the Bellman updated upper bound at b is given by

$$
\begin{array}{l} \bar {v} ^ {b} := \left[ T ^ {\ell} U ^ {\ell + 1} \right] (b) = \max _ {\gamma^ {i} \in \bar {\mathcal {A}} ^ {i}} U ^ {\ell + 1} \left(\tau^ {\ell} \left(b, \gamma^ {i}\right)\right) \\ = \max_{\gamma^{i}\in \bar{\mathcal{A}}^{i}}\max_{\substack{y\in \mathbb{R}^{\mathcal{S}^{i}}\\ \mathbf{M}^{i}y\leq w^{i}}}\sum_{(s^{\ell},a^{i})\in \mathcal{S}^{\ell}\times \mathcal{A}^{i}}b(s^{\ell})\gamma^{i}(a^{i}|m_{s^{\ell}}^{i})y(s^{\ell},a^{i}) \\ =: \max_ {\substack {\gamma^ {i} \in \bar {\mathcal {A}} ^ {i}}} \max_ {\substack {y \in \mathbb {R} ^ {\mathcal {S} ^ {i}} \\ \mathbf {M} ^ {i} y \leq w ^ {i}}} J ^ {i} (b, \gamma^ {i}, y) \tag{17} \\ \end{array}
$$

Now, notice that if we treat each $\gamma ^ { i }$ as a 0-1 indicator vector in $[ 0 , 1 ] ^ { \mathcal { A } ^ { i } \times \mathcal { M } ^ { i } }$ , then $J ^ { i } ( b , \gamma ^ { i } , y )$ is bilinear in $\gamma ^ { i }$ and y: It is linear in $\gamma ^ { i }$ for each fixed y and linear in y for each fixed $\gamma ^ { i }$ . Therefore, (17) is a bilinear programming (BP) problem, which has been studied extensively in the optimization literature [27], [28], [29] as well as decentralized control literature. [30], [31]. Notably, in [30], the author provided a method to convert bilinear programs to MILPs2. Gurobi Optimization™ has also developed specialized solvers for bilinear programs [32].
