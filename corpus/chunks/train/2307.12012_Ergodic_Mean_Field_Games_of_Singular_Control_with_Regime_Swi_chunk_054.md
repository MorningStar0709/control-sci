$$
\begin{array}{l} A ^ {\epsilon} \leq \widehat {\mathbb {E}} \left[ \int_ {0} ^ {\infty} e ^ {- c t} \left| \pi_ {x} (\widehat {X} _ {t} ^ {x + \epsilon}, \theta) - \pi_ {x} (\widehat {X} _ {t} ^ {x}, \theta) \right| d t \right] \tag {A.6} \\ + \widehat {\mathbb {E}} \bigg [ \int_ {0} ^ {\infty} \left| e ^ {\int_ {0} ^ {t} b _ {x} (\widehat {X} _ {s} ^ {x + \epsilon}, Y _ {s} ^ {i}) d s} - e ^ {\int_ {0} ^ {t} b _ {x} (\widehat {X} _ {s} ^ {x}, Y _ {s} ^ {i}) d s} \right| \cdot \left| \pi_ {x} (\widehat {X} _ {t} ^ {x}, \theta) \right| d t \bigg ]. \\ \end{array}
$$

Following the same arguments as in Theorem 38 from Chapter V.7 in [48], we can prove that $x \mapsto \widehat { X } _ { t } ^ { x }$ is continuous $\widehat { \mathbb { P } } \otimes d t { - } \mathrm { a . s . }$ , so that

(A.7) $\Big | e ^ { \int _ { 0 } ^ { t } b _ { x } ( \widehat { X } _ { s } ^ { x + \epsilon } , Y _ { s } ^ { i } ) d s } - e ^ { \int _ { 0 } ^ { t } b _ { x } ( \widehat { X } _ { s } ^ { x } , Y _ { s } ^ { i } ) d s } \Big | \to 0 , \epsilon \downarrow 0 \widehat { \mathbb { P } } \otimes d t \mathrm { - a . s . }$

Furthermore,

(A.8) $\big | e ^ { \int _ { 0 } ^ { t } b _ { x } ( \widehat { X } _ { s } ^ { x + \epsilon } , Y _ { s } ^ { i } ) d s } - e ^ { \int _ { 0 } ^ { t } b _ { x } ( \widehat { X } _ { s } ^ { x } , Y _ { s } ^ { i } ) d s } \big | \leq 2 e ^ { - c t } .$

Hence, combining (A.7), (A.8) with Condition (5) in Assumption 2.2 we obtain that the second expectation on the right-hand side of (A.6) vanishes as $\epsilon \downarrow 0$ . On the other hand, one can prove a version of comparison principle for hybrid-SDEs 1, i.e. $\widehat { X } _ { t } ^ { x } \leq \widehat { X } _ { t } ^ { x + \epsilon } , \mathbb { P } \otimes d t \mathrm { - a . s }$ . and using that $\pi _ { x } ( x , \theta ) \geq 0$ for any $( x , \theta ) \in \mathcal { I } \times \mathbb { R } _ { + }$ and $x \mapsto \pi _ { x } ( x , \theta )$ is decreasing for any $\theta \in \mathbb { R } _ { + }$ , we obtain

(A.9) $\left. \pi _ { x } ( \widehat { X } _ { t } ^ { x + \epsilon } , \theta ) - \pi _ { x } ( \widehat { X } _ { t } ^ { x } , \theta ) \right. \leq 2 \pi _ { x } ( \widehat { X } _ { t } ^ { x } , \theta ) , \quad \widehat { \mathbb { P } } \otimes d t \mathrm { - a . s . }$

Therefore, using again Condition (5) in Assumption 2.2, we can let $\epsilon \downarrow 0$ and we observe that also the first expectation on the right-hand side of (A.5) converges to zero, due to the dominated convergence theorem.
