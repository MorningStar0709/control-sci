# B. Unit-quaternion

Unit-quaternion is a useful tool for singularity-free attitude (orientation) representation in 3D space. Q is a unit-quaternion vector such that $Q = [ q _ { 0 } , q _ { 1 } , q _ { 2 } , q _ { 3 } ] ^ { \top } = [ q _ { 0 } , \pmb { q } ^ { \top } ] ^ { \top } \in \mathbb { R } ^ { 4 }$ and ||Q|| = 1 [12], [42] where $q _ { 0 } \in \mathbb { R }$ and $\pmb q = [ q _ { 1 } , q _ { 2 } , q _ { 3 } ] ^ { \top } \in \mathbb { R } ^ { 3 }$ . The unit-quaternion four elements are given as follows [42]:

$$Q = q _ {0} + i q _ {1} + j q _ {2} + k q _ {3}$$

where $i , j ,$ , and k refers to the standard basis-vectors. $Q ^ { - 1 }$ denoted inversion of $Q$ and is given by [11]:

$$Q ^ {- 1} = Q ^ {*} = \left[ q _ {0}, - q _ {1}, - q _ {2}, - q _ {3} \right] ^ {\top} = \left[ q _ {0}, - \boldsymbol {q} ^ {\top} \right] ^ {\top} \in \mathbb {R} ^ {4}$$

Consider $\otimes$ to be quaternion multiplication and let $Q \ =$ $[ q _ { 0 } , q ^ { \top } ] ^ { \top }$ and $Q _ { d } = [ q _ { 0 d } , \pmb { q } _ { d } ^ { \top } ] = [ q _ { 0 d } , q _ { 1 d } , q _ { 2 d } , q _ { 3 d } ] ^ { \top }$ be two quaternion vectors describing true and desired UAV attitude, respectively, where $q _ { 0 } , q _ { 0 d } \in \mathbb { R }$ and $q , q _ { d } \in \mathbb { R } ^ { 3 }$ . The quaternion multiplication of $Q$ and $Q _ { d }$ is given by:

$$
Q \otimes Q _ {d} = \left[ \begin{array}{c} q _ {0} q _ {0 d} + \boldsymbol {q} ^ {\top} \boldsymbol {q} _ {d} \\ q _ {0 d} \boldsymbol {q} - q _ {0} \boldsymbol {q} _ {d} + [ \boldsymbol {q} ] \times \boldsymbol {q} _ {d} \end{array} \right] = \left[ \begin{array}{c} e _ {0} \\ e \end{array} \right]
$$

Quaternion multiplication is associative but not commutative. The UAV attitude representation in form of a rotational matrix described with respect to the Lie Group of the Special Orthogonal Group $S O ( 3 )$ is described by

$$R = \left(q _ {0} ^ {2} - \boldsymbol {q} ^ {\top} \boldsymbol {q}\right) \mathbf {I} _ {3} + 2 \boldsymbol {q} \boldsymbol {q} ^ {\top} + 2 q _ {0} [ \boldsymbol {q} ] _ {\times} \in S O (3) \subset \mathbb {R} ^ {3 \times 3}$$

Which is equivalent

$$
R = \left[ \begin{array}{l l l} 1 - 2 \left(q _ {2} ^ {2} + q _ {3} ^ {2}\right) & 2 \left(q _ {1} q _ {2} - q _ {0} q _ {3}\right) & 2 \left(q _ {1} q _ {3} + q _ {0} q _ {2}\right) \\ 2 \left(q _ {2} q _ {1} + q _ {0} q _ {3}\right) & 1 - 2 \left(q _ {1} ^ {2} + q _ {3} ^ {2}\right) & 2 \left(q _ {2} q _ {3} - q _ {0} q _ {1}\right) \\ 2 \left(q _ {3} q _ {1} - q _ {0} q _ {2}\right) & 2 \left(q _ {3} q _ {2} + q _ {0} q _ {1}\right) & 1 - 2 \left(q _ {1} ^ {2} + q _ {2} ^ {2}\right) \end{array} \right] \in S O (3) \tag {2}
$$

where $d e t ( R ) \ = \ + 1$ and $R R ^ { \top } \ = \ { \bf I } _ { 3 }$ with $d e t ( \cdot )$ being determinant of a matrix and $R \in \{ B \}$ .
