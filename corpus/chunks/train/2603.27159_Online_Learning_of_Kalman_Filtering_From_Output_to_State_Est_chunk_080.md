$$\sum_ {t = 2 s _ {0}} ^ {T - 1} \left(\left\| y _ {t} - \hat {y} _ {t} ^ {\mathrm{SK}} \right\| ^ {2} - \left\| y _ {t} - \hat {y} _ {t} ^ {\mathrm{KF}} \right\| ^ {2}\right) = \mathcal {O} (\log^ {2} T) + \mathcal {O} \left(\frac {\log^ {3} T}{T}\right) = \mathcal {O} (\log^ {2} T), \tag {C.23}$$

where we used the fact $T \geq \log T$ .

Upper bound on the summation of $\left\| y _ { t } - \hat { y } _ { t } ^ { \mathrm { S K } } \right\| ^ { 2 } - \left\| y _ { t } - \hat { y } _ { t } ^ { \mathrm { K F } } \right\| ^ { 2 }$ over $t \in \{ 0 , \ldots , 2 s _ { 0 } - 1 \}$ . We have

$$
\begin{array}{l} \sum_ {t = 0} ^ {2 s _ {0} - 1} \left(\left\| y _ {t} - \hat {y} _ {t} ^ {\mathrm{SK}} \right\| ^ {2} - \left\| y _ {t} - \hat {y} _ {t} ^ {\mathrm{KF}} \right\| ^ {2}\right) \stackrel {(a)} {\leq} \sum_ {t = 0} ^ {2 s _ {0} - 1} \left(4 \left\| y _ {t} \right\| ^ {2} + 2 \left\| y _ {t} ^ {\mathrm{SK}} \right\| ^ {2} + 2 \left\| \hat {y} _ {t} ^ {\mathrm{KF}} \right\| ^ {2}\right) \\ \stackrel {(b)} {\leq} \sum_ {t = 0} ^ {2 s _ {0} - 1} \left(4 R _ {y} + 2 R _ {y} \| L \| \| C \| \frac {\kappa_ {F}}{1 - \gamma_ {F}} + \mathcal {O} (R _ {y} ^ {2} t ^ {2})\right) \\ \leq 2 s _ {0} \big (\mathcal {O} (\sqrt {\log T}) + \mathcal {O} ((2 s _ {0} - 1) ^ {2} \log T) \big) \\ \stackrel {(c)} {\leq} \mathcal {O} (\log^ {4} T), \tag {C.24} \\ \end{array}
$$

where (a) follows from the fact $\left\| a + b \right\| ^ { 2 } \leq 2 \left\| a \right\| ^ { 2 } + 2 \left\| b \right\| ^ { 2 }$ . To obtain (b), we first recall the upper bounds $\| y _ { t } \| \le R _ { y }$ and the upper bound on $\left\| y _ { t } ^ { \mathbb { S K } } \right\|$ given by (C.9) in the proof of Lemma 4.5, for all $t \in \{ 0 , \ldots , T - 1 \}$ . Next, we notice from $( \mathrm { C . \ddot { 1 1 } } )$ in Lemma C.1 that $\left\| \hat { x } _ { t } ^ { \mathrm { K F } } \right\| = \mathcal { O } ( R _ { y } t )$ , which implies that $\left\| \hat { y } _ { t } ^ { \mathrm { K F } } \right\| ^ { 2 } \leq \left\| C \right\| ^ { 2 } \left\| \hat { y } _ { t } ^ { \mathrm { K F } } \right\| ^ { 2 } = \mathcal { O } ( R _ { y } ^ { 2 } t ^ { 2 } )$ for all $t \in \{ 0 , \ldots , T - 1 \}$ . Finally, (c) follows from the choice of $s _ { 0 } = \mathcal { O } ( \log T )$ .

Combining (C.23) and (C.24) completes the proof of Lemma 4.6.
