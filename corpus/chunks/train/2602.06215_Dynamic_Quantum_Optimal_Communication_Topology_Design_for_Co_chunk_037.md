# 5 Numerical Results

We present numerical examples that demonstrate the proposed dynamic quantum topology design method. All experiments are generated by a single Python script that follows Algorithm 2. The simulations are carried out on a classical computer, and the quantum routines are executed on a Qiskit simulator rather than on real quantum hardware. We use Qiskit version 0.46.3 and qiskit-algorithms version 0.3.1. The QITE solver is im-

plemented using VarQITE from qiskit-algorithms, combined with the ImaginaryMcLachlanPrinciple and an EfficientSU2 ansatz with linear entanglement. The complete code is available at GitHub.
