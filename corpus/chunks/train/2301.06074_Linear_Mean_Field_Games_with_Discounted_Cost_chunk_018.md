(c) The transition probability $\bar { p } : \mathsf { X } \times \mathsf { A } \times \mathsf { X } \to \mathcal { P } ( \mathsf { X } )$ is Lipschitz continuous with Lipschitz constants $( \mathsf { K } _ { 1 } , \mathsf { K } _ { 2 } , \mathsf { K } _ { 3 } )$ ; that is, for any $( x , a , y ) , ( z , b , r ) \in \mathsf { X } \times \mathsf { A } \times \mathsf { X }$ , we have

$$W _ {1} (\bar {p} (\cdot | x, a, y), \bar {p} (\cdot | z, b, r)) \leq \mathsf {K} _ {1} d _ {\mathsf {X}} (x, z) + \mathsf {K} _ {2} \| a - b \| + \mathsf {K} _ {3} d _ {\mathsf {X}} (y, r)$$

One can also prove that c and p are also Lipschitz continuous with the same Lipschitz constants as in (b) and (c), respectively. To define the next assumption, let us introduce the following continuous function

$$\mathbb {L}: [ 0, \infty) \times [ 0, \infty) \times [ 0, \infty) \to [ 0, \infty) \times [ 0, \infty)$$

(d) For any Lipschitz continuous $J : \mathsf { X } \times \mathcal { P } ( \mathsf { X } ) \times \mathsf { X } \to [ 0 , \infty )$ with Lipschitz constants $\left( \mathsf { R } _ { 1 } , \mathsf { R } _ { 2 } , \mathsf { R } _ { 3 } \right)$ , define

$$F ^ {J} (x, a, \mu) := c (x, a, \mu) + \beta \int_ {\mathsf {X}} J (y, \mu , x) p (d y | x, a, \mu)$$

We assume that for any $( x , \mu ) \in \mathsf X \times \mathcal P ( \mathsf X ) , F ^ { J } ( x , \cdot , \mu )$ is ρ-strongly convex; that is, for any $a , b \in \mathsf { A }$ , we have

$$F ^ {J} (x, b, \mu) - F ^ {J} (x, a, \mu) \geq \langle \nabla_ {a} F ^ {J} (x, a, \mu), b - a \rangle + \frac {\rho}{2} \| b - a \| ^ {2}$$

where $\nabla _ { a } F ^ { J } ( x , a , \mu )$ is the gradient of $F ^ { J } ( x , a , \mu )$ with respect to a. Moreover, the gradient $\nabla _ { a } F ^ { J } ( x , a , \mu )$ is Lipschitz continuous in $( x , \mu )$ for any $a \in \mathsf { A }$ with Lipschitz constants $\mathbb { L } ( \mathsf { R } _ { 1 } , \mathsf { R } _ { 2 } , \mathsf { R } _ { 3 } ) = ( \mathbb { L } _ { 1 } ( \mathsf { R } _ { 1 } , \mathsf { R } _ { 2 } , \mathsf { R } _ { 3 } ) , \mathbb { L } _ { 2 } ( \mathsf { R } _ { 1 } , \mathsf { R } _ { 2 } , \mathsf { R } _ { 3 } ) )$ ; that is

$$\sup _ {a \in \mathsf {A}} \| \nabla_ {a} F ^ {J} (x, a, \mu) - \nabla_ {a} F ^ {J} (z, a, \nu) \| \leq \mathbb {L} _ {1} (\mathsf {R} _ {1}, \mathsf {R} _ {2}, \mathsf {R} _ {3}) d _ {\mathsf {X}} (x, z) + \mathbb {L} _ {2} (\mathsf {R} _ {1}, \mathsf {R} _ {2}, \mathsf {R} _ {3}) W _ {1} (\mu , \nu)$$

for all $( x , \mu ) , ( z , \nu ) \in \mathsf { X } \times \mathcal { P } ( \mathsf { X } )$ .
