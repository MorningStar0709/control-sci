This specifies a minor follower’s best response at time t given the history of the mean-field state and its private type up to time t and the leader’s strategy from time t on-wards. Note that this mapping specifies a complete policy for the follower for all time t.

Similarly we define the best response of major follower $j$ as

$$
\begin{array}{l} B R _ {t} ^ {m, j} (\underline {{\pi_ {t}}}, z _ {1: t}, a _ {1: t - 1} ^ {l, m}, x _ {1: t} ^ {m, j}, \sigma_ {t: T} ^ {l}, \sigma_ {t: T} ^ {m, - j}, \sigma_ {t: T} ^ {f}) \\ := \arg \max _ {\sigma^ {m, j}} \mathbb {E} ^ {\sigma_ {t: T} ^ {l} \sigma_ {t: T} ^ {m, j} \sigma_ {t: T} ^ {m, - j}, \sigma_ {t: T} ^ {f}, \underline {{\pi}} _ {t}} \left\{\sum_ {n = t} ^ {T} \delta^ {n - t} R _ {n} ^ {m, j} (Z _ {n}, X _ {n} ^ {l, m}, A _ {n} ^ {l, m}) | \underline {{\pi}} _ {t}, z _ {1: t}, a _ {1: t - 1} ^ {l, m}, x _ {1: t} ^ {m, j} \right\} \tag {5} \\ \end{array}
B R ^ {m, j} (z, \sigma^ {l}, \sigma^ {m, - j}, \sigma^ {f}) := \bigcap_ {t} \bigcap_ {a _ {1: t - 1} ^ {m, j}} \bigcap_ {x _ {1: t} ^ {m, j}} B R _ {t} ^ {m, j} (\underline {{\pi}} _ {t}, z _ {1: t}, a _ {1: t - 1} ^ {l, m}, x _ {1: t} ^ {m, j}, \sigma_ {t: T} ^ {l}, \sigma_ {t: T} ^ {m, - j}, \sigma_ {t: T} ^ {f}). \tag {6}$$

With some abuse of notation, we will also say $\sigma _ { t } ^ { f } \in B R _ { t } ^ { f } ( z _ { 1 : T } , \sigma _ { 1 : T } ^ { l , m } )$ if there exists $\hat { \sigma } ^ { f } \in B R ^ { f } ( z _ { 1 : T } , \sigma _ { 1 : T } ^ { l , m } )$ such that $\sigma _ { t } ^ { f } = \hat { \sigma } _ { t } ^ { f }$ . Similarly we say $\sigma _ { t } ^ { m , j } \in B R _ { t } ^ { m , j } ( z _ { 1 : T } , \sigma _ { 1 : T } ^ { l } , \sigma _ { 1 : T } ^ { m , - j } , \sigma _ { 1 : T } ^ { f } )$ if there exists $\hat { \sigma } ^ { m , j } \in B R ^ { m , j } ( z _ { 1 : T } , \sigma _ { 1 : T } ^ { l } , \sigma _ { 1 : T } ^ { m , - j } , \sigma _ { 1 : T } ^ { f } )$ such that $\sigma _ { t } ^ { m , j } = \hat { \sigma } _ { t } ^ { m , j }$

We now define best response of Stackelberg leader i as follows

$$B R ^ {l, i} \left(z _ {1: T}, \sigma^ {l, - i}\right) := \bigcap_ {t} \bigcap_ {a _ {1: t - 1} ^ {l, i}} \bigcap_ {x _ {1: t} ^ {l, i}} \arg \max _ {\sigma^ {l, i}} \mathbb {E} ^ {\sigma^ {l}, \sigma^ {m}, \sigma^ {f}, \underline {{\pi}} _ {t}} \left\{\sum_ {n = t} ^ {T} \delta^ {n - t} R _ {n} ^ {l, i} \left(Z _ {n} X _ {n} ^ {l, m}, A _ {n} ^ {l, m}\right) | \underline {{\pi}} _ {t}, z _ {1: t}, a _ {1: t - 1} ^ {l, m}, x _ {1: t} ^ {l, i} \right\}, \tag {7}\text { where, } \hat {\sigma} ^ {f} \in B R ^ {f} (z, \sigma^ {l}, \hat {\sigma} ^ {m}), \hat {\sigma} ^ {m} \in B R ^ {f} (z, \sigma^ {l}, \hat {\sigma} ^ {m}, \hat {\sigma} ^ {f}) \tag {8}$$
