# II. PRELIMINARIES

Given a vector field $X : \mathbb { R } ^ { n } \times \mathbb { R } _ { > 0 } \to \mathbb { R } ^ { n }$ , a set of initial states $\mathcal { A } \subset \mathbb { R } ^ { n }$ , and a time interval $\mathcal { T } \subseteq \mathbb { R } _ { \geq 0 } ,$ the flow generated by X is defined as the map $\Phi _ { X } : \mathcal { A } ^ { - } \times \mathcal { T }  \mathbb { R } ^ { n }$ , e.g., [24]–[26], satisfying $\Phi _ { X } ( x , 0 ) = x$ and

$$\frac {\partial}{\partial t} \Phi_ {X} (x, t) = X (\Phi_ {X} (x, t), t). \tag {1}$$

Fixing an initial state $x _ { 0 } = \Phi _ { X } ( x _ { 0 } , 0 ) \in \mathcal { A } ,$ , (1) becomes $\begin{array} { r } { \frac { d } { d t } \Phi _ { X } ( x _ { 0 } , t ) = X ( \Phi _ { X } ( x _ { 0 } , t ) , t ) } \end{array}$ , which is equivalent to

$$\frac {d}{d t} x (t) = X (x (t), t), \quad x (0) = x _ {0}, \tag {2}$$

whose solution $x ( t ) ~ = ~ \Phi _ { X } ( x _ { 0 } , t )$ is called a trajectory (or integral curve, e.g., [25]). Such $x ( t )$ exists and is unique for each $x _ { 0 } ~ \in ~ { \mathcal { A } }$ on a maximal interval $[ 0 , \tau _ { \operatorname* { m a x } } ( x _ { 0 } ) )$ , if the time-varying1 vector field $X ( x , t )$ is continuous in time t and is locally Lipschitz in state x uniformly on compact time intervals (see, e.g., [26]–[28]). Throughout this paper, we further assume that the dynamical system of interest in the form of (2) (equivalently, the corresponding vector field $X ( x , t ) )$ is forward complete, i.e., $\tau _ { \mathrm { m a x } } ( x _ { 0 } ) = \infty , \forall x _ { 0 } \in \mathcal { A } .$ , so the solution $\Phi _ { X } ( x _ { 0 } , t )$ is well-defined $\forall t \geq 0 , \forall x _ { 0 } \in \mathcal { A } .$ .

For a real-valued function $F : \mathbb { R } ^ { n } \times \mathbb { R } _ { \ge 0 } \to \cdot$ R of class $C ^ { 1 }$ in x and t, the total derivative (or Lagrangian derivative) of F along a trajectory $x ( t ) = \Phi _ { X } ( x _ { 0 } , t )$ of (2) is the sum of the Lie derivative and the Eulerian derivative (see, e.g., [29]), i.e.,

$$\frac {d}{d t} F (x (t), t) = L _ {X} F (x (t), t) + \frac {\partial}{\partial t} F (x (t), t), \tag {3}$$

where the Lie derivative is calculated as

$$L _ {X} F (x, t) = \nabla_ {x} F (x, t) X (x, t),$$
