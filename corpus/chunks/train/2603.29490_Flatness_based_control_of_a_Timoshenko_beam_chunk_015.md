# V. STATE FEEDBACK TRACKING CONTROLLER

To design a tracking controller that stabilizes the transition of the Timoshenko beam, we assume given reference trajectories

$$
y _ {i, \mathrm{r}} (t) = \left\{ \begin{array}{l l} y _ {i, 0}, & t <   t _ {0} \\ p _ {i} (t), & t _ {0} \leq t \leq t _ {T} \\ y _ {i, T}, & t > t _ {T}, \end{array} \right. \tag {21}
$$

$i = 1 , 2$ , for the flat output ${ \mathbf { } } _ { { \mathbf { } } ^ { y ( t ) } }$ , where $p _ { i } ( t )$ are continuous functions satisfying $p _ { i } ( t _ { 0 } ) = y _ { i , 0 }$ and $p _ { i } ( t _ { T } ) = y _ { i , T }$ . The trajectory represents the transition of the system between the stationary solutions $( w _ { 0 } ( z ) , \varphi _ { 0 } ( z ) )$ and $( w _ { T } ( z ) , \varphi _ { T } ( z ) )$ of (2). The associated initial and terminal values $\pmb { y } _ { 0 }$ and ${ \pmb y } _ { T }$ are obtained via the preliminary transformations from Section III (cf. Remark 2). Notably, by evaluating the input parametrization (17) for $\begin{array} { r } { \pmb { y } ( \cdot ) ~ = ~ \pmb { y } _ { \mathrm { r } } ( \cdot ) } \end{array}$ , the corresponding feedforward control ${ \pmb u } _ { \mathrm { r } } ( t )$ can be derived.

To stabilize the reference trajectory (21), a tracking controller is derived. For this purpose, a new input ${ \pmb v } ( t )$ is introduced via the state feedback

$$\bar {u} _ {1} (t) = - v _ {1} (t) \tag {22a}\bar {u} _ {2} (t) = - v _ {2} (t) + \int_ {\tau_ {1}} ^ {2 \tau_ {1}} \bar {a} _ {0} ^ {-} (\tau_ {1} + \tau_ {2} - \tau) \eta_ {1} (\tau , t) \mathrm{d} \tau+ \int_ {0} ^ {\Delta \tau + \tau_ {2}} \tilde {a} _ {0} (\tau) v _ {1} (t + \tau) \mathrm{d} \tau , \tag {22b}$$

which decouples the HCF (20) into two independent systems of transport equations

$$\partial_ {t} \eta_ {i} (\tau , t) = \partial_ {\tau} \eta_ {i} (\tau , t), \quad \tau \in [ 0, 2 \tau_ {i}) \tag {23a}\eta_ {i} (2 \tau_ {i}, t) = v _ {i} (t) \tag {23b}$$

for $\begin{array} { r l r } { i } & { { } = } & { 1 , 2 } \end{array}$ . This structure can be interpreted as the hyperbolic counter-part to the Brunovsky form (see, e.g., [7]). The control design for the system in the form (23) is particularly simple, since it essentially corresponds to designing controllers for two independent SI systems (see, $\mathrm { e . g . }$ ., [18]). To this end, we define the tracking errors

$$e _ {i} (t + \tau) = \eta_ {i} (\tau , t) - y _ {i, \mathrm{r}} (t + \tau), \quad i = 1, 2. \tag {24}$$
