Thus the stability results for linear systems are obtained in terms of certain Linear Matrix Inequalities (LMIs). For instance, the following result by Hespanha and Morse in [12] provides a sufficient condition for stability of the reset switched system (2) under arbitrary switching.

Theorem 4.1 (Hespanha and Morse [12]). Suppose there exists a collection of symmetric, positive definite matrices $\{ Q _ { p } \in \ M _ { n } ( \mathbf { \bar { \mathbb { R } } } ) \colon \ p \in \mathcal { P } \}$ such that

$$Q _ {p} A _ {p} + A _ {p} ^ {\top} Q _ {p} < 0, \text {for all} p \in \mathcal {P}, \text {and} R _ {(q, p)} ^ {\top} Q _ {p} R _ {(q, p)} \leq Q _ {q}, \text {for all} p, q \in \mathcal {P},$$

then the reset switched system (2) is stable under arbitrary switching.

Another result of our interest is the following result by Geromel and Colaneri in [10] which gives a sufficient condition for stability of switched system (1) under a dwell time switching strategy.

Theorem 4.2 (Geromel and Colaneri [10]). Assume that for some $\tau > 0$ , there exists a collection of symmetric, positive definite matrices $\{ Q _ { p } \in \ M _ { n } ( \mathbb { R } ) \colon \ p \in \mathcal { P } \}$ such that

$$Q _ {p} A _ {p} + A _ {p} ^ {\top} Q _ {p} < 0, \text {for all} p \in \mathcal {P}, \text {and} \mathrm{e} ^ {A _ {q} ^ {\top} \tau} Q _ {p} \mathrm{e} ^ {A _ {q} \tau} \leq Q _ {q}, \text {for all} p, q \in \mathcal {P},$$

then the switched system (1) is stable for all signals $\sigma \in \mathcal { S } _ { s } [ \tau ]$ .

The above two results motivate us to obtain a sufficient condition for stability of the reset switched system (2) under dwell time switching strategy, thereby addressing issue I.3. This condition is given in the following result.

Theorem 4.3. Given the reset switched system (2), assume that for some $\tau > 0$ , there exists a collection of symmetric, positive definite matrices $\{ Q _ { p } \in \ M _ { n } ( \mathbb { R } ) \colon \ p \in \mathcal { P } \}$ such that

$$Q _ {p} A _ {p} + A _ {p} ^ {\top} Q _ {p} < 0 \quad \text { for all } p \in \mathcal {P}, \tag {7}\mathrm{e} ^ {A _ {q} ^ {\top} \tau} R _ {(q, p)} ^ {\top} Q _ {p} R _ {(q, p)} \mathrm{e} ^ {A _ {q} \tau} \leq Q _ {q} \text { for all } p, q \in \mathcal {P}, \tag {8}$$

then the reset switched system (2) is stable for all signals $\sigma \in \mathcal { S } _ { s } [ \tau ]$ .
