Condition 3. Select $\tau _ { \mathrm { m a x } } ^ { i } > 0$ and $\tau _ { \mathrm { M A D } } ^ { i } > 0 , i \in \mathcal { N } ;$ , with $\tau _ { \mathrm { m a x } } ^ { i } \geq \tau _ { \mathrm { M A D } } ^ { i } + \tau _ { \mathrm { M A S P } } ^ { i }$ such that

$$\tilde {\gamma} _ {i} (0) \phi_ {0, i} (\tau_ {\max} ^ {i}) \geqslant \lambda_ {i} ^ {2} \tilde {\gamma} _ {i} (1) \phi_ {1, i} (0), \tag {26}\tilde {\gamma} _ {i} (1) \phi_ {1, i} (\tau_ {i}) \geqslant \tilde {\gamma} _ {i} (0) \phi_ {0, i} (\tau_ {i}), \text { for all } \tau_ {i} \in [ 0, \tau_ {\mathrm{MAD}} ^ {i} ], \tag {27}$$

where $\phi _ { l , i } , l \in \{ 0 , 1 \}$ , evolves according to

$$\frac {d}{d \tau_ {i}} \phi_ {l, i} = - \left(2 \tilde {L} _ {i} (l) \phi_ {l, i} + \tilde {\gamma} _ {i} (l) \left(\frac {1}{\mu_ {i} \epsilon_ {i}} \phi_ {l, i} ^ {2} + 1\right)\right), \tag {28}$$

for some fixed initial conditions $\phi _ { l , i } ( 0 ) , \ l \ \in \ \{ 0 , 1 \}$ , that satisfy $\tilde { \gamma } _ { i } ( 1 ) \phi _ { 1 , i } ( 0 ) \geqslant \tilde { \gamma } _ { i } ( 0 ) \phi _ { 0 , i } ( 0 ) > \lambda _ { i } ^ { 2 } \tilde { \gamma } _ { i } ( 1 ) \phi _ { 1 , i } ( 0 ) > 0 .$ , where, for each $i \in \mathcal N$ and $l \in \{ 0 , 1 \}$ , the functions $\tilde { L } _ { i } : \{ 0 , 1 \} \to \mathbb { R } _ { \geqslant 0 }$ and $\tilde { \gamma } _ { i } : \{ 0 , 1 \} \to \mathbb { R } _ { \geqslant 0 }$ are given by

$$\tilde {L} _ {i} (l) := \lambda_ {i} ^ {- l} \sqrt {N _ {i}} L _ {i}, \quad \tilde {\gamma} _ {i} (l) := \lambda_ {i} ^ {- l} \gamma_ {i}, \tag {29}$$

with $N _ { i }$ the cardinality of $\mathcal { V } _ { i } ^ { \mathrm { o u t } }$ and where $\mu _ { i }$ and $\gamma _ { i }$ satisfy Condition 2. The constants $\lambda _ { i } ~ \in ~ ( 0 , 1 )$ and $\epsilon _ { i } ~ \in ~ ( 0 , 1 ]$ , $i \in \mathcal { N } ,$ are tuning parameters. If the above conditions are satisfied, $\tau _ { \mathrm { M I E T } } ^ { i }$ is defined as $\tau _ { \mathrm { M I E T } } ^ { i } : = \tau _ { \mathrm { m a x } } ^ { i } - \tau _ { \mathrm { M A S P } } ^ { i } .$ ✷

Condition 3 can always be ensured, as long as sufficiently fast sampling is available. In practice, based on the constants $\gamma _ { i } , \ ( \tau _ { \mathrm { m a x } } ^ { \bar { i } } , \tau _ { \mathrm { M A D } } ^ { i } )$ curves can be generated to intuitively select appropriate values for $\lambda _ { i } , \phi _ { 0 , i } ( 0 )$ and $\phi _ { 1 , i } ( 0 )$ .

These conditions are similar to the conditions in Dolk et al. (2017), even though PETC or the effect of sampling is not considered. Indeed, in the continuous-time case, i.e., when τMASP approaches zero, $\tau _ { \mathrm { M I E T } } ^ { i } = \tau _ { \mathrm { m a x } ; } ^ { i }$ This statement underlines that, if faster sampling is used, the continuoustime ETC behavior is recovered in the proposed setup.
