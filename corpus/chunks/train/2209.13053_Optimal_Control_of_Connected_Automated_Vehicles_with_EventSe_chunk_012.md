$$b _ {3} (\mathbf {x} _ {i} (t)) = \nu_ {\max} - \nu_ {i} (t), \tag {16}b _ {4} (\mathbf {x} _ {i} (t)) = \nu_ {i} (t) - \nu_ {\min}. \tag {17}$$

This yields:

$$\underbrace {- 1} _ {L _ {g} b _ {3} (\mathbf {x} _ {i} (t))} u _ {i} (t) + \underbrace {k _ {3} \left(v _ {\max} - v _ {i} (t)\right)} _ {\gamma_ {3} \left(b _ {3} (\mathbf {x} _ {i} (t))\right)} \geq 0 \tag {18}\underbrace {1} _ {L _ {g} b _ {4} (\mathbf {x} _ {i} (t))} u _ {i} (t) + \underbrace {k _ {4} \left(v _ {i} (t) - v _ {\min}\right)} _ {\gamma_ {4} \left(b _ {4} (\mathbf {x} _ {i} (t))\right)} \geq 0, \tag {19}$$

for the maximum and minimum velocity constraints, respectively.

Inclusion of soft constraints in (8). As a last step in the OCBF approach, we can exploit the versatility of the CBF method to include soft constraints expressed as terminal state costs in (8), e.g., the CAV achieving a desired terminal speed. This is accomplished by using a Control Lyapunov Function (CLF) to track specific state variables in the reference trajectory if desired. A CLF $V ( \mathbf { x } _ { i } ( t ) )$ is similar to a CBF (see [22]). In our problem, letting $V ( \mathbf { x } _ { i } ( t ) ) = ( \nu _ { i } ( t ) - \nu _ { i } ^ { \mathrm { r e f } } ( t ) ) ^ { 2 }$ we can express the CLF constraint associated with tracking the CAV speed to a desired value $\nu _ { i } ^ { \mathrm { r e f } } ( t )$ (if one is provided) as follows:

$$L _ {f} V (\mathbf {x} _ {i} (t)) + L _ {g} V (\mathbf {x} _ {i} (t)) u _ {i} (t) + \epsilon V (\mathbf {x} _ {i} (t)) \leq e _ {i} (t), \tag {20}$$

where $\epsilon > 0$ and $e _ { i } ( t )$ makes this a soft constraint.

Now that all the original problem constraints have been transformed into CBF constraints, we can formulate the OCBF problem as follows:

$$\min _ {u _ {i} (t), e _ {i} (t)} J _ {i} \left(u _ {i} (t), e _ {i} (t)\right) := \int_ {t _ {i} ^ {0}} ^ {t _ {i} ^ {f}} \left[ \frac {1}{2} \left(u _ {i} (t) - u _ {i} ^ {\text { ref }} (t)\right) ^ {2} + \lambda e _ {i} ^ {2} (t) \right] d t \tag {21}$$

subject to vehicle dynamics (1), the CBF constraints (11), (15), (18), (19), the control constraint (6), and CLF constraint (20). Note that this is a decentralized optimization problem, as it only requires information sharing with a small number of “neighbor” CAVs, i.e. CAV $i _ { p }$ and $i _ { c }$ (if they exist). We denote this set of CAV neighbors by $\mathcal { R } _ { i } ( t )$ at time t :

$$\mathcal {R} _ {i} (t) = \{i _ {p} (t), i _ {c} (t) \}. \tag {22}$$
