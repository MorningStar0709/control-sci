We see that the mathematical model of the microactuator is third order: one first-order ODE for the comb circuit and one second-order ODE for the mechanical mass. The dynamic variables are the capacitor voltage $e _ { C }$ and drive-comb displacement x, and the system input variable is source voltage $e _ { \mathrm { i n } } ( t )$ .
