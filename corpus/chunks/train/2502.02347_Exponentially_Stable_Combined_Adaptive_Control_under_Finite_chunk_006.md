$$u = \mathbf {k} _ {x} ^ {T} \mathbf {x} + k _ {r} r - \boldsymbol {\theta} ^ {T} \phi (\mathbf {x}). \tag {6}$$

Let the tracking error denoted by e is defined as ${ \bf e } = { \bf x } - { \bf x } _ { r }$ , the tracking error dynamics using (3), (4) and (6),

$$\dot {\mathbf {e}} = \mathbf {A} _ {r} \mathbf {e} + \left(\mathbf {A} + \mathbf {b} k _ {p} \mathbf {k} _ {x} ^ {T} - \mathbf {A} _ {r}\right) \mathbf {x} + \left(\mathbf {b} k _ {p} k _ {r} - \mathbf {b} _ {r}\right) r. \tag {7}$$

The origin of the tracking error is exponentially stable when the controller gains $\mathbf { k } _ { x }$ and $k _ { r }$ satisfy the following matching condition,

$$\mathbf {A} + \mathbf {b} k _ {p} \mathbf {k} _ {x} ^ {T} = \mathbf {A} _ {r}, \quad \mathbf {b} k _ {p} k _ {r} = \mathbf {b} _ {r}. \tag {8}$$

However, A and $k _ { p }$ are unknown, the matching condition (8) would not generate the controller gains, and θ is also unknown, hence the control law (9) is modified as below,

$$u = \hat {\mathbf {k}} _ {x} ^ {T} \mathbf {x} + \hat {k} _ {r} r - \hat {\boldsymbol {\theta}} ^ {T} \boldsymbol {\phi} (\mathbf {x}), \tag {9}$$

where $\hat { \mathbf { k } } _ { x } , ~ \hat { k } _ { r } ,$ and $\hat { \pmb { \theta } }$ denote the estimate of $\mathbf { k } _ { x } , \ k _ { r } ,$ and $\theta ,$ respectively. Let the error in the estimation of the controller gains denoted by $\tilde { \mathbf { k } } _ { x } , ~ \tilde { k } _ { r }$ , and θ˜ are defined as $\tilde { \mathbf { k } } _ { x } = \hat { \mathbf { k } } _ { x } - \mathbf { k } _ { x } , \tilde { k } _ { r } = \hat { k } _ { r } - k _ { r }$ , and $\tilde { \pmb { \theta } } = \hat { \pmb { \theta } } - \pmb { \theta }$ , respectively. The tracking error dynamics e˙ in (7) is as follows after putting (9) in (3), and using (4) and (8),

$$\dot {\mathbf {e}} = \mathbf {A} _ {r} \mathbf {e} + \mathbf {b} k _ {p} \tilde {\mathbf {k}} _ {x} ^ {T} \mathbf {x} + \mathbf {b} k _ {p} \tilde {k} _ {r} r - \mathbf {b} k _ {p} \tilde {\boldsymbol {\theta}} ^ {T} \boldsymbol {\phi}. \tag {10}$$

The second control objective is the convergence of the controller gains’ estimate $\hat { \mathbf { k } } _ { x } , \hat { k } _ { r }$ , and θˆ to their ideal values $\mathbf { k } _ { x } , \ k _ { r } ,$ and $\theta ,$ respectively, under the finite excitation condition of the regressor vector,

$$\boldsymbol {\varphi} = \left[ \mathbf {x} ^ {T} u \boldsymbol {\phi} ^ {T} (\mathbf {x}) \right] ^ {T} \in \mathbb {R} ^ {n + 1 + p}. \tag {11}$$
