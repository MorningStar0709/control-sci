# 4.1. General Constrained DDP

The standard DDP method (Jacobson and Mayne, 1970) does not account for constraints, so, for the needs of the problem presented in this work, extensions of DDP, proposed by Murray and Yakowitz (1979) for linear constraints and by Yakowitz (1986) for nonlinear constraints, is utilized.

Consider a discrete-time optimal control problem with states 𝒙 and controls 𝒂 with the recursive Bellman equation $V ( x ( k ) , k ) = \operatorname* { m i n } \{ \phi ( x , a , k ) + V ( f ( x , a , k ) , k + 1 ) \} , { \mathrm { f o r ~ } } k = K - 1 , \ldots , 0 ,$ , where 𝑉 is the optimal cost function, 𝜙 is the objective function of the optimal control problem, 𝒇 is the right-hand side of the state equation, and the minimisation must be carried out for all feasible controls 𝒂(𝑘) that satisfy any inequality constraints present. The DDP procedure undertakes, at each time step of each iteration, a quadratic approximation of the term to be minimized in the recursive Bellman equation, i.e., of $\phi ( { \pmb x } , { \pmb a } , { \pmb k } ) + V ( { \pmb f } ( { \pmb x } , { \pmb a } , { \pmb k } ) , { \pmb k } + 1 )$ ). The quadratic approximation is taken around nominal trajectories $( \bar { \pmb x } ( k ) , \bar { \pmb a } ( k ) )$ , which are the initial trajectories of each iteration. Define $\delta { \pmb x } = { \pmb x } - { \bar { \pmb x } }$ and $\delta \pmb { a } = \pmb { a } - \bar { \pmb { a } }$ . The aim is to find the optimal control law for 𝛿𝒂, which minimizes the quadratic approximation, subject to some inequality constraints. The procedure of the DDP algorithm for each iteration is described in brief as follows, see (Yakowitz, 1986; Murray and Yakowitz, 1979) for more details.

For $k = K - 1 , \ldots , 0 .$ , the quadratic approximation $Q ( { \pmb x } , { \pmb a } , k )$ of the term $\phi ( { \pmb x } , { \pmb a } , { \pmb k } ) + V ( f ( { \pmb x } , { \pmb a } , { \pmb k } ) , { \pmb k } + 1 )$ around a nominal point (𝒙̄(𝑘), 𝒂̄(𝑘)), is required and can be expressed as follows (for simplicity, the indexes 𝑘 are omitted)

$$Q (\boldsymbol {x}, \boldsymbol {a}) = \frac {1}{2} \delta \boldsymbol {x} ^ {T} \boldsymbol {A} \delta \boldsymbol {x} + \delta \boldsymbol {x} ^ {T} \boldsymbol {B} \delta \boldsymbol {a} + \frac {1}{2} \delta \boldsymbol {a} ^ {T} \boldsymbol {C} \delta \boldsymbol {a} + \boldsymbol {D} ^ {T} \delta \boldsymbol {a} + \boldsymbol {E} ^ {T} \delta \boldsymbol {x} \tag {18}$$
