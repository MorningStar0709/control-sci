# A. Model and Problem Statement

We consider general nonlinear plants of the form

$$\dot {x} = f (x, u), \tag {20}$$

where $x \in \mathbb { R } ^ { n }$ is the state, $u \in \mathcal { L } _ { N } ^ { \infty }$ is a measurable and essentially bounded control input, and $f : \mathbb { R } ^ { n } \times \mathbb { R } ^ { N } \to \mathbb { R } ^ { n }$ is a continuous function that satisfies the following assumption:

Assumption 3: There exists a ${ \mathcal { C } } ^ { 1 }$ ℓ-Lipschitz function $h :$ $\mathbb { R } ^ { N } \to \bar { \mathbb { R } ^ { n } }$ and constants $\gamma , \ell > 0$ such that:

(a) $f ( h ( u ) , u ) = 0$ for all $u \in \mathbb { R } ^ { N }$ .   
(b) There exists $a > 0 , p \in ( 0 , 1 )$ , and $q > 1$ such that the following implication holds:

$$\left| x - h (\hat {u}) \right| \geq \gamma | u - \hat {u} | \Longrightarrow \tag {21}(x - h (\hat {u})) ^ {\top} f (x, u) \leq - \frac {a}{2} | x - h (\hat {u}) | ^ {2 p} - \frac {a}{2} | x - h (\hat {u}) | ^ {2 q},$$

for all $x \in \mathbb { R } ^ { n } , u \in \mathbb { R } ^ { N }$ , and $\boldsymbol { \hat { u } } \in \mathbb { R } ^ { N }$ .

Remark 5: In the literature, the mapping $h ( u )$ is traditionally referred to as the quasi-steady state mapping for the dynamics (20). It is standard in the feedback optimization literature to assume that this mapping exhibits appropriate stability properties uniformly in u, see, e.g [30], [36]. We impose a similar condition, but in the context of FxT-ISS. Essentially, item (b) of Assumption 3 asks that for each fixed reference input uˆ, the deviation $x - h ( \hat { u } )$ is FxT-ISS with respect to the deviation on the “input” $u - \hat { u }$ , which is established via the quadratic-like FxT ISS Lyapunov function $| x - h ( \hat { u } ) | ^ { 2 }$ . We require that this property holds uniformly in x, u, and uˆ. □

Example 1: One class of systems satisfying Assumption 3 are those of the following form

$$\dot {x} = - \frac {A _ {1} (x - h (u))}{| x - h (u) | ^ {\tilde {p}}} - \frac {A _ {2} (x - h (u))}{| x - h (u) | ^ {\tilde {q}}}, \tag {22}$$

where $A _ { 1 } , A _ { 2 } \ \in \ \mathbb { R } ^ { n \times n }$ with $A _ { 1 } , A _ { 2 } ~ \succ ~ 0 , ~ \tilde { p } ~ \in ~ ( 0 , 1 )$ , and $\tilde { q } < 0$ .
