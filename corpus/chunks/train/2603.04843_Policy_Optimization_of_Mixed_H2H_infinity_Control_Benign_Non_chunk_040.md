# 6.1 Numerical approaches and setup

We compare four approaches:

(1) Analytical solution: For the single-channel case, we solve the Riccati equation (22b) and obtain the optimal policy from (22a).   
(2) Policy iteration: Inspired by the optimality conditions in Corollaries 2 and 3, we apply iterative policy updates to solve (10) and its single-channel case. The details are given below.   
(3) LMI-based convex optimization: Solve the convex reformulation in (27) with the conic solver MOSEK [49].   
(4) HIFOO: A sophisticated nonsmooth optimization package (v3.501 with Hanso 2.01) for fixed-order $\mathcal { H } _ { 2 } / \mathcal { H } _ { \infty }$ controller synthesis [50].

Policy iteration. The method starts from a feasible policy and alternates between policy evaluation (solving a Riccati equation) and policy improvement. It can be viewed as a fixed-point iteration for solving a stationary point $\nabla J _ { \mathrm { m i x } } ( \cdot ) = \mathrm { \bar { 0 } }$ . In the single-channel case with $z _ { 2 } = z _ { \infty }$ , using (18), the update simplifies to:

$$K ^ {\prime} = - R ^ {- 1} B ^ {\mathsf {T}} P _ {K}, \tag {28a}$$

which is equivalent to the Gauss-Newton update: $K ^ { \prime } = K -$ $\eta R ^ { - 1 } \nabla J _ { \mathrm { m i x } } ( K ) \Lambda _ { K } ^ { - 1 }$ with $\eta = 1 / 2$ −. This update preserves feasibility, and its convergence is established in [24]. For the general two-channel setting, we adopt an analogous fixedpoint iteration based on (17):

$$K ^ {\prime} = - R _ {2} ^ {- 1} B ^ {\mathsf {T}} \Gamma_ {K} (I + \beta^ {- 2} \alpha^ {2} X _ {K} \Gamma_ {K}) ^ {- 1}. \tag {28b}$$

While a formal convergence analysis remains open, we conjecture that the method converges for sufficiently large $\beta ,$ and leave this as future work.

Setup. All experiments are conducted in MATLAB R2024b. The stabilizing Riccati solution is computed using MATLAB’s icare. Policy iteration is run until convergence $\| K ^ { \prime } - K \| < 1 0 ^ { - 5 }$ . For LMI-based methods, we use MOSEK with its default high-accuracy stopping criteria.

We compare these methods in terms of 1) the runtime complexity; 2) the square root of $J _ { \mathrm { m i x } }$ of the converged policy; 3) the resulting $\mathcal { H } _ { 2 }$ and $\mathcal { H } _ { \infty }$ norms.
