# IV. FULLY DISTRIBUTED CONSENSUS CONTROL FOR STOCHASTIC MASS

In this section, the unified fully distributed protocol (2) is specifically designed to solve the consensus problem of stochastic MASs for the cases of undirected and directed graphs, respectively.

As we all know that the algebraic Riccati equation is an important tool for the control design of linear deterministic systems [31], [32]. For linear stochastic systems, we introduce the following stochastic algebraic Riccati equation(SARE) to design the controller parameters

$$A ^ {T} P + P A - P B B ^ {T} P + C ^ {T} P C + I _ {n} = 0. \tag {9}$$

Note that the solvability of this SARE has been well investigated in [33], which demonstrates that the existence and uniqueness of the positive definite solution P is equivalent to that the corresponding stochastic system $d x ( t ) = [ A x ( t ) +$ $B u ( t ) ] d t + C x ( t ) d w ( t )$ is m.s. stabilizable. So in this work, SARE (9) is always assumed to have a positive-definite solution P .

A. Fully distributed consensus control for stochastic MASs with the directed topology

Lemma IV.1 ( [34]). Let $\tilde { \mathcal { G } }$ be a directed graph that contains a spanning tree. Then the node set V can be partitioned into two subsets Vl and $\mathcal { V } _ { f }$ satisfying the following properties:

(i) $\mathcal { V } _ { l } \cup \mathcal { V } _ { f } = \mathcal { V }$ and $\begin{array} { r } { \mathcal { V } _ { l } \cap \mathcal { V } _ { f } = \emptyset ; } \end{array}$   
(ii) the subgraph induced by Vl is strongly connected;   
(iii) there is no edge from any node in Vl to any node in $\mathcal { V } _ { f }$ .

By relabeling the nodes such that $\mathcal { V } _ { l } = \{ n _ { 1 } , . . . , n _ { M } \}$ and $\mathcal { V } _ { f } ~ = ~ \{ n _ { M + 1 } , . . . , n _ { N } \}$ with $1 \le M \le N$ , the Laplacian matrix $\dot { \boldsymbol { \mathcal { L } } } \in \mathbb { R } ^ { N \times N }$ of $\ddot { \mathcal { G } }$ can be written in the following block form:

$$
\mathcal {L} = \left[ \begin{array}{c c} \mathcal {L} _ {1 1} & 0 \\ \mathcal {L} _ {2 1} & \mathcal {L} _ {2 2} \end{array} \right],
$$

where $\mathcal { L } _ { 1 1 } \in \mathbb { R } ^ { M \times M }$ is the Laplacian matrix of the strongly connected subgraph induced by $\dot { \mathcal { V } } _ { l } , \mathcal { L } _ { 2 2 } \in \mathbb { R } ^ { ( N - M ) \times ( N - M ) } \dot { \mathrm { i s } }$ a nonsingular M-matrix, and $\mathcal { L } _ { 2 1 }$ represents the connectivity between $\mathcal { V } _ { l }$ and $\mathcal { V } _ { f }$ .
