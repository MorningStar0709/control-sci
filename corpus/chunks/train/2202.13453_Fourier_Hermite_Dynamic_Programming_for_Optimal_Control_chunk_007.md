# Algorithm 1 Differential dynamic programming

Input: Initial state ${ \hat { x } } _ { 1 } .$ , nominal control $\hat { u } _ { k }$ for $k = 1 , \ldots , T - 1 ;$ and nominal states $\hat { x } _ { k }$ for $k = 2 , \ldots , T$

Output: Updated $\hat { u } _ { k }$ and $\hat { x } _ { k }$

1: Backward pass:

2: Compute terminal cost, $\ell _ { T } ( \hat { x } _ { T } )$ and its derivatives $L _ { x } ( \hat { x } _ { T } )$ and $L _ { x x } ( \hat { x } _ { T } )$

3: $V _ { T } ^ { 0 }  \ell _ { T } ( \hat { x } _ { T } ) , v _ { T }  - L _ { x } ( \hat { x } _ { T } )$ , and $S _ { T }  L _ { x x } ( \hat { x } _ { T } )$

4: for $k = T - 1$ to 1 do

5: Evaluate the partial derivatives $L _ { x } , L _ { u } , L _ { x x } , L _ { u u } , L _ { x u }$ of $\ell _ { k }$ and $F _ { x } , F _ { u } , F _ { x x } , F _ { u u } , F _ { x u }$ of $f _ { k }$ at $( \hat { x } _ { k } , \hat { u } _ { k } )$ .

6: Compute the coefficients of $Q _ { k } ( x _ { k } , u _ { k } )$ using (8).

7: Compute d and K using (10), and $v _ { k }$ and $S _ { k }$ using (12).

8: end for

9: Forward pass:

10: Start from ${ \hat { x } } _ { 1 }$

11: for $k = 1$ to $T - 1$ do

12: $\hat { \boldsymbol u } _ { k } \gets \boldsymbol u _ { k } + \delta \boldsymbol u _ { k } .$ , where $\delta u _ { k }$ is given by (11).

13: $\hat { x } _ { k + 1 } \gets f _ { k } ( \hat { x } _ { k } , \hat { u } _ { k } )$

14: end for

15: Repeat from Step 1 until convergence.
