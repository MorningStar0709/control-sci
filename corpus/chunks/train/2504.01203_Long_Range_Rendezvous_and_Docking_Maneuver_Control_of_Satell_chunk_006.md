# B. Rigid Body Dynamics and Kinematics of Chaser Satellite

Let the position vector of the chaser satellite with respect to $E _ { c }$ be denoted by $\tilde { p } _ { c } .$ , whereas the translational and angular velocity of chaser satellite w.r.t $E _ { i }$ frame be denoted by $\tilde { v } _ { c }$ and $\tilde { \omega } _ { c } .$ , respectively. The kinematics and dynamics of the translational motion of the chaser satellite can be described as

$$M _ {c} \dot {\tilde {v}} _ {c} + M _ {c} \boldsymbol {\Omega} (\tilde {\omega} _ {c}) \tilde {v} _ {c} = \tilde {F} _ {d c} + \tilde {F} _ {j _ {2}, c} + \tilde {F} _ {c} \tag {5}\dot {\tilde {p}} _ {c} + \boldsymbol {\Omega} (\tilde {\omega} _ {c}) \tilde {p} _ {c} = \tilde {v} _ {c} \tag {6}$$

where $M _ { c }$ is the mass of the chaser satellite, $\tilde { F } _ { d c } , \tilde { F } _ { c }$ and $\tilde { F } _ { j _ { 2 } , c }$ are the external disturbance force, control input force and $J _ { 2 }$ perturbation force acting on the chaser satellite. The attitude dynamics and kinematics of the chaser satellite are given by:

$$\mathbf {J} _ {c} \dot {\tilde {\omega}} _ {c} = - \boldsymbol {\Omega} (\tilde {\omega} _ {c}) \mathbf {J} _ {c} \tilde {\omega} _ {c} + \tilde {T} _ {d c} + T _ {g c} + \tilde {T} _ {c} \tag {7}$$

where $\mathbf { J _ { c } }$ represents the inertia matrix of the chaser satellite, $\tilde { T } _ { d c } , \tilde { T } _ { c }$ and $\tilde { T } _ { g c }$ are the external disturbance torque, control input torque and gravity-gradient torque acting on the chaser satellite [18]. The attitude kinematics of the chaser satellite is given below:

$$\dot {\boldsymbol {B}} _ {c} ^ {i} = \boldsymbol {B} _ {c} ^ {i} \boldsymbol {\Omega} (\tilde {\omega} _ {c}) \tag {8}$$

where $B _ { c } ^ { i } \in \ S O ( 3 )$ is the rotation matrix which transforms from the chaser body-fixed frame to ICRF frame.
