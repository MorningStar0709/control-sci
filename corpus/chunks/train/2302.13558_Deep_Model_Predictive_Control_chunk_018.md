# A Deep Neural Network

Any continuous function h on a compact set $\mathcal { X }$ can be approximated by a multi-layer network with number of layers $L \geqslant 2$ such that

$$h (x) = W _ {L} ^ {\top} \psi_ {L} \left[ W _ {L - 1} ^ {\top} \psi_ {L - 1} [ \dots [ \psi_ {1} (x) ] ] \right] + \varepsilon^ {*} (x), \tag {16}$$

where $x \in \mathcal { X } , \psi _ { i } , W _ { i }$ for $i = 1 , \ldots , L ,$ , are activation functions and ideal weights, respectively, in the $i ^ { \mathrm { { t h } } }$ layer. The reconstruction error function $\varepsilon ^ { * }$ is bounded by a known constant $\bar { \varepsilon } ^ { * } > 0$ for each $x \in \mathcal { X } , \mathrm { i . } \mathrm { e . } \ \| \varepsilon ^ { * } ( x ) \| \leqslant \bar { \varepsilon } ^ { * }$ . Therefore, we can represent $h ( x _ { t } )$ with the help of a neural network with a desired accuracy. If the neural network is not minimal then the ideal weights may not be unique. However, for the neural-adaptive controller design only the existence of ideal weights is assumed, which is always guaranteed when h is a continuous function on a compact set $[ \bar { 3 7 } , \ \ S 7 . 1 ]$ . Let us define $\phi ^ { \ast } ( x ) : = \psi _ { L } \left[ W _ { L - 1 } ^ { \top } \psi _ { L - 1 } \left[ \cdot \cdot \cdot \left[ \psi _ { 1 } \left( x \right) \right] \right] \right]$ as the output of the last activation layer under the ideal weights of hidden layers and $W ^ { \ast } : = W _ { L } \in \mathbb { R } ^ { ( n _ { L } + 1 ) \times m }$ be the ideal weights of the output layer, then

$$h (x _ {t}) = W ^ {* \top} \phi^ {*} (x _ {t}) + \varepsilon^ {*} (x _ {t}). \tag {17}$$

There are $n _ { L }$ number of neurons in the output layer. The first row of $W ^ { * }$ represents the bias term and the first element of $\phi ^ { * } \in \mathbb { R } ^ { n _ { L } + 1 }$ is 1. The ideal hidden layer weights defining $\phi ^ { \ast } ( \cdot )$ are neither known nor unique.
