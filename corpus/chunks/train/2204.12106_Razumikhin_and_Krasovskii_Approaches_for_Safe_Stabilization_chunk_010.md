# 3.1. Control Lyapunov-Razumikhin Functions

We first recall the classic control Lyapunov-Razumikhin function.

Definition 2 ([4, 5]). For the system (1), a function $V _ { \mathsf { c } } \quad \in$ $C ( \mathbb { R } ^ { n } , \mathbb { R } ^ { + } )$ is called a control Lyapunov-Razumikhin function (CLRF-I), if

(i) there exist $\alpha _ { 1 } , \alpha _ { 2 } \in { \mathcal { K } } _ { \infty }$ such that, for all $x \in \mathbb { R } ^ { n } , \alpha _ { 1 } ( | x | ) \leq$ $V _ { \mathtt { C } } ( x ) \leq \alpha _ { 2 } ( | x | ) ;$ ;   
(ii) there exist $\gamma _ { \mathsf { c } } , \rho \in \mathcal { K }$ with $\rho > \mathrm { I d }$ such that, for all $\phi \in { }$ $C ( [ - \Delta , 0 ] , \mathbb { R } ^ { n } )$ with $\phi ( 0 ) = x , \mathrm { i f } \rho ( V _ { \mathsf { c } } ( x ) ) \geq V _ { \mathsf { c } } ( \phi ( \theta ) )$ for all $\theta \in [ - \Delta , 0 ]$ , then

$$\inf _ {u \in \mathbb {U}} \left\{L _ {f} V _ {\mathrm{c}} (\phi) + L _ {g} V _ {\mathrm{c}} (\phi) u \right\} \leq - \gamma_ {\mathrm{c}} (V _ {\mathrm{c}} (x)), \tag {2}$$

$\begin{array} { r l } & { \mathrm { w h e r e } \ L _ { f } V _ { \mathbb { C } } ( \phi ) : = \frac { \partial V _ { \mathbb { C } } ( x ) } { \partial x } f ( \phi ) \mathrm { a n d } \ L _ { g } V _ { \mathbb { C } } ( \phi ) : = \frac { \partial V _ { \mathbb { C } } ( x ) } { \partial x } g ( \phi ) . } \end{array}$

In Definition 2, the subscript $\cdot _ { \mathsf { C } } ,$ is to show that the CLRF is ‘classic’, which is to distinguish with the CLRF proposed afterwards. In (2), $L _ { f } V _ { \mathsf { c } } ( \phi ) + L _ { g } V _ { \mathsf { c } } ( \phi ) u$ is an alternative form of the upper Dini derivative of $V _ { \mathsf { c } } ,$ and here the upper Dini derivative equals to the usual derivative. The effects of time delays are shown via the Razumikhin condition: $\rho ( V _ { \mathsf { c } } ( x ) ) \geq V _ { \mathsf { c } } ( \phi ( \theta ) )$ for all $\theta \in [ - \Delta , 0 ]$ , which can be written equivalently as $\rho ( V _ { \mathsf { c } } ( x ) ) \geq$ $\| V _ { \mathsf { c } } ( \phi ) \| ;$ ; see [5, 20]. If $L _ { g } V _ { \mathsf { c } } ( \phi ) = 0$ , then Definition 2 is reduced to the one in [4].

In the delay-free case, the existence of the continuous controller is verified via the SCP [3]. However, in the time-delay case, whether the Razumikhin condition is satisfied results in additional difficulties in the controller design, and the SCP is unavailable.
