# 11.3 Environment Layer: Distribution Grid Topology

The environment is a graph $G = ( V , E )$ where V are transformers/feeders and E represent distribution lines. Each node $j$ has capacity $C _ { j }$ and a set of connected consumers. Power flows are represented in simplified form through node-level loads $L _ { j , t } ;$ full AC power flow equations are not needed for load-balanced demand response studies.
