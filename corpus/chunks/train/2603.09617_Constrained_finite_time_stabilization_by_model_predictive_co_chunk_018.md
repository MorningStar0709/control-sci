# D. Extension to constrained nonlinear systems

In Section II, it is assumed that the nonlinear system (13) is feedback linearizable in a compact set $\mathcal { D }$ containing the origin. Here, we suppose that the nonlinear transformation (or nonlinear diffeomorphism) is $z = M ( x )$ , and the corresponding control is $u = \gamma ( x , \nu )$ , such that the linearized system can be calculated by

$$z (k + 1) = G z (k) + F v (k),$$

where $( G , F )$ is controllable. For detailed criteria for feedback linearization of discrete-time nonlinear systems, Please see [26].

The optimization for the nonlinear finite-time MPC can be formulated by

$$
\begin{array}{l} [ U ^ {*} (k), X ^ {*} (k) ] = \arg \min _ {U (k), X (k)} \sum_ {i = n} ^ {N - 1} \left(\| x (i | k) \| _ {Q} ^ {2} + \| u (i | k) \| _ {R} ^ {2}\right) \\ + \| x (N | k) \| _ {P} ^ {2}, \tag {41} \\ \end{array}
$$

subject to the initial constraint (6) and

$$x (i + 1 | k) = f (x (i | k), u (i | k)), \quad \text { for } i = 0, 1, \dots , N - 1, \tag {42}u (i | k) \in \mathcal {U}, \text { for } i = 0, 1, \dots , N - 1, \tag {43}x (i | k) \in \mathcal {X} \cap \mathcal {D}, \text { for } i = 1, \dots , N, \tag {44}x (N | k) \in \mathcal {X} _ {f}, \tag {45}$$

where $\mathcal { X } _ { f } \subset \mathcal { X } \cap \mathcal { D }$ is the terminal invariant set, and there exists $u = - K x$ fulfilling

$$- K x \in \mathcal {U}, f (x, - K x) \in \mathscr {X} _ {f}, \forall x \in \mathscr {X} _ {f}.$$

The matrix P is solved from Lyapunov equation (10) by using Jacobian matrices:

$$A = \left. \frac {\partial f}{\partial x} \right| _ {x = 0, u = 0}, b = \left. \frac {\partial f}{\partial u} \right| _ {x = 0, u = 0}.$$

The constrained nonlinear finite-time MPC is implemented by

$$u (k) = [ 1, 0, \dots , 0 ] _ {1 \times N} U ^ {*} (k), \tag {46}$$

where $U ^ { * } ( k )$ is solved from nonlinear optimization (41).

Theorem 4: Consider the nonlinear plant (13) subject to constraints (4). If the optimization (41)–(45) is feasible initially, and the implementation (46) is applied, then

1) the optimization (41)–(45) is feasible recursively;   
2) a finite time $T > 0$ exists such that $x ( k ) = 0$ for all $k > T$ .

Proof: 1) By using the tail method, it is straightforward to prove the recursive feasibility of optimization.

2) It is also straightforward to claim $x ^ { * } ( n | k )$ converges to the origin as $k \to + \infty$ by using the optimal cost function

$$J _ {n} ^ {*} (k) = \sum_ {i = n} ^ {N - 1} \left(\| x ^ {*} (i | k) \| _ {Q} ^ {2} + \| u ^ {*} (i | k) \| _ {R} ^ {2}\right) + \| x ^ {*} (N | k) \| _ {P} ^ {2}$$
