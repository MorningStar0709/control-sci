$$
\begin{array}{l} \mathrm{KL} \left[ \prod_ {k = 1} ^ {N} q \left(\theta_ {k}\right) | | p (\Lambda | Z) \right] = \int \prod_ {k = 1} ^ {N} q \left(\theta_ {k}\right) \log \left(\frac {\prod_ {k = 1} ^ {N} q \left(\theta_ {k}\right)}{p (\Lambda | Z)}\right) d \Lambda \\ = - \mathcal {L} \left(\prod_ {k = 1} ^ {N} q (\theta_ {k})\right) + \log p (Z), \tag {4} \\ \end{array}
$$

where $\begin{array} { r l } { \mathcal { L } \left( \prod _ { k = 1 } ^ { N } q ( \theta _ { k } ) \right) } & { { } } \end{array}$ is the evidence lower bound, formulated as

$$\mathcal {L} \left(\prod_ {k = 1} ^ {N} q (\theta_ {k})\right) = \int \log p (\Lambda , Z) \prod_ {k = 1} ^ {N} q (\theta_ {k}) d \theta_ {k} \tag {5}- \sum_ {k = 1} ^ {N} \int q (\theta_ {k}) \log q (\theta_ {k}) d \theta_ {k}.$$

Based on the variational inference theory, minimizing the $\mathrm { K L }$ divergence (4) is equivalent to maximizing the evidence lower bound (5), which yields the optimal solution

$$q ^ {*} (\theta_ {k}) = \arg \max _ {q (\theta_ {k})} \mathcal {L} \left(\prod_ {k = 1} ^ {N} q (\theta_ {k})\right). \tag {6}$$

This optimal solution is calculated as [24]

$$\log q ^ {*} (\theta_ {k}) = E _ {\Lambda^ {- \theta_ {k}}} ^ {*} \left\{\log p (\Lambda , Z) \right\} + c _ {\theta_ {k}}, \tag {7}$$

where $\Lambda ^ { - \theta _ { k } }$ denotes the set of all unknown variables Λ except $\theta _ { k }$ , and $c _ { \theta _ { k } }$ is a constant independent of $\theta _ { k }$ . When these variational parameters are coupled, the fixed-point iteration method [25] can be adopted to obtain an approximate optimal solution for (4) as

$$\log q ^ {i + 1} (\theta_ {k}) = E _ {\Lambda^ {- \theta_ {k}}} ^ {i} \{\log p (\Lambda , Z) \} + c _ {\theta_ {k}}, \tag {8}$$

where i denotes the n-th iteration.
