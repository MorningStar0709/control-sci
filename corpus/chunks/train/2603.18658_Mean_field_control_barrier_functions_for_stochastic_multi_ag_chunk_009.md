# III. KERNEL-WEIGHTED OVERLAP AS MEAN-FIELDCONTROL BARRIER FUNCTION

The MF-CBF framework developed in Section II applies to any continuously differentiable functional ${ \mathcal { H } } ;$ in the remainder of this paper, we adopt a specific construction based on kernel-weighted overlaps, which provides a natural and computationally tractable way to quantify the interaction between the population density and dangerous regions. Specifically, we define a kernel-weighted overlap between probability measures and show that bounding this quantity suffices to limit the mass of the population density inside a dangerous region. To this end, assume that $O \subset \Omega$ is a dangerous region and $\rho _ { O }$ is a probability measure on O. We call $O ^ { c }$ the complement of O w.r.t. Ω. The kernel-weighted overlap between probability measures $\rho$ and $\rho _ { O }$ is defined as

$$R _ {k} (\boldsymbol {\rho}, \boldsymbol {\rho} _ {O}) = \iint_ {\Omega \times \Omega} k (\mathbf {x}, \mathbf {y}) \boldsymbol {\rho} (\mathbf {x}) \boldsymbol {\rho} _ {O} (\mathbf {y}) d \mathbf {x} d \mathbf {y}, \tag {21}$$

where $k ( \mathbf { x } , \mathbf { y } )$ is a positive-definite kernel [22] measuring some type of similarity between states $\mathbf { x } , \mathbf { y } . \mathbf { A }$ typical choice is to use the Gaussian kernel $\begin{array} { r } { k ( \mathbf { x } , \mathbf { y } ) = \mathrm { e x p } \left( - \frac { \| \mathbf { \bar { x } } - \mathbf { y } \| ^ { 2 } } { 2 \sigma ^ { 2 } } \right) } \end{array}$

Furthermore, consider the quantity

$$B (\mu) = \inf _ {\int_ {O} \rho (\mathbf {x}) \mathrm{d} \mathbf {x} = \mu} R _ {k} (\rho , \rho_ {O}), \tag {22}$$

where $\mu \in [ 0 , 1 ] . \ B ( \mu )$ represents the minimum kernelweighted overlap achievable by any density that places exactly mass $\mu$ inside O. In what follows, we prove that B is a strictly increasing function, so that the inequality $B ( \mu ) < \varepsilon$ guarantees that $\textstyle \int _ { O } \rho ( \mathbf { x } ) d x < \mu$ for a suitably chosen ε. Before we proceed, we introduce the notation

$$C (\mathbf {x}) = \int_ {\Omega} k (\mathbf {x}, \mathbf {y}) \rho_ {O} (\mathbf {y}) \mathrm{d} \mathbf {y}, \quad \mathbf {x} \in \Omega . \tag {23}$$

Theorem 3. The function B admits the explicit representation

$$B (\mu) = \mu \cdot \inf _ {\mathbf {x} \in O} C (\mathbf {x}) + (1 - \mu) \inf _ {\mathbf {y} \in O ^ {c}} C (\mathbf {y}), \quad 0 \leq \mu \leq 1. \tag {24}$$
