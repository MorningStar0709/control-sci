# C.2 Proof of Lemma 4.2

Consider any $t \geq h$ . By the α0-strong convexity of $f _ { t ; h } ( \cdot )$ shown in Lemma 4.1, we know from Definition 3.1 that

$$f _ {t; h} (N _ {t}) - f _ {t; h} (N ^ {\star}) \leq \nabla f _ {t; h} (N _ {t}) ^ {\top} (N _ {t} - N ^ {\star}) - \frac {\alpha_ {0}}{2} \| N _ {t} - N ^ {\star} \| ^ {2}. \tag {C.1}$$

Next, we aim to provide an upper bound on $f _ { t ; h } ( N _ { t } ) ^ { \top } ( N _ { t } - N ^ { \star } )$ . We first obtain that

$$
\begin{array}{l} \left\| N _ {t + 1} - N ^ {\star} \right\| ^ {2} \leq \left\| N _ {t} - N ^ {\star} - \eta_ {t} \nabla f _ {t} (N _ {t}) \right\| \\ = \| N _ {t} - N ^ {\star} \| ^ {2} + \eta_ {t} ^ {2} \| \nabla f _ {t} (N _ {t}) \| ^ {2} - 2 \eta_ {t} \nabla f _ {t} (N _ {t}) ^ {\top} (N _ {t} - N ^ {\star}), \tag {C.2} \\ \end{array}
$$

where the inequality follows from the fact that $N _ { t + 1 } = \Pi _ { K _ { N } } ( N _ { t } - \eta _ { t } \nabla f _ { t } ( N _ { t } ) )$ and [67, Proposition 2.2]. It then follows from (C.2) that

$$2 \nabla f _ {t} (N _ {t}) ^ {\top} (N _ {t} - N ^ {\star}) \leq \frac {\| N _ {t} - N ^ {\star} \| ^ {2} - \| N _ {t + 1} - N ^ {\star} \| ^ {2}}{\eta_ {t}} + \eta_ {t} \| \nabla f _ {t} (N _ {t}) \| ^ {2}.$$

Noting that $\nabla f _ { t } ( N _ { t } ) = \nabla f _ { t ; h } ( N _ { t } ) + \varepsilon _ { t } ^ { s }$ , we have

$$\langle \nabla f _ {t} (N _ {t}), N _ {t} - N ^ {\star} \rangle = \langle \nabla f _ {t; h} (N _ {t}), N _ {t} - N ^ {\star} \rangle + \langle \varepsilon_ {t} ^ {s}, N _ {t} - N ^ {\star} \rangle .$$

$$\nabla f _ {t; h} (N _ {t}) ^ {\top} (N _ {t} - N ^ {\star}) \leq \frac {\| N _ {t} - N ^ {\star} \| ^ {2} - \| N _ {t + 1} - N ^ {\star} \| ^ {2}}{2 \eta_ {t}} + \frac {\eta_ {t}}{2} \| \nabla f _ {t} (N _ {t}) \| ^ {2} - \langle \varepsilon_ {t} ^ {s}, N _ {t} - N ^ {\star} \rangle . \tag {C.3}$$
