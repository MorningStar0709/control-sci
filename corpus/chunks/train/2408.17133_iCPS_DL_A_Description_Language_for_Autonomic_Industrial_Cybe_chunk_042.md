$$\left\{\mathsf {p} _ {\mathrm{i}}: \mathsf {t} \mid i \in I \right\}, \left\{\mathsf {p} _ {\mathrm{j}}: T _ {j} \mid j \in J \right\} \vdash G.$$

The inductive hypothesis ensures,

$$
\begin{array}{l} \left\{\mathrm{p} _ {\mathrm{i}}: \mathrm{t} \mid i \in I \right\}, \left\{\mathrm{p} _ {\mathrm{j}}: T _ {j} \mid j \in J \right\} \vdash G \\ \in O (| \{\mathsf {p} _ {\mathrm{i}}: \mathsf {t} \mid i \in I \}, \{\mathsf {p} _ {\mathrm{j}}: T _ {j} \mid j \in J \} |). \\ \end{array}
$$

Moreover,

$$
\begin{array}{l} \left| \left\{\mathrm{p} _ {\mathrm{i}}: \text { end } \mid i \in I \right\}, \left\{\mathrm{p} _ {\mathrm{j}}: \text { loop   t. } T _ {j} \mid j \in J \right\} \right| \\ = | \{\mathsf {p} _ {\mathrm{i}}: \mathsf {t} \mid i \in I \}, \{\mathsf {p} _ {\mathrm{j}}: T _ {j} \mid j \in J \} |, \\ \end{array}
$$

concluding that

$$
\begin{array}{l} \left\{\mathrm{p} _ {\mathrm{i}}: \text { e   n   d } \mid i \in I \right\}, \left\{\mathrm{p} _ {\mathrm{j}}: \text { l   o   o   p   t   . } T _ {j} \mid j \in J \right\} \vdash \text { l   o   o   p   t   . } G \\ \in O (| \{\mathsf {p} _ {\mathrm{i}}: \text { end } \mid i \in I \}, \{\mathsf {p} _ {\mathrm{j}}: \text { loop   t. } T _ {j} \mid j \in J \} \vdash \text { loop   t. } G |). \\ \end{array}
$$

The inductive step concludes the proof.
