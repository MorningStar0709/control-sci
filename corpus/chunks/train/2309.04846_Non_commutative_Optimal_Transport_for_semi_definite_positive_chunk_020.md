# 3.1 Weak duality

Let $\mathcal { H } _ { 1 } , ~ \mathcal { H } _ { 2 }$ be finite-dimensional Hilbert spaces with dimensions $d _ { 1 }$ and $d _ { 2 }$ , respectively. Consider the dual functional $\mathcal { D } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } : \mathrm { H } ( \mathcal { H } _ { 1 } ) \times \mathrm { H } ( \mathcal { H } _ { 2 } )  \mathbb { R }$ defined by

$$\mathcal {D} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (U, V) = - \tau_ {1} \operatorname{Tr} [ (\exp \left(- \frac {U}{\tau_ {1}}\right) - \mathbb {I} _ {d}) \rho ] - \tau_ {2} \operatorname{Tr} [ (\exp \left(- \frac {V}{\tau_ {2}}\right) - \mathbb {I} _ {d}) \sigma ] - \varepsilon \operatorname{Tr} [ \exp \left(\frac {U \oplus V - C}{\varepsilon}\right) ], \tag {3.3}$$

where $U \oplus V = U \otimes \mathbb { I } _ { d } + \mathbb { I } _ { d } \otimes V$ denotes the Kronecker sum of U and V .

In the following, similarly to the Proposition 2.3 for the classical case, we will derive the noncommutative weak duality formulation of (3.2) using the Fenchel-Young inequality, which we will restrict to Hermitian elements $\mathrm { H } ( \mathcal { H } ) \subset \mathrm { B } ( \mathcal { H } )$ , instead of considering compact and trace class operators, which in the finite-dimensional case are the same as B(H).
