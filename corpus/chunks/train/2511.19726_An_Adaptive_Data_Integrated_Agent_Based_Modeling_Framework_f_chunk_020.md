# 6.2 Control (Policy) Search

Control or policy search treats the MAS as a noisy black-box mapping $P \mapsto J ( P ; L )$ [26, 43, 56]. An external optimizer updates $P _ { t }$ based on performance estimates. A simple hill-climbing algorithm explores a neighborhood of $P _ { t }$ and moves to candidates with higher J if improvements exceed a tolerance. More sophisticated search procedures (e.g. evolutionary algorithms, Bayesian optimization, policy gradient methods) can be plugged into the same architecture.
