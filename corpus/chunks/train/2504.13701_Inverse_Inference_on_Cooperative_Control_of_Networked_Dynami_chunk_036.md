$$\sum_ {j = 1} ^ {N} Q _ {i j} = \mathbf {0}, \quad \sum_ {j = 1} ^ {N} P _ {i j} = \mathbf {0}, \forall i \in \mathcal {V}. \tag {77}$$

Proof. The proof is based on the the requirement of consensus (synchronization) on the NDS when the time goes to infinity. Let $x _ { c } \in \mathbb { R } ^ { n }$ represent the synchronized state vector for all nodes. Notice that if the consensus is achieved, the global control input u is zero regardless the value of $x _ { c } ,$ i.e.,

$$u = \tilde {K} (\mathbf {1} _ {N} \otimes x _ {c}) = (L \mathbf {1} _ {N} \otimes B K x _ {c}) = \mathbf {0}. \tag {78}$$

Based on this fact, multiplying the global ARE (76) with $\left( \mathbf { 1 } _ { N } \otimes x _ { c } \right)$ yields that

$$
\begin{array}{l} (\tilde {A} ^ {\intercal} P + P \tilde {A} - P \tilde {B} R ^ {- 1} \tilde {B} ^ {\intercal} P + Q) (\mathbf {1} _ {N} \otimes x _ {c}) \\ = (\tilde {A} ^ {\intercal} P + P \tilde {A} + Q) \left(\mathbf {1} _ {N} \otimes x _ {c}\right) = \mathbf {0} \\ \Leftrightarrow \left(\sum_ {j = 1} ^ {N} a _ {j i} \sum_ {\ell = 1} ^ {N} P _ {j \ell} + \sum_ {j = 1} ^ {N} P _ {i j} A + \sum_ {j = 1} ^ {N} Q _ {i j}\right) x _ {c} = \mathbf {0}, \forall i \in \mathcal {V} \\ \Rightarrow \sum_ {j = 1} ^ {N} \left(a _ {j i} \sum_ {\ell = 1} ^ {N} P _ {j \ell}\right) + \left(\sum_ {j = 1} ^ {N} P _ {i j}\right) A + \sum_ {j = 1} ^ {N} Q _ {i j} = \mathbf {0}, \forall i \in \mathcal {V}. \tag {79} \\ \end{array}
$$

where $P \tilde { B } R ^ { - 1 } \tilde { B } ^ { \boldsymbol { \mathsf { T } } } P = P \tilde { B } \tilde { K }$ is applied in the first equality, and the last row is derived due to the arbitrariness of $x _ { c } .$ . Next, note that the condition (79) should also hold regardless of the value of A. Therefore, when A = 0, we obtain

$$\sum_ {j = 1} ^ {N} Q _ {i j} = \mathbf {0}, \forall i \in \mathcal {V}. \tag {80}$$

Then, substituting (80) into (79) yields that

$$\sum_ {j = 1} ^ {N} \left(a _ {j i} \sum_ {\ell = 1} ^ {N} P _ {j \ell}\right) + \left(\sum_ {j = 1} ^ {N} P _ {i j}\right) A = \mathbf {0} \Rightarrow \sum_ {j = 1} ^ {N} P _ {i j} = \mathbf {0}, \tag {81}$$

where the arbitrariness of all columns in A is used for the derivation. The proof is completed. □
