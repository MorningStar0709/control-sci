$$\sup _ {\mathbf {u} \in \mathcal {U}} \left[ \sum_ {i \in \mathcal {V}} \left(w _ {r} \sum_ {k \in \mathcal {B} _ {i} (t)} \hat {E} _ {k} (t) \frac {\partial h _ {k}}{\partial x _ {i}} \right. \right. \tag {19a}\left. \left. + w _ {c} \sum_ {j \in \mathcal {N} _ {i} (t)} E _ {i j} ^ {c} (\mathbf {x}) \frac {\partial h _ {i j} ^ {\mathrm{col}}}{\partial x _ {i}}\right) \dot {x} _ {i} \right] =\sum_ {i \in \mathcal {V}} \sup _ {u _ {i} \in \mathcal {U} _ {i}} \left[ \left(w _ {r} \sum_ {k \in \mathcal {B} _ {i} (t)} \hat {E} _ {k} (t) \frac {\partial h _ {k}}{\partial x _ {i}} \right. \right. \tag {19b}\left. \left. + w _ {c} \sum_ {j \in \mathcal {N} _ {i} (t)} E _ {i j} ^ {c} (\mathbf {x}) \frac {\partial h _ {i j} ^ {\mathrm{col}}}{\partial x _ {i}}\right) \dot {x} _ {i} \right] \geq - \sum_ {i \in \mathcal {V}} \alpha \left(\phi_ {i} (\mathbf {x}, \mathbf {w})\right)\geq - \alpha (\phi (\mathbf {x}, \mathbf {w})), \tag {19c}$$

which completes the proof.

Lemma 3 shows that each robot $i \in \mathcal V$ satisfying its own constraint (15) leads to all robots collectively satisfying (12) with $h _ { k } ( \mathbf { x } ( t ) )$ ) being replaced by $\hat { h } _ { k } ( t )$ , regardless of whether $\hat { h } _ { k } ( t ) \ = \ h _ { k } ( \mathbf { x } ( t ) )$ ) or not. We also highlight that all the components of (15) for robot $i \in \mathcal V$ can be derived only using its local information. Using this fact, we now design our distributed controller.
