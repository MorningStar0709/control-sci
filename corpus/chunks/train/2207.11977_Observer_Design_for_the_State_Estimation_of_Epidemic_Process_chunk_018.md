Corollary 1.1. The ARI (17) holds if and only if there exist positive definite matrices $P , Q \in \mathbb { R } ^ { n _ { x } \times n _ { x } } , K \in \mathbb { R } ^ { k \times n _ { y } }$ , and $R , S \in \mathbb R ^ { n _ { x } \times n _ { y } }$ such that the following LMIs are feasible

$$
\left[ \begin{array}{c c} \operatorname{sym} (P A - R C A - S C) + Q & (P - R C) G \\ G ^ {\mathrm{T}} (P - R C) ^ {\mathrm{T}} & - I _ {n _ {f}} \end{array} \right] <   0 \tag {19a}

\left[ \begin{array}{c c} - Q & (H - K C) ^ {\mathrm{T}} \\ H - K C & - \frac {1}{\ell^ {2}} I _ {k} \end{array} \right] \leq 0 \tag {19b}
$$

where $\operatorname { s y m } ( X ) = X + X ^ { \mathtt { T } } .$ , and $J = P ^ { - 1 } S$ and $L = P ^ { - 1 } R$ .

Proof: Assume that (19) holds. Then, by Schur complement lemma [55, Chapter 1], (19b) is equivalent to

$$\ell^ {2} (H - K C) ^ {\mathrm{T}} (H - K C) \leq Q.$$

Thus, if

$$
\begin{array}{l} (A - L C A - J C) ^ {\mathrm{T}} P + P (A - L C A - J C) \\ + P (I - L C) G G ^ {\mathrm{T}} (I - L C) ^ {\mathrm{T}} P + Q <   0 \\ \end{array}
$$

then (17) holds. Substituting $R = P L$ and $S = P J$ , we can write

$$\operatorname{sym} (P A - R C A - S C) + (P - R C) G G ^ {\mathrm{T}} (P - R C) ^ {\mathrm{T}} + Q < 0. \tag {20}$$

Again, by Schur complement lemma, (20) is equivalent to (19a). Thus, if (19) holds, then (17) is satisfied.

For proving the necessity, assume (19a) holds but not (19b), then $\ell ^ { \bar { 2 } } ( H - \bar { K } C ) ^ { \mathrm { T } } ( H - \bar { K } C ) > Q$ . Thus, even if (19a) holds, (17) cannot be guaranteed. Second, assume (19b) holds but not (19a), then, even for $\ell ^ { 2 } ( H - K C ) ^ { \mathrm { { T } } } ( H - K C ) = Q .$ , (20) doesn’t hold. Thus, (17) doesn’t hold either.

The LMI condition (19) is less restrictive than the design criteria of Luenberger-like and Arcak-Kokotovi´c observers presented in the previous section. However, using the generalized Lipschitz condition, we can further reduce the conservativeness of the above design criterion.
