# A. Contraction via full state feedback

We begin with an LMI-based design for a full-state feedback controller.

Proposition 27 (Contraction via state feedback controller): Consider a plant described by an FRNN (18) with a MONE nonlinearity, with access to the full state, i.e. $C = I _ { n }$ . For the control law $u = K x ,$ for $K \in \mathbb { R } ^ { m \times n }$ , the following hold:

(i) The closed-loop system is an FRNN with state x, and the closed-loop weight matrix is $W _ { c l } = W + B K$ .   
(ii) The closed loop system is S-contracting for a given contraction rate $c \in ( 0 , 1 ]$ if and only if there exist a positive definite $\ b { X } \in \mathbb { R } ^ { n \times n }$ , a positive diagonal matrix D, and a matrix $Y \in \mathbb { R } ^ { m \times n }$ satisfying the LMI:

$$
\left[ \begin{array}{c c} - 2 (1 - c) X & D + X W ^ {\top} + Y ^ {\top} B ^ {\top} \\ D + W X + B Y & - 2 D \end{array} \right] \preceq 0. \tag {52}
$$

If (52) holds, the state feedback gain guaranteeing contraction is given by $K = Y X ^ { - 1 }$ .

Proof: The proof is presented in Appendix III-A

Corollary 28 (Necessity of linear stabilizability): An FRNN with synaptic matrix W , and input matrix B can be rendered S-contracting via full-state feedback with some $c > 0$ only if $( W - I _ { n } , B )$ is stabilizable.

Proof: Pre- and post-multiplying (52) by $\begin{array} { r l } { [ I _ { n }  } & { { } I _ { n } ] } \end{array}$ and its transpose, respectively, and simplifying yields

$$W X + X W ^ {\top} - 2 X + B Y + Y ^ {\top} B ^ {\top} + 2 c X \preceq 0.$$

Using the fact that $c > 0 .$ , and $Y = K X$ , we obtain

$$(W - I _ {n} + B K) X + X (W - I _ {n} + B K) ^ {\top} \prec 0.$$

which is the condition for stabilizability of $\left( W - I _ { n } , B \right)$ .

Similar to the problem of control design for linear systems, we may also apply the projection lemma [6, Section 2.6.2].

Corollary 29 (Necessary and sufficient conditions for S– contraction via full state feedback): Let $\Pi _ { B }$ denote a matrix whose columns form a basis for the null space of $B ^ { \top }$ , such that $B ^ { \top } \Pi _ { B } = 0$ . An FRNN with synaptic matrix W , and input matrix B can be rendered S-contracting via full state feedback with some $c > 0$ if and only if the following inequality holds

$$
\left[ \begin{array}{c c} I _ {n} & 0 \\ 0 & \Pi_ {B} ^ {\top} \end{array} \right] \left[ \begin{array}{c c} - 2 X & D + X W ^ {\top} \\ D + W X & - 2 D \end{array} \right] \left[ \begin{array}{c c} I _ {n} & 0 \\ 0 & \Pi_ {B} \end{array} \right] \prec 0. \tag {53}
$$
