# A. Switched State Feedback Control Law

Theorem 2: Consider the buffered NCS (4) with a switched state feedback controller

$$
\boldsymbol {u} _ {k} = - \boldsymbol {K} _ {x, i} \boldsymbol {x} _ {k} = - \hat {\boldsymbol {K}} _ {i} \boldsymbol {\xi} _ {k} = - \left[ \begin{array}{l l} \boldsymbol {K} _ {x, i} & \boldsymbol {0} \end{array} \right] \boldsymbol {\xi} _ {k} \tag {17}
$$

and consider the scalar $\gamma$ and the matrices as in the Theorem 1 with the difference that the constant matrices $X _ { 1 }$ and $z$ are replaced by matrices $X _ { 1 , i }$ and $\boldsymbol { Z } _ { i }$ of the same size $( \forall i = 1 , 2 , \ldots , { \bar { \delta } } )$ , respectively. If the resulting LMIs of the form (13) are satisfied $\forall i , j \in \left\{ 1 , 2 , \ldots , { \bar { \delta } } \right\}$ , then the buffered NCS is asymptotically stable. The switched control law (17) is obtained by $K _ { x , i } = Z _ { i } X _ { 1 , i } ^ { - 1 }$ leading to $\bar { \delta }$ values of the control signal $\mathbf { \Delta } \mathbf { u } _ { k }$ computed based on single data packet containing xk.

Proof: The proof follows as a direct consequence of the Theorem 1.

Please note that in this case, an additional mechanism implemented in the buffer is necessary. Since it is not known at the moment of the computation of the controller which $\hat { A } _ { i }$ mode is active, the data packet containing all the $\bar { \delta }$ values for $\mathbf { \Delta } \mathbf { u } _ { k }$ is sent to the actuator node. Based on the timestamp attached by the sensor, the total delay can be calculated which allows the buffer to determine which control law (17) should be applied.
