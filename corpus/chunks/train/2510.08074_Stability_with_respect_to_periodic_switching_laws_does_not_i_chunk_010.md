# 3. Proof of Theorem 3

Throughout this section we fix two matrices $A _ { 0 } , A _ { 1 } \in M _ { 2 } ( \mathbb { R } )$ ) having the properties described in Theorem 2, and let $\alpha > 0$ and $B _ { 0 } , B _ { 1 } , B _ { 2 } , B _ { 3 }$ be as in the statement of Theorem 3. We begin by establishing a result which will allow us to rewrite switching laws with values in B in terms more directly compatible with the statement of Theorem 2:

Proposition 1. There exist two linear maps $\ell _ { 0 } , \ell _ { 1 } \colon M _ { 4 } ( \mathbb { R } ) \to \mathbb { R }$ such that for every $B \in { \mathsf { B } }$ we have

$$B - B _ {0} = \ell_ {0} (B - B _ {0}) \cdot (B _ {1} - B _ {0}) + \ell_ {1} (B - B _ {0}) \cdot (B _ {2} - B _ {0}).$$

and additionally $0 \le \ell _ { 0 } ( B - B _ { 0 } ) , \ell _ { 1 } ( B - B _ { 0 } ) \le 1$ .

To prove the proposition we require two lemmas.

Lemma 2. Let C and D be $2 \times 2$ real matrices. $I f C \otimes I = I \otimes D$ then $C = D = \gamma I$ for some real number γ.

Proof. If $C \otimes I = I \otimes D$ with

$$
C = \left( \begin{array}{c c} c _ {1 1} & c _ {1 2} \\ c _ {2 1} & c _ {2 2} \end{array} \right), \quad D = \left( \begin{array}{c c} d _ {1 1} & d _ {1 2} \\ d _ {2 1} & d _ {2 2} \end{array} \right)
$$

then the equation $C \otimes I = I \otimes D$ becomes

$$
\left( \begin{array}{c c c c} c _ {1 1} & 0 & c _ {1 2} & 0 \\ 0 & c _ {1 1} & 0 & c _ {1 2} \\ c _ {2 1} & 0 & c _ {2 2} & 0 \\ 0 & c _ {2 1} & 0 & c _ {2 2} \end{array} \right) = \left( \begin{array}{c c c c} d _ {1 1} & d _ {1 2} & 0 & 0 \\ d _ {2 1} & d _ {2 2} & 0 & 0 \\ 0 & 0 & d _ {1 1} & d _ {1 2} \\ 0 & 0 & d _ {2 1} & d _ {2 2} \end{array} \right)
$$

and we immediately find that $c _ { 1 2 } = c _ { 2 1 } = d _ { 1 2 } = d _ { 2 1 } = 0$ and $c _ { 1 1 } = d _ { 1 1 } = c _ { 2 2 } = d _ { 2 2 }$ as required. 

Lemma 3. The matrices $B _ { 1 } - B _ { 0 }$ and $B _ { 2 } - B _ { 0 }$ are linearly independent.

Proof. We have

$$B _ {1} - B _ {0} = \left(A _ {1} - A _ {0}\right) \otimes I,B _ {2} - B _ {0} = \alpha \cdot I \otimes (A _ {1} - A _ {0}),$$

and if these two matrices are not linearly independent then an equation of the form

$$\left(A _ {1} - A _ {0}\right) \otimes I = \gamma \cdot I \otimes \left(A _ {1} - A _ {0}\right)$$

must hold. By Lemma 2 this is only possible if $A _ { 1 } - A _ { 0 }$ is a scalar multiple of the identity matrix, which contradicts Theorem 2(i). 

We now prove Proposition 1:

Proof. Since $B _ { 1 } - B _ { 0 }$ and $B _ { 2 } - B _ { 0 }$ are linearly independent, there exist linear maps $\ell _ { 0 } , \ell _ { 1 } \colon M _ { 4 } ( \mathbb { R } ) \to$ R such that

$$\ell_ {0} (B _ {1} - B _ {0}) = 1, \quad \ell_ {0} (B _ {2} - B _ {0}) = 0,\ell_ {1} (B _ {1} - B _ {0}) = 0, \quad \ell_ {1} (B _ {2} - B _ {0}) = 1.$$
