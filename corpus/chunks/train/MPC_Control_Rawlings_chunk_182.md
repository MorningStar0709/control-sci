# Exercise 1.56: Unconstrained tracking problem

(a) For an unconstrained system, show that the following condition is sufficient for feasibility of the target problem for any $r _ { \mathrm { s p } }$ .

$$
\operatorname{rank} \left[ \begin{array}{c c} I - A & - B \\ H C & 0 \end{array} \right] = n + n _ {c} \tag {1.69}
$$

(b) Show that (1.69) implies that the number of controlled variables without offset is less than or equal to the number of manipulated variables and the number of measurements, $n _ { c } \le m$ and $n _ { c } \le p$ .

(c) Show that (1.69) implies the rows of H are independent.

(d) Does (1.69) imply that the rows of C are independent? If so, prove it; if not, provide a counterexample.

(e) By choosing H, how can one satisfy (1.69) if one has installed redundant sensors so several rows of C are identical?
