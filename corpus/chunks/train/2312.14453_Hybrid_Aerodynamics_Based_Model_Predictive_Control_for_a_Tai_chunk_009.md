# A. Dynamic Model of the Quadrotor Tail-Sitter UAV

To model the dynamics of the tail-sitter UAV, we assume it to be a 6-DOF rigid body which is defined in both inertial frame $\Gamma _ { I }$ and body frame $\Gamma _ { B }$ , as shown in Fig. 2. The tail-sitter UAV position, velocity, and acceleration in $\Gamma _ { I }$ are defined by $\textbf { p } = \ [ x \ y \ z ] ^ { \mathrm { T } } \ \in \ \bar { \mathbb { R } } ^ { 3 } , \ \textbf { v } = \ [ u \ v \ w ] ^ { \mathrm { T } } \ \in \ \mathbb { R } ^ { 3 }$ , and $\mathbf { a } ~ = ~ \left[ a _ { x } ~ a _ { y } ~ a _ { z } \right] ^ { \mathrm { T } } ~ \in ~ \mathbb { R } ^ { 3 }$ . Its orientation is denoted by $\mathbf { R } _ { \mathrm { B } } ^ { \mathrm { I } } \in \mathrm { S O } ( 3 )$ and its angular velocity is denoted by $\omega .$

The kinetic and dynamic model of the tail-sitter UAV is constructed based on Newton–Euler formalism, as follows

$$\dot {\mathbf {p}} = \mathbf {v}
\begin{array}{l} m \dot {\mathbf {v}} = \mathbf {R} _ {\mathrm{B}} ^ {\mathrm{I}} \mathbf {F} _ {\mathrm{B}} \\ \therefore \mathbf {I} = \mathbf {I} \end{array} \tag {1}
\dot {\mathbf {R}} _ {\mathrm{B}} ^ {\mathrm{I}} = \mathbf {R} _ {\mathrm{B}} ^ {\mathrm{I}} \boldsymbol {\omega} _ {\times}\mathbf {I} \dot {\boldsymbol {\omega}} = - \boldsymbol {\omega} \times (\mathbf {I} \boldsymbol {\omega}) + \mathbf {M} _ {\mathrm{B}},$$

where m is the mass and I is the inertial matrix of the tail-sitter UAV. $\omega _ { \times }$ is the skew-symmetric matrix, such that $\omega _ { \times } { \bf V } = $ $\omega \times \mathbf { V }$ for any vector $\bar { \mathbf { V } } \in \mathbb { R } ^ { 3 }$ . The total force and moments applied in $\Gamma _ { B }$ are denoted by $\mathbf { F } _ { \mathrm { B } }$ and $\mathbf { M } _ { \mathrm { B } }$ , which can be expanded as follows:

$$\mathbf {F} _ {\mathrm{B}} = \mathbf {F} _ {\text {prop}} + \mathbf {F} _ {\text {aero}} + \mathbf {R} _ {\mathrm{I}} ^ {\mathrm{B}} \mathbf {F} _ {\mathrm{g}} + \mathbf {F} _ {\mathrm{d}} \tag {2}\mathbf {M} _ {\mathrm{B}} = \mathbf {M} _ {\text {prop}} + \mathbf {M} _ {\text {aero}} + \mathbf {M} _ {\mathrm{d}},$$

where $\mathbf { F } _ { \mathrm { p r o p } }$ and $\mathbf { M } _ { \mathrm { p r o p } }$ denote the force and moment generated by the propellers, respectively; $\mathbf { F } _ { \mathrm { a e r o } }$ and $\mathbf { M } _ { \mathrm { a e r o } }$ denote the force and moments generated by the aerodynamic effects, respectively; $\mathbf { F _ { d } }$ and $\mathbf { M } _ { \mathrm { d } }$ denote the external disturbance force and moments, respectively; and $\mathbf { F } _ { \mathrm { g } }$ is the gravitational force vector.
