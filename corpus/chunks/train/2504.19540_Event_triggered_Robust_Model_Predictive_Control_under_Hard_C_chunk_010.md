# IV. ROBUST MODEL PREDICTIVE CONTROL

Based on this ET, we design an MPC, i.e., how the central computer calculates ψ. The intuition behind our design stems from Lemma 1. Treating the difference between predicted state and the real state as a bounded disturbance, we leverage an MPC robust against such disturbances. In the following, we drop the index i of the system for brevity.

\begin{array}{l} x (k + 1 | t) = f (x (k | t), u (k | t), x (0 | t) = x (t), \\ x (k | t) \in \tilde {\mathcal {X}} _ {k}, u (k | t) \in \tilde {\mathcal {U}} _ {k}, x (N | t) \in \mathcal {X} _ {\mathrm{f}}, \tag {13} \\ \end{array}
$$

where $\begin{array} { r l r } { l ( x ( k | t ) , u ( k | t ) ) } & { { } = } & { | | x ( k | t ) | | _ { \mathrm { Q } } ^ { 2 } + | | u ( k | t ) | | _ { \mathrm { R } } ^ { 2 } } \end{array}$ , $Q , R , P \ \succ \ 0$ and $\mathcal { X } _ { \mathrm { f } }$ as final state constraint.
