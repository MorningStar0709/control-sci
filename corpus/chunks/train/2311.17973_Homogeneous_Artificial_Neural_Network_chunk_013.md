$$\sigma_ {1} (\| x \| _ {\mathbf {d}}) \leq \| x \| \leq \sigma_ {2} (\| x \| _ {\mathbf {d}}), \quad \forall x \in \mathbb {R} ^ {n}; \tag {4}$$

$3 ) \parallel \cdot \parallel$ is locally Lipschitz continuous on $\mathbb { R } ^ { n } \backslash \{ \mathbf { 0 } \} ; ~ 4 ) ~ \| ~ \cdot ~ \| _ { \mathbf { d } }$ is continuously differentiable on $\mathbb { R } ^ { n } \backslash \{ \mathbf { 0 } \}$ provided that $\| \cdot \|$ is continuously differentiable on $\mathbb { R } ^ { n } \backslash \{ \mathbf { 0 } \}$ and d is strictly monotone.
