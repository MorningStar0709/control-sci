# III. CONTROLLER DESIGN AND STABILITY ANALYSIS

Fig. 2 shows a schematic diagram of the proposed control system. The model dynamics in (4) can be written in statespace form for the purpose of control design as follows:

$$\dot {\eta} = \mathcal {Q} (\eta) + b (\eta) \mathcal {U} \tag {11}$$

where $\mathcal { Q } ( \eta )$ and $b ( \eta ) ~ \ne ~ 0$ are nonlinear smooth functions, $\mathcal { U } \ = \ [ \mathcal { U } _ { t h } , \ \mathcal { U } _ { m } ] ^ { \top } \ \in \ \mathbb { R } ^ { 4 }$ is the control input, and $\eta = [ x , \dot { x } , y , \dot { y _ { , } } z , \dot { z } , \phi , \dot { \phi _ { , } } \theta , \dot { \theta } , \psi , \dot { \psi } ] ^ { \top } \in \mathbb { R } ^ { 1 2 }$ is the state vector of the system. Assuming that for small angles applications $[ \dot { \phi } , \dot { \theta } , \dot { \psi } ] ^ { \dagger } ~ = ~ [ p , q , r ] ^ { \top }$ , the dynamic equations in (11) is extended as follows:

$$
\begin{array}{l} \dot {x} = \nu_ {s _ {x}} \\ \dot {\nu} _ {s _ {x}} = \left[ - k _ {d l} \nu_ {s _ {x}} + (s \theta c \psi + s \phi c \theta s \psi) \mathcal {U} _ {t h} \right] / m _ {s} \\ \dot {y} = \nu_ {s _ {y}} \\ \dot {\nu} _ {s _ {y}} = \left[ - k _ {d l} \nu_ {s _ {y}} + (s \theta s \psi - s \phi c \theta c \psi) \mathcal {U} _ {t h} \right] / m _ {s} \\ \dot {z} = \nu_ {s _ {z}} \\ \dot {\nu} _ {s _ {z}} = \left[ - k _ {d l} \nu_ {s _ {z}} - m _ {s} g + (c \phi c \theta) \mathcal {U} _ {t h} \right] / m _ {s} \\ \dot {\phi} = p \tag {12} \\ \end{array}

\begin{array}{l} \dot {\nu} _ {s _ {z}} = \left[ - k _ {d l} \nu_ {s _ {z}} - m _ {s} g + (c \phi c \theta) \mathcal {U} _ {t h} \right] / m _ {s} \\ \dot {\phi} = p \tag {12} \\ \end{array}
\ddot {\phi} = [ - k _ {d r} \dot {\phi} + (\mathcal {I} _ {s y} - \mathcal {I} _ {s z}) \dot {\theta} \dot {\psi} + \mathcal {U} _ {m 1} ] / \mathcal {I} _ {s x}\dot {\theta} = q\ddot {\theta} = \left[ - k _ {d r} \dot {\theta} + \left(\mathcal {I} _ {s z} - \mathcal {I} _ {s x}\right) \dot {\phi} \dot {\psi} + \mathcal {U} _ {m 2} \right] / \mathcal {I} _ {s y}\dot {\psi} = r\ddot {\psi} = [ - k _ {d r} \dot {\psi} + (\mathcal {I} _ {s x} - \mathcal {I} _ {s y}) \dot {\phi} \dot {\theta} + \mathcal {U} _ {m 3} ] / \mathcal {I} _ {s z}$$

Remark 2. From (12) it can be realized that there are two coupled subsystems. The first one $\left( \dot { x } \ t o \ \dot { \nu } _ { s _ { z } } \right)$ is the translation subsystem with one control input $\left( \mathcal { U } _ { t h } \right)$ and three outputs $( x , y , z )$ . The second subsystem $( \dot { \phi } t o \ddot { \psi } )$ has three control inputs $( { { \mathcal { U } } _ { m 1 } } , { { \mathcal { U } } _ { m 2 } } , { { \mathcal { U } } _ { m 3 } } )$ and three outputs $( \phi , \theta , \psi )$ . Therefore, we need to account for such problem in our controller design.
