# A. Preliminaries in behavioral system theory

Following [5], we introduce some basic notions and results on behavioral systems.

Definition 1. A dynamical system Σ is a triple $\Sigma =$ $\left( \mathbb { Z } _ { + } , \mathbb { R } ^ { q } , B \right)$ , where $\mathbb { Z } _ { + }$ is the time set, $\mathbb { R } ^ { q }$ is the signal space, and $B \subseteq ( \mathbb { R } ^ { q } ) ^ { \mathbb { Z } _ { + } }$ is the behavior of the system.

Definition 2. A dynamical system $\Sigma ~ = ~ ( \mathbb { Z } _ { + } , \mathbb { R } ^ { q } , \boldsymbol { B } )$ is linear if B is a linear subspace of $( \mathbb { R } ^ { q } ) ^ { \mathbb { Z } + }$ , time invariant if $\sigma ^ { t } ( B ) \subseteq B$ for all $t \in \mathbb { Z } _ { + }$ , and complete if B is closed in the topology of pointwise convergence.

The structure of an LTI dynamical system is characterized by a set of integer invariants known as structure indices [4, Section 7]. The most important ones are the number of inputs (or free variables) m, the lag l, and the order n. The structure indices are intrinsic properties of a dynamical system, as they do not depend on its representation. The complexity of a dynamical system is defined as $c = ( m , l , n )$ . The class of all complete linear, time invariant systems (with complexity c) is denoted by $\mathcal { L } ^ { q } \left( \mathcal { L } ^ { q , c } \right)$ . By a convenient abuse of notation, we shall also write $B \in \mathcal { L } ^ { q } \ ( B \in \mathcal { L } ^ { q , c } )$ .

Definition 4. A dynamical system $B \in \mathcal { L } ^ { q }$ is controllable if for every $T \in \mathbb { N } , \ w ^ { 1 } \in B | _ { T }$ , and $w ^ { 2 } \in B$ there exists $T ^ { \prime } \in \mathbb { Z } _ { + }$ , and $w \in B$ such that $w _ { t } = w _ { t } ^ { 1 }$ for $t \in \mathbf { T }$ and $w _ { t } = w _ { t - T - T ^ { \prime } } ^ { 2 }$ for $t > T + T ^ { \prime }$ .

In other words, a dynamical system is controllable if any two trajectories can be patched together in finite time.
