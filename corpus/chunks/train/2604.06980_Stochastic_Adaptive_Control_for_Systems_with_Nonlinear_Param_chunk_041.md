Note that the system can be rewritten as

$$
\begin{array}{l} x _ {t + 1} = f \left(A ^ {*} x _ {t} + B ^ {*} \pi_ {\hat {\theta} _ {t - 1}} (x _ {t}) + B ^ {*} v _ {t}\right) + w _ {t + 1} \tag {B.11} \\ = f \left(\bar {A} x _ {t} + \bar {B} \pi_ {\bar {\theta}} (x _ {t}) + \bar {v} _ {t}\right) + w _ {t + 1}. \\ \end{array}
$$

From Assumption 4 and (B.10), we obtain

$$\sum_ {\tau = 0} ^ {t} \| x _ {\tau} \| ^ {\gamma} = o (\bar {r} _ {t}) + O (t), a. s., t \rightarrow \infty . \tag {B.12}$$

Since $\pi _ { \theta ^ { \ast } }$ is Lipschitz continuous, there exists a constant $L _ { 3 } > 0$ such that $\| \pi _ { \theta ^ { * } } ( x _ { t } ) \| \le L _ { 3 } \| x _ { t } \| + \| \pi _ { \theta ^ { * } } ( 0 ) \|$ . Moreover, by (26), we also know $\| \pi _ { \hat { \theta } _ { t - 1 } } ( x _ { t } ) - \pi _ { \theta ^ { * } } ( x _ { t } ) \| \leq$ $L _ { 1 } \| \theta ^ { * } - \hat { \theta } _ { t - 1 } \| \| x _ { t } \| + L _ { 2 }$ , which implies $\| \pi _ { \hat { \theta } _ { t - 1 } } ( x _ { t } ) \| =$ $O ( \| x _ { t } \| )$ almost surely as $t \to \infty$ . Therefore, by (B.12),

$$
\begin{array}{l} \sum_ {\tau = 0} ^ {t} \| u _ {\tau} \| ^ {\gamma} = O \left(\sum_ {\tau = 0} ^ {t} \| \pi_ {\hat {\theta} _ {\tau - 1}} (x _ {\tau}) \| ^ {\gamma} + \sum_ {\tau = 0} ^ {t} \| v _ {\tau} \| ^ {\gamma}\right) \\ = o (\bar {r} _ {t}) + O (t), a. s. \\ \end{array}
$$

Since $\phi _ { t } ~ = ~ [ x _ { t } ^ { \top } , u _ { t } ^ { \top } ] ^ { \top }$ , the above result gives $\begin{array} { r l } { \bar { r } _ { t } } & { { } = } \end{array}$ $\begin{array} { r l r } { \sum _ { \tau = 0 } ^ { t } \| \phi _ { \tau } \| ^ { \gamma } } & { { } = } & { o ( \bar { r } _ { t } ) + O ( t ) } \end{array}$ , a.s., which implies $\bar { r } _ { t } = \mathcal { O } ( t )$ , a.s. Finally, substituting this into (B.12), we conclude that

$$\sum_ {\tau = 0} ^ {t} \| \phi_ {\tau} \| ^ {\gamma} = O (t), a. s., \tag {B.13}$$

and Theorem 2 holds.
