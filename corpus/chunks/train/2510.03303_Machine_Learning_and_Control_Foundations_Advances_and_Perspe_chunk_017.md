Here, at any given time t, only a single neuron is active. The vector field driving the dynamics vanishes on one half-space and is linear on the other. This structure allows us to design an inductive strategy that builds piecewise constant controls $( w , a , b )$ , sequentially steering each input point $x _ { i }$ to its prescribed target $y _ { i }$ , thereby yielding the following result.

Theorem 3.1 (Simultaneous controllability, $P = 1 , [ 3 4 ] )$ . Let the dimension $d \ge 2 , \ N \ge 1$ , and $T > 0$ be fixed. Given a consistent dataset $\{ ( x _ { i } , y _ { i } ) \} _ { i \in [ N ] } \subset \mathbb { R } ^ { d } \times \mathbb { R } ^ { d }$ with $x _ { i } \neq x _ { j }$ and $y _ { i } \ne y _ { j } i f i \ne j$ , there exists piecewise constant controls $( w , a , b ) = ( w ( t ) , a ( t ) , \dot { b } ( t ) ) \in \mathcal { S } _ { 1 }$ with at most $L = 3 N$ switches, such that the flow map $\Phi _ { T }$ generated by (3.6) satisfies

$$\Phi_ {T} (x _ {i}; w, a, b) = y _ {i}, \quad \forall i \in [ N ]. \tag {3.7}$$

The representation of a finite dataset, as considered thus far, is naturally related to the universal approximation problem. In the present framework, the latter is understood as the ability to approximate any continuous function on a compact set by means of the flow map associated with the NODE. More precisely, the following approximation result holds in $\dot { L } ^ { 2 }$ :

Theorem 3.2 (Universal approximation theorem, $P = 1 , [ 3 4 ] )$ . Let the dimension $d \ge 2 , T > 0$ be fixed, and consider a bounded set $\Omega \subset \mathbb { R } ^ { d }$ . Then, for any $f \in L ^ { 2 } ( \Omega ; \mathbb { R } ^ { d } )$ and $\varepsilon > 0$ , there exist piecewise constant controls $( w , a , b ) = ( w ( t ) , a ( t ) , b ( t ) ) \in \mathcal S _ { 1 }$ with a finite number of discontinuities, such that the flow map $\Phi _ { T }$ generated by (3.6) satisfies

$$\| \Phi_ {T} - f \| _ {L ^ {2} (\Omega)} < \varepsilon . \tag {3.8}$$

The proof combines the simultaneous controllability property of NODEs with approximation arguments based on simple functions defined on hyperrectangles, together with the ability of NODEs to compress and expand the regions where these simple functions take constant values.
