# A. Notation

We use uppercase letters for random variables and lowercase for their realizations. For any variable, subscripts represent time indices and superscripts represent player identities. We use notation −i to represent all players other than player i i.e. $- i = \{ 1 , 2 , \ldots i - 1 , i + 1 , \ldots , N \}$ . We use notation $a _ { t : t ^ { \prime } }$ to represent the vector $( a _ { t } , a _ { t + 1 } , \ldots a _ { t ^ { \prime } } )$ when $t ^ { \prime } \geq t$ or an empty vector if $t ^ { \prime } < t .$ . We use $a _ { t } ^ { - i }$ to mean $( a _ { t } ^ { 1 } , a _ { t } ^ { 2 } , \ldots , a _ { t } ^ { i - 1 } , a _ { t } ^ { i + 1 } \ldots , a _ { t } ^ { m } )$ . We remove superscripts or subscripts if we want to represent the whole vector, for example $a _ { t }$ represents $( a _ { t } ^ { 1 } , \ldots , a _ { t } ^ { m } )$ ). We denote the indicator function of any set A by 1{A}. For any finite set $s ,$ , ${ \mathcal { P } } ( S )$ ) represents space of probability measures on $s$ and |S| represents its cardinality. Given a set ${ \mathcal { A } } ,$ we denote its n-fold Cartesian product by $( A ) ^ { m }$ . We denote the set of real numbers by R. For a probabilistic strategy profile of players $( \sigma _ { t } ^ { i } ) _ { i \in [ N ] }$ where probability of action $a _ { t } ^ { i }$ conditioned on $z _ { 1 : t } , x _ { 1 : t } ^ { i }$ is given by $\sigma _ { t } ^ { i } ( a _ { t } ^ { i } | \boldsymbol { z } _ { 1 : t } , \boldsymbol { x } _ { 1 : t } ^ { i } )$ , we use the short hand notation $\sigma _ { t } ^ { - i } ( a _ { t } ^ { - i } | z _ { 1 : t } , x _ { 1 : t } ^ { - i } )$ to represent $\begin{array} { r } { \prod _ { j \neq i } \sigma _ { t } ^ { j } ( a _ { t } ^ { j } | \boldsymbol { z } _ { 1 : t } , \boldsymbol { x } _ { 1 : t } ^ { j } ) } \end{array}$ . We denote by $P ^ { \sigma }$ (or $E ^ { \sigma } )$ the probability measure generated by (or expectation with respect to) strategy profile σ. All equalities and inequalities involving random variables are to be interpreted in the a.s. sense. For any variable a, we define $ { \boldsymbol { S } } _ { a }$ as the space of all possible a.
