We have already shown that the first term of the above inequality goes to 0. Also, since $\mathcal { T } _ { T , n } ^ { * } ( x ) = \mathcal { I } _ { T , n } ( x , v ^ { n * } )$ and $\mathcal { I } _ { T , n } ( x , v ^ { n * } )$ is the discretization of $\mathcal { I } _ { T } ( x , v ^ { n * } ( \cdot ) )$ , we have that ${ \mathcal { I } } _ { T , n } ^ { * } ( x )$ and $\mathcal { I } _ { T } ( x , v ^ { n * } ( \cdot ) )$ converge to the same limit as $n \to \infty$ . Thus, we conclude that

$$\lim _ {n \to \infty} \mathcal {J} _ {T} (x, v ^ {n *} (\cdot)) = \inf _ {m \in \mathfrak {U} _ {\mathrm{WRC}}} \mathcal {J} _ {T} (x, m).$$

This completes the proof of the theorem .
