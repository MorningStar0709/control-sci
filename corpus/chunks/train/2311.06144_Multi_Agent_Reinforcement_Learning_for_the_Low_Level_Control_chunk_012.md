# IV. MULTI-AGENT RL FOR QUADROTOR CONTROL

Given a goal state $g$ that consists of the desired position $x _ { d } \in \mathbb { R } ^ { 3 } .$ , desired velocity $v _ { d } \in \mathbb { R } ^ { 3 }$ , desired heading direction $b _ { 1 _ { d } } \in \mathsf { S } ^ { 2 }$ , and desired angular velocity $\Omega _ { d } \in \mathbb { R } ^ { \bar { 3 } }$ , we aim to find an optimal policy such that $s _ { t } \ \to \ g _ { t }$ as $t  \infty$ from arbitrary initial states. During training, the goal state is assumed to be fixed, i.e., $x _ { d } = v _ { d } = \Omega _ { d } = [ 0 , 0 , 0 ] ^ { T }$ along with $b _ { 1 _ { d } } = [ 1 , 0 , 0 ] ^ { T }$ . Leveraging error states, we achieve trajectory tracking by shifting the goal state.
