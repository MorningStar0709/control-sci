where $\tau _ { 1 } , \tau _ { 2 } > 0$ are positive numbers, $C \in \mathrm { H } ( \mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 } )$ is a Hermitian operator in $\mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 }$ , $\rho \in \mathrm { H } _ { \geq } ( \mathcal { H } _ { 1 } )$ and $\sigma \in \mathrm { { H } } _ { \geq } ( \mathcal { H } _ { 2 } )$ are Hermitian semi-definite positive operators on $\mathcal { H } _ { 1 }$ and $\mathcal { \mathrm { H } } _ { 2 }$ respectively, $\Gamma _ { 1 } , \Gamma _ { 2 }$ are, respectively, the partial traces of $\Gamma _ { 1 } = \mathrm { T r } _ { 2 } [ \Gamma ] , \Gamma _ { 2 } = \mathrm { T r } _ { 1 } [ \Gamma ]$ in $\mathcal { \mathrm { H } } _ { 2 }$ and $\mathcal { H } _ { 1 }$ . Finally, the functional $\mathcal { E } [ \gamma _ { 1 } | \gamma _ { 2 } ]$ is the Umegaki relative entropy between semi-definite positive and Hermitian operators $\gamma _ { 1 }$ and $\gamma _ { 2 }$

$$
\mathcal {E} [ \gamma_ {1} | \gamma_ {2} ] = \left\{ \begin{array}{l l} \mathrm{Tr} [ \gamma_ {1} (\log \gamma_ {1} - \log \gamma_ {2} - \mathbb {I} _ {d}) + \gamma_ {2} ] & \text {if \ker \gamma_{1} \subset \ker \gamma_{2}}, \\ + \infty & \text {otherwise.} \end{array} \right.
$$

The (von Neumann entropy regularized) Unbalanced Non-commutative Optimal Transport (1.2) relaxes the constraint $\Gamma \mapsto ( \rho , \sigma )$ in (1.1) and, in particular, the trace of the matrices $\rho$ and σ must be finite (and not necessarily equal to one), allowing for unbalanced semi-definite positive and Hermitian operators.

The classical theory of Unbalanced Optimal Transport between positive measures has been introduced, independently, by Liero, Mielke and Savaré [18], Chizat, Peyré, Schmitzer and Vialard [7], and Kondratyev, Monsaingeon and Vorotnikov [16, 17]. This work proposes a generalization of the (static) Shannon-Entropy regularization of Unbalanced Optimal Transport introduced in [15] into the non-commutative setting.
