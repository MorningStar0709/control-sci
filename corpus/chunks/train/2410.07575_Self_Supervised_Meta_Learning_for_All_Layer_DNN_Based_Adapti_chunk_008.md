# IV. SSML-AC FRAMEWORK

Although recent meta-learning approaches for adaptive control [1], [38]–[40] address trajectory tracking under dynamic disturbances, these methods rely on linear parameterization with nonlinear features of disturbance structure, limiting the potential of DNNs in online adaptation. We propose the Self-Supervised Meta-Learning for All-Layer DNNbased Adaptive Control (SSML-AC) framework, which pretrains the DNN via self-supervised meta-learning IV-A and adapts the full DNN online using speed-gradient composite adaptation IV-B.
