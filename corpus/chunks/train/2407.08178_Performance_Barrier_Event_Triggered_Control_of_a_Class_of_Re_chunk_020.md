The term $W ^ { p } ( t ) \triangleq e ^ { - b ^ { * } t } V _ { 0 } - V ^ { p } ( t )$ is the so-called performance residual, which is the difference between the value of the performance-barrier $e ^ { - b ^ { * } t } V _ { 0 }$ and the Lyapunov function ${ \dot { V ^ { p } } } ( t )$ . The decay rate $b ^ { * }$ is the nominal exponential convergence rate guaranteed under the R-CETC briefed in Section 3.1. We introduce this residual into the triggering mechanism allowing the Lyapunov function to occasionally increase, provided it stays below the nominal performancebarrier $e ^ { - b ^ { * } t } V _ { 0 }$ .
