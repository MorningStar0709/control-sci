# 3.1.1 MDP Model

A MDP is a discrete time finite state-transition model that captures random state changes, action-triggered transitions and states-dependent rewards. A MDP is a four tuple $\left( S , A , P _ { a } , R _ { a } \right)$ comprising a set of n states $S = \{ 0 , 1 , \ldots , n - 1 \}$ , a set of m actions $A = \left\{ 0 , 1 , \ldots , m - 1 \right\}$ , a transition probability function $P _ { a }$ and a reward function $R _ { a } .$ . The evolution of a MDP is paced by a discrete clock. At time t, the MDP is in state $s _ { t } ,$ , such that $t = 0 , 1 , 2 , \ldots$ . The MDP model starts in an initial state $s _ { 0 } = 0$ . The transition probability function, denoted as

$$P _ {a} \left(s, s ^ {\prime}\right) = P r \left[ s _ {t + 1} = s ^ {\prime} \mid s _ {t} = s, a _ {t} = a \right] \tag {1}$$

defines the probability of making a transition to state $s _ { t + 1 }$ equal to $s ^ { \prime }$ at time t + 1, when at time t the state is $s _ { t } = s$ and action a is performed. The reward function $R _ { a } ( s , s ^ { \prime } )$ defines the immediate reward associated with the transition from state s to $s ^ { \prime }$ and action a. It has domain $S \times S \times A$ and co-domain R.
