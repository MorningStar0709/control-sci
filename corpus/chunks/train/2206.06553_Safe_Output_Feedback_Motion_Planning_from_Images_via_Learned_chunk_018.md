Two errors drive $w _ { e } ( t )$ : the perception error $\hat { h } ^ { - 1 } ( h ( x , \theta ) , \theta ) - C _ { r } x$ , and the runtime observation noise $B _ { y } w _ { y }$ . Combining with the dynamics error gives $w _ { q } ( t ) \doteq B _ { w } ( t ) w _ { x } ( t ) +$ $w _ { e } ( t ) . \ B _ { w } ( t ) w _ { x } ( t )$ can be bounded as in Lemma 1, but $w _ { e } ( t )$ is harder to bound. Let $y _ { p } = h ( x , \theta )$ and $y = h ( x , \theta ) + B _ { y } w _ { y }$ . We rewrite the norm of the red term in (16) as:

$$
\begin{array}{l} \| \hat {h} ^ {- 1} (y, \theta) - C _ {r} x \| = \| \hat {h} ^ {- 1} (y, \theta) - \hat {h} ^ {- 1} (y _ {p}, \theta) + \hat {h} ^ {- 1} (y _ {p}, \theta) - C _ {r} x \| \\ \leq \underbrace {L _ {\hat {h} ^ {- 1}} \| B _ {y} w _ {y} \|} _ {\text { from   measurement   noise }} + \underbrace {\left\| \hat {h} ^ {- 1} \left(y _ {p} , \theta\right) - C _ {r} x \right\|} _ {\text { from   learning   error } \dot {=} \epsilon (x, \theta)}. \tag {17} \\ \end{array}
$$

Here, $L _ { \hat { h } ^ { - 1 } }$ is the local Lipschitz constant of the learned inverse function in y, i.e.,

$$\left\| \hat {h} ^ {- 1} (\tilde {y}, \theta) - \hat {h} ^ {- 1} (\check {y}, \theta) \right\| \leq L _ {\hat {h} ^ {- 1}} \| \tilde {y} - \check {y} \|, \quad \forall \tilde {y}, \check {y} \in D _ {y} \oplus \mathcal {Y} _ {d}, \quad \forall \theta \in D _ {\theta}, \tag {18}$$

where $D _ { y } = h ( D _ { r } , D _ { \theta } )$ is the image of the training data domains, ⊕ is the Minkowski sum, and $\mathcal { V } _ { d } = \{ B _ { y } w _ { y } \mid \| w _ { y } \| \leq \bar { w } _ { y } \}$ is the set of feasible measurement noise. The first braced term in (17) bounds the effect of measurement error on the reduced observation and is valid for all $( x , \theta ) \in D _ { r } \times D _ { \theta }$ and observation noise satisfying $\lVert w _ { y } \rVert \leq \bar { w } _ { y }$ .
