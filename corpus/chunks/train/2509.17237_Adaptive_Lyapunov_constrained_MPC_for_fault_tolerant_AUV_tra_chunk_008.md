# D. Augmented Error Model

To explicitly capture persistent, low–frequency mismatches—such as steady currents, unmodeled hydrodynamic loads, or residual allocation errors—we introduce a slowly varying bias in the generalized-force channel and treat it as an augmented state. This bias acts like a integral action inside MPC: it cancels constant/slow disturbances to remove steadystate offset, while remaining bounded when the mismatch disappears. We augment the generalized force with a bias $\pmb { d } \in \bar { \mathbb { R } } ^ { 3 \times 1 }$ :

$$\boldsymbol {\tau} _ {\text { eff }} = \boldsymbol {T} (\boldsymbol {\theta}) \boldsymbol {\Gamma} \boldsymbol {u} + \boldsymbol {d}, \tag {7}\dot {\boldsymbol {d}} = - \boldsymbol {\Lambda} \boldsymbol {d}, \quad \boldsymbol {\Lambda} = \operatorname{diag} (\lambda_ {i}) \succ 0.$$

Here Λ sets the bias dynamics: each $\lambda _ { i } > 0$ is a designerchosen decay rate (units $\mathbf { s } ^ { - 1 } )$ , with time constant $\tau _ { i } = 1 / \lambda _ { i }$ . Small $\lambda _ { i }$ lets $^ d$ track slow drifts; larger $\lambda _ { i }$ makes d decay faster. In practice, choose $\tau _ { i }$ on the order of several sampling periods but slower than the dominant closed-loop bandwidth.

Using (4), (5), (6), and (7), the augmented error dynamics become

$$
\dot {\tilde {\boldsymbol {x}}} = \left[ \begin{array}{c} \boldsymbol {J} (\psi) \boldsymbol {\nu} - \dot {\boldsymbol {\eta}} _ {d} \\ \boldsymbol {M} ^ {- 1} \big (\boldsymbol {\tau} _ {\text { eff }} - \boldsymbol {C} (\boldsymbol {\nu}) \boldsymbol {\nu} - \boldsymbol {D} (\boldsymbol {\nu}) \boldsymbol {\nu} \big) - \dot {\boldsymbol {\nu}} _ {d} \end{array} \right] + \boldsymbol {w}, \tag {8}
$$

where the residual w collects fast, zero-mean disturbances not captured by d. and the augmented state is $\tilde { \pmb { x } } _ { \mathrm { a u g } } = \left\lceil \tilde { \pmb { x } } ^ { \top } \ \pmb { d } ^ { \top } \right\rceil ^ { \top }$ . The bias dynamics in (7) act as an integral action in the generalized-force domain: constant or slowly varying disturbances are absorbed by $^ { d , }$ enabling offset-free tracking while guaranteeing that $^ d$ decays when no longer needed. This augmented model will be used in the LMPC design (Sec. III-B) and in the multi-model estimator (Sec. IV).
