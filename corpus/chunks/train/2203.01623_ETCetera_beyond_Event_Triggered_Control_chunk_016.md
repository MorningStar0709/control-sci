The state-space ?? of the original system is now mapped to the set $\tilde { X } \ = \ \{ ( x , 1 ) \ \in \ \mathbb { R } ^ { n _ { \zeta } + 1 } \ : \ x \ \in \ X \}$ . The user has to specify parameters $( p _ { 1 } , p _ { 2 } , . . . , p _ { n _ { \zeta } } )$ (grid\_points\_per\_dim), such that ??˜ is partitioned into $p _ { 1 } { \times } p _ { 2 } { \times } \cdots { \times } p _ { n _ { \zeta } }$ equal hyper-rectangles.
