# 2 Problem Setup

We consider M discrete-time linear time-invariant (LTI) systems, among which an unknown f systems may behave adversarially. The remaining $m = M - f$ honest systems evolve according to

$$

$$

where $\Theta _ { \star } ^ { ( i ) } = [ A _ { \star } ^ { ( i ) } \ B _ { \star } ^ { ( i ) } ]$ denotes system i’s model, and $\{ w _ { t } ^ { ( i ) } \}$ t has elements that are i.i.d. mean-zero σ2w-sub-Gaussian, for some positive variance proxy $\sigma _ { w } ^ { 2 }$ [Vershynin, 2018].

Clusters. Each honest system belongs to one of $N _ { c }$ clusters $\{ \mathcal { C } _ { j } \} _ { j = 1 } ^ { N _ { c } }$ , grouping systems with similar dynamics. Systems within the same cluster share model parameters that are “close” in the Frobenius norm (Assumption 1). The separation between clusters is characterized by the minimum and maximum distances, $\begin{array} { r } { \Delta _ { \operatorname* { m i n } } \triangleq \operatorname* { m i n } _ { j \neq j ^ { \prime } } \| \Theta _ { j } - \Theta _ { j ^ { \prime } } \| } \end{array}$ and $\Delta _ { \operatorname* { m a x } } \triangleq \operatorname* { m a x } _ { j \neq j ^ { \prime } } \Vert \Theta _ { j } - \Theta _ { j ^ { \prime } } \Vert$ , respectively.
