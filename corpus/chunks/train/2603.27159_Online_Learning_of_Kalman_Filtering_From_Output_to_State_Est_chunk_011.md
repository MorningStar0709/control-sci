In the remainder of this paper, we will assume that there exist known constants $\alpha _ { 0 } , \alpha _ { 1 } , \psi , \bar { \sigma } \in \mathbb { R } _ { > 0 }$ such that

$$\alpha_ {0} I _ {n} \preceq W \preceq \alpha_ {1} I _ {n}, \alpha_ {0} I _ {p} \preceq V \preceq \alpha_ {1} I _ {p}, \| C \| \leq \psi , \| \Sigma \| \leq \bar {\sigma}. \tag {13}$$

In addition, we denote

$$\kappa_ {F} = \sqrt {\frac {\bar {\sigma}}{\alpha_ {0}}}, \gamma_ {F} = 1 - \frac {\alpha_ {0}}{2 \bar {\sigma}}, \tag {14}$$

where $0 < \gamma _ { F } < 1$ holds by definition (see Lemma E.4 in Appendix E).

We now prove the following properties for the parameter $M ^ { \star } = [ M _ { s } ^ { \star } ] _ { s \in [ h ] }$ associated with $\hat { x } _ { t } ^ { \mathrm { T K } }$ ; the proof is included in Appendix B.

Lemma 2.1. For any $s \in \mathbb { Z } _ { \ge 1 } , \ \lVert M _ { s } ^ { \star } \rVert \le \kappa _ { F } ^ { 2 } \gamma _ { F } ^ { s - 1 } , \ \lVert M _ { s } ^ { \star } \rVert _ { F } \le \sqrt { \operatorname* { m i n } \{ p , n \} } \kappa _ { F } ^ { 2 } \gamma _ { F } ^ { s - 1 }$ and $\Vert M ^ { \star } \Vert _ { F } \leq$ $\begin{array} { r } { \sqrt { \operatorname* { m i n } \{ p , n \} } \frac { \kappa _ { F } ^ { 2 } } { 1 - \gamma _ { F } } } \end{array}$ , where $M _ { s } ^ { \star }$ given by (8).

Lemma 2.1 shows that the dependence of $\hat { x } _ { t } ^ { \mathrm { T K } }$ on the past information $y _ { t - s }$ decays exponentially as $s \in \mathbb { Z } _ { \geq 1 }$ increases, which will guide our choice of the interval length $h \in \mathbb { Z } _ { \geq 1 }$ used in $\hat { x } _ { t } ^ { \mathrm { T K } }$ . In addition, Lemma 2.1 specifies a set of matrices that the optimal parameter $M ^ { \star }$ belongs to, i.e., $M ^ { \star } \in { \cal K } _ { M }$ with

$$\mathcal {K} _ {M} \triangleq \Big \{M = [ M _ {s} ] _ {s \in [ h ]}: \| M \| _ {F} \leq \sqrt {\min \{p , n \}} \frac {\kappa_ {F} ^ {2}}{1 - \gamma_ {F}} \Big \}. (1 5)$$

Based on the above arguments, one can show that optimizing over all $M \in { \cal K } _ { M }$ will give us the parameter $M ^ { \star }$ in the truncated steady-state Kalman filter $\hat { x } _ { t } ^ { \mathrm { T K } } ,$ ; the specific form of the corresponding optimization problem will be specified later in the next subsection.
