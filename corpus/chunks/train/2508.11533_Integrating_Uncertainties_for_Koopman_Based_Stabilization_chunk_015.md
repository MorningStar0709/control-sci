# 2.2 Petersen’s Lemma

Since we investigate the influences of approximation error and disturbances on feedback stabilization, necessary tools of robust control should be introduced. In this paper, we advocate the use of Petersen’s lemma [3].

Lemma 2 Consider matrices $\mathbf { C } \in \mathbb { R } ^ { n \times n }$ , M $\in \mathbb { R } ^ { n \times p } .$ , $\overline { { \mathbf { D } } } \in \mathbb { R } ^ { q \times q }$ , $\mathbf { N } \in \mathbb { R } ^ { q \times n }$ with $\mathbf { C } = \mathbf { C } ^ { \top }$ and $\overline { { \mathbf { D } } } = \overline { { \mathbf { D } } } ^ { \prime } \succeq 0$ . Let the set of matrices

$$\mathcal {D} := \{\mathbf {D} \in \mathbb {R} ^ {p \times q}: \mathbf {D} ^ {\top} \mathbf {D} \preceq \overline {{\mathbf {D}}} \}. \tag {12}$$

Then,

$$\mathbf {C} + \mathbf {M D N} ^ {\top} + \mathbf {N D} ^ {\top} \mathbf {M} ^ {\top} \prec 0, \forall \mathbf {D} \in \mathcal {D} \tag {13}$$

if and only if there exists $\mu > 0$ such that

$$\mathbf {C} + \mu \mathbf {M M} ^ {\top} + \mu^ {- 1} \mathbf {N} \overline {{\mathbf {D}}} \mathbf {N} ^ {\top} \prec 0. \tag {14}$$

Petersen’s lemma serves as an ideal approach for matrix elimination [3], since we need not to verify the expected inequality for all matrices satisfying the given norm bound. This explains why it can become a useful tool for robust control. Also we have a corresponding non-strict version, given in Lemma 3.

Lemma 3 Consider matrices $\mathbf { C } \in \mathbb { R } ^ { n \times n }$ , M $\mathbf { \tau } \in \mathbb { R } ^ { n \times p }$ , D ∈ Rq×q, N ∈ Rq×n with $\mathbf { C } = \mathbf { C } ^ { \top }$ and $\overline { { \mathbf { D } } } = \overline { { \mathbf { D } } } ^ { \prime } \succeq 0$ . Let the set D be defined the same as (12), and assume $\mathbf { M } \neq 0 , \mathbf { \overline { { D } } } \succ 0$ and N $\neq 0$ . Then,

$$\mathbf {C} + \mathbf {M D N} ^ {\top} + \mathbf {N D} ^ {\top} \mathbf {M} ^ {\top} \preceq 0, \forall \mathbf {D} \in \mathcal {D} \tag {15}$$

if and only if there exists $\mu > 0$ such that

$$\mathbf {C} + \mu \mathbf {M M} ^ {\top} + \mu^ {- 1} \mathbf {N} \overline {{{{\mathbf {D}}}}} \mathbf {N} ^ {\top} \preceq 0. \tag {16}$$

The proofs of Lemmas 2 and 3 can be found in [3], and are therefore omitted here due to space constraints.
