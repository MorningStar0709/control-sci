在上述例子中，对给定的对象参数 $a_1 = 1, a_2 = 1$ 来说，PID增益 $k_1$ 可取 $|k_1 - 50| \leqslant 20$ 范围的值，而给定的PID增益 $k_0 = k_1 = k_2 = 50$ ，却都能控制满足 $|a_1 - 1| \leqslant 20, |a_2 - 1| \leqslant 20$ 的参数所代表的对象。尽管参数 $a_1 = 1$ 的对象和 $a_1 = 20$ 的对象，其时间尺度相差 $\sqrt{20} \approx 4.47$ 倍，但可以用同一个PID增益 $k_0 = k_1 = k_2 = 50$ 来实现 $T_0 = 2$ 的过渡过程。

如果把 $k_{0}^{*} = k_{1}^{*} = k_{2}^{*} = 50$ 作为标准PID增益，那么控制以 $a_1 = 20$ 为参数的对象所需要的PID增益 $k_{0},k_{1},k_{2}$ ，是用时间尺度 $\rho = 1 / \sqrt{20} = 0.2236$ ，按公式(1.5.14）计算得到

$$k _ {0} = 5 0 / \rho^ {3} = 4 4 7 2, k _ {1} = 5 0 / \rho^ {2} = 1 0 0 0,k _ {2} = 5 0 / \rho = 2 2 3. 6$$

当然，这组增益能更好地跟踪前面安排的 $T_{0} = 2$ 的过渡过程。以 $a_1 = 20$ 为参数的系统具有快速响应的能力，对这个系统来说， $T_{0} = 2$ 的过渡过程是很慢的过程了。由于 $a_1 = 20$ 表代表的系统时间 $t$ 和把它化成标准系统的时间 $\tau$ 之间有关系 $t = \rho \tau$ ，因此相当于标准系统 $T_{0} = 2$ 的过渡过程，在 $t$ 时间尺度上应该是 $T_{0} = \rho \times 2 = 2/\sqrt{20} = 0.447$ ，即新的增益能控制 $a_1 = 20$ 的对象应该很好地跟踪 $T_{0} = 0.447$ 的过渡过程(图1.5.6)。
