In the discrete-time setting4, within the behavioral framework, Willems’ fundamental lemma (Willems et al., 2005a) provides a representation of LTI system using a single sufficiently informative and long trajectory. Consider a trajectory $\{ \tilde { u } _ { k } , \tilde { y } _ { k } \} _ { k = 0 } ^ { N - 1 }$ from an LTI system (12) that satisfies a condition , called persistent excitation. Then, we can verify whether any arbitrary trajectory $\{ u _ { k } , y _ { k } \} _ { k = 0 } ^ { L - 1 }$ of shorter length L comes from , the same LTI system as long as $N \ge ( n _ { u } + 1 ) ( n _ { x } + L ) - 1$ . In detail, let $\tilde { \textbf { u } } = \left[ \tilde { u } _ { 0 } ^ { \top } \quad \cdot \cdot \cdot \quad \tilde { u } _ { N - 1 } ^ { \top } \right] ^ { \top } , \tilde { \textbf { y } } = \left[ \tilde { y } _ { 0 } ^ { \top } \quad \cdot \cdot \quad \tilde { y } _ { N - 1 } ^ { \top } \right] ^ { \top }$ $\mathbf { u } \ = \ \left\lceil u _ { 0 } ^ { \top } \quad \cdots \quad u _ { L - 1 } ^ { \top } \right\rceil ^ { \top }$ , and $\mathbf { y } ~ = ~ \left\lceil y _ { 0 } ^ { \top } \quad \cdots \quad y _ { L - 1 } ^ { \top } \right\rceil ^ { \top }$ . The input sequence u˜ is persistently exciting of order $k \in [ 1 , N ]$ if the Hankel matrix $\mathcal { H } _ { k } ( \tilde { u } )$ of depth k constructed from u˜ has full row rank (Willems et al., 2005a). If u˜ is persistently exciting of order $n _ { x } + L ,$ then u, y is a trajectory of (12) if there exists a vector $\alpha \in \mathbb { R } ^ { N - L + 1 }$ such that

$$
\left[ \begin{array}{l} \mathcal {H} _ {L} (\tilde {\mathbf {u}}) \\ \mathcal {H} _ {L} (\tilde {\mathbf {y}}) \end{array} \right] \alpha = \left[ \begin{array}{l} \mathbf {u} \\ \mathbf {y} \end{array} \right]. \tag {13}
$$

Intuitively, the algebraic condition (13) states that any trajectory of a discrete-time LTI system with finite length is a linear combination of one sufficiently informative and long trajectory.
