# Control Deployment

Control law:

$$\dot {x} _ {\mathrm{c}} = (F + G K) x _ {\mathrm{c}} + L y, \quad u = K x _ {\mathrm{c}}, \tag {11}$$

with arbitrary $x _ { \mathrm { c } } ( 0 ) \in \mathbb { R } ^ { \mu } .$ .

The central idea is to build a non-minimal realization of the plant (Section III-A) that has a structure amenable to observer design. This allows us to split the dynamic controller into an observer of the non-minimal realization and a feedback law depending on the observer states (Section III-B). In particular, the feedback gain is designed to quadratically stabilize all realizations consistent with the data (Section III-C) by employing a data-based LMI derived from the matrix S-lemma [7, Thm. 4.10] (Section III-D).
