# 5.1 Measurement noise

To be able to implement the obtained state-feedback control law, we assume the system states are available through estimation or measurement, which may be contaminated by a zero-mean additive noise $\eta _ { x } ( t ) \in \mathbb { R } ^ { m }$ as follows

$$\tilde {x} (t) = x (t) + \eta_ {x} (t).$$

For the measurements (37) made during the learning process in Algorithm 1, we make the following assumption.

Assumption 5. The stochastic process η has a zero mean, i.e., $\mathbb { E } ( \eta ( t ) ) = 0$ for $t \in [ 0 , \bar { t } _ { f } ]$ , and there is some $e > 0$ such that $\| \dot { \eta } ( t ) \| _ { \infty } \leq e$ holds almost surely.

Assumption 5 entails that η(t) also has a bounded variance. Similar to the finite horizon case with squareintegrable noises considered in Section 4.1, the next theorem proves that estimation (40) is also unbiased with bounded variance when $t _ { f }  + \infty$ , provided that the measurement noise is bounded.

Theorem 5. Let Assumptions 1, 4 and 5 hold. Assume (48) and that $u _ { 0 } \in \bar { \mathcal { L } } _ { \infty } ^ { m } ( 0 , t _ { f } )$ is deterministic. Estimation (40) is unbiased, i.e.,

$$\mathbb {E} \big (\nu_ {k} \big) = 0, \quad k \in \mathbb {N} \tag {70}$$

and it is of bounded variance as $t _ { f } \to \infty$ .

Proof. see appendix A.3.

Now assume that Algorithm 1 is run for k iterations in the presence of a measurement noise satisfying Assumption 5. Applying the control input (40) obtained from Algorithm 1 results in

$$\tilde {x} _ {k} (t) = x _ {k} (t) + \mathcal {G} _ {x} \nu_ {k} (t) + \eta_ {x} (t), \quad t \in [ 0, t _ {f} ]. \tag {71}$$

The optimal feedback gain can then be estimated by solving

$$\left(\tilde {X} _ {t _ {1}: t _ {n}} ^ {\prime} \otimes I _ {m}\right) \operatorname{vec} (\tilde {K} _ {k, t _ {f}}) = - \operatorname{vec} (\tilde {U} _ {t _ {1}: t _ {n}}), \tag {72}$$

where the matrices

$$\tilde {X} _ {t _ {1}: t _ {n}} = \left[ \tilde {x} _ {k} (t _ {1}) \tilde {x} _ {k} (t _ {2}) \dots \tilde {x} _ {k} (t _ {n}) \right],
\tilde {U} _ {t _ {1}: t _ {n}} = \left[ \begin{array}{l l l l} \tilde {u} _ {k} (t _ {1}) & \tilde {u} _ {k} (t _ {2}) & \dots & \tilde {u} _ {k} (t _ {n}) \end{array} \right] \tag {73}
$$
