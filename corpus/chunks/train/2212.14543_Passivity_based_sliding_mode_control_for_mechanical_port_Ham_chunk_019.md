Remark 2. In most cases of multiple input multiple output (MIMO) systems, many sliding mode controllers are designed so that each element of sliding variables $\sigma _ { i }$ ’s are decoupled as in (4). On the other hand, Theorem 1 shows that sliding mode control can be achieved if

$$\dot {\sigma} = - A \operatorname{sgn} \sigma$$

holds with a positive definite matrix $A \in \mathbb { R } ^ { m \times m }$ . This implies that sliding mode is realized in the MIMO case even if the matrix A is not diagonal. This result is useful in the design of usual MIMO sliding-mode controllers.
