Algorithm 1 Multitask Certainty-Equivalent Control   
1: Initialize: $\hat{K}_{1}^{(i)} \leftarrow K_{0}^{(i)} \forall i \in [M], T \leftarrow \tau_{1}2^{k_{\mathrm{fin}}-1}$ 2: for $k = 1, 2, \ldots, k_{\mathrm{fin}}$ 3: for all systems $i = 1, \ldots, M$ (in parallel) // Data collection
4: for $t = \tau_{k-1}, \tau_{k-1} + 1, \ldots, \tau_{k}$ 5: If $\|x_{t}^{(i)}\|^{2} \geq x_{b}^{2} \log T$ or $\|\hat{K}_{k}^{(i)}\| \geq K_{b}$ 6: Abort and play $K_{0}^{(i)}$ forever
7: Play $u_{t}^{(i)} = \hat{K}_{k}^{(i)} x_{t}^{(i)} + \sigma_{k} g_{t}^{(i)}$ // Exploration
8: end for
9: end for
10: $\widehat{\Theta}_{k}^{(1:M)} \leftarrow \mathbb{R}\mathbb{C}\mathbb{S}\mathbb{I}(\widehat{\Theta}_{k-1}^{(1:M)}, \mathcal{D}_{\tau_{k}}^{(1:M)}, N_{c}, N, \eta)$ // Robust SysID
11: Update the controller: $\hat{K}_{k+1}^{(i)} \leftarrow K(\widehat{\Theta}_{k}^{(1:M)}), \forall i \in [M]$ // Controller update
12: $\tau_{k+1} \leftarrow 2\tau_{k}$ 13: end for

Algorithm 2 RCSI: Robust Clustered System Identification   
1: for $n = 0, 1, \ldots, N - 1$ do
2: Systems receive all the current cluster estimated models $\{\widehat{\Theta}_{j}\}_{j\in[N_{c}]}$ 3: for $i = 1, \ldots, M$ (in parallel) do // Cluster identity estimation
4: $\hat{j} = \arg\min_{j \in [N_{c}]} \|X^{(i)} - \widehat{\Theta}_{j}Z^{(i)}\|^{2}$ 5: end for
6: Construct the cluster identity set $C_{\hat{j}}$ 7: $G_{\ell}(\widehat{\Theta}^{(i)}) \leftarrow (X^{(\ell)} - \widehat{\Theta}^{(i)}Z^{(\ell)})Z^{(\ell)\top}(Z^{(\ell)}Z^{(\ell)\top})^{-1}, \forall \ell \in \mathcal{C}_{\hat{j}} // Update$ 8: for $i = 1, \ldots, M$ (in parallel) do // Model estimation
9: $\widehat{\Theta}^{(i)} \leftarrow \widehat{\Theta}^{(i)} + \eta F\left(\left\{G_{\ell}(\widehat{\Theta}^{(i)})\right\}_{\ell \in \mathcal{C}_{\hat{j}}}\right) // Aggregation$ 10: end for
11: end for
