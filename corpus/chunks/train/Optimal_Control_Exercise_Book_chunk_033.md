# Solution:

From the definition of first variation, if we take the following function:

$$h (\alpha) := J (y + \alpha \eta (x)) \tag {60}$$

then the first variation functional will be defined by:

$$\delta J _ {y} (\eta (x)) = h ^ {\prime} (0) \tag {61}$$

where $\eta ( x )$ a perturbation and $\alpha \in \mathbb { R }$ . The function becomes:

$$J (y) = \int_ {0} ^ {1} g ^ {\prime} (y (x + \alpha \eta (x)) d x \tag {62}$$

Thus, we calculate the derivative

$$h ^ {\prime} (0) = \frac {\partial}{\partial \alpha} h (0) = \int_ {0} ^ {1} g ^ {\prime} (y (x + \alpha \eta (x))) \eta (x) d x = \int_ {0} ^ {1} g ^ {\prime} (y (x)) \eta (x) d x \tag {63}$$

And finally, since $h ^ { \prime } ( 0 ) = \delta J _ { y } ( \eta )$ , then:

$$\delta J _ {y} (\eta (x)) = \int_ {0} ^ {1} g ^ {\prime} (y (x)) \eta (x) d x \tag {64}$$
