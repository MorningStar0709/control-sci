Definition 1 (Shift Operator, [11]). Let $f = \{ f [ k ] \} _ { k \in \mathbb { Z } }$ denote a discrete-time sequence, where $f [ k ] ~ \in ~ \mathbb { R } ^ { n }$ represents the value of the sequence at index $k \in \mathbb { Z } .$ . Assume that $f [ k ] \ i s$ obtained by sampling a continuous-time signal $f _ { a } ( t ) \in \mathbb { R } ^ { n }$ with sampling period $\epsilon > 0 ,$ , i.e., $f [ k ] ~ = ~ f _ { a } ( k \epsilon )$ , $k \in \mathbb { Z } .$ The forward shift operator q is defined by $q f [ k ] = f [ k + 1 ]$ . Similarly, the backward shift operator is $q ^ { - 1 } f [ k ] = f [ k - 1 ]$ ]. In the z-domain, q is identified with the complex variable $z \in \mathbb { C } .$ .
