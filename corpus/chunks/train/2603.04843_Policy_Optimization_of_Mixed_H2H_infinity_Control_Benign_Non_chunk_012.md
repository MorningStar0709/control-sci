# 2.2 Policy optimization for mixed $\mathcal { H } _ { 2 } / \mathcal { H } _ { \infty }$ design

Despite simple formulation, problem (5) is challenging to solve [40, 41]. A classical approach from [4] is to minimize an upper bound on the $\mathcal { H } _ { 2 }$ cost over the feasible set $\displaystyle { { \cal { K } } _ { \beta } }$ .

Recall that the $\mathcal { H } _ { 2 }$ cost for any stabilizing policy $K \in \kappa$ is $\| { \bf T } _ { 2 } ( K ) \| _ { \mathcal { H } _ { 2 } } ^ { 2 } = \mathrm { t r } ( ( Q _ { 2 } + K ^ { \top } R _ { 2 } K ) \hat { X } _ { K } )$ , where $\hat { X } _ { K }$ is the unique solution to the Lyapunov equation

$$A _ {K} \hat {X} _ {K} + \hat {X} _ {K} A _ {K} ^ {\mathsf {T}} + W = 0. \tag {7}$$

To enforce the $\mathcal { H } _ { \infty }$ constraint in (6), we instead consider the stabilizing solution $X _ { K }$ to the Riccati equation [3–5]

$$A _ {K} X _ {K} + X _ {K} A _ {K} ^ {\mathsf {T}} + \beta^ {- 2} X _ {K} S _ {K} X _ {K} + W = 0, \tag {8}$$

where $S _ { K } : = Q _ { \infty } + K ^ { \mathsf { T } } R _ { \infty } K$ . By the bounded real lemma (see Lemma 1), a policy $K \in \mathcal { K } _ { \beta }$ if and only if (8) admits a unique stabilizing solution $X _ { K } \succeq 0$ such that the matrix $A _ { K } + \mathsf { \bar { \beta } } ^ { - 2 } X _ { K } S _ { K }$ is Hurwitz. Subtracting (7) from (8) gives

$$A _ {K} (X _ {K} - \hat {X} _ {K}) + (X _ {K} - \hat {X} _ {K}) A _ {K} ^ {\intercal} + \beta^ {- 2} X _ {K} S _ {K} X _ {K} = 0.$$

Since $A _ { K }$ is Hurwitz and the last term is positive semidefinite, it follows that $X _ { K } { - } \hat { X } _ { K } { \succeq } 0$ , and hence the $\mathcal { H } _ { 2 }$ cost admits an upper bound $\| { \bf T } _ { 2 } ( K ) \} | _ { \mathcal { H } _ { 2 } } ^ { 2 } \le \mathrm { t r } ( ( Q _ { 2 } + K ^ { \mathsf { T } } R _ { 2 } K ) X _ { K } )$ for all $K \bar { \in } \mathcal { K } _ { \beta }$ . Therefore, we define the mixed cost

$$J _ {\text { mix }} (K) := \operatorname{tr} ((Q _ {2} + K ^ {\mathsf {T}} R _ {2} K) X _ {K}), \quad \forall K \in \mathcal {K} _ {\beta}, \tag {9}$$

where $X _ { K }$ is the unique stabilizing solution to (8).

In sum, we consider the following mixed design [3–5]

$$\inf _ {K \in \mathcal {K} _ {\beta}} J _ {\text { mix }} (K). \tag {10}$$

Unlike the $\mathcal { H } _ { 2 }$ cost, $J _ { \mathrm { m i x } }$ is evaluated using the Riccati solution $X _ { K }$ from (8) rather than the Lyapunov solution $\hat { X } _ { K }$ from (7). Since the cost and constraint in (10) are in general defined by distinct signals $z _ { 2 }$ and $z _ { \infty } .$ we refer to this formulation as the two-channel mixed $\mathcal { H } _ { 2 } / \mathcal { H } _ { \infty }$ design.
