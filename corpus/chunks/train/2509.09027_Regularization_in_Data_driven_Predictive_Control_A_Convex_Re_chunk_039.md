# 6.1 Non-deterministic LTI system

Experiment setup. We first consider an LTI system from [18], which is a triple-mass-spring system with n = 8 states, m = 2 inputs (two stepper motors), and $p = 3$ outputs (disc angles). We consider data collection with additive Gaussian measurement noises $\omega \sim \mathcal { N } ( 0 , \sigma I )$ . We utilize the data sets with various sizes and noise levels that are $T = 4 0 0 , 6 0 0$ , 800 and σ from 0.02 to 0.1 in increments of 0.02. The prediction horizon and the initial sequence are chosen as $N = 4 0$ and $T _ { \mathrm { i n i } } = 4 .$ , respectively. We choose $Q = I , R = 0 . 1 I , \mathcal { U } = [ - 0 . 7 , 0 . 7 ]$ and $\Gamma , \mathcal { V } = \mathbb { R } ^ { 3 }$ unless otherwise specified.
