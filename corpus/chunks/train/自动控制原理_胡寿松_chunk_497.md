$$r (t) = 1 (t), \quad c ^ {*} (0) = c ^ {*} (T) = 0;$$

$$c (0) = c (T) = 0, \quad r (n T) = n (n = 0, 1, 2, \dots);$$

(3) $c(k + 3) + 6c(k + 2) + 11c(k + 1) + 6c(k) = 0,$

$$c (0) = c (1) = 1, \quad c (2) = 0;$$
