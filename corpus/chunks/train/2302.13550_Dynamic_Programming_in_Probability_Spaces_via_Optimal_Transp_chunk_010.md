Problem 3.2 (Discrete-time optimal control in probability spaces). Let $N \in$ $\mathbb { N } _ { \geq 1 }$ . For dynamics $f _ { k } : X _ { k } \times U _ { k } \to X _ { k + 1 }$ , costs $g _ { k } : X _ { k } \times U _ { k } \times R _ { k } \to \bar { \mathbb { R } } _ { \ge 0 }$ and $g _ { N } : X _ { N } \times R _ { N } \to \bar { \mathbb { R } } _ { \ge 0 }$ , initial condition $\mu \in \mathcal P ( X _ { 0 } )$ , and reference trajectory $\rho =$ $( \rho _ { 0 } , \ldots , \rho _ { N } ) \in \mathcal { P } ( R _ { 0 } ) \times \ldots \times \mathcal { P } ( R _ { N } )$ , find the joint state-input distribution $\lambda _ { k } \in$ $\mathcal { P } ( X _ { k } \times U _ { k } )$ which solve

$$
J(\mu ,\rho) = \inf_{\substack{\mu_{k}\in \mathcal{P}(X_{k})\\ \lambda_{k}\in \mathcal{P}(X_{k}\times U_{k})}}\mathcal{K}[g_{N}](\mu_{N},\rho_{N}) + \sum_{k = 0}^{N - 1}\mathcal{K}[g_{k}](\lambda_{k},\rho_{k})
$$

$\mathrm { s . t . } \qquad \mu _ { k + 1 } = f _ { k \# } \lambda _ { k } , \quad \mu _ { 0 } = \mu ,$

$$(\mathrm{proj} _ {X _ {k}} ^ {X _ {k} \times U _ {k}}) _ {\#} \lambda_ {k} = \mu_ {k}.$$
