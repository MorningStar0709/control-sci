# 2.5 Discretization

We present a space-discretized version of the GPL dynamic (10). The domain  is divided into $N \in \mathbb { N }$ $( N \geq 2 )$ cells, where $\Omega _ { i } = \left[ \left( i - 1 \right) / N , i / N \right) \ \left( i = 1 , 2 , . . . , N - 1 \right)$ and $\Omega _ { \scriptscriptstyle N } = \left[ 1 - 1 / N , 1 \right]$ . The midpoint of $\Omega _ { i }$ is $x _ { i - 1 / 2 } = \left( i - 1 / 2 \right) / N$ . The dynamic (10) with $A = \Omega _ { i }$ is discretized as

$$
\begin{array}{l} \frac {\mathrm{d} \hat {\mu} (t , \Omega_ {i})}{\mathrm{d} t} \\ = \frac {1}{N} \sum_ {j = 1} ^ {N} a \left(x _ {j - 1 / 2}, x _ {i - 1 / 2}, \hat {\mu}\right) \hat {\mu} (t, \Omega_ {j}) - \left(\frac {1}{N} \sum_ {j = 1} ^ {N} a \left(x _ {i - 1 / 2}, x _ {j - 1 / 2}, \hat {\mu}\right)\right) \hat {\mu} (t, \Omega_ {i}) \quad \text { for } i = 1, 2, \dots , N \text { and } t > 0, \tag {14} \\ \end{array}
$$

where $\hat { \mu } \big ( t , \Omega _ { i } \big )$ is the discretization of $\mu { \left( t , \Omega _ { i } \right) }$ . We used $\int _ { \Omega _ { i } } \mathrm { d } \boldsymbol { x } = \boldsymbol { N } ^ { - 1 } \quad ( \ i = 1 , 2 , . . . , N )$ . The initial condition $\mu ( 0 , \cdot ) = \mu _ { 0 } \in \mathscr { P }$ is discretized on each $\Omega _ { i }$ . The semi-discretized dynamic (14) serves as an approximation to the original one (10), particularly it can be seen as a nonlinear Markov chain with the transition matrix  . $\left\{ a \left( x _ { i - 1 / 2 } , x _ { j - 1 / 2 } , \hat { \mu } \right) \right\} _ { 1 \leq i , j \leq N }$

To continue, we assume that the utility has the form

$$U (x, \mu) = g \left(\int_ {\Omega} f (x, y) \mu (\mathrm{d} y)\right) \text {for any} x \in \Omega \text {and} \mu \in \mathcal {P}, \tag {15}$$

where $f : \Omega \times \Omega  \mathbb { R }$ is continuous on $\Omega \times \Omega$ ，and $g : \mathbb { R }  \mathbb { R }$ is Lipschitz continuous such that $\left| g \left( z _ { 1 } \right) - g \left( z _ { 2 } \right) \right| \leq L _ { g } \left| z _ { 1 } - z _ { 2 } \right|$ for any $z _ { 1 } , z _ { 2 } \in \mathbb { R }$ , with a constant $L _ { g } > 0$ . This utility satisfies the condition (3). If $g \left( z \right) = z$ , then (15) is the average payoff as conventionally considered in evolutionary games (Sandholm, 2020). Our setting generalizes it in a nonlinear way. We discretize (15) as

$$U \left(x _ {j - 1 / 2}, \hat {\mu}\right) = g \left(\sum_ {k = 1} ^ {N} f \left(x _ {j - 1 / 2}, x _ {k - 1 / 2}\right) \hat {\mu} \left(t, \Omega_ {k}\right)\right), \quad j = 1, 2, \dots , N. \tag {16}$$
