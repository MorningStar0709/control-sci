# 1 Introduction

Let $\varepsilon > 0$ be a positive number, $\mathcal { H } _ { 1 }$ and $\mathcal { \mathrm { H } } _ { 2 }$ be finite-dimensional Hilbert spaces, $C \in$ $\mathrm { H } ( \mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 } )$ be a Hermitian operator on $\mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 } , \rho$ and σ be density matrices, respectively, on $\mathcal { H } _ { 1 }$ and $\mathcal { \mathrm { H } } _ { 2 }$ . The von Neumann entropy regularized Non-commutative Optimal Transport [14] is given by

$$\mathcal {Q O T} _ {\varepsilon} [ \rho , \sigma ] = \inf \left\{\mathrm{Tr} [ C \Gamma ] + \varepsilon \mathcal {S} [ \Gamma ] : \Gamma \mapsto (\rho , \sigma) \right\}, \tag {1.1}$$

where $\ S [ \Gamma ] = \mathrm { T r } [ \Gamma \left( \log \Gamma - \mathbb { I } _ { d } \right) ]$ is the von Neumann Entropy and $\Gamma \mapsto ( \rho , \sigma )$ denotes the set of density matrices Γ in $\mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 }$ having partial traces given by $\rho$ and $\sigma$ .

The static formulations of Non-commutative Optimal Transport $( \mathrm { e . g . , } [ 4 , 5 , 8 , 1 1 ] )$ ) and its von Neumann entropy regularized counterpart [14, 20] are motivated by extending the (static) Optimal Transport Theory for probability measures (e.g., [1, 9, 21, 22]) into quantum states (e.g., density operators or density matrices).

In this note instead, we focus on the extension of the static formulation of von Neumann entropy regularized Non-commutative Optimal Transport for positive semi-definite and Hermitian operators (or Unbalanced Non-commutative Optimal Transport) given by

$$\mathcal {Q O T} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} [ \rho , \sigma ] = \inf \left\{\mathrm{Tr} [ C \Gamma ] + \varepsilon \mathcal {S} [ \Gamma ] + \tau_ {1} \mathcal {E} [ \Gamma_ {1} | \rho ] + \tau_ {2} \mathcal {E} [ \Gamma_ {2} | \sigma ]: \Gamma \geq 0 \text {and} \Gamma^ {*} = \Gamma \right\}, (1. 2)$$
