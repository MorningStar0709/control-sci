# 3. Main Algorithms and Results

In this section, we propose a distributed algorithm for multi-agent formation control that aims to achieve the objectives stated in equations (2)-(5). To accomplish this, we decompose the overall objective into two sets of tasks. The first set (Section 3.1), represented by equations (2)- $\left( 4 \right) .$ , focuses on high-level objectives and involves the development of a nominal control approach. The second set (Section $_ { 3 . 2 ) }$ , represented by equation (5), pertains to a lower-level task designed as an inherent capability of the system and incorporates the concept of control barrier functions with the nominal control. Motivated by [44, 45], the proposed control law for the SRFC for agent i is given as a solution to QP problem:

$$u _ {i} ^ {*} = \arg \min _ {z _ {i}} \frac {1}{2} \left\| z _ {i} - \bar {u} _ {i} \right\| ^ {2},$$

such that

$$A _ {i k} + B _ {i k} ^ {T} z _ {i} \geq 0 \forall i \in \mathcal {V}, k = 1, 2, \dots , K,\frac {1}{2} \bar {A} _ {i j} + \bar {B} _ {i j} ^ {T} z _ {i} \geq 0 \forall j \in \mathcal {N} _ {i}; j \neq i; i \in \mathcal {V},\left\| z _ {i} \right\| _ {\infty} \leq u _ {\max} \forall i \in \mathcal {V},$$

where ${ \bar { u } } _ { i }$ is the nominal control of the agent solving the high-level objectives. The definitions $A , { \bar { B } } , { \bar { A } } .$ , and B¯ are discussed later in Section 3.2. The nominal control is given by

$$\bar {u} _ {i} = \mathbb {1} _ {\Omega} \dot {v} _ {d} - c _ {1} \sum_ {j \in \mathcal {N} _ {i}} \nabla_ {p _ {i}} \Phi_ {i j} - c _ {2} \sum_ {j \in \mathcal {N} _ {i}} \hat {v} _ {i j}- c _ {3} \mathbb {1} _ {i \in \mathcal {L}} \mathbb {1} _ {\Omega^ {-}} \frac {p _ {i} - \mathcal {P} _ {\Omega} (p _ {i})}{\| p _ {i} - \mathcal {P} _ {\Omega} (p _ {i}) \|} - c _ {4} \mathbb {1} _ {i \in \mathcal {L}} \mathbb {1} _ {\Omega} \tanh (c _ {5} (p _ {i} - \gamma_ {i})), \tag {6}$$

where $\mathbb { 1 } _ { i \in { \mathcal { L } } }$ is an indicator function whose value is 1 if agent i is a stealthy leader, $\mathbb { 1 } _ { \Omega }$ and $\mathbb { 1 } _ { \Omega ^ { - } }$ are indicator functions corresponding on whether the agent is inside the target region or not. Here, $\Phi _ { i j }$ represents the artificial potential function, which is defined later in Section $3 { \cdot } 1 ,$ , and $\mathcal { P } _ { \Omega } ( \boldsymbol { p } _ { i } )$ ) refers to the projection of the point $p _ { i }$ onto the set Ω. The estimate of the relative velocity vector $\hat { v } _ { i j }$ is defined using an auxiliary variable $\phi _ { i j }$ as

$$\hat {v} _ {i j} = - \eta (\phi_ {i j} - (p _ {i} - p _ {j})), \tag {7}\dot {\phi} _ {i j} = - \eta (\phi_ {i j} - (p _ {i} - p _ {j})), \tag {8}$$

and auxiliary variable $\gamma _ { i }$ is defined as

$$\dot {\gamma} _ {i} = v _ {d} + c _ {5} (p _ {i} - \gamma_ {i}). \tag {9}$$

Here $c _ { 1 } , c _ { 2 } , c _ { 3 } , c _ { 4 } , c _ { 5 } .$ , and η are strictly positive, $\phi _ { i j } ( 0 )$ ) is initialized with $( p _ { i } ( 0 ) - p _ { j } ( 0 ) )$ , whereas $\gamma _ { i } ( 0 )$ can take any arbitrary value.
