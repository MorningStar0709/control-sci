\begin{array}{l} \left(R - \frac {c}{\mu}\right) \sum_ {j = 0} ^ {k - 1} \frac {(k \rho) ^ {j}}{j !} + \frac {k ^ {k}}{k !} \sum_ {j = k} ^ {k + N - 1} \left(R - \frac {c (j + 1)}{k \mu}\right) \rho^ {j} \\ = \left(R - \frac {k c \rho}{\lambda}\right) \sum_ {j = 0} ^ {k - 1} \frac {(k \rho) ^ {j}}{j !} + \frac {k ^ {k}}{k !} \sum_ {j = k} ^ {k + N - 1} \left(R - \frac {c (j + 1) \rho}{\lambda}\right) \rho^ {j} \\ = \sum_ {j = 0} ^ {k - 1} \left(R - \frac {c j}{\lambda}\right) \frac {(k \rho) ^ {j}}{j !} - \frac {c k ^ {k} \rho^ {k}}{\lambda (k - 1) !} + \frac {k ^ {k}}{k !} \sum_ {j = k + 1} ^ {k + N - 1} \left(R - \frac {c j}{\lambda}\right) \rho^ {j} + \frac {k ^ {k}}{k !} \left(R \rho^ {k} - \frac {c (N + k)}{\lambda} \rho^ {k + N}\right) \\ = \sum_ {j = 0} ^ {k - 1} \left(R - \frac {c j}{\lambda}\right) \frac {(k \rho) ^ {j}}{j !} + \frac {k ^ {k}}{k !} \sum_ {j = k} ^ {k + N - 1} \left(R - \frac {c j}{\lambda}\right) \rho^ {j} - \frac {c k ^ {k} (N + k)}{\lambda k !} \rho^ {k + N}, \tag {41} \\ \end{array}
$$

where in the penultimate step we combine terms in both summations with a shift in the second term. By Descartes’ rule of signs, (41) has exactly one positive root, denoted as $\rho ^ { * }$ , which gives the threshold service rate $\begin{array} { r } { \mu ^ { * } = \frac { \lambda } { k \rho ^ { * } } } \end{array}$ . Note that, in the Erlang-B queueing system, we previously identified $\mu ^ { * }$ as $c / R$ in Section 3.1.
