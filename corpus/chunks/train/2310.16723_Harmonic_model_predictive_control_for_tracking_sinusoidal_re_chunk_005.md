# II. ADMISSIBLE HARMONIC SIGNALS

Definition 1 (Harmonic signal). A trajectory $\boldsymbol { v } ( \cdot ) \in \mathbb { R } ^ { m }$ is a harmonic signal if it satisfies

$$v (t) = v _ {e} + v _ {s} \sin (w t) + v _ {c} \cos (w t), \tag {1}$$

for some parameters $v _ { e } , v _ { s } , v _ { c } \in \mathbb { R } ^ { m }$ and frequency $w > 0 .$ .

In the following, we use a bold $\textbf { v } \doteq \ ( v _ { e } , v _ { s } , v _ { c } )$ as a shorthand to indicate the parameters of a harmonic signal $v ( \cdot )$ , where their dimension is inferred from the dimension of v(·).

Consider a system described by a controllable linear timeinvariant state-space model

$$x (t + 1) = A x (t) + B u (t), \tag {2}$$

where $\boldsymbol { x } ( t ) \in \mathbb { R } ^ { n _ { \boldsymbol { x } } }$ and $u ( t ) \in \mathbb { R } ^ { n _ { u } }$ are the state and control input at the discrete time instant t, respectively, subject to

$$\underline {{y}} \leq E x (t) + F u (t) \leq \overline {{y}}, \forall t, \tag {3}$$

where $E \in \mathbb { R } ^ { n _ { y } \times n _ { x } } , F \in \mathbb { R } ^ { n _ { y } \times n _ { u } } , y , \overline { { y } } \in \mathbb { R } ^ { n _ { y } }$ and $y < { \overline { { y } } } .$ .
