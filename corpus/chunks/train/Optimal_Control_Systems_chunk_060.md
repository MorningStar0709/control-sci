# 2.1.1 Function and Functional

We discuss some fundamental concepts associated with functionals along side with those of functions.

(a) Function: A variable $x$ is a function of a variable quantity $t$ , (written as $x(t) = f(t)$ ), if to every value of $t$ over a certain range of $t$ there corresponds a value $x$ ; i.e., we have a correspondence: to a number $t$ there corresponds a number $x$ . Note that here $t$ need not be always time but any independent variable.
