Finally, a system that has parametric uncertainty may be modeled as

$$x ^ {+} = f (x, u, \theta)$$

in which $\theta$ represents parameters of the system that are known only to the extent that they belong to a compact set Θ. A much studied example is

$$x ^ {+} = A x + B u$$

in which $\theta : = ( A , B )$ may take any value in $\Theta : = \operatorname { c o } \{ ( A _ { i } , B _ { i } ) | i \in { \mathcal { I } } \}$ where $\mathcal { I } = \{ 1 , 2 , \ldots , I \}$ , say, is an index set.

It is possible, of course, for all these types of uncertainty to occur in a single application. In this chapter we focus on the first and third types of uncertainty, namely, additive disturbance and parameter uncertainty. Output MPC, where the controller employs an estimate of the state, rather than the state itself, is treated in Chapter 5.
