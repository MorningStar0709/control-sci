# C. Relative Kinematics and Dynamics of Satellites

Let the relative translational position and the relative translational velocity of the chaser satellite with respect to the target satellite is given by $\tilde { e } _ { p } = \tilde { p } _ { c } - \mathbf { B } _ { \mathbf { t } } ^ { \mathbf { c } } \tilde { p } _ { t }$ and $\tilde { e } _ { v } = \tilde { v } _ { c } - \mathbf { B _ { t } ^ { c } } \tilde { v } _ { t } .$ , respectively. The relative acceleration of chaser satellite with respect to the target satellite can described as shown below.

$$
\begin{array}{l} \dot {\tilde {e}} _ {v} = \frac {\tilde {F} _ {c}}{M _ {c}} + \frac {\tilde {F} _ {j _ {2} , c}}{M _ {c}} + \frac {\tilde {F} _ {d c}}{M _ {c}} + (\boldsymbol {\Omega} (\tilde {e} _ {\omega}) + \boldsymbol {\Omega} (\tilde {\omega} _ {t})) B _ {t} ^ {c} \tilde {v} _ {t} \\ - \mathbf {\Omega} (\tilde {\omega} _ {c}) \tilde {v} _ {c} - \mathbf {B} _ {t} ^ {c} \frac {\tilde {F} _ {d t}}{M _ {t}} \\ \end{array}
$$

Relative translational dynamics is obtained by multiplying the above equation with the mass of the chaser satellite $M _ { c }$ yields

$$
\begin{array}{l} M _ {c} \dot {\tilde {e}} _ {v} + M _ {c} \boldsymbol {\Omega} (\tilde {\omega} _ {c}) \tilde {v} _ {c} = \tilde {F} _ {c} + \tilde {F} _ {j _ {2}, c} + \tilde {F} _ {d c} - M _ {c} B _ {t} ^ {c} \frac {\tilde {F} _ {d t}}{M _ {t}} + \\ M _ {c} \left(\boldsymbol {\Omega} \left(\tilde {e} _ {\omega}\right) + \boldsymbol {\Omega} \left(\tilde {\omega} _ {t}\right)\right) B _ {t} ^ {c} \tilde {v} _ {t} \tag {9} \\ \end{array}
$$

The relative attitude tracking error of chaser satellite w.r.t target satellite is given by $\tilde { e } _ { r } = l o g ( [ \boldsymbol { B } _ { t } ^ { i } ] ^ { T } [ \boldsymbol { B } _ { c } ^ { i } ] )$ , where $l o g ( . )$ is the logarithmic mapping from $S O ( 3 )$ to $\mathbb { R } ^ { 3 }$ . The relative angular velocity vector of chaser satellite w.r.t target satellite is expressed as $\tilde { e } _ { \omega } = \tilde { \omega } _ { c } - \mathbf { B } _ { \mathbf { t } } ^ { \mathbf { c } } \tilde { \omega } _ { t }$ , where $\mathbf { B _ { t } ^ { c } } \in \mathbb { R } ^ { 3 \times 3 }$ denotes the rotation matrix from the $E _ { t }$ to $E _ { c }$ and is formulated as $\mathbf { B _ { t } ^ { c } } = \left( \mathbf { B _ { c } ^ { i } } \right) ^ { T } \mathbf { B _ { t } ^ { i } }$ . The relative angular acceleration is given by performing time-derivative as shown below.

$$\dot {\tilde {e}} _ {\omega} = \dot {\tilde {\omega}} _ {c} + \tilde {\omega} \times \mathbf {B _ {d} ^ {c}} \omega_ {d} - \mathbf {B _ {d} ^ {c}} \dot {\tilde {\omega}} _ {d}$$

where subscript $( . ) _ { d }$ represents the desired position or orientation to be achieved by the chaser satellite at the end of docking phase. Relative rotational dynamics is obtained by multiplying the above equation by $\mathbf { J } _ { c }$ as shown below.

$$\mathbf {J} _ {\mathbf {c}} \dot {\tilde {e}} _ {\omega} = \mathbf {J} _ {\mathbf {c}} \dot {\tilde {\omega}} _ {c} + \mathbf {J} _ {\mathbf {c}} \tilde {\omega} \times \mathbf {B} _ {\mathbf {d}} ^ {\mathbf {c}} \omega_ {d} - \mathbf {J} _ {\mathbf {c}} \mathbf {B} _ {\mathbf {d}} ^ {\mathbf {c}} \dot {\tilde {\omega}} _ {d} \tag {10}$$
