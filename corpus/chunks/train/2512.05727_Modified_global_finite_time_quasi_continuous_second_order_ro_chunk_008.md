# 3.3 Numerical illustration

The obtained analytic solutions (7), (8) and (11)–(14) are compared with numerical simulation of the closed-loop system (1), (2) for the case $d \ = \ 0$ and three different $x ( 0 ) \in \dot { U } \vee \dot { C } _ { a }$ . The assign feedback gain is $\gamma = 1 0 0 . \mathrm { ~ A ~ }$ standard fixed-step numerical solver of Matlab/Simulink is used with the assigned sampling rate of 10 kHz. Also the control values $u ( t )$ for both, the analytic solutions and numerical simulation, are compared for the sake of completeness. The results are depicted in Fig. 3.
