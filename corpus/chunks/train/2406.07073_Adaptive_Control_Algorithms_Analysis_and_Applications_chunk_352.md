Given the desired dynamics of the observer specified by a polynomial $P _ { D } ( q ^ { - 1 } ) =$ det $\boldsymbol { I } - \boldsymbol { z } ^ { - 1 } ( \boldsymbol { A } - \boldsymbol { k } \boldsymbol { \dot { c } } ^ { T } ) ]$ , one has:

$$k ^ {T} = \left[ k _ {1}, \dots , k _ {n} \right] \tag {7.287}$$

with:

$$k _ {i} = p _ {i} - a _ {i} \tag {7.288}$$

(where $p _ { i }$ are the coefficients of the monic polynomials $P _ { D } )$ . This observer corresponds also to a steady state optimal Kalman filter for a disturbance $\bar { \boldsymbol { v } } ( t )$ added to (7.277) and characterized by:

$$\bar {v} (t) = C (q ^ {- 1}) w (t); \quad C (q ^ {- 1}) = P _ {D} (q ^ {- 1}) \tag {7.289}$$

where $w ( t )$ is a white noise sequence. Therefore the control law expressed in terms of $\bar { e } _ { u } ( t )$ takes the form:

$$\bar {e} _ {u} (t) = - \frac {b ^ {T} \Gamma A}{b ^ {T} \Gamma b + \lambda} \hat {x} (t) = - l ^ {T} \hat {x} (t) \tag {7.290}$$

where ${ \hat { x } } ( t )$ is given by (7.285) and (7.286) and Γ is the solution of (7.284). Taking into account (7.290), the observer equation in closed loop becomes:

$$\hat {x} (t + 1) = A \hat {x} (t) - b l ^ {T} \hat {x} (t) - k c ^ {T} \hat {x} (t) + k e _ {y} (t) \tag {7.291}$$

and the transfer function between $e _ { y } ( t )$ and $e _ { u } ( t )$ will be given by:

$$H _ {c} (z) = \frac {R (z)}{S (z)} = l ^ {T} (z I - A + k c ^ {T} + b l ^ {T}) ^ {- 1} k \tag {7.292}$$

where:

$$S ^ {\prime} (z) = \det (z I - A + k c ^ {T} + b l ^ {T}) \tag {7.293}R ^ {\prime} (z) = l ^ {T} \operatorname{adj} \left(z I - A + k c ^ {T} + b l ^ {T}\right) k \tag {7.294}$$

This allows to write the controller under the form:

$$
\begin{array}{l} S ^ {\prime} (q ^ {- 1}) H _ {S} (q ^ {- 1}) e _ {u} (t) + R ^ {\prime} (q ^ {- 1}) H _ {R} (q ^ {- 1}) e _ {y} (t) \\ = S (q ^ {- 1}) e _ {u} (t) + R (q ^ {- 1}) e _ {y} (t) \tag {7.295} \\ \end{array}
$$

from which one obtains the final RST controller form by taking in account the expressions for $e _ { u } ( t )$ and $e _ { y } ( t )$ :

$$S (q ^ {- 1}) u (t) = - R (q ^ {- 1}) y (t) + T (q ^ {- 1}) y ^ {*} (t + d + 1) \tag {7.296}$$

where:

$$T \left(q ^ {- 1}\right) = \beta P = \beta \left[ A S + q ^ {- d - 1} B ^ {*} R \right] \tag {7.297}$$

This controller assures certain tracking performances (defined by $y ^ { * }$ and $B ^ { * } )$ independently of the resulting closed-loop poles (which will include the poles of the observer defined by $P _ { D } )$ .

For further details on this control strategy see Samson (1982), Lam (1982), M’Saad et al. (1990), M’Saad and Sanchez (1992).
