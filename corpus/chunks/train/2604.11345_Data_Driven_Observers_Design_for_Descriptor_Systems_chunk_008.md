For descriptor systems, the singularity of matrix E implies that the state response consists of not only smooth dynamic modes but also algebraic behaviors (or noncausal modes) residing in the fast subsystem. The following assumption on the observability of the fast subsystem is standard to guarantee the existence of a normal observer [23,14,10,7].

Assumption 1 The fast subsystem of (1) is observable, i.e. $r k ( [ C _ { 2 } ^ { \top } , R ^ { \top } C _ { 2 } ^ { \top } , \cdots , ( R ^ { n _ { 2 } - 1 } ) ^ { \top } C _ { 2 } ^ { \top } ] ^ { \top } ) = n _ { 2 }$ , which is equivalent to (see [23])

$$
r k \left[ \begin{array}{l} E \\ C \end{array} \right] = n. \tag {8}
$$

Remark 1 Condition (8) is also called the dual normalizability condition [23, Chap 8]. This condition guarantees that the state x(k) at any time point k is uniquely determined by initial condition x(0) and the former inputs $u ( i )$ , together with former measurements $y ( i )$ , $i = 0 , 1 , . . . , k$ (i.e., the so-called Y-observability). How to check this assumption from data will be discussed in Corollary 1 subsequently.

If condition (8) holds, there exists a full row rank matrix $[ T \_ N ]$ (not unique) such that

$$T E + N C = I _ {n}. \tag {9}$$

By using (9), system (1) is rewritten

$$x (k + 1) = T A x (k) + T B u (k) + N y (k + 1). \tag {10}$$

Equation (10) reveals that the state $x ( k + 1 )$ can be explicitly expressed in terms of the current state, input, and one-step-ahead future output. In the non-singular case, i.e., $E = I _ { n }$ , we can choose $T = I _ { n } , N = 0$ so that (10) reduces to a standard state-space system.

Definition 3 A linear observer $\hat { \Sigma }$ is designed in the following form:

$$\hat {x} (k + 1) = A _ {O} \hat {x} (k) + B _ {O} ^ {u} u (k) + B _ {O} ^ {y} y (k) + N _ {O} y (k + 1), \tag {11}$$
