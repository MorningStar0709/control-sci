Direct policy gradient methods. Reinforcement learning (RL) is proposed to update θ through gradient descent, where the gradient $\nabla _ { \pmb { \theta } } J ( \pmb { \theta } )$ is approximated from trajectories of the system under the control policy $\pi _ { \theta }$ . For example, the policy gradient methods in (Sutton and Barto, 2018) shows that the gradient $\nabla _ { \pmb { \theta } } J ( \pmb { \theta } )$ can be equivalently computed by

$$\nabla_ {\boldsymbol {\theta}} J (\boldsymbol {\theta}) = \mathbb {E} _ {\boldsymbol {\tau} \sim p _ {\pi_ {\boldsymbol {\theta}}} (\boldsymbol {\tau})} \left[ c (\boldsymbol {\tau}) \sum_ {k = 0} ^ {K - 1} \nabla_ {\boldsymbol {\theta}} \log p _ {\pi_ {\boldsymbol {\theta}}} (\boldsymbol {u} (k) \mid \boldsymbol {y} (k)) \right], \tag {3}$$

where K is the length of the trajectory. The terms inside the expectation can be computed purely from observations and actions in the trajectory.

If $C = { \pmb I } _ { n }$ in (1), the states are directly measured and the control law is called state-feedback control. Otherwise, the control law is called output-feedback control. Note that if C is not full column rank, then $( { \pmb y } ( t ) , { \pmb u } ( t ) )$ cannot uniquely determine $\mathbf { \nabla } \mathbf { \boldsymbol { y } } ( t + 1 )$ ). Namely, the system is not a Markov decision process with respect to $( { \pmb y } ( t ) , { \pmb u } ( t ) )$ and it is difficult to use generic RL algorithms based on the quality function $Q ( ( \pmb { y } ( t ) , \pmb { u } ( t ) )$ (Jin et al., 2020). For illustration purpose, this paper focus on the policy gradient algorithm (3), and we consider both state and output feedback control.
