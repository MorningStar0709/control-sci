For a square symmetric matrix M , we denote its largest and smallest eigenvalues by $\lambda _ { \operatorname* { m a x } } ( M )$ and $\lambda _ { \operatorname* { m i n } } ( M )$ , respectively, and use tr[M] to denote the trace of M.We use the shorthand notation $[ n ] = \{ 1 , 2 , \dots , n \}$ for any positive integer n. Regularity of Functions. A mapping $f : \mathbb { R } ^ { n }  \mathbb { R } ^ { n }$ is said to be α−strongly monotone on K if there exists a constant $\alpha > 0$ such that $( x - y ) ^ { \top } ( f ( x ) - f ( y ) ) \geq$ $\alpha \| x - y \| ^ { 2 } , \forall x , y \in K$ . The mapping $f$ is said to be Lipschitz continuous on a set K if there exists a constant $\nu > 0$ such that $\| f ( x ) - f ( y ) \| \leq \nu \| x - y \|$ , $\forall x , y \in K$ . In this case, we also say that f is ν-Lipschitz continuous.
