# III. APPROXIMATION BY TRANSNNS

To derive TransNNs that approximate the virus spread dynamics (1), we introduce the following assumptions.

(A3) At time $k \ge 0 , \{ W _ { i j } ^ { k } : i , j \in [ n ] \}$ are independent and for each $i , j \in [ n ] , \dot { W } _ { i j } ^ { k }$ is independent of $\{ X _ { \ell } ( k ) : \ell \in$ $[ n ] , \ell \neq j \}$ .   
(A4) The events $\{ \{ X _ { i } ( k ) = 1 \} : i \in [ n ] , 0 \leq k \leq T \}$ are independent, with T as the terminal time.

The assumption (A3) introduces the independence of transmissions over different links at the current time k and the independence of transmissions with respect to the states except the current state at the sending node. Hence the assumption (A3) is stronger than the assumption (A2).

Assumptions (A1), (A3) and (A4) together allow us to write the update of the expected state by taking the (total) expectation inside the product on the right-hand side as follows: for $i \in [ n ]$ ,

$$
\begin{array}{l} \mathbb {E} \left(1 - X _ {i} (k + 1)\right) = \mathbb {E} \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - W _ {i j} ^ {k} X _ {j} (k)\right) \tag {6} \\ = \prod_ {j \in N _ {i} ^ {\circ k}} \mathbb {E} \left(1 - W _ {i j} ^ {k} X _ {j} (k)\right). \\ \end{array}
$$

Under (A1), (A3) and (A4), the joint probability distribution for the random variables $\{ Z _ { i j } ( k ) \triangleq 1 - W _ { i j } ^ { k } X _ { j } ( k ) , j \in N _ { i } ^ { \circ k } \}$ on the right-hand side is equal to the product of its marginal distributions. If (A3) and (A4) are not satisfied, the righthand side of (6) constitutes an approximation of the joint distribution by the product of the marginals, referred to as the naive mean-field approximation [25] or individual-based mean-field approximation [10].

Let $w _ { i j } ^ { k } \triangleq \operatorname* { P r } ( W _ { i j } ^ { k } = 1 | X _ { j } ( k ) = 1 )$ denote the conditional probability of the successful transmission of the virus from node j to node i at time step k. Then

$$\mathbb {E} \big (1 - W _ {i j} ^ {k} X _ {j} (k) \big) = \big (1 - w _ {i j} ^ {k} \mathbb {E} X _ {j} (k) \big), \quad \forall i, j \in [ n ].$$

Hence under (A1), (A3) and (A4), the dynamics in (6) lead to the virus spread model in [21]:

$$1 - p _ {i} (k + 1) = \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - w _ {i j} ^ {k} p _ {j} (k)\right), \quad i \in [ n ], \tag {7}$$
