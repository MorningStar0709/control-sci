# D. Contraction Theory

Informally, a system described by an ODE of the form

$$\dot {x} (t) = f (x), \quad f \in C ^ {1} (\mathbb {R} ^ {n}, \mathbb {R} ^ {n}) \tag {4}$$

is contractive if there exists a uniformly positive definite metric tensor, meaning $M ( x ) \in C ^ { 1 } ( \mathbb { R } ^ { n } , S _ { n } ^ { + + } )$ and $\alpha I \ \preceq$ $M ( x )$ for some $\alpha > 0 .$ , such that the associated Riemannian differential distance between trajectories decreases with time, that is $\frac { d ( \delta x ^ { \top } M ( x ) \delta x ) } { d t } < 0$ . This concept leads to the following definition:

Definition 1 ([10]). The system (4) is said to be exponentially contracting, if there exists a uniformly positive definite metric $M \ \in \ C ^ { 1 } ( \mathbb { R } ^ { n } , S _ { n } ^ { + + } )$ , a real number $\lambda ~ > ~ 0$ such that the following matrix inequality holds ∀x $\in \mathbb { R } ^ { n }$ :

$$\frac {\partial f (x)}{\partial x} ^ {\top} M (x) + M (x) \frac {\partial f (x)}{\partial x} + \dot {M} (x) \prec - 2 \lambda M (x). \tag {5}$$

In the sequel, by a slight abuse of notation we will refer to systems satisfying (5) as contractive. Theorem 1 of [10] shows that if (5) is satisfied then there exists a single global exponentially stable equilibrium point.
