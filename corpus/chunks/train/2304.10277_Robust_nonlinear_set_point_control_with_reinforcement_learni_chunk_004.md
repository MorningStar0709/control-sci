# II. PROBLEM STATEMENT

Consider a discrete-time nonlinear state space model

$$x _ {t + 1} = f _ {p} (x _ {t}, u _ {t}, w _ {t}) \tag {1}y _ {t} = h _ {p} (x _ {t}) \tag {2}$$

where $f _ { p }$ and $h _ { p }$ are unknown functions parameterized by $p , \ x _ { t } \ \in \mathbb { R } ^ { n _ { x } }$ is the current state of the system, $u _ { t } \in \mathbb { R } ^ { n _ { \mathfrak { i } } }$ u is the input, $y _ { t } \in \mathbb { R } ^ { n _ { y } }$ is the output, and $w _ { t } \in \mathbb { R } ^ { n _ { w } }$ is the system noise. The task considered in the paper is to use RL-methods to train a state feedback set-point controller to track a piecewise constant set-point $y _ { t } ^ { \mathrm { r e f } }$ . To handle multiple set-points in RL, the multi-goal RL framework in [14] is considered. The aim is thus to find a controller

$$u _ {t} = g _ {\theta} (x _ {t}, y _ {t} ^ {\mathrm{ref}}), \tag {3}$$

where $\theta$ is a parameter vector, that maximizes the discounted return

$$G _ {t} = \sum_ {k = 0} ^ {\infty} \gamma^ {k} r _ {t + k} \tag {4}$$

where $\gamma \in [ 0 , 1 ]$ is the discount factor and $r _ { t } = $ $R ( x _ { t } , u _ { t } , y _ { t } ^ { \mathrm { r e f } } )$ is the reward function at time step t.

As will be seen, state-of-the-art RL-methods like PPO struggle with this problem even in seemingly simple cases such as a cascade water tank system. To overcome the shortcomings of RL-methods, two simple extensions are proposed – making use of an integrator to eliminate static errors and making use of a prior controller to improve the excitation of the system during the learning phase.
