The main result is summarized in the following theorem.

Theorem 1: The controller comprising (22) and (26)–(28), with parameters $| \gamma _ { i } | < 1 , i = 1 , 2$ , ensures exponential convergence of the state $\eta ( \cdot , t )$ of the HCF (20) to the reference $y _ { \mathrm { r } } ( \cdot )$ , and hence of ${ \pmb x } ( \cdot , t )$ in (5) to $\mathbf { \boldsymbol { x } } _ { \mathrm { { r } } } ( \cdot , t )$ .

Proof: Applying (22) and (26)–(28) to the HCF (20) results in the closed-loop error dynamics (25) with $\alpha _ { i } ( \tau ) =$ $0 , i = 1 , 2$ . Choosing $| \gamma _ { i } | < 1$ ensures that $e _ { i } ( t )$ converges to zero and thus $\eta _ { i } ( \tau , t )$ to $\eta _ { i , \mathrm { r } } ( \tau , t ) = y _ { i , \mathrm { r } } ( t + \tau ) , i = 1 , 2$ , for $t \to \infty$ . This implies exponential convergence of $\pmb { x } ( z , t )$ to ${ \pmb x } _ { \mathrm { r } } ( z , t )$ according to Lemma 1, where ${ \pmb x } _ { \mathrm { r } } ( z , t )$ is obtained by inserting ${ \mathbf { } } y _ { \mathrm { { r } } } ( t )$ into (16) and the inverse of (9).
