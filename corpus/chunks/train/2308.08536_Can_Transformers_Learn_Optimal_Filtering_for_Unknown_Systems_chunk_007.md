# III. EXPERIMENTS

In this section, we present the experimental results for the transformer-based MOP in different scenarios. In each scenario, during the training, we fix the number of source systems M = 20000 and training trajectory length $T = 5 0 .$ . To evaluate the performance of MOP on different unseen test systems, for each experimental setup, we randomly generate $N = 1 0 0 0$ systems and record the prediction error $\left\| \hat { \mathbf { y } } _ { t } - \mathbf { y } _ { t } \right\|$ over trajectories each with length $T = 5 0$ , where $\hat { \mathbf { y } } _ { t }$ denotes the prediction for $\mathbf { y } _ { t }$ . We use GPT-2 [22] architecture with 12 layers, 8 attention heads and 256 embedding dimensions. In each experimental setup, the transformer model is trained for 10000 training steps with batch size 64. The $\ell _ { 2 ^ { - } }$ norm is selected as the training loss function.
