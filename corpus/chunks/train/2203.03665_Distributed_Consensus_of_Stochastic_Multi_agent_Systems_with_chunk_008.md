# D. Prescribed Performance

This subsection provides preliminary knowledge on the prescribed performance control (PPC) [12]. The aim of PPC is to prescribe the evolution of the relative position $\overline { { x } } _ { k } ( t )$ within some predefined region which can be expressed in the form of the following inequality

$$- \rho_ {k} (t) < \overline {{{x}}} _ {k} (t) < \rho_ {k} (t) \tag {9}$$

for all $t ~ \in ~ \mathbb { R } _ { 0 } ^ { + }$ , where $\rho _ { k } ~ : ~ \mathbb { R } _ { 0 } ^ { + } ~ \to ~ \mathbb { R } ^ { + } , ~ k ~ \in ~ [ 1 ; m ]$ are positive, smooth, and strictly decreasing performance functions that introduce the desired predefined bounds for the relative positions. In this work, we consider the following performance function

$$\rho_ {k} (t) = (\rho_ {k 0} - \rho_ {k \infty}) \mathrm{e} ^ {- \epsilon_ {k} t} + \rho_ {k \infty}, \tag {10}$$

where $\rho _ { k 0 } , \rho _ { k \infty } ,$ , and $\epsilon _ { k }$ are positive constants with $\rho _ { k 0 } >$ $\rho _ { k \infty }$ and $\begin{array} { r } { \rho _ { k \infty } = \operatorname* { l i m } _ { t  \infty } \rho _ { k } ( t ) } \end{array}$ represents relative positions at steady state. Now by normalizing $\overline { { x } } _ { k }$ with respect to the performance function $\rho _ { k }$ , we define the modulating error as $\begin{array} { r } { \dot { \hat { x } } _ { k } ( t ) = \frac { \overline { { x } } _ { k } ( t ) } { \rho _ { k } ( t ) } } \end{array}$ and the corresponding prescribed performance region $\hat { \mathcal { D } } _ { k } : = \{ \hat { x } _ { k } ~ | ~ \hat { x } _ { k } \in ( - 1 , 1 ) \}$ }. Then the modulated error is transformed through a transformation function $T _ { k } : \hat { \mathcal { D } } _ { k } $ R such that $T _ { k } ( 0 ) = 0$ and is chosen as

$$T _ {k} (\hat {x} _ {k}) = \ln \left(\frac {1 + \hat {x} _ {k}}{1 - \hat {x} _ {k}}\right). \tag {11}$$

The transformed error is then defined as $\xi _ { k } = T _ { k } ( \hat { x } _ { k } )$ . By differentiating $\xi _ { k }$ with respect to time, we obtain transformed error dynamics as

$$\dot {\xi} _ {k} = \phi_ {k} (\hat {x} _ {k}, t) [ \dot {\overline {{{x}}}} _ {k} + \alpha_ {k} (t) \overline {{{x}}} _ {k} ], \tag {12}$$
