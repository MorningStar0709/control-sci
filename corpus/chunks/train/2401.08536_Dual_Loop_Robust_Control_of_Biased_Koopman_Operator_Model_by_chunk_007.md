# A. Koopman Operator Theory

Consider a discrete-time nonlinear dynamical system

$$x _ {t + 1} = f \left(x _ {t}\right). \tag {1}$$

where $x _ { t + 1 } \in \mathcal { X } \subset \mathbb { R } ^ { n }$ is the state at time step $t + 1$ which evolves in time by non-linear state transition map $f : \mathcal { X }  \mathcal { X }$ . Given any observable $\psi \in { \mathcal { F } }$ , the Koopman operator $\kappa : { \mathcal { F } }  { \mathcal { F } }$ is defined as

$$\mathcal {K} \psi (x _ {t}) = \psi (f (x _ {t})) = \psi (x _ {t + 1}) \tag {2}$$

Here, the Koopman operator  linearly evolves a nonlinear dynamics using observables $\psi ,$ which lift the finite-dimensional states $\boldsymbol { x } _ { t } \in \mathbb { R } ^ { n }$ to an infinite-dimensional observable space where original nonlinear dynamics can be represented by an infinite-dimensional linear system.

Because the infinite-dimensional observables are practically infeasible to implement the theoretical transformations, an approximation-based approach is usually implemented to get a finite-dimensional approximation of the Koopman operator. This work uses extended dynamic mode decomposition (EDMD) [23] with controls to get approximate transition matrices in observable space.
