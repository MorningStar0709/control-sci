$$L _ {2} + \lambda_ {2} ^ {\top} f _ {2} = L _ {1} + \lambda_ {1} ^ {\top} f _ {1} \tag {9}\Rightarrow \Delta L = - \Delta \lambda^ {\top} f _ {1} - \lambda_ {2} ^ {\top} \Delta f.$$

Similarly,

$$L _ {2} + \lambda_ {2} ^ {\top} f _ {2} = L _ {1} + \lambda_ {1} ^ {\top} f _ {1} \tag {10}\Rightarrow \Delta L = - \lambda_ {1} ^ {\top} \Delta f - f _ {2} \Delta \lambda^ {\top}.$$

Adding (9) to (10) leads to

$$\Delta \lambda^ {\top} \langle f \rangle = - \left(\Delta L + \langle \lambda^ {\top} \rangle \Delta f\right). \tag {11}$$

Multiply both sides of (11) by $\left( \frac { \partial g ( x ) } { \partial x } | _ { x ^ { o } } \right)$ |xo (i.e., the gradient w.r.t. the state at the interface under the optimal control) to obtain

$$\Delta \lambda^ {\top} \langle f \rangle \left(\frac {\partial g (x)}{\partial x} | _ {x ^ {o}}\right) = - (\Delta L + \langle \lambda^ {\top} \rangle \Delta f) \left(\frac {\partial g (x)}{\partial x} | _ {x ^ {o}}\right). \tag {12}$$

The optimality condition derived in (6) shows that under the optimal control, $\Delta \lambda \in ( T _ { x ^ { * } } g ( x ) ) ^ { \bot } \ ( \mathrm { i . e . , } \ \Delta \lambda$ is parallel to the gradient of the interface), thus we can change the left side of (12) to the following,

$$\Delta \lambda^ {\top} \langle f \rangle \left(\frac {\partial g (x)}{\partial x} | _ {x ^ {o}}\right) = \left(\frac {\partial g (x)}{\partial x} | _ {x ^ {o}}\right) ^ {\top} \langle f \rangle \Delta \lambda . \tag {13}$$

Proof: Denote $\begin{array} { r } { m = \left( \frac { \partial g ( x ) } { \partial x } | _ { x ^ { o } } \right) } \end{array}$ |xo . First, we show that

$$\Delta \lambda^ {\top} \langle f \rangle m = m ^ {\top} \langle f \rangle \Delta \lambda .$$

Since m is parallel to $\Delta \lambda ,$ we can write them respectively as $m = t _ { 1 } v , \Delta \lambda = t _ { 2 } v$ , where v is a direction vector, $t _ { 1 } , t _ { 2 }$ are scalars. Then $\Delta \lambda ^ { \top } \langle f \rangle m = t _ { 1 } v ^ { \top } \langle f \rangle t _ { 2 } v = t _ { 1 } t _ { 2 } v ^ { \top } \langle f \rangle v ,$ $m ^ { \top } \langle f \rangle \Delta \lambda = t _ { 1 } v ^ { \top } \langle f \rangle t _ { 2 } v = t _ { 1 } t _ { 2 } v ^ { \top } \langle f \rangle v .$ .

Substitute (13) into (12), we have the following equation:

$$\left(\frac {\partial g (x)}{\partial x} | _ {x ^ {o}}\right) ^ {\top} \langle f \rangle \Delta \lambda = - \left(\Delta L + \langle \lambda^ {\top} \rangle \Delta f\right) \left(\frac {\partial g (x)}{\partial x} | _ {x ^ {o}}\right), \tag {14}$$

which gives the relation

$$\Delta \lambda = - \frac {\left(\Delta L + \langle \lambda^ {\top} \rangle \Delta f\right) m}{m ^ {\top} \langle f \rangle}. \tag {15}$$

where $\begin{array} { r } { m = \frac { \partial g ( x ) } { \partial x } | _ { x ^ { o } } } \end{array}$ . This relation specifies the quantitative behavior of the co-states at the interface.
