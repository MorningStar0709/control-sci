• Misclassification Error: Now let us control the misclassification error term in (16). For this purpose, we define the misclassification event for agent $\ell \in \mathcal { C } _ { \hat { j } }$ as follows:

$$\mathcal {E} _ {\mathrm{mis}} ^ {(\ell)} := \left\{\exists j ^ {\prime} \neq j: \left\| X ^ {(\ell)} - \widehat {\Theta} _ {j} Z ^ {(\ell)} \right\| _ {F} ^ {2} > \left\| X ^ {(\ell)} - \widehat {\Theta} _ {j ^ {\prime}} Z ^ {(\ell)} \right\| _ {F} ^ {2} \right\},$$

where we recall that $X ^ { ( \ell ) }$ and $Z ^ { ( \ell ) }$ denote the data matrices corresponding to system $\ell ,$ and ${ \widehat { \Theta } } _ { j ^ { \prime } }$ is the estimated model for cluster $\boldsymbol { j ^ { \prime } } \neq \boldsymbol { j }$ . Substituting $X ^ { ( \ell ) } = \Theta _ { j } Z ^ { ( \ell ) } + W ^ { ( \ell ) }$ , where $\Theta _ { j } = \Theta _ { \star } ^ { ( \ell ) }$ , we obtain

$$\left\| (\Theta_ {j} - \widehat {\Theta} _ {j}) Z ^ {(\ell)} + W ^ {(\ell)} \right\| _ {F} ^ {2} > \left\| (\Theta_ {j} - \widehat {\Theta} _ {j ^ {\prime}}) Z ^ {(\ell)} + W ^ {(\ell)} \right\| _ {F} ^ {2}.$$

Rearranging terms, this event is equivalent to

$$D := \left\| \Delta_ {j ^ {\prime}} Z ^ {(\ell)} + W ^ {(\ell)} \right\| _ {F} ^ {2} - \left\| \Delta_ {j} Z ^ {(\ell)} + W ^ {(\ell)} \right\| _ {F} ^ {2} < 0,$$

where we define $\Delta _ { j ^ { \prime } } : = \Theta _ { j } - \widehat { \Theta } _ { j ^ { \prime } }$ and $\Delta _ { j } : = \Theta _ { j } - \widehat \Theta _ { j }$ as the estimation residuals for the incorrect and correct cluster models, respectively. We proceed to analyze this term by defining

$$D _ {1} := \left\| \Delta_ {j ^ {\prime}} Z ^ {(\ell)} \right\| _ {F} ^ {2} - \left\| \Delta_ {j} Z ^ {(\ell)} \right\| _ {F} ^ {2}, \text {and} D _ {2} := 2 \mathrm{tr} \left(W ^ {(\ell) \top} (\Delta_ {j ^ {\prime}} - \Delta_ {j}) Z ^ {(\ell)}\right),$$

where $D = D _ { 1 } + D _ { 2 }$ . Then the probability of misclassification probability becomes

$$\mathbb {P} (\mathcal {E} _ {\mathrm{mis}} ^ {(\ell)}) = \mathbb {P} (D < 0) = \mathbb {P} (D _ {1} < - D _ {2}).$$

As $D _ { 1 }$ is a linear with respect to the assumed Gaussian noise $W ^ { ( \ell ) }$ matrix, it is sub-Gaussian with variance bounded as follows:

$$\operatorname{Var} (D _ {2}) \leq C _ {D _ {2}} \sigma_ {w} ^ {2} \mathbb {E} \left\| (\Delta_ {j ^ {\prime}} - \Delta_ {j}) Z ^ {(\ell)} \right\| _ {F} ^ {2}.$$
