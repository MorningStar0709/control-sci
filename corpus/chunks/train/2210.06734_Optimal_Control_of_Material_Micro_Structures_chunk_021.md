# B. Algorithms Tested

a) Deep Deterministic Policy Gradient: Deep Deterministic Policy Gradient (DDPG) is a policy-gradient based off-policy reinforcement learning algorithm that operates in continuous state and action spaces. It relies on two function approximation networks one each for the actor and the critic. The critic network estimates the $Q ( s , a )$ value given the state and the action taken, while the actor network engenders a policy given the current state. Neural networks are employed to represent the networks.
