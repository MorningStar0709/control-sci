# 3.2 Clairvoyant estimator

A non-causal observer does away with the causal constraint (15c), estimating states based on both past and future measurements. This means that the optimal clairvoyant observer is given by a gain matrix $\mathcal { L } _ { n c } \doteq \mathbb { R } ^ { n T \times m T }$ , which is full and such that the resulting estimation error minimizes the cost (2). While not implementable, ${ \mathcal { L } } _ { n c }$ can give useful information about how an observer should react to noise. In what follows, we show how to explicitly compute ${ \mathcal { L } } _ { n c }$ using the same SLS framework as in Section 3.1.

Lemma 3. For quadratic, positive definite loss functions $\ell _ { t } ~ = ~ e _ { t } ^ { \top } Q _ { t } e _ { t } ~ \boldsymbol { \hat { w i t h } } ~ \boldsymbol { 0 } ~ \prec ~ \boldsymbol { \hat { Q } _ { t } } ~ \in ~ \mathbb { R } ^ { \tilde { n } \times n }$ , the prediction error maps $\Phi _ { v , n c } , \Phi _ { w , n c } \ o f$ the clairvoyant observer ${ \mathcal { L } } _ { n c } =$ $\Phi _ { w , n c } ^ { - 1 } \Phi _ { v , n c }$ are given in the SLS form by

$$
\begin{array}{l} \{\Phi_ {v, n c}, \Phi_ {w, n c} \} = \underset {\Phi_ {v}, \Phi_ {w}} {\arg \min} \left\| \mathcal {Q} [ \Phi_ {v}, \Phi_ {w} ] \left[ \begin{array}{c c} \Sigma_ {v} ^ {\frac {1}{2}} & 0 \\ 0 & \Sigma_ {w} ^ {\frac {1}{2}} \end{array} \right] \right\| _ {F} ^ {2}, (2 3 a) \\ \mathrm{s.t.} (1 5 \mathrm{b}), \\ \end{array}

\in \underset {\Phi_ {v}, \Phi_ {w}} {\arg \min} \left\| \mathcal {Q} [ \Phi_ {v}, \Phi_ {w} ] \left[ \begin{array}{c c} \mathcal {H} _ {v} ^ {- 1} & 0 \\ 0 & \mathcal {H} _ {w} ^ {- 1} \end{array} \right] \right\| _ {2} ^ {2} \tag {23b}
\mathrm{s.t.} (1 5 \mathrm{b}),$$

where $\underline { { { \mathcal Q } } } ~ = ~ b l k d i a g ( Q _ { 0 } ^ { \frac { 1 } { 2 } } , \dots , Q _ { T + 1 } ^ { \frac { 1 } { 2 } } )$ . The corresponding optimal cost is

$$
\min _ {\mathcal {L}} \operatorname{cost} (e (\boldsymbol {v}, \boldsymbol {w}, \mathcal {L})) = \left\| \mathcal {Q} [ \Phi_ {v, n c}, \Phi_ {w, n c} ] \left[ \begin{array}{c} \boldsymbol {v} \\ \boldsymbol {w} \end{array} \right] \right\| _ {2} ^ {2} \tag {24}
$$

Proof. Using (11), one can always obtain $\{ \Phi _ { v , n c } , \Phi _ { w , n c } \}$ from ${ \mathcal { L } } _ { n c }$ using

$$\Phi_ {w, n c} = \left(\left(I - \mathcal {Z} \mathcal {A}\right) - \mathcal {L} _ {n c} \mathcal {C} \mathcal {Z}\right) ^ {- 1}, \tag {25a}\Phi_ {v, n c} = \Phi_ {w, n c} \mathcal {L} _ {n c}. \tag {25b}$$
