After substituting (45) and (48) into (36), for a given $\varepsilon ,$ the sufficient condition in Theorem 1 reduces to

$$\underline {{\rho}} (\sigma ; \varepsilon) \leq \rho ,$$

where

$$\underline {{\rho}} (\sigma ; \varepsilon) = \frac {\left(1 + \sqrt {\alpha_ {S _ {u}} (\varepsilon) \beta_ {\min} (\sigma ; \varepsilon)}\right) ^ {2}}{1 - \varepsilon - \alpha_ {\Gamma_ {U}} (\varepsilon) \beta_ {\min} (\sigma ; \varepsilon)}.$$

$$\max _ {\sigma} \quad \sigma\text { s.t. } \quad 0 < \sigma < \frac {1}{\alpha_ {S} (\varepsilon)}, \tag {49}1 - \varepsilon - \alpha_ {\Gamma_ {U}} (\varepsilon) \beta_ {\min} (\sigma ; \varepsilon) > 0,\underline {{\rho}} (\sigma ; \varepsilon) \leq \rho .$$

This is an optimization problem over a one-dimensional feasible set. Moreover, since $\beta _ { \operatorname* { m i n } } ( \sigma ; \varepsilon )$ is strictly increasing in $\sigma ,$ the feasible set is an interval, and hence a bisection method can be used to compute its largest feasible point numerically. We denote the value obtained by this bisection by $\boldsymbol { \sigma } ^ { \star } ( \varepsilon )$ .

Finally, we design ε and accordingly determine the other parameters. Since $\underline { { \rho } } ( \sigma ; \varepsilon )$ is increasing in ε, a larger ε makes the performance constraint more restrictive.
