We aim to show the right-hand side of (33) is bounded by a constant independent of $n .$ From Wald’s equation (Durrett 2019, Theorem 4.8.6), we have that $\mathbb { E } \left[ \sum _ { i = 1 } ^ { \tau _ { n } } T _ { \beta _ { n } + i } \right]$ is bounded by a constant. For the second term, we use Wald’s second equation (Durrett 2019, Exercise 4.8.4) for i.i.d. random variables $\{ \tilde { T } _ { i } \} _ { i = 1 } ^ { n }$ defined as $\begin{array} { r } { \tilde { T } _ { i } : = T _ { \beta _ { n } + i } - \frac { 1 } { \lambda } } \end{array}$ , with $\mathbb { E } [ \tilde { T } _ { i } ] = 0$ for all i. We take $\begin{array} { r } { \tilde { S } _ { n } : = \sum _ { i = 1 } ^ { n } \tilde { T } _ { i } } \end{array}$ . From Wald’s second equation, for stopping time $\tau _ { n }$ with finite expectation, E $\begin{array} { r } { \left[ \tilde { S } _ { \tau _ { n } } ^ { 2 } \right] = \frac { 1 } { \lambda ^ { 2 } } \mathbb { E } \left[ \tau _ { n } \right] } \end{array}$ . In addition, from the definition of ${ \tilde { S } } _ { n } .$ , we have $\begin{array} { r } { \mathbb { E } \big [ \tilde { S } _ { \tau _ { n } } ^ { 2 } \big ] = \mathbb { E } \big [ \big ( \sum _ { i = 1 } ^ { \tau _ { n } } T _ { \beta _ { n } + i } - \frac { \tau _ { n } } { \lambda } \big ) ^ { 2 } \big ] } \end{array}$ . Finally, we bound the second term on the right-hand side of (33) with a constant as below

$$
\begin{array}{l} \mathbb {E} \Big [ \Big (\sum_ {i = 1} ^ {\tau_ {n}} T _ {\beta_ {n} + i} \Big) ^ {2} \Big ] = \frac {1}{\lambda^ {2}} \mathbb {E} [ \tau_ {n} ] + \frac {2}{\lambda} \mathbb {E} \Big [ \tau_ {n} \sum_ {i = 1} ^ {\tau_ {n}} T _ {\beta_ {n} + i} \Big ] - \frac {1}{\lambda^ {2}} \mathbb {E} \Big [ \tau_ {n} ^ {2} \Big ] \\ \leq \frac {1}{\lambda^ {2}} \mathbb {E} [ \tau_ {n} ] + \frac {1}{\lambda} \mathbb {E} \left[ \sum_ {i = 1} ^ {\tau_ {n}} 2 \tau_ {n} T _ {\beta_ {n} + i} \right] \leq \frac {1}{\lambda^ {2}} \mathbb {E} [ \tau_ {n} ] + \frac {1}{\lambda} \mathbb {E} \left[ \sum_ {i = 1} ^ {\tau_ {n}} T _ {\beta_ {n} + i} ^ {2} \right] + \frac {1}{\lambda} \mathbb {E} \left[ \tau_ {n} ^ {3} \right]. \tag {34} \\ \end{array}
$$
