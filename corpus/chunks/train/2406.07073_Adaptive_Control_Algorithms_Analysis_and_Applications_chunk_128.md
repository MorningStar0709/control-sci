Case $2 Q \neq 0 ; \eta ^ { 2 } = 1 ; \alpha ( t ) \equiv 1$

This corresponds to the problem of estimating a time-varying parameter with a constant average value. The magnitude of this variation will depend upon $Q = \mathbf { E } \{ v ( t ) v ^ { T } ( t ) \}$ . In order to be able to follow such parameter variations, it is necessary to add a constant term (in the expression of the adaptation gain given by (3.45)). If such a term is introduced only at certain instants, for reinitializing the value of the adaptation gain, this can be interpreted as a jump of $v ( t )$ and therefore as a jump of the values of the parameters to be estimated.

This technique is currently used in practice for the reinitialization of the adaptation gain.
