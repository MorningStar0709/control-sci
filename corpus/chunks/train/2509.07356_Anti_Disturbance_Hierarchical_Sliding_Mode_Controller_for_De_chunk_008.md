# 3.2 Control Law Design

The control law integrates hierarchical sliding mode control, adaptive control, and neural network compensation, defined as:

$$u (t) = u _ {s m 0} (t) + u _ {s m 1} (t) + u _ {s m 2} (t) + \sigma (t) u _ {a d} (t) + u _ {\Psi} (t) \tag {9}$$

where $u _ { s m 0 } ( t ) , \ u _ { s m 1 } ( t )$ , and $u _ { s m 2 } ( t )$ are the sliding mode control terms, while $\sigma ( t ) u _ { a d } ( t )$ and $u _ { \Psi } ( t )$ represent the adaptive compensation term and the neural network compensation term, respectively.
