# 3.1 Preliminaries

In [29, 30], the authors propose observer-based R-CETC approaches for a class of reaction-diffusion PDEs with constant coefficients. The full-state feedback design for the case with a spatially varying reaction coefficient follows essentially the same steps. Therefore, we briefly present the results of the design below without going into the details of the derivations.

Events under the R-CETC are generated by the following dynamic event-trigger:

$$t _ {j + 1} ^ {r} = \inf \left\{t \in \mathbb {R} _ {+} | t > t _ {j} ^ {r}, \Gamma^ {r} (t) > 0, j \in \mathbb {N} \right\}, \tag {29}$$

with $t _ { 0 } ^ { r } = 0$ . The function $\Gamma ^ { r } ( t )$ is given by

$$\Gamma^ {r} (t) = d ^ {2} (t) - \gamma m ^ {r} (t), \tag {30}$$

where $\gamma > 0$ is an event-trigger design parameter. The function $d ( t )$ satisfies (25) for all $\mathbf { \bar { \Phi } } _ { t } \in [ t _ { i } ^ { r } , t _ { i + 1 } ^ { \bar { r } } ) , j \in \mathbb { N }$ , and after the control input is updated at each $t \doteq t _ { j } ^ { r } , d ( t _ { j } ^ { r } )$ becomes zero by the virtue of (25). The dynamic variable $m ^ { r } ( t )$ evolves according to the ODE

$$\dot {m} ^ {r} (t) = - \eta m ^ {r} (t) - \rho d ^ {2} (t) + \beta_ {1} \| u [ t ] \| ^ {2} + \beta_ {2} u ^ {2} (1, t), \tag {31}$$

valid for all $t \in ( t _ { j } ^ { r } , t _ { i + 1 } ^ { r } ) , j \in \mathbb { N }$ with $m ^ { r } ( t _ { 0 } ^ { r } ) = m ^ { r } ( 0 ) > 0$ and $m ^ { r } ( t _ { i } ^ { r - } ) = m ^ { r } ( t _ { j } ^ { r } ) \dot { = } m ^ { r } ( t _ { i } ^ { r + } )$ , and $\eta , \rho , \beta _ { 1 } , \beta _ { 2 } > 0$ being event-trigger parameters.

Under Assumption 1 with appropriately chosen eventtrigger parameters (see Section 3.2.1), and by applying the event-triggered control input

$$U _ {j} ^ {r} = \int_ {0} ^ {1} k (y) u (y, t _ {j} ^ {r}) d y, \tag {32}$$

in a zero-order hold manner between events generated by the event-trigger (29)-(31), it can be shown that

• The time between two consecutive events is positively and uniformly lower bounded by a constant $\tau > 0$ , i.e., $t _ { j + 1 } ^ { r } - t _ { j } ^ { r } \geq \tau , j \in \mathbb { N }$ , where

$$\tau = \frac {1}{a} \ln \left(1 + \frac {\sigma a}{(1 - \sigma) (a + \gamma \rho)}\right), \tag {33}$$

with $\sigma \in ( 0 , 1 )$ and

$$a = 1 + \rho_ {1} + \eta > 0. \tag {34}$$

Here, $\rho _ { 1 }$ is given by

$$\rho_ {1} = 3 \varepsilon^ {2} k ^ {2} (1), \tag {35}$$

where $k ( y )$ is given by (14).
