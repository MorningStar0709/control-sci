# 4.6 The Servo Problem

Only the regulator problem has been discussed so far. The criteria have been to eliminate impulse disturbances and to drive the states of the system to zero. The servo problem is another important prototype problem. For that problem the objective is to make the states and the outputs of the system respond to command signals in a specified way.
