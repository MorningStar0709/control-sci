Proof. For a delay factor $e ^ { - s \tau }$ , the phase lag at frequency ω is $\omega \tau ,$ , while the loop magnitude is unchanged. If τ satisfies $\begin{array} { r } { 0 \ \le \ \tau \ \le \ \frac { \phi } { \omega _ { \mathrm { m a x } } } } \end{array}$ then $\omega \tau \ \leq \ \bar { \phi }$ for every $\omega \in$ ωmax $\Omega _ { \mathrm { g c } }$ . Thus the delay contributes no more phase lag than the certified phase bound at any gain-crossover frequency. By the delay boundary condition associated with (40), this excludes imaginary-axis crossings of the delayed active loop. Hence the delayed active-loop equation has no solutions in $\{ \Re s \ge 0 \}$ for all such τ . □
