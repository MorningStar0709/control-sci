# 5.2 Online Learning Algorithm with Informative Measurements

From the fundamental limitation results given by Theorem 5.1, one cannot hope for sublinear regret in T for the state estimation problem if the online learning algorithm only has access to the outputs $y _ { 0 } , y _ { 1 } , . . . .$ One natural idea is then to consider that the algorithm has access to limited yet more informative measurements from system (1). In particular, we assume that at certain time steps $t \geq 0 ,$ , the following measurements can potentially be made available to the learner:

$$\tilde {x} _ {t} = x _ {t} + \tilde {v} _ {t}, \tag {38}$$

where the measurement noise satisfies that $\tilde { v } _ { t } \stackrel { i . i . d . } { \sim } \mathcal { N } ( 0 , \tilde { V } )$ with $\tilde { V } \succeq 0$ and $\{ \tilde { v } _ { t } \} _ { t \geq 0 }$ is assumed to be independent of $\{ w _ { t } \} _ { t \ge 0 }$ and $\{ v _ { t } \} _ { t \geq 0 }$ in system (1). For example, the measurement $\tilde { x } _ { t }$ may come from a sensor on system (1) with measurement matrix $C = I _ { n }$ or a sensor with a known and full column rank measurement matrix $C \in \mathbb { R } ^ { p \times n }$ . While being more informative, it can be costly to use such a sensor to collect measurements from system (1). This motivates us to consider the scenario where the learner may only query for the measurements $\tilde { { x } } _ { t }$ at a limited number of time steps, as will be detailed in our algorithm design next. Finally, we note that the measurement model in (38) has been widely adopted in the literature on learning for unknown linear systems [53–56].
