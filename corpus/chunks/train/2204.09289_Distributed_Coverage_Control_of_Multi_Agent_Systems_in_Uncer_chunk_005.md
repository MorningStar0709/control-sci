# 2.1 Region partition

Consider the 2-dimensional workspace $D \ \subset \ \mathbb { R } ^ { 2 }$ with Lipschitz continuous boundary for N mobile agents to complete the coverage task, and the location of the agent i can be represented by spatial coordinates $\mathbf { s _ { i } } = \left( \mathbf { x _ { i } } , \mathbf { y _ { i } } \right)$ . In addition, any point in the workspace is expressed as $\mathbf { x } =$ $( x , y )$ . The communication topology of multi-agent systems can be expressed as an undirected graph $G = ( V , E )$ , where V represents a collection of N nodes, and each node corresponds to an individual agent. Here E represents the set of connections between nodes, that is, the interactive relationship between agents. For $i , j \in \{ 1 , 2 , 3 , . . . , N \}$ , let $w _ { i j } \in \{ 0 , 1 \}$ represent the communication or connection status between Agent i and Agent j. When $w _ { i j } = 1$ , it means that Agent i is connected with Agent $j .$ . In this problem, we assume that there is communication relationship between agents with overlapping edges in the subregion after Voronoi partition in [14]. For the agent i, its Voronoi cell $D _ { i }$ is defined as

$$D _ {i} = \{\mathbf {x} \in D | d (\mathbf {x}, \mathbf {s _ {i}}) \leq \mathbf {d} (\mathbf {x}, \mathbf {s _ {j}}), \forall \mathbf {j} \in \{\mathbf {1}, \mathbf {2}, \mathbf {3},..., \mathbf {N} \}, \mathbf {i} \neq \mathbf {j} \}, \tag {1}$$

where $d ( \mathbf { x } , \mathbf { s _ { i } } ) = | | \mathbf { x } - \mathbf { s _ { i } } | |$ is the Euclidean distance between the location of x and si. The partition method is shown in Figure 1.
