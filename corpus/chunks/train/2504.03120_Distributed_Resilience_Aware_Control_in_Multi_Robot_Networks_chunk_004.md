# A. Multi-Robot Network System

Consider a multi-robot system of n robots. Robot $i \in$ $\{ 1 , \ldots , n \}$ has a physical state $x _ { i } ( t ) \in \mathbb { R } ^ { m }$ and control input $u _ { i } ( t ) \in \mathcal { U } _ { i } \subset \mathbb { R } ^ { m }$ , denoting position and velocity at time t, respectively. Each robot i has single integrator dynamics as

$$\dot {x} _ {i} (t) = u _ {i} (t). \tag {1}$$

We model robots as circular disks with radii $\frac { \Delta _ { d } } { 2 }$ , where $\Delta _ { d } \in$ $\mathbb { R } _ { + }$ . We also define the state and control input of the entire system as $\mathbf { x } ( t ) = \left[ x _ { 1 } ( t ) ^ { T } \cdot \cdot \cdot x _ { n } ( t ) ^ { T } \right] ^ { T } \in \bar { \mathbb { R } } ^ { n m }$ and ${ \bf u } ( t ) =$ $\left[ u _ { 1 } ( t ) ^ { T } \cdot \cdot \cdot u _ { n } ( t ) ^ { T } \right] ^ { T } \in \mathcal { U } = \mathcal { U } _ { 1 } \times \cdot \cdot \cdot \times \mathcal { U } _ { n } \subset \mathbb { R } ^ { n m }$ . Then the system dynamics is

$$\dot {\mathbf {x}} (t) = \mathbf {u} (t). \tag {2}$$

The distance between robots i and $j$ is denoted as $\Delta _ { i j } ( \mathbf { x } ( t ) ) ~ = ~ \| \boldsymbol { x } _ { i } ( t ) - \boldsymbol { x } _ { j } ( t ) \|$ . We model the multi-robot network as a time-varying simple, undirected graph $\mathcal { G } ( t ) =$ $( \nu , \mathcal { E } ( t ) )$ , where the vertex set $\nu = \{ 1 , \ldots , n \}$ includes robots, and the edge set $\mathcal { E } ( t ) = \{ ( i , j ) \ | \ \Delta _ { i j } ( \mathbf { x } ( t ) ) \leq R \}$ defines connections (links) between two robots within a communication radius $R \in \mathbb { R } _ { + }$ . The set of neighbors of robot $i \in \nu$ at time t is denoted as $\mathcal { N } _ { i } ( t ) = \{ j \in \mathcal { V } | ( i , j ) \in \mathcal { E } ( t ) \}$ . We also denote $B _ { i } ( t ) = \mathcal { N } _ { i } ( t ) \cup \{ i \}$ . Each robot can observe states of its own and of its neighbors. The degree of robot i at time t is denoted as $\delta _ { i } ( \mathcal { G } ( t ) ) = | \mathcal { N } _ { i } ( t ) |$ . Then, the minimum degree of $\mathcal { G } ( t )$ is denoted as $\delta _ { \mathrm { m i n } } ( { \mathcal G } ( t ) ) = \mathrm { m i n } _ { i \in \mathcal { V } } \delta _ { i } ( { \mathcal G } ( t ) )$ ).

We smoothly approximate the network connections through the weighted adjacency matrix ${ \bf A } ( { \bf x } ( t ) )$ , whose entries are

$$
a _ {i j} (\mathbf {x} (t)) = \left\{ \begin{array}{l l} \frac {q _ {1}}{1 + e ^ {- q _ {2} \left(R ^ {2} - \Delta_ {i j} (\mathbf {x} (t)) ^ {2}\right) ^ {2}}} - \frac {q _ {1}}{2} & \Delta_ {i j} (\mathbf {x} (t)) \leq R, \\ 0 & \text { otherwise }, \end{array} \right. \tag {3}
$$
