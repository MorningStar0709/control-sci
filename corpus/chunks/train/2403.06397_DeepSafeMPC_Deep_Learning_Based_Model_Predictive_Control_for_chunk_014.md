In SQP, each iteration involves solving a al Quadratic Programming (QP) subproblem to obtain a search direction for the optimization variables. The $\mathrm { Q P }$ subproblem is formulated based on a quadratic approximation of the objective function and a linear approximation of the constraints around the current iterate $k .$ And the QP subproblem for a particular iteration $\left( s ^ { k } , \mathbf { a } ^ { k } \right)$ can be stated as follows:

$$
\begin{array}{l} \min _ {\Delta s, \Delta a} \frac {1}{2} \left[ \begin{array}{c} \Delta s \\ \Delta \mathbf {a} \end{array} \right] ^ {\mathrm{T}} H ^ {k} \left[ \begin{array}{c} \Delta s \\ \Delta \mathbf {a} \end{array} \right] + \left[ \begin{array}{c} \nabla_ {s} C (s ^ {k}, \mathbf {a} ^ {k}) ^ {\mathrm{T}} \\ \nabla_ {\mathbf {a}} C (s ^ {k}, \mathbf {a} ^ {k}) ^ {\mathrm{T}} \end{array} \right] ^ {\mathrm{T}} \left[ \begin{array}{c} \Delta s \\ \Delta \mathbf {a} \end{array} \right], \\ s _ {\min} - s ^ {k} \leq \Delta s \leq s _ {\max} - s ^ {k}, \\ \mathbf {a} _ {\min} - \mathbf {a} ^ {k} \leq \Delta \mathbf {a} \leq \mathbf {a} _ {\max} - \mathbf {a} ^ {k}. \tag {16} \\ \end{array}
$$

where $H ^ { k }$ is an approximation to the Hessian of the Lagrangian, ∆s and ∆a are the steps in the state and control variables, respectively. $\nabla _ { s } C ( s ^ { k } , \mathbf { a } ^ { k } )$ and $\nabla _ { a } C ( s ^ { k } , \mathbf { a } ^ { k } )$ are the gradients of the cost function with respect to the state and control variables at iteration k. The inequalities for ∆s and ∆a ensure that the updated state and control variables remain within their respective bounds.
