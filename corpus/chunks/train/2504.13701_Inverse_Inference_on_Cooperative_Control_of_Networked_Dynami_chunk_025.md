Note that in some practical scenarios, the observations are not sufficiently large, and the constraint $\| \widehat { A } _ { d } - I \| < 1$ may not hold. To avoid the occurrence of $\| \widehat { A } _ { d } - I \| \ge 1$ under limited observations, we can formulate a semidefinite program optimization problem with constraints to obtain a feasible $\widehat { A } _ { d } .$ Notice that by using the Schur complement, the constraint $\| A _ { d } - I \| < 1$ is equivalent to the following linear matrix inequality

$$
\left[ \begin{array}{c c} I & A _ {d} - I \\ (A _ {d} - I) ^ {\intercal} & I \end{array} \right] \succ 0. \tag {48}
$$

Besides, it is worth noting that there also exists an implicit constraint for $A _ { d } .$ In the continuous model (2), the closed-loop system matrix $A _ { c }$ satisfies

$$
\begin{array}{l} A _ {c} \left(\mathbf {1} _ {N} \otimes I _ {n}\right) = \left(I _ {N} \otimes A - L \otimes B K\right) \left(\mathbf {1} _ {N} \otimes I _ {n}\right) = \mathbf {1} _ {N} \otimes A \\ \Rightarrow A _ {c} ^ {k} (\mathbf {1} _ {N} \otimes I _ {n}) = \mathbf {1} _ {N} \otimes A ^ {k}, \tag {49} \\ \end{array}
$$

where the fact $L \mathbf { 1 } _ { N } = \mathbf { 0 } _ { N }$ is used. Then, multiplying $A _ { d } =$ $e ^ { \boldsymbol { A } _ { c } \tau }$ with $\left( \mathbf { 1 } _ { N } \otimes I _ { n } \right)$ , it follows that

$$
\begin{array}{l} A _ {d} \left(\mathbf {1} _ {N} \otimes I _ {n}\right) = e ^ {A _ {c} \tau} \left(\mathbf {1} _ {N} \otimes I _ {n}\right) = \sum_ {k = 0} ^ {\infty} \frac {\left(A _ {c} \tau\right) ^ {k}}{k !} \left(\mathbf {1} _ {N} \otimes I _ {n}\right) \\ = \sum_ {k = 0} ^ {\infty} \frac {\mathbf {1} _ {N} \otimes (A \tau) ^ {k}}{k !} = \mathbf {1} _ {N} \otimes e ^ {A \tau}, \tag {50} \\ \end{array}
$$
