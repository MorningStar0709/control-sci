# 3 Unbalanced Non-commutative Optimal Transport

Let $\mathcal { H } _ { 1 }$ and $\mathcal { \mathrm { H } } _ { 2 }$ be Hilbert spaces with dimensions, respectively, $d _ { 1 }$ and $d _ { 2 } , C \in \mathrm { H } ( \mathfrak { R } _ { 1 } \otimes \mathcal { H } _ { 2 } )$ be a Hermitian operator on $\mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 } , \rho \in \mathrm { H } _ { \ge } ( \mathcal { H } _ { 1 } )$ and $\sigma \in \mathrm { { H } } _ { \geq } ( \mathcal { H } _ { 2 } )$ be Hermitian and semi-definite positive operators on $\mathcal { H } _ { 1 }$ and $\mathcal { \mathrm { H } } _ { 2 }$ respectively.

Given $\varepsilon , \ \tau _ { 1 } , \ \tau _ { 2 } > 0$ , define the functional $\mathcal { F } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } : \mathrm { H } _ { \geq } ( \mathcal { H } _ { 1 } \otimes \mathcal { H } _ { 2 } )  \mathbb { R }$ a s

$$\mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\Gamma) = \mathrm{Tr} [ C \Gamma ] + \varepsilon \mathcal {S} [ \Gamma ] + \tau_ {1} \mathcal {E} [ \Gamma_ {1} | \rho ] + \tau_ {2} \mathcal {E} [ \Gamma_ {2} | \sigma ], \tag {3.1}$$

where S[Γ] is the von Neumann Entropy

$$\mathcal {S}: \mathrm{H} _ {\geq} (\mathcal {H} _ {1} \otimes \mathcal {H} _ {2}) \to \mathbb {R}, \qquad \mathcal {S} [ \Gamma ] = \mathrm{Tr} [ \Gamma (\log \Gamma - \mathbb {I} _ {d}) ],$$

the matrices $\Gamma _ { 1 } , \Gamma _ { 2 }$ are respectively, the partial traces of $\Gamma _ { 1 } = \mathrm { T r } _ { 2 } [ \Gamma ] , \ \Gamma _ { 2 } = \mathrm { T r } _ { 1 } [ \Gamma ]$ in $\mathcal { \mathrm { H } } _ { 2 }$ and $\mathcal { H } _ { 1 }$ and the functional $\mathcal { E } [ \cdot | \cdot ]$ is the quantum (Umegaki) relative entropy

$$
\mathcal {E}: \mathrm{H} _ {\geq} (\mathcal {H}) \times \mathrm{H} _ {\geq} (\mathcal {H}) \to \mathbb {R}, \quad \mathcal {E} [ \rho_ {1} | \rho_ {2} ] = \left\{ \begin{array}{l l} \mathrm{Tr} [ \rho_ {1} (\log \rho_ {1} - \log \rho_ {2} - \mathbb {I} _ {d}) + \rho_ {2} ] & \text {if} \ker \rho_ {1} \subset \ker \rho_ {2}, \\ + \infty & \text {otherwise} \end{array} \right.
$$

between $\Gamma _ { 1 } ~ ( \mathrm { r e s p . } ~ \Gamma _ { 2 } )$ and $\rho \ ( { \mathrm { r e s p . } } \ \sigma ) $ .

The von Neumann entropy regularized Unbalanced Non-commutative Optimal Transport is given by

$$\mathcal {Q O T} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} [ \rho , \sigma ] = \inf \left\{\mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\Gamma): \Gamma \in \mathrm{H} _ {\geq} (\mathcal {H} _ {1} \otimes \mathcal {H} _ {2}) \right\}. \tag {3.2}$$
