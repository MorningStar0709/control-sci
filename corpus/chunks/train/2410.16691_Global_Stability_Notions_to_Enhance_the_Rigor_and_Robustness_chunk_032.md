$$\dot {V} (x, d) = \nabla V (x) f (x, d) = - \left(z + k + \sigma (\theta + z) ^ {2}\right) y ^ {2} + d y \tag {66}\leq - \left(k - \frac {1}{4 \sigma} - \theta\right) y ^ {2} + | d | | y |$$

The function  defined by (65) satisfies inequality (60) with $a ( s ) = s ^ { 2 } / 2$ . Moreover, due to (66) we obtain the implication

$$\frac {1}{2 \varepsilon^ {2}} | d | ^ {2} \leq V (x) \Rightarrow \dot {V} (x, d) = \nabla V (x) f (x, d) \leq - \varepsilon y ^ {2} = - 2 \varepsilon V (x) \tag {67}$$

where

$$\varepsilon := \frac {1}{2} \left(k - \frac {1}{4 \sigma} - \theta\right) > 0 \tag {68}$$

Consequently, implication (61) holds with $\varphi ( s ) = s ^ { 2 } / ( 2 { \varepsilon } ^ { 2 } ) , \ \rho ( s ) = 2 { \varepsilon } s$ and $r = 0$ . We conclude from Theorem 6 that (62) satisfies the IOS property with gain function $\gamma ( s ) = s / \varepsilon ^ { 2 }$
