# C. Singularity Analysis

At this stage, to be able to proceed, a singularity of the controller needs to be investigated, which is caused by the $\hat { x } _ { 0 } -$ $\varepsilon _ { 1 }$ term in the denominator of the $\widehat { \omega } _ { r }$ in Eq. (37). $\widehat { x } _ { 0 }$ is the nominal value of $x _ { 0 }$ and was chosen in Eq. (39). A singularity happens when $\varepsilon _ { 1 }  { \widehat { x } } _ { 0 } ,$ which makes the denominator zero. To avoid the singularity at this specific moment, ${ \widehat { x } } _ { 0 }$ should be chosen in a way to make the nominator of the $\widehat { \omega } _ { r }$ zero as well. If we consider $\hat { x } _ { 0 } = 0$ , then

$$\widehat {\omega} _ {r} = \frac {c _ {2} h _ {2} | _ {\widehat {x} _ {0} = 0}}{\varepsilon_ {1}} =
\begin{array}{l} \frac {c _ {2}}{\varepsilon_ {1}} \left\{- \omega \left(\omega \varepsilon_ {2} + v _ {x} ^ {d} \cos \varepsilon_ {3} - v _ {x}\right) \right. \\ - \varepsilon_ {1} \left(- \frac {c _ {5}}{c _ {2}} v _ {x} \omega - \frac {c _ {6}}{c _ {2}} \omega\right) \tag {55} \\ + \dot {v} _ {x} ^ {d} \sin \varepsilon_ {3} + (\omega^ {d} - \omega) v _ {x} ^ {d} \cos \varepsilon_ {3} \\ \left. - \lambda_ {2} (- \varepsilon_ {1} \omega + v _ {x} ^ {d} \sin \varepsilon_ {3}) \right\} \\ \end{array}
$$

In the above equation, if $\varepsilon _ { 1 } \to 0$ , then according to Eq. (24), two situations are expected.

$$
\left\{\begin{array}{l l}1. \varepsilon_ {2} \rightarrow 0,&\varepsilon_ {3} \rightarrow 0 \xrightarrow {\text { yields }} \left\{\begin{array}{l}v _ {x} \rightarrow v _ {x} ^ {d}\\\omega \rightarrow \omega^ {d}\end{array}\right.\\2. \varepsilon_ {2} \rightarrow 0,&\varepsilon_ {3} \rightarrow \pi \xrightarrow {\text { yields }} \left\{\begin{array}{l}v _ {x} \rightarrow - v _ {x} ^ {d}\\\omega \rightarrow \omega^ {d}\end{array}\right.\end{array}\right. \tag {56}
$$

In both cases, we have $\widehat { \omega } _ { r } = \frac { 0 } { 0 }$ and as a result, L'Hôpital's rule can be applied [42]. For both scenarios

$$
\begin{array}{l} \widehat {\omega} _ {r} = \lim _ {\varepsilon_ {1} \to 0} \frac {c _ {2} \frac {d h _ {2} | _ {\widehat {x} _ {0} = 0}}{d \varepsilon_ {1}}}{\frac {d \varepsilon_ {1}}{d \varepsilon_ {1}}} \\ = \lim _ {\varepsilon_ {1} \rightarrow 0} \frac {\left(- \frac {c _ {5}}{c _ {2}} v _ {x} \omega - \frac {c _ {6}}{c _ {2}} \omega\right) + \lambda_ {2} \omega}{1} \tag {57} \\ = \tau \in \mathbb {R}. \\ \end{array}
$$

Eq. (57) indicates that the singularity for the designed controller can be avoided if zero is chosen for the nominal value of the xcomponent of the robot’s $I C R , { \mathrm { e . g . } } , { \hat { x } } _ { 0 } = 0$ .
