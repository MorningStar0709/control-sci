Theorem 2. Consider the dynamics (7), there exist a unique matrix L satisfying (9a) and (9b) if and only if the following achievability condition is satisfied

$$\Phi_ {w} (I - \mathcal {Z A}) + \Phi_ {v} \mathcal {C Z} = I. \tag {11}$$

Moreover $\mathcal { L } = \Phi _ { w } ^ { - 1 } \Phi _ { v }$ .

Proof. From (9), one has that $\Phi _ { v } = \Phi _ { w } \mathcal { L } .$ , and (7) can be rewritten as

$$\Phi_ {w} (\boldsymbol {w} + \mathcal {L} \boldsymbol {v}) = \mathcal {Z A} \Phi_ {w} (\boldsymbol {w} + \mathcal {L} \boldsymbol {v}) - \mathcal {L C Z} \Phi_ {w} (\boldsymbol {w} + \mathcal {L} \boldsymbol {v})+ (\boldsymbol {w} + \mathcal {L} \boldsymbol {v}), \tag {12}$$

which is satisfied for all $\pmb { w } \in \mathbb { W }$ and $\boldsymbol { v } \in \mathbb { V }$ if and only if

$$\Phi_ {w} = \mathcal {Z} \mathcal {A} \Phi_ {w} - \mathcal {L C Z} \Phi_ {w} + I. \tag {13}$$

In order to prove the theorem, we only need to prove that $\Phi _ { w }$ is invertible and that (11) is equivalent to (13) with $\mathcal { L } = \Phi _ { w } ^ { - 1 } \Phi _ { v }$ .

First, $\Phi _ { w } \mathcal { Z } \mathcal { A }$ is lower block-triangular with a zero blockdiagonal because it is the product of (i) a lower blocktriangular (ii) a block downshift operator, which zeroes the block-diagonal of $\Phi _ { w } \mathcal { Z }$ , and (iii) a block diagonal matrix which does not change the sparsity pattern. Moreover, $\Phi _ { v } \mathcal { C } \mathcal { Z }$ follows the same pattern because $\Phi _ { v }$ and $\mathcal { C }$ are also lower block-triangular and block-diagonal, respectively. Hence, the diagonal of $\Phi _ { w }$ must be equal to the identity on the right hand side of (11), which means that all its eigenvalues are equal to one if (11) is satisfied. This proves that $\Phi _ { w }$ is invertible.

Second, by post-multiplying and pre-multiplying (13) by $\Phi _ { w } ^ { - 1 }$ and $\Phi _ { w } ,$ respectively, we obtain

$$\Phi_ {w} = \Phi_ {w} \mathcal {Z} \mathcal {A} - \Phi_ {w} \mathcal {L C Z} + I. \tag {14}$$

To obtain (11) from (14), one must replace L by $\Phi _ { w } ^ { - 1 } \Phi _ { v }$ and rearrange the terms, which proves the theorem. □
