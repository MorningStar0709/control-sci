SOBSYN is based on a bisection over γ. After an update of the sample set $s$ in each iteration, we compute a minimizer for L at the current γ-level using the BFGS-based optimization solver implemented in Mogensen and Riseth (2018). If the minimum α is lower than $\varepsilon _ { 2 } ,$ the solver has managed to reduce all singular values of the closed-loop transfer function evaluated at all sample points below the current γ-level up to the termination tolerance $\varepsilon _ { 2 }$ . In this case, we reduce the upper bound to $\gamma _ { \mathrm { { u } } } : = \gamma ;$ otherwise, we increase the lower bound to $\gamma _ { 1 } : = \gamma$ . We terminate the bisection, when the relative difference between $\gamma _ { \mathrm { u } }$ and $\gamma _ { 1 }$ is lower than the bisection tolerance $\varepsilon _ { 1 }$ .
