2.2.
Optimal transport.
Given a nonnegative transportation cost $c : X \times Y $ $\bar { \mathbb { R } } _ { \geq 0 }$ , the optimal transport discrepancy $\mathcal { K } [ c ] : \mathcal { P } ( X ) \times \mathcal { P } ( Y ) \to \bar { \mathbb { R } } _ { \geq 0 }$ between two probability measures $\mu \in { \mathcal { P } } ( X )$ and $\nu \in \mathcal P ( Y )$ is

$$\mathcal {K} [ c ] (\mu , \nu) := \inf _ {\boldsymbol {\gamma} \in \Gamma (\mu , \nu)} \int_ {X \times Y} c (x, y) \mathrm{d} \boldsymbol {\gamma} (x, y), \tag {2.1}$$

where $\Gamma ( \mu , \nu ) : = \left\{ \gamma \in { \mathcal { P } } ( X \times Y ) | ( \mathrm { p r o j } _ { X } ^ { X \times Y } ) _ { \# } \gamma = \mu , ( \mathrm { p r o j } _ { Y } ^ { X \times Y } ) _ { \# } \gamma = \nu \right\}$ is the set of couplings. A prominent example of optimal transport discrepancy is, for some $p \geq 1$ , the $p ^ { \mathrm { t h } }$ power of the p-Wasserstein distance, obtained when $X = Y$ and the transportation cost c is a metric that induces the topology on X [5, §7].

Remark 2.1. When the transportation cost does not depend on one of the two variables (e.g., there exists $\tilde { c } \in \textit { X } \to \bar { \mathbb { R } } _ { \geq 0 }$ such that $c ( x , y ) = \widetilde { c } ( x ) )$ , the optimal transport discrepancy reduces to an expected value; i.e., $\mathcal { K } [ c ] ( \mu , \nu ) = \mathbb { E } ^ { \mu } \left[ \tilde { c } \right]$ .
