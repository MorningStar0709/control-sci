For compact $X ,$ , the Wasserstein distance of order 1 metrizes the weak topology on the set of probability measures ${ \mathcal { P } } ( X )$ (see [47, Corollary 6.13, p. 97]). However, in general, it is stronger than weak topology.

For each fixed $\gamma \in \mathbb { R } _ { > 0 }$ , we set a domain for $Q \cdot$ -functions as follows

$$\mathcal {C} _ {\gamma} = \left\{Q: X \times A \to [ 0, \infty): \| Q \| _ {w} \leq \frac {\gamma M}{1 - \beta \alpha}, \| Q _ {\mathrm{min}} \| _ {\mathrm{Lip}} \leq \frac {\gamma L _ {2}}{1 - \beta K _ {2}} \right\}$$

where constants $M , \alpha , L _ { 2 } $ , and $K _ { 2 }$ are defined below. We define mean-field equilibrium operator as $H _ { \gamma } : { \mathcal { C } } _ { \gamma } \times { \mathcal { P } } ( X ) \to { \mathcal { C } } _ { \gamma } \times { \mathcal { P } } ( X ) : ( Q , \mu ) \to ( H _ { 1 } ^ { \gamma } ( Q , \mu ) , H _ { 2 } ^ { \gamma } ( Q , \mu ) )$ , where the iterations of the Q-functions is handled by the operator

$$H _ {1} ^ {\gamma} (Q, \mu) (x, a) = \gamma c (x, a, \mu) + \beta \int_ {X} Q _ {\mathrm{min}} (y) p (d y | x, a, \mu)$$

and by using the function $f _ { \gamma } ( x , Q , \mu ) = \underset { a \in A } { \mathrm { a r g m i n } } H _ { 1 } ^ { \gamma } ( Q , \mu ) ( x , a )$ , we define the following ergodicity operator to handle the iterations on the state-measures

$$H _ {2} ^ {\gamma} (Q, \mu) (\cdot) = \int_ {X} p (\cdot | x, f _ {\gamma} (x, Q, \mu), \mu) \mu (d x).$$
