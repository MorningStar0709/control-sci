# Dataset Generation

To train the NN-based controller, the geometric controller (8) is used to reach 200 randomly sampled desired target points $\{ ( x _ { r } ^ { i } , y _ { r } ^ { i } ) \} _ { i = 1 } ^ { 2 0 0 }$ from random initial robot poses. For each target, the robot state and control inputs have been logged using a timestep of 0.2 s, resulting in a dataset of approximately 100,000 entries of the form:

$$\left(x _ {r}, y _ {r}, \theta , x _ {d}, y _ {d}, \omega_ {l}, \omega_ {r}\right). \tag {9}$$

where $\omega _ { l }$ and $\omega _ { r }$ are the controller outputs at each timestep.
