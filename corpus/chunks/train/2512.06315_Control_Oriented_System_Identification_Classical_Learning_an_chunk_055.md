# 4.2.3. Sparse Identification of Nonlinear Dynamics (SINDy)

Recently, Sparse Identification of Nonlinear Dynamical systems (SINDy) (Brunton et al., 2016) has emerged as a widelyadopted data-driven method for nonlinear system identification. In many physical systems, the dynamics can be parameterized by a small number of nonlinear function terms, making the governing equations naturally sparse in the space of candidate nonlinear functions. The key idea in SINDy is to identify a sparse set of active terms from the candidate nonlinear function space by transforming the system identification problem into a sparse regression problem.
