# V. SENSITIVITY ANALYSIS AND PARAMETER OPTIMIZATION

From Section IV, especially in (26) and (27), it is evident that the WT-GSC switched system stability is influenced by many factors, including the wind turbine’s external connection impedance and internal control dynamics, which are represented by the line impedance, reactive current proportional coefficient, current inner-loop PI parameters, etc. The analytic expression for the stability index concerning these influencing parameters is hard to obtain. To solve this high-dimensional nonlinear parameter optimization problem for system stability improvement, which is to max(max $\mu )$ by adjusting the aforementioned parameters, we employ the Sobol’ global sensitivity analysis method to identify dominant parameters, followed by searching for the optimal parameter group via the PSO algorithm in this section.
