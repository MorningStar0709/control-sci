# III. THE PROPOSED OBSERVER-FREE SLIDING SURFACE

In the general case of a nonlinear system with relative degree r, the proposed sliding variable is structured as:

$$s = \alpha (x, \dot {x}, \dots , x ^ {(r - 1)}) - \beta (x, u), \quad \text { with } \quad \beta (x, u) = \frac {u}{g (x)}$$

Here, α defines the desired internal dynamics of the system. This structure allows control design without requiring full state observers or estimation of higher-order derivatives, as the control input appears explicitly in the formulation.

In what follows, we illustrate the method using the commonly encountered second-order case:

$$s = \alpha (x, \dot {x}) - \beta (x, u)$$

with

$$\alpha (x, \dot {x}) = \dot {x} + k _ {1} x, \quad \beta (x, u) = \frac {u}{g (x)} \tag {6}$$

For clarity, we denote $\alpha ( x , { \dot { x } } )$ and $\beta ( x , u )$ simply as α and $\beta ,$ respectively, when the arguments are understood from context. The control law is:

$$u = - \lambda \tanh (\alpha) \tag {7}$$

leading to:

$$\beta = - \frac {\lambda}{g (x)} \tanh (\alpha)$$

and thus

$$s = \alpha + \frac {\lambda}{g (x)} \tanh (\alpha)$$

Although $\beta$ does not appear explicitly in the control law, it is structurally embedded via the definition $\beta = u / g ( x )$ . Thus, the control effect is inherently integrated into the sliding surface, making $s = \alpha - \beta$ both a design and analysis tool.

It should be noted that the control law is not a function of the error s, but of the system’s intrinsic dynamics α. thus, this formulation:

• avoids chattering,   
• eliminates the use of discontinuous sign functions,   
• produces a smooth, bounded control input,   
• and is structurally robust without requiring state observers.

It is worth noting that in our formulation, the damping behavior required for smooth convergence arises naturally from the structure of $\alpha = \dot { x } + k _ { 1 } x$ . Unlike classical or higherorder SMC approaches where dissipation must be introduced externally or through additional tuning, our approach embeds this behavior structurally, eliminating the need for external compensation mechanisms.
