Proof of Lemma 8 WLOG, we assume $\mu \in ( \mu ^ { * } , + \infty )$ . In (49) and the arguments following it, we showed that for fixed $m , n , t$ such that $m + n \ge 1$ , the ratio $\frac { p ^ { \prime } ( m , n , t ; \mu ) } { p ( m , n , t ; \mu ) }$ is strictly decreasing with respect to $\mu .$ By repeating the same arguments as in (57)-(58), we can show that when $N _ { i } + A _ { i } \geq 1$ , the expectation $\mathbb E \left[ \frac { p ^ { \prime } \left( M _ { i + 1 } , N _ { i + 1 } , T _ { i + 1 } ; \mu ^ { * } \right) } { p \left( M _ { i + 1 } , N _ { i + 1 } , T _ { i + 1 } ; \mu ^ { * } \right) } \bigm | X _ { i } , N _ { i } , \alpha _ { i } , A _ { i } \right]$ is positive. Define $\tilde { \delta } _ { 1 }$ as

$$\tilde {\delta} _ {1} := \min _ {n, a \text { s.t.} n + a > 0} \mathbb {E} \left[ \frac {p ^ {\prime} \left(M _ {i + 1} , N _ {i + 1} , T _ {i + 1} ; \mu^ {*}\right)}{p \left(M _ {i + 1} , N _ {i + 1} , T _ {i + 1} ; \mu^ {*}\right)} \mid N _ {i} = n, A _ {i} = a \right] > 0.$$

By repeating the same arguments as in the proof of Lemma 3, we have

$$

$$

where the last line follows from (14). In the proof of Lemma 9, we showed that for $i \in \mathbb { N }$

$$\mathbb {E} \left[ D _ {i + 1} \mid X _ {i}, N _ {i}, \alpha_ {i}, A _ {i} \right] = \mathbb {E} \left[ \frac {p ^ {\prime} \left(M _ {i + 1} , N _ {i + 1} , T _ {i + 1} ; \mu^ {*}\right)}{p \left(M _ {i + 1} , N _ {i + 1} , T _ {i + 1} ; \mu^ {*}\right)} \mid X _ {i}, N _ {i}, \alpha_ {i}, A _ {i} \right],$$
