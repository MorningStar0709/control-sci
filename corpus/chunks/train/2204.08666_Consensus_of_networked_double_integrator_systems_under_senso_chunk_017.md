# A. Bias Estimation

The matrix $Y _ { i } ( { \dot { q } } _ { i } , { \ddot { q } } _ { i } , t )$ in (7) is dependent on the acceleration term ${ \ddot { q } } _ { i }$ and so cannot be used in our adaptation law for bias estimation. In order to facilitate relaxation of the persistence of excitation condition, a filter is designed for each agent as proposed in [28],

$$
\begin{array}{l} \dot {Y} _ {F _ {i}} = - \beta Y _ {F _ {i}} + Y _ {i} (\dot {q} _ {i}, \ddot {q} _ {i}, t), \quad Y _ {F _ {i}} (0) = 0 \tag {14} \\ \dot {w} _ {F _ {i}} = - \beta w _ {F _ {i}} + w _ {i}, \qquad w _ {F _ {i}} (0) = 0 \\ \end{array}
$$

where $\beta > 0$ is the scalar filter gain, $Y _ { F _ { i } } \in \mathbb { R } ^ { m \times ( m n + 2 ) }$ and $\boldsymbol { w } _ { F _ { i } } \in \mathbb { R } ^ { m }$ . Solving the above equations explicitly we obtain,

$$Y _ {F _ {i}} (t) = e ^ {- \beta t} \int_ {0} ^ {t} e ^ {\beta r} Y _ {i} (\dot {q} _ {i}, \ddot {q} _ {i}, r) d r \tag {15}w _ {F _ {i}} (t) = e ^ {- \beta t} \int_ {0} ^ {t} e ^ {\beta r} w _ {i} \mathrm{d} r. \tag {16}$$

Utilizing the relation $Y _ { i } ( { \dot { q } } _ { i } , { \ddot { q } } _ { i } , t ) \theta = w _ { i }$ we get $Y _ { F _ { i } } \theta = w _ { F _ { i } }$ from (15) and (16). $Y _ { F _ { i } }$ in (14) cannot be solved explicitly as $Y _ { i } ( { \dot { q } } _ { i } , { \ddot { q } } _ { i } , t )$ is not measured. Therefore, we split $Y _ { i } ( { \dot { q } } _ { i } , { \ddot { q } } _ { i } , t )$ into measured and non-measured parts as,

$$Y _ {i} (\dot {q} _ {i}, \ddot {q} _ {i}, t) = Y _ {1 _ {i}} (\ddot {q} _ {i}) + Y _ {2 _ {i}} (\dot {q} _ {i}, t)$$

where

$$
\begin{array}{l} Y _ {1 _ {i}} (\ddot {q} _ {i}) = \left[ \begin{array}{c c c} \ddot {q} _ {i}, & 0 _ {m \times 1}, & 0 _ {m \times m n} \end{array} \right] \\ Y _ {2 _ {i}} (\dot {q} _ {i}, t) = \\ \left[ 0 _ {m \times 1}, k (t) \dot {q} _ {i}, \frac {1}{2} k (t) [ Q _ {i 1} (t) I _ {m}, Q _ {i 2} (t) I _ {m},..., Q _ {i n} (t) I _ {m} ] \right] \\ \end{array}
$$

This implies that $Y _ { F _ { i } } = Y _ { F _ { 1 _ { i } } } + Y _ { F _ { 2 _ { i } } }$ , where,

$$
\begin{array}{l} \dot {Y} _ {F _ {1 _ {i}}} = - \beta Y _ {F _ {1 _ {i}}} + Y _ {1 _ {i}}, \quad Y _ {F _ {1 _ {i}}} (0) = 0 \tag {17} \\ \dot {Y} _ {F _ {2 _ {i}}} = - \beta Y _ {F _ {2 _ {i}}} + Y _ {2 _ {i}}, \qquad Y _ {F 2 _ {i}} (0) = 0 \\ \end{array}
$$
