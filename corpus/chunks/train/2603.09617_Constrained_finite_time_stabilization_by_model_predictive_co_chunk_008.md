# C. Problem statement

The objective is to design a stabilizing MPC framework that achieves finite-time convergence property for the constrained linear system (3)–(4). Compare with the terminalcost formulation in [23], the proposed scheme offers a significant improvement in initial feasibility.

With appropriate modifications, the finite-time MPC with improved initial feasibility can be extended to:

• constrained multi-input linear systems

$$x (k + 1) = A x (k) + \sum_ {j = 1} ^ {m} b _ {j} u _ {j} (k) = A x (k) + B u (k), \tag {11}$$

subject to

$$x \in \mathcal {X} \subset \mathbb {R} ^ {n}, u \in \mathcal {U} \subset \mathbb {R} ^ {m}, m > 1, \tag {12}$$

where $u = [ u _ { 1 } , \cdots , u _ { m } ] ^ { T } ; B = [ b _ { 1 } , \cdots , b _ { m } ]$ ; and (A, B) is controllable.

• constrained nonlinear plant

$$x (k + 1) = f (x (k), u (k)), \tag {13}u \in \mathcal {U} \subset \mathbb {R}, x \in \mathcal {X} \subset \mathbb {R} ^ {n} \tag {14}$$

where $f : \mathbb { R } ^ { n } \times \mathbb { R } \to \mathbb { R } ^ { n }$ is continuously differentiable in both arguments and satisfies $f ( 0 , 0 ) = 0$ . The nonlinear plant (13) is assumed to be feedback linearizable in a compact set $\mathcal { D } \subseteq \mathcal { X } \ [ 2 6 ]$ .
