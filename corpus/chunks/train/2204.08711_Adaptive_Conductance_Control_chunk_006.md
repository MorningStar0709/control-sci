The constants $\mu _ { \mathrm { i o n } } > 0$ and $E _ { \mathrm { i o n } } \in \mathbb { R }$ are called (intrinsic) maximal conductances and reversal potentials, respectively. The static activation functions $\sigma _ { m , \mathrm { i o n } } ( v )$ and $\sigma _ { h , \mathrm { i o n } } ( v )$ , and time-constant functions $\tau _ { m , \mathrm { i o n } } ( v )$ and $\tau _ { h , \mathrm { i o n } } ( v )$ , model the nonlinear gating of the ionic conductance. Because $\sigma _ { m , \mathrm { i o n } } : \mathbb { R } \to ( 0 , 1 )$ and $\sigma _ { h , \mathrm { i o n } } : \mathbb { R }  ( 0 , 1 )$ are monotonically increasing and decreasing, respectively, the states $m _ { \mathrm { i o n } }$ and $h _ { \mathrm { i o n } }$ are called activation and inactivation gating variables, respectively. The time-constant functions vary in shape, but always respect the bounds

$$0 < \underline {{\tau}} _ {\mathrm{ion}} \leq \tau_ {m, \mathrm{ion}} (v), \tau_ {h, \mathrm{ion}} (v) \leq \overline {{\tau}} _ {\mathrm{ion}}$$

for all $v \in \mathbb { R }$ and some $\underline { { \tau } } _ { \mathrm { i o n } } , \overline { { \tau } } _ { \mathrm { i o n } } > 0$ . The exponents $p _ { \mathrm { i o n } }$ and $q _ { \mathrm { i o n } }$ in (2a) are natural numbers (including zero).
