# 3. Problem Formulation

Consider a power network modeled as in Section 2. Under the condition (4), the unforced system admits a unique locally asymptotically stable equilibrium point $\left( \lambda ^ { \infty } , \omega ^ { \infty } \mathbf { 1 } _ { n } \right)$ . However, in the presence of disturbances, the transient frequency can enter unsafe regions before convergence to the equilibrium. This can be caused, for instance, by a sudden change in load or generation. If such frequency excursions exceed certain bounds, they might in fact lead to failure of loads or generators. To address this issue, we need to design feedback controllers $\{ u _ { i } \} _ { i \in \mathbb { Z } }$ to ensure each nodal frequency ωi stays within its safety bounds $[ \underline { { \omega } } _ { i } , \overline { { \omega } } _ { i } ]$ during transients, while preserving the asymptotic stability of (3).
