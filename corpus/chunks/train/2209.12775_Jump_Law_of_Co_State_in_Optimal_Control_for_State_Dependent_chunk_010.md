# A. Problem formulation

Consider the following general switched system with a time-varying switching interface $g ( x , t ) = 0 ;$

$$
\dot {x} = \left\{ \begin{array}{l l} f _ {1} (x, u, t), & \text { if   } g (x, t) \leq 0 \\ f _ {2} (x, u, t), & \text { if   } g (x, t) \geq 0 \end{array} \right., \tag {16}
$$

and stage cost

$$
L = \left\{ \begin{array}{l} L _ {1} (x, u, t), \text {   if   } g (x, t) \leq 0 \\ L _ {2} (x, u, t), \text {   if   } g (x, t) \geq 0 \end{array} \right.. \tag {17}
$$

Suppose the initial state $x _ { 0 } ,$ , the initial time $t _ { 0 } ,$ , and final time $t _ { f }$ are given.
