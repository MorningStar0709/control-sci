# 2.2.1 Displacements and Derivatives

We shall analyze the state and the motion of translational systems in terms of

Position, x(t), Velocity, x˙ (t), and Acceleration, x¨(t). Where each dot represents a time derivative:

$$\dot {x} = \frac {d}{d t} x (t) \qquad \ddot {x} = \frac {d ^ {2}}{d t ^ {2}} x (t)$$

Often we will omit the time dependence, i.e. ${ \dot { x } } = { \dot { x } } ( t )$ .
