# D. Leaders’ control

We choose u in (4a) such that leaders track the timevarying reference $\bar { \rho } ^ { L }$ in (21) with α and C as in Sec. III-B. To reject the bounded perturbations on the leaders, we select u such that

$$\left[ \rho^ {L} (x, t) u (x, t) \right] _ {x} = q ^ {L} (x, t), \tag {30}$$

where

$$q ^ {L} (x, t) = - k _ {p} ^ {L} e ^ {L} (x, t) - k _ {s} ^ {L} \text { sign } [ e ^ {L} (x, t) ] + \bar {\rho} _ {t} ^ {L} (x, t) + \delta (t), \tag {31}$$

$e ^ { L } = \bar { \rho } ^ { L } - \rho ^ { L } , k _ { p } ^ { L } ( t ) > 0 , k _ { s } ^ { L } > k \| \bar { \rho } ^ { L } ( \cdot , t ) \| _ { \infty }$ , and δ is chosen to satisfy boundary conditions. This control law guarantees global exponential stability in $\mathcal { L } ^ { 2 } ( \Omega )$ , since we can write

$$V _ {t} ^ {L} (t) \leq - k _ {p} ^ {L} V ^ {L} (t), \tag {32}$$

where $V ^ { L } = \textstyle { \frac { 1 } { 2 } } \| e ^ { L } \| _ { 2 } ^ { 2 }$ ; the proof follows from [16, Theorem 1].
