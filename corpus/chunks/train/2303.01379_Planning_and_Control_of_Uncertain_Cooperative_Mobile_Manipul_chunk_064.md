$$p _ {E _ {i}} (q _ {i}) = p _ {O} + R _ {i} (q _ {i}) p _ {E _ {i} / O} ^ {E _ {i}}, \tag {2.8a}\eta_ {E _ {i}} (q _ {i}) = \eta_ {O} + \eta_ {E _ {i} / O}, \tag {2.8b}$$

$\forall i \in \mathcal N$ , where $R _ { i } : = R _ { i } ( q _ { i } ) : \mathbb { R } ^ { n _ { i } }  \mathbb { S } \mathbb { O } ( 3 )$ is the i’s end-effector rotation matrix, and $p _ { E _ { i } / o } ^ { E _ { i } } , \eta _ { E _ { i } / o } \in \mathbb { R } ^ { 3 }$ E are the constant distance and orientation offset vectors between $\{ O \}$ and $\{ E _ { i } \}$ . Following (2.8), along with the fact that, due to the grasping rigidity, it holds that $\omega _ { E _ { i } } = \omega _ { o } , \forall i \in \mathcal { N }$ , one obtains

$$v _ {i} = J _ {O _ {i}} v _ {O}, \tag {2.9}$$

where $J _ { O _ { i } } : = J _ { O _ { i } } ( q _ { i } ) : \mathbb { R } ^ { n _ { i } }  \mathbb { R } ^ { 6 \times 6 }$ is the object-to-agent Jacobian matrix, with

$$
J _ {O _ {i}} (x) := \left[ \begin{array}{c c} I _ {3} & - S (R _ {i} (x) p _ {E _ {i} / O} ^ {E _ {i}}) \\ 0 & I _ {3} \end{array} \right], \forall x \in \mathbb {R} ^ {n _ {i}}, \tag {2.10}
$$

which is always full-rank. Moreover, from (2.9), one obtains

$$\dot {v} _ {i} = J _ {O _ {i}} \dot {v} _ {O} + \dot {J} _ {O _ {i}} v _ {O}. \tag {2.11}$$

In addition, it can be proved for $J _ { O _ { i } }$ that

$$\left\| J _ {O _ {i}} \left(q _ {i}\right) \right\| \leq \left\| p _ {O / E _ {i}} ^ {E _ {i}} \right\| + 1, \forall q _ {i} \in \mathbb {R} ^ {n _ {i}}, i \in \mathcal {N}, \tag {2.12}$$

which will be used in the subsequent analysis.

The kineto-statics duality along with the grasp rigidity suggest that the force $h _ { O }$ acting on the object’s center of mass and the generalized forces $h _ { i } , i \in \mathcal { N }$ , exerted by the agents at the grasping points, are related through:

$$h _ {O} = G h, \tag {2.13}$$

where $G : = G ( q ) : \mathbb { R } ^ { n }  \mathbb { R } ^ { 6 \times 6 N }$ , with $G ( q ) : = [ J _ { o _ { 1 } } ^ { \top } , \ldots , J _ { o _ { N } } ^ { \top } ]$ , is the full row-rank grasp matrix. By substituting (2.4) into (2.13), we obtain:

$$h _ {o} = G \left(u - M \dot {v} - C v - g - d\right),$$

which, after substituting (2.9), (2.11) , (2.5), and rearranging terms, yields the overall system coupled dynamics:

$$\widetilde {M} \dot {v} _ {o} + \widetilde {C} v _ {o} + \widetilde {g} + \widetilde {d} = G u, \tag {2.14}$$

where

$$\widetilde {M} := \widetilde {M} (x) := M _ {o} + G M G ^ {\top} \tag {2.15a}\widetilde {C} := \widetilde {C} (x) := C _ {o} + G C G ^ {\top} + G M \dot {G} ^ {\top} \tag {2.15b}\widetilde {g} := \widetilde {g} (x) := g _ {o} + G g. \tag {2.15c}\widetilde {d} := \widetilde {d} (x, t) := d _ {o} + G d \tag {2.15d}$$
