In the network-free case [9], $\varepsilon _ { i } \equiv 0$ and $0 < h _ { i } ^ { i } \le T _ { i }$ . If $\varepsilon _ { i } = 0$ in the networked case, the results derived in this paper are available for the periodic ETC case [9], [39]; otherwise, the ETM needs to be re-designed; see, e.g., [4] for the continuous ETC case.

To reduce the transmission frequency, a local ETM is implemented for each network. That is, at each sampling time $t _ { j } ^ { i }$ , only when the event-triggered condition for the i-th network is satisfied can the sampled information be transmitted via the $i -$ th network.
