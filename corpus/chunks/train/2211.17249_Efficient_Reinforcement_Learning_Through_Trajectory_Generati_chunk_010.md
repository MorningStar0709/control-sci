# 3.1. Span of historic trajectories

Let ${ \pmb u } _ { d , [ 0 , L - 1 ] }$ and ${ \pmb x } _ { d , [ 0 , L - 1 ] }$ be the a length-L input and state from past trajectory, and let the corresponding Hankel matrix $\mathcal { H } \in \mathbb { R } ^ { ( T m + T n ) \times ( L - T + 1 ) } )$ defined as

$$
\underbrace {\left[ \begin{array}{c} \boldsymbol {U} _ {0 , T , L - T + 1} \\ \boldsymbol {X} _ {0 , T , L - T + 1} \end{array} \right]} _ {\mathcal {H}} := \left[ \begin{array}{c c c c} \boldsymbol {u} _ {d} (0) & \boldsymbol {u} _ {d} (1) & \dots & \boldsymbol {u} _ {d} (L - T) \\ \vdots & \vdots & \ddots & \vdots \\ \boldsymbol {u} _ {d} (T - 1) & \boldsymbol {u} _ {d} (T) & \dots & \boldsymbol {u} _ {d} (L - 1) \\ \hline \boldsymbol {x} _ {d} (0) & \boldsymbol {x} _ {d} (1) & \dots & \boldsymbol {x} _ {d} (L - T) \\ \vdots & \vdots & \ddots & \vdots \\ \boldsymbol {x} _ {d} (T - 1) & \boldsymbol {x} _ {d} (T) & \dots & \boldsymbol {x} _ {d} (L - 1) \end{array} \right]. \tag {5}
$$

By the state-space version of Fundamental Lemma (De Persis and Tesi, 2019) shown below, any linear combination of the columns of the Hankel matrix is a length-T input-state trajectory of (1). The proof is provided in (De Persis and Tesi, 2019) and we supplement it in Appendix 7.1 for completeness.

Lemma 1 ( Fundamental Lemma) If rank $\left\lceil \begin{array} { c } { { U _ { 0 , T , L - T + 1 } } } \\ { { X _ { 0 , L - T + 1 } } } \end{array} \right\rceil = n + T m$ X0,L−T +1 , then any length-T input/state trajectory of system (1) can be expressed as $\begin{array} { r } { \Big [ \mathbf { \bar { u } } _ { [ 0 , T - 1 ] } \Big ] = \Big [ \frac { U _ { 0 , T , L - T + 1 } } { X _ { 0 , T , L - T + 1 } } \Big ] \mathbf { g } _ { \mathrm { ~ , ~ } } } \end{array}$ h U0,T ,L−T +1 g, where g ∈ RL−T +1. $\pmb { g } \in \mathbb { R } ^ { L - T + 1 }$

For the rank condition in Lemma 1 to hold, the minimum requirement on the length of the collected trajectory is $L - T + 1 = n + T m$ , namely, $L = ( m + 1 ) T - 1 + n$ . When the rank condition holds, linear combination of the columns of the Hankel matrix is also a length-T trajectory of the system. Thus, we generate a trajectory of length T using

$$
\left[ \tilde {\boldsymbol {u}} (0); \dots ; \tilde {\boldsymbol {u}} (T - 1); \tilde {\boldsymbol {x}} (0); \dots ; \tilde {\boldsymbol {x}} (T - 1) \right] = \underbrace {\left[ \begin{array}{c} \boldsymbol {U} _ {0 , T , L - T + 1} \\ \boldsymbol {X} _ {0 , T , L - T + 1} \end{array} \right]} _ {\mathcal {H}} \boldsymbol {g}. \tag {6}
$$

For convenience, we adopt the notation $\mathcal { H } _ { u } = U _ { 0 , T , L - T + 1 } , \mathcal { H } _ { x } = X _ { 0 , T , L - T + 1 }$ in the following sections. To represent the rows of blocks starting from the time $k = 0 , \cdots , T - 1$ , we denote

$$\mathcal {H} _ {x} ^ {k} := \left[ \boldsymbol {x} _ {d} (k) \boldsymbol {x} _ {d} (k + 1) \dots \boldsymbol {x} _ {d} (L - T + k) \right], \mathcal {H} _ {u} ^ {k} := \left[ \boldsymbol {u} _ {d} (k) \boldsymbol {u} _ {d} (k + 1) \dots \boldsymbol {u} _ {d} (L - T + k) \right]. \tag {7}$$
