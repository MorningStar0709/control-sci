In addition to the direct comparison of DMD-predicted and experimentally measured time histories, calculating the probability density function (PDF) of the variables under prediction is of interest for the application in irregular waves. To statistically assess the PDF estimator obtaining confidence intervals, a moving block bootstrap (MBB) method is applied to time histories from EFD and BHDMDc-based predictions. For the MBB analysis, a single long time history of T samples is obtained for the EFD and the BHDMDc, respectively, by joining all the measured and predicted test time series. Starting from this time signal with $\tau$ samples, a number $C = \mathcal { T } - r + 1$ of moving blocks is used, each defining a time history with $\xi _ { i }$ time samples $i = c , \ldots , c + r - 1$ , where c is the block index and $r = ( 2 \varphi / a ) ^ { 2 / 3 } T ^ { 1 / 3 }$ is an optimal block length [50] with

$$\varphi = \frac {\mathcal {T} \sum_ {i = 1} ^ {\mathcal {T}} \left[ \xi_ {i + 1} - \mathrm{EV} (\xi) \right] \left[ \xi_ {i} - \mathrm{EV} (\xi) \right]}{\left(\mathcal {T} - 1\right) \sum_ {i = 1} ^ {\mathcal {T}} \left[ \left(\xi_ {i} - \mathrm{EV} (\xi)\right) \right]} \tag {24}$$

and $a = ( 1 - \varphi ) ( 1 + \varphi )$ . From the original set of C blocks, a number of $C ^ { \prime } = \tau / r$ blocks are drawn at random with replacement and concatenated in the order they are picked, forming a new bootstrapped series of size $\tau$ . The PDF of each bootstrapped time series is obtained using kernel density estimation [51] as follows:

$$\operatorname{PDF} (\xi , y) = \frac {1}{\mathcal {T} h} \sum_ {i = 1} ^ {\mathcal {T}} K \left(\frac {y - \xi_ {i}}{h}\right). \tag {25}$$

Here, K is a normal kernel function defined as

$$K (\xi) = \frac {1}{\sqrt {2 \pi}} \exp \left(- \frac {\xi^ {2}}{2}\right), \tag {26}$$

where

$$h = 1. 0 6 \min (S D [ \xi ], \mathrm{IQR} (\xi)) \mathcal {T} ^ {- 1 / 5} \tag {27}$$

is the bandwidth [52], and IQR(ξ) is the inter-quartile range for the variable $\xi$
