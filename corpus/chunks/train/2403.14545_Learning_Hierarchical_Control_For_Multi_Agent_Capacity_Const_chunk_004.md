# II. NOTATION

Unless otherwise indicated, all introduced vectors are column vectors. Parentheses ( ) stack vectors vertically, and brackets [ ] stack vectors horizontally. The notation $a = \mathbf { 0 } _ { b }$ indicates a vector $a \in \mathbb { R } ^ { b }$ , where $a _ { i } = 0 , \ i \in [ 1 , b ]$ .
