# B. Design criterion using generalized Lipschitz property

Lipschitz property is considered to be conservative, especially when the Lipschitz constant turns out to be very large. The generalized Lipschitz condition, introduced in [56], is considered to be less conservative, which states that, for every $x , { \hat { x } } \in { \mathcal { X } } .$ , there exist positive definite matrices $V \in \mathbb { R } ^ { n _ { f } \times n _ { f } }$ and $W \in \mathbb { R } ^ { k \times k }$ such that

$$\| f (H x) - f (H \hat {x}) \| _ {V} \leq \| H (x - \hat {x}) \| _ {W} \tag {21}$$

where the norms $\| a \| _ { V } : = { \sqrt { a ^ { \mathrm { T } } V a } }$ and $\| b \| _ { W } : = \sqrt { b ^ { \mathrm { T } } W b }$ for $a \in \mathbb { R } ^ { n _ { f } }$ and $b \in \mathbb { R } ^ { k }$ .

Lemma 2. If the generalized Lipschitz property (21) holds for some positive definite matrices V and W , then, for every $\Gamma \in \mathbb { R } ^ { n _ { f } \times k }$ and $x , { \hat { x } } \in { \mathcal { X } }$ , it holds

$$2 \xi^ {\mathrm{T}} \Gamma^ {\mathrm{T}} \tilde {f} \leq \| \Gamma \xi \| _ {V ^ {- 1}} ^ {2} + \| \xi \| _ {W} ^ {2}$$

where $\xi : = H ( x - { \hat { x } } )$ and $\tilde { f } : = f ( H x ) - f ( H \hat { x } )$ .

Proof: It holds that

$$
\begin{array}{l} 0 \leq (V ^ {- \frac {1}{2}} \Gamma \xi - V ^ {\frac {1}{2}} \tilde {f}) ^ {\mathrm{T}} (V ^ {- \frac {1}{2}} \Gamma \xi - V ^ {\frac {1}{2}} \tilde {f}) \\ = \| \Gamma \xi \| _ {V ^ {- 1}} ^ {2} + \| \tilde {f} \| _ {V} ^ {2} - 2 \xi^ {\mathrm{T}} \Gamma^ {\mathrm{T}} \tilde {f} \\ \leq \| \Gamma \xi \| _ {V ^ {- 1}} ^ {2} + \| \xi \| _ {W} ^ {2} - 2 \xi^ {\mathrm{T}} \Gamma^ {\mathrm{T}} \tilde {f}. \\ \end{array}
$$
