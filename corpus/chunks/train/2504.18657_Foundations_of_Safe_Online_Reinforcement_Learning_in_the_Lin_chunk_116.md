# G.7 Proof of Lemma 25

proof. By Lemma 28, $F _ { \mathrm { o p t } } ( \theta )$ is the value of $\begin{array} { r } { K \in \left( \frac { a - 1 } { b } , \frac { a } { b } \right] } \end{array}$ that minimizes the function $\frac { q + r K ^ { 2 } } { 1 - ( a - b K ) ^ { 2 } }$ (note that we ignore the constant $\sigma _ { \mathcal { D } } ^ { 2 }$ as this is a positive constant and does not change the minimization problem). Taking the derivative of this function and equating to 0, we have that $F _ { \mathrm { o p t } } ( \theta )$ is the solution to

$$\frac {2 K r (1 - (a - b K) ^ {2}) - 2 b (a - b K) (q + r K ^ {2})}{(1 - (a - b K) ^ {2}) ^ {2}} = 0.$$

Simplifying, we have

$$a b r K ^ {2} + (b ^ {2} q + r - a ^ {2} r) K - a b q = 0$$

Applying the quadratic formula, we get that the positive root is

$$F _ {\mathrm{opt}} (\theta) = \frac {a ^ {2} r - b ^ {2} q - r + \sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r}}{2 a b r}.$$

We also observe that

$$\left(a ^ {2} r + b ^ {2} q + r\right) ^ {2} - \left(\sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r}\right) ^ {2} = 4 a ^ {2} r ^ {2},$$

which implies that

$$
\begin{array}{l} \left(a ^ {2} r + b ^ {2} q + r\right) - \left(\sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r}\right) \\ = \frac {4 a ^ {2} r ^ {2}}{(a ^ {2} r + b ^ {2} q + r) + (\sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r})}. \\ \end{array}
$$

Because $\underline { { a } } \geq a \geq \bar { a } , \underline { { b } } \geq b \geq \bar { b } .$ , and $r > 0$ , this implies that there exists a constant $c _ { \mathrm { L 2 5 } } ^ { F 1 } > 0$ such that

$$
\begin{array}{l} \frac {a}{b} - F _ {\mathrm{opt}} (\theta) = \frac {a ^ {2} r + b ^ {2} q + r - \sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r}}{2 a b r} \\ = \frac {4 a ^ {2} r ^ {2}}{2 a b r \left(a ^ {2} r + b ^ {2} q + r + \sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r}\right)} \\ \geq \frac {4 \underline {{a}} ^ {2} r ^ {2}}{2 \bar {a} \bar {b} r \left(\bar {a} ^ {2} r + \bar {b} ^ {2} q + r + \sqrt {(\bar {b} ^ {2} q + r - \underline {{a}} ^ {2} r) ^ {2} + 4 \bar {a} ^ {2} \bar {b} ^ {2} q r}\right)} \\ := c _ {\mathrm{L25}} ^ {F 1} \\ > 0. \\ \end{array}
$$

Similarly, we have that

$$\left(r (a - 1) ^ {2} + b ^ {2} q\right) ^ {2} - \left(\sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r}\right) ^ {2} = - 4 a r \left((a - 1) ^ {2} r + b ^ {2} q\right).$$

which implies that

$$
\begin{array}{l} \left(r (a - 1) ^ {2} + b ^ {2} q\right) - \left(\sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r}\right) \\ = \frac {- 4 a r ((a - 1) ^ {2} r + b ^ {2} q)}{(r (a - 1) ^ {2} + b ^ {2} q) + (\sqrt {(b ^ {2} q + r - a ^ {2} r) ^ {2} + 4 a ^ {2} b ^ {2} q r})}. \\ \end{array}
$$
