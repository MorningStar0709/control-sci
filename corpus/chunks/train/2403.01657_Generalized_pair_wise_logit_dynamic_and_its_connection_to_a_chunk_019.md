# Proposition 3

Solutions to (10) and (21) with the same initial conditions are denoted as $\mu$ and $\mu _ { { } _ { N } }$ , respectively.

Assume that $\operatorname* { l i m } _ { N  + \infty } \| \mu _ { N } ( 0 , \cdot ) - \mu ( 0 , \cdot ) \| = 0$ for any $A \in { \mathfrak { B } }$ . Then, it follows that

$$\lim _ {N \rightarrow + \infty} \| \mu_ {N} (t, \cdot) - \mu (t, \cdot) \| = 0 \text { at each } t > 0. \tag {23}$$

Remark 5: According to Proof of Proposition 2, the accuracy of the discretization depends on $\operatorname* { m a x } _ { u , \nu \in \Omega } \left. f _ { N } \left( u , \nu \right) - f \left( u , \nu \right) \right. \cdot \mathrm { ~ I f ~ } \left. f \left( u _ { 1 } , \nu _ { 1 } \right) - f \left( u _ { 2 } , \nu _ { 2 } \right) \right. \leq L _ { f } \left( \left. u _ { 1 } - u _ { 2 } \right. + \left. \nu _ { 1 } - \nu _ { 2 } \right. \right)$ with a constant $L _ { f } > 0$ for any $u _ { 1 } , u _ { 2 } , \nu _ { 1 } , \nu _ { 2 } \in \Omega$ ,then

$$
\begin{array}{l} \max _ {u, v \in \Omega} \left| f _ {N} (u, v) - f (u, v) \right| = \max _ {1 \leq i, j \leq N} \max _ {u \in \Omega_ {i}, v \in \Omega_ {j}} \left| f _ {N} (u, v) - f (u, v) \right| \\ = \max _ {1 \leq i, j \leq N} \max _ {u \in \Omega_ {i}, v \in \Omega_ {j}} \left| f _ {N} \left(x _ {i - 1 / 2}, x _ {j - 1 / 2}\right) - f (u, v) \right| \\ \leq \max _ {1 \leq i, j \leq N} \max _ {u \in \Omega_ {i}, v \in \Omega_ {j}} L _ {f} \left(\left| x _ {i - 1 / 2} - u \right| + \left| v - x _ {j - 1 / 2} \right|\right), \tag {24} \\ \leq \max _ {1 \leq i, j \leq N} \max _ {u \in \Omega_ {i}, v \in \Omega_ {j}} L _ {f} \left(\frac {1}{2 N} + \frac {1}{2 N}\right) \\ = \frac {L _ {f}}{N} \\ \end{array}
$$

implying the first-order convergence $\left\| \mu _ { N } \left( t , \cdot \right) - \mu ( t , \cdot ) \right\| = O \left( N ^ { - 1 } \right)$ for a large $N \in \mathbb N$
