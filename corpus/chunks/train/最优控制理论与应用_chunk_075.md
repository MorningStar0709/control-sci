$$\lambda (2) = \frac {\partial \phi}{\partial x (2)} = \frac {\partial \left[ \frac {1}{2} x ^ {2} (2) \right]}{\partial x (2)} = x (2) \tag {4-104}$$

控制方程为

$$\frac {\partial H (k)}{\partial u (k)} = u (k) + \lambda (k + 1) = 0 \tag {4-105}u (k) = - \lambda (k + 1) \tag {4-106}$$

因协态为常数,故控制也是常数,令

$$u (k) = u \tag {4-107}$$

现在来解系统的状态方程,由初始条件 $x(0)=1$ 可得

$$x (1) = 1 + u \tag {4-108}x (2) = x (1) + u = 1 + 2 u \tag {4-109}$$

因为

$$x (2) = \lambda (2) = - u = 1 + 2 u \tag {4-110}$$

故

$$u = u ^ {*} = - \frac {1}{3} \tag {4-111}$$

于是最优控制为

$$u ^ {*} (k) = - \frac {1}{3} \quad k = 0, 1 \tag {4-112}$$

代入系统状态方程,可求得最优状态为

$$x ^ {*} (0) = 1 \quad x ^ {*} (1) = \frac {2}{3} \quad x ^ {*} (2) = \frac {1}{3} \quad (4 - 1 1 3)$$

例4-5 在 $N$ 级换热器系列的最优设计中，设 $x(k)$ 为流出第 $k$ 个换热器的油料温度， $u(k - 1)$ 是第 $k$ 个换热器的换热面积， $T_{k}$ 是第 $k$ 个换热器的热载体温度， $Q_{k}$ 是第 $k$ 个换热器的正常数。则状态方程为

$$x (k + 1) = \frac {x (k) + Q _ {k + 1} T _ {k + 1} u (k)}{1 + Q _ {k + 1} u (k)} \quad (k = 0, 1, \dots , N - 1) \tag {4-114}$$

方程右端对 $u(k)$ 是非线性的。这里 k 表示加热器级数，是空间离散变量，但在求解时与时间离散问题一样。

边界条件为

$$x (0) = a \quad x (N) = b \tag {4-115}$$

性能指标是使换热总面积最小,即

$$J = \sum_ {k = 0} ^ {N - 1} u (k) \tag {4-116}$$

最小。

解 这里 $u(k)$ 无约束, 可用变分法求解。

作哈密顿函数

$$H (k) = u (k) + \lambda (k + 1) \frac {x (k) + Q _ {k + 1} T _ {k + 1} u (k)}{1 + Q _ {k + 1} u (k)} \tag {4-117}$$

协态方程为

$$\lambda (k) = \frac {\partial H (k)}{\partial x (k)} = [ 1 + Q _ {k + 1} u (k) ] ^ {- 1} \lambda (k + 1)$$

即

$$\lambda (k + 1) = [ 1 + Q _ {k + 1} u (k) ] \lambda (k) \tag {4-118}$$

控制方程为

$$\frac {\partial H (k)}{\partial u (k)} = 0$$

即

$$1 + \lambda (k + 1) \frac {Q _ {k + 1} [ T _ {k + 1} - x (k) ]}{[ 1 + Q _ {k + 1} u (k) ] ^ {2}} = 0 \tag {4-119}$$

由上式求出 $\lambda (k + 1)$ 比求 $u(k)$ 容易，故解得

$$\lambda (k + 1) = - \frac {\left[ 1 + Q _ {k + 1} u (k) \right] ^ {2}}{Q _ {k + 1} \left[ T _ {k + 1} - x (k) \right]} \tag {4-120}$$

将式(4-120)代入协态方程(4-118)，消去 $\lambda(k)$ ，得

$$\frac {1 + Q _ {k + 1} u (k)}{Q _ {k + 1} \left[ T _ {k + 1} - x (k) \right]} = \frac {\left[ 1 + Q _ {k} u (k - 1) \right] ^ {2}}{Q _ {k} \left[ T _ {k} - x (k - 1) \right]} \tag {4-121}$$

由状态方程(4-114)可解出 $u(k)$

$$u (k) = \frac {x (k + 1) - x (k)}{Q _ {k + 1} [ T _ {k + 1} - x (k + 1) ]} \tag {4-122}$$

令 $k = k - 1$ ，由上式可得

$$u (k - 1) = \frac {x (k) - x (k - 1)}{Q _ {k} [ T _ {k} - x (k) ]} \tag {4-123}$$

将式(4-122)、(4-123)代入式(4-121)，消去 $u(k)$ ，可得

$$x (k - 1) = x (k) + \left[ T _ {k} - x (k) \right] \left\{1 - \frac {Q _ {k} \left[ T _ {k} - x (k) \right]}{Q _ {k + 1} \left[ T _ {k + 1} - x (k + 1) \right]} \right\} \tag {4-124}$$

式(4-124)是关于 $x(k)$ 的非线性差分方程,若已知 $x(k)$ 和 $x(k+1)$ ,就可递推求出 $x(k-1)$ ,故从终端 k=N 向后递推比较方便。已知 $x(N)=b$ ,但不知 $x(N-1)$ ,只能先假定一个 $x(N-1)$ ,由式(4-124)算出 $x(N-2)$ ;再循环用式(4-124)可依次递推求得 $x(N-3)$ , $\cdots$ , $x(1)$ , $x(0)$ 。若最后求出的 $x(0)$ 等于或很接近于给定的初始条件 a,则这组序列 $\{u(k)\}$ 就是最优状态轨迹;否则另取 $x(N-1)$ 再重算,直到 $x(0)\approx a$ ,这组序列 $\{x(k)\}$ 就是最优状态轨迹。把 $\{x(k)\}$ 代入式(4-122)就可求出最优控制序列 $\{u(k)\}$ 。从上面的说明可知,要反复试凑以满足 $x(k)$ 的边界条件,这是非线性两点边值问题所引起的。这里因为 $x(k)$ 的初始和终端条件都给定,这里采用的解题技巧是消去协态量 $\lambda(k)$ ,直接求解 $x(k)$ 。
