A common way to solve POMGs is Multi-Agent Actor Critic, proposed in [17], adopting the Centralized Training with Decentralized Execution (CTDE) framework. Consider N agents, whose policies $\pi = ( \pi _ { 1 } , \cdot \cdot \cdot , \pi _ { N } )$ are parameterized by $\theta = ( \theta _ { 1 } , \cdot \cdot \cdot , \theta _ { N } )$ . In this approach, each agent has its own actor network determining a deterministic action $a _ { i }$ from its local observation $o _ { i }$ . The joint state $s =$ $\left( o _ { 1 } , \cdots , o _ { N } \right)$ and joint action $a = ( a _ { 1 } , \cdots , a _ { N } )$ correspond to the observations and actions of all agents, respectively. The centralized critic network estimates the joint Q-value $Q _ { i } ^ { \pi } ( s , a )$ based on the joint state-action pairs $( s , a )$ , providing Q-value approximations to decentralized actor networks. To train agents, an experience replay buffer D stores tuples $\tau = ( s , s ^ { \prime } , a , r _ { 1 } , \cdot \cdot \cdot , r _ { N } )$ recording transitions for all agents. Then, a batch of these transitions is randomly sampled from the buffer during training. Next, the one-step lookahead TD error is computed to update the centralized critic network as

$$\mathcal {L} (\theta_ {i}) = \mathbb {E} _ {\tau \sim \mathcal {D}} \big [ (Q _ {i} ^ {\pi} (s, a) - y) ^ {2} \big ], \tag {5}$$

where $y = r _ { i } + \gamma Q _ { i } ^ { \pi ^ { \prime } } ( s ^ { \prime } , a ^ { \prime } ) | _ { a _ { i } ^ { \prime } = \pi _ { i } ^ { \prime } ( o _ { j } ) }$ for the set of target policies $\pi ^ { \prime } .$ Lastly, the deterministic policy gradient updates each agent’s actor parameters as

$$\nabla_ {\theta_ {i}} J (\pi_ {i}) = \underset {\tau \sim \mathcal {D}} {\mathbb {E}} \left[ \nabla_ {\theta_ {i}} \pi_ {i} (a _ {i} | o _ {i}) \nabla_ {a _ {i}} Q _ {i} ^ {\pi} (s, a) | _ {a _ {i} = \pi_ {i} (o _ {i})} \right]. \tag {6}$$

Importantly, if the actions of all agents are known, the environment remains stationary even as policies change, addressing non-stationarity, since $P ( s ^ { \prime } | s , a , \pi _ { 1 } , \cdot \cdot \cdot , \pi _ { N } ) =$ $P ( s ^ { \prime } | s , a ) = P ( s ^ { \prime } | s , a , \pi _ { 1 } ^ { \prime } , \cdot \cdot , \pi _ { N } ^ { \prime } )$ for any $\pi _ { i } \neq \pi _ { i } ^ { \prime } .$
