The problem of minimizing $\boldsymbol { J } ( \boldsymbol { x } , t )$ seems to be solved by Lemma III.1. However, its decentralized realization in multiagent systems is nontrivial in general, which is the main focus of this papaer. One key problem arises from sudden changes of the underlying graph topology, i.e., edges defining adjacency can suddenly appear or disappear (with a finite number of switching instants within a finite time interval). In preparation for the next section where we address this problem, we adapt the nonsmooth CLF framework in [23] (see, e.g., [40] for earlier work on nonsmooth CLFs) to the time-varying case.
