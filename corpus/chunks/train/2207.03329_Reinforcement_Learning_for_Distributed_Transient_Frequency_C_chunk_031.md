# Algorithm 1 Training Process of RLb Algorithm

Require: Total time stages K, number of episodes $I ,$ batch size $H ,$ , learning rate α, parameters in (14)

Input: Initial neural network parameters $\phi$

1: for episode $i = 1$ to I do   
2: Randomly generate initial states $\lambda ^ { h } ( 0 ) , \omega ^ { h } ( 0 )$ for h-th batch, $h = 1 , . . . , H$   
3: Reset the state of cells in h-th batch as $\{ \lambda ^ { h } ( 0 ) , \omega ^ { h } ( 0 ) \}$   
4: RNN cells compute through K stages to obtain the output $\gamma \| \omega ( k ) - \bar { \omega ^ { \infty } } \mathbf { 1 } _ { n } \| ^ { 2 } + \| \mathbf { \bar { \boldsymbol { u } } } _ { \phi } ( k ) \| ^ { 2 }$ for $k = 1 , . . . , K$   
5: Calculate the average loss of all the batches Loss = $\begin{array} { r } { \frac { 1 } { H } \sum _ { h = 1 } ^ { H } \sum _ { k = 1 } ^ { K } \gamma \| \omega ^ { h } ( k ) - \omega ^ { \infty } \mathbf { 1 } _ { n } \| ^ { 2 } + \| u _ { \phi } ^ { h } ( k ) \| ^ { 2 } } \end{array}$   
6: Update neural network parameters by passing Loss to Adam optimizer: $\phi  \phi - \alpha \mathrm { A d a m } ( \mathsf { L o s s } )$   
7: end for

Output: Trained controllers $u _ { \phi }$
