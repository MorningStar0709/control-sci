# A. Modified LNN for Learning Dynamics

In LNNs, FCNNs are used to represent the Lagrangian function. In doing so, the approximated inertia matrix $\begin{array} { r } { \hat { M } = \ \frac { \partial ^ { 2 } \mathcal { L } ( q , \dot { q } ; \gamma ^ { * } ) } { \partial \dot { q } \partial \dot { q } } } \end{array}$ and the Coriolis matrix $\hat { C } = \hat { \hat { M } } ( q ) \textrm { - }$ $\begin{array} { r } { \frac { 1 } { 2 } \dot { M } ( \pmb q ) ^ { \top } } \end{array}$ may not satisfy the Property 1 and Property 2. To solve this problem, we use an ICNN $\mathcal { L } _ { T } ( \pmb { q } , \pmb { \dot { q } } ; \gamma _ { 1 } )$ with the parameter $\gamma _ { 1 }$ to approximate the kinetic energy $T ( \pmb q , \dot { \pmb q } )$ in ${ \mathcal { L } } ,$ which is convex w.r.t. $\dot { \pmb q } .$ Moreover, we use an FCNN $\mathcal { L } _ { V } ( q ; \gamma _ { 2 } )$ with the parameter $\gamma _ { 2 }$ and input $\pmb q$ to learn $V ( q )$ in ${ \mathcal { L } } .$ . Then we construct the Lagrangian neural network as follows.

$$\mathcal {L} (\boldsymbol {q}, \dot {\boldsymbol {q}}; \boldsymbol {\gamma}) = \mathcal {L} _ {T} (\boldsymbol {q}, \dot {\boldsymbol {q}}; \boldsymbol {\gamma_ {1}}) - \mathcal {L} _ {V} (\boldsymbol {q}; \boldsymbol {\gamma_ {2}}),$$

where $\gamma = \{ \gamma _ { 1 } , \gamma _ { 2 } \}$ , the activation function we choose is softplus. Through this design, we can ensure that $\hat { M } \ =$ $\begin{array} { r } { \frac { \partial ^ { 2 } \mathcal { L } ( q , \dot { q } ; \gamma ^ { * } ) } { \partial \dot { q } \partial \dot { q } } = \frac { \partial ^ { 2 } \mathcal { L } _ { T } ( q , \dot { q } ; \gamma ^ { * } ) } { \partial \dot { q } \partial \dot { q } } } \end{array}$ is positive definite and $\dot { \hat { M } } - 2 \hat { C }$ is skew-symmetric.

Using LNNs to approximate the dynamic system model may introduce approximation errors. With $\hat { M } , \hat { \hat { C } } , \hat { G }$ , the system dynamics can be modeled as

$$\hat {M} \ddot {\boldsymbol {q}} + \hat {C} \dot {\boldsymbol {q}} + \hat {G} + \delta (\boldsymbol {q}, \dot {\boldsymbol {q}}) = \boldsymbol {u} + \boldsymbol {\tau} ^ {d}, \tag {14}$$

where $\delta ( { \pmb q } , \dot { \pmb q } ) = - \hat { M } M ^ { - 1 } \{ { \pmb u } + { \pmb \tau } ^ { d } - C \dot { \pmb q } - { \pmb G } \} + \{ { \pmb u } + { \pmb \tau } ^ { d } -$ ${ \hat { C } } { \dot { q } } - { \hat { G } } \}$ is the model mismatch.

Since δ is a function of q and $\dot { \mathbf { q } } ,$ we can employ the firstorder Taylor expansion and assume that the uncertain term satisfies the inequality

$$\| \delta (\boldsymbol {q}, \dot {\boldsymbol {q}}) \| \leq a \| \boldsymbol {q} \| + b \| \dot {\boldsymbol {q}} \| + c, \tag {15}$$

where $a , b , c$ are positive constants.
