# A. Closed-form Solution for $\mathbf { \Delta } \mathbf { u } _ { \mathrm { s } }$

If the conditions stated in Subsection III-D hold, the solution to the inequality (22) is not necessarily unique. Therefore, it becomes possible to formulate an optimization problem aimed at discovering safeguarding controllers that are less intrusive, potentially achieved by minimizing $\lVert \boldsymbol { u } _ { \mathrm { s } } \rVert$ . Another option is to seek a closed-form solution for the safeguarding control. To this end, our strategy is to manipulate just one of the control signals. In other words, for a system with more than one control input, we add the safeguarding control action to one of the control inputs.
