# 4.2 Hyperboloid foliation

Let us now fix an initial point on the hyperboloid:

$$\eta_ {0} \in H.$$

Define the half spaces

$$\Pi^ {+} = \left\{M \in \mathfrak {s l} _ {2} (\mathbb {R}) \mid \operatorname{tr} (M A) > \operatorname{tr} (M B) \right\},\Pi^ {-} = \left\{M \in \mathfrak {s l} _ {2} (\mathbb {R}) \mid \operatorname{tr} (M A) < \operatorname{tr} (M B) \right\},$$

and the half hyperboloids

$$H ^ {+} = H \cap \Pi^ {+},H ^ {-} = H \cap \Pi^ {-}.$$

From condition (3.7), we can see that if $\eta _ { 0 } \in H ^ { + }$ , the control u, in order to satisfy Pontryagin Maximum Principle, must be equal to 1 until the trajectory η meets the switching curve S. Notice that, as far as $u ( t ) = 1$ , the Hamiltonian function of the problem is $H ( \eta ( t ) , 1 ) = \operatorname { t r } ( \eta ( t ) A )$ , hence the quantity $\mathrm { t r } ( \eta ( t ) A )$ is constant.

$\mathrm { S o } , \mathrm { i f } \eta _ { 0 } \in H ^ { + }$ is such that $\mathrm { t r } ( \eta _ { 0 } A ) = c _ { 0 }$ , then the trajectory η is contained in the curve $H ^ { + } \cap \Pi _ { A , c _ { 0 } }$ (see Figure 3). In particular, we can suppose that geometrically the trajectory η coincides with the whole curve $H ^ { + } \cap \Pi _ { A , c _ { 0 } }$ . Indeed, if η accumulates at some point, then the control $u ( t )$ is identically equal to 1 or to 0 for every t sufficiently large and, as we will see, the corresponding Lyapunov exponents are easy to compute.

Now, we want to study all the possible configuration of H, S and the planes $\Pi _ { A , c _ { 0 } } , \Pi _ { B , c _ { 0 } }$ . Consider the straight line given by the intersection of the three planes $\Pi , \Pi _ { A , c _ { 0 } } , \Pi _ { B , c _ { 0 } }$ :

$$

$$

There are three cases: either this line does not intersect the hyperboloid H, either it is tangent to the hyperboloid, or it has two distinct intersection with the hyperboloid.
