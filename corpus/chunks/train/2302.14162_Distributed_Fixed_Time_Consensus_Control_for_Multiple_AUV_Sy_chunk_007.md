The fuzzy rules of fuzzy logic system can be described as:

$$\text { Rule } j: \text { If } z _ {1} \text { is } A _ {1} ^ {j} \text { and ... and } z _ {n} \text { is } A _ {n} ^ {j} \text { then } y \text { is } B ^ {j} \tag {5}$$

$$y = \sum_ {j = 1} ^ {h} w _ {j} \left(\prod_ {i = 1} ^ {n} \mu_ {A _ {i} ^ {j}} (z _ {i})\right) / \sum_ {j = 1} ^ {h} \left(\prod_ {i = 1} ^ {n} \mu_ {A _ {i} ^ {j}} (z _ {i})\right) = \mathrm{w} ^ {T} \Psi (Z) \tag {6}$$

$$\Psi_ {j} (Z) = \frac {\prod_ {i = 1} ^ {n} \mu_ {A _ {i} ^ {j}} (z _ {i})}{\sum_ {j = 1} ^ {h} \left(\prod_ {i = 1} ^ {n} \mu_ {A _ {i} ^ {j}} (z _ {i})\right)}. \tag {7}$$
