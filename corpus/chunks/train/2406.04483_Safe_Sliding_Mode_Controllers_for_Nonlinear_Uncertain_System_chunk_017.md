# C. Strategies to Achieve Less Invasive Safeguarding Control

In the implementation of the proposed method, it is of our interest to achieve a safeguarding control with limited interference on the functionality of ${ \pmb u } _ { \mathrm { s m c } }$ . To this end, we employ two different strategies. First, the designed safeguarding control mechanism is activated only when the system state tends to approach the unsafe set. To formulate this strategy, we define a subset of the safe set C in which the safeguarding control is activated. This “risky set” is shown with $\mathcal { C } _ { \mathrm { R } }$ and is defined as follows:

$$\mathcal {C} _ {\mathrm{R}} = \{\boldsymbol {x} \in \mathbb {R} ^ {n} \mid 0 \leq h (\boldsymbol {x}) \leq \bar {h} \}, \tag {27}$$

where $\bar { h }$ is a strictly positive constant. If $t _ { 1 }$ shows the time when the system trajectory intersects the risky set for the first time, the safeguarding control law is then set to zero for $t < t _ { 1 }$ and to $\mathbf { \Delta } \pmb { u } _ { s }$ such that $\begin{array} { r } { \pmb { a } ( \pmb { x } , z ) \pmb { u } _ { \mathrm { s } } - b ( \pmb { x } , z ) \| \pmb { u } _ { \mathrm { s } } \| _ { \infty } \geq c ( \pmb { x } , z ) } \end{array}$ for $t \geq t _ { 1 }$ . It is clear that $t _ { 1 }$ is zero if $h ( { \pmb x } ( 0 ) ) \le h$ .

On the other hand, if the sliding manifold does not cross the unsafe set, it is always possible to find $\Omega \subset { \mathcal { C } }$ such that $\mathbf { 0 } \in \Omega$ and ⌦ is forward invariant when $\mathbf { { u } } _ { \mathrm { { s } } } = \mathbf { { 0 } }$ . ⌦ can be considered as any subset of $\{ \pmb { x } \in \mathbb { R } ^ { n } \ \mid \ \pmb { s } ^ { \mathrm { T } } \pmb { s } <$ < dist $( s \ =$ $\mathbf { 0 } , \partial { \mathcal { C } } ) \} \cap { \mathcal { R } } _ { \mathrm { A } }$ . This is due to the fact that $\dot { V } _ { \mathrm { s m c } } ( s )$ is always negative for $\mathbf { \boldsymbol { x } } \in \Omega ,$ and hence, ⌦ is forward invariant. The second strategy is to set $\mathbf { { u } } _ { \mathrm { { s } } } = \mathbf { { 0 } }$ when the system trajectory reaches the set ⌦.
