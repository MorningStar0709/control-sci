# 4. Decentralized Information.

4.1. Decentralized Model with Local Measurements. Consider a continuoustime process $\{ X _ { t } \}$ on a Euclidean space $\mathbb { R } ^ { d }$ , controlled by a collection of control process $\mathbf { U } _ { t } : = ( U _ { t } ^ { k } , i = 1 , \cdots , N )$ with each $U _ { t } ^ { i }$ taking values in a compact Borel action space $\mathbb { U } ^ { i }$ , and with an associated observation process $\{ Y _ { t } ^ { i } \}$ taking values in $\mathbb { R } ^ { M }$ , where $0 \leq t \leq T$ . Let $\mathbf { Y } = ( Y ^ { 1 } , \cdots , Y ^ { N } )$ . The evolution of $\{ X _ { t } , Y _ { t } ^ { i } , i = 1 , \cdot \cdot \cdot , N \}$ is given by the stochastic differential equations

$$d X _ {t} = b (X _ {t}, U _ {t} ^ {1}, \dots , U _ {t} ^ {N}) d t + \sigma (X _ {t}) d W _ {t},d Y _ {t} ^ {i} = g ^ {i} (X _ {t}) d t + d B _ {t} ^ {i}, \quad i = 1, \dots , N. \tag {4.1}$$

Where, W and $B ^ { i } , i = 1 , \cdots , N$ are independent standard Wiener processes with values in $\mathbb { R } ^ { d }$ and $\mathbb { R } ^ { M }$ , respectively and $g ^ { i } : \bar { \mathbb { R } ^ { d } }  \mathbb { R } ^ { M } , i = 1 , \dots , N$ are given continuous and bounded functions .

In this section, we assume that the drift term b and the diffusion matrix σ satisfies similar conditions as in (A1) and (A2). In particular, they satisfy the following:

(D1) Local Lipschitz continuity: The function $\begin{array} { r } { b \colon \mathbb { R } ^ { d } \times \prod _ { k = 1 } ^ { N } \mathbb { U } ^ { k } \to \mathbb { R } ^ { d } } \end{array}$ is jointly continuous in $( x , \zeta )$ and $\sigma = \lceil \sigma ^ { i j } \rceil : \mathbb { R } ^ { d }  \mathbb { R } ^ { d \times d }$ is locally Lipschitz continuous in x. In particular, for some constant $C _ { R } > 0$ depending on $R > 0$ , we have

$$\| \sigma (x _ {1}) - \sigma (x _ {2}) \| ^ {2} \leq C _ {R} | x _ {1} - x _ {2} | ^ {2}$$

for all $x _ { 1 } , x _ { 2 } \in \mathcal { B } _ { R }$ , where $\| \boldsymbol { \sigma } \| : = \sqrt { \mathrm { T r } ( \boldsymbol { \sigma } \boldsymbol { \sigma } ^ { \mathsf { T } } ) }$ .

(D2) Boundedness: The functions b and σ are uniformly bounded, i.e.,

$$\sup _ {\zeta \in \prod_ {k = 1} ^ {N} \mathbb {U} ^ {k}} | b (x, \zeta) | + \| \sigma (x) \| ^ {2} \leq C \quad \forall x \in \mathbb {R} ^ {d},$$

for some constant $C > 0$ .
