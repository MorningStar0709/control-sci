# 2.3 Reference dynamics stability proof

The reference dynamics equations are recalled from [1]

$$\overline {{\mathbf {x}}} (k + 1) = \mathbf {A} _ {R} \overline {{\mathbf {x}}} (k) + \mathbf {B} _ {R} \overline {{\mathbf {r}}} (k),
\begin{array}{l} \overline {{{\mathbf {y}}}} (k) = \mathbf {C} _ {R} \overline {{{\mathbf {x}}}} (k), \\ \mathbf {A} = \mathbf {A} - \mathbf {B}, \mathbf {K}. \end{array} \tag {6}
\mathbf {A} _ {R} = \mathbf {A} _ {c} - \mathbf {B} _ {c} \mathbf {K} _ {R},\mathbf {B} _ {R} = \mathbf {B} _ {c} \mathbf {N} _ {R}, \quad \mathbf {C} _ {R} = \mathbf {C} _ {c}.$$

The reference trajectory $\overline { { \mathbf { r } } } ( k )$ is obtained by exploiting a static-state feedback, with as nominal control input $\overline { { \mathbf { u } } } ( k ) = - \mathbf { K } _ { R } \overline { { \mathbf { x } } } ( k ) + \mathbf { N } _ { R } \overline { { \mathbf { r } } } ( k )$ , where ${ \bf K } _ { { \scriptscriptstyle R } } =$ $k _ { \scriptscriptstyle R }$ and $\mathbf { N } _ { R } = n _ { R }$ are the reference generator gains.

The reference dynamics internal and BIBO stability can be studied by exploiting (6) into frequency domain:

$$\overline {{{\mathbf {X}}}} (z) = \underbrace {z [ z \mathbb {I} - \mathbf {A} _ {R} ] ^ {- 1}} _ {\overline {{{\mathbf {X}}}} _ {z i} (z)} \mathbf {X} (0) + \underbrace {\mathbf {B} _ {R} [ z \mathbb {I} - \mathbf {A} _ {R} ] ^ {- 1}} _ {\overline {{{\mathbf {X}}}} _ {z s} (z)} \overline {{{\mathbf {R}}}} (z) \tag {7}$$

where $\overline { { \mathbf { X } } } _ { z i } ( z )$ denotes the zero-input response (hence when $\mathbf { R } ( z ) = 0 ) , \ : \overline { { \mathbf { X } } } _ { z s } ( z )$ the zero state response when $\mathbf { X } ( 0 ) = 0$ , R the reference trajectory input r translated in frequency domain. The stability depends on the term $[ \mathcal { z } \mathbb { I } - \mathbf { A } _ { \scriptscriptstyle R } ] ^ { - 1 }$ . Indeed, by applying the pole-placement technique, the continuoustime eigenvalue $\mathbf { A } _ { R } = a _ { R }$ is user-defined to lie in the right-half plane, to guarantee stability in frequencydomain.
