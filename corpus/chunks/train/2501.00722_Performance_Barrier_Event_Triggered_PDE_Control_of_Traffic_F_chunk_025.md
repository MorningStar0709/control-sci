# C. Regular Self-triggered Control (R-STC)

In this subsection, we present the R-STC strategy for the system (13)-(21). Since this method can be derived via the P-STC design in Section IV-C, we will only present its structure here to prevent redundancy.

Let $\bar { I } ^ { r } ~ = ~ \{ \check { t } _ { 0 } ^ { r } , \check { t } _ { 1 } ^ { r } , \check { t } _ { 2 } ^ { r } , \dots \}$ denote the sequence of eventtimes associated with R-STC. Let the event-trigger parameters $\theta , \eta , \theta _ { m } , \kappa _ { 1 } , \kappa _ { 2 } , \kappa _ { 3 } > 0$ be selected as outlined in Assumption 2. The proposed R-STC strategy consists of two parts:

1) An ETC input $\check { U } _ { k } ^ { r }$

$$\check {U} _ {k} ^ {r} := U \left(\check {t} _ {k} ^ {r}\right), \tag {66}$$

for $t \in \left[ \check { t } _ { k } ^ { r } , \check { t } _ { k + 1 } ^ { r } \right)$ , k $\in \mathbb { N }$ where $U ( t )$ is given by (24). Then, the boundary condition (33) becomes

$$\bar {v} (\ell , t) = r _ {1} \check {U} _ {k} ^ {r}. \tag {67}$$

2) A self-trigger determining event-times

$$\check {t} _ {k + 1} ^ {r} = \check {t} _ {k} ^ {r} + G ^ {r} \left(H (\check {t} _ {k} ^ {r}), m ^ {r} \left(\check {t} _ {k} ^ {r}\right)\right), \tag {68}$$

with $\check { t } _ { 0 } ^ { r } \ = \ 0$ and $G ^ { r } ( \cdot , \cdot ) ~ > ~ 0$ is a positively and uniformly lower-bounded function

$$
\begin{array}{l} G ^ {r} (H (t), m ^ {r} (t)) \\ := \max \left\{\tau_ {d}, \frac {1}{\varrho^ {\star} + \eta} \ln \left(\frac {\theta m ^ {r} (t) + \frac {\theta \theta_ {m} H (t)}{\varrho^ {\star} + \eta}}{H (t) + \frac {\theta \theta_ {m} H (t)}{\varrho^ {\star} + \eta}}\right) \right\}. \tag {69} \\ \end{array}
$$

Here, $\tau _ { d }$ is R-CETC minimum dwell-time given by (52)-(54), mr(t) satisfies the dynamics (43) along the solution of (34)-(38) for $t \in \left( \check { t } _ { k } ^ { r } , \check { t } _ { k + 1 } ^ { r } \right) , k \in \mathbb { N }$ . The constant ${ \varrho ^ { \star } > 0 }$ is defined as

$$\varrho^ {\star} := r _ {0} ^ {2} r _ {1} ^ {2} e ^ {\frac {\mu \ell}{(\gamma p ^ {\star} - v ^ {\star})}} \varrho , \tag {70}$$

where

$$\varrho = \frac {4}{r _ {1} ^ {2}} \max \left\{v ^ {\star} \tilde {L} ^ {2 1} e ^ {\frac {\mu \ell}{v ^ {\star}}}, \frac {(\gamma p ^ {\star} - v ^ {\star}) \tilde {L} ^ {2 2}}{r _ {0} ^ {2}} \right\}, \tag {71}$$

with

$$\tilde {L} ^ {2 1} = \int_ {0} ^ {\ell} (L ^ {2 1} (\ell , \xi)) ^ {2} d \xi , \quad \tilde {L} ^ {2 2} = \int_ {0} ^ {\ell} (L ^ {2 2} (\ell , \xi)) ^ {2} d \xi . \tag {72}$$
