# C. Small-signal State-space Switched System Model

Combining the mathematical models derived in (1)-(9) yields the complete nonlinear model of the GCIs under GFL and GFM switching control. By applying small-signal linearization around a steady-state operating point, the full-order small-signal state-space model is obtained as

$$\Delta \dot {\boldsymbol {x}} = \boldsymbol {A} \Delta \boldsymbol {x} + \boldsymbol {B} \Delta \boldsymbol {u} \tag {10}$$

where $\Delta$ denotes the small-signal perturbation; $\pmb { x } \in \mathbb { R } ^ { n }$ represents the system state vector with respect to time; $\pmb { u } \in \mathbb { R } ^ { m }$ signifies the control input vector; $A \in \bar { \mathbb { R } } ^ { n \times n }$ and $B \in \mathbb { R } ^ { n \times m }$ correspond to the system state matrix and input matrix, respectively; n and m indicate the number of state variables and input variables, respectively.

As introduced in literature [27], [28], a typical switched system contains a collection of continuous-time subsystems and associated switching laws that orchestrates the transitions among these subsystems. The dynamic evolution process of the switched system is determined by the continuous dynamics of active subsystems and the discrete switching events.

Specifically, the GFL subsystem is formulated as a 12thorder state-space model with $\pmb { x } _ { 1 } = [ \zeta , \delta , \gamma _ { d } , \gamma _ { q } , \xi _ { d } , \xi _ { q } , i _ { d } , i _ { q } ,$ $i _ { L d } , i _ { L q } , v _ { d } , \overline { { v _ { q } } } ] ^ { \mathrm { T } }$ and $\pmb { u } _ { 1 } ~ = ~ \left[ P _ { \mathrm { r e f } } , Q _ { \mathrm { r e f } } \right] ^ { \mathbf { 1 } }$ . The GFM subsystem is characterized by a 13th-order state-space model with $\mathbf { x } _ { 2 } = [ \delta , \omega , E , \gamma _ { d } , \gamma _ { q } , \xi _ { \scriptscriptstyle d } , \xi _ { q } , i _ { d } , i _ { q } , i _ { L d } , i _ { L q } , \bar { v } _ { d } , v _ { q } ] ^ { \mathrm { T } }$ and $\pmb { u } _ { 2 } = [ P _ { \mathrm { r e f } } , Q _ { \mathrm { r e f } } , v _ { d \mathrm { r e f } } , v _ { q \mathrm { r e f } } ] ^ { \mathbf { 1 } }$ . The design and analysis of the specific switching laws between the GFL and GFM subsystems will be thoroughly discussed in Section IV.
