$$
\begin{array}{l} h _ {a} = \mathcal {R} _ {o _ {1}} ^ {\top} \left(\mathcal {R} _ {o _ {1}} (M ^ {- 1} + G ^ {\top} M _ {o} ^ {- 1} G) \mathcal {R} _ {o _ {1}} ^ {\top}\right) ^ {\dagger} (\dot {\mathcal {R}} _ {o _ {1}} v - \dot {\mathcal {R}} _ {o _ {1}} G ^ {\top} v _ {o} - \mathcal {R} _ {o _ {1}} \dot {G} ^ {\top} v _ {o} \\ + \mathcal {R} _ {O _ {2}} M ^ {- 1} (u - g - C v) + \mathcal {R} _ {O _ {1}} G ^ {\top} M _ {o} ^ {- 1} (g _ {o} + C _ {o} v _ {o})) \\ = \mathcal {R} _ {o _ {1}} ^ {\top} \left(\mathcal {R} _ {o _ {1}} (M ^ {- 1} + G ^ {\top} M _ {o} ^ {- 1} G) \mathcal {R} _ {o _ {1}} ^ {\top}\right) ^ {\dagger} \mathcal {R} _ {o _ {1}} \bigg (- \dot {G} ^ {\top} v _ {o} + M ^ {- 1} (u - g - C v) \\ \left. + M _ {o} ^ {- 1} (g _ {o} + C _ {o} v _ {o})\right). \\ \end{array}

Denote now for convenience $M _ { G } : = M ^ { - 1 } { + } G ^ { \top } M _ { o } ^ { - 1 } G$ . According to Theorem 3.8 of [178], it holds that $\mathcal { R } _ { { O } _ { 1 } } ^ { \top } \left( \mathcal { R } _ { { O } _ { 1 } } M _ { G } \mathcal { R } _ { { O } _ { 1 } } ^ { \top } \right) ^ { \dagger } \mathcal { R } _ { { O } _ { 1 } } = M _ { G } ^ { - \frac { 1 } { 2 } } \left( \mathcal { R } _ { { O } _ { 1 } } M _ { G } ^ { \frac { 1 } { 2 } } \right) ^ { \dagger } \mathcal { R } _ { { O } _ { 1 } }$ Next, note that $\mathcal { R } _ { O _ { 1 } }$ has linearly independent columns and hence

$$\left(\mathcal {R} _ {o _ {1}} M _ {G} ^ {\frac {1}{2}}\right) ^ {\dagger} = \left(M _ {G} ^ {\frac {1}{2}} \mathcal {R} _ {o _ {1}} ^ {\top} \mathcal {R} _ {o _ {1}} M _ {G} ^ {\frac {1}{2}}\right) ^ {- 1} M _ {G} ^ {\frac {1}{2}} \mathcal {R} _ {o _ {1}} ^ {\top} = M _ {G} ^ {- \frac {1}{2}} \left(\mathcal {R} _ {o _ {1}} ^ {\top} \mathcal {R} _ {o _ {1}}\right) ^ {- 1} \mathcal {R} _ {o _ {1}} ^ {\top},$$
