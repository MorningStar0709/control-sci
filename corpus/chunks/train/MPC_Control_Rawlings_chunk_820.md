# A.3 Range and Nullspace of Matrices

Suppose $A \in \mathbb { R } ^ { m \times n }$ . Then R(A), the range of A, is the set $\{ A x \mid x \in$ Rn}; R(A) is a subspace of Rm and its dimension, i.e., the number of linearly independent vectors that span $\mathcal { R } ( A )$ , is the rank of A. For example, if A is the column vector $\left[ { } _ { 1 } ^ { 1 } \right]$ , then $\mathcal { R } ( A )$ is the subspace spanned by the vector $\left\lceil { } _ { 1 } ^ { 1 } \right\rceil$ and the rank of A is 1. The nullspace ${ \mathcal { N } } ( A )$ is the set of vectors in ${ \bar { \mathbb { R } } } ^ { n }$ that are mapped to zero by A so that ${ \mathcal { N } } ( A ) =$ $\{ x \mid A x \mid = 0 \}$ . The nullspace ${ \mathcal { N } } ( A )$ is a subspace of $\mathbb { R } ^ { n }$ . For the example above, N (A) is the subspace spanned by the vector $\left\lceil \begin{array} { l } { 1 } \\ { - 1 } \end{array} \right\rceil$ . It is an important fact that R(A0) ⊕ $\mathcal { N } ( A ) \ = \ \mathbb { R } ^ { n }$ or, equivalently, that ${ \mathcal { N } } ( A ) \ = \ ( { \mathcal { R } } ( A ^ { \prime } ) ) ^ { \perp }$ where $A ^ { \prime } \in \mathbb { R } ^ { n \times m }$ is the transpose of A and $S ^ { \perp }$ denotes the orthogonal complement of any subspace S; a consequence is that the sum of the dimensions $\mathcal { R } ( A )$ and ${ \mathcal { N } } ( A )$ is n. If A is square and invertible, then $n = m$ and the dimension of $\mathcal { R } ( A )$ is n so that the dimension of ${ \mathcal { N } } ( A )$ is 0, i.e., the nullspace contains only the zero vector, ${ \mathcal { N } } ( A ) = \{ 0 \}$ .
