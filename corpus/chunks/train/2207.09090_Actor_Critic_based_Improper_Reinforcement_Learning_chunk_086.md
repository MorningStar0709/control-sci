# J. Proofs for (Natural) Actor-critic based improper learning

We will begin with some useful lemmas.

Lemma J.1. For any $\theta , \theta \in \mathbb { R } ^ { M }$ , we have $\| \psi _ { \boldsymbol { \theta } } ( m ) - \psi _ { \boldsymbol { \theta } ^ { \prime } } ( m ) \| _ { 2 } \leqslant \| \boldsymbol { \theta } - \boldsymbol { \theta } ^ { \prime } \| _ { 2 } .$

Proof. Recall, $\psi _ { \theta } ( m ) : = \nabla _ { \theta }$ log πθ(m). Fix $m ^ { \prime } \in [ M ]$ ,

$$
\begin{array}{l} \frac {\partial \log \pi_ {\theta} (m)}{\partial \theta_ {m ^ {\prime}}} = \frac {\partial \log \left(\frac {e ^ {\theta_ {m}}}{\sum_ {j = 1} ^ {M} e ^ {\theta_ {j}}}\right)}{\partial \theta_ {m ^ {\prime}}} \\ = \frac {\partial}{\partial \theta_ {m ^ {\prime}}} \left(\theta_ {m} - \log \left(\sum_ {j = 1} ^ {M} e ^ {\theta_ {j}}\right)\right) \\ = \mathbb {1} \{m ^ {\prime} = m \} - \frac {e ^ {\theta_ {m ^ {\prime}}}}{\sum_ {j = 1} ^ {M} e ^ {\theta_ {j}}} \\ = \mathbb {1} \{m ^ {\prime} = m \} - \pi_ {\theta} (m ^ {\prime}). \\ \end{array}

\begin{array}{l} \| \psi_ {\theta} (m) - \psi_ {\theta^ {\prime}} (m) \| _ {2} \leqslant \| \theta - \theta^ {\prime} \| _ {2} = \| \nabla_ {\theta} \log \pi_ {\theta} (m) - \nabla_ {\theta} \log \pi_ {\theta^ {\prime}} (m) \| _ {2} \\ = \left\| \pi_ {\theta} (.) - \pi_ {\theta^ {\prime}} (.) \right\| _ {2} \\ \leqslant^ {(*)} \| \theta - \theta^ {\prime} \| _ {2}. \\ \end{array}
$$

Here (\*) follows from the fact that the softmax function is 1-Lipschitz (Gao & Pavel, 2017).

Lemma J.2. For all $m \in [ M ]$ and $\theta \in \mathbb { R } ^ { M } , \| \psi _ { \theta } ( m ) \| _ { 2 } \leqslant \sqrt { 2 } .$

Proof. Proof follows by noticing that $\lVert \psi _ { \boldsymbol { \theta } } ( m ) \rVert _ { 2 } = \lVert \nabla _ { \boldsymbol { \theta } } \log \pi _ { \boldsymbol { \theta } } ( m ) \rVert _ { 2 } \leqslant \sqrt { 2 } .$ , where the last inequality follows because the 2-norm of a probability vector is bounded by 1. □

Lemma J.3. For all $\begin{array} { r } { \theta , \theta ^ { \prime } \in \mathbb { R } ^ { M } , \| \pi _ { \theta } ( . ) - \pi _ { \theta ^ { \prime } } ( . ) \| _ { T V } \leqslant \frac { \sqrt { M } } { 2 } \| \theta - \theta ^ { \prime } \| _ { 2 } . } \end{array}$

Proof.

$$
\begin{array}{l} \left\| \pi_ {\theta} (.) - \pi_ {\theta^ {\prime}} (.) \right\| _ {T V} = \frac {1}{2} \left\| \pi_ {\theta} (.) - \pi_ {\theta^ {\prime}} (.) \right\| _ {1} \\ \leqslant \frac {\sqrt {M}}{2} \left\| \pi_ {\theta} (.) - \pi_ {\theta^ {\prime}} (.) \right\| _ {2}. \\ \end{array}
$$
