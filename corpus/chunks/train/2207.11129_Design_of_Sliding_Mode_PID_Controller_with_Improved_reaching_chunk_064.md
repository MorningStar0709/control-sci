# 2.7.2. NONLINEAR EXAMPLE

Van der pol’s equation [75] is used as a test problem in the last example, for simulation and to verify the proposed algorithm. Van-der-Pol (VDPO) oscillator is used for representing nonlinear deterministic systems in delta domain. VDPO oscillator is an oscillator with non-linear damping and it evolves for the control of the position. The dynamics of non-linear system is

$$\dot {x} _ {1} = x _ {2}\dot {x} _ {2} = - 2 x _ {1} + 3 \left(1 - x _ {1} ^ {2}\right) x _ {2} + u \tag {2.47}y = x _ {1}$$
