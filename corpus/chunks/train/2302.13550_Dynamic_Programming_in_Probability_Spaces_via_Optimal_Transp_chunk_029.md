6. Proof of Theorem 4.1 and Corollary 4.2. For the proof Theorem 4.1 and Corollary 4.2, we need a few preliminary lemmata. For ease of notation, let $X : = X _ { 1 } \times . . . \times X _ { k } , Y : = Y _ { 1 } \times . . . \times Y _ { h }$ , and $Z : = Z _ { 1 } \times \ldots \times Z _ { k }$ . To start, we introduce a variation of (2.2), in which only the first k marginals $\mu _ { i } \in \mathcal P ( X _ { i } )$ are fixed. Namely,

$$\mathcal {J} [ c ] (\mu_ {1} \ldots , \mu_ {k}) := \inf _ {\left(\operatorname{proj} _ {X} ^ {X \times Y}\right) _ {\#} \boldsymbol {\gamma} \in \Gamma (\mu_ {1}, \ldots , \mu_ {k})} \int_ {X \times Y} c \mathrm{d} \boldsymbol {\gamma},$$

where $c : X \times Y \to { \bar { \mathbb { R } } } _ { \geq 0 }$ is the transportation cost. When $c \in X \to { \bar { \mathbb { R } } } _ { \geq 0 } ( { \mathrm { i . e . } }$ ., there are no free marginals), we conveniently write $\mathcal { I } [ c ] ( \mu _ { 1 } \dots , \mu _ { k } ) = K [ c ] ( \mu _ { 1 } \dots , \mu _ { k } )$ . Furthermore, given a collection of maps $\{ l _ { k } : X _ { k } \to Z _ { k } \} _ { k = i } ^ { j } ,$ we denote by $l : = l _ { i } \times . . . \times l _ { j }$ the map $X _ { i } \times . . . \times X _ { j }  Z _ { i } \times . . . \times Z _ { j }$ defined pointwise as $( x _ { i } , \dots , x _ { j } ) \mapsto ( l _ { i } ( x _ { i } ) , \dots , l _ { j } ( x _ { j } ) )$ . Given the probability measures $\{ \mu _ { i } \in \mathcal P ( X _ { i } ) \} _ { i = 1 } ^ { k } , \mu : = ( \mu _ { 1 } , \dots , \mu _ { k } )$ , we conveniently write $l _ { \# } \mu : = \left( l _ { 1 \# } \mu _ { 1 } , \ldots , l _ { k \# } \mu _ { k } \right)$ . A measure-valued map $X \ni x \mapsto \mu \in { \mathcal { P } } ( X )$ is Borel if and only if, for any Borel set $B \subseteq X$ , the map $x \mapsto \mu ( B )$ is Borel.
