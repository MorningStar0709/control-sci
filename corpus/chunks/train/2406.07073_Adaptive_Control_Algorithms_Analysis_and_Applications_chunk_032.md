The above problem can be reformulated as nonlinear stochastic control with incomplete information. The unknown parameters are considered as auxiliary states (therefore the linear models become nonlinear: $\dot { x } = a x \Longrightarrow \dot { x } _ { 1 } = x _ { 1 } x _ { 2 } , \dot { x } _ { 2 } = v$ where v is a stochastic process driving the parameter variations). Unfortunately, the resulting solutions (dual control) are extremely complicated and cannot be implemented in practice (except for very simple cases). Adaptive control techniques can be viewed as approximation for certain classes of nonlinear stochastic control problems associated with the control of processes with unknown and time-varying parameters.
