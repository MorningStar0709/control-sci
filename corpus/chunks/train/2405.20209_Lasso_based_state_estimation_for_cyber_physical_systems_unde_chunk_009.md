$$
\left( \begin{array}{c c} \mathcal {O} & I _ {\mathcal {S}} \end{array} \right) ^ {\top} \left( \begin{array}{c c} \mathcal {O} & I _ {\mathcal {S}} \end{array} \right) \binom{x ^ {\star} - \widetilde {x}}{a _ {\mathcal {S}} ^ {\star} - \widetilde {a} _ {\mathcal {S}}} + \lambda \binom{0}{\operatorname{sign} (a _ {\mathcal {S}} ^ {\star})} = 0 \tag {9}

I _ {\bar {S}} ^ {\top} \left( \begin{array}{c c} \mathcal {O} & I _ {S} \end{array} \right) \binom{x ^ {\star} - \widetilde {x}}{a _ {S} ^ {\star} - \widetilde {a} _ {S}} + \lambda \zeta_ {\bar {S}} = 0. \tag {10}
$$

We notice that

$$
\left( \begin{array}{c c} \mathcal {O} & I _ {\mathcal {S}} \end{array} \right) ^ {\top} \left( \begin{array}{c c} \mathcal {O} & I _ {\mathcal {S}} \end{array} \right) = \left( \begin{array}{c c} \mathcal {O} ^ {\top} \mathcal {O} & \mathcal {O} _ {\mathcal {S}} ^ {\top} \\ \mathcal {O} _ {\mathcal {S}} & I _ {h} \end{array} \right) \tag {11}
$$

where $I _ { h } \in \{ 0 , 1 \} ^ { h , h }$ is the identity matrix of dimension h. Then, from (9) and (11), we compute

$$
\binom {x ^ {\star} - \widetilde {x}} {a _ {\mathcal {S}} ^ {\star} - \widetilde {a} _ {\mathcal {S}}} = - \lambda \left( \begin{array}{c c} \mathcal {O} ^ {\top} \mathcal {O} & \mathcal {O} _ {\mathcal {S}} ^ {\top} \\ \mathcal {O} _ {\mathcal {S}} & I _ {h} \end{array} \right) ^ {- 1} \binom {0} {\operatorname{sign} (a _ {\mathcal {S}} ^ {\star})}. \tag {12}
$$

The inverse matrix exists because the rows of $\begin{array} { r l } { \big ( \mathcal { O } } & { { } I _ { \mathcal { S } } \big ) ^ { \top } \ \in \ \mathbb { R } ^ { n + h , p \tau } , \ n + } \end{array}$ $h < p \tau$ , are linearly independent by assumption. Moreover, from (12), the difference $a _ { S } ^ { \star } - \widetilde { a } _ { S }$ is proportional to $\lambda ;$ therefore, if λ is sufficiently small, $\mathrm { s i g n } ( a _ { S } ^ { \star } ) = \mathrm { s i g n } ( \tilde { a } _ { S } )$ . We remark that, in the noise-free case, we can design $\lambda$ eas arbitrarily small without loss of generality.

By replacing (12) in (10), we obtain;

$$
\zeta_ {\bar {S}} = I _ {\bar {S}} ^ {\top} \left( \begin{array}{c c} \mathcal {O} & I _ {S} \end{array} \right) \left( \begin{array}{c c} \mathcal {O} ^ {\top} \mathcal {O} & \mathcal {O} _ {S} ^ {\top} \\ \mathcal {O} _ {S} & I _ {h} \end{array} \right) ^ {- 1} \binom {0} {\operatorname{sign} (\widetilde {a} _ {S})}. \tag {13}
$$
