Suppose the initial state x satisfies $| x | \ \leq \ \delta$ where δ is small enough to ensure that $\delta \mathcal { B } \subset \mathbb { X } _ { f }$ . Then $V _ { N } ^ { 0 } ( x ) \leq \alpha _ { 2 } ( | x | )$ and, since $\{ V _ { N } ^ { 0 } ( x ( i ; x ) ) \}$ is nonincreasing, $V _ { N } ^ { 0 } ( x ( i ; x ) ) \leq \alpha _ { 2 } ( | x | )$ for all $i \in \mathbb { I } _ { \geq 0 }$ . for all $x \in \mathcal { X } _ { N }$ , all $i \in \mathbb { I } _ { \geq 0 }$ .
