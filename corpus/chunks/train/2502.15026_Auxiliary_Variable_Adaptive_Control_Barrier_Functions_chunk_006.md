# II. PRELIMINARIES

Consider an affine control system expressed as

$$\dot {\boldsymbol {x}} = f (\boldsymbol {x}) + g (\boldsymbol {x}) \boldsymbol {u}, \tag {1}$$

where $\pmb { x } \in \mathbb { R } ^ { n } , f : \mathbb { R } ^ { n }  \mathbb { R } ^ { n }$ and $g : \mathbb { R } ^ { n }  \mathbb { R } ^ { n \times q }$ are locally Lipschitz, and $\pmb { u } \in \mathcal { U } \subset \mathbb { R } ^ { q }$ denotes the control constraint set, which is defined as

$$\mathcal {U} := \{\boldsymbol {u} \in \mathbb {R} ^ {q}: \boldsymbol {u} _ {\min} \leq \boldsymbol {u} \leq \boldsymbol {u} _ {\max} \}, \tag {2}$$

with ${ \pmb u } _ { m i n } , { \pmb u } _ { m a x } \in \mathbb { R } ^ { q }$ (the vector inequalities are interpreted componentwise).

Definition 1 (Class κ function [33]). A continuous function α : $\ \cdot \ [ 0 , a ) \to [ 0 , + \infty ] , a > 0$ is called a class κ function if it is strictly increasing and $\alpha ( 0 ) = 0$ .

Definition 2. A set $\mathcal { C } \subset \mathbb { R } ^ { n }$ is forward invariant for system (1) if its solutions for some $\mathbf { \boldsymbol { u } } \in \mathcal { U }$ starting from any $\pmb { x } ( 0 ) \in \mathcal { C }$ satisfy $\pmb { x } ( t ) \in \mathcal { C } , \forall t \geq 0$ .

Definition 3. The relative degree of a differentiable function $b : \mathbb { R } ^ { n }  \mathbb { R }$ is the minimum number of times we need to differentiate it along dynamics (1) until every component of u explicitly shows.

The relative degree defined above can also be referred to as the high-order relative degree, which does not include the case of mixed relative degree (discussed in [16]).
