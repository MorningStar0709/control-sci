Equation (2.48) is a single linear, second-order ODE modeling equation for the dual-disk mechanical system. In the case of equal disk inertias it represents the same system dynamics as Eqs. (2.47a) and (2.47b). However, Eq. (2.48) uses the relative angular displacement $\Delta \theta$ as the dynamic variable and hence its solution only provides information about the relative angle between the two disks. While the use of the relative angular twist Δ?? as the single dynamic variable may at first seem restrictive, it can be used to compute the system’s important performance measures. For example, consider the net input power: the power input to a mechanical system is the product of the input force (or torque) and velocity (or angular velocity). In this case, input torque $T _ { \mathrm { i n } } ( t )$ is defined as positive when it is in the same direction as positive rotation of the cylinder disk $\theta _ { 2 }$ and negative rotation of the piston disk $\theta _ { 1 }$ (see Fig. 2.26). Therefore, the net input power is

$$P _ {\text { in }} = T _ {\text { in }} (t) \dot {\theta} _ {2} - T _ {\text { in }} (t) \dot {\theta} _ {1} = T _ {\text { in }} (t) \Delta \dot {\theta} \tag {2.49}$$

Equation (2.49) shows that the net input power can be computed from the relative angular velocity between the two disks, which can be determined from the solution of the single modeling equation (2.48).
