Fig. 1. Control scheme of GFM-RES with virtual synchronous generator.

The power synchronous loop (PSL) and Q-V loop enable the converter to provide basic frequency support and voltage regulation for the system. During the transient period, the current limiter outputs the preset reference $\overline { { I } } _ { \mathrm { F M , d q } } ^ { \mathrm { s a } }$ to prevent overcurrent issues when $I _ { \mathrm { F M , d q } } ^ { \mathrm { r e f } }$ exceeds the threshold. At the same time, the current saturation flag steps from 0 to 1, which freezes the integrator in the inner voltage loop, preventing abnormal accumulation.
