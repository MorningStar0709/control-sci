# A. Numerical Results with Point Agents

In this section, we present the numerical simulation examples to verify the performance of our method using four agents with single integrator dynamics $\dot { \mathbf { x } } _ { i } = \mathbf { u } _ { i }$ for agent i. We take the CLF as $V ( \mathbf { \bar { x } } _ { i } ) = | | \mathbf { \bar { x } } _ { i } - \mathbf { x } _ { i } ^ { \mathrm { g o a l } } | | ^ { 2 }$ , which drives the agent towards its goal, and CBF as $h _ { i j } ( \mathbf { x } ) = | | \mathbf { x } _ { i } - \mathbf { x } _ { j } | | ^ { 2 } -$ $( r _ { i } + r _ { j } ) ^ { 2 }$ , which defines the pairwise agents’ safety. We compare our method with the CLF-CBF method Eq. (6) and the CLF-CBF with deadlock resolution contextualized from [14] with the same parameters setting. The initial positions for the four agents are $( - 1 0 , 0 ) , ( 0 , 1 0 ) , ( 1 0 , 0 ) , ( 0 , - 1 0 )$ .
