$$
\begin{array}{l} \dot {H} _ {\mathrm{smc}} = - \frac {1}{4} \left\| \Lambda \frac {\partial \phi}{\partial \eta} ^ {- \top} \eta + 2 \binom{0}{\dot {\sigma} _ {\mathrm{rm}}} \right\| _ {\Lambda^ {- 1}} ^ {2} - \frac {1}{4} \left\| \frac {\partial \phi}{\partial \eta} ^ {- \top} \eta \right\| _ {\Lambda} ^ {2} \\ \prec 0, \\ \end{array}
$$

in the state space of the equivalent system where $\sigma _ { \mathrm { s m } } = 0$ . Moreover, it holds in the sliding mode $\sigma = 0$ that

$$\dot {\sigma} = \Lambda (q, \eta) \nabla_ {\sigma} U (\sigma) = 0.$$

Then the time derivative of $H _ { \mathrm { s m c } }$ is calculated as

$$\dot {H} _ {\mathrm{smc}} = - \frac {1}{2} \left\| \frac {\partial \phi}{\partial \eta} ^ {- \top} \eta \right\| _ {\Lambda} ^ {2} \prec 0$$

on the sliding surface $\sigma = 0$ . Thus, the time derivative of $H _ { \mathrm { s m c } }$ is negative in reaching mode, sub-sliding mode, and sliding mode. Therefore, the origin of the closed-loop system is asymptotically stable with the Lyapunov function $H _ { \mathrm { s m c } }$ for all cases. This completes the proof. □

This theorem shows that the proposed passivity-based sliding mode controller achieves sliding mode control and Lyapunov stability simultaneously. It also realizes sliding mode control with arbitrary smoothness (arbitrary convergence speed), since a parameterization of stabilizing controllers consisting of both standard passivity-based controllers and sliding mode ones is obtained. In addition to that, we can freely adjust the trade-off between robustness against external disturbances and chattering vibration because the input can be changed from discontinuous inputs to continuous ones as depicted in Fig 1. The reason for these advantages is that two Lyapunov functions $H _ { \mathrm { s m c } } ( \bar { q } , \eta )$ and $U ( \sigma )$ are used simultaneously to ensure stability. The Hamiltonian function $H _ { \mathrm { s m c } } ( \boldsymbol { q } , \boldsymbol { \eta } )$ guarantees Lyapunov stability of the whole system, while $U ( \sigma )$ ensures finite time convergence of the sliding variable σ. Examples of the potential function $U ( \sigma )$ satisfying Assumption (2) are

$$U (\sigma) = k \| \sigma \| _ {s} ^ {r}, k > 0, 1 \leq r < 2, 1 \leq s, \tag {23}U (\sigma) = \alpha \| \sigma \| _ {1} + \frac {\beta}{2} \| \sigma \| ^ {2}, \alpha > 0, \beta > 0.$$
