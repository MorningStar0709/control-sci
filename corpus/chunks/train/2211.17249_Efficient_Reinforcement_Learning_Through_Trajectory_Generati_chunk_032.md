# 7.2. Policy gradient algorithm

Algorithm 2 Policy Gradient with trajectory generation   
Require: The length T of trajectory, the learning rate $\alpha$ , total number of episode I

Policy Gradient with Data generation: Initialisation : Initial weights $\theta$ for control network

1: for episode = 1 to I do

2: Generate a batch of Q trajectories $[\tau_{1}, \cdots, \tau_{Q}] = \text{TrajectoryGen}(\mathcal{H}, \theta, \mathcal{D}, Q)$ 3: Compute the gradient $\nabla J(\theta) = \frac{1}{Q} \sum_{i=1}^{Q} c(\tau_{i}) \sum_{t=1}^{T} \nabla_{\theta} \log \pi_{\theta} (\tilde{\boldsymbol{u}}_{i}(t) | \tilde{\boldsymbol{x}}_{i}(t))$ 4: Update weights in the neural network by gradient descent: $\theta \leftarrow \theta - \alpha \nabla J(\theta)$ 5: end for
