# 4. SMP for affine rough controlled stochastic systems

In this section, we introduce rough stochastic control problems and establish a Stochastic Maximum Principle (SMP) for rough control problems when the rough driver is affine in the state variable. Using an affine rough Doss-Sussmann transformation we show that the rough SMP can be reduced to a SMP for standard stochastic control problems.

Recall that ${ \mathcal { C } } _ { g } ^ { 0 , \alpha }$ is the geometric rough path space, and let $B ( \mathcal { C } _ { g } ^ { 0 , \alpha } )$ be its Borel σ-algebra. Here we always assume $\pmb { \eta } = ( \eta , \eta ^ { ( 2 ) } ) \in \mathcal { C } _ { g } ^ { 0 , \alpha }$ is a geometric rough path. We also assume that controls take values in a convex subset V of a finite-dimensional space with norm | · |. The set of admissible controls is denoted by

$$\mathcal {A} := \{u: [ 0, T ] \times \Omega \rightarrow V, \text { progressively measurable, } \sup _ {t} \mathbb {E} [ | u _ {t} | ^ {p} ] < \infty , \forall p \geq 2 \}. \tag {4.1}$$

Remark 4.1. The integrability of controls guarantees integrability of the coefficients so Proposition 2.6 is applicable. It is possible to relax the integrability by carefully revisiting the proof of [7, Proposition 2.15].

For any admissible control $u \in { \mathcal { A } }$ and initial condition $x _ { 0 } \in \mathbb { R } ^ { d _ { X } }$ we consider the controlled rough $\mathrm { S D E ^ { 3 } }$

$$d X _ {t} ^ {\boldsymbol {\eta}, u} = b _ {t} (X _ {t} ^ {\boldsymbol {\eta}, u}, u _ {t}) d t + \sigma_ {t} (X _ {t} ^ {\boldsymbol {\eta}, u}, u _ {t}) d W _ {t} + (F _ {t} (\boldsymbol {\eta}) X _ {t} ^ {\boldsymbol {\eta}, u} + f _ {t} (\boldsymbol {\eta})) d \boldsymbol {\eta} _ {t}, \quad X _ {0} = x _ {0} \tag {4.2}$$

and introduce the cost functional

$$J (u; \boldsymbol {\eta}) = \mathbb {E} \left[ \int_ {0} ^ {T} h _ {t} (X _ {t} ^ {\boldsymbol {\eta}, u}, u _ {t}) d t + g (X _ {T} ^ {\boldsymbol {\eta}, u}) \right]. \tag {4.3}$$

The corresponding value function is denoted

$$\mathcal {V} (\boldsymbol {\eta}) := \inf _ {u \in \mathcal {A}} J (u; \boldsymbol {\eta}). \tag {4.4}$$
