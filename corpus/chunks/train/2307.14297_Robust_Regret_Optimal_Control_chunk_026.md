# B. Output Feedback Control Design

We first show that the robust $( \gamma _ { d } , \gamma _ { J } )$ -regret feasibility condition can be reduced to a robust $H _ { \infty }$ condition. The bound on the right side of the robust regret condition (20) does not depend on the uncertainty. Following the same arguments as in Section II-D, this bound can be written as $\| \hat { e } \| _ { 2 } ^ { 2 }$ where eˆ is the output of the system (14). Moreover, $\| \hat { e } \| _ { 2 } ^ { 2 }$ is equal to $\| F d \| _ { 2 } ^ { 2 }$ by Lemma 2 where F is a spectral factor that depends on the choice of $( \gamma _ { d } , \gamma _ { J } )$ . Thus the robust $( \gamma _ { d } , \gamma _ { J } ) \mathrm { - r e g r e t }$ in Definition 2 can be written as:

$$J (K, d, \Delta) < \| F d \| _ {2} ^ {2} \quad \forall d \in \ell_ {2}, d \neq 0 \text { and } \forall \Delta \in \Delta .$$

The left side $J ( K , d , \Delta )$ is equal to the closed-loop cost $\| e \| _ { 2 } ^ { 2 } = \| C L ( P , K , \Delta ) d \| _ { 2 } ^ { 2 }$ achieved with controller K and uncertainty $\Delta \in \Delta$ . (Figure 3). The spectral factor F is stable with a stable inverse. Define $\hat { d } = F d$ so that $d = F ^ { - 1 } { \hat { d } } .$ The set of all $d \in \ell _ { 2 }$ maps 1-to-1 on the set of all $\hat { d } \in \ell _ { 2 }$ . Thus we can rewrite the robust regret bound as

$$\left\| C L (P, K, \Delta) F ^ {- 1} \hat {d} \right\| _ {2} ^ {2} < \| \hat {d} \| _ {2} ^ {2} \tag {21}\forall \hat {d} \in \ell_ {2}, \hat {d} \neq 0 \text {and} \forall \Delta \in \boldsymbol {\Delta}.$$

The system $C L ( P , K , \Delta ) F ^ { - 1 }$ corresponds to the closed-loop ${ \mathit { C L } } ( P , K , \Delta )$ with the disturbance channel weighted by $F ^ { - 1 }$ . Thus (21) corresponds to a robust performance condition on $C L ( P , K , \Delta ) F ^ { - 1 }$ that must hold for all uncertainties $\Delta \in \Delta$ . This yields the following main result connecting the robust regret to the robust $H _ { \infty }$ condition.

Theorem 3. A controller K achieves robust $( \gamma _ { d } , \gamma _ { J } )$ -regret relative to the optimal non-causal controller $K ^ { o }$ if and only i $\ C L ( P , K , \Delta ) F ^ { - 1 }$ is well-posed, stable and $\| C L ( P , K , \Delta ) F ^ { - 1 } \| _ { \infty } < 1$ for all $\Delta \in \Delta$ .
