Next, apply Newton’s second law and sum all external forces on mass m with the sign convention as positive to the right

$$+ \rightarrow \sum F = F _ {\mathrm{em}} - k x - b \dot {x} = m \ddot {x}$$

It is common practice to place all “dynamic” or “output” variables and their derivatives on one side of the equal sign and all input variables on the other side. Therefore, rearranging the previous equation, we get

$$m \ddot {x} + b \dot {x} + k x = F _ {\mathrm{em}} \tag {2.21}$$

Equation (2.21) is the mathematical model of the mechanical component of the solenoid actuator. It is known as a mass–spring–damper system because it consists of an inertia, stiffness, and friction element. The mathematical model (2.21) is a linear, second-order ODE. In general, we obtain one second-order ODE for each inertia element of a mechanical system, provided that we use displacements of the inertia elements in the mathematical model. This fact arises from applying F = mẍ to each mass, which results in a second-order ODE. The simple mass–spring–damper system in Fig. 2.9 is a one degree-of-freedom (1-DOF) system, as only one independent coordinate variable (x in this case) is needed to determine the position of the single inertia element. The electromagnetic force $F _ { \mathrm { e m } }$ is the input to the system. We revisit this solenoid example in Chapter 3 and discuss how the electromagnetic force is generated by the current in the coil.
