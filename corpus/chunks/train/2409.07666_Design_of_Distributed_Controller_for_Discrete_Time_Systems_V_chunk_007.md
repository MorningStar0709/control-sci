Lemma $2 \ ( \ I I 5 7 )$ : Consider undirected graph $\mathcal { G } = ( \mathcal { N } , \mathcal { E } )$ with clique index set $\begin{array} { r l r } { \mathcal { Q } _ { \mathcal { G } } } & { { } = } & { \left\{ 1 , \ldots , q \right\} } \end{array}$ . The next assumption on graph structure is required for the block-diagonal factorization of sparse matrices. This assumption means that all the nodes and edges are covered with cliques in $\mathcal { Q } _ { \mathcal { G } }$ and can hold by appropriate choice of $\mathcal { Q } _ { \mathcal { G } }$ for any graph.

Assumption 1: For an undirected communication graph $\mathcal { G }$ and the clique index set $\mathcal { Q } _ { \mathcal { G } } = \{ 1 , \ldots , q \} , \mathcal { Q } _ { \mathcal { G } } ^ { i }$ satisfies

• $\mathcal { Q } _ { \mathcal { G } } ^ { i } \neq \emptyset$ for all $i \in \mathcal N .$ .   
$\bullet \ \mathcal { Q } _ { \mathcal { G } } ^ { \bar { i } } \cap \mathcal { Q } _ { \mathcal { G } } ^ { j } \neq \emptyset \ \Longleftrightarrow \ ( i , j ) \in \mathcal { E } .$

The next lemma shows that for all undirected graphs, every matrix $K \in S$ has a block-diagonal factorization.

Lemma 3 ( [15]): Suppose Assumption 1 holds. Then, the following transformation exists.

$$\mathcal {S} = \left\{E ^ {\top} \tilde {G} E: \tilde {G} = \operatorname{blkdiag} (\dots , \tilde {G} _ {k}, \dots), \right.\tilde {G} _ {k} \in \mathbb {R} ^ {n _ {\mathcal {C} _ {k}} \times n _ {\mathcal {C} _ {k}}}, k \in \mathcal {Q} _ {\mathcal {G}} \}$$

Also, the following lemma plays a significant role in transformations of matrix inequalities:

Lemma 4 (Finsler’s lemma [17]): Let $x \in \mathbb { R } ^ { n } , Q = Q ^ { \top } \in$ $\mathbb { R } ^ { n \times n }$ , and $M \in \mathbb { R } ^ { r \times n }$ such that ran $\mathfrak { s } ( M ) < n$ . The following statements are equivalent:

1) $x ^ { \top } Q x < 0 , \ \forall x \neq 0 \ { \mathrm { s . t . } } \ M x = 0 .$   
2) $M ^ { \bot \top } Q M ^ { \bot } \prec O .$   
3) $\exists \rho \in \mathbb { R } { \mathrm { ~ s . t . ~ } } Q + \rho M ^ { \top } M \prec O .$   
4) $\exists X \in \mathbb { R } ^ { n \times r } { \mathrm { ~ s . t . ~ } } Q + M ^ { \top } X ^ { \top } + X M \prec O .$
