# 3.3 Gamma-convergence when $\varepsilon \to 0 ^ { + }$ and when $\tau _ { 1 } = \tau _ { 2 } \to + \infty$

We focus on computing the limit cases when $\varepsilon \to 0 ^ { + }$ or when $\tau _ { 1 } = \tau _ { 2 } \to + \infty$ of the primal unbalanced functional $\mathcal { F } _ { \varepsilon } ^ { \tau _ { 1 } , \tau _ { 2 } } ( \Gamma )$ . Consider the families

$$\{\mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} \} _ {\varepsilon > 0} = \{\mathcal {F} _ {\varepsilon} ^ {\tau_ {1}, \tau_ {2}} (\cdot) = \mathrm{Tr} [ C (\cdot) ] + \varepsilon \mathcal {S} [ \cdot ] + \tau_ {1} \mathcal {E} [ (\cdot) _ {1} | \rho ] + \tau_ {2} \mathcal {E} [ (\cdot) _ {2} | \sigma ], \varepsilon > 0 \}, \tag {3.12}$$

and

$$\left\{\mathcal {F} _ {\varepsilon} ^ {\tau} \right\} _ {\tau > 0} = \left\{\operatorname{Tr} [ C (\cdot) ] + \varepsilon \mathcal {S} [ \cdot ] + \tau \left(\mathcal {E} [ (\cdot) _ {1} | \rho ] + \mathcal {E} [ (\cdot) _ {2} | \sigma ]\right), \tau > 0 \right\}. \tag {3.13}$$

First, we need to use the following notions.

Definition 3.4 (Γ-convergence, [3]). Let (X, T) be a topological space. Given a family of functionals $\mathcal { F } _ { \alpha } : \mathcal { X }  \mathbb { R }$ we say that it Γ-converges to $\mathcal { F } : \mathcal { X }  \mathbb { R }$ in the topology T if it satisfies

(i) lim inf inequality

For every $x _ { \alpha } \to x$ in T holds

$$\mathcal {F} (x) \leq \liminf _ {\alpha} \mathcal {F} _ {\alpha} (x _ {\alpha}) \tag {3.14}$$

(ii) lim sup inequality

For every $x \in { \mathcal { X } }$ there exists a sequence $x _ { \alpha } \to x$ in T such that

$$\mathcal {F} (x) \geq \limsup _ {\alpha} \mathcal {F} _ {\alpha} (x _ {\alpha}). \tag {3.15}$$

Or equivalently, if $\{ \mathcal { F } _ { \alpha } \}$ α satisfies (3.14), the condition (3.15) is equivalent to

(ii)’ recovery sequence

For every $x \in { \mathcal { X } }$ there exists a sequence $x _ { \alpha } \to x$ in T such that

$$\mathcal {F} (x) = \lim _ {\alpha} \mathcal {F} _ {\alpha} (x _ {\alpha}). \tag {3.16}$$

Definition 3.5 (Equi-coercivity). Let $( { \mathcal { X } } , { \mathcal { T } } )$ be a topological space. Given a family of functionals $\mathcal { F } _ { \alpha } : \mathcal { X }  \mathbb { R }$ we say that it is equi-coercive if for any sequence $\{ x _ { \alpha } \} _ { \alpha }$ such that

$$\sup _ {\alpha} \left| \mathcal {F} _ {\alpha} (x _ {\alpha}) \right| < \infty \tag {3.17}$$

there exists a converging subsequence in topology T.
