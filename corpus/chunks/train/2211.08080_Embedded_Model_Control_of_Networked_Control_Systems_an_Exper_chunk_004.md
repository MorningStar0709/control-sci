# 2.1 Internal model stability proof

The internal model of the EMC unit can be formulated in the form of state equations as follows, [3]

$$
\mathbf {x} (k + 1) = \left[ \begin{array}{c} \mathbf {x} _ {c} \\ \mathbf {x} _ {d} \end{array} \right] (k + 1) = \underbrace {\left[ \begin{array}{c c} \mathbf {A} _ {c c} & \mathbf {H} _ {c c} \\ 0 & \mathbf {A} _ {d} \end{array} \right]} _ {\mathbf {A}} \mathbf {x} (k) +

+ \underbrace {\left[ \begin{array}{l} \mathbf {B} _ {c} \\ \mathbf {B} _ {d} \end{array} \right]} _ {\mathbf {B}} \mathbf {u} (k) + \underbrace {\left[ \begin{array}{l} \mathbf {G} _ {c} \\ \mathbf {G} _ {d} \end{array} \right]} _ {\mathbf {G}} \overline {{\mathbf {w}}} (k), \tag {1}

\mathbf {y} _ {m} (k) = \underbrace {\left[ \begin{array}{c c} \mathbf {C} _ {c c} & \mathbf {C} _ {d} \end{array} \right]} _ {\mathbf {C}} \mathbf {x} (k),
$$

where $\mathbf { x } _ { c }$ and $\mathbf { x } _ { d }$ refer to the controllable (or canonical) and rejector states, respectively.

On the other side, the matrices A, B, C, are related respectively to: (i) the state x, (ii) the control input $\mathbf { u } ,$ and (iii) the estimated output from the internal model ${ \bf y } _ { m }$ . In addition the matrix G is present, for the estimation of the noise vector w. In particular, the matrix A is made by the sub-matrices: $\mathbf { A } _ { c } ,$ related to $\mathbf { x } _ { c } , \mathbf { A } _ { d }$ related to $\mathbf { x } _ { d } ,$ and $\mathbf { H } _ { c } .$ which considers the interactions of $\mathbf { x } _ { d }$ on $\mathbf { x } _ { c }$ . Similarly, we have the sub-matrices $\mathbf { B } _ { c }$ and $\mathbf { B } _ { d }$ for B matrix, $\mathbf { G } _ { c }$ and $\mathbf { G } _ { d }$ for G matrix, $\mathbf { C } _ { c }$ and $\mathbf { C } _ { d }$ for C matrix.

Equation (1) for the internal model can be converted from time to frequency domain, as follows

$$
\begin{array}{l} \mathbf {X} (z) = \underbrace {\{z \mathbb {I} - [ \mathbf {A} - \mathbf {G L C} ] \} ^ {- 1}} _ {\mathbf {X} _ {z i} (z)} \mathbf {X} (0) + \\ + \underbrace {\mathbf {B} \left\{z \mathbb {I} - [ \mathbf {A} - \mathbf {G L C} ] \right\} ^ {- 1}} _ {\mathbf {X} _ {z s, 1} (z)} \mathbf {U} (z) + \tag {2} \\ + \underbrace {\mathbf {G L} \{z \mathbb {I} - [ \mathbf {A} - \mathbf {G L C} ] \} ^ {- 1}} _ {\mathbf {X} _ {z s, 2} (z)} \mathbf {Y} (z) \\ \end{array}
$$
