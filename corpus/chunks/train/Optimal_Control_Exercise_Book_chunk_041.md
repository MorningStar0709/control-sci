# Exercise 2.27 (The Brachistochrone Problem)

Use the no x result to show that extremals for the brachistochrone problem are indeed given by

$$x (\theta) = a + c (\theta - \sin (\theta), \quad y (\theta) = c (1 - \cos (\theta)) \tag {89}$$

where the parameter θ takes values between 0 and 2π and $c > 0$ is constant.
