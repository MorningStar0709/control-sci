# VI. k-HOP PPSO-BASED CONTROLLER

The proposed k-hop PPSO allows each agent $i \in \mathcal V$ to estimate the state of all agents $N _ { i } ^ { i } \in \mathcal { N } _ { i } ^ { k \mathrm { - h o p } }$ . This estimation capability enables the synthesis of a closed-loop control law that leverages local state estimates to accomplish the team’s objective.

Consider the vectorized form of the dynamics in (1), $\mathrm { i . e . , ~ } \dot { { \pmb x } } ( t ) = { \pmb f } ( { \pmb x } ( t ) ) + { \pmb g } ( { \pmb u } ( t ) ) + { \pmb w } ( { \pmb x } , t )$ , where ${ \mathbf { } } x ( t ) =$ $\begin{array} { l l l } { [ x _ { 1 } ( t ) , \dots , x _ { N } ( t ) ] ^ { \top } , } & { { \pmb f } ( { \pmb x } ( t ) ) } & { = } & { [ f _ { 1 } ( x _ { 1 } ) , \dots , f _ { N } ( x _ { N } ) ] ^ { \top } } \end{array}$ , $\begin{array} { r l r } { \pmb { g } ( \pmb { u } ( t ) ) } & { { } } & { = { } \qquad [ g _ { 1 } ( u _ { 1 } ) , \dots , g _ { N } ( u _ { N } ) ] ^ { \top } , \quad \pmb { w } ( \pmb { x } , t ) } \end{array}$ = $[ w _ { 1 } ( { \pmb x } , t ) , \dots , w _ { N } ( { \pmb x } , t ) ] ^ { \top }$ and ${ \pmb u } ( t ) = [ u _ { 1 } ( t ) , \dots , u _ { N } ( t ) ] ^ { \top }$ represents a nonlinear state-feedback control input of the form:

$$\boldsymbol {u} = \boldsymbol {\psi} (\boldsymbol {x}) = \left[ \psi_ {1} (\bar {\boldsymbol {x}} _ {1}, \boldsymbol {x} ^ {1}), \dots , \psi_ {N} (\bar {\boldsymbol {x}} _ {N}, \boldsymbol {x} ^ {N}) \right] ^ {\top}, \tag {22}$$

where, for each $i \in \mathcal { V } , \pmb { x } ^ { i }$ is defined as in (2), and $\bar { \mathbf { x } } _ { i }$ contains the state information of agent i and of all $j \in \mathcal N _ { i }$ .

Since $\mathbf { x } ^ { i }$ is not locally available, the controller in (22) is implemented using ${ \hat { \mathbf { x } } } ^ { i }$ for all $i \in \nu$ , i.e.:

$$\boldsymbol {u} = \boldsymbol {\psi} (\bar {\boldsymbol {x}}, \hat {\boldsymbol {x}}) = \left[ \psi_ {1} (\bar {\boldsymbol {x}} _ {1}, \hat {\boldsymbol {x}} ^ {1}), \dots , \psi_ {N} (\bar {\boldsymbol {x}} _ {N}, \hat {\boldsymbol {x}} ^ {N}) \right] ^ {\top}. \tag {23}$$
