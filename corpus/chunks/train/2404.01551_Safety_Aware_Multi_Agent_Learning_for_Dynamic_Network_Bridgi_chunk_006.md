# C. Distributed autonomous swarm formation for dynamic network bridging

We formulate our problem as a Dec-POMDP, where each agent has restricted sensing capabilities given by a specific communication range, leading to incomplete information about the global state of the system outside such area. However, whenever two entities (agents or targets) are within range, a communication channel becomes available for them to exchange information. This channel enables the possibility for agents to know each other features and execute distributed strategies requiring message exchange to coordinate. At the beginning of every time-step, agents make decentralized decisions, generating target points w, utilizing their local observation to choose the direction of their next movement. To solve the Dec-POMDP that corresponds to our task we used a graph neural network (GNN) approximator that encodes the graph structure of the problem.
