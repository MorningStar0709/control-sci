Definition 1 (Admissible Pair). For an admissible control process $u \in \mathcal { U } .$ , the pair $( x , u )$ , where x satisfies (2) with u as the control, is called an admissible pair for the Problem MFT.

Definition 2 (Solvability and Optimal Pair). Problem MFT is said to be (uniquely) solvable if there exists a (unique) minimizer of $J ( \cdot )$ ; that is, a (unique) admissible control $u ^ { \ast } \in \mathcal { U }$ such that $J ( u ^ { * } ) \leq J ( u )$ for all $u \in \mathcal { U }$ . The corresponding state $x ^ { * }$ , which satisfies (2) under the control $u ^ { * }$ , is called the optimal trajectory, and the pair $( \boldsymbol { x } ^ { * } , \boldsymbol { u } ^ { * } )$ is referred to as the optimal pair for the Problem MFT.

The dependence of the model in Problem MFT on the conditional expectation of the state, i.e., $\mathbb { E } [ x _ { t } | \mathcal { F } _ { t } ^ { 0 } ]$ , classifies it as a Mckean-Vlasov (or equivalently, mean-field type or mean-field) optimal control problem. Similar to other optimal control problems, it can be addressed through three main steps: (i) establishing the well-posedness of the state SDE (2) and the cost functional (1), (ii) proving the solvability of Problem MFT, and (iii) characterizing the corresponding optimal control.
