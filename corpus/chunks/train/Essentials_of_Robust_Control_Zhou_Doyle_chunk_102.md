# $\mathcal { H } _ { \infty }$ Space

$\mathcal { H } _ { \infty }$ is a (closed) subspace of $\mathcal { L } _ { \infty }$ with functions that are analytic and bounded in the open right-half plane. The $\mathcal { H } _ { \infty }$ norm is defined as

$$\| F \| _ {\infty} := \sup _ {\operatorname{Re} (s) > 0} \overline {{\sigma}} [ F (s) ] = \sup _ {\omega \in \mathbb {R}} \overline {{\sigma}} [ F (j \omega) ].$$

The second equality can be regarded as a generalization of the maximum modulus theorem for matrix functions. (See Boyd and Desoer [1985] for a proof.) The real rational subspace of $\mathcal { H } _ { \infty }$ is denoted by $\mathcal { R } \mathcal { H } _ { \infty }$ , which consists of all proper and real rational stable transfer matrices.
