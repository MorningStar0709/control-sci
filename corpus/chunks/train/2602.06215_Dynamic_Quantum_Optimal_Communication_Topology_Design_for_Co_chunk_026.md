To study how $V ^ { ( k ) }$ changes from one iteration to the next, we write

$$
\begin{array}{l} V ^ {(k + 1)} - V ^ {(k)} = \mathcal {L} \left(z ^ {(k + 1)}, f ^ {(k + 1)}, r ^ {(k + 1)}, s ^ {(k + 1)}, \lambda^ {(k + 1)}\right) \\ - \mathcal {L} \left(z ^ {(k)}, f ^ {(k)}, r ^ {(k)}, s ^ {(k)}, \lambda^ {(k)}\right) \\ + \frac {\kappa}{2} \left(\left\| d ^ {(k + 1)} \right\| _ {2} ^ {2} - \left\| d ^ {(k)} \right\| _ {2} ^ {2}\right). \tag {30} \\ \end{array}
$$

For notational convenience, we denote by

$$\mathcal {L} ^ {k} (\lambda) := \mathcal {L} \big (z ^ {(k)}, f ^ {(k)}, r ^ {(k)}, s ^ {(k)}, \lambda \big)$$

the augmented Lagrangian evaluated at the primal variables of iteration k and at a generic dual vector λ. Then we can rewrite (30) as

$$
\begin{array}{l} V ^ {(k + 1)} - V ^ {(k)} = \mathcal {L} ^ {k + 1} (\lambda^ {(k + 1)}) - \mathcal {L} ^ {k} (\lambda^ {(k)}) \\ + \frac {\kappa}{2} \left(\left\| d ^ {(k + 1)} \right\| _ {2} ^ {2} - \left\| d ^ {(k)} \right\| _ {2} ^ {2}\right). \tag {31} \\ \end{array}
$$

We now split the change in the augmented Lagrangian into a dual part and a primal part. We add and subtract

$\mathcal { L } ^ { k + 1 } ( \lambda ^ { ( k ) } )$ inside the first difference:

$$
\begin{array}{l} \mathcal {L} ^ {k + 1} \left(\lambda^ {(k + 1)}\right) - \mathcal {L} ^ {k} \left(\lambda^ {(k)}\right) = \left[ \mathcal {L} ^ {k + 1} \left(\lambda^ {(k + 1)}\right) - \mathcal {L} ^ {k + 1} \left(\lambda^ {(k)}\right) \right] \\ + \left[ \mathcal {L} ^ {k + 1} (\lambda^ {(k)}) - \mathcal {L} ^ {k} (\lambda^ {(k)}) \right]. \tag {32} \\ \end{array}
$$

The first bracket corresponds to the effect of updating the dual variable from $\overset { \bullet } { \lambda } ( k )$ to $\lambda ^ { ( k + 1 ) }$ while the primal variables are fixed at $( z ^ { ( k + 1 ) } , f ^ { ( k + 1 ) } , r ^ { ( k + 1 ) } , s ^ { ( k + 1 ) ^ { \bot } } )$ . The second bracket corresponds to the effect of updating the primal variables from iteration k to iteration $k + 1$ while the dual variable remains equal to $\lambda ^ { ( k ) }$ .

We first consider the dual part. In the augmented Lagrangian in (21), the dual variable appears only in the linear term $\lambda ^ { \top } ( z \textrm { -- } r + s )$ . Therefore, for any fixed $( z , f , r , s )$ and any two dual vectors λ and $\tilde { \lambda } ,$ we have

$$\mathcal {L} (z, f, r, s, \tilde {\lambda}) - \mathcal {L} (z, f, r, s, \lambda) = (\tilde {\lambda} - \lambda) ^ {\top} (z - r + s).$$
