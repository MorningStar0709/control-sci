# 3 General properties of extremals

In the following, if it is not important to specify the control we will denote with X the solution of (2.2) instead of $X ( \cdot ; u )$ .

The solution η to equation (2.5) can be expressed as

$$\eta (t) = X (t) ^ {- 1} \eta_ {0} X (t). \tag {3.1}$$

Indeed, differentiating the previous expression and recalling that

$$\frac {d}{d t} [ X (t) ^ {- 1} ] = - X (t) ^ {- 1} \dot {X} (t) X (t) ^ {- 1},$$

one obtains equation (2.5).

Thus, from (3.1), we can see that the determinant along any solution of (2.5) is constant. For $M \in$ ${ \mathfrak { s l } } _ { 2 } ( \mathbb { R } )$ , we have the formula $2 \operatorname* { d e t } ( M ) = - \mathrm { t r } ( M ^ { 2 } )$ . In what follows, it will be more convenient to use the expression $\mathrm { t r } ( M ^ { 2 } )$ instead of det M .

Notice that from (2.7), it follows that

$$\mathrm{tr} (\eta (T) ^ {2}) = \frac {1}{2} \bigl (\mathrm{tr} X (T) \bigr) ^ {2} - 2.$$

$\mathrm { S o }$ , since $X ( T ) \in \mathrm { S L } _ { 2 } ( \mathbb { R } )$ , then $\operatorname { t r } ( X ( T ) ) ^ { 2 } \geq 4 .$ , at least for large T , and so $\mathrm { t r } ( \eta ( T ) ^ { 2 } ) > 0$ . Moreover, since $\operatorname { t r } ( \eta ^ { 2 } )$ is constant, we can deduce that $\mathrm { t r } ( \eta ( t ) ^ { 2 } ) > 0$ for all $t \in [ 0 , T ]$ . In particular, $\eta ( t )$ must have real eigenvalues for all $t \in [ 0 , T ]$ .
