# 5 EXPERIMENTAL SETUP

For our experiments, we instantiated MAS on the AutoGen platform (Microsoft, 2025) using the popular, open-source, general-purpose Magentic-One configuration (Fourney et al., 2024). By default, it consists of an orchestrator that directs the web surfer, file surfer, video surfer, coder, and executor sub-agents. We implemented CONTROLVALVE as part of the orchestrator and did not modify the sub-agents. Unless otherwise noted, the orchestrator uses OpenAI’s o4-mini, the sub-agents use GPT-4o. For the emailing tasks in the CFH-Hard dataset, we implemented an agent that receives an instruction and an email address, writes an email, and simulates sending it. For AgentDojo, we implemented two custom agents to interface with the Slack environment on the platform.

To reduce the evaluation cost, we limited our experiments to this configuration, but we expect that the results transfer to other configurations. As shown by Triedman et al. (2025), control-flow hijacks transfer across configurations, LLMs, and multi-agent platforms. CONTROLVALVE does not rely on any specific implementation details of the multi-agent system or individual agents.
