# 3.1 Behavior-based identification for predictive control

From the offline noisy data $H _ { \mathrm { d } }$ , we aim to get a new trajectory library $H ^ { * }$ for predictive control

$$H ^ {*} := \operatorname{col} (U _ {\mathrm{P}} ^ {*}, Y _ {\mathrm{P}} ^ {*}, U _ {\mathrm{F}} ^ {*}, Y _ {\mathrm{F}} ^ {*}),$$

where each column of $H ^ { * }$ is a “purified” trajectory of the system. Let $\mathcal { W } _ { L }$ (where $L = T _ { \mathrm { i n i } } + N )$ denotes the space of all possible length-L trajectories of an LTI system, $i . e . ,$

$$
\mathcal {W} _ {L} = \bigg \{\left[ \begin{array}{c} \bar {u} \\ \bar {y} \end{array} \right] | \exists x _ {0} \in \mathbb {R} ^ {n}, (1) \text { holds   with } \bar {x} (0) = x _ {0} \bigg \}.
$$

Ideally, we may consider the following problem

$$H ^ {*} = \underset {H} {\operatorname{argmin}} J _ {\mathrm{id}} (H, H _ {\mathrm{d}}) := \| H - H _ {\mathrm{d}} \| _ {F} ^ {2} \tag {8}$$

$\mathrm { s u b j e c t ~ t o ~ } \mathrm { I m } ( H ) \subseteq { \mathcal { W } } _ { L } .$

If the offline data $H _ { \mathrm { d } }$ is noise-free and satisfies the persistent excitation, the fundamental lemma ensures that Im $( H _ { \mathrm { d } } ) = \mathcal { W } _ { L }$ . In this case, the optimal solution to (8) is $\dot { H } ^ { * } = H _ { \mathrm { d } }$ . In general, the constraint Im $( H ) \subseteq \mathcal { W } _ { L }$ is too abstract and not tractable. For an explicit expression, we have several necessary conditions.

• First, the trajectory matrix H should be of low rank, i.e., we have rank $\Breve { H } ^ { * } \leq m L + n$ .   
• Second, the trajectory matrix H should satisfy linearity, i.e., the future output is a linear function of the past data and future input, and we have

$$Y _ {\mathrm{F}} ^ {*} = K \operatorname{col} (U _ {\mathrm{P}}, Y _ {\mathrm{P}}, U _ {\mathrm{F}}), \tag {9}$$

for some coefficient matrix K.

Third, the trajectory matrix H should satisfy causality, i.e., the coefficient matrix K in (9) should have a block partition $K = \left\lceil K _ { \mathrm { p } } K _ { \mathrm { f } } \right\rceil$ , and $K _ { \mathrm { f } } \in { \mathcal { L } }$ encodes a block-lower triangle structure for causality.   
• Finally, we may also impose a Hankel structure on $H ^ { \ast } \in { \mathcal { H } }$ if each column of $H _ { \mathrm { d } }$ is a one-step shifted trajectory. Representing the trajectory library with a Hankel structure is widely used in both theory [19, 33] and practical applications [27, 29] of DDPC.
