The robust performance condition in Theorem 3 depends on the uncertainty. We next show that this is equivalent to a condition that does not depend on the uncertainty. Consider the feedback interconnection in Figure 4. This corresponds to the closed-loop ${ \mathit { C L } } ( P , K , \Delta )$ with the uncertainty channels “open” and with the disturbance channel weighted by $F ^ { - 1 }$ . The system

from inputs $\left[ \begin{array} { c } { w } \\ { \hat { d } } \end{array} \right]$ to outputs [ ve ] is denoted by:

$$
M := F _ {L} (P, K) \left[ \begin{array}{c c} I _ {n _ {w}} & 0 \\ 0 & F ^ {- 1} \end{array} \right]. \tag {22}
$$

M is constructed from $( P , K , { P } ^ { - 1 } )$ but this dependence is suppressed to simplify the notation. Note that $C L ( P , K , \Delta ) F ^ { - 1 } = F _ { U } ( M , \Delta )$ .

Theorem 4. A controller K achieves robust $( \gamma _ { d } , \gamma _ { J } ) – r e g r e t$ relative to the optimal non-causal controller $K ^ { o }$ if and only if (i) M is stable, and (ii) there exists $D : [ 0 , 2 \pi ] \to ( 0 , \infty )$ such that
