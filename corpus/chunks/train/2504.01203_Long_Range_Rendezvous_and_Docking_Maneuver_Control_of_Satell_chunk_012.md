# A. Cross-Feedback Linear Sliding Surface

Instead of designing independent sliding surfaces for translational and rotational motion, the proposed cross-feedback sliding surface of translational motion consists of translational error and cross-feedback terms of rotational error and vice versa. The translational sliding surface denoted by $\tilde { s } _ { p }$ (11) and the rotational sliding surface denoted by $\tilde { s } _ { r } ~ ( 1 2 )$ is defined based on relative position $\tilde { e } _ { p }$ and orientation error $\tilde { e } _ { r }$ as shown below.

$$\tilde {s} _ {p} = \dot {\tilde {e}} _ {p} + \lambda_ {p} \tilde {e} _ {p} + \mu_ {p} \tilde {e} _ {r} \tag {11}\tilde {s} _ {r} = \dot {\tilde {e}} _ {r} + \lambda_ {r} \tilde {e} _ {r} + \mu_ {r} \tilde {e} _ {p} \tag {12}$$

where $\lambda _ { p } , \lambda _ { r } > 0$ and $\mu _ { p } , \mu _ { r } > 0$ are cross-feedback gains. An exponential reaching law for both translational and rotational motion is defined to ensure that the system error states reaches the sliding surface as, $\dot { \tilde { s } } _ { p } = - k _ { p } \tilde { s } _ { p }$ and $\dot { \tilde { s } } _ { r } = - k _ { r } \tilde { s } _ { i }$ r
