For the hidden number of neurons, let us first bound the $r ( p )$ sequence from (14). Since $r ( p )$ is strictly increasing, then:

$$
\begin{array}{l} r (p) = r \left(2 ^ {\log_ {2} p}\right) \leq r \left(2 ^ {\lceil \log_ {2} p \rceil}\right) \stackrel {(1 4)} {=} \sum_ {i = 1} ^ {\lceil \log_ {2} p \rceil} 2 ^ {i} = 2 ^ {\lceil \log_ {2} p \rceil + 1} - 2 \\ = 2 (2 ^ {\lceil \log_ {2} p \rceil} - 1) <   2 (2 ^ {\log_ {2} p + 1} - 1) = 2 (2 p - 1). \tag {17} \\ \end{array}
$$

Using $| \mathcal { C } _ { i } | \geq 1$ and $\left| \mathcal { C } _ { i } \right| = \bar { m } _ { i } \le m$ in (16), we obtain:

$$
\begin{array}{l} r (l) + \sum_ {i = 1} ^ {l} r (m) + 2 (\zeta (m) - \zeta (1)) = r (l) + l (r (m) + 2 \lceil \log_ {2} m \rceil) \\ \leq 2 l \left(1 + 2 m + \lceil \log_ {2} m \rceil\right) - 2, \\ \end{array}
$$

where the last inequality is given by (17). This proves our statements. □

Note that in the previous theorem, we treat the scalar case. Thus, we provide next the extension to the vector case.

Theorem 2. Any min-max function $\mathcal { M } : \mathbb { X } \to [ \underline { { \mathbf { u } } } , \overline { { \mathbf { u } } } ] ,$ , with $\mathbf { u } , \bar { \mathbf { u } } \in$ $\mathbb { R } ^ { n _ { u } }$ and X a compact set, defined as:

$$\mathcal {M} (x) = \min _ {1 \leq i \leq l} \max _ {j \in \mathscr {C} _ {i}} C _ {j} x + d _ {j},$$

with $\mathcal { C } _ { 1 } , \hdots , \mathcal { C } _ { l } \subseteq \left\{ 1 , \hdots , m \right\}$ , can be exactly represented by a parallel connection of $\cdot _ { n _ { u } }$ HTNNs:

$$
\left[ \begin{array}{c} f ^ {[ 1 ]} (x; \hat {\Theta} ^ {[ 1 ]}) \\ \vdots \\ f ^ {[ n _ {u} ]} (x; \hat {\Theta} ^ {[ n _ {u} ]}) \end{array} \right], \tag {18}
$$

where each HTNN, $f ^ { [ i ] } { } _ { j }$ , has $\zeta _ { i }$ number of layers, $r _ { i }$ number of hidden neurons and wi maximum width, for $i = 1 : n _ { u } .$ . Then, the $n _ { u }$ parallel HTNN has:

$$\zeta = \max _ {i = 1: n _ {u}} \zeta_ {i}, \quad w = \sum_ {i = 1} ^ {n _ {u}} \max (w _ {i}, 2),r = \sum_ {i = 1} ^ {n _ {u}} r _ {i} + 2 (\zeta - \zeta_ {i}).$$
