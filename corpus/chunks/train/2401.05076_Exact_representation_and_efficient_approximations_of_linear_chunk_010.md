# 2.2. First order optimization methods

Since U is a simple polyhedral set, we can consider primal first order methods (FOM) to solve problem (4), since the projection onto U is easy (see [21]). A standard algorithm is the Projected Gradient Descent (PGD) [21]. Providing an initial point $\mathbf { u } _ { 0 } .$ the PGD iteration has the expression:

$$\mathbf {u} _ {k + 1} = \Pi_ {\mathbb {U} ^ {N}} \left(\left(\mathrm{I} _ {v} - \alpha_ {k} \mathbf {H}\right) \mathbf {u} _ {k} - \alpha_ {k} \mathbf {q} (x _ {0})\right), \tag {7}$$

where $\Pi _ { \mathbb { U } ^ { N } }$ denotes the projection operator onto the set $\mathbb { U } ^ { N }$ , $\alpha _ { k }$ is an appropriate stepsize (e.g., $\alpha _ { k } = 1 / \lVert \mathbf { H } \rVert )$ and $\mathrm { I } _ { \nu }$ denotes the identity matrix of dimension ν. Having $\mathbf { H } \succ 0 .$ , PGD converges linearly with a rate depending on the condition number of the matrix H [21]. For faster convergence, one can use the accelerated version of PGD, namely the Accelerated Projected Gradient Descent (APGD) [21]. The improvement in convergence is payed by growing a little the complexity per iteration by adding an extrapolation step:

$$\mathbf {u} _ {k + 1} = \Pi_ {\mathbb {U} ^ {N}} \left(\left(\mathrm{I} _ {v} - \alpha_ {k} \mathbf {H}\right) \mathbf {y} _ {k} - \alpha_ {k} \mathbf {q} \left(x _ {0}\right)\right) \tag {8}\mathbf {y} _ {k + 1} = (1 + \boldsymbol {\beta} _ {k}) \mathbf {u} _ {k + 1} - \boldsymbol {\beta} _ {k} \mathbf {u} _ {k},$$

where ${ \bf u } _ { 0 } = { \bf y } _ { 0 }$ , αk is an appropriate stepsize (usually chosen as in PGD) and the extrapolation parameter $\beta _ { k }$ can be chosen even constant when the objective function is strongly convex (i.e., H ≻ 0 in (4)), see [21] for more details. One can immediately notice that if we choose $\beta _ { k } = 0$ , then we recover PGD algorithm. It is known that APGD converges faster than PGD, i.e., it has a linear rate depending on the square root of the condition number of the matrix H ≻ 0 [21].
