3.1. Stationary Infinite Population Limit. In this section, we introduce the stationary infinite population limit of the game introduced in the preceding section. In the limiting case, for each $t \geq 0$ , we pretend that the mean-field term $\mathsf { e } [ \cdot | \mathbf { x } ( t ) ]$ converges (in some sense) to the deterministic probability measure $\mu$ as $N \to \infty$ . Therefore, under this convergence assumption, given the limiting distribution $\mu .$ which characterizes the collective behavior of all agents in the infinite population limit, a generic agent has the following state dynamics

$$x (t + 1) = f \left(x (t), a (t), z (t), w (t)\right) \tag {4}$$

where

$$z (t) \sim \mu (\cdot) \text { and } w (t) \sim \mathbb {P} _ {\Omega} (\cdot)$$

Here, we can view $z ( t )$ as a noise since its distribution does not depend on the state. Hence, state process $\{ x ( t ) \} _ { t \geq 0 }$ becomes a Markov decision process (MDP) with the following transition probability

$$x (t + 1) \sim p (\cdot | x (t), a (t), \mu) := \int_ {\mathsf {X}} \bar {p} (\cdot | x (t), a (t), z) \mu (d z)$$

In this MDP, the one-stage cost function c¯ is a function of $x ( t ) , a ( t )$ , and $z ( t )$ at time t or alternatively, one-stage cost function c is a function of $x ( t ) , a ( t )$ , and $\mu$ when one views $z ( t )$ as a noise. In this model, for any policy $\pi \in \Pi _ { l }$ , the discounted cost of a generic agent is the following

$$
\begin{array}{l} J (\mu ; \pi) := \mathbb {E} _ {\mu} ^ {\pi} \left[ \sum_ {t = 0} ^ {\infty} \beta^ {t} \bar {c} (x (t), a (t), z (t)) \right] \\ = \mathbb {E} _ {\mu} ^ {\pi} \left[ \sum_ {t = 0} ^ {\infty} \beta^ {t} c (x (t), a (t), \mu) \right] \\ \end{array}
$$

where we also assume that $x ( 0 ) \sim \mu ;$ that is, the initial distribution is the same as the limiting mean-field term $\mu .$ . Here, in the infinite population limit, a generic agent is only allowed to use its local state information when applying its action since the corresponding MDP is timehomogeneous. Therefore, we only consider policies in the policy space Πl. A policy $\gamma \in \Pi _ { l }$ is optimal for $\mu$ if

$$J (\mu ; \gamma) = \inf _ {\pi \in \Pi_ {l}} J (\mu ; \pi)$$

In view of this, let us define the following set-valued map

$$\Lambda (\mu) := \{\pi \in \Pi_ {l}: \pi \text { is optimal for } \mu \}$$

Under mild regularity conditions on $\bar { p }$ and $\bar { c } , \Lambda ( \mu )$ is non-empty for all $\mu \in { \mathcal { P } } ( \mathbb { X } )$ .
