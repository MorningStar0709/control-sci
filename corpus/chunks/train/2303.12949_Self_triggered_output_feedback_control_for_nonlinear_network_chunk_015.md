Since neither $V ( x _ { p } ( s _ { j } ) )$ nor its values from the past $m -$ 1 transmission times can be measured, we use instead $V ( x _ { o } ( s _ { j } ) )$ and the values of the Lyapunov function from the past $m - 1$ transmission times at transmission time $s _ { j }$ to determine the next transmission time $s _ { j + 1 }$ .
