# 3.1 Parameter estimation algorithm

Denote by $\phi _ { t } = \left[ x _ { t } ^ { \top } , u _ { t } ^ { \top } \right] ^ { \top } \in \mathbb { R } ^ { n + m }$ the regression vector. To construct an online estimation algorithm, for each $t \geq 0$ , we consider the weighted least-squares loss function given by

$$S _ {t} (\theta) = \frac {1}{\mu_ {t}} \mathbb {E} \left[ \left\| x _ {t + 1} - f (\theta^ {\top} \phi_ {t}) \right\| ^ {2} \mid \mathcal {F} _ {t} \right], \theta \in \Theta ,$$

where $\{ \mu _ { t } \} _ { t \ge 0 }$ is a weight sequence to be designed and is assumed to be adapted to the filtration $\{ { \mathcal F } _ { t } \} _ { t \ge 0 } .$ . The loss function is minimized at $\theta ^ { * }$ , and mi $\bar { \mathsf { 1 } } _ { \theta \in \Theta } \bar { S } _ { t } ( \theta ) =$ $S _ { t } ( \theta ^ { * } ) = \mu _ { t } ^ { - 1 } \mathbb { E } [ \left\| w _ { t + 1 } \right\| ^ { 2 } | \mathcal { F } _ { t } ] , \forall t \geq 0 .$ .

To estimate $\theta ^ { * }$ by minimizing $S _ { t } ( \theta )$ , we develop an online WLS-type algorithm, summarized in Algorithm 1.

We now give an intuitive and constructive interpretation of Algorithm 1. Since $\phi _ { t }$ and $\mu _ { t }$ are $\mathcal { F } _ { t } .$ −measurable and $\mathbb { E } [ w _ { t + 1 } \mid \mathcal { F } _ { t } ] = 0$ , it follows that

$$S _ {t} (\theta) - S _ {t} (\theta^ {*}) = \mu_ {t} ^ {- 1} \| f (\theta^ {* ^ {\top}} \phi_ {t}) - f (\theta^ {\top} \phi_ {t}) \| ^ {2}. \tag {12}$$

By Assumption 1, the nonlinear map $f ( \cdot )$ satisfies

$$
\begin{array}{l} d _ {t} \| \theta^ {* ^ {\top}} \phi_ {t} - \theta^ {\top} \phi_ {t} \| ^ {2} \\ \begin{array}{l} \omega_ {t \mid \mid} \phi_ {t} - \phi_ {t} - \phi_ {t \mid \mid} \\ \leq (\theta^ {* ^ {\top}} \phi_ {t} - \theta^ {\top} \phi_ {t}) ^ {\top} [ f (\theta^ {* ^ {\top}} \phi_ {t}) - f (\theta^ {\top} \phi_ {t}) ], \end{array} \tag {13} \\ \end{array}
$$

where $d _ { t }$ depends on the monotonicity function $\alpha ( \cdot )$ and is specified in Algorithm 1. On the other hand, applying the Cauchy–Schwarz inequality yields

$$
\begin{array}{l} \left(\theta^ {* ^ {\top}} \phi_ {t} - \theta^ {\top} \phi_ {t}\right) ^ {\top} \left[ f \left(\theta^ {* ^ {\top}} \phi_ {t}\right) - f \left(\theta^ {\top} \phi_ {t}\right) \right] \\ \leq \left\| \theta^ {* ^ {\top}} \phi_ {t} - \theta^ {\top} \phi_ {t} \right\| \left\| f \left(\theta^ {* ^ {\top}} \phi_ {t}\right) - f \left(\theta^ {\top} \phi_ {t}\right) \right\|. \\ \end{array}
$$
