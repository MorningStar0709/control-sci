# A. Assumptions

The starting point in many reinforcement learning analyses is the requirement that (1) be stabilizable [36]–[39]. In this work, we make the following assumptions that are both implied by stabilizability [40].

Assumption 1: There exists a locally Lipschitz continuous, positive-definite function $L : \mathbb { R } ^ { n } \to \mathbb { R } _ { + } .$ , a diverging continuous positive definite function $\nu : \mathbb { R } ^ { n } \to \mathbb { R } _ { + }$ and $\alpha _ { \mathrm { l o w } } , \alpha _ { \mathrm { u p } } \in$ $\kappa _ { \infty }$ s. t. for any compact $\bar { \mathbb { X } } \subset \mathbb { X } ,$ there exists a compact set $\bar { \mathbb { U } } \subset \mathbb { U }$ and it holds that, for any $x \in { \bar { \mathbb { X } } }$ ,

i) L has a decay rate satisfying

$$\inf _ {u \in \bar {\mathbb {U}}} \mathcal {L} _ {f (x, u)} L (x) \leq - \nu_ {0} (x), \tag {3}$$

ii) $\alpha _ { \mathrm { l o w } } ( \| x \| ) \leq L ( x ) \leq \alpha _ { \mathrm { u p } } ( \| x \| )$ .

In the above, $\mathcal { L } _ { f ( x , u ) } L ( x )$ is a suitable generalized directional differential operator of $L ( x )$ in the direction of $f ( x , u ) , \mathrm { e . g . }$ ., directional subderivative [41].

Assumption 2: There exists a stabilizing sample-and-hold policy $\eta : \mathbb { X } \to \mathbb { U }$ associated with the latter Lyapunov function $L , \mathrm { i . } \mathrm { e . }$ .,

$$\forall \delta \in (0, \bar {\delta} ]:U _ {t} := \eta (X _ {t \bmod \delta}) \impliesL (X _ {(k + 1) \delta}) - L (X _ {k \delta}) \leq - \delta \nu (X _ {k \delta}), \tag {4}$$

where $\nu ( x ) > 0$ whenever $\bar { S } _ { \sigma _ { \operatorname* { m a x } } , \bar { \delta } } > \| x \| > \bar { s } _ { \sigma _ { \operatorname* { m a x } } , \bar { \delta } } .$

A recent stochastic stability analysis [31] has shown that as long as the noiseless system

$$\mathrm{d} X _ {t} ^ {\text { noiseless }} = f (X _ {t} ^ {\text { noiseless }}, U _ {t}) \mathrm{d} t \tag {5}$$

is stabilizable, assumptions 1-2 hold. Furthermore it is true that

$$\bar {s} _ {\sigma_ {m a x}, \bar {\delta}} \rightarrow 0, \text { as } \sigma_ {\max} \rightarrow 0 \text { and } \bar {\delta} \rightarrow 0,\bar {S} _ {\sigma_ {m a x}, \bar {\delta}} \rightarrow + \infty , \text { as } \sigma_ {\max} \rightarrow 0 \text { and } \bar {\delta} \rightarrow 0. \tag {6}$$
