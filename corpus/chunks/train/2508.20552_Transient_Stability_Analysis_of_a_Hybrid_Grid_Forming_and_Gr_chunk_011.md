# C. Power Flow Model of the Hybrid Power System

The typical topology of the GFM/GFL-RES hybrid system is given in Fig. 3, where the main grid is modeled as an ideal voltage source to simplify the subsequent muti-machine interaction analysis. Bus 1 to 3 are active nodes, and bus 4 is passive node. The algebraic admittance model can be described as:

$$
\left[ \begin{array}{l} \mathbf {I} _ {\mathrm{s}} \\ 0 \end{array} \right] = \left[ \begin{array}{l l} \mathbf {Y} _ {\mathrm{a}} & \mathbf {Y} _ {\mathrm{b}} \\ \mathbf {Y} _ {\mathrm{c}} & \mathbf {Y} _ {\mathrm{d}} \end{array} \right] \left[ \begin{array}{l} \mathbf {U} _ {\mathrm{s}} \\ \mathbf {U} _ {\mathrm{ns}} \end{array} \right] \tag {7}
$$

where subscript $^ { 6 6 } \mathrm { \vec { s } } ^ { , 5 }$ and $" \mathrm { n } \mathrm { s } ^ { \mathrm { , } \mathrm { , } }$ indicate active node and passive node, respectively. and denote voltage vector and current vector. $\mathbf { Y } _ { \mathrm { a } } , \mathbf { Y } _ { \mathrm { b } } , \mathbf { Y } _ { \mathrm { c } }$ , and $\mathbf { Y _ { d } }$ are corresponding submatrices of the admittance matrix, which are also affected by system fault and load changes. Employing the Kron-reduction method to eliminate the passive nodes, (7) can be rewritten as:

$$\mathbf {I} _ {\mathrm{s}} = \left(\mathbf {Y} _ {\mathrm{a}} - \mathbf {Y} _ {\mathrm{b}} \mathbf {Y} _ {\mathrm{d}} ^ {- 1} \mathbf {Y} _ {\mathrm{c}}\right) \mathbf {U} _ {\mathrm{s}} = \mathbf {Y} _ {\mathrm{r}} \mathbf {U} _ {\mathrm{s}} \tag {8}$$

The expanded form of (8) is:

$$
\left[ \begin{array}{l} \mathbf {I} _ {\mathrm{V}} \\ \mathbf {I} _ {\mathrm{I}} \end{array} \right] = \left[ \begin{array}{l l} \mathbf {Y} _ {\mathrm{ra}} & \mathbf {Y} _ {\mathrm{rb}} \\ \mathbf {Y} _ {\mathrm{rd}} & \mathbf {Y} _ {\mathrm{rc}} \end{array} \right] \left[ \begin{array}{l} \mathbf {U} _ {\mathrm{V}} \\ \mathbf {U} _ {\mathrm{I}} \end{array} \right] \tag {9}
$$

where subscript $\mathbf { \bar { \Psi } } ^ { \mathfrak { s } } \mathbf { V } ^ { \mathfrak { s } }$ and $^ { 6 6 } \mathrm { I } ^ { 5 }$ indicate the voltage source and current source.
