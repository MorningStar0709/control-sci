# Exercise 4.15

Solve the problem

$$u ^ {*} := \underset {u: [ 0, T ] \rightarrow \mathbb {R}} {\operatorname{argmin}} J (u) = \int_ {0} ^ {T} \sqrt {1 + u (t) ^ {2}} d t \tag {308}$$

subject to

$$\dot {x} (t) = u (t), \quad x (0) = x _ {0}, \quad x (T) = x _ {1} \tag {310}$$

where $x _ { 0 } \in \mathbb { R }$ is an initial state, $T > 0$ is a fixed final time.
