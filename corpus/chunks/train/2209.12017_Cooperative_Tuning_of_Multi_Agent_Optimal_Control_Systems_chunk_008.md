Let $L _ { i } ( \pmb { \xi } _ { i } , \pmb { \theta } _ { i } )$ denote a scalar function, which is an additional ‘partial’ performance index or loss function (independent of $J _ { i } )$ used as a contribution to a group or team-average performance index or loss function, and reflecting to agent i’s trajectory $\pmb { \xi } _ { i } ( \pmb { \theta } _ { i } )$ as well as the associated performance index; thus ultimately, $L _ { i }$ is just a function of $\theta _ { i }$ because $\xi _ { i } ( \theta _ { i } )$ is in effect determined by the minimization of $J _ { i }$ . Correspondingly the global average $\begin{array} { r } { \frac { 1 } { N } \sum _ { i = 1 } ^ { N } L _ { i } ( \pmb { \xi } _ { i } , \pmb { \theta } _ { i } ) } \end{array}$ evaluates the performance of the whole multi-agent system. Note that the objective function $J _ { i }$ reflects a task specification that is only related to agent-i, whereas the loss function $L _ { i }$ indicates a new task specification which might be also related to other agents.

The problem of interest is to develop an iterative rule for each agent i to update $\theta _ { i }$ such that all the $\theta _ { i }$ reach a consensus at a common parameter $\pmb { \theta } ^ { * }$ , which minimizes the global average loss, i.e.

$$\left\{\boldsymbol {\theta} ^ {*} \right\} _ {i = 1} ^ {N} = \underset {\left\{\boldsymbol {\theta} _ {i} \right\} _ {i = 1} ^ {N}} {\arg \min} \frac {1}{N} \sum_ {i = 1} ^ {N} L _ {i} \left(\boldsymbol {\xi} _ {i} \left(\boldsymbol {\theta} _ {i}\right), \boldsymbol {\theta} _ {i}\right) \tag {2}$$

$\mathrm { s . t . } \qquad \xi _ { i } \ \mathrm { o b t a i n e d \ b y \ ( 1 ) \ u n d e r } \ \theta _ { i } .$

Note that the notation $\pmb { \xi } _ { i }$ is shorthand for the whole optimal trajectory (input and state) of agent-i from $t = 0 \mathrm { \ t o \ } t = T$ .
