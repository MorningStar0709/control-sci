# B. CBF-based safe RL

The coefficients used in training the RL agent in the reward function (17) are $\pmb { r } _ { 1 } = [ 2 , 2 ] , r _ { 2 } = 4 0 , r _ { 3 } = 2 5 , r _ { 4 } = 1 5$ , $r _ { 5 } = 2 0 0$ , and $r _ { 6 } = 1 0 ^ { 5 } .$

Fig. 3 presents the episode and average rewards (over 100 episodes) obtained by the RL agent during training. The increasing and converging average reward indicates that the agent successfully learns an AGC policy. The episode rewards drop below −200 in a few episodes in the beginning, marked by the red crosses in Fig. 3, when the CBF alerts to the RL agent’s unsafe actions. However, they never fall below $- 1 0 ^ { 5 }$ , confirming the effectiveness of the CBF in preventing safety violation.
