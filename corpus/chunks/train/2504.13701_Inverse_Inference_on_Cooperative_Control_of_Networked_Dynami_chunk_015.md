where $U \in \mathbb { R } ^ { N n \times N n }$ is a unitary matrix and $V \in \mathbb { R } ^ { T \times N \tau }$ n a block of a unitary matrix, and $\mathbf { \bar { \boldsymbol { \Lambda } } } \in \mathbb { R } ^ { N n \times N n }$ is a diagonal matrix consisting of the singular values of $Y _ { T } ^ { - }$ . Then, denote the inference error matrix as $E _ { 1 } = \widehat { A } _ { d } ( T ) - \grave { A _ { d } } .$ , and it satisfies

$$
\begin{array}{l} E _ {1} = (\Phi_ {T} ^ {+} - A _ {d} \Phi_ {T} ^ {-}) (Y _ {T} ^ {-}) ^ {\intercal} (Y _ {T} ^ {-} (Y _ {T} ^ {-}) ^ {\intercal}) ^ {- 1} \\ = \left(\Phi_ {T} ^ {+} - A _ {d} \Phi_ {T} ^ {-}\right) V \Lambda U ^ {\intercal} \left(U \Lambda V ^ {\intercal} V \Lambda U ^ {\intercal}\right) ^ {- 1} \\ = (\Phi_ {T} ^ {+} - A _ {d} \Phi_ {T} ^ {-}) V \Lambda^ {- 1} U ^ {\intercal} \\ \Rightarrow \| E _ {1} \| \leq \| \Phi_ {T} ^ {+} - A _ {d} \Phi_ {T} ^ {-} \| \cdot \| V \| \cdot \| \Lambda^ {- 1} \| \cdot \| U ^ {\intercal} \| \\ \leq \frac {\left\| \Phi_ {T} ^ {+} \right\| + \left\| A _ {d} \right\| \left\| \Phi_ {T} ^ {-} \right\|}{\sigma_ {\min} (Y _ {T} ^ {-})}, \tag {17} \\ \end{array}
$$

where $\sigma _ { \mathrm { m i n } } ( \cdot )$ is the minimum singular value of a matrix.

Recalling $\dot { Y } _ { T } ^ { - } = X _ { T } ^ { - } + \Phi _ { T } ^ { - } ;$ since $X _ { T } ^ { - }$ is deterministic and independent of $\Phi _ { T } ^ { - } , Y _ { T } ^ { - }$ can be regarded as a shifted version of the random matrix $\Phi _ { T } ^ { - }$ with independent entries. Therefore, we can directly apply the Theorem 1 in [42] to obtain a lower bound for $\sigma _ { \operatorname* { m i n } } ( Y _ { T } ^ { - } )$ as

$$\operatorname * {P r} \{\sigma_ {\min} (Y _ {T} ^ {-}) \geq u \sqrt {T} \} \geq 1 - \exp (- v T), \tag {18}$$

where u and v are positive constants that do not depend on T . Next, for the noise matrix $\Phi _ { T } ^ { + }$ where the entries are i.i.d. zero-mean Gaussian noises, we can apply the concentration of measure in Gaussian space [43] to $\Phi _ { T } ^ { + }$ and obtain that with probability at least $1 - 2 \exp \left( - r ^ { 2 } / 2 \right)$

$$\sigma_ {\max} (\Phi_ {T} ^ {+}) \leq \sqrt {T} + \sqrt {N n} + r. \tag {19}$$

By setting $r = { \sqrt { T } } ,$ we directly have

$$\operatorname * {P r} \{\sigma_ {\max} (\Phi_ {T} ^ {+}) \leq 2 \sqrt {T} + \sqrt {N n} \} \geq 1 - 2 \exp (- T / 2). \tag {20}$$

The above upper bound with probability guarantee also holds for $\sigma _ { \operatorname* { m a x } } \bigl ( \Phi _ { T } ^ { - } \bigr )$ .
