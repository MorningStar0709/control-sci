# C. Control Fusion and Supervisory Logic

At each sampling instant, the mode–conditioned LMPC for $i \in \mathcal { M }$ returns a feasible input $\boldsymbol { u } _ { i } ^ { \star } ( k )$ and the corresponding generalized force ${ \pmb \tau } _ { i } ^ { \star } ( k ) = { \pmb T } ( \theta _ { i } ) { \bf { T } } _ { i } { \pmb u } _ { i } ^ { \star } ( k )$ . We apply a probabilistic blending of these candidates which form the convex combination in force space [33], [44], [45]

$$\boldsymbol {\tau} ^ {\text { blend }} (k) = \sum_ {i \in \mathcal {M}} p _ {i} (k) \boldsymbol {\tau} _ {i} ^ {\star} (k), \tag {30}$$

and map it to thruster commands via a least–squares allocation using the current most–probable parameters $( { \bar { \theta } } , { \bar { \Gamma } } ) =$ arg $\operatorname* { m a x } _ { i } p _ { i } ( k )$ :

$$\hat {\boldsymbol {u}} (k) = \left(\boldsymbol {T} (\bar {\theta}) \bar {\boldsymbol {\Gamma}}\right) ^ {\dagger} \boldsymbol {\tau} ^ {\text { blend }} (k), \boldsymbol {u} ^ {\text { blend }} (k) = \Pi_ {\mathcal {U}} \bigl (\hat {\boldsymbol {u}} (k) \bigr), \tag {31}$$

where $\Pi _ { \mathcal { U } }$ denotes projection onto the admissible input set. This procedure yields smooth control during probability transitions and guarantees feasibility by construction through $\Pi _ { \boldsymbol { u } } .$ .

For each mode i, the associated LMPC enforces the same first–step contraction

$$V (F (\boldsymbol {\xi} _ {k}, \boldsymbol {u} _ {i} ^ {\star})) - V (\boldsymbol {\xi} _ {k}) \leq - \alpha \| \tilde {\boldsymbol {x}} _ {k} \| ^ {2}, \quad \alpha > 0. \tag {32}$$

Over one prediction step, the augmented error dynamics are affine in the generalized force and $V ( \cdot )$ is quadratic; hence for the unprojected blend uˆ one has the Jensen-type bound

$$V (F (\boldsymbol {\xi} _ {k}, \hat {\boldsymbol {u}})) \leq \sum_ {i \in \mathcal {M}} p _ {i} (k) V (F (\boldsymbol {\xi} _ {k}, \boldsymbol {u} _ {i} ^ {\star})). \tag {33}$$

Combining (32)–(33) and $\textstyle \sum _ { i } p _ { i } = 1$ yields

$$V (F (\boldsymbol {\xi} _ {k}, \hat {\boldsymbol {u}})) - V (\boldsymbol {\xi} _ {k}) \leq - \alpha \| \tilde {\boldsymbol {x}} _ {k} \| ^ {2}. \tag {34}$$
