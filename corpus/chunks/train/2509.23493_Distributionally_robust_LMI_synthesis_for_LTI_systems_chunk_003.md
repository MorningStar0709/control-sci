Here, $\chi ^ { \top } ( t ) \ = \ \left( x ^ { \top } ( t ) x _ { c } ^ { \top } ( t ) \right)$ is the combined state of the closed-loop system between (1) and (2). The closed-loop matrices $A , B , \mathcal { C } , \mathcal { D }$ are given by

$$
\left( \begin{array}{c c} \mathcal {A} & \mathcal {B} \\ \hline \mathcal {C} & \mathcal {D} \end{array} \right) := \left( \begin{array}{c c c} A + B _ {u} D _ {c} C _ {y} & B _ {u} C _ {c} & B _ {w} + B _ {u} D _ {c} D _ {y w} \\ B _ {c} C _ {y} & A _ {c} & B _ {c} D _ {y w} \\ \hline C _ {z} + D _ {z u} D _ {c} C _ {y} & D _ {z u} C _ {c} & D _ {z w} + D _ {z u} D _ {c} D _ {y w} \end{array} \right).
$$

Like the controller matrices, the closed-loop matrices are colored in blue, since they are affine functions of the controller parameters.

The ambiguity set $\mathcal { W }$ confines the probability distributions of the disturbance w using the 2-Wasserstein metric. It may take one of the following forms:

1) The ambiguity set ${ \mathcal { W } } _ { \mathrm { c o r } }$ contains all sequences of random disturbances $v ~ : ~ \mathbb { N } _ { 0 } ~ \to ~ L _ { 2 } \big ( \Omega , \mathbb { R } ^ { n _ { w } } \big )$ that satisfy $\mathbb { W } ( \mathbb { P } _ { w ( t ) } , \mathbb { P } _ { \mathrm { n o m } } ) \leq \gamma$ for all $t \in  { \mathbb { N } } _ { 0 }$ with a fixed, zero mean, normal distribution $\mathbb { P } _ { \mathrm { n o m } }$ with $\Sigma _ { \mathrm { n o m } } : = \int v v ^ { \top } \mathrm { d } \mathbb { P } _ { \mathrm { n o m } } ( v )$ .

2) The ambiguity set ${ \mathcal { H } } _ { \mathrm { i n d } }$ contains all sequences of independent, zero mean random disturbances $w : \mathbb { N } _ { 0 } \ \to$ $L _ { 2 } ( \Omega , \mathbb { R } ^ { n _ { w } } )$ that satisfy $\mathbb { W } ( \mathbb { P } _ { w ( t ) } , \mathbb { P } _ { \mathrm { n o m } } ) \leq \gamma$ for all $t \in  { \mathbb { N } } _ { 0 }$ for a fixed, zero mean, normal distribution $\mathbb { P } _ { \mathrm { n o m } }$ with $\Sigma _ { \mathrm { n o m } } : = \int \boldsymbol { v } \boldsymbol { v } ^ { \intercal } \mathrm { d } \mathbb { P } _ { \mathrm { n o m } } ( \boldsymbol { v } )$ .
