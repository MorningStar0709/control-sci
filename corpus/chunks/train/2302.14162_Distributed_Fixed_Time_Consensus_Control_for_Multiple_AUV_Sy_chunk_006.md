# A. Fixed-time stability

A typical nonlinear system can be represented as follows [44]:

$$\dot {\xi} (t) = f (\xi (t)), \quad \xi (t _ {0}) = \xi_ {0}, \quad \xi \in \Re^ {n} \tag {1}$$

where $f ( \cdot ) : \Re ^ { n }  \Re ^ { n }$ is a possibly discontinuous vector field. The fixed time convergence is determined for system (1) when it is globally finite-time stable and its convergent time is bounded regardless the initial states of the system, i.e., $\forall \xi _ { 0 } \in \mathfrak { R } ^ { n } , T ( \xi _ { 0 } ) \le T _ { \mathrm { m a x } }$ is satisfied, where $T _ { \mathrm { m a x } }$ is a positive constant.

Lemma 1 ( [44]): If a positive definite continuous function $V ( \xi ) \ : \ \mathfrak { R } ^ { n } \ \to$ < for system (1) satisfies $\dot { V } ( \boldsymbol { \xi } ) ~ \leq$ $- \chi _ { 1 } V ^ { \varrho } ( \xi ) - \chi _ { 2 } V ^ { \varsigma } ( \xi )$ for some $\chi _ { 1 } > 0 , \chi _ { 2 } > 0 , \varrho > 1$ , and $0 < \varsigma < 1$ , then system (1) is determined as a globally fixedtime stable system. The convergence time can be calculated independently with the initial states of system (1) as follows:

$$T (\xi_ {0}) \leq \frac {1}{\chi_ {1} (\varrho - 1)} + \frac {1}{\chi_ {2} (1 - \varsigma)}. \tag {2}$$

Lemma 2 ( [44]): If a positive definite continuous function $V ( \xi ) \ : \ \mathfrak { R } ^ { n } \ \to \ \mathfrak { R }$ for system (1) satisfies $\dot { V } ( \xi ) \ \leq$ $- \chi _ { 1 } V ^ { p } ( \xi ) - \chi _ { 2 } V ^ { q } ( \xi ) + \varphi$ for some $\chi _ { 1 } > 0 , \chi _ { 2 } > 0 , p > 1 .$ , $0 \textless q \textless 1$ , and $0 < \varphi < \infty$ , then system (1) is called a practically fixed-time stable system. Futhermore, the solution of system (1) has a residual set:

$$\lim _ {t \rightarrow T} \xi | \| \xi \| \leq \min \left\{\chi_ {1} ^ {\frac {- 1}{p}} \left(\frac {\varphi}{1 - \kappa}\right) ^ {\frac {1}{p}}, \chi_ {2} ^ {\frac {- 1}{q}} \left(\frac {\varphi}{1 - \kappa}\right) ^ {\frac {1}{q}} \right\} \tag {3}$$

where κ satisfies $0 ~ < ~ \kappa ~ < ~ 1$ . The settling time can be calculated independenly with the initial states of the system as follows:

$$T (\xi_ {0}) \leq \frac {1}{\chi_ {1} \kappa (1 - p)} + \frac {1}{\chi_ {2} \kappa (q - 1)}. \tag {4}$$
