Observations:

1. As ζ approaches zero (dark blue), the magnitude plot has a sharper and sharper peak near $\omega = \omega _ { n }$   
2. As ζ approaches 1.0 (magenta), the magnitude plot smoothly curves through the point 6dB below the intersection of the two straight line asymptotes.   
3. The high frequency asymptote has a slope of −40dB/decade.   
4. The Phase plot goes from $0 ^ { \circ } ~ \mathrm { t o } ~ - 9 0 ^ { \circ }$ as ω increases beyond $\omega _ { n } .$   
5. As ζ approaches zero, the phase plot has a sharper transition $0 ^ { \circ }  - 9 0 ^ { \circ } .$   
6. As ζ approaches 1.0, the phase plot acts like two real poles at $\omega = \omega _ { n } .$
