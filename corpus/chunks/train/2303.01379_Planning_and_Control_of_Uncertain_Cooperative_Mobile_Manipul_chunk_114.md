# 2.2.8 Decentralized NMPC

In this section, in order to reduce the computational complexity of the NMPC, we develop a decentralized counterpart, where each robotic agent calculates its own control signal.

We first decouple the dynamics (2.14) for each agent’s MPC. We define $x _ { O _ { i } } : \mathbb { R } ^ { n _ { i } } \to \mathbb { M } , v _ { O _ { i } } : \mathbb { R } ^ { 2 n _ { i } } \to \mathbb { R } ^ { 6 }$ with $x _ { O _ { i } } ( q _ { i } ) : = [ p _ { O _ { i } } ( q _ { i } ) ^ { \top } , \eta _ { O _ { i } } ( q _ { i } ) ^ { \top } ] ^ { \top } \in \mathbb { M }$ ,

$$p _ {O _ {i}} \left(q _ {i}\right) := p _ {E _ {i}} \left(q _ {i}\right) + R _ {i} \left(q _ {i}\right) p _ {O / E _ {i}} ^ {E _ {i}} \tag {2.58a}\eta_ {O _ {i}} (q _ {i}) := \eta_ {E _ {i}} (q _ {i}) + \eta_ {O / E _ {i}} \tag {2.58b}$$

i $\in \mathcal { N } .$ , as well as

$$v _ {O _ {i}} (q _ {i}, \dot {q} _ {i}) := [ \dot {p} _ {O _ {i}} (q _ {i}) ^ {\top}, \omega_ {O _ {i}} (q _ {i}, \dot {q} _ {i}) ^ {\top} ] ^ {\top} := J _ {i _ {O}} (q _ {i}) v _ {i} (q _ {i}, \dot {q} _ {i}), \forall i \in \mathcal {N}, \tag {2.59}$$
