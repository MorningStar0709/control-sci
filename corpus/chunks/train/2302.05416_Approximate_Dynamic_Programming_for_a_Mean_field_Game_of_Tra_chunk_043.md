# Step 4. Boundedness of Approximate Solutions.

Now, we must show that there is a limit point of this sequence. We will briefly comment on boundedness of the approximate solutions now. From the bounds given by Pazy in [28], and from a ω−contractivity of $\mathcal { L } _ { n } ( t ; \tilde { u } ^ { n } ( \cdot ) , \tilde { w } ^ { n } ( \cdot ) )$ , we know that (using C as an arbitrary non-negative constant):

$$| | \rho^ {n} (t) | | _ {D (\mathcal {L} _ {n})} \leq e ^ {\omega t} | | \rho_ {0} | | _ {D (\mathcal {L} _ {n})} \leq e ^ {\omega T} | | \rho_ {0} | | _ {D (\mathcal {L} _ {n})} \leq C | | \rho_ {0} | | _ {\mathrm{H} ^ {2} (Y; \mathbb {R})} =: \Re , \tag {53}$$

since $D ( \mathcal { L } _ { n } ) \subset \mathrm { H } ^ { 2 } ( Y ; \mathbb { R } )$ . As for $A ^ { n } , B ^ { n }$ , we have from the proof of Lemma 1:

$$| | A ^ {n} (t) | | _ {F} + | | B ^ {n} (t) | | _ {F} \leq \dots| | A _ {0} | | _ {F} + | | B _ {0} | | _ {F} + \int_ {0} ^ {t} | | \mathfrak {G} (A ^ {n} (s), B ^ {n} (s), \rho^ {n - 1} (s)) | | _ {F} d s + \dots\int_ {0} ^ {t} | | \mathfrak {H} (A ^ {n} (s), B ^ {n} (s), \rho^ {n - 1} (s)) | | _ {F} d s \leq \dots\left| \left| A _ {0} \right| \right| _ {F} + \left| \left| B _ {0} \right| \right| _ {F} + C t + \int_ {0} ^ {t} \left| \left| A ^ {n} (s) \right| \right| _ {F} + \left| \left| B ^ {n} (s) \right| \right| _ {F} d s$$

again using our result from (39). Now, by the Grönwall-Bellman [14] inequality:

$$| | A ^ {n} (t) | | _ {F} + | | B ^ {n} (t) | | _ {F} \leq (| | A _ {0} | | _ {F} + | | B _ {0} | | _ {F} + C T) e ^ {T} =: \mathfrak {D},$$

hence $( A , B , \rho ) ^ { n } ( \cdot )$ is uniformly bounded in the usual supremum-norm topology on $C ( [ 0 , T ] ; \mathbb { H } ^ { 2 } )$ . We also know that our classical solution $\rho ^ { n } ( \cdot )$ is s.t.:

$$\frac {\mathrm{d}}{\mathrm{d} t} \rho^ {n} (t) = \mathcal {L} _ {n} (t; u ^ {n} (t), w ^ {n} (t)) \rho^ {n} (t), \tag {54}$$

in $\operatorname { L } ^ { 2 } ( Y ; \mathbb { R } )$ and from (53):

$$| | \frac {\mathrm{d}}{\mathrm{d} t} \rho^ {n} (t) | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} = | | \mathcal {L} _ {n} (t; u ^ {n} (t), w ^ {n} (t)) \rho^ {n} (t) | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} \leq | | \rho^ {n} (t) | | _ {D (\mathcal {L} _ {n})} \leq \Re .$$
