# C. Control Objective and Standing Assumptions

The objective is to design a fault-tolerant tracking controller that achieves high-accuracy tracking in nominal conditions, and detects and accommodates thruster faults with minimal delay, and reallocates control effort while preserving closedloop stability under thruster degradation. We make the following standard assumptions:

• $M \ \succ \ 0 , \ \exists \ d _ { \operatorname* { m i n } } \ > \ 0 \ s . { \mathrm { t . } } \ D ( \nu ) \ \succeq \ d _ { \operatorname* { m i n } } I \qquad \forall \nu \ =$ $[ u , v , r ] ^ { \top }$ with $| u | \ \leq \ \bar { u } , \ | v | \ \leq \ \bar { v } , \ | r | \ \leq \ \bar { r } .$ where $\bar { \boldsymbol { \nu } } = [ \bar { u } , \bar { v } , \bar { r } ] ^ { \top }$ denotes the operating-range bounds on the body-frame velocities.   
• disturbances w are bounded; inputs $\pmb { u } \in [ u _ { \operatorname* { m i n } } , u _ { \operatorname* { m a x } } ]$ with known rate limits;   
• $( \Gamma , \theta )$ are piecewise-constant and change at isolated instants (Sec. IV).

The constant $d _ { \operatorname* { m i n } } > 0$ is a uniform damping lower bound. Let $\pmb { x } _ { d } = [ \pmb { \eta } _ { d } ^ { \top } , \pmb { \nu } _ { d } ^ { \top } ] ^ { \top }$ denote the desired trajectory and its compatible body-frame velocity, with

$$\boldsymbol {\nu} _ {d} = \boldsymbol {J} ^ {- 1} (\psi_ {d}) \dot {\boldsymbol {\eta}} _ {d}, \tag {6}$$

Define the tracking error $\tilde { \pmb { x } } = \pmb { x } - \pmb { x } _ { d } .$
