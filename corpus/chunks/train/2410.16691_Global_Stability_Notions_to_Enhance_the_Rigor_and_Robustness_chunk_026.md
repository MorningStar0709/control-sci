Discussion of Theorem 4: Theorem 4 in the same spirit with the results in [1] for systems defined on $\mathbb { R } ^ { n }$ (it is a generalization of certain results in [1]) and with the results in [13] for abstract systems defined on normed linear spaces. However, in contrast with the results in [1], Theorem 4 does not require the set $S _ { r } = \left\{ \left. x \in \mathbb { R } ^ { n } : H ( x ) \leq r \right. \right\}$ to be bounded for all $r > \operatorname* { i n f } \left\{ H ( x ) { : } x \in \mathbb { R } ^ { n } \right\}$ , i.e., Theorem 4 does not require  to be radially unbounded. Instead, Theorem 4 requires boundedness of the set $f ( S _ { r } \times D _ { r } )$ for all $r > \operatorname* { i n f } \left\{ H ( x ) { : } x \in \mathbb { R } ^ { n } \right\}$ ; this condition is automatically satisfied if is radially unbounded (notice that in this case $S _ { r } = \left\{ \vphantom { \sum _ { i } ^ { n } } x \in \mathbb { R } ^ { n } : H ( x ) \leq r \right\}$ is compact for all $r > \operatorname* { i n f } \left\{ H ( x ) { : } x \in \mathbb { R } ^ { n } \right\}$ and since $D \subseteq \mathbb { R } ^ { p }$ is closed it also follows that $D _ { r } = \left\{ d \in D \colon | d | \leq r \right\}$ is compact for all $r > \operatorname* { i n f } \left\{ H ( x ) { : } x \in \mathbb { R } ^ { n } \right\} .$ ).

Example 5: Consider the following system

$$\dot {x} = f (x, d) \tag {49}x \in \mathbb {R} ^ {n}, d \in \mathbb {R} ^ {p}$$

where $f \in C ^ { 1 } \left( \mathbb { R } ^ { n } \times \mathbb { R } ^ { p } \right)$ is a mapping that satisfies the following estimate for all $x \in \mathbb { R } ^ { n } , d \in \mathbb { R } ^ { p }$

$$\left| f (x, d) \right| \leq \left(L + \kappa (\left| d \right|)\right) | x | + \kappa (\left| d \right|) \tag {50}$$

where $\kappa \in K _ { \infty }$ and $L > 0$ is a constant. Define the radially unbounded function

$$H (x) = \frac {1}{2} | x | ^ {2} \tag {51}$$

for which we get (using (50) and completing the squares) for all $x \in \mathbb { R } ^ { n } , d \in \mathbb { R } ^ { p }$

$$
\begin{array}{l} \dot {H} (x, d) = \nabla H (x) f (x, d) = x ^ {\prime} f (x, d) \\ \leq | x | | f (x, d) | \leq \left(L + \kappa (| d |)\right) | x | ^ {2} + | x | \kappa (| d |) \\ \leq \left(L + \kappa (| d |) + \frac {1}{2}\right) | x | ^ {2} + \frac {1}{2} \kappa^ {2} (| d |) \tag {52} \\ = (2 L + 2 \kappa (| d |) + 1) H (x) + \frac {1}{2} \kappa^ {2} (| d |) \\ \end{array}
$$

It follows from (52) that (42) holds for appropriate non-decreasing functions $c , \sigma \in C ^ { 0 } \left( \mathbb { R } _ { + } ; \mathbb { R } _ { + } \right)$ . We conclude from Theorem 4 that system (49) is forward complete.
