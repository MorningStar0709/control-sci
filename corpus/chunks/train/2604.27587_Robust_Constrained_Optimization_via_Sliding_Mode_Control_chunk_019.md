# 3.3 Finite-Time Nonsingular TSMC Design for Global Optimality in Convex Optimization

Conventional sliding mode control $S \ = \ h ( x )$ achieves finite-time constraint satisfaction via $\dot { S } = - K \mathrm { s g n } ( S )$ , but slows near $h \ = \ 0$ unless large gains are used. In the primal dynamics, only the normalized gradient of f as in [15] is used rather than $\nabla { \mathcal { L } } = \nabla \phi + { \bar { J } } _ { h } ^ { \top } \lambda$ , since SMC guarantees $h ( x ) = 0$ in finite time; once the manifold is reached, $\lambda ^ { \top } \dot { h } = 0$ and the optimization reduces to minimizing φ on the constraint manifold. This decouples the roles: SMC robustly enforces feasibility, while the gradient flow drives the state to the optimum without redundancy or interference from λ-terms, under bounded disturbance $\| \xi ( t ) \| \le D$ .
