block transformation of a T −periodic $n \times n$ matrix function $A \ = \ ( a _ { i j } ) _ { i , j = 1 , \cdots , n } \in L ^ { 2 } ( [ 0 \ T ] , \mathbb { R } ^ { n \times n } )$ , denoted ${ \mathcal { A } } = { \mathcal { T } } ( A )$ , defines a constant $n \times n$ Toeplitz Block (TB) and infinite dimensional matrix:

$$
\mathcal {A} = \left( \begin{array}{c c c} \mathcal {A} _ {1 1} & \dots & \mathcal {A} _ {1 n} \\ \vdots & \ddots & \vdots \\ \mathcal {A} _ {n 1} & \dots & \mathcal {A} _ {n n} \end{array} \right) \tag {1}
$$

where $\mathcal { A } _ { i j } = \mathcal { T } ( a _ { i j } ) , i , j = 1 , \cdots , n .$

The m−truncation of the $n \times n$ Toeplitz block matrix A denoted by $\mathcal { T } _ { m } ( A )$ , is defined by the m−truncation $\mathcal { T } _ { m } ( a _ { i j } )$ for all its entries $( i , j )$ . Similarly, the $n \times n$ Hankel block matrices $\mathcal { H } ( A ^ { + } ) , \ \mathcal { H } ( A ^ { - } )$ are also defined respectively by $\mathcal { H } ( A ^ { + } ) _ { i j } = \mathcal { H } ( a _ { i j } ^ { + } )$ and $\mathcal { H } ( A ^ { - } ) _ { i j } = \mathcal { H } ( a _ { i j } ^ { - } )$ for $i , j = 1 , \cdots , n .$ . Their principal submatrices $\mathcal { H } ( A ^ { + } ) _ { ( p , q ) } , \mathcal { \hat { H } } ( A ^ { - } ) _ { ( p , q ) }$ for $p > 0 ,$ $q > 0$ are obtained by considering the principal submatrices of the entries $\mathcal { H } ( a _ { i j } ^ { + } ) _ { ( p , q ) }$ and $\mathcal { H } ( a _ { i j } ^ { - } ) _ { ( p , q ) }$ for $i , j = 1 , \cdots , n .$ .

Recall that the product of two TB matrices is a TB matrix only in infinite dimension. In finite dimension, we have the following result [8].

Theorem 1: Let A, B be two n×n TB matrices and $\mathcal { C } = A B$ Then,

$$
\begin{array}{l} \mathcal {T} _ {m} (A) \mathcal {T} _ {m} (B) = \mathcal {T} _ {m} (C) - \mathcal {H} _ {(m, \eta)} \left(A ^ {+}\right) \mathcal {H} _ {(\eta , m)} \left(B ^ {-}\right) \\ - \mathcal {J} _ {n, m} \mathcal {H} _ {(m, \eta)} (A ^ {-}) \mathcal {H} _ {(\eta , m)} (B ^ {+}) \mathcal {J} _ {n, m}, \tag {2} \\ \end{array}
$$

where ${ \mathcal { T } } _ { n , m } = I d _ { n } \otimes J _ { m }$ and $\eta \in \mathbb { Z } ^ { + } \cup \left\{ + \infty \right\}$ is such that $2 \eta \geq$ min $( d ^ { o } A , d ^ { o } B )$ with $d ^ { o } A$ the largest harmonic (non vanishing Fourier coefficient) of A.
