# 4.1. Learning Trackability

Using the state estimator and our models for the transition and emission $f$ and $g ,$ , we can create rollouts of a regular MPC policy. These rollouts have the form $\mathcal { D } = \{ \mathbf { z } _ { 1 : T } ^ { i } , \mathbf { h } _ { 1 : T } ^ { i } , \mathbf { e } _ { 1 : T - 1 } ^ { i } \} _ { i = 1 } ^ { N }$ , where we denote the state estimation error at time t by $\mathbf { e } _ { t } : = \epsilon ( \mathbf { z } _ { t } , \mathbf { h } _ { t } )$ .

Given D, we can train the neural network by minimising:

$$\min _ {\phi} \sum_ {i} ^ {N} (\phi (\mathbf {z} _ {1} ^ {i}) - J ^ {\mathrm{err}} (\mathbf {z} _ {1} ^ {i}; \lambda , \phi)) ^ {2}, \tag {5}$$

where $J ^ { \mathrm { e r r } } ( \mathbf { z } _ { 1 } ^ { i } ; \lambda , \phi )$ is the λ-return of $\mathbf { z } _ { 1 } ^ { i }$ defined as:

$$J ^ {\mathrm{err}} (\mathbf {z} _ {1} ^ {i}; \lambda , \phi) = (1 - \lambda) \sum_ {k = 1} ^ {T - 1} \lambda^ {k - 1} J _ {k} ^ {\mathrm{err}} (\mathbf {z} _ {1} ^ {i}; \phi), \qquad J _ {k} ^ {\mathrm{err}} (\mathbf {z} _ {1} ^ {i}; \phi) = \sum_ {t = 1} ^ {k} \beta^ {t - 1} \mathbf {e} _ {t} ^ {i} + \beta^ {k} \phi (\mathbf {z} _ {k + 1} ^ {i}).$$
