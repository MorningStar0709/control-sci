# 3.4 时域指标

一个控制系统设计的性能指标通常包含与系统时间响应相关的一些要求。阶跃响应的这些要求由图 3.23 所示的标准数量来表示。

![](image/71522485696e32b520974bc97f96321057d593f96ee68213fbe7b51c61ceca9b.jpg)

<details>
<summary>line</summary>

| t | Value |
| --- | --- |
| t_r | 0.1 |
| t_s | 0.9 |
| t_p | 1.0 |
| M_p | 0.9 |
| ±1% | 1.0 |
</details>

图3.23 上升时间 $t_{\mathrm{r}}$ 、调节时间 $t_{\mathrm{s}}$ 和超调 $M_{\mathrm{p}}$ 的定义

(1) 上升时间 $t_{r}$ 就是系统达到新设定点附近所需的时间。  
(2) 调节时间 $t_{s}$ 就是系统暂态衰减所需的时间。  
（3）超调 $M_{p}$ 就是系统调节过程中超过终值的最大值除以终值所得的结果（通常以百分比的形式表示）。  
（4）峰值时间 $t_{p}$ 就是系统达到最大超调点所需的时间。
