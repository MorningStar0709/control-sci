# 2.1 Backstepping Control Design

Assumption 1 The parameters $q , \varepsilon > 0 , \lambda \in C ^ { 2 } ( [ 0 , 1 ] ; \mathbb { R } _ { + } )$ satisfy the following relation:

$$q > \frac {\lambda_ {m a x}}{2 \varepsilon} + \frac {\theta_ {1}}{2}, \tag {7}$$

where

$$\lambda_ {m a x} \triangleq \max _ {x \in [ 0, 1 ]} \lambda (x). \tag {8}$$

Assumption 1 is important in ensuring the stability of the target system under PDE backstepping control with dynamic event-triggering. This is because we intentionally avoid using the signal $u ( 1 , t )$ in the nominal control law. Such avoidance is crucial for dynamic ETC design due to the challenges associated with obtaining a meaningful bound on the rate of change of $u ( 1 , t )$ . However, an eigenfunction expansion of the solution of $( 1 ) ‐ ( 3 )$ with $U ( t ) \bar { = } 0$ shows that the system is unstable when $\mathrm { m i n } _ { x \in [ 0 , 1 ] } \lambda \left( x \right) > \varepsilon \pi ^ { 2 } / 4 ^ { \theta _ { 1 } }$ , no matter what q > 0. $q > 0$

Consider the invertible backstepping transformation

$$w (x, t) = u (x, t) - \int_ {0} ^ {x} K (x, y) u (y, t) d y, \tag {9}$$

defined in the domain $0 \leq y \leq x \leq 1$ , where $K ( x , y )$ satisfies

$$K _ {x x} (x, y) - K _ {y y} (x, y) = \frac {\lambda (y)}{\varepsilon} K (x, y), \tag {10}\theta_ {1} K _ {y} (x, 0) = - \theta_ {2} K (x, 0), \tag {11}K (x, x) = - \frac {1}{2 \varepsilon} \int_ {0} ^ {x} \lambda (y) d y, \tag {12}$$

for $0 \leq y \leq x \leq 1$ . Then, using standard arguments in PDE backstepping boundary control, we can show that the transformation (9)-(12) and a control law $U ( t )$ chosen as

$$U (t) = \int_ {0} ^ {1} k (y) u (y, t) d y, \tag {13}$$

where

$$k (y) = \wp K (1, y) + K _ {x} (1, y), \tag {14}$$

with

$$\wp = q - \frac {1}{2 \varepsilon} \int_ {0} ^ {1} \lambda (y) d y, \tag {15}$$

map the system (1)-(3) into the following target system:

$$w _ {t} (x, t) = \varepsilon w _ {x x} (x, t), \tag {16}\theta_ {1} w _ {x} (0, t) = - \theta_ {2} w (0, t), \tag {17}w _ {x} (1, t) = - \wp w (1, t). \tag {18}$$

The inverse transformation of (9)-(12) is given by

$$u (x, t) = w (x, t) + \int_ {0} ^ {x} L (x, y) w (y, t) d y, \tag {19}$$

where $L ( x , y )$ satisfies

$$L _ {x x} (x, y) - L _ {y y} (x, y) = - \frac {\lambda (y)}{\varepsilon} L (x, y), \tag {20}\theta_ {1} L _ {y} (x, 0) = - \theta_ {2} L (x, 0), \tag {21}L (x, x) = - \frac {1}{2 \varepsilon} \int_ {0} ^ {x} \lambda (y) d y, \tag {22}$$

for $0 \leq y \leq x \leq 1$ . Following arguments similar to those provided in [21], [33], it can be shown that the PDE systems $( 1 0 ) – ( 1 2 )$ and (20)-(22) admit unique $C ^ { 3 }$ solutions in the domain $0 \leq y \leq x \leq 1$ , provided that $\lambda \in C ^ { 2 } ( [ 0 , 1 ] )$ .
