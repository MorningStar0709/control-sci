The left hand side can be rewritten as

$$\mathrm{tr} (e ^ {s B} e ^ {t A} A) = \mathrm{tr} (e ^ {s B} A e ^ {t A}) = \mathrm{tr} (e ^ {t A} e ^ {s B} A) = \mathrm{tr} (X (T) A).$$

Similarly, the right hand side of (4.11) is equal to tr(X(T )B). Hence, it follows that also

$$\operatorname{tr} (X (T) A) = \operatorname{tr} (X (T) B), \tag {4.12}$$

which implies that T (and thus, since η is T -periodic, also 0) is a switching time.

Remark 4.13. If we consider a solution of (2.2) with one switch, the final point is in the form

$$X (T) = X (t, s) = e ^ {t A} e ^ {s B}$$

where $T = t + s$ . So, we can see the cost functional as a function of t and s:

$$\Phi (t, s) = \mathrm{tr} (e ^ {t A} e ^ {s B}).$$

With this notation, if t is a switching point, from the previous proof follows that X(T ) must satisfy Equation (4.12), which can be rewritten as

$$\frac {\partial \Phi}{\partial t} (t, s) = \frac {\partial \Phi}{\partial s} (t, s). \tag {4.14}$$

To summarize, given any t > 0, equation (4.14) give us a condition to find s > 0 as a function of t such that the control

$$
u (\tau) = \left\{ \begin{array}{l l} 1 & \text {if} \quad \tau \in [ 0, t ], \\ 0 & \text {if} \quad \tau \in [ t, t + s ], \end{array} \right.
$$

satisfies Pontryagin Maximum Principle on the time interval $[ 0 , t + s ]$ . The corresponding solution to the adjoint system η satisfies transversal condition on the same time interval, so one can extend this control by periodicity on the whole line $\lbrack 0 , + \infty )$ .

Concerning the value of the principal Lyapunov exponent, when the adjoint trajectory is periodic it can be computed as follows:

$$
\begin{array}{l} \ell_ {\text { per }} (T) = \limsup _ {n \to + \infty} \frac {1}{n T} \log \| X (n T) \| = \\ = \operatorname * {l i m s u p} _ {n \to + \infty} \frac {1}{n T} \log \| X (T) ^ {n} \| = \\ = \operatorname * {l i m s u p} _ {n \to + \infty} \frac {1}{n T} \log \lambda (T) ^ {n} = \frac {1}{T} \log \lambda (T), \tag {4.15} \\ \end{array}
$$

Where $\lambda ( T )$ is the greatest eigenvalue of X(T ). We can further simplify the expression for $\ell _ { \mathrm { p e r } } ( T )$ . Indeed, the functional of our Optimal Control Problem does not appear explicitly in the formula for $\ell _ { \mathrm { p e r } } ( T )$ . However, we can deduce the value the eigenvalue from the trace solving

$$\lambda (T) ^ {2} - \mathrm{tr} (X (T)) \lambda (T) + 1 = 0,$$

which gives

$$\lambda (T) = \frac {\mathrm{tr} (X (T)) + \sqrt {\mathrm{tr} (X (T)) ^ {2} - 4}}{2}.$$

Using the identity arcoshx $= \log ( x + { \sqrt { x ^ { 2 } - 1 } } )$ , we obtain

$$\ell_ {\mathrm{per}} (T) = \frac {\operatorname{arcosh} \left(\frac {\operatorname{tr} (X (T))}{2}\right)}{T}.$$
