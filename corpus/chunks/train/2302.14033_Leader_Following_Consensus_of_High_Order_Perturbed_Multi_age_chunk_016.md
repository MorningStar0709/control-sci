Proof of Theorem 2. Let $X \in \mathbb { R } ^ { n }$ and let $M \in \mathbb { R } ^ { n \times n }$ , then

$$M < 0 \quad \Leftrightarrow \quad X ^ {\prime} M X < 0 \quad \forall X \neq 0 \tag {41}$$

According to (41), we can write

$$X ^ {\prime} (M _ {1} + \xi M _ {2}) X = X ^ {\prime} M _ {1} X + \xi X ^ {\prime} M _ {2} X < 0 \tag {42}$$

By denoting $\lambda _ { m a x } ^ { { \cal M } _ { 1 } } < 0$ and $\lambda _ { m a x } ^ { { \cal M } _ { 2 } } > 0$ be the maximum eigenvalues of $M _ { 1 }$ and $M _ { 2 } ,$ , (42) can be recast as follows

$$X ^ {\prime} M _ {1} X + \xi X ^ {\prime} M _ {2} X \leq (\lambda_ {m a x} ^ {M _ {1}} + \xi \lambda_ {m a x} ^ {M _ {2}}) \| X \| _ {2} ^ {2} \tag {43}$$

it follows that

$$\lambda_ {m a x} ^ {M _ {1}} + \xi \lambda_ {m a x} ^ {M _ {2}} < 0 \quad \Rightarrow \quad \xi \lambda_ {m a x} ^ {M _ {2}} < - \lambda_ {m a x} ^ {M _ {1}}\xi < - \frac {\lambda_ {m a x} ^ {M _ {1}}}{\lambda_ {m a x} ^ {M _ {2}}} = \frac {| \lambda_ {m a x} ^ {M _ {1}} |}{| \lambda_ {m a x} ^ {M _ {2}} |} \tag {44}$$

Since $M _ { 1 } < 0 , M _ { 2 } > 0$ and $\| \boldsymbol { M } \| = \lambda _ { m a x } ^ { \phantom { a b c } M }$ , we can thus rewrite (44) as follows

$$\xi < \frac {\left\| M _ {1} \right\|}{\left\| M _ {2} \right\|} \tag {45}$$

This concludes the proof.

Proof of Theorem 3. By substituting (12) into the MAS dynamics (8), we obtain:

$$\dot {x} _ {i} (t) = A x _ {i} (t) - B \left(\sum_ {j = 0} ^ {N} \alpha_ {i j} k _ {i j} \left(x _ {i} \left(t - \tau_ {i j} (t)\right) - x _ {j} \left(t - \tau_ {i j} (t)\right)\right) \right.\left. + \rho \operatorname{sign} \left(s _ {i} (t)\right) - \omega_ {i} (t)\right) \tag {46}$$

and then, by computing the time derivative of (13) along with the trajectories of (14), we reach to:

$$\dot {s} _ {i} (t) = b \left(\omega_ {i} (t) - \rho \operatorname{sign} \left(s _ {i} (t)\right)\right) \tag {47}$$

Let us now select the following Lyapunov function:

$$\bar {V} (t) \frac {1}{2} \sum_ {i = 1} ^ {N} s _ {i} (t) ^ {2} \tag {48}$$

so that the time derivative of V¯ (t) correspondingly takes the form:

$$\dot {\bar {V}} (t) = \sum_ {i = 1} ^ {N} s _ {i} (t) \dot {s} _ {i} (t) = \sum_ {i = 1} ^ {N} b \big (s _ {i} (t) \omega_ {i} (t) - \rho | s _ {i} (t) | \big) \tag {49}$$

Then by referring to Assumption 1, we manipulate (49) as:

$$\dot {\bar {V}} (t) \leq - \sum_ {i = 1} ^ {N} b (\rho - \Gamma) \cdot | s _ {i} (t) | < 0 \quad \forall i, \quad \rho > \Gamma . \tag {50}$$
