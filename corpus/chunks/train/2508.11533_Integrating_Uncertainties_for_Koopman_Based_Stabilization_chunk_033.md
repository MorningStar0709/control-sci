# 5.1 Validity of the Lifted-State Feedback Controller

In Theorems 1 and 2, designed controllers given by $u =$ $K z = L ^ { \top } P ^ { - 1 } z$ exhibit a linear dependence on the liftedstate. A natural question arises as to whether this structure is sufficient for addressing bilinear system control problems, as exemplified by the infinite-horizon optimal control [1]. However, investigating the robust stabilization problem using the designed controllers $u = K z { \mathrm { ~ r e } } -$ mains to be justified. The following theorem seeks to confirm that the linear component with respect to $z = \Psi ( x )$ is far more critical to guarantee closed-loop stability.

Theorem 3 Consider the lifted bilinear system (28). Assume the feedback controller that linearly dependent on the lifted-state $u = K z$ obtained by (29) stabilizes (28). Then if an additional nonlinear term $u _ { n } ( z )$ is added to the original controller resulting in a modified one $u =$ $K z + u _ { n } ( z )$ , where $u _ { n } ( z )$ is a high-order small term with respect to z, i.e.,

$$\lim _ {z \rightarrow 0} \frac {\| u _ {n} (z) \|}{\| z \|} = 0, \tag {56}$$

the closed-loop system controlled by $u = K z + u _ { n } ( z )$ is also locally stable.
