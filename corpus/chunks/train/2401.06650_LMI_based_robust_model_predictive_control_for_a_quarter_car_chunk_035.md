$$\tilde {f} _ {k} = \tilde {C} _ {f} \tilde {x} _ {k} + \tilde {D} _ {f u} u _ {k} \tag {42}\tilde {f} _ {N} = \tilde {C} _ {f N} \tilde {x} _ {N}$$

where $\tilde { C } _ { f } = [ C _ { f } \ 0 ] , \ \tilde { C } _ { f N } = [ \hat { C } _ { f } \ 0 ]$ and $\tilde { D } _ { f u } = D _ { f u }$ . The upper bounds $\bar { f } _ { k }$ and $\bar { f } _ { N }$ remain the same.

It is noted that the minimum and maximum values of the PMSM electrical power $P , - { \underline { { P } } }$ (power flow to the PMSM) and

P (power flow from the PMSM), respectively,

$$- \underline {{P}} \leq P \leq \overline {{P}}, \tag {43}$$

are not possible to include as constraints in the RMPC synthesis because of the nonlinearity of such constraints. Instead, they are only imposed in the actuator control model (block ‘Single-link actuator’ in Fig. 5) in the simulation of the nonlinear multibody model, via the PMSM inner (current) and outer (velocity) control loops, by saturating the dq voltage references to the DC-AC converter of the PMSM and the magnetizing q current reference to the inner control loop, respectively, as explained in Sections 4.3 and 4.4 of [5].
