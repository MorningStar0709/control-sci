# 2.2 An Optimal Distributed $\mathcal { H } _ { \infty }$ Controller

For systems described by (1), it was shown in Lidström et al. (2017) that an optimal distributed $\mathcal { H } _ { \infty }$ controller is given by

$$K = B ^ {\top} (A - I) ^ {- 1}, \tag {6}$$

as long as the dynamics of each node $i \in \mathcal V$ satisfies the following condition

$$a _ {i} ^ {2} + 2 b ^ {2} d _ {i} < a _ {i}. \tag {7}$$

The condition (7) is related to the speed of information propagation through the network as well as its connectivity defined using the parameters $a _ { i } , b , d _ { i } ,$ , and the bound on the maximum eigenvalue of the symmetric normalized Laplacian matrix of the underlying network’s graph. More details on the local condition of nodes described by $( 7 )$ is available in subsection IV.B of Lidström et al. (2017). Further, (7) can be equivalently written using the compact notations as

$$A ^ {2} + B B ^ {\top} \prec A. \tag {8}$$

Since $a _ { i } \in ( 0 , 1 )$ , (1) is inherently stable. Then, it is best to quantify the amplification caused by just the disturbance to the system when a zero control is applied. We have the following lemma that computes the $\ell _ { 2 }$ gain of the subsystem $\pmb { \Sigma _ { \mathbf { i } } }$ with zero control inputs.

Lemma 1. Let $u _ { i } ( t ) = u _ { j } ( t )$ for all $j \in \mathcal N _ { i }$ corresponding to node $i \in \nu$ in (1). Then,

$$\left\| \boldsymbol {\Sigma} _ {\mathbf {i}} \right\| _ {\mathcal {H} _ {\infty}} \leq \frac {1}{1 - \left(\frac {1}{2} + \sqrt {\frac {1}{4} - 2 b ^ {2} d _ {i}}\right)}, \quad i f, \tag {9}\frac {1}{2} - \sqrt {\frac {1}{4} - 2 b ^ {2} d _ {i}} < a _ {i} < \frac {1}{2} + \sqrt {\frac {1}{4} - 2 b ^ {2} d _ {i}}, \quad a n d \tag {10}b < \sqrt {\frac {1}{8 \mathbf {d}}}. \tag {11}$$

Proof. We can equivalently rewrite (7) as

$$
\begin{array}{l} \left(a _ {i} - \frac {1}{2}\right) ^ {2} + 2 b ^ {2} d _ {i} - \frac {1}{4} <   0 \\ \Longleftrightarrow \frac {1}{2} - \sqrt {\frac {1}{4} - 2 b ^ {2} d _ {i}} <   a _ {i} <   \frac {1}{2} + \sqrt {\frac {1}{4} - 2 b ^ {2} d _ {i}}. \\ \end{array}
$$

Since $a _ { i } \in ( 0 , 1 )$ and $d _ { i } \geq 1 , \forall i \in \mathcal { V }$ , we see that

$$\frac {1}{4} - 2 b ^ {2} d _ {i} > 0 \iff b < \sqrt {\frac {1}{8 d _ {i}}}.$$

Since the above condition is true $\forall i \in \nu$ , we get (11). Note that (1) relates to a first-order stable system with zero control input for which its $\ell _ { 2 }$ gain is given by

$$\| \boldsymbol {\Sigma} _ {\mathbf {i}} \| _ {\mathcal {H} _ {\infty}} \leq \frac {1}{1 - \left(\frac {1}{2} + \sqrt {\frac {1}{4} - 2 b ^ {2} d _ {i}}\right)}.$$
