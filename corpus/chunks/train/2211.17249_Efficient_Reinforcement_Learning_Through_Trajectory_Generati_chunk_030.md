# 7.1. Fundemental Lemma

For state feedback control, ${ \pmb y } ( t ) = { \pmb x } ( t ) ( { \pmb C } = { \pmb I } _ { n } )$ and the system transition dynamics is reduced to $\pmb { x } ( k + 1 ) = \pmb { A } \pmb { x } ( k ) + \pmb { B } \pmb { u } ( k )$ . Expanding the dynamics for $k = 0 , \cdots , T$ gives the input-state response over $[ 0 , T - 1 ]$ as

$$
\left[ \begin{array}{l} \boldsymbol {u} _ {[ 0, T - 1 ]} \\ \boldsymbol {x} _ {[ 0, T - 1 ]} \end{array} \right] = \left[ \begin{array}{c c} \boldsymbol {I} _ {T m} & \mathbb {O} _ {T m \times n} \\ \mathcal {T} _ {[ 0, T - 1 ]} & \mathcal {O} _ {[ 0, T - 1 ]} \end{array} \right] \left[ \begin{array}{c} \boldsymbol {u} _ {[ 0, T - 1 ]} \\ \boldsymbol {x} (0) \end{array} \right], \tag {18}
$$

where $\mathcal { T } _ { [ 0 , T - 1 ] }$ and $\mathcal { O } _ { [ 0 , T - 1 ] }$ are the Toeplitz and observability matrices of order T represented as (De Persis and Tesi, 2019)

$$
\mathcal {T} _ {[ 0, T - 1 ]} := \left[ \begin{array}{c c c c c} \boldsymbol {B} & \mathbb {0} _ {n \times m} & \mathbb {0} _ {n \times m} & \dots & \mathbb {0} _ {n \times m} \\ \boldsymbol {A B} & \boldsymbol {B} & \mathbb {0} _ {n \times m} & \dots & \mathbb {0} _ {n \times m} \\ \vdots & \vdots & \vdots & \ddots & \vdots \\ \boldsymbol {A} ^ {T - 2} \boldsymbol {B} & \boldsymbol {A} ^ {T - 3} \boldsymbol {B} & \boldsymbol {A} ^ {T - 4} \boldsymbol {B} & \dots & \mathbb {0} _ {n \times m} \end{array} \right] \quad \mathcal {O} _ {[ 0, T - 1 ]} := \left[ \begin{array}{c} \boldsymbol {I} _ {n} \\ \boldsymbol {A} \\ \vdots \\ \boldsymbol {A} ^ {T - 1} \end{array} \right].
$$

On this basis, the Hankel Matrix H can be represented as

$$
\left[ \begin{array}{l} \boldsymbol {U} _ {0, T, L - T + 1} \\ \boldsymbol {X} _ {0, T, L - T + 1} \end{array} \right] = \left[ \begin{array}{c c} I _ {T m} & \mathbb {O} _ {T m \times n} \\ \mathcal {T} _ {[ 0, T - 1 ]} & \mathcal {O} _ {[ 0, T - 1 ]} \end{array} \right] \left[ \begin{array}{c} \boldsymbol {U} _ {0, T, L - T + 1} \\ X _ {0, L - T + 1} \end{array} \right], \tag {19}
$$

where $X _ { 0 , L - T + 1 } : = { \left\lceil \begin{array} { l l l l } { \mathbf { \Delta x } _ { d } ( 0 ) } & { \mathbf { \Delta x } _ { d } ( 1 ) } & { \dots } & { \mathbf { \Delta x } _ { d } ( L - T ) } \end{array} \right\rceil }$ .
