# 4 Statistical variables and performance metrics

To evaluate the predictions made by the deterministic and Bayesian models and compare them with the ground truth from the experiments, three error indices are employed: the average normalized mean square error (ANRMSE) [14], its time-resolved version (ε), and the normalized average minimum/maximum absolute error (NAMMAE) [14].

The ANRMSE quantifies the root mean square error between the predicted values $\tilde { \mathbf { x } } _ { t }$ and the measured (reference) values $\mathbf { x } _ { t }$ at different time steps, normalizing the result for each variable by k times the standard deviation of the measured value (k = 1 in this work), and averaging over the N variables in x:

$$\text { ANRMSE } = \frac {1}{N} \sum_ {i = 1} ^ {N} \sqrt {\frac {1}{\mathcal {T} (k S D [ x _ {i} ]) ^ {2}} \sum_ {j = 1} ^ {\mathcal {T}} \left(\tilde {x} _ {i} (t _ {j}) - x _ {i} (t _ {j})\right) ^ {2}}, \tag {19}$$

where $\tau$ is the number of considered time instants in the considered time window and $S D [ x _ { i } ]$ is the standard deviation of the measured values in the considered time window for the variable $x _ { i } { \mathrm { : } }$

$$S D [ \xi ] = \sqrt {\frac {1}{\mathcal {T} - 1} \sum_ {i = 1} ^ {\mathcal {T}} (\xi_ {i} - E V [ \xi ]) ^ {2}} \tag {20}$$

and

$$E V [ \xi ] = \frac {1}{\mathcal {T}} \sum_ {i = 1} ^ {\mathcal {T}} \xi_ {i} \tag {21}$$

For the same time window, the time-resolved variant of the ANRMSE called ε is evaluated as the average on the variables of the time evolution of the square root difference between the reference and predicted signal, normalized by its standard deviation in the considered time interval:

$$\varepsilon (t) = \frac {1}{N} \sum_ {i = 1} ^ {N} \sqrt {\frac {(\tilde {x} _ {i} (t) - x _ {i} (t)) ^ {2}}{(k S D [ x _ {i} ]) ^ {2}}} \tag {22}$$

This is used to monitor potential trends in the prediction error, i.e., whether the accuracy decreases for longer predictions.
