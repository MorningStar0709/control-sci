# C. Problem Under Consideration

The main objective of the control problem addressed in this paper is to develop a state feedback controller that ensures the system state remains within a predetermined safe set . Moreover, if possible, the control design aims to achieve asymptotic stability of the system’s origin despite the presence of $\delta .$ The safe set is defined by a continuously differentiable function $h ( \pmb { x } ) : \mathbb { R } ^ { n } $ R as follows:

$$\mathcal {C} = \{\boldsymbol {x} \in \mathbb {R} ^ {n} \mid h (\boldsymbol {x}) \geq 0 \}, \tag {13}$$

where it is assumed that ${ \textbf { 0 } } \in { \mathcal { C } } .$ . In the rest of this paper, In $( \mathcal { C } ) = \{ \pmb { x } \in \mathbb { R } ^ { n } \ | \ h ( \pmb { x } ) > 0 \}$ and $\partial \mathcal { C } = \{ \pmb { x } \in \mathbb { R } ^ { n } \ \vert \ h ( \pmb { x } ) =$ 0} are the interior and boundary of C, respectively. If for each initial condition $\pmb { x } ( 0 ) = \pmb { x } _ { 0 } \in \mathcal { C } , \pmb { x } ( t ) \in \mathcal { C }$ for all $t \geq 0 , \mathcal { C }$ is said to be forward invariant [14]. The unsafe set ¯ is assumed to be open and bounded. We also assume that the safety and stability requirements are compatible, meaning that within the safe set , it is possible to find a control input that satisfies both conditions [21].
