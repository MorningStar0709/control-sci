# A. Problem Setup

This work is concerned with nonlinear dynamical systems

$$\dot {x} = f (x) + F (x) \theta + g (x) u, x (0) = x _ {0}, t \geq 0, \tag {1}$$

with state $x \in \mathbb { R } ^ { n }$ , input $u \in \mathbb { R } ^ { m }$ and parameter $\theta \in \mathbb { R } ^ { p }$ , where $f : \mathbb { R } ^ { n } \to \mathbb { R } ^ { n } , g : \mathbb { R } ^ { n } \to \mathbb { R } ^ { n \times m } , F : \mathbb { R } ^ { n } \to \mathbb { R } ^ { n \times p }$ are continuously differentiable and $f ( 0 ) = 0 , F ( 0 ) = 0$ . For brevity, denote $\mathcal F ( x , u , \theta ) : = f ( x ) + F ( x ) \theta + g ( x ) u$ .

The control task motivating the later proposed approach is to find a control $u ( \cdot )$ that minimizes

$$J (x _ {0}, u (\cdot)) := \int_ {0} ^ {\infty} r (x (t), u (t)) \mathrm{d} t \tag {2}$$

along the solution $x ( \cdot )$ to (1) under $u ( \cdot )$ for any $x _ { 0 } .$ , where $r : \mathbb { R } ^ { n } \times \mathbb { R } ^ { m } \to \mathbb { R } _ { > 0 }$ is a continuous positive-definite $( \mathrm { p . d . } )$ running cost, i. e., satisfying $r ( 0 , 0 ) = 0$ and $r ( x , u ) \ > \ 0$ otherwise. Denote the optimal cost $J ^ { * } ( x ) : = J ( x , u ^ { * } ( \cdot ) )$ ) with the associated minimizing control policy $u ^ { * } ( \cdot ) .$ .

Under the Bellman’s principle of optimality, J∗ satisfies the Hamilton-Jacobi-Bellman (HJB) equation

$$0 = \min _ {u \in \mathbb {R} ^ {m}} r (x, u) + \nabla_ {x} ^ {\top} J ^ {*} (x) \mathcal {F} (x, u, \theta) \tag {3}$$

for all $x \in \mathbb { R } ^ { n }$ (assuming the minimum exists). Due to the curse of dimensions, finding $J ^ { * }$ analytically is challenging [25], though certain techniques exist to compute the latter approximately on $\mathbb { R } ^ { n }$ .

Herein, the parameter θ is assumed unknown which poses a challenge and will be addressed via adaptive control.
