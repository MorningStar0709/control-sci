3.4. Robust stability. A more detailed introduction to the ISS theory can be found in [9, 40].

Definition 5. The system (3.4) is said to be ISS if there exist $\beta \in \mathcal { K L }$ and $\iota \in { \mathcal { K } }$ such that

$$| x (t, x _ {0}, d) | \leq \beta (\| x _ {0} \|, t) + \iota (\| d \| _ {\infty})$$

for all $x _ { 0 } \in C _ { \eta } , d \in \mathcal { L } _ { \infty } ^ { m }$ and $t \geq 0$ .

If $d = 0$ , then we recover the conventional global asymptotic stability property [9].
