Lemma 1. If an interval $[ \kappa _ { 1 } , \kappa _ { 2 } ] \subset \mathbb { R } _ { > 0 }$ is LMI-certifiable, then every subinterval $\left[ \hat { \kappa } _ { 1 } , \hat { \kappa } _ { 2 } \right] ~ \subseteq ~ \left[ \kappa _ { 1 } , \kappa _ { 2 } \right]$ is also LMIcertifiable.

Proof. Let $P \succ 0$ certify $[ \kappa _ { 1 } , \kappa _ { 2 } ]$ . For any $\hat { \kappa } \in [ \kappa _ { 1 } , \kappa _ { 2 } ]$ , write $\hat { \kappa } = \mu \kappa _ { 1 } + ( 1 - \mu ) \kappa _ { 2 }$ , with $\mu \in [ 0 , 1 ]$ . Since $\tilde { A } ( \kappa )$ depends affinely on $\kappa ,$ ,

$$
\begin{array}{l} \tilde {A} (\hat {\kappa}) ^ {\top} P + P \tilde {A} (\hat {\kappa}) = \mu \left(\tilde {A} \left(\kappa_ {1}\right) ^ {\top} P + P \tilde {A} \left(\kappa_ {1}\right)\right) \\ + (1 - \mu) \big (\tilde {A} (\kappa_ {2}) ^ {\top} P + P \tilde {A} (\kappa_ {2}) \big), \\ \end{array}
$$

which is negative definite because $P$ certifies the endpoints. Hence the same $P$ certifies every subinterval. □

Lem. 1 implies that the endpoints ${ \underline { { \kappa } } } _ { \mathrm { L M I } }$ and $\overline { { \kappa } } _ { \mathrm { L M I } }$ can be computed by repeated feasibility tests on refined intervals using the LMIs in Prop. 5. In the next subsection, these LMIs are used directly as synthesis constraints to design controllers with prescribed gain robustness.
