$$
\begin{array}{l} \left[ \begin{array}{c} \boldsymbol {\mathcal {F}} \\ \boldsymbol {\tau} \end{array} \right] = \left[ \begin{array}{c c} m \mathbf {I} _ {3 \times 3} & \mathbf {0} _ {3 \times 3} \\ \mathbf {0} _ {3 \times 3} & \mathbf {M} (\boldsymbol {\Omega}) \end{array} \right] \left[ \begin{array}{c} \ddot {\boldsymbol {\eta}} \\ \ddot {\boldsymbol {\Omega}} \end{array} \right] \tag {1} \\ + \left[ \begin{array}{c c} \mathbf {0} _ {3 \times 3} & \mathbf {0} _ {3 \times 3} \\ \mathbf {0} _ {3 \times 3} & \mathbf {C} (\boldsymbol {\Omega}, \dot {\boldsymbol {\Omega}}) \end{array} \right] \left[ \begin{array}{c} \dot {\boldsymbol {\eta}} \\ \dot {\boldsymbol {\Omega}} \end{array} \right] + \left[ \begin{array}{c} m g \mathbf {e} _ {3} \\ 0 \end{array} \right], \\ \end{array}
$$

Here, M(Ω) and $\mathbf { C } ( \Omega , \dot { \Omega } )$ represent the rotational inertia and Coriolis matrices, respectively, and their structures are detailed in the appendix. Additionally, $\mathbf { e } _ { 3 }$ is a unit vector in the vertical direction.

The dynamic model can be rewritten in its more compact form as:

$$\bar {\mathbf {M}} (\mathbf {q}) \ddot {\mathbf {q}} + \bar {\mathbf {C}} (\mathbf {q}, \dot {\mathbf {q}}) \dot {\mathbf {q}} + \bar {\mathbf {G}} = \mathbf {T}, \tag {2}$$

where $\bar { \bf M } ( { \bf q } )$ is the generalized inertia matrix, $\bar { \bf C } ( { \bf q } , \dot { \bf q } )$ is the Coriolis generalized matrix, G¯ represents the gravitational forces, and ${ \bf T } = [ { \mathcal F } ^ { \mathrm { ~ \tiny ~ \tau ~ } } ] ^ { \mathrm { ~ \tiny ~ \mathsf ~ { ~ \tiny ~ \tau ~ } ~ } }$ corresponds to the applied forces and torques.
