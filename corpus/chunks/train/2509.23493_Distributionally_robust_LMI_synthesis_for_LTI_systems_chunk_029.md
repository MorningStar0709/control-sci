# VIII. SIMULATION RESULTS

To validate the proposed distributionally robust controller synthesis framework, we conduct simulations on a linearized wind turbine blade control system, inspired by [26], modeled as a discrete-time linear time-invariant (LTI) system. The system state is $x = ( \omega , h , \dot { h } , \phi , \dot { \phi } , \beta , \dot { \beta } ) ^ { \intercal }$ , where ω is the rotor speed (rad/s), h is the flapwise displacement (m), ϕ is the torsional angle (rad), and $\beta$ is the pitch angle (rad). The control input is u, representing the pitch rate command (rad/s), and the disturbance input is $\boldsymbol { w } = \left( v _ { x } , v _ { z } \right) ^ { \top }$ , representing wind speed components (m/s) in the axial and vertical directions, respectively. The performance outputs are $z = ( \omega , h , \phi , u ) ^ { \intercal }$ , and the measured outputs are $y ~ = ~ ( \omega , h , \phi ) ^ { \intercal }$ . The control objective is to minimize the root-mean-square (RMS) values of flapwise displacement (h, target RMS ≤ 2 m) and torsional angle (ϕ, target $\mathrm { R M S } ~ \le ~ 0 . 0 2$ rad) under stochastic wind turbulence.

The system dynamics are described by:

$$x (t + 1) = A x (t) + B _ {w} w (t) + B _ {u} u (t), \tag {20a}z (t) = C _ {z} x (t) + D _ {z w} w (t) + D _ {z u} u (t), \tag {20b}y (t) = C _ {y} x (t) + D _ {y w} w (t), \tag {20c}$$

where $A ~ \in ~ \mathbb { R } ^ { 7 \times 7 } , ~ B _ { w } ~ \in ~ \mathbb { R } ^ { 7 \times 2 } , ~ B _ { u } ~ \in ~ \mathbb { R } ^ { 7 \times 1 } , ~ C _ { z } ~ \in ~ \mathbb { R } ^ { 4 \times 7 } .$ , $D _ { z w } \ \in \ \mathbb { R } ^ { 4 \times 2 } , \ D _ { z u } \ \in \ \mathbb { R } ^ { 4 \times 1 } , \ C _ { y } \ \in \ \mathbb { R } ^ { 4 \times 7 }$ , and $D _ { y w } \ \in \ \mathbb { R } ^ { 4 \times 2 }$ are the system matrices, resulting from a continuoustime model which is discretized with a sampling time of $\Delta t { \it \Delta \Psi } = 0 . 0 5 { \it \Delta \Psi } { \it s } .$ . The disturbance $w ( t )$ is modeled as a zero-mean random process with nominal covariance $\Sigma _ { \mathrm { n o m } } ~ = ~ I _ { 2 }$ . Details, such as the exact specification of the system matrices, are provided in the accompanying repository https://github.com/SphinxDG/Distri-\ butionallyrobustLMIsynthesisforLTIsystems.
