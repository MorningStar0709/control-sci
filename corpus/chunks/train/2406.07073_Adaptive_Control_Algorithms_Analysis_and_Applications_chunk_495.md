# 10.7 A Robust Parameter Estimation Scheme

Figure 10.7 illustrates how the various modifications of the basic parameter adaptation algorithm discussed in this chapter can be integrated in the open-loop type parameter estimation schemes used for adaptive control. When using closed-loop output error parameter estimation schemes, the filtering effect is automatically present as a consequence of the structure of the adjustable predictor. Therefore, in general, the input/output data filtering stage indicated in Fig. 10.7 is no more necessary.
