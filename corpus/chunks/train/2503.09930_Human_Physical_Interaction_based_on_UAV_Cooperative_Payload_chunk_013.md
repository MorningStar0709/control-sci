where $\begin{array} { r l r } { X _ { d } } & { { } = } & { \left[ x _ { d } , y _ { d } , z _ { d } \right] ^ { \top } } \end{array}$ and $\begin{array} { r l r l } { K _ { p } } & { { } \in } & { \mathbb { R } ^ { 3 \times 3 } } & { { } = } \end{array}$ $d i a g ( k _ { p 1 } , k _ { p 2 } , k _ { p 3 } ) > 0$ . Based on (17), the origin $( E _ { p } = 0 )$ is globally asymptotically stable.
