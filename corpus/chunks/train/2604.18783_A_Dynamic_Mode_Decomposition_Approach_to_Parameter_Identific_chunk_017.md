Once the a system $\hat { F }$ is obtained, we solve for $\scriptstyle { \hat { \pi } } \ \mathbf { a s }$ in (4). The specific approach used in this paper uses a grid of candidate parameters in the set Π. Every point in this grid is used as an initial condition for the last p number of states, where p is the dimension of π. Then, the prediction of $\hat { F }$ is compared with the query dataset as in (4), and the grid point that minimizes the error metric is πˆ (see Algorithm 1).
