The (2, 2) block can be simplified using $A _ { F } = \hat { A } - \hat { Y } \hat { A } ^ { - \top } \hat { K } _ { x } ^ { \top } \hat { H } \hat { K } _ { x }$ and $\hat { Y } \hat { A } ^ { - \top } ~ = ~ A _ { F } \hat { Y }$ Combining these two expressions gives $A _ { F } ( I + \hat { Y } \hat { K } _ { x } ^ { \top } \hat { H } \hat { K } _ { x } ) = \hat { A }$ . Hence the lower right block is $A _ { F } ^ { - \top }$ . This can also be used to simplify the (1, 2) block as follows:

$$
\begin{array}{l} - \hat {A} ^ {- \top} \hat {K} _ {x} ^ {\top} \hat {H} \hat {K} _ {x} = - A _ {F} ^ {- \top} (I + \hat {K} _ {x} ^ {\top} \hat {H} \hat {K} _ {x} \hat {Y}) ^ {- 1} \hat {K} _ {x} ^ {\top} \hat {H} \hat {K} _ {x} \\ = - A _ {F} ^ {- \top} (I - \hat {K} _ {x} ^ {\top} \hat {W} ^ {- 1} \hat {K} _ {x} \hat {Y}) \hat {K} _ {x} ^ {\top} \hat {H} \hat {K} _ {x} = - A _ {F} ^ {- \top} \hat {K} _ {x} ^ {\top} \hat {W} ^ {- 1} \hat {K} _ {x}. \\ \end{array}
$$

The second equality makes use of the matrix inversion lemma.

Similar simplifications can be made after applying $T$ to the remaining state matrices of $\hat { F } ^ { \sim } \hat { F }$ Thus the coordinate transformation gives the following realization for $\hat { F } ^ { \sim } \hat { F }$ :

$$
\begin{array}{l} \left[ \begin{array}{c} \tilde {x} _ {t + 1} \\ \bar {x} _ {t + 1} \end{array} \right] = \left[ \begin{array}{c c} A _ {F} & 0 \\ - A _ {F} ^ {- \top} \hat {K} _ {x} ^ {\top} \hat {W} ^ {- 1} \hat {K} _ {x} & A _ {F} ^ {- \top} \end{array} \right] \left[ \begin{array}{c} \hat {x} _ {t} \\ \bar {x} _ {t} \end{array} \right] + \left[ \begin{array}{c} B _ {F} \\ - A _ {F} ^ {- \top} \hat {K} _ {x} ^ {\top} \hat {W} ^ {- 1} \end{array} \right] d _ {t} \\ \bar {d} _ {t} = \left[ (I - B _ {F} ^ {\top} A _ {F} ^ {- \top} \hat {K} _ {x} ^ {\top}) \hat {W} ^ {- 1} \hat {K} _ {x} \quad B _ {F} ^ {\top} A _ {F} ^ {- \top} \right] \left[ \begin{array}{c} \hat {x} _ {t} \\ \bar {x} _ {t} \end{array} \right] \\ + (I - B _ {F} ^ {\top} A _ {F} ^ {- \top} \hat {K} _ {x} ^ {\top}) \hat {W} ^ {- 1} d _ {t}. \\ \end{array}
$$

It can be shown that this is also a realization for $F ^ { \sim } F$ where F is the system in Equation (39). Hence we have shown that $\hat { P } ^ { \sim } \hat { P } = \hat { F } ^ { \sim } \hat { F } = F ^ { \sim } F$ .
