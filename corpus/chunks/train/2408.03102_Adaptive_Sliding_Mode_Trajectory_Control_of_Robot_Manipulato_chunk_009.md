# 3.2. Analysis and Control Design

For the design purpose that fulfills our objective, we define the following Lyapunov function candidate

$$V := \frac {1}{2} \eta^ {T} M (q) \eta + \frac {1}{2} \tilde {\phi} ^ {T} \Lambda^ {- 1} \tilde {\phi} \tag {4}$$

where $V \in \Re ; \ \tilde { \phi }$ is subsequent term that will be defined later; $\boldsymbol { \Lambda } \in \Re ^ { n \times n }$ is a diagonal matrix. Taking the time derivative of Eq. (4) gives

$$
\begin{array}{l} \dot {V} = \frac {1}{2} \dot {\eta} ^ {T} M (q) \eta + \frac {1}{2} \eta^ {T} \dot {M} (q) \eta + \frac {1}{2} \eta^ {T} M (q) \dot {\eta} \tag {5} \\ + \frac {1}{2} \dot {\tilde {\phi}} ^ {T} \Lambda^ {- 1} \tilde {\phi} + \frac {1}{2} \tilde {\phi} ^ {T} \Lambda^ {- 1} \dot {\tilde {\phi}}. \\ \end{array}
$$

Since all parts in the above equation are scalar quantities, we can take the transpose of $\begin{array} { l } { \frac { 1 } { 2 } { \dot { \eta } } ^ { T } M ( q ) \eta } \end{array}$ and ${ \scriptstyle \frac { 1 } { 2 } } \dot { \tilde { \phi } ^ { T } } \Lambda ^ { - 1 } \tilde { \phi }$ which simplifies Eq. (5) to be

$$\dot {V} = \frac {1}{2} \eta^ {T} \dot {M} (q) \eta + \eta^ {T} M (q) \dot {\eta} + \tilde {\phi} ^ {T} \Lambda^ {- 1} \dot {\tilde {\phi}}. \tag {6}$$

Utilizing the term $M ( q ) \dot { \eta }$ in Eq. (6), we take the derivative of (3) and multiply it by $M ( q )$ to obtain

$$M (q) \dot {\eta} = M (q) \Delta \ddot {q} + \sigma M (q) \Delta \dot {q}. \tag {7}$$

Equation (7) can be extended by substituting $\Delta \ddot { q } = \ddot { q } _ { d } - \ddot { q }$ to obtain

$$M (q) \dot {\eta} = M (q) \ddot {q} _ {d} - M (q) \ddot {q} + \sigma M (q) \Delta \dot {q}. \tag {8}$$

Using $M ( q ) \ddot { q } = \tau - \tau _ { v } - \tau _ { l } - V _ { m } ( q , \dot { q } ) \dot { q } - G ( q )$ from Eq. (1), we can write Eq. (8) as

$$
\begin{array}{l} M (q) \dot {\eta} = M (q) \ddot {q} _ {d} - \tau + \tau_ {v} + \tau_ {l} \\ + V _ {m} (q, \dot {q}) \dot {q} + G (q) \tag {9} \\ + \sigma M (q) \Delta \dot {q}. \\ \end{array}
$$

Since $\dot { \boldsymbol { q } } = \dot { \boldsymbol { q } } _ { d } - \Delta \dot { \boldsymbol { q } }$ and $\Delta \dot { q } = \eta - \sigma \Delta q$ , we will replace ̇ so we rewrite Eq. (9) as

$$M (q) \dot {\eta} = M (q) \ddot {q} _ {d} - \tau + \tau_ {v} + \tau_ {l}+ V _ {m} (q, \dot {q}) [ \dot {q} _ {d} - \eta + \sigma \Delta q ] \tag {10}+ G (q) + \sigma M (q) \Delta \dot {q}.$$

We define

$$Y \phi := V _ {m} (q, \dot {q}) \dot {q} _ {d} + V _ {m} (q, \dot {q}) \sigma \Delta q+ \sigma M (q) \Delta \dot {q} + G (q) \tag {11}+ M (q) \ddot {q} _ {d}$$
