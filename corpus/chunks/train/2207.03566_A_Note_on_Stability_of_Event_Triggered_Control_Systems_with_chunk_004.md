# II. Problem Formulation

To make this note self-contained, we adopt the notations from [1] and briefly introduce the event-triggered control problem formulated in [1] in this section.

Denote by N the set of positive integers, R the set of real numbers, $\mathbb { R } ^ { + }$ the set of nonnegative reals, and ${ \mathbb { R } } ^ { n }$ the n-dimensional real space equipped with the Euclidean norm denoted by $\| \cdot \| .$ For $a , b \in \mathbb { R }$ with $b > a ,$ we define

$$
\begin{array}{l} \mathcal {P C} ([ a, b ], \mathbb {R} ^ {n}) = \{\phi : [ a, b ] \rightarrow \mathbb {R} ^ {n} \mid \phi \text {   is   piecewise   right- } \\ \text { continuous } \} \\ \end{array}
\mathcal {P C} ([ a, \infty), \mathbb {R} ^ {n}) = \{\phi : [ a, \infty) \rightarrow \mathbb {R} ^ {n} | \phi | _ {[ a, c ]} \in \mathcal {P C} ([ a, c ], \mathbb {R} ^ {n})\text { for all } c > a \}$$

where $\phi | _ { [ a , c ] }$ is a restriction of $\phi$ on interval $[ a , c ] .$ . Let $C ( J , \mathbb { R } ^ { n } )$ denote the set of continuous functions from interval J to ${ \mathbb { R } } ^ { n }$ . Given $\tau > 0$ , the linear space $C ( [ - \tau , 0 ] , \mathbb { R } ^ { n } )$ is equipped with the supremum norm $\| \phi \| _ { \tau } : = \operatorname* { s u p } _ { s \in [ - \tau , 0 ] } \| \phi ( s ) \|$ for $\phi \in C ( [ - \tau , 0 ] , \mathbb { R } ^ { n } )$ ). A function $\alpha : \mathbb { R } ^ { + }  \mathbb { R }$ is said to be of class K and we write $\alpha \in \mathcal { K }$ , if  is continuous, strictly increasing, and satisfies $\alpha ( 0 ) = 0$ If $\alpha \in \mathcal K$ and also $\alpha ( s ) \to \infty { \mathrm { ~ a s ~ } } s \to \infty$ , we say that $\alpha$ is of class $\mathcal { K } _ { \infty }$ and we write $\alpha \in \mathcal { K } _ { \infty }$ A continuous function $\beta : \mathbb { R } ^ { + } \times \mathbb { R } ^ { + } \to \mathbb { R } ^ { + }$ is said to be of class $\mathcal { K L }$ and we write $\beta \in { \mathcal { K L } } .$ , if the function $\beta ( \cdot , t )$ is of class K for each fixed $t \in \mathbb { R } ^ { + }$ , and the function $\beta ( s , \cdot )$ is decreasing and $\beta ( s , t )  0$ as $t \to \infty$ for each fixed $s \in \mathbb { R } ^ { + }$ .
