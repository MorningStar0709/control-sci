In subsequent two sections, we prove some properties of $V ^ { * }$ . The first one is the symmetry property.

3.3.1. Symmetry of $V ^ { * }$ . In this section, we prove that $V ^ { * }$ is symmetric in the variables $( x _ { 2 } , \ldots , x _ { N } )$ . To this end, we define

$$C _ {\text { sym }} (\mathsf {X} ^ {N}) := \left\{V \in C _ {b} (\mathsf {X} ^ {N}): V \text { is symmetric in } (x _ {2}, \ldots , x _ {N}) \right\}$$

We also define the optimality operator $T$ as follows

$$T V (\mathbf {x}) := \min _ {a \in \mathsf {A}} \left[ C (\mathbf {x}, a) + \beta \int_ {\mathsf {X} ^ {N}} V (\mathbf {y}) P (d \mathbf {y} | \mathbf {x}, a) \right]$$

for all $\mathbf { x } \in \mathsf { X } ^ { N }$ . Note that $T$ is a β-contraction on $C _ { b } ( \mathsf { X } ^ { N } )$ with an unique fixed point $V ^ { * }$ .

LEMMA 3.2. T maps $C _ { \mathrm { s y m } } ( \mathsf { X } ^ { N } )$ into itself.

$$T V (\mathbf {x}) = \min _ {a \in \mathsf {A}} \left[ c (x _ {1}, a, \mathsf {e} [ \cdot | \mathbf {x} ]) + \beta \int_ {\mathsf {X} ^ {N}} V (\mathbf {y}) \prod_ {i = 2} ^ {N} p (d y _ {i} | x _ {i}, \pi^ {*} (x _ {i}), \mathsf {e} [ \cdot | \mathbf {x} ]) \bigotimes p (d y _ {1} | x _ {1}, a, \mathsf {e} [ \cdot | \mathbf {x} ]) \right]$$
