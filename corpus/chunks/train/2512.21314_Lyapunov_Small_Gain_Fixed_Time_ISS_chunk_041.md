$$| \overline {{{G}}} (\tilde {x}, \tilde {u}) | \leq K | \tilde {x} - (h (\tilde {u} + u ^ {*}) - h (u ^ {*})) |\leq K (| \tilde {x} | + \ell | \tilde {u} |)\leq K \left(\frac {1}{\delta \sqrt {2 \mu}} + \frac {\ell}{\mu}\right) | \nabla P (\tilde {u} + u ^ {*}) |= \frac {K (\ell + \varepsilon + \gamma)}{\mu} | G (\tilde {u}) |. \tag {49}$$

Note that $\begin{array} { r } { \frac { K ( \ell + \varepsilon + \gamma ) } { \mu } < \frac { 1 } { N } } \end{array}$ , hence by Lemma 12 in the Appendix there exists $c > 0$ such that:

$$\dot {W} (\tilde {u}) = - \sum_ {i = 1} ^ {N} G _ {i} (\tilde {u}) \mathcal {F} _ {\xi} (G _ {i} (\tilde {u}) + \overline {{G}} _ {i} (\tilde {x}, \tilde {u}))\leq - c | G (\tilde {u}) | ^ {2 - \xi_ {1}} - c | G (\tilde {u}) | ^ {2 - \xi_ {2}}\leq - c (2 \mu) ^ {1 - \frac {1}{2} \xi_ {1}} W ^ {1 - \frac {1}{2} \xi_ {1}} (\tilde {u}) - c (2 \mu) ^ {1 - \frac {1}{2} \xi_ {2}} W ^ {1 - \frac {1}{2} \xi_ {2}} (\tilde {u}),$$

where the bottom inequality uses the following fact

$$| \nabla P (\tilde {u} + u ^ {*}) | \geq \sqrt {2 \mu} W ^ {\frac {1}{2}} (\tilde {u}),$$

which follows directly from (48b). Since

$$\delta^ {2} \tilde {\gamma} ^ {2} = \left(\frac {\gamma}{\gamma + \varepsilon}\right) ^ {2} < 1,$$

we can find $\hat { \delta } > 0$ such that $( \delta + { \hat { \delta } } ) ( \tilde { \gamma } + { \hat { \delta } } ) < 1$ . By choosing $\hat { \gamma } _ { 1 } ( s ) = ( \tilde { \gamma } + \hat { \delta } ) ^ { 2 } s$ and $\hat { \gamma } _ { 2 } ( s ) = ( \delta + \hat { \delta } ) ^ { 2 } s .$ , we observe that the small gain condition of Theorem 1 is satisfied. This establishes the result.

Remark 8: The condition (46) is highly reminiscent of condition (30), but we now require the lower bound on $\mu$ to be proportional to the number of players in the game. Hence, as the player count increases, u needs to be weighted much more heavily than x in the pseudogradient $\mathcal { G } ( x , u )$ . This is unsurprising, since we are primarily concerned with players implementing the decoupled pseudogradient flow (43), where it is typically the case that the dimension should be accounted for [8]. Meanwhile, if the players choose to implement a centralized fixed-time pseudogradient flow, then the problem can be studied by emulating the results from Section IV. □
