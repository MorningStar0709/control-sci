Note that the generated initial state is given by $\tilde { \pmb { x } } ( 0 ) = \left[ \begin{array} { l } { \mathcal { H } _ { \ b { x } } ^ { 0 } } \end{array} \right] \pmb { g } .$ Combining with (9) gives

$$
\underbrace {\left[ \begin{array}{c} \mathcal {H} _ {u} - (\boldsymbol {I} _ {T} \otimes \boldsymbol {\theta}) \mathcal {H} _ {x} \\ \mathcal {H} _ {x} ^ {0} \end{array} \right]} _ {\boldsymbol {G} _ {\boldsymbol {\theta}}} \boldsymbol {g} = \left[ \begin{array}{c} \boldsymbol {w} (0) \\ \vdots \\ \boldsymbol {w} (T - 1) \\ \boldsymbol {x} (0) \end{array} \right]. \tag {10}
$$

Note that the matrix $G _ { \pmb { \theta } } \in \mathbb { R } ^ { ( T m + n ) \times ( L - T + 1 ) }$ is not a square matrix and its rank is determined by the length of historic trajectory L. When the trajectory is sufficiently long and rank $( { \mathcal { H } } ) =$ $n + T m$ , we have $L - T + 1 > T m + n$ and thus there might be multiple $\pmb { g }$ where (10) holds. We use the minimum-norm solution of (10) given by

$$
\boldsymbol {g} ^ {*} = \boldsymbol {G} _ {\boldsymbol {\theta}} ^ {\top} \left(\boldsymbol {G} _ {\boldsymbol {\theta}} \boldsymbol {G} _ {\boldsymbol {\theta}} ^ {\top}\right) ^ {- 1} \left[ \begin{array}{l} \boldsymbol {w} (0); \dots ; \boldsymbol {w} (T - 1); \boldsymbol {x} (0) \end{array} \right]. \tag {11}
$$

In the next Theorem, we prove the existence and uniqueness of the trajectory generated by (10). The goal is to show that given $( { \pmb w } ( 0 ) , \cdot \cdot \cdot , { \pmb w } ( T - 1 ) , { \pmb x } ( 0 ) )$ ), any $\pmb { g }$ that satisfies (10) will generate the same trajectory using $\mathcal { H } g$ . So it is suffice to choose the closed-form solution in (11).
