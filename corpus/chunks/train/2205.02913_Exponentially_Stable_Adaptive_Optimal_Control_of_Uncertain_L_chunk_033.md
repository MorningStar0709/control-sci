# 3.4.2 Second group

First of all, to provide the finite excitation of the initial regressor $\Psi ( t ) \in \mathrm { F E }$ , the reference signal ?? for the proposed adaptive control system is to be different from zero $r \neq 0 _ { m }$ . This condition follows from the fact that, when $r = 0 _ { m }$ , then:

$$\exists \ell_ {i} \neq 0 \sum_ {i = 1} ^ {m + n} \ell_ {i} \Psi_ {i} (t) = \sum_ {i = 1} ^ {n} \ell_ {i} x _ {i} (t) + \sum_ {i = 1} ^ {m} \ell_ {i} u _ {i} (t) = \sum_ {i = 1} ^ {n} \ell_ {i} x _ {i} (t) + \sum_ {i = 1} ^ {m} \ell_ {i} k _ {x} ^ {i} x (t) = 0, K _ {x} ^ {\mathrm{T}} = \left[ k _ {x} ^ {1} k _ {x} ^ {2} \dots k _ {x} ^ {m} \right], \tag {34}$$

where $\Psi _ { i } \left( t \right)$ is the $i ^ { \mathrm { { t h } } }$ element of the regressor $\Psi ( t ) , \ell _ { i } \neq 0$ is an arbitrary constant. In accordance with the results $\mathrm { o f } ^ { 4 3 }$ , Proposition 1, the linear dependence beetween regressor elements (34) is the sufficient condition for $\boldsymbol { \Psi } \left( t \right) \not \in \mathrm { F E }$ .

Therefore, the choice of a nonzero reference value ?? for the proposed adaptive control system is a condition to propagate the excitation of the regressor through the parameterizations (23), (25), and (31). It should be noted that $r \neq 0 _ { m }$ is a necessary but not sufficient condition for $\Psi ( t ) \in \mathrm { F E } .$ , since, even when $r \neq 0 _ { m }$ , it holds that $\Psi ( t ) \notin$ FE if $\exists \ell _ { i } > 0$ such that:

$$
\begin{array}{c} \sum_ {i = 1} ^ {m + n} \ell_ {i} \Psi_ {i} (t) = \sum_ {i = 1} ^ {n} \ell_ {i} x _ {i} (t) + \sum_ {i = 1} ^ {m} \ell_ {i} u _ {i} (t) = \sum_ {i = 1} ^ {n} \ell_ {i} x _ {i} (t) + \sum_ {i = 1} ^ {m} \ell_ {i} k _ {x} ^ {i} x (t) + \sum_ {i = 1} ^ {m} \ell_ {i} k _ {r} ^ {i} r = - \sum_ {i = 1} ^ {m} \ell_ {i} k _ {r} ^ {i} r + \sum_ {i = 1} ^ {m} \ell_ {i} k _ {r} ^ {i} r = 0, \\ K _ {r} = \left[ k _ {r} ^ {1} k _ {r} ^ {2} \dots k _ {r} ^ {m} \right] \end{array} \tag {35}
$$

In this case, the requirement $\Psi \left( t \right) \in \mathrm { F E }$ seems possible to be met by addition of a noise or harmonic component to the control signal $u \left( t \right)$ in accordance with the results $\mathrm { o f } ^ { 4 4 }$ .
