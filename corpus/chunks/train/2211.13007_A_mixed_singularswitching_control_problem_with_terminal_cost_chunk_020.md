From here and using (H3) and (2.5), it gives

$$
\begin{array}{l} 0 \leq \operatorname{tr} \left[ a _ {\iota_ {\circ}} \left(x _ {\circ}\right) \mathrm{D} ^ {2} w _ {\ell_ {\circ}, \iota_ {\circ}} \left(x _ {\circ}\right) \right] \\ = c _ {\iota_ {\circ}} (x _ {\circ}) w _ {\ell_ {\circ}, \iota_ {\circ}} (x _ {\circ}) - \varrho h _ {\iota_ {\circ}} (x _ {\circ}) + \varrho \Xi_ {\ell_ {\circ}, \iota_ {\circ}} w (x _ {\circ}) \leq c _ {\iota_ {\circ}} (x _ {\circ}) w _ {\ell_ {\circ}, \iota_ {\circ}} (x _ {\circ}). \\ \end{array}
$$

Since $c _ { \iota _ { \circ } } > 0$ on ${ \overline { { \mathcal { O } } } } ,$ it follows that $w _ { \ell _ { \circ } , \iota _ { \circ } } ( x _ { \circ } ) \geq 0$ . Therefore $w _ { \ell , \iota } ( x ) \geq w _ { \ell _ { \circ } , \iota _ { \circ } } ( x _ { \circ } ) \geq 0$ , for all $x \in { \overline { { \mathcal { O } } } }$ and $( \ell , \iota ) \in \mathbb { M } \times \mathbb { I }$ . Supposing now $( x ^ { \circ } , \ell ^ { \circ } , \iota ^ { \circ } ) \in \overline { { \mathcal { O } } } \times \mathbb { M } \times \mathbb { I }$ such that

$$w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) = \max _ {(x, \ell , \iota) \in \overline {{\mathcal {O}}} \times \mathbb {M} \times \mathbb {I}} w _ {\ell , \iota} (x),$$

taking into account that

$$
\begin{array}{l} \mathrm{D} ^ {1} w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) = 0, \operatorname{tr} [ a _ {\iota^ {\circ}} (x ^ {\circ}) \mathrm{D} ^ {2} w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) ] \leq 0, \\ w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) - w _ {\ell^ {\circ}, \kappa} (x ^ {\circ}) \geq 0 \text {for} \kappa \in \mathbb {I} \setminus \{\iota^ {\circ} \}, \\ w _ {\ell^ {\circ}, \iota^ {\circ}} (x ^ {\circ}) - w _ {\ell^ {\prime}, \iota^ {\circ}} (x ^ {\circ}) \geq 0 \text {for} \ell^ {\prime} \in \mathbb {M} \setminus \{\ell^ {\circ} \}. \\ \end{array}
$$

and arguing in a similar way as before, the reader can easily see that (2.8) is true. With this remark, we conclude the proof.
