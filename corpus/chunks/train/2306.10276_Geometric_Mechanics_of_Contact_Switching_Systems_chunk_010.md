# C. Contact-switching Interpolation Extension

In order to leverage existing tools to analyze systems with stratified kinematics, we introduce a single, switching-mode (or hybrid) local connection $( \tilde { A } _ { i  j } )$ which combines multiple distinct local connections $( A _ { i }$ and $A _ { j } )$ . To accomplish this in our case, we introduce a discrete contact shape variable, $\beta \in$ $\{ - 1 , 1 \} \cap \mathbb { Z }$ and an interpolated contact shape, $\tilde { \beta } \in [ - 1 , 1 ] \cap$ R to smoothly interpolate between the two vector fields. To complete the formulation, we define a continuous version, $, ^ { 2 } \tilde { c _ { i } } .$ , of our previous defined contact states, $c _ { i } .$ , for each leg (shown in Fig. 3a) as:

$$\tilde {c} _ {1} (\tilde {\beta}) = \frac {1 + \cos \frac {\pi}{2} (\tilde {\beta} + 1)}{2}, \tilde {c} _ {2} (\tilde {\beta}) = \frac {1 - \cos \frac {\pi}{2} (\tilde {\beta} + 1)}{2} \tag {5}$$

As the next step, we define the switching-mode local connection, $\tilde { A } _ { 1  2 }$ (shorthanded hereafter as $\tilde { A } )$ as:

$$\tilde {\boldsymbol {A}} _ {1 \leftrightarrow 2} (\alpha , \tilde {\beta}) = \tilde {c _ {1}} (\tilde {\beta}) \boldsymbol {A} _ {1} (\alpha) + \tilde {c _ {2}} (\tilde {\beta}) \boldsymbol {A} _ {2} (\alpha) \tag {6}$$

The discrete version of the switching-mode connection is a piecewise function (either $A _ { 1 }$ or $\pmb { A } _ { 2 } )$ based on the discrete contact shape (β) as highlighted in Fig. 3(b).
