# Notation

The sets of integer, real, and complex numbers are ℤ, ℝ, and ℂ, respectively, with subsets $\mathbb { N } _ { \nu } : = \{ i \in \mathbb { Z } \mid 1 \leq i \leq \nu \}$ , $\mathbb { C } _ { 0 } : = \left\{ s \in \mathbb { C } \ \lvert \ \mathrm { R e } s > 0 \right\}$ , and $\bar { \mathbb { C } } _ { 0 } : = \left\{ s \in \mathbb { C } \ \lvert \ \mathrm { R e } \ s \geq 0 \right\}$ . By $I _ { \nu }$ and $\mathbb { 1 } _ { \nu }$ we denote the $\nu \times \nu$ identity matrix and 𝜈-dimensional vector of ones, respectively. When the dimension is immaterial or clear from context, we use 𝐼 and 𝟙. The complex-conjugate transpose of a matrix 𝐴 is denoted by $A ^ { \top }$ , the set of all its eigenvalues by spec( 𝐴), and its minimal singular value by $\underline { { \sigma } } ( A )$ . The notation diag $\left\{ A _ { i } \right\}$ stands for a block-diagonal matrix with diagonal elements $A _ { i } .$ . The image (range) and kernel (null) spaces of a matrix 𝐴 are notated Im 𝐴 and ker 𝐴, respectively. Given two matrices 𝐴 and 𝐵, 𝐴 ⊗ 𝐵 denotes their Kronecker product.

By the stability of a system 𝐺 we understand its 𝐿2-stability. It is known (Curtain and Zwart, 2020, §A.6.3) that a 𝑝 × 𝑚 LTI system is causal and stable iff its transfer function $G ( s )$ belongs to $H _ { \infty } ^ { p \times m }$ , which is the space of holomorphic and bounded functions $\mathbb { C } _ { 0 } \to \mathbb { C } ^ { p \times m }$ (we write $H _ { \infty }$ when the dimensions are clear). Given a real-rational transfer function $G ( s )$ , its McMillan degree is denoted by deg(𝐺). By nrank $G ( s )$ we understand the normal rank of a function $G ( s )$ .

A digraph $\mathcal { G } = ( \mathcal { V } , \mathcal { E } )$ consists of a vertex set $_ \mathrm { ~  ~ }$ and an edge set $\mathcal { E } \subset \mathcal { V } \times \mathcal { V }$ , see (Godsil and Royle, 2001) for more details. The (oriented) incidence matrix of $\mathcal { G }$ is denoted by $E _ { \mathcal { G } }$ or simply 𝐸 when the association with a concrete graph is clear. It is a $\lvert \mathcal { V } \rvert \times \lvert \mathcal { E } \rvert$ matrix, whose (𝑖, 𝑗 ) entry is

$$
[ E _ {\mathcal {G}} ] _ {i j} = \left\{ \begin{array}{l l} 1 & \text { if   vertex   } i \text {   is   the   head   of   edge   } j \\ - 1 & \text { if   vertex   } i \text {   is   the   tail   of   edge   } j \\ 0 & \text { if   vertex   } i \text {   does   not   belong   to   edge   } j \end{array} \right..
$$
