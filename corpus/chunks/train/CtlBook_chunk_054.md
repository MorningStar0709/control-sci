# 2.2.3 Mechanical Network Schematic Diagram

We must usually simplify the mechanical system into a purely translational one to apply the analyis of this chapter. To do so, we identify the mass for each moving part and draw it as a box labeled $M _ { i } \ ( \mathrm { F i g u r e \ 2 . 1 } )$ . What each box actually represents is a point mass. Each point mass has a displacement, xi which indicates its location along the axis of linear motion.

Springs and dampers are then connected between the moving parts. Alternatively, one end of a spring or damper may be connected to ground (a point at which $x = { \dot { x } } = { \ddot { x } } = 0 )$ .
