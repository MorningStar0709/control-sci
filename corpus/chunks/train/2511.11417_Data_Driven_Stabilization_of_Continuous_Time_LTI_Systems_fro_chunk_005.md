# Assumption 1. The pair $( A , B )$ is controllable.

Suppose that an experiment is performed on system (3) over the time interval [0, T ], with $T \ > \ 0$ . Specifically, given an unknown initial condition $x ( 0 ) \in \mathbb { R } ^ { n p }$ , an input trajectory $u ( \cdot ) : [ 0 , T ]  \mathbb { R } ^ { m }$ , and unknown noise signals $w ( \cdot ) : [ 0 , T ]  \mathbb { R } ^ { q } , v ( \cdot ) : [ 0 , T ]  \mathbb { R } ^ { p }$ , let the corresponding state and output trajectories be $x ( \cdot ) : [ 0 , T ] \to \mathbb { R } ^ { n p }$ and $y ( \cdot ) \ : \ [ 0 , T ] \ \to \ \mathbb { R } ^ { p }$ , with $x ( \cdot )$ an absolutely continuous function. Then, we suppose that the following continuoustime dataset is collected during the experiment:

$$\{(t, u (t), y (t)): t \in [ 0, T ] \}. \tag {4}$$

Our goal is to use (4) to design a stabilizing controller for system (1) (equivalently, system (3)) without any prior knowledge of the matrices $A _ { i } , B _ { i } , i \in \{ 1 , . . . , n - 1 \}$ except for the order n. Specifically, we aim to compute the matrices $A _ { \mathrm { c } } , B _ { \mathrm { c } } , C _ { \mathrm { c } } ,$ , and $D _ { \mathrm { c } }$ of a dynamic controller of the form

$$\dot {x} _ {\mathrm{c}} = A _ {\mathrm{c}} x _ {\mathrm{c}} + B _ {\mathrm{c}} y \tag {5}u = C _ {\mathrm{c}} x _ {\mathrm{c}} + D _ {\mathrm{c}} y,$$

with $x _ { \mathrm { c } } \in \mathbb { R } ^ { \mu }$ the controller state, so that the state matrix of the closed-loop interconnection of (3) and (5)

$$
\left[ \begin{array}{c c} A + B D _ {\mathrm{c}} C & B C _ {\mathrm{c}} \\ B _ {\mathrm{c}} C & A _ {\mathrm{c}} \end{array} \right] \tag {6}
$$

is Hurwitz.
