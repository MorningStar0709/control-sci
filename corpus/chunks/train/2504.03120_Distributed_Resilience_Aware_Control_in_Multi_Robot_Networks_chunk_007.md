In [3], the W-MSR (Weighted-Mean Subsequent Reduced) algorithm was introduced to guarantees resilient consensus. A normal robot $i \in \mathcal L$ performs W-MSR algorithm with parameter $F$ at every time step $t _ { p }$ as follows:

1) Robot i collects values $y _ { j } ( t _ { p } )$ from its neighbors $j \in$ $\mathcal { N } _ { i } ( t _ { p } )$ and sorts them.   
2) If there are at least F strictly larger values than its own $y _ { i } ( t _ { p } )$ , it discards the largest $F ;$ otherwise, it removes

all larger values. Similarly, it does the same for values less than $y _ { i } ( t _ { p } )$ .

3) Let $\mathcal { R } _ { i } ( t _ { p } )$ be the set of robots whose values are discarded in Step 2. Then, it applies

$$y _ {i} (t _ {p + 1}) = \sum_ {j \in \mathcal {B} _ {i} (t _ {p}) \backslash \mathcal {R} _ {i} (t _ {p})} w _ {i j} (t _ {p}) y _ {j} (t _ {p}), \tag {7}$$

where $w _ { i j } ( t _ { p } )$ satisfies the same conditions stated for the nominal consensus update protocol (6).
