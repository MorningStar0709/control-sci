A hydraulic actuator generates the force based on a control input u. The nominal hydraulic actuator model is first-order with a time constant of ${ \frac { 1 } { 6 0 } } \sec$ , i.e. $\begin{array} { r } { A _ { 0 } ( s ) = \frac { 6 0 } { s + 6 0 } } \end{array}$ . The maximum actuator displacement is 0.05m. This places a constraint on the suspension displacement to be less than 0.05m. The actuator dynamics are assumed to uncertain. This is modeled with an input

multiplicative uncertainty:

$$A (s) = A _ {0} (s) \cdot (1 + W _ {u n c} (s) \Delta (s)),$$

where $\begin{array} { r } { W _ { u n c } ( s ) = \frac { 3 s + 1 8 . 5 } { s + 4 6 . 3 } } \end{array}$ and $\Delta ( s )$ is a stable, LTI system satisfying $\| \Delta \| _ { \infty } \leq 1$ . The weight $W _ { u n c } ( s )$ has DC gain of 0.4, $| W _ { u n c } ( j \infty ) | = 3$ , and a zero near $- 6 . 2 \mathrm { r a d / s e c }$ . This represents a relatively small uncertainty (40%) uncertainty at low frequencies with increasing uncertainty above 6.2rad/sec. The continuous-time quarter car and hydraulic actuator models are discretized using zero-order hold and sample time $T _ { s } = 0 . 0 0 2 \ { \mathrm { s e c } } .$ ||

Figure 11 shows the interconnection for the control design. There are three disturbances associated with the road input and measurement noise. Each of these has a constant weights that (roughly) model the amplitudes of the corresponding input disturbances: $W _ { r o a d } = 0 . 0 7$ , $W _ { d _ { 2 } } ~ = ~ 0 . 0 1$ , and $W _ { d _ { 3 } } ~ = ~ 0 . 5$ . There are three generalized errors that model the competing objectives of actuator commands, body acceleration, and suspension travel. The corresponding performance weights are:

$$W _ {a c t} (s) = 0. 8 \frac {s + 5 0}{s + 5 0 0},W _ {s _ {d}} (s) = \frac {0 . 0 0 6 2 5 s + 0 . 5}{0 . 0 0 5 s + 0 . 0 4},W _ {a _ {b}} (s) = \frac {0 . 0 3 s + 4 . 5}{8 s + 3 . 6}.$$

These weights are also discretized using a zero-order hold with $T _ { s } = 0 . 0 0 2 \mathrm { s e c }$ . These weights correspond to a “balanced” objective between comfort and handling. Additional details on the design objectives can be found in [35], [36].
