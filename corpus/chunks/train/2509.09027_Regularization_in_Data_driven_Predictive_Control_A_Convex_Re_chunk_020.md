The reasoning above critically relies on the conceptual optimality constraint $H \in { \mathcal { C } } _ { \mathrm { o p t } }$ , associated with the inner problem (7b). Dropping any constraint in (7b) results in a different optimality set, meaning that the corresponding penalized single-level problem may no longer offer a valid relaxation. This subtle but important point may be easily overlooked; for example, some relaxation statements in [1] are not fully accurate without further clarification. We consider a simple example below.

Example 1 Consider a simple bi-level formulation

$$\min _ {x \in \mathbb {R}} x ^ {2}$$

$\mathrm { s u b j e c t ~ t o } \quad y ^ { * } x = 1 ,$ (14a)

$$\text { where } y ^ {*} \in \arg \min _ {y \in \mathbb {R}} y ^ {2} \tag {14b}$$

$\mathrm { s u b j e c t ~ t o ~ 2 } \leq y \leq 4 .$ (14c)

This inner problem has an optimality condition as $y ^ { \ast } \in$ $\mathcal { C } _ { \mathrm { o p t } } : = \{ 2 \bar  \}$ . We consider a penalty function $p ( y ) =$ $| y - 2 |$ , which satisfies (12). Then, a penalized single-level optimization is

$$\min _ {x \in \mathbb {R}, y \in \mathbb {R}} x ^ {2} + \lambda | y - 2 | \tag {15}$$

$\mathrm { s u b j e c t ~ t o ~ } y x = 1 .$

It can be verified that when $\lambda \geq \frac { 1 } { 4 }$ (see Fig. 2 for illustration), (15) has the same optimal solution as (14). If $\lambda < \textstyle { \frac { 1 } { 4 } }$ , the optimal value of (15) is a lower bound of (14). If we relax the constraint (14c) as $y \leq 4 ,$ , then the inner problem has a different optimal solution $y ^ { * } = 0$ . In this case, the outer problem in (14a) becomes infeasible. This illustrates that dropping constraints from the inner problem does not necessarily reduce the optimal value of the outer problem. This nuance was overlooked in [1].
