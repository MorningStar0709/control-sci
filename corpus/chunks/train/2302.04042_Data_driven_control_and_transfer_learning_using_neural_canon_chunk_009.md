# B. Trajectory Planning

For a desired change of the operating point from an initial position $x _ { 0 }$ to a final position $x _ { N }$ in N sampling steps, the trajectory is planned in Brunovsky coordinates. For this purpose, the initial and target coordinates $z _ { 0 } = \Phi _ { x } ( x _ { 0 } )$ and $z _ { N } = \Phi _ { x } ( x _ { N } )$ are determined using the state parametrization. The first Brunovsky state $y _ { 1 } ( k )$ is parameterized with an approach $y _ { d , 1 } ( k ) ~ = ~ \lambda ( k ) ^ { T }$ α linear in the coefficients $\alpha \in \mathbb { R } ^ { n _ { \alpha } }$ . The parameters $\lambda ( k ) ^ { T } = [ \lambda _ { 1 } ( k ) , \dots , \lambda _ { n _ { \alpha } } ( k ) ]$ ] with $\lambda _ { i } \in$ R have to satisfy the initial and end conditions. A polynomial approach for lambda with $\lambda ( k ) ^ { T } = [ 1 , k , . . . , \overset { . } { k ^ { n _ { \alpha } } } ]$ and $n _ { \alpha } ~ = ~ 2 n - 1$ was chosen. Due to the shift-register structure of the Brunovsky system, the remaining Brunovsky states are determined by $\bar { y _ { d , i } } ( k ) ~ = ~ \lambda ( k + i { \bf \bar { \sigma } } - 1 ) ^ { T } c$ α for $i = 1 , \ldots , n$ and thus

$$
z _ {d} (k) = \left[ \begin{array}{c} y _ {d, 1} (k) \\ \vdots \\ y _ {d, n} (k) \end{array} \right] = \left[ \begin{array}{c} \lambda (k) ^ {T} \\ \vdots \\ \lambda (k + n - 1) ^ {T} \end{array} \right] \alpha = \mathcal {S} (k) \alpha .
$$

The unknown coefficients α can be determined by solving the linear equation system of the boundary conditions

$$
\left[ \begin{array}{c} z _ {0} \\ z _ {N} \end{array} \right] = \left[ \begin{array}{c} z _ {d} (0) \\ z _ {d} (N) \end{array} \right] = \left[ \begin{array}{c} \mathcal {S} (0) \\ \mathcal {S} (N) \end{array} \right] \alpha .
$$

The target input is defined by $\boldsymbol { v } _ { d } ( k ) = \lambda ( k + n ) ^ { T } \boldsymbol { \alpha } .$ .
