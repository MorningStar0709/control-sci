Definition 13 (Asymptotic stability (constrained – KL version)). Suppose $X \subset \mathbb { R } ^ { n }$ is positive invariant for $x ^ { + } ~ = ~ f ( x )$ , that A is closed and positive invariant for $x ^ { + } = f ( x )$ , and that A lies in the interior of X. The set A is asymptotically stable with a region of attraction X for $x ^ { + } = f ( x )$ ) if there exists a KL function $\beta ( \cdot )$ such that, for each $x \in X$

$$\left| \phi (i; x) \right| _ {\mathcal {A}} \leq \beta (| x | _ {\mathcal {A}}, i) \quad \forall i \in \mathbb {I} _ {\geq 0} \tag {7}$$

Notice that we simply replace Rn with the set X in Definition 9 to obtain Definition 13. We then have the following result, analogous to Theorem B.13, connecting a Lyapunov function to the KL version of asymptotic stability for the constrained case.

Theorem 14 (Lyapunov function for asymptotic stability (constrained case – KL version)). Suppose $X \subset \mathbb { R } ^ { n }$ is positive invariant $f o r x ^ { + } = f ( x )$ , that A is closed and positive invariant for $x ^ { + } = f ( x )$ , and that A lies in the interior of X. If there exists a Lyapunov function in X for the system $x ^ { + } = f ( x )$ and set A with $\alpha _ { 3 } ( \cdot ) \ a \ \mathcal { K } _ { \infty }$ function, then A is asymptotically stable $f o r x ^ { + } = f ( x )$ with a region of attraction X under Definition 13.

The proof of this result is similar to that of Theorem 12 with $\mathbb { R } ^ { n }$ replaced by X.
