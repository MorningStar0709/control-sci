# Step 5. Bounding the Resolvent.

Take the weak formulation of the previous elliptic problem (45):

$$B [ \phi , \psi ; \lambda ] = \langle f, \psi \rangle_ {\mathrm{L} ^ {2} (Y; \mathbb {R})}$$

which is solved by $\phi \in D ( \mathcal { L } _ { n } )$ for every ψ $\in H ^ { 1 } ( Y ; \mathbb { R } )$ by Lax-Milgram and and elliptic regularity with $\lambda > \omega$ . From the energy estimate and letting $\psi = \phi$ ,

$$| | \nabla_ {y} \phi | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} ^ {2} + (\lambda - \omega) \langle \phi , \phi \rangle_ {\mathrm{L} ^ {2} (Y; \mathbb {R})} \leq \langle f, \phi \rangle_ {\mathrm{L} ^ {2} (Y; \mathbb {R})}.$$

So,

$$(\lambda - \omega) \langle \phi , \phi \rangle_ {\mathrm{L} ^ {2} (Y; \mathbb {R})} \leq \langle f, \phi \rangle_ {\mathrm{L} ^ {2} (Y; \mathbb {R})} \leq | | f | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} | | \phi | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})}$$

Now, since $\phi$ is a weak solution to the elliptic problem (45):

$$\phi = \mathcal {R} (\lambda ; \mathcal {L} _ {n} (t; u, w)) f$$

with $\lambda > \omega$ . Thus:

$$(\lambda - \omega) | | \mathcal {R} (\lambda ; \mathcal {L} _ {n} (t; u, w)) f | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} ^ {2} \leq | | f | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})} | | \mathcal {R} (\lambda ; \mathcal {L} _ {n} (t; u, w)) f | | _ {\mathrm{L} ^ {2} (Y; \mathbb {R})},$$

so:

$$| | \mathcal {R} (\lambda ; \mathcal {L} _ {n} (t; u, w)) | | _ {\mathcal {L} (\mathrm{L} ^ {2} (Y; \mathbb {R}); \mathrm{L} ^ {2} (Y; \mathbb {R}))} \leq \frac {1}{\lambda - \omega}.$$

So, by Corollary 3.8. of [28], a version of the Hille-Yosida theorem, $\mathcal { L } _ { n } ( t ; u , w )$ generates an ω-contractive semigroup over $\operatorname { L } ^ { 2 } ( Y ; \mathbb { R } )$ . Since this holds for every $t \in [ 0 , T ]$ , this completes the proof of the original claim. □
