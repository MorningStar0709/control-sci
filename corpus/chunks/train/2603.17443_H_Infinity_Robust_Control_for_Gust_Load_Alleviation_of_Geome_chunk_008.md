# 2.1 Coupled System

The aeroelastic-flight dynamic model integrates three subsystems. The structural dynamics employ geometricallyexact nonlinear beam theory [Hodges, 2003, Palacios, 2011] with displacement-based finite elements carrying six DOF per node. The aerodynamics use unsteady strip theory based on Theodorsen’s framework [Theodorsen, 1935], with Wagner [Wagner, 1925] and Küssner [Küssner, 1936] indicial functions providing time-domain unsteady loads. The flight dynamics use Newton–Euler equations with quaternion-based attitude propagation [Tantaroudas et al., 2015].

The coupled system is expressed in first-order state-space form:

$$\dot {\mathbf {w}} = \mathbf {R} (\mathbf {w}, \mathbf {u} _ {c}, \mathbf {u} _ {d}) \tag {1}$$

where $\mathbf { w _ { \lambda } } \in \mathbb { R } ^ { n }$ is the state vector partitioned into aerodynamic, structural, and rigid-body components; $\mathbf { u } _ { c }$ is the control input (flap deflection); and $\mathbf { u } _ { d }$ is the gust disturbance.

Linearisation about the trimmed equilibrium $\mathbf { w } _ { 0 }$ yields:

$$\dot {\mathbf {w}} = \mathbf {A} \mathbf {w} + \mathbf {B} _ {c} \mathbf {u} _ {c} + \mathbf {B} _ {g} \mathbf {u} _ {d} + \mathbf {F} _ {\mathrm{nl}} (\mathbf {w}) \tag {2}$$

where A is the Jacobian, $\mathbf { B } _ { c }$ and $\mathbf { B } _ { g }$ are input matrices, and $\mathbf { F } _ { \mathrm { n l } }$ contains the higher-order nonlinear terms.
