From (8.41), one can see that the term $1 / P _ { D } ( z ^ { - 1 } ) P _ { F } ( z ^ { - 1 } )$ will introduce a stronger attenuation in the frequency domain than the term $1 / P _ { D } ( z ^ { - 1 } )$ ) if the auxiliary poles $P _ { F } ( z ^ { - 1 } )$ are real (aperiodic) and asymptotically stable. However, since $S ^ { \prime } ( z ^ { - 1 } )$ depends upon the poles through (8.44), one cannot guarantee this property for all the values of $P _ { F } ( z ^ { - 1 } )$ .

The auxiliary poles are generally chosen as high-frequency real poles under the form:

$$P _ {F} (z ^ {- 1}) = (1 - p _ {1} z ^ {- 1}) ^ {n _ {F}}; \quad 0. 0 5 \leq p _ {1} \leq 0. 5$$

where:

$$n _ {F} \leq n _ {p} - n _ {D}; \quad n _ {p} = (\deg P) _ {\max}; \quad n _ {D} = \deg P _ {D}$$

The effect of the introduction of the auxiliary poles is illustrated in Fig. 8.16 one observes that the auxiliary poles “squeezes” the modulus of the output sensitivity function around 0 dB axis in the high-frequency range.

Note that in many applications the introduction of the high-frequency auxiliary poles allows the requirements for robustness margins to be met.

Property 8.7 Simultaneous introduction of a fixed part $H _ { S _ { i } }$ and of a pair of auxiliary poles $P _ { F _ { i } }$ in the form

$$\frac {H _ {S _ {i}} (z ^ {- 1})}{P _ {F _ {i}} (z ^ {- 1})} = \frac {1 + \beta_ {1} z ^ {- 1} + \beta_ {2} z ^ {- 2}}{1 + \alpha_ {1} z ^ {- 1} + \alpha_ {2} z ^ {- 2}} \tag {8.48}$$
