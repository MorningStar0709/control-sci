$$N > \left\{- 2 \left(\frac {d R}{d t}\right) / (v _ {m} \cos (\gamma_ {m} - \lambda)) \right\} \quad \text { for } \quad \cos (\gamma_ {m} - \lambda) > 0. \tag {4.30}$$

The condition cos $( \gamma _ { m } - \lambda )$ means that the missile’s direction of flight forms an angle with the LOS to the target. Substituting d $R / d t$ from (4.25b) into (4.30), one obtains

$$N > 2 \{1 - [ \cos (\gamma_ {t} - \lambda) / (\kappa \cos (\gamma_ {m} - \lambda)) ] \}, \tag {4.31}$$

where we have substituted $\kappa = v _ { m } / v _ { t }$ . We can now write (4.31) as [17]

$$
\begin{array}{l} N = N ^ {\prime} \{1 - [ \cos (\gamma_ {t} - \lambda) / (\kappa \cos (\gamma_ {m} - \lambda)) ] \} \\ = - N ^ {\prime} \left\{\left(\frac {d R}{d t}\right) / (v _ {m} \cos (\gamma_ {m} - \lambda)) \right\}, \tag {4.32a} \\ \end{array}
$$

or

$$N ^ {\prime} = - N \left\{(v _ {m} \cos (\gamma_ {m} - \lambda)) / \left(\frac {d R}{d t}\right) \right\}, \tag {4.32b}$$

where $N ^ { \prime } \left( N ^ { \prime } > 2 \right)$ is commonly called the effective navigation ratio, and $- d R / d t$ is the missile’s closing velocity $( \mathrm { i . e . , - } ( d R / d t ) = v _ { c } )$ . We note from (4.28) that when $d ^ { 2 } \lambda / d t ^ { 2 }$ remains finite, then as $R  0 , ( d \lambda / d t )  0$ also.

Since the missile velocity vector cannot be controlled directly, the missile normal acceleration $a _ { n }$ is defined as

$$a _ {n} = v _ {m} \left(\frac {d \gamma_ {m}}{d t}\right), \tag {4.33}$$

where $d \gamma _ { m } / d t$ is the missile’s turning rate. Substituting (4.26) into (4.33), we have

$$a _ {n} = v _ {m} \left(\frac {d \gamma_ {m}}{d t}\right) = v _ {m} N \left(\frac {d \lambda}{d t}\right). \tag {4.34}$$

Furthermore, substituting (4.32) into (4.34) results in

$$a _ {n} = \left\{- N \dot {R} v _ {m} / \left(v _ {m} \cos \left(\gamma_ {m} - \lambda\right)\right) \right\} \left(\frac {d \lambda}{d t}\right) = \left\{N v _ {c} / \left(\cos \left(\gamma_ {m} - \lambda\right)\right) \right\} \left(\frac {d \lambda}{d t}\right), \tag {4.35}$$
