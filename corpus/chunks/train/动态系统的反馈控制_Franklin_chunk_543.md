# 8.3.5 几种数字化近似法的比较

针对一阶滞后环节，将四种近似方法得到的频率响应的幅值进行数值比较，设

$$D _ {c} (s) = \frac {5}{s + 5}$$

在两个不同的采样速率下，比较结果如图8.15所示。图8.15使用的 $D_{\mathrm{d}}(z)$ 的计算结果如表8.2所示。值得注意的是，MMPZ法和ZOH法在这个例子中的结果是完全相同的。但如果有多个零极点，结果就不总是这样了。另外MPZ法和图斯蒂法的结果也是完全相同的；但由表8.2可知，它们的 $D_{\mathrm{d}}(z)$ 有些微小的差别。四种近似法的主要差别就是带有 $(z + 1)$ 零点的两种方法都在 $\omega_{\mathrm{s}} / 2$ 时有凹陷，而这往往是我们需要的特性。

由图8.15看出，所有的近似法在低于约 $1 / 5$ 采样速率（或 $\omega_{\mathrm{s}} / 5$ 时都取得了十分理想的效果。如果 $\omega_{\mathrm{s}} / 5$ 远大于滤波器的截止频率（也就是说如果采样速率足够快），则滞后环节的截止特性将被精确复制。图斯蒂法和两种MPZ法都在 $\omega_{\mathrm{s}} / 2$ 处有凹陷，因为它们都有 $(z + 1)$ 项的零点 $z = -1$ 。除了在接近 $\omega_{\mathrm{s}} / 2$ 处存在较大差异外，四种方法的精度相似，但这种差异一般不是我们所关心的。
