Remark 1. The key of estimator (12) lies in the invertibility of $\Sigma _ { 0 } ( T )$ . Note that the observations $\{ y ( k ) \} _ { k = 1 } ^ { T }$ are corrupted by independent random noises. Since $I _ { N } \otimes \Gamma$ is a diagonal matrix and we do not consider the case that the state magnitude is in the same scale as the noise magnitude, the invertibility of $( \Sigma _ { 0 } ( T ) - I _ { N } \otimes \Gamma )$ is also guaranteed almost surely.
