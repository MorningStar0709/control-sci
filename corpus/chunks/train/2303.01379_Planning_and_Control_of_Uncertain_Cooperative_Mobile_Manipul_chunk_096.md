What remains to be shown is that $\tau _ { \operatorname* { m a x } } = \infty$ . We can conclude from the aforementioned analysis, Assumption $2 . 3 .$ , and $( 2 . 4 5 ) , ( 2 . 4 9 )$ that the solution $\sigma ( t )$ remains in a compact subset $\Omega ^ { \prime }$ of $\Omega , \forall t \in [ 0 , \tau _ { \operatorname* { m a x } } )$ , namely $\sigma ( t ) \in \Omega ^ { \prime } , \forall t \in [ 0 , \tau _ { \operatorname* { m a x } } )$ . Hence, according to Theorem $\mathrm { A . 2 }$ of Appendix $\mathrm { A } .$ , it holds that $\tau _ { \operatorname* { m a x } } = \infty$ . Thus, all closed loop signals remain bounded and moreover $\sigma ( t ) \in \Omega ^ { \prime } \subset \Omega , \forall t \in \mathbb { R } _ { \geq 0 }$ . Finally, by multiplying (2.45) by $\rho _ { k } ( t ) , k \in \mathcal { K }$ , we obtain

$$- \rho_ {s _ {k}} (t) < - \bar {\xi} _ {s} \rho_ {s _ {k}} (t) \leq e _ {s _ {k}} (t) \leq \bar {\xi} _ {s} \rho_ {s _ {k}} (t) < \rho_ {s _ {k}} (t), \tag {2.51}$$

$\forall t \in \mathbb { R } _ { \geq 0 }$ , which leads to the conclusion of the proof.

□

Remark 2.5 (Prescribed Performance). From the aforementioned proof it can be deduced that the Prescribed Performance Control scheme achieves its goal without resorting to the need of rendering the ultimate bounds $\bar { \varepsilon } _ { s } , \bar { \varepsilon } _ { v }$ of the modulated pose and velocity errors $\varepsilon _ { s } , \varepsilon _ { v }$ arbitrarily small by adopting extreme values of the control gains $g _ { s }$ and $g _ { v }$ (see (2.44) and (2.48)). More specifically, notice that (2.45) and (2.49) hold no matter how large the finite bounds $\bar { \varepsilon } _ { s } , \bar { \varepsilon } _ { v }$ are. In the same spirit, large uncertainties involved in the coupled model (2.14) can be compensated, as they affect only the size of $\varepsilon _ { v }$ through $\bar { B } _ { v }$ , but leave unaltered the achieved stability properties. Hence, the actual performance given in (2.51), which is solely determined by the designed-specified performance functions $\rho _ { s _ { k } } ( t ) , \rho _ { v _ { k } } ( t ) , k \in \mathcal { K }$ , becomes isolated against model uncertainties, thus extending greatly the robustness of the proposed control scheme.
