# A. Preliminary on graph theory

For the considered group of underwater vehicles, it is assumed that each vehicle can be viewed as a node, and the communication topology associated with the networked system among multiple AUVs can then be represented by a weighted directed graph $G = \{ V , E , A \}$ . As for a simple time-invariant graph G, it is composed of the vertex set $V = \{ \nu _ { 1 } , \nu _ { 2 } , . . . , \nu _ { N } \}$ , the edge set $E \subseteq V \times V$ , and the weighted adjacency matrix $A = \mathsf { \bar { [ } } a _ { i j } \mathsf { ] } \in \mathbb { R } ^ { \mathrm { { N } \times \mathrm { { N } } } }$ . The element $\nu _ { i }$ in vertex set V denotes vehicle i, and the index i belongs to a finite index set $\Gamma = \{ 1 , \dots , N \}$ . If there exists the information exchange between AUV i and AUV j, then, say, there is a edge between AUVs i and $j , \mathrm { i . e . , } ( \nu _ { i } , \nu _ { j } ) \in E .$ , and $a _ { i j } = a _ { j i } > 0$ . Particularly, call vehicle j a neighbor of vehicle i, and the set of neighbors is represented by $N _ { i } ~ = ~ \{ j | ( \nu _ { i } , \nu _ { j } ) \in E \}$ . Otherwise, there is no edge among them, and $a _ { i j } = a _ { j i } = 0$ . Moreover, we define $a _ { i i } = 0$ for all $i \in \Gamma$ , and the out-degree $\begin{array} { r } { d _ { i } = \sum _ { j \in N _ { i } } a _ { i j } } \end{array}$ associated with the node i. Afterwards, the degree matrix together with the Laplacian matrix of the graph G can then be defined as $D = \mathrm { d i a g } \mathbf { \bar { \{ d _ { 1 } , \dots , { d _ { N } } \} } } \in \mathbb { R } ^ { \mathbf { N } \times \mathbf { \bar { N } } }$ and $L = D - A$ , respectively. A path in graph is a sequence that is consisted of a set of successive adjacent nodes, starting from node i and ending at node j. If any two nodes in a graph G have at least one path, then, say, graph G is connected.

In order to make the group of AUVs move along with the desired path as a whole, a reference trajectory must be defined ahead of time. The availability to the information of reference trajectory for i-th AUV is indicated by a parameter $b _ { i } ;$ that is, if AUV i is able to access this information,then $b _ { i } > 0 ;$ $b _ { i } = 0 .$ , otherwise. Let B = diag $\{ b _ { 1 } , \dotsc , b _ { N } \}$ .

Assumption 1. For the considered multi-AUV formation control network, graph G is connected, and moreover there is at least one AUV able to receive the information of reference trajectory, i.e., the elements of B are not all equal to zero.
