# 3 Preliminaries and Problem Statement

We consider uncertain continuous-time control-affine nonlinear systems (which include many common mechanical systems of interest [15]) with output observations

$$\dot {x} (t) = f (x (t)) + B u (t) + B _ {w} (t) w _ {x} (t) \tag {1a}y (t) = h (x (t), \theta) + B _ {y} w _ {y} (t) \tag {1b}$$

where $\begin{array} { r } { f : \mathcal { X }  \mathcal { X } , \mathcal { X } \subseteq \mathbb { R } ^ { n _ { x } } , B \in \mathbb { R } ^ { n _ { x } \times n _ { u } } , B _ { w } : [ 0 , \infty )  \mathbb { R } ^ { n _ { x } \times n _ { w _ { x } } } , B _ { y } \in \mathbb { R } ^ { n _ { x } \times n _ { u } } , } \end{array}$ $\mathbb { R } ^ { n _ { y } \times n _ { w _ { y } } } , \mathcal { U } \subseteq \mathbb { R } ^ { n _ { u } }$ , and $w _ { x } \in \mathbb { R } ^ { n _ { w _ { x } } }$ is a possibly stochastic state disturbance where $\lVert w _ { x } ( t ) \rVert \leq \bar { w } _ { x }$ , for all t. Without loss of generality, we assume $\| B _ { w } ( t ) \| \le 1$ , for all t. Norms $\| \cdot \|$ without subscript are the (induced) 2-norm. We obtain high-dimensional observations $y \in \mathcal { Y } \subseteq \mathbb { R } ^ { n _ { y } } \mathrm { ~ } ( \mathrm { e . g . } , N \times N$ -pixel RGB-D images, leading to $n _ { y } =$ $4 N ^ { 2 } )$ , generated by a deterministic, nonlinear function $h ( x , \theta ) : \mathcal { X } \times \theta  \mathcal { Y }$ which is unknown to the robot; here, $\theta \in \mathbb { R } ^ { n _ { p } }$ are external parameters (e.g., location of obstacles, map of environment, etc.). The observations may be corrupted by (possibly stochastic) sensor noise $w _ { y } ( t ) \in \mathbb { R } ^ { n _ { w _ { y } } }$ , where $\| w _ { y } ( t ) \| \le \bar { w } _ { y }$ , for all t. We note that our results also apply to time-varying $B ( t )$ under some conditions on its null-space.
