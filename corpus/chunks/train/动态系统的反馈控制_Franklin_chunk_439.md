# 7.6.2 对称根轨迹(SRL)

线性控制系统中一个非常有效且广泛使用的方法是，最优线性二次调节器（LQR）。LQR问题的简化描述即为寻找一个控制律使得以下性能指标：

$$\mathcal {J} = \int_ {0} ^ {+ \infty} [ \rho z ^ {2} (t) + u ^ {2} (t) ] \mathrm{d} t \tag {7.106}$$

对如下的系统最小：

$$\dot {x} = A x + B u \tag {7.107a}z = C _ {1} x \tag {7.107b}$$

其中：式(7.106)中的 $\rho$ 为设计者选取的权系数。值得注意的是，使得 J 最小化的控制律由下式的线性状态反馈给出：

$$u = - K x \tag {7.108}$$

这里，K 的最优值为能使闭环极点配置在如下对称根轨迹(SRL)方程(1980 年，凯拉斯(Kailath))的稳定根(都在 LHP)位置上的 K 值：

$$1 + \rho G _ {0} (- s) G _ {0} (s) = 0 \tag {7.109}$$

其中： $G_{0}$ 为从 $u$ 到 $z$ 的开环传递函数：

$$G _ {0} (s) = \frac {Z (s)}{U (s)} = C _ {1} (s I - A) ^ {- 1} B = \frac {N (s)}{D (s)} \tag {7.110}$$

注意，这就是在第5章中讨论过的关于参数 $\rho$ 的根轨迹问题，在性能指标方程(7.106)中参数 $\rho$ 权衡了(跟踪误差) $z^2$ 对于被控量 $u^2$ 的相对损耗。同样可以看出， $s$ 与 $-s$ 以同样的方式影响式(7.109)，因此，对于式(7.109)的任意一个根 $s_0$ ，也必将有一个根 $-s_0$ 。我们称这样所得到的根轨迹为对称根轨迹，因此位于左半平面的轨迹在右半平面将有一镜像；即根轨迹关于虚轴对称。所以，我们通过先选取矩阵 $C_1$ 来选取最优闭环极点， $C_1$ 决定了跟踪误差，且设计者希望跟踪误差保持为很小的值，然后选取 $\rho$ ，确保跟踪误差以及控制量的重要性相平衡。注意将输出选取为跟踪误差而不需要选择对象传感器输出。这就是式(7.107)中的输出用 $z$ 表示而不用 $y$ 表示的原因。

从式(7.109)的解中选取一组稳定的极点得到期望的闭环极点，然后将这些极点应用在极点配置公式中，例如阿克曼公式(式(7.88))，计算得到K。由于在实传递函数 $G_{0}$ 的所有根轨迹中，每一条轨迹关于实轴也都是对称的；也就是说，根轨迹关于实轴和虚轴都是对称的。将对称根轨迹方程写成如下的标准根轨迹形式：

$$1 + \rho \frac {N (- s) N (s)}{D (- s) D (s)} = 0 \tag {7.111}$$

将从 U 到 Z 的传递函数的开环极点通过虚轴映射（得到双倍的零极点数），这样就得到了全部的零极点，然后可以画出根轨迹。注意，根轨迹可以是 $0^{\circ}$ 或 $180^{\circ}$ ，这取决于式 (7.109) 中的 $G_{0}(-s)G_{0}(s)$ 的符号。决定用哪种类型 ( $0^{\circ}$ 或 $180^{\circ}$ ) 的根轨迹的一个快捷的方法就是选取与虚轴没有交点的那个。绘制根轨迹的实轴规则揭示了这一点。在此我们假设系统是可控的，并且假设所有系统的状态矢量都出现在被选的输出 z 当中，那么一定保证最优闭环系统是稳定的；因此根轨迹的所有部分都不在虚轴上。
