# II. PROBLEM STATEMENT

We consider a model predictive control problem for trajectory tracking of an unknown linear system. In particular, we consider a discrete-time linear time invariant system

$$
\begin{array}{l} x _ {t + 1} = A x _ {t} + B u _ {t}, \quad x _ {t} \in \mathcal {M} \subset \mathbb {R} ^ {n}, \\ u _ {t} \in \mathcal {U} \subset \mathbb {R} ^ {m}, \quad t \in \mathbb {N} _ {0}, \tag {1} \\ \end{array}
$$

where both A and B matrices are unknown. We construct an MPC trajectory problem on an identified system as follows:

$$
\begin{array}{l} \min _ {\left\{u _ {k \mid t}, x _ {k + 1 \mid t} \right\} _ {k = t} ^ {t + T _ {h} - 1}} \sum_ {k = t} ^ {t + T _ {h} - 1} \left[ \left(x _ {k \mid t} - x _ {k \mid t} ^ {\text { ref }}\right) ^ {\top} Q (x _ {k \mid t} - x _ {k \mid t} ^ {\text { ref }}) + u _ {k \mid t} ^ {\top} R u _ {k \mid t} \right] \\ + \left(x _ {t + T _ {h} | t} - x _ {t + T _ {h} | t} ^ {\text { ref }}\right) ^ {\top} Q _ {f} (x _ {t + T _ {h} | t} - x _ {t + T _ {h} | t} ^ {\text { ref }}) \tag {2} \\ \end{array}
$$

$\mathrm { s u b j e c t ~ t o : ~ } x _ { k + 1 | t } = \hat { A } x _ { k | t } + \hat { B } u _ { k | t } , \ k = t , \ldots , ( t + T _ { h } - 1 ) ,$

$$x _ {k + 1 | t} \in \mathcal {M}, u _ {k | t} \in \mathcal {U}, x _ {t | t} = x _ {t}, x _ {t + T _ {h} | t} \in \mathcal {X} _ {f}.$$

where $\hat { A }$ and $\hat { B }$ are the identified system matrices, $T _ { h }$ is the MPC horizon, and $\chi _ { f }$ is a terminal constraint required for the MPC stability. If no terminal constraint exists, then $\chi _ { f } = M .$ We assume that $Q \succeq 0$ and $R \succ 0$ and known, as customary in standard linear quadratic (LQ) problems. The terminal cost matrix $Q _ { f }$ is also added for proving the stability of the MPC as described in Section IV. The optimal value of the cost function depends on the current state $x _ { t }$ and is denoted as $J ^ { * } ( x _ { t } )$ . The input computed from this MPC problem is applied to the original system (1), and we expect the original system to track the reference trajectory $x ^ { \mathrm { r e f } }$ . The resultant QSID-MPC algorithm is described in Algorithm 1.
