The optimal non-causal estimator ${ \mathcal { L } } _ { n c }$ is a known result, which can be found in Hassibi et al. (1999). It is unique and provides estimates that minimize both $\mathcal { H } _ { 2 }$ and $\bar { \mathcal { H } } _ { \infty }$ costs. Using (25), this means that the corresponding pair of transfer functions $\{ \Phi _ { v , n c } , \Phi _ { w , n c } \}$ exists, is unique, and solves both problems in (23).

The problem (23a) is strictly convex because, by definition, $\mathcal { Q } \ \succ \ 0 , \ \Sigma _ { v } \ \succ \ \dot { 0 } .$ and $\Sigma _ { w } \ \succ \ 0$ . Hence, its solution is unique. Since a solution solving both problems exists, and the solution to (23a) is unique, we conclude that one only needs to solve (23a). Finally, (24) follows from (10).
