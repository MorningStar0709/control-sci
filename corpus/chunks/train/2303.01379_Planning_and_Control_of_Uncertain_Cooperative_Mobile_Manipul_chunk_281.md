Remark 4.3. The proof can be trivially extended to the 2D case on the horizontal plane where there is no gravity, i.e., $g = 0$ . In addition, it is worth noting that in that case, the estimation parameter $\hat { m } ( t )$ will converge to a constant value different than the mass m, as revealed by a careful inspection of the closed-loop system. Moreover, the condition $\begin{array} { r } { k _ { \phi } > \frac { \alpha } { 2 } } \end{array}$ of Theorem 4.1 is only sufficient and not necessary, as will be shown in the simulation results.
