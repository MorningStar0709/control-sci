# A. Optimal MPC

In this subsection we review the properties of the optimal mapping $\mu ^ { \star } ( x )$ , which are then used for the analysis of the “perturbed” suboptimal dynamics (6). We start with the regularity properties of the optimal mapping.

Lemma $\begin{array} { r l } { \mathit { { l } } : } & { { } [ 9 , } \end{array}$ Corollary 2] For any $x , y \in \mathbb { R } ^ { n }$ , the optimal solution mapping, $\mu ^ { \star } ( x )$ , satisfies

$$\| \mu^ {\star} (x) - \mu^ {\star} (y) \| \leq \| H ^ {- \frac {1}{2}} \| \| G (x - y) \| _ {H ^ {- 1}} \leq L \| x - y \|,\langle \mu^ {\star} (x) - \mu^ {\star} (y), G (x - y) \rangle \leq - \| \mu^ {\star} (x) - \mu^ {\star} (y) \| _ {H} ^ {2},$$

with a Lipschitz constant $L : = \| H ^ { - \frac { 1 } { 2 } } \| \cdot \| H ^ { - \frac { 1 } { 2 } } G \|$ .

The proof follows from the parametric quadratic program structure of the MPC problem and can be found in [9] or [14] with an explicit MPC point of view.

As shown in [12], [15], system (2) is asymptotically stable with the ROA estimate

$$\Gamma_ {N} := \{x \in \mathbb {R} ^ {n} \mid \psi (x) \leq r _ {N} \},$$

where $\psi ( x ) ~ : = ~ \sqrt { V ( x ) } , ~ d = c \cdot \lambda ^ { - } ( Q ) / \lambda ^ { + } ( P ) , ~ r _ { N } ~ =$ $\sqrt { N d + c }$ and $c > 0$ is such that the following set is nonempty

$$\Omega = \{x \in \mathbb {R} ^ {n} \mid \| x \| _ {P} ^ {2} \leq c, - K x \in \mathcal {U} \}.$$

Lemma 2: The value function V , is a local Lyapunov function for the optimal closed-loop system (2) with a ROA estimate $\Gamma _ { N }$ , satisfying

$$\| x \| _ {P} ^ {2} \leq V (x) \leq \| x \| _ {W} ^ {2} \tag {8}V (f (x)) \leq \beta^ {2} V (x), \tag {9}$$
