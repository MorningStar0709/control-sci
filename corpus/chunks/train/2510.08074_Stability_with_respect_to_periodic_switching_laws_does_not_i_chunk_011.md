If B belongs to the convex hull of $B _ { 0 } , B _ { 1 } , B _ { 2 } , B _ { 3 }$ then we may write

$$B = \sum_ {i = 0} ^ {3} \beta_ {i} B _ {i}$$

where $\textstyle \sum _ { i = 0 } ^ { 3 } \beta _ { i } = 1$ and where $0 \leq \beta _ { i } \leq 1$ for every i. The identity

$$B _ {0} + B _ {3} = B _ {1} + B _ {2}$$

follows from the definition of the matrices $B _ { i } .$ , so

$$
\begin{array}{l} B - B _ {0} = (\beta_ {0} - 1) B _ {0} + \beta_ {1} B _ {1} + \beta_ {2} B _ {2} + \beta_ {3} B _ {3} \\ = \left(\beta_ {0} - 1\right) B _ {0} + \beta_ {1} B _ {1} + \beta_ {2} B _ {2} + \beta_ {3} \left(B _ {1} + B _ {2} - B _ {0}\right) \\ = (\beta_ {1} + \beta_ {3}) (B _ {1} - B _ {0}) + (\beta_ {2} + \beta_ {3}) (B _ {2} - B _ {0}). \\ \end{array}
$$

Applying $\ell _ { 0 }$ and $\ell _ { 1 }$ to both sides of the above equation immediately yields

$$\ell_ {0} (B - B _ {0}) = \beta_ {1} + \beta_ {3},\ell_ {1} (B - B _ {0}) = \beta_ {2} + \beta_ {3}.$$

In particular

$$B - B _ {0} = \ell_ {0} (B - B _ {0}) (B _ {1} - B _ {0}) + \ell_ {1} (B - B _ {0}) (B _ {2} - B _ {0})$$

and $0 \le \ell _ { i } ( B - B _ { 0 } ) \le 1$ for i = 0, 1 as required.

For the remainder of this section we fix linear maps $\ell _ { 0 } , \ell _ { 1 } \colon M _ { 4 } ( \mathbb { R } ) \to \mathbb { R }$ as given by Proposition 1, and let $e _ { 0 } , e _ { 1 }$ denote the standard basis for $\mathbb { R } ^ { 2 }$ .
