# A.2 Proof of Theorem 1

We prove the set inclusions in both directions. The first is stated below, whose proof relies critically on the global optimality property of state-feedback $\mathcal { H } _ { \infty }$ cost function.

Lemma 8 Under Assumptions 1 and 2, we have

$$\left\{K \in \mathcal {K} \mid \| \mathbf {T} _ {\infty} (K) \| _ {\mathcal {H} _ {\infty}} \leq \beta \right\} \subseteq \operatorname{cl} \left(\mathcal {K} _ {\beta}\right).$$

Proof: We claim that for any $K \in \mathcal { K }$ with $\| \mathbf { T } _ { \infty } ( K ) \| _ { \mathcal { H } _ { \infty } } \leq$ β, there exists a sequence $\{ K _ { j } \} \subset \mathcal { K } _ { \beta }$ such that $K _ { j }  K$ , and hence $K \in \mathrm { c l } ( \bar { \mathcal { K } } _ { \beta } )$ .

Suppose for contradiction that this is false. Then there exists a neighborhood $U \subset K$ of K such that $U \cap \mathcal { K } _ { \beta } = \emptyset$ , implying that $\| { \bf T } _ { \infty } ( K ^ { \prime } ) \| _ { \mathcal { H } _ { \infty } } \ge \beta$ for all $K ^ { \prime } \in U$ . Thus, K (with $\| \bar { \mathbf { T } } _ { \infty } ( K ) \| _ { \mathcal { H } _ { \infty } } \dot { \leq } \dot { \beta } )$ is a local minimizer of $\| \mathbf { T } _ { \infty } ( \cdot ) \| _ { \infty }$ over U . It is clear by continuity that $\| \mathbf { T } _ { \infty } ( K ) \| _ { \mathcal { H } _ { \infty } } = \ddot { \beta }$ .

However, by [23, Corollary 4.1], every local minimizer of $\| \mathbf { T } _ { \infty } ( \cdot ) \| _ { \infty }$ is a global one that attains the infimum value $\beta ^ { * } { \mathrel { \mathop : } } \mathrm { = i n f } _ { K \in { \mathcal K } } \| \bar { \mathbf T } _ { \infty } ( K ) \| _ { \mathcal { H } _ { \infty } }$ . Therefore, we must have $\| \mathbf { T } _ { \infty } ( K ) \| _ { \mathcal { H } _ { \infty } } = \beta ^ { * }$ , which is a contradiction since $\beta \neq \beta ^ { * }$ . Hence, such a neighborhood U cannot exist. ✷
