Nonuniqueness of a Set of State Variables. It has been stated that a set of state variables is not unique for a given system.Suppose that $x _ { 1 } , x _ { 2 } , \ldots , x _ { n }$ are a set of state variables.

Then we may take as another set of state variables any set of functions

$$\hat {x} _ {1} = X _ {1} \left(x _ {1}, x _ {2}, \dots , x _ {n}\right)\hat {x} _ {2} = X _ {2} \big (x _ {1}, x _ {2}, \dots , x _ {n} \big)\hat {x} _ {n} = X _ {n} \left(x _ {1}, x _ {2}, \dots , x _ {n}\right)$$

provided that, for every set of values ${ \hat { x } } _ { 1 } , { \hat { x } } _ { 2 } , \ldots , { \hat { x } } _ { n }$ there corresponds a unique set of, values $x _ { 1 } , x _ { 2 } , \ldots , x _ { n }$ , and vice versa. Thus, if x is a state vector, then $\hat { \mathbf { x } }$ where ,

$$\hat {\mathbf {x}} = \mathbf {P} \mathbf {x}$$

is also a state vector, provided the matrix P is nonsingular. Different state vectors convey the same information about the system behavior.
