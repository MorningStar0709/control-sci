Theorem 10 tells us that we will not be able to connect any two arbitrary signed measures together, only the ones where the positive and negative couplings are c-CM compatible. This is restrictive for some versions of signed optimal transport that want to be able to connect any two signed measures in the same way as probability measures, but it is an inherent limitation from the approach focusing on c-CM as the salient feature of optimality.

Unfortunately, in two dimensions and higher it is not possible to partition the space of signed measures into classes such that there exists a kernel between all measures within a class and that any target for which there is a kernel is an element of the class. We demonstrate this with the following example.

However they are in one dimension, and there is a simple tool that we can use to signify when two signed measures are in the same equivalence class.
