Notations and definitions: The set of real numbers are denoted by $\mathbb { R } ,$ the n - dimensional Euclidean space by $\mathbb { R } ^ { n }$ , and the space of $m \times n$ real-valued matrices by $\mathbb { R } ^ { m \times n }$ . The identity matrix of order $n \times n$ is denoted by $\mathbf { I } _ { n } .$ For a matrix $\textbf { M } \in \mathbb { R } ^ { m \times n }$ , the notation $\mathbf { M } _ { ( i , : ) }$ denotes the $i ^ { t h }$ row, $\mathbf { M } _ { ( : , j ) }$ denotes the $j ^ { t h }$ column, and $\mathbf { M } _ { ( i : p , j : q ) }$ denotes the submatrix containing rows from i to $p$ and columns from j to $q .$ The p−norm of a vector $\mathbf { x } \in \mathbb { R } ^ { n }$ is defined as $\left. \mathbf { \textbf { x } } \right. _ { p } = ( \sum _ { i = 1 } ^ { n } | x _ { i } | ^ { p } ) ^ { \frac { 1 } { p } }$ . The Frobenious norm of a matrix $\mathbf { A } \in \mathbb { R } ^ { m \times n }$ is defined as k $\begin{array} { r } { \textbf { M } \| _ { F } = \sqrt { \sum _ { i = 1 } ^ { m } \sum _ { j = 1 } ^ { n } M _ { i j } ^ { 2 } } = \sqrt { \mathrm { t r a c e } ( \textbf M ^ { \top } \textbf M ) } } \end{array}$ . The block Hankel matrix consists of $\mathrm { N } \times \mathrm { H }$ samples of the vector v, starting from the index k is defined as:

$$
\mathbf {H} _ {\left(\mathrm{N}, \mathrm{H}, \mathbf {v} _ {k}\right)} = \left[ \begin{array}{c c c c} \mathbf {v} _ {k} & \mathbf {v} _ {k + 1} & \dots & \mathbf {v} _ {k + \mathrm{H} - 1} \\ \mathbf {v} _ {k + 1} & \mathbf {v} _ {k + 2} & \dots & \mathbf {v} _ {k + \mathrm{H}} \\ \vdots & \vdots & & \vdots \\ \mathbf {v} _ {k + \mathrm{N} - 1} & \mathbf {v} _ {k + \mathrm{N}} & \dots & \mathbf {v} _ {k + \mathrm{N} + \mathrm{H} - 2} \end{array} \right] = \left[ \begin{array}{l l l l} \mathbf {V} _ {k} & \mathbf {V} _ {k + 1} & \dots & \mathbf {V} _ {k + \mathrm{H} - 1} \end{array} \right]. \tag {1}
$$

A general optimization problem can be represented as:

$$\inf _ {\mathbf {z}} J (\mathbf {z})\text { subject to } \mathbf {z} \in \mathbb {Z} \subseteq \mathbb {R} ^ {h} \tag {2}$$

where z is the decision vector, J is the cost function (loss function), and Z is the constraint set. The optimum value of the decision vector is denoted by $\mathbf { z } ^ { \ast }$ . Constraint sets can be defined using linear inequalities: $\mathbb { Z } = \left\{ \mathbf { z } \in \mathbb { R } ^ { n } : \mathbf { F } \mathbf { z } \leq \mathbf { g } \right\}$ or $\mathbb { Z } = \left\{ \mathbf { z } \in \mathbb { R } ^ { n } : \mathbf { z } _ { m i n } \leq \mathbf { z } \leq \mathbf { \bar { z } } _ { m a x } \right\}$ .
