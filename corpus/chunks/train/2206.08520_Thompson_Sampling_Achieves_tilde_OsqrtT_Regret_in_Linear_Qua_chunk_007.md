# 2.1. Setting

Suppose we are given a discrete time linear time-invariant system with the following dynamics,

$$x _ {t + 1} = A _ {*} x _ {t} + B _ {*} u _ {t} + w _ {t}, \tag {1}$$

where $x _ { t } \in \mathbb { R } ^ { n }$ is the state of the system, $u _ { t } \in \mathbb { R } ^ { d }$ is the control input, $w _ { t } \in \mathbb { R } ^ { n }$ is i.i.d. process noise at time t. At each time step t, the system is at state $x _ { t }$ where the agent observes the state. Then, the agent applies a control input $u _ { t }$ and the system evolves to $x _ { t + 1 }$ at time $t + 1$ . The underlying system (1) can be represented as $x _ { t + 1 } = \Theta _ { * } ^ { \intercal } z _ { t } + w _ { t }$ , where $\Theta _ { * } ^ { \intercal } = \left[ A _ { * } \ B _ { * } \right]$ and $\boldsymbol { z } _ { t } = [ \boldsymbol { x } _ { t } ^ { \intercal } ~ \boldsymbol { u } _ { t } ^ { \intercal } ] ^ { \intercal }$ . In this work, we consider stabilizable linear dynamical systems $\Theta _ { \ast } .$ , such that there exists a controller K where $\rho ( A _ { * } + B _ { * } K ) < 1$ . More precisely, the systems with the following property:
