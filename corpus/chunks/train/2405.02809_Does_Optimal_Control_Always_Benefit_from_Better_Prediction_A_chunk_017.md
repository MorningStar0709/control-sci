# 3.2.2. The A Priori Truth

One may argue that there are intrinsic uncertainties in s as the future values of the i.i.d. environment disturbance $\left[ r _ { 1 } , r _ { 2 } , . . . , r _ { N - 1 } \right]$ can never be known at step 0. In a prediction, the true probability distribution should be determined by what has happened by the time of the prediction, not from the future. This probability distribution should be determined at step 0. Therefore for the belief $\bar { W } _ { b } = \mathcal { P } ( o ( s ) )$ , the truth is the following conditional probability distribution

$$\bar {w} (S) \big | S _ {z _ {0}} = s _ {z _ {0}}, S _ {r _ {0}} = s _ {r _ {0}}, \tag {11}$$

where $s _ { z _ { 0 } } , \ s _ { r _ { 0 } }$ are the $z _ { 0 }$ and $r _ { 0 }$ component of s respectively. We call this a priori true probability distribution $\hat W _ { t n } \big ( s _ { z _ { 0 } } , s _ { r _ { 0 } } \big )$ , or $\bar { W } _ { t n } ( s )$ for short.
