# Algorithm 3: Recursive MPC Algorithm

1) Given: Initial state $\Phi _ { 0 } ,$ time horizon $^ { - } { }$ cost $\begin{array} { r } { c ( \Phi , U ) = l ( \Phi ) + \frac { 1 } { 2 } r U ^ { 2 } , } \end{array}$ , and terminal cost $c _ { T } ( \Phi )$ .   
2) Set $H = T , \Phi _ { i } \bar { = } \Phi _ { 0 }$

while $H > 0$ do

1) Solve the open-loop(deterministic) optimal control problem for initial state $\Phi _ { i }$ and horizon H. Let the optimal sequence be $U ^ { * } = \{ U _ { 0 } , \bar { U _ { 1 } } , . . . U _ { H - 1 } \} .$ .   
2) Apply the first control $U _ { 0 }$ to the stochastic system, and observe the next state $\Phi _ { n } .$ .   
3) Set $H = H - 1 , \Phi _ { i } = \Phi _ { n }$

end while

The feedback policy obtained appears to be near-optimal and highly robust for low noise levels (Fig 7), which is as expected from the $O ( \epsilon ^ { 4 } )$ -optimality of the algorithm used [27]. But when operating in high noise regimes, the robustness of the feedback policy quickly degenerates. Thus, to reduce the variance of the policy in these domains, we advocate the use of a rapid open loop solver along with replanning at every step, i.e. a Model-Predictive Control approach, to solve the RL problem, since MPC should be able to recover the global optimality for these problems [29].

The recursive MPC algorithm implemented, which uses a fast and reliable local-planner, is summarised in Algorithm 3, and Figure 8 compares the robustness to noise between the stochastic policy obtained from the recursive MPC with the decoupled closed-loop feedback policy at varying noise regimes.
