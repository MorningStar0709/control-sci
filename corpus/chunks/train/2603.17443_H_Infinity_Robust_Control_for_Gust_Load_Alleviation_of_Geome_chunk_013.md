# 3.3 $\mathcal { H } _ { \infty }$ Problem Formulation

The generalised plant is constructed following the standard two-port framework:

$$
\left[ \begin{array}{l} \mathbf {z} _ {\text { perf }} \\ \mathbf {y} _ {\text { meas }} \end{array} \right] = \left[ \begin{array}{l l} \mathbf {P} _ {1 1} & \mathbf {P} _ {1 2} \\ \mathbf {P} _ {2 1} & \mathbf {P} _ {2 2} \end{array} \right] \left[ \begin{array}{c} \mathbf {w} _ {\text { ext }} \\ u \end{array} \right] \tag {7}
$$

where $\mathbf { z } _ { \mathrm { p e r f } }$ is the performance output and $\mathbf { y } _ { \mathrm { m e a s } }$ is the measured output.

An input-shaping weighting function $K _ { c }$ is applied to penalise the control effort:

$$
\mathbf {z} _ {\text { perf }} = \left[ \begin{array}{l} \mathbf {C} _ {z} \mathbf {x} \\ K _ {c} \cdot u \end{array} \right] \tag {8}
$$

where $\mathbf { C } _ { z }$ selects the states to be regulated (typically wing-tip displacement and bending slope).

The $\mathcal { H } _ { \infty }$ optimisation finds a stabilising controller $K ( s )$ that minimises the closed-loop $\mathcal { H } _ { \infty }$ norm:

$$\gamma^ {*} = \min _ {K \text { stabilising }} \| \mathcal {F} _ {l} (\mathbf {P}, K) \| _ {\infty} = \min _ {K} \sup _ {\omega} \bar {\sigma} \left[ \mathcal {F} _ {l} (\mathbf {P}, K) (j \omega) \right] \tag {9}$$

where $\mathcal { F } _ { l }$ denotes the lower linear fractional transformation and σ¯ is the maximum singular value. The optimal $\gamma ^ { * }$ represents the minimum achievable worst-case amplification from gust to performance output.
