# D. Robust Guarantees of the Adaptation Process

In this section we highlight the input-to-state (ISS) stability nature of the proposed adaptive friction estimator and the passivity property of the friction phenomenon [19]. Based on ISS and passivity, we analyze some strategies for robusifying the adaptive estimation process.

Following the above analysis, the derivative of the Lyapunov function of $V _ { 1 }$ satisfies

$$
\begin{array}{l} \dot {V} _ {1} (t) = - \mathbf {s} ^ {T} (\mathbf {K} _ {D} \mathbf {s} - \mathbf {Y} \tilde {\pi}) \leq - \mathbf {s} ^ {T} \mathbf {K} _ {D} \mathbf {s} + | \mathbf {s} | | \mathbf {Y} | | \tilde {\pi} | \\ \leq - \mathbf {s} ^ {T} \mathbf {K} _ {D} \mathbf {s} + c | \tilde {\boldsymbol {\pi}} | \quad \leq - \alpha (| \tilde {\mathbf {x}} |) + \beta (| \tilde {\boldsymbol {\pi}} |), \\ \end{array}
$$

with |Y| the induced norm of the regressor matrix, $c =$ $\operatorname { s u p } ( | \mathbf { s } | ) | \mathbf { Y } |$ a positive constant, $\tilde { \mathbf { x } } = \left\lceil \tilde { \mathbf { q } } \tilde { \mathbf { q } } \right\rceil ^ { T }$ the system’s tracking error, and the class $\mathcal { H } _ { \infty }$ functions [1] α and $\beta .$ The last inequality is easy to check by noticing that $c | \tilde { \pi } |$ is a class $\mathcal { H } _ { \infty }$ function and by scaling the $l _ { 2 }$ norm of $\left| { \tilde { \mathbf { X } } } \right|$ with a positive constant k we can have $k | \tilde { \mathbf { x } } | > | \mathbf { s } |$ . Hence linear scaling with the rate k can be a candidate α function. The last inequality establishes ISS, namely the tracking error is bounded by parameter variations. In the derivation, the second inequality comes from the boundedness of the regressor matrix and sliding mode tracking error due to stability.

From the second inequality, it is suggested to use configurations that have a small induced norm of the regressor matrix to reduce the disturbance. In this paper, we compensate for the friction between the joints, which depends on the relative motion of two adjacent joints. The friction models between different joints can then be considered decoupled; we can estimate friction parameters for each joint independently.
