Now we transform the control-affine nonlinear system via the Koopman operator. The time derivative of Ψ is

$$
\begin{array}{l} \frac {\mathrm{d} \Psi}{\mathrm{d} t} = \nabla \Psi \cdot f (x) + \sum_ {i = 1} ^ {m} u _ {i} \nabla \Psi \cdot g _ {i} (x) + \nabla \Psi \cdot \bar {d} \tag {9} \\ = \mathcal {L} _ {f} \Psi + \sum_ {i = 1} ^ {m} u _ {i} \mathcal {L} _ {g _ {i}} \Psi + d. \\ \end{array}
$$

We further assume $\Psi ( 0 ) = 0 , \Psi ^ { - 1 } ( 0 ) \cap \mathbb { X } = \{ 0 \}$ and $\exists C \ \in \ \mathbb { R } ^ { n \times N } , \mathrm { s . t . } \ x \ \stackrel { } { = } \ C \Psi ( x )$ to facilitate Lyapunov based stability analysis. As shown in [8], if the eigenspace of $\mathcal { L } _ { f }$ is an invariant subspace of $\dot { \mathcal { L } } _ { g _ { i } } , i = \bar { 1 } , \cdot \cdot \cdot , m .$ , system (9) (hence (7)) without process disturbance is bilinearizable. If this is not satisfied, a projection error term can be introduced such that

$$\frac {\mathrm{d} \Psi (x)}{\mathrm{d} t} = A \Psi (x) + B _ {0} u + \sum_ {i = 1} ^ {m} u _ {i} B _ {i} \Psi (x) + r (\Psi (x), u) + d. \tag {10}$$

In data-driven control problem where $A , B _ { 0 }$ and $B _ { i } , i =$ $1 , \cdots , m$ are obtained from data, not only projection error but also estimation error resulting from finite data should be considered. The following lemma gives a proportional error bound on the approximation error.

Lemma 1 Suppose the collected data are independent and identically distributed (i.i.d.) in X. Then, there exist constants $c _ { 1 } , c _ { 2 }$ such that the approximation error term in (10) including projection error and estimation error is bounded by (11) for all $( x , u ) \in \mathbb { X } \times \mathbb { U }$ .

$$\left\| r (\Psi (x), u) \right\| \leq c _ {1} \| \Psi (x) \| + c _ {2} \| u \| \tag {11}$$
