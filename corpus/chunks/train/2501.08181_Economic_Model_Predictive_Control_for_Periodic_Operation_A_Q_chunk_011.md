# A. Proposed algorithm

Next, we introduce our single-layer E-MPC algorithm inspired by Newton’s optimization method, where the linearization point is selected to be the optimal artificial trajectory of the optimization problem solved at the previous time step. The resulting procedure is described in Algorithm 1. It can be noticed from (12) that $u _ { 0 } ^ { * }$ depends on the exogenous parameter ˆζ. A suitable initial value for $\hat { \boldsymbol { \zeta } }$ could be obtained by solving (13) first with $\widetilde O$ as any quadratic offset cost function (since this initial point does not need to be optimal but only feasible).

Algorithm 1 Approximated single-layer periodic E-MPC   
Require: Initial feasible $\hat{\zeta} = (\hat{\xi},\hat{\nu})$ .  
At each sampling time $k\geq 0$ :  
1: Read: $x,p$ 2: Solve QP: $(z^{*},\zeta^{*}) = \arg \widetilde{\mathbb{P}}_{k}(x,\hat{\zeta},p)$ 3: Apply: $u_0^*$ 4: Update: $\zeta\colon \hat{\xi}\gets \xi_{1}^{*},\hat{\nu}\gets [\nu_{1}^{*},\dots ,\nu_{T - 1}^{*},\nu_{0}^{*}]$
