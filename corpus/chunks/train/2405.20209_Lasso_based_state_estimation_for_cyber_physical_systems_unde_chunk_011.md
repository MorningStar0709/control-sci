Since $\rho : = \left\| \mathcal { O } _ { \bar { S } } ^ { \dagger \top } \mathcal { O } _ { S } ^ { \top } \right\| _ { \infty } = \operatorname* { m a x } _ { j } \left\| \mathcal { O } _ { S } \left( \mathcal { O } _ { \bar { S } } ^ { \dagger } \right) _ { j } \right\| _ { 1 }$ , a qualitative interpretation of (6) is as follows: the term $\rho$ must be small, i.e., the rows of $\mathcal { O } _ { \mathcal { S } }$ must be “sufficiently orthogonal” to the columns of $\mathcal { O } _ { \bar { s } } ^ { \dagger }$ . This implies some observations. Since $\widetilde { x } = \mathcal { O } _ { \bar { s } } ^ { \dagger } y _ { \bar { s } }$ , then

$$\mathcal {O} _ {\mathcal {S}} \widetilde {x} = \mathcal {O} _ {\mathcal {S}} \mathcal {O} _ {\bar {\mathcal {S}}} ^ {\dagger} y _ {\bar {\mathcal {S}}}. \tag {19}$$

In particular, $\mathcal { O } _ { S } \widetilde { x } = 0$ would be the ideal case to identify the attacks, which ewould be directly observable from $y s = a s$ . Nevertheless, this is not realistic, in particular because it depends on the specific initial state x. However,

$$\left\| \mathcal {O} _ {\mathcal {S}} \widetilde {x} \right\| _ {1} \leq \rho \left\| y _ {\bar {\mathcal {S}}} \right\| _ {1} \tag {20}$$

that is, $\rho$ controls the “orthogonality” between the rows of O indexed in $s$ and x: a small irrepresentable term $\rho$ implies a small energy of $\mathcal { O } _ { S } \widetilde { x }$ , which, in turn, implies that the attacks are more exposed to identification.

Differently from previous conditions considered in the literature, see for example Theorem 4.4 in [19], the irrepresentable condition well captures the ability of CPS to identify attacks. We illustrate this point with a simple illustrative example.
