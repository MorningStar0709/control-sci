Being able to write an evolution that satisfies both φ and ψ fairly suffices to show the game is collaborative.
Two different agent modes affect how the game plays out.
An agent is deemed social if it is aware of the other agent in the environment and is willing to cooperate with it towards the best mutual payoff, and unsocial otherwise.
An agent is deemed optimistic if it assumes its messages are perceived by the environment in the same order as the agent issues them, and that all frames of reference are synchronized, and realistic otherwise.
