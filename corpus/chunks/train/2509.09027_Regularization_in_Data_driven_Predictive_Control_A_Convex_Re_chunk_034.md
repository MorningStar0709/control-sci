# 5.1 Sequential optimization

Our key idea is to adopt a sequential optimization strategy that addresses the constraints in (34) one at a time. The detailed procedure is described below.

Step 1: Low-rank approximation. Since the input data $u _ { \mathrm { d } }$ has no noise and satisfies the persistent excitation, we have rank $\left( H _ { u } \right) = m L$ . In contrast, the output data yd may contain “variance” noise and “bias” error. Thus, the rank of the raw data H $( i . e . , \mathrm { c o l } ( H _ { u } , H _ { y } ) )$ is larger than mL+n. We first consider the constraint (34a) that approximates H with a low-rank matrix, which is

$$\min _ {\tilde {H} _ {y}} \| H _ {y} - \tilde {H} _ {y} \| _ {F} \tag {35}$$

$\mathrm { s u b j e c t \ t o ~ \ r a n k } ( \tilde { H } ) = m L + n .$

This problem is slightly different from the standard lowrank approximation, as we do not change the input data $U _ { \mathrm { P } }$ and $\bar { U } _ { \mathrm { F } }$ . Still, we can get an analytical solution of (35) via singular value decomposition.

Proposition 5 Consider problem (35), where the input data $u _ { \mathrm { d } }$ has no noise and satisfies the persistent excitation. Let $\Pi _ { 2 } = H _ { u } ^ { \dagger } H _ { u }$ be the orthogonal projector onto the row space of $H _ { u }$ , and denote SVD for the component of $H _ { y }$ in the null space of $H _ { u }$ as $\begin{array} { r } { H _ { y } ( I - \Pi _ { 2 } ) = \sum _ { i = 1 } ^ { p L } \bar { \sigma } _ { i } \bar { u } _ { i } \bar { v } _ { i } ^ { \top } } \end{array}$ . Then, the optimal solution $H _ { y _ { 1 } }$ to (35) is given by

$$H _ {y _ {1}} := H _ {y} \Pi_ {2} + \sum_ {i = 1} ^ {n} \bar {\sigma} _ {i} \bar {u} _ {i} \bar {v} _ {i} ^ {\mathsf {T}}. \tag {36}$$

Since we have rank $\left( \begin{array} { l } { { \displaystyle H _ { u } \right) \ = \ m L } } \end{array}$ , the key insight for Proposition 5 is to ensure the part of $H _ { y }$ in the null space of $H _ { u }$ has rank n. Specifically, we first divide $H _ { y }$ into two parts that are $H _ { y } \Pi _ { 2 }$ in the row space of $H _ { u }$ and $H _ { y } ( I - \hat { \Pi } _ { 2 } )$ in the null space of $H _ { u }$ . We then perform an SVD of $H _ { y } ( I - \Pi _ { 2 } )$ to estimate its rank-n approximation, and finally combine it with the component of $H _ { y }$ in the row space of $H _ { u }$ as in (36). A detailed proof is provided in Appendix B.5.

For notational simplicity, we denote the mapping from the data $H _ { y }$ to the optimal solution $H _ { y _ { 1 } }$ of (35) as $\Pi _ { \mathrm { L } }$ .
