# Lemma 2 Suppose Assumptions 1 and 2 hold. The set $\displaystyle { { \cal { K } } _ { \beta } }$ in (6) satisfies the properties below:

(1) It is always nonempty, open, and path-connected;   
(2) It is, in general, nonconvex and unbounded.

Proof: The nonemptiness of $\displaystyle { { \cal { K } } _ { \beta } }$ follows directly from Assumption 1. To establish openness, we apply Lemma 1 to the transfer function $\mathbf { T } _ { \infty } ( \bar { K } )$ . This ensures that $K \in \mathcal { K } _ { \beta }$ if and only if there exists $X \succ 0$ (the positive definiteness is ensured by $W \succ 0 )$ satisfying the strict Riccati inequality

$$A _ {K} X + X A _ {K} ^ {\mathsf {T}} + \beta^ {- 2} X S _ {K} X + W \prec 0. \tag {14}$$

Since the inequality is strict, the same $X \succ 0$ ensures that the inequality holds for all $\dot { K ^ { \prime } }$ in a sufficiently small neighborhood around K. Hence, $\displaystyle { { \cal { K } } _ { \beta } }$ is open.

Path-connectivity can be established using a strategy similar to that in [22, Section III $\mathrm { A l }$ . By applying the Schur complement to (14) and introducing the change of variables $K \stackrel { \cdot } { = } Y X ^ { - 1 }$ , we see that $K \in \mathcal { K } _ { \beta }$ if and only if there exist matrices $X \succ 0$ and Y satisfying $\mathrm { F } _ { \mathrm { c v x } } \prec 0$ , where

$$
\mathrm{F} _ {\mathrm{cvx}} := \left[ \begin{array}{c c c} A X + X A ^ {\mathsf {T}} + B Y + Y ^ {\mathsf {T}} B ^ {\mathsf {T}} + W & X Q _ {\infty} ^ {1 / 2} & Y ^ {\mathsf {T}} R _ {\infty} ^ {1 / 2} \\ Q _ {\infty} ^ {1 / 2} X & - \beta^ {2} I & 0 \\ R _ {\infty} ^ {1 / 2} Y & 0 & - \beta^ {2} I \end{array} \right]. \tag {15}
$$

The above inequalities define a convex set in the lifted variables $( X , Y )$ , and since the map $K = Y X ^ { - 1 }$ is continuous, it follows that $\displaystyle { { \cal { K } } _ { \beta } }$ is path-connected.

While the nonconvexity of $\displaystyle { { \cal { K } } _ { \beta } }$ is well-known, its unboundedness has not been discussed 1 . The example below illustrates the nonconvexity and unboundedness of $\displaystyle { { \cal { K } } _ { \beta } }$ . ✷
