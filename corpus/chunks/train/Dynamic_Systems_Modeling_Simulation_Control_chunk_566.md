Equation (10.32) shows that the constant $K _ { \mathrm { s p } }$ is equal to the DC gain of the forward transfer function, or $G ( s = 0 )$ . Next, observe how the system type, Eq. (10.31), affects $K _ { \mathrm { s p } } \mathrm { : }$ if N = 0 (type 0 system), then the DC gain $G ( s = 0 )$ is finite and hence $K _ { \mathrm { s p } }$ is a finite number. However, if $N \geq 1$ (type 1 or higher), then computing the DC gain $G ( s = 0 )$ results in dividing by zero and $K _ { \mathrm { s p } } = \infty$ . Consequently, the steady-state error for a unit-step input is $e ( \infty ) = 1 / \infty = 0$ . In other words, the closed-loop system output can perfectly track a step input at steady state if we have a type 1 (or higher) system. It is important to note that the single “free integrator” may reside naturally in the plant transfer function $G _ { P } ( s )$ or it can be inserted into the forward path by using a PI or PID controller.
