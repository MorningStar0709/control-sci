# IV. DISTRIBUTED BACKSTEPPING FUZZY SLIDING MODE CONTROLLER WITH INPUT SATURATION

The backstepping SMC presented in section III has two main shortcomings: (i) the bigger sliding gain is chosen based on Assumption 1, for which if the disturbance is big, the controller provides a big chattering in the system, (ii) the saturated control torque effects have not been considered. In this section, we introduce an auxiliary variable and a fuzzy approximation to overcome these shortcomings.

The designed control input $\tau _ { i } ( t ) \in \mathfrak { R } ^ { n }$ is affected by the saturation nonlinearity and can be expressed as [41]:

$$
u _ {i} (\tau_ {i} (t)) = \left\{ \begin{array}{l l} \text { sign } (\tau_ {i} (t)) \tau_ {\max _ {i}}, & | \tau_ {i} (t) | \geq \tau_ {\max _ {i}}, \\ \tau_ {i} (t), & | \tau_ {i} (t) | <   \tau_ {\max _ {i}}, \end{array} \right. \tag {33}
$$

where $\tau _ { \mathrm { m a x } _ { i } }$ represents the maximum control torque allowed for joint i.

Furthermore, considering the input saturation, the saturated control torque can be approximated by

$$u _ {i} (\tau_ {i}) = g _ {i} (\tau_ {i}) + \varsigma_ {i} (\tau_ {i}), \tag {34}$$

where $g _ { i } ( \tau _ { i } )$ is a smooth function. $\varsigma _ { i } \big ( \tau _ { i } \big )$ is the bounded approximation error. $g _ { i } ( \tau _ { i } )$ can be chosen as [41]:

$$
\begin{array}{l} g _ {i} \left(\tau_ {i}\right) = \tau_ {\max _ {i}} \times \tanh \left(\frac {\tau_ {i}}{\tau_ {\max _ {i}}}\right) \tag {35} \\ = \tau_ {\max _ {i}} \frac {e ^ {\tau_ {i} / \tau_ {\max _ {i}}} - e ^ {- \tau_ {i} / \tau_ {\max _ {i}}}}{e ^ {\tau_ {i} / \tau_ {\max _ {i}}} + e ^ {- \tau_ {i} / \tau_ {\max _ {i}}}}. \\ \end{array}
$$

The approximation error $\varsigma _ { i } ( \tau _ { i } )$ is bounded by

$$\left| \varsigma_ {i} \left(\tau_ {i}\right) \right| = \left| u _ {i} \left(\tau_ {i}\right) - g _ {i} \left(\tau_ {i}\right) \right| \leq \tau_ {\max _ {i}} (1 - \tanh (1)) = \bar {\Delta} _ {i}. \tag {36}
\begin{array}{l l} \text {Let:} g (\tau (t)) & = [ g _ {1} (\tau_ {1} (t)), g _ {2} (\tau_ {2} (t)),..., g _ {N} (\tau_ {N} (t)) ] ^ {T}, \\ \varsigma (\tau (t)) = & [ \varsigma_ {1} (\tau_ {1} (t)), \varsigma_ {2} (\tau_ {2} (t)),..., \varsigma_ {N} (\tau_ {N} (t)) ] ^ {T}. \end{array} \quad \text {To}
$$

compensate for the saturated controller, an adaptive auxiliary variable $\mu$ is introduced as:

$$\dot {\mu} = - \mu + \bar {J} (\eta_ {2}) \bar {\Pi} (g (\tau) - \tau). \tag {37}$$
