# Example 5.17

Use the computer to plot the BAMP of

$$G (s) = \frac {(s ^ {2} + 2 \zeta 1 0 0 s + 1 0 ^ {4})}{(s + 1 0) (s + 3 0) (s + 5 0 0)}$$

For ζ = {0.05, 0.1, 0.25, 0.5, 0.75, 0.99}. Choose a frequency range which shows all the features of G(s) (all the poles and zeros). Make all the plots superimposed on the same axes.
