$$f (H x) - f (H \hat {x}) = \mathbf {D} f (H c) (H x - H \hat {x}) \tag {9}$$

where D is the Jacobian operator. However, (9) is well-known to be an incorrect extension of MVT to vector-valued functions. A counter example is $f ( H x ) = [ \quad$ cos(x1) sin(x1) ]T for $x = { \left[ \begin{array} { l l } { x _ { 1 } } & { x _ { 2 } } \end{array} \right] } ^ { \mathtt { T } }$ and $\hat { x } = [ \begin{array} { l l } { \hat { x } _ { 1 } } & { \hat { x } _ { 2 } } \end{array} ] ^ { \mathtt { T } }$ with $x _ { i } , { \hat { x } } _ { i } \in \mathbb { R }$ . Now, for $x _ { 1 } = 2 \pi , { \hat { x } } _ { 1 } = 0 .$ , and any $x _ { 2 } , { \hat { x } } _ { 2 } \in \mathbb { R }$ , we have the left-hand side of (9) equal to zero; however, the right-hand side is non-zero with $H x - H { \hat { x } } \neq 0$ for $x - { \hat { x } } \not \in \ker ( H )$ and

$$
\mathbf {D} f (H c) = \left[ \begin{array}{c c} - \sin (c _ {1}) & 0 \\ 0 & \cos (c _ {1}) \end{array} \right] \neq 0,   \forall   c \in \mathbb {R} ^ {2}.
$$

Therefore, the results (8) and (9) claimed in [44] do not hold in general. See [29, Section 1.3.2 (b)] for a correct extension of MVT, which is in an integral form.

3) Design using one-sided Lipschitz and quadratically inner boundedness properties: A function $f : \mathcal X \to { \mathbb { R } } ^ { n _ { f } }$ is said to be one-sided Lipschitz (OSL) if, for every $x , { \hat { x } } \in { \mathcal { X } }$ , there exists $\varphi \in \mathbb { R }$ such that

$$\langle G (f (H x) - f (H \hat {x})), x - \hat {x} \rangle \leq \varphi \| x - \hat {x} \| ^ {2}. \tag {10}$$

It is said to be quadratically inner bounded (QIB) if, for every $x , { \hat { x } } \in { \mathcal { X } }$ , there exist $\varrho _ { 1 } , \varrho _ { 2 } \in \mathbb { R }$ such that

$$
\begin{array}{l} \left\| G (f (H x) - f (H \hat {x})) \right\| ^ {2} \leq \varrho_ {1} \| x - \hat {x} \| ^ {2} \tag {11} \\ + \varrho_ {2} \langle G (f (H x) - f (H \hat {x})), x - \hat {x} \rangle . \\ \end{array}
$$

See [45] for methods to compute the constants $\varphi , \varrho _ { 1 } , \varrho _ { 2 }$

The design criteria of a Luenberger-like observer (7) using the OSL and QIB properties was first provided in [46]. However, the design is still conservative as the one-sided Lipschitz property cannot be used directly. $$\dot {V} = e ^ {\mathrm{T}} \left[ (A - L C) ^ {\mathrm{T}} P + P (A - L C) \right] e + 2 e ^ {\mathrm{T}} P G \tilde {f}$$
