# 2. CONTROL SYSTEM

We consider a generic class of perturbed second-order dynamic systems described by

$$\dot {x} _ {1} (t) = x _ {2} (t),\dot {x} _ {2} (t) = u (t) + d (t), \tag {1}$$

where the unknown disturbance value $\ b { d } ( t ) \in \mathbb { R }$ is matched with the control quantity u(t), and is (Lebesgue) measurable, while $\begin{array} { r } { \| d \| _ { \infty } = \widehat { \mathrm { e s s } \operatorname* { s u p } } _ { t \geq 0 } | d ( t ) | \ \leq \ D . } \end{array}$ . The dynamic state variables are assumed to be available for a feedback control and denoted in the vector form (for the sake of a brief notation in course of analysis) by $x ( t ) =$ $\left( x _ { 1 } ( t ) , x _ { 2 } ( t ) \right) ^ { \top } \in \mathbb { R } ^ { 2 }$ . The upper-bound $0 < D < \infty$ o f dynamic disturbances is assumed to be known. Further we note that the inertial and input-gaining terms, in case of a physical plant model equivalent to (1), can be correspondingly incorporated into the control parameters when designing u(t).

In Ruderman and Efimov (2025), a novel quasi-continuous robust control $u ( t )$ with a fast nonlinear damping law (introduced in Ruderman (2021) and detailed with experiments in Ruderman (2022)) was proposed for (1). The existence of solutions, uniform asymptotic stability, finite time convergence, and robustness against the disturbances $\| d \| _ { \infty } \leq D$ of the closed-loop system were shown with an accompanying analysis in Ruderman and Efimov (2025). Note that this control law was only defined on the invariant for the closed-loop system set $\{ x \in \mathbb { R } ^ { 2 } : x _ { 1 } \neq 0 \} \cup \{ 0 \}$ . The phase portrait of that control system is qualitatively shown in Fig. 1 by the thin blue lines. Note that the depicted notation of the quadrants I.-IV. in the phase-plane will further be used in analysis of the control system.
