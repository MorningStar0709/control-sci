$$
\left[ \Pi_ {\mathbb {U} ^ {N}} (\mathbf {u}) \right] _ {i} = \left\{ \begin{array}{l l} \underline {{\mathbf {u}}} _ {i}, & \mathbf {u} _ {i} \leq \underline {{\mathbf {u}}} _ {i} \\ \mathbf {u} _ {i}, & \underline {{\mathbf {u}}} _ {i} <   \mathbf {u} _ {i} <   \overline {{\mathbf {u}}} _ {i} \\ \overline {{\mathbf {u}}} _ {i}, & \mathbf {u} _ {i} \geq \overline {{\mathbf {u}}} _ {i}. \end{array} \right.
$$

It is obvious that min-max functions are closed under addition, multiplication, minimization and maximization operations. Hence, combining the min-max expression of the projection operator, $[ \Pi _ { \mathbb { U } ^ { N } } ( \mathbf { u } ) ] _ { i } = \operatorname* { m i n } ( \overline { { \mathbf { u } } } _ { i } , \operatorname* { m a x } ( \mathbf { u } _ { i } , \mathbf { \underline { { u } } } _ { i } ) )$ ), with the update rules for the first order methods presented before (PGD and APGD), one can notice that after K iterations of one of these methods, (7) or (8), rolling backward, we obtain again a min-max representation for the last iterate ${ \mathbf { u } } _ { K } .$ , denoted ${ \bf u } _ { K } ^ { \tt F O M } ( x _ { 0 } )$ , which depends on the starting point $\mathbf { u } _ { 0 } .$ , initial state x0 and problem data:

$$\mathbf {u} _ {K} ^ {\mathrm{FOM}} (x _ {0}) = \mathcal {M} _ {i n} ^ {\mathrm{FOM}} \mathcal {M} _ {a x} ^ {\mathrm{FOM}} (x _ {0}; \mathbf {H}, \mathbf {q}, \underline {{\mathbf {u}}}, \overline {{\mathbf {u}}}, \mathbf {u} _ {0}, K). \tag {9}$$

From the exiting convergence guarantees for first order methods (i.e., for PGD and APGD), one can easily obtain a certificate about how far is the approximate min-max map (9) from the optimal min-max map (6), see [21]:

$$\left\| \mathbf {u} ^ {*} (x _ {0}) - \mathbf {u} _ {K} ^ {\mathrm{FOM}} (x _ {0}) \right\| ^ {2} \leq \mathcal {O} \left((1 - \kappa^ {\mathrm{FOM}}) ^ {K}\right) \quad \forall x _ {0} \in \mathbb {X}, \tag {10}$$

where $\kappa ^ { \mathsf { F O M } } = \kappa$ for PGD and $\kappa ^ { \mathsf { F O M } } = \sqrt { \kappa }$ for APGD (here, κ denotes the condition number of the matrix H). Note that the bound (10) is usually tight and is uniform in x0 $\in \mathbb { X }$ (the initial state of the system), provided that U is bounded set (i.e., in $\mathcal { O } ( \cdot )$ there is no dependence on $x _ { 0 } .$ , but on the diameter of U).
