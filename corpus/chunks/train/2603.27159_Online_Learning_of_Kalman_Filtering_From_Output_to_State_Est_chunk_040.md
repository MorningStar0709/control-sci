Lemma 5.6. Under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ , it holds with probability at least $1 - \delta / 1 2$ that

$$\sum_ {t = \tau} ^ {T - 1} \left(f _ {t} (M _ {t}) - f _ {t} (M ^ {\star}) - \mathbb {E} _ {b} \big [ f _ {t} (M _ {t}) - f _ {t} (M ^ {\star}) \big ]\right) \leq \left(8 R _ {x} ^ {2} + 8 h R _ {y} ^ {2} R _ {M} ^ {2}\right) \sqrt {2 T \log \frac {2 4}{\delta}}. \tag {43}$$

Recalling from Lemma 3.1 that $\mathbb { P } ( \mathcal { E } _ { w } \cap \mathcal { E } _ { v } ) \geq 1 - 2 \delta / 3$ and $\mathbb { P } ( \mathcal { E } _ { \tilde { v } } ) \geq 1 - \delta / 1 2$ as argued at the beginning of Section 5.3, we can combine Lemmas 5.4-5.6 and obtain from a union bound that under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v } ,$ , the following holds with probability at least $1 - \delta / 3$ :

$$\sum_ {t = \tau} ^ {T - 1} \big (f _ {t} (M _ {t}) - f _ {t} (M ^ {\star}) \big) = \mathcal {O} \big (\tau (\log^ {4} T) \log (T / \tau) + \sqrt {T} \log^ {2} T \big),$$

where we again use the facts deduced from (13)-(14), (26) and (41) that $h = \mathcal { O } ( \log T ) , l _ { x } = \mathcal { O } ( \log ^ { 2 } T )$ , $R _ { y } = \mathcal { O } ( \sqrt { \log T } ) , R _ { \tilde { v } } = \mathcal { O } ( \sqrt { \log T } ) , \beta _ { x } = \mathcal { O } ( \log ^ { 2 } T )$ and $R _ { M } = \mathcal { O } ( 1 )$ , where $\mathcal { O } ( \cdot )$ hides the factors stated in Theorem 5.2. In addition, we get from Lemma 3.3 that under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ ,

$$\sum_ {t = 0} ^ {\tau - 1} \left(f _ {t} (M _ {t}) - f _ {t} (M ^ {\star})\right) \leq \tau (R _ {x} ^ {2} + 4 h R _ {y} ^ {2} R _ {M} ^ {2}) = \mathcal {O} (\tau \log^ {2} T).$$

Combining the above two inequalities, we obtain that under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ , the following holds with probability at least $1 - \delta / 3$ :

$$\sum_ {t = 0} ^ {T - 1} \left(f _ {t} (M _ {t}) - f _ {t} (M ^ {\star})\right) = \mathcal {O} \big (\tau (\log^ {4} T) \log (T / \tau) + \sqrt {T} \log^ {2} T \big), \tag {44}$$

which completes upper bounding the online optimization regret in (42).

Truncation and steady-state regret. Investigating the proofs of Lemma 4.5-4.6, one can observe that the proofs also show that under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ ,

$$\sum_ {t = 0} ^ {T - 1} \left(f _ {t} (M ^ {\star}) - \left\| x _ {t} - \hat {x} _ {t} ^ {\mathrm{KF}} \right\| ^ {2}\right) = \mathcal {O} (\log^ {4} T). \tag {45}$$

Overall regret upper bound. Combining (44)-(45) and noting that $\mathbb { P } ( \mathcal { E } _ { w } \cap \mathcal { E } _ { v } ) \ge 1 - 2 / ( 3 \delta )$ , we complete the proof of Theorem 5.2.
