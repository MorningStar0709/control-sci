# B. 3rd order consensus

It has been shown that for $n \geq 3$ it is not possible to achieve scalable stability for any graph family $\left\{ G _ { N } \right\}$ where the corresponding graph Laplacian $L _ { N }$ has an eigenvalue with vanishing real part as the graph is growing, i.e. if lim ${ \cal N } {  } { \infty } \operatorname { R e } ( \lambda _ { 2 } ( L _ { N } ) ) = 0$ .
