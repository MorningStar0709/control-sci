# 2.1.1 Reduced optimality system

Problem (2.1)-(2.2) is equivalent to the following KTT system for (¯y, p, ¯ u¯) [34, Chap. 2, Theorem 1.4]:

$$
\left\{ \begin{array}{l l} - \Delta y = f + u \text {   in   } \Omega & \text { with } \quad y = 0 \text {   on   } \partial \Omega , \\ - \Delta p = y - y _ {d} \text {   in   } \Omega & \text { with } \quad p = 0 \text {   on   } \partial \Omega , \\ u = - \lambda^ {- 1} p \text {   in   } \Omega . \end{array} \right. \tag {2.3}
$$

The last line is the first-order necessary optimality condition of problem (2.1)–(2.2). Using the condition $u = - \lambda ^ { - 1 } p ,$ problem (2.1)-(2.2) reduces to the following system in y and $p \mathrm { : }$

$$
\left\{ \begin{array}{l l} - \Delta y = f - \lambda^ {- 1} p \text {   in   } \Omega & \text { with } \quad y = 0 \text {   on   } \partial \Omega , \\ - \Delta p = y - y _ {d} \text {   in   } \Omega & \text { with } \quad p = 0 \text {   on   } \partial \Omega . \end{array} \right. \tag {2.4}
$$

This problem has a unique solution tuple $( { \bar { y } } , { \bar { p } } )$ , and the optimal ¯u can be recovered ast $\bar { u } = - \lambda ^ { - 1 } \bar { p } .$

In view of (2.4) and the principle of PDE residual minimization, we define the following continuous loss

$$\mathcal {L} (y, p) = \left\| \Delta y + f - \lambda^ {- 1} p \right\| _ {L ^ {2} (\Omega)} ^ {2} + \alpha_ {i} \left\| \Delta p + y - y _ {d} \right\| _ {L ^ {2} (\Omega)} ^ {2} + \alpha_ {b} ^ {y} \left\| y \right\| _ {L ^ {2} (\partial \Omega)} ^ {2} + \alpha_ {b} ^ {p} \left\| p \right\| _ {L ^ {2} (\partial \Omega)} ^ {2}, \tag {2.5}$$

where $\alpha _ { i } > 0 , \alpha _ { b } ^ { y } > 0$ and $\alpha _ { b } ^ { p } > 0$ are penalty parameters to approximately enforce the PDE residual $\left( \operatorname { o f } p \right)$ and zero Dirichlet boundary conditions of y and p. Below we let ${ \pmb { \alpha } } = ( \alpha _ { i } , \alpha _ { b } ^ { y } , \alpha _ { b } ^ { p } )$ . The idea of utilizing NNs to parameterize the solution and then learning the NN parameters using a loss given by the least-squares of the PDE residual and boundary fitting was first proposed in the 1990s [32, 30, 31], and later greatly popularized by [43], under the name of physics informed neural networks (PINN). It has been successfully applied to a range of problems [27, 10]. We explore the use of NN for solving elliptic optimal control.
