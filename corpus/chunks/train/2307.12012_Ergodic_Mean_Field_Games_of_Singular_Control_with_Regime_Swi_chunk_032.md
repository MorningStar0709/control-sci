We next show that the map $\theta \mapsto \alpha _ { i } ( \theta )$ is right-continuous. Fix a sequence $\{ \theta ^ { n } \} _ { n \in \mathbb { N } } \subseteq \Theta$ such that $\theta ^ { n } \searrow \theta$ . Again by the monotonicity of the map $\theta \mapsto \alpha _ { i } ( \theta )$ , arguing by contradiction we assume that there exists $\delta > 0$ such that $\alpha _ { i } ( \theta ^ { n } ) \leq \alpha _ { i } ( \theta ) - \delta$ for any $n \in \mathbb { N }$ . Similarly to the first part of the proof, we find

$$\int_ {\theta} ^ {\theta^ {n}} \big (- v _ {\theta} (\alpha_ {i} (\theta^ {n}), i; y) \big) d y = \int_ {\alpha_ {i} (\theta^ {n})} ^ {\alpha_ {i} (\theta)} \big (- v _ {x} (y, i; \theta^ {n}) \big) d y.$$

which leads to the following inequality

$$0 < \min _ {y \in [ \alpha_ {i} (\theta) - \delta , \alpha_ {i} (\theta) - \epsilon ]} | v _ {x} (y, i; \theta) | (\delta - \epsilon) \leq \max _ {(x, y) \in \mathcal {K} \times \Theta} | v _ {\theta} (x, i; y) | (\theta^ {n} - \theta) \leq C (\theta - \theta^ {n}),$$

for K as in (4.1) and a constant C not depending on n. Thus, letting $n \to \infty$ , we obtain a contradiction.

By repeating the same argument, one can show that the map $\theta \mapsto \beta _ { i } ( \theta )$ is continuous, thus completing the proof. 
