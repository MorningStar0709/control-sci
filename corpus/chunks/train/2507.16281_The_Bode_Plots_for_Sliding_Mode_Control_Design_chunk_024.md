# A. Slow Motions due to Constant Perturbations

Let the exogenous input be defined as $\begin{array} { r } { f _ { 0 } ( \mathrm { s } ) = \frac { \eta } { \mathrm { s } } } \end{array}$ , where the magnitude satisfies $| \eta | \ll A K _ { n }$ . The corresponding bias level in the steady-state response (6) can be computed from the sensitivity TF (34) as

$$\sigma_ {0} ^ {*} = \lim _ {\mathrm{s} \rightarrow 0} \mathrm{s} \sigma_ {0} (\mathrm{s}) \Bigg | _ {f _ {0} (\mathrm{s}) = \frac {\eta}{\mathrm{s}}} = \lim _ {\mathrm{s} \rightarrow 0} \frac {\eta G (\mathrm{s})}{1 + K _ {n} W (\mathrm{s})}. \tag {35}$$

Once the EG given in (33) has been determined, the bias component in the sliding variable (6) can be computed using the expression (35), namely

$$\sigma_ {0} ^ {*} = \lim _ {\mathrm{s} \rightarrow 0} \frac {\eta G (\mathrm{s})}{1 + K _ {n} W (\mathrm{s})} = \eta \mu , \tag {36}$$

where $G ( \mathrm { s } ) = 1 / \mathrm { s } , G _ { a } ( \mathrm { s } ) = 1 / ( \mu \mathrm { s } + 1 ) ^ { 2 }$ , and $K _ { n } = 1 / \mu$ . Further simulations are carried out under the same conditions as those described in Example 2, namely with controller gain ρ = 5 and actuator time constant $\mu = 0 . 0 5$ .
