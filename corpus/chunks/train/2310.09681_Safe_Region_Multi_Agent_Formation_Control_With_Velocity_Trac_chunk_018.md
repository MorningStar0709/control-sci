Theorem 2 (refer [41]). Let $\mathcal { C } \subset \mathbb { R } ^ { n }$ be a set defined as the super level set of a continuously differentiable function h : $D \subset \mathbb { R } ^ { n }$ . If h is a control barrier function on D and $\textstyle { \frac { \partial h } { \partial x } } ( x ) \neq$ 0 for all $x \in \partial { \mathcal { C } } ^ { 1 }$ , then any Lipschitz continuous controller $u ( x ) \in K _ { c b f } ( x )$ for the system (16) renders the set C safe. Additionally, the set C is asymptotically stable in D.

One of the limitations of the CBF defined by (17) is that it requires the control input to appear in the first derivative of CBF which leads to complex CBFs for higherorder systems. The same approach can be extended to CBFs with arbitrarily high relative-degree. Such functions are known as exponential control barrier functions (ECBF). The ρth derivative of h(x) can be written as

$$h ^ {\rho} (x) = L _ {f} ^ {\rho} h (x) + L _ {g} L _ {f} ^ {\rho - 1} h (x) u. \tag {19}$$

We define $\eta _ { b }$ as-

$$\eta_ {b} (x) := [ h (x) \dot {h} (x)... h ^ {\rho - 1} (x) ] ^ {T}. \tag {20}$$

A $\rho$ times continuously differentiable function h as defined by (15) is an ECBF if there exists a row vector $K _ { \alpha } \in \mathbb { R } ^ { n }$ such that for the above control system (16), the following holds:

$$\sup _ {u \in U} [ L _ {f} ^ {\rho} h (x) + L _ {g} L _ {f} ^ {\rho - 1} h (x) u ] \geq - K _ {\alpha} \eta_ {b} (x) \forall x \in \mathcal {D}. \tag {21}$$

For the second-order CBF, the forward invariance condition boils down to

$$\ddot {h} + k _ {1} \dot {h} + k _ {0} h \geq 0, \tag {22}$$

which is the same as

$$L _ {f} ^ {2} h (\mathbf {x}) + L _ {g} L _ {f} h (\mathbf {x}) u + k _ {1} L _ {f} h (\mathbf {x}) + k _ {0} h (\mathbf {x}) \geq 0, \tag {23}$$

where the parameters $k _ { 0 } , k _ { 1 }$ should be selected such that the two roots of the polynomial $s ^ { 2 } + k _ { 1 } s + k _ { 0 }$ are negative real2.

Let $s _ { i k }$ be the closest point from the position of agent i $( p _ { i } ) _ { \cdot }$ , to the kth unsafe region inside the sensing region of agent i (partial or complete) as shown in figure (1). Using $\delta _ { e x }$ (≪ r) as the required safe distance for external safety, we define ECBF for interaction between agent i and kth unsafe region as

$$h _ {i k} = \left\| p _ {i} - s _ {i k} \right\| ^ {2} - \delta_ {e x} ^ {2}. \tag {24}$$
