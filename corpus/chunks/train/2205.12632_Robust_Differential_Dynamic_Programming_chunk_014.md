# B. Dualization for convexification

In general, we cannot assume that subblocks of $\overline { { \pmb { Q } } } _ { t }$ do not depend on decision variables. Nevertheless, convexification can be performed for a large class of $\overline { { \mathbf { Q } } } _ { t }$ . This class contains, for example, the case where $Q _ { t }$ is fixed and $M _ { t }$ satisfies (8).
