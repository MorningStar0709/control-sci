# 3.4 Parametric Convergence

$$\lim _ {t \to \infty} b _ {i} (t) = b _ {i}; \quad i = 1, \ldots , n _ {B}$$

is assured if:

(1) One uses a PAA which assures

$$\lim _ {t \to \infty} \varepsilon (t + 1) = \lim _ {t \to \infty} [ y (t + 1) - \hat {y} (t + 1) ] = 0$$

(2) The orders $n _ { A }$ and nB are known exactly.   
(3) The plant model to be identified is characterized by an irreducible transfer function in $z ^ { - 1 } ( i . e . , A ( q ^ { - 1 } )$ and $B ( q ^ { - 1 } )$ are relatively prime).   
(4) The input u(t) is a persistently exciting signal of order $n = n _ { A } + n _ { B }$ .

Proof The proof is a generalization of the example considered in Sect. 3.4.1 combined with the definition and properties of persistently exciting signals.

Under the assumption that $\begin{array} { r } { \operatorname* { l i m } _ { t \to \infty } \varepsilon ( t + 1 ) = 0 , } \end{array}$ , the adjustable predictor becomes asymptotically a fixed predictor described by:

$$y (t + 1) = \hat {\theta} ^ {T} \varphi (t) \tag {3.323}$$

where:

$$\hat {\theta} ^ {T} = \left[ \hat {a} _ {1}, \dots , \hat {a} _ {n _ {A}}, \hat {b} _ {1}, \dots , \hat {b} _ {n _ {B}} \right] \tag {3.324}$$

contains the final values of the estimated parameters $\begin{array} { r } { ( \hat { a } _ { i } = \operatorname* { l i m } _ { t  \infty } \hat { a } _ { i } ( t ) , \ \hat { b } _ { i } = } \end{array}$ $\mathrm { l i m } _ { t  \infty } \hat { b } _ { i } ( t ) )$ , and one has:

$$\lim _ {t \rightarrow \infty} \varepsilon (t + 1) = - \left(\sum_ {i = 1} ^ {n _ {A}} \left(a _ {i} - \hat {a} _ {i}\right) q ^ {- i + 1}\right) y (t) + \left(\sum_ {i = 1} ^ {n _ {B}} \left(b _ {i} - \hat {b} _ {i}\right) q ^ {- i + 1}\right) u (t) = 0 \tag {3.325}$$

Taking into account assumption (3) and replacing y(t) by:

$$y (t) = \frac {B (q ^ {- 1})}{A (q ^ {- 1})} u (t) \tag {3.326}$$

Equation (3.325) becomes:

$$
\begin{array}{l} \left[ - \left(\sum_ {i = 1} ^ {n _ {A}} (a _ {i} - \hat {a} _ {i}) q ^ {- i + 1}\right) B (q ^ {- 1}) + \left(\sum_ {i = 1} ^ {n _ {B}} (b _ {i} - \hat {b} _ {i}) q ^ {- i + 1}\right) A (q ^ {- 1}) \right] u (t) \\ = \left(\sum_ {i = 0} ^ {n _ {A} + n _ {B} - 1} \alpha_ {i} q ^ {- i}\right) u (t) = 0 \tag {3.327} \\ \end{array}
$$

with:

$$\alpha_ {0} = \Delta b _ {1} \tag {3.328}
\begin{array}{l} [ \alpha_ {1}, \dots , \alpha_ {n _ {A} + n _ {B} - 1} ] = \Delta b _ {1} [ a _ {1}, \dots , a _ {n _ {A}}, 0, \dots , 0 ] \\ + \left[ \Delta b _ {2}, \dots , \Delta b _ {n _ {B}}, - \Delta a _ {n _ {A}}, \dots , - \Delta a _ {1} \right] R (a _ {i}, b _ {i}) \tag {3.329} \\ \end{array}
$$

and condition (4) assures the desired persistence of excitation.

Further insight on the influence of the input characteristics upon the convergence of various adaptive schemes can be found in Anderson and Johnson (1982).
