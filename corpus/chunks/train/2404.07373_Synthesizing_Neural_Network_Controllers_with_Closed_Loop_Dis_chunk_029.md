# 5.1 Nonlinear Inverted Pendulum

In this example we consider the following inverted pendulum model:

$$\dot {\boldsymbol {x}} _ {\mathbf {1}} (t) = \boldsymbol {x} _ {\mathbf {2}} (t)\dot {\pmb {x}} _ {\mathbf {2}} (t) = - \frac {\mu}{m \ell^ {2}} \pmb {x} _ {\mathbf {2}} (t) + \frac {g}{\ell} \sin (\pmb {x} _ {\mathbf {1}} (t)) + \frac {1}{m \ell^ {2}} \pmb {u} (t)\pmb {y} (t) = \pmb {x _ {1}} (t)$$

where $x _ { 1 }$ is the angle of the pendulum from vertical, x2 is the angular velocity of the pendulum, u is the control input torque, $m = 0 . 1 5$ kg is the mass, $\ell = 0 . 5$ m is the length of the pendulum, and $\mu = 0 . 0 5$ Nms/rad is the coefficient of friction. This is modeled in the form of (1) by setting $\Delta _ { p } ( x _ { 1 } ) = \sin ( x _ { 1 } )$ . We characterize this uncertainty by the sector bound of [0, 1], which holds over $[ - \pi , \pi ]$ , and construct $M _ { \Delta _ { p } } = \left[ \begin{array} { c c } { { \bar { 0 } } } & { { 1 } } \\ { { 1 } } & { { - 2 } } \end{array} \right]$ .

Note: an alternative, equivalent, formulation would be to manipulate the dynamical model such that the LTI system is the linearization, and $\Delta _ { p } ( x _ { 1 } ) = \sin ( x _ { 1 } ) - x _ { 1 }$ . This can be implemented with $\begin{array} { r } { M _ { \Delta _ { p } } = \left[ \begin{array} { c c } { 0 } & { - 1 } \\ { - 1 } & { - 2 } \end{array} \right] } \end{array}$ to represent a sector bound of [−1, 0], which holds over $[ - \pi , \pi ]$ .

We compare the performances of the controllers on this inverted pendulum with a reward constructed to incentivize minimal control effort. The reward for each step of a rollout, which is of length 201, is set to $r ( x , \bar { u } ) ~ = ~ \mathrm { e x p } ( - u ^ { 2 } )$ . Initial conditions for the inverted pendulum are sampled uniformly at random with $x _ { 1 } \in \left[ - 0 . 6 \pi , 0 . 6 \pi \right]$ rad and $x _ { 2 } \in [ - 2 , 2 ]$ rad/s. If, during a rollout, $| x _ { 1 } | > \pi$ rad or $| x _ { 2 } | > \mathrm { \AA } \mathrm { r a d / \bar { s } }$ , then the rollout is terminated and the remainder of the rollout is given a reward of 0. The control input is limited (saturated) to remain in the interval [−2, 2]. We simulate using a time step of 0.01 seconds.
