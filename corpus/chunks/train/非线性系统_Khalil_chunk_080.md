# 2.30 一个生化反应器可由模型

$$\dot {x} _ {1} = \left(\frac {\mu_ {m} x _ {2}}{k _ {m} + x _ {2}} - d\right) x _ {1}, \quad \dot {x} _ {2} = d (x _ {2 f} - x _ {2}) - \frac {\mu_ {m} x _ {1} x _ {2}}{Y (k _ {m} + x _ {2})}$$

表示,其中的状态变量和非负常数 $d, \mu_{m}, k_{m}, Y$ 和 $x_{2f}$ 与习题 1.22 中的定义相同, 设 $\mu_{m} = 0.5, k_{m} = 0.1, Y = 0.4, x_{2f} = 4$ 。

(a) 求出当 d > 0 时系统的所有平衡点, 并确定各平衡点的类型。  
(b) 研究当 d 变化时系统发生的分岔。  
(c) 构造 d=0.4 时的相图, 并讨论系统的特性。
