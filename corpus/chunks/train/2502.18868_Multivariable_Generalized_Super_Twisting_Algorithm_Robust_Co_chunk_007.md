# 3 Robust MGSTA Lyapunov control design

The main purpose of this paper is to provide an LMIbased procedure for MGSTA controller design, extending its applicability to a broader class of linear multivariable dynamic systems.

Naturally, we consider that system (1) can be viewed as a feedback connection of the linear, time-invariant subsystem $S _ { L }$ , with state space realization

$$\dot {\zeta} = A \zeta + E \sigma \tag {3}y = C \zeta + D \sigma \tag {4}$$

and the nonlinear subsystem $\mathcal { S } _ { N }$ that emerges from the synthesis of the MSTA control, that is

$$\dot {\sigma} = y + B u + f \tag {5}\dot {\eta} = c (\sigma) ^ {2} \sigma \tag {6}u = K _ {0} \zeta + c (\sigma) K _ {1} \sigma + K _ {2} \eta \tag {7}$$

The matrices $K _ { 1 } , K _ { 2 } \in \mathbb { R } ^ { m \times n }$ are the MSTA control gains, while $K _ { 0 } \in \mathbb { R } ^ { m \times r }$ is an additional gain introduced to facilitate the handling of internal dynamic coupling. The positive scalar function $c ( \sigma )$ is given by

$$c (\sigma) = \frac {1}{\sqrt {\| \sigma \|}} + \alpha : \mathbb {R} ^ {n} \rightarrow \mathbb {R} _ {+}, \quad \alpha > 0 \tag {8}$$

and is well-defined and differentiable for all $\sigma \neq 0$ .

Assuming, with no loss of generality, see [4], that matrix $B K _ { 2 } \ \in \ \bar { \mathbb { R } } ^ { n \times n }$ is non singular, applying the change of variables

$$z = \eta + (B K _ {2}) ^ {- 1} f \tag {9}$$

it is readily verified that, defining the state variable $x =$ $[ \sigma ^ { T } ~ z ^ { T } ] ^ { T } ~ \in ~ \mathbb { R } ^ { 2 n }$ , the nonlinear subsystem $\mathcal { S } _ { N }$ , has the state space realization

$$\dot {x} = L (\sigma) A _ {K} R (\sigma) x + L (\sigma) F _ {0} w + G _ {0} ^ {T} y \tag {10}\sigma = G _ {0} x \tag {11}$$

where w ${ \bf \Xi } ( t ) = c ( \sigma ) ^ { - 1 } ( B K _ { 2 } ) ^ { - 1 } \dot { f } ( t )$ . Moreover, the matrices $L ( \sigma ) = \mathrm { d i a g } ( I _ { n } , c ( \sigma ) I _ { n } ) , R ( \sigma ) = \mathrm { d i a g } ( c ( \sigma ) I _ { n } , I _ { n } )$ are diagonal and well-defined for all ${ \boldsymbol { \sigma } } \neq 0 , { \boldsymbol { F } } _ { 0 } = [ 0 \ I _ { n } ] ^ { T } \in$ R2n×n, $G _ { 0 } = [ I _ { n } 0 ] \in \mathbb { R } ^ { n \times 2 n }$ and, as expected, the equality ${ A _ { K } } = { A _ { 0 } } \dot { + } \dot { B _ { 0 } } { \bf \bar { K } }$ is established by denoting

$$
A _ {0} = \left[ \begin{array}{l l} 0 & 0 \\ I _ {n} & 0 \end{array} \right], B _ {0} = \left[ \begin{array}{l} B \\ 0 \end{array} \right], K = \left[ \begin{array}{l l} K _ {1} & K _ {2} \end{array} \right] \tag {12}
$$
