Cost function of the state estimation problem. Recall that in the problem of learning Kalman filtering for state estimation, the cost function is given by $f _ { t } ( M ) = \mathbf { \bar { \| } } x _ { t } - \hat { x } _ { t } ( M ) \mathbf { \| ^ { 2 } }$ with $\begin{array} { r } { \hat { x } _ { t } ( M ) = \sum _ { s = 1 } ^ { h } M _ { s } y _ { t - s } } \end{array}$ and $M = [ M _ { s } ] _ { s \in [ h ] }$ . Note that we may write

$$
\begin{array}{l} \hat {x} _ {t} (M) = \sum_ {s = 1} ^ {h} M _ {s} y _ {t - s} = [ M _ {s} ] _ {s \in [ h ]} [ y _ {t - s} ^ {\top} ] _ {s \in [ h ]} ^ {\top} \\ = \left(I _ {p} \otimes [ y _ {t - s} ^ {\top} ] _ {s \in [ h ]}\right) \cdot \boldsymbol {v e c} (M), \\ \end{array}
$$

where $v e c ( M ) \in \mathbb { R } ^ { n p h }$ concatenates the rows of M into a (column) vector. Further denoting

$$Y _ {t - 1: t - h} \triangleq I _ {p} \otimes [ y _ {t - s} ^ {\top} ] _ {s \in [ h ]}, \tag {22}$$

for a given $M \in \mathbb { R } ^ { n \times ( p h ) }$ , we can compactly write

$$f _ {t} (\pmb {v e c} (M)) = (Y _ {t - 1: t - h} \pmb {v e c} (M) - x _ {t}) ^ {\top} (Y _ {t - 1: t - h} \pmb {v e c} (M) - x _ {t}),$$

which yields

$$\nabla f _ {t} (\boldsymbol {v e c} (M)) = 2 Y _ {t - 1: t - h} ^ {\top} \left(Y _ {t - 1: t - h} \boldsymbol {v e c} (M) - x _ {t}\right), \tag {23}\nabla^ {2} f _ {t} (\boldsymbol {v e c} (M)) = 2 Y _ {t - 1: t - h} ^ {\top} Y _ {t - 1: t - h}. \tag {24}$$

To proceed, we introduce the following probabilistic events regarding the noise $w _ { t } , v _ { t }$ in system (1):

$$\mathcal {E} _ {w} = \left\{\max _ {0 \leq t \leq T - 1} \| w _ {t} \| \leq \sqrt {5 \operatorname{tr} (W) \log \frac {3 T}{\delta}} \right\}, \tag {25}\mathcal {E} _ {v} = \Big \{\max _ {0 \leq t \leq T - 1} \| v _ {t} \| \leq \sqrt {5 \operatorname{tr} (V) \log \frac {3 T}{\delta}} \Big \},$$

where $\delta \in ( 0 , 1 )$ . It can be shown that the above events hold with high probability; the proof of Lemma 3.1 is given in Appendix B.
