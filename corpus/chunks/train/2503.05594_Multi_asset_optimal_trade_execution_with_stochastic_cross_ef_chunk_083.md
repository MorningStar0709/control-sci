Remark B.3. Note that if there exists $\varepsilon \in ( 0 , \infty )$ such that $\mathcal { R } - \varepsilon I _ { n }$ is $\mathbb { S } _ { \geq 0 } ^ { n }$ -valued, then Assumption B.2 holds with $F = \mathcal { R } ^ { - 1 } \mathcal { Q }$ . In general, the process in Assumption B.2, if it exists, is not necessarily unique (up to $d P \times d s – n u l l \ \mathrm { s e t s } )$ . When we assume that Assumption B.2 holds, we choose and fix a process F with the properties detailed in Assumption B.2 and use it for all objects involved. Observe that in the case without a risk term, that $\mathrm { i s } , \mathcal { Q } = 0$ , we have that Assumption B.2 is satisfied with $F = 0$ .
