# A. Simulation Setup

The following numerical simulation was conducted in a MATLAB environment on a 2.8GHz desktop computer with 16GB of RAM. The scenario involves 5 quadrotor UAVs, governed by dynamics (1), which are initially located inside a cuboid inspection region of dimensions 180m × 180m × 40m. The objective of the multi-UAV system is to inspect an object of interest of dimensions 156m × 78m × 26m. More specifically, a set of 132 target points, being part of the object’s surface, have to be inspected by the multi-UAV system. To achieve this goal, density functions are used with the parameters $\alpha = 1$ and $\beta \ : = \ : 0 . 0 0 7 5$ while the sensing range of each UAV is chosen as r = 10m. The control gains are selected as $k _ { p } = 0 . 3 2 , k _ { d } = 0 . 8 6$ , and $\mu _ { o } = 1 0 0 0$ for a safety region $d _ { o } = 1 2 \mathrm { m }$ .
