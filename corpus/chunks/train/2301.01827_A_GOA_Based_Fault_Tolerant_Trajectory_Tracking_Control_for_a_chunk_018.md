The switching item $\pmb { \tau _ { c } }$ in sliding mode control can be defined as

$$\boldsymbol {\tau} _ {c} = - \mathbf {K} _ {1} \mathbf {s} - \mathbf {K} _ {2} | s | ^ {r} \text { sign } (\mathbf {s}), \tag {17}$$

where $s i g n ( \mathbf { s } )$ is the nonlinear sign function of s; $\mathbf { K } _ { 1 }$ and $\mathbf { K } _ { 2 }$ are positive coefficients, $\mathbf { K } _ { 1 } \ge \boldsymbol { \eta } + \boldsymbol { F }$ and ${ \bf K } _ { 2 } \ge \eta + { \bf \Delta }$ $F , ~ \eta$ is the design parameter which is always chosen as a positive constant; $0 ~ < ~ r ~ < ~ 1$ ; and F represents the upper bound of the difference between the system actual output and the estimation,

$$F = | f (\mathbf {v}) - \hat {f} (\mathbf {v}) |. \tag {18}$$

Additionally, an adaptive variation term $\widetilde { \tau } _ { e s t }$ is added to the control law, where $\hat { \widetilde { \tau } } _ { e s t } = \Gamma \mathbf { s }$ e and Γ represents a positive constant. Hence the final sliding mode control law is defined as

$$
\begin{array}{l} \pmb {\tau} = \hat {\pmb {\tau}} + \widetilde {\tau} _ {e s t} + \tau_ {\mathbf {c}} \\ = \hat {\mathbf {M}} (\dot {\mathbf {v}} _ {\mathbf {c}} + \frac {- k \dot {\mathbf {e}} _ {\mathbf {v}}}{2 \lambda} + \frac {\lambda}{2} \mathbf {e} _ {\mathbf {v}}) + \hat {\mathbf {C}} \mathbf {v} + \hat {\mathbf {D}} \mathbf {v} + \hat {\mathbf {g}} \\ + \widetilde {\tau} _ {e s t} - \mathbf {K} _ {1} \mathbf {s} - \mathbf {K} _ {2} | s | ^ {r} \text {sign} (\mathbf {s}). \tag {19} \\ \end{array}
$$
