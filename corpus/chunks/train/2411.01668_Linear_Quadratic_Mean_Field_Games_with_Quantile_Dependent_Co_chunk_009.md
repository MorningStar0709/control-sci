Remark 4.2 The solution above has a special structure that leads to further simplifications. If we denote $H _ { t } = \Pi _ { t } + P _ { t }$ , then summing up both sides of the dynamics for Π and $P$ yields the dynamics for H as follows:

$$- \dot {H} _ {t} = 2 a H _ {t} - \frac {b ^ {2}}{r} (H _ {t}) ^ {2}, \tag {28}$$

with the terminal condition $H _ { T } = 0$ . Clearly the solution is $H _ { t } = 0$ for all $t \in [ 0 , T ]$ ]. Then the MFG solution can be replaced by

$$u _ {i} ^ {*} (t) = - \frac {b}{r} \left(\Pi_ {t} x _ {i} (t) + \left(H _ {t} - \Pi_ {t}\right) \bar {x} (t)\right) = - \frac {b}{r} \Pi_ {t} \left(x _ {i} (t) - \bar {x} (t)\right) \tag {29}$$

where the control gains are given by

$$- \dot {\Pi} _ {t} = 2 a \Pi_ {t} - \frac {b ^ {2}}{r} \Pi_ {t} ^ {2} + q _ {t} ^ {\alpha}, \quad \Pi_ {T} = 0 \tag {30}\dot {V} (t) = 2 (a - \frac {b ^ {2}}{r} \Pi_ {t}) V (t) + \sigma^ {2}, \quad V (0) = V _ {0} \tag {31}$$

with $q _ { t } ^ { \alpha } = q ( 1 + e ^ { \sqrt { V ( t ) } \Phi _ { \alpha } ^ { - 1 } } ) \geq 0$ , and the mean field dynamics are given by

$$\dot {\overline {{{x}}}} = (a - \frac {b ^ {2}}{r} H _ {t}) \overline {{{x}}} = a \overline {{{x}}}, \quad \overline {{{x}}} (0) = \mu_ {0}, \tag {32}$$

consistent with the solution in (17). Such a decomposition of the control gains can be viewed as a special case of the decomposition for network-coupled control problems detailed in [10] using a spectral decoupling approach.

However, in general for mean field game problems with mean field couplings in the dynamics, the solution for $H _ { t } : = P _ { t } + \Pi _ { t }$ may not necessarily be zero since the boundary condition could be nonzero or the zero order term in the dynamics for $H _ { t }$ could be non-zero, and for such cases an extra condition is needed for the time horizon to account for the finite escape time. In addition, in cases where the state dimension is higher than 1, one cannot have the structure $H _ { t } = P _ { t } + \Pi _ { t }$ as the Riccati equation for $\Pi _ { t }$ will be a non-symmetric Riccati equation (see e.g., [9, 15]).
