We proceed by further estimating $\mathrm { s u p } _ { \xi ^ { n } \in \widehat { A } _ { b } } R _ { N } ( \xi ^ { n } )$ . From the latter inequality, since $\otimes _ { \ell \neq n } \nu ^ { \ell } \otimes \nu ^ { \xi ^ { n } }$ has compact support, we can use Assumption 5.1 to obtain

(5.8)

$$R _ {N} (\xi^ {n}) \leq C \int_ {\mathcal {O} ^ {N}} (1 + | x _ {n} | ^ {\beta}) \bigg | F \bigg (\sum_ {\ell \neq n} \frac {f (x _ {\ell})}{N - 1} \bigg) - F \big (\langle f, \nu^ {\theta^ {*}} \rangle \big) \bigg | \otimes_ {\ell \neq n} \nu^ {\ell} (d x _ {\ell}, i _ {\ell}) \otimes \nu^ {\xi^ {n}} (d x _ {n}, i _ {n})\leq C _ {\widehat {K}} \bigg (\int_ {\mathcal {O} ^ {N - 1}} \bigg | F \bigg (\sum_ {\ell \neq n} \frac {f (x _ {\ell})}{N - 1} \bigg) - F \big (\langle f, \nu^ {\theta^ {*}} \rangle \big) \bigg | \otimes_ {\ell \neq n} \nu^ {\ell} (d x _ {\ell}, i _ {\ell}) \bigg),$$

where the constant $C _ { \widehat { K } }$ depends only on the compact $\widehat { K }$ , since $\nu ^ { \xi ^ { n } }$ ha support in the compact $\widehat { K }$ $( \xi ^ { n } \in { \widehat { A } } _ { b }$ as in (5.3)). Next, using the local Lipschitz property of F (cf. Assumption (2b)-(2.3)), and again the fact that $\nu ^ { \ell }$ are supported in the compact $\widehat K \times \mathbb Y$ , we obtain

$$
\begin{array}{l} \sup _ {\xi^ {n} \in \widehat {A} _ {b}} R _ {N} (\xi) \leq C _ {\widehat {K}} \int_ {\mathcal {O} ^ {N - 1}} \left(1 + \sum_ {\ell \neq n} \frac {| f (x _ {\ell}) |}{N - 1} + \langle f, \nu^ {\theta^ {*}} \rangle\right) ^ {\frac {1}{\beta} - 1} \bigg | \sum_ {\ell \neq n} \frac {f (x _ {\ell})}{N - 1} - \langle f, \nu^ {\theta^ {*}} \rangle \bigg | \underset {\ell \neq n} {\otimes} \nu^ {\ell} (d x _ {\ell}, i _ {\ell}) \\ \leq \widehat {C} _ {\widehat {K}} \int_ {\mathcal {O} ^ {N - 1}} \left| \frac {1}{N - 1} \sum_ {\ell \neq n} f (x _ {\ell}) - \langle f, \nu^ {\theta^ {*}} \rangle \right| \otimes_ {\ell \neq n} \nu^ {\ell} (d x _ {\ell}, i _ {\ell}). \\ \end{array}
$$

Finally, by using the strong law of large numbers (cf. Theorem 4.23 in [36]) and the dominated convergence theorem, a limit as $N \to \infty$ in the latter inequality leads to (5.6). This completes the proof of the theorem. 
