Hence, if $\begin{array} { r } { \operatorname* { i n f } _ { \mathbf { x } \in O } C ( \mathbf { x } ) > \operatorname* { i n f } _ { \mathbf { y } \in O ^ { c } } C ( \mathbf { y } ) } \end{array}$ , then $B ( \mu )$ is strictly increasing and

$$R _ {k} (\rho , \rho_ {O}) < B (\mu) \Longrightarrow \int_ {O} \rho (\mathbf {x}) d \mathbf {x} < \mu .$$

Proof. Let $\mu \in [ 0 , 1 ]$ and ρ be such that $\begin{array} { r } { \int _ { O } \rho ( \mathbf { x } ) \mathrm { d } \mathbf { x } = \mu } \end{array}$ , and $\rho _ { i n } ( \mathbf { x } ) = \rho ( \mathbf { x } \mid O )$ , and $\rho _ { o u t } ( { \bf x } ) = \rho ( { \bf x } \mid O ^ { c } )$ . Then we have that

$$\rho = \mu \rho_ {i n} + (1 - \mu) \rho_ {o u t},$$

and

$$
\begin{array}{l} R _ {k} (\rho , \rho_ {O}) = R _ {k} \left(\mu \rho_ {\text { in }} + (1 - \mu) \rho_ {\text { out }}, \rho_ {O}\right) \\ = \mu R _ {k} (\rho_ {i n}, \rho_ {O}) + (1 - \mu) R _ {k} (\rho_ {o u t}, \rho_ {O}) \\ = \mu \int_ {O} C (\mathbf {x}) \rho_ {i n} (\mathbf {x}) d \mathbf {x} + (1 - \mu) \int_ {O ^ {c}} C (\mathbf {y}) \rho_ {o u t} (\mathbf {y}) d \mathbf {y} \\ \geq \mu \cdot \inf _ {\mathbf {x} \in O} C (\mathbf {x}) + (1 - \mu) \cdot \inf _ {\mathbf {y} \in O ^ {c}} C (\mathbf {y}). \\ \end{array}
$$

Hence, we have that

$$B (\mu) \geq \mu \cdot \inf _ {\mathbf {x} \in O} C (\mathbf {x}) + (1 - \mu) \cdot \inf _ {\mathbf {y} \in O ^ {c}} C (\mathbf {y}).$$

Furthermore, let $( z _ { n } ) \subset O$ and $( w _ { n } ) \subset O ^ { c }$ be such that

$$\lim _ {n \to \infty} C (z _ {n}) = \inf _ {\mathbf {x} \in O} C (\mathbf {x}), \qquad \lim _ {n \to \infty} C (w _ {n}) = \inf _ {\mathbf {y} \in O ^ {c}} C (\mathbf {y}).$$

Then for $\rho _ { n } = \mu \delta _ { z _ { n } } + ( 1 - \mu ) \delta _ { w _ { n } }$ n (or suitable smooth approximation of these) we have that

$$
\begin{array}{l} R _ {k} (\rho_ {n}, \rho_ {O}) = \mu \cdot C (z _ {n}) + (1 - \mu) \cdot C (w _ {n}) \\ \longrightarrow \mu \inf _ {\mathbf {x} \in O} C (\mathbf {x}) + (1 - \mu) \inf _ {\mathbf {y} \in O ^ {c}} C (\mathbf {y}), \\ \end{array}
$$

and so

$$B (\mu) \leq \mu \cdot \inf _ {\mathbf {x} \in O} C (\mathbf {x}) + (1 - \mu) \cdot \inf _ {\mathbf {y} \in O ^ {c}} C (\mathbf {y}),$$

which proves (24). The remaining statements follow readily from (24). □

Corollary 1. Under the hypotheses of Theorem 3, for every $\mu \in ( 0 , 1 )$ there exists $\varepsilon > 0$ such that $R _ { k } ( \rho , \rho _ { O } ) < \varepsilon$ implies $\begin{array} { r } { \int _ { O } \rho ( \mathbf { x } ) \mathrm { d } \mathbf { x } < \mu } \end{array}$ .

Proof. Taking $\varepsilon = B ( \mu )$ , the result follows from Theorem 3. □
