# 4.1 Coordinate transformation to error system

This subsection gives the closed-loop port-Hamiltonian system whose state variable is the tracking error between the state and its desired value.

For the port-Hamiltonian system (3), let us consider the desired trajectory $q ^ { \mathrm { d } } ( t )$ of the configuration variable q which is assumed to be bounded and twice differentiable. Let us define the target momentum $\boldsymbol { \eta } ^ { \mathrm { d } } ( \boldsymbol { q } , t )$ by

$$\eta^ {\mathrm{d}} (q, t) \equiv T (q) ^ {- 1} \dot {q} ^ {\mathrm{d}} (t),$$

and define the error coordinate of the state variable by

$$\binom{\tilde {q}}{\tilde {\eta}} = \binom{q - q ^ {\mathrm{d}} (t)}{\eta - \eta^ {\mathrm{d}} (q, t)}.$$

The following lemma shows the system (3) is converted into the error port-Hamiltonian system.

Lemma 2. Consider the system (3) with the input

$$u = G (q) ^ {- 1} \left(D (q, \eta) \eta^ {\mathrm{d}} (q, t) + \frac {\partial \eta^ {\mathrm{d}} (q , t)}{\partial q} T (q) \eta + T (q) ^ {- 1} \ddot {q} ^ {\mathrm{d}} (t) + v\right) \tag {24}$$

with $v \in \mathbb { R } ^ { n }$ . Then the system is converted into the following closed-loop system with Hamiltonian function $\tilde { H } ( \tilde { \eta } ) =$ $( 1 / 2 ) \| \tilde { \eta } \| ^ { 2 }$

$$\binom {\dot {\tilde {q}}} {\dot {\tilde {\eta}}} = \binom {0 _ {n} T (q)} {- T (q) ^ {\top} - D (q, \eta)} \binom {\nabla_ {\tilde {q}} \tilde {H} (\tilde {\eta})} {\nabla_ {\tilde {\eta}} \tilde {H} (\tilde {\eta})} + \binom {0 _ {n}} {I _ {n}} v. \tag {25}$$

Proof. The time derivative of q˜ is calculated as

$$
\begin{array}{l} \dot {\tilde {q}} = \dot {q} - \dot {q} ^ {\mathrm{d}} (t) \\ = T (q) \eta - T (q) \eta^ {\mathrm{d}} (t) = T (q) \tilde {\eta}. \tag {26} \\ \end{array}
$$

On the other hand, the time derivative of $\boldsymbol { \eta } ^ { \mathrm { d } } ( \boldsymbol { q } , t )$ is derived as

$$
\begin{array}{l} \dot {\eta} ^ {\mathrm{d}} (q, t) = \frac {\partial \eta^ {\mathrm{d}} (q , t)}{\partial q} \dot {q} + T (q) ^ {- 1} \ddot {q} ^ {\mathrm{d}} (t) \\ = \frac {\partial \eta^ {\mathrm{d}} (q , t)}{\partial q} T (q) \eta + T (q) ^ {- 1} \ddot {q} ^ {\mathrm{d}} (t). \tag {27} \\ \end{array}
$$

Substituting (24) into the system (3) and using (27), we obtain

$$
\begin{array}{l} \dot {\eta} = - D (q, \eta) \eta + G (q) u \\ = - D (q, \eta) (\eta - \eta^ {\mathrm{d}} (q, t)) + \dot {\eta} ^ {\mathrm{d}} (q, t) + v \\ \dot {\tilde {\eta}} = - D (q, \eta) \tilde {\eta} + v. \tag {28} \\ \end{array}
$$
