# 5.5.1 Problem Formulation

Consider $N > 1$ robotic agents operating in a workspace with $M > 0$ objects; W is a bounded open ball in 3D space, i.e., $\mathcal { W } : = \boldsymbol { B } ( \boldsymbol { 0 } , \boldsymbol { r } _ { 0 } )$ , where $r _ { 0 } \in$ $\mathbb { R } _ { > 0 }$ is the radius of . The objects are represented by rigid bodies whereas the robotic agents are fully actuated and consist of a fully actuated holonomic moving part (i.e., mobile base) and a robotic arm, having, therefore, access to the entire workspace. Within there exist $K > 1$ smaller spheres around points of interest, which are described by $\pi _ { k } : = \bar { B } ( p _ { \pi _ { k } } , r _ { \pi _ { k } } )$ , where $p _ { \pi _ { k } } \in \mathbb { R } ^ { 3 }$ is the center and $r _ { \pi _ { k } } \in \mathbb { R } _ { > 0 }$ the radius of $\pi _ { k }$ . We denote the set of all $\pi _ { k }$ as $\Pi : = \{ \pi _ { 1 } , \ldots , \pi _ { K } \}$ and ${ \ K _ { \mathcal { R } } } : = \{ 1 , \ldots , K \}$ . Moreover, we introduce disjoint sets of atomic propositions $\Psi _ { i } , \Psi _ { j } ^ { o }$ , expressed as boolean variables, that represent services provided to agent $i \in \mathcal N$ and object $j \in \mathcal { M } : = \{ 1 , \dots , M \}$ in Π. The services provided at each region $\pi _ { k }$ are given by the labeling functions $\mathcal { L } _ { i } : \Pi \ :  \ : 2 ^ { \Psi _ { i } } , \mathcal { L } _ { i } ^ { o } : \Pi \ :  \ : 2 ^ { \Psi _ { j } ^ { O } }$ , which assign to each region $\pi _ { k } , k \in \mathcal { K } _ { \mathcal { R } }$ , the subset of services $\Psi _ { i }$ and $\Psi _ { j } ^ { o }$ , respectively, that can be provided in that region to agent $i \in \mathcal N$ and object $j \in \mathcal { M }$ , respectively. In addition, we consider that the agents and the object are initially $( t = 0 )$ in the regions of interest $\pi _ { i n i t ( i ) } , \pi _ { i n i t _ { O } ( j ) }$ , where the functions init : $\mathcal { N } \to \mathcal { K } _ { \mathcal { R } }$ $i n i t _ { o } : \mathcal { M } \to \mathcal { K } _ { \mathcal { R } }$ specify the initial region indices.
