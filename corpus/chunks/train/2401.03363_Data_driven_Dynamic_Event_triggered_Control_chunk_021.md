# APPENDIX A PROOF OF THEOREM 1

Some useful lemmas are introduced in the following.

Lemma 1: [23] Consider the system (1) with Assumptions 1–4. Then it holds that $A + B K = ( \chi _ { 1 } - \mathcal { D } _ { 0 } ) \mathcal { G }$ , where the matrix $\mathcal { G } \in \mathbb { R } ^ { \tau \times n }$ is any solution of

$$
\left[ \begin{array}{c} K \\ \mathbf {I} \end{array} \right] = \left[ \begin{array}{c} \mathcal {U} _ {0} \\ \mathcal {X} _ {0} \end{array} \right] \mathcal {G}. \tag {25}
$$

Lemma 2: [23] Suppose that Assumptions 1–4 and (10) hold. Then, for any $\mathcal { D } \in \operatorname { D } ,$ it holds that

$$(\mathcal {X} _ {1} - \mathcal {D}) \mathcal {Y} + \mathcal {Y} ^ {T} (\mathcal {X} _ {1} - \mathcal {D}) ^ {T} + \Omega \prec 0. \tag {26}$$

It is time to present the proof of Theorem 1.

Proof The Lyapunov function is constructed as

$$V (t) = x (t) ^ {T} \mathcal {P} x (t) + \frac {1}{\delta} f (t), \tag {27}$$

which is well-defined since $\mathcal { P } = ( \mathcal { X } _ { 0 } \mathcal { Y } ) ^ { - 1 } \succ 0$ and $f ( t ) \geq 0$ .

Substituting (6) and (8) into (1) yields the following closedloop system

$$\dot {x} (t) = (A + B K) x (t) + B K e (t) + d (t). \tag {28}$$

Note from Lemma 1 that $A + B K = ( \chi _ { 1 } - \mathcal { D } _ { 0 } ) \mathcal { G }$ and from (12) that $B K = B \mathcal { U } _ { 0 } \mathcal { Q } = ( \mathcal { X } _ { 1 } - A \mathcal { X } _ { 0 } - \mathcal { D } _ { 0 } ) \mathcal { Q } = ( \mathcal { X } _ { 1 } - \mathcal { D } _ { 0 } ) \mathcal { Q } .$
