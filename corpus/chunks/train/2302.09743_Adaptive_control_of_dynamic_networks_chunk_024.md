Figure 4 illustrates the AC algorithm on an example dynamic network. Initially, in temporal subgraph $\mathrm { G } _ { 1 }$ , the metric $\{ \mathsf { q } _ { 1 } = 0 , \mathsf { q } _ { 2 } =$ $0 , \mathbf { q } _ { 3 } = 0 , \mathbf { q } _ { 4 } = 0 \}$ is calculated by formula (2), resulting in an MDS of $\left\{ \mathbf { V } _ { 3 } , \mathbf { V } _ { 4 } \right\}$ . Subsequently, when the network evolves to $\mathrm { G } _ { 2 } .$ , the node metric q is recalculated by formula (2), and an augmenting path search is performed by prioritizing the metric q in ascending order, leading to an MDS of $\left\{ \mathbf { V } _ { 3 } , \mathbf { V } _ { 4 } \right\}$ (Figure 4A). This procedure is repeated until the network no longer undergoes any further changes.
