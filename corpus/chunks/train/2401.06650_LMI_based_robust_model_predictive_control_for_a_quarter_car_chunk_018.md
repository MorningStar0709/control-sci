# A. System Description

Following the representation described in Section II-C, the general form of the linear discrete-time system, subject to bounded disturbances and norm-bounded structured uncertainty is illustrated as [23]:

$$
\left[ \begin{array}{c} x _ {k + 1} \\ q _ {k} \\ f _ {k} \\ z _ {k} \end{array} \right] = \begin{array}{c} n \\ n _ {q} \\ n _ {f} \\ n _ {z} \end{array} \left[ \begin{array}{c c c c} A & B _ {u} & B _ {p} & B _ {d} \\ C _ {q} & D _ {q u} & 0 & 0 \\ C _ {f} & D _ {f u} & D _ {f p} & D _ {f d} \\ C _ {z} & D _ {z u} & D _ {z p} & D _ {z d} \end{array} \right] \left[ \begin{array}{c} x _ {k} \\ u _ {k} \\ p _ {k} \\ d _ {k} \end{array} \right], p _ {k} = \Delta_ {k} q _ {k}, \tag {8}

\left[ \begin{array}{c} q _ {N} \\ f _ {N} \\ z _ {N} \end{array} \right] = \left[ \begin{array}{c c} \hat {C} _ {q} & 0 \\ \hat {C} _ {f} & \hat {D} _ {f p} \\ \hat {C} _ {z} & \hat {D} _ {z p} \end{array} \right] \left[ \begin{array}{c} x _ {N} \\ p _ {N} \end{array} \right], \qquad \qquad p _ {N} = \Delta_ {N} q _ {N},
$$

where $x _ { k } \in \mathbb { R } ^ { n } , u _ { k } \in \mathbb { R } ^ { n _ { u } } , d _ { k } \in \mathbb { R } ^ { n _ { d } } , f _ { k } \in \mathbb { R } ^ { n _ { f } } , z _ { k } \in \mathbb { R } ^ { n _ { z } } , p _ { k } \in \mathbb { R } ^ { n _ { p } }$ and $q _ { k } \in \mathbb { R } ^ { n _ { q } }$ are the state, input, disturbance, constraint, cost, and input and output uncertainty vectors, respectively, with $k { \in { \mathcal { N } } } { : = } \{ 0 , 1 , . . . , N { - } 1 \}$ , where N is the horizon length. In this study, it is assumed that all the states are measurable and the description includes terminal cost signal and state constraints to ensure closed-loop stability [12]. $\Delta _ { k } \in \mathcal { B } \pmb { \Delta }$ where $\pmb { \Delta } \subseteq \mathbb { R } ^ { n _ { p } \times n _ { q } }$ is a subspace that captures the uncertainty structure. Finally, the disturbance $d _ { k }$ is assumed to belong to the set $\mathcal { D } _ { k } = \{ d _ { k } \in$ $\mathbb { R } ^ { n _ { d } } \colon - \bar { d } _ { k } \le d _ { k } \le \bar { d } _ { k } \big \rbrace$ , where the disturbance’s upper bound is $\bar { d } _ { k } > 0$ and assumed known or approximated by the application specification (see Section V for example).

Remark 3. The constrain signal fk and the cost signal $z _ { k }$ are design signals linearly associated with the state and the input signal of the linear equivalent model and are used at the RMPC formulation, as will be presented in Sections III-B, IV and V.
