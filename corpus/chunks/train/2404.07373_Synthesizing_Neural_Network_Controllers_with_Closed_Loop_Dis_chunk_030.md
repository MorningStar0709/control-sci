We train a Dissipative RINN as per Algorithm 1, with $e = \left[ { \begin{array} { l } { x _ { 1 } } \\ { x _ { 2 } } \end{array} } \right]$ ] and $X = 0$ . This guarantees closed-loop stability. The state size, $n _ { k }$ , of the Dissipative RINN is the same as that of the plant, and the nonlinearity size, $n _ { \phi } ,$ is 16. We initialize the Dissipative RINN with an LTI controller. The LTI controller is then recovered from the dissipative $\hat { \theta }$ using the process described in the proof of Theorem 3. We use a $t _ { R S }$ value of 1.5 in (25) for all projections, and train this model using a learning rate of $1 0 ^ { - 3 }$ .
