# Example 1.8

Linearize

$$f _ {2} (x) = 0. 7 2 3 x ^ {3} - 4. 3 7 x ^ {2} + 6 7 x$$

about the point $x = 0$

$$f _ {2} (0) = 0\frac {d}{d x} f _ {2} (x) = 2. 1 6 9 x ^ {2} - 8. 7 4 x + 6 7\frac {d}{d x} f _ {2} (0) = 6 7\hat {f} _ {2} (x) = 6 7 x$$

Note that we are simply taking the linear term of $f _ { 2 } ( x )$ , 67x. However we only could get away with this because 1) we linearized about $\overset { - } { x } = 0$ and 2) the function was a polynomial.
