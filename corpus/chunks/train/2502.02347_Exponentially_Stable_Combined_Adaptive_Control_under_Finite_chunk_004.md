# 2. PRELIMINARIES

The following notations and definitions are used throughout this article, a scalar variable is denoted in small q, a vector in small boldface ${ \bf q } ,$ and a matrix in capital boldface Q. The Euclidean norm of a vector is denoted by kqk, in addition, $\lambda _ { \operatorname* { m i n } } ( \mathbf { Q } ) , \lambda _ { \operatorname* { m a x } } ( \mathbf { Q } )$ , and det(Q), denote the minimum and maximum eigenvalues, and determinant of a square matrix Q, respectively. An orthogonal matrix Q satisfies $\mathbf { Q Q } ^ { T } = \mathbf { I } \dot { }$ , where I is an identity matrix.

Definition 1. ((Yuan and Wonham, 1977)). A bounded signal $\ b { \phi } ( t ) \in \mathbb { R } ^ { n }$ is said to be persistently exciting if there exist positive constants $\gamma$ and T such that for every $t > 0$ , there is a sequence of q numbers in $t _ { i } \in [ t , t + T ] , i \in \mathbb { N }$ , with

$$\left\| \left[ \phi (t _ {1}) \phi (t _ {2}) \dots \phi (t _ {q}) \right] ^ {- 1} \right\| \leq \gamma . \tag {1}$$

Definition 2. A bounded signal $\phi ( t ) \in \mathbb { R } ^ { n }$ is said to be finitely exciting if there exist positive constants $\gamma$ and T so that there is at least a sequence of q numbers in $t _ { i } \in [ t , t + T ] , i \in \mathbb { N }$ , with

$$\left\| \left[ \phi (t _ {1}) \phi (t _ {2}) \dots \phi (t _ {q}) \right] ^ {- 1} \right\| \leq \gamma , \tag {2}$$

where $\gamma$ is the excitation level of the signal $\phi \left( t \right)$ .
