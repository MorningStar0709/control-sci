Recall that the neuron in Example 1 exhibits spiking due to the presence of both fast-activating negative conductance and slower-activating positive conductance. The model in Example 2 exhibits robust bursting because this pairing is replicated at the timescale of the burst [19]. The T- and L- type calcium currents $I _ { \mathrm { T } }$ and $I _ { \mathrm { L } }$ provide negative conductance in the slower timescale of calcium activation, and even slower positive conductance is provided by the calciumactivated potassium current $I _ { \mathrm { K C a } }$ .

Synaptic currents arise from electrochemical connections between neurons [20, Chapter 7]. They are modelled as current sources in the same way as intrinsic currents; the only difference is that the voltage dependence of their conductances is on the presynaptic neuron, noted $v _ { p }$ in the example below.

Example 3. For the sake of illustration, all the models in this paper only use one type of (inhibitory) synapse, called a GABA synapse in neurophysiology, obeying the standard model reproduced from [21]:

$$I _ {\mathrm{syn,p}} = \mu_ {\mathrm{syn,p}} s _ {\mathrm{syn,p}} (v - E _ {\mathrm{syn}}) \tag {4a}\dot {s} _ {\mathrm{syn,p}} = 0. 5 3 \sigma_ {\mathrm{syn}} (v _ {\mathrm{p}}) (1 - s _ {\mathrm{syn,p}}) - 0. 1 8 s _ {\mathrm{syn,p}} \tag {4b}$$

with a synaptic activation function $\sigma _ { \mathrm { s y n } }$ given by a sigmoid function of the form

$$\sigma_ {\mathrm{syn}} (v _ {\mathrm{p}}) = \frac {1}{1 + \exp (- (v _ {\mathrm{p}} - 2) / 5)}. \tag {5}$$

Here, $s _ { \mathrm { s y n , p } }$ is the synaptic gating variable and $v _ { \mathrm { p } }$ is the membrane voltage of the presynaptic neuron. The constants $\mu _ { \mathrm { s y n , p } } > 0$ and $E _ { \mathrm { s y n } } \in \mathbb { R }$ are (synaptic) maximal conductances and reversal potentials, respectively. This model can represent excitatory or inhibitory synapses, depending on the value of $E _ { \mathrm { s y n } }$ . In the present paper we set $E _ { \mathrm { s y n } } = - 9 0$ , which models the inhibitory synapses encountered in central pattern generators.
