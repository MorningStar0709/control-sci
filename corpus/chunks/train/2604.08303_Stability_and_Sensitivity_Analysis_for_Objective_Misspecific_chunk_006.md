# A. Continuous Action Monotone Games

A mathematical game formulation consists of agents, their actions, and cost functions: $G = ( N , \{ \mathcal { U } _ { i } \} _ { i \in N } , \{ J _ { i } \} _ { i \in N } )$ , where $N = \{ 1 , 2 , \dots , n \}$ is the set of players, $\mathcal { U } _ { i } \in \mathbb { R } ^ { m _ { i } }$ is the action set of player i, and $J _ { i } : \mathcal { U } _ { 1 } \times \mathcal { U } _ { 2 } \times \cdot \cdot \cdot \times \mathcal { U } _ { n } \to$ R is the cost function of player i. The collective behavior of the group is captured by the joint action profile consisting of the individual actions from each agent is represented by $u = ( u _ { 1 } , \ldots , u _ { n } ) \in \mathbb { R } ^ { m }$ , where $\begin{array} { r } { m = \sum _ { i \in N } m _ { i } } \end{array}$ . We denote the action of all players except i as u−i.

In game-theoretic contexts, the emergent behavior of the agents is often modeled by a Nash equilibrium. Within the context of multi-agent planning, this solution concept can serve as a prediction of the collective behavior. In many multi-agent interactions, however, the constraints on agents’ control actions may be coupled [12], [13], i.e., additional constraints of the form $u \in { \mathcal { C } } .$ . Throughout this work, we will consider this more general setting where the joint-action space is denoted by ${ \mathcal { Z } } = \mathcal { U } \cap { \mathcal { C } }$ , as well as the Generalized Nash equilibrium (GNE) game solution concept.

Definition 1. (Generalized Nash equilibrium): Consider an n-player game with joint action space ${ \mathcal { Z } } = \mathcal { U } \cap { \mathcal { C } }$ and cost functions $J _ { i } : \mathcal { Z } \to \mathbb { R }$ . The joint action profile $u ^ { \star } = ( u _ { 1 } ^ { \star } , \ldots , u _ { n } ^ { \star } )$ is a Generalized Nash equilibrium if the following holds for every player i:

$$J _ {i} (u _ {i} ^ {*}, u _ {- i} ^ {*}) \geq J _ {i} (u _ {i}, u _ {- i} ^ {*}) \quad \forall u _ {i} \in \mathcal {U} _ {i} \cap \mathcal {C}, \forall i \in N. \tag {1}$$
