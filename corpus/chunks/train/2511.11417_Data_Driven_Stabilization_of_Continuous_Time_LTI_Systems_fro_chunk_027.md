# A. Derivation of the Input–Output Model

To obtain the differential equation (1) from system (3), we perform an elimination of x as in [21, §6.2.2]. In particular, (3) can be written as

$$\mathcal {R} _ {x} \left(\frac {\mathrm{d}}{\mathrm{d} t}\right) x = \mathcal {R} _ {y} \left[ u ^ {\top} \quad v ^ {\top} \quad w ^ {\top} \quad y ^ {\top} \right] ^ {\top}, \tag {59}$$

where

$$
\mathcal {R} _ {x} (\xi) := \left[ \begin{array}{c} \xi I _ {n p} - A \\ C \end{array} \right], \mathcal {R} _ {y} := \left[ \begin{array}{c c c c} B & 0 _ {n p \times p} & E & 0 _ {n p \times p} \\ 0 _ {p \times m} & - I _ {p} & 0 _ {p \times q} & I _ {p} \end{array} \right]. \tag {60}
$$

Define the polynomial matrix

$$
\mathcal {W} (\xi) := \left[ \begin{array}{l l l l l} I _ {p} & \xi I _ {p} & \dots & \xi^ {n - 1} I _ {p} & - \mathcal {D} (\xi) \end{array} \right]. \tag {61}
$$

Then, by post-multiplying both sides of (59) by $\mathcal { W } ( \mathrm { d } / \mathrm { d } t )$ and exploiting the structure of A, B, C, E in (3), we obtain exactly (1). Note, in particular, that straightforward computations yield $\begin{array} { l } { { \mathcal { W } ( \xi ) \mathcal { R } _ { x } ( \xi ) } } \end{array} = \begin{array} { l } { 0 } \end{array}$ and $\begin{array} { r l } { \mathcal { W } ( \xi ) \mathcal { R } _ { y } } & { { } = } \end{array}$ $[ N ( \xi ) \mathcal { D } ( \xi ) \mathcal { N } _ { w } ( \xi ) - \mathcal { D } ( \xi ) ]$ .
