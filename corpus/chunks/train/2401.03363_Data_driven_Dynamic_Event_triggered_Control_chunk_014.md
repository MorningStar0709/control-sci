# IV. COMPARISONS

In this section, some existing dynamic ETMs for data-driven event-triggered control are briefly reviewed for comparisons, where some notations are changed to facilitate representation.

A. Dynamic ETMs with a prescribed timer or a dynamic internal variable

The dynamic ETM proposed in [21] is as follows:

$$t _ {k + 1} = \inf \left\{t > t _ {k} + \tilde {\rho} | \tilde {\eta} (t) \leq 0 \right\}, \tag {15}$$

where the update law of $\tilde { \eta } ( t )$ can be referred to [21, (19)] and the setting of the timer $\tilde { \rho }$ depends not only on an LMI [21, (21)], but also on some inequalities, e.g., [21, (22) and (23)].

The dynamic ETM proposed in [22] is presented in the following:

$$t _ {k + 1} = t _ {k} + h \min \left\{j > 0 | \tilde {\eta} (\mu_ {k} ^ {j}) + \tilde {\vartheta} \tilde {\rho} (\mu_ {k} ^ {j}) < 0 \right\}, \tag {16}$$

where $\mu _ { k } ^ { j } = t _ { k } + j h , j = 0 , 1 , . . .$ , with h being the prescribed sampling period, the constant ϑ˜ is specified in [22, Theorem 1], and the explicit expression of $\tilde { \rho } ( \mu _ { k } ^ { j } )$ and the update law of the dynamic internal variable $\widetilde { \eta } \big ( \mu _ { k } ^ { j } \big )$ can be found in [22, (12)] and [22, (13)], respectively. Moreover, the choice of the gain matrix designed in [22, (12)] is determined by a couple of LMIs [22, (18) and (19)] with complex components detailed in [22, Theorem 1].

Compared with the dynamic ETMs proposed in [21] and [22], less conservative conditions are required to implement the dynamic ETM proposed in (4)–(7). Specifically, the design of a prescribed timer is removed and the designed parameters, ${ \mathrm { e . g . } } , \ \alpha , \beta ,$ , are determined only by a data-driven LMI (14) whose components are much simpler than those in [22].

B. A dynamic ETM with a prescribed timer and a dynamic internal variable

A dynamic ETM with a prescribed timer $\bar { \rho }$ as well as an interval variable $\eta ( t )$ is designed in [23], as shown below:

$$t _ {k + 1} = \inf \left\{t \geq t _ {k} + \bar {\rho} | \vartheta (z (t) ^ {T} \Psi z (t) - \nu) - \eta (t) \geq 0 \right\}, \tag {17}$$

where the constants $\nu , \vartheta \geq 0 ,$ , the matrix Ψ can be set as [23, (28)], the timer $\bar { \rho } \in \{ 0 , \rho \}$ , with $\rho > 0$ defined in [23, (41)], which is dependent on $\mathcal { X } _ { 0 } , \mathcal { X } _ { 1 }$ and $\mathcal { U } _ { \mathrm { 0 } }$ . Moreover, the interval variable $\eta ( t )$ is dynamically updated by [23, (54)].

Note that a strictly positive MIET can be ensured when applying (17) with $\nu > 0 \mathrm { \ o r \ } \bar { \rho } = \rho .$ . When $\nu > 0$ , the system (1) is only with practical exponential ISS property. When $\bar { \rho } =$ $\rho ,$ the event-based controller will not update until at least time $\rho$ has elapsed, which will prolong the time for the closedloop system to achieve stability. In this paper, the existence of a strictly positive MIET is guaranteed, without requiring a prescribed timer and degrading system stability.
