# B. Extended State Feedback Control Law

Theorem 3: Consider the buffered NCS (4) with an extended state feedback controller

$$
\boldsymbol {u} _ {k} = - \hat {\boldsymbol {K}} \boldsymbol {\xi} _ {k} = - \left[ \begin{array}{l l} \boldsymbol {K} _ {x} & \boldsymbol {K} _ {u} \end{array} \right] \boldsymbol {\xi} _ {k} \tag {18}
$$

which includes not only the state variables but also part of the control input history. Furthermore, consider the scalar γ and the matrices as in the Theorem 1 with the difference that the matrices $X _ { i }$ and $\hat { Z }$ are replaced by one constant matrix X and a full matrix $\hat { \pmb { Z } } = \bar { \lceil } \pmb { Z } _ { x } \pmb { Z } _ { u } \rceil ( \forall i = 1 , 2 , . . . , \bar { \delta } )$ , respectively. If the resulting LMIs of the form (13) are satisfied $\forall i , j \in \{ 1 , 2 , \dotsc , \bar { \delta } \}$ , then the buffered NCS is asymptotically stable for the extended control law (18) obtained by $\dot { \boldsymbol { K } } = \hat { Z } \boldsymbol { X } ^ { - 1 }$ .

Proof: The proof follows as a direct consequence of the Theorem 1. 

Please note that if an extended control law is used, an additional restriction that there are no dropouts in the connection path from the plant to the controller must hold. In addition, the controller must wait until all the previous control signals are available before computing $\mathbf { \Delta } \mathbf { u } _ { k }$ , which could add an additional delay in the system and hence leads to the reformulation of the sensor-to controller delay to $\tau _ { k } ^ { s c } = \operatorname* { m a x } \{ k T _ { d } + \tau _ { k } ^ { s c } , j T _ { d } + \tau _ { j } ^ { s c } \} , \forall j < k$ . However, the maximal round trip time and therefore the value of $\bar { \delta } ( 2 )$ remain unchanged, see Assumption 2, allowing that all the considerations directly follow from Theorem 1.
