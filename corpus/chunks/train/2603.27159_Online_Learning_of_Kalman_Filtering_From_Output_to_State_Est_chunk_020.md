# 4 Learning Kalman Filtering for Output Estimation

In this section, we study the output estimation problem, which corresponds to the full information setting in the online optimization framework as we discussed in Section 2.2. Recall that the cost of the output estimation problem is given by $f _ { t } ( N _ { t } ) = \left\| y _ { t } - \hat { y } _ { t } ( N _ { t } ) \right\| ^ { 2 }$ with $\begin{array} { r } { \hat { y } _ { t } ( N _ { t } ) = \sum _ { s = 1 } ^ { h } N _ { t , s } y _ { t - s } , } \end{array}$ where the filter parameter $N _ { t } = [ N _ { t , s } ] _ { s \in [ h ] }$ is chosen from the set $\kappa _ { N }$ defined in (19).

Algorithm 1: Online learning for output estimation   
Input: Step sizes $\{\eta_t\}_{t\geq 0}$ , truncated filter length $h$ .  
1 Initialize $N_0 = 0$ .  
2 for $t = 0,1,\ldots$ do  
3 Estimate $\hat{y}_t(N_t) = \sum_{s=1}^h N_{t,s}y_{t-s}$ 4 Obtain $f_t(N_t) = \| y_t - \hat{y}_t(N_t)\|^2$ 5 Update $N_{t+1} = \Pi_{\mathcal{K}_N}(N_t - \eta_t\nabla f_t(N_t))$ Output: $N_0,N_1,\ldots$ .
