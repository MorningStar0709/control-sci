$$
\begin{array}{l} c ^ {2} \| \tilde {\boldsymbol {x}} \| ^ {2} = c ^ {2} \sum_ {j = 1} ^ {3} \sum_ {i \in \mathbb {S} _ {j}} | \tilde {x} _ {i} | ^ {2} \geq \sum_ {i \in \mathbb {S} _ {2}} c ^ {2} | \tilde {x} _ {i} | ^ {2} + \sum_ {i \in \mathbb {S} _ {3}} c ^ {2} | \tilde {x} _ {i} | ^ {2} \\ > \sum_ {i \in \mathbb {S} _ {2}} N \underline {{\eta}} ^ {2} + \sum_ {i \in \mathbb {S} _ {3}} \left(\hat {\eta} _ {i} ^ {2} (x _ {i}) + (N - 1) \underline {{\eta}} ^ {2}\right) \tag {26} \\ = | \mathbb {S} _ {2} \cup \mathbb {S} _ {3} | N \underline {{\eta}} ^ {2} + \sum_ {i \in \mathbb {S} _ {3}} \Big (\eta_ {i} ^ {2} (x _ {i}) - \underline {{\eta}} ^ {2} \Big). \\ \end{array}
$$

Similarly, the value of $\| \hat { \pmb { \eta } } ( \pmb { x } ) \| ^ { 2 }$ is written as

$$
\begin{array}{l} \| \hat {\boldsymbol {\eta}} (\boldsymbol {x}) \| ^ {2} = \sum_ {j = 1} ^ {3} \sum_ {i \in \mathbb {S} _ {j}} \hat {\eta} _ {i} ^ {2} (x _ {i}) \leq \sum_ {i \in \mathbb {S} _ {1} \cup \mathbb {S} _ {2}} \underline {{\eta}} ^ {2} + \sum_ {i \in \mathbb {S} _ {3}} \hat {\eta} _ {i} ^ {2} (x _ {i}) \\ = N \underline {{\eta}} ^ {2} + \sum_ {i \in \mathbb {S} _ {3}} \left(\hat {\eta} _ {i} ^ {2} (x _ {i}) - \underline {{\eta}} ^ {2}\right). \tag {27} \\ \end{array}
$$

Apply the inequalities (26) and (27) into (21), and then due to the positive c, (21) is bounded by

$$
\begin{array}{l} \dot {V} _ {c} \leq - \frac {2 \| \tilde {\boldsymbol {x}} \|}{c \| \tilde {\boldsymbol {x}} \| + \| \hat {\boldsymbol {\eta}} (\boldsymbol {x}) \|} (c ^ {2} \| \tilde {\boldsymbol {x}} \| ^ {2} - \| \hat {\boldsymbol {\eta}} (\boldsymbol {x}) \| ^ {2}) \\ <   - \frac {2 \| \tilde {\boldsymbol {x}} \|}{c \| \tilde {\boldsymbol {x}} \| + \| \hat {\boldsymbol {\eta}} (\boldsymbol {x}) \|} (| \mathbb {S} _ {2} \cup \mathbb {S} _ {3} | N \underline {{\eta}} ^ {2} - N \underline {{\eta}} ^ {2}). \tag {28} \\ \end{array}
$$

Now, we consider the case with $\| \tilde { \pmb { x } } \| ~ > ~ 2 c ^ { - 1 } N \eta$ . This indicates at least one agent satisfying $| \tilde { x } _ { i } | > 2 c ^ { - \overline { { 1 } } } \sqrt { N } \eta .$ , which is proven by considering the contradiction with all agents having $c | \tilde { x } _ { i } | \leq 2 \sqrt { N } \underline { { \eta } }$ and

$$\| \tilde {\boldsymbol {x}} \| ^ {2} \leq \sum_ {i \in \mathcal {V}} 4 c ^ {- 2} N \underline {{\eta}} ^ {2} = (2 c ^ {- 1} \sqrt {N} \underline {{\eta}}) ^ {2}. \tag {29}$$
