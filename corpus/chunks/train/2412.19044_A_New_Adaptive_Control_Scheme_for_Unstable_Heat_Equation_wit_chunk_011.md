# 5 Proof of Theorem 2.1

Before proving Theorem 2.1, we first give two lemmas.

Lemma 5.1. Suppose that $b \neq 0 , c _ { 1 } > 0$ and $u _ { 0 } \in L _ { \mathrm { l o c } } ^ { 2 } ( 0 , \infty )$ . Then, for any initial state $( \tilde { w } ( \cdot , 0 ) , \tilde { \zeta } ( 0 ) ) \in \mathcal { H } \times \mathbb { R }$ and for any $T > 0$ , system (2.7) admits a unique weak solution $( \tilde { w } ( \cdot , t ) , \tilde { \zeta } ( t ) ) \in$ $C ( 0 , T ; \mathcal { H } \times \mathbb { R } )$ such that

$$\int_ {0} ^ {t} \int_ {0} ^ {1} \tilde {w} _ {x} ^ {2} (x, s) d x + c _ {1} \tilde {w} ^ {2} (1, s) d s = F (0) - F (t) \tag {5.1}$$

and

$$\int_ {0} ^ {t} \int_ {0} ^ {1} \tilde {w} _ {x} ^ {2} (x, s) d x + b u _ {0} (s) \tilde {\zeta} (s) \tilde {w} (1, s) + c _ {1} \tilde {w} ^ {2} (1, s) d s = E (0) - E (t), \tag {5.2}$$

where $0 < t \leq T$ and

$$F (t) = E (t) + \frac {| b |}{2} \tilde {\zeta} ^ {2} (t), \quad E (t) = \frac {1}{2} \int_ {0} ^ {1} \tilde {w} ^ {2} (x, t) d x. \tag {5.3}$$

Proof. We will prove Lemma 5.1 by Galerkin method [2]. Let

$$
\left\{ \begin{array}{l l} \phi_ {n} (x) = \sqrt {2} \sin \sqrt {\lambda_ {n}} x, & \\ \lambda_ {n} = \left(n - \frac {1}{2}\right) ^ {2} \pi^ {2}, & \end{array} \quad x \in [ 0, 1 ], \quad n \geq 1. \right. \tag {5.4}
$$

Then, $\{ \phi _ { n } ( \cdot ) \} _ { n = 1 } ^ { \infty }$ forms an orthonormal basis for H. Fix a positive integer N, let $\mathcal { P } _ { N }$ be the orthogonal projection of H onto $V _ { N } : = \mathrm { t h e }$ linear span of $\{ \phi _ { j } : j = 1 , 2 , \cdot \cdot \cdot , N \}$ . Let

$$\tilde {w} _ {N} (\cdot , t) = \sum_ {j = 1} ^ {N} \tilde {w} _ {N, j} (t) \phi_ {j}, \quad \tilde {\zeta} _ {N} (t) \tag {5.5}$$

be the approximate solutions that satisfy the following system of ordinary differential equations:

$$
\left\{ \begin{array}{l} \langle \tilde {w} _ {N t} (\cdot , t), \phi_ {j} \rangle_ {\mathcal {H}} + \langle \tilde {w} _ {N x} (\cdot , t), \phi_ {j x} \rangle_ {\mathcal {H}} = - \phi_ {j} (1) \left[ b \tilde {\zeta} _ {N} (t) u _ {0} (t) + c _ {1} \tilde {w} _ {N} (1, t) \right], \\ \dot {\tilde {\zeta}} _ {N} (t) = \operatorname{sgn} (b) u _ {0} (t) \tilde {w} _ {N} (1, t), \\ \tilde {w} _ {N} (\cdot , 0) = \mathcal {P} _ {N} \tilde {w} (\cdot , 0), \quad \tilde {\zeta} _ {N} (0) = \tilde {\zeta} (0), \quad j = 1, 2 \dots , N. \end{array} \right. \tag {5.6}
$$

Clearly the initial value satisfies

$$\lim _ {N \rightarrow + \infty} \| \tilde {w} _ {N} (\cdot , 0) - \tilde {w} (\cdot , 0) \| _ {\mathcal {H}} = 0 \tag {5.7}$$
