# 5.3. Log-Likelihood: A Probably-Fine Measure

If we use the probability-based measure log-likelihood P , in the discrete case,

$$
\begin{array}{l} E _ {P} (\bar {W} _ {t o}, \bar {W} _ {b}) = \underset {\bar {w} \sim \bar {W} _ {t o}} {\hat {\mathbb {E}}} P (\bar {w}, \bar {W} _ {b}) \\ = - \underset {\bar {w} \sim \bar {W} _ {t o}} {\hat {\mathbb {E}}} \log \hat {\mathbb {P}} _ {\bar {W} _ {b}} (\bar {w}) \tag {33} \\ = - \sum_ {\bar {w}} \mathbb {P} (\bar {w} | o) \log \hat {\mathbb {P}} _ {\bar {W} _ {b}} (\bar {w}). \\ \end{array}
$$

This expectation is related to two probability distributions: the subjective belief and the observable true probability distribution (it is different from the entropy $- \sum _ { \bar { w } } \hat { \mathbb { P } } _ { \bar { W } _ { b } } ( \bar { w } )$ log $\hat { \mathbb P } _ { \bar { W } _ { b } } ( \bar { w } )$ , which describes the property of one probability distribution). The best predictor is $\hat { \mathbb { P } } _ { \bar { W } _ { b } } ( \bar { w } ) = \hat { \mathbb { P } } _ { \bar { W } _ { t o } } ( \bar { w } )$ for all w¯ due to convexity, which means this measure is best-P-lowest-C. However, there is no guarantee to make it better-P-lower-C.
