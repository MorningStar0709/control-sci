# Assumption 2.1. The following hold:

(1) The functions $b ( \cdot , i )$ and $\sigma ( \cdot , i )$ are twice continuously differentiable, for every $i \in \mathbb { Y } .$ .   
(2) There exists $C > 0 ,$ , such that $| b ( x , i ) | + | \sigma ( x , i ) | \leq C ( 1 + | x | )$ , for any $( x , i ) \in \mathcal { I } \times \mathbb { Y } .$   
(3) There exists $c > 0 _ { i }$ , such that $b _ { x } ( x , i ) \leq - c , f o r a n y \left( x , i \right) \in \mathcal { T } \times \mathbb { Y } .$ .   
(4) For any $( x , i ) \in \mathcal { T } \times \mathbb { Y } , \sigma ( x , i ) > 0 .$

Conditions (1) and (2) in Assumption 2.1 imply that $( X ^ { 0 } , Y )$ is regular, meaning that there exists a sequence of stopping times $\{ \beta _ { n } \} _ { n \ge 0 }$ , with βn := inf $\{ t \ge 0 : | X _ { t } ^ { 0 } | = n \}$ , such that $\beta _ { \infty } : =$ $\begin{array} { r } { \operatorname* { l i m } _ { n  \infty } \beta _ { n } = \infty } \end{array}$ , P-a.s.; for further details see Section 2.3 in [53].

For our subsequent analysis, we introduce the F-adapted process $\{ \widehat { X } _ { t } \} _ { t \geq 0 }$ , which evolves as

$$d \widehat {X} _ {t} = (b (\widehat {X} _ {t}, Y _ {t}) + \sigma \sigma_ {x} (\widehat {X} _ {t}, Y _ {t})) d t + \sigma (\widehat {X} _ {t}, Y _ {t}) d \widehat {W} _ {t}, \quad (\widehat {X} _ {0}, Y _ {0}) = (x, i) \in \mathcal {I} \times \mathbb {Y}, \tag {2.4}$$

for an F-adapted Brownian motion $\widehat { W }$ . Notice that Equation (2.4) also admits a unique strong solution $( \widehat { X } ^ { x } , Y ^ { i } )$ which is regular, due to Assumption 2.1.
