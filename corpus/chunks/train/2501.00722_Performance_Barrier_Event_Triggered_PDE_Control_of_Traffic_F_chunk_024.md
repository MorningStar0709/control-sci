# B. Regular Periodic Event-triggered Control (R-PETC)

In this subsection, we the introduce R-PETC approach applied to the system (13)-(21). Since the R-PETC design can be derived via the P-PETC approach detailed in Section IV-B, we will only present its structure here to prevent redundancy.

Let $\tilde { I } ^ { r } = \left\{ \tilde { \tilde { t } } _ { 0 } ^ { r } , \tilde { t } _ { 1 } ^ { r } , \tilde { t } _ { 2 } ^ { r } , \dots \right\}$ denote the sequence of event-times associated with R-PETC, and let the event-trigger parameters $\theta , \eta , \theta _ { m } , \kappa _ { 1 } , \kappa _ { 2 } , \kappa _ { 3 } > 0$ be selected as outlined in Assumption 2. The proposed R-PETC strategy consists of two parts:

1) An ETC input $\tilde { U } _ { k } ^ { r }$

$$\tilde {U} _ {k} ^ {r} := U (\tilde {t} _ {k} ^ {r}), \tag {61}$$

for $t \in \left[ \tilde { t } _ { k } ^ { r } , \tilde { t } _ { k + 1 } ^ { r } \right) , k \in \mathbb { N }$ , where $U ( t )$ is given by (24). Then, the boundary condition (33) becomes

$$\bar {v} (\ell , t) = r _ {1} \tilde {U} _ {k} ^ {r}. \tag {62}$$

2) A periodic event-trigger determining event-times

$$
\begin{array}{l} \tilde {t} _ {k + 1} ^ {r} = \inf \left\{t \in \mathbb {R} _ {+} \mid t > \tilde {t} _ {k} ^ {r}, \tilde {\Gamma} ^ {r} (t) > 0, t = n h, \right. \tag {63} \\ h > 0, n \in \mathbb {N}, k \in \mathbb {N} \}, \\ \end{array}
$$

with $\tilde { t } _ { 0 } ^ { r } = 0$ . Here h is the sampling period selected as

$$0 < h \leq \tau_ {d}, \tag {64}$$

where $\tau _ { d }$ is given by $( 5 2 ) – ( 5 4 )$ , and $\tilde { \Gamma } ^ { r } ( t )$ is the triggering function defined as

$$\tilde {\Gamma} ^ {r} (t) := \left(a + \theta \theta_ {m}\right) e ^ {a h} d ^ {2} (t) - \theta \theta_ {m} d ^ {2} (t) - \theta a m ^ {p} (t), \tag {65}$$

where a is defined in (53). Further, $d ( t )$ is defined in (34), and $m ^ { r } ( t )$ satisfies the ODE given by (43), along the solution of (13)-(15),(17)-(21),(61)-(65) for all $t \in$ $\left[ \tilde { t } _ { k } ^ { r } , \tilde { t } _ { k + 1 } ^ { r } \right) , k \in \mathbb { N } .$ .

The main difference between the continuous-time eventtrigger (41)-(43) and the periodic event-trigger (63)-(65) lies in that the triggering function $\Gamma ^ { r } ( t )$ of R-CETC has to be monitored continuously while the triggering function $\tilde { \Gamma } ^ { r } ( t )$ of R-PETC requires only periodic evaluations.
