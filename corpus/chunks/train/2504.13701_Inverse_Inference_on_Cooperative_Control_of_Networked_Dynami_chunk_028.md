Algorithm 1 Alternating minimization based topology inference algorithm   
Input: Surrogate matrix $\tilde{L} \in \mathbb{R}^{N \times N}$ , stepsize factor $\alpha$ , inner iteration limit $\bar{k}$ , outer iteration limit $\bar{t}$ , perturbation magnitude $\epsilon$ , threshold parameter $\epsilon_L$ Output: Estimate of matrix $L$ ;

1: for $t = 1, \ldots, \bar{t}$ do

2: Initialize $\tilde{L} = \tilde{L}^{(t-1)}$ ;

3: for $k = 1, \ldots, \bar{k}$ do

4: Compute gradient: $\nabla f(L) = 2(L - \tilde{L})$ ;

5: Gradient descent update: $L \leftarrow L - \alpha \nabla f(L)$ ;

6: Project $L$ to satisfy row-sum constraint: $L \leftarrow L - \frac{1L1^T}{N}$ ;

7: Project $L$ to satisfy non-negativity constraint on off-diagonal:

8: for all $i \neq j$ do

9: $L_{ij} \leftarrow \max(L_{ij}, 0)$ ;

10: end for

11: end for

12: if $\|L - L^{(t-1)}\| \leq \epsilon_L$ and $L$ is simple then

13: break;

14: end if

15: Add random perturbation: $L \leftarrow L + \epsilon \cdot \text{randn}(N, N)$ ;

16: Update $L^{(t)} \leftarrow L$ ;

17: end for

18: Return $L^{\bar{t}}$
