$$\underline {{\omega}} _ {i} ^ {\mathrm{th}} = (\omega^ {\infty} - \underline {{\omega}} _ {i}) \varsigma (v _ {i} ^ {+}) + \underline {{\omega}} _ {i}, \overline {{\omega}} _ {i} ^ {\mathrm{th}} = (\omega^ {\infty} - \overline {{\omega}} _ {i}) \varsigma (v _ {i} ^ {-}) + \overline {{\omega}} _ {i},$$

where $v _ { i } ^ { + } , v _ { i } ^ { - } \in \mathbb { R }$ are biases. Then $\underline { { \omega } } _ { i } ^ { \mathrm { t h } }$ and $\overline { { \omega } } _ { i } ^ { \mathrm { t h } }$ approximate any values in $( \underline { { \omega } } _ { i } , \omega ^ { \infty } )$ , and $\left( \omega ^ { \infty } , \overline { { \omega } } _ { i } \right)$ , respectively.

The proof of Lemma 5.2 readily follows the definition of the sigmoid function.
