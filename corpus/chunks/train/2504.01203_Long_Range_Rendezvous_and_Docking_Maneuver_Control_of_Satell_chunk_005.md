# A. Rigid Body Dynamics and Kinematics of Target Satellite

Let the position vector of the target satellite with respect to $E _ { t }$ be denoted by $\tilde { p } _ { t }$ , whereas the translational and angular velocity of target satellite w.r.t $E _ { i }$ frame be denoted by $\tilde { v } _ { t }$ and $\tilde { \omega } _ { t } .$ , respectively. The translational kinematics and dynamics of the target satellite can be described as [18]

$$M _ {t} \dot {\tilde {v}} _ {t} + M _ {t} \boldsymbol {\Omega} (\tilde {\omega} _ {t}) \tilde {v} _ {t} = \tilde {F} _ {d t} + \tilde {F} _ {j _ {2}, t} \tag {1}\dot {\tilde {p}} _ {t} + \boldsymbol {\Omega} (\tilde {\omega} _ {t}) \tilde {p} _ {t} = \tilde {v} _ {t} \tag {2}$$

where $M _ { t }$ is the mass of the target satellite, taken as 1000 $k g .$ , $\tilde { F } _ { d }$ is the external disturbance force and $\tilde { F } _ { j _ { 2 } , t }$ is gravity and $J _ { 2 }$ perturbation force acting on the target satellite [18], respectively. $\pmb { \Omega } ( \tilde { \omega } _ { t } )$ denotes the skew-symmetric matrix representation of angular velocity vector $\tilde { \omega } _ { t } )$ as defined below.

$$
\boldsymbol {\Omega} (\tilde {\omega} _ {\mathbf {t}}) = \left[ \begin{array}{c c c} 0 & - \omega_ {t _ {z}} & \omega_ {t _ {y}} \\ \omega_ {t _ {z}} & 0 & - \omega_ {t _ {x}} \\ - \omega_ {t _ {y}} & \omega_ {t _ {x}} & 0 \end{array} \right]
$$

The rotational kinematics and dynamics of the target satellite is described as given below.

$$\mathbf {J} _ {\mathbf {t}} \dot {\tilde {\omega}} _ {t} = - \boldsymbol {\Omega} (\tilde {\omega} _ {t}) \mathbf {J} _ {\mathbf {t}} \tilde {\omega} _ {t} + \tilde {T} _ {d t} + \tilde {T} _ {g t} (3)$$

where $\tilde { T } _ { d t }$ represent the external disturbance torque and $\tilde { T } _ { g t }$ denote the gravity gradient torque acting on the target satellite $[ 1 ] ; \mathbf { J } _ { t } \in \mathbb { R } ^ { \bar { 3 } \times 3 }$ denotes the inertia matrix of the target satellite. The attitude kinematics of the target satellite is given below:

$$\dot {\boldsymbol {B}} _ {t} ^ {i} = \boldsymbol {B} _ {t} ^ {i} \boldsymbol {\Omega} (\tilde {\omega} _ {t}) \tag {4}$$

where $B _ { t } ^ { i } \in \ S O ( 3 )$ is the rotation matrix which transforms from the target body-fixed frame to ICRF frame.
