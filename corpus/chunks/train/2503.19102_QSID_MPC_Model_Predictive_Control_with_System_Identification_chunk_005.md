# A. System Identification

The system matrices A and B are identified from quantized input and state data snapshots $\{ \tilde { x } _ { t } \} _ { t = 0 } ^ { T }$ and $\{ \tilde { u } \} _ { t = 0 } ^ { T - 1 }$ as follows:

$$
\begin{array}{l} \hat {G} = [ \hat {A}, \hat {B} ] = \underset {\mathcal {A} \in \mathbb {R} ^ {n \times n}, \mathcal {B} \in \mathbb {R} ^ {n \times m}} {\operatorname{argmin}} \frac {1}{T} \| X ^ {+} - \mathcal {A} X - \mathcal {B} U \| ^ {2} \tag {3} \\ = \underset {\mathcal {G} \in \mathbb {R} ^ {n \times (n + m)}} {\operatorname{argmin}} \frac {1}{T} \| X ^ {+} - \mathcal {G} \Psi \| ^ {2}, \\ \end{array}
$$
