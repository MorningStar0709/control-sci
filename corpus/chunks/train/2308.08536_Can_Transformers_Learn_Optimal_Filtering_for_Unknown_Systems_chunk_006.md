$$\left\| f _ {i} ^ {(t)} (\mathbf {x}, \mathcal {W}) - f _ {i} ^ {(t)} \left(\mathbf {x} ^ {\prime}, \mathcal {W}\right) \right\| \leq C _ {\rho} \rho^ {t} \| \mathbf {x} - \mathbf {x} ^ {\prime} \|. \tag {2}$$

We define the notation $\begin{array} { r } { L _ { \rho } : = \frac { C _ { \rho } } { 1 - \rho } } \end{array}$ Cρ . When the class of dynamical systems we are sampling from consists of linear systems with $f _ { i } ( \mathbf { x } ) = \mathbf { A } _ { i } \mathbf { x }$ , then Assumption 1 is satisfied when the spectral radius $\rho ( \mathbf { A } _ { i } ) ~ < ~ 1$ for all i. It is also satisfied by systems that are contracting [20] or exponentially incrementally input-to-state stable [21] with input w.

In this work, we seek to predict system output using a transformer [5], which is a deep sequence model $\operatorname { T F } _ { \theta } ( \cdot )$ that maps system output sequences $\mathcal { V } _ { t } : = \mathbf { y } _ { 0 : t } \tan \hat { \mathbf { y } } _ { t + 1 } : = \mathrm { T F } _ { \theta } ( \mathcal { V } _ { t } )$ , an estimation of the true output $\mathbf { y } _ { t + 1 }$ at time $t + 1$ . The trainable parameters of the transformer are denoted by $\theta \in \Theta$ for some parameter set Θ. The transformer structure allows the sequence length t to be varying.

Assuming the access to M length-T output trajectories $\{ \mathbf { y } _ { i , 0 : T } \} _ { i = 1 } ^ { M }$ generated by each of the M source systems, the goal in this work is to train a transformer model that, at each time t, can predict the output $_ { \mathbf { y } _ { 0 , t + 1 } }$ of the target system $S _ { 0 }$ only using the past outputs ${ \mathbf { y } } _ { 0 , 0 : t } .$ Let $\mathbf { \mathcal { V } } _ { i , t } : = \mathbf { y } _ { i , 0 : t }$ denote the outputs up to time $t ,$ which is also known as the prompt (to predict $\mathbf { y } _ { i , t + 1 } )$ ), then the transformer is trained by solving

$$\widehat {\mathrm{TF}} = \underset {\mathrm{TF} \in \mathcal {A}} {\arg \min} \frac {1}{M T} \sum_ {i = 1} ^ {M} \sum_ {t = 1} ^ {T} \ell (\mathbf {y} _ {i, t}, \mathrm{TF} (\mathcal {Y} _ {i, t - 1})), \tag {3}$$

where $\mathcal { A } : = \{ \mathrm { T } \mathbb { F } _ { \theta } ~ : ~ \theta ~ \in ~ \Theta \}$ and $\ell ( \cdot , \cdot ) \geq 0$ is the loss function. To apply $\widehat { \mathrm { T F } } ( \cdot )$ to the target systems $ { \boldsymbol { S } } _ { 0 }$ , we simply take $\widehat { \mathrm { T F } } ( \mathcal { V } _ { 0 , t } )$ as the prediction for $_ { \mathbf { y } _ { 0 , t + 1 } }$ .

Training a model as in (3) where the data comes from a diversity of sources is also known as in-context learning. As a result of the training diversity, the transformer can achieve good performance on any of the source systems as well as demonstrate generalization ability for the unseen target system $ { \boldsymbol { S } } _ { 0 }$ . Hence, we refer to the obtained transformer $\overbrace { \mathrm { T F } } ^ { }$ as metaoutput-predictor (MOP).

In what follows, we first empirically demonstrate the performance of MOP in Section III under various setups, which is followed by theoretical analysis in Section IV.
