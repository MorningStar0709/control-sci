Noise qt has covariance matrix Q:

$$Q = d i a g\left(\left[ 0. 0 0 2 \Delta t \quad 0. 0 0 2 \Delta t \quad 0. 0 0 2 \Delta t \quad 0. 4 5 \quad 0. 4 5 \quad 0. 4 5 \right]\right) \tag {26}$$

diag() transforms a 1 x n vector into an n x n diagonal matrix. The position variance is multiplied by Δ? because error increases with more delay between updates. The velocity variance, however, is independent of the value of Δ? These variances are not obtained from measurements on an actual robot. They are just arbitrary values that can be tweaked to adjust the simulation behaviour. For example, when experimenting solely with the accuracy of the prediction model, the values in Q can be set to near zero, which will cause the KF to “ignore” the measurement.
