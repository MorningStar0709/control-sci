# A. ESO and Estimation Error Bound

In practice, we only have output measurements y. To implement CBF-QP, the state x and disturbance d in (7) need to be estimated by ESOs. After converting the original nonlinear control-affine system into a normal form and adding the corresponding disturbance for each measurement, we use x to represent state z in (3) just for symbolic consistence for CBF. Equation (3) can be written in a matrix form:

$$
\left\{ \begin{array}{l} \dot {x} = A _ {0} x + B _ {0} (b (x) + a (x) u _ {i} + d _ {i}) \\ y _ {i} = C _ {0} x \end{array} \right. \tag {8}
$$

where $A _ { 0 } = \left[ \begin{array} { l l l l } { 0 } & { 1 } & { \cdots } & { 0 } \\ { \vdots } & { \vdots } & { \ddots } & { \vdots } \\ { 0 } & { 0 } & { \cdots } & { 1 } \\ { 0 } & { 0 } & { \cdots } & { 0 } \end{array} \right] _ { r \times r } , B _ { 0 } = \left[ \begin{array} { l } { 0 } \\ { \vdots } \\ { 0 } \\ { 1 } \end{array} \right] _ { r \times 1 } , C _ { 0 } =$ r×r r×1

$[ 1 , 0 , \cdots , 0 ] _ { 1 \times r } , r$ is the relative degree of yi with respect to ui for $i \in [ 1 , m ]$ , di is the corresponding disturbance for $y _ { i } , a ( x ) = \bar { L _ { g _ { i } } } \bar { L _ { f } ^ { r - 1 } } \zeta _ { i } ( \phi ^ { - 1 } ( x ) )$ , and $b ( x ) \stackrel { - } { = } L _ { f } ^ { r } \zeta _ { i } ( \Phi ^ { - 1 } ( x ) )$ .

Since the disturbance for each measurement can be treated as an extended state [14], the augmented system becomes:

$$
\left\{ \begin{array}{l} {\left[ \begin{array}{l} \dot {x} \\ \dot {f} _ {i} \end{array} \right] = A \left[ \begin{array}{l} x \\ f _ {i} \end{array} \right] + B (b (\hat {x}) + a (\hat {x}) u _ {i}) + D \dot {f} _ {i}} \\ y _ {i} = C \left[ \begin{array}{l l} x & f _ {i} \end{array} \right] ^ {T} \end{array} \right. \tag {9}
$$

$A ~ = ~ \left[ { \begin{array} { c c } { A _ { 0 } } & { B _ { 0 } } \\ { 0 _ { 1 \times r } } & { 0 } \end{array} } \right] _ { ( r + 1 ) \times ( r + 1 ) } , ~ B ~ = ~ \left[ { \begin{array} { c } { B _ { 0 } } \\ { 0 } \end{array} } \right] _ { ( r + 1 ) \times 1 } ,$ $C = [ 1 , 0 , \cdot \cdot \cdot , 0 ] _ { 1 \times ( r + 1 ) } , D = [ 0 , \cdot \cdot \cdot , 0 , 1 ] _ { 1 \times ( r + 1 ) } ^ { T } ,$ $f _ { i } = b ( x ) - b ( \hat { x } ) + ( a ( x ) - a ( \hat { x } ) ) u _ { i } + d _ { i }$ rather than $d _ { i }$ .

An ESO is designed upon the formulation in (9):

$$
\left[ \begin{array}{l} \dot {\hat {x}} \\ \dot {\hat {f}} _ {i} \end{array} \right] = A \left[ \begin{array}{l} \hat {x} \\ \hat {f} _ {i} \end{array} \right] + B (b (\hat {x}) + a (\hat {x}) u _ {i}) + L \left(y _ {i} - C \left[ \begin{array}{l} \hat {x} \\ \hat {f} _ {i} \end{array} \right]\right) \tag {10}
$$
