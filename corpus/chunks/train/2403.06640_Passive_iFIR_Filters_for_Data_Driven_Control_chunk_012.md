# A. Fitting of passive FIR filters

We identify passive FIR filters to illustrate the effectiveness of (7), (10) and (14) in dealing with the targets

$$C _ {q} (s) = \frac {1}{(0 . 5 s + 1) ^ {q}} \quad q \in \{1, 2, 3 \}, \tag {18}$$

where $C _ { 1 }$ is passive, and $C _ { 2 }$ and $C _ { 3 }$ show increase shortage of passivity. For each target, we take $\gamma = 0$ and (5) paired with passivity constraints. The input/output data (e, u) are generated as $u = C _ { i } e$ for $i \in \{ 1 , 2 , 3 \}$ . The probing signal is a filtered step given by $\begin{array} { r } { e = \mathcal { L } ^ { - 1 } \left( \frac { 1 } { 0 . 2 s + 1 } \frac { 1 } { s } \right) } \end{array}$ , where ${ \mathcal { L } } ^ { - 1 }$ is the inverse Laplace transform. Results are shown in Figure 2. The top-left plot shows that all three approaches give perfect fitting. The top-right plot shows the results for conservative ϵ (10d). Indeed, this parameter can be used to enforce an excess of passivity, to compensate for shortages in the plant. The left and right bottom plots are related to the non-passive targets $C _ { 2 }$ and $C _ { 3 }$ . All three approaches achieve a passive FIR filter. As expected, the match with the target systems degrades as the passivity shortage increases. These results are obtained using CVXPY [13] with MOSEK solver [24].
