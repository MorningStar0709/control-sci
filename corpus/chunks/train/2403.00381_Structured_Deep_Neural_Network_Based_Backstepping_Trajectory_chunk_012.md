# B. Stability of the NBS Controller

In the following theorems, we analyze the performance of the NBS tracking controller. First, we consider the case without disturbances.

Theorem 1. Consider the system (2) without disturbance $\tau ^ { d } { } _ { ; }$ , if the controller is designed as (9), where $\Phi ( z _ { 1 } )$ is strongly convex in ${ z } _ { 1 }$ with a unique minimum at $z _ { 1 } = 0$ satisfying $\Phi ( 0 ) = 0 ,$ , and $D ( z _ { 2 } )$ is positive definite, then the closed-loop system is globally asymptotically stable at ${ \boldsymbol { z } } _ { 1 } = \mathbf { 0 } , { \boldsymbol { z } } _ { 2 } = \mathbf { 0 }$ .

Proof. For simplifing notations, we shall ignore the DNN parameters $\theta _ { 1 } , \theta _ { 2 }$ and hyperparameters $S ,$ m in (9). According to (8) and (9), (2) can be reformulated as

$$
\begin{array}{l} M \dot {z} _ {2} + C z _ {2} + G = M \ddot {q} - M \dot {\phi} + C \dot {q} - C \phi + G \\ = \boldsymbol {u} - M \dot {\phi} - C \phi = G - \frac {\partial \Phi}{\partial z _ {1}} - D (z _ {2}) z _ {2}, \tag {11} \\ \end{array}
$$

Consider the following candidate Lyapunov function

$$\boldsymbol {V} (\boldsymbol {z _ {1}}, \boldsymbol {z _ {2}}) = \boldsymbol {\Phi} (\boldsymbol {z _ {1}}) + \frac {1}{2} \boldsymbol {z _ {2} ^ {\top}} M \boldsymbol {z _ {2}}.$$
