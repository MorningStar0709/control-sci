# 2 The forward McKean-Vlasov SDE

In this section, we establish several exponential convergence results for McKean-Vlasov SDEs under two different sets of assumptions $( \mathcal { H } _ { S D E } 1 )$ and $( \mathcal { H } _ { S D E } 2 )$ . Let us consider the following McKean-Vlasov SDE:

$$
\left\{ \begin{array}{l l} \mathrm{d} X _ {t} ^ {s, [ \theta ]} = b (t, X _ {t} ^ {s, [ \theta ]}, [ X _ {t} ^ {s, [ \theta ]} ]) \mathrm{d} t + \sigma (X _ {t} ^ {s, [ \theta ]}, [ X _ {t} ^ {s, [ \theta ]} ]) \mathrm{d} W _ {t}, & t \geq s, \\ X _ {s} ^ {s, [ \theta ]} = \theta \in \mathrm{L} ^ {p} (\Omega , \mathbb {R} ^ {d}) \text {is} \mathcal {G} \text {-measurable}, \end{array} \right. \tag {MV-SDE}
$$

where $p \geq 2 .$ , ?? and ?? satisfy the following assumptions.

Assumption 1. $b : \mathbb { R } _ { + } \times \mathbb { R } ^ { d } \times \mathcal { P } _ { p } \mapsto \mathbb { R } ^ { d }$ and $\sigma : \mathbb { R } ^ { d } \times \mathcal { P } _ { p } \mapsto \mathbb { R } ^ { d \times d }$ are two measurable functions such that

1. for all $t \geq 0 \ b ( t , \cdot , \cdot )$ is uniformly Lipschitz,   
2. for all $\boldsymbol { x } \in \mathbb { R } ^ { d }$ and $\mu \in \mathcal P _ { p } , t \mapsto b ( t , x , \mu )$ is bounded,   
3. $\sigma$ is Lipschitz.

We emphasize that Assumption 1 is only here for the well-posedness of (MV-SDE) and this set of assumptions is supposed to be fulfilled throughout the article. If we consider a r.v. $\theta ^ { \prime } \in \mathrm { L } ^ { p } ( \Omega , \mathbb { R } ^ { d } )$ that is not G-measurable, then we can construct a G-measurable r.v. $\theta _ { \mu }$ that only depends on $\mu : = [ \theta ^ { \prime } ]$ and such that $[ \theta ] = \mu = [ \theta ^ { \prime } ]$ since the probability space (Ω, G, P) is atomless: see e.g. [10, Page 352]. In this case, we define it by a slight abuse of the notation $X ^ { s , [ \theta ^ { \prime } ] } : = X ^ { s , [ \theta _ { \mu } ] }$ . Moreover, we can remark that if $\theta ^ { \prime }$ is $\mathcal { F } _ { s }$ -measurable but not $\mathcal { G }$ -measurable, then $X ^ { s , [ \theta ^ { \prime } ] }$ is not equal to the solution of (MV-SDE) starting from $\theta ^ { \prime }$ at time ?? but these two processes are equal in distribution.
