$$c _ {1} [ J (L) - J (L ^ {*}) ] + c _ {2} \| L - L ^ {*} \| _ {F} ^ {2} \leq \langle \nabla J (L), \nabla J (L) \rangle , \tag {11a}c _ {3} \| L - L ^ {*} \| _ {F} ^ {2} \leq J (L) - J (L ^ {*}), \tag {11b}\left\| \nabla J (L) - \nabla J \left(L ^ {\prime}\right) \right\| _ {F} \leq \ell \| L - L ^ {\prime} \| _ {F}, \tag {11c}$$

Remark 4. The proposition above implies that $J ( . )$ has the Polyak-Łojasiewicz (PL) property (aka gradient dominance) on every $S _ { \alpha } ; \mathrm { i . e . }$ ., for any $L \in { \dot { S _ { \alpha } } }$ we have $\begin{array} { r } { J ( L ) - \dot { J ( L ^ { * } ) } \leq \frac { 1 } { c _ { 1 } ( \alpha ) } \langle \dot { \nabla } \dot { J ( L ) } , \dot { \nabla } J ( \bar { L } ) \rangle } \end{array}$ .
