In this paper, since the state dynamics is time-homogeneous, we assume that agents are only allowed to use stationary policies, meaning that the policies do not change over time.

For a given joint policy $\pmb { \pi } = ( \pi _ { 1 } , . . . , \pi _ { N } ) \in \Pi ^ { N }$ , the cost of agent i is the following discounted cost

$$
\begin{array}{l} J _ {i} (\mu ; \pmb {\pi}) := \mathbb {E} _ {\mu} ^ {\pmb {\pi}} \left[ \sum_ {t = 0} ^ {\infty} \beta^ {t} \bar {c} (x _ {i} (t), a _ {i} (t), z _ {i} (t)) \right] \\ = \mathbb {E} _ {\mu} ^ {\boldsymbol {\pi}} \left[ \sum_ {t = 0} ^ {\infty} \beta^ {t} c (x _ {i} (t), a _ {i} (t), \mathrm{e} [ \cdot | \mathbf {x} (t) ]) \right] \\ \end{array}
$$

where $\beta \in ( 0 , 1 )$ is the discount factor and $\mu$ is the common initial distribution; that is,

$$(x _ {1} (0), \dots , x _ {N} (0)) \sim \mu^ {\otimes N} := \bigotimes_ {i = 1} ^ {N} \mu$$

Let us now give the definition of ε-Nash equilibrium, which is the canonical optimality notion adopted in game theory.

DEFINITION 1 (ε-Nash equilibrium). For a given $\varepsilon \ge 0$ , a joint policy $\pi ^ { * } = ( \pi _ { 1 } ^ { * } , \ldots , \pi _ { N } ^ { * } )$ is ε-Nash equilibrium if

$$J _ {i} (\mu ; \boldsymbol {\pi} ^ {*}) \leq \inf _ {\pi \in \Pi} J _ {i} (\mu ; \pi , \boldsymbol {\pi} _ {- i} ^ {*}) + \varepsilon$$

for all $i = 1 , \ldots , N$ , where $\pi _ { - i } ^ { * } : = \{ \pi _ { j } ^ { * } \} _ { j \neq i } . \operatorname { I f } \varepsilon = 0$ , then we have Nash equilibrium.

In this paper, our goal is to obtain approximate Nash equilibrium for N -agent games, where N is assumed to be sufficiently large, via studying the stationary infinite population limit. More precisely, we establish that if all agents apply the equilibrium policy in the infinite population limit, which is introduced in the next section, this joint policy is proved to be $\varepsilon ( N )$ -Nash equilibrium for N -agent games, where there is an explicit relation between $\varepsilon ( N )$ and N .
