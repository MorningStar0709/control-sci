# 5.2 Data-driven predictive control with approximated linear data-driven representation

We use the approximately optimal linear model $H _ { \mathrm { o p } } ^ { * }$ from Algorithm 1 as the predictor in data-driven predictive control, which leads to

$$
\min_ {\substack {g, \sigma_ {y} \in \Gamma , \\ u \in \mathcal {U}, y \in \mathcal {Y}}} \| u \| _ {R} ^ {2} + \| y \| _ {Q} ^ {2} + \lambda_ {y} \| \sigma_ {y} \| _ {2} ^ {2} + \lambda_ {1} \| g \| _ {1} \tag{38}
$$

$\mathrm { s u b j e c t ~ t o } \quad H _ { \mathrm { o p } } ^ { * } g { = } \mathrm { c o l } ( u _ { \mathrm { i n i } } , y _ { \mathrm { i n i } } + \sigma _ { y } , u , y ) ,$

where the l1-norm regularizer is used to implicitly select the trajectory in $H _ { \mathrm { o p } } ^ { * }$ . We call this formulation (38) as Approximation-based DDPC (A-DDPC). We note that $H _ { \mathrm { o p } } ^ { * }$ only needs to be computed once via the offline collected data, and the initial trajectories $u _ { \mathrm { i n i } } , y _ { \mathrm { i n i } }$ are updated online at each step.

The SLRA [52, 53], which utilizes SVD and projection onto the set of Hankel matrices iteratively, has been wellstudied. Sufficient conditions for its convergence guarantee are discussed in [54]. However, these results cannot be directly applied to Algorithm 1 as we require extra separation steps to divide $H _ { y }$ into the row and null spaces of $H _ { u }$ and the causality projection. In numerical implementation, Algorithm 1 is applied one time offline and was observed to converge in all our experiments with various pre-collected trajectories. Establishing a formal theoretical proof and analyzing its convergence rate remain open, which we leave for future work.
