$$
\begin{array}{l} \int_ {X \times Y} c (x, y) d \omega (x, y) = \int_ {\Gamma} (c (x, y)) f (x) d \gamma (x, y) \\ \leq \int_ {\Gamma} (A c _ {X} (x) + B) f (x) d \gamma (x, y) \\ = \int_ {X} (A c _ {X} (x) + B) f (x) d \mu (x) \\ = A \int_ {X} c _ {X} (x) d \eta (x) + B <   \infty . \\ \end{array}
$$

Clearly $\omega \ll \gamma .$ with Radon-Nikodym derivative $\begin{array} { r } { \frac { d \omega } { d \gamma } ( x , y ) = f ( x ) } \end{array}$ . Thus ω satisfies the conditions of Corollary 1 and is an optimal transport plan between the marginals η and ζ.

Finally, we show that ζ(y) has finite cY -integral by noting that

$$
\begin{array}{l} \int_ {Y} c _ {Y} (y) d \zeta = \int_ {X \times Y} c _ {Y} (y) d \omega (x, y) \\ = \int_ {\Gamma} c _ {Y} (y) d \omega (x, y) \\ <   \int_ {\Gamma} A c _ {X} (x) + B d \omega (x, y) \\ = \int_ {X \times Y} A c _ {X} (x) + B d \omega (x, y) \\ = \int_ {X} A c _ {X} (x) + B d \eta (x) <   \infty \\ \end{array}
$$

Corollaries 2 and 3 are two complementary means of ensuring the new coupling ω has finite cost. Note, that only Corollary 3 ensures us that $\zeta ( y ) = ( \mathrm { p r o j } _ { Y } ) _ { \# } ( f ( x ) \gamma ( x , y ) )$ has finite cY -integral.

Based on the results of the section, we have shown that we can use existing optimal couplings to guarantee the optimality of absolutely continuous couplings (when finite). This led to two corollaries that allow us to send an absolutely continuous marginal ‘through’ an optimal coupling. These corollaries show that we can treat optimal couplings like Markov Kernels, which we will show in the next section.
