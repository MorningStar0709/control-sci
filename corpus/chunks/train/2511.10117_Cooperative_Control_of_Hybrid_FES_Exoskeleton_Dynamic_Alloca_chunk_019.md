Now, given the definition $S = \mathrm { d i a g } ( \pmb { \sigma } )$ , see (7), for $\tau ^ { N } \geq 0 .$ we have $\| S \zeta \| = | \zeta _ { 1 } |$ , and from (16), it follows that $\dot { \zeta } _ { 2 } = 0$ meaning $\zeta _ { 2 }$ remains constant. Similarly, for $\tau ^ { N } < 0 .$ we obtain $\| S \zeta \| = \| \zeta _ { 2 } |$ , and again from (16), $\dot { \zeta } _ { 1 } = 0 .$ so $\zeta _ { 1 }$ remains constant. This guarantees that all signals within the allocation dynamics remain bounded for any bounded external input τ¯, thereby ensuring that the allocation dynamics is inputto-state stable with respect to τ¯ (see Appendix B for more details). Consequently, the redistribution term $g _ { \perp } ^ { * } S \zeta$ in (11a) is invisible to $\Sigma _ { H } \ ( 1 )$ .

Moreover, the matrices K and $W ( t )$ are chosen in a way to comply with two other conditions in (15), namely, $\tau ^ { F } \in \mathbb { A } _ { F }$ and $\tau ^ { E } \in \mathbb { A } _ { E }$ . ■

Remark 8 (Switching Allocation Dynamics). The allocation dynamics defined in (16) represents a time-dependent switching system [44], where the switching signal is determined by the sign of $\tau ^ { N } ( t )$ as defined by S.
