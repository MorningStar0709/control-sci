As the models of time-varying delays τupk $\tau _ { k } ^ { \mathrm { u p } }$ an d $\tau _ { k } ^ { \mathrm { d w } }$ , this paper consider stochastic processes. We formally introduce the 2-dimensional stochastic process

$$
\xi = \left(\xi_ {k}\right) _ {k \in \mathbf {N} _ {0}}, \quad \xi_ {k} = \left[ \begin{array}{l l} \tau_ {k} ^ {\mathrm{up}} & \tau_ {k} ^ {\mathrm{dw}} \end{array} \right] ^ {T}. \tag {8}
$$

Then, only the following is used as the essential restriction on $\xi$ in this paper.

Assumption $\mathit { \Pi } _ { l : \ \xi _ { k } }$ is independent and identically distributed (i.i.d.) with respect to $k \in \mathbf { N } _ { 0 }$ .

This assumption does not deny the dependence between $\tau _ { k } ^ { \mathrm { u p } }$ and $\tau _ { k } ^ { \mathrm { d w } }$ at common k. The distribution of $\xi _ { k }$ is arbitrary, and the support is not required to be unbounded; although another assumption will be additionally introduced on the NCS Σ in the following section, it is a minimal requirement for defining second-moment stability, and does not imply the degradation of applicability of the proposed approach.

Remark 1: Since the delays generally have physical lower bounds, we may also assume that $\tau _ { k } ^ { \mathrm { u p } }$ and $\tau _ { k } ^ { \mathrm { d w } }$ are positive in practice.
