# III. CATL+ SYNTAX AND SEMANTICS

In this section we introduce Capability Temporal Logic plus (CaTL+), a logic for specifying requirements for multiagent systems. CaTL+ has two layers: the inner logic and the outer logic. The inner logic is identical to STL [2] and is defined on an individual trajectory. In the previous example, the inner logic can specify “eventually visit $V _ { 1 }$ within 25 time units”. The outer logic specifies behaviors of the team trajectory. With reference to the same example, the outer logic could specify “3 agents with capability “Delivery” should eventually visit $V _ { 1 }$ within 25 time units”.
