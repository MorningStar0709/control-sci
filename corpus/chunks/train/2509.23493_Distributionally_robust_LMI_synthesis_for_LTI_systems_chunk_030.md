We use the correlated ambiguity set ${ \mathcal { W } } _ { \mathrm { c o r } }$ instead of the independent set ${ \mathcal { W } } _ { \mathrm { i n d } }$ because wind turbine blades are sensitive to temporally correlated wind disturbances. Wind turbulence exhibits low-frequency components that persist over time, causing sustained loads on the blade. The ambiguity set ${ \mathcal { W } } _ { \mathrm { c o r } }$ allows disturbances to capture these temporal dependencies, enabling the controller to mitigate worst-case scenarios where disturbances amplify specific modes, such as flapwise or torsional oscillations.

The controller synthesis is based on the problem (3) with the correlated ambiguity set $\mathcal { W } _ { \mathrm { c o r } } .$ . The nominal covariance matrix is $\Sigma _ { \mathrm { n o m } } = I _ { 2 }$ , and the Wasserstein radius is set to $\gamma = 0 . 5$ .

We synthesized two controllers: a standard $H _ { 2 }$ controller, minimizing the nominal $H _ { 2 }$ cost, and a DR controller, solving (3) for ${ \mathcal { W } } _ { \mathrm { c o r } } .$ . The DR controller is designed using the convex LMI formulation in Theorem 6.4. The $H _ { 2 }$ controller is synthesized using standard LMI-based $H _ { 2 }$ synthesis. Both controllers are implemented as output-feedback controllers of the form (2), with $A _ { c } , B _ { c } , C _ { c } , D _ { c }$ derived from the LMI solutions via the parametrization in Theorem 6.3.
