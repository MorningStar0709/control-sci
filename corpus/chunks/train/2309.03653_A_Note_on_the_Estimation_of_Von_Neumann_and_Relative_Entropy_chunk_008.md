# III. ESTIMATION OF VON NEUMANN & RELATIVE ENTROPY

Claude Shannon introduced the idea of the entropy [17] of a random variable: given a random variable x with (finite) probability distribution

$$(P _ {1}, P _ {2}, \dots , P _ {n}) = (p (x _ {1}), p (x _ {2}), \dots , p (x _ {n})) \tag {12}$$

the Shannon entropy is

$$Q (\boldsymbol {x}) = Q (P _ {1}, P _ {2}, \dots , P _ {n}) \triangleq - \sum_ {i = 1} ^ {n} p _ {i} \log p _ {i}. \tag {13}$$

This entropy can be seen as both the average information gain when we learn the value of x, and, the average amount of uncertainty before we learn the value of x. The definition 0 log $0 \overset { \triangle } { = } 0$ is used for the value of the entropy at the origin. Later in [2], von Neumann introduced the von Neumann quantum entropy

$$S (\boldsymbol {\varrho}) \triangleq - \operatorname{tr} (\boldsymbol {\varrho} \log \boldsymbol {\varrho}) \tag {14}$$

of the density $\varrho \in { \mathcal { S } }$ . In this case, S measures the $^ { 6 6 } \mathrm { u n - }$ certainty” of a quantum density operator. A pure state has entropy zero; however, the entropy of a mixed state is non-zero since it represents an ensemble of systems in various states.

Because $\varrho$ is Hermitian, it can be diagonalized by a unitary $Q$ such that $\varrho = Q A Q ^ { \dagger }$ . Using this fact and standard properties of the trace and matrix logarithm, one can show

$$S (\boldsymbol {\varrho}) = S (\boldsymbol {\Lambda}) = - \sum_ {k = 1} ^ {d} \lambda_ {k} \log \lambda_ {k} \tag {15}$$

which is also the Shannon entropy

$$Q (\boldsymbol {\lambda}) \equiv - \sum_ {k = 1} ^ {d} \lambda_ {k} \log \lambda_ {k} \tag {16}$$

given $\pmb { \lambda } = \mathrm { d i a g } \{ \pmb { A } \}$ . It is known, c.f. [18], that $S ( \varrho )$ is nonnegative and bounded:

$$0 \leq S (\boldsymbol {\varrho}) \leq \log d. \tag {17}$$

Definition 1. A bounded function $f : \mathbb { R } _ { + } \to \mathbb { C } ^ { d \times d }$ is said to be essentially exponentially convergent if there exists a finite time $T \in ( 0 , \infty )$ and positive constants M and σ such that

$$\| f (t) \| \leq M e ^ {- \sigma t}, \quad \forall t \geq T.$$
