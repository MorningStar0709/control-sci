# 13.2.1 Plant with Uncertainty

Plant model is supposed to be a member of a very large set of admissible models given by:

$$\mathcal {G} = \bigcup_ {\theta \in \Theta} G (\theta) \tag {13.1}$$

where $\Theta$ is a compact set and each $G ( \theta )$ is a family of models centered around a nominal model $G _ { 0 } ( \theta )$ . For example, if we consider multiplicative uncertainty, the plant model can be represented by

$$G (\theta) = G _ {0} (\theta) [ 1 + W \varDelta ]$$

where $W$ is the uncertainty filter and $\varDelta$ is a stable transfer function with $\| \varDelta \| _ { \infty } < 1$ . $\mathbf { A } s$ a result, $\mathcal { G }$ represents the parametric uncertainty (or variation) and for each fixed $\theta .$ , the subfamily of $G ( \theta )$ represents the unmodeled dynamics.
