# 3.2 An Iterative Solution

Problems (15) and (16) are non-convex even for Gaussian uncertainties. This subsection derives an SQP algorithm to solve (15) and (16) under the following assumption.

Assumption 7. 1) The uncertainties are Gaussian, 2) the Page construction is used, and 3) $\Sigma _ { \nu } ^ { d } ( \tau ) = \mathbf { 0 } _ { n \times n }$ for all $\tau > 0$ .

Assumption 7.1 specifies a particular $f ( \cdot )$ , whereas Assumptions 7.2 and 7.3 lead to a block diagonal $\Sigma _ { g } ( \mathbf { g } )$ . This iterative algorithm can serve as a computationally efficient approximate solution to the problem in practice, even when Assumption 7 is not satisfied. It also demonstrates the relations between the MAP framework and existing algorithms for data-driven prediction and control.

Under Assumption 7, (15) is equivalent to

$$\hat {\mathbf {g}} = \operatorname{argmin} _ {\mathbf {g}} J (\mathbf {g}), J (\mathbf {g}) := \log \det \left(\Psi (\mathbf {g})\right) + \left\| \boldsymbol {\delta} _ {\zeta} (\mathbf {g}) \right\| _ {\Psi^ {- 1} (\mathbf {g})} ^ {2},$$

where $\Psi ( \mathbf { g } ) = \left\| \mathbf { g } \right\| _ { 2 } ^ { 2 } \Phi \left( \mathbb { I } _ { L } \otimes \Sigma _ { \nu } ^ { d } ( 0 ) \right) \Phi ^ { \top } + \Sigma _ { \epsilon }$ from (9). Let $\Lambda : = \Phi \left( \mathbb { I } _ { L } \otimes \Sigma _ { \nu } ^ { d } ( 0 ) \right)$ Φ⊤ and $\hat { \bf g } _ { k }$ be the k-th iterate of the iterative algorithm. We approximate $J ( \mathbf { g } )$ locally at gˆk with a quadratic function as

$$J (\mathbf {g}) \approx J (\hat {\mathbf {g}} _ {k}) + \operatorname{tr} \left(\Lambda \Psi^ {- 1} (\hat {\mathbf {g}} _ {k})\right) \left(\| \mathbf {g} \| _ {2} ^ {2} - \| \hat {\mathbf {g}} _ {k} \| _ {2} ^ {2}\right)- \left\| \Psi^ {- 1} (\hat {\mathbf {g}} _ {k}) \boldsymbol {\delta} _ {\zeta} (\hat {\mathbf {g}} _ {k}) \right\| _ {\Lambda} ^ {2} \left(\| \mathbf {g} \| _ {2} ^ {2} - \| \hat {\mathbf {g}} _ {k} \| _ {2} ^ {2}\right)+ \left(\| \boldsymbol {\delta} _ {\zeta} (\mathbf {g}) \| _ {\Psi^ {- 1} (\hat {\mathbf {g}} _ {k})} ^ {2} - \| \boldsymbol {\delta} _ {\zeta} (\hat {\mathbf {g}} _ {k}) \| _ {\Psi^ {- 1} (\hat {\mathbf {g}} _ {k})} ^ {2}\right),$$
