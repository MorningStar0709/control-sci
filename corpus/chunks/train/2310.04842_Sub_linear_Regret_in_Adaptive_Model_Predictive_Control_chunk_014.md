$$
\begin{array}{l} R _ {T} (\pi) = \sum_ {t = 0} ^ {T - 1} \left(x _ {t} ^ {\pi^ {\top}} Q (x _ {t} ^ {\pi} - x _ {t} ^ {\star}) + (x _ {t} ^ {\pi} - x _ {t} ^ {\star}) ^ {\top} Q x _ {t} ^ {\star}\right) \\ + \sum_ {t = 0} ^ {T - 1} \left(u _ {t} ^ {\pi^ {\top}} R (u _ {t} ^ {\pi} - u _ {t} ^ {\star}) + (u _ {t} ^ {\pi} - u _ {t} ^ {\star}) ^ {\top} R u _ {t} ^ {\star}\right) \\ \leq \sum_ {t = 0} ^ {T - 1} (\| x _ {t} ^ {\pi} \| + \| x _ {t} ^ {\star} \|) \| Q \| \| x _ {t} ^ {\pi} - x _ {t} ^ {\star} \| \\ + \sum_ {t = 0} ^ {T - 1} (\| u _ {t} ^ {\pi} \| + \| u _ {t} ^ {\star} \|) \| R \| \| u _ {t} ^ {\pi} - u _ {t} ^ {\star} \| \\ = \bar {Q} \sum_ {t = 0} ^ {T - 1} \| x _ {t} ^ {\pi} - x _ {t} ^ {\star} \| + \bar {R} \sum_ {t = 0} ^ {T - 1} \| u _ {t} ^ {\pi} - u _ {t} ^ {\star} \|, \\ \end{array}
$$

where the first inequality follows from Cauchy-Schwartz inequality, and the second by defining ${ \bar { Q } } : = 2 \| Q \| { \bar { x } }$ and ${ \bar { R } } : = 2 \| R \| { \bar { u } }$ with ${ \bar { x } } \ = \ \operatorname* { m a x } _ { ( x , u ) \in { \mathcal { C } } } \left\{ \| x \| \right\}$ and $\bar { u } \ =$ $\operatorname* { m a x } _ { ( x , u ) \in { \mathcal { C } } } \{ \| u \| \}$ . Note that $\bar { Q }$ and R¯ are well defined since the constraint set C is compact by Assumption 2.3. Further note that $( x _ { t } , u _ { t } ) \in \mathcal { C }$ is guaranteed by the recursive feasibility of STT-MPC (Theorem 3.4).

We seek to upper bound the quantities $\| x _ { t } ^ { \pi } - x _ { t } ^ { \star } \|$ and $\parallel u _ { t } ^ { \pi } -$ $\boldsymbol u _ { t } ^ { \star } \parallel$ in terms of $\lVert { \boldsymbol { \theta } } _ { t } - { \boldsymbol { \theta } } ^ { \star } \rVert$ . To this end, we make use of the following theorem, presented originally in (Shin et al., 2022) (Theorem 4.5) for graph-structured nonlinear optimization problems, and specialized to constrained nonlinear systems by (Lin et al., 2022) (Theorem H.1.).

Theorem 4.2. For any tube-MPC algorithm $\pi \in \Pi ,$ , there exists $R > 0$ such that, for $x _ { i } , x _ { i } ^ { \prime } \in B ( x _ { i } ^ { \star } , R )$ , where $i \in$ $\{ 0 , \ldots , t \}$ , the following perturbation bounds hold:

$$\left\| z _ {t} ^ {\pi} \left(x _ {i}, \theta\right) - z _ {t} ^ {\pi} \left(x _ {i} ^ {\prime}, \theta\right) \right\| \leq q _ {1} (t - i) \| x _ {i} - x _ {i} ^ {\prime} \|, \tag {14}$$

and

$$\left\| v _ {t} ^ {\pi} \left(x _ {i}, \theta\right) - v _ {t} ^ {\pi} \left(x _ {i}, \theta^ {\prime}\right) \right\| \leq q _ {2} (t - i) \| \theta - \theta^ {\prime} \|, \tag {15}$$
