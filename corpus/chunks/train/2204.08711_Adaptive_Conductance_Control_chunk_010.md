# 2.2. Conductance-based model of a neural network

A conductance-based neural network is given by the interconnection of $n _ { v } \in$ N single neurons. We denote the index set of neurons in the network by $\mathcal { N } : =$ $\{ 1 , \ldots , n _ { v } \}$ and we describe the dynamics of the $i ^ { \mathrm { t h } }$ neuron in the network by attaching an $i \in \mathcal N$ subscript to all the variables in (1)-(4), except for reversal potentials, time-constant functions, and activation functions.

The network state-space model gathers all neuronal membrane voltages in the vector

$$\boldsymbol {v} = (v _ {1}, \dots , v _ {n _ {v}}) ^ {\mathsf {T}},$$

and all other state variables in the vector

$$w = \operatorname{col} (w ^ {(1)}, \dots , w ^ {(n _ {v})}),$$

where the vector $w ^ { ( i ) }$ collects the intrinsic and synaptic gating variables of the $i ^ { \mathrm { t h } }$ neuron, that is, $m _ { \mathrm { i o n } , i } , h _ { \mathrm { i o n } , i }$ , and $s _ { \mathrm { s y n } , p , i }$ , as well as the calcium concentration [Ca]i. Notice that with this notation, $\mathcal { P } \subseteq \mathcal { N }$ .

In addition to synaptic current interconnections, neuronal networks may include electrical gap junctions, modelled as ohmic resistive currents flowing between neurons. Gap junction currents are thus passive components of the network, just as leak currents are passive components in a single neuron. It follows that the model of a conductance-based neural network has a form completely analogous to that of a single neuron, given by (1). The network model is given by the vector equations

$$c \dot {v} = - I _ {\text { leak }} - I _ {\text { int }} - I _ {\text { ext }} + u \tag {6a}\dot {w} = g (v, w) \tag {6b}$$

where $I _ { \mathrm { l e a k } }$ is the overall leaky current, given by

$$
I _ {\mathrm{leak}} = \left( \begin{array}{c} \mu_ {\mathrm{leak}, 1} (v _ {1} - E _ {\mathrm{leak}}) + \sum_ {i \in \mathcal {N} \setminus \{1 \}} \mu_ {\mathrm{gap}, 1, i} (v _ {1} - v _ {i}) \\ \vdots \\ \mu_ {\mathrm{leak}, n _ {v}} (v _ {n _ {v}} - E _ {\mathrm{leak}}) + \sum_ {i \in \mathcal {N} \setminus \{n _ {v} \}} \mu_ {\mathrm{gap}, n _ {v}, i} (v _ {n _ {v}} - v _ {i}) \end{array} \right)
$$
