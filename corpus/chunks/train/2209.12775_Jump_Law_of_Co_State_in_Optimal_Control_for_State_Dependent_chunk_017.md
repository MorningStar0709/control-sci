# B. Examples

We propose an iterative algorithm (called GEL) to solve the optimal control of such systems efficiently leveraging the (co-state) jump laws in (15) and (20). The main idea is to construct the following function:

$$
\begin{array}{l} F (\tau) = \Delta \lambda (\tau) - G (\lambda_ {+}, \lambda_ {-}, \tau , x _ {-}, x _ {+}) (36) \\ = \lambda_ {+} - \lambda_ {-} - G (\lambda_ {+}, \lambda_ {-}, \tau , x _ {-}, x _ {+}), (37) \\ \end{array}
$$

where

$$
\begin{array}{l} G (\lambda_ {+}, \lambda_ {-}, \tau , x _ {-}, x _ {+}) = \\ - \frac {(L _ {+} (\tau) - L _ {-} (\tau) + \langle \lambda (\tau) \rangle^ {\top} \Delta f (\tau)) m (\tau)}{\langle m (\tau) ^ {\top} f (\tau) \rangle}. \\ \end{array}
$$

Note that this option is not unique. We can also choose other boundary constraints such as (32) to be the function $F ( \tau )$ . By using MATLAB “bvp4c”, we can solve the optimal control problem of this switching system as two boundary value problems with multiple boundary conditions. Then we find τ by Newton’s Method (a root-founding process), which is

$$\tau_ {k + 1} = \tau_ {k} - \alpha \frac {F (\tau)}{F ^ {\prime} (\tau)}, \tag {38}$$

where α is the step size. The stopping criterion is then $| \tau _ { k + 1 } - \tau _ { k } | \leq$ tol and $| F ( \tau _ { k } ) - 0 | <$ tol. For this derivative part $\begin{array} { r } { \frac { d F } { d \tau } = \frac { F ( \tau + \delta \tau ) - F ( \tau ) } { \delta \tau } } \end{array}$ F (τ+δτ)−F (τ)δτ , a perturbation scheme is used to obtain $F ( \tau + \delta \tau )$ . We summarize the proposed approach in Algorithm 1.

Algorithm 1 Proposed algorithm (named GEL) for solving hybrid system optimal control problem numerically   
Initialize $\tau_{0}=0.5$ , tolerance tol=0.0001, $\delta\tau=0.1$ for k=0:iter do
    Solve two-point boundary value problem (26), (28), (25), (27) using bvp4c
    Calculate $F(\tau_{k})$ if $|F(\tau_{k})-0|<tol$ then
    break
    else
    Add a small perturbation $\tau_{k}^{\prime}=\tau_{k}+\delta\tau$ Calculate $F(\tau_{k}^{\prime})$ Calculate $F^{\prime}(\tau_{k})=\frac{F(\tau_{k}^{\prime})-F(\tau_{k})}{\delta\tau}$ Newton Update: $\tau_{k+1}=\tau_{k}-\frac{F(\tau_{k})}{F^{\prime}(\tau_{k})}$ if $|\tau_{k+1}-\tau_{k}|<tol$ then
    break
    end if
    end if
end for
