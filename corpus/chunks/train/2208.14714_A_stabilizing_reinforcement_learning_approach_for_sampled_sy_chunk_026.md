# B. Adaptive Cruise Control

As a second example, consider a simplified adaptive cruise control problem as investigated in [2], [45]. The system dynamics are given by

$$\dot {v} = - \frac {1}{m} \left(f _ {0} + v f _ {1} + v ^ {2} f _ {2}\right) + \frac {1}{m} u \tag {36}$$

with v describing the velocity of a vehicle. The task is bring the velocity to a desired value $v ^ { * }$ while $f _ { 0 , 1 , 2 }$ are unknown frictional force parameters. Suppose henceforth $v ^ { * } ~ = ~ 1 4$ . These parameters will be estimated by (21b) to yield $\hat { \theta } \stackrel { } { = }$ $[ \hat { f } _ { 0 } \hat { f } _ { 1 } \hat { f } _ { 2 } ] ^ { \top }$ . It was shown in [2] that the function

$$V (v, \theta) = V (v) = \tilde {v} ^ {2} \tag {37}$$

with $\tilde { v } = v - v ^ { * }$ is an adaptive control Lyapunov function associated with the controller

$$\mu (v, \hat {\theta}) = - \varepsilon \frac {m}{2} \tilde {v} + \hat {f} _ {0} + \hat {f} _ {1} v + \hat {f} _ {2} v ^ {2} \tag {38}$$

for some $\varepsilon > 0$ . The effective Lyapunov function $V _ { c }$ is thus given by

$$V _ {c} (x, \tilde {\theta}) = \tilde {v} ^ {2} + \frac {1}{2} \tilde {\theta} ^ {\top} \Gamma^ {- 1} \tilde {\theta}. \tag {39}$$

Again, for simplicity, take $\varphi = [ \tilde { v } ^ { 2 } ] ^ { \top }$ such that $w ^ { \# } = 1$ and $V ( x ) \leq \hat { J } ( x , w )$ for $w \ge 1$ . All system and control associated parameters can be found in [2]. Therein, it was observed that the stabilizing input $\mu$ may take high values. To comply with the desirable bound $\varepsilon _ { w }$ as to ensure decay of $\mathcal { V } _ { c }$ (refer to Theorem 1), the learning rate $l _ { w }$ and the weight matrix R of the stage cost $r ( x , u ) = x ^ { \top } Q x + u ^ { \top } R u$ with

$$r (v, u) = \tilde {v} ^ {2} + 1 0 ^ {- 7} u ^ {2}$$

are chosen sufficiently small. Note that by choosing a small learning rate, switches the the backup controller may occur less frequent. Note further that R can be chosen freely and the condition can be met by the choice of $l _ { w } .$ The sampling time is set to $\delta = 0 . 0 1$ and $w _ { 0 } = w ^ { \# }$ .

Fig. 4 depicts the practically stable state with respect to $v ^ { * }$ and the associated control values $u _ { k }$ together with the weight $w _ { k }$ of the critic and the effective Lyapunov function of the system $\mathcal { V } _ { c } .$ It was observed that $\mathcal { C } _ { \% } ( v _ { 0 } = 1 0 ) \approx 0 . 1 2$ which indicates a significant performance improvement.
