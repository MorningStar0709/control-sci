# 10.6 Regimes Instantiation

The emissions framework instantiates the four regimes as follows:

CPCA: Constant Policy, Constant Agents. $\lambda _ { t } , \tau _ { t } , \sigma _ { t }$ fixed; no technological learning $( \eta _ { i } = 0 )$ . Represents baseline or static compliance scenarios.

CPVA: Constant Policy, Variable Agents. Policy fixed; agents adapt via efficiency gains or technology adoption $( \eta _ { i } > 0 )$ .

VPCA: Variable Policy, Constant Agents. Policymaker adapts $P _ { t } ;$ firms do not change technology.

VPVA: Variable Policy, Variable Agents. Both policy and behavior adapt; fundamental feedbacks emerge, often yielding oscillatory or drifting emissions trajectories.
