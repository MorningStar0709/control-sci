# 4.1 Practical homogeneity degree

Let us study the problem of identification of an unknown homogeneity degree of the function h under the assumption that the linear dilation $\mathbf { d } ( s ) = e ^ { s G _ { \mathbf { d } } } , s \in \mathbb { R }$ is known.

By Definition 2, the scalar-valued function $h : \mathbb { R } ^ { n } \mapsto \mathbb { R }$ is d-homogeneous of degree $\nu \in \mathbb { R }$ if

$$h (e ^ {s G _ {\mathbf {d}}} x) = e ^ {\nu s} h (x), \quad \forall x \in \mathbb {R} ^ {n}, \quad \forall s \in \mathbb {R}, \tag {16}$$

where $\mathbf { d } ( s ) = e ^ { s G _ { \mathbf { d } } }$ is a linear dilation in $\mathbb { R } ^ { n }$ . If $j _ { \epsilon }$ approximates a d-homogeneous function $g = h$ on Ω with a high enough precision then we should have

$$g _ {\epsilon} (\mathbf {d} (s) x) \approx e ^ {\nu s} g _ {\epsilon} (x), \quad x \in \Omega , \quad s \in \mathbb {R}: \mathbf {d} (s) x \in \Omega .$$

Let a norm $\left\| \cdot \right\|$ in $\mathbb { R } ^ { n }$ and the set Ω be such that $S = \{ x \in \mathbb { R } ^ { n } : \| x \| = 1 \} \subset \Omega$ and the dilation d be monotone with respect to $\| \cdot \|$ . In this case, due to $\mathbf { d } ( - \ln \| x \| _ { \mathbf { d } } ) x \in S$ , the above approximate identity implies

$$g _ {\epsilon} (\mathbf {d} (- \ln \| x \| _ {\mathbf {d}}) x) \approx \| x \| _ {\mathbf {d}} ^ {- \nu} g _ {\epsilon} (x), \quad \forall x \in \Omega , \tag {17}$$

where $\| \cdot \| _ { \mathbf { d } }$ is the canonical homogeneous norm induced by the norm $\| \cdot \|$ . Hence, for $x \neq \mathbf { 0 }$ we have

$$\nu \approx \frac {1}{\ln \| x \| _ {\mathbf {d}}} \ln \frac {g _ {\epsilon} (x)}{g _ {\epsilon} (\mathbf {d} (- \ln \| x \| _ {\mathbf {d}}) x)} \tag {18}$$

provided that

$$0 < \frac {g _ {\epsilon} (x)}{g _ {\epsilon} (\mathbf {d} (- \ln \| x \| _ {\mathbf {d}}) x)} < + \infty . \tag {19}$$

Notice that the implication

$$h (x) \neq 0 \quad \Rightarrow \quad 0 < \frac {h (x)}{h (\mathbf {d} (s) x)} < + \infty , \quad \forall s \in \mathbb {R}$$

is necessary for a function h to be homogeneous. Indeed, if $h ( x ) \neq 0$ and $\frac { h ( x ) } { h ( - \mathbf { d } ( \ln \| x \| _ { \mathbf { d } } ) x ) } \ < \ 0$ then the identity (16) is impossible and h is surely nonhomogeneous. Therefore, the condition (19) can be utilized for detection of the non-homogeneity of $g _ { \epsilon }$ .
