# C. Fast Motions

To investigate the existence of limit cycles in the undisturbed closed-loop system—i.e., when $f ( t ) ~ = ~ 0 ,$ , implying $\sigma _ { 0 } ( t ) = 0 \mathrm { ~ }$ —the SM controller is approximated by its SIDF, $\overline { { N _ { 1 } } } ( A , \omega )$ , and the linear subsystem by its frequency response W (jω). This yields a quasi-linearized of Figure 5, wherein the steady-state oscillation $\sigma ( t ) ~ = ~ A \sin ( \omega t )$ leads to the following frequency-domain relation

$$
\left[ \begin{array}{c c} \overline {{N _ {1}}} (A, \omega) & - 1 \\ 1 & W (j \omega) \end{array} \right] \left[ \begin{array}{l} \sigma (j \omega) \\ \bar {u} (j \omega) \end{array} \right] = 0. \tag {22}
$$

The matrix equation (22) admits non-trivial solutions if and only if its determinant vanishes. This condition is classically reformulated as the harmonic balance (HB) equation,

$$\overline {{{N _ {1}}}} (A, \omega) W (j \omega) + 1 = 0, \tag {23}$$

whose solutions $( A ^ { * } , \omega ^ { * } )$ can represent different response modes, some of which may be unstable [28].

Consider any linear block $W ( j \omega )$ operating in closed-loop with the relay-type controller characterized by the SIDF (19). The HB equation (23) can be generally expressed as

$$\underbrace {\frac {4 \rho}{\pi A} + \operatorname{Re} \left\{\frac {1}{W (j \omega)} \right\}} _ {\mathbf {U} (A, \omega)} + j \underbrace {\operatorname{Im} \left\{\frac {1}{W (j \omega)} \right\}} _ {\mathbf {V} (\omega)} = 0, \tag {24}$$

whose solution consists of finding

$$\omega^ {*} > 0: \mathbf {V} (\omega^ {*}) = 0 \Rightarrow A ^ {*} > 0: \mathbf {U} (A ^ {*}, \omega^ {*}) = 0. \tag {25}$$

The linear block is constructed as the cascade connection of the critically damped second-order actuator dynamics, described by (4), and the nominal (undisturbed) tracking error dynamics given in (1). The resulting transfer function (TF) is therefore given by

$$W (s) = \frac {1}{s (\mu s + 1) ^ {2}}, \tag {26}$$

where $0 ~ < ~ \mu ~ \ll ~ 1$ denotes the actuator time constant. Following the HB-based algorithm (25) applied to the linear block (26), the predicted amplitude and frequency of the main chattering harmonics (5) are given by

$$\omega^ {*} = \frac {1}{\mu}, \quad A ^ {*} = \frac {2 \rho}{\pi} \mu . \tag {27}$$
