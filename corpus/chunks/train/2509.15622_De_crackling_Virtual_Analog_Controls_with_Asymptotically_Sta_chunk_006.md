# 3.1. Stable conditioned models

We will consider the networks in their autonomous state (with zero input). That is, we consider each recurrent network as a system

$$\mathbf {h} _ {t} = f \left(\mathbf {h} _ {t - 1}, \mathbf {p} _ {t}\right) \tag {1}$$

where $\mathbf { h } _ { t - 1 }$ is the previous hidden state and $\mathbf { p } _ { t } \in \mathbb { R } ^ { p }$ is the controlconditioning vector.

For this system to be asymptotically stable, the hidden state $\mathbf { h } _ { t }$ should decay towards and to an equilibrium $\mathbf { h } _ { e }$ . We will also restrict the norm of the equilibrium to be zero to eliminate DC offset in the output. That is, we want

$$\lim _ {t \rightarrow \infty} \| \mathbf {h} _ {\mathbf {t}} - \mathbf {h} _ {e} \| = \lim _ {t \rightarrow \infty} \| \mathbf {h} _ {\mathbf {t}} \| = 0. \tag {2}$$
