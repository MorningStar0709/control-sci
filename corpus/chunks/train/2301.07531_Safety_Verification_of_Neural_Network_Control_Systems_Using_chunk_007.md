# III. GUARANTEED NEURAL NETWORK MODEL REDUCTION

Given a large-scale neural network Φ, there exist a large number of neural network model reduction methods as in survey paper [8] to obtain its reduced-size version Φˆ as below:

$$
\left\{ \begin{array}{l} \hat {\boldsymbol {\eta}} _ {\ell} = \hat {\phi} _ {\ell} (\hat {\mathbf {W}} _ {\ell} \hat {\boldsymbol {\eta}} _ {\ell - 1} + \hat {\mathbf {b}} _ {\ell}), \ell = 1, \dots , \hat {L} \\ \hat {\boldsymbol {\eta}} _ {L} = \hat {\Phi} (\hat {\boldsymbol {\eta}} _ {0}) \end{array} \right. \tag {10}
$$

To enable guaranteed neural network model reduction, the key is how to rigorously compute the output difference between the original neural network Φ and its reducedsize version Φˆ . Without loss of generality, the following assumption is given for neural network Φ and its reducedsize version Φˆ .

Assumption 1: The following assumptions hold for neural network Φ and its reduced-size version Φˆ :

1) The number of inputs of two neural networks are the same, $\begin{array} { r } { \mathrm { i } . \mathrm { e } . , n _ { 0 } = \hat { n } _ { 0 } ; } \end{array}$   
2) The number of outputs of two neural networks are the same, i.e., $n _ { L } = n _ { \hat { L } }$ ;   
3) The number of hidden layers of neural network Φ is greater than or equal to the number of hidden layers of neural network Φˆ , i.e., $L \geq \hat { L }$ .

To characterize the output difference between Φ and $\hat { \Phi } .$ , we define the following metric for model reduction precision.

Definition ${ \it 6 : }$ Consider neural network Φ and its reducedsize version $\hat { \Phi }$ with one same input set U and their corresponding output sets $\mathcal { V }$ and $\hat { \mathcal { Y } } ,$ , we define the distance between the outputs of Φ and Φˆ with respect to the input set U by

$$\rho (\Phi , \hat {\Phi}, \mathcal {U}) = \sup _ {\boldsymbol {\eta} _ {0} = \hat {\boldsymbol {\eta}} _ {0}, \boldsymbol {\eta} _ {0}, \hat {\boldsymbol {\eta}} _ {0} \in \mathcal {U}} \left\| \Phi (\boldsymbol {\eta} _ {0}) - \hat {\Phi} (\hat {\boldsymbol {\eta}} _ {0}) \right\| \tag {11}$$

where $\rho ( \Phi , { \hat { \Phi } } , \mathcal { U } )$ is called model reduction precision.
