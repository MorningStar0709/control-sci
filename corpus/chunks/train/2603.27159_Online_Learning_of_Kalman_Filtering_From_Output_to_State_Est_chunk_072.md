# C.4 Proof of Lemma 4.4

We first decompose $X _ { h } ( N ^ { \star } ) , \ldots , X _ { T - 1 } ( N ^ { \star } )$ into martingale difference (sub)sequences. Defining time indices $t _ { i , j } = h + i + ( j - 1 ) h$ for all $i \in \{ 0 , \ldots , h - 1 \}$ and all $j \in [ T _ { i } ]$ , where $T _ { i } = \operatorname* { m a x } \{ j : t _ { i , j } \leq T - 1 \}$ , we may write $\begin{array} { r } { \sum _ { t = h } ^ { T - 1 } X _ { t } = \sum _ { i = 1 } ^ { h } \sum _ { i = 1 } ^ { T _ { i } } X _ { t _ { i , j } } } \end{array}$ . Since $\mathbb { E } [ X _ { t } | \mathcal { F } _ { t - h } ] = 0$ as shown by Lemma 4.3 for all $t \geq h$ and $t _ { i , j } = t _ { i , j - 1 } + h$ , we have $\mathbb { E } [ X _ { t _ { i , j } } | \mathcal { F } _ { t _ { i , j } - 1 } ] = 0$ . Considering any $i \in \{ 0 , \ldots , h - 1 \}$ and the corresponding filtration $\{ \mathcal { F } _ { t _ { i , j } } \} _ { j \geq 1 }$ , we see that $\{ X _ { t _ { i , j } } \} _ { j \geq 1 }$ forms a martingale difference sequence with respect to $\{ \mathcal { F } _ { t _ { i , j } } \} _ { j \geq 1 }$ . In addition, under the event $\mathcal { E } _ { w } \cap \mathcal { E } _ { v }$ , we have that for any $t \geq h$ ,

$$| X _ {t} (N ^ {\star}) | \leq \| \nabla (f _ {t} - f _ {t; h}) (N _ {t - h}) \| \| N _ {t - h} - N ^ {\star} \| + | f _ {t} (N _ {t - h}) - f _ {t} (N ^ {\star}) | + | f _ {t; h} (N ^ {\star}) - f _ {t; h} (N _ {t - h}) |\stackrel {(a)} {\leq} 2 l _ {y} \left\| N _ {t - h} - N ^ {\star} \right\| + 2 l _ {y} \left\| N _ {t - h} - N ^ {\star} \right\|,$$

where (a) follows from the fact that both $\| \nabla f _ { t } ( N ) \|$ and $\| \nabla f _ { t ; h } ( N ) \|$ are upper bounded by $l _ { y }$ for all $N \in { \cal K } _ { N }$ and the fact that both $f _ { t } ( \cdot )$ and $f _ { t ; h } ( \cdot )$ are ly-Lipschitz continuous, as shown by Lemma 3.4.

Now, for any $i \in \{ 0 , \ldots , h - 1 \}$ , we can apply the Azuma inequality (Lemma E.3) and obtain

$$\mathbb {P} (\sum_ {j = 1} ^ {T _ {i}} X _ {t _ {i, j}} (N ^ {\star}) \geq \epsilon) \leq \exp \Big (- \frac {\epsilon^ {2}}{3 2 l _ {y} ^ {2} \sum_ {j = 1} ^ {T _ {i}} \left\| N _ {t _ {i , j} - h} - N ^ {\star} \right\| ^ {2}} \Big). \tag {C.8}$$
