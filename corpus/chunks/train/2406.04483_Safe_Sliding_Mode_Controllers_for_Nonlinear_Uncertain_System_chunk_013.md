# A. Dynamics of Augmented State

As mentioned above, the dynamics of z is selected to retain the system stability. In the sliding mode control, the asymptotic stability of the origin is achieved by constructing a stable manifold and reaching this manifold in a finite time. Thus, the added state z must preserve these properties. Let us consider the following dynamics for the augmented state z:

$$\dot {z} = - 2 \frac {\lambda \sqrt {| z |} + \boldsymbol {s} ^ {\mathrm{T}} \hat {\boldsymbol {G}} \boldsymbol {E} \boldsymbol {u} _ {\mathrm{s}} + \| \boldsymbol {s} \| _ {\infty} \| \boldsymbol {E} \| _ {\infty} \varrho_ {2} \| \boldsymbol {u} _ {\mathrm{s}} \| _ {\infty}}{c _ {z}} \operatorname{sign} (z), \tag {16}$$

where $c _ { z }$ and λ are two strictly positive constants. We can see that $( \pmb { s } = \mathbf { 0 } , z = 0 )$ is the equilibrium point of (16). Consider the composite Lyapunov function candidate $V ( \pmb { \mathscr { s } } , z ) = ( 1 / 2 ) \pmb { \mathscr { s } } ^ { \mathrm { T } } \pmb { \mathscr { s } } + V _ { z } ( z )$ where $s ~ = ~ \zeta ~ - ~ \phi ( \eta )$ and $V _ { z } ( z ) = ( c _ { z } / 2 ) | z |$ . When the system (1) is under the control law $\pmb { u } = \pmb { u } _ { \mathrm { s m c } } + \pmb { u } _ { \mathrm { s } }$ in which ${ \pmb u } _ { \mathrm { s m c } }$ is computed by (15), the derivative of $V ( s , z )$ satisfies the inequality

$$\dot {V} (\boldsymbol {s}, z) \leq - g _ {0} \beta_ {0} \| \boldsymbol {s} \| _ {1} + \boldsymbol {s} ^ {\mathrm{T}} \boldsymbol {G} (\boldsymbol {x}) \boldsymbol {E} (\boldsymbol {x}) \boldsymbol {u} _ {\mathrm{s}} + \frac {c _ {z} z \dot {z}}{2 | z |}. \tag {17}$$

By substituting (16) in (17) and thanks to Assumption 2, we have

$$
\begin{array}{l} \dot {V} (\boldsymbol {s}, z) \leq - g _ {0} \beta_ {0} \| \boldsymbol {s} \| _ {1} + \boldsymbol {s} ^ {\mathrm{T}} \boldsymbol {G} (\boldsymbol {x}) \boldsymbol {E} (\boldsymbol {x}) \boldsymbol {u} _ {\mathrm{s}} - \lambda \sqrt {| z |} \\ - \boldsymbol {s} ^ {\mathrm{T}} \hat {\boldsymbol {G}} \boldsymbol {E} \boldsymbol {u} _ {\mathrm{s}} - \| \boldsymbol {s} \| _ {\infty} \| \boldsymbol {E} \| _ {\infty} \varrho_ {2} \| \boldsymbol {u} _ {\mathrm{s}} \| _ {\infty} \tag {18} \\ \leq - g _ {0} \beta_ {0} \| \boldsymbol {s} \| _ {1} - \lambda \sqrt {| z |}. \\ \end{array}
$$
