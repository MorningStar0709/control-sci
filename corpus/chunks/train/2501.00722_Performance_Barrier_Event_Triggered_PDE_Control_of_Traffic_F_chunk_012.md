Given a topological set S, and an interval $I \subseteq \mathbb { R }$ , the set $ { \mathcal { C } } ^ { 0 } ( I ; S )$ is the set of continuous functions $g : I  S$ . Variables and functions related to regular ETCs are denoted with a superscript r while those of performance-barrier ETCs are denoted with $p .$
