# 3.3.4 Simulation Results

This section provides simulation results using 4 identical UR5 robotic manipulators in the realistic dynamic environment V-REP [180]. The 4 agents are rigidly grasping an object of 40 kg in an initial configuration as shown in Fig. 3.15. In order to verify the theoretical findings of the previous sections, we apply the controller (3.62) to achieve tracking of a desired trajectory by the object’s center of mass. We simulate the closed loop system for two cases of $G ^ { * }$ , namely the proposed one $G _ { 1 } ^ { * } = M G ^ { \top } ( G M G ^ { \top } ) ^ { - 1 }$ as well as the more standard choice $G _ { 2 } ^ { * } = G ^ { \top } ( G G ^ { \top } ) ^ { - 1 }$ .
