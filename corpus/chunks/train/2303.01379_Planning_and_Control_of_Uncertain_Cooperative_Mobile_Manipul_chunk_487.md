# Control Design

We first recap the dynamics linear parameterization with respect to the aforementioned unknown parameters, denoted by $\theta \in \mathbb { R } ^ { l } , \ l \in \mathbb { N }$ . More specifically, similarly to Chapter 2, it holds that

$$B (a) d + C (a, b) c + g (a) = Y (a, b, c, d) \theta , \tag {6.4}$$

a $\in \mathbb { T } , b , c , d \in \mathbb { R } ^ { 3 n }$ , where $Y ( \cdot )$ is a matrix independent of θ. Let $q _ { \mathrm { d } } : =$ $[ q _ { \mathrm { d } , t r } ^ { \top } , q _ { \mathrm { d } , r } ^ { \top } ] ^ { \top } : [ 0 , t _ { f } ]  \mathbb { T }$ be a reference trajectory, with $q _ { \mathrm { d } , t r } \in \mathbb { R } ^ { n _ { t r } }$ and $q _ { \mathrm { d } , r } \in [ 0 , 2 \pi ) ^ { n _ { r } }$ being its translational and rotational parts, respectively. Such a trajectory will be the output of the sampling-based motion planning algorithm that will be developed in the next section. We wish to design the control input $\tau$ of (6.3) such that $q ( t )$ converges close to $q _ { \mathrm { d } } ( t )$ , despite the uncertainty in $\theta .$ We show how such a design can be used in the motion planning of a robotic system with dynamical uncertainties by developing a suitable variant of a standard adaptive control scheme.
