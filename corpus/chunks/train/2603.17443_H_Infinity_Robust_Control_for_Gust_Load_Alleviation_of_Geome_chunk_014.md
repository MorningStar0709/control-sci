# 3.4 Role of the Weighting Parameter $K _ { c }$

The weighting parameter $K _ { c }$ penalises control effort: larger values of $K _ { c }$ produce smaller flap deflections at the expense of reduced gust alleviation. The value $K _ { c } ~ = ~ 1$ was used for all results presented in this paper, following Tantaroudas and Da Ronch [2017].
