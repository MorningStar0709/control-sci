# C. Variational Approximation: $\gamma _ { k } = 1$

This subsection studies the case where $z _ { k }$ is transmitted by the sensor at step $k ,$ i.e., $\gamma _ { k } = 1 \quad$ . The following unknown parameters will be jointly estimated:

$$\Lambda_ {1} \triangleq \{x _ {k}, P _ {k | k - 1}, R _ {k}, \lambda_ {k}, \mu_ {k} \}. \tag {41}$$

The joint pdf $p ( \Lambda _ { 1 } , z _ { k } , \gamma _ { k } )$ is factorized as

$$p \left(\Lambda_ {1}, z _ {k}, \gamma_ {k} \mid \mathcal {I} _ {k}\right) = p \left(\gamma_ {k} \mid z _ {k}, x _ {k}, Y _ {k}, \mathcal {I} _ {k - 1}\right) p \left(x _ {k} \mid P _ {k \mid k - 1}, \mathcal {I} _ {k - 1}\right)\times p (z _ {k} | x _ {k}, R _ {k}, \mathcal {I} _ {k - 1}) p (P _ {k | k - 1} | \lambda_ {k})\times p (\lambda_ {k} | \mu_ {k}) p (\mu_ {k}) p (R _ {k}),$$

where $\begin{array} { r } { p ( \gamma _ { k } = 1 | z _ { k } , x _ { k } , Y _ { k } , \mathcal { T } _ { k - 1 } ) = 1 - \exp ( - \frac { 1 } { 2 } e _ { k } ^ { T } Y _ { k } e _ { k } ) . } \end{array}$

The techniques are similar to that in the above subsection, and the results about $P _ { k | k - 1 } , \lambda _ { k } , \mu _ { k }$ are the same as that for the case of $\gamma _ { k } = 0$ situation. Here, the results about $x _ { k }$ and $R _ { k }$ are given.
