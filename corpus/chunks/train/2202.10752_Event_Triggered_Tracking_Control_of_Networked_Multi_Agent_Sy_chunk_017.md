At the arrival time $r _ { i } ^ { i } , j \in  { \mathbb { N } } _ { + }$ , whether $\hat { z } _ { i }$ is updated via the latest information depends on the local ETM at $t _ { j } ^ { i }$ . Here, we assume that the event-triggered condition for the i-th network is given by $\Gamma _ { i } \geq 0$ , where the function $\Gamma _ { i } : \mathbb { R } _ { \geq 0 }  \mathbb { R }$ will be designed explicitly in Section V-B. $\Gamma _ { i } \geq 0$ implies that the sampled measurement needs to be transmitted, and $\hat { z } _ { i }$ is updated with the latest measurement. That is, $\hat { z } _ { i }$ is updated by

$$
\hat {z} _ {i} (r _ {j} ^ {i +}) = \left\{ \begin{array}{l l} z _ {i} (t _ {j} ^ {i}) + \mathbf {h} _ {z} ^ {i} (\kappa_ {i} (t _ {j} ^ {i}), \mathbf {e} _ {i} (t _ {j} ^ {i})), & \Gamma_ {i} (t _ {j} ^ {i}) \geq 0, \\ \hat {z} _ {i} (r _ {j} ^ {i}), & \Gamma_ {i} (t _ {j} ^ {i}) <   0, \end{array} \right. \tag {13}
$$

where $\kappa _ { i } : \mathbb { R } _ { \geq 0 }  \mathbb { N }$ is a counter to record the number of the successful transmission events. That is, $\kappa _ { i } ( t _ { j } ^ { i } { } ^ { + } ) = \kappa _ { i } ( t _ { j } ^ { i } ) + 1$ if $\Gamma _ { i } ( t _ { i } ^ { i } ) \geq 0$ , and $\kappa _ { i } ( t _ { j } ^ { i } { } ^ { + } ) = \kappa _ { i } ( t _ { j } ^ { i } )$ otherwise. $\mathbf { h } _ { z } ^ { i } \in \mathbb { R } ^ { n _ { z } }$ is the update function and depends on the protocol. Let $\mathbf { h } _ { z } ^ { i } : = \mathbf { \partial }$ $( \mathbf { h } _ { \mathrm { p } } ^ { i } , \bar { \mathbf { h } } _ { \mathrm { r } } ^ { i } , \mathbf { h } _ { \mathrm { c } } ^ { i } , \mathbf { h } _ { \mathrm { f } } ^ { i } )$ , and then we can rewrite (13) as

$$
\begin{array}{l} \hat {z} _ {i} (r _ {j} ^ {i ^ {+}}) = (1 - \Upsilon (\Gamma_ {i} (t _ {j} ^ {i}))) \hat {z} _ {i} (r _ {j} ^ {i}) \\ + \Upsilon (\Gamma_ {i} (t _ {j} ^ {i})) [ z _ {i} (t _ {j} ^ {i}) + \mathbf {h} _ {z} ^ {i} (\kappa_ {i} (t _ {j} ^ {i}), \mathbf {e} _ {i} (t _ {j} ^ {i})) ], \tag {14} \\ \end{array}
$$

where $\Upsilon : \mathbb { R }  \{ 0 , 1 \}$ is defined as $\Upsilon ( \Gamma _ { i } ) = 1 \mathrm { i f } \Gamma _ { i } \geq 0$ and $\Upsilon ( \Gamma _ { i } ) = 0$ otherwise. From (14), the error $\mathbf { e } _ { i }$ is updated by

$$
\begin{array}{l} \mathbf {e} _ {i} (r _ {j} ^ {i ^ {+}}) = \hat {z} _ {i} (r _ {j} ^ {i ^ {+}}) - z _ {i} (r _ {j} ^ {i ^ {+}}) \\ = \mathbf {e} _ {i} (r _ {j} ^ {i}) + \Upsilon (\Gamma_ {i} (t _ {j} ^ {i})) [ \mathbf {h} _ {z} ^ {i} (\kappa_ {i} (t _ {j} ^ {i}), \mathbf {e} _ {i} (t _ {j} ^ {i})) - \mathbf {e} _ {i} (t _ {j} ^ {i}) ]. \\ \end{array}
$$
