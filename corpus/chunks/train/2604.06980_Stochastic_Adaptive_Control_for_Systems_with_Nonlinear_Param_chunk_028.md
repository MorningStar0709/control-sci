Define the system parameter as $\theta ^ { * } = [ \alpha ^ { * } I _ { n } + \tilde { A } ^ { * } ] ^ { \top }$ . Given $c _ { L }$ and a parameter $\theta \in \mathbb { R } ^ { 2 n \times n }$ , we can employ the pinning control law

$$u _ {t} = \kappa (\theta) x _ {L}, \tag {34}$$

where $\kappa ( \theta ) > 0$ is the control gain which depends on the graph topology $\tilde { A } ^ { * }$ and reinforcement factor $\alpha ^ { * }$ . Under this pinning control design, the number of nodes whose opinions eventually agree in sign with the leader’s opinion is maximized under certain conditions (see (Ancona et al., 2023, Section IV-A) for details on the construction of $\kappa ( \theta )$ and the corresponding results). Here, “sign” refers to the binary-opinion setting.

When the graph topology and reinforcement factor, i.e., the parameter $\theta ^ { * }$ , is unknown, the following numerical example illustrates that our proposed method can simultaneously learn $\theta ^ { * }$ and synthesize an adaptive control $\hat { u } _ { t } .$ , such that the resulting closed-loop opinion trajectory converges to that generated by the oracle control $\boldsymbol { u } _ { t } ^ { * }$ . Here, $\boldsymbol { u } _ { t } ^ { * }$ is the controller by setting $\theta = \theta ^ { * }$ in (34).
