The user must set the parameters Q\_in, P\_atm, and P0 (the initial tank pressure for the integrator block) in the MATLAB workspace before running the Simulink model in Fig. 6.18. One way to do this is to write a MATLAB program (or script) called an M-file, which is simply a collection of the required single-line MAT-LAB commands. MATLAB M-file 6.1 (called run\_tank\_NL.m) sets the required parameters and executes the Simulink model tank\_NL.mdl (Fig. 6.18) using the sim command. This M-file also plots the tank pressure P.
