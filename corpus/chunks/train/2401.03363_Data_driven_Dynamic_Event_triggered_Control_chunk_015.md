# V. EXTENSIONS

In this section, we aim to extend the main results developed in Section III to the scenario where state information is quantized by various quantizers $q _ { * } : \mathbb { R } \to \mathbb { R }$ , where $* = u$ or $* \ = \ l$ represents the uniform quantizer or logarithmic quantizer, respectively.

Consider the same $q _ { u }$ and $q _ { l }$ as in [25], which are respectively detailed as follows:

$$q _ {u} (x _ {i} (t)) = \theta \left\lfloor \frac {x _ {i} (t)}{\theta} \right], \tag {18}q _ {l} (x _ {i} (t)) = \mathrm{sign} (x _ {i} (t)) e ^ {q _ {u} [ \ln | x _ {i} (t) | ]}, \tag {19}$$

where the constant $\theta > 0$ represents the quantizer parameter, and $x _ { i } ( t ) \ \in \ \mathbb { R }$ represents the ith component of x(t). Let $q _ { * } ( x ( t ) )$ be the column stack vector of $q _ { * } ( x _ { i } ( t ) ) , i = 1 , . . . , n$ . Denoted by $\epsilon _ { * } ( t ) = q _ { * } ( x ( t ) ) - x ( t )$ the quantization error. It holds that $\begin{array} { r } { \| \epsilon _ { u } ( t ) \| \le \frac { \sqrt { n } \theta } { 2 } } \end{array}$ and $\| \epsilon _ { l } ( t ) \| \le ( e ^ { \frac { \theta } { 2 } } - 1 ) \| x ( t ) \|$ .

Under uniform or logarithmic quantization effect, the dynamic ETM developed in (4)–(7) is modified as

$$t _ {k + 1} = \inf \left\{t > t _ {k} \mid f _ {*} (t) \leq 0 \right\}, \tag {20}$$

where the update algorithm of $f _ { * } ( t ) , \ t \ \in \ \left[ t _ { k } , t _ { k + 1 } \right)$ , is proposed as

$$\dot {f} _ {*} (t) = \min \left\{- z _ {*} (t) ^ {T} \Phi_ {*} z _ {*} (t), 0 \right\} - f _ {*} (t), f _ {*} \left(t _ {k} ^ {+}\right) = \bar {f} _ {*}, \tag {21}$$

where the constant $\bar { f } _ { * } > 0 , z _ { * } ( t ) = [ q _ { * } ( x ( t ) ) ^ { T } ~ e _ { * } ( t ) ^ { T } ] ^ { T }$ , with

$$e _ {*} (t) = q _ {*} (x (t _ {k})) - q _ {*} (x (t)), \tag {22}$$

and

$$
\Phi_ {u} = \left[ \begin{array}{c c} - \frac {\alpha}{2} \mathbf {I} & \mathbf {0} \\ \mathbf {0} & \beta \mathbf {I} \end{array} \right], \Phi_ {l} = \left[ \begin{array}{c c} - e ^ {- \theta} \alpha \mathbf {I} & \mathbf {0} \\ \mathbf {0} & \beta \mathbf {I} \end{array} \right], \tag {23}
$$

where the constants $\alpha , \beta > 0 .$

$$u (t) = K q _ {*} (x (t _ {k})), t \in [ t _ {k}, t _ {k + 1}), \tag {24}$$
