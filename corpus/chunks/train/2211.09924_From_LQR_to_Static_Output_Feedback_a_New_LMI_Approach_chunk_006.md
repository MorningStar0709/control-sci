Theorem 2.6: Assume that the linear system (1) is given with $( A , B )$ controllable and $( A , C )$ observable. Assume further that $Q = Q ^ { T } \geq 0 , \dot { R } = \dot { R } ^ { T } > 0$ , are such that the pair $( A , { \sqrt { Q } } )$ is observable. If there is a solution of the LMI

$$
\left[ \begin{array}{c c} Q - P B F C - C ^ {T} F ^ {T} B ^ {T} P + N & P \overline {{B}} \\ \overline {{B}} ^ {T} P & R \end{array} \right] > 0 \tag {14}
$$

where

$$N = P B (F D R ^ {- 1} + R ^ {- 1} D ^ {T} F ^ {T}) B ^ {T} P, \tag {15}\overline {{{B}}} = B (I _ {m} + F D), \tag {16}$$

and P is the unique positive definite solution of the Riccati equation (6), then the control law $u = F y$ is a stabilizing static output feedback for system (1) in the absence of disturbances and the system is quadratically dissipative with supply rate $W ( w ) \ = \ w ^ { T } R w .$ Conversely, if the system (1) is static output feedback stabilizable in the absence of disturbances and quadratically dissipative with supply rate $W ( w ) = w ^ { T } R w$ , there always exist $F , P > 0 , R > 0 , Q$ verifying LMI (14).

Proof: Since $R > 0$ one can use the Schur complement defined in Lemma 2.4 to rewrite LMI (14) equivalently as

$$Q - P B F C - C ^ {T} F ^ {T} B ^ {T} P + N - P \overline {{B}} R ^ {- 1} \overline {{B}} ^ {T} P > 0$$

Using the definition of N and $\overline { B }$ from (15) and (16), respectively, and the Riccati equation (6), this inequality can be rewritten as

$$- A ^ {T} P - P A - P B F C - C ^ {T} F ^ {T} B ^ {T} P - \overline {{{N}}} > 0 \tag {17}$$

where $\overline { { { N } } } \ = \ P B F D R ^ { - 1 } D ^ { T } F ^ { T } B ^ { T } P$ . Using Schur complement inequality (17) is equivalent to

$$
\left[ \begin{array}{c c} (A + B F C) ^ {T} P + P (A + B F C) & P B F D \\ D ^ {T} F ^ {T} B ^ {T} P & - R \end{array} \right] <   0 \tag {18}
$$

We first prove that the LMI (14) or, equivalently, LMI (18) is a sufficient condition. Assume that there is a solution of the LMI (18). Pre-multiplying this inequality by $\boldsymbol { v } = [ x ^ { T } ~ w ^ { T } ]$ and post-multiplying by $v ^ { T }$ yields

$$x ^ {T} P \dot {x} + \dot {x} ^ {T} P x - w ^ {T} R w < 0 \tag {19}$$

where ${ \dot { x } } \ = \ A x + B u$ with $u \ = \ F C x + F D w$ are the closed-loop dynamics. Choosing the storage function $V = x ^ { T } P x$ , the inequality (19) proves dissipativity with supply rate $W ( w ) = w ^ { T } R w$ according to definition 2.1. In the absence of disturbances one can prove asymptotic stability of the closed-loop system using the Lyapunov function $V = x ^ { T } P x$ .
