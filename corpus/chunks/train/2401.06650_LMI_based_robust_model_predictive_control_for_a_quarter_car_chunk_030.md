$$
\begin{array}{l} \tilde {A} = \left[ \begin{array}{c c} A + B _ {u} K _ {p} r & B _ {u} K _ {i} \\ r & 0 \end{array} \right], \quad \tilde {B} _ {u} = \left[ \begin{array}{c} B _ {u} \\ 0 \end{array} \right], \tag {29} \\ \tilde {B} _ {d} = \left[ \begin{array}{c} B _ {d} \\ 0 \end{array} \right], \qquad \qquad \qquad \tilde {B} _ {p} = \left[ \begin{array}{c} B _ {p} \\ 0 \end{array} \right], \\ \end{array}
$$

in which the matrices A, $\begin{array} { r } { B _ { u } , B _ { d } , B _ { p } , } \end{array}$ are defined in (5).

Then the model after discretisation using a zero-order hold method is defined as:

$$x _ {k + 1} = \tilde {A} \tilde {x} _ {k} + \tilde {B} _ {u} u _ {k} + \tilde {B} _ {p} p _ {k} + \tilde {B} _ {d} d _ {k}, \tag {30}$$

Remark 4. Note that the distribution matrices $\tilde { A } , \tilde { B } _ { u } , \tilde { B } _ { p } , \tilde { B } _ { d } ,$ , and so on, in (30) are the discretized version of those in (28) and the same notation is used for simplicity.
