# A. Definition and Proof of Switched System Stability

Definition 1 (Switched System Stability [22]): For the switched system in (1), if there exist a positive constant δ and a function $\beta ( r , t ) : [ 0 , \infty ) \times [ 0 , \infty )  [ \bar { 0 } , \infty )$ , such that for all switching signals $\sigma ,$ the solutions of the switched system with $\lVert \mathbf { \boldsymbol { x } } ( 0 ) \rVert \leq \mathbf { \widetilde { \delta } }$ satisfy the inequality:

$$\| \boldsymbol {x} (t) \| \leq \beta (\| \boldsymbol {x} (0) \|, t), \forall t \geq 0 \tag {15}$$

where $\beta \left( r , t \right)$ is a strictly monotonic increasing continuous function with $\beta \left( 0 , t \right) = { \dot { 0 } }$ for each fixed $t \geq 0 ,$ , and $\beta \left( r , t \right)$ decreases to zero as $t \to \infty$ for each fixed $r \geq 0$ . Then, the switched system is asymptotically stable. If this inequality holds for all switching signals and initial conditions, the switched system is globally asymptotically stable.

Definition 2 (Common Lyapunov Function [22]): For the switched system in (1), given a positive definite continuously differentiable function $\breve { V } ( \pmb { x } ) : \mathbf { \bot } \breve { \mathbb { R } } ^ { N } \  \ \mathbb { R }$ , if there exists a positive definite continuous function $W ( { \pmb x } ) : \mathbb { R } ^ { N }  \mathbb { R }$ such that

$$\frac {\partial V}{\partial \boldsymbol {x} ^ {\mathrm{T}}} \boldsymbol {f} _ {\sigma} (\boldsymbol {x}) \leq - W (\boldsymbol {x}), \forall \sigma \in S \tag {16}$$

then the function $V \left( { \pmb x } \right)$ is called the common Lyapunov function of the switched system.

The matrix convex combinations of subsystems in the switched system are defined as

$$
\left\{ \begin{array}{l} \boldsymbol {A} _ {\boldsymbol {\lambda}} = \sum_ {i = 1} ^ {n} \lambda_ {i} \boldsymbol {A} _ {i}, \quad \boldsymbol {B} _ {\boldsymbol {\lambda}} \boldsymbol {u} _ {\boldsymbol {\lambda}} = \sum_ {i = 1} ^ {n} \lambda_ {i} \boldsymbol {B} _ {i} \boldsymbol {u} _ {i} \\ \boldsymbol {\Lambda} = \{\boldsymbol {\lambda} = [ \lambda_ {1}, \lambda_ {2}, \dots , \lambda_ {n} ] \in \mathbb {R} ^ {n}, \lambda_ {i} \geq 0, \sum_ {i = 1} ^ {n} \lambda_ {i} = 1 \} \end{array} \right. \tag {17}
$$

where $\lambda _ { i }$ can be interpreted as the proportion of subsystem i on the switched system solution trajectory.
