# § 2. Problem statement

We consider a discrete-time switched linear system

$$x (t + 1) = A _ {\nu (t)} x (t) + b _ {\nu (t)} \mu (t), x (0) = x _ {0}, t = 0, 1, \ldots , T - 1, \tag {2.1}$$

where $x ( t ) \in \mathcal { X } \subseteq \mathbb { R } ^ { d }$ is the vector of states at time t, and $\mu ( t ) \in \mathbb { R }$ and $\nu ( t ) \in \mathcal { N } : =$ $\{ 1 , 2 , \ldots , N \}$ are the continuous control and discrete control at time t, respectively.

The pair $( \nu ( t ) , \mu ( t ) )$ is called a hybrid control at time t. Let $\mu _ { T } = \left( \begin{array} { c } { { \mu ( 0 ) } } \\ { { \mu ( 1 ) } } \\ { { \cdots } } \\ { { \mu ( T - 1 ) } } \end{array} \right)$ and

$\nu _ { T } = \left( \begin{array} { c } { { \nu ( 0 ) } } \\ { { \nu ( 1 ) } } \\ { { \cdots } } \\ { { \nu ( T - 1 ) } } \end{array} \right)$ denote the continuous control and discrete control sequences, respectively.

We will call the pair $\left( \nu _ { T } , \mu _ { T } \right)$ a hybrid control sequence.

Let the set of admissible switches between the subsystems,

$$\mathcal {E} (\mathcal {N}) := \{(i, j) \mid i \text { to } j \text { is allowed }, i, j \in \mathcal {N} \} \subseteq \mathcal {N} \times \mathcal {N}$$

and the set of admissible continuous control inputs, $\mathcal { U } \subseteq \mathbb { R }$ , be given. We will let $\mathcal { N } _ { s } \subseteq \mathcal { N }$ denote the set of all $i \in \mathcal { N }$ such that $( i , i ) \in \mathcal { E } ( \mathcal { N } )$ .

Definition 1. The hybrid control sequence, $\left( \nu _ { T } , \mu _ { T } \right)$ , is admissible, if the following holds:

$$(\nu (t), \nu (t + 1)) \in \mathcal {E} (\mathcal {N}), t = 0, 1, \ldots , T - 2, \tag {2.2}$$

and

$$\mu (t) \in \mathcal {U}, t = 0, 1, \ldots , T - 1. \tag {2.3}$$

Let H denote the set of all admissible hybrid control sequences.

In the sequel we will assume that the subsystems dynamics, $\left( A _ { i } , b _ { i } \right) \in \mathbb { R } ^ { d \times d } \times \mathbb { R } ^ { d } , i \in \mathcal { N } .$ , the set of admissible switches between the subsystems, $\mathcal { E } ( \mathcal { N } )$ , the set of admissible control inputs, U, and the time horizon, $T \in \mathbb { N } .$ , are pre-specified.

Given $\boldsymbol { \xi } \in \mathbb { R } ^ { d }$ , we define

$$\mathcal {C} _ {\xi , T} := \left\{\left(\nu_ {T}, \mu_ {T}\right) \in \mathcal {H} \mid x (0) = \xi , \right.x (t + 1) = A _ {\nu (t)} x (t) + b _ {\nu (t)} \mu (t), t = 0, 1, \ldots , T - 1,\left. x (T) = 0 _ {d} \right\}.$$
