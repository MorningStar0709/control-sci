# A. Fixed- and finite-time stability

To review some findings regarding fixed- and finite-time stability, Consider the following system:

$$\dot {x} (t) = f (x (t)), \quad x (t) \in \mathbb {R} ^ {n}, \quad x (0) = x _ {0}. \tag {1}$$

with the continuous function $f .$

Definition 1: From [3]: a system denoted as (1) is considered globally finite-time stable if it satisfies Lyapunov stability and for any initial condition vector $x _ { 0 }$ in $\mathbb { R } ^ { n }$ , there exists a settling time function $T ( x _ { 0 } ) \geq 0$ dependent on the initial conditions. Specifically, for any solution x(·) of system (1) with the initial condition $x ( 0 ) = x _ { 0 }$ , it holds that as t approaches $T ( x _ { 0 } )$ , the norm of $x ( t )$ converges to zero, i.e., lim $_ { t  T ( x _ { 0 } ) } | x ( t ) | ~ = ~ 0$ . In other words, the norm of $x ( t )$

remains identically zero for all $t \geq T ( x _ { 0 } )$ . The function $T ( x _ { 0 } )$ is referred to as the settling time.

Definition 2: (see [6]): System (1) is considered globally FxT stable if it satisfies the following conditions: 1) It is globally finite-time stable; 2) The settling-time function $T ( x _ { 0 } )$ is upper-bounded by a constant $T _ { \mathrm { m a x } } > 0$ , meaning that for all $x _ { 0 } \ \in \ \mathbb { R } ^ { n } , \ T ( x _ { 0 } ) \ \leq \ T _ { \operatorname* { m a x } }$ , and $T _ { \mathrm { m a x } }$ remains constant regardless of the initial conditions.

The following lemmas are necessary for the subsequent analysis.

Lemma 1: (see [6]): If there exists a continuously differentiable positive definite radially unbounded function $V$ : $\mathbb { R } ^ { n } \to \mathbb { R } ^ { \dagger }$ such that

$$\dot {V} (x) \leq - c _ {1} V (x) ^ {h _ {1}} - c _ {2} V (x) ^ {h _ {2}}$$

where V is called the Lyapuov function and $x \in \mathbb { R } ^ { n } , c _ { i } > 0 .$ , and $0 < h _ { 1 } < 1 < h _ { 2 }$ , then system (1) is globally FxT stable with a settling time bounded by

$$T (x _ {0}) \leq \frac {1}{c _ {1} (1 - h _ {1})} + \frac {1}{c _ {2} (h _ {2} - 1)}. \tag {2}$$

Lemma 2: [24] Consider the following system:

$$\dot {x} (t) = - \alpha \frac {\sqrt {\pi}}{2} \exp (x ^ {2}) \operatorname{sign} (x) + d (t), \quad x (0) = x _ {0}, \tag {3}$$

where $\begin{array} { r } { \alpha > \frac { 2 } { \sqrt { \pi } } \bar { d } , x \in \mathbb { R } . } \end{array}$ , and $d \in \mathbb { R }$ . Then, system (3) is globally fixed-time stable with a settling time $T ( x _ { 0 } ) \leq T _ { \mathrm { m a x } }$ where $\begin{array} { r } { \dot { T } _ { \mathrm { m a x } } = \frac { 1 } { \alpha - \frac { 2 } { \sqrt { \pi } } \bar { d } } . } \end{array}$
