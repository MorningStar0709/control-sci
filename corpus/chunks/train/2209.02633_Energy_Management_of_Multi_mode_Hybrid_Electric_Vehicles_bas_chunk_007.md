# 1) Engine model

The engine model is used to determine the fuel consumption rate $\dot { m } _ { f } \left( \mathsf { g } / \mathsf { s } \right)$ based on a 2D look-up table, which is a function of the engine speed, $n _ { e n g } ( t )$ , and the engine torque, $T _ { e n g } ( t )$ , by

$$\dot {m} _ {f} (t) = f (n _ {e n g} (t), T _ {e n g} (t)) \tag {7}$$
