For $\tau > 0$ and $\eta > 0$ , we define the class of G-admissible signals with dwell time $\tau$ in stable and flee time η in unstable subsystems as follows: $\bar { S ^ { \mathcal { G } } } [ \tau , \eta ] = \bar { S ^ { \mathcal { G } } } \cap \bar { S } [ \tau , \eta ]$ . Hence, when all subsystems are stable, we define $\begin{array} { r } { S _ { s } ^ { \mathcal { G } } [ \tau ] = S ^ { \mathcal { G } } \cap S _ { s } [ \tau ] } \end{array}$ . Similarly, when all subsystems are unstable, we define $\begin{array} { r } { S _ { u } ^ { \mathcal { G } } [ \eta ] = S ^ { \mathcal { G } } \cap S _ { u } [ \eta ] } \end{array}$ .

Consider the subgraphs $\mathcal { G } _ { s }$ and $\mathcal { G } _ { u }$ of G for which $\mathcal { V } ( \dot { \mathcal { G } } _ { s } ) = \mathcal { V } ( \mathcal { G } ) = \mathcal { V } ( \mathcal { G } _ { u } )$ and

$$\mathcal {E} (\mathcal {G} _ {s}) = \left\{\left(A _ {p}, A _ {q}\right) \in \mathcal {E} (\mathcal {G}): p \in \mathfrak {s} \right\}, \quad \mathcal {E} (\mathcal {G} _ {u}) = \left\{\left(A _ {p}, A _ {q}\right) \in \mathcal {E} (\mathcal {G}): p \in \mathfrak {u} \right\}.$$

That is, ${ \mathcal { E } } ( { \mathcal { G } } _ { s } )$ consists of those edges of the graph $\mathcal { G }$ which originate from a stable subsystem and $\mathcal { E } ( \mathcal { G } _ { u } )$ consists of those edges of the graph G which originate from an unstable subsystem.

Henceforth we will use the shorthand $( p , q )$ for the directed edge $( A _ { p } , A _ { q } ) \in \mathcal { E } ( \mathcal { G } )$ .
