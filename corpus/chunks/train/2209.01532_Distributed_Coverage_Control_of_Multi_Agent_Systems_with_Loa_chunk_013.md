# Lemma 3.6.

$$\lim _ {t \to \infty} \varphi_ {i} (t) = \varphi_ {i} ^ {*}, \quad i \in I _ {N}$$

with the constant $\boldsymbol { \varphi } _ { i } ^ { * }$ .

Proof. It follows from Lemma 3.1 and Equation (2) that

$$\lim _ {t \rightarrow \infty} \dot {\varphi} _ {i} (t) = \lim _ {t \rightarrow \infty} \kappa_ {\varphi} [ m _ {i} (t) - m _ {i - 1} (t) ] = 0.$$

In light of Lemma 3.3, there exist positive constants c1 and $c _ { 2 }$ such that

$$\dot {\varphi} _ {i} (t) \leq | \dot {\varphi} _ {i} (t) | = \kappa_ {\varphi} | m _ {i} (t) - m _ {i - 1} (t) | \leq \kappa_ {\varphi} c _ {1} e ^ {- c _ {2} t},$$

which leads to

$$\lim _ {t \rightarrow \infty} \varphi_ {i} (t) - \varphi_ {i} (0) = \int_ {0} ^ {+ \infty} \dot {\varphi} _ {i} (t) d t \leq \int_ {0} ^ {+ \infty} | \dot {\varphi} _ {i} (t) | d t \leq \kappa_ {\varphi} c _ {1} \int_ {0} ^ {+ \infty} e ^ {- c _ {2} t} d t = \frac {\kappa_ {\varphi} c _ {1}}{c _ {2}} < + \infty .$$

According to Theorem 10.33 in [20], ϕ˙ (t) is improper Riemann-integrable on $[ 0 , + \infty )$ , and thus the improper integral $\int _ { 0 } ^ { + \infty } \dot { \varphi } _ { i } ( t ) d t$ exists, which implies the existence of $\scriptstyle \operatorname* { l i m } _ { t \to \infty } \varphi _ { i } ( t )$ because of the following equality

$$\lim _ {t \rightarrow \infty} \varphi_ {i} (t) = \varphi_ {i} (0) + \int_ {0} ^ {+ \infty} \dot {\varphi} _ {i} (t) d t.$$

Therefore, $\varphi _ { i } ( t ) , i \in I _ { N }$ converges to the constant value $\boldsymbol { \varphi } _ { i } ^ { * }$ as time t goes to the positive infinity.
