where $u _ { t } ^ { m } \in \mathbb { R } ^ { u }$ is the actuation input applied to the mth agent at time step t. The states and inputs are constrained as $x _ { t } \in \mathcal { X }$ and $u _ { t } \in \mathcal { U } .$ A subset of each agent’s state $c \in$ $\mathbb { R } ^ { L } , \ L \leq$ n describes its L capacities, which may include any limited resource, such as battery charge or travel duration:

$$x _ {t} ^ {m} = \left(c _ {t} ^ {m}, \chi_ {t} ^ {m}\right) \tag {2}c _ {t} ^ {m} = (c _ {t, 1} ^ {m}, c _ {t, 2} ^ {m}, \dots , c _ {t, L} ^ {m}), \tag {3}$$

where $c _ { t , l } ^ { m }$ is the value of the lth capacity state of agent m at time step t, and $\chi _ { t } ^ { m } \in \mathbb { R } ^ { n - L }$ are the agent’s non-capacity states. The capacity and non-capacity states evolve as

$$c _ {t + 1} ^ {m} = c _ {t} ^ {m} + g _ {c} (\chi_ {t} ^ {m}, u _ {t} ^ {m}) \tag {4}\chi_ {t + 1} ^ {m} = g _ {\chi} (\chi_ {t} ^ {m}, u _ {t} ^ {m}), \tag {5}$$

where $g _ { c } ( \cdot , \cdot ) \geq 0$ (component-wise positive), and are subject to constraints

$$c _ {t, l} ^ {m} \in \mathcal {C} _ {l}, l \in [ 1, L ], \chi_ {t} ^ {m} \in X, \tag {6}$$

where $\mathcal { C } _ { l } = [ 0 , C _ { l } ]$ and capacity state constraints are jointly denoted as $c _ { t } ^ { m } \in \mathcal { C }$ . The state constraints $x _ { t } ^ { m } \in \mathcal { X }$ are satisfied if and only if both $c _ { t } ^ { m } \in \mathcal { C }$ and $\chi _ { t } ^ { m } \in X$ .

We introduce the set ${ \mathcal { N } } _ { j }$ to describe that agent m is at node $v _ { j }$ at time step t if $x _ { t } ^ { m } \in \mathcal N _ { j }$ :

$$\mathcal {N} _ {j} = \big \{(c, \chi) \in \mathbb {R} ^ {n} \mid a (\chi) = j, \quad \chi = \bar {\chi} _ {j} \big \}, \tag {7}$$

where the function $a ( \cdot )$ maps a non-capacity state $\chi$ to a node index $j .$ . For all states in ${ \mathcal { N } } _ { j }$ , the the non-capacity states $\chi$ have fixed value $\bar { \chi } _ { j }$ , though this value can depend on $j .$ .

Our goal is to find a feasible input sequence for each agent (1) that optimizes a function of the agent task assignment; here we maximize the total number of unique nodes visited. We formulate the corresponding optimal control problem as

$$\max _ {v, x, u, T} \sum_ {m \in \mathcal {M}} \sum_ {i, j \in \mathcal {V}} v _ {i, j} ^ {m} \tag {8a}\text { s.t. } \quad b (v, x, u, T) \leq 0 \tag {8b}$$
