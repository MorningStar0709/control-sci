# The Modification Algorithm

1. Define β(t) as follows:

$$\beta^ {T} (t) = \left[ \sigma (t), \sigma (t) ^ {m}, \sigma (t) ^ {m ^ {2}}, \dots , \sigma (t) ^ {m ^ {(m - 1)}} \right] \tag {12.67}$$

where:

$$m = n _ {A} + n _ {B} + d; \quad m ^ {(m - 1)} = l ^ {\prime} \tag {12.68}$$

2. $\sigma ( t )$ takes values in the set D

$$D = \left[ \sigma_ {1}, \sigma_ {2}, \dots , \sigma_ {l} \right]; \quad l = m ^ {m} \tag {12.69}$$

with

$$\sigma_ {i} \in R; \quad \sigma_ {i} \geq \sigma_ {i - 1} + 1; \quad i = 1, 2, \dots , l \tag {12.70}$$

The values of $\sigma ( t )$ will switch from a value to another in D such that an appropriate $\beta ( t )$ can be found. In order to stop the searching procedure, a switching function with a constant hysteresis width $\mu > 0 ( \mu \ll 1 )$ is introduced. Denote:

$$x (t, \sigma) = | \det M [ \hat {\theta} (t) + F (t) ^ {- 1 / 2} (\sigma , \sigma^ {m}, \dots , \sigma^ {l ^ {\prime}}) ^ {T} ] | \tag {12.71}$$

3. The hysteresis switching function defining $\sigma ( t )$ is defined as follows:

$$
\sigma (t) = \left\{ \begin{array}{l l} \sigma (t - 1) & \text { if   } x (t, \sigma_ {j}) <   (1 + \gamma) x [ t, \sigma (t - 1) ] \\ & \text { for   all   } \sigma_ {j} \in D \\ \sigma_ {j} & \text { if   } j \text {   is   the   smallest   integer   such   that } \\ & x (t, \sigma_ {j}) \geq (1 + \gamma) x [ t, \sigma (t - 1) ] \\ & \text { and   } x (t, \sigma_ {j}) \geq x (t, \sigma_ {i}) \quad \forall \sigma_ {i} \in D \end{array} \right. \tag {12.72}
$$

It is shown in Lozano and Zhao (1994) that using this type of algorithm, one has the following result:

Theorem 12.3 Subject to the assumptions that the plant model polynomials $A ( q ^ { - 1 } )$ and $B ( q ^ { - 1 } )$ do not have common factors and their orders as well the delay d are known, the parameter estimation algorithm (given by (12.5) through (12.13)) combined with the modification procedure defined by (12.67) through (12.72) assures:

(i) a lower bound for the absolute value of the determinant of the Sylvester matrix, $i . e .$ ,

$$| \det M [ \bar {\theta} (t) ] | \geq \alpha \delta_ {0}; \quad \alpha > 0 \tag {12.73}$$

where $\delta _ { 0 }$ is a measure of the controllability of the unknown system (related to the condition number)

$$0 < \delta_ {0} \leq | \det M (\theta) | \tag {12.74}$$

(ii) convergence in finite time toward the modified value $\bar { \theta } ( t )$ assuring the property above.

Analysis A full proof of this result is beyond our scope. In what follows, we would like to outline the basic steps and the rationale behind the choice proposed for $\beta ( t )$ . The key ideas are the following:

1. Using the proposed modification it is possible to write:

$$\det M [ \bar {\theta} (t) ] = g ^ {T} (t) v [ \beta (t) ] \tag {12.75}$$

where $g ( t )$ contains combinations of the coefficient of the unmodified parameter vector ${ \hat { \theta } } ( t )$ and $v ( \beta ( t ) )$ contain combinations of $\sigma _ { i }$ at various powers.
