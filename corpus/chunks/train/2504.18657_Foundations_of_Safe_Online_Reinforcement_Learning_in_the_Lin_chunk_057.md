proof. We have four cases depending on the values of $x , y$ . We will prove the results of Lemma 4 and Lemma 6 for each of these cases separately. WLOG assume that $x \leq y$ .

Case 1: $\begin{array} { r } { \frac { D _ { L } } { a - b K } \leq x \leq y \leq \frac { D _ { U } } { a - b K } } \end{array}$ a−bK .

In this case, $C _ { K } ^ { \theta } ( x ) = - K \colon$ x and $C _ { K } ^ { \theta } ( y ) = - K y$ , and therefore the following two equations hold. Case 1 Lemma 6:

$$| a ^ {*} x + b ^ {*} C _ {K} ^ {\theta} (x) - a ^ {*} y - b ^ {*} C _ {K} ^ {\theta} (y) | = | a ^ {*} - b ^ {*} K | d = (| a ^ {*} - b ^ {*} K | + O _ {T} (\epsilon)) d.$$

Case 1 Lemma 4:

$$| C _ {K} ^ {\theta} (x) - C _ {K} ^ {\theta} (y) | = K d \leq \frac {a}{b} \cdot d \leq \frac {\bar {a}}{\underline {{b}}} \cdot d = O _ {T} (d).$$

Case 2: $\begin{array} { r } { \frac { D _ { U } } { a - b K } \leq x \leq y \mathrm { ~ o r ~ } x \leq y \leq \frac { D _ { L } } { a - b K } } \end{array}$ (which is W of Lemma 6).

First, assume the former is true. Then $\begin{array} { r } { C _ { K } ^ { \theta } ( x ) = \frac { D _ { U } - a x } { b } } \end{array}$ and likewise $\begin{array} { r } { C _ { K } ^ { \theta } ( y ) = \frac { D _ { U } - a y } { b } } \end{array}$ Therefore the following equations hold.

Case 2 Lemma 6:

$$
\begin{array}{l} \left| a ^ {*} x + b ^ {*} C _ {K} ^ {\theta} (x) - a ^ {*} y - b ^ {*} C _ {K} ^ {\theta} (y) \right| = d \left| a ^ {*} - \frac {a}{b} b ^ {*} \right| \\ = d \left| \frac {a ^ {*} b - a b ^ {*}}{b} \right| \\ \leq d \frac {\max \left((a + \epsilon) b - a (b - \epsilon) , | (a - \epsilon) b - a (b + \epsilon) |\right)}{b} \\ = d \frac {\epsilon b + \epsilon a}{b} \\ \leq d \frac {\epsilon \bar {b} + \epsilon \bar {a}}{\underline {{b}}} \\ \leq O _ {T} (\epsilon) d. \\ \end{array}
