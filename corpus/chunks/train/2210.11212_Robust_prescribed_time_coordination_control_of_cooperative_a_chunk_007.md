Notations: $\mathbb { R } ^ { k } \left( \mathbb { C } ^ { k } \right) , \ \mathbb { R } ^ { k \times r } \left( \mathbb { C } ^ { k \times r } \right)$ , and $\mathbb { R } \left( \mathbb { R } _ { \geq 0 } \right)$ denote respectively the sets of k-dimensional real (complex) vectors, $k \ \times \ r$ real (complex) matrices, and (nonnegative) real numbers. $I _ { k }$ indicates the $k \times k$ unit matrix, 0 denotes the zero matrix with compatible sizes, $1 _ { k }$ stands for the k-dimensional all-one vector, and sign (·) refers to the sign function. We denote by $\begin{array} { r c l } { \mathcal { I } _ { N } } & { = } & { \{ 1 , 2 , \dots , N \} } \end{array}$ , by $\mathbb { G } _ { N } ~ = ~ \left\{ G = \mathrm { d i a g } \left\{ g _ { 1 } , \dots , g _ { N } \right\} : g _ { k } \in \left\{ \pm 1 \right\} , k \in \mathcal { I } _ { N } \right\}$ , by diag $\{ \boldsymbol { { r } } _ { 1 } , \ldots , \boldsymbol { { r } } _ { N } \}$ a diagonal block matrix whose kth diagonal block is $\varGamma _ { k } , \mathrm { b y } \mid \cdot$ | the absolute value of a scalar or matrix, and by k·k the Euclidian norm of a vector. For any Hermitian matrix $A \in \mathbb { C } ^ { k \times k } , \lambda _ { \operatorname* { m a x } } \left( A \right) ( \lambda _ { \operatorname* { m i n } } \left( A \right) )$ ) represents its largest (smallest) eigenvalue, and $A \succ 0 ( \succeq 0 )$ indicates that A is positive (semi-)definite. Given any matrix $B = [ b _ { i j } ] \in \mathbb { R } ^ { r \times r }$ , the comparison matrix of B is defined by $\mathcal { M } \left( B \right) = \left[ m _ { i j } \right] \in \mathbb { R } ^ { r \times r }$ with $m _ { i i } = | b _ { i i } |$ and $m _ { i j } = - \vert b _ { i j } \vert , i \neq j$ . B is termed a Z-matrix when $b _ { i j } \leq 0 , i \neq j$ , and a singular (nonsingular) M-matrix if further its eigenvalues all possess nonnegative (positive) real parts. It is termed an H-matrix provided its comparison matrix $\mathcal { M } \left( B \right)$ is a singular or nonsingular Mmatrix.
