Now consider a trajectory $\left[ \hat { \pmb u } _ { [ 0 , T - 1 ] } ; \hat { \pmb x } _ { [ 0 , T - 1 ] } \right]$ starting from an initial state ${ \hat { \mathbf { x } } } ( 0 )$ and evolves with the sequence of actions $\hat { \pmb u } _ { [ 0 , T - 1 ] }$ . If $\left[ \begin{array} { c } { { U _ { 0 , T , L - T + 1 } } } \\ { { X _ { 0 , L - T + 1 } } } \end{array} \right]$ X0,L−T +1 is full row rank, namely rank( $\begin{array} { r } { \left[ { \pmb U } _ { 0 , T , L - T + 1 } \right] ) = } \\ { \left[ X _ { 0 , L - T + 1 } \right] } \end{array}$ $n + T m$ , then there exists $\hat { \pmb g } \in \mathbb { R } ^ { L - T + 1 }$ such that $\begin{array} { r } { \Big [ \hat { \pmb { u } } _ { [ 0 , T - 1 ] } \Big ] = \Big [ \pmb { U } _ { 0 , T , L - T + 1 } \Big ] \hat { \pmb { g } } . } \end{array}$ . By (18),

$$
\begin{array}{l} \left[ \begin{array}{l} \hat {\boldsymbol {u}} _ {[ 0, T - 1 ]} \\ \hat {\boldsymbol {x}} _ {[ 0, T - 1 ]} \end{array} \right] = \left[ \begin{array}{c c} \boldsymbol {I} _ {T m} & \mathbb {O} _ {T m \times n} \\ \mathcal {T} _ {[ 0, T - 1 ]} & \mathcal {O} _ {[ 0, T - 1 ]} \end{array} \right] \left[ \begin{array}{c} \boldsymbol {U} _ {0, T, L - T + 1} \\ X _ {0, L - T + 1} \end{array} \right] \hat {\boldsymbol {g}} \tag {20} \\ = \left[ \begin{array}{c} \boldsymbol {U} _ {0, T, L - T + 1} \\ \boldsymbol {X} _ {0, T, L - T + 1} \end{array} \right] \hat {\boldsymbol {g}}, \\ \end{array}
$$

where the second equation follows from the relation in (19). This complete the proof of Lemma (1).
