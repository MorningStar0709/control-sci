# A.2 Proof of Theorem 1

Our proof below adapts the idea of partial calmness [50] into the quadratic optimization problems in Theorem 1. We let $f ( x ) { : = } x ^ { \top }$ M x for simplicity. Since $\left\| \cdot \right\| _ { p }$ is a norm, the constraint $\| D x _ { 2 } \| _ { p } = 0$ is equivalent to $\dot { D } x _ { 2 } = 0$ . It suffices to prove that there exists a finite $\lambda _ { w } ^ { \ast } > 0$ such that for every $\lambda _ { w } > \lambda _ { w } ^ { * }$ , every global minimizer $( x _ { 1 } , x _ { 2 } )$ of (18) must satisfy $D x _ { 2 } = 0$ . Indeed, if this holds, then all minimizers of (18) are feasible for (17). On the set $\{ ( x _ { 1 } , x _ { 2 } ) : D x _ { 2 } = { \dot { 0 } } \}$ , the penalty term vanishes, so both problems have the same objective values and constraints. Therefore, the two problems have the same optimal value and the same set of optimal solutions.

Step 1: Every minimizer $( x _ { 1 } , x _ { 2 } )$ of (18) must satisfy $D x _ { 2 } = 0$ . We claim that there exist constants $\delta > 0$ and ${ \lambda } _ { w } ^ { \ast } > 0$ such that for any $\lambda _ { w } > \lambda _ { w } ^ { * }$ , the inequality

$$f (x _ {1}) + \lambda_ {w} \| D x _ {2} \| _ {p} \geq f (x _ {1} ^ {*}) + (\lambda_ {w} - \lambda_ {w} ^ {*}) \| D x _ {2} \| _ {p} \tag {A.1}$$

holds for all feasible $( x _ { 1 } , x _ { 2 } )$ of (18) satisfying

$$\left\| x _ {1} - x _ {1} ^ {*} \right\| _ {p} \leq \delta , \quad \left\| D x _ {2} \right\| _ {p} \leq \delta , \tag {A.2}$$

where $( x _ { 1 } ^ { * } , x _ { 2 } ^ { * } )$ is the given optimal solution of (17). We note that, the existence of such feasible solution is guaranteed as we can trivially choose $x _ { 1 } = x _ { 1 } ^ { * }$ and $x _ { 2 } = x _ { 2 } ^ { * }$ .

Assuming (A.1) for now, fix any $\lambda _ { w } > \lambda _ { w } ^ { * }$ . Moreover, (A.1) implies that any feasible point in the neighborhood (A.2) has objective value at least $f ( x _ { 1 } ^ { * } )$ , with strict inequality whenever $D x _ { 2 } \neq 0$ . Hence $( x _ { 1 } ^ { * } , x _ { 2 } ^ { * } )$ is a local minimizer of (18). Since (18) is convex, every local minimizer is global, so $( x _ { 1 } ^ { * } , x _ { 2 } ^ { * } )$ is a global minimizer of (18).
