$$
\begin{array}{l} J \left(\mathbf {u} ^ {\prime}, \theta_ {n}\right) = \left\langle \left[ \begin{array}{c c} Q _ {0} + \widetilde {A} ^ {*} \widetilde {Q} _ {P} \widetilde {A} & 0 \\ 0 & V _ {0} \end{array} \right] \theta_ {n}, \theta_ {n} \right\rangle \tag {7} \\ + \langle H \mathbf {u} ^ {\prime}, \mathbf {u} ^ {\prime} \rangle + 2 \langle \mathbf {u} ^ {\prime}, F \theta_ {n} \rangle , \\ \end{array}

Since we want to minimize $J ( \mathbf { u } ^ { \prime } , \theta _ { n } )$ by varying $\mathbf { u } ^ { \prime } ,$ for a fixed, given $\theta _ { n } ~ = ~ \left[ \begin{array} { c } { { x _ { n } } } \\ { { u _ { n - 1 } } } \end{array} \right]$ , the first term in (7) does not influence the location of the minimum, and it can be omitted in the optimization.

The particular case when H is coercive is important, i.e., when $H ^ { * } = H$ and

$$\langle H \mathbf {u} ^ {\prime}, \mathbf {u} ^ {\prime} \rangle \geq \varepsilon \| \mathbf {u} ^ {\prime} \| ^ {2}, \quad \mathbf {u} ^ {\prime} \in U ^ {N}, \tag {9}$$

for some $\varepsilon > 0$ independent of $\mathbf { u } ^ { \prime } .$ This is guaranteed, $\mathrm { e . g . }$ , if $\widetilde { V }$ is coercive (see Lemma 2 below); indeed observe that

$$H - \widetilde {V} := \widetilde {B} ^ {*} \widetilde {Q} _ {P} \widetilde {B} + \widetilde {R} + \widetilde {B} ^ {*} \widetilde {M} + \widetilde {M} ^ {*} \widetilde {B} \geq 0,$$

since $\langle ( H - \widetilde { V } ) \mathbf { u } ^ { \prime } , \mathbf { u } ^ { \prime } \rangle$ equals ${ \cal J } ( { \bf u } ^ { \prime } , 0 )$ in $( 7 )$ , or equivalently in $( 3 )$ , with $V _ { k } = 0$ for all $k ,$ and we assumed $\left\lceil \begin{array} { l l } { Q _ { k } } & { \bar { M } _ { k } } \\ { M _ { k } ^ { * } } & { R _ { k } } \end{array} \right\rceil , P \geq 0 .$ Moreover, H inherits coercivity from $\widetilde { R }$ in case $M _ { k } = 0$ for all $k = 1 , \ldots , N - 1$ .
