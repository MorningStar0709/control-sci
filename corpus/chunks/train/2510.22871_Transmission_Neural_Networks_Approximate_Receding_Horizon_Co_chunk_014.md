# A. MDP Solution for Markovian SIS Dynamics

Consider the stochastic dynamics in (1) with $w _ { i j } ^ { k }$ replaced by the controlled transmission probability $m _ { i j } ^ { k } ( u _ { i } ( k ) )$ , and the cost below

$$J _ {1} = \mathbb {E} \sum_ {k = 0} ^ {T - 1} \mathbf {1} _ {n} ^ {\intercal} (c X (k) + u (k)) \triangleq \mathbb {E} \sum_ {k = 0} ^ {T - 1} l (X (k), u (k))$$

with transition probabilities given by (4) and ${ \bf 1 } _ { n }$ as the ndimensional vector of ones. In this case, the state space is $\{ 0 , 1 \} ^ { n }$ and control action space is $\{ 0 , 1 \} ^ { n }$ . Let $V _ { k }$ : $\{ 0 , 1 \} ^ { n } \to \mathbb { R }$ denote the value function defined by

$$V _ {k} (x) \triangleq \min _ {u _ {k} \dots u _ {T - 1}} \mathbb {E} \sum_ {\tau = k} ^ {T - 1} l (X (\tau), u (\tau)), \quad X (k) = x.$$

By Dynamic Programming (see e.g. [27]), the value function satisfies the Bellman equation

$$V _ {k} (x) = \min _ {u} \left[ l (x, u) + \sum_ {x ^ {\prime} \in \{0, 1 \} ^ {n}} \operatorname * {P r} (x ^ {\prime} | x, u) V _ {k + 1} (x ^ {\prime}) \right]$$

where $V _ { T } = 0 $ , the transition probability following Proposition 1 is specified by

$$
\begin{array}{l} \operatorname * {P r} (X (k + 1) = q | X (k) = x, u (k) = v) \\ = \prod_ {i = 1} ^ {n} \left(q _ {i} \rho_ {i} (k + 1) + (1 - q _ {i}) (1 - \rho_ {i} (k + 1))\right) \tag {17} \\ \end{array}
$$

with $\begin{array} { r } { \rho _ { i } ( k + 1 ) = 1 - \prod _ { j \in N _ { i } ^ { \circ k } } \left( 1 - m _ { i j } ^ { k } ( v _ { i } ) x _ { j } ( k ) \right) } \end{array}$ and

$$m _ {i j} ^ {k} (v _ {i}) = v _ {i} w _ {i j} ^ {k} \beta + (1 - v _ {i}) w _ {i j} ^ {k}, \forall i, j \in [ n ]. \tag {18}$$

The optimal control is then given by the arguments of the minima on the right-hand side of the Bellman equation.
