Given the phase field vector $\phi _ { t }$ and the control input vector inputs $u _ { t }$ at any time t, as a stack of all states $\{ \dot { u } _ { t } ^ { i , j } \} = \{ T _ { t } ^ { i , j } , h _ { t } ^ { i , j } \} ^ { T }$ t respectively, we can simulate $\{ \phi _ { t } ^ { i , j } \}$ and control the phase-field at the next time step from Eq. (6) and (7). We also observe the dynamics to be affine in control for both cases.

The system can, thus, be described in the discretized statespace form as

$$\Phi_ {t + 1} = f (\Phi_ {t}) + g (\Phi_ {t}) U _ {t}, \tag {8}$$

where $\Phi _ { t } ~ = ~ \{ \phi _ { t } ^ { i , j } \}$ is the phase variable on the spatially discretized FD grid at time t, and $U _ { t } = \{ u _ { t } ^ { i , j } \}$ represents the control variables on the same grid at time t.
