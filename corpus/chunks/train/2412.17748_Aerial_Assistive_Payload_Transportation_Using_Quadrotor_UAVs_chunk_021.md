# A. Sliding surface design

Let $\boldsymbol { e } ~ \in ~ \mathbb { R } ^ { 6 }$ be the tracking error of the system which is given as:

$$e = \chi_ {d} - \chi \tag {20}$$

where $\chi = [ x , y , z , \phi , \theta , \psi ] ^ { \top } \mathbf { a n d } ~ \chi _ { d } = [ x _ { d } , y _ { d } , z _ { d } , \phi _ { d } , \theta _ { d } , \psi _ { d } ] ^ { \top }$ are the actual and desired position and orientation of the entire system, respectively. For the system in Equation (16) and the tracking error in Equation (20), the Non-singular Fast Terminal Sliding Mode (NFTSM) surfaces are considered as follows:

$$
\left\{ \begin{array}{l l} S _ {x} & = \dot {e _ {x}} + \xi e _ {x} + \eta s g n (e _ {x}) ^ {a} \\ S _ {y} & = \dot {e _ {y}} + \xi e _ {y} + \eta s g n (e _ {y}) ^ {a} \\ S _ {z} & = \dot {e _ {z}} + \xi e _ {z} + \eta s g n (e _ {z}) ^ {a} \\ S _ {\phi} & = \dot {e _ {\phi}} + \xi e _ {\phi} + \eta s g n (e _ {\phi}) ^ {a} \\ S _ {\theta} & = \dot {e _ {\theta}} + \xi e _ {\theta} + \eta s g n (e _ {\theta}) ^ {a} \\ S _ {\psi} & = \dot {e _ {\psi}} + \xi e _ {\psi} + \eta s g n (e _ {\psi}) ^ {a} \end{array} \right. \tag {21}
$$

where $\xi > 0 , \eta > 0 , a \geq 1 , s g n ( e ) ^ { a } = | e | ^ { a } s g n ( e )$ and its derivative is given as [47] $\begin{array} { r } { \frac { d } { d t } ( s g n ( e ) ^ { a } ) = a | e | ^ { a - 1 } \dot { e } . } \end{array}$ , and sgn(.) is the sign function which is given in Equation (22):

$$
s g n (e) := \left\{ \begin{array}{l l} 1 & \text { if   } e > 0, \\ 0 & \text { if   } e = 0, \\ - 1 & \text { if   } e <   0 \end{array} \right. \tag {22}
$$

For simplicity of analysis and calculation, let $\begin{array} { r l } { S } & { { } = } \end{array}$ $[ S _ { x } , S _ { y } , \bar { S } _ { z } , \bar { S _ { \phi } } , S _ { \theta } , S _ { \psi } ] ^ { \top }$ represent the general NFTSM surface of all system variables above, and it is given as follows:

$$S = \dot {e} + \xi e + \eta | e | ^ {a} s g n (e) \tag {23}$$

Taking the time derivative of the sliding surface in Equation (23) results in:

$$
\begin{array}{l} \dot {S} = \ddot {e} + \xi \dot {e} + \eta a | e | ^ {a - 1} \dot {e} \\ = \ddot {\chi} _ {d} - \ddot {\chi} + \xi \dot {e} + \eta a | e | ^ {a - 1} \dot {e} \tag {24} \\ \end{array}
$$

Substituting Equation (16) into Equation (24) results in:

$$\dot {S} = \ddot {\chi_ {d}} - F - \delta - b u + (\xi + \eta a | e | ^ {a - 1}) \dot {e} \tag {25}$$

Considering uˆ as an equivalent controller (continuous control law) to achieve ${ \dot { S } } = 0 ,$ results in the following:

$$\hat {u} = \frac {1}{b} \left[ \ddot {\chi} _ {d} - F - \delta + (\xi + \eta a | e | ^ {a - 1}) \dot {e} \right] \tag {26}$$

To address dynamic uncertainties and external disturbances, as well as to increase the speed of convergence to the sliding surface, the control law can be formulated by incorporating a discontinuous term into uˆ in Equation (26) as follows:

$$u = \hat {u} + \frac {1}{b} \left[ \lambda_ {1} S + \lambda_ {2} s g n (S) \right] \tag {27}$$

where $\lambda _ { 1 } > 0$ and $\lambda _ { 2 } > 0 .$
