$$
\begin{array}{l} J (\mathbf {u} ^ {\prime}, [ \begin{array}{c} x _ {n} \\ u _ {n - 1} \end{array} ]) := \langle P x _ {N} ^ {\prime}, x _ {N} ^ {\prime} \rangle + \langle V _ {N} u _ {N - 1} ^ {\prime}, u _ {N - 1} ^ {\prime} \rangle \\ + \sum_ {k = 0} ^ {N - 1} \left(\left\langle \left[ \begin{array}{l l} Q _ {k} & M _ {k} \\ M _ {k} ^ {*} & R _ {k} \end{array} \right] \left[ \begin{array}{l} x _ {k} ^ {\prime} \\ u _ {k} ^ {\prime} \end{array} \right], \left[ \begin{array}{l} x _ {k} ^ {\prime} \\ u _ {k} ^ {\prime} \end{array} \right] \right\rangle \right. \tag {3} \\ \left. + \langle V _ {k} (u _ {k} ^ {\prime} - u _ {k - 1} ^ {\prime}), u _ {k} ^ {\prime} - u _ {k - 1} ^ {\prime} \rangle\right), \\ \end{array}
$$

where the terminal penalty $P = P ^ { * }$ and the weights $Q _ { k } =$ $\begin{array} { r } { Q _ { k } ^ { * } \in \mathcal { L } ( X ) , R _ { k } = R _ { k } ^ { * } , V _ { k } = V _ { k } ^ { * } \in \mathcal { L } ( U ) } \end{array}$ and $M _ { k } \in \mathcal { L } ( U , X )$ satisfy $P , V _ { N } , V _ { k } , \left\lceil \begin{array} { l l } { Q _ { k } } & { M _ { k } } \\ { M _ { k } ^ { \ast } } & { R _ { k } } \end{array} \right\rceil \geq 0$ for all $k = 0 , \ldots , N - 1$ . The minimization of the cost functional (3) is subject to constraints of the form

$$
\left\{ \begin{array}{c c} x _ {k + 1} ^ {\prime} = A ^ {\prime} x _ {k} ^ {\prime} + B ^ {\prime} u _ {k} ^ {\prime}, \\ x _ {0} ^ {\prime} = x _ {n}, \\ u _ {- 1} ^ {\prime} = u _ {n - 1}, \end{array} \quad \left[ \begin{array}{c} x _ {N} ^ {\prime} \\ u _ {N - 1} ^ {\prime} \end{array} \right] \in \mathcal {T}, \right. \tag {4}

\left[ \begin{array}{c} x _ {k} ^ {\prime} \\ u _ {k} ^ {\prime} \\ u _ {k - 1} ^ {\prime} \end{array} \right] \in \mathcal {W} _ {k}, \quad 0 \leq k \leq N - 1,
$$

where $\mathcal { W } _ { k }$ and  represent some stage and terminal constraints introduced in $( 5 ) – ( 6 )$ , respectively, and ideally the prediction model is perfect, i.e., $( A ^ { \prime } , B ^ { \prime } ) = ( A , B )$ . The obtained control step $u _ { n }$ is applied to the plant and at the next time step, a new optimization is carried out. We assume that the MPC procedure is stabilizing and feasible, and refer to [15, Sect. $5 \mathrm { - } 7 ]$ for an introduction to these topics.
