Rate-dependent usable capacity is described by Peukert’s law (Yang et al., 2019):

$$Q _ {\text { eff }} = Q _ {\text { all }} \left(\frac {C _ {\text { ref }}}{C _ {\text { now }}}\right) ^ {n - 1}, \tag {40}$$

where $Q _ { \mathrm { a l l } }$ is the rated capacity, $C _ { \mathrm { r e f } }$ the reference C-rate, $C _ { \mathrm { n o w } }$ the operating C-rate, and n Peukert’s constant. The cell SOC keeps its standard definition based on the rated capacity:

$$\mathrm{SOC} (k + 1) = \mathrm{SOC} (k) - \frac {I (k) \Delta t}{Q _ {\text { all }}}. \tag {41}$$

Electrolyte diffusion is represented by two concentrationdeviation states at the positive and negative current collectors (Li et al., 2018):

$$\Delta c _ {1} (k + 1) = \Delta c _ {1} (k) + \frac {\Delta t}{\tau_ {e}} \big [ P _ {\mathrm{con}, a} I (k) - \Delta c _ {1} (k) \big ], \tag {42}\Delta c _ {2} (k + 1) = \Delta c _ {2} (k) + \frac {\Delta t}{\tau_ {e}} \big [ P _ {\mathrm{con}, b} I (k) - \Delta c _ {2} (k) \big ], \tag {43}$$

where $\tau _ { e }$ is the electrolyte diffusion time constant and $P _ { \mathrm { c o n } , a } , P _ { \mathrm { c o n } , b }$ are cell-dependent coefficients.

The corresponding concentration overpotential is

$$\eta_ {\mathrm{con}} (k) = \frac {2 R T}{F} (1 - t _ {+}) \ln \left(\frac {c _ {0} + \Delta c _ {1} (k)}{c _ {0} - \Delta c _ {2} (k)}\right), \tag {44}$$

with R the gas constant, T the temperature, F the Faraday constant, $t _ { + }$ the transference number, and $c _ { 0 }$ the nominal electrolyte concentration.

The reaction overpotential is obtained from Butler–Volmer kinetics:

$$
\begin{array}{l} \eta_ {\mathrm{rxn}} (k) = \frac {2 R T}{F} \left[ \ln \left(\sqrt {m _ {p} ^ {2} (k)} + 1 + m _ {p} (k)\right) \right. \\ \left. + \ln \left(\sqrt {m _ {n} ^ {2} (k)} + 1 + m _ {n} (k)\right) \right]. \tag {45} \\ \end{array}
