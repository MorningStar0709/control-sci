# D. Infer the Laplacian Topology Matrix

In this part, we demonstrate how to extract an estimate for L from the inferred continuous-time closed-loop matrix $\widehat { A } _ { c } .$ .

First, recalling (49) reveals that $A _ { c } ( \mathbf { 1 } _ { N } \otimes I _ { n } ) = \mathbf { 1 } _ { N } \otimes A ,$ which indicates that all N blocks of the sum $A _ { c } ( \mathbf { 1 } _ { N } \otimes I _ { n } )$ are exactly the matrix A. This identity can be seen as an intrinsic disclosure vulnerability of NDSs satisfying (2). Leveraging this property, an external observer easily extracts the estimate of A from $\widehat { A } _ { c } ,$ given by

$$\widehat {A} = \frac {1}{N} \sum_ {i \in \mathcal {V}} \sum_ {j \in \mathcal {V}} \widehat {A} _ {c} [ i, j ]. \tag {53}$$

Since $\widehat { A }$ is a sum of finite blocks in $\widehat { A } _ { c } ,$ the convergence rate of $\| \widehat { A } ( T ) - A \|$ is the same as $\| \widehat { A } _ { c } ( T ) - A _ { c } \|$ and it holds that limT→∞ $\| \widehat { A } ( \widehat { T } ) - A \| = 0$ . Then, define an auxiliary matrix as

$$W = I \otimes \widehat {A} - \widehat {A} _ {c}. \tag {54}$$

With the estimated $\widehat { A } _ { c }$ and ${ \widehat { A } } ,$ it is desired that the deviation $\| W - L \otimes B K \|$ should be as close to zero as possible. Since BK is tied up with L through a Kronecker product, it implies that W contains multiple block-wise linear equations about BK and $L .$ Specifically, based on the definition of L, we have

$$\sum_ {i, j = 1, j \neq i} L _ {i j} B K / N = - s _ {L} B K, \tag {55}$$

where $\begin{array} { r } { s _ { L } = \frac { 1 } { N } \sum _ { i = 1 } L _ { i i } } \end{array}$ represents the average row-sum for $A .$ Utilizing this property, a scalar ambiguity estimate of $B K$ is designed as

$$Z = \frac {1}{2 N} \sum_ {i = 1} ^ {N} \left(W _ {i i} - \sum_ {j = 1, j \neq i} ^ {N} W _ {i j}\right), \tag {56}$$

which leverages all information contained in the matrix W . It is worth mentioning that Z will approximate $s _ { L } B K$ instead of $B K$ as the observation number goes to infinity, and thus we call it a scalar ambiguity estimate.

Next, we estimate the elements of L consequently by taking the elements of Z as benchmarks. Note that either $B _ { \ell _ { 1 } } = 0$ or $K _ { \ell _ { 2 } } = 0$ will result in $[ B K ] _ { \ell _ { 1 } \ell _ { 2 } } = 0$ , and thus the nearzero elements in $Z$ can be excluded for estimation. Define the index set of the non-zero elements of $Z$ as

$$\mathcal {I} _ {Z} = \{(\ell_ {1}, \ell_ {2}): | Z _ {\ell_ {1} \ell_ {2}} | > \varepsilon_ {Z}, \ell_ {1}, \ell_ {2} = 1, \dots , n \}, \tag {57}$$
