# 2.4 Quadrature representation

Based on the above derivation, we can describe the cavity system coupled to a waveguide at two distant points as

$$
\begin{array}{l} \dot {a} (t) = \bar {A} a (t) + \bar {A} _ {d} a (t - T) + \bar {B} b _ {\mathrm{in}} (t) + \bar {B} _ {d} b _ {\mathrm{in}} (t - T), \\ b _ {\mathrm{out}} (t) = \bar {C} a (t) + \bar {C} _ {d} a (t - T) + \bar {D} _ {d} b _ {\mathrm{in}} (t - T), \tag {19} \\ \end{array}
$$

where the coefficient matrices are written as $\bar { A } = - \mathrm { i } \omega _ { c } -$ $\begin{array} { r } { \frac { \gamma } { 2 } , \bar { A } _ { \underline { { d } } } = - \frac { \gamma } { 2 } , \bar { B } = \bar { B } _ { d } = - \mathrm { i } \sqrt { \frac { \gamma } { 2 } } , \bar { C } = \bar { C } _ { d } = - \mathrm { i } \sqrt { \frac { \gamma } { 2 } } } \end{array}$ and $\bar { D } _ { d } = 1$ .

To obtain real coefficient equations, we convert the above equations into an quadrature representation and thus obtain

$$
\begin{array}{l} \mathrm{d} x (t) = A x (t)   \mathrm{d} t + A _ {d} x (t - T)   \mathrm{d} t + \tilde {B} \left[ \begin{array}{c} \mathrm{d} w (t) \\ \mathrm{d} w (t - T) \end{array} \right], \\ \mathrm{d} y (t) = C x (t) \mathrm{d} t + C _ {d} x (t - T) \mathrm{d} t + D _ {d} \mathrm{d} w (t - T), \tag {20} \\ \end{array}
$$

which constitutes a linear Gaussian system, since the state and output dynamics are linear in the system variables and are driven by Gaussian quantum noise processes.
