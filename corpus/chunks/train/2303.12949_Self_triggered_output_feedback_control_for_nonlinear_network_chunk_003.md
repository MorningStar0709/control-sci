# Notation and definitions

The nonnegative real numbers are denoted by $\mathbb { R } _ { > 0 }$ . The natural numbers are denoted by N, and we define $\mathbb { N } _ { 0 } : =$ N∪{0}. We denote the Euclidean norm by |·|. A continuous function $\alpha : \mathbb { R } _ { \geq 0 }  \mathbb { R } _ { \geq 0 }$ is a class K function if it is strictly increasing and $\alpha ( 0 ) \stackrel { \textstyle = } { = } 0 .$ . It is a class $\kappa _ { \infty }$ function if it is a class K function and it is unbounded. A continuous function $\beta : \mathbb { R } _ { \ge 0 } \times \mathbb { R } _ { \ge 0 } \to \mathbb { R } _ { \ge 0 }$ is a class KL function, if $\beta ( \cdot , t )$ is a class $\kappa$ function for each $t \in \mathbb { R } _ { > 0 }$ and $\beta ( q , \cdot )$ is nonincreasing and satisfies lim $\beta ( q , t ) \ \stackrel { - } { = } \ 0$ for each t→∞ $q \in \mathbb { R } _ { > 0 }$ . A function $\beta : \mathbb { R } _ { \geq 0 } \times \bar { \mathbb { R } } _ { \geq 0 } \times \mathbb { R } _ { \geq 0 } \to \mathbb { R } _ { \geq 0 }$ is a class KLL function if for each $r \geq \bar { 0 } , \beta ( \cdot , r , \cdot )$ and $\bar { \beta } ( \cdot , \cdot , r )$ are class KL functions.

We use (Carnevale et al., 2007, Definitions 1-3), that are originally taken from Goebel and Teel (2006), to characterize a hybrid model of the considered NCS and corresponding hybrid time domains, trajectories and solutions. Moreover, we adapt the definitions of maximal solutions and t−completeness from Goebel and Teel (2006).
