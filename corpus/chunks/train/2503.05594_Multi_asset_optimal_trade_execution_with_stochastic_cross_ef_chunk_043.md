# 5.1. Proofs for Section 2

Proof of Proposition 2.5. Integration by parts, (4), (1), and the fact that ν is continuous and of finite variation imply that

$$
\begin{array}{l} d (\nu (s) D ^ {X} (s)) = \nu (s) d D ^ {X} (s) + (d \nu (s)) D ^ {X} (s) + d [ \nu , D ^ {X} ] (s) \\ = - \nu (s) \rho (s) D ^ {X} (s) d s + \nu (s) \gamma (s) d X (s) + \nu (s) \rho (s) D ^ {X} (s) d s \tag {45} \\ = \nu (s) \gamma (s) d X (s), \quad s \in [ 0, T ]. \\ \end{array}
$$

We use this and the fact that $\Delta D ^ { X } ( s ) = \gamma ( s ) \Delta X ( s ) , s \in [ 0 , T ]$ , to obtain that

$$
\begin{array}{l} \int_ {[ 0, T ]} \left(2 (D ^ {X} (s -)) ^ {\top} + (\Delta X (s)) ^ {\top} \gamma (s)\right) d X (s) \\ = \int_ {[ 0, T ]} \left(2 (D ^ {X} (s -)) ^ {\top} + (\Delta D ^ {X} (s)) ^ {\top}\right) \gamma^ {- 1} (s) \nu^ {- 1} (s) d (\nu (s) D ^ {X} (s)) \tag {46} \\ = \int_ {[ 0, T ]} \left(2 (\widetilde {D} (s -)) ^ {\top} + (\Delta \widetilde {D} (s)) ^ {\top}\right) \widetilde {\nu} (s) d \widetilde {D} (s), \\ \end{array}
$$

where we introduced the abbreviations $\widetilde { D } = \nu D ^ { X }$ and $\widetilde { \nu } = ( \nu ^ { - 1 } ) ^ { \top } \gamma ^ { - 1 } \nu ^ { - 1 }$ . Note that, since $\widetilde { D } = \nu D ^ { X }$ has finite variation, it holds for all $r \in [ 0 , T ]$ that

$$
\begin{array}{l} \int_ {[ 0, r ]} \left(\Delta \widetilde {D} (s)\right) ^ {\top} \widetilde {\nu} (s) d \widetilde {D} (s) = \sum_ {\substack {k = 1 \\ n}} ^ {n} \int_ {[ 0, r ]} \Delta \left(\left(\widetilde {D} (s)\right) ^ {\top} \widetilde {\nu} (s)\right) _ {k} d \widetilde {D} _ {k} (s) \tag{47} \\ = \sum_ {k = 1} ^ {n} \left[ \left(\widetilde {D} ^ {\top} \widetilde {\nu}\right) _ {k}, \widetilde {D} _ {k} \right] (r) = \left[ \widetilde {D} ^ {\top} \widetilde {\nu}, \widetilde {D} \right] (r). \\ \end{array}
$$

Furthermore, integration by parts and using that $\widetilde { D } = \nu D ^ { X }$ has finite variation and $\widetilde { \nu } = ( \nu ^ { \top } ) ^ { - 1 } \gamma ^ { - 1 } \nu ^ { - 1 }$ is continuous shows for all $s \in [ 0 , T ]$ that

$$d \big ((\widetilde {D} (s)) ^ {\top} \widetilde {\nu} (s) \big) = \big (d (\widetilde {D} (s)) ^ {\top} \big) \widetilde {\nu} (s) + (\widetilde {D} (s)) ^ {\top} d \widetilde {\nu} (s). \tag {48}$$
