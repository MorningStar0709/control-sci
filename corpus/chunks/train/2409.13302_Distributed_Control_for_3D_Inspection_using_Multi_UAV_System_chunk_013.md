# C. Object Avoidance

During the mission of the multi-UAV system, each UAV $i \in \mathcal V$ moves toward the centroid $\mathbf { C } _ { V _ { i } }$ of its Voronoi region $V _ { i } .$ . However, this movement generates a trajectory that may cause a collision between the UAV and the object of interest. Consequently, an object avoidance technique is adopted to maintain the UAVs’ positions $\mathbf { p } _ { i } \in \mathcal { Q } - \mathcal { W }$ . As mentioned above, the object of interest is represented by a set of target points $\mathcal { Q } _ { T }$ that all UAVs should avoid due to disastrous consequences. As a result, a repulsive function is utilized and given as follows:

$$
U _ {o, i} = \left\{ \begin{array}{c l} \sum_ {l = 1} ^ {| \mathcal {Q} _ {T} |} \frac {1}{2} \epsilon \left(\frac {1}{\| \mathbf {p} _ {i} - \mathbf {q} _ {T} ^ {l} \|} - \frac {1}{d _ {o}}\right) ^ {2}, & \text { if } \| \mathbf {p} _ {i} - \mathbf {q} _ {T} ^ {l} \| \leq d _ {o} \\ 0, & \text { otherwise } \end{array} \right. \tag {9}
$$

where $\epsilon , d _ { o } > 0$ are a positive gain and the safety distance, that is, the minimum allowable distance of the UAV $i \in \mathcal V$ from each target point $\mathbf { q } _ { T } ^ { l }$ , respectively.
