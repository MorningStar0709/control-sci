# D.3. Proof of Lemma 9

The following lemma guarantees existence of a stable neighborhood around any stable matrix.

Lemma 13 Let $A _ { c } \in \mathcal { M } _ { S c h u r } , i . e . , \rho ( A _ { c } ) < 1$ . Then, there exists $\epsilon > 0$ such that for any $\Delta \in \mathcal { M } _ { n }$ with $\| \Delta \| _ { F } \le 1$ , we have that $A _ { c } + \epsilon \Delta \in \mathcal { M } _ { S c h u r } , i . e . , \rho ( A _ { c } + \epsilon \Delta ) < 1$ .

Proof Per Gelfand’s formula, we have that for any $\delta > 0$ , there exists $N _ { \delta } \in \mathbb { N }$ such that

$$\rho (A _ {c}) \leq \| A _ {c} ^ {k} \| _ {F} ^ {1 / k} < \rho (A _ {c}) + \delta \tag {53}$$

for any $k \geq N _ { \delta }$ . Since the mapping $A _ { c } \mapsto \left. A _ { c } ^ { k } \right. _ { F } ^ { 1 / k }$ is smooth for any $k \in \mathbb N .$ we can write the following expansion by Taylor’s theorem for any $t \in \mathbb { R }$

$$\left. \left\| \left(A _ {c} + t \Delta\right) ^ {k} \right\| _ {F} ^ {1 / k} = \left\| A _ {c} ^ {k} \right\| _ {F} ^ {1 / k} + t \frac {\mathrm{d}}{\mathrm{d} t} \left\| \left(A _ {c} + t \Delta\right) ^ {k} \right\| _ {F} ^ {1 / k} \right| _ {\lambda t} \tag {54}$$

where $\lambda \in [ 0 , 1 ]$ . For a given $t \in \mathbb { R }$ , there exists a constant $M _ { k , t } > 0$ such that for any $\| \Delta \| _ { F } \le 1$ , we have that $\begin{array} { r } { \left| \frac { \mathrm { d } } { \mathrm { d } t } \| ( A _ { c } + t \Delta ) ^ { k } \| _ { F } ^ { 1 / k } \right| \leq M _ { k , t } } \end{array}$ by Taylor’s theorem. Then, we can write the following upper bound

$$\left\| \left(A _ {c} + t \Delta\right) ^ {k} \right\| _ {F} ^ {1 / k} \leq \left\| A _ {c} ^ {k} \right\| _ {F} ^ {1 / k} + | t | M _ {t, k} \tag {55}$$

Using the relation (53) and the upper bound (55), we have that for any $\delta > 0 , t > 0$ , and $\| \Delta \| _ { F } \le 1$ , there exists $N _ { \delta } \in \mathbb { N }$ and ${ M _ { t , N _ { \delta } } } > 0$ such that

$$\rho (A _ {c} + t \Delta) \leq \| (A _ {c} + t \Delta) ^ {N _ {\delta}} \| _ {F} ^ {1 / N _ {\delta}} \leq \| A _ {c} ^ {N _ {\delta}} \| _ {F} ^ {1 / N _ {\delta}} + t M _ {t, N _ {\delta}} \tag {56}< \rho (A _ {c}) + \delta + t M _ {t, N _ {\delta}} \tag {57}$$

Fix a $\delta > 0$ such that $\rho ( A _ { c } ) + \delta < 1$ and fix $\mathrm { ~ a ~ } t > 0$ . Then, we can find $0 < \epsilon \leq t$ such that $\rho ( A _ { c } ) + \delta + \epsilon M _ { t , N _ { \delta } } < 1$ and thus

$$\rho (A _ {c} + \epsilon \Delta) < \rho (A _ {c}) + \delta + \epsilon M _ {t, N _ {\delta}} < 1 \tag {58}$$

for any $\| \Delta \| _ { F } \le 1$ by (57).
