Employing the approach adopted in Section 3.2.4, it can be shown, if $\mathbb { \Delta } _ { d }$ is sufficiently small, that there exist two sublevel sets $S _ { b } ( \hat { d } , \bar { r } ) : = \{ z \mid V _ { N } ^ { 0 } ( z , \hat { d } , \bar { r } ) \leq b \}$ and $S _ { c } ( \hat { d } , \bar { r } ) : = \{ z \ | \ V _ { N } ^ { 0 } ( z , \hat { d } , \bar { r } ) \ \le c \} , \ c > b$ such that $z ( 0 ) \in S _ { c } ( \hat { d } ( 0 ) , \bar { r } )$ implies the existence of a finite time $i _ { 0 }$ such that $z ( i ) \in S _ { b } ( \hat { d } ( i ) , \bar { r } )$ for all $i \geq i _ { 0 }$ .
