# Output Error with Extended Prediction Model

From (3.279) and (3.283), it results that the extended output error adjustable predictor should have the structure:

$$\hat {y} ^ {0} (t + 1) = \hat {\theta} ^ {T} (t) \phi (t) + \sum_ {i = 1} ^ {n _ {A}} \hat {c} _ {i} (t) \varepsilon (t + 1 - i) = \hat {\theta} _ {e} ^ {T} (t) \phi_ {e} (t) \tag {3.284}\hat {y} (t + 1) = \hat {\theta} (t) \phi (t) + \sum_ {i = 1} ^ {n _ {A}} \hat {c} _ {i} (t + 1) \varepsilon (t + 1 - i) = \hat {\theta} _ {e} ^ {T} (t + 1) \phi_ {e} (t) \tag {3.285}$$

where:

$$\hat {\theta} _ {e} ^ {T} (t) = \left[ \hat {\theta} ^ {T} (t), \hat {c} _ {1} (t), \dots , \hat {c} _ {n _ {A}} (t) \right] \tag {3.286}\phi_ {e} ^ {T} (t) = \left[ \phi^ {T} (t), \varepsilon (t), \dots , \varepsilon \left(t - n _ {A} + 1\right) \right] \tag {3.287}$$

Subtracting now (3.285) from (3.275), one gets:

$$
\begin{array}{l} \varepsilon (t + 1) = y (t + 1) - \hat {y} (t + 1) \\ = \phi^ {T} (t) [ \theta - \hat {\theta} (t + 1) ] + \sum_ {i = 1} ^ {n _ {A}} \left[ - a _ {i} - \hat {c} _ {1} (t + 1) \right] \varepsilon (t + 1 - i) \\ = \phi_ {e} ^ {T} (t) [ \theta_ {e} - \hat {\theta} _ {e} (t + 1) ] \tag {3.288} \\ \end{array}
$$

with:

$$\theta_ {e} = \left[ a _ {1}, \dots , a _ {n _ {A}}, b _ {1}, \dots , b _ {n _ {B}}, - a _ {1}, \dots , - a _ {n _ {A}} \right] \tag {3.289}$$

By applying Theorem 3.1 directly, one concludes that $\begin{array} { r } { \operatorname* { l i m } _ { t \to \infty } \varepsilon ( t + 1 ) = 0 } \end{array}$ when using the adaptation algorithm of (3.231) through (3.233) in which $\hat { \theta } ( t ) , \phi ( t ) , \nu ( t )$ are replaced by $\hat { \theta } _ { e } ( t ) , \phi _ { e } ( t ) , \varepsilon ( t )$ given by (3.284), (3.285) and (3.286) and without any positive real condition to be satisfied. For similar solutions see also Sect. 5.5, Anderson and Landau (1994) and Shah and Franklin (1982).
