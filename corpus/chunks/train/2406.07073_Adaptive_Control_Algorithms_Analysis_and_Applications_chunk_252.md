For $y _ { I V } ( t )$ to be representative of $y ( t )$ , an estimation of $a _ { i }$ and $b _ { i }$ must be available. This is why the method should be initialized by the recursive least squares (initialization horizon: 5 to 10 times the number of parameters). For a fixed value $\hat { \theta }$ , the equation of the prediction error can be written as (see Problem 5.2):

$$\varepsilon (t + 1, \hat {\theta}) = \frac {\hat {A} (q ^ {- 1} , \hat {\theta})}{A (q ^ {- 1})} [ \theta - \hat {\theta} ] \phi_ {I V} (t, \hat {\theta}) + w (t + 1) \tag {5.100}$$

$\phi _ { I V } ( t , { \hat { \theta } } )$ and $w ( t + 1 )$ are uncorrelated and the averaging method clearly indicates that $\hat { A }$ should be close enough to A, in order that the algorithm converge $( \hat { A } / A - \lambda _ { 2 } / 2$ should be strictly positive real). For a detailed coverage of instrumental variable methods see Ljung and Söderström (1983).
