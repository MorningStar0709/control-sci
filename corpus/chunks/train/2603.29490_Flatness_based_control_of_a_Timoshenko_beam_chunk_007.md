# A. Preliminary transformations

We introduce the state ${ \pmb x } ( z , t )$ which allows rewriting the system $( 2 ) - ( 4 )$ of two second-order PDEs as a system of four first-order PDEs:

$$
\begin{array}{l} \partial_ {t} \boldsymbol {x} ^ {-} (z, t) = \boldsymbol {\Lambda} (z) \partial_ {z} \boldsymbol {x} ^ {-} (z, t) \\ + \boldsymbol {A} ^ {- -} (z) \boldsymbol {x} ^ {-} (z, t) + \boldsymbol {A} ^ {- +} (z) \boldsymbol {x} ^ {+} (z, t) \tag {5a} \\ \end{array}
\partial_ {t} \boldsymbol {x} ^ {+} (z, t) = - \boldsymbol {\Lambda} (z) \partial_ {z} \boldsymbol {x} ^ {+} (z, t)+ \boldsymbol {A} ^ {+ -} (z) \boldsymbol {x} ^ {-} (z, t) + \boldsymbol {A} ^ {+ +} (z) \boldsymbol {x} ^ {+} (z, t) \tag {5b}\boldsymbol {x} ^ {+} (0, t) = - \boldsymbol {x} ^ {-} (0, t) \tag {5c}\boldsymbol {x} ^ {-} (1, t) = \boldsymbol {x} ^ {+} (1, t) + \boldsymbol {B u} (t), \tag {5d}$$

where $\pmb { \Lambda } ( z ) = \mathrm { d i a g } ( \lambda _ { 1 } ( z ) , \lambda _ { 2 } ( z ) )$ , and $a _ { i i } ( z ) = 0 , i =$ $1 , \ldots , 4 ,$ for the diagonal elements of $A ( z )$ . The specific expressions for $\pmb { x } ( z , t ) , \lambda _ { 1 } ( z ) , \lambda _ { 2 } ( z ) , \pmb { A } ( z )$ and B depend on the parameters of the Timoshenko beam, as detailed in Remark 1. The system (5) in Riemann coordinates comprises four heterodirectional transport equations: two propagating in the negative (dim $\pmb { x } ^ { - } ( z , t ) = 2 )$ and two in the positive z-direction (dim $\pmb { x } ^ { + } ( z , t ) = 2 )$ . Both subsystems (5a) and (5b) share the same absolute transport velocities $\lambda _ { 1 } ( z ) ~ >$ $\lambda _ { 2 } ( z ) ~ > ~ 0$ as they are derived from the original pair of wave equations (2). Since the transport velocities are sorted in decreasing order, the respective first transport processes associated with $x _ { 1 } ^ { \mp } ( z , t )$ are faster than those associated with $x _ { 2 } ^ { \mp } ( z , t )$ . Defining strictly monotonically increasing positive functions $\begin{array} { r } { \phi _ { i } ( z ) = \int _ { 0 } ^ { z } \frac { \mathrm { d } \zeta } { \lambda _ { i } ( \zeta ) } , i = 1 , 2 } \end{array}$ , together with their corresponding inverse functions ψi satisfying $\psi _ { i } ( \phi _ { i } ( z ) ) = z$ yields the system’s transport times $\tau _ { i } = \phi _ { i } ( 1 )$ between the boundaries $z = 0$ and $z = 1$ , with $\tau _ { 2 } > \tau _ { 1 } > 0$ .
