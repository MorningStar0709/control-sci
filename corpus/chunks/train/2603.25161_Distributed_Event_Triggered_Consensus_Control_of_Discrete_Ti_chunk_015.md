Recalling that $\tilde { x } [ k ] = ( U ^ { \top } \otimes I _ { n } ) x [ k ]$ and $U ^ { \top } L U = \Lambda$ , we have

$$
\begin{array}{l} x ^ {\top} [ k ] (L \otimes Q) x [ k ] = \tilde {x} ^ {\top} [ k ] (\Lambda \otimes Q) \tilde {x} [ k ] \\ = \sum_ {i = 2} ^ {N} \lambda_ {i} (L) \tilde {x} _ {i} ^ {\top} [ k ] Q \tilde {x} _ {i} [ k ]. \tag {41} \\ \end{array}
$$

Combining (40) and (41) yields

$$\sum_ {k = 0} ^ {\infty} \sum_ {i = 2} ^ {N} \lambda_ {i} (L) \tilde {x} _ {i} ^ {\top} [ k ] Q \tilde {x} _ {i} [ k ] < \infty .$$

Since the graph is connected, we have $\lambda _ { i } ( L ) > 0$ for all $i \in \{ 2 , \ldots , N \}$ , and since $Q \succ 0 .$ , it follows that

$$\sum_ {k = 0} ^ {\infty} \| \tilde {x} _ {i} [ k ] \| ^ {2} < \infty , \quad \forall i \in \{2, \dots , N \}.$$

Therefore, we obtain

$$\lim _ {k \to \infty} \tilde {x} _ {i} [ k ] = 0, \quad \forall i \in \{2, \dots , N \},$$

which completes the proof.
