# 5.1 Construction of Quantized Mean-field Games

In this subsection, we assume that our state space X is compact. Since X is compact, we can find a sequence $( \{ x _ { n , i } \} _ { i = 1 } ^ { k _ { n } } ) _ { n \geq 1 }$ of finite grids in X such that for all n and $x \in X$ ,

$$\min _ {i \in \{1, 2, \dots , k _ {n} \}} d _ {X} (x, x _ {n, i}) < \frac {1}{n}.$$

Let $X _ { n } = \{ x _ { n , i } \} _ { i = 1 } ^ { k _ { n } } \subset X$ and define $T _ { n } : X \to X _ { n }$ via the relation

$$T _ {n} (x) = \operatorname * {a r g m i n d} _ {x _ {n, i} \in X} d _ {X} (x, x _ {n, i}),$$

where ties are broken so that $T _ { n }$ is measurable. The map $T _ { n }$ is called a quantization map (or a state aggregation map) and each element of $X _ { n }$ is called a representative (for the quantization). We use pullbacks of $T _ { n }$ to partition X as $\{ S _ { n , i } \} _ { i = 1 } ^ { k _ { n } }$ , where each $S _ { n , i }$ is given by

$$S _ {n, i} = \{x \in X: T _ {n} (x) = x _ {n, i} \}.$$

Since $T _ { n }$ is measurable, each $S _ { n , i }$ is a measurable subset of X and has the diameter $\begin{array} { r } { \operatorname* { s u p } _ { x , y \in S _ { n , i } } d _ { X } ( x , y ) < } \end{array}$ < ${ \begin{array} { l } { { \frac { 2 } { n } } . } \end{array} }$ .

Since $X _ { n } = \{ x _ { n , i } \} _ { i = 1 } ^ { k _ { n } } \subset X$ is a discrete space, we will equip it with the discrete metric $d _ { X _ { n } }$ :

$$d _ {X _ {n}} (x _ {n, i}, x _ {n, j}) = 1 _ {i \neq j}.$$
