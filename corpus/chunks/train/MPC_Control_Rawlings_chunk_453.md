$$\left| x (k) - \hat {x} (k) \right| \leq \overline {{\beta}} \left(\underline {{\gamma}} _ {x} ^ {- 1} (\delta) + \gamma_ {x} ^ {- 1} (\delta)\right) + \left(\gamma_ {1} + \gamma_ {2}\right) \left(\underline {{\gamma}} _ {w} ^ {- 1} (\delta) + \gamma_ {w} ^ {- 1} (\delta)\right)$$

in which $\overline { { \beta } } ( s ) : = \beta ( s , 0 )$ is a K-function. Finally we choose $\delta$ such that the right-hand side is less than ε, which is possible since the right-hand side defines a K-function, which goes to zero with $\delta .$ This gives for all $k \geq 0$

$$| x (k) - \hat {x} (k) | \leq \varepsilon$$

and part (b) of the robust GAS definition is established.
