# 3.1 Ho-Kalman-Kung Algorithm based D-LMPC

One of the initial approaches for linear state space identification is the Ho-Kalman algorithm, which is a realizationbased method [10]. Ho-Kalman algorithm uses the discrete impulse response data for identifying the system parameters: A, B, C matrices. The discrete impulse response of a linear system refers to the system’s output when subjected to a unit pulse input, often referred to as an impulse input. The Ho-Kalman-Kung algorithm is an extension of the Ho-Kalman algorithm proposed by Kung [11] which is based on the SVD-based factorization of the Hankel matrix constructed using the impulse response data. A Hankel matrix is characterized by constant entries along each ascending skew-diagonal. In other words, a Hankel matrix H is defined by its elements $h _ { i j }$ , where the entry at position $( i , j )$ is constant for all pairs $i + j = c ,$ where c is the skew-diagonal index. This structure is particularly useful in various system identification and signal processing tasks, as it captures the system’s temporal or spatial dependencies in a compact form. Consider the discrete time unit pulse function defined as:

$$
u _ {k} = \left\{ \begin{array}{l l} 1, & \text {   for   } k = 0 \\ 0, & \text {   for   } k = 1, 2, 3, \dots \end{array} \right. \tag {34}
$$

Applying this impulse input to a linear system with $\mathbf { x } _ { 0 } = \mathbf { 0 }$ results in the unit-pulse response as:

$$y _ {1} = \mathbf {C x} _ {1} = \mathbf {C} [ \mathbf {A x} _ {0} + \mathbf {B u} _ {0} ] = \mathbf {C B}y _ {2} = \mathbf {C x} _ {2} = \mathbf {C} \left[ \mathbf {A} ^ {2} \mathbf {x} _ {0} + \mathbf {A B} u _ {0} + \mathbf {B} u _ {1} \right] = \mathbf {C A B} \tag {35}
\begin{array}{c} \vdots \\ y _ {k} = \mathbf {C x} _ {k} = \mathbf {C A} ^ {k - 1} \mathbf {B}. \end{array}
$$
