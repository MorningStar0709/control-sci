# Theorem 10.4 (PAA with Data Normalization)

• Assume that the plant output can be described by:

$$y (t + 1) = \theta^ {T} \phi (t) + w (t + 1) \tag {10.115}$$

where $w ( t + 1 )$ represents the effects of the unmodeled dynamics or of a disturbance (or both).

• Assume that:

$$| w (t + 1) | \leq d _ {1} + d _ {2} \eta (t); \quad 0 < d _ {1}, d _ {2} < \infty \tag {10.116}$$

where:

$$\eta^ {2} (t) = \mu^ {2} \eta (t - 1) + \| \phi (t) \| ^ {2} \tag {10.117}$$

• Define the normalized input-output variables

$$\bar {y} (t + 1) = \frac {y (t + 1)}{m (t)}; \quad \bar {u} (t) = \frac {u (t)}{m (t)}; \quad \bar {\phi} (t) = \frac {\phi (t)}{m (t)} \tag {10.118}$$

where:

$$m ^ {2} (t) = \mu^ {2} m ^ {2} (t - 1) + \max [ \| \phi (t) \| ^ {2}, 1 ]; \quad m (0) = 1 \tag {10.119}$$

• Consider an adjustable model of the form:

$$\hat {y} ^ {0} (t + 1) = \hat {\theta} ^ {T} (t) \bar {\phi} (t) \tag {10.120}\hat {y} (t + 1) = \hat {\theta} ^ {T} (t + 1) \bar {\phi} (t) \tag {10.121}$$

• Consider the PAA of the form:

$$\hat {\theta} (t + 1) = \hat {\theta} (t) + F (t) \bar {\phi} (t) \bar {\nu} (t + 1) \tag {10.122}F (t + 1) ^ {- 1} = \lambda_ {1} (t) F (t) ^ {- 1} + \lambda_ {2} (t) \bar {\phi} (t) \bar {\phi} (t) ^ {T}\lambda_ {1} (t) \equiv 1; 2 > \lambda_ {2} (t) \geq 0 \tag {10.123}\bar {\nu} (t + 1) = \frac {\bar {y} (t + 1) - \hat {y} ^ {0} (t + 1)}{1 + \bar {\phi} ^ {T} (t) F (t) \bar {\phi} (t)} \tag {10.124}$$

Then:

$$\sum_ {t = 0} ^ {N} \bar {v} ^ {2} (t + 1) \leq c _ {1} + c _ {2} \sum_ {t = 0} ^ {N} \bar {w} ^ {2} (t + 1); \quad 0 < c _ {1}, c _ {2} < \infty \tag {10.125}$$

where:

$$\bar {w} (t + 1) = \frac {w (t + 1)}{m (t)} \tag {10.126}$$

and

$$\frac {1}{N} \sum_ {t = 0} ^ {N} \bar {\nu} ^ {2} (t + 1) \leq c _ {2} \delta^ {2} + c _ {1} / N \tag {10.127}$$

where:

$$\delta = d _ {2} + d _ {1} \tag {10.128}$$

Proof The proof of Theorem 10.4 is a straightforward application of Theorem 10.1. By using normalized variables, the normalized output equation takes the form:

$$\bar {y} (t + 1) = \theta^ {T} \bar {\phi} (t) + \bar {w} (t + 1) \tag {10.129}$$

where now w(t¯ + 1) is strictly bounded. Applying now the results of Theorem 10.1 to this normalized plant model, one gets (10.125) from (10.20). On the other hand, (10.127) is a direct consequence of (10.125) and (10.116). -

Corollary 10.1 Under the same hypotheses as Theorem 10.4, if one defines the unnormalized a posteriori prediction error as:

$$\varepsilon (t + 1) = y (t + 1) - \hat {\theta} ^ {T} (t + 1) \phi (t) \tag {10.130}$$

one has:

$$\varepsilon^ {2} (t + 1) \leq \bar {\nu} ^ {2} (t + 1) m ^ {2} (t) \tag {10.131}$$

Proof of Corollary 10.1 From the definition of $\bar { \varepsilon } , \bar { y } , \bar { \phi } ( t )$ one gets:

$$\varepsilon (t + 1) = \bar {y} (t + 1) m (t) - \hat {\theta} ^ {T} (t + 1) \bar {\phi} (t) m (t) = \bar {\nu} (t + 1) m (t) \tag {10.132}$$

and (10.131) results.
