# 2.4 Problem Statement

Due to the stochasticity introduced for privacy preservation, the control objective in (2) cannot be achieved exactly. Accordingly, the cooperative control problem is reformulated and formally stated as follows.

Problem 1. Design a differentially private distributed cooperative control mechanism for the MAS described by (1) such that:

(1) The tracking error $\xi _ { l _ { k } } ( t ) \triangleq x _ { i } ( t ) - x _ { j } ( t ) - d _ { l _ { k } }$ converges to a finite random limit $x _ { l _ { k } } ^ { \ast }$ in mean square and almost surely, for all $l _ { k } = ( i , j ) \in \mathcal { E } ;$   
(2) The mechanism guarantees bounded $( \epsilon , \delta )$ -differential privacy from time $t = 0$ to infinity for all $d _ { l _ { k } } , \ l _ { k } =$ $( i , j ) \in \mathcal { E }$ under the adjacency relation defined in Definition 1.

The first objective ensures the convergence of the tracking error in a stochastic sense, while the second enforces privacy protection on the reference signals. To facilitate the subsequent convergence analysis in Section 3.2, the following result from martingale theory is needed.

Lemma 3. (Ash, 2014) Let $\{ X ( k ) , { \mathcal { F } } ( k ) \}$ be a martingale sequence satisfying sup $\mathbb { E } \| X ( k ) \| ^ { 2 } \leq \infty .$ Then X(k) cont≥0 verges almost surely and in mean square.
