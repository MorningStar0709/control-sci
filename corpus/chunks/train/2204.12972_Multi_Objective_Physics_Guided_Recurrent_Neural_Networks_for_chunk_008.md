The new hidden state $h _ { k }$ results as a linear combination of the preceding hidden state $\boldsymbol { h } _ { k - 1 }$ and the candidate state $\tilde { h } _ { k } , \mathrm { i . e . , } h _ { k } = ( 1 - z _ { k } ) \odot h _ { k - 1 } + z _ { k } \odot \tilde { h } _ { k }$ . Here, the gate activation function $\sigma _ { g }$ and state activation function $\pmb { \sigma } _ { s }$ are the logistic and tanh-function, respectively, while indicates element-wise multiplication. The GRU’s output at time t is the current hidden state $h _ { N _ { t } }$ . Following up the GRU with a consecutive layer of neurons, the high dimensional hidden state $h _ { N _ { t } }$ is transformed back into state space $\hat { \mathbb X }$ and output space $\mathbb { Y } , \mathrm { i . e . , ~ } \dot { \hat { \mathbf { x } } } _ { t } : = \pmb { w } _ { b } ^ { T } \pmb { h } _ { N _ { t } } + \pmb { b } _ { b } ,$ , ${ \pmb w } _ { b }$ and $b _ { b }$ being the consecutive layer’s weights and biases. The output $\hat { \pmb y } _ { t }$ is retrieved analogously.
