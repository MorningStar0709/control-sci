We evaluate our method on an autonomous aircraft taxiing system (AATS) [20] that is based on Boeing’s TaxiNet [29]. For the reachability analysis we used Continuous Reachability Analyzer (CORA) [3]. We provide a complete comparison between our proposed method and the existing baseline method [20].

Algorithm 2: Forward Reachable Set Computation   
1 Input: Initial Cell Set $C_{0}$ , Cell Abstraction Graph $\langle C, E \rangle$ ;
2 Output: Reachable Cells $C_{end}$ ;
3 $n \leftarrow 0$ ;
// while reachable cells haven't converged
4 while $C_{n+1} \neq C_{n}$ do
5 $n \leftarrow n + 1$ ;
6 $C_{n+1} \leftarrow \emptyset$ ;
7    for c in $C_{n}$ do
8    // get successor cells from abstraction graph
9 $C' \leftarrow \{c' | (c, c') \in E\}$ ;
// append successor cells
9 $C_{n+1} \leftarrow C_{n+1} \cup C'$ ;
10    end
11 end
