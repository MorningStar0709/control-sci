The sensitivity result is crucial in characterizing the impact of misspecification on the equilibrium point in a quantified manner. This result helps to quantify and understand the misalignment of the game models used by different agents both within themselves and with the real game model, and the impact on the steady state of the feedback dynamics. As $\nabla _ { \boldsymbol { \bar { \theta } } } \bar { u } ( \bar { \boldsymbol { \theta } } )$ increases, the sensitivity of the equilibrium point $x ^ { * } ( { \bar { \theta } } )$ to ¯θ increases. This can be interpreted as, the more misspecified games become, the more sensitive the equilibrium point becomes to changes in misspecifications. This phenomenon is exemplified in the next section.
