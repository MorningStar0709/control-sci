# B. Dynamic Feedback Linearization (DFL)

DFL also known as the dynamic extension algorithm, is comprehensively discussed in ( [37], Chapter 6). By analyzing the decoupling matrix $A ( x )$ , it becomes apparent that $u _ { 2 }$ is the main problem as it does not appear in any of the two outputs resulting in a zero-column which leads to $d e t ( A ) = 0$ . A possible solution is a decoupling matrix $A ( x )$ with ??2. This can be done by adding a differential delay to $u _ { 1 }$ where an integrator would allow ??2 to appear in the $A ( x )$ matrix. An integrator in $u _ { 1 }$ adds a new state ?? resulting in a new state vector $\bar { x } \in \mathbb { R } ^ { 4 } : \bar { x } = [ x _ { 1 } , x _ { 2 } , x _ { 3 } , \zeta ] ^ { \top }$ and new vector fields ${ \bar { f } } ( { \bar { x } } ) , { \bar { g } } _ { 1 } ( { \bar { x } } )$ , and $\bar { g } _ { 2 } ( \bar { x } )$ as well as new control vector $\mathcal { U } =$ $[ \mathcal { U } _ { 1 } , \mathcal { U } _ { 2 } ] ^ { \top }$ . The new state $\bar { x } \in \mathbb { R } ^ { 4 } : \bar { x } = [ x _ { 1 } , x _ { 2 } , x _ { 3 } , \zeta ] ^ { \top }$ and dynamics can be written as follows:

$$
\bar {\dot {x}} = \left[ \begin{array}{c} \zeta \cos (x _ {3}) \\ \zeta \sin (x _ {3}) \\ \mathcal {U} _ {2} \\ \mathcal {U} _ {1} \end{array} \right], \quad y _ {1} = x _ {1} \quad y _ {2} = x _ {2} \tag {23}
$$

In [4], the authors proved that the extended model in (23) is feedback linearizable such that the extended system will be equivalent to a two-chain of double integrators using the control as follows:

$$
\mathcal {U} = \left[ \begin{array}{c} v _ {1} \cos (x _ {3}) + v _ {2} \sin (x _ {3}) \\ - v _ {1} \frac {\sin (x _ {3})}{\zeta} + v _ {2} \frac {\cos (x _ {3})}{\zeta} \end{array} \right] \tag {24}
$$
