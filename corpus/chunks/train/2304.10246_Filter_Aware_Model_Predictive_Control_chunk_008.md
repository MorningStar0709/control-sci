# 3.2. Model Predictive Control for Imperfect State Information

A variant of MPC related to open-loop feedback control (Dreyfus, 1965) approximates this problem by devising an optimal plan for the first K steps and using its first step:

$$\pi_ {\mathrm{MPC}} (\mathbf {x} _ {1: t}, \mathbf {u} _ {1: t - 1}) = \arg \min _ {\mathbf {u} _ {t}} \min _ {\mathbf {u} _ {t: t + K - 1}} \mathbb {E} _ {\mathbf {z} _ {t} | \mathbf {x} _ {1: t}, \mathbf {u} _ {1: t - 1}} \left[ \sum_ {k = t} ^ {t + K - 1} \beta^ {k - t} c (\mathbf {z} _ {k}, \mathbf {u} _ {k}) + \beta^ {K} \hat {J} (\mathbf {z} _ {t + K}) \right]. (2)$$

A terminal cost function $\hat { J }$ is used to alleviate some of the bias resulting from omitting the future.
