# 3.3 The nonlinear subsystem $\boldsymbol { \mathcal { S } } _ { N }$

We are now in position to deal with the nonlinear subsystem $\mathcal { S } _ { N }$ , defined by equations (10)-(11). The algebraic manipulations are much more involved and some difficulties must be overcome. In order to perform the time derivative of $V ( x )$ along an arbitrary trajectory of $\mathcal { S } _ { N }$ , let us determine

$$
\begin{array}{l} R (\sigma) \dot {x} = R (\sigma) \left(L (\sigma) A _ {K} R (\sigma) x + L (\sigma) F _ {0} w + G _ {0} ^ {T} y\right) \\ = c (\sigma) \left(A _ {K} R (\sigma) x + F _ {0} w + G _ {0} ^ {T} y\right) \tag {21} \\ \end{array}
$$

and also, see [4]

$$\dot {R} (\sigma) x = - \left(\frac {1}{\sqrt {\| \sigma \|}}\right) E _ {0} \Pi_ {\sigma} (B K R (\sigma) x + y) \tag {22}$$

in which expression $E _ { 0 } ~ = ~ [ ( 1 / 2 ) I _ { n } ~ 0 ] ^ { T } ~ \in ~ \mathbb { R } ^ { 2 n \times n }$ and $\Pi _ { \sigma } = ( \sigma / \lVert \boldsymbol { \hat { \sigma } } \rVert ) ( \sigma / \lVert \sigma \rVert ) ^ { T } \in \mathbb { R } ^ { n \times n }$ is a rank-one matrix with the key property $0 \leq \Pi _ { \sigma } \leq I _ { n }$ for all $0 \neq \sigma \in \mathbb { R } ^ { n }$ . Hence, with $( 2 1 ) \AA - \bigl ( 2 2 \bigr )$ and performing the factorization of the output variable $y ,$ the time derivative of the Lyapunov function along with an arbitrary trajectory of $\mathcal { S } _ { N }$ , can be expressed as

$$
\begin{array}{l} \dot {V} (x) = 2 c (\sigma) x ^ {T} R (\sigma) P \left(A _ {K} R (\sigma) x + F _ {0} w\right) \\ - \left(\frac {2}{\sqrt {\| \sigma \|}}\right) x ^ {T} R (\sigma) P E _ {0} \Pi_ {\sigma} B K R (\sigma) x \\ + 2 c (\sigma) x ^ {T} R (\sigma) P \Gamma_ {\sigma} y \tag {23} \\ \end{array}
$$

where, as it can be verified, the matrix $\Gamma _ { \sigma }$ is given by $\Gamma _ { \sigma } = G _ { 0 } ^ { T } - \rho _ { \sigma } E _ { 0 } \Pi _ { \sigma }$ with $\rho _ { \sigma } = 1 / \big ( c ( \sigma ) \sqrt { \| \sigma \| } \big )$ . Fortunately, from the definition (8), the scalar variable is such that $\rho _ { \sigma } \in ( 0 , 1 )$ for all $\sigma \neq 0$ which allows the determination of the upper bound

$$
\begin{array}{l} 0 \leq \Gamma_ {\sigma} \Gamma_ {\sigma} ^ {T} \\ = \left[ \begin{array}{c} I _ {n} - (\rho_ {\sigma} / 2) \Pi_ {\sigma} \\ 0 \end{array} \right] \left[ \begin{array}{c} I _ {n} - (\rho_ {\sigma} / 2) \Pi_ {\sigma} \\ 0 \end{array} \right] ^ {T} \\ \leq G _ {0} ^ {T} G _ {0} \tag {24} \\ \end{array}
$$
