# G.9 Proof of Lemma 28

proof. Let $x _ { 0 } , x _ { 1 } , . . . ,$ be the series of states when using controller $C _ { K } ^ { \mathrm { u n c } }$ under dynamics θ with $x _ { 0 } = 0$ . Then we have the recursive relationship that $x _ { 0 } = 0$ and $x _ { i + 1 } = ( a - b K ) x _ { i } + w _ { i }$ for all $i \geq 0$ . Using this recursive relationship, we have that

$$x _ {t} = \sum_ {i = 0} ^ {t - 1} w _ {i} (a - b K) ^ {t - 1 - i}. \tag {107}$$

If $\begin{array} { r } { K = \frac { a - 1 } { b } } \end{array}$ , then $a - b K = 1$ . This implies that $x _ { t } ^ { 2 } \longrightarrow \infty$ , and therefore $\bar { J } ( \theta , C _ { K } ^ { \mathrm { u n c } } , T ) = \infty$ .

For the rest of this proof, assume $\begin{array} { r } { K \in ( \frac { a - 1 } { b } , \frac { a } { b } ] } \end{array}$ . Recall that $u _ { i } = - K x _ { i }$ for all $i \geq 0$

Define $\rho = ( a - b K ) ^ { 2 }$ . Using the above expression for $x _ { t } .$ , we have that

$$
\begin{array}{l} \bar {J} (\theta , C _ {K} ^ {\mathrm{unc}}, T) = \frac {1}{T} \mathbb {E} \left[ q x _ {T} ^ {2} + \sum_ {t = 0} ^ {T - 1} q x _ {t} ^ {2} + r u _ {t} ^ {2} \right] \\ = \frac {1}{T} \left(q \mathbb {E} [ x _ {T} ^ {2} ] + \sum_ {t = 1} ^ {T - 1} (q + r K ^ {2}) \mathbb {E} [ x _ {t} ^ {2} ]\right) \quad [ x _ {0} = u _ {0} = 0 ] \\ = - \frac {r K ^ {2} \mathbb {E} [ X _ {T} ^ {2} ]}{T} + \frac {1}{T} \left(\sum_ {t = 1} ^ {T} (q + r K ^ {2}) \mathbb {E} [ x _ {t} ^ {2} ]\right). \\ \end{array}
$$

Furthermore, we have

$$
\begin{array}{l} \frac {1}{T} \left(\sum_ {t = 1} ^ {T} (q + r K ^ {2}) \mathbb {E} [ x _ {t} ^ {2} ]\right) \\ = \frac {1}{T} \sum_ {t = 1} ^ {T} (q + r K ^ {2}) \mathbb {E} \left[ \left(\sum_ {i = 0} ^ {t - 1} w _ {i} (a - b K) ^ {t - 1 - i}\right) ^ {2} \right] \tag {Equation(107)} \\ = \frac {1}{T} \sum_ {t = 1} ^ {T} (q + r K ^ {2}) \mathbb {E} \left[ \sum_ {i = 0} ^ {t - 1} \sum_ {j = 0} ^ {t - 1} w _ {i} w _ {j} (a - b K) ^ {t - 1 - i} (a - b K) ^ {t - 1 - j} \right] \\ = \frac {1}{T} \sum_ {t = 1} ^ {T} (q + r K ^ {2}) \sum_ {i = 0} ^ {t - 1} \sigma_ {\mathcal {D}} ^ {2} (a - b K) ^ {2 (t - 1 - i)} \\ = \frac {\sigma_ {\mathcal {D}} ^ {2}}{T} \sum_ {t = 1} ^ {T} (q + r K ^ {2}) \sum_ {i = 0} ^ {t - 1} (a - b K) ^ {2 i} \\ = \frac {\sigma_ {\mathcal {D}} ^ {2} (q + r K ^ {2})}{T} \sum_ {t = 1} ^ {T} \sum_ {i = 0} ^ {t - 1} \rho^ {i} \\ = \frac {\sigma_ {\mathcal {D}} ^ {2} (q + r K ^ {2})}{T} \sum_ {t = 1} ^ {T} \frac {1 - \rho^ {t}}{1 - \rho} \\ = \frac {\sigma_ {\mathcal {D}} ^ {2} (q + r K ^ {2})}{T (1 - \rho)} \left(T - \sum_ {t = 0} ^ {T - 1} \rho^ {t}\right) \\ = \frac {\sigma_ {\mathcal {D}} ^ {2} (q + r K ^ {2})}{1 - \rho} \Big (1 - \frac {1 - \rho^ {T}}{T (1 - \rho)} \Big). \\ \end{array}
$$
