In general, pure Nash equilibria (and thus GNE) may not exist for a game, or there may exist multiple within a single game [14], making their role as predictors unreliable. As such, many works have studied the specific class of strongly monotone games. This specific class of games consist of non-cooperative agents who choose their strategies from a convex set of actions and the pseudo-gradient of each agent’s cost, defined as $F ( u ) = \left( \nabla _ { u _ { i } } J _ { i } ( u _ { i } , u _ { - i } ) \right) _ { i \in N } .$ , is strongly monotone [15].

Definition 2. (Strong monotonicity): An operator $F ( x ) :$ $\mathcal { X } \subseteq \mathbb { R } ^ { n } \to \mathbb { R } ^ { n }$ is strongly monotone in X if there exists $\rho > 0$ such that,

$$\left(F (x _ {1}) - F (x _ {2})\right) ^ {\top} \left(x _ {1} - x _ {2}\right) \geq \rho \| x _ {1} - x _ {2} \| ^ {2},$$

for all $x _ { 1 } , x _ { 2 } \ \in \ { \mathcal { X } }$ , where ρ is the strong monotonicity constant.

GNE of monotone games where Z is convex can be equivalently defined in the form of a variational inequality (VI) problem of the pseudo-gradient F over Z.

Definition 3. (Variational inequality): Given a subset Z of $\mathbb { R } ^ { m }$ and a mapping $F : \mathcal { Z }  \mathbb { R } ^ { m }$ , a vector $u \in { \mathcal { Z } }$ solves the variational inequality problem, denoted $V I ( { \mathcal { Z } } , F )$ , if

$$(y - u) ^ {T} F (u) \geq 0, \forall y \in \mathcal {Z}. \tag {2}$$

The set of solutions to this problem is denoted $S ( \mathcal { Z } , F )$ .

In the game context, the solutions satisfying (2) are referred to as variational generalized Nash equilibria (or $\operatorname { v G N E } )$ . If the pseudo-gradient F is strongly monotone and $J _ { i }$ continuously differentiable over Z, then the solution set $S ( \mathcal { Z } , F )$ is equivalent to the set of Nash equilibria of the game as given below.
