Using that $\alpha \geq \bar { \alpha }$ and $| t - s | ^ { \beta ^ { \prime } } K _ { 1 } < 1$ , taking the $( \bar { \alpha } + \beta ) { \cdot } \mathrm { H } \ddot { \mathrm { o l d e r } }$ norm on both sides of the inequality (3.17) we obtain

$$\Psi_ {s, t} \leq C (K _ {1} \theta + K _ {1} K \Gamma_ {s, t} + K _ {1} ^ {2} | t - s | ^ {\beta^ {\prime}} \Xi_ {s, t} + K _ {1} (t - s) ^ {\beta^ {\prime}} \Psi_ {s, t}). \tag {3.18}$$

$$\| \delta X _ {s, t} \| _ {m} \lesssim K _ {1} \theta (t - s) ^ {\bar {\alpha}} + K _ {1} K \Gamma_ {s, t} (t - s) ^ {\alpha} + K _ {1} \Xi_ {s, t} (t - s) ^ {\alpha + \beta}.$$
