# 2. Linear Model Predictive Control (MPC)

We consider discrete-time linear systems, defined by the following linear difference equations:

$$x _ {t + 1} = A x _ {t} + B u _ {t}, \tag {1}$$

where $x _ { t } \in \mathbb { R } ^ { n _ { x } }$ the state at time t and $u _ { t } \in \mathbb { R } ^ { n _ { u } }$ the associated control input. We also impose input constraints: $u _ { t } \in \mathbb { U }$ for all $t \geq 0$ . We assume that U is a simple polyhedral set, i.e., it is described by linear inequality constraints $( { \mathrm { e . g . , } } { \underline { { u } } } \leq u \leq { \overline { { u } } } )$ . For system (1), we consider convex quadratic stage and final costs:

$$\ell (x, u) = 1 / 2 \left(x ^ {\top} Q x + u ^ {\top} R u\right), \quad \ell_ {f} (x) = 1 / 2 x ^ {\top} P x,$$

respectively. The matrices $P , Q \in \mathbb { R } ^ { n _ { x } \times n _ { x } }$ and $R \in \mathbb { R } ^ { n _ { u } \times n _ { u } }$ are positive (semi-)definite. For a prediction horizon of length N, the MPC problem for system (1), given the initial state x0, is:

$$\min _ {x _ {t}, u _ {t}} \sum_ {t = 0} ^ {N - 1} \ell (x _ {t}, u _ {t}) + \ell_ {f} (x _ {N}) \tag {2}\text { s.t.: } \quad x _ {t + 1} = A x _ {t} + B u _ {t}, \quad u _ {t} \in \mathbb {U} \forall t = 0: N - 1, \quad x _ {0} \text { given. }$$

In order to present a compact reformulation of the MPC problem as an optimization problem, we define the matrices:

$$
\mathbf {A} = \left[ \begin{array}{c c c c} B & 0 & \dots & 0 \\ A B & B & \dots & 0 \\ A ^ {2} B & A B & \dots & 0 \\ \vdots & \vdots & \vdots & \vdots \\ A ^ {N - 1} B & A ^ {N - 2} B & \dots & B \end{array} \right], \quad \mathbf {A} _ {N} = \left[ \begin{array}{c} A \\ A ^ {2} \\ A ^ {3} \\ \vdots \\ A ^ {N} \end{array} \right], \tag {3}
\mathbf {Q} = \operatorname{diag} (\underbrace {Q , \cdots , Q} _ {N - 1 \text { times }}, P) \quad \text { and } \quad \mathbf {R} = \operatorname{diag} (\underbrace {R , \cdots , R} _ {N \text { times }}).$$

Further, by eliminating the state variables, problem (2) is reformulated as a Quadratic Program (QP) [23]:

$$\min _ {\mathbf {u} \in \mathbb {U} ^ {N}} \frac {1}{2} \mathbf {u} ^ {T} \mathbf {H} \mathbf {u} + (\mathbf {q} (x _ {0})) ^ {T} \mathbf {u}, \tag {4}$$

where the decision variable contains the entire sequence of commands over the prediction horizon

$$\mathbf {u} = [ u _ {0} ^ {T}, u _ {1} ^ {T}, \dots , u _ {N - 1} ^ {T} ] ^ {T} \in \mathbb {R} ^ {N n _ {u}}$$

and

$$\mathbf {H} = \mathbf {R} + \mathbf {A} ^ {T} \mathbf {Q A} \quad \text { and } \quad \mathbf {q} (x _ {0}) = \mathbf {A} ^ {T} \mathbf {Q A} _ {N} x _ {0}.$$

Throughout the paper we assume that the matrices $Q , P$ and R are chosen such that problem (4) is strongly convex, i.e., H ≻ 0. Throughout the paper, we denote $N n _ { u } = \nu .$ . In the next sections we provide several strategies to solve the QP problem (4).
