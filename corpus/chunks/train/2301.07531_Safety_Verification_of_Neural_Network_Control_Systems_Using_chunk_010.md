$$
\tilde {\mathbf {W}} _ {\ell} \tilde {\boldsymbol {\eta}} _ {\ell - 1} + \tilde {\mathbf {b}} _ {\ell} = \left[ \begin{array}{c} \mathbf {W} _ {\ell} \boldsymbol {\eta} _ {\ell - 1} + \mathbf {b} _ {\ell} \\ \hat {\boldsymbol {\eta}} _ {\ell - 1} \end{array} \right], \hat {L} \leq \ell \leq L - 1 \tag {21}
$$

and

$$\boldsymbol {\eta} _ {L - 1} = \phi_ {L - 1} \circ \dots \circ \phi_ {\hat {L}} (\mathbf {W} _ {\hat {L}} \boldsymbol {\eta} _ {\hat {L} - 1} + \mathbf {b} _ {\hat {L} - 1})$$

which leads to

$$
\tilde {\boldsymbol {\eta}} _ {L - 1} = \left[ \begin{array}{l} \boldsymbol {\eta} _ {L - 1} \\ \hat {\boldsymbol {\eta}} _ {\hat {L} - 1} \end{array} \right] \tag {22}
$$

Then, when $\ell = L ,$ , it yields that

$$
\tilde {\mathbf {W}} _ {L} \tilde {\boldsymbol {\eta}} _ {L - 1} + \tilde {\mathbf {b}} _ {L} = \left[ \begin{array}{l} \mathbf {W} _ {L} \boldsymbol {\eta} _ {L - 1} + \mathbf {b} _ {L} \\ \hat {\mathbf {W}} _ {\hat {L}} \hat {\boldsymbol {\eta}} _ {\hat {L} - 1} + \hat {\mathbf {b}} _ {\hat {L}} \end{array} \right] \tag {23}
$$

Thus, one can obtain

$$
\tilde {\boldsymbol {\eta}} _ {L} = \left[ \begin{array}{l} \boldsymbol {\eta} _ {L} \\ \hat {\boldsymbol {\eta}} _ {\hat {L}} \end{array} \right] \tag {24}
$$

At last, when $\ell = L + 1$ , the following result can be obtained

$$
\tilde {\boldsymbol {\eta}} _ {L + 1} = \tilde {\mathbf {W}} _ {L + 1} \tilde {\boldsymbol {\eta}} _ {L} + \tilde {\mathbf {b}} _ {L + 1} = \left[ \begin{array}{c c} \mathbf {I} _ {n _ {L}} & - \mathbf {I} _ {n _ {\hat {L}}} \end{array} \right] \left[ \begin{array}{c} \boldsymbol {\eta} _ {L} \\ \hat {\boldsymbol {\eta}} _ {\hat {L}} \end{array} \right]
$$

which implies that $\tilde { \eta } _ { L + 1 } = \eta _ { L } - \hat { \eta } _ { \hat { L } }$ . Therefore, we can conclude that $\tilde { \Phi } ( \tilde { \eta } _ { 0 } ) = \Phi ( \eta _ { 0 } ) - \hat { \Phi } ( \bar { \hat { \eta } } _ { 0 } )$ as long as $\tilde { \eta } _ { 0 } =$ $\hat { \eta } _ { 0 } = \hat { \eta } _ { 0 }$ and

$$\rho (\Phi , \hat {\Phi}, \mathcal {U}) = \sup _ {\tilde {\eta} _ {0} \in \mathcal {U}} \left\| \tilde {\Phi} (\tilde {\boldsymbol {\eta}} _ {0}) \right\| \tag {25}$$

The proof is complete.
