# Solution:

We first show that p is constant along the extremals in the ”no y” case. Let’s consider during the proofs, for the easier notation’s sake, that $y ( x ) = y , y ^ { \prime } ( x ) = y , p ( x ) = p .$ . The Hamilton’s canonical equation for p is:

$$\frac {d p}{d x} = - H _ {y} (x, y, y ^ {\prime}, p), \quad x \in [ a, b ] \tag {112}$$

The Hamiltonian can be expressed as:

$$H (x, y, y ^ {\prime}, p) = p y ^ {\prime} - L (x, y, y ^ {\prime}) \tag {113}$$

Given the ”no $y '$ case, then we can write:

$$H (x, y ^ {\prime}, p) = p y ^ {\prime} - L (x, y ^ {\prime}) \tag {114}$$

Thus, the Hamilton’s canonical form yields

$$\frac {d p}{d x} = - H _ {y} (x, y ^ {\prime}, p), \quad x \in [ a, b ] \tag {115}= - \frac {d}{d y} \left(p y ^ {\prime} - L \left(x, y ^ {\prime}\right)\right) \tag {116}= - \frac {d}{d y} p y ^ {\prime} + \frac {d}{d y} L \left(x, y ^ {\prime}\right) \tag {117}= 0 \tag {118}$$

Given that dp = 0 then $\begin{array} { r } { { \frac { d p } { d x } } = 0 } \end{array}$

$$p (x) = \text { constant } \tag {119}$$

thus showing that in the ”no $y '$ case p is constant along extremals.

Now, let’s show that H is constant along extremals in the ”no $x '$ case. The Hamiltonian can be expressed as:

$$H (y, y ^ {\prime}, p) = p y ^ {\prime} - L (y, y ^ {\prime}) \tag {120}$$

Given the ”no $y '$ case, then we can write:

$$H (y, y ^ {\prime}, p) = p y ^ {\prime} - L (y, y ^ {\prime}) \tag {121}$$

We can derive with respect to x the Hamiltonian yielding

$$\frac {d}{d x} H \left(y, y ^ {\prime}, p\right) = \frac {d}{d x} \left(p y ^ {\prime} - L \left(y, y ^ {\prime}\right)\right) \tag {122}= \frac {d}{d x} p y ^ {\prime} - \frac {d}{d x} L \left(y, y ^ {\prime}\right) \tag {123}= 0 \tag {124}$$

Hence, given that $\begin{array} { r } { \frac { d } { d x } H ( y , y ^ { \prime } , p ) = 0 } \end{array}$ we can write that:

$$H (y, y ^ {\prime}, p) = \text { constant } \tag {125}$$

thus showing that in the ”no $x '$ case the Hamiltonian $H ( x , y , y ^ { \prime } , p )$ is constant along extremals. 
