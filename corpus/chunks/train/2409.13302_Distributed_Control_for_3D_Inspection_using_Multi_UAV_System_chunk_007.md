# III. PROBLEM STATEMENT

Consider a group of N quadrotor UAVs, with the dynamics described by (1), located in an arbitrary area within the bounded and convex inspection region $\mathcal { Q } \subset \mathbb { R } ^ { 3 }$ . It is assumed that each UAV i ∈ V, where $\mathcal { V } = \{ 1 , \ldots , N \}$ , is equipped with a gimballed camera that has the ability to rotate its fieldof-view (FoV) to any direction in Q, capturing important information about the environment. Let an object of interest, e.g. a structure, be represented by a surface from which a set of points $\mathcal { Q } _ { T } = \{ \mathbf { q } _ { T } ^ { l } \bar  \} , l \in \{ 1 , \dots , | \mathcal { Q } _ { T } | \}$ is extracted, where $\mathbf { q } _ { T } ^ { l }$ is the ${ { l } ^ { \mathrm { { t h } } } }$ target point on the object’s boundary ∂W, and the total number of these points is denoted by $\lvert \mathcal { Q } _ { T } \rvert$ . The target points that need to be inspected are already known to UAVs. The 3D inspection problem addressed in this work can be stated as follows: Find for each quadrotor $U A V i \in \mathcal { V }$ a distributed control law, such that, for any initial position ${ \bf p } _ { i } ( 0 )$ of each $U A V i \in \mathcal { V } ,$ , the multi-UAV system cooperatively inspects all target points QT on the object’s boundary ∂W, and all signals remain bounded. The aforementioned problem can be expressed at a high level, as formulated in Problem (P1).

Problem (P1): 3D Inspection Problem

$$\underset {\boldsymbol {u} _ {1} (t), \dots , \boldsymbol {u} _ {N} (t)} {\operatorname{argmin}} \mathcal {H}, \quad t \geq 0 \tag {2a}$$

subject to:

$$\dot {\mathbf {p}} _ {i} (t) = \boldsymbol {v} _ {i} (t), \quad \dot {\boldsymbol {v}} _ {i} (t) = \boldsymbol {u} _ {i} (t) \quad \forall i \in \mathcal {V}, \forall t \geq 0 \tag {2b}\mathbf {p} _ {i} (t) \in \mathcal {Q} - \mathcal {W} \quad \forall i \in \mathcal {V}, \forall t \geq 0 \tag {2c}\mathbf {p} _ {i} (t) \neq \mathbf {p} _ {j} (t) \quad \forall i, j \in \mathcal {V}, \forall t \geq 0 \tag {2d}$$

The objective is to design, for each UAV i ∈ V, a control input ${ \bf } u _ { i } ( t )$ that the trajectories of the multi-UAV system drive the inspection cost function H to the minimizer, subject to the constraints in (2b)-(2d). The constraint in (2b) corresponds to the UAVs’ dynamics introduced in (1) while the constraint (2c) ensures that the position of any UAV i ∈ V for $\forall t \geq 0 ,$ , belongs to the inspection region Q avoiding collision with the object of interest W. Finally, the collision avoidance between agents is represented by the constraint in (2d).
