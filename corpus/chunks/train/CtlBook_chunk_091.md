# Example 3.1

Find the equation or equations of motion for the following system

There is one inertia (J) so there is only one EOM:

$$J \ddot {\theta} + B (\dot {\theta} - 0) = \tau (t)$$

or

$$J \ddot {\theta} + B \dot {\theta} = \tau (t)$$
