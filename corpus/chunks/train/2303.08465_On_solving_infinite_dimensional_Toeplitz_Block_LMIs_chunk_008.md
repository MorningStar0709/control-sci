$$\dot {x} (t) = A (t) x (t) + B (t) u (t) \quad x (0) = x _ {0} \tag {3}$$

If, x is a solution associated to the control u ∈ $L _ { l o c } ^ { 2 } ( \mathbb { R } , \mathbb { C } ^ { m } )$ of the linear time periodic (LTP) system (3) then, $X = { \mathcal { F } } ( x )$ is a solution associated to $U = \mathcal { F } ( u )$ of the linear time invariant (LTI) system:

$$\dot {X} (t) = (\mathcal {A} - \mathcal {N}) X (t) + \mathcal {B} U (t), \quad X (0) = \mathcal {F} (x) (0) \tag {4}$$

where $\mathcal { A } = \mathcal { T } ( A ) , B = \mathcal { T } ( B )$ and

$$\mathcal {N} = I d _ {n} \otimes d i a g (\mathrm{j} \omega k, k \in \mathbb {Z}) \tag {5}$$

Reciprocally, if $X \in H$ is a solution to (4) with $U \in H ,$ then their representatives x and u $( \mathrm { i . e . ~ } X = \mathcal { F } ( x )$ and $\boldsymbol { U } = \mathcal { F } ( \boldsymbol { u } ) )$ 号 are a solution to (3). In addition, it is proved in [2] that one can reconstruct time trajectories from harmonic ones, that is:

$$x (t) = \mathcal {F} ^ {- 1} (X) (t) = \sum_ {k = - \infty} ^ {+ \infty} X _ {k} (t) e ^ {\mathrm{j} \omega k t} + \frac {T}{2} \dot {X} _ {0} (t)$$

Remark 1: In this paper, we use a TB matrix representation instead of a more standard Block Toeplitz (BT) matrix representation. The main reason is that it allows to obtain a structure of the harmonic equations similar to the one in the time domain (see for example (1)). This is more suitable for analysis and control design purposes. To obtain a BT structure as in [2], [13], one has to define $\cdot$ by $\cdot$ $( \cdots , X _ { - 1 } , X _ { 0 } , X _ { 1 } , \cdots )$ where $\cdot$ refers to the $\cdot$ phasors instead of $-$ . Obvioulsly, we can always switch from one representation to another by applying a permutation matrix.
