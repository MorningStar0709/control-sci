# 3. THE MAXIMUM A POSTERIORI FRAMEWORK

To solve Problem 1, this work considers a MAP framework, where $\zeta$ is treated as the observation and the Willems’ fundamental lemma characterization is used as prior information.

In detail, $\mathbf { z } ^ { 0 }$ is estimated by solving

$$\hat {\mathbf {z}} = \underset {\mathbf {z} ^ {0}} {\operatorname{argmax}} p \left(\mathbf {z} ^ {0} | \boldsymbol {\zeta}\right) = \underset {\mathbf {z} ^ {0}} {\operatorname{argmax}} p \left(\boldsymbol {\zeta} | \mathbf {z} ^ {0}\right) p \left(\mathbf {z} ^ {0}\right). \tag {5}$$

The conditional distribution of $\zeta$ given $\mathbf { z } ^ { 0 }$ is given by

$$\boldsymbol {\zeta} | \mathbf {z} ^ {0} \sim \mathcal {E} _ {f} \left(\Phi \mathbf {z} ^ {0}, \Sigma_ {\epsilon}\right).$$

The prior distribution of $\mathbf { z } ^ { 0 }$ can be obtained from Theorem 3 using the equation

$$\mathbf {z} ^ {0} = H \mathbf {g} - \Delta_ {H} \mathbf {g},$$

where g is treated as a hyperparameter. To quantify the uncertainty of $\Delta _ { H } \mathbf { g }$ given g, we follow a similar derivation as in Yin et al. (2023) by vectorizing $\Delta _ { H } \mathbf { ; }$

$$\Delta_ {H} \mathbf {g} = \left(\mathbf {g} ^ {\top} \otimes \mathbb {I} _ {n L}\right) \operatorname{vec} \left(\Delta_ {H}\right).$$

Let vec $\left( \Delta _ { H } \right) \sim \mathcal { E } _ { f } \left( \mathbf { 0 } _ { n L M } , \Sigma _ { d } \right)$ , where $\Sigma _ { d } \ \in \ \mathbb { S } _ { + } ^ { n L M }$ . For $i , j \in \mathbb { N } _ { [ 1 , M L ] }$ , define $i = : ( i _ { 1 } - 1 ) L + i _ { 2 } , j = : ( j _ { 1 } - 1 ) L + j _ { 2 }$ , where $i _ { 1 } ^ { \setminus } , j _ { 1 } \in \mathbb { N } _ { [ 1 , M ] }$ and $i _ { 2 } , j _ { 2 } \in \mathbb { N } _ { [ 1 , L ] }$ . Then, $\Sigma _ { d }$ is given by

$$\left(\Sigma_ {d}\right) _ {i, j} = \Sigma_ {\nu} ^ {d} (| t _ {i _ {1}} + i _ {2} - (t _ {j _ {1}} + j _ {2}) |), \tag {6}$$

where $\left( \Sigma _ { d } \right) _ { i , j }$ is the $( i , j )$ -th $n \times n \cdot$ -block element of $\Sigma _ { d }$

This leads to $\Delta _ { H } \mathbf { g } | \mathbf { g } \sim \mathcal { E } _ { f } \left( \mathbf { 0 } _ { n L } , \boldsymbol { \Sigma } _ { g } ( \mathbf { g } ) \right)$ , where

$$\Sigma_ {g} (\mathbf {g}) := \left(\mathbf {g} ^ {\top} \otimes \mathbb {I} _ {n L}\right) \Sigma_ {d} \left(\mathbf {g} \otimes \mathbb {I} _ {n L}\right) \in \mathbb {S} _ {+} ^ {n L}, \tag {7}$$

and thus the prior distribution: $\mathbf { z } ^ { 0 } | \mathbf { g } \sim \mathcal { E } _ { f } \left( H \mathbf { g } , \Sigma _ { g } ( \mathbf { g } ) \right)$ .
