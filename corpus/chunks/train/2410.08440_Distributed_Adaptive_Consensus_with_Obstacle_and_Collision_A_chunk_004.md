where $x _ { 0 } ^ { k } \in \mathbb { R }$ is the k-th component of the leader agent’s state, and $x _ { 0 } = [ x _ { 0 } ^ { 1 } , x _ { 0 } ^ { 2 } , . . . , x _ { 0 } ^ { n } ]$ is its state vector.

The unknown function $f _ { 0 } ( x _ { 0 } , t ) : [ 0 , \infty ) \times \mathbb { R } ^ { n } \to \mathbb { R }$ in the leader dynamics is considered to be piecewise continous in t and locally Lipschitz in $x _ { 0 }$ , with $f _ { i } ( 0 , t ) = 0$ for all $t \geq 0$ , and all $x _ { 0 } \in \mathbb { R } ^ { n }$ .

In the consensus problem, it is desired (see, e.g., [11]) that

$$\lim _ {t \to \infty} [ (x _ {i} ^ {k} - \psi_ {i}) - (x _ {0} ^ {k} - \psi_ {0}) ] = 0 \tag {4}\lim _ {t \rightarrow \infty} \left[\left(x _ {i} ^ {k} - \psi_ {i}\right) - \left(x _ {j} ^ {k} - \psi_ {j}\right)\right] = 0 \tag {5}$$

for each $i \in \mathcal N$ and for all $j \in \mathcal { N } , j \neq i$ , where $\psi _ { i } , \psi _ { j }$ and $\psi _ { 0 }$ denote the desired offsets for agents $i , j$ and $0 ,$ respectively. However, such a strong level of connectivity which requires communication among all agents is often not feasible in networked control systems due to practical constraints such as limited bandwidth, packet loss, and delays (e.g., see [12]).

In this paper, we seek a decentralized consensus policy where, in addition to its own state $x _ { i } ,$ each agent determines its input $u _ { i }$ based solely on the states of a limited number of other agents, which may or may not include the leader agent. Furthermore, each agent must also account for obstacle and collision avoidance.

To address collision avoidance between agents as well as obstacle avoidance, we employ potential functions as an extension of [10]. These potential functions guide the agents’ motions, ensuring that they avoid both collisions with each other and obstacles in their environment.

For collision avoidance between agent i and agent j positions, we define $m _ { i j }$ as follows:

$$
m _ {i j} := \left\{ \begin{array}{c c} 0 & \| x _ {i} ^ {1} - x _ {j} ^ {1} \| \geq \psi_ {i j} \\ \frac {\chi}{\| x _ {i} ^ {1} - x _ {j} ^ {1} \|} & \| x _ {i} ^ {1} - x _ {j} ^ {1} \| <   \psi_ {i j} \end{array} \right. \tag {6}
$$

Similarly, for collision avoidance between agent i and the leader agent 0, we define $m _ { i 0 }$ as follows:

$$
m _ {i 0} := \left\{ \begin{array}{c c} 0 & \| x _ {i} ^ {1} - x _ {0} ^ {1} \| \geq \psi_ {i 0} \\ \frac {\chi}{\| x _ {i} ^ {1} - x _ {0} ^ {1} \|} & \| x _ {i} ^ {1} - x _ {0} ^ {1} \| <   \psi_ {i 0} \end{array} \right. \tag {7}
$$
