Corollary 1 (Disturbance Robustness). Consider the proposed HC-FFTSD (12) in Theorem 1 under perturbation, $\mathbf { \bar { \boldsymbol { \delta } } } = ( \delta _ { 1 } ^ { \mathrm { T } } , \delta _ { 2 } ^ { \mathrm { T } } ) ^ { \mathrm { T } } , \delta _ { 1 } , \delta _ { 2 } \in \mathbb { R } ^ { n }$ , and δ is bounded as $\left| \left| \delta \right| \right| \leq \bar { \delta } .$ . Thereafter, the differentiator under perturbation is as

$$
\begin{array}{l} \begin{array}{r l} \dot {e} _ {1} & = - k _ {1} \phi_ {1} (e _ {1}) + e _ {2} + \delta_ {1}, \\ \dot {\cdot} & = k _ {1} \phi_ {1} (e _ {1}) + \delta_ {1}. \end{array} \tag {32} \\ \dot {e} _ {2} = - k _ {2} \phi_ {2} (e _ {1}) + \delta_ {2}. \\ \end{array}
$$

When $\gamma _ { 1 }$ in (31) fulfills $\gamma _ { 1 } ~ \geq ~ \lambda _ { \operatorname* { m a x } } \left\{ \mathcal { P } \right\} / \lambda _ { \operatorname* { m i n } } \left\{ \mathcal { P } \right\}$ , (32) is Practically Finite-Time Stable (PFTS).

Proof. Consider the Lyapunov stability analysis in Theorem 1. With the Lyapunov-candidate defined by (20) and the expression of the differentiator under perturbation in (32), we express the time derivative of (20) as follows:

$$\dot {V} \leq - \gamma_ {1} V - \gamma_ {2} V ^ {\frac {1}{p}} + 2 \lambda_ {\max} \left\{\mathcal {P} \right\} \bar {\delta} | | \zeta | |. \tag {33}$$

By applying Cauchy-Schwarz inequality and (21), from (33), we obtain,

$$
\begin{array}{l} \dot {V} \leq - \gamma_ {1} V - \gamma_ {2} V ^ {\frac {1}{p}} + \lambda_ {\max} \left\{\mathcal {P} \right\} \left| | \zeta | \right| ^ {2} + \lambda_ {\max} \left\{\mathcal {P} \right\} \bar {\delta} ^ {2}   (  \lambda_ {\max} \left\{\mathcal {P} \right\}) ^ {- 1} = 0. (34) \\ \leq - \left(\gamma_ {1} - \frac {\lambda_ {\max} \{\mathcal {P} \}}{\lambda_ {\min} \{\mathcal {P} \}}\right) V - \gamma_ {2} V ^ {\frac {1}{p}} + \lambda_ {\max} \{\mathcal {P} \} \bar {\delta} ^ {2}. (34) \\ \end{array}
$$

Therefore, according to Lemma 3, with inequality (34), we conclude that the system (32), which is the differentiator (12) under disturbance $\delta ,$ is practical finite time stable at the origin. □
