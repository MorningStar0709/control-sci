# Approximate Adaptive Minimum Variance Tracking and Regulation

The case $d = 0$ Taking into account (11.94), one can consider approximating $\hat { y } ( t ) , \hat { y } ( t - 1 ) , \ldots \ b \mathbf { y } \ y ^ { * } ( t ) , y ^ { * } ( t - 1 ) , \ldots$ . both in the controller equation and in the observation vector used in the PAA of (11.88) through (11.90). This corresponds to the use of a priori predictions instead of a posteriori predictions in the observation vector. In this case the adjustable controller is given by:

$$\hat {\theta} _ {C} ^ {T} (t) \phi_ {C} (t) = \hat {S} (t) u (t) + \hat {R} (t) y (t) - \hat {C} ^ {*} (t) y ^ {*} (t) = y ^ {*} (t + 1) \tag {11.99}$$

where $\hat { \theta } _ { c } ( t )$ is given by (11.85) and

$$\phi_ {C} ^ {T} (t) = [ u (t), \dots , u (t - n _ {S}), y (t), \dots , y (t - n _ {R}), - y ^ {*} (t), \dots , - y ^ {*} (t - n _ {C} + 1) ] \tag {11.100}$$

and the PAA is given by:

$$\hat {\theta} _ {C} (t + 1) = \hat {\theta} _ {C} (t) + F (t) \phi_ {C} (t) \varepsilon (t + 1) \tag {11.101}F (t + 1) ^ {- 1} = F (t) ^ {- 1} + \lambda_ {2} (t) \phi_ {C} (t) \phi_ {C} ^ {T} (t)0 < \lambda_ {2} (t) < 2; F (0) > 0 \tag {11.102}\varepsilon (t + 1) = \frac {y (t + 1) - y ^ {*} (t + 1)}{1 + \phi_ {C} ^ {T} (t) F (t) \phi_ {C} (t)} \tag {11.103}$$

The case $d > 0$ One considers a $d + 1$ steps ahead adaptive predictor of the form (see Sect. 6.3):

$$\hat {y} ^ {0} (t + d + 1) = - \hat {C} ^ {*} (t) \hat {y} ^ {0} (t + d) + \hat {F} (t) y (t) + \hat {G} (t) u (t) \tag {11.104}$$

The control u(t) is obtained by forcing:

$$\hat {y} ^ {0} (t + d + 1) = y ^ {*} (t + d + 1) \tag {11.105}$$

which yields:

$$\theta_ {C} ^ {T} (t) \phi_ {C} (t) = \hat {R} (t) y (t) + \hat {S} (t) u (t) - C ^ {*} (t) y ^ {*} (t + d) = y ^ {*} (t + d + 1) \tag {11.106}$$

with:

$$\theta_ {C} ^ {T} (t) = [ \hat {s} _ {0} (t), \dots , \hat {s} _ {n _ {S}} (t), \hat {r} _ {0} (t), \dots , \hat {r} _ {n _ {R}} (t), \hat {c} _ {1} (t), \dots , \hat {c} _ {n _ {C}} (t) ] \tag {11.107}\phi_ {C} ^ {T} (t) = [ u (t), \ldots , u (t - n _ {S}), y (t), \ldots , y (t - n _ {F}),\left. - y ^ {*} (t + d), \dots , - y ^ {*} (t + d - n _ {C} + 1) \right] \tag {11.108}$$

In the above equations, $\hat { F } ( t )$ and $\hat { G } ( t )$ have been replaced by $\hat { R } ( t )$ and $\hat { \boldsymbol { S } } ( t )$ similar to the known parameter case and $\hat { y } ( t + d ) , \hat { y } ^ { 0 } ( t + d - 1 ) , \dots \ b \ y \ y ^ { * } ( t + d ) , y ^ { * } ( t +$ $d - 1 ) , \ldots$ . taking into account (11.105). The used PAA is given by:
