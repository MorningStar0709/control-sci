# Notation

A cone $\mathcal { P } \subseteq \mathbb { R } ^ { n }$ is said to be proper if it is convex, closed, pointed and has a non-empty interior. The dual cone of a proper cone P is also proper, and given by

$$\mathcal {P} ^ {*} = \left\{\lambda : \lambda^ {\top} x \geq 0 \text { for all } x \in \mathcal {P} \right\}.$$

All further non-standard notation will be introduced as it is used.
