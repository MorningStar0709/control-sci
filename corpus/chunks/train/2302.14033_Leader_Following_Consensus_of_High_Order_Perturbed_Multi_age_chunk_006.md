# III. PROBLEM STATEMENT

We consider a continues-time MAS with topology represented by a directed graph $\mathcal { G } = ( \nu , \mathcal { E } )$ where the agents have the dynamics described by

$$\dot {x} _ {i} (t) = A x _ {i} (t) + B u _ {i} (t) + B \omega_ {i} (t) \tag {8}$$

where $x _ { i } \in \mathbb { R } ^ { n }$ represents the state of the i-th agent and $u _ { i } ( t ) \in \mathbb { R }$ is a control protocol which needs to be designed. The partial function of $\omega _ { i } ( t ) \in \mathbb { R }$ , denotes unknown exogenous perturbations. And also $A \in \mathbb { R } ^ { n \times n }$ and $B \in \mathbb { R } ^ { n }$ have the following expression:

$$
A = \left[ \begin{array}{c c c c c} 0 & 1 & 0 & \dots & 0 \\ 0 & 0 & 1 & \ddots & 0 \\ \dots & \dots & \dots & \ddots & \vdots \\ 0 & 0 & 0 & \ddots & 1 \\ - a _ {1} & - a _ {2} & \dots & \dots & - a _ {n} \end{array} \right] \quad B = \left[ \begin{array}{c} 0 \\ \vdots \\ 0 \\ b \end{array} \right] \tag {9}
$$

with $b > 0$ and $a _ { 1 } , a _ { 2 } , \ldots , a _ { n }$ are positive constants.

Assumption 1. We assume that for the system (8), the unknown perturbations are bounded:

$$\left| \omega_ {i} (t) \right| \leq \Gamma_ {i} \leq \Gamma \quad \Gamma = \max _ {\mathrm{i} \in \boldsymbol {\mathcal {V}}} \left\{\Gamma_ {i} \right\} \leq \infty , \quad \forall i \in \boldsymbol {\mathcal {V}} \tag {10}$$

where $\Gamma _ { i }$ is a positive known constant.

Our objective is to design a continues-time distributed consensus protocol $u _ { i } ( t )$ for agents as in Eq. (8) to enable each agent to track the time-varying virtual leader despite delayed communications among agents, wherein the reference dynamic is described as

$$\dot {x} _ {0} = A x _ {0} (t) \tag {11}$$
