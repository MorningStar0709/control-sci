# D.1. Proof of Lemma 7

Given a stabilizable system $\Theta = ( A , B ) ^ { \intercal }$ , and a stabilizing linear feedback controller K, we can find the LQR cost as follows

$$J (\Theta , K) = \lim _ {T \rightarrow \infty} \frac {1}{T} \mathbb {E} \left[ \sum_ {t = 1} ^ {T} x _ {t} ^ {\intercal} Q x _ {t} + u _ {t} ^ {\intercal} R u _ {t} \right], \tag {31}= \lim _ {T \rightarrow \infty} \frac {1}{T} \mathbb {E} \left[ \sum_ {t = 1} ^ {T} \operatorname{tr} \left(\left(Q + K ^ {\intercal} R K\right) x _ {t} x _ {t} ^ {\intercal}\right)\right], \tag {32}= \lim _ {T \rightarrow \infty} \operatorname{tr} \left((Q + K ^ {\intercal} R K) \frac {1}{T} \sum_ {t = 1} ^ {T} \mathbb {E} \left[ x _ {t} x _ {t} ^ {\intercal} \right]\right), \tag {33}= \operatorname{tr} \left(\left(Q + K ^ {\intercal} R K\right) \Sigma (\Theta , K)\right) \tag {34}$$

where $\begin{array} { r } { \Sigma ( \Theta , K ) : = \operatorname* { l i m } _ { T \to \infty } \frac { 1 } { T } \sum _ { t = 1 } ^ { T } \mathbb { E } \left[ x _ { t } x _ { t } ^ { \intercal } \right] } \end{array}$ is the stationary state covariance of the closed-loop system. In (32), we used the feedback control policy relation $u _ { t } = K x _ { t }$ and trace trick for inner products of vectors. Note that the closed-loop system evolves as

$$x _ {t + 1} = (A + B K) x _ {t} + w _ {t}. \tag {35}$$

The covariance of the state at time t can be written as a recursive relation

$$\mathbb {E} \left[ x _ {t + 1} x _ {t + 1} ^ {\intercal} \right] = \mathbb {E} \left[ ((A + B K) x _ {t} + w _ {t}) ((A + B K) x _ {t} + w _ {t}) ^ {\intercal} \right] \tag {36}= (A + B K) \mathbb {E} \left[ x _ {t} x _ {t} ^ {\intercal} \right] (A + B K) ^ {\intercal} + \mathbb {E} \left[ w _ {t} w _ {t} ^ {\intercal} \right] \tag {37}= (A + B K) \mathbb {E} \left[ x _ {t} x _ {t} ^ {\intercal} \right] (A + B K) ^ {\intercal} + \sigma_ {w} ^ {2} I \tag {38}$$

where (37) is because $\mathbb { E } \left[ w _ { t } \right] = 0$ and $w _ { t }$ and $x _ { t }$ are independent. Since $\rho ( A + B K ) < 1$ , the above iteration converges to a finite fixed-point. Furthermore, we have the following relation

$$\frac {1}{T} \sum_ {t = 1} ^ {T} \mathbb {E} \left[ x _ {t + 1} x _ {t + 1} ^ {\intercal} \right] = (A + B K) \frac {1}{T} \sum_ {t = 1} ^ {T} \mathbb {E} \left[ x _ {t} x _ {t} ^ {\intercal} \right] (A + B K) ^ {\intercal} + \sigma_ {w} ^ {2} I \tag {39}$$
