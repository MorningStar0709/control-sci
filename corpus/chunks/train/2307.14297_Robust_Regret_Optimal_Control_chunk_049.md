# A. Discrete-Time Spectral Factorization

This appendix summarizes existing results for discrete-time spectral factorizations. Good overviews can be found in Section 8.3 of [37] or Chapter 13 of [16]. The specific construction given here for the spectral factorization, stated as Lemma 4 below, is based on results in [38].

Consider a discrete-time, linear time-invariant system $\hat { P }$ with the following state-space representation:

$$\hat {x} _ {t + 1} = \hat {A} \hat {x} _ {t} + \hat {B} d _ {t} \tag {35}\hat {e} _ {t} = \hat {C} \hat {x} _ {t} + \hat {D} d _ {t}.$$

The input/output dimensions of $\hat { P }$ are $n _ { \hat { e } } \times n _ { d }$ . We assume that $\hat { P }$ is stable although not necessarily causal. An input $d \in \ell _ { 2 }$ generates a state $\hat { x } \in \ell _ { 2 }$ and output $\hat { e } \in \ell _ { 2 }$ . The boundary conditions for the state are $\hat { x } _ { - \infty } = 0$ and $\hat { x } _ { \infty } = 0 . { } ^ { \dagger } { } ^ { \dagger }$ We’ll denote $\hat { e } = \hat { P } d$ as the output generated by the input d $\in \ell _ { 2 }$ subject to these boundary conditions. Let $\hat { J } : = \langle \hat { e } , \hat { e } \rangle$ be the resulting cost.

The appendix is rewrites $\hat { J }$ in a more useful form using two ingredients. The first ingredient is the para-Hermitian conjugate of ${ \hat { P } } ,$ denoted $\hat { P } ^ { \sim }$ , and defined as follows:

$$\hat {A} ^ {\top} \bar {x} _ {t + 1} = \bar {x} _ {t} - \hat {C} ^ {\top} \bar {e} _ {t} \tag {36}\bar {d} _ {t} = \hat {B} ^ {\top} \bar {x} _ {t + 1} + \hat {D} ^ {\top} \bar {e} _ {t}.$$

The input/output dimensions for $\hat { P } ^ { \sim }$ are $n _ { d } \times n _ { \hat { e } }$ . If $\bar { e } _ { t } \in \ell _ { 2 }$ then the state of $\hat { P } ^ { \sim }$ satisfies the boundary conditions $\bar { x } _ { - \infty } = 0$ and $\bar { x } _ { \infty } = 0$ . The para-Hermitian conjugate $\hat { P } ^ { \sim }$ also satisfies the adjoint property stated in the next lemma.
