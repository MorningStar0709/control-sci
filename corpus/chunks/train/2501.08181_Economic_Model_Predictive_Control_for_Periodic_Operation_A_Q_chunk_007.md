# III. ECONOMIC MPC FOR CHANGINGPERIODIC OPERATION

First, we summarize the MPC control formulation proposed in [18] that addresses the aforementioned control objective. To keep notation compact, we write $z : = ( x , \mathbf { u } ) \in$ $\mathbb { R } ^ { \check { n } + m N }$ when designating the pair composed of a given initial state x and a sequence of N control inputs u.

One of the main building blocks of our E-MPC control scheme is a MPC controller which tracks periodic references [18]. To this aim, we introduce an artificial periodic trajectory $\zeta = ( \xi , \nu ) \in \mathbb { R } ^ { n + m T }$ as a decision variable: the trajectory is determined by an initial state ξ and a sequence $\pmb { \nu } : = [ \nu _ { 0 } , \dots , \nu _ { T - 1 } ] \in \mathbb { R } ^ { m T }$ of $T \geq N$ control inputs $\nu .$ The deviation between the predicted and artificial trajectories is penalized through the stage cost $S : \mathbb { R } ^ { n + m N } \times \mathbb { R } ^ { n + m T } \to \mathbb { R }$ , defined as

$$S (\boldsymbol {z}, \boldsymbol {\zeta}) = \sum_ {i = 0} ^ {N - 1} \ell^ {s} (x _ {i} - \xi_ {i}, u _ {i} - \nu_ {i}), \tag {5}$$

where $\ell ^ { s } : \mathbb { R } ^ { n } \times \mathbb { R } ^ { m } $ R is defined as

$$\ell^ {s} (v, w) = \| v \| _ {Q} ^ {2} + \| w \| _ {R} ^ {2},$$

and $Q , R$ are positive definite matrices. The economic cost of the artificial trajectory $\zeta$ along the horizon T is evaluated by the offset cost function $O _ { k } \colon \bar { \mathbb { R } } ^ { n + m T } $ R, defined as:

$$O _ {k} (\boldsymbol {\zeta}, p) = \sum_ {j = 0} ^ {T - 1} \ell_ {k + j} ^ {e} (\xi_ {j}, \nu_ {j}, p), \tag {6}$$

where $\ell _ { k + j } ^ { e }$ is as in (4a). Combining (5) and (6), the cost function of the proposed single-layer MPC problem, $V _ { k } :$ : $\mathbb { R } ^ { n + m N } \times \mathbb { R } ^ { n + \hat { m } T } \stackrel { \cdot } {  } \mathbb { R }$ , is defined as:

$$V _ {k} (\boldsymbol {z}, \boldsymbol {\zeta}, p) := S (\boldsymbol {z}, \boldsymbol {\zeta}) + O _ {k} (\boldsymbol {\zeta}, p). \tag {7}$$

Then, for a given state x and parameter p at time $k ,$ the single-layer MPC control problem is formulated as:

$$\mathbb {P} _ {k} (x, p): \min _ {\boldsymbol {z}, \boldsymbol {\zeta}} V _ {k} (\boldsymbol {z}, \boldsymbol {\zeta}, p) \tag {8a}$$
