# 2.1 Network Model

We model such networked dynamical system in discrete time setting with a graph $\mathcal { G }$ comprising a node set V representing $| \nu | = N$ subsystems and edge set $\mathcal { E } \subset \mathcal { V } \times \mathcal { V }$ representing a set of $| { \mathcal { E } } | = E$ communication links amongst the subsystems. The incidence matrix encoding the edge set information is denoted by $\mathcal { T } \in \mathbb { R } ^ { N \times E }$ . We denote by $\mathcal { N } _ { i } = \{ j \in \mathcal { V } : ( j , i ) \in \mathcal { E } \}$ , the neighbor set of agent i, whose states are available to agent i through E. The degree of node $i \in \mathcal V$ is denoted as $d _ { i } : = | { \mathcal { N } } _ { i } |$ . The set of inclusive neighbors of agent i is denoted by $\mathcal { I } _ { i } : = \{ i \} \cup \mathcal { N } _ { i }$ . Let $\mathbf { d } : = \operatorname* { m a x } \{ d _ { i } \mid i \in \mathcal { V } \}$ . We associate with each node $i \in \mathcal V .$ , a state $x _ { i } ( t ) \in  { \mathbb { R } }$ at time $t \in \mathbb { N }$ . Each subsystem $\pmb { \Sigma _ { \mathbf { i } } }$ corresponding to node $i \in \mathcal V$ updates its own states by interacting with its neighbors as

$$x _ {i} (t + 1) = a _ {i} x _ {i} (t) + b \sum_ {j \in \mathcal {N} _ {i}} (u _ {i} (t) - u _ {j} (t)) + w _ {i} (t), \tag {1}$$

where $a _ { i } \in ( 0 , 1 ) , b > 0$ . The additive disturbance $w _ { i } ( t ) \in \mathbb { R }$ affecting the node $i \in \nu$ is adversarial in nature. The control input between two nodes $( i , j ) \in \mathcal { E }$ is such that what is drawn from subsystem $j$ is added to subsystem i and hence it only affects two nodes. Note that the dynamics of each node in the network is coupled with the other nodes only through their control inputs. The concatenated states of all nodes in (1), the corresponding control inputs along the edges and the adversarial disturbances acting on each node are denoted as

$$
x (t) = \left[ \begin{array}{c c c} x _ {1} (t) & \dots & x _ {N} (t) \end{array} \right] ^ {\top} \in \mathbb {R} ^ {N}, \tag {2}
u (t) = \left\{u _ {i} (t) - u _ {j} (t) \right\} _ {(i, j) \in \mathcal {E}} \in \mathbb {R} ^ {E}, \text { and } \tag {3}w (t) = \left[ w _ {1} (t) \quad \dots \quad w _ {N} (t) \right] ^ {\top} \in \mathbb {R} ^ {N} \tag {4}$$

respectively. Hence, the system described by (1) can be equivalently written compactly as

$$x _ {t + 1} = A x _ {t} + B u _ {t} + w _ {t}, \tag {5}$$

with $A \in \mathbb { R } ^ { N \times N }$ being symmetric and Schur stable, and $B = b \mathcal { T }$ , with $\mathcal { T } \in \mathbb { R } ^ { N \times E }$ being the incidence matrix of the underlying graph. Note that, $B B ^ { \top } = b ^ { 2 } \mathcal { T } \mathcal { T } ^ { \top } = b ^ { 2 } \mathbf { L }$ , where L denotes the Laplacian matrix associated with the graph $\mathcal { G }$ .
