# Example 5.4

Use your computer to get the steady state response of the following system

$$G (s) = \frac {5 0}{(s + 0 . 0 5 + j 0 . 4) (s + 0 . 0 5 - j 0 . 4)}$$

to a sinusoidal input of the form:

$$X (t) = \sin (\omega t) u (t)$$

for ω = 1.25 (the units of ω are radians per second).
