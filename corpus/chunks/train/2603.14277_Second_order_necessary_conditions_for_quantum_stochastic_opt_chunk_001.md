$$\Psi (z) := C (z) + A (J z), z \in \mathscr {H},$$

where $C ( z )$ and $A ( z )$ are the creation operator and the annihilation operator, respectively. Additionally, $J$ is the complex conjugation operator on $L ^ { 2 } ( \mathbb { R } ^ { + } , d s ) \ ( \mathrm { i . e . , } \ J f = \overline { { f } } )$ . $$\{\Psi (v): v \in L ^ {2} (\mathbb {R} ^ {+}, d s) \text { and ess supp } v \subset [ 0, t ] \},$$
