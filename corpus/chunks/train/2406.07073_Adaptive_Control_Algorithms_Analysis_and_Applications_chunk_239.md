# 5.3.4 Recursive Maximum Likelihood (RML)

This algorithm is also dedicated to ARMAX type plant + disturbance model. It belongs to the class of RPEM. It can be interpreted as a modification of the ELS in order to eliminate (locally) the positive real condition for convergence.

Consider the equation of the a posteriori prediction error for the ELS but for a fixed value of the estimated parameter vector $\hat { \theta }$ (which is used in Theorem 4.1). From (5.28) one obtains:

$$\nu (t + 1, \hat {\theta}) = \frac {1}{C (q ^ {- 1})} [ \theta - \hat {\theta} ] ^ {T} \phi (t, \hat {\theta}) + e (t + 1) \tag {5.34}$$

In the vector $\hat { \theta } .$ , an estimation of the coefficients of the polynomials $C ( q ^ { - 1 } )$ is available. This estimated polynomial will be denoted by $\hat { C } ( \overset { \cdot } { q } ^ { - 1 } , \overset { \cdot } { \theta } )$ ). Introduce the filtered regressor vector:

$$\phi_ {f} (t, \hat {\theta}) = \frac {1}{\hat {C} (q ^ {- 1} , \hat {\theta})} \phi (t, \hat {\theta}) \tag {5.35}$$

Taking into account that $\hat { \theta }$ is fixed in (5.34), this equation can be rewritten as:

$$\nu (t + 1, \hat {\theta}) = \frac {\hat {C} (q ^ {- 1} , \hat {\theta})}{C (q ^ {- 1})} [ \theta - \hat {\theta} ] \phi_ {f} (t, \hat {\theta}) + e (t + 1) \tag {5.36}$$

In other words, the positive real condition will be relaxed in particular around the correct value $\hat { \theta } = \theta$ . This suggests to use of $\phi _ { f } ( t )$ as observation vector. Therefore, in the RML, with respect to ELS, one replaces the observation vector $\phi ( t )$ by its filtered version given in (5.35). One filters the regressor through the current estimate of the polynomial $C ( q ^ { - 1 } )$ to generate the observation vector (i.e. one filters the components of $\phi ( t ) )$ . More specifically, one has: (for $n _ { A } = n _ { B } = n _ { C } )$

$$
\phi_ {f} (t) = \left[ \begin{array}{c c c} \hat {c} & 0 & 0 \\ 0 & \hat {c} & 0 \\ 0 & 0 & \hat {c} \end{array} \right] \phi_ {f} (t - 1) + \left[ \begin{array}{c c c} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{array} \right] \phi (t)
$$

where:

$$
\hat {c} = \left[ \begin{array}{c c c c} \hat {c} _ {1} (t) & \ldots & \ldots & \hat {c} _ {n} (t) \\ 0 & & & \\ 0 & & I _ {n - 1} & \\ 0 & & & \end{array} \right]
$$
