# (1) Principles of Quadcopter Operation

A quadrotor consists of four symmetrically distributed rotors, with typical configurations being the $\mathbf { \epsilon } ^ { \ast } \mathbf { X } ^ { \ast }$ form and the ‘+’ form. The total thrust of the quadrotor is the sum of all four rotor thrusts:

$$F _ {t o t a l} = F _ {1} + F _ {2} + F _ {3} + F _ {4} = k _ {t} \big (\omega_ {1} ^ {2} + \omega_ {2} ^ {2} + \omega_ {3} ^ {2} + \omega_ {4} ^ {2} \big) (1)$$

where $F _ { i }$ is the thrust force generated by the i-th rotor $( N ) , \omega _ { i }$ is the angular velocity of the i-th motor (rad/s), and $k _ { t }$ is the thrust coefficient (N·s²/rad²). In hover or steady flight, the total reactive torque is balanced:

$$\tau_ {t o t a l} = k _ {m} (\omega_ {1} ^ {2} + \omega_ {3} ^ {2}) - k _ {m} (\omega_ {2} ^ {2} + \omega_ {4} ^ {2}) = 0 \tag {2}$$

Where $k _ { m }$ is the torque coefficient (N·m·s²/rad²). The torque direction follows the right-hand rule.

Figure 1 Principles of Quadcopter Operation

A quadrotor achieves six-degree-of-freedom motion control by differentially adjusting the rotational speeds of its four motors. As illustrated in Fig. 1, the operating principles demonstrate how motors control altitude and attitude through varying speeds. Before proceeding, we must define the coordinate frames.

Earth-fixed Inertial Frame $\mathcal { F } _ { E } = \{ O _ { E } ; x _ { E } , y _ { E } , z _ { E } \} \colon \boldsymbol { \mathrm { A } }$ fixed reference frame attached to the ground, used to describe the absolute position and attitude of the vehicle. We adopt the standard ENU (East-North-Up) convention, where $x _ { E }$ points east, $y _ { E }$ points north, $\mathbf { z } _ { E }$ and points vertically upward.   
Body-fixed Frame $\mathcal { F } _ { \mathcal { B } } = \{ O _ { B } ; x _ { B } , y _ { B } , z _ { B } \}$ :A moving reference frame attached to the quadrotor's center of mass, where $x _ { B }$ points toward the nose (rotor 1 direction), $\scriptstyle { y _ { B } }$ points to the left side (rotor 4 direction), and ${ \pmb z } _ { B }$ points vertically upward perpendicular to the body plane. The vehicle's linear velocity and angular velocity in the body frame are represented by vectors $\pmb { v _ { b } } = [ u , v , w ] ^ { T }$ and $\begin{array} { r } { \pmb { \omega _ { b } } = [ p , q , r ] ^ { T } } \end{array}$ , respectively.
