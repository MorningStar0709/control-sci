# 2 Problem Statement

By following [5, 19], we consider CPSs that can be modeled as DT LTI dynamical systems

$$
\begin{array}{l} x (k + 1) = A x (k) \\ (1) = C _ {0} (1) + \dots (1) \end{array} \tag {1}
y (k) = C x (k) + a (k)$$

where $\ b { x } ( k ) \in \mathbb { R } ^ { n }$ is the state, $\ b { y } ( k ) \in \mathbb { R } ^ { p }$ is the measurement vector, $a ( k ) \in$ $\mathbb { R } ^ { p }$ is the attack vector, $A \in \mathbb { R } ^ { n , n }$ and $C \in \mathbb { R } ^ { p , n }$ . We assume that each sensor i takes a measurement $y _ { i } ( k )$ ; if $a _ { i } ( k ) \neq 0$ , sensor i is under attack. We assume that $a ( k )$ is sparse, i.e., few sensors are under attack at each k. Since the presence of a known input does not impact on the formulation of the problem, see [19], for simplicity of notation we consider a zero-input model. Moreover, as in [5, 19], we consider a noise-free model for our theoretical analysis, while we envisage measurement noise in some numerical experiments.

The SSE problem is as follows.

Problem 1. For some $\tau \leq n$ and $k \geq \tau - 1$ given $A , C$ and $y = ( y ( k -$ $\tau + 1 ) ^ { \top } , \ldots , y ( k ) ^ { \top } ) ^ { \top } \in \mathbb { R } ^ { p \tau }$ , estimate the τ -delayed state $x ( k - \tau + 1 )$ in the presence of sparse sensor attacks.

Let us denote $\widetilde { \boldsymbol { a } } = ( a ( k - \tau + 1 ) ^ { \top } , \ldots , a ( k ) ^ { \top } ) ^ { \top } \in \mathbb { R } ^ { p \tau }$ and $\widetilde { \boldsymbol { x } } = \boldsymbol { x } ( k - \tau + 1 ) \in$

Rn, while $I \in \{ 0 , 1 \} ^ { p \tau , p \tau }$ is the identity matrix. From (1), we have

$$
y = \left( \begin{array}{c c} \mathcal {O} & I \end{array} \right) \binom{\widetilde {x}}{\widetilde {a}}, \text {   where   } \mathcal {O} = \left( \begin{array}{c} C \\ C A \\ \vdots \\ C A ^ {\tau - 1} \end{array} \right) \in \mathbb {R} ^ {p \tau , n} \tag {2}
$$

If $\tau = n$ , O is the observability matrix of the attack-free system. We assume that the attack-free system is observable, i.e., rank $( \mathcal { O } ) \ : = \ : n$ . In principle, we can estimate $( \widetilde x , \widetilde a )$ by solving

$$
y = \left( \begin{array}{c c} \mathcal {O} & I \end{array} \right) \binom{x}{a} \tag {3}
$$
