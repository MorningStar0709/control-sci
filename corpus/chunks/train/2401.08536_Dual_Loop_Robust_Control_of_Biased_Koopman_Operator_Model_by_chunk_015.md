# IV. DUAL-LOOP ROBUST CONTROL

Consider the discrete-time Koopman model matrices from perturbed data points (40). It is assumed that $( A , B _ { 2 } )$ are stabilizable, and $( C _ { 2 } , A )$ are detectable, such that there exist stabilizing controllers for the system. The $H _ { \infty }$ control usually produces conservative performance since it aims to handle the worst-case model uncertainty. However, the dual-loop control can relieve the conservativeness by introducing the compensation block [21].
