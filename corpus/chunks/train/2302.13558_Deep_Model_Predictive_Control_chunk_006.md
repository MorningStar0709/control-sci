In order to design the online reference tracking MPC, we first choose an optimization horizon $N \in \mathbb { Z } _ { + }$ and positive definite matrices $Q , R \succ { \bf 0 }$ , which can be different from those chosen for the reference governor. Let

$$c _ {\mathrm{s}} (x _ {t + i | t}, u _ {t + i | t}) := \left\| x _ {t + i | t} - x _ {t + i} ^ {r} \right\| _ {Q} ^ {2} + \left\| u _ {t + i | t} - u _ {t + i} ^ {r} \right\| _ {R} ^ {2}$$

be the cost per stage at time $t + i$ predicted at time t and let $c _ { \mathrm { f } } ( x ) : = x ^ { \top } Q _ { f } x$ be the terminal cost with $Q _ { f } \succ 0 .$ . The terminal cost $c _ { \mathrm { f } }$ is treated as a local control Lyapunov function within a terminal set

$$\mathcal {X} _ {f} := \{x \in \mathbb {R} ^ {d} \mid c _ {\mathrm{f}} (x) \leqslant \alpha ; \alpha > 0 \} \tag {7}$$

as in [41] by making the following assumption:

Assumption 1. There exists a control $u ^ { \prime } \in \mathbb { U } ^ { \prime }$ such that the following holds

$$c _ {\mathrm{f}} \left(\bar {f} (x, u ^ {\prime})\right) - c _ {\mathrm{f}} (x) \leqslant - c _ {\mathrm{s}} (x, u ^ {\prime}) \text { for every } x \in \mathcal {X} _ {f}. \tag {8}$$

The above assumption is standard in the literature. Refer to [41, §4] for more details with a minor modification, which we made here for simplicity. Let us define

$$V _ {m} (x _ {t | t}, (u _ {t + i | t}) _ {i = 0} ^ {N - 1}) := c _ {\mathrm{f}} (x _ {t + N | t}) + \sum_ {i = 0} ^ {N - 1} c _ {\mathrm{s}} (x _ {t + i | t}, u _ {t + i | t}). \tag {9}$$

The online reference tracking MPC minimizes (9) at each time instant t under the following constraints:

$$x _ {t \mid t} = x _ {t} \tag {10}u _ {t \mid t} + u _ {t} ^ {a} \in \mathbb {U} \tag {11}x _ {t + i + 1 \mid t} = \bar {f} (x _ {t + i \mid t}, u _ {t + i \mid t}) \text { for } i = 0, \dots , N - 1 \tag {12}u _ {t + i \mid t} \in \mathbb {U} ^ {\prime} \text { for } i = 1, \dots , N - 1. \tag {13}$$

Notice that the constraint (11) is different from the constraints present in the tube-based MPC formulation [41]. We define the underlying optimal control problem as follows:

$$V _ {m} (x _ {t}) := \min _ {(u _ {t + i | t}) _ {i = 0} ^ {N - 1}} \{(9) \mid (1 0), (1 1), (1 2), (1 3) \}. \tag {14}$$

Let the optimizer of the above problem be $( u _ { t + i | t } ^ { * } ) _ { i = 0 } ^ { N - 1 }$ . Then the optimal cost will be $V _ { m } ( x _ { t } ) : =$ $V _ { m } ( x _ { t } , ( u _ { t + i | t } ^ { * } ) _ { i = 0 } ^ { N - 1 } )$ . The first control $\boldsymbol { u } _ { t \mid t } ^ { * }$ is called the MPC component $u _ { t } ^ { m }$ and is applied along with $u _ { t } ^ { a }$ to the system at time t.
