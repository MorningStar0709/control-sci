# 2 The game

Consider the following controlled linear mean-field type stochastic differential equation (SDE):

$$
\left\{ \begin{array}{l} d x (t) = \left\{A _ {0} (t) x (t) + \bar {A} _ {0} (t) \mathbb {E} [ x (t) ] + B _ {0} (t) u (t) + \bar {B} _ {0} (t) \mathbb {E} [ u (t) ] \right\} d t \\ \quad + \sum_ {j = 1} ^ {r} \left\{A _ {j} (t) x (t) + \bar {A} _ {j} (t) \mathbb {E} [ x (t) ] + B _ {j} (t) u (t) + \bar {B} _ {j} (t) \mathbb {E} [ u (t) ] \right\} d w _ {j} (t) \\ x (s) = x _ {s} \in \mathbb {R} ^ {n} \end{array} \right. \tag {1}
$$

$t \in [ s , \ T ]$ . In (1), $\{ \boldsymbol { w } ( t ) \} _ { t \geq 0 } ~ ( \boldsymbol { w } ( t ) = ( \boldsymbol { w } _ { 1 } ( t ) , . . . , \boldsymbol { w } _ { r } ( t ) ) ^ { \top } )$ is an r-dimensional standard Wiener process defined on a given probability space $( \Omega , { \mathcal { F } } , { \mathcal { P } } )$ . Let $\mathcal { F } _ { t } , \ t \ \geq \ 0$ denotes the family of $\sigma \mathrm { - }$ algebras $\mathcal { F } _ { t } = \sigma ( w ( s ) , 0 \leq s \leq t )$ . We denote by $\mathbb { E } [ \cdot ]$ the mathematical expectation. We assume that: $\begin{array} { r } { t  A _ { k } ( t ) : [ 0 , T ]  \mathbb { R } ^ { n \times n } , t  \bar { A } _ { k } ( t ) : [ 0 , T ]  \mathbb { R } ^ { n \times n } , t  B _ { k } ( t ) : [ 0 , T ]  \mathbb { R } ^ { n \times m } ; } \end{array}$ $t  \bar { B } _ { k } ( t ) : [ 0 , T ]  \mathbb { R } ^ { n \times m } , 0 \leq k \leq r ;$ , are continuous matrix valued functions. We define the class $\mathcal { U } _ { a d }$ of the admissible controls consisting of all stochastic processes, $u = \{ u ( t ) \} _ { t \geq s } \in L _ { w } ^ { 2 } ( [ s , T ] ; \mathbb { R } ^ { m } )$ . In this work, the space $L _ { w } ^ { 2 } ( \mathcal { T } ; \mathbb { R } ^ { d } )$ stands for the vector space of stochastic processes v : I × Ω → Rd which are non-anticipative with respect to the filtration $\{ \mathcal { F } _ { t } \} _ { t \ge 0 }$ and satisfy

$$\int_ {\mathcal {I}} \mathbb {E} [ | v (t) | ^ {2} ] d t < \infty .$$

For more details regarding the properties of such stochastic processes, we refer to section 1.9 from [6]. Invoking Proposition 2.1 in [28], one deduces that (1) admits a unique solution $x ( \cdot ) \in L _ { w } ^ { 2 } ( [ s , T ] ; \mathbb { R } ^ { n } )$ .
