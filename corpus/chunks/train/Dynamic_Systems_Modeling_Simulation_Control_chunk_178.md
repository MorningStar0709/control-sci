Equation (4.74) is our fundamental modeling equation for a pneumatic system. We see that for a constant-volume vessel $( { \dot { V } } = 0 )$ , Eq. (4.74) is identical to the differential equation for pressure (4.68) that was derived from the pneumatic capacitance definition. Furthermore, the pneumatic pressure-rate equation (4.74) has a structure that is very similar to the compressible hydraulic pressure-rate equation (4.40): both equations demonstrate an increase in pressure for a positive net flow into the CV and a decrease in pressure when the CV expands (i.e., $\dot { V } > 0 )$ . For the case of constant volume $( \dot { V } = 0 )$ ), the fluid capacitance of the hydraulic system in Eq. (4.40) is $V / \beta$ while the pneumatic capacitance in Eq. (4.74) is $V / n R T$ , which matches Eq. (4.67).
