# C.1 Proof of Lemma 3

In order to prove Lemma 3, we will use the following lemma that has a similar result but holds conditional on an event that depends on x.

Lemma 5. There exists an $L = \tilde { O } _ { T } ( 1 )$ such that the following holds. Suppose that $| x | , | y | \leq$ $4 \log ^ { 2 } ( T )$ and $d : = | x - y | \leq { \frac { 1 } { \log ^ { 1 0 } ( T ) } }$ . In the setting of Problem 1, for any $\theta \in \Theta$ such that $\begin{array} { r } { \epsilon : = \lVert \theta - \theta ^ { * } \rVert _ { \infty } \leq \frac { 1 } { \log ^ { 4 6 } ( T ) } , t \leq T , \dot { W } ^ { \prime } = \{ w _ { i } \} _ { i = 0 } ^ { t - 1 } } \end{array}$ , and any $K \in [ \frac { a - 1 } { b } , \frac { a } { b } ]$ , there exists $\mathcal { V } _ { \mathrm { L 5 } } \in \mathbb { R } ^ { t }$ that only depends on x, K and θ such that the event $E _ { \mathrm { L 5 } } ( x , K , \theta , W ^ { \prime } ) : = \left\{ W ^ { \prime } \in \mathcal { V } _ { \mathrm { L 5 } } \right\}$ satisfies $\mathbb { P } ( E _ { \mathrm { L 5 } } ( x , K , \theta , W ^ { \prime } ) ) = 1 - o _ { T } ( 1 / T ^ { 2 0 } )$ and the following holds. Define $d _ { i }$ as the difference in state at time $i \leq t$ when starting at $x _ { 0 } = x$ versus starting at $x _ { 0 } = y$ and using controller $C _ { K } ^ { \theta }$ with noise variables $W ^ { \prime }$ . Then for sufficiently large $T ,$ conditional on $E _ { \mathrm { L 5 } } ( x , K , \theta , W ^ { \prime } )$ ,

$$
d _ {i} \leq \left\{ \begin{array}{l l} \left(1 - \frac {1}{\log^ {1 0} (T)}\right) ^ {i} \cdot d, & i f i > L \\ 2 d + \tilde {O} _ {T} (\epsilon) & i f i \leq L. \end{array} \right. \tag {10}
$$

The proof of Lemma 5 can be found in Appendix C.2.

Now we need to find a single event $E _ { \mathrm { L 3 } } ( K , \theta , W ^ { \prime } )$ such that Equation (8) holds for all $| x | , | y | \leq 4 \log ^ { 2 } ( T )$ under this event. Define the set

$$G := \left\{- 4 \log^ {2} (T) + \frac {i}{\log^ {1 0} (T)} \right\} _ {i \in [ 0: 8 \log^ {1 2} (T) ]},$$

i.e. G is a grid of points evenly spaced $\frac { 1 } { \log ^ { 1 0 } ( T ) }$ apart. Note that $| G | = \tilde { O } _ { T } ( 1 )$ . Now, take

$$E _ {\mathrm{L3}} (K, \theta , W ^ {\prime}) = \bigcap_ {g \in G} E _ {\mathrm{L5}} (g, K, \theta , W ^ {\prime}).$$

First, we note that because $\mathbb { P } ( E _ { \mathrm { L 5 } } ( g , K , \theta , W ^ { \prime } ) ) = 1 - o _ { T } ( 1 / T ^ { 2 0 } )$ for all g and because $| G | =$ ${ \cal \tilde { O } } _ { T } ( 1 )$ , by a union bound $\mathbb { P } ( E _ { \mathrm { L 3 } } ) = 1 - o _ { T } ( 1 / T ^ { 1 0 } )$ .
