$$
\mathbf {P} = \left[ \begin{array}{l l l} \mathbf {P} _ {1} & \mathbf {P} _ {2} & \mathbf {B} _ {\mathbf {Y}} \end{array} \right] \in \mathbb {R} ^ {p N \times (m N + p N + m N)} \quad \mathbf {S} = \left[ \begin{array}{l} \mathbf {U} _ {\mathrm{p}} \\ \mathbf {Y} _ {\mathrm{p}} \\ \mathbf {U} _ {\mathrm{f}} \end{array} \right] \in \mathbb {R} ^ {(m N + p N + m N) \times H} \tag {52}
$$

Eq. (51) can be compactly written as:

$$\mathbf {Y} _ {\mathrm{f}} = \mathbf {P S} \tag {53}$$

which gives the parameter matrix P as:

$$\mathbf {P} = \mathbf {Y} _ {\mathrm{f}} \mathbf {S} ^ {\dagger} \tag {54}$$

where $\mathbf { S } ^ { \dagger } = ( \mathbf { S } ^ { \top } \mathbf { S } ) ^ { - 1 } \mathbf { S } ^ { \top }$ is the pseudoinverse of S. Note that Eq. (53) shows that the future output can be represented as a linear function of the past outputs and past inputs. The identified parameters $\mathbf { P } _ { 1 } , \mathbf { P } _ { 2 } , \mathbf { B } _ { \mathbf { Y } }$ can be used for computing the predicted output sequence $\mathbf { Y } _ { k + 1 }$ in Eq. (9) by defining

$$
\begin{array}{l} \mathbf {Y} _ {k + 1} = \mathbf {Y} _ {\mathrm{f} _ {1}} = \mathbf {H} _ {\left(\mathrm{N}, 1, \mathbf {y} _ {k + 1 \mid k}\right)} \\ \mathbf {H} _ {k - 1} = \mathbf {H} _ {k - 1} = \mathbf {H} _ {k - 1} \end{array} \quad \begin{array}{l} \mathbf {Y} _ {\mathrm{p} _ {1}} = \mathbf {H} _ {\left(\mathrm{N}, 1, \mathbf {y} _ {k - \mathrm{N} + 1 \mid k}\right)} \\ \mathbf {H} _ {k - 1} = \mathbf {H} _ {k - 1} \end{array} \tag {55}
\mathbf {U} _ {k} = \mathbf {U} _ {\mathrm{f} _ {1}} = \mathbf {H} _ {(\mathrm{N}, 1, \mathbf {u} _ {k | k})} \qquad \mathbf {U} _ {\mathrm{p} _ {1}} = \mathbf {H} _ {(\mathrm{N}, 1, \mathbf {u} _ {k - \mathrm{N} + 1 | k})}$$

Now, using Eq. (51), $\mathbf { Y } _ { k + 1 }$ can be represented as:

$$\mathbf {Y} _ {k + 1} = \mathbf {P} _ {1} \mathbf {U} _ {\mathrm{p} _ {1}} + \mathbf {P} _ {2} \mathbf {Y} _ {\mathrm{p} _ {1}} + \mathbf {B} _ {\mathbf {Y}} \mathbf {U} _ {k} \tag {56}$$
