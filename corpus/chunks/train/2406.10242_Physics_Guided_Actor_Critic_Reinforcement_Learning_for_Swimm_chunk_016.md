# C. BK Model under Proportional Control

The Batchelor-Kraichnan (BK) model is a special case of the Batchelor model where the velocity gradient is not only large scale but is also assumed to be Gaussian and delta-correlated in time (see [25] for the general discussion of the model’s historical roots and validity. The BK model in d-dimensions is described by the following paircorrelation function of the velocity gradient matrix, $\sigma ,$ entering Eq. (7):

$$
\begin{array}{l} \mathbb {E} \big [ \sigma_ {i j} (t) \sigma_ {k l} (t ^ {\prime}) \big ] = D (d + 1) \delta (t - t ^ {\prime}) \\ \times \left(\delta_ {j l} \delta_ {i k} - \frac {\delta_ {i j} \delta_ {k l} + \delta_ {j k} \delta_ {i l}}{d + 1}\right), \forall i, j, k, l = 1, \dots , d, \\ \end{array}
$$

where $\delta ( \cdot )$ and $\delta _ { i j }$ are the δ-function and the Kronecker symbol respectively.

Assuming that ϕ is sufficiently large we derive from the stochastic ODE (7) the following Fokker-Planck (FP) equation for the spherically symmetric probability density of s: (see [29] for details):

$$\mathcal {L} P (\boldsymbol {s} \mid \phi) = 0, \tag {14}\mathcal {L} = s ^ {1 - d} \frac {\mathrm{d}}{\mathrm{d} s} s ^ {d} \left(\phi + \frac {1}{2} \left(D (d - 1) s + \frac {\kappa}{s}\right) \frac {\mathrm{d}}{\mathrm{d} s}\right),$$

where $s = | s | ;$ and κ stands for variance of the thermal noise in Eq. (7). Solution of Eq. (14) is

$$P (\pmb {s} | \phi) = N ^ {- 1} \bigg (1 + \frac {(d - 1) D r ^ {2}}{\kappa} \bigg) ^ {- \phi / (D (d - 1))},$$

with

$$N \equiv \left(\frac {\pi \kappa}{(d - 1) D}\right) ^ {d / 2} \frac {\Gamma \big (\phi / (D (d - 1)) - d / 2 \big)}{d \Gamma \big (\phi / (D (d - 1)) \big)},$$

where N is the normalization coefficient which guarantees that, $\begin{array} { r } { \int _ { 0 } ^ { \infty } \Omega _ { s } \mathrm { d } s P ( s | \phi ) = 1 , \Omega _ { s } = ( \pi ^ { d / 2 } / \Gamma ( d / 2 + 1 ) ) s ^ { d - 1 } } \end{array}$ . The solution is valid, i.e. the normalization integral is bounded, if $\phi > \left( d - 1 \right) d D / 2$ .
