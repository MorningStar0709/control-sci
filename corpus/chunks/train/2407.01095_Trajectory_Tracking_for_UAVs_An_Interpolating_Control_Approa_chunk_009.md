# A. Planar UAV Model

For easier analysis and better insight, a planar UAV model (see Figure 1) will be employed, which exhibits similar behavior but is reduced both in the number of state variables and in the complexity of the equations of motion. The dynamics along ⃗y and ⃗z-axis with attitude φ as rotation around axis ⃗x are considered.

The nonlinear dynamics of the planar UAV is described by the following equations

$$\ddot {y} (t) = - \frac {F _ {T} (t)}{m} \sin (\phi (t)), \ddot {z} (t) = - g + \frac {F _ {T}}{m} \cos (\phi (t)), \ddot {\phi} (t) = \frac {\tau_ {R _ {x}} (t)}{J _ {x}}, \tag {10}$$

where $F _ { T }$ is the collective thrust in [N], m is the mass of the UAV in [kg], $\tau _ { R _ { x } }$ is the collective torque in [N  m] generated by rotors around ⃗x-axis and $J _ { x }$ is the moment of inertia around ⃗x in $\left\lceil \mathrm { k g } \cdot \mathrm { m } ^ { 2 } \right\rceil$ .

The attitude control is handled by the UAV’s autopilot, the proposed controller can control the translation in ⃗y and ⃗z via desired acceleration, which is recalculated based on linearized equations (10) around hover state to attitude control reference as

$$\bar {\phi} (t) = - \frac {\ddot {\bar {y}} (t)}{g}, F _ {T} (t) = m (\ddot {\bar {z}} (t) + g), \tag {11}$$

where φ¯ and $F _ { T }$ are desired angle and collective thrust, respectively. Moreover, as a side effect, the controllers are independent of the $\mathrm { U A V } \mathbf { \hat { s } }$ parameters m and $J _ { x } ,$ as they are only compensated in the calculations for the attitude and thrust control setpoint.
