# EXAMPLE 5.10 Static nonlinear system

Consider a static system that is described by the nonlinear equation

$$y (t) = f (u (t))$$

For all norms we have

$$| y (t) | \leq \max _ {u} | f (u (t)) |$$

The gain of the system is thus given by

$$\gamma = \max _ {u} \frac {| f (u) |}{| u}$$

The gain of a static system has a simple interpretation. A function whose norm is $\gamma$ can be bounded between the straight lines $y = \pm \gamma u$ , as is illustrated in Fig. 5.15.

Having defined the gain of a system, we can now define stability.
