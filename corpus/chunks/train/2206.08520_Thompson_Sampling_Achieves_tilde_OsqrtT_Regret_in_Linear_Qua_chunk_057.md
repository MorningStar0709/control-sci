Denoting by $\begin{array} { r } { \Sigma _ { T } ( \Theta , K ) : = \frac { 1 } { T } \sum _ { t = 1 } ^ { T } \mathbb { E } \left[ x _ { t } x _ { t } ^ { \top } \right] } \end{array}$ the finite averaged state covariance, we have the following

$$\Sigma_ {T} (\Theta , K) + \frac {\mathbb {E} \left[ x _ {T + 1} x _ {T + 1} ^ {\intercal} \right] - \mathbb {E} \left[ x _ {1} x _ {1} ^ {\intercal} \right]}{T} = (A + B K) \Sigma_ {T} (\Theta , K) (A + B K) ^ {\intercal} + \sigma_ {w} ^ {2} I \tag {40}$$

Taking the limit of both sides as $T \to \infty$ and noting that $\mathbb { E } \left[ x _ { T + 1 } x _ { T + 1 } ^ { \intercal } \right]$ has a finite value at the limit, we obtain the following Lyapunov equation

$$\Sigma (\Theta , K) = (A + B K) \Sigma (\Theta , K) (A + B K) ^ {\intercal} + \sigma_ {w} ^ {2} I \tag {41}$$

whose solution is given by the following convergent infinite sum

$$\Sigma (\Theta , K) = \sum_ {t = 0} ^ {\infty} (A + B K) ^ {t} \sigma_ {w} ^ {2} I ((A + B K) ^ {\intercal}) ^ {t} \tag {42}$$

It is well known that the optimal control policy of infinite-horizon LQR systems can be achieved by stationary linear feedback controllers Bertsekas (1995). Therefore, we can find the optimal LQR cost of a stabilizable system by minimizing its closed-loop cost among all stabilizing stationary linear feedback controllers.

Suppose $\Theta \in \mathcal { S } ^ { \operatorname { s u r r } } , i . e . , J ( \Theta , K ( \Theta _ { * } ) ) \le J ( \Theta _ { * } , K ( \Theta _ { * } ) )$ . Then, the optimal LQR cost of Θ is given as

$$J (\Theta) = J (\Theta , K (\Theta)) = \min _ {K \in \mathbb {R} ^ {d \times n}} J (\Theta , K) \tag {43}\leq J (\Theta , K (\Theta_ {*})) \stackrel {(a)} {\leq} J (\Theta_ {*}, K (\Theta_ {*})) = J (\Theta_ {*}) \tag {44}$$

where (a) is due to $\Theta \in { \mathcal { S } } ^ { \mathrm { s u r r } }$ . Thus, $\Theta \in S ^ { \mathrm { o p t } }$ .
