# 4. Theoretical Results

Define the Hilbert space for integer $p \geq 0$

$$\mathbb {H} ^ {p} := \mathbb {R} ^ {K \times K} \times \mathbb {R} ^ {K \times K} \times \mathrm{H} ^ {p} (Y; \mathbb {R}), \tag {20}$$

with norm $( | | \cdot | | _ { F }$ is the Frobenius norm):

$$\left| \left| (A, B, \rho) \right| \right| _ {\mathbb {H} ^ {p}} ^ {2} := \left| \left| A \right| \right| _ {F} ^ {2} + \left| \left| B \right| \right| _ {F} ^ {2} + \left| \left| \rho \right| \right| _ {\mathrm{H} ^ {p} (Y; \mathbb {R})} ^ {2}, \tag {21}$$

which are induced by the usual inner products in the respective spaces. $\mathrm { H } ^ { 0 } ( Y ; \mathbb { R } )$ is just $\operatorname { L } ^ { 2 } ( Y ; \mathbb { R } )$ . Let:

$$
\mathbf {L} := \left[ \begin{array}{c c} 0 & 0 \\ 0 & \epsilon \end{array} \right]. \tag {22}
$$

Define:

$$D (\mathcal {A}) := \{f \in \mathrm{H} ^ {2} (Y; \mathbb {R}): (\nabla_ {y} ^ {\top} f) \mathbf {L n} | _ {\partial Y} = 0 \text { almost everywhere } \}, \tag {23}$$

which is the (vector) space of $\mathrm { H } ^ { 2 } ( Y ; \mathbb { R } )$ functions satisfying the Neumann BCs in the trace sense. Recall the earlier definition in (19) of the form of linear time-varying operator $- \mathcal { A } ( t ; u , w )$ and consider it to be of type:

$$- \mathcal {A} (t; u, w): D (\mathcal {A}) \to \mathrm{L} ^ {2} (Y; \mathbb {R}), \tag {24}$$

with $u , w$ assumed to satisfy the control-disturbance constraints, and act as parameters for the (unbounded) time-varying linear operator $- \mathcal { A } ( \cdot )$ . Also, define ${ \mathfrak { G } } , { \mathfrak { H } } : \mathbb { H } ^ { 2 } \to \dot { \mathbb { R } } ^ { K \times K }$ as
