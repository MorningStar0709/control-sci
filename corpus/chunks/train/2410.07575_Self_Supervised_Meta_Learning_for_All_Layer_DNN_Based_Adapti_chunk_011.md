Algorithm 1 Self-Supervised Meta-Learning   
Input: Trajectory dataset $\mathcal{D} = \{\mathcal{D}_k\}$ Initialize: Neural network weight $\theta_0$ Result: Trained neural network weight $\theta_0$ while not done do
    Sample batch of tasks $\mathcal{D}_k$ from $\mathcal{D}$ for all $\mathcal{D}_k$ do $\mathcal{B}_k^a, \mathcal{B}_k^t \leftarrow \mathbf{D}_k$ $\theta_k \leftarrow \theta_0$ ▷ Gradient descent on $\mathcal{B}_k^a$ $\Delta \theta_k \leftarrow -\alpha \nabla_\theta \mathcal{L}(\theta_0, \mathcal{B}_k^a)$ end for $\mathcal{L}_{\text{meta}}(\theta_0) = \sum_{\mathcal{D}_k \sim p(\mathcal{D})} [\mathcal{L}(\theta_0 + \Delta \theta_k, \mathcal{B}_k^t) + \lambda_{\text{dir}} \mathcal{L}(\theta_0, \mathcal{B}_k^t)] + \lambda_{\text{norm}} \| \theta_0\|^2$ $\theta_0 \leftarrow \theta_0 - \beta \nabla_\theta \mathcal{L}_{\text{meta}}(\theta_0)$ ▷ Update $\theta_0$ end while
