1: $V[x(k_{\max}), k_{\max})] \leftarrow 0 \quad \forall x(k_{\max}) \in X$ 2: for each $k = k_{\max} - 1, \ldots, 0$ do
3:    for each discrete state $x(k) \in X$ do
4:    for each discrete control $a(k) \in U$ do
5:    Calculate $x(k + 1), v(k + 1)$ 6:    if $x(k + 1) \notin X$ then
7: $\Phi[x(k), a(k), k + 1] \leftarrow \infty$ 8:    continue
9:    end if
10: $\Phi[x(k), a(k), k + 1] \leftarrow \frac{1}{2}a(k)^2 + p(0|k) \cdot J^{*}_{DG}[x(k), a(k), k + 1] + [1 - p(0|k)] \cdot V[x(k + 1), k + 1]$ 11:    end for
12: $V[x(k), k] \leftarrow \min \Phi[x(k), a(k), k + 1] \quad \forall a(k) \in U$ 13: $R[x(k), k] = a^{*}(k) \leftarrow \arg \min_{a(k) \in U} \{\Phi[x(k), a(k), k + 1]\}$ ,
14:    with $a^{*}(k)$ the optimal control of point $[x(k), k]$ 15:    end for
16: end for

As mentioned, this algorithm delivers a globally optimal control law $a ^ { * } ( k ) = R [ { \pmb x } ( k ) , k ]$ for the full state domain 𝑿. Note that general stochastic optimal control problems do not possess a solution trajectory for specific initial states $\mathbf { \delta x } _ { 0 } ,$ , because the state evolution is uncertain in presence of the stochastic variables. However, in the specific stochastic GLOSA problem addressed here, the evolution of the vehicle states (4) and (5) is not affected by the stochastic variable 𝑧(𝑘), which concerns only the signal switching time. Thus, for a given initial state, i.e., vehicle position and speed at time 0, the optimal control law may be used to produce an optimal vehicle trajectory that the vehicle should pursue; until the signal switching actually occurs, in which case the vehicle must instantly behave according to the deterministic GLOSA solution.

Note that the SDP algorithm may take several minutes to execute in an ordinary PC, as will be reported in Section 5. This implies that the solution cannot be obtained on the vehicle side in real time.
