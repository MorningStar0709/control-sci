# B. Local Synchronization Error

Given the limited information available to the $i ^ { \mathrm { { t h } } }$ agent, we define its $k ^ { \mathrm { { t h } } }$ order weighted synchronization error as:

$$
\begin{array}{l} e _ {i} ^ {k} = - \nu_ {1} \sum_ {j = 1} ^ {N} a _ {i j} \left[ \left(x _ {i} ^ {k} - \psi_ {i}\right) - \left(x _ {j} ^ {k} - \psi_ {j}\right) \right] \\ - \nu_ {2} b _ {i} ^ {0} \left[ (x _ {i} ^ {k} - \psi_ {i}) - (x _ {0} ^ {k} - \psi_ {0}) \right] \tag {18} \\ \end{array}
$$

where the parameters $\nu _ { 1 }$ and $\nu _ { 2 } .$ , are positive scalar gains that adjust the influence between agents and the leader, providing flexibility in varying degrees of influence across the network.

Using the simplifying notations: $\bar { x } _ { i } ^ { k } : = x _ { i } ^ { k } - \psi _ { i } , \bar { x } _ { j } ^ { k } : =$ $x _ { j } ^ { k } - \psi _ { j }$ and $\bar { x } _ { 0 } ^ { k } : = x _ { 0 } ^ { k } - \psi _ { 0 }$ , we rewrite equation (18) as:

$$e _ {i} ^ {k} = - \nu_ {1} \sum_ {j = 1} ^ {N} a _ {i j} \left[ \bar {x} _ {i} ^ {k} - \bar {x} _ {j} ^ {k} \right] - \nu_ {2} b _ {i} ^ {0} \left[ \bar {x} _ {i} ^ {k} - \bar {x} _ {0} ^ {k} \right] \tag {19}$$

Here, $- \nu _ { 1 } \sum _ { i = 1 } ^ { N } a _ { i j } ( \hat { x } _ { i } ^ { k } - \hat { x } _ { j } ^ { k } )$ ensures consensus among agent i and its neighbors, while $- \nu _ { 2 } b _ { i } ^ { 0 } ( \bar { x } _ { i } ^ { k } - \bar { x } _ { 0 } ^ { k } )$ couples the agent’s state to the leader state by penalizing their difference. Defining, $e ^ { k } = [ e _ { 1 } ^ { k } , e _ { 2 } ^ { k } , e _ { 3 } ^ { k } , \ldots , \dot { e } _ { N } ^ { k } ] ^ { \top } , \underline { { { \bar { x } } } } _ { 0 } ^ { k } = [ \bar { x } _ { 0 } ^ { k } , \ldots , \bar { x } _ { 0 } ^ { k } ] ^ { \top }$ , $\bar { x } ^ { k } = [ \bar { x _ { 1 } ^ { k } } , \dots , \bar { x _ { N } ^ { k } } ] ^ { \bar { \top } }$ , we reformulate (19) in global form as:

$$e ^ {k} = - (\nu_ {1} L + \nu_ {2} B) (\bar {x} ^ {k} - \underline {{\bar {x}}} _ {0} ^ {k}) \tag {20}$$

Lemma 1: Under Assumption 2.1, the matrix $\nu _ { 1 } L + \nu _ { 2 } B$ is nonsingular.

Proof: Let J denote the set of strictly diagonally dominant nodes of $\pmb { \mathcal { L } } : = \nu _ { 1 } L + \nu _ { 2 } B ,$ i.e,

$$J = \{i: | £_ {i i} | > \sum_ {j \neq i} | £_ {i j} | \}.$$

If $\begin{array} { r } { I } { { \bf \Pi } = { \bf \Pi } \mathcal { N } , } \end{array}$ , i.e., the matrix £ is strictly diagonally dominant, then $\operatorname* { d e t } ( { \pmb { \mathscr { L } } } ) ~ \neq ~ 0$ is directly obtained by the application of the Gershgorin circle theorem [14].
