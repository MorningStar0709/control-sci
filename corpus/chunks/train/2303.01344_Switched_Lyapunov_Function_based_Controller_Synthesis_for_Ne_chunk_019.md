# V. ILLUSTRATIVE EXAMPLE

The performance and efficiency of the proposed algorithms with respect to the reduced complexity are highlighted using a simulation example for a rotary servo plant described by a continuous-time plant. By choosing the angle $x _ { 1 } = \varphi ( t )$ and rotational speed $x _ { 2 } = \dot { \varphi } ( t ) = \dot { x } _ { 1 }$ as the state variables, the mathematical model (1) with

$$
\boldsymbol {A} _ {c} = \left[ \begin{array}{c c} 0 & 1 \\ 0 & - \left(\Theta k _ {G} k _ {M} + \frac {k _ {R}}{J _ {M}}\right) \end{array} \right], \quad \boldsymbol {b} _ {c} = \left[ \begin{array}{l} 0 \\ \Theta \end{array} \right] \tag {22}
$$

and $\begin{array} { r } { \Theta = \frac { \eta k _ { G } k _ { M } } { J _ { M } R _ { M } } } \end{array}$ is derived [9]. The values of the parameters for (22) are listed in Table I. The sampling time of $T _ { d } =$ 20ms is chosen and the network induced delays are bounded by $0 \leq \tau _ { k } \leq 4 T _ { d } , \forall k$ . Furthermore, the used network ensures the safe transmission of data, i.e. no dropouts occur, leading to $\bar { d } = 4 , \bar { p } = 0$ and hence $\bar { \delta } = 4$ . The considered NCS can therefore be described by a discrete-time model (4) with

$$
\boldsymbol {A} _ {d} = \left[ \begin{array}{c c} 1 & 0. 0 1 0 6 \\ 0 & 0. 2 3 4 7 \end{array} \right] \text {and} \boldsymbol {b} _ {d} = \left[ \begin{array}{c} 0. 0 0 9 8 \\ 0. 7 9 5 3 \end{array} \right].
$$
