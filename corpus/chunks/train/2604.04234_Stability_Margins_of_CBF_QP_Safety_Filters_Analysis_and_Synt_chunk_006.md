# B. Closed-Form Controller and Two-Mode Dynamics

We next recall from [21] the explicit CBF–QP solution and the induced two-mode closed-loop dynamics. The resulting safety-filtered closed loop is continuous and piecewise affine, with an active mode that differs from the nominal mode by a rank-one term; this structure is central to the robustness analysis below. Define $a \ : = \ B ^ { \top } c , \ \beta \ : = \ a ^ { \top } G ^ { - 1 } a .$ , and $g ( x ) : = c ^ { \top } ( A _ { 0 } + \alpha I ) x +$ αd. The function $g ( x )$ determines whether the nominal input satisfies the CBF constraint. Accordingly, define the inactive and active regions

$$\mathcal {R} _ {+} := \{x \in \mathbb {R} ^ {n}: g (x) \geq 0 \}, \quad \mathcal {R} _ {-} := \mathbb {R} ^ {n} \setminus \mathcal {R} _ {+}. \tag {4}$$

Define

$$U := \frac {B G ^ {- 1} a}{\beta}, \quad V ^ {\top} := c ^ {\top} (A _ {0} + \alpha I), \tag {5}$$

and $\tilde { A } = A _ { 0 } - U V ^ { \top } , \tilde { b } = - \alpha d U .$

The next result recalls the explicit form of the safety filter and the induced two-mode closed loop. The key point is that, in the active region, the safety filter adds a correction in a fixed direction, which leads to an affine mode whose linear part is a rank-one modification of the nominal closed loop.

Proposition 1 ([21]). The optimizer of (3) is

$$
u ^ {\star} (x) = \left\{ \begin{array}{l l} - K x, & x \in \mathcal {R} _ {+}, \\ - K x - \frac {g (x)}{\beta} G ^ {- 1} a, & x \in \mathcal {R} _ {-}, \end{array} \right. \tag {6}
$$

and the corresponding closed-loop dynamics are

$$
\dot {x} = \left\{ \begin{array}{l l} A _ {0} x, & x \in \mathcal {R} _ {+}, \\ \tilde {A} x + \tilde {b}, & x \in \mathcal {R} _ {-}. \end{array} \right. \tag {7}
$$

Remark 1. In the active region R− in (4), the safety-filtered input can be written as

$$u ^ {\star} (x) = u _ {\mathrm{nom}} (x) + u _ {\mathrm{corr}} (x), \quad u _ {\mathrm{nom}} (x) = - K x, \tag {8}$$

with correction term

$$u _ {\text { corr }} (x) := - \frac {g (x)}{\beta} G ^ {- 1} a. \tag {9}$$

Thus, in the active region, the safety filter modifies the nominal controller through the additive correction term $u _ { \mathrm { c o r r } } ( x )$ .
