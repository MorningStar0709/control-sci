# IV. FEASIBILITY OF THE LMI

We now exploit Theorem 1 to provide a sufficient condition based on data such that the LMI (S) is feasible.

Under Assumption 3, E can be rewritten as a bounded matrix ellipsoid. By defining $\tilde { \Theta } : = \Theta - \hat { \Theta }$ , where

$$\hat {\Theta} := - X ^ {\top} Z ^ {- 1} \tag {39}$$

is the least-squares estimate of $\Theta ^ { \star }$ , we can change the coordinates in the QMI of (29) to obtain

$$
\left[ \begin{array}{l l} I _ {p} & \Theta \end{array} \right] N \left[ \begin{array}{c} I _ {p} \\ \Theta^ {\top} \end{array} \right] = \left[ \begin{array}{l l} I _ {p} & \tilde {\Theta} \end{array} \right] \left[ \begin{array}{c c} S _ {N} & 0 _ {p \times (n + \mu)} \\ 0 _ {(n + \mu) \times p} & - Z \end{array} \right] \left[ \begin{array}{c} I _ {p} \\ \tilde {\Theta} ^ {\top} \end{array} \right], \tag {40}
$$

where $S _ { N } : = \Delta - Y + X ^ { \top } Z ^ { - 1 } \ d { X }$ X is the Schur complement of the (2, 2) block of N in (30). As a consequence, (29) becomes

$$\boldsymbol {\mathcal {E}} = \left\{\Theta \in \mathbb {R} ^ {p \times (n + \mu)}: (\Theta - \hat {\Theta}) Z (\Theta - \hat {\Theta}) ^ {\top} \preceq S _ {N} \right\}. \tag {41}$$

Since the matrix (28) is positive semidefinite and $Z \succ 0 ,$ it holds from the Schur complement that $Y - X ^ { \top } Z ^ { - 1 } X \succeq 0$ . As a consequence,

$$\boldsymbol {\mathcal {E}} \subseteq \hat {\boldsymbol {\mathcal {E}}} := \left\{\Theta \in \mathbb {R} ^ {p \times (n + \mu)}: (\Theta - \hat {\Theta}) Z (\Theta - \hat {\Theta}) ^ {\top} \preceq \Delta \right\}. \tag {42}$$

Exploiting the fact that, for any $\boldsymbol \Theta \in \hat { \pmb \mathcal { E } }$ ,

$$\lambda_ {\min} (Z) (\Theta - \hat {\Theta}) (\Theta - \hat {\Theta}) ^ {\top} \preceq (\Theta - \hat {\Theta}) Z (\Theta - \hat {\Theta}) ^ {\top} \preceq \Delta , \tag {43}$$

we obtain the following bound:

$$\left| \Theta - \hat {\Theta} \right| ^ {2} \leq \frac {\lambda_ {\max} (\Delta)}{\lambda_ {\min} (Z)} =: \rho , \tag {44}$$

where, for $\rho > 0 , \mathtt { S N R } : = \rho ^ { - 1 }$ is the worst-case signalto-noise ratio according to Assumptions 2 and 3. Note that $\Theta ^ { \star } \in \mathcal { E }$ , thus it follows from the triangle inequality that

$$\left| \Theta - \Theta^ {\star} \right| = \left| (\Theta - \hat {\Theta}) - \left(\Theta^ {\star} - \hat {\Theta}\right) \right| \leq 2 \sqrt {\rho}. \tag {45}$$

We achieve the following feasibility condition, whose proof is given in Appendix II-C.

Corollary 1. Let Assumption 1 hold. Then, there exists $\rho ^ { \star } >$ 0 such that, if the data satisfy Assumptions 2 and 3, with $\rho : =$ $\lambda _ { \operatorname* { m a x } } ( \Delta ) / \lambda _ { \operatorname* { m i n } } ( Z ) \in [ 0 , \rho ^ { \star } ] ,$ , then the LMI (S) is feasible.
