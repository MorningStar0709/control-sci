# 3.2 Universal value functions

An important concept in RL is the state-value function, which is the expected value of $G _ { t }$ in (4). This shows how rewarding the current state is in terms of expected future returns. In order to handle the multiple-goal setting, the value function can be extended to the universal value function (Schaul et al., 2015), which depends not just on the state, but also the set point $y _ { t } ^ { \mathrm { r e f } }$ , and is defined as

$$V _ {\pi} (x _ {t}, y _ {t} ^ {\text { ref }}) = \mathbb {E} _ {\pi} [ G _ {t} | x _ {t}, y _ {t} ^ {\text { ref }} ], \tag {6}$$

where $\mathbb { E } _ { \pi } [ \cdot ]$ denotes the expected value while using policy π and t is any time step. Similarly, the so called $Q -$ -value function, estimating the value of a state-input pair, is defined as

$$Q _ {\pi} (x _ {t}, u _ {t}, y _ {t} ^ {\text { ref }}) = \mathbb {E} _ {\pi} [ G _ {t} | x _ {t}, u _ {t}, y _ {t} ^ {\text { ref }} ], \tag {7}$$

which indicates the expected value starting from the state xt with input $u _ { t } ,$ and after that following the policy π. When improving the policy, it is relevant to know how much better a specific input is in terms of a certain state as compared to the input given by the current policy. The advantage function is defined as

$$A _ {\pi} (x _ {t}, u _ {t}, y _ {t} ^ {\mathrm{ref}}) = Q _ {\pi} (x _ {t}, u _ {t}, y _ {t} ^ {\mathrm{ref}}) - V _ {\pi} (x _ {t}, y _ {t} ^ {\mathrm{ref}}), \tag {8}$$

and it corresponds to how much better the input $u _ { t }$ is than taking an input u from the policy distribution π. The value functions $V _ { \pi } , \ Q _ { \pi }$ and $A _ { \pi }$ can be estimated, for example, using Monte Carlo methods or Temporal-Difference Learning algorithms (Sutton and Barto, 2018). There are several choices for the advantage estimator. The estimator used in this paper is the generalized advantage estimator (Schulman et al., 2016) to reduce the variance.
