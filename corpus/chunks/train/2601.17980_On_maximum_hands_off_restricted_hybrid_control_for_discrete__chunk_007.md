It follows that $\mathcal { C } _ { \xi , T } \subseteq \mathcal { H }$ is the set of all admissible hybrid control sequences, $\left( \nu _ { T } , \mu _ { T } \right)$ , that steer the state of the switched system (2.1) from $x ( 0 ) = \xi$ to the origin in $T$ time instants, $\mathrm { i . e . , } x ( T ) = 0 _ { d }$ . We are interested in the sparsest element of the set $\mathcal { C } _ { \xi , T }$ .

Let us define $\Delta : \mathcal { N } ^ { T } \to \mathbb { Z } ^ { T - 1 }$ as

$$
\nu_ {T} \mapsto \Delta (\nu_ {T}) := \left( \begin{array}{c} \nu (1) - \nu (0) \\ \nu (2) - \nu (1) \\ ... \\ \nu (T - 1) - \nu (T - 2) \end{array} \right).
$$

Definition 2. An admissible hybrid control sequence, $( \nu _ { T } ^ { * } , \mu _ { T } ^ { * } ) \in \mathcal { C } _ { \xi , T }$ , is called a maximum hands-off hybrid control sequence for the switched system (2.1), if it is a solution to the following optimization problem:

$$\underset {(\nu_ {T}, \mu_ {T}) \in \mathcal {C} _ {\xi , T}} {\text { minimize }} \| \Delta (\nu_ {T}) \| _ {0} + \| \mu_ {T} \| _ {0}. \tag {2.4}$$
