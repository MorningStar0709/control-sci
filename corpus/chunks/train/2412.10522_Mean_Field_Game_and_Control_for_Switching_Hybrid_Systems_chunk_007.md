# 2.4 PDE system for MFG

Next, the optimality conditions for MFG in Definition 1 can be expressed as a system of partial differential equations (PDE).

Define the Lagrangian as a function of the running cost:

$$L (x, m, \alpha , p) = f (x, m, \alpha) + \langle b (x, m, \alpha), p \rangle , \tag {13}$$

and the corresponding Hamiltonian for the representative agent is

$$H (x, m, p) = \max _ {\alpha} \left\{- L (x, m, \alpha , p) \right\}. \tag {14}$$

It is assumed that H is strictly convex with respect to p.

Remark 1. In this setting of continuous state-action spaces, the optimality conditions are described by a set of coupled PDEs.

$$- \frac {\partial u}{\partial t} (t, x) - \nu \Delta u (t, x) + H (x, m (t, x), \nabla u (t, x)) = 0 \tag {15}
\begin{array}{l} \frac {\partial m}{\partial t} (t, x) - \nu \Delta m (t, x) \\ - \operatorname{div} (m (t, x) \partial_ {p} H (x, m (t, x), \nabla u (t, x))) = 0 \tag {16} \\ \end{array}
u (T, x) = g (x, m (T, x)), \quad m (0, x) = m _ {0} (x) \tag {17}$$

where $\textstyle \nu = { \frac { \sigma ^ { 2 } } { 2 } }$ and $\partial _ { p } H$ is the partial derivative of H w.r.t. the third variable which is p in (14).

Here, the optimal control for a typical agent satisfies the Hamilton-Jacobi-Bellman (HJB) equation through the value function, $u ( t , x )$ . Meanwhile, the density $m ( t , x )$ evolves according to the Fokker-Planck-Kolmogorov (FPK) equation. The HJB equation in (15) is formulated backward in time from a terminal value $u ( T , x )$ , while the FPK equation in (16) evolves forward in time from m(0, x).

Given the equilibrium values of population density and the value function, the optimal control for a representative agent is:

$$\hat {\alpha} (t, x) = \arg \max _ {\alpha} \left\{- L (x, m (t, x), \alpha , \nabla u (t, x)) \right\}. \tag {18}$$
