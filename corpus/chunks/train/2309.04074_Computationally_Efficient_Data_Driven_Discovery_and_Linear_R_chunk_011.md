# D. LQR Controller

An LQR controller is employed with the learned dynamics. The objective of the LQR controller is to regulate

the states of a system through the use of a quadratic loss function. The loss function $\mathcal { I }$ is given by

$$\mathcal {J} = \int_ {0} ^ {\infty} \frac {1}{2} [ \boldsymbol {\Phi} (x) ^ {T} \mathbf {Q} \boldsymbol {\Phi} (x) + \mathbf {u} ^ {T} \mathbf {R u} ] d t, \tag {17}$$

${ \mathrm { w h e r e ~ } } \mathbf { Q } = { \left[ \begin{array} { l l l l } { I _ { n \times n } } & { \cdot } & { \cdot } & { 0 } \\ { \cdot } & { \cdot } & { \cdot } & { 0 } \\ { \cdot } & { \cdot } & { \cdot } & { 0 } \\ { 0 } & { 0 } & { 0 } & { 0 } \end{array} \right] } \in R ^ { N \times N } { \mathrm { ~ a n d ~ } } \mathbf { R } = 1 { \mathrm { ~ a r e } }$ In×n

the tuning matrices for states and control respectively. It can be noted that since the first n rows of the observable $\Phi ( x )$ are the states themselves, it is straightforward to weight the controllable states. The LQR control effort is given as

$$u (t) = - K _ {L Q R} \Phi (x). \tag {18}$$

This constant gain is found in MATLAB by calling the LQR function with the appropriate input variables. The trajectory of the LTI system can then easily be found by integrating it with the RK4 function. For purposes of control, we converted the Koopman matrix K and input matrix B in continuous form such that

$$\mathbf {A} = \frac {\mathbf {K} - \mathbf {I}}{\Delta t} \text { and } \mathbf {B} _ {\mathrm{con}} = \frac {\mathbf {B}}{\Delta t}, \tag {19}$$

where $\Delta t$ is the RK4 time step of 0.01 seconds.
