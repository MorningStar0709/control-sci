$$\frac {\partial}{\partial x} \mathcal {H} (x (t), u (t), \lambda (t)) = - \frac {d}{d t} \lambda (t), \quad \lambda (t _ {f}) = 0 \quad (2 0 \mathrm{a})\frac {\partial}{\partial u} \mathcal {H} (x (t), u (t), \lambda (t)) = 0. \tag {20b}$$

Since $\mathcal { H } ( x , u , \lambda )$ is convex in x, u for any $\lambda \in \mathbb { R } ^ { n }$ , the joint optimality condition (20) is also sufficient. The first condition (20a) can be written as

$$
\begin{array}{l} - \frac {d}{d t} \lambda (t) = \frac {\partial}{\partial x} \mathcal {H} \bigl (x (t), u (t), \lambda (t) \bigr) \\ = A ^ {\prime} \lambda (t) + C ^ {\prime} Q C x (t) + C ^ {\prime} Q D u (t), \tag {21} \\ \end{array}
$$

where $\lambda ( t _ { f } ) = 0 \nonumber$ . Equation (21) may be written in reversed time $\tau = t _ { f } - t$ as follows

$$\frac {d}{d \tau} \hat {\lambda} (\tau) = A ^ {\prime} \hat {\lambda} (\tau) + C ^ {\prime} Q C \hat {x} (\tau) + C ^ {\prime} Q D \hat {u} (\tau), (2 2)$$

where $\hat { \lambda } ( 0 ) = 0 .$ . As the system (5) is externally symmetric, one may invoke Proposition 3 to write

$$\frac {d}{d \tau} \hat {\lambda} (\tau) = T S \hat {\lambda} (\tau) + T B \Sigma_ {e} Q C \hat {x} (\tau) + T B \Sigma_ {e} Q D \hat {u} (\tau),$$

where, multiplying the both sides by $T ^ { - 1 }$ gives

$$
\begin{array}{l} \frac {d}{d \tau} \big (T ^ {- 1} \hat {\lambda} (\tau) \big) \\ = S \hat {\lambda} (\tau) + B \Sigma_ {e} Q C \hat {x} (\tau) + B \Sigma_ {e} Q D \hat {u} (\tau) \\ = A T ^ {- 1} \hat {\lambda} (\tau) + B \Sigma_ {e} Q \big (C \hat {x} (\tau) + D \hat {u} (\tau) \big), \tag {23} \\ \end{array}
$$

where $\hat { \lambda } ( 0 ) = 0$ . Equation (23) indicates that the signal $T ^ { - 1 } \hat { \lambda }$ coincides with the state trajectory of the system (5) in the range $[ 0 , t _ { f } ]$ driven by the input signal $\Sigma _ { e } Q \left( \dot { C } \hat { x } + D \hat { u } \right)$ from the origin. The output signal associated with such a trajectory is given by

$$\hat {w} (\tau) := C T ^ {- 1} \hat {\lambda} (\tau) + D \Sigma_ {e} Q \big (C \hat {x} (\tau) + D \hat {u} (\tau) \big). \tag {24}$$
