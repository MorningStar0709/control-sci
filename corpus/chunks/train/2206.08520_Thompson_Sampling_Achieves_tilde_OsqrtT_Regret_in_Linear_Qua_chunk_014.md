Stabilizing TS After guaranteeing the design of stabilizing policies with improved exploration in the first phase, TSAC starts the adaptive control with only TS. In particular, for the remaining timesteps, TSAC deploys $u _ { t } = K ( \tilde { \Theta } _ { t } ) x _ { t }$ for $\tilde { \Theta } _ { t } = \mathcal { R } _ { S } ( \hat { \Theta } _ { t } + \beta _ { t } ( \delta ) V _ { t } ^ { - 1 / 2 } \eta _ { t } )$ and updates the sampled model parameter in every $\tau _ { 0 }$ time-steps. Note that, even though all the policies during this phase are stabilizing, frequent policy changes can still cause undesirable state growth. TSAC prevents this possibility by applying the same control policy for $\tau _ { 0 }$ time-steps in this phase as well. During this phase, TSAC decays the possible state blow-ups in the first phase and maintains stable dynamics.
