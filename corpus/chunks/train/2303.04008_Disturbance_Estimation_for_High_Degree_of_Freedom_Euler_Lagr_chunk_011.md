where $x ( t ) = \left[ \zeta ^ { \top } ( t ) \xi ^ { \top } ( t ) \right] ^ { \top }$ is the state variable, $A \in \mathbb { R } ^ { n \times n } , C \in \mathbb { R } ^ { n \times 2 n } , E \in \mathbb { R } ^ { 2 n \times n }$ are parametric matrices formulated as

$$
A = \left[ \begin{array}{c c} 0 _ {n} & I _ {n} \\ 0 _ {n} & 0 _ {n} \end{array} \right],   C = \left[ \begin{array}{c c} I _ {n} & 0 _ {n} \end{array} \right],   E = \left[ \begin{array}{c} 0 _ {n} \\ I _ {n} \end{array} \right],
$$

where $I _ { n } , 0 _ { n } \in \mathbb { R } ^ { n \times n }$ are respectively the n-dimensional identity and zero matrices, ζ(t) serves as the measurement output, u(t) is the auxiliary input that reads

$$
u = \left[ \begin{array}{c} \dot {\hat {M}} (q) q \\ \tau + \dot {\hat {M}} (q) \bar {q} - \hat {N} (q, \bar {q}) - \hat {G} (q) - \hat {F} (\bar {q}) \end{array} \right],
$$

and the disturbance term $d ( t ) = \tau _ { \mathrm { d } } ( t ) + \eta ( t )$ contains both the unknown external torques $\tau _ { \mathrm { d } } ( t )$ and the unmodeled system uncertainties $\eta ( t )$ . Note that $d ( t )$ is also bounded, i.e., $\lVert d ( t ) \rVert \leq \alpha _ { \tau } + \epsilon _ { \eta } ,$ , and we claim $d \approx \tau _ { \mathrm { d } } , \mathrm { i f } \epsilon _ { \eta } \ll$ $\alpha _ { \tau }$ . Under this condition, reconstruction of the unknown input $d ( t )$ in the linearized model (2) is equivalent to the estimation of the disturbance $\tau _ { \mathrm { d } } ( t )$ of system (1). To ensure the feasibility of the observer-based solution for the disturbance estimation problem, we present the following lemma.

Lemma 1. For the linearized dynamics in $( 2 ) , ( A , E )$ is controllable and $( A , C )$ is observable.

Proof. Calculating the controllability matrix $\left[ E A E \right]$ and the observability matrix $\left[ C ^ { \top } \left( C A \right) ^ { \top } \right] ^ { \top }$ for (2), we obtain

$$
\operatorname{rank} \left[ \begin{array}{c} C \\ C A \end{array} \right] = \operatorname{rank} \left[ \begin{array}{c c} I _ {n} & 0 _ {n} \\ 0 _ {n} & I _ {n} \end{array} \right] = 2 n,
$$

and

$$
\operatorname{rank} \left[ \begin{array}{c c} E & A E \end{array} \right] = \operatorname{rank} \left[ \begin{array}{c c} 0 _ {n} & I _ {n} \\ I _ {n} & 0 _ {n} \end{array} \right] = 2 n.
$$

Thus, (A, E) is controllable and (A, C) is observable.
