# (f) Derivation of Translation Equations $( Z _ { b } { \cdot } \mathbf { B o d y } \ \mathbf { A x i s } )$

The forces acting along the $Z _ { b } { \mathrm { - } } { \mathrm { b o d y } }$ axis of a missile consist of the $Z _ { b } \mathrm { - c o m p o n e n t }$ of weight and the Z -component of the aerodynamic normal force. We have noted earlier that the $Z _ { b } \mathrm { - c o m p o n e n t }$ of the aerodynamic normal force is

$$F _ {N} = C _ {N} q S$$

and that the $Z _ { b } \mathrm { - c o m p } { \cdot }$ onent of the weight is

$$z _ {m g} = m g \cos \phi \cos \theta .$$

Summing the force components along the $Z _ { b } \mathrm { - a x i s }$ , we have

$$F _ {z} = z _ {m g} - F _ {N} = m g \cos \phi \cos \theta - C _ {N} q S. \tag {3.50}$$

Substituting the above results into (3.42) results in the following equations:

$$
\begin{array}{l} \frac {d u}{d t} = R v - Q w + \left(F _ {x} / m\right) = R v - Q w + \left[ \left(T _ {\text {boost}} - T _ {\text {sust}}\right) / m \right] \\ - g \sin \theta - (C _ {A} q S / m), \tag {3.51a} \\ \end{array}
\frac {d v}{d t} = P w - R u + (F _ {y} / m) = P w - R u + g \cos \theta \sin \phi - (C _ {Y} q S / m), \tag {3.51b}\frac {d w}{d t} = Q u - P v + \left(F _ {z} / m\right) = Q u - P v + g \cos \phi \cos \theta - \left(C _ {N} q S / m\right). \tag {3.51c}
$$

Finally, in order to obtain the translational velocity components u, v, and w, (3.51) must be integrated. Thus,

Longitudinal Component of Velocity

$$u = \int_ {0} ^ {t} \left(\frac {d u}{d t}\right) d t. \tag {3.52a}$$

Lateral Component of Velocity

$$v = \int_ {0} ^ {t} \left(\frac {d v}{d t}\right) d t. \tag {3.52b}$$

Vertical Component of Velocity

$$w = \int_ {0} ^ {t} \left(\frac {d w}{d t}\right) d t. \tag {3.52c}$$
