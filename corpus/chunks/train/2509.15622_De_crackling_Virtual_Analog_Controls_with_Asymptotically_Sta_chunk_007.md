# 3.2. Common definitions and properties

The element-wise standard logistic function is notated using $\sigma ( \cdot )$ and has the following property:

$$\sigma (x) \in (0, 1), \quad x \in \mathbb {R}. \tag {3}$$

The element-wise hyperbolic tangent is notated using $\phi ( \cdot )$ . Since it is monotonic and 1-Lipschitz continuous, it has the following property [11]:

$$\| \phi (\mathbf {x}) \| \leq \| \mathbf {x} \|. \tag {4}$$

The matrix norm induced by a vector norm can be defined as

$$\| \mathbf {A} \| \triangleq \max _ {\mathbf {x} \neq \mathbf {0}} \frac {\| \mathbf {A x} \|}{\| \mathbf {x} \|}. \tag {5}$$
