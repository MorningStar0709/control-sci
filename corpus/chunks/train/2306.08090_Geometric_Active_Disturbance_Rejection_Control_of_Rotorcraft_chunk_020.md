# D. Tracking error kinematics and dynamics

Let $g ^ { d } ( t ) \ \in \ \mathrm { S E } ( 3 )$ be the desired pose generated by a guidance scheme [27]. Let $v ^ { d }$ and $\nu ^ { d }$ denote the desired translational velocity in the inertial frame I and the body frame B, respectively, and $\Omega ^ { d }$ denote the body’s reference angular velocity in the body frame. Then, the tracking error is given by,

$$
\bar {h} = (\mathrm{g} ^ {d}) ^ {- 1} \mathrm{g} = \left( \begin{array}{c c} Q & x \\ 0 & 1 \end{array} \right) \in \operatorname{SE} (3), \tag {45}
$$

where $Q = ( R ^ { d } ) ^ { \mathrm { T } } R$ is the attitude tracking error, and $x =$ $( R ^ { d } ) ^ { \mathrm { T } } ( \dot { b } - b ^ { \dot { d } } ) = ( R ^ { d } ) ^ { \mathrm { T } } \tilde { b }$ is the position tracking error, both in the body-fixed frame. Also, the translation velocity tracking error is given by

$$\tilde {v} = v - v ^ {d}, \tag {46}$$

and the angular velocity tracking error is given by,

$$\omega = \Omega - Q ^ {\mathrm{T}} \Omega^ {d}. \tag {47}$$

Thus, in the inertial frame I, the translational tracking error kinematics and dynamics can be summarized as

$$\dot {\tilde {b}} = \tilde {v}, \tag {48}m \dot {\tilde {v}} = m g \mathbf {e} _ {3} - f R \mathbf {e} _ {3} + \varphi_ {D} - m \dot {v} ^ {d}.$$

in the body-fixed frame B, the attitude tracking error kinematics and dynamics can be summarized as

$$
\begin{array}{l} \dot {Q} = Q \omega^ {\times}, \\ J \dot {\omega} = \tau + \tau_ {D} + J \left(\omega^ {\times} Q ^ {T} \Omega^ {d} - Q ^ {T} \dot {\Omega} ^ {d}\right) + (J \Omega) \times \Omega . \tag {49} \\ \end{array}
$$

The rotational error dynamics is decoupled from the translational error dynamics such that the translation control force, f , is obtained in the inertial frame followed by the appropriate attitude tracking control law, τ , in body frame to track the desired trajectory, $b ^ { d } .$ .
