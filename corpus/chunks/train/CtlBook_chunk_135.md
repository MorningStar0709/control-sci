# 5.4.1 Magnitude

The rst task of frequency response analysis of a system described by $G ( s )$ is to compute $\left| G ( j \omega ) \right|$ over some frequency range of interest, $\omega _ { m i n } < \omega < \omega _ { m a x }$ . $\left| G ( j \omega ) \right|$ is computed by 1) plugging in s = jω and 2) evaluating the magnitude of each pole and zero term, and 3) combining the terms.
