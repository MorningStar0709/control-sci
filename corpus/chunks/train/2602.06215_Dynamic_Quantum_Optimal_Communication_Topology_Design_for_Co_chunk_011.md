# 2.1.1 First-Order Consensus Dynamics

Consider a network of n agents with first-order dynamics

$$\dot {\boldsymbol {x}} (t) = \boldsymbol {u} (t), \tag {1}$$

where $\pmb { x } ( t ) = [ x _ { 1 } ( t ) , \ldots , x _ { n } ( t ) ] ^ { \top } \in \mathbb { R } ^ { n }$ is the stacked vector of agent states and ${ \pmb u } ( t ) ^ { \top } = [ u _ { 1 } ( t ) , \dots , u _ { n } ( t ) ] ^ { \top } \in$ $\mathbb { R } ^ { n }$ is the vector of control inputs. Consensus is achieved $\mathrm { i f }$

$$\lim _ {t \to \infty} | x _ {i} (t) - x _ {j} (t) | = 0, \quad \forall i, j \in \{1, \dots , n \}.$$

As proposed in [19], consensus in first-order MASs can be achieved using the classical distributed control law

$$u _ {i} (t) = - \sum_ {j \in \mathcal {N} _ {i}} a _ {i j} \left(x _ {i} (t) - x _ {j} (t)\right),$$

where ${ \mathcal { N } } _ { i }$ is the set of neighbors of agent i and $a _ { i j } \geq 0$ are the edge weights of the communication graph. In vector form, this becomes

$$\boldsymbol {u} (t) = - L \boldsymbol {x} (t),$$

where $\ b { L } \in \mathbb { R } ^ { n \times n }$ is the graph Laplacian matrix. Substituting this controller into (1) yields the closed-loop system

$$\dot {\boldsymbol {x}} (t) = - L \boldsymbol {x} (t). \tag {2}$$

If the graph is connected, then L is positive semidefinite with a single zero eigenvalue associated with the eigenvector 1, and all other eigenvalues are strictly positive.

In the presence of a time-varying communication topology, the Laplacian becomes time-dependent, and the closed-loop dynamics are

$$\dot {\boldsymbol {x}} (t) = - L (t) \boldsymbol {x} (t), \tag {3}$$

where $L ( t )$ is the Laplacian of the communication graph at time t. In our framework, $L ( t )$ is determined by the optimizer at each step and held constant until the next update.
