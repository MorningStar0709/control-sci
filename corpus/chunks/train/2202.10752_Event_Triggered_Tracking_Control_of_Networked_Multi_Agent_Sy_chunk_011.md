The graphs $\mathcal { G } _ { \mathrm { p } }$ and $\mathcal { G } _ { \mathrm { r } }$ are not necessarily the same, and will not play a central role in our results. To track the reference system (3), assume that in the absence of the network, the controller for the i-th agent is designed as

$$u _ {i} = u _ {\mathrm{c}} ^ {i} + u _ {\mathrm{f}} ^ {i}, \tag {4}$$

where $u _ { \mathrm { c } } ^ { i } \in \mathbb { R } ^ { n _ { u } ^ { i } }$ is the feedback control input, and is generated by the following feedback controller

$$\dot {x} _ {\mathrm{c}} ^ {i} = f _ {\mathrm{c}} ^ {i} (x _ {\mathrm{c}} ^ {i}, y _ {\mathrm{p}} ^ {i}, y _ {\mathrm{r}} ^ {i}, w _ {\mathrm{c}}), \quad u _ {\mathrm{c}} ^ {i} = g _ {\mathrm{c}} ^ {i} (x _ {\mathrm{c}} ^ {i}), \tag {5}$$

where $\boldsymbol { x } _ { \mathrm { c } } ^ { i } \in \mathbb { R } ^ { n _ { \mathrm { c } } ^ { i } }$ is the controller state, and $w _ { \mathrm { c } } \in \mathbb { R } ^ { n _ { 3 } }$ is the external disturbance. Therefore, in the absence of the network and under the controller $( 4 ) , x _ { \mathrm { p } } ^ { i }$ should converge to $x _ { \mathrm { r } } ^ { i }$ as close as possible. That is, the tracking goal is achieved, if

$$\left| x _ {\mathrm{p}} ^ {i} (t) - x _ {\mathrm{r}} ^ {i} (t) \right| \leq \beta \left(\left| x _ {\mathrm{p}} ^ {i} (0) - x _ {\mathrm{r}} ^ {i} (0) \right|, t\right) + \gamma (\| w \|), \tag {6}$$

where $\beta \in \mathcal { K L } , \gamma \in \mathcal { K } _ { \infty }$ , and $w : = ( w _ { \mathrm { p } } , w _ { \mathrm { r } } , w _ { \mathrm { c } } ) \in \mathbb { R } ^ { n _ { w } }$ with $n _ { w } : = n _ { 1 } + n _ { 2 } + n _ { 3 }$ .

Remark 1: For each agent, the dynamics (2) is general and recovers the single/double integrators [13]–[15] and heterogenous models [19], [33] as special cases. The controller (5) depends only on the corresponding agent and reference system, which implies that the controllers are not affected by other agents and reference systems. Such controller exists for both MAS [33], [34] and NCS [4], [35], [36]. 
