# 4.1 Data-Driven Identification via EDMD Algorithm

EDMD algorithm has been widely utilized to approximate the Koopman operator and its generator, also leading to the lifted bilinear system. The convergence of EDMD algorithm has been proved in [15], with applications on a variety of control problems including robots [4], quadrotors [21], smart grids [12], etc.

Despite the existence of its variants, fundamental theoretical principles and the most general methodology remain the same [29]. The collected data and notations are the same as (18), which lead to (20) and equivalently

$$Z _ {1} = \left[ A B _ {0} B _ {1} \dots B _ {m} \right] W _ {0} + D _ {0}. \tag {43}$$

Discarding the process disturbance, EDMD algorithm leads to the least square problem

$$\min _ {A, B _ {0}, B _ {1}, \dots , B _ {m}} \| Z _ {1} - [ A B _ {0} B _ {1} \dots B _ {m} ] W _ {0} \| \tag {44}$$

which is solved by

$$\left[ \hat {A} \hat {B} _ {0} \hat {B} _ {1} \dots \hat {B} _ {m} \right] := \hat {\mathbf {Z}} ^ {\top} = Z _ {1} W _ {0} ^ {\dagger} \tag {45}$$

and identifies the bilinear system (28) as

$$\frac {\mathrm{d} z}{\mathrm{d} t} = \hat {A} z + \hat {B} _ {0} u + \sum_ {i = 1} ^ {m} u _ {i} \hat {B} _ {i} z + r (z, u). \tag {46}$$

The identified matrix $\hat { A }$ is usually used to compute the Koopman eigenfunctions [11]. Assume that Aˆ is diagonalizable, i.e., $\exists V \in \mathbb { R } ^ { N \times N } , \mathrm { s . t . } \hat { \Lambda } = V ^ { - 1 } \hat { A } V$ , where the diagonal elements of Λ are corresponding eigenvaluesˆ and the columns of $V = [ v _ { 1 } \ \cdots \ v _ { N } ]$ are also eigenvectors of ${ \hat { A } } .$ . Then the approximated Koopman eigenfunctions are ${ \hat { \Phi } } ( x ) = V ^ { \top } \Psi ( x ) = \left[ v _ { 1 } ^ { \top } \Psi ( x ) \ \cdots \ v _ { N } ^ { \top } \Psi ( x ) \right]$  . In the case of non-diagonalizable ${ \hat { A } } , { \hat { \Lambda } }$ is of Jordan canonical form, $v _ { j } ^ { \top } \Psi ( x ) , \top = 1 , \cdot \cdot \cdot , N$ are generalized Koopman eigenfunctions [31]. The corresponding identified dynamics

$$
\begin{array}{l} \frac {\mathrm{d} \hat {\Phi} (x)}{\mathrm{d} t} = \hat {\Lambda} ^ {\top} \hat {\Phi} (x) + V ^ {\top} B _ {0} u \\ + \sum_ {i = 1} ^ {m} u _ {i} V ^ {\top} \hat {B} _ {i} (V ^ {\top}) ^ {- 1} \hat {\Phi} (x) + \hat {r} (\hat {\Phi} (x), u) \tag {47} \\ \end{array}
$$

is equivalent to (46).
