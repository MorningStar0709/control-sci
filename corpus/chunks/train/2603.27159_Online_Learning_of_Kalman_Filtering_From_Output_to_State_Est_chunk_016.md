# 3 Cost Functions of the State and Output Estimation Problems

In this section, we elaborate on the expression of the cost functions in the state and output estimation problems. We begin with the following definitions.

Definition 3.1. (Strongly-convexity) A differentiable function $f : \mathcal { X } \to \mathbb { R }$ is µ-strongly convex with $\mu > 0 \ i f$

$$f (y) - f (x) \geq \langle y - x, \nabla f (x) \rangle + \frac {\mu}{2} \left\| y - x \right\| ^ {2}, \forall x, y \in \mathcal {X}.$$

Definition 3.2. (Smoothness) A differentiable function $f : \mathcal { X } \to \mathbb { R }$ is $\beta _ { - s m o o t h }$ with $\beta > 0 \ i f$

$$\left\| \nabla f (y) - \nabla f (x) \right\| \leq \beta \| y - x \|, \forall x, y \in \mathcal {X}.$$

Definition 3.3. (Lipschitzness) A function f : $\mathcal { X }  \mathbb { R }$ is l-Lipschitz continuous with $l > 0$ if

$$| f (y) - f (x) | \leq l \| y - x \|, \forall x, y \in \mathcal {X}.$$

Some useful facts are summarized below. For a twice differentiable function $f : \mathcal { X }  \mathbb { R }$ with $\mathcal { X } \subseteq \mathbb { R } ^ { d }$ , it is well-known that µ-strongly convex (resp., β-smooth) as per Definition 3.1 (resp., Definition 3.2) is equivalent to $\nabla ^ { 2 } f ( x ) \succeq \mu I _ { d }$ (resp., $\nabla ^ { 2 } f ( x ) \preceq \beta I _ { d } )$ for all $x \in \mathcal { X }$ (see, e.g. [46, Chapter 3]), where we also note that $\nabla ^ { 2 } f ( x ) \preceq \left\| \nabla ^ { 2 } f ( x ) \right\| I _ { d }$ . Moreover, a direct consequence of β-smoothness is that $f ( y ) - f ( x ) \leq \nabla f ( x ) ^ { \top } ( y - x ) + { \textstyle { \frac { \beta } { 2 } } } \| y - x \| ^ { 2 } [ 4 6$ , Chapter 3]. For a differentiable function $f : \mathcal { X }  \mathbb { R }$ , the condition $\| \nabla f ( x ) \| \leq l$ for all $x \in \mathcal { X }$ implies that $f ( \cdot )$ is l-Lipschitz [46, Chapter 3].

We will make the following assumption, which can also be found in existing work for learning in partially observed LTI systems [8, 28, 40, 47].

Assumption 3.1. The system matrix A satisfies that $\rho ( A ) < 1$ , i.e., there exist $\kappa _ { A } \in \mathbb { R } _ { \geq 1 }$ and $\gamma _ { A } \in \mathbb { R }$ with $\rho ( A ) < \gamma _ { A } < 1$ such that $\| A ^ { k } \| \le \kappa _ { A } \gamma _ { A } ^ { k }$ for all $k \in \mathbb { Z } _ { \geq 0 }$ .

Note that the existence of $\kappa _ { A }$ and $\gamma _ { A }$ in Assumption 3.1 follows from the Gelfand formula [48, Chapter 5].
