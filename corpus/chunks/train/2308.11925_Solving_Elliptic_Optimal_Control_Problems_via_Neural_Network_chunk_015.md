where $\eta ^ { d } \in L ^ { 2 } ( \Omega )$ and $\eta ^ { b } \in L ^ { 2 } ( \partial \Omega )$ are Lagrangian multipliers for the constraints $F ( y , u ) = 0$ in Ω and $y = 0$ on $\partial \Omega .$ , respectively, which may also be realized by NNs. Given $( \eta _ { k } ^ { d } , \eta _ { k } ^ { b } )$ , we minimize $\mathcal { L } _ { \mathrm { a l m } } \big ( y _ { \theta } , u _ { \kappa } , \eta _ { k } ^ { d } , \eta _ { k } ^ { b } \big )$ in $\begin{array} { r } { ( y _ { \theta } , u _ { \kappa } ) \colon ( y _ { \theta ^ { k } } , u _ { \kappa ^ { k } } ) = \arg \operatorname* { m i n } _ { ( y _ { \theta } , u _ { \kappa } ) \in \mathcal { V } \times \mathcal { P } } \mathcal { L } _ { \mathrm { a l m } } ( y _ { \theta } , u _ { \kappa } , \eta _ { k } ^ { d } , \eta _ { k } ^ { b } ) } \end{array}$ , and then update the Lagrangian multipliers $( \eta _ { k + 1 } ^ { d } , \eta _ { k + 1 } ^ { b } )$ by $\eta _ { k + 1 } ^ { d } = \eta _ { k } ^ { d } + \mu F ( y _ { \theta ^ { k } } , u _ { \kappa ^ { k } } )$ and $\eta _ { k + 1 } ^ { b } = \eta _ { k } ^ { b } + \mu \alpha y _ { \theta ^ { k } }$ . In practice, the integrals are approximated by evaluations on fixed collocation points, and it suffices to discretize the multipliers point-wise [35].

AONN is also based on the KKT system (2.3), but utilizes the adjoint $p$ to compute the gradient of the reduced cost $\boldsymbol { J } ( u ) = \boldsymbol { J } ( u ) , \boldsymbol { u } )$ in u only. The total derivative $\mathrm { d } _ { u } J ( y , u ) = \alpha u + p$ is used to update u:

$$u _ {\kappa^ {k + 1}} = \arg \min _ {u _ {\kappa} \in \mathcal {U}} \| u _ {\kappa} - (u _ {\kappa^ {k}} - s ^ {k} \mathrm{d} _ {u} J (y _ {\theta^ {k}}, u _ {\kappa^ {k}})) \| _ {L ^ {2} (\Omega)} ^ {2}, \tag {2.12}$$

where the step size $s ^ { k } > 0$ should be suitably chosen. The minimization step projects the gradient update into the set $\mathcal { U } , \mathrm { i . e . }$ , u is approximated by an NN $u _ { \kappa } \in \mathcal { U }$ .

The presence of box constraint $u \in U$ requires a slightly different treatment. In PM, one can add an additional penalty term:

$$\mathcal {L} _ {\mathrm{pm-c}} (y _ {\theta}, u _ {\kappa}) = J (y _ {\theta}, u _ {\kappa}) + \mu \mathcal {L} _ {\mathrm{nn}} (y _ {\theta}, u _ {\kappa}) + \frac {\mu^ {\prime}}{2} \| u _ {\kappa} - P _ {U} (u _ {\kappa}) \| _ {L ^ {2} (\Omega)} ^ {2}, \tag {2.13}$$
