# 4.2 Results & Discussion

The bounds from Lemma 2 and Theorem 2 were found out to be 13.5936 and 16.0161 respectively. We observed that both the distributed minimax adaptive control policy and the distributed $\mathcal { H } _ { \infty }$ control policy were stabilising. That is, the behaviour of the distributed minimax adaptive and the distributed $\mathcal { H } _ { \infty }$ controllers get very similar after certain point in time as shown in Figure 1, where the stabilised states of node 1 is shown for illustrative purpose. The respective control inputs behaving very similar to each other is shown in Figure 1 where the first edge input from both the controller is shown. The time when both these policies coincide is still an open problem as we conjecture that there might exist adversarial disturbance policies for some node in the network that can keep the controller continuously guessing about its local dynamics model uncertainty.
