# 2.4.4 Monotonicity of the Value Function

If Assumptions 2.12 (and 2.13) hold, the value function sequence $\{ V _ { j } ^ { 0 } ( \cdot ) \}$ has an interesting monotonicity property, first established for the unconstrained linear quadratic regulator problem, namely, for given x, the value $V _ { j } ^ { 0 } ( x )$ decreases as the time to go j increases. We prove this in Lemma 2.15.

Lemma 2.15 (Monotonicity of the value function). Suppose, as usual, that Assumptions 2.2 and 2.3 hold, and that Assumptions 2.12 (and 2.13) hold. Then

$$V _ {j + 1} ^ {0} (x) \leq V _ {j} ^ {0} (x) \quad \forall x \in \mathcal {X} _ {j}, \forall j \in \mathbb {I} _ {0: N - 1}V _ {N} ^ {0} (x) \leq V _ {f} (x) \quad \forall x \in \mathbb {X} _ {f}$$
