$$
\begin{array}{l} \frac {d}{d t} e (t) = \left(\frac {d}{d t} x (t)\right) - \left(\frac {d}{d t} \tilde {x} (t)\right) \\ = (A x + B u) - (A \tilde {x} + B u + L (y - \tilde {y})) \\ \begin{array}{l} = (A x + B u) - \left(A \tilde {x} + B u + L C x - L C \tilde {x}\right) \\ = A (x - \tilde {x}) + L C (\tilde {x} - x) \end{array} \tag {30} \\ = A (x - \tilde {x}) + L C (\tilde {x} - x) \\ = A (x - \tilde {x}) - L C (x - \tilde {x}) \\ = (A - L C) (x - \tilde {x}) \\ = (A - L C) e \\ \end{array}
$$

Equation (30) is an estimate error dynamics system, with e(t) → 0 as t → ∞ for each initial tracking error e(t0) guaranteed by the characteristic equation in Eq. (31) by having all roots on the left side of the imaginary axis in the plane s. Therefore, the observer design process determines the L Matrix so that the roots of the characteristic equation are on the left side of the imaginary plane. This is achieved if the system has fully observable.

$$\left| \lambda I - (A - L C) \right| = 0 \tag {31}$$

After the w and v matrices have been determined, the gain L, also known as the Kalman gain can be acquired by executing the command ${ } ^ { " } L = l q e ( A , w , C , w , \nu ) ^ { " }$ in MATLAB. When the Bicopter has noise in the form of process noise (w) and measurement noise (v) as in Eq. (32) so the estimated error dynamics system becomes Eq. (33).

$$\dot {x} = A x + B u + w \tag {32}y = C x + v\dot {e} = A x + B u + w - A \tilde {x} - B u - L (C x + v - C \tilde {x}) \tag {33}= (A - L C) e + w - L v$$

In Eq. (33), there exist w and v, which means that the estimation is not going to be zero in most cases, so if we want the error to remain small we can do it with the right choice L, the optimal choice L is given by the Kalman gain.
