# 4.1.1 ε-PACS optimal control problem

Before presenting the second part of the verification lemma, let us first construct the control $( \xi ^ { \varepsilon , * } , \varsigma ^ { \varepsilon , * } )$ which turns out to be the optimal strategy for the $\varepsilon { \mathrm { - P A C S } }$ control problem. Let us first introduce the switching regions.

For any $\ell \in \mathbb { M }$ , let $S _ { \ell } ^ { \varepsilon }$ be the set defined by

$$\mathcal {S} _ {\ell} ^ {\varepsilon} = \{(x, \iota) \in \mathcal {O} \times \mathbb {I}: u _ {\ell} ^ {\varepsilon} (x, \iota) - \mathcal {M} _ {\ell} u ^ {\varepsilon} (x, \iota) = 0 \}.$$

The complement $\mathcal { C } _ { \ell } ^ { \varepsilon }$ of $S _ { \ell } ^ { \varepsilon }$ in $\mathcal { O } \times \mathbb { I }$ , where is optimal to stay in the regime $\ell ,$ is the so-called continuation region

$$\mathcal {C} _ {\ell} ^ {\varepsilon} = \{(x, \iota) \in \mathcal {O} \times \mathbb {I}: u _ {\ell} ^ {\varepsilon} (x, \iota) - \mathcal {M} _ {\ell} u ^ {\varepsilon} (x, \iota) < 0 \}.$$

The set $S _ { \ell } ^ { \varepsilon }$ satisfies the following property.

Lemma 4.3. Let ℓ be in M. Then, $\begin{array} { r } { S _ { \ell } ^ { \varepsilon } = \widetilde { S } _ { \ell } ^ { \varepsilon } { : = } \bigcup _ { \ell ^ { \prime } \in \mathbb { N } \backslash \{ \ell \} } S _ { \ell , \ell ^ { \prime } } ^ { \varepsilon } } \end{array}$ where

$$\mathcal {S} _ {\ell , \ell^ {\prime}} ^ {\varepsilon} := \{(x, \iota) \in \mathcal {C} _ {\ell^ {\prime}} ^ {\varepsilon}: u _ {\ell} ^ {\varepsilon} (x, \iota) = u _ {\ell^ {\prime}} ^ {\varepsilon} (x, \iota) + \vartheta_ {\ell , \ell^ {\prime}} \}.$$
