$$\frac {d}{d t} \frac {\partial T}{\partial \dot {x} _ {0}} - \dot {\theta} \frac {\partial T}{\partial \dot {y} _ {0}} = \sum F _ {x} \tag {2.66a}\frac {d}{d t} \frac {\partial T}{\partial \dot {y} _ {0}} + \dot {\theta} \frac {\partial T}{\partial \dot {x} _ {0}} = \sum F _ {y} \tag {2.66b}\frac {d}{d t} \frac {\partial T}{\partial \dot {\theta}} + \dot {x} _ {0} \frac {\partial T}{\partial \dot {y} _ {0}} - \dot {y} _ {0} \frac {\partial T}{\partial \dot {x} _ {0}} = \sum M _ {0} \tag {2.66c}$$

where the equation for the $q _ { i }$ remains unaltered. In accounting for the terms in these equations, the partials $\partial T / \partial { \dot { x } } _ { 0 }$ and $ \partial T / \partial \dot { y } _ { 0 }$ will be recognized as the general momenta, and $\partial T / \partial { \dot { \theta } }$ as the generalized angular momentum. The linear momentum can then be represented by the expression

$$\boldsymbol {p} = (\partial T / \partial \dot {x} _ {0}) \mathbf {i} + (\partial T / \partial \dot {y} _ {0}) \mathbf {j}. \tag {2.67}$$

Since the force equation is the rate of change of the linear momentum, we have

$$\boldsymbol {F} = [ d \boldsymbol {p} / d t ] + \boldsymbol {\omega} \times \boldsymbol {p}. \tag {2.68}$$

The terms of (2.66a) and (2.66b) are immediately accounted for. Moreover, the terms of (2.66c) can be identified from the expression

$$\mathbf {M} _ {0} = \left(\frac {d \boldsymbol {h} _ {0}}{d t}\right) + \left(\frac {d \boldsymbol {R} _ {0}}{d t}\right) \times \sum m _ {i} \left(\frac {d \boldsymbol {r} _ {i}}{d t}\right). \tag {2.69}$$

The term $\partial T / \partial { \dot { \theta } }$ is the angular momentum $\pmb { h } _ { 0 }$ , and the remaining two terms are equal to $\left( d { \pmb R } _ { 0 } / d t \right) \times \sum m _ { i } ( d { \pmb r } _ { i } / d t )$ , where $d { \pmb R } _ { 0 } / d t = ( d x _ { 0 } / d t ) { \bf i } + ( d y _ { 0 } / d t ) { \bf j }$ .
