# Step 2. First Pass for the Density.

We seek a classical solution to abstract problem:

$$\frac {\mathrm{d}}{\mathrm{d} t} \rho^ {1} (t) = \mathcal {L} _ {1} (t; \tilde {u} ^ {1} (\cdot), \tilde {w} ^ {1} (\cdot)) \rho^ {1} (t) \text { in } (0, T ] \times \operatorname{int} Y; \tag {51}\rho^ {1} (0) = \rho_ {0} \in D (\mathcal {L} _ {1}).$$

We present Theorem 5.5.3. from [28] to aid us now:

Theorem $\it { 2 . - } \left( P a z y \right)$ Let U be a Banach space, and $\{ \mathbf { A } ( t ) \} _ { t \in [ 0 , T ] }$ be a stable family of generators of C -semigroups on $U ~ s . t . ~ \mathbf { A } ( t ) : D \subset U \setminus U$ for each $t \in [ 0 , T ]$ , and for each $f \in D , \mathbf { A } ( t ) f \in C ^ { 1 } ( [ 0 , T ] ; U )$ . $I f g \in C ^ { 1 } ( [ 0 , T ] ; U )$ , then for every $x _ { 0 } \in D$ , the abstract Cauchy problem:

$$\frac {\mathrm{d}}{\mathrm{d} t} x (t) = \mathbf {A} (t) x (t) + g (t), x (0) = x _ {0}$$

has a unique classical solution in the sense of U :

$$x \in \mathrm{C} ^ {0} ([ 0, T ]; D) \cap \mathrm{C} ^ {1} ([ 0, T ]; U)$$

where for each $t \in [ 0 , T ]$ :

$$x (t) = \Phi_ {\mathbf {A}} (t, 0) x _ {0} + \int_ {0} ^ {t} \Phi_ {\mathbf {A}} (t, s) g (s) d s,$$

where Φ $\mathbf { \delta A } ( \cdot , \cdot ) : [ 0 , T ] \times [ 0 , T ]  { \mathcal { L } } ( D ; D )$ is the evolution system [28] associated to $\{ \mathbf { A } ( t ) \} _ { t \in [ 0 , T ] } . \ I f \mathbf { A } ( t )$ each generate ω-contractive $C _ { 0 }$ semigroups, then:

$$| | \Phi_ {\mathbf {A}} (t, s) | | _ {\mathcal {L} (D; D)} \leq e ^ {\omega (t - s)}.$$

This result is the infinite-dimensional analogue of the usual finitedimensional LTV systems theory [8].

Let $D : = D ( \mathcal { L } _ { 1 } ) , U : = \mathrm { L } ^ { 2 } ( Y ; \mathbb { R } )$ . We have from Lemma 2 for $n \geq 1$ that $\mathcal { L } _ { n } ( t ; u , w ) f \in \mathrm { C } ^ { 1 } ( [ 0 , T ] ; U )$ for each $f \in D$ . Now, we invoke Lemma 3 and Theorem 2, and conclude that:

$$\rho^ {1} (\cdot) \in \mathrm{C} ^ {0} ([ 0, T ]; D (\mathcal {L} _ {1})) \cap \mathrm{C} ^ {1} ([ 0, T ]; \mathrm{L} ^ {2} (Y; \mathbb {R}))$$

uniquely solves the previously given PDE in the sense of $\operatorname { L } ^ { 2 } ( Y ; \mathbb { R } )$ . Note that $D ( \mathcal { L } _ { n } ) = D ( \mathcal { A } ) = D$ for every $n \geq 1$ due to the structure of $\partial Y$ as in Remark 1.
