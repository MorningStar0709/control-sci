# B. Frequency-domain Willems’ fundamental lemma

Next, we recall the frequency-domain version of WFL presented in [18], [23]. To this end, we consider discretetime LTI systems, governed by

$$
\Sigma : \left\{ \begin{array}{l l} x _ {k + 1} & = A x _ {k} + B u _ {k}, \\ y _ {k} & = C x _ {k} + D u _ {k}, \end{array} \right. \tag {3a}
$$

where $x _ { k } \in \mathbb { R } ^ { n _ { x } } , \ u _ { k } \ \in \mathbb { R } ^ { n _ { u } }$ , and $y _ { k } \in \mathbb { R } ^ { n _ { y } }$ denote, respectively, the state, input, and output at time $k \in \mathbb { Z }$ . The system (3), i.e., the quadruplet of matrices $( A , B , C , D )$ , is unknown. Let $\ell _ { \Sigma } \in \mathbb { Z } _ { [ 1 , n _ { x } ] }$ denote the observability index of Σ, given by

$$\ell_ {\Sigma} = \min \arg \max _ {k \in \mathbb {Z} _ {\geqslant 0}} \operatorname{rank} \mathcal {O} _ {k},$$

where $\mathcal { O } _ { k }$ denotes the k-step observability matrix, i.e.,

$$
\mathcal {O} _ {k} = \left[ \begin{array}{c c c c} C ^ {\top} & (C A) ^ {\top} & \dots & (C A ^ {k - 1}) ^ {\top} \end{array} \right] ^ {\top}. \tag {4}
$$
