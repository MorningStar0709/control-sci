$$
\begin{array}{l} \mathbb {E} (1 - X _ {i} (k + 1)) = \mathbb {E} \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - W _ {i j} ^ {k} X _ {j} (k)\right) \\ \geq \prod_ {j \in N _ {i} ^ {\circ k}} \mathbb {E} \left(1 - W _ {i j} ^ {k} X _ {j} (k)\right) \tag {11} \\ = \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - w _ {i j} \mathbb {E} X _ {j} (k)\right) \\ \end{array}
$$

where $w _ { i j } \triangleq \operatorname* { P r } ( W _ { i j } = 1 | X _ { j } = 1 )$ denotes the conditional probability of the successful virus transmission from node $j$ to node i given that node j is infected. Thus we obtain

$$\mathbb {E} X _ {i} (k + 1) \leq 1 - \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - w _ {i j} \mathbb {E} X _ {j} (k)\right), \quad \forall i \in [ n ]. \tag {12}$$

Firstly, we note that this inequality is element-wise (i.e. it holds for each $i \in [ n ] )$ . Starting from the initial condition $p _ { i } ( 0 ) = \mathbb { E } X _ { i } ( 0 )$ for all $i \in [ n ]$ , we obtain $p _ { i } ( 1 ) \geq \mathbb { E } X _ { i } ( 1 )$ for all $i \in [ n ]$ . Secondly, we observe that the function

$$f (y _ {1},..., y _ {n}) \triangleq 1 - \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - w _ {i j} y _ {j}\right) \tag {13}$$

corresponding to the right-hand side of (12) is monotonically increasing with respect to $y _ { j }$ for all $j \in N _ { i } ^ { \circ k }$ . Thus,

$$
\begin{array}{l} p _ {i} (2) = 1 - \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - w _ {i j} p _ {j} (1)\right) \\ \geq 1 - \prod_ {j \in N _ {i} ^ {\circ k}} \left(1 - w _ {i j} \mathbb {E} X _ {j} (1)\right) \geq \mathbb {E} X _ {i} (2), \quad \forall i \in [ n ]. \\ \end{array}
$$

Thus by induction we obtain $p _ { i } ( k ) \geq \mathbb { E } X _ { i } ( k )$ for all $i \in [ n ]$ , for all $k \in \mathbb N$ . Finally, since the state $X _ { i } \in \{ 0 , 1 \}$ is binary, we obtain $\mathbb { E } X _ { i } ( k ) = \operatorname* { P r } ( X _ { i } ( k ) = 1 )$ and hence (10).
