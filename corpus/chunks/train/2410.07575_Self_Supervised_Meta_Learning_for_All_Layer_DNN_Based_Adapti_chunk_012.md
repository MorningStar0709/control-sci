# B. Adaptive Control or Composite adaptation

In the online adaptation stage, the goal is to minimize position tracking error. Given a twice-differentiable reference trajectory $q _ { d }$ , we define the composite velocity error

$$s = \dot {\tilde {q}} + \Lambda \tilde {q} = \dot {q} - \dot {q} _ {r} \tag {12}$$

where $\tilde { q } = q - q _ { d }$ is the position tracking error, $\boldsymbol { \Lambda } \in \mathbb { R } ^ { n \times n }$ is a positive definite matrix, and $q _ { r }$ is the reference position. The nominal PD controller with disturbance compensation is defined as

$$u = M (q) \ddot {q} _ {r} + C (q) \dot {q} _ {r} + g (q) - K s - f (q, \dot {q}, \theta) \tag {13}$$

We propose the following adaptive controller for the system (1) and nominal feedback controller (13) using the velocity gradient descent method from [43].

The adaptive law is

$$
\begin{array}{l} \bar {V} = s ^ {\top} M (q) s + \left(f (q, \dot {q}, \theta) - y\right) ^ {\top} \Gamma (f (q, \dot {q}, \theta) - y) (14) \\ \dot {\theta} = - \gamma \nabla_ {\theta} \dot {\bar {V}} (\theta) - \lambda (\theta - \theta_ {0}) \\ = - \gamma J ^ {\top} [ s + \Gamma (f - y) ] - \lambda (\theta - \theta_ {0}), \quad \theta (0) = \theta_ {0} (15) \\ \end{array}
$$

where $\bar { V }$ is a Lyapunov-like function, $y = d + \epsilon$ is the measured disturbance with measurement noise ϵ, and $J =$ $\frac { \partial f } { \partial \theta } ( q , \dot { q } , \theta )$ is the Jacobian of f . K, Γ are both positive definite gain matrices and $\gamma , \lambda$ are positive constant.

To constrain the Jacobian of f, we apply layer spectral normalization on θ during online adaptation:

$$
\theta_ {i} ^ {+} = \operatorname{Proj} (\theta_ {i} ^ {-}) = \left\{ \begin{array}{c c} \theta_ {i} ^ {-} & \text { if } \| \theta_ {i} ^ {-} \| <   \nu \\ \theta_ {i} ^ {-} / \| \theta_ {i} ^ {-} \| & \text { otherwise } \end{array} \right. \tag {16}
$$

Thus, the network parameter θ belongs to a bounded, compact set Θ during the online adaptation, further bound the Jacobian J if the robot states are bounded.

The control law and adaptive law are designed to ensure the closed-loop system remains robust to imperfect learning and time-varying environment conditions. In the next section, we discuss the theoretical guarantees of the proposed control and adaptive scheme.
