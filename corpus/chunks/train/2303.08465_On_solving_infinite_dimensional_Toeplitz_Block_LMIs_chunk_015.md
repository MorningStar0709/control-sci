Now for assertion 2), by Assumption 1, the only entry of the TBLMI not bounded on $\overline { { \ell ^ { 2 } } }$ is $\mathcal { N }$ . Fortunately, as $\mathcal { N }$ is diagonal, $\mathcal { N } = \mathcal { N } _ { b ( p ) }$ for any $p \geq 0$ and thus $\mathcal { N }$ does not play any role. If P is a solution to (12), then by assumption $\mathcal { L } ( \mathcal { P } ; \mathrm  ~  ~ { ~ \vert ~ \} ~ }$ must be a bounded operator on $\ell ^ { \dot { 2 } }$ . LMIs being continuous with respect to their entries, there exists a constant $C _ { 2 }$ depending of $\mathcal { P }$ and $\cdot$ such that for any $p > 0$

$$
\begin{array}{l} \left\| \mathcal {L} (\mathcal {P}; \mathcal {A} _ {s}, s \in \mathbb {S}) - \mathcal {L} (\mathcal {P}; \mathcal {A} _ {s _ {b (p)}}, s \in \mathbb {S}) \right\| _ {\ell^ {2}} \\ \leq C _ {2} \sum_ {s \in \mathbb {S}} \left\| \mathcal {A} _ {s} - \mathcal {A} _ {s _ {b (p)}} \right\| _ {\ell^ {2}} \\ \end{array}
$$

$$\left\| \mathcal {L} \big (\mathcal {P}; \mathcal {A} _ {s}, s \in \mathbb {S} \big) - \mathcal {L} \big (\mathcal {P}; \mathcal {A} _ {s _ {b (p)}}, s \in \mathbb {S} \big) \right\| _ {\ell^ {2}} < \epsilon$$

Finally, to show the last assertion, as all $\mathcal { A } _ { s _ { b ( p ) } } , ~ s ~ \in \mathrm { ~ S ~ }$ in $\mathcal { L } ( \mathcal { P } ; \mathcal { A } _ { s _ { b ( p ) } } , s \in \mathbb { S } ) < 0$ are banded, only the unknown $\mathcal { P }$ is possibly not banded. As the product of infinite dimensional banded TB operators is a banded TB operator (which is not true in finite dimension), the terms in the TBLMI involving operator $\mathcal { P }$ have the generic form: $\mathcal { U P V }$ where U and V are polynomial functions of banded entries $\cdot$ and are therefore banded. Applying $\Pi _ { m }$ on $\begin{array} { r } { \mathcal { L } ( \mathcal { P } ; { } ) \qquad } \end{array}$ leads to compute $\Pi _ { m } ( \mathcal { U P V } )$ . Using (13), we have:

$$

where $\eta _ { 1 }$ is the first integer greater than ${ \scriptstyle { \frac { 1 } { 2 } } } d ^ { o } U$ and where $\Pi _ { m } ( { \mathcal { P V } } )$ is determined using (13) with η the first integer greater than ${ \scriptstyle { \frac { 1 } { 2 } } } d ^ { o } V$ .
