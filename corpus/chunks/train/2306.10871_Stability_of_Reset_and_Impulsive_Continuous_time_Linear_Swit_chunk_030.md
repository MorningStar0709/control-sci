# 6. Mode-Dependent Constraints

Finding dwell time constraint is one of the most common ways to establish stability of a given switched system when all subsystems are stable. However, the requirement of spending more than a fixed amount of time after every switching instance can be relaxed by allowing different dwell time constraints depending on the active subsystem (or mode). As a result, a larger class of stabilizing signals can be obtained, refer to Example 7.2. This also allows us to select the switching times ‘on the $\mathrm { H y } ^ { \prime }$ to ensure stability, if the information of the active mode is known as a feedback at every switching instance. Mode-dependent dwell time constraints have been used in literature to stabilize switched systems both in the continuous-time setting [6], and discrete-time setting [9]. The more general modedependent average dwell time constraints have also been widely used, in the literature, to stabilize switched systems with both stable and unstable subsystems, for instance [30].

Consider a switched system (1) and suppose the increasing sequence of switching times of the signal σ is given by $\{ t _ { k } \} _ { k \geq 1 }$ . Then σ is said to have mode-dependent dwell times $\{ \tau _ { p } \} _ { p \in \mathfrak { s } }$ if $t _ { k + 1 } - t _ { k } \ge \tau _ { p }$ , whenever $\sigma ( t _ { k } ) = p$ . Further σ is said to have mode-dependent flee times $\{ \eta _ { p } \} _ { p \in \mathfrak { u } } \mathrm { i f } \ t _ { k + 1 } - t _ { k } \le \eta _ { p }$ , whenever $\sigma ( t _ { k } ) = p$ . Note that, we have earlier looked at situations when the dwell time and flee time are independent of the mode. Mode-dependent constraints allow for more effective stabilization of switched systems.
