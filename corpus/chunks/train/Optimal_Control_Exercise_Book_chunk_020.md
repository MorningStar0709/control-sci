# Proposition 2 (Second-order necessary condition for constrained optimality):

Suppose that f is a $\mathcal { C } ^ { 2 }$ (continuously differentiable) function and $x ^ { * }$ is its local minimum. Then,

$$d ^ {T} \nabla^ {2} f (x ^ {*}) d \geq 0 \tag {14}$$

for all feasible directions d such that

$$\langle \nabla f (x ^ {*}), d \rangle = 0 \tag {15}$$
