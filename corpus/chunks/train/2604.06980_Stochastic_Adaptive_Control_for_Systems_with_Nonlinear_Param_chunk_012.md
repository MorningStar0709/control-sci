# 2.2 Problem formulation

Denote the system parameter by $\theta ^ { * } \ = \ \left[ A ^ { * } , B ^ { * } \right] ^ { \top } \ \in$ $\mathbb { R } ^ { ( n + m ) \times n }$ , and assume that the true parameter $\theta ^ { \ast } \in \Theta$ , where Θ is a compact convex set in $\mathbb { R } ^ { ( n + m ) \times n }$ . We first consider the parameter estimation problem for the unknown parameter $\theta ^ { * }$ .

Problem 1 At each time $t \geq 0$ , given the trajectory observations $\{ x _ { \tau } , u _ { \tau - 1 } \} _ { 1 \leq \tau \leq t } ,$ , construct an online estimate $\widehat { \theta } _ { t }$ of the unknown parameter $\theta ^ { * }$ , and determine conditions on the trajectory data $\{ \boldsymbol { x } _ { \tau } , \boldsymbol { u } _ { \tau - 1 } \}$ under which

$$\lim _ {t \to \infty} \left\| \hat {\theta} _ {t} - \theta^ {*} \right\| ^ {2} = 0, a. s.$$

With the estimation algorithm provided in Problem 1, we further solve the adaptive control problem.

Let the space of admissible feedback control laws be

$$\mathcal {U} = \left\{\mathbf {u}: \mathbb {R} ^ {n} \rightarrow \mathbb {R} ^ {m} \mid \mathbf {u} \text { is Lipschitz continuous } \right\}.$$

Let π be a given control design mechanism, which defines a mapping from Θ to U, and let $\{ x _ { t } ^ { * } \}$ denote the corresponding reference state trajectory generated by system (1) under the feedback controller $u _ { t } ^ { * } = \pi _ { \theta ^ { * } } ( x _ { t } )$ , i.e.,

$$x _ {t + 1} ^ {*} = f (A ^ {*} x _ {t} ^ {*} + B ^ {*} \pi_ {\theta^ {*}} (x _ {t} ^ {*})) + w _ {t + 1}, x _ {0} ^ {*} = x _ {0}, \mathrm{a.s.} (8)$$

We will study the following adaptive control problem.

Problem 2 Given a control design mechanism $\pi . _ { i }$ , without knowing the true parameter $\theta ^ { * }$ , design an adaptive feedback control algorithm together with the online parameter estimation algorithm

$$u _ {t} = \mathcal {G} (x _ {t}, \hat {\theta} _ {t}),\hat {\theta} _ {t} = \mathcal {H} (\{u _ {\tau} \} _ {\tau \leq t - 1}, \{x _ {\tau} \} _ {\tau \leq t}), \quad \forall t \geq 0,$$

where $\{ x _ { t } \}$ is the closed-loop trajectory of system (1) with adaptive feedback control $\bar { \{ { u } _ { t } \} }$ , such that the resulting trajectory $\{ x _ { t } , u _ { t } \}$ starting from any initial state $x _ { 0 }$ has the following two properties:

• Adaptive stabilization: the closed-loop system is globally stable, $i . e . _ { \cdot }$ ,

$$\limsup _ {t \to \infty} \frac {1}{t} \sum_ {\tau = 0} ^ {t} \| x _ {\tau} \| ^ {2} < \infty , a. s.$$

• Asymptotic tracking: the averaged tracking error converges to zero almost surely, i.e.,

$$\lim _ {t \to \infty} \frac {1}{t} \sum_ {\tau = 0} ^ {t} \left(\left\| x _ {\tau} - x _ {\tau} ^ {*} \right\| ^ {2} + \left\| u _ {\tau} - u _ {\tau} ^ {*} \right\| ^ {2}\right) = 0, a. s.,$$

where $\{ x _ { t } ^ { * } \} _ { t \geq 0 }$ and $\{ u _ { t } ^ { * } \} _ { t \geq 0 }$ are given in (8).

These two problems outlined above will be addressed in Section 3 and Section 4, respectively.
