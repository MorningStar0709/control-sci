# Example 2.6

Given the functional

$$J (x (t)) = \int_ {t _ {0}} ^ {t _ {f}} [ 2 x ^ {2} (t) + 3 x (t) + 4 ] d t, \tag {2.1.17}$$

evaluate the variation of the functional.

Solution: First, we form the increment and then extract the variation as the first order approximation. Thus

$$
\begin{array}{l} \Delta J \triangleq J (x (t) + \delta x (t)) - J (x (t)), \\ = \int_ {t _ {0}} ^ {t _ {f}} [ 2 (x (t) + \delta x (t)) ^ {2} + 3 (x (t) + \delta x (t)) + 4) \\ \left. - \left(2 x ^ {2} (t) + 3 x (t) + 4\right) \right] d t, \\ = \int_ {t _ {0}} ^ {t _ {f}} [ 4 x (t) \delta x (t) + 2 (\delta x (t)) ^ {2} + 3 \delta x (t) ] d t. \tag {2.1.18} \\ \end{array}
$$

Considering only the first order terms, we get the (first) variation as

$$\delta J (x (t), \delta x (t)) = \int_ {t _ {0}} ^ {t _ {f}} (4 x (t) + 3) \delta x (t) d t. \tag {2.1.19}$$
