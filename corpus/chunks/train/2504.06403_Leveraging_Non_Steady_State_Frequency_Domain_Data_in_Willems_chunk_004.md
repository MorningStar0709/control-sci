# II. NOTATION AND PRELIMINARIES

Let R denote the field of reals, C the complex plane, and Z the integers. We denote $\mathbb { Z } _ { [ m , n ] } = \{ m , m + 1 , \ldots , n \}$ and $\mathbb { Z } _ { \ge m } = \{ m , m + 1 , \ldots \}$ , where n, m ∈ Z and n > m. The imaginary unit is denoted $j , \mathrm { i . e . , } j ^ { 2 } = - 1$ . For a complexvalued matrix $A \in \mathbb { C } ^ { n \times m } , \dot { A } ^ { \top } , A ^ { \mathsf { H } }$ , and A∗ are its transpose, its complex-conjugate transpose, and its complex conjugate.

For any complex-valued function $v : \mathbb { Z } \to \mathbb { C } ^ { n _ { v } }$ , let

$$v _ {[ r, s ]} := \left[ v _ {r} ^ {\mathsf {H}} \quad v _ {r + 1} ^ {\mathsf {H}} \quad \ldots \quad v _ {s} ^ {\mathsf {H}} \right] ^ {\mathsf {H}}$$

denote the vectorized restriction of v to the interval $\mathbb { Z } _ { [ r , s ] } .$ . With some abuse of notation, we also use $v _ { [ r , s ] }$ to refer to the sequence $\{ v _ { k } \} _ { k \in \mathbb { Z } _ { [ r , s ] } }$ . Similarly, we use $0 _ { [ r , s ] }$ to refer to the length- $\boldsymbol { \cdot } \boldsymbol { s } - \boldsymbol { r } + 1$ sequence of null vectors of appropriate dimensions or their vectorized form

$$
0 _ {[ r, s ]} = \left[ \begin{array}{c c c c} 0 & 0 & \ldots & 0 \end{array} \right] ^ {\top}
$$

depending on the context. Finally, $0 _ { m \times n }$ denotes the m-by-n zero matrix, ⊗ denotes the Kronecker product, and ⌈·⌉ the ceiling function.
