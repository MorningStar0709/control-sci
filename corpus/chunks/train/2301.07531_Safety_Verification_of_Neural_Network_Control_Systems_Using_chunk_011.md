Remark 1: In the process of augmenting neural networks Φ and Φˆ into $\tilde { \Phi } ,$ , the case of $\ell = 1$ in (14)–(16) ensures that augmented neural network Φ˜ takes the one same input $\pmb { \eta } _ { 0 }$ for the subsequent calls involving both processes of Φ and its reduced-size version Φˆ . Then, for $1 < \ell \leq \hat { L } - 1$ , augmented neural network $\tilde { \Phi }$ conducts the computation of Φ and and its reduced-size version Φˆ parallelly for the hidden layers of $1 < \ell \leq \hat { L } - 1$ . When $\hat { L } \leq \ell \leq L - 1$ , the hidden layers of reduced-size neural network Φˆ which has fewer hidden layers are expanded to match the number of layers of the original neural network Φ with a larger number of hidden layers, but the expanded layers are forced to pass the information to subsequent layers without any changes, i.e., the weight matrices of the expanded hidden layers are identity matrices, and the bias vectors are zero vectors. This expansion is formalized as in the case of $\hat { L } \leq \ell \leq L - 1$ in (14)–(16). Moreover, as $\ell = L$ , this layer is a combination of output layers of both Φ and Φˆ to generate the same outputs of Φ and Φˆ . At last, a comparison layer $L + 1$ is added to compute the exact difference between the original neural network Φ and its reduced-size version Φˆ .

Remark 2: As shown in Theorem 1, the key of computing model reduction precision $\rho ( \Phi , { \hat { \Phi } } , \mathcal { U } )$ is to compute the maximal output value of augmented neural network Φ˜ with respect to input set U. This can be efficiently done by neural network reachability analysis. For instance, as in NNV neural network reachability analysis tool, the reachable sets are in the form of a family of polyhedral sets [16], and in the IGNNV tool, the output reachable set is a family of interval sets [13], [14]. With the reachable set $\tilde { \mathcal { V } } ,$ , the model reduction precision $\rho ( \Phi , { \hat { \Phi } } , \mathcal { U } )$ can be easily obtained by searching for the maximal value of $\| \tilde { \eta } _ { L + 1 } \|$ in Y˜, e.g., testing throughout a finite number of vertices in polyhedral sets.
