# 2.2 Optimal control with box constraint

Now we turn to the constrained case, i.e., the control u is from a box constrain set $U = \{ u \in L ^ { 2 } ( \Omega ) : u _ { a } \leq$ $u \leq u _ { b } ~ \mathrm { a . e }$ . in $\Omega \}$ , with $u _ { a } < u _ { b }$ . Then the optimal tuple $( \bar { y } , \bar { p } , \bar { u } )$ satisfies the following first-order necessary optimality system [49, p. 67]

$$
\left\{ \begin{array}{l l} - \Delta y = f + u \text {   in   } \Omega & \text { with } \quad y = 0 \text {   on   } \partial \Omega , \\ - \Delta p = y - y _ {d} \text {   in   } \Omega & \text { with } \quad p = 0 \text {   on   } \partial \Omega , \\ (u + \lambda^ {- 1} p, v - u) \geq 0, & \forall v \in U. \end{array} \right. \tag {2.8}
$$

Using the pointwise projection operator $P _ { U }$ into the set $U _ { ; }$ defined by $P _ { U } ( v ) ( x ) = \operatorname* { m i n } ( \operatorname* { m a x } ( v ( x ) , u _ { a } ) , u _ { b } )$ , we can rewrite the variational inequality in (2.8) as $u = P _ { U } ( - \lambda ^ { - 1 } p )$ , and the system is reduced to

$$
\left\{ \begin{array}{l l} - \Delta y = f + P _ {U} (- \lambda^ {- 1} p) \text {in} \Omega & \text {with} \quad y = 0 \text {on} \partial \Omega , \\ - \Delta p = y - y _ {d} \text {in} \Omega & \text {with} \quad p = 0 \text {on} \partial \Omega . \end{array} \right. \tag {2.9}
$$

The system (2.9) naturally motivates the following continuous loss:

$$
\begin{array}{l} \mathcal {L} (y _ {\theta}, p _ {\sigma}) = | \Omega | \mathbb {E} _ {X \sim U (\Omega)} \left[ \left(\Delta y _ {\theta} (X) + f (X) + P _ {U} (- \lambda^ {- 1} p _ {\sigma} (X))\right) ^ {2} \right] + \alpha_ {b} ^ {y} | \partial \Omega | \mathbb {E} _ {Y \sim U (\partial \Omega)} \left[ \left(y _ {\theta} (Y)\right) ^ {2} \right] \\ + \alpha_ {i} | \Omega | \mathbb {E} _ {X \sim U (\Omega)} \left[ \bigl (\Delta p _ {\sigma} (X) + y _ {\theta} (X) - y _ {d} (X) \bigr) ^ {2} \right] + \alpha_ {b} ^ {p} | \partial \Omega | \mathbb {E} _ {Y \sim U (\partial \Omega)} \left[ \bigl (p _ {\sigma} (Y) \bigr) ^ {2} \right]. \\ \end{array}
$$
