Since F is Hurwitz, there exists Lyapunov function

$$V (\boldsymbol {z}) = \sqrt {\boldsymbol {z} ^ {T} \boldsymbol {P} \boldsymbol {z}} \tag {43}$$

with $P > 0$ satisfying

$$\boldsymbol {P} \boldsymbol {F} + \boldsymbol {F} ^ {T} \boldsymbol {P} = - \boldsymbol {I}.$$

Suppose that the domain of analysis is $z \in \mathcal { D }$ , where $\mathcal { D } = \{ z \in \mathbb { R } ^ { 6 } | \| z \| \le r _ { z } \}$ . Then, the time-derivative of V along trajectories of the (37) satisfies

$$\dot {V} = \left(- \frac {1}{2} \| \boldsymbol {z} \| ^ {2} + \boldsymbol {z} ^ {T} \boldsymbol {P} \boldsymbol {g} (\boldsymbol {z}, t)\right) / V\leq - \frac {\sigma \| \boldsymbol {z} \| ^ {2}}{V} + \frac {b \| \boldsymbol {z} \| \| \tilde {\boldsymbol {\theta}} \|}{V}. \tag {44}$$

where $\begin{array} { r } { \sigma = \frac 1 { 2 \lambda } - \kappa _ { 1 } r _ { \theta } , b = \bar { \lambda } ( \kappa _ { 2 } r _ { z } ^ { 2 } + \kappa _ { 0 } ) } \end{array}$ , and $\overline { { \lambda } } = \lambda _ { \mathrm { m a x } } ( P )$ . Using the lower and upper bounds of the Lyapunov function,

$$\sqrt {\underline {{\lambda}}} \| \boldsymbol {z} \| \leq V \leq \sqrt {\overline {{\lambda}}} \| \boldsymbol {z} \|, \tag {45}$$

where $\underline { { \lambda } } = \lambda _ { \operatorname* { m i n } } ( P )$ , into (44) yields the following Bernoulli differential inequality

$$\dot {V} < - \sigma V + \frac {b}{\sqrt {\underline {{\lambda}}}} \| \tilde {\boldsymbol {\theta}} \|. \tag {46}$$

Here, the coefficient σ is positive if

$$r _ {\theta} < \frac {1}{2 \kappa_ {1} \overline {{\lambda}}} \tag {47}$$

is satisfied. Then, according to the comparison lemma [46, 47, p. 222], the solution of (46) must satisfy the inequality

$$V \leq V (0) e ^ {- \sigma t} + \frac {b}{\sqrt {\underline {{\lambda}}}} \int_ {0} ^ {t} e ^ {- \sigma (t - \tau)} \| \tilde {\boldsymbol {\theta}} (\tau) \| d \tau .$$

Using the upper and lower limits (45) in the above inequality gives

$$\| \boldsymbol {z} (t) \| \leq \| \boldsymbol {z} (0) \| \sqrt {\gamma} e ^ {- \sigma t} + \frac {b}{\underline {{\lambda}}} \int_ {0} ^ {t} e ^ {- \sigma (t - \tau)} \| \tilde {\boldsymbol {\theta}} (\tau) \| d \tau \tag {48}$$

where $\gamma = \overline { { \lambda } } / \underline { { \lambda } }$ is the condition number of matrix P . It can be verified that

$$\| \boldsymbol {z} (0) \| \leq \frac {1}{\sqrt {\gamma}} r _ {z} \quad \text { and } \quad r _ {\theta} \leq \frac {\sigma \underline {{\lambda}}}{b} r _ {z} \tag {49}$$
