$$\dot {H} _ {d} (q, \dot {q}) = - \dot {q} ^ {T} J _ {c} (q) ^ {T} \mathrm{diag} (b) J _ {c} (q) \dot {q} + \dot {z} _ {e} ^ {T} F _ {e} \leq \dot {z} _ {e} ^ {T} F _ {e}$$

where the inequality follows from diag(b)  0. This proves passivity.

We now show the local gain inequality. Recall that $z _ { c } =$ $h _ { c } ( q )$ , take $z _ { i } ^ { * } ~ = ~ h _ { c } ( q _ { i } ^ { * } )$ ), and consider the local storage function

$$
V (z _ {c} - z _ {i} ^ {*}, \dot {z}) = \left[ \begin{array}{c} z _ {c} - z _ {i} ^ {*} \\ \dot {z} _ {c} \end{array} \right] ^ {T} P \left[ \begin{array}{c} z _ {c} - z _ {i} ^ {*} \\ \dot {z} _ {c}, \end{array} \right]
$$

where P is defined in (9). Then, (8) implies that there exists some (small) ε > 0 such that

$$\dot {V} \leq - \frac {1}{\gamma} \left(| W _ {1} (z _ {d} - h _ {d} (q _ {i} ^ {*})) | _ {2} ^ {2} + | W _ {2} \dot {z} _ {d} | _ {2} ^ {2}\right) + \gamma | F _ {d} | _ {2} ^ {2}$$

as long as $| z _ { c } - z _ { i } ^ { * } | + | \dot { z } _ { c } | \leq \varepsilon$ . For the initial condition $z _ { c } ( 0 ) = z _ { i } ^ { * }$ , this implies that

$$\int_ {0} ^ {t} | W _ {1} (z _ {d} (\tau) - h _ {d} (q _ {i} ^ {*})) | _ {2} ^ {2} + | W _ {2} \dot {z} _ {d} (\tau) | _ {2} ^ {2} d \tau \leq \gamma^ {2} \int_ {0} ^ {t} | F _ {d} (\tau) | _ {2} ^ {2} d \tau$$

Thus, if the perturbation $F _ { d }$ is small enough to keep $| z _ { c } ( t ) -$ $z _ { i } ^ { * } | + | \dot { z } _ { c } ( t ) | \le \varepsilon$ for all $t \geq 0 ,$ we can extend the interval of integration to $t \to \infty$ , arriving at

$$
\left\| \left[ \begin{array}{c} W _ {1} (z _ {d} - h _ {d} (q _ {i} ^ {*})) \\ W _ {2} \dot {z _ {d}} \end{array} \right] \right\| _ {2} ^ {2} \leq \gamma^ {2} \| F _ {d} \| _ {2} ^ {2}  .
$$
