# C. Adaptive Control of Robot Manipulators

Most of the adaptive control design for trajectory tracking tasks of robot manipulators is based on the certainty equivalence principle (CE) adaptive updating law proposed in [24]. The CE based control and parameter update laws are

$$\boldsymbol {\tau} = \mathbf {Y} \hat {\pi} - \mathbf {K} _ {D} \mathbf {s}, \quad \dot {\hat {\pi}} = - \boldsymbol {\Gamma} \mathbf {Y} ^ {T} \mathbf {s},$$

with the tracking error defined on the sliding surface ${ \bf s } =$ $\dot { \tilde { \mathbf { q } } } + \Sigma \tilde { \mathbf { q } }$ , the position $\tilde { \mathbf { q } } = \mathbf { q } - \mathbf { q } _ { d } ,$ and velocity tracking error $\dot { \tilde { \mathbf { q } } } = \dot { \mathbf { q } } - \dot { \mathbf { q } } _ { d }$ , the desired position $\mathbf { q } _ { d } ( t )$ , a gain matrix Σ to determine the sliding surface, a gain matrix Γ for the parameter updating law and a gain matrix $\mathbf { K } _ { D }$ for the sliding mode tracking error. In [24], the authors also discussed the case only updating part of the parameters, showing that an additional sliding mode controller $\boldsymbol { \tau } = \mathbf { Y } \hat { \boldsymbol { \pi } } - \mathbf { K } _ { D } \mathbf { s } - \mathbf { k } \mathrm { s } \mathrm { g n } ( \mathbf { s } )$ can guarantee the asymptotic tracking performance.
