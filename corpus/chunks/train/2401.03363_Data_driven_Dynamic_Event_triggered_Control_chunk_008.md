# III. MAIN RESULTS

In this section, a data-driven dynamic event-triggered control scheme is developed, and some necessary discussions are provided.

The event sequence is denoted as $\{ t _ { k } \} _ { k \in \mathbb { N } } .$ , with $t _ { 0 } = 0$ , which is determined by the following event-triggering rule:

$$t _ {k + 1} = \inf \left\{t > t _ {k} \mid f (t) \leq 0 \right\}, \tag {4}$$

where $f ( t )$ represents the dynamic event-triggering function. For $t \in [ t _ { k } , t _ { k + 1 } )$ , the update algorithm of $f ( t )$ is proposed as

$$\dot {f} (t) = \min \left\{- z (t) ^ {T} \Phi z (t), 0 \right\} - f (t), f (t _ {k} ^ {+}) = \bar {f}, \quad (5)$$

where the constant $\bar { f } > 0 .$ , and $\boldsymbol { z } ( t ) = [ \boldsymbol { x } ( t ) ^ { T } ~ \boldsymbol { e } ( t ) ^ { T } ] ^ { T }$ with

$$e (t) = x (t _ {k}) - x (t), \tag {6}$$

and

$$
\Phi = \left[ \begin{array}{c c} - \alpha \mathbf {I} & \mathbf {0} \\ \mathbf {0} & \beta \mathbf {I} \end{array} \right], \tag {7}
$$

where the constants $\alpha , \beta > 0$ are determined by a data-driven LMI to be proposed later. Once an event is triggered by (4), $e ( t )$ will be 0 and f (t) will be instantly reset to ${ \bar { f } } .$ It is easy to verify that $\dot { f } ( t ) \leq 0$ and $0 \leq f ( t ) \leq \bar { f } .$

Using the state information sampled at event instants, the event-based controller is proposed as

$$u (t) = K x \left(t _ {k}\right), t \in \left[ t _ {k}, t _ {k + 1}\right), \tag {8}$$

where $K \in \mathbb { R } ^ { m \times n }$ represents the data-driven feedback gain matrix designed off-line, of the form

$$K = \mathcal {U} _ {0} \mathcal {Y} (\mathcal {X} _ {0} \mathcal {Y}) ^ {- 1}, \tag {9}$$

where the matrix $\mathcal { V } \in \mathbb { R } ^ { \tau \times n }$ is a solution of

$$
\left[ \begin{array}{c c} \mathcal {X} _ {1} \mathcal {Y} + (\mathcal {X} _ {1} \mathcal {Y}) ^ {T} + \Omega + \gamma \Delta \Delta^ {T} & \mathcal {Y} ^ {T} \\ \mathcal {Y} & - \gamma \mathbf {I} \end{array} \right] \prec 0, \tag {10}
\mathcal {X} _ {0} \mathcal {Y} \succ 0, \tag {11}$$

where $\gamma > 0$ is a proper constant, and $\Omega \succ 0$ is a given matrix.

Let $\mathcal { P } = ( \mathcal { X } _ { 0 } \mathcal { Y } ) ^ { - 1 }$ , which is positive definite under (11), and let $\mathcal { Q } \in \mathbb { R } ^ { \tau \times n }$ be any solution of

$$
\left[ \begin{array}{c} K \\ \mathbf {0} \end{array} \right] = \left[ \begin{array}{c} \mathcal {U} _ {0} \\ \mathcal {X} _ {0} \end{array} \right] \mathcal {Q}, \tag {12}
$$

which exists under Assumption 3.
