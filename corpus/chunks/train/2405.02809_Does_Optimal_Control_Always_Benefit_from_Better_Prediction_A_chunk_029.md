# 4.4. Predictor Evaluation Using Available Data

With one $\{ o , \bar { w } \}$ pair, given a predictor ${ \mathcal P } ,$ we can compute a belief $\bar { W } _ { b } =$ ${ \mathcal { P } } ( o )$ . Using a beleif $\bar { W } _ { b }$ and the corresponding realized disturbance sequence $\bar { w }$ , we can define some one-time prediction performance measures. With multiple $\{ o , \bar { w } \}$ pairs, the predictor ${ \mathcal { P } } _ { \mathrm { { s } } }$ performance can be evaluated by aggregating these one-time prediction performance measures.
