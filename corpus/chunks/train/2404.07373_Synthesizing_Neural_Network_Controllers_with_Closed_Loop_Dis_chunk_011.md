# 2.3 Integral Quadratic Constraints

The block $\Delta _ { p }$ may represent a variety of uncertainties, including unmodeled dynamics, nonlinearities, and uncertain parameters. To characterize $\Delta _ { p }$ in a manner convenient for dissipativity analysis, we describe the relationship between its inputs and outputs using a form of

time-domain integral quadratic constraints (IQCs) [11] [17] [18] [21].

Define a stable filter $\Psi _ { p }$ by

$$
\left[ \begin{array}{c} \dot {\boldsymbol {\psi}} _ {\boldsymbol {p}} (t) \\ \boldsymbol {z} _ {\boldsymbol {p}} (t) \end{array} \right] = \left[ \begin{array}{c c c} A _ {\psi} & B _ {\psi v} & B _ {\psi w} \\ C _ {\psi} & D _ {\psi v} & D _ {\psi w} \end{array} \right] \left[ \begin{array}{c} \boldsymbol {\psi} _ {\boldsymbol {p}} (t) \\ \boldsymbol {v} _ {\boldsymbol {p}} (t) \\ \boldsymbol {w} _ {\boldsymbol {p}} (t) \end{array} \right] \tag {7}
$$

where $\psi _ { p } ( 0 ) = 0$ . Then, with $M _ { \Delta _ { p } } = M _ { \Delta _ { p } } ^ { \top }$ , we say $\Delta _ { p }$ satisfies the IQC defined by $( \Psi _ { p } , M _ { \Delta _ { p } } )$ if

$$\int_ {0} ^ {T} \boldsymbol {z} _ {\boldsymbol {p}} (t) ^ {\top} M _ {\Delta_ {p}} \boldsymbol {z} _ {\boldsymbol {p}} (t) d t \geq 0 \quad \forall T \geq 0 \tag {8}$$

for all $v _ { p }$ and ${ \pmb w _ { p } = \Delta _ { p } ( \pmb v _ { p } ) }$ . This is often referred to as $\mathrm { a \ ^ { * } h a r d ^ { , * } I Q C \ [ 1 i ] }$ . In this paper, we only use hard IQCs, so we henceforth refer to them just as IQCs.

As a special case, we say $\Delta _ { p }$ satisfies a static IQC defined by $M _ { \Delta _ { p } }$ if $\Delta _ { p }$ satisfies the IQC defined by $( I , M _ { \Delta _ { p } } )$ . This is the case where $z _ { p } ( t ) = \Big \lceil \pmb { v _ { p } } ( t ) ^ { \top } \ \pmb { w _ { p } } ( t ) ^ { \top } \Big \rceil ^ { \top }$ . As a further special case, we say $\Delta _ { p }$ satisfies the quadratic constraint (QC) defined by $\overline { { { M _ { \Delta _ { p } } } } }$ if

$$
\left[ \begin{array}{c} \boldsymbol {v _ {p}} (t) \\ \boldsymbol {w _ {p}} (t) \end{array} \right] ^ {\top} M _ {\Delta_ {p}} \left[ \begin{array}{c} \boldsymbol {v _ {p}} (t) \\ \boldsymbol {w _ {p}} (t) \end{array} \right] \geq 0 \quad \forall t \geq 0.
$$

We say dynamic IQC to refer to IQCs with filters when it is not clear from context if a special case of IQC is being referred to.

The controller nonlinearity $\phi ,$ due to each scalar nonlinearity being sector-bounded in [0, 1], satisfies the quadratic constraint defined by

$$
M _ {\phi} = \left[ \begin{array}{c c} 0 & \Lambda \\ \Lambda & - 2 \Lambda \end{array} \right] \tag {9}
$$

where $\Lambda \succeq 0$ and diagonal [5].
