# VII. RELATION TO ROBUST $H _ { 2 }$ SYNTHESIS

The problem studied in this article is closely related to a specific robust $H _ { 2 }$ synthesis problem. Indeed, we show in this section that the robust analysis problem (13) can be interpreted as robust $H _ { 2 }$ analysis for an uncertain system interconnection depicted in Figure 1.

This uncertain system interconnection is given by a linear (closed-loop) system with a real block $\Delta$ in the input of the performance channel, i.e., we study the uncertain system

$$
\binom {\chi_ {k + 1}} {z _ {k + 1}} = \left( \begin{array}{c c} \mathcal {A} & \mathcal {B} \Delta \\ \mathcal {C} & \mathcal {D} \Delta \end{array} \right) \binom {\chi_ {k}} {\hat {w} _ {k}} = \left( \begin{array}{c c} \mathcal {A} & \mathcal {B} _ {\Delta} \\ \mathcal {C} & \mathcal {D} _ {\Delta} \end{array} \right) \binom {\chi_ {k}} {\hat {w} _ {k}}. \tag {16}
$$

Here, $\boldsymbol { B } _ { \Delta }$ and $\mathcal { D } _ { \Delta }$ denote perturbed versions of the matrices B and D, respectively. The goal is to analyze the $\Sigma _ { \mathrm { n o m } }$ -weighted $H _ { 2 }$ norm of the transfer function from wˆ to z, which is the closed-loop transfer function of a system with a controller. With $\scriptstyle \sum _ { \mathrm { n o m } ^ { - } } \mathrm { w e i g h t e d }$ , we mean that the input matrices $\boldsymbol { B } _ { \Delta }$ and $\mathcal { D } _ { \Delta }$ are scaled with the static weight $\scriptstyle \sum _ { \mathrm { n o m } } ^ { \frac { 1 } { 2 } }$ . In this course, we invoke the following result from robust control theory [23].

Theorem 7.1 (Bounding the $H _ { 2 }$ norm [23]): Let A be stable. The $\left( \Sigma _ { \mathrm { n o m } } \mathrm { - w e i g h t e d } \right) H _ { 2 }$ norm of (16) is smaller than $\rho$ if and only if there exists a matrix $\mathcal { P } \succ 0$ such that

$$\mathcal {A} ^ {\top} \mathcal {P} \mathcal {A} - \mathcal {P} + \mathcal {C} ^ {\top} \mathcal {C} < 0, \tag {17a}\operatorname{tr} \left(\left(\mathcal {B} _ {\Delta} ^ {\top} \mathcal {P} \mathcal {B} _ {\Delta} + \mathcal {D} _ {\Delta} ^ {\top} \mathcal {D} _ {\Delta}\right) \Sigma_ {\text {nom}}\right) < \rho^ {2}. \tag {17b}$$

We assume that $\Delta$ is fixed but unknown except for the fact that it satisfies $\Vert ( I - \Delta ) \Sigma _ { \mathrm { n o m } } ^ { \frac { 1 } { 2 } } \Vert _ { F } \leq \gamma$ , i.e., it is γ-close to the identity matrix in the weighted Frobenius norm.
