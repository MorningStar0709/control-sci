# II. SYSTEM MODEL AND NOTATION

Let |S| be the cardinality of a set S. We use bold symbols to represent trajectories and calligraphic symbols for sets. For $z \in \mathbb { R }$ , we define $[ z ] _ { + } = \operatorname* { m a x } \{ 0 , z \}$ and $[ z ] _ { - } = - [ - z ] _ { + }$ . Consider a team of agents labelled from a finite set J . Let Cap denote a finite set of agent capabilities. We assume that the agents operate in a continuous workspace $S \subseteq \mathbb { R } ^ { n _ { s } }$ .

Definition 1. An agent $j \in \mathcal { I }$ is a tuple $\begin{array} { r l } { A _ { j } } & { { } = } \end{array}$ $\langle { \mathcal { X } } _ { j } , { \boldsymbol { x } } _ { j } ( 0 ) , C a p _ { j } , { \mathcal { U } } _ { j } , f _ { j } , l _ { j } \rangle$ where: $\mathcal { X } _ { j } \subseteq \mathbb { R } ^ { n _ { x , j } }$ is its state space; $x _ { j } ( 0 ) \in \mathcal { X } _ { j }$ is its initial state; $C a p _ { j } \ \subseteq \ C a p$ is its finite set of capabilities; $\mathcal { U } _ { j } \ \subseteq \ \mathbb { R } ^ { n _ { u , j } }$ is its control space; $f _ { j } : \mathcal { X } _ { j } \times \mathcal { U } _ { j } \to \mathcal { X } _ { j }$ is a differentiable function giving the discrete time dynamics of agent j:

$$x _ {j} (t + 1) = f _ {j} \big (x _ {j} (t), u _ {j} (t) \big), \quad t = 0, 1, \dots , H - 1, \tag {1}$$
