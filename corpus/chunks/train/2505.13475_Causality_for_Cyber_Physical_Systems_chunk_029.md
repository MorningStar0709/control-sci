Note that, since the braking coefficient is constant, the valuation of brakes over time is a constant function. To discretise time in order to perform causal analysis and identify when the cause occurred, we first need to define the notion of time interval.

Definition 11 (Time interval). A left-closed right-open interval $[ i , j )$ , where $\textit { i } < \textit { j }$ , is defined as a convex subset $o f \mathbb { R } _ { + }$ , such that, ∀x $\in \mathbb { R } _ { + } , x \in [ i , j ) \iff i \leq x < j$ .

In this work, a cause is defined using sub-trajectories of variables X over one or more time intervals, as shown in Figure 4. To formalise that, we define trajectory slices.

Definition 12 (Trajectory slice). Given a set of variables $\vec { V } _ { ; }$ , a trajectory $x \in T r a j s ( \vec { V } )$ , and an interval $[ i , j )$ such that $[ i , j ) \subseteq \mathrm { d o m } ( x )$ , a trajectory slice $x _ { [ i , j ) } \in T r a j s ( \vec { V } )$ is defined as a function $x _ { [ i , j ) } : [ i , j )  V a l ( \vec { V } )$ , such that $\forall t \in [ i , j ) , x _ { [ i , j ) } ( t ) = x ( t )$ .
