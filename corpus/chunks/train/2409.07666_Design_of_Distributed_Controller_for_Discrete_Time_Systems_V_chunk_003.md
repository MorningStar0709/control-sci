# II. PROBLEM STATEMENT

Consider the system with

$$x (k + 1) = A x (k) + B u (k), \tag {1}$$

consisting of N subsystems, where $x = [ x _ { 1 } ^ { \top } , \cdots , x _ { N } ^ { \top } ] ^ { \top } \in \mathbb { R } ^ { n }$ , · · · , x⊤N ] , the state of i-th subsystem xi ∈ Rni with P i=1 ni = n, u = $x _ { i } \in \mathbb { R } ^ { n _ { i } }$ $\begin{array} { r } { \sum _ { i = 1 } ^ { N } n _ { i } = n , } \end{array}$ N $[ u _ { 1 } ^ { \top } , \cdot \cdot \cdot \ l , u _ { N } ^ { \top } ] ^ { \top } \in \mathbb { R } ^ { \dot { m } }$ u N ] , the input of i-th subsystem $u _ { i } \in \mathbb { R } ^ { m _ { i } }$ with N $\textstyle \sum _ { i = 1 } ^ { N } m _ { i } = m , A \in \mathbb { R } ^ { n \times n }$ , and $B \in \mathbb { R } ^ { n \times m }$ . We assume that $( A , B )$ is stabilizable. Suppose that a communication network between subsystems is modeled by a time-invariant undirected graph $\mathcal { G } = ( \mathcal { N } , \mathcal { E } )$ with $\mathcal { N } = \{ 1 , \ldots , N \}$ and an edge set $\mathcal { E } .$ Namely, pairs of different nodes $i , j \in \mathcal N$ satisfy $( i , j ) \in \mathcal { E } \Leftrightarrow ( j , i ) \in \mathcal { E } .$ A distributed state feedback controller is defined as

$$u (k) = K x (k), K \in \mathcal {S}, \tag {2}$$

where S represents the set of matrices with graph-induced sparsity defined as

$$\mathcal {S} = \left\{K \in \mathbb {R} ^ {m \times n}: K _ {i j} = O _ {m _ {i} \times n _ {j}} \text { if } (i, j) \notin \mathcal {E}, i \neq j \right\}. \tag {3}$$

For simplicity in the following notation, we assume $n _ { i } = m _ { i }$ for all $i \in \mathcal N$ without loss of generality. Please refer to Appendix A for the generalized notation where $n _ { i } \neq m _ { i }$ .

The set of all state feedback gains K that stabilize the system (1), denoted by $\kappa _ { \mathrm { a l l } }$ , can be characterized as follows:

$$\mathcal {K} _ {\text {all}} = \{K \in \mathcal {S}: \exists P \succ O \tag {4}\text { s.t. } (A + B K) ^ {\top} P (A + B K) - P \prec 0 \}.$$

Conventionally, this problem would be convexified using the Schur complement without the constraints on the distributedness, $\kappa \in S$ . This involves transforming the matrix variables $Q = P ^ { - 1 }$ and $K = Z Q ^ { - 1 }$ . This convexification is ineffective for (4) because $K \in S$ is transformed into a non-convex constraint, $Z Q ^ { - 1 } \in { \mathcal { S } } .$ To the best of the authors’ knowledge, an exact convex formulation of this problem has not yet been found.
