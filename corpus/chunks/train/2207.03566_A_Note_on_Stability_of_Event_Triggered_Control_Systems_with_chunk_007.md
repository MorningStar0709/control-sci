# III. Halanay-Type Inequality

For a continuous function $g : \mathbb { R }  \mathbb { R }$ , the Dini-derivatives $\mathrm { D } ^ { + } g ( t )$ and $\mathrm { D } _ { - } g ( t )$ are defined as follows:

$$\mathrm{D} ^ {+} g (t) = \limsup _ {\varepsilon \to 0 ^ {+}} \frac {g (t + \varepsilon) - g (t)}{\varepsilon}$$

and

$$\mathrm{D} _ {-} g (t) = \liminf _ {\varepsilon \to 0 ^ {-}} \frac {g (t + \varepsilon) - g (t)}{\varepsilon}.$$

The following lemma from [3] provides a relationship between $\mathrm { D } ^ { + } g ( t )$ and $\mathbf { D } _ { - } g ( t )$ , which will be used to construct our Halanay-type inequality.

Next, we introduce a new Halanay-type inequality which allows us to bound the states of the event-triggered control system with time delays.

Lemma 2. Let $g : [ t _ { 0 } - r , t _ { 0 } + \Gamma ) \to \mathbb { R } ^ { + }$ be a continuous function satisfying

$$\mathrm{D} ^ {+} g (t) \leq \gamma_ {1} g (t _ {0}) + \gamma_ {2} \| g _ {t} \| _ {r} \quad f o r \quad t _ {0} \leq t < t _ {0} + \Gamma \tag {8}$$

where $r , \Gamma , \gamma _ { 1 }$ , and $\gamma _ { 2 }$ are positive constants. Then

$$g (t) \leq \| g _ {t _ {0}} \| _ {r} e ^ {\lambda (t - t _ {0})} \quad f o r \quad t _ {0} \leq t < t _ {0} + \Gamma$$

where $\lambda = \gamma _ { 1 } + \gamma _ { 2 } .$

Proof. Define

$$
w (t) = \left\{ \begin{array}{l l} \| g _ {t _ {0}} \| _ {r} e ^ {\lambda (t - t _ {0})}, & \text { if } t _ {0} <   t <   t _ {0} + \Gamma \\ \| g _ {t _ {0}} \| _ {r}, & \text { if } t _ {0} - r \leq t \leq t _ {0} \end{array} \right.
$$

and let $K > 1$ be an arbitrary constant. Then, for $t \in [ t _ { 0 } - r , t _ { 0 } ]$ , we have

$$g (t) \leq \| g _ {t _ {0}} \| _ {r} = w (t) < K w (t), \tag {9}$$

that is, $g ( t ) < K w ( t )$ for $t \in [ t _ { 0 } - r , t _ { 0 } ]$ .

Next, we use a contradiction argument to show that $g ( t ) < K w ( t )$ for $t \in ( t _ { 0 } , t _ { 0 } + \Gamma )$ ). Suppose there exits some $t \in ( t _ { 0 } , t _ { 0 } + \Gamma )$ such that $g ( t ) \geq K w ( t )$ , then we define

$$\bar {t} = \inf \left\{t \in (t _ {0}, t _ {0} + \Gamma) \mid g (t) \geq K w (t) \right\}.$$

From the continuities of g and w, we have

$$g (t) < K w (t) \text { for } t _ {0} < t < \bar {t} \tag {10}$$

and

$$g (\bar {t}) = K w (\bar {t}). \tag {11}$$
