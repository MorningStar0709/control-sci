# Initial Choice of d and $\mathbf { \nabla } \mathbf { \pmb { n } } _ { B }$

If no knowledge of the time delay is available, $d = 0$ is chosen as an initial value. If a minimum value is known, an initial value $d = d _ { \operatorname* { m i n } }$ is chosen. If the time delay has been underestimated, during identification the first coefficients of $B ( q ^ { - 1 } )$ will be very low. $n _ { B }$ must then be chosen so that it can both indicate the presence of the time delays and identify the transfer function numerator. $n _ { B } = ( d _ { \operatorname* { m a x } } - d _ { \operatorname* { m i n } } ) + 2$ is then chosen as the initial value. At least two coefficients are required in $B ( q ^ { - 1 } )$ because of the fractional delay which is often present in applications. If the time delay is known, $n _ { B } \ge 2$ is chosen, but 2 remains a good initial value.
