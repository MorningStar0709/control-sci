# A. System and Safety Filter

Consider the linear system

$$\dot {x} = A x + B u,$$

where $x \in \mathbb { R } ^ { n }$ and $u \in \mathbb { R } ^ { m }$ . Let $u _ { \mathrm { n o m } } ( x ) = - K x$ be a nominal state-feedback controller, and define $A _ { 0 } : = A - B K$ . Throughout the paper, we assume that $A _ { 0 }$ is Hurwitz. Safety is specified by the affine constraint $h ( x ) : = c ^ { \top } x + d \geq 0 ,$ with associated safe set

$$\mathcal {C} := \{x \in \mathbb {R} ^ {n}: h (x) \geq 0 \}. \tag {1}$$

We assume $d > 0 ,$ , so that the origin lies in the interior of C, and $c ^ { \top } B \neq 0 ,$ , so that the input enters the barrier derivative. To enforce safety, we use the exponential CBF condition

$$c ^ {\top} (A x + B u) + \alpha h (x) \geq 0, \tag {2}$$

where $\alpha > 0 .$ . The safety-filtered input is defined by the quadratic program

$$u ^ {\star} (x) = \underset {u \in \mathbb {R} ^ {m}} {\arg \min} \frac {1}{2} (u + K x) ^ {\top} G (u + K x) \tag {3}\text { s.t. } c ^ {\top} (A x + B u) + \alpha h (x) \geq 0,$$

where $G = G ^ { \top } \succ 0$ .

For every x, (3) admits a unique optimizer, since the objective is strictly convex and the constraint is affine. It therefore returns the closest safe input to $u _ { \mathrm { n o m } } ( x )$ in the G-weighted norm.
