At $\omega = 0$ , the inequality (24) is reformulated as:

$$\Delta V _ {k} + | | e _ {k} | | ^ {2} \leq 0.$$

It yields the exponential stability condition

$$\Delta V _ {k} \leq - \sigma V (e _ {k}),$$

along with $\begin{array} { r } { \sigma = \frac { 1 } { \lambda _ { \operatorname* { m a x } } ( P ) } > 0 } \end{array}$ . Hence, the proposed LMIs ensure the exponential stability of the error dynamic (19) when $\omega = 0$ .
