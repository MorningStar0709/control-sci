Filter-aware MPC increases the success rate to 64%. We learn the trackability using 900 rollouts of the MPC policy, having 200 time steps each. We plot the learned trackability function and a set of filter-aware rollouts in fig. 2 (c). The filter-aware controller avoids the left corridor in all cases. We compare filter-aware MPC, vanilla MPC and vanilla-easy, which is vanilla MPC run on a setup where both corridors are safe in fig. 2 (d). Filter-aware MPC outperforms vanilla MPC by 16% when both are run on the same problem and approaches the performance of vanilla-easy, which achieves a success rate of 69%.
