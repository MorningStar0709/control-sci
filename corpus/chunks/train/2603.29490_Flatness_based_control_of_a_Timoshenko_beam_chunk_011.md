# B. Derivation of the parametrization

The solution of the spatial Cauchy problem associated with the representation (10) is straightforward thanks to the cascaded transport equations in (10a) and (10b). Integration along the characteristics yields

$$\bar {x} _ {1} ^ {-} (z, t) = \bar {x} _ {1} ^ {-} (0, t + \phi_ {1} (z)) \tag {14a}\bar {x} _ {2} ^ {-} (z, t) = \bar {x} _ {2} ^ {-} (0, t + \phi_ {2} (z))- \int_ {0} ^ {z} \frac {a _ {0} ^ {-} (\zeta)}{\lambda_ {2} (\zeta)} \bar {x} _ {1} ^ {-} (0, t + \phi_ {2} (z) - \phi_ {2} (\zeta)) d \zeta \tag {14b}\bar {x} _ {1} ^ {+} (z, t) = \bar {x} _ {1} ^ {+} (0, t - \phi_ {1} (z)) \tag {14c}\bar {x} _ {2} ^ {+} (z, t) = \bar {x} _ {2} ^ {+} (0, t - \phi_ {2} (z))+ \int_ {0} ^ {z} \frac {a _ {0} ^ {+} (\zeta)}{\lambda_ {2} (\zeta)} \bar {x} _ {1} ^ {+} (0, t - \phi_ {2} (z) + \phi_ {2} (\zeta)) d \zeta , \tag {14d}$$

which depends on the boundary values ${ \bar { \pmb x } } ^ { - } ( 0 , t )$ and ${ \bar { \pmb x } } ^ { + } ( 0 , t )$ , respectively. The boundary value ${ \bar { \pmb x } } ^ { - } ( 0 , t )$ can be replaced with ${ \bar { \pmb x } } ^ { + } ( 0 , t )$ by using the BC (10c), after which the solutions are rewritten in terms of ${ \bar { \pmb x } } ^ { + } ( 1 , t )$ , exploiting the transport character of (10a) and (10b). This allows us to parametrize the state $\bar { \mathbf { x } } ( \cdot , t )$ by the flat output

$$\boldsymbol {y} (t) = \bar {\boldsymbol {x}} ^ {+} (1, t) \tag {15}$$

resulting in

$$\bar {x} _ {1} ^ {-} (z, t) = - y _ {1} \left(t + \tau_ {1} + \phi_ {1} (z)\right) \tag {16a}
\begin{array}{l} \bar {x} _ {2} ^ {-} (z, t) = - y _ {2} \left(t + \tau_ {2} + \phi_ {2} (z)\right) \\ + \int_ {0} ^ {1} \frac {a _ {0} ^ {+} (\zeta)}{\lambda_ {2} (\zeta)} y _ {1} (t + \tau_ {1} + \phi_ {2} (z) + \phi_ {2} (\zeta)) d \zeta \\ + \int_ {0} ^ {z} \frac {a _ {0} ^ {-} (\zeta)}{\lambda_ {2} (\zeta)} y _ {1} (t + \tau_ {1} + \phi_ {2} (z) - \phi_ {2} (\zeta)) d \zeta \tag {16b} \\ \end{array}
\bar {x} _ {1} ^ {+} (z, t) = y _ {1} \left(t + \tau_ {1} - \phi_ {1} (z)\right) \tag {16c}\bar {x} _ {2} ^ {+} (z, t) = y _ {2} \left(t + \tau_ {2} - \phi_ {2} (z)\right)- \int_ {z} ^ {1} \frac {a _ {0} ^ {+} (\zeta)}{\lambda_ {2} (\zeta)} y _ {1} (t + \tau_ {1} - \phi_ {2} (z) + \phi_ {2} (\zeta)) d \zeta . \tag {16d}$$
