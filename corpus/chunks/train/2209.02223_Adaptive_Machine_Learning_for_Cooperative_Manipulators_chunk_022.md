# Appendix A

Consider the following linear parametrization form

$$\boldsymbol {y} _ {k} = \boldsymbol {W} _ {k} \boldsymbol {a} _ {k} + \boldsymbol {e} _ {k}. \tag {52}$$

The least-squares method finds the current parameter estimate $\hat { \pmb { a } } _ { k }$ which minimizes the cost function

$$J = \sum_ {i = 1} ^ {k} e ^ {- \mu (t _ {k} - t _ {i})} \| (\boldsymbol {y} _ {i} - \boldsymbol {W} _ {i} \hat {\boldsymbol {a}} _ {k}) \| ^ {2} \tag {53}$$

The parameter update law is

$$\hat {\boldsymbol {a}} _ {k + 1} = \hat {\boldsymbol {a}} _ {k} + \boldsymbol {K} _ {k} (\boldsymbol {y} _ {k} - \hat {\boldsymbol {y}} _ {k}) \tag {54}$$

where $\hat { \pmb { y } } _ { k } = { \pmb { W } } _ { k } \hat { \pmb { a } } _ { k } , \varrho = e ^ { - \mu h }$ , and

$$\boldsymbol {K} _ {k} = \boldsymbol {P} _ {k} \boldsymbol {W} _ {k} ^ {T} \left(\varrho \boldsymbol {I} + \boldsymbol {W} _ {k} \boldsymbol {P} _ {k} \boldsymbol {W} _ {k} ^ {T}\right) ^ {- 1}\boldsymbol {P} _ {k + 1} = (\boldsymbol {I} - \boldsymbol {K} _ {k} \boldsymbol {W} _ {k}) \boldsymbol {P} _ {k} / \varrho$$

are the gain and covariance matrices.
