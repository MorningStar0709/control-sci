2) Constraint for Inter-Agent Collision Avoidance: For realistic deployment, we also incorporate collision avoidance between agents to ensure safe navigation. In this work, we construct candidate CBFs similar to those in [13], [19] to enforce collision avoidance between agents i and j:

$$h _ {i j} ^ {\mathrm{col}} (\mathbf {x}) = \Delta_ {i j} (\mathbf {x}) ^ {2} - \Delta_ {d} ^ {2}. \tag {10}$$

When $h _ { i j } ^ { \mathrm { c o l } } ( { \bf x } ) \geq 0$ , this implies robots i and j are at least $\Delta _ { d }$ away from each other, ensured to be free of collision.
