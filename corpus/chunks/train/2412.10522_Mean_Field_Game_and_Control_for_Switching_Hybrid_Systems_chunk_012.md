$\forall n ~ \in ~ \left\{ 0 , \ldots , N _ { T } - 1 \right\} , ~ i ~ \in ~ \left\{ 0 , \ldots , N _ { X } \right\}$ with the appropriate boundary conditions in space and the terminal condition $U _ { i } ^ { N _ { T } } = \breve { g } ( x _ { i } , M ^ { N _ { T } } )$ .

Meanwhile, the discrete FPK equation (16) is expressed as

$$(D _ {t} M _ {i}) ^ {n} - \nu (\Delta_ {h} M ^ {n + 1}) _ {i} - \mathcal {T} _ {i} (U ^ {n}, M ^ {n + 1}) = 0, \tag {24}$$

$\forall n ~ \in ~ \left\{ 0 , \ldots , N _ { T } - 1 \right\} , ~ i ~ \in ~ \left\{ 0 , \ldots , N _ { X } \right\}$ starting from $M _ { i } ^ { 0 } = \stackrel { \cdot } { m } _ { 0 } ( x _ { i } )$ . Here, the discrete transport operator $\mathcal { T } _ { i }$ has the following structure

$$
\begin{array}{l} \mathcal {T} _ {i} (U, M) = \frac {1}{h} \left(M _ {i} \partial_ {p _ {1}} \tilde {H} (x _ {i}, M _ {i}, [ \nabla_ {h} U ] _ {i}) \right. \\ - M _ {i - 1} \partial_ {p _ {1}} \tilde {H} (x _ {i - 1}, M _ {i - 1}, [ \nabla_ {h} U ] _ {i - 1}) \\ + M _ {i + 1} \partial_ {p _ {2}} \tilde {H} (x _ {i + 1}, M _ {i + 1}, [ \nabla_ {h} U ] _ {i + 1}) \\ \left. - M _ {i} \partial_ {p _ {2}} \tilde {H} (x _ {i}, M _ {i}, [ \nabla_ {h} U ] _ {i})\right), \tag {25} \\ \end{array}
$$

which ensures that the total mass of the density matrix is constant over time.

Both the discrete HJB and FPK equations are implicit schemes in time. When (23), (24) are combined, the system of equations can be expressed as a root finding problem:

$$\varphi (Z) = 0, \tag {26}$$

${ \begin{array} { l } { { \mathrm { w h e r e } } \quad Z ~ = ~ ( U , M ) ~ { \mathrm { a n d } } \quad \varphi ~ : ~ \mathbb { R } ^ { 2 \times ( N _ { T } + 1 ) \times ( N _ { X } + 1 ) } \quad  } \\ { \mathbb { R } ^ { 2 \times ( N _ { T } + 1 ) \times ( N _ { X } + 1 ) } . } \end{array} }$

The simplest method to obtain the solution is the Picard iteration approach, where each of the equations is iteratively updated using the most recent estimates of the value function and the density. However, this procedure has significant challenges in terms of convergence to a solution. Therefore, the chosen robust approach is to solve via Newton iterations from an initial guess. Starting from $Z ^ { ( 0 ) }$ where $Z ^ { ( k ) } = ( U ^ { ( k ) } , M ^ { ( k ) } )$ ), the solution is the asymptotic iterate of

$$Z ^ {(k + 1)} = Z ^ {(k)} - J _ {\varphi} (Z ^ {(k)}) ^ {- 1} \varphi (Z ^ {(k)}). \tag {27}$$

where $J _ { \varphi } ( Z ^ { ( k ) } )$ is the Jacobian of $\varphi$ at $Z ^ { ( k ) }$
