# 2.2.1 Homogeneous functions

Below we study functions being symmetric with respect to a linear dilation d. The dilation symmetry introduced by the following definition is known as the generalized homogeneity [24], [12], [23], [2], [21].

Definition 2 [12] Let d be a dilation in $\mathbb { R } ^ { n }$ . A function $h : \mathbb { R } ^ { n } \mapsto \mathbb { R }$ is dhomogeneous of degree $\nu \in \mathbb { R } \ i f$

$$h (\mathbf {d} (s) x) = e ^ {\nu s} h (x), \quad \forall s \in \mathbb {R}, \quad \forall x \in \mathbb {R} ^ {n}. \tag {7}$$

Local behavior of any homogeneous function h can be expanded globally taking into account the homogeneity degree of the function. The sign of the homogeneity degree also defines some properties of the function [2].

Proposition 1 Let $h : \mathbb { R } ^ { n } \mapsto \mathbb { R }$ be a d-homogeneous function of degree $\nu \in \mathbb { R }$ , $h \neq \mathbf { 0 }$ and $\textstyle \operatorname* { s u p } _ { x \in S } | h ( x ) | < + \infty$ .

$$\lim _ {\| x \| \to 0} h (x) = 0 \quad a n d \quad \operatorname * {l i m s u p} _ {\| x \| \to + \infty} | h (x) | = + \infty .$$

• $I f \nu < 0$ then h is bounded on any set $\mathbb { R } ^ { n } \backslash B _ { \bf d } ( r )$ ,

$$\lim _ {\| x \| \to + \infty} h (x) = 0 \quad a n d \quad \operatorname * {l i m s u p} _ {\| x \| \to 0} | h (x) | = + \infty .$$

• $I f \deg _ { \bf d } ( h ) = 0$ then h is uniformly bounded on $\mathbb { R } ^ { n }$ ; moreover, $h \equiv \mathrm { c o n s t }$ $i f h$ is continuous at 0.
