$$\mathbb {E} \big [ \langle \nabla (f _ {t} - f _ {t; h}) (N _ {t - h}), N _ {t - h} - N ^ {\star} \rangle | \mathcal {F} _ {t - k} \big ] = \langle \mathbb {E} [ \nabla (f _ {t} - f _ {t; h}) (N _ {t - h}) | \mathcal {F} _ {t - h} ], N _ {t - h} - N ^ {\star} \rangle = 0,$$

where we also recall the definition $f _ { t ; h } ( N _ { t - k } ) = \mathbb { E } [ f _ { t } ( N _ { t - h } ) | \mathcal { F } _ { t - k } ]$ . We now upper bound the remaining two terms on the right-hand side of (C.5). We have

$$
\begin{array}{l} \sum_ {t = h} ^ {T - 1} \langle \nabla (f _ {t} - f _ {t; h}) (N _ {t - h}), N _ {t} - N _ {t - k} \rangle + \langle \nabla (f _ {t} - f _ {t; h}) (N _ {t}) - \nabla (f _ {t} - f _ {t; h}) (N _ {t - h}), N _ {t} - N ^ {\star} \rangle \\ \leq \sum_ {t = h} ^ {T - 1} \Big (\| \nabla (f _ {t} - f _ {t; h}) (N _ {t - h}) \| \| N _ {t} - N _ {t - h} \| \\ + \left(\| \nabla f _ {t} (N _ {t}) - \nabla f _ {t} (N _ {t - h}) \| + \| \nabla f _ {t; h} (N _ {t - h}) - \nabla f _ {t; h} (N _ {t}) \|\right) \| N _ {t} - N ^ {\star} \|\left. \right) \\ \stackrel {(a)} {\leq} \sum_ {t = h} ^ {T - 1} \left(2 l _ {y} \left\| N _ {t} - N _ {t - k} \right\| + 2 \beta_ {y} \left\| N _ {t} - N _ {t - h} \right\| \left\| N _ {t} - N ^ {\star} \right\|\right) \\ \stackrel {(b)} {\leq} 2 (l _ {y} + 2 R _ {N} \beta_ {y}) \sum_ {t = h} ^ {T - 1} \| N _ {t} - N _ {t - h} \|, \\ \end{array}
$$

where to obtain (a), we use the upper bound on $\lVert \nabla f _ { t } ( N _ { t - h } ) \rVert$ shown by Lemma 3.4, which also implies via the definition $f _ { t ; h } ( N _ { t - h } ) = \mathbb { E } [ f _ { t } ( N _ { t - h } ) | \mathcal { F } _ { t - h } ]$ that $\nabla f _ { t ; h } ( N _ { t - k } ) = \mathbb { E } [ \nabla f _ { t } ( N _ { t - k } ) | \mathcal { F } _ { t - k } ]$ and thus $\left\| \nabla f _ { t ; h } ( N _ { t - h } ) \right\| \le l _ { y } ;$ to obtain (b), we use the $\beta _ { y }$ -smoothness of $f _ { t ; h } ( \cdot )$ , which is a direct consequence of the $\beta _ { y }$ -smoothness of $f _ { t } ( \cdot )$ shown by Lemma 3.4. It remains to upper bound

$$
\begin{array}{l} \sum_ {t = h} ^ {T - 1} \left\| N _ {t} - N _ {t - h} \right\| \leq \sum_ {t = h} ^ {T - 1} \sum_ {i = 0} ^ {h - 1} \left\| N _ {t - i} - N _ {t - i - 1} \right\| \\ \stackrel {(a)} {\leq} \sum_ {t = h} ^ {T - 1} \sum_ {i = 0} ^ {h - 1} \| \eta_ {t - i - 1} \nabla f _ {t} (N _ {t - i - 1}) \| \\ \stackrel {(b)} {\leq} l _ {y} \sum_ {t = h} ^ {T - 1} \sum_ {i = 0} ^ {h - 1} \eta_ {t - i - 1} \\ \stackrel {(c)} {\leq} \frac {2 l _ {y} h}{\alpha_ {0} \log^ {2} T} \sum_ {t = 1} ^ {T - 1} \frac {1}{t} \leq \frac {2 l _ {y} h}{\alpha_ {0} \log^ {2} T} (1 + \log T), \tag {C.6} \\ \end{array}
$$
