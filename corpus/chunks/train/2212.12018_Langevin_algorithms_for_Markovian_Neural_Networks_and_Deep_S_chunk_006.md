# 2.1 Stochastic optimal control

We consider the following SOC problem in continuous time:

$$\min _ {u} J (u) := \mathbb {E} \left[ \int_ {0} ^ {T} G (t, X _ {t}) d t + F (X _ {T}) \right], \tag {2.1}d X _ {t} = b (X _ {t}, u _ {t}) d t + \sigma (X _ {t}, u _ {t}) d W _ {t}, t \in [ 0, T ] \tag {2.2}$$

where $b : \mathbb { R } ^ { d _ { 1 } } \times \mathbb { R } ^ { d _ { 3 } }  \mathbb { R } ^ { d _ { 1 } } , \sigma : \mathbb { R } ^ { d _ { 1 } } \times \mathbb { R } ^ { d _ { 3 } }  \mathcal { M } _ { d _ { 1 } , d _ { 2 } } ( \mathbb { R } )$ , W is a $\mathbb { R } ^ { d _ { 2 } }$ -valued Brownian motion and u is a $\mathbb { R } ^ { d _ { 3 } }$ -valued continuous adapted process, $T > 0 , G : [ 0 , T ] \times \mathbb { R } ^ { d _ { 1 } }  \mathbb { R }$ and $F : \mathbb { R } ^ { d _ { 1 } }  \mathbb { R }$ .

We first approximate the continuous SDE $( X _ { t } ) _ { t \in [ 0 , T ] }$ with its Euler-Maruyama scheme and the control $u _ { t }$ with a discrete-time control. For $N \in \mathbb { N }$ being the number of discretization times, we consider the regular subdivision of [0, T ]:

$$t _ {k} := k T / N, k \in \{0, \dots , N \}, h := T / N \tag {2.3}$$

and we approximate the control applied at times $t _ { 0 } , \ldots , t _ { N - 1 }$ as the output of a single neural network depending on t, or as the output of N neural networks, one for each discretization instant $t _ { k }$ :

$$u _ {t _ {k}} = \bar {u} _ {\theta} (t _ {k}, X _ {t _ {k}}) \quad \text { or } \quad u _ {t _ {k}} = \bar {u} _ {\theta^ {k}} (X _ {t _ {k}}) \tag {2.4}$$

where ${ \bar { u } } _ { \theta }$ is a neural function with finite-dimensional parameter $\theta \in \mathbb { R } ^ { d }$ . Indeed, since (2.2) defines a Markovian process, we can assume that $u _ { t }$ depends only on t and on $X _ { t }$ instead of t and $( X _ { s } ) _ { s \in [ 0 , t ] }$ .

The SOC problem (2.1) is numerically approximated by:

$$\min _ {\theta} \bar {J} (\bar {u} _ {\theta}) := \sum_ {k = 0} ^ {N - 1} (t _ {k + 1} - t _ {k}) G (t _ {k + 1}, \bar {X} _ {t _ {k + 1}} ^ {\theta}) + F (\bar {X} _ {t _ {N}} ^ {\theta}), \tag {2.5}\bar {X} _ {t _ {k + 1}} ^ {\theta} = \bar {X} _ {t _ {k}} ^ {\theta} + (t _ {k + 1} - t _ {k}) b \big (\bar {X} _ {t _ {k}} ^ {\theta}, \bar {u} _ {k, \theta} (\bar {X} _ {t _ {k}} ^ {\theta}) \big) + \sqrt {t _ {k + 1} - t _ {k}} \sigma \big (\bar {X} _ {t _ {k}} ^ {\theta}, \bar {u} _ {k, \theta} (\bar {X} _ {t _ {k}} ^ {\theta}) \big) \xi_ {k + 1}, \tag {2.6}\xi_ {k} \underset {\text { i . i . d . }} {\sim} \mathcal {N} \left(0, I _ {d _ {2}}\right) \tag {2.7}$$
