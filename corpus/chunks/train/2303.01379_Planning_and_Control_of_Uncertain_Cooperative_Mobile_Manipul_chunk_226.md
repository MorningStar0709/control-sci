Hence, all the motions produced by the nullspace of $\mathcal { \widetilde { R } } _ { \mathcal { G } }$ for an infinitesimally rigid framework must correspond to trivial motions, i.e., coordinated translations and rotations. Moreover, given (3.44), it follows that $( \mathcal { G } , p _ { \mathcal { G } } , R _ { \mathcal { G } } )$ is D&B infinitesimally rigid in SE(3) if and only if

$$\mathrm{null} (\mathcal {R} _ {\mathcal {G}}) = \{x = P _ {R} y \in \mathbb {S E} (3) ^ {N}: y \in \mathrm{null} (\widetilde {\mathcal {R}} _ {\mathcal {G}}) \}, \tag {3.46}$$

i.e., the nullspace of $\mathcal { R } _ { \mathcal { G } }$ consists of the vectors of $\operatorname { n u l l } ( \widetilde { \mathcal { R } } _ { \mathcal { G } } )$ whose elements are permutated by $P _ { R }$ .

It is worth noting that the aforementioned results are not valid if the rigidity matrix loses rank, i.e., rank $( \mathcal { R } _ { \mathcal { G } } ) < \operatorname* { m a x } \{ \operatorname { r a n k } ( \mathcal { R } _ { \mathcal { G } } ( x ) ) , x \in \mathbb { S } \mathbb { E } ( 3 ) \}$ . These are degenerate cases that correspond, for example, to when all agents are aligned along a direction $\mathbf { v } \in \mathbb { S } ^ { 2 }$ . For more discussion on these degenerate cases, the reader is referred to [174].

As a last remark, we observe that frameworks over the complete graph, $( K _ { N } , p _ { K _ { N } } , R _ { K _ { N } } )$ , are (except for the degenerate configurations), infinitesimally rigid. That is, rank $( \widetilde { \mathcal { R } } _ { K _ { N } } ) = 6 N - 6$ . This leads to the following corollary.

Corollary 3.1. Consider the D&B frameworks $( \mathcal { G } , p _ { \mathcal { G } } , R _ { \mathcal { G } } )$ and $( \mathcal { K } _ { N } , p _ { \mathcal { G } } , R _ { \mathcal { G } } )$ for nondegenrate configurations $( p _ { \mathcal { G } } , R _ { \mathcal { G } } )$ . Then $( \mathcal { G } , p _ { \mathcal { G } } , R _ { \mathcal { G } } )$ is D&B infinitesimally rigid if and only if

$$\mathrm{rank} (\widetilde {\mathcal {R}} _ {\mathcal {G}}) = \mathrm{rank} (\widetilde {\mathcal {R}} _ {\mathcal {K} _ {N}}) = 6 N - 6.$$
