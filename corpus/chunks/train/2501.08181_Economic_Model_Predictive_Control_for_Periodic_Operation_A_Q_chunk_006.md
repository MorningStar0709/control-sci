$$(x ^ {\circ}, \mathbf {u} ^ {\circ}) = \arg \mathbb {D} _ {k} (p): \min _ {x _ {0}, \mathbf {u}} \sum_ {j = 0} ^ {T - 1} \ell_ {k + j} ^ {e} (x _ {j}, u _ {j}, p) \tag {4a}\text { s.t. } \quad x _ {j + 1} = f _ {k + j} (x _ {j}, u _ {j}), x _ {T} = x _ {0}, \tag {4b}\left(x _ {j}, u _ {j}\right) \in \mathcal {Z} _ {k + j}, \quad \forall j \in \mathbb {Z} _ {[ 0, T - 1 ]}, \tag {4c}$$

where $\mathbf { u } : = [ u _ { 0 } , \dots , u _ { T - 1 } ] \in \mathbb { R } ^ { m T }$ , and $\mathbb { Z } _ { [ 0 , T - 1 ] }$ is the set of integers from 0 to $T - 1$ .

Assumption 2: The function $\ell _ { k } ^ { e }$ is convex in $( x _ { k } , u _ { k } )$ and lower-bounded for all value of $( \boldsymbol { x } _ { k } , \boldsymbol { u } _ { k } ) \in \mathcal { Z } _ { k } , p$ and k. In addition, there exists $\alpha \in { \mathcal { K } } _ { \infty }$ such that

$$
\begin{array}{l} \ell_ {k + j} ^ {e} (x _ {j}, u _ {j}, p) - \ell_ {k + j} ^ {e} (x _ {j} ^ {\circ}, u _ {j} ^ {\circ}, p) \geq \\ \alpha (\| (x _ {j}, u _ {j}) - (x _ {j} ^ {\circ}, u _ {j} ^ {\circ}) \|), \forall k, j. \\ \end{array}
$$

We note that $( x ^ { \circ } , \mathbf { u } ^ { \circ } )$ is unique under Assumption 2.

Assumption 3: The gradient of $\ell _ { k } ^ { e } ( x , u , p )$ with respect to $( x , u )$ is ρ-Lipschitz continuous, i.e.,

$$
\begin{array}{l} \left\| \nabla_ {(x, u)} \ell_ {k} ^ {e} (x, u, p) - \nabla_ {(v, w)} \ell_ {k} ^ {e} (v, w, p) \right\| \leq \\ \rho \| (x, u) - (v, w) \|, \forall p, \\ \end{array}
$$

for all $( x , u ) , ( v , w ) \in \mathcal { Z } _ { k }$ and some $\rho > 0$ .

In the next section, the goal is to design a controller that— for any given value of p—stabilizes the system and steers it to the optimal trajectory $( x ^ { \circ } ( p ) , \mathbf { u } ^ { \circ } ( p ) )$ , while also satisfying the constraints during transients.
