Proof. Since $h _ { \mathrm { i n t , d } } \ \in \ \mathrm { n u l l } ( G ) \ = \ \mathrm { r a n g e } ( \mathcal { R } _ { \mathcal { G } } ^ { \top } )$ , it holds that $M ^ { - \frac { 1 } { 2 } } h _ { \mathrm { i n t , d } } ~ \in$ rang $\mathrm { ; e } ( M ^ { - \frac { 1 } { 2 } } \mathcal { R } _ { \mathcal { G } } ^ { \top } ) = \mathrm { r a n g e } ( \mathcal { R } _ { \mathcal { G } } M ^ { - \frac { 1 } { 2 } } ) ^ { \dagger }$ . Therefore, it holds that

$$\left(\mathcal {R} _ {\mathcal {G}} M ^ {- \frac {1}{2}}\right) ^ {\dagger} \mathcal {R} _ {\mathcal {G}} M ^ {- 1} h _ {\text { int,d }} = \left(\mathcal {R} _ {\mathcal {G}} M ^ {- \frac {1}{2}}\right) ^ {\dagger} \mathcal {R} _ {\mathcal {G}} M ^ {- \frac {1}{2}} (M ^ {- \frac {1}{2}} h _ {\text { int,d }}) = M ^ {- \frac {1}{2}} h _ {\text { int,d }}. \tag {3.65}$$

Hence, (3.57) yields the resulting internal forces

$$
\begin{array}{l} h _ {\mathrm{int}} = M ^ {\frac {1}{2}} (\mathcal {R} _ {\mathcal {G}} M ^ {- \frac {1}{2}}) ^ {\dagger} \mathcal {R} _ {\mathcal {G}} M ^ {- 1} (I - M G ^ {\top} (G M G ^ {\top}) ^ {- 1}) h _ {\mathrm{int,d}} \\ = M ^ {\frac {1}{2}} \left(\mathcal {R} _ {\mathcal {G}} M ^ {- \frac {1}{2}}\right) ^ {\dagger} \mathcal {R} _ {\mathcal {G}} M ^ {- 1} h _ {\text { int,d }} \\ = M ^ {\frac {1}{2}} M ^ {- \frac {1}{2}} h _ {\text {int,d}} = h _ {\text {int,d}}, \\ \end{array}
$$

where we have used (3.65) and the fact that $\mathcal { R } _ { \mathcal { G } } G ^ { \top } = 0$ from Theorem 3.2.

Finally, in view of Theorem 3.2, one can also verify the consistency of the expressions of $h , h _ { o }$ in (3.52) with the grasp-matrix rigidity constraint $h _ { o } = G ( x ) h$ (see (3.38)). Indeed, Theorem 3.2 dictates that $G \mathcal { R } _ { \mathcal { G } } ^ { \top } = 0$ . Therefore, by combining (3.55) and (3.54) we conclude that $h _ { o } = { \cal { G } } ( x ) h$ . Note also that, in view of Corollary 3.2, the result is still valid if different $\gamma _ { \bar { \mathcal { G } } }$ and $\mathcal { R } _ { \bar { \mathcal { G } } }$ are chosen.
