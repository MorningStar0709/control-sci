# 4.2. Trajectory generation

Using the transition dynamics (13), the probability density function of a length-T trajectory is

$$p _ {\pi_ {\boldsymbol {\theta}}} (\boldsymbol {\tau}) = p \left(\mathcal {X} \left(T _ {0} - 1\right)\right) \prod_ {k = T _ {0} - 1} ^ {T - 1} p \left(\boldsymbol {\theta} \boldsymbol {y} (k) + \boldsymbol {w} (k) | \mathcal {X} (k)\right) p \left(\mathcal {X} (k + 1) | \mathcal {X} (k), \boldsymbol {\theta} \boldsymbol {y} (k) + \boldsymbol {w} (k)\right),$$

which is uniquely determined by $\mathcal { X } \left( T _ { 0 } - 1 \right)$ and the sequences w $( T _ { 0 } - 1 ) , \cdot \cdot \cdot , \pmb { w } ( T - 1 )$ .

In analogy with the derivation in (9), we aim to generate the trajectory starting from X $( T _ { 0 } - 1 )$ under perturbations on actions given by ${ \pmb w } ( T _ { 0 } - 1 ) , \cdots , { \pmb w } ( T - 1 )$ . From the control policy ${ \pmb u } ( k ) =$ $\pmb { \theta } \pmb { y } ( k ) + \pmb { w } ( k )$ , the trajectory subject to the perturbations ${ \pmb w } ( T _ { 0 } ) , \dots , { \pmb w } ( T - 1 )$ should satisfy

$$
\left[ \mathcal {H} _ {u} ^ {T _ {0} - 1: T - 1} \right] \boldsymbol {g} = \left(\boldsymbol {I} _ {T - T _ {0}} \otimes \boldsymbol {\theta}\right) \left[ \mathcal {H} _ {y} ^ {T _ {0} - 1: T - 1} \right] \boldsymbol {g} + \left[ \begin{array}{c} \boldsymbol {w} (T _ {0} - 1) \\ \vdots \\ \boldsymbol {w} (T - 1) \end{array} \right]. \tag {15}
$$

Note that the generated extended initial state is given by $\tilde { \mathcal { X } } \left( T _ { 0 } - 1 \right) : = \left[ \mathcal { H } _ { y } ^ { 0 : T _ { 0 } - 1 } ; \mathcal { H } _ { u } ^ { 0 : T _ { 0 } - 2 } \right] g$ Together with the constraints in (15) gives

$$
\underbrace {\left[ \begin{array}{c} \mathcal {H} _ {u} ^ {T _ {0} - 1 : T - 1} - \left(\boldsymbol {I} _ {T - T _ {0}} \otimes \boldsymbol {\theta}\right) \mathcal {H} _ {y} ^ {T _ {0} - 1 : T - 1} \\ \hline \mathcal {H} _ {y} ^ {0 : T _ {0} - 1} \\ \mathcal {H} _ {u} ^ {0 : T _ {0} - 2} \end{array} \right]} _ {G _ {\theta}} \boldsymbol {g} = \underbrace {\left[ \begin{array}{c} \boldsymbol {w} (T _ {0} - 1) \\ \vdots \\ \boldsymbol {w} (T - 1) \\ \hline \mathcal {X} (T _ {0} - 1) \end{array} \right]} _ {\boldsymbol {R}} \tag {16}
$$
