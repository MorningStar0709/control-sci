# F.2. Bounding $R _ { T } ^ { \bf R L S }$

Bounding this term is achieved by manipulating the similar bounds in Abeille and Lazaric (2017); Abbasi-Yadkori and Szepesv´ari (2011) to our setting and TS algorithm. We first have the following result from regularized least squares estimate.

$\begin{array} { r } { E _ { T } , f o r X _ { s } = \frac { \left( 1 2 \kappa ^ { 2 } + 2 \kappa \sqrt { 2 } \right) \sigma _ { w } } { \gamma } \sqrt { 2 n \log ( n ( T - T _ { w } ) / \delta ) } } \end{array}$

$$
\begin{array}{l} \sum_ {t = 0} ^ {T} \| (\Theta_ {*} - \tilde {\Theta} _ {t}) ^ {\intercal} z _ {t} \| ^ {2} \leq 2 (\beta_ {T} (\delta) + v _ {T} (\delta)) ^ {2} \left(\left(1 + \frac {(1 + \kappa^ {2}) (n + d) ^ {2 (n + d)}}{\mu}\right) ^ {\tau_ {0} + 1} \log \frac {\det (V _ {T _ {r}})}{\det (\mu I)} \right. \\ + \left(1 + \frac {(1 + \kappa^ {2}) X _ {s} ^ {2}}{\mu}\right) ^ {\tau_ {0} + 1} \log \frac {\det (V _ {T})}{\det (V _ {T _ {r}})} \Bigg). \\ \end{array}
$$

Proof Let $\tau \leq t$ be the last time step before t, when the policy was updated. Using Cauchy-Schwarz inequality, we have:

$$\sum_ {t = 0} ^ {T} \left\| \left(\Theta_ {*} - \tilde {\Theta} _ {t}\right) ^ {\intercal} z _ {t} \right\| ^ {2} \leq \sum_ {t = 0} ^ {T} \left\| V _ {t} ^ {\frac {1}{2}} \left(\tilde {\Theta} _ {t} - \Theta_ {*}\right) \right\| ^ {2} \| z _ {t} \| _ {V _ {t} ^ {- 1}} ^ {2} \leq \sum_ {t = 0} ^ {T} \frac {\det (V _ {t})}{\det (V _ {\tau})} \left\| V _ {\tau} ^ {\frac {1}{2}} \left(\tilde {\Theta} _ {\tau} - \Theta_ {*}\right) \right\| ^ {2} \| z _ {t} \| _ {V _ {t} ^ {- 1}} ^ {2}. \tag {119}$$

Note that $t - \tau \leq \tau _ { 0 }$ due to policy update rule. Moreover, we have

$$\det (V _ {t}) = \det (V _ {\tau}) \prod_ {i = 0} ^ {t - \tau} (1 + \| z _ {t} \| _ {V _ {t - i} ^ {- 1}} ^ {2}) \leq \det (V _ {\tau}) \left(1 + \frac {\| z _ {t} \| ^ {2}}{\mu}\right) ^ {\tau_ {0}}.$$
