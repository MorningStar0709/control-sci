Figure 3: Sketch of ${ \mathcal { C } } ,$ two distinct points $X _ { \alpha } , Y _ { \alpha } \ \in \ { \mathcal { C } } _ { \beta }$ , and the line $\theta X _ { \alpha } +$ $\left( 1 - \theta \right) Y _ { \alpha }$ . Points in the interior of $\mathcal { C }$ are positive definite, and points on the boundary are only semi-definite. Since C is compact, the line must pass through its boundary. This means that there is a value of θ such that $g \left( \theta \right)$ is positive semi-definite, but not positive definite. It also follows that when θ becomes sufficiently large, $g \left( \theta \right) \notin { \mathcal { C } }$ . These two observations are used in section 2.3 to deduce that for each $\alpha > 0$ , the function $f \left( \cdot \right)$ has a unique fixed point.
