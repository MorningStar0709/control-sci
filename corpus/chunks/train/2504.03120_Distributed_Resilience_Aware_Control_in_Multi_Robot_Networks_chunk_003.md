# II. PRELIMINARIES AND PROBLEM STATEMENT

We denote the cardinality of a set S as |S|. We use $\mathbb { R } _ { + } .$ , $\mathbb { R } _ { \geq 0 } .$ , and $\mathbb { Z } _ { \geq 0 }$ to denote sets of positive real, non-negative real, and non-negative integer numbers, respectively. We also use $\| \cdot \| : \mathbb { R } ^ { m } \to \mathbb { R } _ { \geq 0 }$ to represent the Euclidean norm.
