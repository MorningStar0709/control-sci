# 3.2.4 P-STC

In this subsection, we develop the P-STC approach based on the P-CETC (46)-(55). This is accomplished by determining an upper bound on $d ^ { 2 } ( t )$ and lower bound on $m ^ { p } ( t )$ , both of which are the constituent terms of the continuoustime event triggering function $\Gamma ^ { p } ( t )$ given by (49). We prove that updating the control input at specific times prescribed by the self-trigger ensures $\dot { \Gamma } ^ { p } ( t )$ remains non-positive and $m ^ { p } ( t )$ remains positive along the P-STC closed-loop system solution. As a result, convergence properties equivalent to R4 and R5 in Theorem 1 holds for the P-STC closed-loop system as well.

Let $\check { I } ^ { p } = \{ \check { t } _ { 0 } ^ { p } , \check { t } _ { 1 } ^ { p } , \check { t } _ { 2 } ^ { p } , \dots \}$ denote the sequence of eventtimes associated with the P-STC. Let the parameters $\gamma , c , \beta _ { 1 } , \beta _ { 2 } , \rho > 0$ be selected as outlined in Section 3.2.1, and the parameter $\eta > 0$ be selected as in (78). The proposed P-STC strategy consists of two components:

(1) An event-triggered boundary control input

$$\check {U} _ {j} ^ {p} = \int_ {0} ^ {1} k (y) u (y, \check {t} _ {j} ^ {p}) d y, \tag {94}$$

for all $t \in [ \check { t } _ { i } ^ { p } , \check { t } _ { i + 1 } ^ { p } ) , j \in \mathbb { N }$ . Accordingly, the boundary condition (3) becomes

$$u _ {x} (1, t) + q u (1, t) = \check {U} _ {j} ^ {p}. \tag {95}$$

(2) A self-trigger determining the event-times

$$\check {t} _ {j + 1} ^ {p} = \check {t} _ {j} ^ {p} + G ^ {p} (\check {t} _ {j} ^ {p}), \tag {96}$$

with $\check { t } _ { 0 } ^ { p } = 0$ where $G ^ { p } ( t )$ is a uniformly and positively lower-bounded function defined as

$$G ^ {p} (t) \triangleq \max \left\{\tau , \check {\tau} (t) \right\}. \tag {97}$$

Here, $\tau > 0$ is given by (33) and

$$\check {\tau} (t) \triangleq \frac {1}{2 \lambda_ {m a x} + \eta + c} \ln \left(\frac {\gamma m ^ {p} (t) + \frac {\gamma \rho H (t)}{2 \lambda_ {m a x} + \eta} + \frac {c}{\rho} W ^ {p} (t)}{H (t) + \frac {\gamma \rho H (t)}{2 \lambda_ {m a x} + \eta}}\right), \tag {98}$$

where the dynamics of $m ^ { p } ( t )$ satisfy (50)-(55) along the solution of $( 1 ) , ( 2 ) , ( 9 4 ) , ( \dot { 9 } \dot { 5 } )$ for $t \in ( \check { t } _ { j } ^ { p } , \check { t } _ { j + 1 } ^ { p } ) , j \in \mathbb { N } .$ and

$$H (t) \triangleq 2 \| k \| ^ {2} \left(2 + \frac {\varepsilon^ {2} \| k \| ^ {2}}{\lambda_ {m a x} ^ {2}}\right) \| u [ t ] \| ^ {2}, \tag {99}$$

with k(y) given by (14).
