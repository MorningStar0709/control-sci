# B. Structural Properties of Robot Dynamics

Robot manipulators belong to the Euler-Lagrange (EL) family, which possess structural properties as follows [18].

Property 1: The joint space inertia matrix M(q) is positive definite and bounded from above and below [18], namely, the eigen value of the joint space inertia matrix satisfies $\sigma _ { \mathrm { m i n } } < \sigma ( \mathbf { M } ( \mathbf { q } ) ) < \sigma _ { \mathrm { m a x } }$ , where $\sigma _ { \mathrm { m i n } }$ and $\sigma _ { \mathrm { m a x } }$ are the minimum and maximum eigenvalues of the joint space inertia matrix, respectively.

Property 2: The centrifugal and Coriolis matrix $\mathbf { C } ( \mathbf { q } , \dot { \mathbf { q } } )$ satisfies $| \mathbf { C } ( \mathbf { q } , { \dot { \mathbf { q } } } _ { 1 } ) { \dot { \mathbf { q } } } _ { 2 } | \leq c _ { 0 } | { \dot { \mathbf { q } } } _ { 1 } | | { \dot { \mathbf { q } } } _ { 2 } |$ , where $c _ { 0 }$ is a positive constant.

Property 3: The gravity term ${ \bf g } ( { \bf q } )$ can be bounded by a positive constant $c _ { 1 } ,$ namely, $| | \mathbf { g ( q ) } | | \leq c _ { 1 }$ . These properties can subsequently be used to derive the following proposition.

Proposition 1: Due to the kinematic limit of the robot, which results in boundedness of joint position, velocities, and accelerations, the linear regressor is also bounded $| | \mathbf { Y } | | \leq c _ { 2 }$ , where $c _ { 2 }$ is a positive constant.
