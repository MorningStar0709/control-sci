It is advantageous for both ELS and RML to replace the residual in the regression vector by the posterior residual defined as

$$\varepsilon_ {p} (t) = y (t) - \varphi^ {T} (t - 1) \hat {\theta} (t)$$

that is, the latest value of $\hat{\theta}$ is used to compute $\varepsilon_{p}$ .

Another possibility to model the correlated noise is to use the model

$$y (t) = \frac {B (q)}{A (q)} u (t) + \frac {C (q)}{D (q)} e (t)$$

instead of Eq. (2.38). Recursive parameter estimates for this model can be derived in the same way as for Eq. (2.38).

Details about the extended least-squares method and the recursive maximum likelihood method are found in the references at the end of the chapter.
