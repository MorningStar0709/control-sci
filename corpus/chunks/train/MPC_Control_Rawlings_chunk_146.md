# Exercise 1.27: Forecasting and variance increase or decrease

Given positive definite initial state variance P (0) and process disturbance variance Q, the variance after forecasting one sample time was shown to be

$$P ^ {-} (1) = A P (0) A ^ {\prime} + Q$$

(a) If A is stable, is it true that $A P ( 0 ) A ^ { \prime } < P ( 0 ) ?$ If so, prove it. If not, provide a counterexample.   
(b) If A is unstable, is it true that $A P ( 0 ) A ^ { \prime } > P ( 0 ) ?$ If so, prove it. If not, provide a counterexample.   
(c) If the magnitudes of all the eigenvalues of A are unstable, is it true that $A P ( 0 ) A ^ { \prime } >$ $P ( 0 ) ?$ If so, prove it. If not, provide a counterexample.
