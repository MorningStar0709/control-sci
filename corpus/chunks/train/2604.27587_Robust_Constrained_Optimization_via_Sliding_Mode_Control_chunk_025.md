# 4.2 Obstacle Avoidance

We illustrate the proposed sliding–mode based constrained optimization framework using a mobile robot navigation problem in a cluttered planar environment. The robot state is $\mathbf { q } = [ x y ] ^ { T } \in \mathbf { \bar { \mathbb { R } } ^ { 2 } }$ with first–order dynamics $\dot { \mathbf { q } } = \mathbf { u }$ . The task is to steer the system from the initial configuration q0 to the goal ${ \bf q } _ { g }$ while satisfying the distance–based safety constraints $h _ { i } ( \mathbf { q } ) \ =$ $| | { \bf q } - { \bf q } _ { \mathrm { o b s } } ^ { i } | | - R _ { s } = 0 , \quad { \bf q } _ { \mathrm { o b s } } ^ { 1 } = { \bf \Psi } _ { \mathrm { 1 } } ^ { \left\{ 3 \right\} } , ~ { \bf q } _ { \mathrm { o b s } } ^ { 2 } = { \bf \Psi } _ { \mathrm { - 1 } } ^ { \left\{ 3 \right\} }$ 2o , with safety radius $R _ { s } = 0 . 8$ . The objective function to be minimized is the quadratic potential $\begin{array} { r } { \phi ( \mathbf { q } ) = \frac { 1 } { 2 } \lVert \mathbf { q } - \mathbf { q } _ { g } \rVert ^ { 2 } } \end{array}$ . For baseline comparison, we implement the classical artificial potential field $( \mathrm { A P F } )$ controller [16] $\mathbf { \mathbf { \mathbf { \mathbf { \mathbf { \mathbf { \mathbf { \mathbf { \mathbf } { \mathbf } { A P F } } } } } } } } = \mathbf { \mathbf { \mathbf { \mathbf { \mathbf { \ s } } } } } - \nabla \Phi ( \mathbf { \mathbf { \mathbf { q } } } )$ , where $\begin{array} { r l r } { \Phi ( { \bf q } ) } & { { } = } & { \Phi _ { \mathrm { a t t } } ( { \bf q } ) + \sum _ { i } \Phi _ { \mathrm { r e p } } ^ { i } ( { \bf q } ) } \end{array}$ combines a quadratic attractive term and inverse–distance repulsive potentials. Due to the symmetric obstacle configuration, the repulsive gradients balance along the corridor, and the robot becomes trapped at a non-goal equilibrium

$\mathbf { q } ^ { * } \neq \mathbf { q } _ { g }$ satisfying $\nabla \Phi ( \mathbf { q } ^ { * } ) = 0$ . This behavior confirms the well-known local-minima drawback of APF navigation. The projected gradient flow corresponding to the constrained optimization problem
