# II. PROBLEM STATEMENT

We consider a Timoshenko beam without internal damping, which is described by two coupled wave equations, i.e., second-order partial differential equations (PDEs)

$$
\begin{array}{l} \varrho (z) S (z) \partial_ {t} ^ {2} w (z, t) = \partial_ {z} (\kappa (z) (\partial_ {z} w (z, t) - \varphi (z, t))) (2a) \\ J (z) \partial_ {t} ^ {2} \varphi (z, t) = \partial_ {z} \bigl (E (z) I (z) \partial_ {z} \varphi (z, t) \bigr) \\ + \kappa (z) \big (\partial_ {z} w (z, t) - \varphi (z, t) \big), (2b) \\ \end{array}
$$

where $w ( z , t )$ denotes the transverse displacement along the beam axis and $\varphi ( z , t )$ is the rotation angle of the normal of the cross-section relative to the beam axis (see Figure 2). Without loss of generality, we assume $w ( z , t )$ and $\varphi ( z , t )$ defined on $[ 0 , 1 ] \times [ 0 , \infty )$ , which can always be achieved by normalization of the spatial domain. In $( 2 ) , \varrho ( z )$ denotes the (linear) density, $\kappa ( z ) = k _ { s } ( z ) G ( z ) S ( z )$ the product of shear correction factor $k _ { s } ( z )$ , shear modulus $G ( z )$ and crosssection area $S ( z )$ , $J ( z ) ~ = ~ \varrho ( z ) I ( z )$ the rotary moment of inertia, $I ( z )$ the moment of inertia, and $E ( z )$ Young’s modulus of elasticity. The beam is assumed to be clamped at $z = 0$ , i.e., $w ( 0 , t ) = 0$ and $\varphi ( 0 , t ) = 0$ . This implies boundary conditions (BCs)

$$\partial_ {t} w (0, t) = 0 \quad \text { and } \quad \partial_ {t} \varphi (0, t) = 0. \tag {3}$$

for the corresponding velocities, which are used for the firstorder representation in Section III-A. The end at $z = 1$ is fully actuated, leading to the BCs

$$\kappa (1) \big (\partial_ {z} w (1, t) - \varphi (1, t) \big) = u _ {1} (t) \tag {4a}E (1) I (1) \partial_ {z} \varphi (1, t) = u _ {2} (t), \tag {4b}$$

where the input components $u _ { 1 } ( t )$ and $u _ { 2 } ( t )$ represent a force and a bending moment, respectively. We consider general initial conditions $( \mathrm { I C s } ) ~ w ( z , 0 ) = \bar { w } _ { 0 } ( z ) , \varphi ( z , 0 ) = \bar { \varphi } _ { 0 } ( z )$ and $\partial _ { t } w ( z , 0 ) = \bar { w } _ { 1 } ( z ) , \partial _ { t } \varphi ( z , 0 ) = \bar { \varphi } _ { 1 } ( z )$ .
