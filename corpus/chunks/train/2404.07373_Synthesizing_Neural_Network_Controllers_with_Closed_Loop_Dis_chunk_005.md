# 1.1 Notation

We let the symbol $\mathbb { R } _ { \geq 0 }$ denote the set of nonnegative real numbers, ${ \dot { \mathcal { L } } } _ { 2 } ^ { n }$ the space of square-integrable functions from $\mathbb { R } _ { > 0 }$ to $\mathbb { R } ^ { n }$ , and $\mathcal { L } _ { 2 e } ^ { n }$ the space of functions from $\mathbb { R } _ { > 0 }$ to ${ \overline { { \mathbb { R } } } } ^ { n }$ which are square-integrable on the interval [0, T ] for all $T > 0$ . We drop the superscript denoting the dimension of the codomain if it is clear from context. We use bold symbols to denote signals and their non-bold counterparts to denote points. For example, $\scriptstyle { \mathbf { { \vec { x } } } } _ { p }$ denotes the element of $\mathcal { L } _ { 2 e } ^ { n _ { p } }$ and $x _ { p }$ denotes an element of $\mathbb { R } ^ { n _ { p } }$ . For brevity, we use $( \star ) ^ { \top } \dot { X } y$ to denote $y ^ { \top } X y$ when the expression that appears in place of y is lengthy.
