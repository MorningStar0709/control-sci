# B. Learning a feedback controller

To design the feedback law, we follow the approach introduced in [26], which gives simple formulas for controller design and returns a data-based representation of the closedloop dynamics that is useful to determine later on the triggering policy. The next two results are taken from [26]; we recall them for convenience. We start with an auxiliary result.

Lemma 1: Let Assumption 1 hold and consider any matrix $K \in \mathbb { R } ^ { m \times n }$ . Then $A + B K = X _ { 1 } G$ where $G \in \mathbb { R } ^ { T \times n }$ is any solution to the system of equations

$$
\left[ \begin{array}{l} K \\ I \end{array} \right] = \left[ \begin{array}{l} U _ {0} \\ X _ {0} \end{array} \right] G. \tag {5}
$$

![](image/27faa6f3d4d18dc66f917a0d11bd648a2b66087b8b47898d1d9553d86670f640.jpg)

Proof. By Assumption 1, there exists a $T \times n$ matrix G such that (5) holds. Hence, $A + B K = \left[ B \ A \right] \left[ { \begin{array} { l } { K } \\ { I } \end{array} } \right] = \left[ B \ A \right] \left[ { \begin{array} { l } { U _ { 0 } } \\ { X _ { 0 } } \end{array} } \right] G$ where the second identity follows from (5). The result follows because the elements of $U _ { 0 } , X _ { 0 }$ and of $X _ { 1 }$ satisfy the relation $\dot { x } ( t ) = A x ( t ) + B u ( t ) , t \in \{ 0 , T _ { s } \ldots , ( T - 1 ) T _ { s } \}$ , which, in compact form, gives $X _ { 1 } = A X _ { 0 } + B U _ { 0 }$ . 

We exploit Lemma 1 to design K such that $A + B K$ is Hurwitz, despite the fact that A and B are unknown. As a result, the origin of system (1) with $d \equiv 0$ in closed-loop with the continuous-time controller $u = K x$ will be globally exponentially stable, i.e., there exist $c _ { 1 } \geqslant 1$ and $c _ { 2 } > 0$ such that any solution x satisfies $| x ( t ) | \leqslant c _ { 1 } e ^ { - c _ { 2 } t } | x ( 0 ) |$ | for any t ě 0. We derive for this purpose a convex program, specifically a SDP. As A and B are unknown, we use the fact that for any matrix $K \in \mathbb { R } ^ { n \times m } , X _ { 1 } G = A + B K$ where $X _ { 1 }$ and $G$ are known from D in (3) in view of Lemma 1, to select suitable K based on $U _ { 0 } , X _ { 0 }$ and $X _ { 1 }$ in the next theorem.

Theorem 1: Consider stabilizable system (1) with $d \equiv 0$ and suppose Assumption 1 holds. Consider the next SDP in the decision variable $Y \in \mathbb { R } ^ { T \times n }$

$$X _ {1} Y + \left(X _ {1} Y\right) ^ {\top} < 0, \tag {6a}X _ {0} Y \succ 0. \tag {6b}$$
