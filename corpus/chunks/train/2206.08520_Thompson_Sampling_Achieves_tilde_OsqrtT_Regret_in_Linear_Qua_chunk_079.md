# Appendix E. Regret Decomposition

Denote the optimal expected average cost of an LQR system Θ with process noise covariance W by $J _ { * } ( \Theta , W ) = \mathrm { t r } ( P ( \Theta ) W )$ ). Note that during the initial exploration period, we have that $u _ { t } = \bar { u } _ { t } + \nu _ { t }$ for $t \leq T _ { w }$ and after the initial exploration, we have that $u _ { t } = \bar { u } _ { t }$ for $t > T _ { w }$ where we denote by $\bar { u } _ { t } : = K ( \bar { \Theta } _ { t } ) x _ { t }$ the optimal control action assuming the system $\tilde { \Theta } _ { t }$ . Since initial exploration period injects independent random perturbations through the optimal control input, ${ { \bar { u } } _ { t } } ,$ for sampled system, $\tilde { \Theta } _ { t }$ , the state dynamics can be reformulated in order to take the external perturbations into account by adding it to the process noise:

$$x _ {t + 1} = A _ {*} x _ {t} + B _ {*} \bar {u} _ {t} + \zeta_ {t}, \tag {103}$$

where $\bar { u } _ { t } = K ( \tilde { \Theta } _ { t } ) x _ { t } , \zeta _ { t } = B _ { * } \nu _ { t } + w _ { t }$ for $t \leq T _ { w }$ , and $\zeta _ { t } = w _ { t }$ for $t > T _ { w }$ . We can write the regret explicitly as

$$R _ {T} = \sum_ {t = 0} ^ {T} \left\{x _ {t} ^ {\intercal} Q x _ {t} + u _ {t} ^ {\intercal} R u _ {t} - J _ {*} (\Theta_ {*}, \sigma_ {w} ^ {2} I) \right\} = R _ {T w} ^ {\exp} + R _ {T} ^ {\mathrm{noexp}}, \tag {104}$$

where

$$R _ {T _ {w}} ^ {\mathrm{exp}} := \sum_ {t = 0} ^ {T _ {w}} \left(2 \bar {u} _ {t} ^ {\intercal} R \nu_ {t} + \nu_ {t} ^ {\intercal} R \nu_ {t}\right), \& R _ {T} ^ {\mathrm{noexp}} := \sum_ {t = 0} ^ {T} \left\{x _ {t} ^ {\intercal} Q x _ {t} + \bar {u} _ {t} ^ {\intercal} R \bar {u} _ {t} - J _ {*} (\Theta_ {*}, \sigma_ {w} ^ {2} I) \right\}$$

Since $E _ { s } \subset E _ { t }$ for any $0 \leq s \leq t$ , we have that

$$
\begin{array}{l} R _ {T} ^ {\mathrm{noexp}} \mathbb {1} _ {E _ {T}} = \sum_ {t = 0} ^ {T} \left\{x _ {t} ^ {\intercal} Q x _ {t} + \bar {u} _ {t} ^ {\intercal} R \bar {u} _ {t} - J _ {*} (\Theta_ {*}, \sigma_ {w} ^ {2} I) \right\} \mathbb {1} _ {E _ {T}} \\ \leq \sum_ {t = 0} ^ {T} \left\{x _ {t} ^ {\intercal} Q x _ {t} + \bar {u} _ {t} ^ {\intercal} R \bar {u} _ {t} - J _ {*} (\Theta_ {*}, \sigma_ {w} ^ {2} I) \right\} \mathbb {1} _ {E _ {t}}, \tag {105} \\ \end{array}
R _ {T _ {w}} ^ {\exp} \mathbb {1} _ {E _ {T}} = \sum_ {t = 0} ^ {T _ {w}} \left(2 \bar {u} _ {t} ^ {\intercal} R \nu_ {t} + \nu_ {t} ^ {\intercal} R \nu_ {t}\right) \mathbb {1} _ {E _ {T}} \leq \sum_ {t = 0} ^ {T _ {w}} \left(2 \bar {u} _ {t} ^ {\intercal} R \nu_ {t} + \nu_ {t} ^ {\intercal} R \nu_ {t}\right) \mathbb {1} _ {E _ {t}}. \tag {106}$$
