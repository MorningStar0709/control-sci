Proof. In order to prove Theorem.2, let Lemma 1. holds true, and the following Lyapunov function is chosen as:

$$\mathcal {L} _ {s} = \frac {1}{2} s ^ {\top} s + \frac {1}{2 \lambda} \tilde {\boldsymbol {\Lambda}} ^ {\top} \tilde {\boldsymbol {\Lambda}} \tag {36}$$

where $\tilde { \pmb { \Lambda } } = \hat { \pmb { \Lambda } } - \pmb { \Lambda }$ . One can find the derivative of (36) as:

$$\dot {\mathcal {L}} _ {s} = s \left(\gamma_ {1} \dot {\tilde {\boldsymbol {q}}} + \dot {\tilde {\omega}}\right) + \frac {1}{\lambda} \tilde {\boldsymbol {\Lambda}} ^ {\top} \dot {\tilde {\boldsymbol {\Lambda}}} \tag {37}$$

Based on (28) and substituting (34), the above equation is simplified as:

$$\dot {\mathcal {L}} _ {s} = - \mu_ {1} | | s | | ^ {2} - s \hat {\boldsymbol {\Lambda}} \operatorname{sign} (s) + \frac {1}{\lambda} \tilde {\boldsymbol {\Lambda}} ^ {\top} \dot {\hat {\boldsymbol {\Lambda}}} \tag {38}$$

such that

$$\dot {\mathcal {L}} _ {s} = - \mu_ {1} | | s | | ^ {2} - s (\tilde {\boldsymbol {\Lambda}} + \boldsymbol {\Lambda}) \operatorname{sign} (s) + \frac {1}{\lambda} \tilde {\boldsymbol {\Lambda}} ^ {\top} \lambda | | s | | \tag {39}\leq - \mu_ {1} | | s | | ^ {2} - \boldsymbol {\Lambda} | | s | | \leq 0$$

Thus, the attitude system (6) is asymptotically stable with the proposed controller (34). ■
