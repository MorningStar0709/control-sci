# 2.1 Mathematical Formulation

We consider a stochastic optimal control problem in which an agent makes sequential decisions within a random environment over a finite time horizon, denoted as $\mathbb { T } = \{ 0 , 1 , \ldots , T \}$ . At each time $t \in \mathbb { T }$ , the state of the environment is represented by $\pmb { \mathscr { s } } _ { t } \in \mathbb { S } \subseteq \mathbb { R } ^ { m }$ , and the agent selects an action $\pmb { a } _ { t } \in \mathbb { A } _ { t } \subseteq \mathbb { R } ^ { n }$ . The agent receives a reward $r _ { t } ( \pmb { s } _ { t } , \pmb { a } _ { t } )$ , and the environment transitions to the next state $\pmb { s } _ { t + 1 }$ according to the dynamics:

$$\boldsymbol {s} _ {t + 1} = f _ {t} (\boldsymbol {s} _ {t}, \boldsymbol {a} _ {t}, \xi_ {t + 1}), \tag {1}$$

where $\xi _ { t + 1 }$ is a random variable taking values in $\Xi \subseteq \mathbb { R } ^ { d }$ . The action set $\mathbb { A } _ { t }$ is characterized by a set of equality and inequality constraints:

$$\mathbb {A} _ {t} = \{a \in \mathbb {R} ^ {n}: g _ {i} (s _ {t}, a) = 0, 1 \leq i \leq E; g _ {j} (s _ {t}, a) \geq 0, 1 \leq j \leq I \}, \tag {2}$$

with continuously differentiable constraint functions $\{ g _ { i } \} _ { i = \cdot } ^ { E }$ 1 and $\{ g _ { j } \} _ { j = 1 } ^ { I }$ .
