# B. Electric power system

1) Frequency dynamic model of electric buses: Two types of buses are considered: $i \in \mathcal { E } ^ { \bar { D } G }$ , which connect only to distributed generators (DGs), and $i \in \mathcal { E } ^ { H P }$ , which host both DGs and heat pumps (HPs), and $\mathcal { E } = \mathcal { E } ^ { D G } \cup \mathcal { E } ^ { H P }$ . Let $P _ { i }$ denote the net active power flowing out of bus i. The frequency deviation, damping coefficient, controllable generation, control input, electrical disturbance, and inertia at bus i are denoted by $\omega _ { i } , D _ { i } , P _ { i } ^ { G } , u _ { i } , P _ { i } ^ { L }$ , and $M _ { i }$ , respectively. Here, $\omega _ { i } , u _ { i } , P _ { i } ^ { L }$ $P _ { i }$ , and $P _ { i } ^ { G }$ represent deviations from their nominal steadystate values. To capture the lag in the response of generation, each controllable unit is modeled with a first-order system:

$$T _ {g, i} \dot {P} _ {i} ^ {G} = - P _ {i} ^ {G} + u _ {i}, \quad i \in \mathcal {E}, \tag {2}$$

where $T _ { g , i } > 0$ is the effective turbine–governor time constant.

The bus frequency dynamics are then given by

$$M _ {i} \dot {\omega} _ {i} = - D _ {i} \omega_ {i} - P _ {i} + P _ {i} ^ {G}, \quad i \in \mathcal {E} ^ {D G}, \tag {3a}M _ {i} \dot {\omega} _ {i} = - D _ {i} \omega_ {i} - P _ {i} + P _ {i} ^ {G} - p _ {i} ^ {H}, \quad i \in \mathcal {E} ^ {H P}, \tag {3b}$$

where $p _ { i } ^ { H }$ is the active power consumed by the HP at bus $i ,$ computed according to (1b). The network power flow and angle dynamics are:

$$P _ {i} = \sum_ {j} | B _ {i j} | V _ {i} V _ {j} \sin (\theta_ {i} - \theta_ {j}) - P _ {i} ^ {L}, i \in \mathcal {E}, \dot {\boldsymbol {x}} _ {\boldsymbol {\theta}} = \boldsymbol {R} _ {\boldsymbol {I}} \boldsymbol {\omega}, (4)$$

where $\scriptstyle { \mathbf {  { x } } } \theta$ is the phase-angle deviation vector excluding the reference bus $r ,$ ω denotes frequency vector, and $R _ { I } = [ I _ { n ^ { \varepsilon } - 1 } , - \mathbf { 1 } _ { n ^ { \varepsilon } - 1 } ] . n ^ { \varepsilon }$ denotes the number of electric buses, while ${ \cal I } _ { x }$ and ${ \bf 1 } _ { x }$ represent the identity matrix and the all-ones vector of dimension x, respectively.

Assumption 1. For any $\bar { P } ^ { L }$ and $\bar { p } ^ { H }$ , the EPS (2-4) admits an equilibrium $( x _ { \theta } ^ { * } , \theta ^ { * } , \omega ^ { * } , P ^ { G * } , P ^ { * } , { u } ^ { * } )$ , where $\omega ^ { * } =$ $\omega ^ { c o m } \mathbf { 1 } _ { n } \varepsilon$ and $\omega ^ { c o m }$ is the synchronized EPS frequency.
