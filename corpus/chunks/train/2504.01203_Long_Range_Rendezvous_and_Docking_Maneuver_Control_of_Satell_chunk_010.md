The trajectory optimization framework is designed to act as higher-level guidance layer generating feasible trajectory of chaser satellite. As $\triangle v$ represents change in velocity at discrete time step due to impulsive thrust. These impulses are optimized to minimize the total velocity change from initial state to the desired docking state. The optimized impulse $\triangle v$ is used to compute discrete-time state propagation of chaser satellite, $\widetilde { p } _ { c } ( \boldsymbol { k } + 1 ) = \widetilde { p } _ { c } ( \boldsymbol { k } ) + h [ \boldsymbol { B } _ { c } ^ { i } ] ^ { T } \Delta v$ where $h = t _ { k + 1 } - t _ { k }$ is fixed-time step size.
