X(N ) in (5.115) is a function that decreases with N . For example, in the so called BICLS (n, N ) ˆ criterion, X(N ) = log N $B I C _ { L S } ( \hat { n } , N )$ $\begin{array} { r } { X ( N ) = \frac { \log N } { N } } \end{array}$ (other choices are possible—see Ljung 1999; Söderström and Stoica 1989; Duong and Landau 1996) and the order nˆ is selected as the one which minimizes $C V _ { L S }$ given by (5.114). Unfortunately, the results are unsatisfactory in practice because in the majority of situations, the conditions for unbiased parameter estimation using least squares are not fulfilled.

In Duong and Landau (1994) and Duong and Landau (1996), it is proposed to replace the matrix R(n)ˆ by an instrumental variable matrix $Z ( \hat { n } )$ whose elements will not be correlated with the measurement noise. Such an instrumental matrix Z(n)ˆ can be obtained by replacing in the matrix $R ( { \hat { n } } )$ , the columns $Y ( t - 1 )$ , $Y ( t - 2 ) , Y ( t - 3 )$ by delayed version of $U ( t - L - i )$ , i.e., where $L > n \colon$ :

$$Z (\hat {n}) = [ U (t - L - 1), U (t - 1), U (t - L - 2), U (t - 2), \dots ] \tag {5.116}$$

and therefore, the following criterion is used for the order estimation:

$$C V _ {P J} (\hat {n}, N) = \min _ {\hat {\theta}} \frac {1}{N} \| Y (t) - Z (\hat {n}) \hat {\theta} \| ^ {2} + \frac {2 \hat {n} \log N}{N} \tag {5.117}$$

and:

$$\hat {n} = \min _ {\hat {n}} C V _ {P J} (\hat {n}) \tag {5.118}$$

A typical curve of the evolution of the criterion (5.117) as a function of $\hat { n }$ is shown in Fig. 5.7. Comparisons with order estimation criterions are also provided in this reference.
