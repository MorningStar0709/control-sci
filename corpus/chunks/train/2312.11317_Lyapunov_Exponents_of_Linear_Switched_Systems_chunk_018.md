$$c _ {0} > \frac {1}{\sqrt {2}} \sqrt {\frac {\operatorname{tr} ([ A , B ] ^ {2})}{2 \operatorname{tr} (A B) - \operatorname{tr} (A ^ {2}) - \operatorname{tr} (B ^ {2})}} \quad \text {or} \quad c _ {0} < - \frac {1}{\sqrt {2}} \sqrt {\frac {\operatorname{tr} ([ A , B ] ^ {2})}{2 \operatorname{tr} (A B) - \operatorname{tr} (A ^ {2}) - \operatorname{tr} (B ^ {2})}}.$$

As already mentioned in the Introduction, if the control u in equation (2.2) is constant equal to 1, then the corresponding principal Lyapunov exponent is equal to the positive eigenvalue of A, i.e.

$$\ell_ {u = 1} (A, B) = \sqrt {\frac {\mathrm{tr} (A ^ {2})}{2}}.$$

Analogously, if the control u is constant equal to 0, then

$$\ell_ {u = 0} (A, B) = \sqrt {\frac {\mathrm{tr} (B ^ {2})}{2}}.$$

Clearly, since we are assuming $\operatorname { t r } ( A ^ { 2 } ) \geq \operatorname { t r } ( B ^ { 2 } )$ , we have $\ell _ { u = 1 } \geq \ell _ { u = 0 }$
