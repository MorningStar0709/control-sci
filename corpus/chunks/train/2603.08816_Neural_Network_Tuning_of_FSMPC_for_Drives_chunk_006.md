# 2.1 Performance indicators

The performance indicators considered include the mechanical, electro-magnetic and electronic variables listed and defined below. Each indicator is given a number $i ,$ so that it can be referred to as $\pi _ { i }$ .

• Speed overshoot $( \pi _ { 1 } = P O )$ , defined in (8).   
• Speed rise time $\left( \pi _ { 2 } = T _ { r } \right)$ , defined in (9).   
• Integral Time Absolute Error $( \pi _ { 3 } = I T A E )$ , defined in (10).   
• Torque ripple $( \pi _ { 4 } = R _ { t } )$ , defined in (11).   
• Harmonic content $( \pi _ { 5 } = E _ { x y } )$ , defined in (12).   
• Average switching frequency $( \pi _ { 6 } = A S F )$ , defined in (13).

The mathematical expressions of the performance indicators are:

$$P O = 1 0 0 \cdot \frac {\max \omega - \omega^ {*}}{\omega^ {*}} \tag {8}T _ {r} = \underset {t \geq 0} {\operatorname{argmin}} \omega^ {*} (t) - \omega (t) \tag {9}I T A E = \frac {1}{N} \sum_ {k = 1} ^ {N} \frac {\left\| \omega^ {*} (k) - \omega (k) \right\|}{\omega^ {*} (k)} k \tag {10}R _ {t} = \sqrt {\frac {1}{N} \sum_ {k = 1} ^ {N} \left(T ^ {*} (k) - T (k)\right) ^ {2}} \tag {11}E _ {x y} = \sqrt {\frac {1}{N} \sum_ {k = 1} ^ {N} e _ {x y} ^ {2} (k)} \tag {12}A S F = \frac {1}{5 N T _ {s}} \sum_ {k = 1} ^ {N} S C (k) \tag {13}$$
