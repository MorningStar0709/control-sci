# B. Safe shepherding control

We consider a shepherding problem [10] in which leaders must confine followers into a goal region while both populations avoid dangerous areas. Most existing approaches to shepherding in the presence of dangerous regions rely on the assumption of cohesion among followers, which simplifies the confinement task by preserving group compactness. To the best of our knowledge, only few works address this problem without cohesion assumptions, including a reinforcement learning approach [24] and a heuristic strategy [25], both operating at the agent level. Here, we tackle this problem in a large-scale, density-based setting following [26], where analytical and computational tractability are facilitated by the continuum formulation.

Following [9], leaders and followers have densities $\rho _ { i }$ : $\mathbb { R } _ { \geq 0 } \times \Omega  \mathbb { R } _ { \geq 0 } , i \in \{ L , F \}$ , governed by the coupled PDEs

$$\partial_ {t} \rho_ {L} (t, \mathbf {x}) + \nabla \cdot [ \rho_ {L} (t, \mathbf {x}) \mathbf {u} (t, \mathbf {x}) ] = 0, \tag {29}\partial_ {t} \rho_ {F} (t, \mathbf {x}) + \nabla \cdot [ \rho_ {F} (t, \mathbf {x}) \mathbf {v} _ {F} (t, \mathbf {x}) ] = D \Delta \rho_ {F} (t, \mathbf {x}), \tag {30}$$

where u is the leader control field, $D > 0$ is the follower diffusion coefficient, and ${ \mathbf { v } } _ { F } = ( { \mathbf { g } } * \rho _ { L } ) ( t , { \mathbf { x } } )$ models repulsive leader–follower interaction through the kernel

$$\mathbf {g} (\{\mathbf {x}, \mathbf {y} \} _ {\Omega}) = \frac {\{\mathbf {x} , \mathbf {y} \} _ {\Omega}}{\| \{\mathbf {x} , \mathbf {y} \} _ {\Omega} \|} \exp \left(- \frac {\| \{\mathbf {x} , \mathbf {y} \} _ {\Omega} \|}{L _ {r}}\right), \tag {31}$$

with $L _ { r }$ being the interaction length scale.

Leaders are directly controlled and not subject to diffusion, while followers are driven indirectly through (31) and subject to stochastic perturbations.

The leader control u is designed following [9, Sec. VI] to minimize the $\mathcal { L } ^ { 2 ^ { - } }$ distance between $\rho _ { F }$ and a desired von Mises profile $\bar { \rho } _ { F } ( { \bf x } ) \ = $ $\exp ( k _ { 1 } \cos ( x _ { 1 } ) + k _ { 2 } \cos ( x _ { 2 } ) + \cos ( x _ { 1 } - x _ { 2 } ) )$ with $k _ { 1 } = k _ { 2 } = 9 .$ , chosen to confine 99% of followers within a goal region of radius $r ^ { * } = 1$ centered in the origin. Safety is enforced for both populations via (15) with barrier thresholds $\varepsilon _ { L } = 0 . 0 1 3$ and $\varepsilon _ { F } = 0 . 0 1$ .
