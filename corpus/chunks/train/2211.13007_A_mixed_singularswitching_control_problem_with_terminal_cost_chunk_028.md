# 3.1 Sketch of the proofs of Propositions 1.1 and 1.4

Let $\varepsilon \in ( 0 , 1 )$ and $( \ell , \iota ) \in \mathbb { M } \times \mathbb { I }$ be fixed. Under assumptions (H1)–(H4), by (3.1), (3.7) and (3.8), using Arzel\`a-Ascoli compactness criterion (see [5, p. 718]) and that for each $p \in ( 1 , \infty )$ , $( \mathrm { L } ^ { p } ( B _ { \beta r } ) , \| \cdot \| _ { \mathrm { L } ^ { p } ( B _ { r } ) } )$ , with $B _ { r } \subset \mathcal { O }$ , is a reflexive space (see [1, Thm. 2.46, p. 49]), we get that there exist a sub-sequence $\{ u _ { \ell , \iota } ^ { \varepsilon , \delta _ { \hat { n } } } \} _ { \hat { n } \geq 1 }$ of $\{ u _ { \ell , \iota } ^ { \varepsilon , \delta } \} _ { \delta \in ( 0 , 1 ) }$ and a function $w _ { \ell , \iota } ^ { \varepsilon }$ in $\mathrm { W } _ { \mathrm { l o c } } ^ { 2 , \infty } ( { \mathcal { O } } )$ such that

$$
\begin{array}{l}u _ {\ell , \iota} ^ {\varepsilon , \delta_ {\hat {n}}} \underset {\delta_ {\hat {n}} \rightarrow 0} {\longrightarrow} w _ {\ell , \iota} ^ {\varepsilon} \text {in} C _ {\text {loc}} ^ {1} (\mathcal {O}),\\\mathcal {O} = \varepsilon^ {\delta_ {\hat {n}}} \quad \text {and} \mathcal {O} = \varepsilon^ {\delta_ {\hat {n}}} \text {and} L _ {1} ^ {2} (\mathcal {O}) = f _ {1} + f _ {2} - (1, \dots)\end{array}\tag {3.10}
\partial_ {i j} u _ {\ell , \iota} ^ {\varepsilon , \delta_ {\hat {n}}} \underset {\delta_ {\hat {n}} \to 0} {\longrightarrow} \partial_ {i j} w _ {\ell , \iota} ^ {\varepsilon}, \mathrm{weakly} \mathrm{L} _ {\mathrm{loc}} ^ {p} (\mathcal {O}), \mathrm{foreach} p \in (1, \infty).$$

Taking

$$
u _ {\ell , \iota} ^ {\varepsilon} (x) := \left\{ \begin{array}{l l} w _ {\ell , \iota} ^ {\varepsilon} (x) & \text {   if   } x \in \mathcal {O}, \\ f _ {\iota} (x) & \text {   if   } x \in \partial \mathcal {O}, \end{array} \right. \tag {3.11}
$$

and considering that $0 \leq u _ { \ell , \iota } ^ { \varepsilon , \delta _ { \hat { n } } } \leq v _ { \ell , \iota }$ uℓ,ι on O and $u _ { \ell , \iota } ^ { \varepsilon , \delta _ { \hat { n } } } = v _ { \ell , \iota } = f _ { \iota }$ uℓ,ι in ∂O, it implies that

$$0 \leq u _ {\ell , \iota} ^ {\varepsilon} \leq v _ {\ell , \iota} \text { on } \mathcal {O} \text { and } u _ {\ell , \iota} ^ {\varepsilon} = f _ {\iota} \text { in } \partial \mathcal {O}. \tag {3.12}$$
