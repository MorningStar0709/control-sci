Lemma 1.14 (Stabilizable systems and feasible targets). Consider an unconstrained, stabilizable system (A, B). The target is feasible for any $r _ { \mathrm { s p } }$ if and only if

$$
\operatorname{rank} \left[ \begin{array}{c c} I - A & - B \\ H C & 0 \end{array} \right] = n + n _ {c}
$$
