# 2. PROBLEM STATEMENT

Consider a discrete-time nonlinear state space model

$$x _ {t + 1} = f (x _ {t}, u _ {t}, w _ {t}) \tag {1}y _ {t} = h \left(x _ {t}\right) \tag {2}$$

where f and $h$ are unknown functions, $x _ { t } \ \in \ \mathbb { R } ^ { n _ { x } }$ is the current state of the system, $u _ { t } \in \mathbb { R } ^ { n _ { u } }$ is the input, $\boldsymbol { y } _ { t } \in \mathbb { R } ^ { n _ { \boldsymbol { y } } }$ is the output, $w _ { t } \in \mathbb { R } ^ { n _ { w } }$ is the system noise, and t is the time step.

The problem considered in the paper is to use RL to train a state feedback set point control policy,

$$u (t) = g (x _ {t}, y _ {t} ^ {\mathrm{ref}}) \tag {3}$$

to make the output $y _ { t }$ track a reference level $y _ { t } ^ { \mathrm { r e f } }$ . In order to allow for varying set points, the framework of multi-goal RL will be utilized.
