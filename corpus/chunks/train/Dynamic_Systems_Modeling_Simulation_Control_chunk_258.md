# Step Input

A step input function exhibits a sudden, instantaneous change from one constant value to another constant value. The unit-step input function U(t) “steps up” from zero to unity at time t = 0+

$$
\text { Unit - step   input } \quad U (t) = \left\{ \begin{array}{l l} 0 & \text { for } \quad t \leq 0 \\ 1 & \text { for } \quad t > 0 \end{array} \right. \tag {5.123}
$$

We can represent a step input with magnitude A as $u ( t ) = A U ( t )$ . For example, a sudden constant 30-N force applied at time t = 0 would be expressed as $u ( t ) = 3 0 U ( t ) \mathrm { \Delta N }$ .
