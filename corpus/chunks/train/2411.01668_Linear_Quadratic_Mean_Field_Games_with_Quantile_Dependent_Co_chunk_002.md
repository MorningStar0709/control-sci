# 2 Problem Statement

A quantile of a random variable X (or its distribution $F _ { X } )$ provides a least upper bound x such that a certain fraction α (with $0 \leq \alpha \leq 1 )$ of the probability mass is guaranteed to lie below or at x. More precisely, the α-quantile of a random variable X (resp. its distribution $F _ { X } )$ is denoted by $\mathcal { Q } _ { X } ( \alpha )$ (resp. $\mathcal { Q } _ { F _ { X } } ( \alpha ) )$ and is defined by

$$\mathcal {Q} _ {X} (\alpha) = \mathcal {Q} _ {F _ {X}} (\alpha) := \inf \{x \in \mathbb {R}: F _ {X} (x) \geq \alpha \}, \quad \alpha \in [ 0, 1 ], \tag {1}$$

where $F _ { X } ( x )$ denotes the cumulative distribution function (CDF) of X. For the special case of a Gaussian random variable $X \sim { \mathcal { N } } ( \mu , \sigma )$ with the mean $\mu$ and the standard deviation $\sigma ,$ the quantile function satisfies $\mathcal { Q } _ { X } ( \alpha ) = \mu + \sigma \Phi _ { \alpha } ^ { - 1 }$ where $\Phi _ { ( \cdot ) } ^ { - 1 }$ is the probit function, i.e., the inverse CDF of the standard (scalar-valued) normal distribution (see Figure 1).

$$d x _ {i} (t) = \left(a x _ {i} (t) + b u _ {i} (t)\right) d t + \sigma d w _ {i} (t), \quad x _ {i} (t), u _ {i} (t), w _ {i} (t) \in \mathbb {R}, \quad t \in [ 0, T ], \tag {2}$$
