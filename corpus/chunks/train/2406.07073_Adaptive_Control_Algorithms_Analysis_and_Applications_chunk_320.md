# Tracking (Computation of $T ( q ^ { - 1 } ) )$

The precompensator $T ( q ^ { - 1 } )$ is computed in order to ensure (in accordance to Fig. 7.9) the following transfer operator between the reference r(t) and y(t):

$$H _ {C L} (q ^ {- 1}) = \frac {q ^ {- d - 1} B _ {m} (q ^ {- 1})}{A _ {m} (q ^ {- 1})} = \frac {q ^ {- d - 1} B _ {m} (q ^ {- 1}) T (q ^ {- 1})}{A _ {m} (q ^ {- 1}) P (q ^ {- 1})} \tag {7.105}$$

From (7.105) one obtains:

$$T (q ^ {- 1}) = P (q ^ {- 1}) \tag {7.106}$$

The input to $T ( q ^ { - 1 } )$ at sampling instant t is the desired trajectory $y ^ { \star } , d + 1$ steps ahead $( y ^ { \star } ( t + d + 1 ) )$ .
