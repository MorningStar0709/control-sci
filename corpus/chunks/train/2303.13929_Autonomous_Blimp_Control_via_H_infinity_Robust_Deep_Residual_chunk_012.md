Then according to Table. I and the constraint (19) and assuming average case when agent actions are uniform random and have an average zero, i.e., $\mathbb { E } \left[ a \right] = 0 ,$ we can select any distribution for the mixing factor q as long as the mean of the distribution is positive, i.e., $\mathbb { E } \left[ q \right] \ge 0$ . Or assuming worst-case scenario when we have an adversarial agent, i.e., $\mathbb { E } \left[ a \right] = - u$ , then when E $[ q ] \ge 0 . 0 8$ for TurtleSim or $\mathbb { E } \left[ q \right] \geq 0$ for the blimp simulator, the process will remain stable.

However, because our plant model is likely imperfect and considering other disturbance and noise that is not modeled, the allowed maximum input disturbance will be less than the estimation. In practice, our conservative choice of E $[ q ] = 0 . 5$ for the TurtleSim and $\mathbb { E } \left[ q \right] = 0 . 3$ for the blimp simulator seem to work well.
