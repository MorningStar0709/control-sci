In designing TS-based adaptive control algorithms for multidimensional stabilizable LQRs, one needs to maintain bounded state. In bounding the state, Abeille and Lazaric (2018) rely on the fact that the underlying system is contractive, $\lVert \bar { \boldsymbol { A } } + \tilde { \boldsymbol { B } } \boldsymbol { K } ( \tilde { \boldsymbol { \Theta } } ) \rVert ~ < ~ 1$ . However, under Assumption 1, even if the optimal policy of the underlying system is chosen by the learning agent, the closedloop system may not be contractive since for any symmetric matrix $M , \rho ( M ) \leq \| M \|$ . Thus, to avoid dire consequences of unstable dynamics, TS-based adaptive control algorithms should focus on finite-time stabilization of the system dynamics in the early stages.

Moreover, the lack of contractive closed-loop mappings in stabilizable LQRs, prevent frequent policy changes used in Abeille and Lazaric (2018). From the definition of $( \kappa , \gamma )$ -stabilizability (Assumption 1), for any stabilizing controller $K ^ { \prime }$ , we have that $A _ { * } + B _ { * } K ^ { \prime } = H ^ { \prime } L H ^ { \prime - 1 }$ , with $\| L \| < 1$ for some similarity transformation $H ^ { \prime }$ . Thus, even if all the policies are stabilizing, changing the policies at every time step could cause couplings of these similarity transformations and result in linear growth of state over time. Thus, TS-based adaptive control algorithms need to find the balance in rate of policy updates, so that frequent policy switches are avoided, yet, enough optimistic policies are sampled. In light of these observations, our results hinge on the following:

1) Improved exploration of TSAC, which allows fast stabilization of the system dynamics,   
2) Fixed policy update rule of TSAC, which prevents state blow-up and reduces $R _ { T } ^ { \mathrm { g a p } }$ and $R _ { T } ^ { \mathrm { T S } }$   
3) A novel result that shows TS samples optimistic model parameters with a constant probability for multidimensional LQRs and gives a novel bound on $R _ { T } ^ { \mathrm { T S } }$ .
