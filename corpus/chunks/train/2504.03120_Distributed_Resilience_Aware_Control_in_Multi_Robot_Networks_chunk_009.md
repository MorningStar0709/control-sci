# C. Control Barrier Function

In this work, we leverage Control Barrier Functions (CBFs) [16] to ensure resilient consensus despite an F -total attack. Note we are considering time-invariant CBFs. Let a continuously differentiable function $h : { \mathcal { D } } \subset \mathbb { R } ^ { n m } \to \mathbb { R }$ define a safe set $\mathcal { C } = \{ \mathbf { x } \in \mathbb { R } ^ { n m } \mid h ( \mathbf { x } ) \geq 0 \}$ and its boundary $\partial { \mathcal { C } } = \{ \mathbf { x } \in \bar { \mathbb { R } } ^ { n m } \mid h ( \mathbf { x } ) = 0 \}$ . We say h is a CBF for the system (2) if there exists an extended class $\kappa _ { \infty }$ function $\alpha : \mathbb { R }  \mathbb { R }$ such that

$$\sup _ {\mathbf {u} \in \mathcal {U}} \left[ \frac {\partial h}{\partial \mathbf {x}} \dot {\mathbf {x}} + \alpha (h (\mathbf {x})) \right] \geq 0, \quad \forall \mathbf {x} \in \mathcal {C}. \tag {8}$$

If h is a CBF on a given set C for given system dynamics and $\begin{array} { r } { \frac { \partial h } { \partial \mathbf { x } } ( \mathbf { x } ) \neq 0 \forall \mathbf { x } \in \bar { \partial } \mathcal { C } } \end{array}$ , then any locally Lipschitz continuous controller $\mathbf { u } ^ { * } ( \mathbf { x } ) \in K _ { \mathrm { c b f } } ( \mathbf { x } ) = \{ \mathbf { u } \in \mathcal { U } \mid \frac { \partial h } { \partial \mathbf { x } } \dot { \mathbf { x } } \geq - \alpha ( h ( \mathbf { x } ) ) \}$ renders the set C forward invariant [16, Theorem 2].
