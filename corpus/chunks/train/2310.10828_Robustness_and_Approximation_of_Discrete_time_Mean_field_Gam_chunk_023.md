Hence, for large enough $n \mathrm { { ^ { * } s } }$ , we have $F _ { n } ( x ^ { ( n ) } , a ^ { ( n ) } ) > \operatorname* { m i n } _ { a } F ( x ^ { ( n ) } , a )$ which implies that $( x ^ { ( n ) } , a ^ { ( n ) } ) \notin$ $C _ { n }$ . Then, by [28, Theorem 3.5], for each M we have lim su $\mathrm { l p } _ { n  \infty } \nu _ { n } \otimes \pi _ { n } \big ( B ^ { M } \cap C _ { n } \big ) = 0$ . Therefore, we obtain

$$1 \leq \operatorname * {l i m s u p} _ {n \to \infty} \nu_ {n} \otimes \pi_ {n} \big (C \cap C _ {n} \big) \leq \operatorname * {l i m s u p} _ {n \to \infty} \nu_ {n} \otimes \pi_ {n} (C) \leq \nu \otimes \hat {\pi} (C),$$

where the last inequality follows from Portmanteau theorem [12, Theorem 2.1] and the fact that C is closed. Hence, $\nu \otimes \hat { \pi } ( C ) = 1$ .

Now, let us complete the proof in view of the above result. Note that

$$\nu \otimes \hat {\pi} \left\{(x, a): F (x, a) = \min _ {a} F (x, a) \right\} = 1$$

Hence, for $\nu \mathrm { - a . e . }$ , we have

$$\hat {\pi} \left(\{(x, a): F (x, a) = \min F (x, a) \} \Big | x\right) = 1$$

But note that arg min ${ \bf \Phi } _ { a } F ( x , a )$ is unique for all x, and so, for the x-values, where above equality is true, we have

$$\operatorname{supp} \hat {\pi} (\cdot | x) = \underset {a} {\arg \min} F (x, a)$$

This implies that ${ \hat { \pi } } ( \cdot | x ) = \pi ( \cdot | x ) \ \nu { \mathrm { - a . e } }$ . Hence, $\nu \otimes \hat { \pi } = \nu \otimes \pi$ . This completes the proof.
