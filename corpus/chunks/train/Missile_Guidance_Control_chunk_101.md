# (e) Derivation of Translation Equations $( Y _ { b } { \cdot } \mathbf { B o d y } \ \mathbf { A x i s } )$

The forces acting along the $Y _ { b } – \mathbf { b o d y }$ axis of a missile consist of the $Y _ { b } \mathrm { - c o m p o n e n t s }$ of weight and aerodynamic normal force. Recall that the $Y _ { b }$ -component of the aerodynamic normal force is

$$F _ {Y} = C _ {Y} q S.$$

The weight component of the missile along the $Y _ { b } \mathrm { - a x i s }$ is

$$y _ {m g} = m g \sin \phi \cos \theta , \tag {3.47}$$

where

$$
y _ {m g} = \text { weight component along the } Y _ {b} \text {-axis},
\begin{array}{l} \phi = \text { angle   between   the   projection   of   the   weight   vector   onto   the } Y _ {b} - Z _ {b} \text { plane } \\ \text { and   the } Z _ {b} \text {-axis; this   is   the   roll   angle   and   is   defined   to   be   positive   for   the } \\ \text { counterclockwise   sense   of   rotation   in   the } Y _ {b} - Z _ {b}. \end{array}
$$

Note that the weight vector is directed along the $Z _ { E ^ { - } \mathbf { a x i s } }$ . If we project this vector onto the $Y _ { b } - Z _ { b }$ plane by dropping a perpendicular, we obtain the vector mg cos θ. Next, we drop a perpendicular from this vector onto the $Z _ { b } \mathrm { - a x i s . }$ , thus obtaining the Zb-component of the weight:

$$z _ {m g} = m g \cos \theta \cos \phi . \tag {3.48}$$

The third side of this vector triangle is parallel to the $Y _ { b } – \mathrm { a x i s } .$ , and so is the $Y _ { b } \mathrm { - c o m p o n e n t }$ of the weight. This vector is positive because of its orientation with respect to the $Y _ { b } – \mathrm { a x i s }$ . Now we can sum the force components along the $Y _ { b } \mathrm { - a x i s : }$

$$F _ {y} = y _ {m g} - F _ {Y} = m g \sin \phi \cos \theta - C _ {Y} q S. \tag {3.49}$$
