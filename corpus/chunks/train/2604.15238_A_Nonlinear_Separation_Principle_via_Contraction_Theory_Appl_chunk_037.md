# C. Low gain Integral control

Having designed a controller in Proposition 27, and an observer in Proposition 30, which satisfies the assumptions in our separation principle in Theorem 3, we now proceed to perform reference tracking via an integral controller and a singular perturbation argument. As a first step, we identify sufficient conditions for the contraction of the reduced dynamics.

Lemma 33 (Contractivity of the reduced dynamics): Let a map $x ^ { \star } \colon \mathbb { R } ^ { m }  \mathbb { R } ^ { n }$ exist that satisfies the equation

$$x ^ {\star} (u) = \Psi (W x ^ {\star} (u) + B u) \tag {59}$$

where $W \in \mathbb { R } ^ { n \times n } , B \in \mathbb { R } ^ { n \times m }$ . Let $A = I _ { n } - W , C \in \mathbb { R } ^ { p \times n }$ and $Q \in \mathbb { R } ^ { n \times n }$ be a diagonal positive matrix. Assume:

(A1) Ψ is slope-restricted in [δ, 1], for $\delta > 0$ .   
(A2) There exists $u ^ { \star }$ such that $r = C x ^ { \star } ( u ^ { \star } ) .$

$$

$$

$$\dot {u} = \varepsilon K (r - C x ^ {\star} (u)) \tag {61}$$
