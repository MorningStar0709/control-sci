# 7.2 Parametric Programming

A conventional optimization problem has the form $V ^ { 0 } = \mathrm { m i n } _ { u } \{ V ( u ) \ |$ | $u \in \mathcal { U } \}$ where u is the “decision” variable, $V ( u )$ is the cost to be minimized, and U is the constraint set. The solution to a conventional optimization is a point or set in U; the value $V ^ { 0 }$ of the problem satisfies $V ^ { 0 } = V ( \boldsymbol { u } ^ { 0 } )$ where $u ^ { 0 }$ is the minimizer. A simple example of such a problem is $V ^ { 0 } = \mathrm { m i n } _ { u } \{ a + b u + ( 1 / 2 ) c u ^ { 2 } \mid u \in [ - 1 , 1 ] \}$ where the solution is required for only one value of the parameters a, b and c. The solution to this problem $u ^ { 0 } = - b / c { \mathrm { ~ i f ~ } } | b / c | \ \leq \ 1 , u ^ { 0 } = - 1$ if $b / c \geq 1$ and $u ^ { 0 } = 1 { \mathrm { ~ i f ~ } } b / c \leq - 1$ . This may be written more compactly as $u ^ { 0 } = - \mathsf { s a t } ( b / c )$ where sat(·) is the saturation function. The corresponding value is $V ^ { 0 } = a - b ^ { 2 } / 2 c \mathrm { i f } | b / c | \le 1 , V ^ { 0 } = a - b + c ^ { 2 } / 2$ if $b / c \geq 1$ and $V ^ { 0 } = a + b + c ^ { 2 } / 2$ if $b / c \leq - 1$ .
