# Example 9.1

Find the system type for a closed loop negative feedback system consisting of the following elements:

$$C (s) = \frac {5 0 0}{s (s + 1 0)} \quad P (s) = \frac {(s + 0 . 1)}{s \left(s ^ {2} + 5 0 s + 1 5 0 0\right)} \quad H (s) = 1$$

Every factor in the denominatory of s by itself (i.e. (s + 0) ) is a pole at the origin. The combined system CP H(s) has two poles at the origin $( s ^ { 2 } ) ^ { \ J }$ so it is of type 2.
