# 5.2. Regret: A Good But Computationally Expensive Measure

If the regret-based predictor performance measure is used, then

$$
\begin{array}{l} E _ {R} (\bar {W} _ {t o} (o), \bar {W} _ {b}) = \underset {\bar {w} \sim \bar {W} _ {t o} (o)} {\hat {\mathbb {E}}} R (\bar {w}, \bar {W} _ {b}) \\ = \underset {\bar {w} \sim \bar {W} _ {t o} (o)} {\hat {\mathbb {E}}} \left(J _ {\bar {w}} ^ {\pi_ {\bar {W} _ {b}}} - J _ {\bar {w}} ^ {\pi_ {\bar {w}}}\right) \tag {32} \\ = \underset {\bar {w} \sim \bar {W} _ {t o} (o)} {\hat {\mathbb {E}}} J _ {\bar {w}} ^ {\pi_ {\bar {W} _ {b}}} - \underset {\bar {w} \sim \bar {W} _ {t o} (o)} {\hat {\mathbb {E}}} J _ {\bar {w}} ^ {\pi_ {\bar {w}}}. \\ \end{array}
$$

The second term $\hat { \mathbb { E } } _ { \bar { w } \sim \bar { W } _ { t o } } J _ { \bar { w } } ^ { \pi _ { \bar { w } } }$ is the posteriori optimal cost, which is a constant given $\hat { W } _ { t o }$ . The first term is the same as the expected cost. The regret is essentially the control cost with a constant offset. $\hat { W } _ { t o }$ globally minimizes $E _ { R } ( \bar { W } _ { t o } ( o ) , \cdot )$ . Furthermore, decreasing $E _ { R } ( \bar { W } _ { t o } ( o ) , \bar { W } _ { b } )$ will lead to a decrease of the expected cost. Evaluating the predictors using regret-based error measures is essentially evaluating the control performance after connecting the prediction and the optimal control process. Actually, we can just compute the first term $\hat { \mathbb { E } } _ { \bar { w } \sim \bar { W } _ { t o } ( o ) } J _ { \bar { w } } ^ { \pi _ { \bar { W } _ { b } } }$ πW¯ b , which is the control cost under the predictor, to evaluate this predictor. The regret is both a best-P-lowest-C and a better-P-lower-C measure.
