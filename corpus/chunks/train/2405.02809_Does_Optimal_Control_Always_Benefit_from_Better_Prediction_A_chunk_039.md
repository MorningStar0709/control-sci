# 6.1. A Simple Linear System Example

We provide a numerical example where the predictor measure gets better while the control cost gets worse in a Type I problem, and illustrate the differences of the three predictor measures shown in Table 1. Consider the simple linear system,

$$x _ {k + 1} = x _ {k} + w _ {k} + u _ {k}, \quad k = 0, 1, \tag {34}$$

where $- 1 \le u \le 1 , x _ { 0 } = w _ { 0 } = 0$ , with a quadratic cost function $J = x _ { 2 } ^ { 2 }$ . There is no observation information other than $w .$ . The input $u _ { 1 }$ will be determined after $w _ { 1 }$ is measured. So the key of this predictive optimal control is to forecast the value of $w _ { 1 }$ and determine $u _ { 0 }$ accordingly.

Assume that $w _ { 1 } \in \{ - 3 , 2 \}$ . Therefore, the observable true probability distribution of $w _ { 1 }$ is in the form of

$$\mathbb {P} (w _ {1} = - 3) = p, \quad \mathbb {P} (w _ {1} = 2) = 1 - p. \tag {35}$$

In addition, we assume that $\begin{array} { r } { 0 \leq p \leq { \frac { 2 } { 3 } } } \end{array}$ . In this example, the hidden prediction state s can be considered as $w _ { 1 }$ . If we know the value of $p$ and can use it to design the optimal control accordingly, the ideal optimal policy is

$$
u _ {1} ^ {*} = \left\{ \begin{array}{l l} 1, & \text { if } w _ {1} = - 3, \\ - 1, & \text { if } w _ {1} = 2, \end{array} \right. \tag {36}
u _ {0} ^ {*} = 3 p - 1,$$

and the ideal optimal cost expectation is,

$$\mathbb {E} J _ {\bar {w} (S)} ^ {\pi_ {\bar {W} _ {t o} (o (S))}} = - 9 p ^ {2} + 9 p. \tag {37}$$

A predictor gives a prediction of $p$ as $p _ { b }$ and $\begin{array} { r } { 0 \leq p _ { b } \leq \frac { 2 } { 3 } } \end{array}$ . $p _ { b }$ generates a belief of $w _ { 1 }$ . The predictive optimal solution is in the same form as (36) while replacing $p$ with $p _ { b }$ . With this belief, the predictive optimal control cost expectation is

$$\mathbb {E} J _ {\bar {w} (S)} ^ {\pi_ {\mathcal {P} (o (S))}} = 9 p _ {b} ^ {2} - 1 8 p p _ {b} + 9 p. \tag {38}$$

Obviously, $p _ { b } = p$ minimizes this cost expectation.

When using the MSE, the regret, and the log-likelihood measure, the predictor performance measures are

$$E _ {M} (\bar {W} _ {t o}, \bar {w} _ {b}) = - 5 0 p p _ {b} + 2 5 p _ {b} + 2 5 p, \tag {39}E _ {R} (\bar {W} _ {t o}, \bar {w} _ {b}) = 9 p _ {b} ^ {2} - 1 8 p p _ {b} + 8 p, \tag {40}E _ {P} (\bar {W} _ {t o}, \bar {w} _ {b}) = - p \log p _ {b} - (1 - p) \log (1 - p _ {b}), \tag {41}$$
