$$\widetilde {M} (\bar {x}) \dot {v} _ {o} + \widetilde {C} (\bar {x}) v _ {o} + \widetilde {g} (\bar {x}) = G u, \tag {3.39}$$

where we slightly change the notation with respect to the previous chapter as $\bar { x } : = [ q ^ { \top } , \bar { q } ^ { \top } , \eta _ { o } ^ { \top } , \omega _ { o } ^ { \top } ] ^ { \top } \in \mathbb { X } = \mathsf { S } \times \mathbb { R } ^ { n + 6 } \times \mathbb { T }$ (instead of x).

The vector of interaction forces h among the agents and the object can be decoupled into motion-induced and internal forces

$$h = h _ {\mathrm{m}} + h _ {\text {int}}. \tag {3.40}$$

The internal forces $h _ { \mathrm { i n t } }$ are squeezing forces that the agents exert to the object and belong to the nullspace of $G ( x ) ~ ( \mathrm { i . e . , } ~ G h _ { \mathrm { i n t } } = 0 )$ . Hence, they do not contribute to the acceleration of the coupled system and result in internal stresses that might damage the object. A closed form analytic expression for $h _ { \mathrm { m } }$ and $h _ { \mathrm { i n t } }$ will be given in the next section.

Note from (3.37) that the agent velocities v belong to the range space of $G ^ { \top }$ . Therefore, since G is a matrix that encodes rigidity constraints, this motivates the association of G to the rigidity matrix used in formation rigidity theory, and of the rigid cooperative manipulation scheme to a multiagent rigid formation scheme. To this end, we introduce now the notion of Distance and Bearing Rigidity in SE(3).
