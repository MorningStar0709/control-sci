$$\hat {\mathbf {g}} _ {k + 1} = \underset {\mathbf {g}} {\operatorname{argmin}} c (\hat {\mathbf {g}} _ {k}) \| \mathbf {g} \| _ {2} ^ {2} + \| \boldsymbol {\delta} _ {\zeta} (\mathbf {g}) \| _ {\Psi^ {- 1} (\hat {\mathbf {g}} _ {k})} ^ {2}, \tag {17}$$

$\mathrm { w h e r e } \ c ( \hat { \mathbf { g } } _ { k } ) : = \operatorname { t r } \big ( \Lambda \Psi ^ { - 1 } ( \hat { \mathbf { g } } _ { k } ) \big ) - \big \| \Psi ^ { - 1 } ( \hat { \mathbf { g } } _ { k } ) \pmb { \delta } _ { \zeta } ( \hat { \mathbf { g } } _ { k } ) \big \| _ { \Lambda } ^ { 2 } .$

The SQP algorithm for (16) can be obtained similarly:

$$\hat {\mathbf {g}} _ {k + 1} = \underset {\mathbf {g}} {\operatorname{argmin}} \bar {c} (\hat {\mathbf {g}} _ {k}) \| \mathbf {g} \| _ {2} ^ {2} + \left\| U _ {\Psi , 1} ^ {\top} \boldsymbol {\delta} _ {\zeta} (\mathbf {g}) \right\| _ {\Psi_ {1} ^ {- 1} (\hat {\mathbf {g}} _ {k})} ^ {2} \tag {18}\text { s.t. } U _ {\Psi , 2} ^ {\top} \boldsymbol {\delta} _ {\zeta} (\mathbf {g}) = \mathbf {0},$$

where $\bar { c } ( \hat { \bf g } _ { k } ) : = \mathrm { t r } \left( \bar { \Lambda } \Psi _ { 1 } ^ { - 1 } ( \hat { \bf g } _ { k } ) \right) - { \left. { \Psi _ { 1 } ^ { - 1 } ( \hat { \bf g } _ { k } ) U _ { \Psi , 1 } ^ { \top } \delta _ { \zeta } ( \hat { \bf g } _ { k } ) } \right. } _ { \bar { \Lambda } } ^ { 2 }$ and $\bar { \Lambda } = U _ { \Psi , 1 } ^ { \top } \Lambda U _ { \Psi , 1 } .$ .

Problem (17) can be non-convex when the Hessian $c ( \hat { \mathbf { g } } _ { k } ) \mathbb { I } _ { M } + \dot { H } ^ { \top } \Phi ^ { \top } \Psi ^ { - 1 } ( \hat { \mathbf { g } } _ { k } )$ )ΦH is not positive semidefinite, as commonly observed in SQP. Standard techniques in sequential convex programming can be applied to address this issue, such as projecting the Hessian onto the space of positive semidefinite matrices and utilizing the trust region or cubic regularization method (Nesterov and Polyak, 2006).

If closed-form solutions are desired, one can further approximate (17) or (18) by only running one iteration from the pseudoinverse solution to $\zeta = \Phi H \bar { \bf g } ;$ :

$$\hat {\mathbf {g}} _ {0} = \hat {\mathbf {g}} _ {\mathrm{pinv}} = H ^ {\top} \Phi^ {\top} \left(\Phi H H ^ {\top} \Phi^ {\top}\right) ^ {- 1} \boldsymbol {\zeta}.$$

In this case, $\delta _ { \zeta } ( \hat { \bf g } _ { 0 } ) = { \bf 0 } _ { p }$ and $c ( \hat { \bf g } _ { k } ) , \bar { c } ( \hat { \bf g } _ { k } )$ are positive. 1 So, (17) and (18) are guaranteed to be convex. Similar to (13) and (14), convex quadratic programs with only equality conditions admit closed-form solutions, which we omit here for space constraints. A similar strategy is shown to perform well numerically for the prediction problem (Yin et al., 2021).
