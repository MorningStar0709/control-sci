# 5.2 Introduction

This chapter will introduce calculation of the response of systems which we have described by transfer functions. First we consider step response of a 2nd order system in the time domain. Step response of higher order systems requires numerical evaluation but we can build much intuition from considering 2nd order systems. Secondly, we will study steady state output corresponding to constant sinusoidal inputs and develop methods to hand-sketch magnitude and phase of the output response for a system of any order.

Although it is easier to do both of these computations with the computer, basic hand skills are studied because they develop better intuition and understanding of systems and system response results. The basic analytical approach is to understand the big picture by hand sketching these responses rst, to a practical level of accuracy, and then easily create very accurate plots with the computer for detailed analysis.
