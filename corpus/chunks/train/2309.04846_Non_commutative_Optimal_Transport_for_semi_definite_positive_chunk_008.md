Note that when $\tau _ { 1 } , \tau _ { 2 } \to + \infty$ , the $( c , \tau , \varepsilon )$ -transforms (2.7) and (2.8) become the classical $( c , \varepsilon )$ −transform (or Sinkhorn iterations) [12] for the Shannon-Entropy regularized optimal transport problem (2.3).

The following proposition shows that the Unbalanced $( c , \tau , \varepsilon )$ −transforms are well-defined.

Proposition 2.1. Let $\varepsilon , \tau _ { 1 } , \tau _ { 2 } > 0$ be the positive parameters, X and Y be complete separable metric spaces, $\mu \in \mathcal { M } _ { + } ( X ) , \ \nu \in \mathcal { M } _ { + } ( Y )$ be positive measures, and let $c \in L ^ { \infty } ( X \times Y )$ be a cost function. $I f u \in L ^ { \infty } ( X ) , v \in L ^ { \infty } ( Y )$ , then $u ^ { ( c , \tau _ { 2 } , \varepsilon ) } \in L ^ { \infty } ( Y )$ and $v ^ { ( c , \tau _ { 1 } , \varepsilon ) } \in L ^ { \infty } ( X )$ .

Proof. It suffices to show that the integral part of the transform is bounded, exploiting the fact that $| | c | | _ { \infty } , | | u | | _ { \infty } , | | v | | _ { \infty } < \infty$ . The following estimates will be shown for $u ^ { ( c , \tau _ { 2 } , { \varepsilon } ) }$ , and the identical bound will also hold for $\boldsymbol { v } ^ { ( c , \tau _ { 1 } , \varepsilon ) }$ .

By monotonicity of the exponential, the inequality

$$\exp \left(\frac {- | | u | | _ {\infty} - | | c | | _ {\infty}}{\varepsilon}\right) \leq \exp \left(\frac {u (x) - c (x , y)}{\varepsilon}\right) \leq \exp \left(\frac {| | u | | _ {\infty} + | | c | | _ {\infty}}{\varepsilon}\right)$$

holds $\mu$ and ν−almost everywhere, and since $\mu ( X )$ is a finite positive value, taking the

integrals and logarithms on both sides we easily obtain

$$\left| \left| u ^ {(c, \tau_ {2}, \varepsilon)} + \frac {\tau_ {2} \varepsilon}{\tau_ {2} + \varepsilon} \log (\mu (X)) \right| \right| _ {\infty} \leq \frac {\tau_ {2} \varepsilon}{\tau_ {2} + \varepsilon} \left(\frac {| | u | | _ {\infty} + | | c | | _ {\infty}}{\varepsilon}\right),$$

which concludes the proof.
