# 3. Problem Statement and Notation

A (finite) Markov Decision Process $( S , { \mathcal { A } } , { \mathsf { P } } , r , \rho , \gamma )$ is specified by a finite state space ${ \mathcal { S } } _ { : }$ , a finite action space ${ \mathcal { A } } ,$ a transition probability matrix P, where $\mathbb { P } \left( \tilde { s } | s , a \right)$ is the probability of transitioning into state s˜ upon taking action $a \in { \mathcal { A } }$ in state s, a single stage reward function $r : S \times \mathcal { A }  \mathbb { R }$ , a starting state distribution $\rho$ over $s$ and a discount factor $\gamma \in ( 0 , 1 )$ . A (stationary) policy or controller $\pi : S  { \mathcal { P } } ( A )$ specifies a decision-making strategy in which the learner chooses actions $\left( { { a } _ { t } } \right)$ adaptively based on the current state $\left( { { s _ { t } } } \right)$ , i.e., $a _ { t } \sim \pi ( s _ { t } )$ . π and $\rho ,$ together with P, induce a probability measure $\mathbb { P } _ { \rho } ^ { \pi }$ on the space of all sample paths of the underlying Markov process and we denote by $\mathbb { E } _ { \rho } ^ { \pi }$ the associated expectation operator. The value function of policy $\pi$ (also called the value of policy $\pi )$ , denoted by $V ^ { \pi }$ is the total discounted reward obtained by following π, i.e., $\begin{array} { r } { V ^ { \pi } ( \rho ) : = \mathbb { E } _ { \rho } ^ { \pi } \sum _ { t = 0 } ^ { \infty } \gamma ^ { t } r ( s _ { t } , a _ { t } ) } \end{array}$ .

Improper Learning.
