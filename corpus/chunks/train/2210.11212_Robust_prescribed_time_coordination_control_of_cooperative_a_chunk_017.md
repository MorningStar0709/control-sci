# III. MAIN RESULTS

A. Prescribed-time coordination control without disturbances

In this subsection, the PTCC problems of the nominal CAN (3) are addressed. The control protocol for each agent $k , k \in$ $\mathcal { I } _ { N }$ is designed as

$$u _ {k} = \left(\rho_ {1} + \rho_ {2} \frac {\dot {\varphi} (t , T)}{\varphi (t , T)}\right) \sum_ {l \in \mathcal {N} _ {k}} w _ {k l} [ x _ {l} - \operatorname{sign} (w _ {k l}) x _ {k} ], \tag {4}$$

where $\rho _ { 1 } ~ > ~ 0 , ~ \rho _ { 2 } ~ > ~ 0 , ~ \varphi ( t , T )$ and $\dot { \varphi } \left( t , T \right)$ are defined in Lemma 6 with T specified as $T = T _ { 1 } > 0$ , and $w _ { k l }$ is the $( k , l )$ th element of the adjacency matrix W . It is worth highlighting that the protocol (4) is completely distributed and scalable since it relies upon merely the relative states of neighboring agents, without exploiting any global information.

Note that, in the special case when $\rho _ { 1 } = 1$ and $\rho _ { 2 } = 0$ , the above protocol (4) reduces to the typical Laplacian-type protocol studied in References [1], [30], [32], [33], which was proved to solve the asymptotic stability, bipartite consensus, interval bipartite consensus, and bipartite containment control problems of the nominal CAN (3) under suitable topology conditions. It is noteworthy that the convergence analysis approaches presented in References [1], [30], [32], [33] are no longer applicable here.

Let $\begin{array} { r } { \boldsymbol { e } _ { k } = \sum _ { l \in \mathcal { N } _ { k } } \boldsymbol { w } _ { k l } \left[ \boldsymbol { x } _ { l } - \mathrm { s i g n } \left( \boldsymbol { w } _ { k l } \right) \boldsymbol { x } _ { k } \right] , k \in \mathcal { I } _ { N } } \end{array}$ . Further, denote $\boldsymbol { E } = \left[ e _ { 1 } , \ldots , e _ { N } \right] ^ { \mathrm { T } }$ and $X = \left[ x _ { 1 } , \ldots , x _ { N } \right] ^ { \mathrm { T } }$ . Then we have

$$E = - \mathscr {L} X. \tag {5}$$

In view of (3) and (4), we clearly have

$$\dot {x} _ {k} = \left(\rho_ {1} + \rho_ {2} \frac {\dot {\varphi} (t , T _ {1})}{\varphi (t , T _ {1})}\right) \sum_ {l \in \mathcal {N} _ {k}} w _ {k l} [ x _ {l} - \mathrm{sign} (w _ {k l}) x _ {k} ],$$

k $\in \mathcal { I } _ { N }$ , which can be represented compactly as

$$\dot {X} = - \left(\rho_ {1} + \rho_ {2} \frac {\dot {\varphi} (t , T _ {1})}{\varphi (t , T _ {1})}\right) \mathscr {L} X. \tag {6}$$

This, together with (5), yields

$$\dot {E} = - \left(\rho_ {1} + \rho_ {2} \frac {\dot {\varphi} (t , T _ {1})}{\varphi (t , T _ {1})}\right) \mathscr {L} E. \tag {7}$$
