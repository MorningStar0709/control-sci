# 6. Case Study

We conduct a case study to illustrate the performance of the RLb method. We consider the IEEE 39-bus power network and assume each bus represents an aggregate area containing loads and generators with system parameters from [5, 27]. We consider the time horizon of interest to be $T = 5 0$ seconds and bus 38 encountering a sudden change in power injection during the time interval (0, 2] seconds, with $\Delta p _ { 3 8 } = - p _ { 3 8 } ^ { \mathrm { n o m } }$ . The nominal frequency is 60 Hz, and the safe region is set as [59.8, 60.2] Hz for every bus.
