$$d _ {k, \text {col}} < \| p _ {k _ {2}} (0) - p _ {k _ {1}} (0) \| < d _ {k, \text {con}}, \forall k \in \mathcal {K}. \tag {3.4}$$

The desired formation is specified by the constants $d _ { k , \mathrm { d e s } } \in \mathbb { R } _ { \geq 0 } , R _ { k , \mathrm { d e s } } \in$ $\mathbb { S } \mathbb { O } ( 3 ) , \forall k \in { \cal K }$ , for which, the formation configuration is called feasible if the set $\{ x \in \mathbb { S } \mathbb { E } ( 3 ) ^ { N } : \| p _ { k _ { 2 } } - p _ { k _ { 1 } } \| = d _ { k , \mathrm { d e s } } , \ R _ { k _ { 2 } } ^ { \top } R _ { k _ { 1 } } = R _ { k , \mathrm { d e s } } , \forall k \in \mathcal { K } \}$ their communication capabilities are limited, the control protocol, except from achieving a desired inter-agent formation and maintaining connectivity, should also guarantee for all $t \in \mathbb { R } _ { \geq 0 }$ that the initially neighboring agents avoid collision with each other. Therefore, all pairs $( k _ { 1 } , k _ { 2 } ) \in \mathcal { N } ^ { 2 }$ of agents that initially form an edge must remain within distance greater than $d _ { k , \mathrm { c o l } }$ and less than $d _ { k , \mathrm { c o n } }$ . We also make the following assumptions that on the graph topology:

Assumption 3.1. The communication graph $\mathcal { G }$ is a tree.

Formally, the robust formation control problem under the aforementioned constraints is formulated as follows:

Problem 3.1. Given N agents governed by the dynamics (3.1), under Assumption 3.1 and given the desired inter-agent configuration constants $d _ { k , \mathrm { d e s } } \in \mathbb { R } _ { \geq 0 } , \ R _ { k , \mathrm { d e s } } \in \mathbb { S } \mathbb { O } ( 3 )$ , with $d _ { k , \mathrm { c o l } } < d _ { k , \mathrm { d e s } } < d _ { k , \mathrm { c o n } } .$ , k , design decentralized control laws $u _ { i } \in \mathbb { R } ^ { 6 } , i \in \mathcal { N }$ such that, $\forall k \in \mathcal { K }$ , the following hold:

1. $\operatorname* { l i m } _ { t \to \infty } \| p _ { k _ { 2 } } ( t ) - p _ { k _ { 1 } } ( t ) \| = d _ { k , \mathrm { d e s } } ;$   
2. $\operatorname* { l i m } _ { t  \infty } [ R _ { k _ { 2 } } ( t ) ] ^ { \top } R _ { k _ { 1 } } ( t ) = R _ { k , \mathrm { d e s } } ;$   
$\begin{array} { r } { 3 . \ d _ { k , \mathrm { c o l } } < \Vert p _ { k _ { 2 } } ( t ) - p _ { k _ { 1 } } ( t ) \Vert < d _ { k , \mathrm { c o n } } , \forall \ t \in \mathbb { R } _ { \ge 0 } . } \end{array}$

The term “robust” here refers to robustness of the proposed methodology with respect to the unknown dynamics and external disturbances in (3.1) as well as the unknown noise $\mathsf { n } _ { i } ( \cdot )$ in the velocity feedback.
