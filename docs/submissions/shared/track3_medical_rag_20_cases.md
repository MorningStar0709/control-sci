## 医疗证据工作流入口

医学 RAG 的关键不在于回答更多问题，而在于把文献整理、来源复核、边界判断和私有部署这些高成本环节变成可追溯的证据工作流。下表把样例包中的案例按实际使用场景重新组织，便于从问题类型进入对应证据。

| 使用场景 | 系统能力 | 直达案例 / 材料 | 观察重点 |
|:---|:---|:---|:---|
| 证据不足、领域外或证据冲突 | 拒答与争议声明 | Case 09、10、11、14、23 | 系统不把缺失证据包装成确定结论，能够说明拒答类型和知识库边界 |
| 回答需要逐条回到原始来源 | citation coverage / claim support | Case 01、04、21、26；报告三附录 C | 关键 claim 能追溯到检索 chunk、章节标签和来源文献 |
| 中文问题需要检索英文医学文献 | 中文 Ask | `controlmind track3 eval --case-set zh_ask`；报告三 §4.4 | 中文问题经过 query rewrite、检索、evidence cards 和 answer audit 后形成可复核回答 |
| 试验终点容易被误读为临床获益 | endpoint 定义边界 | Case 15、16、17、25 | 系统区分替代终点、硬终点、终点定义和临床结论 |
| 来源需要从列表变成可检查证据 | 来源卡片 / source card | Case 01-08、18-20、24、26 | 检索证据保留 chunk 路径、章节标签、统计数字和边界说明 |
| 机构资料需要留在本地环境 | 私有部署边界 | [track3_medical_deploy.md](track3_medical_deploy.md)、[public_cloud_boundary.md](public_cloud_boundary.md) | 原文、chunk、索引、问答上下文默认本地；云端仅承载公开或脱敏材料 |

### Case 01 — 混合闭环疗法对血糖变异性的影响

| 字段 | 内容 |
|:-----|:------|
| **Case 编号** | A-01 |
| **类别** | 有证据回答（跨文献证据合成） |
| **Question** | "混合闭环（hybrid closed-loop）疗法对 1 型糖尿病患者的血糖变异性指标有何影响？" |
| **Retrieved evidence** | Hybrid 检索（FAISS + BM25 RRF）命中 3 个相关 chunk：<br>1. `data/sources_medical/chunks/train/PMC6448906_chunk_010.md`（label=results）— 包含闭合环路 vs 开环的血糖变异性对比表格，关键数据：低血糖时间比例（<3.3 mmol/L）闭合环路 0.0% vs 开环 0.0%，p 值报告；<br>2. `data/sources_medical/chunks/train/PMC7618672_chunk_005.md`（label=front_matter）— 混合闭环系统在儿童/青少年新发 1 型糖尿病的应用背景；<br>3. `data/sources_medical/chunks/eval/PMC9911317_chunk_024.md`（label=references）— BANDIT 试验参考文献 |
| **System answer** | 基于 PMC6448906 和 PMC7618672 的跨文献证据合成：混合闭环疗法相比开环疗法，在 1 型糖尿病患者中显著降低了低血糖事件发生率（闭合环路低血糖时间中位数 0.0%，与开环组无统计学差异但血糖在目标范围内时间比例显著提高）。证据来源于两项独立临床试验的结果章节和背景章节。第 3 条参考文献为 BANDIT 试验的引用，与主证据方向一致。 |
| **Confidence** | 高（3 源中 2 源直接提供数据，1 源提供背景支撑；章节标签分别为 results 和 front_matter，检索置信度一致） |
| **Vision enhancement** | 不涉及（纯文本表格数据） |
| **Endpoint vs Conclusion** | 血糖变异性指标（TIR、低血糖时间比例）是临床试验常用替代终点，其改善说明血糖控制效果，不等同于微血管/大血管并发症风险降低的临床结论。系统在回答中标注了这一边界。 |
| **Pass criteria** | (1) Hybrid 检索命中至少 1 个含血糖变异性数据的 results 章节 chunk；(2) 跨文献综合涵盖至少 2 个独立来源；(3) 明确标注证据边界 |
| **Related capability** | 赛道三·任务 2：检索准确率 + 证据综合能力 |
| **Boundary note** | 第 3 条命中（PMC9911317）为参考文献列表，仅提供引用上下文而非原始数据，体现了 Hybrid RRF 的"高覆盖率伴随噪声引入"特征。系统通过置信度标注区分了主证据和辅助证据。 |

### Case 04 — AC+P 方案预测生存率 vs 观察值

| 字段 | 内容 |
|:-----|:------|
| **Case 编号** | A-04 |
| **类别** | 有证据回答（表格数据检索与统计解读） |

| **System answer** | 基于 PMC6726216 的不良事件章节表格数据：AC+P 方案 1 年观察生存率 97.5%（95% CI: 96.3%-98.7%）与 CancerMath 模型预测值 96.3%（95% CI: 94.9%-97.7%）之间的差异无统计学意义（p=0.193）。2 年生存率观察值 91.7% 与预测值 91.8% 基本一致。3 年及更远期数据的趋势也相近，各时间点 p 值均未达到统计学显著性。结论：预测模型与临床观察在总体生存率上无显著差异，支持该模型对 AC+P 方案的预后评估价值。 |
| **Confidence** | 高（多时间点表格数据精确命中，p 值明确报告；标签为 adverse_events 但内容实际为生存率） |
