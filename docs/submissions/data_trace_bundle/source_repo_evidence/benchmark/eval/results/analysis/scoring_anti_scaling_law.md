# 评分者反规模定律验证（Scoring Anti-Scaling Law）

> 分析日期：2026-05-09 | 样本量：80 题分层抽检 | 本地 Judge：6 模型


## 核心命题

> **评分者反规模定律（Scoring Anti-Scaling Law）**：在 LLM-as-Judge 场景中，
模型参数量与评分严格度呈反比——更大的模型倾向于给出更保守、更低的分值，
而非更宽松的评分。


该命题的直观证据：gemma3:4b (4.4 GB) 在 80 题中给出平均分 0.872，
而 qwen3.5:35b (23 GB) 仅给出 0.232。
但表面数值差异可能来自两种根本不同的机制：

1. **尺度差异（Scale Difference）**：所有 Judge 对题目相对排名的看法一致，只是打分的"松紧度"不同。
   → Spearman ρ 高而 Pearson r 低

2. **逻辑差异（Logic Difference）**：不同 Judge 对"什么算正确答案"有根本性分歧，排名不一致。
   → Spearman ρ 和 Pearson r 双低

**上三角均值**：Spearman ρ = 0.230，Pearson r = 0.230

**族内 vs 族外分析**：Qwen 家族内部（2B↔4B↔9B↔35B）的 Spearman ρ 较高（0.29~0.60），
表明同一架构家族的模型排名判断一致性强。然而跨家族配对（尤其是涉及 Llama3.2-3B）的 ρ 趋近于零甚至为负——
这与"评分严格度不同"的解释矛盾：不同架构的 Judge 对答案质量存在真实分歧。

⚠️ **结论**：Spearman ρ 与 Pearson r 接近（Δ < 0.01），不一致来自**逻辑分歧**而非单纯尺度差异。
"评分者反规模定律"的 L1（排名一致性）维度**不成立**——不同 Judge 确实对答案质量有根本性分歧，
尤其在跨架构的配对中。这反而是一个更重要的发现：LLM-as-Judge 的评分者间信度远低于预期。
反规模定律在 L2（零分率）维度仍成立。

## 综合结论


### 评分者反规模定律成立条件


1. **L1 验证**：Spearman ρ（0.230）≈ Pearson r（0.230），Δ < 0.01 → 不一致来自**逻辑分歧**，非单纯尺度差异 → ❌（但族内一致性尚可）

2. **L2 验证**：零分率与模型尺寸正相关 → 最大模型最严格 → ✅

3. **L3 验证**：各 Judge 与 API 共识的 Spearman ρ 分布揭示了谁更接近专家集体判断

4. **L4 验证**：存在领域特化偏差，但宽松/严格倾向跨领域一致


### 核心数据


| 指标 | 值 |
|:--|:--:|
| 6 Judge Spearman ρ（上三角均值） | 0.230 |
| 6 Judge Pearson r（上三角均值） | 0.230 |
| Δ = ρ - r | 0.0005 |
| 最宽松 Judge | gemma3:4b (0.872) |
| 最严格 Judge | qwen3.5:35b (0.232) |
| 尺寸比（最大/最小） | 35B / 4B = 8.75× |
| 评分比（最松/最严） | 3.75× |

### 对 LLM-as-Judge 实践的启示


1. **评分者间信度不足**：Spearman ρ 均值仅 0.230，6 个本地 Judge 的排名判断一致性很低。
   更准确地说：同一架构家族（Qwen）内部一致性尚可（ρ=0.29~0.60），但跨架构配对（尤其涉及 Llama）
   出现负相关——不同架构的 LLM 对"好答案"的标准存在根本分歧。
2. **零分率与模型尺寸强相关**：35B 的零分率（33.8%）是 4B（0~8.8%）的 4~8 倍，这个模式在
   同一架构家族（Qwen 内部：2B 3.8% → 35B 33.8%）内严格单调，暗示大模型对回答质量的标准更高。
3. **API 共识对齐度有限**：最佳本地 Judge（Qwen3.5-4B）与 API 8 Judge 共识的 ρ 仅 0.481——
   没有一个本地 Judge 能可靠地逼近 API 专家集的判断，多 Judge 融合是必要的。
4. **领域偏差较小**：所有 Judge 的跨领域方差均 < 0.01，说明评分倾向在学科间稳定，
   领域特化偏差不是影响评分差异的主要因素。
5. **评分离散度差异显著**：gemma3:4b 只用了 6 个离散分值（步长 0.05~0.1），
   而 qwen3.5:4b 用了 13 个唯一分值。粗粒度 Judge 的"宽松"部分源于
   其无法区分 0.6 和 0.65 之间的细微质量差异——这是量化策略的不同，不一定是"更宽容"。
6. **反规模定律的修正表述**：该定律在零分率维度成立（大模型更严格），但在排名一致性维度不成立——
   Judge 间的分歧源于真实判断差异，而非仅仅是评分尺度的不同。
