# Diagram Source Workspace

This directory stores editable Excalidraw source files for submission diagrams.

Report Markdown files should reference exported PNG files under:

```text
docs/submissions/shared/assets/task1/
docs/submissions/shared/assets/task2/
docs/submissions/shared/assets/task3/
```

Update flow:

1. Edit the `.excalidraw` source file in this directory.
2. Render or export the matching `.png`.
3. Place the exported image in the matching task asset directory.
